行走在羊狮慕，仿佛在梦境之中【首发】

游记 摄影 登山 自助游 自然奇观 行程规划 2018/11/11
【旅行碎碎念】
云海佛光寻常见，莫道仙境是人间。
从来只闻仙与佛，原来真容在此间。
不喜欢吟唱的我，在第一眼看到羊狮慕时变情不自禁的发出了这种感慨。
我无法写出李白那样浪漫的诗来形容它，
我亦无法写出温庭筠那样辞藻华丽的诗句来赞美它，
唯有一腔肺腑之言说与诸君听。
【预告片】
【景区简介以及行程路线】
羊狮慕（杨思慕）景区位于国家级风景名胜区、国家自然遗产地、国家森林公园、国家地质公园江西省安福县武功山境内西北部。景区以花岗岩峰林地貌为主，千姿百态，巍峨壮观。以“奇峰怪石、流泉飞瀑、山花争妍、云海幻境、雾凇飞雪”为主要特色。群峰叠嶂，苍翠欲滴，秀中藏秀，奇中出奇，佛光、云海、日出、晚霞，美不胜收，身临其境，如入仙境。
景区建有悬空栈道、山谷游步道、高山滑道、滑索、户外运动中心、五星级仿古度假酒店，仿古四合院等旅游设施，是集生态游览、休闲度假、户外运动、宗教朝圣为一体的山岳型旅游胜地。
景区内著名的 两‘峰’、五‘福’奇绝景观令人叹为观止，同时与中国传统的道家、佛家‘福’文化不谋而合，处处彰显人们对‘福’的追求，因而又有‘中国福山’之美誉。
武功奇绝独一处，风景尽在羊狮慕，景区总面积37.5平方公里，最高峰天子峰（又‘石笋峰’）海拔1764米，与江西省境内第一高峰武功山金顶一脉相承，号称‘姐妹峰’！

Day1：杭州-宜春-羊狮慕大峡谷观日落
如果有人问我最喜欢江南的哪一种景象，我会思考一下。然后认真的回答他，一定是江南的烟雨。因为这些烟雨拥有无限的柔情，而这柔情恰恰是人们赞美江南时最常用的词语。因为江南柔情满怀的雨我开始喜欢上了其他地方的雨，因为雨在我心中已经变成了柔情的象征。
下午初见羊狮慕，迎接我的竟然是我最喜欢的小雨。淅淅沥沥的小雨打湿了大地，却将我的心情点亮。在我最爱的天气中出行，我总是精神饱满。即使车马劳顿，我仍能借着雨的柔情一扫疲劳。
“羊狮慕”与“杨思慕”都是这里的名字，而二者之间却没有什么联系。“杨思慕”的由来与宋代大诗人杨万里的求学游历息息相关，相传他曾游历到此见山中奇石壮景而流连忘返，日夜思慕。是在观赏了鬼斧神工的石笋峰后，更是留下了憾世诗篇：“笔锋插霄汉，云气蘸锋芒。时时同挥洒，散作甘露香”的佳作，才有了“杨思慕”的美名；又因景区经年云雾蒸腾，常现‘羊’、‘狮’追逐嬉戏于山间的气象景观，而又有了‘羊狮慕’之名。
沿路而上，我们经过正在修建的建筑物顺利的到达了流云台。此处，全然不负它流云的名声。这里可以清晰的看到洁白的云层在山谷之间飘飘然的幻化身形，云层薄处，似仙女轻舞飘过山峰；云层厚处，则如白练一般缠绕在山峰之间。
小雨渐渐停止了它凌乱的脚步，太阳从云层身后探出了头，弯道光芒将天空照的白净如许。而黛绿色的山峰却与明亮的天空形成了鲜明的对比，这一明一暗的对比倒是给此处增添了许多的意境。
随着太阳不断显示它的威力，乌云被慢慢驱散，整个天空开始出现了蓝天白云。明媚的阳光带来的不仅仅是天清气朗，还有满山的绿意。伴随着绿意一起到来的还有赤橙黄绿青蓝紫的彩虹，远远望去，不知是架在了山上还是挂在了天上。
但无论此刻的景色多么美，都无法逃脱“夕阳无限好，只是近黄昏。”的现实规律。落日慢慢的收起了它的光，将身躯悄悄的藏在山后面。我想，明天的朝阳一定很美。
Day2：天子峰齐云台-谿谷道径-栈道（流云台-齐云台）
【天子峰齐云台】
为了验证我昨天赏日落的猜测，我专门起了个打早在山间转悠，只为了能与日出相遇。由于起的确实有点早，周围白茫茫的白雾将我紧紧包围，就像是仙雾缭绕一般。顺着登山梯，我拾阶而上。山中的空气格外新鲜，带着好心情仿佛没有任何疲倦，心情的愉悦使我一口气直登天子峰的齐云台。
山顶还是非常寒冷的，游客一定要记得保暖。这山顶与北方的严寒天气有一拼，嘴里哈出都是白气，再看看周围的白雾，仙雾缭绕也不过如此吧！虽然景色很美，但是伴随着美景而来的还有满头的雾凇，将头发全都打湿了。
白色的云翻滚着，犹如海中波浪一般，而我特别享受这种脚踩白云头顶青天的感觉。当我陶醉于天子峰的美景时，太阳从山后慢慢升起。这时只能通过无人机拍摄到，可惜我没有带，只能在朋友的手机屏幕上看到这别制的日出。
【谿谷道径】
谿谷道径尚未完全建成，也正是如此我才有幸能感受到羊狮慕的“野性”。观赏人工尚未雕琢的景点，我还是第一次，带着好奇与憧憬漫步在山间的小路上。
山中到处都是青草绿树，雾气腾腾更显朦胧美色。因为尚未经过人工打磨，这里落叶满地，青苔面遍布，充满了“野”味。脚踩在落叶上软软的很舒服，只是被雾打湿的落叶有时会留恋于脚间，太过缠绵使我有点不适应。
随着浓雾渐渐散去，一路上我看到了许多老树盘根的形象。四散开来的树根长满了青苔，我经过了解才知道，青苔不是随便长的。必须有非常好的环境才能长出青苔，氧含量必须非常高才行。我听人说这里的人生活在新鲜的富氧空气中，要是进了城就会出现缺氧的情况，非常不适应，不知道是真是假！
虽不是雨季，谿谷道径仍可以看到沿着小路流淌的溪水，一路欢声笑语向前流去。
这些溪水穿行与巨石峭壁之间，有时冲击出多多水花，有时又会在形成落差的地方形成小瀑布，犹如一条白练，洁白如雪，纯净美好。
游行于山间的小路，在落叶的衬托下充满了古韵，而最引起我注意的却是与溪水打闹的石块与随意生长的树木。这些石块都长有青苔，有些石块因为溪水的冲击而变得光滑没了青苔。而随意生长的树木更像是一件件艺术品，大自然真是充满了魔力。
谿谷道径上有一条瀑布，上半部沿着陡峭的石壁直冲下来。忽然遇到石块，这条瀑布瞬间四散开来，伴随着巨响坠入悬崖之中。
【栈道（流云台-齐云台）】
羊狮慕的栈道从流云开始，那个藏着羊狮慕云海的地方。从此地往右走是凌云台，那是看云海最好的视角。位于左边的天子峰是一个充满神秘与神奇的地方，在那里天然形成了时刻字。上面写着“天子”二字，并且还有一个繁体的“梦”字。
这还不是它最神奇的地方，因为这字原来是没有的，在中国提出建设我们的中国梦时，这几个字就不约而同的出现了。这些字的出现恰恰与中国梦的理念相符合，除了说是上天的安排，似乎其他一切的解释都是苍白无力的。
倚崖而建的栈道不仅是一条供游人行走的路，更是福山对人民传播福气的见证。相传当初修建它的工人之中有十几个人误食了毒蘑菇，全部治愈后又在工地上继续工作了。并有传闻说当工人被送进医院时，此地还出现了火凤凰式的云朵，在空中翱翔。
走在栈道上，脚下是升腾的云海，肆意流动，软若无骨，但仙气十足。正当我如痴如醉的贪恋着云海奇景时，太阳从云海中之下射出橘黄色的光，黄与白泾渭分明。还有赶来凑热闹的彩虹，赤橙黄绿青蓝紫把色彩拆开给游客看个透彻。
我从不觉的自己是一个与佛有缘的人，直到我在羊狮慕见到佛光的一刹那。我觉得自己仿佛瞬间皈依了佛门，如果你也想看佛光，羊狮慕绝对是你的最佳选择。
栈道上散落着秋季的黄叶，我突然想起北京金黄色的秋天，和这里一样让人心中陶醉不已。虽然有人说“这看起来有点萧瑟”，但我想到的只有“我言秋日胜春朝。”的美好。
在玻璃栈道上恰逢夕阳西下，霞光满天，此时的云海与夕阳构成了一幅意境深远的画卷。站在玻璃栈道上，看远处霞光与云海相接的地方。此时夕阳的黄色带来无尽的喜悦，变幻的云海静静流淌，一切都是最好的样子。
有人说，这是因为福山知道工人一定会安然无恙的，所以这是在为工人祈福。到福山祈福的人络绎不绝，福山也从不吝啬地将福带给千家万户。
天子峰过后是左将军右宰相，一文一武共同撑起属于他们的美景，而旁边诵经的老者不知道是否是在为他们加油！
当云只能停留在我们的脚底时，我们才会发现原来天并不是遥不可及。所以这也激励着许多人坚持着梦想，因为自己曾经是“腾云驾雾”的王者，只是暂时流落人间而已。
Day3：观日亭
虽然在天子峰齐云台已经看过日出了，但是面对拥有更为开阔视野的观云台我们还是无法忍住自己那颗热爱美景的心。
于是，我们在最后一天又起了一个早，不为别的，就只为那让人欲罢不能的日出。我们觉得自己起的很早了，但是一山更比一山高。等我们到达山顶时，观云台上已经有摄影协会的人在等待日出了。
站在观日台上，看着眼前浩瀚无垠的白色云海，我觉得比面对大海更加能让我打开心扉，或许是因为多了一份新鲜空气的原因。
我看着远处的云似乎未动，便觉得有点煞了我的风景。随着太阳渐渐升起，金色光芒像是万支利箭一样将云层花开，将金色的光芒照进我的眼中。
随着太阳越升越高，直到它完全超过了云海。我才发现这金红色的太阳是如此这般诱人，让我久久不愿离开。
【关于交通】
羊狮慕景区位于江西省吉安市安福县境内、与宜春明月山景区、安福武功山景区毗邻，总面积37.5平方公里，奇峰云瀑和云中草原可移步换景。各地可选择高铁到宜春或者是吉安，然后开车前往景区。
吉安路线：
1、从安福县城平都镇出发至泰山乡方竹坪（约1.5个小时），参观羊狮慕大峡谷（约3.5小时）到达风景区安平门游客服务中心。
2、车行路线：从安福县城平都镇出发至泰山乡羊狮慕风景区南福门游客服务中心（约50分钟）。
3、索道路线：从安福县城平都镇出发至位于泰山乡的羊狮慕景区南福门游客服务中心（约1.5小时）乘坐羊狮慕索道到达景区（单程约20分钟）。
宜春路线：
1、从宜春市出发到达明月山景区游客服务中心（约半小时），参观明月山核心景区（1.5个小时）高山小火车（单程20分钟）到达羊狮慕景区东安门游客服务中心。
2、萍乡路线从萍乡市区出发至槽下村（约1.5小时），登山至羊狮慕景区安平门游客服务中心（约1小时）。
