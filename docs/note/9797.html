【首发】永恒圣境，细数巴拉格宗的“七宗最”

游记 小众 摄影 登山 自助游 自然奇观 2019/01/15

【前言】

香格里拉在天堂
巴拉格宗在人间
天堂太远去不了
但我可以将巴拉格宗游个遍
 
曾经是地图上找不到的地方，如今逐渐出现在大众视野，即便开发仅仅不到十分之一，但足以让无数游客为之着迷。
 
香格里拉太美，而巴拉格宗在画中央，集自然、人文、历史、故事于一身，一个你深入了解会爱上的地方，永恒圣境巴拉格宗，今天来数一数你的“七宗最”。

【行程安排】

此次是从汕头飞香格里拉，因为汕头没有直达香格里拉的机票，所以只能在昆明转机，提前一个星期订票，价格是1200元左右，香格里拉有自己的机场，能直飞的城市有昆明、芒市、成都、重庆、黔江、西安、桂林、百色、北京、上海、杭州、广州、长沙、大连、长治、张家口、拉萨等。中转航班要注意中间转机的时间，尽量不要太短，因为如果前序航班延误，不是同家航空公司的，影响到下一班航班是没有赔付的。
这是香格里拉的景点分布图，香格里拉大峡谷巴拉格宗景区距离古城82公里，开车约一个半小时，可以自驾过去，不过相对比较考验驾驶技术。古城有专线车往返巴拉格宗景区，每天9:15分准时发车，旺季人多情况会适当延迟个10分钟左右，总体而言非常方便，耗时约一个半小时，如果当天回，专线车时间是下午5点左右。
这是巴拉格宗景区景点分布图，虽然这个景区看起来路线复杂，但玩起来其实很简单，主要景点有通天峡，大峡谷，巴拉村，玻璃栈道，香巴拉佛塔，景点之间都有观光车，常规路线一日游就可以了，但精华路线需要两日游。
二日精华游推荐路线：
DAY1：人马驿道——千年菩提树——大峡谷——通天峡——藏族歌舞表演——巴拉格宗藏式生态大酒店
DAY2：香巴拉佛塔——玻璃栈道——巴拉村——返程

【最具佛性的千年菩提：神奇的佛手】

大千世界无奇不有，不走出去你永远不知道这个世界还有很多神奇的事物。巴拉格宗入口的一棵菩提树，就是如此神奇般的存在，不仅在万丈悬崖生存了3000多年，还神奇般地长出了一只佛手，目前也是当地的佛树，每位游客过来都会来此朝拜一番。
香格里拉平均海拔3000多米，这也是目前我见过的，生长海拔最高的菩提树，因为高海拔，所以香格里拉的冬天非常寒冷，也不知道这个菩提是怎么年复一年，历尽磨难存活到今天的。
不仅存活在高海拔的地方，还专挑在最艰苦的悬崖绝壁下生长，而第一眼见到这棵菩提树，你可能会感慨，在如此恶劣的自然环境中，竟然还能生长得如此之茂盛，管口粗的竹杆，枝叶却爬满了整个岩壁，紧贴着岩石生长，看似柔软的菩提，却如刚石般坚硬，缝隙之间还能看到它结下的菩提果，这是多么强大的生命力呀。
而最令人吃惊的地方是，在菩提的树干，竟然还长出了一根佛手，这只佛手非常细长，非常像小孩五指般的小手，一端连着树干的腰部，一端紧紧抓着旁边的岩石，手指伸入岩石缝隙，牢牢的固定在那里，看到的游客无不感慨大自然的神奇和这千年菩提的佛性。
这棵千年菩提也是伴着当地村民从小长到大的神树，在当地人眼中，这颗菩提保佑了他们的和平与安宁，被当地人奉为神树，并用栏杆把树干围起来，担心游客进去不小心破坏它的佛手。
但即便如此，来参观的游客依旧络绎不绝，纷纷将自己身上的现金卷在枝叶上，或者是将身上唯一的黄色哈达批在树干上，把祝福送给这棵神树。整棵菩提树的枝叶趴在岩壁上，像一只长翅的菩提，所以当地人也把这颗菩提命名为：凤凰菩提。
“身是菩提树，心如明镜台，时时勤拂拭，勿使惹尘埃”，或许佛祖想传达给我这么一个道理，让我们每天保持警醒反思的心态，不被纷纷扰扰的外界所诱惑，不被名和利迷惑了双眼，不在世俗中迷失了自己。

【最险峻的山路：巴拉格宗的生命之路】

巴拉格宗，很多人对这个名字似乎不是很熟悉，确实是这样的，因为这个地方曾经在地图上是找不到的，直到2000年初，才被摄影师挖掘出来，而真正让普通游客接触到这一片圣域，是一条让无数公路专家望而退步的通天路。
这个大山深处，还藏有一个千年古村落，村落的村民从山中走出大山到县城需要整整七天七夜的时间，如果有人生病了，需要到县城里去看医生，而七天七夜的时间，让多少可能好转的疾病变成了绝症，很多人没走出大山，就已经病倒在半路上。
因此村中一个叫“斯那定珠”的少年决定去改变这一切，13岁那年起背井离乡，去一线城市打拼，积累下数千万的资产，和当初背井离乡被村民反对一样，他决定放弃自己所有的一切修一条通往家乡的路，也遭到了大家的反对。
但他不顾反对，孤身一人奋战，变卖了自己的火锅城和财产，借钱也要把这条路修好，终于历经十年的光阴，这条在大家看起来不可能修成的路竟然奇迹般地修成了，县城到巴拉格宗，也从原来的七天七夜时间，缩短到现在的1.5小时。
从此巴拉格宗有了自己的路，海拔1800米到4200米，中间由一条盘山公路连起来，拐弯即掉头，开车需要整整一个小时的时间，才能到海拔4200米的地方，这也是我经历过的连续弯卡最多的地方。
一路驶来，一路风景，历经一个多小时的颠簸，到达海拔4200米的高原，竟然有种热泪盈眶的感觉，神峰圣域，一辈子难忘的风景，就在眼前，触手可及。

【最古朴的村落：消失1300多年的巴拉村】

没有人能想到，在这里隐蔽的大山里，竟然还存有一个1300多年的古村落――巴拉村，这个与世隔绝了1300多年的古村落，直到2008年，路修通了，才点亮自己在地图上的坐标。
那么这个神秘的村落又是怎么来的呢？据说在1300多年前，巴塘地区一个的部落首领为了躲避无情的战争给族人带来的伤痛，带领自己的部下去寻找藏经中记载的香巴拉王国，他放下了一切，离开了自己打下的江山，来到了格宗雪山脚下，建立了自己的古村落——巴拉村，从此过上了与世无争，世外桃源般的生活。
如今的巴拉村依旧延续1300多年来的宁静古朴，周边是高山峡谷，抬头即可见到白雪皑皑的雪山，这里的村民勤劳而质朴，男耕女织，家家户户都有饲养牛羊，过着自给自足的幸福生活。
住的当然是藏式民居啦，这里的建筑都是土木结构，如果你细看，你会发现墙都是一块一块的大石头拼成的，而这种建筑风格，也是结合当地的气候来建，这样的建筑冬暖夏凉，抗震性能良好，屋顶都是斜坡式的，可以减少雨雪对屋顶的冲击力。屋内一般分三层，地下一层是饲养牛羊的，而第二层则是自己住，第三层用作于仓库。
在无人打扰的古村落，走进一家藏族民居，感受这里沉淀千年的历史底蕴，喝一口香醇的酥油茶，享受一番不被打扰的休闲时光。
住宿的话可以选择在山脚，也可以选择在海拔3000多米的巴拉村，但如果有机会，行程不赶，我推荐住在这半山腰的巴拉村里边，山里的日出日落，晨光夕照，与世隔绝的清幽宁静，以及沉淀在时光中的文化气息，都能在这里深刻的感受到。
对了，一定要吃一顿当地的藏式小火锅，纯正的牦牛，以及走在路上的藏香猪，都可以在这里品尝到。

【最神奇的山峰，山即是塔，塔即是山】

4000多米的高原上，因为交通不是很方便，所以能见到这座塔的游客非常非常少，也很少人知道有这么一座山峰的存在。在当地人眼中，这是他们心中熟悉的佛塔，因为整座山峰很像一座佛塔，所以当地人给这座山峰取了个形象名字：“香巴拉佛塔”。
很多人去巴拉格宗玩，只玩了一天，而一天的行程是见不到香巴拉佛塔的，只有两天的行程，或者是那些户外深度徒步者，才有机会见识到这座神山的庐山真面目，所以很多去香格里拉玩的人都不知道这座山峰，但这并不妨碍这座山峰在巴拉村人民心中，仅次于第一神山格宗雪山的崇高地位。
初次见到这座山峰，简直是大吃一惊，怎么会有长得如此像佛塔的山峰，一般的山峰都以土黄色为主，但这座山峰却是神奇的雪白色，因为身处海拔4000多米的高原，一开始还以为那是山峰上的雪，但定晴一看，竟然不是，山峰的颜色就是雪白色的，雪白让这座山峰看起来更为神圣。
隐蔽于群山之中的天然大佛塔，与香格里拉市第一高峰格宗雪山遥相呼应，身处海拔4000多米的高原，人迹罕至，若没有长途跋涉的朝圣之心，很难与这座佛塔相遇，即便开车上来，也要经历无数个九转十八弯，耗时1个多小时。
香巴拉佛塔在巴拉村人们心中有崇高无上的份量，相传有老喇嘛在此修炼成仙，最后化成一座大山，不过不管是神话传说也好，不管是大自然的鬼斧神工也有，时时刻刻都要对大自然报以敬畏的心态，人和自然和谐相处了，才得以可持续发展。
从穿短袖到披上大棉袄，也就一个小时的时间，海拔4200米的乃当牧场，温度仅有几度，阳光和煦，微风稍凉，第一次觉得我们离天堂是如此之近，仿佛掂起脚尖就可以触摸到天空，此时此刻，呼吸也变得有些急促，不知是不是这里的高海拔，亦或是内心的激动之情造成的。

【中国海拔最高的玻璃栈道】

玻璃栈道见过不少，但海拔最高的玻璃栈道还是第一次见，3000多米的垂直高度，抬头是神圣的山峰，脚踩的是万丈悬崖，感受的是心跳的声音。
透明的玻璃倒映的是最纯粹的蓝天白云，随手一拍，都是天空之镜的大片，人往玻璃一站，然后请开始你的表演。
底下是海拔1800米的漂流河道，从高处往下望，整个大峡谷尽收眼底，目前在建的还有滑索，没错，穿越大峡谷的滑索，想想都刺激。
换个长焦，感受下竖屏满屏的震撼，这条峡谷风光是可以徒步和漂流的，因为最近下雨比较多，河水有点浑浊，没雨的季节，是碧绿般的清澈。

【香格里拉市最高的山峰：格宗雪山】

海拔5400多米的格宗雪山，是香格里拉市最高的山峰，这也是当地的第一神山，目前没有攀登记录，也不允许外人去攀登。
常年云雾缭绕的格宗雪山，披着神秘的面纱展现在我们面前，偶尔一两阵风吹过，才能隐约的看到它雄伟的轮廓，据说有缘的人，才能见到格宗雪山的庐山真面目，但这样保持神秘的模样也挺好的，至少能勾起大众的好奇，勾起人们对它的向往。
对于我们来说这是向往，对于当地的人就是信仰，西藏就有不少人这么跪拜着去接近他们心中的信仰，修今生，渡来世，他们内心的世界，又是一个怎样的价值体系。
神峰圣域，高洁隽永，也不知集多少天地之灵气才形成今天的模样，人间最后的净土，也让其继续保持神秘吧，没人踏至的地方，或者有神仙居住吧。

【最壮观的峡谷奇观】

两岸四五千米的高峰，把河道挤压在1000多米的地方，纵横深切的大峡谷，像是上帝笔下的艺术作品，在人们心头挥之不去。
河流旁是藏式生态大酒店，是巴拉格宗为数不多的酒店之一，在大山的怀抱里睡觉还是感觉很不错的。而酒店的对面，就是人马驿道了，当年巴拉村人民走出大山就是走这样的道路，还是没有栏杆铁锁的，可想而知，当时的巴拉村人民有多么艰辛。
亲身走在峡谷的栈道中，有种“两岸猿声啼不住，轻舟已过万重山”的感觉，巴拉格宗好玩的地方在于，它是有海拔梯度的，从低到高，而又从高到低，可以感受四季的变化。
走完三公里的大峡谷，就是河流的尽头，这里也是漂流的起点，峡谷源头流出的水，如碧绿般的清澈，有水的地方就有灵性，原来巴拉格宗也有柔美的一面。
流水配高山，通天峡也是巴拉格宗的一个奇观，拾级而上，行走在悬崖峭壁之间，深入峡谷腹部，零距离感受大峡谷的壮美。
一撮清泉无疑是这峡谷画廊的点睛之笔，它柔化了峡谷的刚烈，也让静态的峡谷动了起来。

【后记】

至此，“七宗最”数落完毕，巴拉格宗的精彩，也远远不止这些，这是个年轻的地方，开发至今只有10岁，这也是个很有潜力的地方，我们能玩到的地方，也仅仅是巴拉格宗的十分之一，随着时间的推移，巴拉格宗会给我们一些怎样的惊喜呢？让我们拭目以待！
