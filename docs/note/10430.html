“屿”见厦门-周末小资游

游记 摄影 精品酒店 美食 自助游 2018/11/27
鼓浪屿、环岛路……这些和厦门紧紧联系在一起的词语总是让人能感受到浓浓的海风味，带着一丝清甜的咸。
之前数次去往永定土楼和三明，厦门总是作为一个中转之地匆匆而过，印象中只有机场行李提取处最显眼的“I 💖 XIAMEN”。
而这一次终于有时间能够静下心来好好享受一次厦门的周末。
坐上厦门宸洲洲际酒店的接机专车，从机场到酒店一路畅通，不过20分钟便到了洲际楼下，虽然此前看到的天气预报都是小到中雨，但也许是老天爷也有好客之心，天空竟然还微微泛出一抹天鹅蓝。​
一进酒店大堂，感觉就像是突然换了一个画风。
仿佛刚刚还在城市的水泥森林中穿行，突然就进入到一个海洋世界。
头顶的蓝鲸正保持着在深海前行的姿势，流线型的庞大身姿却给人灵活而光滑的感觉，再仔细看看大堂顶部，无数小黑点其实是汇聚的鱼群，暖色调仿佛是阳光穿过水面直射而下。
在非常有效率的check in之后，把行李交给前台的工作人员，两手空空（当然要带着房卡😆）的进入了电梯，我已经迫不及待想从高处看一看厦门的海了。
啊，忘了说了，这次入住的厦门宸洲洲际酒店就在厦门的东海岸，毗邻厦门国际会议中心，步行到海边也不过5分钟路程，是看日出的不二选择。​
天色还微微有些雨后的沉郁，不过海面已经透出了青蓝色，从窗口望去，海景一览无余。
而大部分时间都身处北京和杭州的我来说，这里的位置刚刚好：远离了遮挡视线的高架和喧嚣的十字路口，得到的是闲散和舒适。​
当然也少不了精致的迎宾甜品，而作为盘底的垫纸上也清晰印刷这关于厦门和鼓浪屿的介绍，让初次来访的客人也能感受被细致照顾的暖心感觉。​
一直很喜欢洲际酒店御用的Agraria柠檬马鞭草香味。
简单的洗了把脸，靠在窗前的沙发上，满眼都是厦门的海景，已经开始期待明天的鼓浪屿之行了。
有时候，一座陌生的城市会让你放松紧绷已久的神经，而这份陌生之下的心安，往往取决于你所感受到的贴心和细致。​
一直喜欢冷暖色调的搭配，喜欢摄影但又从来没有系统学习过课程的自己总是随心所欲，兴至而发，利用洗手台的镜子也能有不错的角度。
因为是早班飞机，几乎没来得及吃早饭，而位于1楼的餐厅已经过了供应早餐的时间，于是前往行政酒廊寻觅能填饱肚子的食物。
通常来说一家酒店的行政酒廊通常代表了酒店自身的格调。
轻奢和偏西式的风格往往意味着优雅，精致的点心和水果足以抚慰早起和旅途带来的躁动和心焦。​
反正今天天气并不算晴好，于是打定主意在酒店里四处探寻。
无意中还钻进了健身房，光线和环境都给人很舒适的感觉。
回到一楼大堂，入口右手边的小酒吧在色调上的搭配吸引了我，用手机试了几个角度，利用对称感和本身色彩的对比性，也是非常适合酒店宣传照的取景点。​
不知不觉就在酒店呆了一天，决定趁着夜色未深，去附近逛一逛。
虽然厦门宸洲洲际酒店地处静僻一隅，其实附近也配置了新开发的商业区和办公区，刚刚入夜的天空被多彩的霓虹照亮。​
这条路作为环岛路的一部分，车流量其实并不大，为了拍摄这一张长曝光，我和朋友硬生生等了十多分钟才等来一波不大的车流。​
回到酒店还不到9点，于是打算去6层的游泳池看看，已经入秋微凉，但没想到泳池旁还有一个室外bar&BBQ。
和朋友就势坐下，点了几串海鲜，海阔天空的聊了起来……​
离开时还是没忍住在泳池一端拍下了附近商业区的夜景，想必在盛夏的酷热里，感受泳池带来的清凉，看着眼前的五彩霓虹，也是一种享受。
内心已经打定主意要在明年再来一次。
带着这种期待，回到房间倒在舒适的大床上……玩手机玩到自然入睡，也是现在快节奏生活之中的一种奢侈啦。​
第二天一大早就被朋友叫醒，说买好了10点半去鼓浪屿的船票，看了看床头的电子钟：9点~这一觉睡得真是舒服，起床-吃饭-出发！​
从内厝澳码头登上鼓浪屿，赵小姐一路都寻找着自己的家业，终于不负众望找到了一家，赶紧打卡巡视~~​
可能是周末的原因，鼓浪屿主道路上的游客也一点不少，虽然没有机动车，但在一些主要的路口交汇处，也是近乎于水泄不通。
好在还有各种精致小店带来的精致和惊喜。​
这样的tiffany蓝搭配一个古风店名“苏小曼”，足以吸引到我的镜头了。​
似乎店主们都深谙“旅行的意义”对年轻人和游客们的吸引力，而明信片和盖章本则是来到鼓浪屿避不开的两种纪念方式。
虽然自从上次去了曼谷大皇宫，也常常被朋友盛赞为“拍出了无人的大皇宫”，但面对鼓浪屿的人潮，似乎也有些无能为力呀~​
想起自己曾经辞职旅居的日子，也遇到过旅行途中突然就留下开店的朋友，那样的勇气一直是自己所羡慕的，只是自己天性懒惰，总还是想走走看看，若是把心思放在经营上，又顿觉索然无趣了。
而这，也是朋友们至今聚会时总会问：“你怎么还在坚持上班？”的由来。​
岛上的建筑其实才是真正的历史遗产，那些文艺小店常换常新，而这些历史建筑则像原住民一样沉默看着更迭变换。
渐渐深入岛内，总是捡着偏僻小径钻，终于是偏离了人潮，找到了曾经在书中看到过的静谧鼓浪屿。​
悠悠闲逛，在鼓浪屿也花了接近5个小时，手里还不能免俗的提着一大袋网红牛轧糖。
朋友说正好有一艘游艇出海，要不要体验一下。
想想自己坐过近海日韩航线的海上邮轮，也感受过欧洲多瑙河的内河游轮，还没有体验过这样的私家游艇，于是欣然前往。
和游艇上的工作人员聊天时，得知这艘游艇目前是港口里最贵的一艘，价值上亿，通常租用一个小时也要近两万元。
嗯，我默默捂住了钱包~~~
上船之前需要换鞋并取下一些随身的尖锐物品，避免损伤船体表面，而我在换鞋的时候无意中看到后舱里还放着一台水上摩托艇，不过今天的天气和温度，应该是不会拿出来的啦。​
上艇之后，看到一层甲板已经摆放好了丰富的下午茶……
内心OS：啊啊啊啊啊，又是甜品，这个假期看来又要长胖了，要不晚上回酒店还是先去健身房吧。
二层的甲板上放着躺椅，抹上防晒霜或者戴好遮阳帽，在阳光晴好的日子里躺下感受海风的吹拂和海浪的韵律，当真是非常舒适的享受。​
听朋友说这次游艇体验就是宸洲洲际的业主提供的，如果入住的客人有需要，也可以提供出租服务。
我特意到驾驶舱看了看，和我在多瑙河内河游轮上看到的结构一模一样，而沙龙区的软装据说都是定制，这也是造价高的主要原因之一，比如游艇上的影音系统就是全套BOSE定制，沙龙区的家居也都是纯手工定制的Poltrona Frau家居。
听朋友说游艇自带的主人房和4间VIP客房也都是有独立卫浴、更衣室以及海景大窗的，突然有点后悔自己只顾着在沙龙区吃吃吃了~~~​
逛了整整一天，回到酒店也该好好享受一番了。
因为事先预定了酒店的润祥中餐厅包厢，因此等我们回到酒店的时候，包厢里已经摆好了餐具和冷盘。
作为厦门的地主，朋友早就帮我们点好了厦门的特色菜以及一些经典粤菜。​
说到这儿还这要插上一句，其实这里的自助餐厅不仅仅有特色海鲜，其实还提供一些特色菜品和表演。
比如我们到的当天就看到了自助餐厅户外区域正有一场印度厨师的飞饼表演，虽然看制作过程很想摊大饼，但是制作工艺和材质上还是有不小的区别。​
话说我们到宸洲洲际的时间还真是巧，一直到11月30日都恰好是感“蟹”有你的美食活动举办期间。
这期间一楼的丰咖啡全日制餐厅都会在晚餐时间提供自助晚餐和海鲜火锅，而现在正是各种螃蟹肥美的时节，光是看着都已经按捺不住自己的胃了。​
各种花壳蟹、三点蟹、梭子蟹以及阿拉斯加蟹腿都是任意取食的，当然也要基于不浪费的原则哟~~~​
当然也有咖喱蟹和葱香蟹的做法，如果你是个爱蟹之人，那么一定不能错过呀。​
不过要说这次印象最深的还要数这一道海盐焗鸡了，用粗海盐包住一整只鸡，倒上茅台酒并点燃，不停翻动，让茅台酒燃烧殆尽之后，挖出埋在粗盐里的鸡，那香味就已经是一种诱惑啦~~~​
这道菜里像金针菇一样的东西，你就更猜不到是什么做的啦~~~
没错，就是豆腐，这样的刀工，也是非常了得啊~~​
嗯嗯嗯，来到厦门，沙茶面自然是少不了的~~不过我能说这让我更加怀念重庆的小面了么​
说到最后，其实自己这么多年旅行的体验里，住宿真心是决定一次旅行品质的重要因素之一，毕竟想一想，至少一天8小时的睡眠时间总归是在酒店里度过的，这就占了一天的三分之一。
话说如果希望住宿的地点还是更加热闹、公共交通更加便捷，宸洲旗下还有一个子品牌：晒途酒店，也是不错的选择，应该也更受年轻人的欢迎。
毕竟由日本乃村工艺集团操刀设计，房间的采光良好，配色也很明快，其中亲子房和情侣房都相当出彩，要是遇到下雨天也没关系，酒店自己的公共区域还提供了苹果电脑以便娱乐和查询信息。
悄悄的说，酒店公共区域和特色房型也非常适合自拍哟，说不定发上抖音也能火一把呢~~​
