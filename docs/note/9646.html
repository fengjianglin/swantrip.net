游记 人文游 古镇 自然奇观 骑行 2019/01/27
常常有客人朋友问：
“什么季节来 沙溪 最好？”“ 沙溪 有什么好玩，好看的？”
面对这个问题我很难确切地回答。
一个人的审美观由他的成长环境、自身阅历、学识、性格爱好等影响，面对同样的人、物、景，不同的人自然感受是不一样的。于 沙溪 而言，春夏秋冬，所到之处皆是景，置身其中，关键在于你的心态，清空内心，敞开胸怀，静默观察，才能感知自然的美好。所以面对 沙溪 游玩推荐，我只能跟你分享下，我眼中看到的，至于其他的，需要你自己开启发现的眼睛。

一年四季 皆可赏玩

一、赏建筑：

【古建筑】欧阳大院、 (古戏台) 、老马店、古街、古寨门
欧阳大院
拍摄者：岑伟健
欧阳大院：  ( 沙溪 古镇) 作为滇藏茶马古道上的重要驿站，有无数马店分布在以四方街为中心的各个巷道中，其特点大都是前铺后院，既可以经商，又可以住宿。对于作为马店的古建筑来说，只有回应了历史，并能再现和反映当时现实生活状况的，才是最有生命力的建筑。饱经沧桑、依旧古朴中透着华丽，建于清末至民国初年的百年马店欧阳大院，就是(寺登街) 上一座具有顽强生命力，并能回应茶马古道历史的老马店。
沙溪寺登街古戏台
拍摄者：岑伟健
 (古戏台) ：  (寺登街) 的灵魂。村村有魁阁、寨寨有戏台是 沙溪 一道亮丽的风景线。从进入 沙溪 坝子的第一个村子甸头村起，19个戏台、魁阁星罗棋布地点缀在 沙溪 坝子中间。一个个戏台。一个个魁阁像一位位历经沧桑的老人，给人们讲述着每一个村寨古老而真实的故事。它们既是茶马古道兴衰的见证者，也是今天人们了解白族传统文化与建筑艺术难得的实物资料。 沙溪 19座魁阁、戏 台中 至今保存完好的有11座，其中以 (寺登街) 、段家登魁阁带戏台最具代表性。
沙溪兴教寺
拍摄者：岑伟健
【古寺】兴教寺
兴教寺：古朴典雅，清幽宜人，建于永乐十三年（1415年），已有近600年历史。兴教寺是(寺登街) 上现存最古老的建筑， (寺登街) 的“寺”指的就是“兴教寺”。(寺登街) 就是以“兴教寺”为中心而建盖，兴教寺是 (寺登街) 的核心。
玉津桥
拍摄者：苏金泉
【古桥】  (玉津桥) 
(玉津桥) ：石可成桥，从今不唱公无渡；津真是玉，到此方知水有源。这是 (玉津桥) 的真实写照。作为茶马古道重镇 沙溪 ，桥是沟通两岸居民与外界的中介，20公里的河道上曾有桥近20座。 (玉津桥) 为单孔石桥，长35.4米，宽5米，净夸12米，高6米，始建年代不可考，明代已有木板桥，乾隆年间改石桥。
【其他】
 (寺登街) ：为茶马古道要冲。2001年10月底，与 中国 长城、 陕西 大秦宝塔和修道院、 上海 欧黑尔、雪切尔犹太教堂并列“2002年世界纪念性建筑遗产保护名录”。
沙溪
拍摄者：不详
城隍庙：在鳌峰 山南 ，距(寺登街) 约800米。其建于清光绪十六年（1890年），为一进两院带花园的寺庙建筑群，有大殿、戏台、照壁、厢房和花园等组成。里边有大殿三间，两边陋角各六间，两边耳房各四间，戏台代大门三间，戏 台东 伙房两间，外院两边各平房六间，伙房东厮房四间，共四十四间。大殿内正中泥塑城隍一尊，木雕城隍一尊。木雕一尊供三月初五城隍盛会，初四迎出，会后送进大殿内。大殿两边泥塑文武两将。东陋角金睡佛一尊，金观音两尊，银观音三尊。西陋角送子观音一堂。抗日战争时期烈士灵碑。之后张策在料总时失火烧毁 (古戏台) ，后由粮管所改建成现状。城隍庙边上有两百多株松柏种于1934年，距今已近80年。每年的农历三月初三到初五， 沙溪 坝子里各村的村民要轮流主办城隍庙会。
白龙潭
拍摄者：李雷明
白龙潭：位于 沙溪 坝子南端 红星 村附近，距 (寺登街) 6公里，历史上 沙溪 四景之一的“玉塔捍门”就在这里。碧绿的龙潭中泉眼喷珠吐玉，水花串串，翠峰古木倒映水中，妙趣横生。龙潭清澈见底，泉水凛冽甘甜。晴天之日，骄阳从浓郁中筛下斑驳的光点，龙潭波光粼粼，像闪烁着金光，其景美不胜收。
古镇晨曦： 沙溪 是茶马古道上唯一幸存的集市，其中心(寺登街) 与万里长城、 意大利 庞贝 古城、 埃及 国王谷、 美国 圣托马斯 教堂被列为世界濒危建筑遗产；完整保留了茶马古道传统的山乡古集市风貌；阿吒力佛教文化、儒家文化和白族民间乡土文化沿袭至今。清晨，对面晨曦，雾霭升腾，仿佛 禾木 。
沙溪
拍摄者：苏金泉
黑潓江： 沙溪 山水环绕、如诗如画，是具有农业古国田园风光的坝子；徐霞客赞誉为“鱼米之乡”。源于 (剑湖) 的黑潓江汇合 桃源 河后出峡进 沙溪 ，纵贯整个坝子，蜿蜒穿行、缓缓南去。白族村落散居两岸，各色杨树矗立河边，处处皆景。
沙溪
拍摄者：苏金泉
 (剑湖) ： 云南 重要的高原湿地之一。(剑湖) 之景物，四时各有特色。 阳春 三月，河堤上莺啼绿柳，湖边芦苇一片青翠，湖面上菱花白得耀眼。更有沿湖田野中蚕豆花、菜子花香气袭人，令人乐也陶陶。炎夏时节，  (剑湖) 东边山脚下果园中的桃李熟得正好，果农们载来了一船船香甜酥脆的五月桃和血红的李子，载来了满船的歌声笑声。此时游览(剑湖) ，既可以到中流击水，也可大饱口福。金秋九月，远山苍苍，湖水盈盈，为一年中湖面最开阔的时光。是时湖边沟江汊里的泥鳅正肥，此时如有兴致在湖边烧一锅新米饭，煮一锅泥鳅豆腐汤，亦是一种难得的美味。十冬腊月的 (剑湖) ，尽管湖边的树木大多已经裸露出了光秃的枝干．但气氛并不冷清。在波光粼粼的湖面上，有众多的鹭鸶和水鸟或遨游于天上，或嬉戏于湖中，特别是嘴馋而大胆的野鸭，较易让人捕获．此时游 (剑湖) ，感受与收获又是不同。
马坪关：位于 沙溪 西部群山中，是一个约1.5平方公里、为群山环抱的圆形谷地。东距 沙溪 15公里，西距傍弥潜井15公里，是古镇四卡中唯一保存完好的关卡。至今留有守关人修建的古建筑（戏台、魁星阁、(本主庙) 、智慧庵、关风桥）和依稀可见的古道遗址。如果您体力充沛而且时间充裕，徒步上马坪关将是您亲身感受茶马古道的最佳选择。
 ( 剑川 ) 古城：始于元代至正末年（公元1341—1370年），已有650多年历史，原有古城墙巍峨壮观，古城墙、樵楼1952年拆除。现四门护城河、壕桥犹存，古城墙基础四至清晰，明代建成的街巷道路走向不变，尺度不变，格局不变，历经沧桑，古貌依旧。古城的民居极富特色，尚保留明代建筑40余处，其中有21个院落保存完整，尚保留清代建筑146处，其余为民国至20世纪60年代土木结构建筑，古民居建筑的保留量占全城民居总数的90%以上。

2、欣赏当地白族剧团歌舞戏乐表演
沙溪
拍摄者：不详
【歌】白族调、白曲弹唱
在 大理 ，白族人依然有对唱的传统，尤其在 ( 剑川 ) 一带，唱的白族调最为正宗。  ( 剑川 ) 白族调白曲弹唱具有广泛的群众性，它来源于群众的生产生活，民情民俗。为白族群众所创作，有着极其深厚的群众基础，它的表演形式方便灵活。其格调一般为“七七七五”的山花体，曲体有长曲、短曲，长曲为本子曲，短曲一般四至八句为一首。白曲体裁有：情歌、风俗歌、劳动歌、一字歌、反意歌、咏物歌、寓意歌，生活歌、儿歌等。演唱者依情、依景、依时、依人现编现唱的一种曲调，歌词贴近日常生活，考验歌者的急智。
著名有大本曲，如《黄氏女对金刚经》，民间故事，如《鸿雁带书》，情歌对唱，如《花配栁》等。一年一度的 ( 剑川 )  (石 宝山 ) 歌会，会有万人约唱白族调的盛大场面。相传在(石 宝山 ) 上，正值花季的少男少女，在山涧相互对唱，若对得上的话，便会凭歌相会，成为恋人。
【舞】霸王鞭
关于霸王鞭的传说：传说三国时期，诸葛亮与司马懿率军相战，诸葛亮一方因寡不敌众被围困于山头，夜晚，只听得四周喊杀震天，情况已万分危急，诸葛亮急中生智，令将士们燃起大堆篝火，又在钢鞭、长矛上拴上铜铃，围着篝火不断跳跃，生性多疑的司马懿突见山头火光冲天，而金属撞击声不绝于耳，以为诸葛亮正在调兵遣将，置下圈套让他往里钻，急忙下令撤兵。诸葛亮南征至滇中以后，这招退兵之计广为流传，于是当地白族都效仿打起霸王鞭，以示钦佩之情。
霸王鞭舞起源于石龙村，石龙霸王鞭舞以其独特的表演形式和艺术风格独树一帜。一根长约二尺四寸的竹棍，上开五个小槽，每个槽内装有五枚铜钱，鞭杆两端扎有红绸花，尾部系两个小铜铃，轻轻一晃就叮当作响。据说竹鞭的长度代表二十四节气，五槽分指“东西南北中”五方，且与“青赤白黑黄”五色，“金木水火土”五行相对应。目前保存有三套：观音扫地。双飞蝴蝶。童子拜佛。其动作有双肩奉送、八步梅花、左右插花、八面威风、左右抬驾、仿莲花、蜻蜓点水等二十多个动作，融动作观赏性和健身性为一体。
沙溪
拍摄者：不详
【乐】白族洞经古乐
沙溪 古乐由4类曲种组成。一是古曲，有《浪淘沙》、《南清宫》等，其中一些古曲属早已失传的唐宋词谱，如《浪淘沙》即由吕洞宾所创。二是洞经古乐，盛行于清代康熙时期，延续至今、演奏洞经古乐时分文班和武班。文班乃洞经会，武班乃古乐队。文班乐器分为大乐、细乐。大乐为唢呐，细乐为竹笛、大胡、二胡、三弦、哑胡、板胡等。武班乐器油提手、小板鼓、大鼓、大锣、金杈、木鱼、鸳鸯锣等。石龙洞经古乐的特点是：武班给文班作伴奏，两者相互协助，不可分割。
著名乐曲有《嫦娥歌》、《醉杨妃》、《八卦腔》等30多首，其中《八卦腔》据考与唐玄宗开元二十九年（公元741年）所创的舞曲《紫微八卦舞》关系密切。过去，洞经会遍及全县各地，参加者都为会演奏乐器的文人学士及民间艺人。三是阿吒力佛腔，有《大佛腔》、《众等皈依》、《香赞》等100多首。四是道腔，有《长梵白鹤》、《集福迎祥》、《五召请》等40多首，其曲调自成一格，有浓郁的地方特色。在 大理 州乃至全省，能较为全面掌握阿吒力佛腔、道腔及其科仪表演的仅有 ( 剑川 ) 的一些老艺人。
沙溪

亮点二：太子会

“二月八”是白族传统重大节庆之一，时值农历二月初八。因县内所在地区及宗教信仰不同，赋予这个节日不同的内容。其一，佛教徒以此为太子释迦牟尼初游四门紀念日。传说释迦牟尼生前为毗罗卫国净饭王的太子，于此日出皇宫到民间察看，见到人世间生、老、病、死等种种苦厄，便于当晚舍弃皇宫生活和太子位，瑜城出家修行，寻求解除百姓种种苦难的道路，最后创立了佛教。这一天，广大佛教信徒聚集在县城 金华 镇和 沙溪 镇(寺登街) ，举行盛大的纪念活动。抬着释迦牟尼太子雕像，一路钟鼓齐鸣，焚香诵经，浩浩荡荡，漫游“四门”，常常聚集万人。村妪老妇牵儿带孙参加游行，并把自家的孩子装扮成“太子”以沾“佛”气。其二，山区半山区，特别是养羊作为主要副业的地区，称二月八为“羊神节”这一天要做“羊子会”，养羊户要以三牲酒礼祭祀羊神，祈求象征财富的羊群兴旺。第三，一般农村视二月八为“蔬果节”，传说这一天在菜园里种瓜种豆不会有虫害。

亮点二：宴遇菌子季

【最佳时节：7月、8月、9月】
一眨眼，又到菌子季。 云南 得天独厚的气候，孕育了野生食用菌二百多种，野生菌味美鲜并富含多种维生素、蛋白质，营养丰富。7月至9月来 沙溪 ，餐桌上怎么能少得了这些“山珍”。
绝对不容错过的当地特色菌子：
（1）松茸
松茸是纯天然的珍稀名贵食用菌，无法人工培植，被誉为"菌中之王"。
松茸有一种独特的香气，入口香脆爽滑，鲜香诱人。
松茸的吃法有很多，时下 沙溪 一种 新兴 的吃法是松茸和桑拿鸡的组合。桑拿鸡是一种利用蒸汽循环原理的创意食法。松茸桑拿鸡，汤底以松茸搭配其他的食材调制，开锅沸腾时，汤底中的松茸香气渗入到鸡肉里。而腌制过的鸡肉中的精华则通过气孔留入汤底中，使得汤、松茸、鸡肉三者巧妙地融合，最终汤鲜肉嫩，美味可口。

（2）牛肝菌
牛肝菌因其肉质肥厚，形似牛肝而得名。该菌菌体较大，肉肥厚，柄粗壮，食味香甜可口，营养丰富，是一种世界性著名食用菌。

鸡枞菌肉厚肥硕，质细丝白，味道鲜甜香脆。含人体所必须的氨基酸、蛋白质、脂肪，还含有各种维生素和钙、磷、核黄酸等物质。鸡枞的吃法很多，可以单料为菜，还能与蔬菜、鱼肉及各种山珍海味搭配，可无论炒、炸、腌、煎、拌、烩、烤、焖，清蒸或做汤，其滋味都很鲜，为菌中之冠。

亮点三：火把节【农历6月25日】

火把节白语称“富物”（意即六月之皇）、“薪毁架”（ 薪火节），汉语书写为“星回节”通称“火把节”，是白族火崇拜的节日，时在农历六月二十五日。
白族古有火崇拜习俗，这一天，各村由年内新添子嗣之家作“会首”，在本村坝中竖一高大的火把，常年扎12台，闰年扎13台，顶端扎升斗旗帜，上书“风调雨顺”、“五谷丰登”字样，下悬花红、梨子、铜钱等物。天擦黑点燃火把，会首们以糖果、炒蚕豆、烟、荼、酒招待相聚在火把下场坝中的本村老幼。儿童则一手持小火把，一手往火把上泼洒松脂粉末相戏，以示对新生的婴儿致贺礼赞。此日，新亡故者之家人须上坟祭祀，在新故者坟前或大门口竖一小火把，如果是3年，表示对死者的缅怀。火把节是白族对“生之礼赞和死之缅怀”的节日，与史书“星回日祭祖”的记载相合。
关于火把节的传说较多，有纪念汉代酋长曼阿南之妻阿南夫人殉夫之事；有三国时群众欢送诸葛亮南征得胜回 成都 之事；有南诏王皮罗阁“火烧松明楼”柏节夫人殉节之事；还有纪念五谷之神“达拨”，祈求五谷丰登，六畜兴旺之事。

亮点四：石宝山歌会

【农历7月27日至29日】
石 宝山 歌会素有“白族情人节”和“ 中国 歌城”之称，是一个让人不思归的盛大民族聚会。石 宝山 传统白曲已列入国际民歌歌目，2008年，石 宝山 歌会被国务院列入国家级非物质文化遗产保护名录。
石 宝山 歌会是滇 西北 地区以白族为主的各少数民族青年男女，用歌声来表达爱情和寻找爱情的盛大聚会。石 宝山 歌会也称“朝山歌会”，在每年硕果累累、稻谷飘香的农历七月二十七至八月初一暨时在石宝山 上举行。歌会期间来自 剑川 、 洱源 、 鹤庆 、 兰坪 、 云龙 、 大理 、 丽江 等三州八县的各族群众身着盛装，身挂龙头三弦，纷至沓来。方圆20多平方公里的石 宝山 山间的小道上，人流如潮，宛如一条条五彩斑斓的长龙。参加歌会的不仅有青年男女，也有中年妇女、壮年男子，还有老翁、老妪，但以青年居多。有的是去年歌会相识，今年来相会；有的相知相爱，但未结成良缘的，借歌会互相诉说相思之情；而更多的青年男女则是用歌声来寻找爱情。
歌会期间，石 宝山 宝相寺、石钟寺、 狮子 关、绿荫旁、小河边都能看到盛装的青年男女用歌声相互吐露芳心。小伙子们弹起明快的龙头三弦，姑娘们拉开明亮的金嗓子，歌声弦声此起彼落，令人陶醉。歌声拉近了一对对青年男女的距离，青年男女在充满爱情和欢乐的聚会中从素不相识到相识相知和相爱，直到成为情侣，石 宝山 山谷成了多情之谷。
石 宝山 歌会是情歌的大荟萃。夜幕降临，堆堆篝火映红半边天，四面八方闪烁的火把与电筒光柱像流星似的在山林中闯荡，耸岩下、山泉旁、树阴下、花丛中、竹林间堆堆篝火旁，男女青年用歌声交流火一样炽热的情感，赞美蜜一般甜美的生活。到了深夜，弦歌鼎沸，纵歌纵情，通宵达旦，三天三夜。石宝山 化作了歌的海洋，有人说要问石 宝山 上有多少歌手、多少情歌、多少情侣，那你先数数天上有多少星星。石 宝山 上的歌手、情歌、情侣，像天上的星星，数也数不清，唱也唱不完，不愧有“白族情人节”和“ 中国 歌城”之称。
石 宝山 上对歌，不但唱情，也比智慧。要求即兴创作演唱，一问一答，不能停断，谁接不上唱词或所唱词不合韵，就算败了。情歌有表达难以尽意的时候，高明的歌手对唱几天几夜也难分高下。许多质朴真挚源远流长的石 宝山 传统民歌，是白族人民口头创作的艺术结晶，是反映白族生活和审美特征最为典型的部分。它透过青年男女对美好生活的向往，对忠贞爱情的追求，反映社会生活的方方面面，在一定程度上揭示了生活的本质，体现了本民族特有的审美趣味，具有较高的艺术价值，是珍贵的民间非物质文化遗产。在白族地区的日常生活中，在田间旷野、花前月下，伴随着劳动生产或休闲时光，随时都可以听到情歌对唱。在白族聚居地区，歌是盐巴，歌是空气，歌是水，天天少不了，没有够的时候。
石 宝山 歌会从石钟寺石刻造像旁许多元明时期游人题记和明代赵炳龙题石 宝山 楹联中“弦歌犹俚俗；亭障亦边关”可知，白族先民到石 宝山 上拜佛进香、求生求育、弹弦对歌的传统具有悠久的历史。清代《丽郡文征》载 剑川 文人赵怀礼《朝山曲》生动地描述了当时朝山歌会的情景，其中有两首写道：
香烟喷作雾迷漫，沏夜僧寮不掩关；
明月乍来还乍去，可怜佳节于名山。
三营浪子土三弦，靡曼山歌断复连；
菩萨 低眉 弥勒 笑，无遮大会奈何天。
可见石宝“名山”赛歌“佳节”在明清时代已形成规模。青年男女用“土三弦”伴奏的“靡曼山歌”已“彻夜”对唱，连寺庙“僧寮”已无法“掩关”闭户，而面对这样的盛会“ 菩萨 ”也只能“低眉”，“ 弥勒 ”也只能“微笑”，可见是一个“神人共欢的盛会”。
从流传历史来看，歌会的起源，应该是白族先民早期朝山进香并祈求生殖生育、风调雨顺等祭祀活动的庙会或香会。歌会与古代周人之“社祭”、宋人之“桑人之祭”、楚人之“ 云梦 ”······男男女女一同相聚祭祀，祈求人口增长、风调雨顺、粮食丰收的“山林野合”遗风相似。

三、秋 绵绵秋意，硕果累累

【9月10月11月】 
秋天款款而来，村民已然开始收割稻谷。
秋高气爽，是最舒服的季节；因为，天总是很蓝，整个村落色彩斑斓得那么自然，守在窗前看远山，牛羊漫步在草甸和山坡，那就是一多彩的幅油 。

亮点一： 徒步季

【最佳时节：10月下旬-11月】
秋季是适合徒步接触大自然、深入村落的好时机。
华丛山三大潭徒步： 沙溪 海拔仅有1900米，沿村进山的道路前行，跟随古老的马道，穿过杜鹃山林，平坦的土路两旁是高大浓密的松树、杉树和栎树，最重要的是枝头都挂着松萝，俗称“树 胡子”，它对环境的要求很高，空气中有一点点污染就不能存活，因此是最好的环境检测器。沿着山花前行后，面对草甸、大潭享受一份宁静而又丰盛的野餐无疑是最浪漫和惬意的旅行体验。

四、冬 温馨暖冬，喜气洋洋

【12月1月2月】（避寒）
冬季做客，是 沙溪 人生活中重要的一部分，主人家办事请客，村里的亲朋好友就去做客。所谓办事包含的内容很多，结婚、订婚、生小孩、小孩满月、家里盖房、上梁、落成、办丧事等等都要请客，而且不是请一顿，是一请就要三到五天，大宴宾客。饭桌上有鱼有肉，有白族特色的八大碗，请得也不是十几二十人，几乎全村有点关系的都要请，吃流水席。如果你在街上看到有当街搭灶台烧火做饭的，千万别惊讶，只要你驻足拍照或者询问，人家一定盛情邀请你吃的！

亮点一 ：剑湖观鸟

【最佳时节：11月-次年4月】
冬季，从来都是剑湖最热闹的季节。
每年的11月至来年4月，
可在此观赏鸟儿与自然风景形成一派和谐、动静皆宜的景致。

亮点二：杀猪季节

【最佳时节： 12月中上旬至1月中旬】
到年底， 沙溪 的村庄中，结束了一年的农忙。婚礼、树房等民俗活动集中在年底陆续举行。
杀猪、腌制腊肉火腿、吃杀猪饭，这些丰富的民俗成为一年中 沙溪 最热闹的季节。
沙溪
沙溪
沙溪
【杀猪菜 】
春节前，家家户户都会“杀年猪”。杀完的猪，根据不同的部位做成不同的食物：肥肉用来榨油，猪腿用来腌制火腿，猪肉猪排可腌制腊肉，大肠小肠则可以用来制作血肠。
中午，全家以及亲朋好友，会围坐在桌前吃烤肉，现杀的新鲜猪肉，烤起来，香喷喷，外酥里嫩，再配着当地的啤酒、大麦酒，大口喝酒大口吃肉，日子简单又喜庆。完了再一起吃杀猪菜。饭菜丰盛，但都是充满当地特色的家常小菜，吃的是一份呢乡愁。

【腌制腊肉火腿 】
在杀猪当天，同时会完成腊肉火腿的腌制。在新鲜的猪腿、猪排、猪头上撒上盐、五香粉等，反复涂抹，之后将其挂起晾晒，水分晒干后放置阴凉处保存，一般放置一年左右味道最佳。
【灌血肠 】
灌血肠。猪大肠和小肠均可用作肠皮，糯米饭蒸熟，放盐，倒入热猪油和凝固的猪血后搅拌。使用竹制工具将肠皮撑开，将血糯米灌入其中，灌好后直接放入蒸锅。蒸好的血肠可以直接吃，也可以煎、烤等。

亮点三：白族婚礼

【婚礼密集期：12月-次年2月】
现在年轻人结婚也都是新式了，如果你在街上看到有接亲的队伍，一定要跟着看一路，人家会不断地往你手里发糖、发烟。新郎从娘家把亲娘子接出来，两个人一路走到新郎家，送亲的人就抬着嫁妆跟着走一路，不仅有被褥衣服，还有柴米油盐，还有家具电器，背冰箱的，沙发的，吹吹打打，甚是热闹。闹洞房甚至从接亲就开始了，接亲的人走一段路，就让新人表演各种，吃苹果、吃糖块等等，各种花招。。。。最后一直走到新郎家，拜父母，吃大餐。
