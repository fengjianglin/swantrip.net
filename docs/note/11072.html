【天鹅之旅首发】我在外伶仃岛的慢时光，回荡着美食与美景的交响乐

游记 情侣游 摄影 海岛 美食 邮轮旅游 2018/10/13

前言

期待一个被海浪声叫醒的清晨，可以咪着惺忪的眼睛，推开窗，呼吸着那夹带海腥味的空气，感受秋日阳光的温暖，对着那一望无际的大海发呆......或许，这就是一个久居城市的人，对海岛生活的渴望。
在广东，找一处海岛度假并不难，特别是有着“百岛之市”别称的珠海市，是珠三角中海洋面积最大、岛屿最多的城市，当中最著名的海岛要算外伶仃岛，它是中国广东省万山群岛的岛屿之一，与香港本岛只有一海之隔。恰逢岛上正在举办第五届海馨海鲜美食节，请跟随我的镜头，一起去与美食、美景约会吧。

行前准备

1、由于需要坐船到达，容易晕船的游客请提前服用晕船药，上船之前请尽量避免饱腹。
2：在感受阳光与海滩之时，别忘记涂一层防晒，带一顶帽子。
3：泳衣是必备的，除此以外，请多带几套更换的衣服，一些水上活动如香蕉船、摩托艇等都会弄湿衣服的。酒店提供的是一次性拖鞋，不适宜在沙滩上走，可以的话再带上一对沙滩鞋吧。
4：由于行船过程中距离港澳较近，偶尔会切换到其网络，建议提前关闭手机漫游，以免产生额外流量。
5：一定要带上证件，登船前边防会核对船票和身份证。

行程开支

来回船票都已经两百多了，还没算岛上的吃喝玩乐，建议购买海鲜美食嘉年华套票，入住岛上最好的酒店——悠活度假酒店，享用全岛最丰盛的海鲜晚宴，具体信息如下：
A 套餐 998元/人
含珠海或深圳至外伶仃来回船票； 含入住悠活渡假酒店（准五）豪华园景房每床位（原价2580元/间）；含酒店早餐、烧烤套餐、豪华海鲜晚餐、免费任饮酒店原装进口红酒、免费享用酒店私家海水泳池及海水温泉泡池。
B套餐 798元/人
含珠海或深圳至外伶仃来回船票；含入住海馨酒店（准四）每床位（原价1180元/间）；含酒店早餐、烧烤套餐、豪华海鲜晚餐、免费任饮酒店原装进口红酒、免费享用酒店私家海水泳池及海水温泉泡池。
Ps:搜索“香洲港”官微可预定哦！

关于酒店

珠海外伶仃海馨·悠活渡假酒店，位于外伶仃岛清风径2号，距离海边20米，步行至码头只需1分钟，到美食街也是3分钟以内，闹中取静，由于地势较高，客房朝海，可以看到不错的海景。
据当地人介绍，悠活度假酒店是外伶仃岛上设备设施最好的酒店，配备了高档会议室、西餐厅等，散步回来后，还能在立体户外海水泳池或温泉池上洗去一天的烦恼。
酒店溶入了浓郁的海岛风情，在露台便能直接纵览海面上过往的船只，颇有草船借箭的感觉。
外伶仃岛里不伶仃，享受阳光倾泻入房间的暖意，走出阳台，感受微风观赏落日。余晖尽散，在阳台上听着隐隐约约的海浪声，与朋友品尝杯红酒，谈天说地，尽情地享受独属于你的悠闲时光，甚是惬意。

香洲港

香洲港位于珠海市东海岸港口路，毗邻情侣中路，拥有秀丽的自然景色，是珠海市连接近邻海岛的重要港口客运站，每日有固定航班前往：桂山岛、东澳岛、外伶仃岛、万山岛。从香洲港前往外伶仃岛，每天有4班船，乘坐高速客轮船程约1小时左右，往返成人票价240元， 60岁以上长者和1.2—1.5米儿童半价120元，1.2米以下儿童免票。这里有个小贴士，如遇海面浪大，请做好晕船的准备，建议坐在船舱的中后部，俗话说车头船尾。
1：外地来的朋友可以乘坐城轨到珠海站（近拱北口岸），班次多，广州南过来约1小时09分，二等座65元，然后滴滴过去，费用约34元，不堵车的话25分钟可到达，如果时间宽裕的话，可以在珠海站换乘11路公交抵达码头，车程40分钟左右。
2：乘坐城轨到明珠站（珠海站的前一个站），这个站离香洲港较近，可惜班次少，广州南过来约54分钟，二等座65元，然后滴滴过去，费用约25元，全程15分钟。
3：香港、深圳的朋友可以先坐船到九洲港码头，然后打车前往香洲港，车程15分钟。
4：从珠海金湾机场出发，需打车到香洲港，车程约1小时。
由于航线座位有限，强烈建议提前在官微“香洲港”预定船票以及海岛酒店，售票窗口旁边就有专属的自助取票区域，取票方式类似我们取高铁票，同时订单详情里面有香洲港码头位置导航，非常方便自驾的朋友，可以说是非常贴心的官微了。
从码头背面就是珠海的新地标——珠海大剧院，它是中国唯一建设在海岛上的歌剧院，选址于野狸岛海滨，与情侣路隔海相对，远远望去，就如两枚大贝壳屹立在海中央，因而被人们称之为“日月贝”。
白天，大剧院就如半张合的贝壳，格外优雅，一到夜晚，五彩斑斓的射灯在贝壳上不断变幻，为市民呈现出惊艳的一面。

外伶仃岛

外伶仃岛，名字颇有诗意，相信大家马上会联想到文天祥的一首诗——《过零丁洋》，里面的“零丁”，与“伶仃”其实是同义的，都代表孤独，由于此岛不在伶仃洋上，所以取名外伶仃，就如一处与世隔绝的胜地。外伶仃岛在星罗棋布的万山群岛中风格独特，岛不大而绮丽，山不高而峻秀，尤以水清石奇为人称道，岛上伶仃湾、塔湾、大东湾的沙滩，沙质细腻，海水湛蓝，清澈见底。岛上集休闲娱乐，吃住行为一体，是悠闲度假的胜地。
一个半小时的船程，终于抵达到外伶仃岛，下船那一刻，就如踏进了天堂，天空、大海那一抹温柔的蓝，随着海浪声深入我心，这是一道我无法抗拒的颜色。
随着渔船的靠岸，市场和餐厅又新进一批又一批新鲜的海鲜，部分渔民在岸边就开始售卖，这些海产都保证新鲜，一旦发现不是活的，马上扔进大海。
你以为这里只有阳光与海滩吗？其实不然，一家十分文艺的书店藏于岛中央，取名“阅潮”，颇有诗意，这里除了出售各种书籍，还能在此享用美味的下午茶和海岛明信片以及珠海本地手信。
步入其中，就被这里的文艺气息所感染，淡淡的书香和浓烈咖啡香交汇在一起，翻一本书，品一杯咖啡，何其惬意。
据了解，阅潮书店在珠海已有十多家连锁，而这里则是南中国海首家海岛书店、咖啡。美女店长并非本土人，但是自从她来到了外伶仃岛，就被岛上的美景和氛围所吸引，如果你也有同感，不妨到此来一场美丽邂逅。
位于阅潮书店旁边，有一处网红打卡点，背景所表达的，或许就是岛民的日常，画中的一男一女，我希望有一天会成为我和他的写照，与他谈天说地，听听风，看看海，享受无忧无虑的生活。
来到外伶仃岛一定要爬伶仃峰，伶仃峰是爬山的中央主峰，爬上伶仃峰，可从上往下俯瞰全岛，在这看日出从海上冉冉升起，水光接天。远处轻雾弥漫，一艘艘渔船荡漾在水面上，壮美盛景，尽收眼底。
我们沿着海岸的小环线徒步，秋日的阳光并不猛烈，在海风的亲吻下，颇感凉快。
最吸引人的，莫过于那延绵数里的石滩，它们形状各异，星罗棋布，经过百年来的海水冲刷和自然风化作用，形成那错综复杂的纹理，不禁感叹大自然的鬼斧神工。
相对于千篇一律的沙滩，我更喜欢石滩的澎湃和激情，喜欢海浪拍打着岩石的韵律，喜欢海水从湛蓝变成白沫的过程。
虽然外伶仃岛开发较早，但大部分景色都较为原始，站在岩石上，看着那蔚蓝的天空，无尽的海平线，吹着海风，感受大自然对我们最温柔的问候。
双子亭，可以说是岛上的标志性建筑，与岩石群一起延伸出海，不少新人的婚纱照都在此取景。双子亭的一则是游泳海域，此处沙质细腻，海水清澈，人们的欢声笑语与海浪声融为一体，自是岛上最动听的乐章。
其实，岛上除了美景，并不缺乏一些历史遗址，比如说沿岸的摩岩石刻，或是藏在岩堆内的碉堡，都是岁月的见证者。

舌尖上的外伶仃

“伶仃归去不尝鲜，尝鲜必至伶仃岛”。赶在日落前回到悠活渡假酒店，参加一年一度的海馨海鲜美食嘉年华，都说美景与美食缺一不可，大部分游客都是冲着海鲜而前来的。这是一场规模宏大的海鲜盛宴，食材都是经过精挑细选的深海野生海鲜，主要打伶仃三宝，大鱼、大虾、大蟹、贝类及深海鱼刺身等常规海鲜十余种。
对于我来说，这次的重头戏要算比巴掌还要大的南海特有小龙虾，先别说其他海产，单是半只龙虾就能填饱肚子了。
龙虾采用了芝士焗制的方法，芝士的口感与龙虾肉的鲜味搭配得天衣无缝。而另一位主角要算原只大螃蟹，清蒸手法保留了螃蟹的原汁原味。
作为日料控的我，最欢喜的就是看见寿司拼盘，重点是不限量供应！除了有三文鱼、北极贝等常规刺身外，还有让人欲罢不能的海胆刺身。
而海胆炒饭更是一个奇妙的组合，丰腴清淡的海胆跟轻柔的米饭搭配在一起，滋味天成，没有腥味。
贝壳类一直是我的心头好，而这份外形较为奇特的就是狗爪螺，是伶仃三宝之一，因其像是狗爪，故名狗爪螺，看似怪异，肉不多但味道却异常鲜美，实属海鲜之珍品。如果你没吃过狗爪螺，那你肯定困惑该吃哪个部位才好！你以为是像剥瓜子那样剥开“狗爪”吗？那就错了，其实后面较软的部位才适合食用，轻轻把皮撕开，便能尝试到那来之不易的螺肉。
皮皮虾，在我们广东人口中又称之为赖尿虾，可以说是餐桌上不可或缺的海鲜之一，其肉质鲜美，让人垂延欲滴，盐焗烹饪手法更是深得我心。
为了打造多元化美食，酒店还提供各种烤肉，烤全羊是最值得期待的，在海岛也能吃出草原的味道。晚餐过后，可以在周边散步，从热闹中回归宁静，欣赏海岛的夜色。
第二天的午餐，由于需要自行解决，所以我们来到了当地的海产市场，购买生猛海产，然后拿到海鲜街加工。
市场不大，经营的都是本地人，海鲜的选择性也较多，“海鲜三宝”这些食材在本地都很常见，购买时可以适当地跟老板砍价哦。
为了保证海鲜的质量，大部分海鲜都不能捞上来拍照，可见老板们对食材的重视。
一些海产干货也值得我们选购，毕竟都是来自附近海域的海鲜晾晒而成的，很地道。
经本地人介绍，第二天午餐我们最终来到丰俭由人的哈味餐厅享用午餐，由于昨晚过于丰盛，把海洋三宝之一的将军帽给遗忘了。将军帽是沿石头生长的贝类食物，其内质颜色为黄绿色，肉质较脆，富有弹性，吃起来韧性如鲍鱼，故又有“假鲍鱼”之称，营养相当丰富。清蒸将军帽，贝类的肉上混着蒜蓉，撒上香葱...甚是鲜美。
何以解忧，唯有旅行。即使没有长假，不能放下手头的一切说“世界这么大，我想去看看”，但我们的压抑情绪还是需要及时释放，这时短途旅行就是最适合的。这里没有车水马龙的凌乱，也没有人声鼎沸的嘈杂，有的只是温柔的大海，淳朴的渔民，让我们卸下面具，做回自己。
