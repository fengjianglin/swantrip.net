游记 人文游 国庆 小众 美食 自助游 2019/01/08

概述

重庆因嘉陵江古称“渝水”，故重庆简称“渝”。1997年6月第三次列为 中华人民共和国中央直辖市。原以为刚刚列为直辖市的重庆不太大，到了重庆火车站，坐车到市区还要一小时。查资料得知重庆面积8万多平方公里，人口有3千万多；而上海只有6340平方公里，人口包括流动人员只有2千万多。重庆机场有3个，上海只有2个机场。来到重庆实地游览，真是开眼界了！立体交通繁忙，名副其实的山城！ 重庆和上海一样也是国际大都市、国家中心城市、超大城市、世界温泉之都、长江上游地区经济中心、金融中心和创新中心等。俗话说：百闻不如一见。 上午抵达重庆，到了市区临近中午，花半天时间自己游览重庆市区各个景点，恐怕来不及，在长浜路上一家旅行社，下单重庆一日游（下午出发也叫一日游）128元/人，工作人员看我们初来乍到，忽悠说是订了长江三峡两日游1000多元，送重庆一日游。一激动全订啊！详情见后。

行程安排

第一天：上海乘火车去重庆； 第二天：重庆、湖广会馆、白公馆、渣滓洞、磁器口、洪崖洞、解放碑； 第三天：参加旅行社“天坑地缝”一日游； 第四天：参加两日船游长江三峡：长寿古镇、丰都鬼城、白帝城。 第五天：船游长江、屈原寺、坛子岭、185平台至宜昌。 第六天：宜昌动车回沪。

出游准备：

1、国内旅游，一定要随身携带好身份证； 2、带好银联卡（信用卡），方便购物。 3、带好相机、手机、充电器，充电宝、数据线等。 4、带好自己的日常用药、晕车药、风油精，重庆火锅油多，带上防腹泻的药品备着。 5、出门前查询天气，带好雨伞，穿舒适的、防雨的旅游鞋。 6、带点干粮、零嘴，以备不时之需。 7、 带好洗漱用品及换洗衣物。 8、带好防嗮用品、口罩等

重庆气候

重庆市年平均气温16~18℃，最热月份平均气温26~29℃，最冷月平均气温4~8℃，采用候温法可以明显地划分四季。重庆市年平均降水量较丰富，年平均相对湿度多在70%~80%，在中国属高湿区。日照百分率仅为25%~35%，为中国年日照最少的地区之一，素有“雾重庆”之称。重庆年平均雾日是104天，有世界雾都之称的英国伦敦年平均雾日只有94天，远东雾都的日本东京也只有55天。重庆是名副其实的“雾都”，而重庆壁山区的云雾山全年雾日多达204天，堪称“世界之最”。

湖广会馆

重庆湖广会馆位于重庆市渝中区东水门正街4号。 湖广会馆始建于明朝万历年间。当时张居正捐私宅修建了全楚会馆、清朝文学家纪晓岚也曾在这里居住过。道光十年，会馆里修起了大戏台，谭鑫培、田桂凤都在这里演过戏。曾国藩在这里办过六十大寿；京城的革命党人也曾多次在这里欢迎孙中山先生，国民党成立大会就是在这里举行的。 门票价格： 30元

洪崖洞

洪崖洞位于重庆市核心商圈解放碑沧白路、长江、嘉陵江两江交汇的滨江地带，东水门大桥旁，坐拥城市旅游景观、商务休闲景观和城市人文景观于一体。以具巴渝传统建筑特色的"吊脚楼"风貌为主体，依山就势，沿江而建，观洪崖滴翠、逛山城老街、赏巴渝文化、看两江汇流、品天下美食的好去处，也是解放碑的会客厅。 洪崖洞有纸盐河酒吧街、天成巷巴渝风情街、盛宴美食街及异域风情城市阳台四条大街。乘坐扶梯或 观光电梯 直上解放碑或达到洪崖洞的任何一层楼（rh任何一条大街）。天成巷巴渝风情街为世人展示的一种时尚潮流盛宴美食街为大家展现的则是一种"另类美食城"的盛景。异域风情城市阳台为重庆最大的一个城市交通转换站。

解放碑

解放碑位于重庆市渝中区，我们步行约莫15分钟，来到民权路，远远看到解放碑，碑体绝大部分呈白色，有些地方是肉色，上端有几架大钟，朝着四面八方，到整点时，便听到悦耳的钟声。解放碑，原名抗战胜利纪功碑，是全中国唯一的一座纪念中华民族抗日战争胜利的国家纪念碑，以纪念重庆对于国家的伟大贡献。解放碑是抗战胜利和重庆解放的历史见证。
建议游览时间：解放碑周围商业圈可以游玩半天~一天。
 
重庆解放碑步行街为全国第一条商业步行街 ，有"西部第一街"美誉的解放碑商贸中心区(CBD)有望跻身中国第三大商圈。 嗨！忙乎一天，住处尚未落实呢！行李还在旅行社，赶紧过去，问了几位警察，说到长滨路没有直达公交，旁边问了出租车，询价30多元，自己百度一下，距离存放行李的旅行社1.6公里，咱就边走边欣赏体感重庆的山城（阶梯走了不少）。

天生三桥

武隆天生桥是全国罕见的地质奇观生态型旅游区，属典型的喀斯特地貌。景区以天龙桥、青龙桥、黑龙桥三座气势磅礴的石拱桥称奇于世，属亚洲最大的天生桥群。 我们游览，是要下到坑底行走。依次走过三个石桥，也只是从桥洞下走过，而不是桥面。桥面是看不到的。其实，我们也可以理解为几个桥洞，把几个天坑连通了。 天龙桥即天生一桥， 桥高200米，跨度300米，因其位居第一，顶天立地之势而得名。一桥桥中有洞，洞中生洞，洞如迷宫，即壮观又神奇。
青龙桥即天生二桥，是垂直高差最大一座天生桥。桥高350米，宽150米，跨度400米，夕阳西下，霞光万道，忽明忽暗，似一条真龙直上青天，故名青龙桥。
黑龙桥即天生三桥，桥孔深黑暗，桥洞顶部岩石如一条黑龙藏身于此，令人胆战心惊。黑龙桥景色以其流态各异的“三迭泉”、“一线泉”、“珍珠泉”、“雾泉”四眼宝泉而独具特色。

丰都鬼城

丰都鬼城，古为"巴子别都"，东汉和帝永元二年置县，距今已有近2000年的历史 ，位于重庆市下游丰都县的长江北岸，现在是长江游轮旅客的一个观光胜地。还是集儒、道、佛为一体的民俗文化艺术宝库，是长江黄金旅游线上最著名的人文景观之一。听说丰都鬼城又称为"幽都"、"鬼国京都 "、"中国神曲之乡"。鬼城以各种阴曹地府的建筑和造型而著名。鬼城内有哼哈祠、天子殿、奈河桥、黄泉路、望乡台、药王殿等多座表现阴间的建筑。看了介绍吓得我们只在景区门外兜了一圈。
门票价格：120元（淡季票价80元/人，旺季票价120元/人。
开放时间： 6:30-17:30
最佳旅游时间：丰都气候温和，除了夏季最热的几天外，一年四季都适宜旅游。
交通信息： 丰都“鬼城”位于忠县和涪陵之间的长江北岸，上游距重庆市172公里，下游距湖北宜昌476公里，既是重庆长江三峡游的必经之地，也是从重庆沿江旅游的第一个著名旅游景区。目前进出丰都鬼城主要有三种交通方式，即公路、铁路与水路。
路线一：重庆朝天门码头到丰都码头轮船，六个小时到达丰都 ,停靠丰都老县城，下船即是景区。
路线二：重庆四公里综合换乘枢纽车站每一个小时有发往丰都的班车，两小时到达丰都，下车为丰都新县城，转乘市内交通109路到景区。
路线三：重庆火车北站乘坐动车到丰都丁庄火车站，一小时左右到达丰都，转乘县内的公交车，到斜南溪公交车站转乘109到达名山风景区。
自驾：重庆主城内环蟠龙立交转G50S（渝涪沿江高速） → 丰都新县城。

长江三峡

长江三峡又名峡江或大三峡，位于中国重庆市、恩施州、宜昌市地区境内的长江干流上，西起重庆市奉节县的白帝城，东至湖北省宜昌市的南津关，全长193千米，由瞿塘峡、巫峡、西陵峡组成。长江三峡是瞿塘峡、巫峡和西陵峡三段峡谷的总称。
交通方式
三峡旅游有其特殊性如景点均遍布两岸两岸群山峻岭不宜行车等所以经水路游三峡最为理想。一般提及游三峡在没有特别指明其它游览方式的情况下即为乘船游。乘坐游轮饱览江景每到名胜登岸观光既免去了每日更换酒店的奔波之苦又可享受每日不同的窗外景观。
旅游路线
1、从重庆顺江而下快节奏地观赏三峡的奇特风光。
2、从上海、南京、武汉逆流而上游览长江沿途美景。
3、从宜昌逆流而上观赏三峡的美丽风光。

屈原祠

屈原祠位于秭归县东1.5 公里长江北岸的向家坪，又称清烈公祠，占地面积约30亩，为纪念屈原而建。屈原祠始建于唐元和十五年(820年)。1978年建葛洲坝水利枢时，将它迁向家坪，且按原貌重建。 屈原祠以 屈原文化为统领，是三峡库区能够把物质文化遗产和非物质文化遗产保护利用充分结合起来的重点区域。

坛子岭

 登坛子岭，俯瞰整个三峡工程，上185平台实地，见证“截断巫山云雨”的雄伟大坝。坛子岭景区是国家首批4A级景区，也是三峡坝区最早开发的景区，因其顶端观景台形似一个倒扣的坛子而得名，亦是大坝建设勘测点，海拔262.48米，是观赏三峡工程全景的最佳位置，不仅仅能欣赏到三峡大坝的雄浑壮伟，还能观看壁立千仞的“长江第四峡”双向五级船闸。  
整个园区以高度的递增从上至下分为三层，主要由模型展示厅、万年江底石、大江截流石、三峡坝址基石、银版天书及坛子岭观景台等景观，还有壮观的喷泉、秀美的瀑布、蜿蜒的溪水、翠绿的草坪贯穿其间，放眼望去，静中有动，动中有静，仿佛置身于美妙的乐园。

站在平台上向下俯看，就如同身临坝顶，可以感受到大坝的高度，同时，现在海拔175米的水位也使我们能在这儿领略到平湖的感觉。 185平台的左侧，有一个船闸，是单线船闸口，这就是：大船爬楼梯，小船坐电梯。

宜昌

从185平台出来，景区大巴把我们送到宜昌九码头散团。有团友在那查看下一步的旅游行程，我们呢，找地方慰问犒劳一下自己的胃，然后找住宿。在九码头对面、万达广场旁边一条马路上，一家古色古香的面店，停下脚步，有人进去观察。
宜昌
他们刚要关门休息，见我们进来，停下手里的活，等待我们决定吃啥。
宜昌
他家小笼是唤醒记忆的味道，那我们就让店家重现手艺的美好，来笼小笼。

宜昌火车站

为了方便早上去火车站，昨晚把酒店定在了火车站附近，10分钟即可到火车站。
宜昌火车站
早上经过餐饮店，吃了包子、鸡蛋、豆浆，精神饱满乘动车，回家啰！
宜昌火车站
宜昌东站门口可罗雀，动车候车室在楼上。回家了！再见了宜昌！

重庆旅游建议：

1、伙伴们来到重庆，不必参加重庆一日游，每个景点停留时间少，自己去，公交地铁很方便！
2、去武隆最好参加两日游：天坑地缝及仙女山，一日游天坑地缝，路途遥远太辛苦！
3、参加长江三峡旅游，事先一定要问清楚游船是否过大坝，货比三家，从中取得价廉物美的一家。我们就是心急，出了重庆火车站，碰上热心的的哥，跟他来到长滨路上的旅行社，定下了两日游，我们是游船上价格最贵的 ！本次是一心想坐船过三峡大坝的，又一次落空。2017年元旦参加天鹅之旅的去武汉自由行，同时预订了武汉——宜昌、三峡大坝一日游，到了那里说是天气不给力，游船不能过坝，扫兴而归。

看样子应了老话：事不过三呐，三峡大坝还得来一次！
