游记 人文游 小众 自然奇观 赏花 雪景 2019/04/24

把片段拼凑成记忆！

孙 中山 先生曾这样评价 南京 ：“乃在一美善之地区，其地有高山、有深水、有平原，此三种天工，钟毓一处，在世界之大都市诚难觅如此佳境也”。 
别的古都，把历史浓缩到宫殿；而 南京 ，把历史溶解于自然。钟山风景区的秋色，一墙（城墙）之隔的玄武湖（旧称后湖）和前湖，天堑长江沿线的带有民国气息的滨江风光带，夫子庙的秦淮风光带，等等等等。。。 城中处处留着历朝痕迹：明孝陵、总统府、颐和路、夫子庙、甘熙故居，等等。甚至你走在老城南生着苔藓的青石板路上，不定就到了城墙边、秦淮人家，又或是先人故居。 中山 码头坐个1元的轮渡过（长）江，就到了当年朱自清父亲爬过的站台。这里既有《人民的名义》中的汉东检察院（ 南京 体院），又有曹雪芹笔下的红楼一梦（ 江宁 织造府）。
这一年， 骑行穿梭在 南京 的每个角落，用手机记录下美好瞬间已经成为骑行的动力之一，毕竟 南京 是我的城，我要为大蓝鲸疯狂打call。可是仅限于盆友圈的9图或是短视频无法链接成为圆，所以，一段段骑行的故事，把它们拼凑成一段完整的记忆。写下第二篇 南京 骑行记！

这里是下马坊遗址公园，600多年前，“孝陵卫”是大明帝国一支精锐部队的番号，其地位不亚于京城内的皇家卫队。当年孝陵卫是皇家禁地，近6000名精锐士兵驻扎于此，文武百官到此必须下马步行。这样的庄严与肃穆只有一个原因，这里是大明开国皇帝朱元璋孝陵的入口。明孝陵国家和皇权的象征，也是“龙脉”所在，孝陵的平安就意味着子孙兴旺发达、国运长久。因此，除了管理孝陵内部的神宫监外，皇室特地在下马坊东侧设立了军事机构“孝陵卫”，以保卫孝陵的安全。
