初恋VS婚姻 西藏VS新疆

只有一个地方让人刻骨铭心——西藏，如入秘境：虔诚的藏人，快乐的小猪和小狗，雪山峡谷的旖旎风光，当然还有缺氧造成的心痛、头痛。虽然物质很贫乏，但是心是快乐的，那永生不忘的心痛，仿佛是初恋的记忆。
△新疆 · 喀纳斯
西藏，是初恋的感觉。而新疆，是婚姻的感觉。再也不能像年轻那样，不怕天不怕地，不怕缺氧，嗖地就去了西藏，谁知冻成了狗，疼得差点在手机里写了遗嘱。
△新疆 · 喀纳斯

雄性魅力

时隔多年之后，终于来到了新疆。而新疆仿佛让我进入了一场婚姻，一场期待多年的婚姻。
△在喀纳斯湖乘坐游船之前，兴许能偶遇到他。
穿制服的男人最帅，会双语（汉语和蒙古语）的男人更酷。
喀纳斯在阿勒泰地区，图瓦族仅有几千人居住，而且世代不离开那里。喀纳斯真的好美，还有无法科学解释的谜团，附近的酒店和县城因旅游而发展，到处是做生意的汉人，只有警察和一些工作人员是少数民族，少数民族的男人异常地帅！草原的汉子，脸上还带着紫外线的光晕。
△新疆 · 喀纳斯地区图瓦族家访表演
新疆的维族、回族、汉族多，但是望春风姐姐发现这里的少数民族哈萨克族和喀纳斯土著图瓦族（突厥民族）男人才是最有雄性气息的。
△新疆 · 喀纳斯地区图瓦族家访表演

△新疆 · 喀纳斯地区图瓦族家访结束后送客
不羁是草原汉子的风格，高山紫外线的肤色，粗犷有型的颧骨，会说话的长睫毛，再加上合身的制服，绝配了，哪怕衬衣下摆没有完全塞好，此刻让礼仪靠边站吧，要的就是草原民族不经意间散发的野性魅力。

△新疆 · 喀纳斯地区图瓦族聚居区的花

面包牛奶VS馕与葡萄

散发的雄性荷尔蒙还不够，牛奶面包才是王道。酸奶、羊肉串、馕、葡萄、瓜果…… 都是天然的味道。不管你是不是吃货，在新疆都是幸福的，少了添加剂的干扰，胃口大开。
△新疆 · 吐鲁番
念念不忘哈密火车站的馕，是进入新疆领域的第一个馕，干吃馕喝白开水，一整个馕一口气就吃完了，省了顿盒饭。
我平时不吃串儿，在这儿的羊肉串是真羊肉，每顿饭都要来一个串，因为每天坐车十个小时左右。

新疆西北部高山逶迤，草原辽阔，水草丰美，自古以来就是中国西部游牧民族繁衍生息的地方。

喀纳斯

喀纳斯景区位于阿勒泰地区的深山密林中，距离首都乌鲁木齐约800公里，包含了喀纳斯湖景区，禾木村，白哈巴村等著名景区。喀纳斯湖以神秘的湖怪、变换颜色的湖水和炫丽的自然风光而让人痴迷。禾木和白哈巴都是当地图瓦人聚集的村落，小木屋颇有瑞士小镇的风采，犹如人间仙境。据说这里是中国九大徒步线路之一，已成为国内户外爱好者心目中的一条天堂线路。
△新疆 · 喀纳斯 · 卧龙湾

△新疆 · 喀纳斯 月亮湾
包车/自驾：是前往喀纳斯景区比较便捷舒适的一种交通方式，还能顺道游览沿途景点。包车参考价：乌鲁木齐-贾登峪约1500-1800元（单程/辆）需要12小时左右（包括中间吃饭时间）；布尔津县城-贾登峪约250元（单程/辆），约150公里，耗时约2小时。价格因淡旺季和车辆路线时间的原因而变化，请仔细甄别。
火车：景区所在的布尔津县没有火车站，距离景区最近的火车站是阿勒泰火车站，省会乌鲁木齐有往返北屯的列车，耗时约12小时，硬座票价约100元，硬卧约180元。到达北屯后，在市区客运站有发往布尔津县的班车，耗时约2小时，全程约90公里。到达布尔津县城后，再换车前往贾登峪（耗时约2.5小时）。
客车：乌鲁木齐碾子沟车站有发往布尔津县城的班车，每天三班，11:15、20:10、20:40，耗时约10小时，全程约700公里，票价约180元。碾子沟汽车站：乌鲁木齐市黑龙江路49号，电话：0991-5878015。乌鲁木齐-布尔津县沿途人烟稀少，还会穿越沙漠戈壁和百里无人区，途中自带些干粮和饮用水非常有必要。

喀纳斯 · 卧龙湾

△新疆 · 喀纳斯 卧龙湾
卧龙湾位于布尔津县去喀纳斯的途中距县城140公里，距喀纳斯10公里，是喀纳斯湖的排水口。面积约9公顷，河湾中心是一块植物茂盛的沙洲，好似一条静卧在水中的巨龙，卧龙湾以此得名。湖四周森林茂密，湖进水处巨石抵中流，波光荡漾。湖的泄水口有座木桥飞架东西，站在桥上向北是如镜的卧龙湾向南是奔腾的喀纳斯河。
△新疆 · 喀纳斯

△新疆 · 喀纳斯

喀纳斯 · 月亮湾

△新疆 · 喀纳斯 月亮湾
月亮湾是喀纳斯最著名的景观之一，因为这部分河流是月牙形状的，而且河水的颜色也比较好看，颜色是苹果绿的。
△新疆 · 喀纳斯月亮湾
喀纳斯湖在这划了一道优美的弧线，犹如弯弯的月亮落入这林木葱茏的峡谷，令无数游人为之陶醉。
这里是摄影爱好者的胜地，从春季到秋季，从清晨到傍晚，在不同的气候作用和光线照耀下，一汪水月不停地改变着充满神秘魅力的颜容，为摄影爱好者提供了取之不尽的精彩画面。

白沙湖

△新疆 · 白沙湖
在新疆的边境线上，有个185团，白沙湖位于185团东北部的三连境内，是独立于浩瀚沙漠中的原生态沙漠湖，水面面积不足185亩，海拔650米。
△新疆 · 白沙湖
白沙湖位于哈巴河县境内，是个独立于浩瀚沙漠中的原生态沙漠湖，没有进水与出水口，沙漠里一般蒸发量都比较大，但这里的水位多年来却没有变化，更不因季节变迁而增多或减少。景致构成十分独特，湖中四周生长着高达1-4米的密密丛生的芦苇、菖蒲等水生植物，湖中还有野荷花孤寂地生存着；湖周50米内，是高大茂密的杨树、白桦混生的林带。

火焰山

因《西游记》三借芭蕉扇的故事家喻户晓，维吾尔语意为“红山”，因其炎热曾把它称作“火山”。
△新疆 · 火焰山
山脉呈东西走向，寸草不生。夏日时分，地表在太阳照耀下蒸腾，似飞腾的火龙十分壮观。
景区内有许多西游记中出现的人物雕像。

天山天池

天山天池，古称“瑶池”，是天山博格达峰北侧的一个半月形湖泊，新疆著名的旅游胜地。
△新疆 · 天山天池
湖水晶莹如玉，倒映着博格达峰，四周群山环抱，山上布满了挺拔、苍翠的云杉、塔松，漫山遍岭，风景犹如仙境。
推荐下午晚些时候去，因为随着光线的变化，湖水会呈现近处碧绿远处蓝色的样子，非常好看。
△新疆 · 天山天池
天山天池在中国古代神话中，是西天王母娘娘沐浴的地方，让这里愈发神秘。
每年的11月中旬到次年的3月底会举办“天山天池冰雪风情节”，游客可以前往风景区附近的天山滑雪场进行滑雪及其他的冰雪项目。

△新疆 · 天山天池
王母娘娘的洗澡盆——天山天池，也可以说是王母娘娘的婚外情，哈哈，每个景点都有自己的故事。据说好多文人都在天池住过，包括邓小平总理。
△新疆 · 天山天池
【主天池】海拔1980米，是一个湖面呈半月形天然的高山湖泊。湖泊内有游船，需要额外购买票。除此，还有【东小天池】【西小天池】，都是比较小的湖泊，有的在公路旁边，只能在车上看看，无法停车。
△新疆 · 天山天池
【定海神针】是湖边的一棵榆树，据说榆树不适宜在这里生长，只有这一棵“榆坚强”生长在这里，每次湖水涨起涨落，都没有腌到这棵树，神奇吧。
【飞龙潭】是很长的瀑布，需要沿途下坡，流下的水静止后是苹果绿色的，非常美丽。
△新疆 · 天山天池 飞龙潭
【穿梭巴士】去时的巴士有讲解员，会告诉途径的几个景点，长得H形的两棵树，一线天的景点。去时巴士分两段，中途下车，穿过天山景区的民俗购物街，就是让游人消费的意思吧。走着大概10几分钟，继续坐穿梭巴士。返程，需要持票排队坐车一程即可到出口。
【天气】由于天山天池是高山景观，山上气候多变，公路部分路段狭窄，下雨以及极端天气，巴士无法正常行驶。需要及时留意景区公告。
△新疆 · 天山天池
【位置】天山天池位于新疆阜康市境内的博格达峰下的半山腰，东距乌鲁木齐110公里。驱车1个多小时即可到达。

图瓦族

△新疆 · 喀纳斯图瓦族聚居区
图瓦族居住在喀纳斯，至今也没离开过，据说是成吉思汗的守卫兵的后代，因为成吉思汗说不要登山，神在看着你，成吉思汗年老，死在了这座山上，卫兵们就一直守卫着他的陵墓。
这位少数民族姑娘给大家讲解的，棱角分明，基因强大。
进景区就可以去家访去。目前每家都开成了文艺小茶室，注册了公司，也成了美女的摄影棚和观看图瓦族歌舞的大舞台。

五彩滩

五彩滩位于布尔津县城西北24公里处的额尔齐斯河北岸，是一个以雅丹地貌著称的河滩。这里的丘陵地质奇特，显示出各种色彩，每到傍晚夕阳照耀时，颜色奇幻绚烂，非常壮观。对岸是一片茂密的胡杨林，倒映在河水中景色优美，每到秋季黄叶灿烂，是拍照摄影的绝佳地点。游玩时要按照路线在栈道上行走，这里地质疏松脆弱需要保护，切勿踩踏和攀爬。保护生态，人人有责哦！
△新疆 · 五彩滩
