【天鹅之旅首发】广州吃货游成都，带你品尝最正宗的成都味道

游记 人文游 摄影 美食 自助游 2018/11/22
一场说走就走的旅行，赋予岁月以智慧。一份地道的本土美食，也足以温暖一生。好玩又好吃的旅行地，能给眼睛和胃一份超大满足！秋日正好，宜“贴秋膘”。这次，我们走进国际美食之都--成都，用52个小时来体验这个“吃货天堂”，品尝地方美味佳肴，感受当地人对“吃”的热爱。
辣是人间最为激烈的味觉体验，初尝时辛而甜，到了舌头中间变麻，一路温热咽到咽喉和肚子，大为酣畅淋漓。
成都是串串的发祥地。各种荤素被串在竹签上，抓上一把签子，在红汤里涮过，蘸上碟料，吃得过瘾。

最接地气的马路边边麻辣烫

我们美食打卡第一站就是最接地气的，马路边边麻辣烫（致民路店）据网上资料查阅，它家的串串在成都串串香热门榜名列第1名。
营业时间：10：00——1：00
地址：致民路48号（地铁3号线新南门站B出口170米）
进店就是满满的怀旧气氛，小时候学校马路边吃串串那种熟悉的场景又回到我们眼前。
这家店卖的是情怀，店内墙壁上有各种涂鸦画，收银台是学校小卖部的外形，大厅设计成学校操场，有电线杆，还有广播体操用的电喇叭，各种老物件仿佛回到了童年……
一切都跃现于眼前，在这草根气质十足的环境下撸串，是不是更有成都味道呢?
虽然串串属低端火锅，但很市井也接地气，来排队的人很多，我们是下午四点半到的，店还没开张，外面板凳上已经坐了几圈排队等位的食客了，也让我等真正见识了成都吃货们的魅力。
菜品分量和价格都比较亲民，串串的种类很多，需要自己过去冰柜里面自选。
正确的打开方式都是一边吃串串，一边喝啤酒，一边擦汗。面对吃香锅里面传出的阵阵香味，相信任何人都抵挡不住这种诱惑。
串串香会配上香麻的花椒粒，火红的辣椒及秘制酱料，荦和素的放进锅里面涮一下，沾上配料其味无穷，好吃到完成停不下嘴。
马路边边麻辣烫的菜单设计也很有特色，类似小学课本那种，特色菜品需要在菜单上单独下单。
特色菜品有火冒三丈兔腰：第一次吃这个东西，看起来小小的一个，吃起来有点脆和弹牙，加辣椒后，腥味不重，可以接受。
凉拌花甲：是一道菜基本所有成都美食店都会有，但用小时候的铁碗盛装，别有风味，口感麻辣入味，味道不错。
鲜鸭肠：本人觉得很赞，这家鸭肠非常脆，涮几秒就可以食了；鸭肠、黄喉、毛肚是火锅必点三大件，衡量一家火锅店好不好吃，看这三样产口的水准就知。
猪脑花：这个是我老婆的最爱，口感粉嫩，不腥，吃里需在芝麻粉和辣椒粉滚过再吃。民间有吃脑补脑的说法。来成都了，值得一试。
凉拌千层肚：这个极力推荐，实在太好吃了。肚尖很脆，拌了蒜泥特别的香。人生第一次吃到，很有成都地方特色的小食。
红糖糍粑：值得试试，也是很经典的地方小吃。吃法比较特别，沾酱油和花生粉，新鲜！

成都最时尚的地方 太古里

成都太古里坐落于成都市中心，将古老街巷、川西风格的青瓦坡与现代化的潮流商业交融于一体的大型商业中心。
网红和模特聚集地使太古里成为成都旅游必打卡必去景点之一，作为成都最时尚的地方，许多国内外一线服装品牌都在太古里设了旗舰店，每天都会有很多妙龄亮丽的美女，穿着时尚衣服在这里为品牌拍视频和写作。
都说四川出美女，美女那里可以看到呢，来太古里就对了，业余摄影师们都喜欢来这里街拍，找灵感。
太古里还有许多的川西风格的古建筑，周边高耸入云的摩天大楼包围下，显得这座繁华的都市时尚又不失传统。
太古里有不少这种玻璃天桥，随便一个点都是不错的摄影背景，路边的咖啡店等也都很有特色，停下来喝一杯，欣赏这里人来人往的美女，也是一个很好的风景线。
在太古里旁边就是IFS商场，喜欢历史的，可以来看看2008年入选全国十大考古新发现的“唐宋街坊遗址”，当时遗址挖掘出土了大量的陶器，瓷器、钱币和骨器等文物，现在原址保护起来，供人参观。
再走不远就是成都最著名的商业街“春熙路步行街”。
街上有不少熊猫造型的车吸引人们排队打卡留影。

没辣味的泰式海鲜火锅

集渔泰式海鲜火锅（太古里店）（泰式火锅，成都火锅热门榜第一，排队前三）
营业时间：11:00-22:00
地址：下东大街169号晶融汇2期洋展购1号门（距地铁2、3号线春熙路站B口390米）
这家店是太古里商圈中十分火的一家泰式火锅店，鲜料冬阴功锅是招牌，以吃海鲜为主，每一道菜品卖相都非常漂亮，美味更是一极棒！
来成都除了不可不试的川味火锅外，近期没辣味的海鲜火锅也成为了成都人追捧的对象。这家来自泰国普吉岛，汤底讲究新鲜，用野生鲫鱼经过四个小时熬制而成，汤色白乳，鲜气浓郁。
锅底上桌时，就能闻到浓浓的椰汁香味，看到锅内有两个布绑着的鲜料包，据说里面精选了几十种进口的香料搭配而成，有泰国的红辣椒、香茅、青柠、南姜等，并非传统干料可比。
就餐时，有服务员站旁边帮我们烫菜，熟了再分摊给每个人，所以服务比较好，不用担心菜品烫几成熟的问题。
我们点的是四人套餐，菜式很丰富，我们点的海鲜餐里面有象拨蚌，生蠔、青口、海虾这些，样样都很新鲜，蘸料有六种味道可供选择。
鲜虾刺身很特别，在一个内陆城市，敢做刺身，可想而知这虾得多新鲜。小小的虾处理掉虾线后，放在柠檬片上。吃起来口感微辣略带清新。
西柚缤纷气泡， 这个瓶子外观非常精美，里面混合了几种的水果，复合水果味，适合女士。
爆浆奶盖蛋糕，当服务员将盖在上面的盖子抽开那一刻，真的很惊艳，因为太美了，一个会爆浆的奶盖。惊喜连连，甜度刚刚好，比传统的蛋糕更好吃多了。
环境设计相对时尚有个性，店一共分为三层，就餐环境全被绿植包围着，餐桌上的镜子是一大特色，让空间更呈宽阔明亮。
因为是网外店，门外等位区较大，并贴心地为大家准备了取暖炉，大理石的地砖加金属质感的楼梯和吊灯，当下各种时尚元素在这店都能找到，不失是美食打卡的一个好地方。

大隐隐于世的成都大慈寺 秋天赏银杏

太古里旁边的大慈寺现在银杏开得艳，现在是观赏古寺秋色的最佳季节，古寺内最大的古银杏树龄已超160年，为清代自信禅师手植，现正值金秋满寺，地上一片金黄，为成都10大著名银杏观景点之一。
历唐末、五代、北宋三百多年，众多知名画师在大慈寺创作了无数精美绝伦的壁画，各种神像、佛像、画像都非常精美，使艺术大放异彩。
寺内的楼、阁、殿、塔，屋檐雕梁画栋，是一座 “举世无双”的艺术宝库，拥有悠久的历史和文化保护价值。
古寺始建于七世纪中叶，唐玄宗赐名 “敕建大圣慈寺”，历经兴废，多次毁于兵火。现存诸殿系清代顺治后陆续重建。主要建筑有天王殿、弥勒殿、观音殿、大雄宝殿和藏经楼等。
正山门殿上“古大圣慈寺”的题刻为清光绪时任成都知府的著名书法家黄云鹄所题。
据记载，西游记的原形-唐代高僧玄奘大师曾在大慈寺修行受戒并踏上了西天取经之路。
抗战时期，故宫文物西迁北路途中，就存藏在成都大慈寺中。
我们来的时候，正好是大慈寺“禅意金秋 古寺银杏”活动，寺内有品茶、禅锈、书画等，活动将持续到12月18号，过来的小伙伴们不要错过了。

成都吃客（致民路店）

餐厅在二楼，进门首入眼帘为餐厅各种获奖证书，大众点评2017、2018必吃榜餐厅；大厅原木色调装饰，屏风上都是老成都的照片、边上写着成都景点的名字，环境很温馨！
营业时间：12：00——13：30，17：00——21：00
地址：致民路48号（地铁3号线新南门B口步行150米）
青椒鸭肠
属四川菜中一个劲爆菜系，虽然很辣，但每一口都是过瘾，超级好吃！鸭肠口感很脆，主要是突出青椒和麻辣的味道。
特色烤鱼片
是火锅，串串和烧烤三种四川特色组合在一起组合的创新菜；将鱼片串起来烤，这个是第一次吃，闻起来就很香，鱼肉细腻肥美，味道为四种复合型味道，根据四川人比较喜欢吃海带和豆干的特点，鱼片下面就垫了这两种美味。
蒜香排骨
选用的是猪的精排，制作时，事前先腌制一晚，第二天用大火猛烤，迅速收干肉排的水份，这样吃到的口感，无腥味，肉质感较强。
绝情娃
选用的是中等的牛蛙，口感细腻，比较清新，能突出四川的香、麻、辣味道。
纸上焗鲍鱼
这个是店的招牌，是网红菜，根据年轻人喜欢吃泡面的特性，结合到了海鲜做配料，让菜的味形更加突出，口感是蒜蓉青花椒味，鲍鱼、蟹肉、丝瓜，配双椒，底下是方便面；创作理念是让海鲜融入的泡面里的味道，适合不吃辣的朋友。
裸奔坨坨虾
顾名思义，裸奔是指虾经处理后去壳去了虾线，这样客人吃起来比较方便。然后选的是海虾，肉质口感有弹性，底下还垫了芋头，这就形成了四川独有的味道，用藤椒精炼出来的香辣油，泡浸着，使虾吃得比较香。
冷吃牛肉
大家都喜欢四川自贡的冷吃兔和冷吃牛肉非常有名，这个菜不单味道上能代表四川菜，餐具也非常有创意，碟子上的面谱就彰显出成都元素！味形上突出麻辣鲜香，用的是贵州的满天星，辣跟香相结合，是成都本人最喜欢的。
相思红颜
属小菜，功能是解油腻，去辣；吃的是小时候妈妈的味道。做法是番茄用开水烫后去皮，加白糖浸泡1小时，后加入秘制桂花糖，调味后，淋上去，就是成品。

旧成都生活标本：宽窄巷子

成都既有喧嚣繁华的闹市，也有古街古巷，在繁华都市与静谧古巷间穿梭漫步，处处有景致，处处有故事。
宽窄巷子就是代表之一，它是成都目前遗留下来的保守完整并成规模的古街道，景区是由三条平行的巷子和四合院落群组成。里面聚集了许多技艺精湛的民间艺人，让前来的游客切身感受到老成都深厚的文化。
商家招睐生意，常请一些专业演员在门口表演川剧，演员穿着那色彩斑斓的服饰常常使人驻足围观。
我们本次入住的民宿就在井巷子内，这条小巷的人文气息最重，进入巷子就进像进入了博物馆，墙上有关于成都古代砖的介绍。井巷子砖历史文化景观墙从山土坯砖，秦砖、唐砖、宋砖、明砖、清砖，民国砖，阐释了成都城关于砖的记忆。
砖历史文化墙上还有18幅历史老照片，照片上描绘的都是老成都市民日常生活，充满快乐的气氛，吸引游客排队合照留念。
我们民宿楼下，就是关于井巷子的起源的雕像，还是口真井，石碑上记述开挖于康熙年间，是八旗军驻防时所挖，辛亥革命后，因巷内有此井，整个巷命名为井巷子。

井巷子的地面上都是关于成都古代名水井的介绍，青铜绘画铺设在地砖上，很文化气息喔。
宽窄巷子是老成都生活的再现，走进这三条小巷中游览，体验到的是最老成都的生活，感受到的是成都的一些几乎要失传了民俗生活风俗场景。它是“旧成都生活标本”。

打卡成都玉林美食地标

成都，成都，被一首成都唱红的玉林，是成都旅游必打卡的美食和文艺地标。
走进天府之国，大街小巷内满城都是火锅味，刺激了味蕾，纵使知道四川人对麻辣的喜爱，但走几步便能看见一家火锅店的密度，就已经让火锅当之无愧成为了它的代名词。
来之前做了功课，玉林商圈火锅人气排名第一就是这家如在蜀老火锅。
营业时间：9:30—凌晨02:00
地址：玉林南路15号（玉林生活广场一楼）（距离地铁1号线倪家桥B口步行1300米）
如在蜀老火锅在玉林生活广场一楼，离地铁1号线倪家桥比较近，不同于其他的火锅店，如在蜀的厨房透明的，整个配菜和打锅全过程都可以看到，这样吃起来，食客也会比较放心。
菜品现切现卖，像我们点的金针菇和香菇，都是服务器端到我们门前，用剪刀剪掉根部，再拿回去清洗，上桌，也保证的食材的新鲜。
来品老成都味儿，我们选了鸳鸯锅，老火锅牛油上还刻了“蜀”字，锅底里面有豆芽和手工现磨的豆花，锅底自带的这两样东西就足够惊艳味蕾了！点半斤毛肚就送锅底！如在蜀也是成都首家免锅底的火锅！
铺盖毛肚是如在蜀的镇店菜，实在太好吃了，我们吃了二盘还不过瘾！毛肚放在辣油锅涮八秒就可以吃了，水牛肚口感超级脆嫩。
双椒霸王牛肉，混合了红、青两种辣椒搭配，牛肉吃起来很有肉味。送白葡萄酒最合适不过了。
红酒浸肥牛
当晚最有创意的一个菜，红酒杯内装着半杯红酒倒放在肥牛上面。肥牛吸引了红酒的精华，涮在辣锅里口感更嫩滑好吃。
茴香现炸酥肉
这个餐厅小吃，几乎每家火锅店都有。
精致入味的还有蜀小碗系列，有脑花、花甲等冒菜，平宜又好吃。
虽说成都是内陆城市，店内海鲜也不少，点了八爪鱼、手打虾滑这些，味道也不错，如在蜀走的是平民低价路线，所有荤菜都是1开头没有超过20元，吃顿火锅吃下来买单的时候，保证让你眼发光，大呼实惠。 
环境来说，就是老成都那味儿，又融入少许草堂元素，让人有点在草堂打火锅的感觉。门口一个骑着自行车穿过小巷的老伯伯，蛮特色的。

跟着唐诗去旅行 浣花溪畔的杜甫草堂

杜甫，自号少陵野老，唐代现实主义诗人，有“诗圣”之称，他的诗被称为“诗史”。后世称其杜拾遗、杜工部、杜少陵、杜草堂。现世共有1500首诗歌被保留了下来，在杜甫草堂有专门的杜工诗集展览厅。
杜甫草堂却是充满感情的园林，景区内每一座小桥、亭台、屋舍，都能感到杜诗所营造出的意境，仿佛回到唐代感受诗人的热情、悲伤、希望和叹息。
草堂除了富有诗情画意外，这里的人文景观和自然景观结合得也相当完美。设有千诗碑，回廊内石板上，所刻画的杜诗、名人诗词刻匾使这里的文化底蕴提升不少。
今日草堂，古朴典雅，幽深宁静。现在秋色最浓，亭台楼阁间的黄叶十分好看。
在院子内闲游时，看到好多老奶奶、老爷爷在打牌，其乐融融。
草堂正门匾額所书“草堂”两字为清代康熙帝17子允礼亲笔。门口悬挂的一副对联“万里桥西宅，西花潭北庄”。正正道出了草堂的地理位置。
草堂内还有一个草堂遗寺博物馆，公无759年杜甫入蜀，借住成都浣花溪旁的古寺中。该寺原名为梵安寺，俗称草堂寺。遗址出土了大量的寺庙建筑残件，印证了杜甫在到达草堂时，浣花溪旁已有古寺的事实。
草堂茅屋还原了唐代诗圣杜甫曾经居住过的模样，墙都有竹子和泥巴外露，很破旧，也很真实。
茅屋外面还有个喝茶的地方，名字起来相当文雅“草堂北邻”，卖的是成都郊外的野生茶，我小试了一杯，味道还可以。

上过天天向上的少数民族的烤肉-醉西昌

营业时间：下午17点———晚上23点
地址：汇源北路333号中粮鸿云13幢3楼(万象城)（距离地铁2号线东大路站G口步行750米）
这里是吃肉爱好者的秘密基地，来自正宗西昌放养小香猪,，独特的彝族风格，让整个店铺变得截然不同。
以大凉山彝族菜为特色，大凉山跑山小香猪厚烤是特色，肉烤熟后，基本不用添加任何佐料，都很好吃，用生菜包着，一口一个香。据说素有“一家煮肉四邻香。七里之遥闻其味”之美称。
天味小香肠，这个是袖珍型的，肠子比筷子还小，烤熟了很鲜美，醇厚浓郁，回味绵长，越嚼越香。
蒜泥牛肉比较特色，反正我是第一次吃，牛肉切的很薄，烤熟后，沾上花生粉味道比想象中好。
食材都很新鲜，五花肉、大排骨、小猪肉，猪小肠这几样都是必点的，加了生菜和大蒜后，也变得清爽好吃。
虽然是烤肉店，但无烟环境让你印象深刻，完全不用担心烤肉的时候油烟过重的问题，只能说他家的玻璃罩油烟机确实好。

锦里 成都版清明上河图

传说，锦里在三国的时候就已经是全国闻名的商业街了。它是成都目前最古老最有商业氛围的老街道之一，具有成都版清明上河图之称。
每个人都带着怀旧的情怀而来，走进锦里大街，地面上是干滑的青石板路，两旁砖木结构的川西民居，马上穿越回到古代。
锦里，保存了不少快要失传的民间手艺人。糖画、剪纸、泥人、皮影……这些全可以在锦里找到。
中国传统文化艺术“吹糖人”，游客可以亲自参与制作一个晶莹剔透的糖人，也许会唤起尘封已久的童年记忆，那一份淡忘已久的欢乐与惊喜。
皮影戏这种传统民俗记忆来自童年。那些是可爱的回忆，现在看到，更多是怀旧。
这里的寻常巷陌间更充满着纯正川味的民俗风情，在溪边的茶楼坐下喝个盖碗茶，到戏台下看场古装大戏，一切都是那种不经意的慵懒。
商店橱窗的萌萌的熊猫又吸引过往游客合照，随处可见大家对于熊猫文化的热爱。
入夜，这里会更浪漫，慢慢去体验，你会发现慢慢喜欢上这种慢生活。

很多大明星去过的火锅店 大龙燚火锅

营业时间：10：30——2：00（凌晨）
地址：东大街下东大街段166号2层（距地铁2、3号线春熙路站B口360米）
大龙燚这家火锅界的传奇，几乎大半个娱乐圈,都来打过卡 范冰冰、谢霆锋、邓超、黄晓明...超多大牌明星都被它圈粉，之前在广州就吃过2次，来到成都，怎样也得尝尝发源地网红店的味道。所以体验安排在离开成都的最后一顿午餐上。
我们选择是的太古里店，主要是这里离高铁站近，只有10几分钟车程，吃完好赶车。
麻辣牛肉依旧鲜嫩，放在牛油锅涮一涮，马上让人欲罢不能。
跳水虾滑，纯手工制作，涮菌菇番茄锅又鲜又嫩最好吃。
到店必点菜大刀毛肚，手机大小，七上八下涮一下，沾蒜末蚝油，又爽又脆，味道一流。
大龙燚的镇店菜为千层肚，整条放入牛油锅涮几下，带着渗透的香辣汤汁，鲜香麻辣，入味好吃。
鸭血下到在沸腾牛油锅里，最后吃时，鲜嫩顺滑，一口入魂，简直让人欲罢不能！
环境，店在二层，比较光亮，桌与桌距离很宽阔，商务宴请都OK。
服务好，虾滑都是服务员帮忙下的。
