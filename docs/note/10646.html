【天鹅之旅首发】品味成都历史，打卡网红拍照圣地，吃最地道成都美味

游记 人文游 小众 摄影 美食 自助游 2018/11/12

前言

对青年人来说，旅行是教育的一部分；对老年人来说，旅行是阅历的一部分。对于我来说，旅行是生活必不可缺的一部分。成都，一座被各大短视频平台强烈力推的城市，她有“慢生活、美食、美女、美景”总是吸引着很多人来到这里，一旦停留就不想驻足、久居，从而房价也一路飘高，尽管如此许多怀揣梦想的年轻人也在这里打拼奋斗，这里没有北上广的快节奏，却又繁忙工作后遛弯赏花的三圣乡，加班后补充能量的串串火锅麻辣烫，充电学习的博物馆、科技馆.....

探访博物馆，寻味老成都

有人说，了解一座城市，不是去走过一条街，吃一道地道小吃。也并非去所谓的名胜古迹，而是去当地的博物馆，在那里可以花最少的钱，深入的了解一座城市的古往今来。
如果你说天府广场就只有一个广场，有什么可玩的，那你就错了。天府广场旁边成都博物馆是成都综合性规模最大的博物馆。也是作为成都周边必去的七大博物馆之一。成都博物馆坐落在成都市区中心天府广场西侧的小河街西侧，与四川科技馆隔空相望，毗邻四川美术馆和四川图书馆。
博物馆要安检，因此为了方便，少带行李，不要携带打火机、食物品等。若带了可以去展馆旁边的寄存区寄存。成都市博物馆新馆的总建筑面积将达64945.87平方米，这将是西南地区规模最大的城市博物馆，而将这样的一个博物馆安放在城市中心地带，可谓体现了一个城市的内涵、眼光和远见。
博物馆有地上六层，地下一层，其中有一到五层为博物馆的主展厅，地下一层是临时画展，时尚展的所在区域，每隔一段时间都会不同。六层为办公区域，也是城市观光和四楼展厅，主要展览的是隋唐五代宋元时期的成都，四楼为近代成都的风俗民情，五楼为成都的川剧皮影文化。
我们进入博物馆后，最先到达就是我们博物馆的一楼展厅，这里主要一个休息区域，一旁是一个临时展厅，有两个购物点，在里面有很多关于成都特色的小物件，比如川剧变脸的小人，油纸伞等等。
乘坐扶梯，就来到了二楼的展厅，这一层主要介绍的是先秦到三国时期的成都，像展厅里面战国的船棺，巴蜀图语的铜勺，秦汉的石犀，都是这一时期的产物。
宝墩文化是迄今为止能追溯到的最早的考古学文化，是成都平原的一支重要的新石器时代考古学文化。宝墩文化可能是由营盘山文化发展而来的。宝墩文化在其自身的发展过程中，陶器制作工艺有所创新，同时还可能接受了来自其它考古学文化的影响。远在距今4500年左右，在成都平原就已经产生了作为蜀文化源头的新石器时代晚期文化--“宝墩文化”。宝墩文化是文明孕育时期的考古文化，宝墩遗址既是这一时期成都平原时代最早的古城址的典型，也是四川即将跨进文明门槛的历史见证。
在成都平原上相继发现的新津宝墩村、都江堰芒城村、崇州双河村和紫竹村、郫县古城村、温江鱼凫村等六座史前遗址群为代表。当时，古人的生产工具主要是石器，也能制作陶器了，主要有绳纹花边陶、敞口圈足尊、喇叭口高领罐、宽沿平地尊为标志。对上述6座古城的发掘与研究证明：它们与三星堆遗址的第一期属于同期文化，它不仅大大丰富了三星堆一期的文化内涵，能够再细分为若干期，并且可与以三星堆古城为代表的夏商时期的三星堆文化或古蜀文明相衔接。于是按考古学文化命名规则被称为“宝墩文化”。
来到秦并巴蜀展区，偶然听到这样一段话“这就是天府广场挖出来那个犀牛嘛，好壮实，长得乖得很。”在二楼展馆，两位老人围着天府广场石犀仔细地转了一圈，觉得它憨得可爱。
其实外表呆萌的石犀牛，可是大有来头。它最早被发现的时间是在1973年，天府广场钟楼挖地基时就被发现之后又被回填，这一“睡”又是几十年。2010年10月7日，代表一代人记忆的钟楼被拆除，将在原址修建天府大剧院。为此，2013年1月8日，石犀牛从大剧院工地请了出来。
它的耳朵、眼睛、下颔和鼻子仍然清晰可辨，局部装饰卷云图案，四肢短粗，身体浑圆，造型极萌。虽是呆萌可爱，但其个头却不小，石犀牛长3.3米，宽1.2米，高1.7米，重达8.5吨。
据考证，石犀牛由整块红砂岩雕刻而成，埋藏在西晋地层以下。最早埋于秦朝、最晚埋于西晋，但它的制作年代早于西晋。是目前成都市区出土的最早最大的圆雕石刻，具有极高的考古与艺术研究价值。出土之时，前国家文物局局长、故宫博物院院长单霁翔曾现场查看并感叹：“（石犀）在一定程度上改写了中国雕塑史。”
走完几层楼的展厅，我最喜欢的还是第五楼的皮影戏。这一展厅主要是民俗文化的展览，不仅有皮影戏、川剧变脸的展示、还有同一时期北方地区的民俗文化科普。白天走进这里，觉得每一处都很新奇，不同的剧影人物，它们身上各式各样的穿戴都觉得十分有意思。可是若晚上一个人走在这个展厅，就觉得是在上演恐怖片。
逛完博物馆，可以看一看，旁边的皇城清真寺。它是中国西南地区著名的清真寺之一,简称皇城寺,皇城清真寺总平面采用中国传统寺庙园林布局，南侧有幽深的庭院。建组造型虽为清初风格，但装修却严格遵守伊斯兰教规，不用动物飞禽，仅采用植物和伊斯兰经文。1997年6月，因建设需要被夷为平地。新寺融阿拉伯建筑风格与仿明清建筑风格为一体。
据考证这是清朝中叶所建，长9米宽4米的照壁耸立，大门广三间，正中门相悬“皇城清真寺”横匾，左右为斗门，门外有栅栏，门内过道两侧，各有逾两百年银杏一株，高大挺拔，枝叶盈庭；再进为三道回拱门，越横廊，过院坝，到达礼拜殿，大殿典雅宏伟，古朴端庄。

成都人的后花园——三圣乡

在我看来成都的慢生活并在人民公园，而是这个被误解为“打牌喝茶圣地”的圣乡。三圣乡距离成都市区15公里左右，算是成都的郊区。这里却有很多人过着陶渊明心之所往的田园生活。
三圣乡是许多网红民宿和私房菜的聚集地，除此之外也是许许多多公司团建的好地方。因为这里有许许多多的农家乐，可以喝茶打麻将，也可以骑行感受户外的氧吧。此外，三圣乡更是买绿植，花卉的好地方，毫不夸张的说这里的花卉仅次于云南昆明花卉市场。
春观桃花，夏赏荷。秋观菊花，冬赏梅。一年四季，景象万千。为什么说三圣乡是成都的后花园呢？幅员辽阔的三圣乡占地面积很广，由 “花乡农居”、“幸福梅林”、“江家菜地”、“荷塘月色”、“东篱菊园”五个主题组成的五朵金花是一个休闲度假的圣地。四季花开不断，蝶舞蜂飞，说它是成都的后花园，实至名归。
在三圣花乡，不仅可以感受到传统的乡村文化，又可以欣赏到优美的乡村景色，体验回归田园、拥抱自然的别样情趣。这里繁多的私房菜与网红民宿更是拍照的圣地。正因网红民宿与私房菜的在这里的流行，许多孩子的百岁宴、年轻人的求婚都会选择在三圣乡，因而这里仿佛是一个制造幸福的工厂。这里是上班族无数个周末欢乐的温暖港湾，也见证了许多人美好的回忆。
收起

一起回味《前任3》的电影碎片

时光荏苒，电影《前任3》的余温早已褪去，但是电影中出现过的地方，依旧是成都人爱去的地方，时尚的万象城，复古的东郊记忆公园，都是拍照圣地。
万象城在二环路边，这是一个时尚的商场，网红美食聚集地，仅次于太古里和大悦城。万象城的旋转楼梯和透明的玻璃门，楼中的空中花园，商场里的网红奶茶店每一处都是拍照的圣地，在这里可以休闲也可以时尚。遥想九月初，鹿角巷在万象城的开业，火热的长队伍，也是一道别致的风景线。
来到成都的东门，东郊记忆音乐公园就一定要去上一次。这里时尚会举行一些表演或漫展，即便没有这些东西，这一座时代遗留下来的复古公园也值得上前去走一走，逛一逛，拍一拍。
多年前的东郊记忆是兵工厂的聚集地，随着军转民、改制、搬迁，东郊记忆已经没有工厂了，老厂房却遗留了下来，成为了时代的见证者，改造成了如今的创意文化园，不禁勾起了许多土著民的儿时回忆。
一缕初冬的斜阳，一段黯淡的过往。如今的东郊意义公园有喧闹的酒吧，火爆的美食，还有夜晚里流浪歌手的清音。遗留下来的火车头成为了拍照的圣地，因此许多小姐姐，新婚燕尔多会来这里记录最美的时刻。

带不走的成都味，一次必须吃个够

成都是世界美食之都，就如同城市宣传标语一样“一座来了就不想走的城市”此言不差。成都的美食基本可以划分为三大类，其一苍蝇馆子；其二，小吃夜市；其三，就是时尚的餐厅。每一大类里，各种串串、火锅、麻辣烫、中餐都是交错纵横，玲琅满目。那么你肯定想说，该吃些什么呢？
在天府之国的成都，这六家颜值与品质极高的美食店，一趟成都行是不容错过的。作为一个在成都生活近十年的吃货，隔三岔五就要去打卡一次。
锅儿匠辣子鸡
“一斤辣椒两斤鸡，一生只做一道菜”光听他家的品牌理念，就知道这家吃鸡店不简单。在这种美食之都里，一家主打吃鸡的小店，在串串火锅独占鳌头的世界里一跃成名，成为了广大嗜辣吃鸡食客聚会的新选择。
“锅儿匠”是旧时川西坝子对操厨师手艺人的习惯称呼，简单的来说，就是乡村厨师。其实现在很多乡野名厨的厨艺并不比国内知名大厨差。这家吃鸡店的大厨，就把普通的鸡肉与辣椒相结合，烹饪出了青红椒辣子鸡、熊阿兵棒棒鸡、辣鸡黑椒意面等美味。银石广场5楼的锅儿匠装潢是中式乡村时尚风，十分迎合了店名，店口的海椒种类，科普了食客们对辣椒的认知。
青红椒辣子鸡是锅儿匠的头牌，事前码料让鸡肉更入味，大火急炒保持了鸡肉的鲜嫩，花生、芝麻又增加了几分香脆，鲜艳青红椒不仅让辣味渗入鸡肉，还让整个菜品颜色呈现出了和谐美丽，搭配店里定制的“金鸡钵钵”，吃鸡也有仪式感，超大一份，物超所值。这道菜堪称成都辣子鸡的翘楚，天下无双。
火爆的辣椒鸡肉穿肠过肚后，不妨来一杯盖碗陈皮红豆羹，软糯甜香老少挚爱。小米排骨健康养胃又好吃，又是一道老少皆宜的良品，照顾了 不吃辣的人群，十分贴心。听说很多人来成都都要去建设路的网红土豆排队打卡，其实来锅儿匠就这对。香甜酥脆的锅巴土豆，是我吃过最好吃的土豆了，完胜建设路某土豆。
熊阿宾棒棒鸡用古风的竹盘装载上菜，薄薄的黄瓜片，红油辣椒，分盘而置，最后在统一汇合，静置一段时间后，方可食用。同传统的雅安棒棒鸡相比这里的棒棒鸡吃起来更有仪式感。辣子鸡黑椒意面一道中西合璧与南北相汇的菜肴，解决了一顿餐食的主食问题。饭后小甜品，一杯盖碗红糖冰粉，解决酷辣与燥热，让这一餐吃的清爽愉悦。
如在蜀老火锅
“下雨天与火锅更配”是在成都经常可以听到的一句话。其实川渝人对于火锅的热爱不仅于此，山城网红火锅温泉足以证明这一点。在最繁华的春熙路，火锅品牌简直让人挑的眼花缭乱。一个在蓉城生活近十年，吃遍春熙路火锅的人告诉你，要吃资格成都老火锅，选择如在蜀老火锅，吃了你就忘不掉。
如在蜀老火锅是成都的资格火锅，中式淡雅的装潢，以“铺盖毛肚”论斤卖的特色，吸引了广大食客。地处春熙路商圈，潮流广场对面的如在蜀老火锅，临近文轩书城。不管你是个文艺青年，还是个时尚小姐姐，这家火锅都将你引入其中，坐下来这一餐火锅让你忘却室外的微凉。来过这家店的人都知道，其实春熙路店的如在蜀，就是一个翻版的“宽窄巷子”。店内外的装饰，就是一个复制的简版宽窄巷子。坐在里面吃火锅，再也不用人挤人的去宽窄巷子拍照了。
如在蜀的锅底是“透明”的，端上来后在加入高汤，打开火炉煮至沸腾就可以烫菜了。红汤锅底有资格的老豆腐，老而不绵，口感极佳。番茄锅底煮上手剪蘑菇，酸鲜健康。红酒肥牛鲜嫩多汁，甜豆花儿鲜香可口，小份儿冒千层肚麻辣鲜香，好吃到停不下来。
马路边边麻辣烫
在火锅、串串、麻辣烫、冒菜越来越相似，越来越分不清彼此的今日，吃到一家很地道的麻辣烫店是不容易的。马路边边麻辣烫一家怀旧又保持着原始麻辣烫形式的麻辣烫店，因而食客广聚。
十里飘香是马路边边麻辣烫的一个显著特点。致民路的马路边边麻辣烫靠近新南门车站，因此刚来成都或即将离开的过客，都会寻着味来到这里撸麻辣烫。最原始的锑锅，最古老的锅底，四处80年代的成都小街生活场景的墙画，眯上眼恍若真的回到了过去。
来到马路边边小卖部的糖油果子是不能错过的，小份儿冒千香气四溢层肚、冒花甲麻辣鲜香,吃完盘子的料不要扨掉，沾麻辣烫超好吃。喜辣就选传统锅，不喜辣就选经典锅，搭配种类繁多的荤素串串，让你的麻辣烫之餐更丰富。
醉西昌
成都人的夜生活除了灯红酒绿，必须来点烤肉。把把烧、烧烤串、韩国烤肉品类那么多，选哪家是个问题，那就来醉西昌吧，这里通通都有。
醉西昌中粮鸿运店是一家户内户外二合一的烤肉店，店内电烤无污染，店外碳烤有情调，灯光幽暗过道曲折，盆景全覆室内外，白天这里是花园烤肉，晚上这里就是blub烤肉。
醉西昌的招牌是醉虾，若不怕生鲜一定要点上一盘。活拨乱跳的小虾在秘制的酸甜味配料的浸泡下，鲜脆可口。碳烤青岛大虾，肉质紧实，外焦里嫩。碳烤鱿鱼须，浓香爽滑。碳烤生菜包牛肉，吃出了韩式烤肉的精髓，碳烤羊排吃出北方的豪迈，西昌小香猪吃出了邛海碧波秀丽。金针菇、韭菜、玉米棒子、西葫芦，素烧烤的四大员，缺一不可，沾上黄豆花生面儿，裹点辣椒面，味道资格。
醉西昌里最享受的就是服务员阿姨帮忙烤肉，负责吃就可以了，也可以自己亲自动手享受烤肉的乐趣，这样的食色夜生活妙趣横生。
集渔泰式海鲜火锅
据说来集渔用餐的小姐姐普遍好看、小哥哥大多高帅，所以一不小心下个转角就会脱单，其实这都不重要，重要的是这家海鲜火锅真的超好吃，汤鲜味美颜值高。
在晶融汇的集渔泰式海鲜火锅，开业当天就拍着长龙般的食客群，如今依旧是络绎不绝的食客。时尚雅致的ins装修风格是许多小姐姐聚会的首选，不仅可以吃到美味的美食还能拍出高逼格的照片，这样的宝地自然不容错过。
集渔泰式海鲜火锅是一家可以喝汤的火锅店，招牌冬阴功锅底让你免却长途飞行的疲劳，仍能品味泰式酸辣的美味。鲫鱼汤锅底，汤鲜味美，强生健体的首选。川式锅底，青红椒的辣味让你吃出“四川辣”的清新感。还有各式各样的甜品、小食、鲜果全程不限量供应。
烫菜可以点套餐，也可以自己单点蟹类、贝类、虾类、素菜类......小选主食香喷喷的海鲜炒饭不能错过，想要日常点的小吃，香脆软糯的虾饼。想喝一杯冰冰凉凉的饮品，气泡水是最佳选择。用餐全程有小姐姐为你烫菜，小哥哥为你剥虾，餐后来个哈根达斯的单球冰淇淋，这一餐海鲜盛宴的火锅就圆满了。
成都吃客
来到锦官城，火锅麻辣烫必不可缺，川菜自然也不可少。在繁多的川菜馆里，一家中西合璧的新派川菜“成都吃客”成为了许多人用餐的首选。不仅仅是因为这里简单大方舒适的环境 ，更因为这滨河路畔的新派川菜太诱人。
走进致民路的成都吃客，优雅的环境，香喷喷的烤鱼串，让你不自觉的就想坐下来享受美食。Q弹的裸奔爬爬虾熊孩子的最爱，去掉了虾壳不仅吃起来更方便，而且更入味。
三国风的圆盘冷吃牛肉，香辣可口，有嚼劲，下酒良品。纸上焗鲍鱼，这是一道每桌必点的菜，可以俗称“鲍鱼面”，因为里面的泡面实在太好吃了，还有蟹肉棒、五六个小鲍鱼、丝瓜片。把把烧式的小串烤五花，一口一个爽到爆。稀溜耙鸡爪，不在路边摊也能吃到的川式小吃。蒜香排骨，香嫩多汁。捞汁花甲，鲜嫩酸爽。藤椒鸭肠，麻辣鲜香。
这是一家道道是精品，口口是满足的新派川菜，想试试成都味，这家川菜界的翘楚，不容错过。
舌尖上的成都 ,好吃嘴的天堂 。在双十一剁手节前，先来一趟味蕾上的享受吧，过了双十一就要吃土了。
