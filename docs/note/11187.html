领略长白山天池绝美风光

长白山西景区
长白山天池是我国最高、最深、最大的高山湖泊，是由岩浆喷发拓成的巨型伞面体。这里夏季融雪，池水湛蓝，美的令人心悸；冬季白雪皑皑，冰面皓白，又是另一番模样。
长白山西景区
长白山西景区
长白山西景区
长白山西景区
长白山西景区
长白山天池每年的11月中旬开始封冻，至翌年6月末完全解冻，时间长达半年以上，这里的年均气温-7.3度，冰层最厚曾达3米，是一处非常神奇的地方。
长白山西景区
在亿万年以来的地质历史上，长白山地区经历了沧海桑田的变迁。最初，这里被海水淹没，到处是一片汪洋大海，后来由于地壳的上升，海水退出，地表重新露出水面，在阳光、雨水和气候变化等外力作用下，地面岩石遭受风化和破坏，最后长白山还经历了火山爆发和冰川的雕塑，形成今天的地貌景观。
长白山西景区
关于天池的秘密非常多，最流行的便是关于水怪的传说，尤其是在上世界，这是被人们津津乐道的话题，如今，每一个亲眼看到天池壮丽美景的人，都会不自觉的相信有水怪的存在，因为有太多秘密是自然和科学解释不了的。
长白山西景区
天池有“龙潭”、“海眼”之说，所谓“海眼”说是其与大海相通达，常有神龙出没。《长白山江岗志略》记载：“天池，在长白山顶…群峰环抱，池高约二十里，故名为天池。土人云:池水平日不见涨落，每至七日一潮，竞其与海水相呼吸，又名海眼。又云：天池水清浅处，可以行人”。又“数年前有猎夫自碧螺山下，渡至补天石旁，其中有热如汤泉，冷如冰海之处，五步外即深不可测，以足试之，滑腻异常，又名温凉泊”。
长白山西景区
天池中似有上古神兽，清代《长白山江岗志略》中记述：“自天池中有一怪物覆出水面，金黄色，头大如盆，方顶有角，长项多须，猎人以为是龙”，近些年出没比较频繁。只是，天池是火山喷发形成的高山湖泊，四周被十六座群峰围拱、草木不生，自然环境险恶；水中有机质及浮游生物极少，但在乘槎河里，也发现有生命体存在，个体并不大。它们是如何在生物链的顶端来生存的，是个叫人琢磨不透的迷。

打卡全景玻璃森系网红公厕

长白山是东北最知名的自然旅游景点，闻名天下的天池、美不胜收的原始森林和各式奇珍异宝无不吸引着国内外游客的脚步。与其他景区不同，在这里，“一个公厕就是一道风景”。长白山人民对于厕所的建设可谓严谨又仔细，从大大小小的景区到骑行道路的路边，每一个公厕都干净整洁，而且颇具设计感。有的时候，人们看到这些美的不像话的建筑，几乎不敢相信眼前就是公厕。
长白山风景区
有个位于长白山脚下的网红公厕就非常有名，本地人都把到这里喝咖啡当作梦想。很多外地人都会专程到这个网红公厕里上个厕所或者拍个照什么的，可以说是很特别的存在了。
长白山风景区
这公测的造型非常别致，透明玻璃镂空阶梯还有绿树环绕，这大概是整个东北颜值最高的公厕了。
长白山风景区
极具设计感的造型，白色玻璃与钢铁相结合，有点日系有点森。公测的周边是茂密的白桦树，夏季的时候，绿叶映衬这通体发光的建筑，非常有感觉。
长白山风景区
拍照最好看的当属比较高的那栋，因为有镂空楼梯的设计，看起来更酷炫！爬到厕所的楼顶就会发现，这里是一个非常赞的观景平台，可以远眺长白山。在这里凹造型也很顺手 ，发了朋友圈，大概每人猜到你是在厕所楼顶拍的。
长白山风景区
厕所里面也很讲究，由于吉林冬季寒冷，天气变化大，按常规建设和管理旅游厕所不适合。为破解这一难题，长白山建设了泡沫封堵、循环水生态环保厕所。生态环保厕所利用泡沫代替水，细菌群将排泄物中的有机物完全分解，并且很好地处理了厕所排放的污水，实现卫生间内空气高度洁净，消除视觉污染，让卫生间也拥有清新气息。
长白山风景区
其实，在长白山景区及城区看到的每一座厕所从外观上都各不相同，步移景异的原则是长白山旅游厕所设计的初衷，五十多座已建成和在建的旅游公厕都不一样，每个旅游公厕都有自身的特点。感兴趣的可以挨个去看看哦。

漫步静谧的二道白河小镇

二道白河镇
长白山脚下的二道白河，是一个来了就不想走的宜居之所。老人喜欢来这里避暑，年轻人喜欢到这里来骑行，对小朋友来说，这里有无尽的森林和虫鸟，是最自然的乐园。这里仿佛具有永恒的吸引力，让人在不经意间，因为一处处别出心裁的景致而心动。
二道白河镇
可以去美人松公园看最美的树。这个不算太大的公园里有美人松千余株，百年以上的古松352株。苑内有长白山特有的鹅卵石铺就的1500米长的甬道， 古朴的草亭，习习清风鸟鸣，片片山花烂漫，漫步园内尽可仰视美人松的洒脱风采。
二道白河镇
在民间，有这样一个传说，从前有一条恶龙霸占了长白山，天池的水不能流出来灌溉良田，人们受尽了干旱的折磨。一个木匠不怕恶龙的淫威，拿着斧子跳入天池，和恶龙展开了搏斗。木匠的妻子抱着水缸在山上等候，经过三天三夜的搏斗，恶龙被木匠砍下了头，天池的水又重新流了下来，人们得救了，可是木匠却再也没有力气上岸了，木匠美丽年轻的妻子不相信自己的亲人会死去，一直守候在山头，期待着丈夫归来，天长日久，渐渐变成了一棵美丽的松树。心地善良，忠实虔诚的人们，在月明风清，万籁俱寂之夜，会看到那棵棵美人松变成美丽的姑娘翩翩起舞，勇敢坚强的小伙儿，还会得到美人的芳心。
二道白河镇

二道白河镇
