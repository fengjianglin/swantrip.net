心动韩国丨我们都需要一次浪漫与烟火气并存的治愈之旅

游记 人文游 摄影 滑雪 美食 2019/02/20
看一场漂亮的烟花表演，去釜山吃一顿正经韩餐烤肉，逛完整条小吃街，去个性的买手店里挑一件喜欢的饰品，总之你可以在这里慵懒自在的旅行生活，发现小确幸之外的温柔自己。来韩国的机会并不少，每一次都有新奇的体验，这一次受旅游局邀请的韩国之旅，有心动、有刺激，三两好友在路边小店里的聚会，都让人难忘。

穿梭首尔大街小巷，我最喜欢你的腼腆

作为世界的第十大城市，首尔自然不缺繁华热闹，随处可见的高楼，大踏步走过的精英族，路边洋溢着青春气息的女孩子，和其他的大城市并无二致。但它可爱的地方在于你永远对这里有新鲜感，藏在每个拐角处的惊喜，总能让你重新向前。

北村韩屋村

北村韩屋村最早并非是供游客观赏修建的民俗村，而是韩国当地人们实际居住生活的村庄，但后来随着韩屋画廊、传统作坊、韩屋餐厅等不断的入驻，这里吸引了越来越多游客的目光。它是首尔最具代表性的传统居住地,且到处都有很多史迹、文化遗产和民俗资料。因此,被称为城市中心的街道博物馆。
北村韩屋村还是《冬季恋歌》、《绅士的品格》和《她很漂亮》等韩剧拍摄地，如果你来到首尔，那这里是你必不可少的“打卡景点”。
漂亮的韩屋群建筑大都是朝鲜时代视为上层社会人士拥有的住所，屋瓦房结构，算是两班贵族的居住地区。最初此地仅有几户有高耸大门的权贵人家和30余户的韩屋，然而现今所呈现的样貌大多主要是在日据时代末期及625韩国光复过后所流传下来的。
熙熙攘攘的人群，耳边传来特有的韩语调子，只要换上一身韩服，你就可以快速融入当地生活。
我们在附近的韩服店装扮好过去拍照，距离这里也就十分钟。我以为的韩服应该是仙气飘飘韩剧里大人的样子，如果你有朋友建议穿成我这种地主家的傻儿子风格去拍照的话，建议把这个朋友拖出去打死算了，真的上头。
这样的店很多，挑一家顺眼的进去，租借时价普遍在1.5万至3万韩元3小时（需要额外的押金），会有手巧的店员帮你免费做好韩国发式，男生只要多加一顶帽子，而我造型太奇特，放弃了这个选项。
站在这种文化街上，需要用心去静静体会品味，你看到什么，喜欢什么，它就是什么样子。旅行不是千篇一律的模仿，而是你我共同度过的美好时光。

街边的韩餐小店

一天的游览让人早已饥肠辘辘，酒店边上一家叫做Han-Ga-Ram的韩餐小店，纯韩语服务，中英都没有，还好同行的有韩语人才，成功解决吃饭问题。
这家店并不太大，就是普通的韩国餐厅，装修的很温馨，摆放在一边的书架也不会显得突兀，有种来朋友家做客的感觉。
 老板贿赂我们，送了小瓶的酒，叫百岁酒，是韩国很有名的一种清酒，配韩餐和中餐都可以。
牛肉鲍鱼上放的那个带叶子小萝卜是人参，可以吃的，一股草味，但应该会是中国人最喜欢的一道菜，我觉得还行。
下图的原样和正确吃法，一片鸭一片猪小段鱼用原味泡菜卷了一口吃掉口感丰富，就是吃起来麻烦，感觉女生不是很喜欢这个菜，可能因为鱼的部分是生的腌鱼的缘故。
两鱼双拼我很喜欢，其中一种是埋在菜丝里的，吃起来更像是酱烤鳗鱼，一点刺也没有。
还一些菜都没拍，要靠人翻译才能吃明白，非常地道的韩国餐厅，韩剧里的约会都是各种吃，米肠粉、韩牛、各种汤饭和海鲜，但是新鲜水果之类的比较少。很多人吃韩餐都想到烧烤和拌饭，其实一些特色的本地菜也是可以尝试的，能不能吃的惯就看自己了。

Nanta乱打秀

第好几次在首尔看Nanta乱打秀了，每次都不一样，这次居然还上台互动了一把，跟不知名的他国姑娘现场接了个婚，由于不让拍照，不上台真的拍照的机会也没有。
Running Man主题体验馆更适合情侣或家人一起去玩，一小时内满80分可以获得天下无敌奖状，满分88分，讲真，太难了，德智体美全面发展的才能做得到，现状我还是达不成的。
最后的仁寺洞爱来魔相艺术馆就在Running Man边上，来了顺便拍拍也好的，切记不要我这般臃肿的去。

爱宝乐园

爱宝乐园因为距离首尔市区较远，所以中国游客知道的不多。不过这儿地方大，距1976年开始营业至今已有50多年的历史，现由5大主题区组成，所以韩国本地游客特别青睐，想要玩个痛快必须得早早出发。
冬天的爱宝乐园人气依然很高，项目的玩乐难度都不高，最适合娇羞的情侣们来这里甜蜜蜜了。
最令人期待的“木制过山车”，冬天不会开放，否则你可以体会坐在过山车上听轨道木头嘎嘎作响的声音，一个将近90度的坡说不定会让你吓破胆。当然，如果你胆子比较小，你可以去旁边的VR室体验这个项目。
惊险刺激之余，去试试少女心爆棚的“ 旋转木马”，无论你是男生女生，在这里大大的旋转木马一定会满足你的“少女心”。IG照片打卡地“梦幻花园”，南瓜马车、城堡、花海等等，一次满足你的公主王子梦。
爱宝乐园的其中一个组成部分就是野生动物园，坐在老虎形状的大巴里被动物们围观。不过只有几种大型动物，它们的生活空间还是挺大的。老虎狮子都极高冷，完全不为所动，只有熊的配合度极高，走路转圈想怎么来怎么来。要不是提醒太大，司机大叔可能还要亲亲抱抱举高高。
入园的时候我们按照提示下载了naver地图以及everland，这两个APP是配合官方园区出的专门的APP，直接搜索就可以，APP是中文韩文双语的用起来非常方便。
更重要的是上面会有关于全园区的项目位置，表演时间地点，以及餐厅地点。让你不会错过每一个精彩的节目，更方便你提前做攻略。

食欲釜山，在这里吃的每一口都幸福炸裂

能吃的好的旅行才算美满，所以每到一个地方，我也会花很长时间去专注于寻找当地美食。而釜山就是一个神奇的地方，在这里就可以满足你对韩国食物的所有期待。

釜山烤肉

我们最先选择的是釜山烤肉，来了韩国不吃一顿正宗的韩牛怎么行。
釜山烤肉No1，海云台有名的母牛排骨店，这个名字也是有点奇特，几十年只卖一款烤肉，一共三种肉可以选择，韩牛排，调味牛排，烤肉，就没有了。
就这么简单的菜式，这家店在釜山的夜色里可都是爆满的节奏。我们点了和牛，配上甜酱超级好吃！
烤完可以配肉汁涮一波土豆面，味道也是美极了。

甘川文化村

在1950年代太极道信徒和6.25避难人民共同聚居形成的甘川文化村，如今经过政府和艺术家们的合力打造，已变成了韩版“圣托里尼”，成为了釜山的代表景点，更有人说这里是亚洲最美村庄。
在旅游咨询信息中心买了一份打卡用的2000韩元的地图，开始了我们在甘川文化村的寻宝之路。
这样一份地图不仅仅包含了线路信息，更重要的是打卡盖章式的，如果你去过厦门，那么对此一定不陌生，等你盖完所有的章，回过头来会发现，你已然走完了这一个整个村，已经收获到了满满的景色与照片。如果你来到了釜山，那么这儿一定不可错过。
花上半天的时间去惬意的感受这个村子彩色小屋层层叠叠，巷弄错落，和村里的“网红”小王子和小狐狸的背影雕像合照，逛一逛村里的咖啡店，精品屋和小吃店。
最后如果你有时间，万里晴空，可以登顶俯瞰整个村落，静静等待落日余晖为村落刷上金色的彩漆。

“1953鱼糕制作”

除了甘川文化村，在釜山你一定还有一个需要知道的，那就是“1953” 。鱼糕是釜山最具代表性的食品，好吃的鱼饼也是多种多样。有人说如果你不知道1953，你的釜山之旅会有遗憾！如果没吃到1953,你的釜山之旅会不尽兴！
三进鱼糕创始于1953年，是釜山乃至韩国历史悠久的鱼糕品牌，因此在他们的店名上就包含了这个年份，如果你在釜山听到“1953” 那么一定指这么一家富有历史的鱼糕品牌店。
三进“鱼糕烘培坊”的主打产品是“黄金鱼球”，这个产品是他们首创的，在金黄色的外皮下包裹着芝士、山芋等多种夹心，口味丰富口感Q弹味而又颜值，一度受到了吃货的疯狂追捧。因而影岛店的这家体验馆逐渐成为了釜山的必到景点之一。
影岛总店以西饼店形式运营，顾客自助选取喜欢的鱼糕后，可以打包带走，也可以拿到用餐区直接食用。
吃完之后去二楼，原来这里还设置了鱼糕体验历史馆，买了一个体验课程尝试着自己做了一下鱼饼，一边听师傅讲三进鱼糕厂的历程，一边学习鱼糕制作。
其实也很简单，把准备好的巨型鱼丸一顿抽打摩擦，再给上面涂好彩色面条，就会被送去蒸炸烹饪，最后等半小时出炉就能直接吃了，现场感很足。
请千万要吃完别带回房间，离开这个环境之后会觉得味道怎么这么大！

札嘎其海鲜市场

我的旅行奇怪理论之一就是：像当地人一样去旅行就要去当地市场。
于是在釜山我们就去了当地最有名的海鲜市场，寻找最新鲜的食材。原本的札嘎其海鲜市场指的是从釜山影岛大桥底下的干鱼市场至忠武洞早市区附近一带。
在朝鲜战争时许多的避难者都迁徙到了现在的釜山，那时南浦洞成为当时釜山的中心。而后随着多年的持续发展，变成札嘎其海鲜市场。
现在的札嘎其海鲜市场，由于靠海，各种各样的海产品非常丰富。当然也有不少进口的海鲜，在这里，所有的摊位.上会将产地清清楚楚的标明海鲜的价格。这里的海鲜个头巨大，种类繁多，海鱼、螃蟹、龙虾等，都让人眼花缭乱。
作为当地黑暗料理闻名于世的应该是活章鱼，切碎蘸酱直接吃，我也不知道自己为什么要做这个挑战。挑好章鱼之后本来还想自己试着剁一下，结果对面的大妈太利索，分分钟就剁碎了，直接放在一次性餐盒里，给几根签字就可以带走了。
刚开始的10分钟特别剧烈，放嘴里很吸。而且吃完之后次日就有4个人微恙，一定是受到了章鱼的诅咒了吧。

奇妙甜甜圈

在釜山一直食欲很好，想到什么好吃的就立马去找，感觉整个人都胖了一圈，纠结了一下午还是没抵挡住诱惑，出门觅食了。
想找一个打卡点，在附近转了很久都快绝望了，才发现是这家甜甜圈店门口的小邮箱里面，太作弊了！
由于逗留了一会儿，秉承一名优秀游客“来都来了”的优良传统，我们就进店开买了。说是甜甜圈，实际上就是小粒的各种口味年糕，快吃完了才反应过来，这个甜点做得好厉害呀！门口那只被书包的狗很多人都有拍过，感觉起到了招财猫的功效。
食物的治愈力是最强的，只要当下满足自己的味蕾，就能产生超级温暖的能量包裹着你，去抵抗严寒。

没有汗蒸房的韩国之旅不完美

无论你是不是韩剧迷，一定不会陌生韩国的汗蒸文化，入乡随俗，我们也大着胆子进了韩国汗蒸房，迷迷糊糊的来了一次汗蒸体验。
Spaland是釜山当之无愧的三温暖胜地，韩剧里能看到的包着羊角帽满地滚，吃着温泉蛋喝着酒酿饮料的画面在这里全部都能实现。
spa land分两层空间很大，工业风的装修设计的非常明亮，十几间不同类型的汗蒸房任君选择，这里甚至还有冻的汗蒸房。
一楼分男女部，里面有淋浴和热水池，公共区域有汗蒸房，热石地板等，户外有温泉足浴池。二楼大多数是收费项目，有spa按摩，理疗床，餐厅，咖啡厅等。
盐巴房是最好看的，满目红砖的就是，不过温度也不低，汗蒸的时候根据自己的实际状况而定哦。

寻找舒适住宿

釜山新罗酒店就在海云台，周边随便走走都是各种打卡景点。一看地图，原来大海就在旁边。装修风格看起来冷淡却又多彩，陈设多样然而又很和谐。有一只比成年人还大的布偶熊坐在大堂的火炉对面，一个小朋友一直在玩着，也就不好意思拍照了。
釜山的游艇大概是全世界最可爱的吧，以前体验过的游艇会让人觉得各种高级，然而这里的游艇会感觉到各种温馨，只是冬天湿冷的雨少了好看的蓝天，所以就窝在船舱里和大家一起下午茶。

阳智松林滑雪场

任性的吃了很多东西，终于醒悟了需要运动，查了攻略找到离首尔很近的阳智松林滑雪场，开车差不多40分钟就能到，还有免费的班车可以搭乘。
穿上装备，不管会不会滑，气势得拿出来！
新手村就是阳智Pine度假村滑雪场，场内设有7条滑雪道，倾斜度在5~35度之间，可满足初学者及滑雪高手的不同需求。为初学者设置的黄色路线是国内最高的斜坡，配备救生用品，坡度平稳、均匀，非常适合初学者进行滑雪练习。
虽然上过几次滑雪课，但是我还是佛系操作，随缘吧。

收官韩餐

安排的满满当当的韩国之旅让人意犹未尽，最后我们去了当地人最喜欢的传统韩餐厅之一자연밥상作为结尾，而这家店非常傲娇，连英文名都没有。
落地窗下就是大海，装修的风格比较年轻化，据说就是为了让更多年轻人从喜好西餐的潮流中回归韩餐而作出的改变，就餐的年轻人也确实很多，只是我们这些外地人连菜单都看不懂，还好有随身翻译，大家如果要去吃的话提前学习下韩语吧。
脆煎腌鱼是最贵的一道菜，也是我吃最多的。
喝几口清酒，再夹上一筷子鱼肉，伴随着食物咽下，旅程尽头的不舍感也扑面而来。从一开始慢慢悠悠的逛街，到后来认识新朋友一起寻找喜欢的餐厅，然后一起放开了玩，旅行的意义从来都不是逃离原有的生活，而是让我们收集温暖和善意，带着这些美好的经历生活，都会让我们接下来的日子更加坚定和清晰。
