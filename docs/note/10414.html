绍兴媳妇儿带你逛绍兴：必打卡和新发现

游记 人文游 古镇 摄影 美食 自助游 2018/11/28

前言

绍兴，一个很难用一两个词语来定义的小城。
这是一个人文气息很浓的小城。从古到今，绍兴地区名人辈出，大禹治水的大禹，卧薪尝胆的勾践、四大美女之首的西施、著名书法家王羲之、诗人陆游等......近代最出名的要数曾经的北大校长蔡元培、著名的民主革命志士秋瑾等......还有周恩来、鲁迅就更不用提了，要让我列出绍兴的名人可能可以列出长长的一串。
绍兴也是一个很有文化底蕴的城市，从唐代开始李白、杜甫等450多位诗人在绍兴的山水之中游走，留下了众多诗文，及至宋、元、明、清，又有大量文人墨客来到绍兴，写下了众多诗歌，其中唐诗就有1500余首。
绍兴是一个很适合带着孩子来研学的小城，《从百草园到三味书屋》、《社戏》、《故乡》、《兰亭集序》、《回乡偶书》、如果说绍兴是在课本里出现过最多的城市之一，应该也没有人反对吧。
作为一个绍兴媳妇，因公因私我也来过绍兴很多次，不过每一次都没有好好地把绍兴走一走看一看。这个仲秋，我终于把绍兴又重走了一遍，也终于把一些必打卡的地方都成功地打上了卡。

交通

住在江浙沪地区的人，去绍兴最方便的当然是自驾咯。不过如果没有开车也可以乘坐高铁，一样很方便，从杭州坐高铁只需要19分钟就可以到达绍兴。绍兴是个不大的城市，乘坐出租车价格也不会太贵，市内交通也很方便，有些景点离的很近，只需要走走就可以到达。

游玩

绍兴古城：
绍兴是座水乡，古城四面环水，内河纵横交错，被誉为“漂在水上的古城”。在绍兴古城内就有好几个游船码头，乌篷船作为水上特色交通工具，也是备受欢迎的游览方式之一，有时间可以感受一下坐在乌篷船上看古城的感觉。
古城内古意盎然、绿树成荫，白墙灰瓦的民居建筑保存很完整，走在小城里能感受到这里浓浓的生活气息，这种生活气息与大城市里截然不同，这里的人不像大城市里的人，永远都行色匆匆，走在绍兴古城里，老城居民们不紧不慢地做着手头上的活，晒晒菜干、藕粉，话话家常，在这里你能感受到久违了的那种慢生活。
走在街头，很容易看到戴着绍兴特有的乌毡帽的中老年男人，这种其貌不扬的帽子之前只在书上、电视上看到过，没想到这绍兴还是很受当地人的欢迎。
绍兴是明代哲学家王阳明的故乡和心学发源地，名副其实的阳明学圣城。这里不仅是王阳明成长和归葬之所，也是他生活时间最长、留存遗迹最多的地方，更是阳明思想发端与成熟之地，王门后学影响极大。
在古城内就有一个王阳明纪念馆，我们也进去看了看。
在王阳明纪念馆附近还有王阳明观象台，是一座用石条砌成的用来占卜星象、认证天体的长方形高台，南北宽十余米，东西长二十米，形同假山，有时间也可以走过去看看。如今这里已经被翻修过，变成了研究、宣传王阳明心学的一个场所。
站在观星台上向下瞭望古城，白墙灰瓦的民居建筑保存的很完整。
绍兴是书圣王羲之的故乡，古城内还有书圣故里，这座戒珠寺就曾是王羲之的故居。
戒珠寺对面有一方水池，名为墨池，据说就是王羲之当年洗毛笔的地方。
蔡元培故居位于绍兴古城区萧山街笔飞弄13号，这是一座中国明清建筑风格的宅子，里面还建有蔡元培纪念馆，纪念馆里有蔡元培的雕像和生平事迹介绍。
花格门窗的建筑风格很有中国特色。
小小的一个绍兴古城，花上半天就能走遍。比起参观各个名胜古迹，我却更享受穿梭在古城小巷中的时光，只有在这小巷中才能感受到浓浓的小城生活气息。
鲁迅故里
众所周知，绍兴是鲁迅的故乡，这里有很多鲁迅描写过的地方出现在课本中，来过绍兴那么多次，不去鲁迅故里打个卡还真的说不过去。所以趁着晚饭后的时间，我们又去鲁迅故里转了转。
鲁迅故里是个免费景区，占了鲁迅中路的一条街区，虽已是晚上，鲁迅故居还是开放的，只是需要在门口扫描二维码预约后才能进入。
鲁迅的卧室：
曾经在课本里看过的百草园、三味书屋，现在终于可以看到真实版本了，不知道大家觉得和想象中是不是一样呢？
比较有意思的是在鲁迅故居里还有出售鲁迅的周边文化产品，也是蛮有趣的。
安昌古镇
兴柯桥的安昌古镇，虽不如江南六大古镇有名，不过它独具特色的本土韵味和没有那么强的商业化气息让这个古镇和其他著名古镇相比，特别的让人流连忘返。
安昌古镇是绍兴历史最悠久的一个古镇，始建于北宋时期，它的建筑风格传承了典型的江南水乡特色。
其实，早在四年前的夏季我就曾经到这个古镇游览，不过据说，最美的安昌古镇还是在冬季。安昌古镇的特产是安昌腊肠、各类酱货和扯白糖。每到秋末冬初，各家各户都已经开始腌制和晾晒腊肠以及，各种酱鸭，酱猪头，酱鱼干，这次我们去刚到古镇门口，就看到了满街都是各种酱货和腊肠的盛景。
古镇的老街沿河而建，沿着青石板路向前走，河的南岸是民居，河的北岸则是商市，两岸之间以古桥相连。河岸边的长廊都有顶棚，遮阳挡雨，现在已经改成各种传统特色的店铺作坊。
年关将近，当地居民都拿出了自己制作的各类腊肠，和酱货、干菜出售，整条街上有上百家腊货店，这里的腊肠都是当地居民手工制作的，香肠灌好以后就挂在房前的河边通风，走在街上，到处都是扑鼻的香味，让人不禁口水直流。
店主们在自家的店铺里守株待兔，不需要吆喝或是做广告，自有来自各地的游客会上门选货、购买。
走累了，可以坐在长廊边，一边赏水乡河景，一边品尝一碗当地特色的面条馄饨等各类小食。
安昌的小桥也非常有特色，千姿百态，古朴典雅。走在小桥上，看着戴着绍兴乌毡帽的船老大，摇着乌篷船从桥下。穿梭而过，这一刻感觉时间仿佛回流了。
安昌还有一样特产，就是酱油。安昌最有名的酱油就数仁昌牌酱油，这里制作的腊肠都是用仁昌牌酱油来调味的。可别小看仁昌酱油，一瓶就要三四十元，不比日料店里的日本酱油便宜多少。仁昌酱园，也在安昌古镇上，从酱园的窗户向里望去，一个个大大的酱缸，排列的整整齐齐，里面都是等着发酵的酱油，这种纯手工酿造的酱油可比配制酱油健康美味的多了。
绍兴是一个很宜居的小城，这里的人民很享受自己按步就班，平平淡淡的小日子，这里的夜生活并不是那么丰富，不像其他那些著名古镇，安昌古镇一到傍晚5点左右，各家店铺都已经准备打烊，古镇也从热热闹闹变得冷冷清清，想要感受真正的水乡古镇生活，来安昌古镇不会错。
东方山水乐园
在绍兴柯桥区南大池的岛屿上有一个东方山水乐园，据说耗资80亿历时四年才建成。没想到这样一个小城市也有这么高端的乐园呢。
乐园由五个庞大的“鹅卵石”状的建筑体组成，还有一家配套的超五星的东方山水金沙酒店。
记得几年前第一次去韩国首尔，第一次感受乐天世界室内乐园的乐趣，当时真的是记忆深刻，觉得乐天世界超好玩，没想到现在在杭州周边的这个小城市里也有了这样好玩的室内乐园。
东方山水乐园分为陆公园和水公园两部分，全部都在室内，所以完全不会受气候因素干扰，一年四季都可以玩。
一进入公园入口，我就被大堂秀所吸引。大堂中央超大的环状LED圆柱运用先进的声光电技术，打造了一场感官盛宴，把人们带往一个如梦如幻的梦想世界。
因为没带泳衣，这次就没有去水公园游玩，而是重点游玩了陆公园。陆公园又分为山之王国、梦想世界和冒险王国以及地下晶窟、珍宝博物馆、国防教育体验馆等。说真的，这个乐园真的很适合带着孩子来研学、游玩。也难怪，虽不是双休日，乐园里的大小孩子都很多，由老师和家长们带领着欢乐地穿梭在各个区域里。
坐在源力飞艇上游客可以自上而下清楚的看到馆内全景。
刺激的幽灵古船（海盗船）摆角的角度特别大，坐在上面游客仿佛置身于怒海惊涛中的飞舟上，这个项目不仅孩子爱玩，喜欢刺激的年轻人也很喜欢。
乐园内每天还有好几场精彩的节目表演，演艺人员还经常会在园区内巡回与游客进行互动。
在国防教育馆里，游客们更可以近距离接触到国内最先进的军械装备，亲身体验航空返回舱，全新的4D影片技术能让游客身临其境的感受到科技文明带来的震撼力。在这里游客们能感受到祖国国防的强大，装备的先进，这一区域特别受男孩子们喜欢。
不过，我更喜欢的位于“山之王国”二层的珍宝馆，馆内汇聚了众多的奇珍异宝与古玩。 阴沉木、景泰蓝、猛犸象牙等都是馆内的精品，估计这80亿的投资里有很大一部分都砸在这个珍宝馆里了吧。
令人没想到的是在山之王国的地下还有一个地下晶窟，洞内有石笋、石塔、石花等钟乳石，水晶幻境在灯光的配合之下，显得特别璀璨梦幻。
园内还有一片热带雨林区域，逼真的装置让人误以为真的踏入了大自然之中。
胆子够大的游客也可以去鬼屋八十一号裁缝铺挑战一下，反正我这个胆小鬼是不敢。
整个乐园一共有好几十个游乐项目，虽然趁着乐园的知名度还不够大，游客不算太多，不用排太长的队，不过想要一天把各个项目都玩遍也是不可能完成的任务，好在乐园离杭州、上海、宁波等城市都不算太远，如果有时间完全可以一来再来
十里荷塘
和前面介绍的鲁迅故居、安昌古镇这些有名的景点比起来，十里荷塘，知道的人少多了。
十里荷塘位于柯桥镜湖湿地公园西面，虽不是夏季，池子里只剩下一些残荷，却别有一番风景。
这是一个很安静的地方，闲来无事可以来这里走走，呼吸一下新鲜空气，释放一下心情，享受一下独处的感觉。
景区里还有一片芦苇荡，是拍照的好地方。
棠棣村
如果在绍兴时间还有多，可以去棠棣村看看。这个村庄被称为“绍兴花木第一村”，村里95%以上的劳动力都直接或间接地从事着花木的生产和经营以出产兰花出名。通往棠棣村的公路上开满了各色鲜花，被誉为绍兴最美公路，如果骑行在这条路上心情一定会很舒畅吧。
绍兴地区还包括新昌县、嵊州市和诸暨市。这一次，我们还顺便去诸暨走了走。
其实诸暨我也去过好几次了，诸暨最有名的景点要属西施故里和五洩景区，这两个景点我也是早就去玩过了，所以这一次我们就去了两个不是那么著名却同样很有特色很有趣的景点。
米果果小镇
米果果小镇是一个适合周末亲子游的度假地，小镇把农业和旅游结合的非常好，园区分种养功能区、种养废弃物资源高效利用功能区、农副产品深加工功能区、乡村休闲旅游功能区、农特产品展销功能区、青少年农业科普教育功能区等六大区域。
别说是孩子，就是对于久居城市的成年人来说，米果果小镇里的很多东西都让人感到很新鲜。在米果果小镇真的可以学习到很多农业知识，也难怪这里有这么多的带孩子的游客，其中还不乏很多外国面孔。
香榧森林公园
香榧是诸暨的特产，因为香榧的产量很少，所以这是一种价格挺高的坚果。相信很多人虽然吃过香榧，可能还从未见过产香榧的树的样子。
在绍兴诸暨的会稽山脉深处，有一座香榧森林公园，在那里可以看看香榧树到底是怎么样的，这一次我们也去了这座公园。
香榧森林公园内有一片香榧林，古老的香榧树盘根错节，有的扭曲多弯，有的高耸入云，令人百看不厌。
要知道香榧树都很高，采摘可不是一件容易的事，有棵香榧树下有一个木梯，是采摘香榧的工人登高采香榧的工具，作为我们普通人来说光看看就觉得很危险。
其实11月底已经过了香榧采摘的季节，不过公园里还有一个香榧文化馆，可以在里面看看，了解一下各种香榧的形态和区别。
香榧森林公园内山清水秀，风景很好，还有很多景点，比如黄四娘潭、黄檀溪景观都非常迷人，在里面走走看看，呼吸一下清新的空气真的是好享受。公园内还有一座吊桥，有兴趣有胆量的游客也可以去走走。
整个公园内植被丰富，深秋季节正是银杏最美的时候，也可以去赏一下黄灿灿的银杏叶。

美食

咸亨酒店
到绍兴来怎能不去咸亨酒店打卡，原来咸亨酒店是鲁迅叔叔家开的啊，之前还真不知道。这家酒店的环境不错，门口立着孔乙己的雕像，里面的柜台上还挂有孔乙己欠十九钱的欠条一张，很有点意思。餐厅的装修也挺有风格，用的餐桌都是绍兴传统的八仙桌，对我们现代人来说稍显小了一点，就算只坐八个人都显得满满当当，菜也放不下。
咸亨酒店的餐具也很有特色，上面都有咸亨的标记，连盛饭的木桶都看起来很古朴。
绍兴当地菜口味以咸、臭为特色，臭豆腐大家都吃过，不过霉菜管可不是什么人都吃的下的，反正我是难以接受，另外孔乙己吃过的茴香豆也是必点的菜肴之一。这家餐馆的菜口味都比较重，很多菜都偏咸，吃的清淡的人可能会不习惯，好在作为绍兴媳妇这不过种咸度对我来说不在话下，这才是最正宗的绍兴特色啊。
古越龙山酒楼
来到黄酒小镇绍兴，怎能错过品尝这里的黄酒，以前只知道古越龙山是著名的黄酒品牌，原来它家也开了酒楼，酒楼的装修很有酒厂的风范，放了很多黄酒坛子做装饰。
既然来到了古越龙山，当然要品尝一下古越龙山的黄酒，各种黄酒来一波，这次一下试了六种酒，不会喝酒的我还是喝白色的米酒更合口味，不过好酒之人应该能品出各种酒的差别吧。作为酒厂开的酒楼，感觉连这里的冷菜醉枣都比别家餐厅的醉枣更醉一点。黄酒配着绍兴特色的醉鱼干，臭豆腐，茴香豆，梅干菜扣肉，才是最正宗的吃法，很满足的一餐。
另外我们还去了两家虽不是非常知名，却在绍兴口碑不错的饭店，当地人也经常会在这两家餐厅吃饭，无论在哪一家吃，炸臭豆腐、醉鱼干和梅干菜扣肉都是少不了的，看来这些菜都深深地印入到了绍兴人的基因里，真正的绍兴人不可一日无臭，不可一日不吃梅干菜和鱼干这样的腌制品啊。
波影大酒店
风光大酒店
在诸暨的米果果小镇，我们直接吃了当地的农家菜，菜的份量十足，味道也十分地道。
其中火龙果米糕是米果果小镇的特色。
小吃
虽然在饭店里几乎每顿饭都有臭豆腐，不过总觉得街边现炸的臭豆腐才是最正宗的，所以我们在安昌古镇又买了臭豆腐吃。
绍兴的黄酒也是非常出名，所以也会有很多用黄酒作为原料做的零食，比如黄酒棒冰、黄酒奶茶。
据说最有名的黄酒奶茶是由三个年轻人开的一家在鲁迅故里景区附近的名为“景绘轩”的店出品的，这家店已经成为了网红店，去绍兴的年轻人都会去打卡品尝。不过绍兴人生性比较安逸，一到晚上五六点就下班了，这家店也是一样，每到傍晚就准时打烊，虽然我存着侥幸心理晚上去咸亨酒店想去碰碰运气，最后还是遗憾而归。
不过黄酒棒冰我还是买到了，虽然叫做黄酒棒冰，不过一点不像想象中那么醉人，咬起来甜甜糯糯的，还有一股淡淡的酒香，味道还真是不错。

住宿

绍兴是个富庶的小城市，这里当然也有不少中高端的酒店。不过，居家过日子总是量力而行比较好，这次我们住的酒店个人认为性价比就很高，我表示很满意。
开元曼居大禹店
开元曼居是开元集团旗下的品牌，价位不高，不过舒适度还是很高的。在绍兴有好几家开元曼居酒店，这一次我们住的是大禹店。
绍兴作为文人墨客和各种名人辈出的城市，这家开元曼居酒店也很有文化气息。
一楼的大厅挑高很高，装饰物都是具有中国文化特色和绍兴特色的一些文化产品，有着浓浓的中国味儿。
酒店还是很注重安全和隐私保护的，坐电梯需要刷房卡。我们的房间在七楼，开元曼居的客房一般都不算太大，不过一点不影响其舒适性。
房间内的整体主色调是米色、灰色和驼色，看起来很明亮，简洁的装饰却不影响日常的使用，让人心情也很舒畅。
客房内的装饰也运用了不少中国元素。
相比客房，开元曼居的洗手间还是比较大的，干湿分离的设计让使用特别方便。浴衣是不同颜色的，以便于住客区分，酒店的设计人员也是很有心。
酒店餐厅设置在二楼，对于我这样的中国胃来说，有现煮的面条、馄饨就是另我满意的一餐。除此之外，早餐还有不少绍兴当地特色的食物，比如梅干菜扣肉、醉鱼干等，让外地游客不出酒店门就能尝到当地美食，必须赞一个。另外，像面包、果酱、饮料、酸奶等也都有供应，相信每一个住客都能吃到适合自己的食物。

写在最后

三天两夜的绍兴之旅就此结束了，这是一个能让你心静下来的城市，这里的居民做什么事都不紧不慢的，晚上也没有太多的夜生活，不过我觉得这里丰富的人文资源还是值得来看一看的，特别是带着孩子，可以感受一下跟着课本游绍兴的感觉。再加上现在绍兴又开了很多适合孩子的娱乐场所，让绍兴游有了更多选择。
