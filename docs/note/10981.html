新版旅图使用指南

游记 摄影 2018/10/23
新版旅图正式上线，小编给大家一一介绍下新版旅图以及正确的发布姿势！

新版旅图有何不同？

新版旅图，把过去一张张分散的图片集合成一个图集的形式呈现出来，更多照片也能分享到朋友圈啦！更能添加大段的文字记录图片中的故事、旅游时的心情。小编告诉大家一个秘诀，图片和文字描述越多，绑定的地点和标签越准确，不仅能被更多人看到，参加活动获奖的机会也越大哦~
PS：图集也是一个小帖子，你可以以某个城市，某个景点等等为你的主题内容，来发布图片和写描述内容哦！

新版旅图发布指南

见下图，即旅图发布页面，一次可发布20张图片，点击查看大图，可删除图片。

新版旅图新颜值

左右滑动浏览图片，切换同一个帖子的图片
顶部右侧增加关注按钮，让你可以快捷关注所喜爱的作者。
文字为整个图集的描述，不随图片的切换而改变。
只想看纯净的图片？点击一下图片，将收起除图片外的其他界面，世界顿时变得纯净~~~
介绍完新版旅图，大家是不是很想体验下新页面呢，打开天鹅之旅APP去发旅图吧~

发布步骤

打开天鹅之旅app—点击页面最下”发现” —点击发现频道右下角绿色图标—选择“发旅图”—撰写标题/内容/选择图片—选择对应位置和标签—勾选“同意天鹅之旅内容产品用户协议”—发布即可
