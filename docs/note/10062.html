羁绊人心的大容山

 
赏识了桂林山水甲天下,追逐过北海银滩的浪花朵朵,我曾以为这就是广西的缩影。兜兜转转,仲夏之际,初次造访“荔枝之乡”北流市。妙游山清水秀的大容山,淡雅脱俗的陶瓷小镇,神乎其神的铜石岭,趣味无穷的五彩田园之后,方知过往的想法真是贻笑大方。广西这个地方的美景之多,确实不容小觑。仲夏之际匆匆一游后,徒有那判若置身北欧秘境的大容山,最让人难忘。圣诞之际再次仙游大容山,其中的喜乐不能一一言表。

岭南第一谷

 
山高云深的大容山暗藏着一个幽谷圣地,从景区步行登山的第一站就是九瀑谷景区,这里林深,水绿,谷幽,置身其中恍若徒步热带林。一路上妙趣横生,可观仰“树包石”的神奇奥妙,倾听她背后的动人传说,亦可亲密无间观赏植物活化石桫椤树,漫步曲径通幽的竹林,聆听那一片片芭蕉林后,时而悠扬婉转,时而高亢激昂的山泉之韵。
 
山的王国,水的天地。藤木拥簇,瀑布遍野。清溪幽谷,奇石异花。曲径通幽,梵音袅袅。时至寒冬,这里却一抹清幽,生机勃勃。九瀑谷景区“岭南第一谷”的美誉实至名归。

雕刻艺术的宝库

 
古有“孟母三迁”,今有“古宅三移”。从清幽的九瀑谷返回到景区入口的桫椤民宿,民宿一旁一座清徽派建筑就引人注目,这便是大容山第二大珍宝的江家大院。江家大院祖级江西,在广东“沉睡”12年后,又“移居”玉林大容山的清代徽式古宅江家大院,终于不在波折动荡,山幽水美的大容山成为了她的最终“归宿”。
 
远看气势恢宏,近观巧夺天工。灰墙、黛瓦、粉壁是她最动人的外表。“砖雕、木雕、石雕”三雕艺术是她待人发现的内在美。高宅、深井、大厅是她庇护的堡垒。这座珍奇的古宅是岭南雕刻艺术的宝库,具有极高的研究观赏价值。

异域风情的欧陆风光

 
号称“南方西岳”的大容山森林公园,却更有异域风情。寒风凛烈,跟随着蜿蜒盘旋的公路,深入大容山腹地,领略那近峰巍峨,远山如黛的景致。车行山间,漫游山间那绿如碧毯的草甸,湛如明镜的湖泊,快如欢歌的溪流,势如龙腾的瀑布。这番雅致幽深源的美景,它历经千年积淀,景象万千,无所不包容,故名“大容山”。
 
汽车戛然而止,映入眼帘的是一座小教堂静谧的毅力在绿草甸上,溪湖围着杉树静静的流淌,宛然一副欧洲小镇的韵味。不久,便抵达来大容山国家森林公园的核心之地莲花景区。恰逢圣诞之际,这里被五彩缤纷的道具装点着,乍一看像级了童话世界的圣诞节狂欢夜。
 
晴空里的莲花景区,天高云淡,风和日丽。茵茵绿草边碧湖随风荡漾,与高耸入林的奥地利风情小教堂,组成里一幅美丽动人的欧陆山水画卷。阴雨中的莲花景区,朦胧缠绵,幻如仙境。白纱笼罩着神秘的森林,俯身看去,晶莹的露珠滚动在杉树叶尖儿,杉林间雨雾交融神秘幽深,像是一层打开魔法学校的结界。
 
夜幕降临,像雾似的雨,像雨似的雾,丝丝缕缕缠绵不断,整个小教堂在雨雾中若隐若现。虽然天公不作美,圣诞电灯仪式,在细雨朦胧中如期举行,前来观赏的游客都热情高涨。精彩都灯光秀后,便在莲花餐厅举行了BBQ的圣诞晚宴,精彩的热舞,滑稽的互动,将整个圣诞狂欢夜点灯仪式推向了高潮。
 
在小莲池的密林旁,有一梅花形状的泉眼,这里便是大容山的圣水。梅花泉的泉水从岩缝中涌出,终年不干涸,水质清冽甘甜,因而途经游客多喜喝上一口,那沁入心脾的甘甜,回味无穷。
 
莲花景区的另外一大看点就是媲美庐山瀑布的莲花瀑布。瀑布落差巨大,三层叠嶂,因在季节不同呈现出不同的形状。丰水期的莲花瀑布犹如一匹巨大的珠帘,从山峰上倾泻而下,声如雷鸣,气势恢宏。十分壮观,值得一游。
 
 

追溯玉林母亲河源头

 
穿过教堂左侧的小径,沿着柏油路前行几步,会看到一道树木繁茂的山林小径,道路入口毅力着一块石碑,碑文刻着“南流江源头”。南流江是北流玉林人的母亲河,难以置信的是源头尽在大容山国家森林公园。一行人沿着通幽的林间小径,拾级而上,约莫十余分钟抵达了南流江源头。
 
江流源头是一处很狭窄的水域,让人吃惊的是四周的树木葱茂。蕨类植物密布树干,树枝盘绕纠缠,似盘龙醉卧山涧。徒步山林,似若穿越在北欧的热带雨林。难怪这里发现了稀有的巨大蜗牛,可见生态环境十分优良。

半山腰的露营天堂

 
大容山是许多露营爱好者的天堂。在大容山的半山腰有专门的露营地,这里地势平坦,视野开阔,是岭南为数不多的露营圣地之一,因而广深露营者的喜爱。露营者不仅可以睡卧自带的帐篷,还可以体验色彩缤纷的“集装箱”旅店,或是体验舒适智能的房车。
第一次入住房车,原来上面设施十分齐全,就像是机场里的胶囊酒店一样,不过房车还可以洗浴,简单的烹饪,有电视,暖气,实在太巴适了,相比帐篷这个更安全舒适,价格也合理,这样的体验者大容山不容错过。
 
晴空里站在露营地就能眺望秀美的天湖景区,闲来无事不妨前往露台湿地露营垂钓,在将新鲜的湖鱼架起在烧烤区烹饪,享受亲近自然,与山同乐的恬静惬意。如遇还天气,还能在在露营地静候漫天星辰作伴的浪漫。

桂东南第一峰

 
设为封面
美丽的大容山最高峰海拔1275.6米,号称“桂东南第一峰”。冬日登上第一峰不仅能体会会当凌绝顶,一览众山小的霸气之感。还能俯瞰黔东南最大的风车阵,一排排的风车屹立苍山之颠,莽莽蓝天苍穹下,迎着朝阳翩翩起舞,像一个个跃跃欲飞的小精灵,像武士挥剑守卫着这座秘境。若冬日气温骤降,运气好的话,还能领略晶莹剔透的雾凇景观。
 
隆冬季节,万物失去了生机,大容山顶却呈现出千树万树梨花开的雾凇景象。当雾气在低于0℃的附着物上直接凝华而成的产物,由白色的小冰粒积聚,包裹在附着物的外面,宛如玉树琼枝。
 
下山途中,途经抗战时期的作战指挥楼,三层楼高的褐色小楼,斑驳的痕迹诉说这过往的艰辛岁月。不远处就是景区里有名的莲花寺庙,祈福朝圣的圣地。

山林饕餮美食

来到大容山,我终于找到了广西表妹出落水灵的原因。山幽水静的自然环境,孕育了生态湖鱼,山野独有的野菜,珍菌,生态家禽。最后在养生餐厅大厨的秘制烹饪下,成就了一道道养生健体的山野美味。
 
毫不夸张的说,这几道美味一下肚,清肠养胃又不长肉,难怪广西表妹个个靓丽十足。听当地人讲,广西的低山沟谷里常常有这样的野菜,农家人经常采摘回家制作美食。因此在野菜的做法上,广西人有着本土的烹饪技艺。这也解释了广西表妹,为何一个个出落水灵,只因山野的美味,让她们健康美丽。
 
野菜羹,光看这绿油油的汤汁就食欲极强,喝起来一点清甜伴随着微微的苦涩,清肠养胃,值得多来几碗。
 
清炒小河虾,大容山有九瀑高谷,多条溪流汇入,有两个美丽的湖泊,因此山野孕育的河虾,味道特别鲜美,没有工业的污染,清炒出来的河虾,带着淡淡的河腥味,又因简单的油炸,有点多酥脆香甜,老少咸宜,更是一道喝酒的下酒菜。
 
水煮野菜,在这个大拼盘里,我们能尝遍大容山所以营养有稀有的山野美味。绿色,红色,黄褐色的野菜,各有各味,或清甜,或清脆,或苦涩,或软糯......每一种菜肴都有独特的味道,来到美丽的大容山,自然不能错过体验这种养生的山野佳品。搭配店里独特的蘸碟,吃起来爽口,又回味无穷。
 
板栗瓜蒸排骨,这是一道招牌菜。美食从来不拘于诞生于什么样的环境,只在于其味道。板栗瓜就很多地方的嫩南瓜相似,但是吃起来又略有不同。松软可口,瓜上的排骨沁入了板栗瓜的清香,吃起来特别鲜美可口。其实大容山的美味不只有这些,还有很多山林的野生菌,山鸡.....
 
美丽的大容山上一个北欧风情的小镇,在山脚下的养生餐厅是一家中西合璧的餐厅。有着国内普通大锅饭式的大圆桌排排布列,又有着北欧古堡式的吊顶,这样的装潢非但不会土气,反倒显得因地制宜,迎合了大容山的底蕴。

山野幽居

大容山景区的住宿并不多,唯一一个景区内恬静的旅居地就是桫椤谷酒店的民宿。独门独栋,一山而建,清幽静谧,不失为一个幽居圣地。在这里可以放松身心,呼吸大容山天然氧吧清新自然的空气,发呆养神。也可以在一旁的露天泳池冬泳,或是在电影院看一场精彩的电影,围在篝火旁随音乐翩翩起舞........

关于交通
 
1.自驾
 
线路一:玉林——北流——民乐镇——大容山
线路二:容县——北流——民乐镇——大容山
 
2.乘车
 
线路一:玉林火车站(或汽车总站)坐车到北流汽车总站,在喷泉广场旁边坐大容山专线车直达大容山。(注:北流——大容山每小时一班车,时间:08:00——16:00 大容山——北流每小时一班车,时间:09:00——17:30)
