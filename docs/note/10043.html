两天一夜百万葵园，我闻香而来，逐香而行

游记 亲子 摄影 精品酒店 美食 赏花 2018/12/26

游玩信息参考

花，有谁不爱？我闻香而来，逐花而行，带着儿子在百万葵园度过了一个完美的假日。
至今，依然在回味，那花、那鸡、那舒适的床铺和温暖的笑容。
行程安排：
从 广州 出发，自驾至 番禺 新垦镇百万葵园。
第一天下午在园区游玩，晚上入住花之恋酒店
第二天上午进入园区游玩，中午退房回家。
交通/门票参考：
除自驾外，可乘坐 广州 地铁4号线到蕉门站B出口，转乘G1直达百万葵园门口。
门票：成人150，学生120，儿童1.2米以下免票。

百万葵园里那美丽的郁金香呵

百万葵园的郁金香开得很旺，仿佛提前给广州带来了春天。
我闻香而来，逐花而行。
空中花园里花开满园，各种不知道名字的花吊在半空中，像童话里的世界。这是一座花之城，梦之乡，充满了浪漫与温馨。
虽然说空中花园一半是用餐区，一半是郁金香展区，但是依然像穿行在梦中，让人不愿意醒来。
最醒目的，莫过于那摆放了将近半室的郁金香。
白色的，黄色的，紫色的，彩色的……淡雅的花香、绚烂缤纷的模样，仿佛大海边穿得花花绿绿的绝色美女，数不清到底有多少株，一丛丛，一簇簇，灿烂的让人不忍移动脚步。大提琴的声音从花丛中白衣美女的指尖流出，飘荡在这如春的时空里，让人恍然以为真的进入了梦中的童话里。
首界郁金香展在百万葵园开展得如火如荼，郁金香被摆放成不同的样子，每一个角度，每一朵花，都是一处风景。展厅里人流攒动，花是景，人也构成其中一景。
有人在画它，有人在赏它，有人在闻它，有人在拍它……总也画不够看不够闻不够拍不够的样子。
那白的，如莲，那黄的，如阳，那红的，似火，那红白相间的，好似新娘头上的红盖头，只等着心上人，走近它，靠近它，去发现它最美丽的一面......
花，妖娆如仙，娇柔似羞。让人想到了女人如花。
有一句话，叫做闻香识女人。女人如花，亦如画，女人如花，亦如诗。
郁金香，便是一幅古画，一首古诗，一曲古词，清雅得让人伶爱无比，越看越爱。
此次百万葵园举办的“首界 中国 郁金香花博秀”共分四季，第一季“世界花后----郁金香”展将从2018年12月23日一直持续到2019年3月31日。
别的地方冷如冰，但郁金香却在 广州 逆季开放，这个全世界开花最早的展成为整个冬季最期待的盛会。现场30多个艺术场景，吸引了一颗颗爱花爱美的心。
百万葵园每天免费赠送给女士300株郁金香，12：00送出100盆，14：00送出100盆，15：30送出100盆。
不用去 荷兰 ，不用等到郁金香正常的季节开放，便是在这个12月，就能看到这盆盆精美的郁金香花开。
女人爱花，如同爱自己，花总是很配女人的。百万葵园懂花，也懂女人。女人懂花，亦懂自己。
在空中花园，一举手，一抬头，一驻步，都是花，都是风景，都是心动。
除了 荷兰 的郁金香，还有 南非 的蓝雪花， 美国 的 紫云 牵牛， 巴西 的龙翅海棠......
入眼之处，皆美如仙境。
赏花，来百万葵园便对了。

那绚烂的薰衣草花海

百万葵园，是 广东 人最爱的地方，是年轻人拍摄婚纱照的最美之地，就连小学生一年里也会过来郊游几次，它还是亲子游的最佳选择地。
假如说空中花园是个小童话世界，那么整个百万葵园便是花的海洋，一步一景，满目是花。紫色的薰衣草，点缀在田间地头，映衬得天地仿佛都变成了紫色，人们热爱这片代表着浪漫的紫色花海，白色的婚纱飘浮在紫色之上，仿佛，连誓言都会变得多了几分诚垦。
真想成为那花间的蜜蜂，亦或只是成为它身边的一棵小草，便能始终欣赏这一片美了。
第一天入园，薰衣草在淡淡的风云中美轮美奂，虽然当晚下了一场小雨，第二天，当我们又来看它们的时候，雨珠还停留在紫色的花瓣上，却给这一片紫色带来更加清新、更让人怜爱的味道。

孩童之心：动漫港

儿子很喜欢葵园里处处点缀的卡通人物，拉着我，不停地告诉我这个叫什么那个叫什么，兴奋得如同被放飞大自然的鸟儿，据说，喜欢动漫的成人心中都住着一个孩子，孩子们对动漫如同没有免疫的纯真。
动漫港，装着卡通的童话世界，满满的一屋子，仿佛盛放着各种各样的故事，走近它，便像打开一本丰富的动漫书，每一页都是精华。
大人喜欢，小孩子更喜欢。
各种叫得出名字的，叫不出名字的卡通形象，摆满了一个橱窗又一个橱窗，吸引了无数热爱卡通的成人或孩子们的目光，一个个欣赏下去，只会越看越喜欢，越看越津津有味，从不会感觉到厌倦。

雨林松鼠谷

百万葵园的热闹和繁华是属于每一个人的，可以静静地欣赏动漫展，可以买一份零食边走边吃，可以看鸭子给金鱼喂食，可以在田间地 头屋 前花海漫行，还可以带着宝贝来到雨林松鼠谷与小松鼠和鹦鹉亲密接触。
如树杆颜色的身体，漆黑的小眼睛，灵巧无比的速度，松鼠谷里的松鼠们如同一个个小精灵般，时而仰着头凝望着游人，时而和同伴追逐着食物，时而隐匿于花间树丛。
看到它们可爱的样子，真想伸出手去抱抱它们，扶摸一下它们光滑的皮毛，只是，它们灵巧地在树丛中钻来钻去，仿佛在做游戏。
这只小松鼠，是在等小伙伴一起玩吗？
陪伴着小松鼠的，是一只只落在树杈上、树屋中的彩色鹦鹉，它们仿佛怕人，又仿佛不怕人地自由自在地穿梭、飞行，偶尔落下几滴鸟粪，仿佛在宣扬着它的地盘。
孩子们欢笑着，瞅着，沉浸在与动物的和谐相处里。
在这里，还可以买点食物去逗喂一下这些小精灵。
孩子、松鼠、鹦鹉，构成和谐的一谷一世界，都是纯真而无邪的。

与孩子一起玩

百万葵园里，也并不单纯只是观赏，孩子们能玩的游戏有很多。
除了机动游戏，还可以荡秋千，划船、拓展……大人和孩子可以同玩，弥补平日缺少陪伴的缺憾。
时光不止，欢乐不止，成长的路上总会有美好的记忆相伴。
汽车、摩托车......可以坐上去，摆个酷酷的POSE。
VR游戏，在真实与虚拟中畅游。
跟孩子一起做拓展游戏，让自己也变成小孩子，一个人的欢乐便会变成双份。
最长情的表白是陪伴。爱，需要表达，需要做，需要和孩子共同成长。

绝色美味：葵花鸡

虽然是冬季，百万葵园的葵花却开了，黄色的花向着太阳的方向，仿佛想把天地间的光明全部收进那大大的圆盘里。只是，这里种植的葵花却并不是给人吃的，而是用来喂鸡的。
所以，游玩百万葵园，在空中花园餐厅吃过葵花鸡才能叫做完美。
葵花鸡的外皮金黄、冰脆，肉质鲜嫩，吃一口满嘴生香，这种只吃葵花籽长大的鸡味道与别的鸡有所不同，肉质也更加冰鲜，吃一口便再也忘不了那种鲜，那种美，那种嫩，那种香……
当然，餐厅的美食绝对丰富，味道堪比五星级米其林大厨。每一口味道，都是沉淀下来的精华。
饭后，可以领着孩子在园子里散散步，漫步在花草丛林，这里的花种类繁多，造型奇异，每一种造型都会成为吸引目光的焦点。
玫瑰花开了，格桑花开了......红的，白的，黄的，花儿像比赛似的，点缀在这个冬季，谁又敢说，这不是春天呢？
一垄垄的红色大花海棠美得似火，花丛中小火车发着呜呜的声音驮着孩子们的笑声在红色的花海中穿梭。
带着孩子也坐上了小火车，小火车欢快地叫着，隆隆地前进着，仿佛带着自己的童年，让人忘记了时光，忘记了时光已远。
风从指尖佛过，仿佛携带着欢乐，飘荡在心间、脸上，映衬得那彩色的如城堡的花之恋酒店格外美丽。

大花海棠花海中的花之恋酒店

入住花之恋酒店，可以无限次畅玩百万葵园。
郁金香是最爱，当然选择郁金香主题。
我们入住的是郁金香大床房，房间空间超大，玻璃上，地毯上，花盆里，都点缀着郁金香的元素，连被子上都绣着美丽的郁金香，床头放着几本彩色绘本和故事书，呆在房间没有外出的时候，可以给孩子讲故事或者一起看书，临睡的时候，可以用故事伴着宝贝入梦乡。
床铺和枕头极为舒适，睡在这样的地方，连梦都会感觉到香甜。
可以用房间的咖啡机泡一杯咖啡，再放上一首舒缓的音乐，享受一个人的寂寞，或者两个人的甜蜜，亦或是一家三口人的欢乐。
就算在浴缸泡个泡泡浴，也能感受到从心底涌起的小小幸福。
此时此刻，便会觉得，日子如诗，生活如花，花如酒，入喉的那刻，便迷了心，喝下去，便醉了一世，刻骨又铭心。
时光那么美，让人不忍辜负，时光这么静，红千紫百皆是梦。

饕餮大餐

酒店的餐饮除了享用品种丰富的饕餮大餐，一定不要忘记品尝酒店餐厅专门为住客提供的葵花鸡粥和葵花萨其玛、葵花鸡蛋、葵花鸡腿、鸡肉、鸡胸……
满足的不仅仅是胃，更是来过便不想离去的人心。
海鲜、日料、甜品、点心、水果、中餐......品种丰富，种类繁多，此时，只恨自己只有一个胃。
在杯与盘，刀与叉的碰撞下，是胃的更是心灵的满足。
入住花之恋酒店，假如想去百万葵园景区游玩，找前台在手上盖个章，便可以无限次畅游百万葵园。
时光是百万葵园的，更是属于自己的。
人生最欢乐的，莫过于，住一间好的酒店，吃一顿好的大餐，去一个好的景点，留下好的回忆。白天的百万葵园无异是热闹的，由人群相伴，花海相拥，夜晚的百万葵园是宁静的，由灯光点缀，星月相照，假如，你从空中俯瞰，一定会发现百万葵园不一样的秘密-----更美，更仙，尤如梦的世界……

尾声

百万葵园的郁金香花展，让我提前见到了 广州 的春天，我爱花，爱它的名贵与普通，爱它的娇柔与坚强，不用去 荷兰 ，带着宝贝在百万葵园度过两天一夜的美好时光，是这个节假日收到的最美好的礼物。
我在，你来么？在花的海洋里，谁人抚琴慢歌赋，奏一回永生难忘的曲......
