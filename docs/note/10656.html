15天可以在石家庄做的事

我很幸运，因为热爱，认识了一群朋友，同样热爱自己的热爱。
在他们尽情生活，努力热爱的同时，作为 浙江 人的我
旁观了太多，也实实在在的体验了很多
15天，带上一颗度假心
你可以在 石家庄 做这些事
1.在24小时的呈明书店看一本喜欢的书看到忘记时间（当然拍照也超好看！）
2.自驾去横山岭野炊烧烤（直接烤钓上来的鱼）
3.找一条有蔷薇花的路散步，那一天的心情都会很美
4.必尝一下 石门 一味的 石家庄 味道（听说所有你知道的和不知道的菜都有）
5.在科大和师大的校园里走一走，哦呜哦~
6.如果是晴天，就在晚饭之后去 正定 古城墙上看一次夕阳吧

凌晨之后的到达，本该睡到日上三竿，却被意料之外的太阳给召唤醒了，比起上次冬天的大雾霾，这次的晴天实在是太友好了！一块钱的早餐包子给了一个好的开头，结束于24小时书店。
从湿气郁结的 杭州 飞到 石家庄 ，本来都还潮潮的外套在第二天已经完全褪去在 杭州 的状态，这样的变化，是把我从南方拽到北方来最直观的惊喜了。
在到达 石家庄 之前，你甚至不知道人们口中的“土味儿”是不是一个好词，一方面它代表着扎实质朴、贴近人情，另一方面也带点荒凉偏执的况味。然后这一切在我到达 石家庄 机场后，渐渐地开始为我们展开属于它的各种画面。
没有特别一尘不染，没有特别规规矩矩，没有特别着急忙慌，不得不说， 石家庄 确实有一种浓浓的“土”味，这种“土”是一种down to earth的生活气息，是菜场的街巷里或坐或走的老年人依然精神矍铄的神气，是汪曾祺先生“不管在什么环境下永远不消沉沮丧。无机心，少俗虑”的生活本质，也是出租车大叔和小食摊大姐一边抱怨着世道艰辛一边微笑着早出晚归的安贫乐道。
 跟 杭州 等南 方城 市盛行的诗意文化不同，北方人似乎总在刻意追求一种不掺杂水分的“原汁原味”，他们精心呵护着这份纯粹。但也正是因为这种“敝帚自珍”式的固执，让这里的人的“憨”成为一种可爱又自信的符号。
白天的晴天迎接得我依然眩晕在真实和梦境之间，街道两旁青葱的绿色让我恍惚于怀疑“这还是我上次来的 石家庄 ”的自问自答里，可见四季分明的城市果然有着不止一次的惊喜啊。

石家庄 也是可以文艺的和你侃大山的嘛 
这一次，住在了小姐姐家里——作为文案公司的非典型CEO，她的日常绝对被我的停留打乱了。
这几天的早餐都是被临近虚脱的饿意叫醒的，差不多到九点半的样子了，但早餐还是一顿没落。
提起 石家庄 ，很少会脱口而出什么经典的味道，也许是因为被火车拉来，人们都习惯于直接带上干粮奔赴下一站了吧，这也让这个地方有了承载全国美食的包容。
西安 肉夹馍，从小姐姐的日常来看，肉夹馍配上清爽的鸡蛋汤是最容易让自己一整天能量满格的早餐了，8元就能吃的很满足。
（肉夹馍里除了加肉，还可以选择鸡蛋和豆腐）纯纯的面香，面饼外酥里筋道，肉汁没有过分渗出到面饼上，反而香气浓郁，各展其能。
我以为 石家庄 没有称得上名的咖啡馆，然而我又错了。
广安 大街上的漫咖啡，三层架空，点上等同于星爸爸价格的咖啡，就能消遣一整天的时光。嗯，这很惊喜。
