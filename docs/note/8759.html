京郊两日 | 麻麻花的山坡 两小无猜与自然的真实邂逅

游记 五一 亲子 家庭游 自驾游 踏青 2019/05/14

吉言碎碎念

每一次的旅行，都能看见你的成长。
在青籽树，你还是不到九个月的宝宝，会坐会爬，对自然事物充满好奇。
在 三亚 ，10个半月的你有了很多第一次的新尝试，也发现在陌生的环境里你会格外粘我。
这一次，在麻麻花的山坡，可以独立行走与人逗趣，快15个月的你给了我更多的惊喜。
虽然每天都有参与你的成长，有时也会忽略忘记一些细微变化，还好有镜头定格分分秒秒，记录你的成长变化与轨迹。
好吧，开篇来自一位母亲的叨（gan）叨（kai）念。
有了娃以后，旅行的频次远不如以前。
但总有那么一个地方，能让你说走就走。
因野菜“麻麻花”而命名的“麻麻花的山坡”乡间民宿，
坐落在淳朴宁静的南峪村山谷之间，位于京冀交界，自驾只需2小时车程。 
春暖花开之际，我们决定带娃来一次 春日 之旅。
在钢筋水泥的城市久了，会想念绿水青山。
在麻麻花的山坡，与好友一家相约，过两日无忧无虑的日子。
看那天真烂漫在风中奔跑，体会乡野间的宁静与快乐。
今年天气暖的快，如果再早一周来这里，会看到桃花开遍山坡，不过也还好，赶上了桃花的尾声和梨花的盛开。生活就是如此，不能事事圆满如意，但也还算是有不断的惊喜。

拍摄设备：佳能5D、佳能6D

初相识 是熟悉的气息

因为去年八月带桃心儿去过隐居乡里的青籽树民宿，所以来到南峪村的麻麻花的山坡，一点都不陌生。
抵达时已到正午，进院后映入眼帘的是一样雅致的院落，相似的装潢和同样笑脸相迎的管家阿姨，初相识也是熟悉的气息。
不同的是14个月多的桃儿心已经可以自己独立行走，可以自由穿梭在院里的每一处空间。处在大运动飞速发展的她，对5号院两处房屋相连的木梯格外欢喜，上上下下，登高攀爬，乐此不疲。
五号院坐落在山坡最高处，与山为伴，与自然融为一体。时而有村民赶羊群从院侧而过，时而有山间野生松鼠跑过，孩子新奇又惊喜。
因为我们先行抵达，闲暇之余，在管家阿姨的帮助下，把我给孩子们准备的帐篷和野餐垫摆在庭院中，坐等桃心儿的茉莉姐姐一家到来，一起嬉戏玩耍。
去年十一还无法骑跨的红色木马，如今终于可以摇摆玩耍，有了对比才发现桃心儿真的有在成长，感慨时间一晃而过。
休整完毕，就来餐厅享用午餐。
这里的民宿院子费用包含一顿下午茶、晚餐和早餐，抵达当日的午餐不是标准餐，如有需要并且安排得下是可以另外付费添加。山坡下的南峪村也有饭馆和农家乐可以解决吃饭问题。
因为带着孩子，为了省事减少奔波，沟通后发现我们周五入住的前一晚院子没有客人，所以就加了顿午饭。
四菜一饭，虽是家常，但也足够丰盛与美味。管家阿姨还自愿承担起给娃喂饭的“重任”，好在桃心儿也配合一口接一口，吃得愉快，不时还与人逗趣互动，让我可以安心吃午饭。
吃完饭，沐浴着 春日 的阳光，泡一壶野生黄芩茶，看窗外山间美景，捧一本书，无限惬意尽在这一方院落和暖暖的午后。
这里，夏秋两季应该会更美。
桃心睡完午觉，茉莉一家也刚好抵达。管家阿姨准备了农家下午茶，纯天然自制山楂汁、蒸南瓜、苹果、玉米糕和炸馓子，都是在地取材，无污染无公害～
两个小家伙吃的开心极了～明明刚刚喝完180ml奶的桃心儿玉米糕抢着吃了一口又一口，丝毫没有停下来的意思。
两岁半的茉莉也深深沉浸在山楂汁酸酸甜甜的的滋味里。
吃饱喝足，开启玩耍模式。
五号院内有个小石磨，连我们都觉得新奇，就别说小孩子了，茉莉还像模像样的学着管家阿姨的样子、听着爸爸的指导，尝试操作小石磨，充满新鲜感的生活真的不错。
希望你们长大也可以对世界保持好奇。

在避世小院 留住春光明媚的尾巴

几寸光阴，几片绿意，还有几分心动的感觉。
两家人，上下院，彼此相连又互不打扰，空间也大过了时间。
一切的一切都浓缩在这一方天地里……
院子里，孩子们可以开心的跑，可以大声的叫。桃心天真无邪悦耳的笑声在空中弥漫，洋溢着无忧无虑的喜悦。这就足以把所有繁忙抛之脑后，慢度时光，体会陪伴是最长情的告白所溢出的幸福感。
玩耍道具准备的足够齐，泡泡机、帐篷、绘本、鲷鱼头套…凑成了午后两小无猜的互动美景，用相机记录孩子眼眸透出无邪的光和天真的笑脸。
有没有觉得茉莉和桃心儿有些相似，相似到好像说是我们一家四口，也毫无违和感。
在院子里玩了会儿后，我们打算去相邻的别的院子转转，在山间走走，偶遇鸡鸭鹅羊，去梨花树下，邂逅春光。
山间有棵很大的梨花树，格外显眼。
就在3号院的空地前，3号院的地理位置也是极佳，景色宜人，所以我们决定趁入住的客人还没到，先去“串个门”。
就是可惜没赶上花期，不然这番景象会更加美丽。
感谢身为亲子摄影师的茉莉妈，帮我留住春光明媚的尾巴。作为朋友必须吹爆她！强推～
感谢扛起抱娃体力活的爸爸们，父~爱~如~山！

风筝小栈 最美的亲子时光

麻麻花的山坡还有一处宝藏之地——精心打造的风筝小栈，这里装着孩子们对于风筝的全部美好想象，四月初到五月底都有风筝节的活动，在这里可以学习绘制风筝的技巧、制作属于孩子自己的专属风筝，体验非遗风筝文化的魅力。
风筝，注定是属于这暖风拂面的春天。
住在麻麻花的山坡，迷恋院子里的悠闲，也别忘抽时间来这里看看。
正确打开美好亲子时光的方式，不就是一次乐趣满满的惊喜体验嘛~
在这里，可以满足你对风筝的所有幻想。春风拂日暖，趣童画纸鸢。
春花的花期尾声，夏花正含苞待放，多彩的纸鸢便成了这天地间最美的风景。
推开门，映入眼帘的是木质窗门的四合院落，传统合院式建筑让人眼前一亮。 北京 四合院的特色越来越少，没想到在南峪村踏进“过去”，当落日余晖洒进院子，时光自是安恬，只想到四个字——岁月静好。
在风筝小栈工作人员招呼下，我们便开始绘制风筝。
桃心儿还小，无法自己完全参与，但还是会蠢蠢欲动有模有样拿起画笔学着涂抹。
虽然她的作品基本由我代笔，也还是会让她选择喜欢的颜色，参与其中。
画好的风筝可以带走留念，也会有专人二次加工让风筝真的可以飞起来，让孩子放飞梦想~
虽然桃心儿可能还无法真正理解自己做了什么，但看她踮起脚尖、学着茉莉姐姐认真的模样，用五颜六色的画笔，趁着微醺的春风，肆意涂抹，这种瞬间，真的很好，可意会而不可言传。
想一直一直记住，这一瞬的时光。
也希望，未来你可以有一个，踮起脚尖就能飞起来的梦想。
除此以外，还有绘本专区，有三十余本风筝专题的图书、绘本等阅读读物，寓教于乐。让孩子对风筝有更好的理解与认知。
时间就是这样，不紧不慢的过去。
画完风筝，读完绘本，就以晚上六点多，好在春夏之季天黑的晚，也好在有管家阿姨烹饪晚饭，我们还可以不慌不忙，伴着夕阳余晖，一路有说有笑，爸爸各自驮着自家娃，返程归“家”。
回到五号院，太阳落了山，管家阿姨端上了特色五花肉豆角焖饭、凉拌嫩豆腐、葱花摊鸡蛋、拌木耳等，一切都恰到好处，喂饱了我们一行人。焖饭看似不起眼，我还头一回吃，阿姨做的味道相当好，我们每人都吃了至少两小碗。
这里每个院子的管家大姐都是专门培训过的，手艺和食材经过考核检验，并不是随便能做出的口味哦。

有烟火的夜晚 格外的美

在院外就看见了烟火的残碎，问了管家阿姨，可以在村里的三个胖嫂超市买到烟火。
茉莉一听可以放烟火，就一直心心念晚上的到来。
而我们一家三口也内心兴奋，因为这可是桃心儿人生第一次近距离看烟花、放烟火。
烟花和女孩，更加接近幸福的夜晚。
忙碌而又充实的一天结束，希望你们的童年如烟花一般绚烂。

一宅院落 感受时间放慢的脚步

第二天一早，依旧是贴近自然的好时光。
经过再造后的院子，精致而又包含浓浓生活气息的。
席慕蓉说过：“我常常会想起我们山坡上的那个开满了花的院子，和天上的那些云彩，白白柔柔的，却飞得好快。不肯回来的，大概也不只是那些云彩了。”
在这里，感受时光放慢的脚步，这种体验是在 北京 市区也难复制的。
“孩子就像需要睡眠和食物一样，需要和自然的接触”
—— 查德•洛夫《林间最后的小孩——拯救自然缺失症儿童》
捏木炭、玩沙水、拔嫩草、捡树枝 ，四处发现“新大陆”，感受春天的可爱。
要知道皮肤是生命最具灵性的部分，让孩子的皮肤可以尽早亲近大自然的阳光、风、草石、沙水等也是必需的。
自然界中的各种变化，种类繁多的动植物，千奇百怪的地形，多种多样的颜色、气味和声音，能够最大限度地调动宝宝的感官，促使宝宝的视觉、听觉、触觉、嗅觉、味觉一起发挥作用，使宝宝的观察能力在无形中得到巨大的提高。
玩水湿了衣服，玩沙玩土脏了手脸，这都没有关系。
带娃出门在外，多带几套换洗衣物就好，请别阻碍孩子的好奇心与探索欲。
迎着明媚的清晨，走，我们吃早饭！
美好的一天，从餐桌的第一顿美食开启~
手工揉的馒头，配上柴鸡蛋和有机小米粥，还有几碟用麻麻花调味的秘制小菜，清新爽口，质朴而纯粹。因为有孩子，管家阿姨还特别细心的为她们蒸了鸡蛋羹。
也因为麻麻花的山坡，桃心儿爱上了她还未猎食的人间美味——馒头。
吃完早饭，我们一行六人，在5号院的院落合影留念，定格这两天一夜的 春日 之旅，留住属于我们与麻麻花的 春日 记忆。
茉莉妈妈——PP酱露脸照在此！想要亲子留念的可以微博找她~
人生总是不断的相逢，又不断的离别。
又到了说再见的时刻，笑脸相迎到笑脸相送，不过短短一日，情谊也埋在了五号院。
我还记得管家阿姨说“有机会你们再来”真诚的眼睛，和对桃心说的那句普通却温柔的“我最喜欢你”！
退房时候，阿姨联系了电动三轮车帮我们搬运行李到坡下的停车场，减轻了不少负担，让我们可以再看看山坡的景色。
一路告别，和广阔的山间、和精致的小院、和暖心的人、和各种小动物……
在市区内没实现的 春日 暖阳，都在南峪村の麻麻花的山坡一 一实现。

实用攻略

★ 麻麻花的山坡
地址： 河北 省 保定 市 涞水 县南峪村（距 野三坡 15分钟车程）
价格：680—3380元/晚
设施：以我们住的两居室院子的话，有两个卧室、餐厅、起居室，室内带地暖+空调，冰箱、儿童餐椅、矿泉水、暖水器等等一应俱全。卧室、餐厅都有卫生间，5号院是上下院落，所以有两个院子。
预订方式：官网 www.nalada.cn 
★ 周边游玩实用信息
1、 野三坡 百里峡：从麻麻花的山坡去 野三坡 百里峡的话，开车大概40分钟， 野三坡 是国家AAAAA级旅游区，时间 富裕 自驾的小伙伴可以去这里看看。
各景区门票价格：百里峡110元、印象 野三坡 90元、溶洞漂流90元。
2、 野三坡 大峡谷
野三坡 大峡谷位于 北京 市西南，与 房山 十渡景区相邻，拒马河流经 野三坡 风景区35公里，两岸独特的山峦交相辉映，形成奇山异水，有“北方小 桂林 ”之美称。
门票： 30元/人
3、拒马河漂流
拒马河漂流是京郊比较好的漂流，天然河道，可尽情打水仗，喜欢刺激项目的朋友体验一下。大约长500米左右，激流口比较多，河道窄，玩的时候要注意安全~
门票：98元/人
4、七彩玻璃栈道
玻璃栈道很是受到各地游客的喜欢，位于 房山 区十五渡的东湖港景区内，这里的玻璃栈道是七彩的，也因此吸引大家前往攀爬挑战，七彩栈道是需要单独买票进入的，网上目前还没有购买渠道，只能景区门口购买，也可以买景区套票。
门票：30元/人
5、 野三坡 白草畔森林公园
白草畔位于 野三坡 的上坡,是 野三坡 景区的最高点。白草畔其实是 北京 房山 、 门头沟 和 河北 交界的地方的几个相连的山峰和高山草甸，在山顶都是连在一起的一大片。每逢花季，白草畔景区内杜鹃花开，还是值得一去的。
门票：成人门票60元；学生、60-70岁老人凭有效证件、1.2米以上儿童享半价30元；1.2米以下儿童、70岁以上老人免门票。
6、鱼谷洞
鱼谷洞也算是 野三坡 众多景区中一个成熟的景区了。这里四面环山，河水滢滢，杨柳低垂，气候宜人、奇泉怪洞相映成趣，是寻奇探险、休闲度假的理想之地。
门票：成人票65元，学生票35元
7、百里峡七彩小镇
百里峡艺术小镇就位于 野三坡 百里峡景区的对面，是京西百渡休闲度假区中的41渡。以“赤橙黄绿青蓝紫”七色作为色彩创作基础，把原来的苟各庄村100多户的农家旅馆进行整体包装改造，采用欧式风格，突出时尚与希望。景观上，邀请了百名艺术家参与，进行了多个节点壁画的艺术创作，实现了色彩、当代艺术 和美 丽乡村的完美融合。
