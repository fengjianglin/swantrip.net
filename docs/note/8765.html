#天鹅之旅首发#吉隆坡-马六甲-槟城，学渣本渣的大马三城记

游记 人文游 摄影 美食 自助游 自然奇观 2019/05/13

学渣本渣（正是在下！）

有鸡汤说，“当你决定出发的时候，其实最困难的一部分就已经完成了”。我也比较认同这句话，但是对于当年英语三级都是补考才能过的学渣来说，出境游还是要考虑许多问题的。虽然也不是第一次出境了，但之前要么跟团，要么有人带着我，不用自己操心，难度几乎为零。
这一次是真正意义上决定的出境自由行，什么事情都要亲力亲为，想想还蛮刺激的。于是目的地顺理成章的选择了马来西亚这个适合新手刷经验的东南亚国家。那里旅游资源丰富，网上攻略多，相对稳定安全，所以没什么好顾虑的。

行程开销

这部分就粗略算下吧，单位人民币。杭州往返吉隆坡的机票是1000出头；吉隆坡往返槟城的机票加起来才85（就是这么难以置信！）；住宿平均200每晚，两人均摊住四晚，每人400块；吃饭每顿20左右，共五天；打车每次10块出头；没有景点门票产生。所以，3000出趟国怎么都够了！

行程安排

DAY1：杭州-吉隆坡（亚洲航空XD7303,14:25-19:35），入住KLCC苹果精品酒店；
DAY2：黑风洞（Batu Caves），老火车站，茨厂街，石油双子塔（KLCC），入住苹果精品酒店；
DAY3：吉隆坡-马六甲，新时代咖啡店，荷兰红屋，马六甲海峡清真寺，鸡场街，入住悦舍酒店（Treasures Hotel&;Suites）；
DAY4：马六甲-吉隆坡机场-槟城（亚洲航空AK6126,13:05-14:10），爱情巷、汕头街，入住槟城皇家酒店（Hotel Royal Penang）；
DAY5：（酒店休息半天），下午太热又在咖啡店里偷懒几个小时！姓氏桥、沿街壁画打卡，槟城-吉隆坡（亚洲航空AK612123:55-00:55）；
DAY6：吉隆坡-杭州（亚洲航空XD7302,08:10-13:10）。
接下来就按照城市分三个章节来写吧！想要实用信息的可以直接拉到最后一个章节【学渣宝典】来找自己想要的东西。当然这样做的话，你会错过一个“去印度庙，吃日本料理，逛唐人街”的精彩行程呦！

PAGE 1，吉隆坡

到达吉隆坡是晚上七点多，因为亚航上没有免费餐食，就在机场随意腐败了一下。吉隆坡机场是东南亚很大的一个中转站，所以里面几乎什么都有，ATM机里取好马币消费就行了。然后辗转巴士和地铁来到吉隆坡的地标KLCC石油双子塔，差不多已经晚上十点了！因为第一次来就把酒店订在了地标附近100米处。
入住的酒店叫做Leapple boutique hotel(苹果精品酒店)，KLCC地铁站出站即到，在房间内拉开窗帘就能观景。对了，这次行程还捡到另一个学渣跟我一起，我们是从不同地方出发的，所以他早我半天已经到酒店了，他叫泽彬，是广东汕头人（先介绍一下地方，槟城部分会有说明）。两个臭皮匠也抵一个诸葛亮吧，毕竟我比较胖！
看酒店的房间，档次相当于国内的如家，不算特别大，但是蛮干净、卫生的。而且这个酒店的价格也很合适，折算成人民币200多块，就相当于你去上海住在东方明珠脚下了。男孩子出门有的睡就好！第二天清晨，拉开窗帘，旅程正式开启。
酒店距离全市最大的换乘中心KL Sentral仅五站，从那里可以去到吉隆坡几乎所有好玩的地方。我给自己规划的首站是距离市区最远的黑风洞，这也是我的惯用策略，由远及近，边玩边退回酒店。
黑风洞（Batu Caves）是吉隆坡的印度教圣地，每天都有大量的信徒前来朝拜，在地铁上我就发现了很多印度人。各种神像虽然我都不认识，但是五颜六色特别好看，也是之前在网上种草，特地过来拍照的。
广场上的人很多，有信徒也有游客。山下的庙（应该可以叫庙吧？就是很多人在里面进行宗教活动的地方）里没有进去，因为不懂也不敢问！若要到洞穴主庙需要爬上272个彩虹阶梯，不算特别高，但是坡度很陡，对服装(特别是女士)也有一定要求。来都来了，肯定要爬的。
一路上有许多猴子，普遍个头不大，好像不是特别怕人，但是胆子也没有肥到抢包的程度。胆子大的还能跟它拍照，但谨防被抓伤。
到达山顶，洞里别有洞天，至少是特别凉快的，忍不住在里面多待一会。遇到的人也很nice，特别是一家印度人，小孩子特别可爱，在他父母允许的情况下，给他拍了张照片，眼镜比我的还大，惭愧惭愧！
下面就是这一天的高光时刻了！到了饭点，肯定要吃午饭吧？因为是印度教，所以周边只有一些印度餐厅。早就听说印度餐很酸爽，没想到在吉隆坡要面对我人生第一顿最“地道“的印度餐！凭眼缘找了一家黄色招牌的，就像咖喱一样的黄，人均RM12元，点了一份咖喱饭且浇上汤汁。
关键是浇上了黏稠的神秘汤汁以后，还需要用手抓！虽然泽彬找到了勺子，但是我为了体验当地人的习惯，还是坚持用了手。结果我只是把那块鸡肉和薄饼吃了，每一手抓起的饭粒不超过10粒！绝对很真实了。浪费食物是可耻的，我想字我检讨一百遍，然后赶紧出去买了个椰子压压惊。
离开黑风洞，我们去往的是市区的老火车站。结果刚一出地铁站就下雨了。赶紧找了个餐厅坐下来，点了一杯奶茶。我的旅行态度就是这么随性，只要有吃的什么都好说！
东南亚的雨来的快，去的也快！雨势小了以后，我们就在地铁站以及老火车站周边随意拍了一拍。
从老火车站步行到茨厂街（唐人街）就不远了。对了，周边还有一座大的清真寺，我觉得那都是女孩子摆拍喜欢去的地方，就没有过去。相对来说，我很喜欢茨厂街的市井和人文气息。能够在异国他乡看到满满的中华元素，本身就是一种自豪感。
茨厂街里所售卖的东西可以说是吸引力不强，类似于义乌小商品市场。但是吃的还是有一些网红店的。由于中午印度手抓饭带来的阴影实在太大，已经不想吃那种茶餐厅了，于是就随意买了份排队超多的豆腐花，甜的！味道感觉也是一般。
那么晚餐吃的什么呢？吃了一顿还算有点排面的日本料理，就在酒店一楼的六甲（ROKKO）日料店.吉隆坡这座城市绝对是包容的，完整的一天居然去了印度庙、逛了唐人街还吃了顿日本料理。
餐厅有整面的玻璃落地窗，宽敞明亮，就餐环境非常舒适。城市临海的地理位置也保障了食材的新鲜，招牌的刺身不但味美，价格也算合适。还点了一些汤水、饮料和烧烤，看得出来都是经过悉心打磨，很有自己的特色。人均消费大约在200RM左右。一扫印度餐的咖喱味~
吃过晚饭时间尚早，于是去到了KLCC转一转，毕竟很近，肯定要去打卡的。感觉建筑本身非常像两根玉米棒插在地上，吃货的眼光总是独到的，那就给它各个角度来一波照片吧！
双子塔的内部除了办公也有商场，需要购物的可以逛一逛，不过貌似价格不算太美丽。背面是一座小公园，到了晚上居然还有很不错的音乐喷泉可以看，人还特别的少。完全不用担心像杭州西湖以及西安大雁塔那样挤，实名推荐！

PAGE 2，马六甲

从吉隆坡到马六甲要乘坐汽车。车站叫做南湖镇终站，英文：Terminal Bersepadu Selatan，简称TBS车站，是个大型的长途车站。值得注意的是汽车站并不像国内的车站，到售票窗口说到哪里就可以。而是有很多家汽车公司，可能他们会有去同一个目的地的车，要分清楚。好在TBS还是很正规的，但马六甲的车站相对较乱，最好提前去，以免耽误时间。还有就是有人说车票需要提前在网上买，不然去了现场可能买不到。去的时候我们是现场买的，回来是在某网站预订的，仅供参考。
到了马六甲车站吃了顿麦当劳，价格比国内还要便宜一些。然后用Grab打车，先去到了预定好的悦舍酒店。网上看它们家是中式四合院形式的，这个四合院的楼还是蛮高的呀！
从吉隆坡到马六甲的时候，最大的感触就是中国元素变多了。郑和下西洋时曾经六次在此停靠，至今还保留着诸多郑和的遗迹。包括入住的Treasures Hotel&Suites不但有中文招牌，还有会说中文的店员和老板，特别有也归属感。（下面是老板本人）
酒店的房间和在吉隆坡住的差不多，但马六甲小城相对更加安静，也没有什么高的建筑，更加适合度假。楼顶处还有一个大泳池。
酒店歇息了一会，总是要出门转转。因为下午太热，就去了老板推荐的一家咖啡店，就在附近。不过咖啡店相当隐蔽，跟着导航才找到。门头的招牌也不大，而且非常老旧，不再三确认下都看不清上面写的Sin See Tai(新时代）。
炎热的下午点杯冰咖啡，十分惬意。这里的咖啡口味非常棒，价格并不算贵，十几马币一杯，和国内的价格差不多。呆了一个小时也就我们两个客人，可见马六甲的安逸。
喝完咖啡，沿着马六甲河就能走到荷兰红屋，这里也是整个马六甲人气最火爆的地方，游客必打卡的地方，旁边也就是鸡场街（唐人街）。
荷兰红屋充斥着各种旅行团的人，但基本上都集中在广场上，想要拍照人少可以到旁边的楼上。不但能拍到红屋的全景，通体红色的楼梯本身也很好看。
还想拍几张自己的游客照，但已经找不到泽彬了。他居然被一种挂满了各种卡通公仔的三轮车所吸引，拿着相机要拍遍所有的款式！我实在没能get到他拍照的点，个人感觉这些车怪怪的，我就不说像什么了吧，浩浩荡荡大概有几十种款式都不带重样的，音乐放的也很high，不乏《海草海草》等抖音神曲，到了晚上甚至还会发光！价格也没问，旁边买了个冰箱贴5马币，参考下物价还是合理的。
马六甲很小，景点也都集中在了这一片。但是有个不得不去的景点靠在海边，就是海峡清真寺。去海峡清真寺所看的不是建筑本身，而是绝美的海边日出和日落，当然有清真寺当前景也是极美的。
在荷兰红屋打车过来很方便，大概是下午五六点钟到达海峡清真寺，这个时间点是有些尴尬的。因为这时候的清真寺已经不对外参观开放了，但是距离日落也还有那么一会会才到。
不同肤色的游客都很有默契的朝着日落方向坐着，等着夕阳的到来。没有喧嚣和吵闹，这一刻的心灵是纯净的，可能这就是旅行的意义吧？什么生活上的烦恼仿佛都没那么重要了。日落到来，咔咔走起！
欣赏落日美景的期间我还偷懒去门口的小摊点买了油炸小螃蟹，十块钱马币，颇有几分网红火鸡面的意思，海的味道，就在这一包！
看完日落，接着打车返回了鸡场街。很明显，又到了饕餮时刻！鸡场街里的小吃太多，以至于让人眼花缭乱。
最后还是凭借眼缘吃了顿我家鸡饭，原因是在门口的照片墙上发现了吴建豪。没错就是那个F4成员，要陪你去看流星雨，却没说去哪看的那个吴建豪。
点了招牌的海南鸡饭，门口大爷那里买了杯凉茶。鸡饭的味道还是蛮不错的，马来西亚的美食很多，最好吃的还是这种偏中式的。鸡饭的价格也不算贵，人均消费大概15马币就可以了。一份饭一杯饮料美滋滋。吃完饭，外面的小吃又来了两份，然后深知长肉的罪恶，步行回酒店啦！

PAGE 3，槟城

槟城的行程可能是这次三个地方，最没有按照预期来的地方。好在这样的未知会更加的有趣味。从吉隆坡起飞到槟城，总共的行程不过一小时，飞机快落地的时候飞过海岸线很漂亮。直接打车去到了酒店。由于酒店和前面两天的档次差不多，就忘记了拍照，倒是拍了一张窗外的风景。
在槟城之前，你很难不对这里的美食充满期待。毕竟这里是全球有名的吃货天堂！真正出门以后发现情况并不简单。由于中午的亚航飞机没有餐食，我和泽彬是卯足了劲要在槟城大吃一番，谁知道下午三四点钟的街上，居然没有餐厅开门！可能是因为我们来的时候算是旅游淡季吧？我们去了汕头街，本来带着汕头人来准备打折的，结果差点饿晕在街头。
一定是旅游淡季没错了！我们辗转来到网红的多春茶座，居然是歇业状态。后来又去了张弼士故居，想看蓝色的房子，也是整修暂停营业！泽彬说它们可能是回家过年了，拜托已经三月份了！不过想想也是，如果是回广东那边的沿海城市过年，呆个一两个月也不过分呀！
后来到了饭点，终于还是在汕头街找到了组织。槟城的美食吃起来并不需要什么仪式感，它们不是摆在高档的餐厅里，需要花费很多钱，而是渗透在市井里的一份份小吃。网红店没去成，也甩掉了探店的包袱，各种美味随缘去点。这些十元左右的各式小吃多为华人经营，可以用中文交流。推荐福建炒面，炒粿条以及蚝煎。
第二天在酒店的楼下排档里也吃到了很不错的一餐。猪脚粉是甜的，我有点吃不习惯。
烤面包是意外的惊喜，香脆可口，这个随意吃到的味道已经很赞了，很难想象多春的招牌烤面包还能比它好吃到什么程度。
半熟鸡蛋也还不错，洒上白胡椒没有腥味，爽滑可口。
离开槟城的最后一餐呢，则是吃了一碗沙叻和红豆冰。说来也巧，在槟城的街头转悠纠结吃什么的时候，几个华人老大爷路过告诉我们一家餐厅叫做Laksa kitchen，因为他们也是来吃红豆冰的。
就试着来吃了碗沙叻，看介绍他们家的沙叻有着很长的年代了，酸酸的又很鲜美，汤底很浓郁，我说好吃的时候，店家老奶奶表示很开心。也学着大爷们点了红豆冰，太甜！不过这么热的天气吃起来还是蛮爽的。
然后出于好奇，我鼓动泽彬点了一份已经忘记了名字的甜品，那上面绿色的丝可不是辣椒啊，但还是像极了黑暗料理。我还是吃我的红豆冰吧，电影《初恋红豆冰》就是在槟城拍的，就是这碗红豆冰。
来槟城，除了探寻美食以外，乔治镇街道上的壁画和铁塑漫画也是不容错过的。起初它只是一位立陶宛艺术家的无心插柳，散落在大街小巷。最知名的可能要数Lebuh Armenian街上的《姐弟共骑》，很多国内的古镇或者商业区都能看到类似模仿的作品。我们第二天找到它的时候围满了拍照的游客，于是就随意拍了一些其它的。
这些老壁画已经略显斑驳，城区内也兴起了一些较新的壁画，和老壁画一样游客都可以和他们互动拍照。壁画散落在城区各个街巷，如果想要全部打卡这些壁画，可以租一辆三轮车，车夫会带你去所有的点拍照。
下面这个不是真的车夫，是泽彬！
他也帮我拍了一些照片，新壁画的好处就是几乎没有游客。反正再过几十年它们也就是老壁画了，哈哈。
在走街串巷的过程中，各种中文招牌也给我留下了深刻的印象。莫名对这些复古的中文招牌特别感兴趣。也许它让我想起了爱看的电影《追龙》。对比了一下，这里的华人和电影里的跛豪应该是在同一个时代背景下漂洋过海，离开家乡。只是槟城的华人并不像电影里一样过着刀刃舔血的生活。而是落地生根，从而造就了一座《孤独星球》榜首的“美食之城” 。
最后两个景点再交待一下！一个是爱情巷，可能是名字起的好吧，否则和周边的街巷差别也不大。街上的酒吧很多，很多露天的摊位，欧美人居多。建议简单的拍照打卡，想喝酒的另算。
另一个是姓氏桥，其中最出名保存最完好的是姓周桥。简单的说就是当年华人沿海在海面上而建的高脚木屋，形成了规模后就变成了村子，通常都是同一个家族也就是同一个姓氏的人住在一起，所以称作姓X桥。
显然姓周桥已经有些商业化了，我还替泽彬找到了姓林桥，明显更加生活化一些。虽然住着这样的老房子，但是装扮的都是非常精致的。那就祝他们生活越来越好吧！

学渣宝典（学霸免看！）

【出入境】
1、去马来西亚之前肯定要办理签证，有网站是可以自己填写信息，然后缴费办理的。但是你能看到这篇学渣宝典，我相信你肯定跟我一样看不懂英文。那么就在马蜂窝或者找万能的某宝代办一下！我花了180块，只要提供护照信息和电子版的照片就好，当天就出签了。
2、尽量把机票、酒店订单打印出来，海关可能会检查。语言不好就不要给自己添麻烦，东西都准备妥当。在吉隆坡下了飞机根据引导到外国人出关的地方出关！没错，落地以后，你就已经是个外国人了。
【货币兑换】
在国内的银行通常是不能直接兑换马币的（马来西亚钱币叫做“林吉特”，但是能跟你说中文的都会把它叫做“马币”），我出发的时候汇率是0.6左右，也就是100RMB换60MYR（大约）。我问了萧山机场的兑换点，每换100你会比汇率亏20块左右。吉隆坡的兑换点，英文我也说不好，所以咱也不敢问！结果在有银联标识的ATM上自助取款的（有中文界面）。用的是中国银行的借记卡，每次扣15元的手续费。所以，有银联标识的ATM机，学渣也要了解使用一下！
【交通出行】
1、吉隆坡的交通非常发达，从机场到市区建议乘坐巴士，在市区游玩建议乘坐地铁或打车。学渣要记住的几个知识点！下载好百度地图和谷歌地图，建议两个地图比较着用，这样不容易出错。吉隆坡最大的交通中转站叫做“KL Sentral”，所以最好记得这个名字。
2、吉隆坡的地铁不像国内的编好1234号线这样很好认，因为有不同的地铁公司，名字都会一串很长，对照起来有些麻烦，比如KTM SEREMBAN LINE，就是去黑风洞的地铁。技巧就是看颜色，这个和国内是同理的，每条线的颜色肯定有区别，对照手机地图上的指引即可。
3、下载Grab打车软件，这个就是东南亚版的滴滴。打车的时候有中文提示，因为基于谷歌地图的支持，上车点标记的特别准确。而且很多地方都有醒目的Grab停靠点，方便定位。
司机发消息来也可以实时谷歌翻译成中文，同理你给他发中文他也能看的明白。付款我是在钱包里绑定了信用卡，回国记得解绑一下即可，据说也可以现金支付。我一般和司机都不用交流，就可以把车打的妥妥的。（有图有真相）我第一个打到的司机居然还是个女司机，还有评分和车牌号等消息，基于我也不知道为什么的原则，还是给她打个马赛克吧！
记住Grab在马六甲和槟城这两座没有地铁的城市特别重要，价格比国内打车还便宜！
【手机上网】
现在可以在境外上网的设备很多，但是我个人不太喜欢移动WiFi，因为多带一个东西，而且耗电特别快。建议买当地电话卡，马来的电话卡还是很便宜的，购买的时候注意下使用方法和套餐内容即可。我用的是“亿点连接”，一张固定的电话卡，可以反复使用。每一次只要在app上购买要去国家的套餐即可，和本地电话卡差不多，省事！
【杂七杂八】
1、在所有开空调的地方不可吸烟，离人5米范围内不可吸烟，否则会有重罚。如果实在拿捏不准，你就看别人在哪抽你就在哪抽吧！当然吸烟有害健康，建议不抽；
2、马来西亚的便利店东西比较贵，跟泰国几乎没法比，特别是可乐折算人民币大概10块一瓶。据说是因为当地对这种含糖高的垃圾食品收了重税，意在保护人民健康；
3、亚航飞机上以及客运大巴上冷气开的都特别足，特别是飞机上，不知道是不是为了多卖毯子给旅客。所以乘坐大交通时，即便是夏天，也建议随身携带一件薄外套；
4、马来西亚大多数人信奉伊斯兰教，当然也有黑风洞这种印度教的圣地。如果可以的话，简单了解一下宗教的习俗和禁忌 ，以免造成什么不必要的误会；
5、马来西亚和国内是同一个时区，没有时差，千万不要装B发一些倒时差之类可笑的朋友圈呀！
能想到的就这么多了吧！如果有什么遗漏的欢迎评论区讨论。收好这份我总结出来的学渣宝典，让我们像学霸一样说走就走吧！
