游记 人文游 小众 摄影 自助游 自然奇观 2018/12/14
克罗地亚 拥有绵长的海岸线，无数的离岛，众多历史悠久的古城遗迹，使得 克罗地亚 像一颗明珠闪耀在蓝色亚得里亚海的东边。
因《权力的游戏》拍摄地而知名的 克罗地亚 ，可能首先会让你想到马克西姆演奏的《 克罗地亚 狂想曲》，而今年在世界杯上“格子军团”又凭借高昂不懈的斗志燃烧了球迷的整个夏日。
而旅途都不止是这些印象与想象，每天的日升日落，清风雨露，好坏意外，与他人的邂逅，与当地人的交流，都成为自己的故事。
旅途中的所见所感，所有的收获，好的坏的，沉淀成为的故事，就是旅行的意义。

失恋博物馆 中所有的故事，

【 失恋博物馆 】
博物馆展览世界各地的人捐赠的失恋／分手时的纪念品，每一个纪念品都配有关于这个它和它原主人的故事。
买票的时候告知你来自的 中国 ，会给你一本中文的失恋故事手册。
【 圣马可教堂 】
圣马可教堂 是一座哥特式建筑，带有绚丽的彩瓦屋顶。屋顶的图案配色与 克罗地亚 国旗是一致的。
鸽子在教堂悠闲。
教堂里陈列着 克罗地亚 著名雕塑家梅什特罗维奇的作品。
【观景平台】

【 萨格勒布 大教堂】
圣母升天大教堂 俗称 萨格勒布 大教堂，原址在1880年的地震中被毁，后来于1899年重建。

斯普利特

海岸线高耸的山峰串起的防线，在 斯普利特 留下了一个半岛。
不似杜城战后修缮的新古城， 斯普利特 古城不大却更显得古朴和历史沧桑感。
在圣多米努斯大教堂邂逅了一场充盈喜悦的当地婚礼，让我们跟随着喜悦一起扬长而去。
我们的小俞飞鸿在Trsteno植物园。
杜布罗夫尼克 开往 斯普利特 的中途，高山大海，十分雄伟。这里是一个弯角，叫做Vruja，狂风肆虐，照片右下方是个纯白圣母雕像，不想下方的海边还有一座古朴的酒店Vruja Resort，有时间的人可以来感受下。
【奥米什 Omiš】
奥米什独特的地形，两座崇山峻岭之间，一条大河开拓而出，入 海口 两岸就是奥米什的城镇。
入 海口 的后方，大河从另一座山体中穿涌而出，山体之上有一座Mila Gojsalić的雕像，远眺峡谷出 海口 。Mila Gojsalić是 克罗地亚 的女性民族英雄，在于奥斯曼 土耳其 的战争中她自愿牺牲贞洁并渗透到土耳其 大军中摧毁了 土耳其 军队的储备，使 克罗地亚 人 成功 击败了 土耳其 军队。
这个视角下的Mila Gojsalić壮怀又有一丝悲凉。

满场跳动着欢喜，最后他们跟着 克罗地亚 旗帜随着吉他手和手风琴手一路唱跳踏出古城。

【Klis城堡】
《权力的游戏》中弥林城堡的取景地
城堡后边有一座足球场，身着 克罗地亚 秋衣的两队在比赛。想想 克罗地亚 刚拿了世界杯亚军，看比赛的时候就不由激动。
在Kaštel Gomilica找《权力的游戏》的拍摄地时，无意间发现这座小房子，墙壁外侧的花盆、窗台、装饰、甚至门帘，都是用贝壳拼起来的。
非常引爆少女心和我的少男心。
当时看到的我直接震惊了！这家主人也太会玩了，非常有创意，而且做这件事得需要足够的耐心，当时房门紧闭，否则非要膜拜一下房主不可。

扎达尔

从 斯普利特 到 扎达尔 ，一路经过的城市都值得停留。
扎达尔 古城充满生活气息，海风琴在这里奏鸣，太阳敬礼在这里闪耀。

萨拉热窝

从 克罗地亚 的斯拉沃尼 亚布罗德 Slavonski Brod入境 波黑 ，要经过 萨瓦 河，这座铁桥横跨 萨瓦河，看起来有些年代感。
伊斯兰教、东正教、天主教、犹太教数世纪以来共存其中，被称为“巴尔干的 耶路撒冷 ”。
从战争中恢复没有多少年，本以为是苦大仇深，城中见到的却都是怡然自得。
从黄堡俯瞰 萨拉热窝 ，老屋和 新建 筑并存的一座不小的城市。
黄堡看向旁边的山上，云雾缭绕。
萨拉热窝 在1885年成为 欧洲 首个也是全世界第二个全天运行路面电车的城市。
各色电车在车上徜徉，想想当中的铁轨最久已经有一百多年历史了不免会觉不可思议。
纪念 萨拉热窝 被害儿童广场前地面上的血迹雕刻。
广场上有一个中间为绿色的雕塑的喷泉，一旁有几个铁制的圆筒，圆筒上标刻着在内战时期遇害的未满18岁的所有 萨拉热窝 孩子的生卒年，是一个主题非常沉重地地方。

【黑山】

【奥斯特洛修道院】
是 塞尔维亚 东正教的一所修道院，上半部分用来供奉圣十字，下半部分供奉着它的建造者奥斯特洛的遗体。整个修道院几乎是嵌在了巨大的山体之中，俗称 黑山 的悬空寺，它是巴尔干半岛地区参拜人数最多的修道院之一。
我们跟随教徒进去参观了奥斯特洛的遗体，放在一个类似于木乃伊的盒子中，盒子长约一米，彩绘华丽，遗骸用华丽的布盖着，布上放了一张奥斯特洛的画。氛围庄严肃穆。
【 基督救世主大教堂 】
位于 黑山 首都 波德戈里察 ，内部壁画非常华丽，不用门票，可以拍照，还会有团体游客参观，我们来的时候只有两个教徒在里面，非常安静，感受很好。
从 波德戈里察 前往Pavlova Strana的途中遭遇山火，冒险从浓烟中穿过，隔着车门都感受到了山火的炙热。
当时风很大，在到达Pavlova Strana观景点之后，山火已经从山路的下侧烧到了山路的上边，我们如果再晚一点就无法通过了，甚至通过的当时如果大风正起，也是非常危险的事情。
Pavlova Strana是一处河湾，奈何镜头不够广，河流在绕过图中的两座山后流入巴尔干最大的湖泊——斯库台湖，虽然也不远，由于时间关系我们就没办法去斯库台湖了，还要赶去圣斯特凡岛看日落。

攻略简要

【路线】
维也纳 → 萨格勒布 → 萨拉热窝 → 莫斯塔尔 → 科托尔 → 杜布罗夫尼克 (两晚)→ 斯普利特 → 扎达尔→ 罗维尼 → 布莱德 (两晚)→ 卢布尔雅那 → 维也纳 
我们贪多，所以行程比较赶。
【租车相关】
导航使用探途离线地图，在国内下载好，到地方直接使用，方便。还可以用Google地图，但是用移动的国外套餐还得得翻墙才行。
加油站都可以刷卡，自助加油或者让工作人员帮忙，加满油后去店内指明是几号油枪，刷卡就可以了。高速收费也全都可以刷卡。
【签证】
而由于签证问题， 克罗地亚 和 黑山 是单独的签证，且是拥有 申根 两次或多次出入签证就可以免签的。也就是说拥有了有效期内的两次或多次 申根 签证 克罗地亚 、 黑山 都可以随意出入。
所以我们选择了航班较多的 维也纳 出入 欧洲 ，这样可以办理两次出入的奥迪利 申根 签证， 斯洛文尼亚 和 意大利 都是 申根 签， 波黑 免签，一举多得，计划通。
tips： 奥地利 签证很抠门，卡着天数和次数给。也可以从 布达佩斯 走也就是申请 匈牙利 签证，比较容易给多次。我办 奥地利 签证时签证中心的人还说“你去 克罗地亚 最久你得先办 克罗地亚 签证”，非常不合理的要求，我硬是要办了。
【货币】
奥地利 、 斯洛文尼亚 、 黑山 都是使用欧元，大多数可以刷信用卡，比较方便，建议还是换些欧元备用。
克罗地亚 使用 克罗地亚 库纳，也收欧元，但很多景区门票只收库纳，但国内无法兑换 克罗地亚 库纳，可以在当地用银联取现。汇率大概是 库纳：人民币=1.1：1。
波黑 使用 波黑 马克，也基本都收欧元，但付欧元都是用马克找零钱，这就很亏了，店铺计算的汇率也非常不好，建议能刷卡就刷卡。汇率大概是 马克：人民币 = 4：1。跟欧元就是1:2了。
【手机通讯】
沃达丰电话卡在 波黑 和 黑山 都没有信号，所以需要在当地现办，但是时间短又不知道，某移动有国外流量套餐倒是可以在 波黑 和 黑山 使用。
移动套餐不能用Google地图，电话卡可以而且便宜，不差钱想省事且不用Google地图导航的的建议开流量套餐，要不然可以安排 波黑 黑山 在行程前段部分，用流量套餐，后半部分用电话卡。

尾巴

各民族、宗教教派混杂在巴尔干半岛上，使得这里一度成为 欧洲 的火药桶。
但在 和平 时期，这些元素给旅行者带来的却是多元的旅行体验。
民族宗教文化建筑受到受 意大利 、 奥地利 、 土耳其 甚至 俄罗斯 的影响，在驱车穿越这几个国家时，几乎每天都是不一样的风情，甚至在同一个城市都会讶异于丰富的所见。
对于老一辈来讲，兄弟社会主义国家南斯拉夫更有让他们印象深刻的电影、文化在当时年轻的他们的群体中疯狂流传。
99年南斯拉夫大使馆被炸，一度认为与他们是亲密关系。但在了解到 塞尔维亚 族与 克罗地亚 族、 斯洛文尼亚 各种不对付才导致南斯拉夫解体时，感情是复杂的。
除此以外，巴尔干半岛上的山与水，点点明珠般的半岛与离岛，亚得里亚海海水的净蓝与壮美的日落，阿尔卑斯山 小支脉的童话镇—— 布莱德 仙境，让在行程中的我们每天都期待出发，渴望到达，然后找一处美地静静的等待夕阳，感受海风。
如果你正在苦思冥想一个可以休闲又有美丽自然风光的地方，别纠结了， 克罗地亚 呀。
