前言

出发前合影镇楼
无意间在网上看到一则关于华纳兄弟主题公园要在 阿布扎比 亚斯岛开业的消息，早就听闻亚斯岛是阿联酋的旅游度假胜地，华纳又是伴着我们童年成长、乃至现今的重要一部分。旅行就是这样，决定好了，我们就出发吧！
这次是一家四口去的，我跟孩子妈妈，还有3岁半的女儿、2岁的儿子。所以决定租车自驾，具体的方法在文末有详细的攻略介绍。
这次去阿布扎比真的特别好玩，每天都很惬意，睡睡懒觉，游游泳，逛逛街，吃吃东西，去游乐场。特别是华纳兄弟主题乐园，弟弟虽然才2岁，只要身高满了90cm园内大部分的项目都可以玩，最重要的是，不用排队，具体我在后面会讲！总之这次的行程超级好玩，超级舒服。
我们定的行程日期是8月27日-9月1日，很多人会讲到这几天会比较热，从我们实际体验来说，户外肯定会温度高一些。其实体感温度还好，大部分时候都是在室内或者车子里，还是比较舒服的。
本篇游记依然会从衣、食、住、行，签证、景点、行程等，为大家尽可能详细介绍一下阿布扎比 。让大家可以参考本游记来游玩阿布扎比 。

华纳兄弟主题乐园

激动人心的时刻到了，华纳兄弟主题乐园是这次带弟弟妹妹来最开心的一个地方了，妹妹一整天能量满满，从早上入园到晚上出园High完全程。只要是碰到的人偶一定是冲过去排队跟人家合影，并且能玩的她都玩，什么跳楼机啊，激流勇进啊，天上飞来飞去的啊，旋转木马啊什么的就更别说了。就连鬼屋都进了，你敢想么？简直开心到爆。还好那个类似过山车的她身高实在是不够，不然……

总结：
1、推荐从岩床区开始玩，一路顺着往下玩即可。
2、如果是成年人，在哥谭市和大都市花费的时间可能会多一些，可以酌情安排一下时间。
3、各路超级英雄玩偶合影的时间是不确定的，所以碰到了想合影，马上就拍，下一次再碰到就不一定咯。
4、里面各个餐厅的美食也不太一样，可以多转几家选择适合自己感觉的尝试一下。
5、餐厅支持银联、mastercard，visa支付，也可以携带现金。
华纳兄弟主题乐园真的是太好玩了，一整天的时间甚至都有点不太够，最可惜的是我们没有能跟其他的几个英雄拍照。最后我们结束了，妹子还跟我转了两圈跑去找他们合影，都没找到。
我想等孩子再大一些再领他们来一次。感受也会不太一样。总之强烈推荐带着孩子来这里体验一下不一样的主题乐园。

法拉利主题公园

2岁、3岁的小孩子来法拉利公园玩，年龄上还是有点小的，再大一点点就会比较好玩。门票可以在现场买，也可以提前在网上买好扫码直接进入。价格大约在400元左右，3岁以上的小孩子需要门票。
现场人不是很多，里面有一些项目还是比较好玩的，园区也不是很大，差不多一下午的时间也就够玩了。
我觉得各位来法拉利一定要体验一下Formula Rossa，同类型中最大的云霄飞车，据说在5秒内直接到达240公里，紧接着爬升至52米高空，然后高速上下带你翻滚摩擦。我年龄大了，享受不了，你们这些喜欢刺激的可以来试试！

我的是这种的，在租租车网站租车的时候直接就勾选办理了。
