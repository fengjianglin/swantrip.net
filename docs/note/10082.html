【天鹅之旅首发】烟台：各种海鲜，各种蓝

游记 亲子 暑假 美食 自助游 2018/12/24

前言

我也是很佩服自己，别人的青岛、威海、烟台三地游才五天四晚，这个暑假我光烟台一个地方就去了六天五晚，还没去到蓬莱、长岛等烟台著名景点。
因为带着一个只想吃坐睡的懒惰中年男子，很多地方都没去。不过，作为一个只要出门旅游就开心的佛系旅行者，而且还有我最爱的蓝天白云和海鲜加持，所以这趟烟台之旅我也并不觉得遗憾。毕竟人生路漫漫，这次有些景点没去成，下回还是有机会的，反而给了我一个下次再来烟台的理由。
旅行多了，越来越不愿意做攻略。酒店、景点、餐厅一切随缘。
大交通：杭州——烟台 飞机来回
市内交通：首汽约车、神州专车、公交车
住宿：
烟台上邦戴斯酒店*4晚
孩子大了，如果是三人睡一个床除非是2米大床，否则真的是有点挤。所以我一般都选择有亲子房的酒店订。不过，这次订的酒店确实有点失策，以为离烟台市政府较近属于市中心，其实离各大景区都有点远。
烟台金子的海边小屋（民宿）*1晚
用餐：
山东鲁菜最为有名，烟台又在海边，最爱的海鲜每天吃起来！

第一天 出发，鲁菜海鲜吃起来！

我自己出门总是想着尽量坐早班机，多利用一天时间。于是，我按照老经验订了清晨7点多起飞的飞机，在飞机上昏昏沉沉的睡了一小觉就到了。
从机场打车到酒店也不过十一点不到，说真的，看到酒店略有点失望。酒店的外观是绿色、灰色的墙面，看起来不太高档，而且我们入住的时候还有部分在施工，略有点吵。不过进了大厅还是可以看到这家酒店定位的是精品酒店，因为内部的装饰还是有点设计感的。
酒店走廊和公共空间也用很鲜明的颜色装饰。
我们的房型价格高，房间自然也挺大，大到简直感觉有点空荡荡的了。房间整体装饰风格很明快，用的颜色都是鲜艳的柠檬黄、绿色、紫色等，可能我还是不太习惯北方人民的审美风格。
洗手间内配有浴缸，干湿分离的设计还是不错的，还有两个洗手台盆，在赶时间的时候还能两个人同时洗漱。
酒店内部自带餐厅，客房内有餐厅的餐单，都是些家常便饭，价格比较适中，不过身在海滨城市没有什么海鲜有点说不过去哦。但是懒惰的中年男子说早上起太早，实在没力气出门吃饭了，所以我们就在餐厅里吃了点。
餐厅名为国王餐厅，因为没睡够，一家三口都食欲不振，就点了三个菜，味道不功不过，不过餐厅的装修风格还是挺讨喜的。
吃完饭继续回房休息，看来我本来想着早点坐飞机不浪费游玩时间的计划落空了，作为一个吃货，那我就利用这时间好好研究晚饭吃什么吧。
小红鲸据说是酒店所在的莱山区排名第一的海鲜餐厅，离我们住的地方只有不到一公里，所以我们当然要到这里去尝一下。这家店不太好找，外面是绿色的建筑工地护栏，绕了一大圈才找到店。说实话店的装修什么的都很一般，感觉就是一家普通的家常馆子，这么有名一定是因为他家的味道特别好。
走进餐厅里面就是海鲜展示台，各种新鲜的海鲜铺了一地。
我们两大一小按照以往的惯例点了六个菜。来烟台前，在网上看到海肠是烟台的特色菜，当然不可以错过。
海肠确实是好东西，不过我们南方人从来没吃过，看这不可描述的样子还真是无福消受，只斗胆吃了几根，白白浪费了。
海胆蒸蛋，味道很醇厚，比我在三亚吃的鲜得多。
我们平时也经常吃的章鱼又叫八爪鱼，在山东被称为八带。山东鲁菜用酱爆的烧法比较多，酱香浓郁，八爪鱼也是用酱爆的烧法烧的。
大家都知道北方海参的质量要好于南方海参，作为浙江人虽然平时也经常吃海鲜，不过海参确实吃的不多，来到海参的原产地，当然要点一盘海参过过瘾。平时在家妈妈烧海参都是用来熬汤的，这里用整条的海参和大葱烧起来，让人吃的就是满足。
海星也从来没吃过，点上！不过味道就有点怪怪的，我也觉得不太好吃。
山东人吃面食比较多，所以我们最后也点了两个馒头作为主食，上来一看，真的被吓了一跳，一个馒头和我脸比也小不了多少，我们三人分食一个也没吃完。
不得不说烟台人民真的很实诚，菜量都是够够的，食材也都是真材实料。这一餐吃了不到500元，量实在是太大了，剩了一大半，以至于我们后来几餐每餐都只敢点三到四个菜。
晚上我们回酒店的时候发现服务员已经帮我们开了夜床，还奉送了两瓶葡萄酒和两份点心。

第二天 去张裕酒文化博物馆品个酒，到烟台山上赏烟台城市美景

烟台最有名的景点是蓬莱阁和长岛等，离我们住的酒店都太远了。如果要去早上就得一大早起来，当然也可以在当地租一辆车，但是某人说出来度假只想休息，而且开车就不能喝酒了，所以我们放弃了去这些著名景点的计划，改为就在市内转转玩玩了。
今天，我们就打算去市内的张裕酒文化博物馆和烟台山景区转转看看。
早上在酒店的国王餐厅吃的饱饱的，准备出发！
张裕酒文化博物馆位于烟台市中心的芝罘区大马路张裕公司原址，它是中国第一家世界级葡萄酒专业博物馆。
这一次我们买了含品鉴酒和亲手制作白兰地酒的套票。
张裕酒文化博物馆由酒文化广场、百年地下大酒窖、综合大厅、历史厅、影视厅、现代厅、字画厅、珍品厅、休闲购物厅等组成。在这里可以了解到葡萄酒以及张裕公司在中国发展的历史。
博物馆地下还有一个酒窖，是亚洲第一大酒窖。这是我第一次参观酒窖，觉得一切都很新鲜。葡萄酒窖内四季恒温，自然保持在摄氏12--18度之间，对酒的陈酿非常有益。
酒窖内还有三个硕大无比的橡木桶，容量为15000公升，被称为桶王。
酒窖内还有一个古朴典雅的酒吧，游客可以在这个酒吧欣赏着欧洲古典音乐、品尝各种美酒，有专人会为游客介绍葡萄酒知识，在这里，游客可以体味张裕古老的历史，接受酒文化的熏陶。
白兰地制作体验其实就是把已酿造好的酒灌入小酒瓶中，盖上酒瓶盖，贴上酒标。其中盖酒瓶盖稍微需要一些技术含量，需要自己用锤子把木塞打入酒瓶中，小朋友在这里玩的不亦乐乎。
下一站我们的目的地是烟台山景区，这里离张裕酒文化博物馆并不远，走过去也不过二十分钟左右。
烟台是个靠海的城市，市中心就能看到海，简直太幸福了，我们一路沿着海岸走过去，远远地就望到了烟台山上的灯塔。
在路边看到了铁板鱿鱼的车子，虽然在杭州也偶尔吃过铁板鱿鱼，但是毕竟这里的铁板鱿鱼更正宗啊。买买买！
烟台的鱿鱼果然比我们杭州的大的多，这么大一个15元，一个吃下去就饱了，午饭都吃不下了。
小朋友看到网红冒烟冰淇淋又忍不住了，我们一人买了一份尝尝，说真的，味道很一般。
山顶的灯塔是烟台市内最高点，可以乘坐灯塔内的电梯上到11层瞭望台，在瞭望台上可以观赏烟台市区美景，塔上的风很大，站在瞭望台上被风吹得几乎站不住，不过看到美景还是觉得不虚此行。
景区内还有众多的老西洋建筑和陈列馆、纪念馆。清朝同治年间，烟台成为山东第一个开埠通商口岸，当时有十几个国家在烟台设立领事或代理领事，我们也去参观了烟台开埠陈列馆（美国领事馆官邸旧址）、中国京剧艺术腊像馆等场馆，也了解到了不少历史知识。
公园内的绿化也很不错，夏季漫步在这绿色长廊中，真的是惬意不过。
从山上下来，漫步在海边，走在礁石上，与大海可以做个亲密接触，不过要注意安全哦。
穿着细带凉鞋的我在礁石上滑了好几下，还是找个地方坐坐，赏赏海景更适合我。
从烟台山景区出来，这一条马路上有很多西式老房子，据说曾经是各国领事馆。这里是拍照的好地方，难怪有新人在这里取景拍摄婚纱照呢。
这条街好像在拆迁中，让我们俩都觉得很费解，这么有特色的街道，立上几个大型遮阳伞，改成咖啡厅多好啊，坐在街边喝喝咖啡，吹吹海风，一定很有情调。
走在街上路过了一家驴肉饺子店，虽然已快到晚餐时间了，某人经受不住诱惑，强烈要求进去尝尝。驴肉大概在北方比较有市场，作为南方人，其实我们没有吃驴肉的习惯，不过某人说小时候来山东玩时曾经吃过驴肉，印象特别深。
我也斗胆尝了一个，味道还不粗，不过总觉得有些心理障碍，剩下的就都给他吃了。
晚饭我们在旺角小渔村吃，这是我在网上找到排名较高的一家店，走进店里还看到墙上挂着网友最喜欢的餐厅的招牌，看来是一家网红店哦。店里的顾客真的是很多，我们大概五点半到，已经没有单独的座位了，只能和另一家人拼桌。
山东人民的网红店，还真的不太注重外在美，应该说内部装修很接地气。
这一回，我们点了山东著名的鲅鱼水饺，水饺很霸气，一个就有差不多拳头大小。
还点了鲍鱼，我们南方的鲍鱼都是按个儿卖的，这里的鲍鱼一上一大盘。
葱烧海参特别好吃，应该是我吃过的海参里味道最好的。
我们还点了这里著名的疙瘩汤，这里的疙瘩汤不像我们南方的面疙瘩都有手指粗细，疙瘩汤里的疙瘩都很小颗。
个人认为这家店是我们在烟台吃的最最美味的一餐，现在想起来还会流口水。

第三天 网红养马岛走起来

春末夏初的时候，抖音上养马岛很红，我也不能免俗地中了毒。所以，第三天我们安排在养马岛泡了一整天。
养马岛在牟平区，我们从市区打车过去开了半个多小时。小朋友只喜欢玩海水、沙滩，所以我们直奔海滨浴场。可能是抖音效果太好了，这里的游客还真不少。
虽然不如抖音上说是中国的马尔代夫那么夸张，养马岛的海水还是湛蓝的，配上金黄的沙滩，景色还是很不错的。不过，可能是因为北方海域一年只有一个季节游客比较多的缘故吧，这里的物价也真的算不得便宜，无论是租用沙滩伞还是用餐，价格比起马来西亚、泰国都高出很大一截。市区15元一串的大鱿鱼，这里最低要30，其他食物也都比市内贵至少一倍。可能是因为这景点刚热起来，基础建设还真的做的不太好，这里的洗手间真心是我去过的景点里最脏乱差的一个了。
不过看在这湛蓝的海水和天空的份上，我觉得这一切还是可以接受的，只是希望它能尽快改造的更好些。
都说这座岛四周盛产海参、扇贝、鲍鱼、对虾、牡蛎、天鹅蛋等海产品，是海鲜爱好者的天堂，我们当然要去品品这里的原汁原味。从网上找到养马岛上最有名的就是一家名为鑫阳烧烤饭店的餐厅，从网上查到离海滨浴场也不算太远，我们就打算一路走过去，也顺便看看养马岛的的风景。一路走来感觉住在这岛上的居民生活还真是幸福，想吃海鲜了可以直接下海去打捞，想吃水果也可以直接吃房前屋后自己种的水果，葡萄、无花果等各种品种都有。
我们不到5点半，就赶到了餐厅，餐厅外观比较原生态，不过来用餐的人很多，停车场停满了汽车，餐厅外也坐满了等着开餐的人。5点半一到，老板就开始接受点餐了。
在餐厅外面我就看到了一箱箱的鱼到货了，我问老板这是什么鱼？老板说这就是鲅鱼，鲅鱼饺子就是用这个鱼现做的。那怎么能错过呢？点上一份！
另外我们还点了红烧小鲍鱼，虾丸汤和一份天鹅蛋。小鲍鱼和鲅鱼饺子一如既往的好吃，虾丸汤更是获得了我们三个的一致好评，比在火锅店里吃到的虾滑更加鲜美。相比较而言，天鹅蛋就比较一般了，因为太老了，有点嚼不动。
吃完饭坐着出租车从跨海大桥上开过，一路上海风吹拂，把暑气完全驱散了，不得不感叹北方城市的夏季真的比南方好过太多了，即使白天再热，一到傍晚就变得很清凉舒爽了。
本来以为北方的海边不如南方海边的太阳晒，一疏忽忘记涂防晒霜了，晚上回到酒店发现全家人都被晒脱皮了，一洗澡就更痛了。听出租车司机说烟台的紫外线一样是很厉害的，难怪司机自己都带着防晒胳膊套呢。

第四天 漫步东炮台海滨风景区

在养马岛被晒了一天，大家都被晒晕了，更没有力气早起去远的景区了，我们决定去市区内的东炮台海滨风景区。
东炮台是一个免费的景点，这里是个爱国主义教育场所，也很适合带孩子去参观。
东炮台始建于1891年，当年，清末北洋大臣李鸿章奏请光绪皇帝在此建造了炮台，以防北来敌船侵袭。耗银达百万两的古炮台建有台门、清兵营房、指挥所、运兵壕沟、隐蔽所等海防设施，安装了当时世界上技术最先进、有“世界海岸炮之王”之美誉的克虏伯大炮。古炮台气势雄伟，中西合璧，是中国近代海防史上保存最为完整的海防设施之一。
这里的自然风景也很美，海水湛蓝，风光旖旎。
景点内还有烟台特产小吃海菜饼，价格不贵，5元一个，清香鲜美的味道让人难忘。
太阳底下天气实在太热，我们想找个有个格调的海边小餐厅吃点冰淇淋。可能是山东的商业氛围真的不如江浙地区吧，沿着海边走了好半天也没有找到符合心目中想要的那种餐厅。
路过的月亮湾，因为实在太热了，实在没有力气走近欣赏，只远远地拍了一张照。
最后决定还是打车到市中心大商场觅食，哈根达斯吃起来。
路过麦当劳，小朋友还是最好这一口。
我们住的酒店离烟台大学比较近，根据经验，离大学近的地方一般都有美食街，烟台也不例外。经我们考察，一致认为烟台的物价比杭州便宜不少，同样的品种价格比杭州都要低至少三分之一。
北方人爱吃饺子，来到山东当然要入乡随俗。晚上我们就在一品香饺子馆随便吃了点，点了50多元的一大盘饺子，真的好好吃，是我吃过饺子里面数一数二的，是海鲜馅料的，食材很新鲜。还点了一大盘虾和羊排，爱吃肉的父子俩对羊排赞不绝口，山东的菜量确实大，我们三个人，点了两菜一汤，一盘饺子都吃的饱饱的了。

第五天 住一天海景房，感受面朝大海，春暖花开

在市区住了四晚，小朋友的爸爸说感觉如果不住海边这趟烟台白来了。他对景点没什么兴致，对于度假的唯一要求就是要住在能看到海的房子里。
说真的，烟台是个季节性很强的旅游城市，淡旺季的住房价格相差很大，所以我们预定了一间有海景的民宿。
民宿位于开发区的招商马尔贝拉小区，不在市中心，所以略微有点偏。不过如果喜欢安静的人，选择住在这边还是挺好的。
这里的楼房都是全白的不规则形的，矗立在海边与蓝色的大海很相配。
房间的设施算不得豪华，不过也能满足日常的生活所需。民宿就是有这样一个特点，短时间内让人感觉自己好像就是这屋子的主人，而不是过客。
住在这里，在客房和阳台都能看到大海，这点对于我们这种平时不住在海边的人来说真的诱惑力很大。
毕竟是北方，八月下旬虽还是夏季，坐在阳台上海风吹起来还是觉得有点凉飕飕，坐了一会儿我就觉得有点受不住了。
傍晚时分，我们去小区里走走，小区就建在海边，这里没有什么游客，听着海涛拍岸的声音，心情真的是很好。
因为人少，这里也是拍照片的好场所，作为一个喜欢自拍的胖子，我在小区里尽情拍了个够。
根据民宿主人的建议，我们打车到附近的永旺城去吃晚饭。永旺城是个大型的商城，里面除了商场和超市还有不少餐饮店。
田大榜疙瘩汤应该是烟台有名的连锁店，我在好多地方都看到了它家的招牌。既然这么有名，当然要去尝一尝喽。这家店就开在永旺城里面，和之前吃过的烟台其他餐厅相比，环境还是挺不错的。
蛤蜊烧鸡，应该也是烟台特有的烧法，海鲜的鲜味与鸡肉的淳香搭配在一起，真的可以被称为绝配，连我这样不爱吃鸡的人也连吃了好几块鸡肉。
爱吃羊肉的父子俩这回又点了羊排，好大一份，也给他们吃光了，看来味道不错哦。
在疙瘩汤店里，当然要点疙瘩汤，果然名不虚传，味道很好。
我们还点了虾酱蒸蛋，虾酱蒸蛋其貌不扬，但是味道真的是很鲜美。
吃完饭回到住处，夜晚的月色迷人，坐在阳台上看月光，觉得住在这里真是好享受。

第六天 在城堡圆一个公主梦

在烟台的最后一天，早晨起来外面狂风大作，下起了大暴雨。
暑假期间有一部电影《西虹市首富》很热卖，里面的豪宅古堡就是在烟台的文成城堡取景的。每个女人心里大概都有一个公主梦，去不了欧洲看古堡，去烟台看城堡也是极好的。可是因为天气原因没有时间去文成城堡了，那就改去附近的张裕卡斯特酒庄看看吧。
这里很适合拍照，带着孩子来了解一下葡萄酒的历史和生产知识也是很有意思的事。
买一张门票就可以在酒庄内逛上半天，好好地消磨一下时光。张裕卡斯特酒庄是由中国葡萄酒业巨头张裕公司和法国葡萄酒业巨头卡斯特公司合资兴建的，酒庄占地面积非常大，里面除了酿酒葡萄园、葡萄长廊外还有几幢城堡式建筑。
酒庄的大门就很显气派：
沿着葡萄长廊一路往里走，这里的葡萄可不能随意采摘品尝哦。
酒庄内的城堡式建筑里有酒窖、品酒室，还有酿酒车间，可以让游客对葡萄酒有个初步的认识。
这里的葡萄酒还有整桶出售的，桶上的标记大概是所有者的签名吧。
在城堡楼上向下眺望景色也不错，大片的绿化，大面积的葡萄园，让来自南方看惯了江南水乡秀丽风景的我又领略了北方景色的大气。
酒庄的出口处是一个商业展示区，看到广告发现这么有特色的庄园里竟然还能办婚宴，而且价格还这么实惠，让我不禁羡慕这里的新娘真的好幸福。
烟台的最后一餐，我们应小朋友的要求吃了自助小火锅吉夻良煮，价格很实惠，好像才三十九一个人，可以吃一个小时。桌上的流水线不停的转，火锅食材放在上面，想吃什么就自取，对小朋友来说是个很新鲜的体验。
吃完晚餐也该返程了。虽然一早经历了大风大雨，傍晚时分又是满天霞光。
六天五晚的烟台之行就此结束，烟台蔚蓝的天空和湛蓝的大海以及物美价廉高品质的海鲜给我留下了深刻印象。我想，有机会我还会再来的，把没去过的景点再查漏补缺一遍，顺便再好好吃上几顿海鲜。
