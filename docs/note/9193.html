题记

波西米亚，一个让人难以捉摸的神秘词汇。一曲《布拉格广场》让这座城市和捷克一同风靡了中国的大江南北，而一部部令人辗转的爱情电影又为这座城市赋予了浓厚的浪漫情怀。来到波西米亚，一切仿佛都变得有点不切实际，漫步在充满异域风情的城市街道之中，无需去刻意追逐那些著名的景点，当你问起我对布拉格的印象，或许我会开窗送你一米阳光，因为我在这儿看到过最动人的落日。
奥地利，这个在世人眼中一个音乐萦绕的国度，多瑙河如宝石般的流水蜿蜒流淌过这片国土。来到奥地利这样的国度里，仿佛每时每刻都置身于美丽的画卷之中，可以远离烦恼，去体味生活的美好，也可以远离喧嚣，去感受自然的瑰丽。或许，无论你是人文爱好者，亦或是自然风光的追求者，这里都值得你为之驻足。
“无意间，在500px上看到了某位摄影大师发的一幅精彩的摄影作品，状着胆子留言询问了一下拍摄的地点，意想不到的竟然得到了十分详细的地址回复，同时在计划最后欧洲环游之旅时发现我到达意大利最后一站米兰后有一张只需9€的廉价机票飞往布拉格，于是便有了接下来的这段捷克、奥地利之旅。”
—— 来到布拉格的因缘巧合
每一次旅行，我都会事先做好尽可能相近的准备，今年以来，开始认真地整理15年来的每一段旅程游记，除去最后一周在瑞士全是阴雨天没能尽情享受旅程以外，这段捷克&奥地利的旅行将会是我至今周游欧洲诸国的最后一篇游记。这次的旅行来的比较突然，行程中我只是确定了几个想去的城镇，然后具体的景点并没有做太多的攻略，可以说是走到哪算哪，所以你们或许会发现本篇游记中介绍的景点数量并不是很多，有时候还多是一些大街小巷上平凡的风景，但还是希望能够通过自己用心的描写，给大家展现当地最美丽的风情。每一次游记，我也都尽量避免记流水账的形式，尽可能多一点地描述景点的风景、历史、文化和体验。为了不影响叙述，游记中会将每个地方的旅行信息放在章节的最后，也方便大家做攻略参考。

想到啥是啥的tips

1. 当地治安
捷克和奥地利的治安都算是比较好的，布拉格老城的夜晚很美，但到了晚上街上行人较少，尽量不要单独出行；维也纳则治安同样很好，当地的居民还算是热情，对于一个在巴黎生活的人来说这里简直是安全得不要不要的，当然无论在哪，看管好自己随身贵重物品都是必须的！至于东欧的这些小镇就更加不用当心了，可以说真的是夜不闭户、路不拾遗。
2. 天气情况
中东欧这一片区的天气四季比较分明，夏季是整个欧洲大陆永远的旅游旺季，而春秋季节则气候更加宜人，适合走进大自然，去森林里徒步、登山等。另外奥地利因为有很多山地，冬天也是滑雪爱好者的热门目的地。
3. 捷克、奥地利的交通——大巴篇
捷克境内的交通主要都是大巴，在捷克坐大巴可以不用提前预定，因为一般都会有空座，所以去周边小镇游玩都是直接前往巴士站现场买票的。（具体交通会在每个城市最后的实用信息介绍）
捷克克鲁姆洛夫到维也纳是有类似私人巴士的，网上没有查到具体的线路和在线预定。我们是到了CK后直接在酒店前台预定的，车子会在预定好的时间在酒店门口等待，除去接人等待的时间，车程约3-4小时，票价15欧上下。
4. 奥地利火车票
奥地利火车和德国类似，条件很好，也比较准时。前往哈尔施塔特的时候是在奥铁网站上购买的火车票，期间只有15min换车时间，居然时间刚刚好，完全不会赶。
5. 当地美食
捷克的饮食口味浓重，比较大众的东欧口味，和德国的饮食有点接近，有代表性的有烤猪肉、烤鸭、甜酸菜、饺子（当地传统主食，不好吃，有点像馒头）。还有水果布丁，就着点酸奶干酪和融化的黄油一起吃。比较喜欢他们餐前的面包大蒜汤（Goulash Soup），味道很独特，可能是我喜欢吃大蒜的缘故jQuery17108917004303075373_1490419152070。 捷克街头常见的一种叫做Trdlo的传统甜点，值得品尝。
奥地利的美食也如同捷克、德国一般，其中维也纳式煎猪排（Winner Schnitzel）或者鸡排是不可错过的，并且烤肘子、德式香肠等也都是值得品尝。
东欧的啤酒可以说是不得不品尝的，味道香浓醇厚而且便宜实惠，来到捷克奥地利旅行，就跟在德国时一样，几乎是每天每顿一大杯啤酒。捷克啤酒，特别是黑啤，十分有名。
6. 住宿预订
这次住宿的地方主要分成华人旅馆以及在网上预定的Hotel。随着随身携带的摄影器材越来越多、越来越贵，为了安全起见，不得不放弃以前每次出行都钟爱的多人床位青年旅馆，改选民宿和小旅馆。事实上，除了一些比较热门的旅游景点或度假景点以外，民宿的价格并不算非常高，但相对青旅更加温馨安静一些。这次全程除了在哈尔施塔特最后一晚换成了单人间比较贵以外，其它地方的住宿大约平均每晚35欧元。
7. 有用的app
a. 因为一个人出行比较多，虽然认路能力很强，但是为了万无一失，我都会把所有的景点标记在离线地图上
b. 墨迹天气，现在一般查天气情况都用这个了，苹果自带的略渣，主要是看24小时的天气情况，还有日出日落时间
c. 对于摄影师来说还推荐一个PlanIt！，这个软件可以查到某地、某日的银河情况、云图、光污染等
d. 美景听听Pro：免费的中文景点语音解说软件，出境游非常有用，大部分景点都有语音介绍，既能听到中文版的当地景点解说，还能省下不少租用讲解器的钱哦。
8. 最佳旅行季节
捷克、奥地利的最适季节都是5-9月，暑假期间7、8月是全欧洲休假最疯狂的时节，到时会是极其恐怖的旅游旺季，住宿、交通、活动甚至是餐厅都需要提前预定好。另外冬季也有一番别样的风情，雪后的哈尔施塔特绝对是人间仙境，将来一定要再冬日里再来一次。
9. 尽量不要带太大的行李箱，小城市里面的道路都是石子路，搬运行走起来会比较不方便。
10. 这里没有强制要小费之说，一般高档点的餐厅小费在结账时已经包含在里面了，如果不包含的话就是不强制了，欧洲给小费的习惯相对美国来说低很多，一般凑个整就行了。
11. 关于签证和保险：好吧，拿着法国1年长居签证的人，从来没有考虑过签证问题……旅游保险嘛，如果是从国内过来玩的，办理签证的时候都要买的，还是买一下好，我现在出国游玩即使签证不要求我也会买一份。我一般会买安联保险，这次欧洲行买的是他们家的申根之王保险。

生命中不能承受之轻

在开篇之前，有必要先吐槽一下在米兰机场受到的不公正待遇。因为买的是瑞安航空的廉价机票，所以是不含任何行李的，所以在买票的时候特意买了一个登机箱的行李额。然而！重点来了，在过登机口的时候，被一个意大利人拦下来了，硬是说我没有购买行李额，我当时就拿出了收到的邮件确认单给他看，然后这货就脑残了，想了想又改口说我背的书包太大，还要再买一份行李！我看着身边背着大包小包的外国人都直接通过了，明显地故意欺负中国人，绝不能轻易妥协！正当我准备去找机场管理人员投诉他们歧视中国人的时候，有一个很友好的英国女士悄悄地走过来说她没有任何行李，可以偷偷地帮我把书包拿进去，经过一番沟通之后，我将装满摄影器材的书包交给了这位女士，那个意大利工作人员看都没看就通过了。于是我再推着箱子走过登机口，这货还专门把我拦下问我的行李呢，我说塞进登机箱里了，才放我通过……虽然旅程已经结束很久了，但至今还能记得他们当时丑陋的嘴脸。
独自一人出行在外，有时候总会遇上这样那样的突发情况，把这段写出来，也是希望大家能了解即便在欧洲，种族歧视或者仗势欺人的现象也是经常会有的，几年的旅途下来，有遇上过小偷、遇上过合伙行骗、遇上过火车上满车厢的偷渡难民、更遇上过被围堵在厕所的紧急情况。希望每个在外行走的旅人们，能够随机应变，保护好自己，然后愉快的享受每一段旅程。

布拉格，神秘的浪漫

== 序章 ==
尼采说：“当我想以一个词来表达神秘时，我只想到了布拉格。”如果说，矗立于山丘之上傲视全城的城堡以及散落于承重的高塔代表了她的恢弘与神秘；那么，蜿蜒于老城的小巷以及伏尔塔瓦河上的游船则彰显了她的浪漫与柔情。
这是一座让我流连忘返的城市，即便这已是第二次来到布拉格，心中依然期望着第三次、第四次、第N次的到来。再次来到这座城市，已然是夜幕降临，在机场与提前一步到达的小伙伴们顺利会和后，在夜色中乘坐着机场公交，慢慢地驶入这神秘的中心。
昆德拉的一本《生命中不能承受之轻》让他的祖国捷克走向了世人，也使之成为了文学的家园；而蔡依林的一首《布拉格广场》给这座城市遮上了一道神秘的面纱。如今，布拉格的的童话依然在继续，夕阳古堡的光晕加上高雅的音乐又赋予这城市一道神秘的浪漫。
== 老城区 ==
来到布拉格的第一个早晨，天空覆上了一层阴翳的乌云，可能是第二次到来的缘故，并没有什么特别想去的旅游景点，只是随意地走在老城街道之上。红黄相间的电车很有节奏的行驶于古老的街头小巷之中，两旁是一排排风格迥异而又富有色彩的古老建筑。

对于一座历史文化名城而言，可以这样淋漓尽致地在最为亲近游客的大街小巷上展示她丰厚的风格底蕴，无疑是值得骄傲的。而在对于这些文化保护的同时，捷克也极力地发展着他们的的文学和艺术修养，一些带有现代元素的雕刻装饰同样在老城街头随处可见。
随着旅游业的逐步发展，如今的老城街道上也开始趋于商业化模式，各式各样的手工艺品、纪念品小店如雨后春笋般林立，闪耀着夺目光芒的波西米亚水晶店铺、挂满可爱风趣的木偶商店，散发着各种芬芳的手工皂小店，为每一位游客提供了多样化的闲逛选择。

== 这是一个绝佳的摄影机位 ==
一大早出发的第一个景点，在老城中闲逛一番之后，终于找到了这个期待已久的地方：黑暗圣母之物（House of the Black Madonna）。或许这里并非是一个大众旅游景点，可能在当地亦同样不为人所知，然而在摄影界里却是非常的著名，在网上也经常流传着关于该楼梯的摄影作品，可以说是几何曲线与摄影结合的巅峰之作了。当我怀揣着激动的心情找到原来记下地址的所在地后，不禁一愣，怎么只是一座略带古典艺术范儿的老房子呢？而入口处更是已然挂上了咖啡馆的标志。站在门口迎接客人的咖啡馆服务小哥或许是看到我背着相机三脚架等器材吧，竟然在我犹豫之际主动上前攀谈，并告知我所寻找的楼梯正是在这建筑内部。感受着服务员的热情友好，步入一楼狭小的正门，一下子就看到了这一设计精妙的场景。原来，楼梯的设计竟能如此巧妙，楼顶更是直通天台露出了天空的光芒。是否，这就像是一盏明灯，漫步前行、向上攀登，终能走向光明的彼端。

== 老城广场 ==
漫步来到老城广场，这里没有许愿池的喷泉、也没有无人的走廊、更没有随处可见的彩绘玻璃窗，取而代之的是众多的餐厅、咖啡馆以及小商店。来到老城广场，最先映入眼帘的便是位于广场中央的市政厅塔楼上的天文钟。
据传，该钟楼建立于1410年，作为一座拥有精美设计的自鸣钟，至今仍能够精准的报时。天文钟初建时仅有其机械部件以及天文表盘，后来在19世纪加上了其它这些精美的设计。传说该天文钟的设计者，一名查理大学的数学、天文学教授，在完成这巨作之后被当权者灼瞎了双眼，为了避免他再次设计出这样的杰作。作文中世纪时期的天文研究设备，天文钟可以显示地球和太阳、月亮的相对位置。现在每到整点时分，天文钟上那两扇蓝色小门会打开，跳出两个表演的小木偶，因此，整点时也是天文钟广场上人群最为密集的时刻。

== 布拉格城堡 ==
在捷克，城堡已然是布拉格的主角，但同样也是它的背景，没有城堡作为背景的照片，画面似乎少了一些主题。位于伏尔塔瓦河边丘陵上的布拉格城堡，是如今世界上最大的城堡之一，也是所有来到布拉格的游客必须游览的景点。


布拉格城堡总共包含了约9个景点，主要是：圣维塔大教堂、旧皇宫、圣乔治大教堂、黄金巷等。城堡通票有A\B\C三种，包含的景点数量不同，可根据兴趣购买门票。全部景点仔都要参观的话需要一整天时间，对于历史和建筑有浓厚兴趣且又有较多时间的朋友，建议购买A套票，在两日内有效。


== 序 ==
坐落于布拉格东南不到100公里的库特纳霍拉（Kutná Hora），只需要1个小时左右的车程，虽然曾经是波西米亚一座最富的城市之一，然而在15世纪20年代的胡斯战争期间的大量屠杀之后，小城开始没落。
当大巴驶入小镇之后，仿佛感觉自己来到了一座被遗忘的城市当中，或许，如若不是身旁不时走过的三三两两挂着相机的游客，都不敢相信这里就是捷克最著名的旅游小镇之一。不知道是因为街道空旷的缘由，还是小镇本身就带着点令人毛骨悚然的阴冷，看着路旁有点破败的公交站牌，小道上门户紧闭着的店面，时不时传来的一阵乌鸦低鸣，总是不自主的心底发冷。
然而，正是这样一座似乎已经无人问津的空城，却常年蝉联了捷克旅游热门城市排行的前三名，因为无论是谁，当你做好了心理准备，都会想来看看那这传说中的人骨教堂。
== 人骨教堂 ==
( The Sedlec Ossuary )
对于很多人来说，总是先知道人骨教堂之名，才了解库特纳霍拉这座小镇。在这近乎被人遗忘的小镇上，这座十分狭小的哥特式教堂却成了最有人气的地方。教堂内部深藏的数万具骸骨也使得整个周边弥漫着一种庄严和阴森。

因为14世纪爆发的黑死病，以及15世纪的胡斯战争，使得整个波西米亚生灵涂炭。教堂最初的建造并非是为了摆放这些骸骨。为了安抚这无数的亡灵，当地虔诚的传教士们将它们安葬在了这里。
从外面观看教堂，似乎是那么普通而又不起眼，平平淡淡的外观建设，并没法让你在第一时间发觉这就是著名的人骨教堂。然而！当跨入那狭小的正门时，一阵阴风袭来，映入眼帘的便是如下的画面……


教堂内部的装饰，全部都是用人骨做成的，包括头骨堆砌而成的灯架、天花板的吊灯、诸多人骨堆成的祭坛、墙壁上装饰的花纹，一切的一切，全是由人骨制作而成的！




教堂中央是一个巨大的人骨做的吊灯，以成年男性的腿骨组成主架，再以下颚骨串成挂帘，整个吊灯的外形独特，诡异中又带着震慑人心的华丽。


虽然整个教堂中遍布着令人恐惧的骸骨，但是其艺术水平却也值得震撼，每一道装饰都通过当年RINT独具匠心的设计，加上几何线条的组合，叠加成一个个独特的艺术品。用摄影圈的一个词形容就是：细节感人……



慢慢地观赏教堂内部的装饰摆设，渐渐地去读懂这片土地的历史，此刻的心情，已然由刚步入教堂的惊讶变得平静，即便被万千骸骨所包围，心中却始终保留着一片圣洁。

== 圣维塔教堂 ==
老城区的地标，此地区中最古老、最奢华的教堂，始建于14世纪初，是一座兼具哥特式和巴洛克式风格的教堂。早在14世纪初罗森伯格家族的彼得一世曾在此兴建一座小教堂，该家族曾统治了克鲁姆洛夫长达三百年之久，于1340年在德国建筑师Linhart的指导下重建了教堂，现在我们看到的是15世纪Jan Staněk主持下又一次大规模重建的晚期哥特式教堂。

维也纳，跃动音符的殿堂

== 序章 ==
气派的皇宫、巴洛克风格的宫廷装饰、遍布全城的歌剧院以及宏大的广场共同营造了维也纳这座充满活力的奥地利之都。这里，被誉为“世界音乐之都”，因为其街头巷里之间无处不充斥着浓厚的古典音乐艺术氛围，在这里总能邂逅到几分优雅的生活情调。同时，维也纳也是一个多民族融合的国际大都市，然而却没有魔都这样超快的生活节奏，人们喜欢总是喜欢在午后的阳光里，静坐在街边的咖啡馆，品上一杯小小的咖啡，虚度一下这段最美妙的时光，放慢各自的生活节奏。
维也纳是一座拥有了1800多年历史的城市，迄今为止，维也纳已然谱写了极其辉煌的历史：在13世纪末成为哈布斯堡王朝的首都，此后维也纳又成为神圣罗马帝国的首都；15世纪以后成为罗马帝国的首都和欧洲的经济中心；19世纪期间，随着奥匈帝国的强盛，维也纳发展成当时欧洲一个最为重要的都会，并于1806年成为奥地利帝国的首都；直至1867年，成为当时强盛无比的奥匈帝国的首都；第二次世界大战后，奥地利与纳粹德国合并，维也纳也于战后遭到了四国联军的划分管理；直至1955年奥地利再度获得独立，维也纳也再度成为了一国之都。

伊丽莎白·阿马利亚·欧根妮（德语：Elisabeth Amalie Eugenie，德语系称之为“奥匈帝国的伊丽莎白（Elisabeth von ?sterreich-Ungarn）”，1837年12月24日－1898年9月10日），通常被家人与朋友昵称为茜茜（Sisi），是巴伐利亚女公爵 与公主，后来嫁给弗朗茨·约瑟夫一世成为奥地利皇后兼匈牙利王后，她的美貌和魅力征服了整个欧洲，被世人称为“世界上最美丽的皇后”。茜茜这个名字经常在小说、电影和动画系列里写为Sissi（法语）。虽然伊丽莎白在奥匈帝国的政治影响上有限，但她已经俨然成为一个文化偶像。伊丽莎白如同其在电影和戏剧制作一样，被认为是一位在自由精神和传统宫廷规矩徘徊的悲剧人物。
1837年12月24日伊丽莎白出生于德国慕尼黑。1853年伊丽莎白随她母亲与18岁的姐姐海伦赴上奥地利的度假村巴德伊舍，原定计划是海伦应当在那里 引起其23岁的表哥，奥地利皇帝弗朗茨·约瑟夫一世的注意。但出乎意外的是弗兰茨·约瑟夫一世竟然爱上了当时15岁的伊丽莎白。两人于1年后的1854年 4月24日在维也纳的奥古斯丁教堂结婚。
16岁的伊丽莎白从一开始就很难接受哈布斯王朝宫廷内所使用的严格的宫廷规矩，因为她本人喜欢骑马、读书和艺术，但这些又是维也纳宫廷无法理解的。因此她 在皇宫里非常孤立。她婚后她生下两个女儿、一个儿子：索非（1855年—1857年）、吉赛拉（1856年—1932年）和太子鲁道夫（1858年— 1889年）。但子女们的抚养权却被婆婆苏菲剥夺，期间她与弗兰茨·约瑟夫之间的关系也开始恶化。1857年悲剧发生，伊丽莎白不顾医生的反对意见，带她 的两位女儿离开了奥地利前往匈牙利度假。两位公主患上腹泻，尽管吉赛拉恢复得很快，但她2岁的姊姊索菲却因此而死亡。长女的死亡导致伊丽莎白她的生活和休 息造成影响，
1889年，，伊丽莎白的儿子30岁的奥地利太子鲁道夫与女友的遗体一起，在他于下奥地利行宫里被发现，经过调查认定是自杀，而并非谋杀。伊丽莎白从此陷 入忧郁症，再也没有恢复过来。伊丽莎白毕生始终在别人身上找错，因此这次她也将她唯一的儿子的死归罪于他人。鲁道夫的死亡，导致伊丽莎白产生其著名的个人 标志。从此以后她只穿黑色衣服，打着一把皮制阳伞，用一把棕色扇子遮住面孔。只有极少数其他服装照片是由摄影师趁她不知道时偷拍出的。
1898年9月10日，60岁的伊丽莎白在瑞士日内瓦和Sztaray伯爵夫人沿着日内瓦湖边的勃朗峰滨湖路步行，准备登上日内瓦号轮船前往蒙特勒时，被 意大利年轻的无政府意义者卢旺伊季·卢切尼用一把磨尖的锉刀刺伤心脏。她遭到袭击后，因为不知道她的受伤程度，还继续登船。直到除去紧身衣，周围人才意识 到问题的严重。伊丽莎白终因流血过多身亡，时年60岁。她留给世人的最后一句话是：“出了什么事？”
据报道，卢切尼本来想暗杀奥尔良公爵作为行动宣传，但奥尔良公爵临时改变了行程。卢切尼事后说：“我只想要杀死一名皇室成员，不管是什么人”。虽然伊丽莎白当时是匿名在日内瓦逗留，但报纸上还是报道了她的行踪，因此卢切尼决定刺杀伊丽莎白。

== 黑死病纪念柱 ==
这是位于维也纳内城壕沟大道(Graben)中央的一座巴洛克式纪念柱，为了纪念17世纪及一直以来死于黑死病的患者，奥地利皇帝利奥波特一世下令建造了这尊纪念柱。
黑死病，可以说是人类历史上最恐怖的瘟疫之一。最早起源于亚洲西南部，于1347年传入西西里群岛，此后开始横扫整个欧洲，因为患者的皮肤会出现许多黑斑，整个过程非常痛苦，因而被称为“黑死病”，在当时的医疗水平下，没有任何治愈的可能，几乎是一种绝症。而在之后的300年里多次席卷欧洲，1679年维也纳的一场黑死病大爆发，吞噬了近三分之二维也纳人的生命。

这座纪念柱上布满了许多繁琐的雕像，描绘了神创天地、黑死病流行、最后的晚餐等故事，下令兴建柱子的国王利奥波德一世也被雕在上面，表现出皇帝半跪在地上祈求上帝保佑自己的臣民的画面，而柱子基座上的碑文则在提示世人，勿忘上帝给这座城市带来的惩罚。

无论你是否极力想从某种压力中脱身而出，我笃信有一个地方定能让你得到全身心的愉悦和放松，那就是哈尔施塔特。如果你想真正的远离喧嚣，远离烦恼，感受大自然的美好，体味生活的美好，这里千姿百态的风光，如梦幻般的自然风景，能为每个来此度假的游客一次独一无二的体验。

走完湖滨小径之后，再度往另一个方向暴走，前往拍摄明信片角度的位置。途中会经过如下图的地方，往上走就能到达位于半山腰的天主教堂。虽然是镇上最重要的教堂，但是相较来说教堂边上的人骨屋却更为有名，里面整齐地堆放着一排排骷髅头，上面绘有装饰图案以及死者的姓名，不仅庄严肃穆，还带有一点小城人民共同的温情，也是是对生命的尊重与诚挚。原来因为小镇墓地有限，有了一个先人为后人让地的规矩，即将先人的头骨留放在教堂的一隅，供人缅怀，因此便有了如今的人骨室。同行小姑娘害怕再次看到一大片的骷髅，最终我们没有前去参观，下图的头骨只是路边小摊兜售的一种仿制的工艺品。

== 达赫施泰因大冰洞 ==
(Dachstein Rieseneish?hle)
来到哈尔市塔特的第二日，早早地起床，乘坐位于酒店附近的公交车前往上特劳恩。许多人或许只听说过哈尔市塔特，却往往忽略了其周边同样有许多美丽的景点，而在上特劳恩小镇后的达赫施泰因山和克里彭施泰因山便是当地著名的户外自然旅游景区。全程共分为两段缆车，联通两座山脉，下图为官网上这一片景区的全景地图。

既然来到此处，我们决定用一整天的时间来慢慢游玩，所以买了全景点的逃票，包括了所有的景观和缆车，而此行的第一站就是位于山腰处的千年古冰洞。在山脚下就需要将门票都买好，根据自己的时间以及每个景点进入的团队的时间选择，然后坐缆车到第一站下来，需要步行20分钟上山。这里的几个洞穴景点都需要导游带入的，所以要在入口等待开团时间，趁此机会同行的小情侣站在山崖边上欣赏山脚下的风景，持续给我发狗粮……

刚进入洞穴便被内部的五光十色的岩壁和灯光效果所震撼。



走了一小段，始终在岩洞当中，游客们纷纷在问：冰呢？冰呢？。不得不说，奥地利整体对于自然景点的开发和保护都做的很好，因为是在冰洞靠近入口的地方，还没有地下冰层出现，但是这里配备了各种灯光和音乐，导游每到一个点会停下来，打开灯光，配合着背景音乐给我们介绍历史故事，而离开时又把灯光关掉，确保洞穴始终保持原始的环境。这里石头上的光表示着在这一片区域曾经发现的熊的骨头化石，表示这里曾经是熊冬眠的地方。



走着走着，洞穴内的温度逐渐下降，终于在一阵冰凉的寒意中看到了冰层了。据说，这些巨大的冰块都是已经存在了成百上千年的历史，而且如今，每年夏天，当地人都会在这冰洞里面举行一次音乐会，我们去的时候正好看见他们在拆之前搭建的舞台。



== 猛犸洞 ==
整个冰洞的参观时间大约需要一个小时，从冰洞出来后可以走到下一个景点：猛犸洞。虽然洞穴被称为猛犸洞，但似乎和猛犸没有什么太大的关系，原来还以为这里能看到猛犸的化石啥的，结果发现只是一个存在无数年悠久历史的史前洞穴，在里面可以参观一下洞穴中的地质地貌，导游还会介绍一些开发洞穴和洞穴探险的故事。

== 克里彭施泰因山 ==
(Krippenstein)
刚刚过了午后，我们的午饭就是随身携带的面包和火腿，一路走回到缆车搭乘处，在正午的阳光下来到了另一座山上，这里有一个著名的观景平台：五指山( five fingers )。这里到了冬天，会变成一片雪白的世界，而夏天则是著名的徒步胜地，大约1个小时的步行路程算起来是十分轻松的徒步线路了，沿途可以欣赏身旁的高山地貌，还有一种一览众山小的感受，而此刻，正午的阳光是如此的明媚。




行走在山顶上，可以随时看到远处流淌在山峦之间的湖泊以及散落在湖泊边上静谧的小镇，此后的五指山观景台则是此处最佳的观景平台，可以一览整个湖区的美丽风景。


在欧洲无处不在的便是教堂，即便是位于原先荒无人烟的高山之上，依然有一座简单的小教堂矗立着，只是不知道如今是否还有修道士居住其中？


一路上的徒步路道设计得十分人性化，途中有许多类似这样的波浪形躺椅，而且最感人的是，这里的躺椅每天似乎都有员工打扫，并不会因为位于室外而积灰，游客随时可以在这里休息。正好看到一对母女躺在椅子上休息，举起相机的时候，小女孩好奇地看着我，于是拍下了这一幕。


说起五指山，可能大家都会联想到西游记里如来佛主镇压孙悟空的五指山，然而，这里的五指山却完全没有所谓的五个山头，而是因为在山巅之上，建造成“五指”形的观景平台，从悬崖边上延伸而出，像一个悬挂在悬崖上的手掌，平台的下方则是用钢铁制造的框架，游人们可以站在指间，看着脚下千米高的惊人风景，也能够轻松地一览这里的湖光山色。




来到哈尔市塔特的第三天，闲来无事，原本打算租辆自行车在这片湖区环湖骑行，但是可能睡得太晚，到湖边租车点时都已经没有好一点的自行车了。干脆在码头乘坐当地的小渔船游湖，从哈尔市塔特对岸到小镇上大约半小时至40分钟左右，一人10欧元好像，船夫会一边划船一边跟大家聊天，介绍这里的风景和当地人的生活，一船也就10人左右，几个英国来的女生在船上不停地自拍，其乐融融……




== 旅途中总是有许多不期而遇 ==
爱上旅行的一个原因，便是对于未来无法预知的惊喜，即使我经常在出行前做好了充足的功课，标记了所有想去的景点，甚至疯狂点还会安排好每个景点的时间，景点之间交通转换的方式，但旅途中也总是有许多未知。
已然是我环游欧洲的第7周，特地在这个度假胜地安排了3天时间，于是最后一天就显得有点无所事事了。再次来到明信片角度拍照，但由于阳光过于强烈，无法得到很好的作品，而拍摄的过程中，有一位很像中国人的小伙子，背着沉重的摄影器材，来到我身边想要寻找拍摄的机位。可能同样都是摄影爱好者，很快我们就开始聊起了怎么拍摄好看（居然是用英语开始聊天的），直到我说了一句我来自中国，才发现对方是一名美籍华裔，我去……我们居然用英语聊了半天……
因为天气的关系，我们决定等傍晚时分再回到此处拍摄，考虑到Keyth租车自驾来的，于是我们一起看了看地图，决定午饭后开车去距离20-30km处的山中湖泊拍照，记得前一天晚上听一个老外摄影师提到这个地方，于是我们就这样在阴天里驱车前往。沿途路过一个小农庄，当地蓄养的牛群在草地上吃草，显得十分祥和。


终于抵达了这神山中的湖泊，可惜天公不作美，下午突然变得阴云密布外加狂风大作，所以只能随手拍几张到此一游照了，全部调成了阴沉的色调……




THE END
人只有不断行走，用心去看这个世界，去拥抱大自然，才能认识到自己的渺小，感受到世界的奇妙。记得曾在深圳青旅遇上过一个意大利男生，毕业后工作了一年，辞职，一路骑自行车历史一年多从意大利来到了深圳。必须承认，很佩服他的勇气，然而，又有多少年轻人能真的拥有这样的勇气呢？
始终有个梦想，想要环游世界，记录下世界上的美丽景色，记录下生命中的每个美好瞬间，记录下各地不同的文化风情。或许，这个梦想看似遥远，但其实又近在眼前。细数过来，留学三年已然走过许多地方，看过许多风景，经历过许多故事。在这些故事淡出记忆之前，赶紧写下每次出行的点滴，将来有一天，或许还能够清晰地回忆起曾经走过的路。
