山海彼岸，漫游台湾，台北花莲九份七日行游记攻略

游记 人文游 情侣游 海岛 2019/01/24

说说这次旅行

台湾之行回来已经一个月了，仍然念念不忘的行程，终于提笔开始写下这篇游记。一直就想去台湾旅行，温暖的气候，亲切的语言，同根同源，像另一个平行世界一样吸引这我，终于在2018年底成行，圣诞前夕，锵锵四人行。

行程安排

上海-台北-花莲-九份-台北-上海

 行前准备

证件：去台湾最好提前一两个月就准备好证件，需要两个必备证件：1、台湾通行证及签注；2、入台证。
台湾通行证及签注的办理方法和护照、港澳通行证类似，自行在当地机关办理。在台湾通行证及签注办理下来后，可以在某宝上找专门办入台证的旅行社来办理入台证了。
入台证长这样。
 
机票：都办理好以后就可以购买机票啦。注意！机票使用台湾通行证账号购买，不要使用护照，如果用护照购买，需要到机场柜台更改成通行证账号，耽误一些时间。
（朋友2017年去台湾的时候是使用护照购买的机票可以值机，但我们在2018年底去的时候工作人员就要求改成台湾通行证账号才能值机了；另外在台湾旅行过程中也完全可以用台湾通行证办理酒店入住、购票、退税等，不需要使用护照）
 

DAY1 上海至台北-看电影-宁夏夜市

上海直飞到台北，因为起了个大早，当天就没安排更多行程，下午入住民宿休息，睡醒了去电影院看电影，晚上散步到宁夏夜市大吃特吃！
桃园机场下飞机取了行李，就看到现金兑换的窗口啦，但是汇率不低，最终决定在自动取款机用内地银行卡取款，手续费同行四人均摊，也不会太贵。
 
继续往捷运（地铁站）方向行走，会看到“台湾大哥大“（这个名字笑死我了！）运营商，当地短期旅游电话卡售卖，只收现金，办一张七天左右的旅游电话卡，可打电话，不限流量，完全够用了。
 
自动贩卖机可以用支付宝和微信付款！骄傲！
 
到了捷运站会有人工柜台售卖交通卡，因为要在台湾玩七天，就办了一张，之后从瑞芳坐台铁回台北时也可以直接刷卡进站。圆形卡是台北市车站到桃园机场的直达票，我们往返都在桃园机场，就买了两张圆形卡和一张交通卡的套餐。
 
特意感受下台湾气氛，这边有一些内地没有上映的电影，而且电影分级制度比较严格，按照年龄会有对应限制观看级别，真的很好，既不会限制电影业发展，也可以照顾到不同年龄段需求。《水行侠》就是《海神》，哈哈哈。
宁夏夜市逛吃逛吃！每一样都好吃！最后再去吃碗牛肉面，在台湾，街边随意进一家牛肉面都有很多渊源和得奖记录，牛嫩、筋软，又不散，面条筋道，大爱！
 

DAY2 台北故宫博物院-淡江高级中学-渔人码头-士林夜市

上午到台北故宫博物院参观，有些非常珍贵的藏馆是不允许拍照的，进门前看清楚，现场也会有工作人员阻止。藏品肯定没有北京故宫的多，但毕竟是内地没有的展品，还是很值得逛一逛的。售票处可支付宝也可刷卡付款，旁边有电子讲解器租赁处，建议租一个，否则自己乱逛，对藏品没有太多研究的话，并不能领略其中意境。
台北故宫博物院的周边文创做得很好，光是逛文创产品就可以逛两个小时。
 
从台北故宫博物院出来以后，四个人拼一辆车到淡江。来到淡水第一站——周杰伦的母校，淡江高级中学，周杰伦的电影《不能说的秘密》中的淡江艺术学院就是在此取景。不过非常不凑巧，去时是工作日，并未开放参观。如果对这个景点很在意的话一定要注意参观开放时间：周六周日上午8点至下午四点。
 
淡江高级中学附近也有很多好看的地方可以走走看看，步行百米就是海边。红毛城和真理大学也都在附近，可以一逛，景色很美。我要是在这念大学，怕是无心读书了，走在路上都觉得浪漫，让人直想谈恋爱。
 
红毛城的文创店有“春一枝”冰棒卖，这是合作文创项目，吃掉冰棒，里面的杆子是一根景点图案的书签。在红毛城买的冰棒里面是红毛城样式书签，在诚品书店买的冰棒里就是诚品书店样式的书签。
提前在天气预报类APP上确认当天日落时间，提前坐公交前往渔人码头，等待日落。虽然看过很多处的日落，但渔人码头的日落真的很美，船只和飞机划过水面和天空，太阳一点点西沉。风超大，一定要带个抗风的外套。
 
坐公交转捷运到达士林夜市，开启今晚的晚饭+夜宵。
 

DAY3 中正纪念堂-诚品书店-台北101大厦

上午坐车去中正纪念堂参观，路过自由广场，飞起的鸽子很美。
“生命的意义在创造宇宙继起之生命，生活的目的在增进人类全体之生活。”
中正纪念堂整点有卫兵交接仪式，有兴趣可以多逗留一会观看。
纪念堂楼下也有史料、展厅参观，还有些有趣的小电影。
 
纪念堂这里正好有家春水堂，据说是珍珠奶茶创始老店！作为一名奶茶爱好者，我是万万不能错过的！没有什么是一杯奶茶解决不了的，如果有，那就来两杯。
 
诚品书店，有趣的音乐和画册。预留了整个下午时间逛诚品，买到了喜欢的漫画图册，听到了好听的音乐。来台北一定不要错过诚品书店。
 
附近有一辆几米的月亮公车路边景点，如果你也喜欢几米的漫画，可以去找找，一个彩蛋。
 
曾经的“最高大厦”，可以俯瞰台北市夜景，我们在网上订了票，现场根据单号在自助机上可以取票。
 

DAY4 台北至花莲-海滨公园-自强夜市

台北前往花莲，火车路过东海岸。
你好呀，太平洋！
 
花莲位于山海之间，我很喜欢。这里还有我最爱的炸弹葱油饼和凤梨湾凤梨酥。
 
因为已经逛了台北的两个夜市，在花莲夜市就没有特别惊喜。趁着日落前跑到海岸线看日落，像是在画里。
 

DAY5 太鲁阁-七星潭

第五天行程因为都是离花莲市区较远的景区，公共交通不太方便，就选择了包车，遇到喜欢的景色随时可以停下观看，而且时间也自由。
 
砂卡当步道一段景色绝美，一片碧色。走到尽头会有摊位卖小吃，香肠和米酒必尝，不是普通景点糊弄人的面粉烤肠。
 
太鲁阁国家公园的特色为峡谷和断崖。太鲁阁也成为台铁之列车名。清水断崖是我主要推荐景点。可惜下起了下雨，否则一定更美。山间铁路也仿佛日本漫画一般清新。
 
七星潭大多都属于砾石滩，让七星潭成为花莲近郊最佳的踏浪捡石好去处。以前所见都是沙滩，第一次见到石滩，据说垒砌的石头越高就越幸运哦！在潭边观景注意，浪很大，小伙伴就被浪追上，湿了鞋！
 

DAY6 花莲出海赏鲸-花莲至瑞芳-九份

太平洋东海岸看日出&晨跑。
 
花莲港出海赏鲸，海蓝得像假的一样，船长说夏天还会更蓝。
真美好啊。要是我跟Leo没吐就更美好了。这里是踩过的坑，因为早饭太好吃，吃了五个包子，上船以后摇摇晃晃的就吐了。大家千万吸取教训。
出海赏鲸可以让酒店、民宿帮忙代订，我们是前一天从七星潭游玩回来，请司机顺便开车带我们到赏鲸公司的柜台自己订的。需要至少一人的台湾通行证证件。
 
乘坐台铁至瑞芳，瑞芳是我们的中转站。瑞芳火车站对面有一个美食街，逛吃一会儿就坐车上到山上的九份了。
(彩蛋：原来七里香是鸡屁股的意思！震惊！)
 
九份是《千与千寻》的“取景地”，据说宫崎骏老爷子当时画画的灵感就是来源于此处。九份随处可见日本游客，相比也是因为这个原因到此处游览的吧。最美的当然是夜景，仿佛飘浮在宇宙中的一处浮岛。我们穿越民居，走到九份对面的观景台。
 
九份芋圆很好吃。散步回民宿，一起重温了《千与千寻》，很有意思的一个纪念。
 

DAY7 九份至台北-西门町-台北至上海

九份下山到瑞芳火车站，可以刷公交卡直接进站坐火车回到台北站。
 
西门町逛一逛，就要返程回上海了。
 

住宿指南

这次旅行四个人出门，主要选择的是民宿，整间预定。在台北住的是京站楼上的商务公寓民宿。在花莲住在重庆市场附近的布谷鸟。九份住在山顶的“热门民宿”（没错，名字就叫热门民宿，台湾同胞太可爱了）。
 

美食指南

台北篇
夜市
去了宁夏和士林两个夜市，相比较更喜欢宁夏夜市，街道好逛。
 台湾朋友推荐的青蛙下蛋，名字太酷了，原来是珍珠奶茶……珍珠就是青蛙下的蛋……
 
牛肉面——建宏牛肉面
在台湾吃的每顿牛肉面都很好吃，这里特别推荐一家，建宏牛肉面，从西门町地铁站步行不远。
 
一兰拉面
朋友特意推荐来吃的日本有名拉面，下午两点钟竟然还排了一会儿队。很“日式”很“正宗”的拉面馆，每人一个小隔间，有需要可以按铃，服务员会打开面前的帘子服务。推荐抹茶啤酒和抹茶杏仁豆腐，喜欢吃抹茶，所以都觉得很喜欢。汤汁鲜美，全干了吧！
 
了凡油鸡饭
之前新加坡之行就推荐过啦，全球最便宜的米其林餐厅，人均只有三四十元。位于台北101地下美食广场。
 
花莲篇
炸弹葱油饼
炸弹葱油饼是花莲一绝，不管是夜市还是街边都有售卖。我推荐的这家是跟当地很多居民打听都反馈最好吃的一家，叫黄车炸弹葱油饼，在复兴街102号，对面有家“柠檬老爹”柠檬茶店。葱油饼只有摊位，没座位，可以买了到对面的柠檬茶店坐着边吃边喝，一口柠檬茶，一口葱油饼，爽！
 
公正包子铺
在花莲吃了两顿公正包子，这家店是24小时营业的，我们晨跑回来路过。包子太好吃了！我早饭一向吃的很少，都一口气吃了五个包子。强烈推荐红茶豆浆，平时吃豆制品不太喜欢豆腥味，这家红茶豆浆用红茶盖住了豆腥，只剩香醇，特别好喝！
 
凤梨酥爱好者
吃到过最好吃的凤梨酥！“凤梨湾凤梨酥“，在花莲买了很多带回去送朋友，两种口味，土凤梨酥口味酸酸甜甜我最喜欢，金钻凤梨酥不酸只甜，都很好吃。
比之前朋友从台北带回去的佳德凤梨酥要好吃的多。很多凤梨酥内馅儿并不是用真的凤梨，而是用冬瓜代替，但这款是实实在在能吃到凤梨纤维的，也不会甜腻。
 
牛奶
瑞穗的鲜奶非常好喝，路边便利店的冷藏柜都有卖，首推水果味和苹果味的！
 

穿搭指南

洋葱式穿法，每件不厚，多层重叠。因为行程里有渔人码头落日，和逛夜市项目，会有点凉，其他时候单件就够。
 

注意事项

1、入台证：入台证在入台时会盖个章，从台湾回来时还需要盖章，这张纸不要丢掉，补办需要另付一笔费用。
2、退税:在诚品书店和商场买了很多东西，可以退税，但需要使用台湾通行证和入台证的原件，记得带在身上，购物前也可以跟店员咨询标准，要购物满一定费用才能退税，各店要求不同。
3、现金：可在到达台湾后，在自动取款机上用内地银行卡直接取款，会扣除手续费，但不会比柜台兑换现金的汇率更贵。
4、便利店：台湾的便利店非常全面，可以买火车票、可以寄快递、可以有atm机取现、可以打印、可以上厕所，所以有什么问题就找路边便利店吧。
 

再见啦！台湾！

度过了温暖又美好的七日时光，到今天依然对台湾之行念念不忘，值得以来再来的地方。行程设计比较轻松，不想太劳累的休闲之旅，2019年决定再去一次，台湾环岛骑行！也想去台南线体验，想去垦丁晴朗的天气看海。总之，期待下一次的宝岛游。
