#冬日极客# 探秘《亲爱的客栈2》拍摄地，在阿尔山遇见诗和远

游记 摄影 滑雪 自助游 自然奇观 雪景 2018/11/06

【序】

最近大热的一档综艺节目《亲爱的客栈2》告别了泸沽湖，终于在阿尔山开播了。刘涛在节目里说：“我时常在想，如果再次出发，我们会去哪里？”没错，他们来到了这个梦幻般色彩的阿尔山，去拥抱这个冬天。
喜欢这个节目，不仅仅是因为带领我们去领略那些不足为外人所知的美景，更是教会了我如何去好好爱一个人。刘涛和王珂的互诉衷肠，一字字饱含深意的情书都使我潸然泪下，也许这就是爱情本来的模样。
你有多久没看过星星？你有多久没看到过蓝天？你会怎样的去生活？你会去结交一些怎样的新朋友？带着对于爱情的美好憧憬，我和朋友踏上了前往阿尔山的路，在那儿去遇见诗和远方。

【关于大交通】

我和同伴相约从天津出发，所以根据我们的始发地给大家安利一下该如何到达阿尔山。说实话，这地方的交通并不是很便利，距离各大城市都比较的远，就连空运的快递也得四五天才能到。也因为这样，有很多人都选择从呼和浩特自驾前往阿尔山。推荐的出行方式有如下几种：
1、飞机：阿尔山有一个全国最小的飞机场——伊尔施机场。其距离市中心大约10km，森林公园55km，阿尔山火车站6.5km，地理位置还算不错。现在这个小机场只与北京、呼和浩特、乌兰浩特开通了直飞的航班，四季通航。我们本来就是计划从北京坐飞机走的，因为前往阿尔山的机票真的是相当的便宜，特价的时候只要两三百就可以买到了。因为我们一行三个人，特价票剩余量不足，我们就改乘火车前往。
2、火车：火车虽然没有飞机那么快，但是却真的可以让你的世界慢下来。前往阿尔山，夕发朝至，睡一晚也就到了，非常节省时间。早上一醒来，外面已经是白茫茫的一片，一下就从秋天穿越到了冬天。阿尔山的风景是极美的，所以在火车上可以欣赏阿尔山沿途的风景。晨雾弥漫，森林静幽，好一番童话秘境。前往阿尔山最理想一趟火车便是K7565，晚上出发早上就到了，一点也不浪费路上的时间。所以所以我们这次也是先从天津高铁到了沈阳，然后从沈阳搭乘这辆列车前往阿尔山。
3、自驾：从各地到阿尔山的路各不相同，所以大家可以自行查找地图进行查看。不过在对于车辆的选择上面，尽量选择越野车或者SUV。因为阿尔山的路况并不是特别好，这些车型地盘较低，安全系数比较高。阿尔山一片的交通标识不是很完善，一定要看清楚道口，以免走错了路。如果假期不是很多，又非常想要自驾的话，建议先前往沈阳、哈尔滨等地，然后从当地租车前往。

【关于当地交通】

阿尔山当地的公共交通不是很发达，如果从大城市过去的甚至都感觉非常的落后。出租车的起步价只要5元，而且阿尔山镇非常的小，起步价就足够绕小镇一圈的了，但是很多出租车司机并不打表，会采用拼车的方式拉客。公交车发班的间隔非常长，外地人一般都看不太懂。当然网约车什么的，在这里自然更是没有的了。
【自驾】

从阿尔山镇前往阿尔山森林公园、好森沟森林公园、奥伦布坎等都可以选择自驾前往。因为沿路的风景都是非常美的，所以看到心仪的地方停下来拍照都可以，只要不影响到车辆正常行驶就行了，能够最大程度的体验到当地的美景。但是冬天的阿尔山地区冰天雪地，一定要有驾驶经验丰富的老司机才建议采用自驾的方式。
【包车（拼车）】

这是从阿尔山镇前往阿尔山森林公园最常见的方式，出了阿尔山火车站就可以看到很多揽客的司机。价格基本上是150元两人，400元包车。当然如果你有三寸不烂之舌，也可以跟司机讲讲价，说不定还能便宜一点，据说拼出租车50元就行了，当然这个看个人的运气了。如果两个人出行的话，150元两个人的拼车出行最为划算。
【景区内观光车】（强制购买）

对，没错。只要进入阿尔山森林公园，里面的景区观光车票是必须强制购买的，无论你需要不需要。基本上景区里面的所有的点，观光车都可以到达。所以很多司机告诉你包车可以包景区内的交通，一点也没必要，拼车只要到森林公园门口就行了，没必要一直包车。

【关于门票】

阿尔山森林公园门票：（学生证半价）

旺季（6月1日至10月15日）：180元/人
淡季（10月16日至5月31日）：130元/人

景区内观光车：105元/人（强制购买，无优惠）

所有的门票都得保留好，去每一个景点和坐车的时候都是需要检票的，可别弄丢了哦。

【关于阿尔山游玩路线】

阿尔山其实并不单纯的就是一座山，这里准确地应该被叫做阿尔山地区。以阿尔山市作为中心可以分为四条游玩路线，供游客们自由选择。主要可以分为阿尔山国家森林公园、阿尔山火山温泉国家地质公园、好森沟国家森林公园等几大景区。
1、阿尔山北线：这是阿尔山旅游最为经典的一条线路，也是我们此行所要走的一条线路。其景点主要集中在阿尔山国家森林公园内。公园内的地貌是由于火山喷发熔岩拥塞和水流切割所造成的，也被称为火山熔岩地貌，也因此形成了一些非常奇特的地貌景观。比如：驼峰岭天池、三潭峡、哈拉哈河等。晚上运气好的话可以看到漫天的繁星就在你的头顶，似乎伸出手就能够摘下来。

2、阿尔山南线：这条线主要前往的是奥伦布坎景区，这里曾经是奔跑吧兄弟的拍摄地。在白狼镇10km处，白狼镇也是《亲爱的客栈2》拍摄地，这里有南兴安隧道，是上世纪抗日战争时期日本为了掠夺中国东北的林木资源所修建的铁路，也是内蒙古最长的铁路隧道。本来也挺想去这里的，但是因为时间原因没能前往，也算是留下点遗憾吧。
3、阿尔山西线：西线通往的是阿尔山口岸，出了口岸就到了中国的邻国蒙古国，与他们的松贝尔口岸相对应。中间会路过玫瑰峰，由十多座山峰所组成的山石奇观，由于其岩土呈现深红色，所以得名玫瑰峰。
4、阿尔山东线：东线的精华主要集中在好森沟国家森林公园，这里开发的并没有阿尔山森林公园成熟，还属于比较原始的状态。这里的信号非常差，所以一定要结伴而行，据说这里的秋天非常好看，但是只有越野车才能开进去。

【关于出行装备】

虽然只是11月份，在上海生活的我还穿着短袖T恤，但是阿尔山却早已入冬了。所以出行装备一定要选对，不然在下雪天里受罪的可是自己了。因为之前自己也有一些徒步探险的经历，所以就给大家来介绍一下我此行的装备，全副武装起来，不仅能够保持温暖，也可以拍出高颜值的靓照哦。这次我选择的是德国有名的户外品牌狼爪，也算是响当当的大品牌了，大家也可以根据自己的喜好选择其他的品牌。
1、冲锋衣：在寒冷地带必备的一件装备。不仅能够防风，也能够防雨、防雪。这次我带的是德国户外品牌狼爪的三合一冲锋衣，里面是一件羽绒服，可以拆卸，贴身保暖。外面的冲锋衣面料防风防水，抵御寒冷～两件组合在雪地里走非常舒适，到室内以后脱掉外衣，单穿羽绒也是时尚又保暖。
2、冲锋裤：防水，防风，透气，耐磨，保暖，这就是我对于裤子的所有要求。而且必须要轻便为主，灵活性高，不会束缚自己的运动。
3、户外鞋：一双鞋子对于这种冰天雪地的户外运动来说是非常重要的。舒适、专业的户外运动鞋，可以有效地减缓你的脚部疲劳，同时也可以防止雨雪浸入你的鞋内，导致整个脚都湿漉漉的，这样可能使你的体温骤降而出现一些危险的情况。所以在购买鞋子上一定不要吝啬，这是对于生命的保障，一定要专业专业专业！
4、太阳眼镜：冰雪天出太阳之后反射光会非常的强烈，对于眼睛伤害比较大，一副太阳眼镜就显得尤为重要了。
5、多功能面巾：面巾非常轻便，所以基本出行我都会随身携带。无论是当围脖还是擦汗或者固定遮挡太阳，多种穿法都非常实用，性价比非常高。

【内蒙古的秘境大门——阿尔山火车站】

如果说阿尔山注定是一场梦幻的旅程，那么开启这场旅程的大门便是在这个极具瑞士风情的阿尔山火车站了。到达阿尔山火车站的列车并不多，一辆绿皮火车呼啸而过，缓缓停住，我们便到了我们此行的开始——阿尔山火车站。
阿尔山火车站被称为中国最美的小火车站，可以和瑞士的火车站相媲美。在时空交错中，阿尔山火车站就站在这里岿然不动，经过了多少风霜雨雪的洗礼，才有了如今这般繁华美丽。
阿尔山火车站始建于1935年六月，到1937年竣工。这是一座典型的日式建筑，集合了日本宗教和日本民居元素为一体，军事交通为一身，拥有着十分重要的历史价值。火车站的一层外围是用花岗岩堆砌的乱插石墙，楼顶则是用赭色水泥修建的，选料质感对比鲜明，色彩处理也十分的相得益彰，不由得让人惊鸿一瞥。
据说《夜宴》曾经就是在这取景拍摄的，两军在这里发生了对峙，九个黑衣骑士策马奔腾而出，桥下是几洼蓝水，周围却是荒草丛生，那种苍茫荒凉的感觉，却又平添了几分油画的意境。
火车站唯一的一个候车室内部面积都不足十平方，售票处和检票处都仅仅只有一个，两边各有一排木制的座椅，也就差不多十几张座位，其他的旅客就只能站着等候，要是人多的时候这一个小地方可一点都挤不下，只能在外面站着等待。
如果时间充裕可以等待黄昏十分来到车站，金灿灿的阳光洒在屋檐上，整个建筑都在散发着耀眼的金色，十分好看。这里的复古气息也成了网红打卡的圣地，到了车站一定别急着走。等到游客全都离开了车站，便就成了自己一个人的拍照全景了。今天的积雪还不算厚，如果是在夜晚，再配上漫天的星空，一定就成了神秘的童话小屋。
【TIPS】

1、阿尔山火车站是必须要打卡的一个点，无论是否需要乘车，进站拍照也是需要购买随便去哪儿的车票的；

2、车站门口有各种揽客的司机，如果你住在镇上大可自己走，因为无论你住在哪儿都超不过十分钟的路程。当然也可以谈好去其他地方的价格，便可以直接上车了。我们就直接拼车前往了阿尔山森林公园。

【驼峰岭天池——上帝的遗珠】

进入阿尔山景区的第一站就直奔了驼峰岭天池，坐上景区的大巴一直到终点站下车便可。阿尔山是一处植被茂密的天然氧吧，一进来就是浓浓的新鲜的味道。这里的冬天更是神秘与浪漫的集合体，本来童话了整个秋天的阿尔山森林，摇身一变披上了一席银装，雾凇雪松漫山遍野地盛开，整个山上都似乎成为了白色的森林秘境。
阿尔山有两个天池，一个叫做阿尔山天池，一个叫做驼峰岭天池。因为这里本来是一座火山，驼峰岭天池正是火山喷发之后积水形成的杰作。睡眠海拔有1284米，湖面整体形状就像人的左脚丫，东西宽约450米，南北约800米，已经有30多万年的历史了。
观光车所能到达的只有在山脚下，想要一览天池的魅力自然还是要付出一番心血的。等待着你的便是998级木制台阶，共2.5公里左右的路程。不过这起起伏伏、上上下下的木栈道，再加上下雪天台阶上的积雪，着实给我们的登山之旅一个下马威。差不多要走一个多小时的路程，中间的路途因为两边的美景也并没有显得那么枯燥无味，阳光透过枝丫投射到我们的身上，虽然已经是零下五度的气温，依旧给我们带来了一点暖意。
山穷水复疑无路，柳暗花明又一村。相信大家都经历过艰难跋涉才到达顶峰看到美丽风景的那种喜悦之情吧。矗立在驼峰岭之巅，看着眼前这大自然的鬼斧神工，这条静养了千年的湖泊，成了最富表现力的一部分。清澈平静的水面，今天上来的人并不是很多，周围安静的好像时间就凝固在了这里。闭上眼睛，深吸一口气，再慢慢地呼出，眼前的天池就好似绝离凡尘的上帝的眼睛，如梦如幻，宁静透亮。
据当地人说，天池有很多的神奇、莫测之处。天池的水深不见底，这里有人曾经想要测量天池的深度，绳子放下去三百多米依然没有到底，有人戏说，这儿是直接通往地心的。天池里面没有一条鱼，当地人曾经往天池里放过鱼苗，这些鱼就再也没有出现过。天池的水位也是常年维持在一个水平线上，无论这里是干旱还是洪涝，这里的水位多年不升不降。
鸟瞰台是最佳的观景平台，站在这里极目远眺，整个大兴安岭的初冬之景尽收眼底。积雪还未消融，星星点点地点缀在金黄的树叶上，煞是好看。那一汪碧蓝澄澈的湖面，好似一个杯盏，盛着一湖秀美，一山新奇。置身此间，如饮一杯甘醇的美酒，不由得深深沉醉。虽然已经过了这里的旅游旺季，依然可以感受到天池的波澜壮阔又宠辱不惊的镇定，这里属于天属于地，属于自然交汇的连结之界。
如果你的力气还算充裕，还可以接着往前走来到湖边。这里可以捡到一种非常奇特的石头，它们虽然看似巨大，但是拿起来丝毫没有分量，上面有着很多密密麻麻的气泡孔，放在湖水上会浮在水面上，当地人把这样的石头叫做“浮石”，其实就是因为火山喷发而行程的火山石。捡一块火山石将对于驼峰岭天池的爱带回家，也是将整个阿尔山也一起装进了口袋。
【TIPS】

1、去看驼峰岭天池需要攀登998个台阶，大约需要一个多小时。

2、阿尔山门票一价全包，如果上山的路上有人再跟你收费，请保持警惕以防上当。

3、湖边嬉戏请注意安全，谨防落水。

【巧夺天工石塘林——来自火山的馈赠】

石塘林乍一听这个名字就觉得一定不简单，更是激发了我们前往一探究竟的兴趣。阿尔山独特的自然地貌是因为火山的运动形成的地质遗迹，是一座纯天然的火山熔岩博物馆。早在第四纪火山喷发的时候，滚烫的岩浆布满了沟壑，经过千百年来的自然风化和地壳运动，这里便形成了亚洲最大的死火山玄武岩地貌。
走在里面的木头栈道上，两边都是火山岩石所堆积出来的大小不等、凹凸不平的地貌，怪石嶙峋，姿态各异。别看这地上一块块的石头外表看着十分的丑陋，甚至一点都不讨人喜欢，但是其存在的年代可是十分的久远的。积雪将火山岩覆盖在了自己的身下，就像是守护着这块土地上的宝藏一般。因火山岩而形成的熔岩洞、熔岩陷谷等等都成了石塘林特有的标志。
里面最为神奇的是，塘内寸土无积，但是落叶松长得高大挺拔，高山柏长达数丈，白桦林也是这地方独有的特色。经过千年风化和流水冲刷，形成了石塘林独具特色的自然地貌，犹如波涛汹涌的熔岩海洋。非常好奇这些树木需要怎样的毅力，才可以在这片熔岩之地生根发芽，长成如此粗壮的树木在这里得以存活，大自然的神奇之处莫过于此。
不过冬日的石塘林可看之处并不是很多，因为这里冬日的养分非常的少，所以能够存活下来的植物都是落叶植物，冬天到来之际也都凋零自己的落叶等待一段白色涅槃重生。阳光透过枝丫倾泻到我们的身上，只感觉这冬日的暖意沁人心脾，在人不多的石塘林兜兜转转，感受大自然的鬼斧神工，也不失为一种悠闲自得的神游快活。

【冬日中流动的精灵——哈拉哈河永不冻】

哈拉哈不冻河就在我们住宿的旁边，非常近。我和朋友相约着来这儿看一场日出，共赴一次阿尔山的山盟海誓。当清晨的第一缕阳光洒向这个大地的时候，整个不冻河似乎一下就被唤醒了，潺潺的流水并没有因为冬日内蒙古的严寒所屈服，反而享受其中，凌冽的天气似乎激发了它的斗志，再河道里自由地奔腾。
我们来的这个季节处于秋冬交替之际，所以内蒙古的河流还有很多都没有冰封。听我们遇到的一位牧民朋友说，这里到了三九严冬，就连在空气中泼洒一盆热水都能瞬间结成冰，就更别说大小的河流了。到了那时候，每一条河流都会冰封，只有哈拉哈河的这一段河流却是有着某种魔力一般成为了唯一在严冬中跳舞的精灵，严寒也不能让它屈服，热气腾腾，流水淙淙，无论多久从来不会结冰
从山坡往下走，便来到了不冻河的岸边。虽不似驼峰岭天池那般的浩瀚，也没有青海湖的雄伟，更没有喀纳斯的奇幻，但是她那灵动的波，轻盈的水流，更似一位活泼热情的少女，美丽大方而又坦荡含蓄，让所有来这里的人都在这儿流连忘返。
现在在狼爪jackwolfskin的微信公众号上正在发起一个#冬日极客#的活动，只要上传一张含有狼旗的冬日照片，就可以参加活动啦。而且还可以得到免费北欧极光游和1000元装备基金，赶紧来试试手气吧。
在岸边找了一块儿地坐了下来，静静地望着眼前，时而又望着远方，环顾群山。周围的一切都好安静，安静的我都不敢大口喘气，生怕我的一下深呼吸惊扰了此刻的宁静。远处水面倒映着蓝天白云，蔚蓝的水面也被映照得格外的澄澈。可以一眼就看到水中的水草，水中的鱼群还在水草之中来回嬉戏，忽而蓝，忽而绿，仿佛天与地已经紧紧相连。不知是游鱼拨动了水草，还是水草惊扰了小鱼，却好似一幅水中镜画，生趣盎然。河面上云蒸霞蔚，水底暗潮涌动，河岸边雾凇晶莹，十分的壮观。
湖面上凝结了一层薄薄的水雾，因为在这段水域底下存在着大量的地热，也只有冬天才能看到如此的美景。不远处不知从哪儿飞来几只水鸟落在了水面上，在身后划出了一道长长的涟漪，一会儿整个水面又归于平静了。闭上眼，感觉就像进入了一个虚无缥缈的幻境，远离了尘嚣，远离了红尘，远离了世俗，仿佛这天地之间就仅剩下我自己一人，胸中的烦闷也随着一口气一吐为快。一下整个脑子都清醒了起来，一片空灵，似有飘飘欲仙之感。
秋天的金黄夹着还未消尽的积雪，犹如黄金中镶嵌着一颗颗璀璨的钻石一般，光彩而又夺目。滩头上，一群只有在阿尔山才能见到的牧牛出现在我们的视线当中，它们身上的毛异常的厚，为了抵御阿尔山极寒的天气而不得不用这样的皮毛来保护自己，还好我们也穿上了冲锋衣，比起他们如此笨重的身躯，我们算是轻便多了。它们饮着不冻河里圣洁的水，吃着阿尔山的天然的馈赠，这一切都是自然的和谐相处。这种属于自然的美好最能令人动容，那一刻，我哭了；这条河流，初次遇见就爱得如此深沉，如此的不可自拔。真该感谢生命中所有的遇见，才能让我如此的刻骨铭心。

【末】

三天在深山的里的日子匆匆而过，对于这次阿尔山的造访似乎是我这段时间做的最疯狂的决定。但是来了却从来没有后悔过这个决定，饱览大兴安岭的雄姿和阿尔山的壮阔，回归自然，让心灵在纯净的白雪中涤荡。在崇山峻岭中，人是那么的渺小；面对大自然，似乎找到了我的诗和远方。
