【行程安排】

地图上看，全南县离广东非常近，基本和去韶关没什么区别，开车过去仅需3个多小时，也可以在天河客运站坐车，120元，但每天只有一班车，下午5点半出发，所以建议还是自驾前往，全南目前还没有动车站。
这是全南景点分布图，还是比较分散的，好在都不远，路况很好，公共交通不太便利，建议开车前往。

【梅子山夜景：越夜越美丽】

下午3点出发，来到全南已经是6点多了，虽然是一个小小的县城，但整体环境还是挺干净舒服的，吃过晚饭就上了梅子山散散步，从山脚到山顶需要15分钟时间，沿途的景色比较单调，不禁小声嘀咕，山顶有啥风景，还只是个小县城，结果到上边一看就打脸了。
夜晚的梅子山顶简单而温馨，有聊天的，有运动的，有散步的，而见证这一幕幕美好瞬间的是这绚丽多彩的百盛塔，一共7层，塔身高56米，会随着时间变换颜色，七彩光韵流转，如梦似幻。
而登上塔顶，能见到整个全南的绚丽的夜色，说实话，登高远眺的那一刻，确实震撼到了我，很难想像一个小小的县城，有如此璀璨的夜色，如浩瀚星辰，似繁星点点，驱散了黑暗，点亮了夜空。
微风徐徐，夜色醉人，远处一江两岸，桃江如带，山城似画，迷人的灯带若长虹卧波，在金黄整齐房屋的点缀下，美轮美奂，让我们对这个小小的县城刮目相看，也对接下来的全南之旅更加期待。

【天龙山：悬崖崛起，三面壁立】

蝉鸣于耳，盛夏已至，但这里依旧能感受到春天的气息。游客不多，清幽宁静，小溪流淌，风景如画，空气中都能闻到芳香的味道，这是我对天龙山的第一印象。
风轻云淡，花静鸟喧，一种平静恬淡的舒适感油然而生，不知是因为这波光粼粼的清澈湖水，还是造型奇特的草木花卉，抑或当地人在湖边观鱼时的谈笑风生。相对于大众化的人挤人热门旅游地，我更喜欢来这种人少古朴景美的地方，而这样的地方，更能让自己回归平静，倾听自己内心的声音。
青山绿水，曲径通幽，浮躁的现代社会，更需要一些简单来缓冲，重返自然，返璞归真，化繁为简，感受平凡生活中的小确幸，这里似乎有种魔力，让人不知不觉慢下脚步，回归心灵的平静。
鼓声响起，寻声探去，原来正在举办“比武招亲”的活动，围观的群众里一层外一层，好不热闹，天龙山很多点都是以武侠小说《天龙八部》中原型建造的，“曼陀山庄”“无崖擂茶”“虚竹艺坊”“乔峰杏子林”“段誉食坊”等八大坊店各有特色，与周边优美的生态环境相融合，也给人们天龙山的衣食住行提供了便利。
有山有水有音乐有美酒，走累了就来木婉清吧小酌一杯，七彩的气球配上清新的盆栽，还是蛮有情调的。
沿着蜿蜒的山路拾级而上，一直到最高的平台，峰峦叠嶂，万千雄姿，而此刻被我们一网打尽，远处云山缠绕，隐隐约约的山峰多了几分模糊美。
这也是我喜欢登高望远的原因，总觉得这样离上天更近一步，总觉得自己和天地之间融为一体。悬崖崛起，三面壁立，酷似巨龙腾飞，故取名天龙山，如此险要的地势和优美的风景，也难怪蒋经国先生会称赞道：“实为楚峦佳地，南国秀岳，真乃天龙胜地也”。
途中有幸偶遇一群广东中山的“修行客”，她们说这个地方风景美还清净，所以不惜千里迢迢驱车赶来，也不清楚这是第几次来了，原来天龙山已经成为不少珠三角游客的小众秘境，真的是孤陋寡闻了。看着她们的修行，感受到前所未有的宁静，连呼吸都生怕打扰到她们，人生何尝不是一场修行，保持真我，不为外界所煽动，又有多少人能到达这境界呢？
褪尽繁华，告却喧嚣，但愿某一天，我也能突破思想的枷锁，轻装上阵，不以奢为尚，不向喧而趋，“心为明镜台”， 应“时时勤拂拭，勿使惹尘埃”。
此外还有一大看点，就是188米长横跨两座险峰的玻璃栈道，还有破裂效果，近期也将开启，在玻璃桥的中间段可以直接看到底部，没有任何树木遮挡，对，玩的就是心跳！冒着绳命危险拍了一组图，不过还是要到现场才能感受到这满满的惊险和刺激。

【雅溪古村：流转千年的慢时光】

青葱的绿叶，潺潺的流水，灰瓦黄墙，引入眼帘的是雅溪千年古村。雅溪，一个优雅的名字，而这里的景色，确实“人如其名”，明黄色的土墙有种淡淡的小清新感觉，看着令人心情愉悦而明快。
见过很多古村古镇，原来对古村已经免疫，未料见到雅溪之时，内心还是有几分波动，除了小清新式的古风建筑风格，还有这儿宁静平和的气氛，游客不多，来往的人群中还能看到村民的身影，一栋栋老房子似乎在讲述着这个村落千年来发生的故事。
古村说大不大，说小不小，路线嘛，延续我一贯的迷失路线，不打路牌，不问路人，随心所欲，流转在千年古村的大街小巷里，看巷里奔跑的小孩，看门前休憩的阿姨，慢慢地感受古村里的脉搏与呼吸。
过了一会，村里突然热闹起来，跟着人群寻声望去，原来今天为江西二套《江山多娇·我和我的家乡》的直播现场，传统的车马灯戏、摔碗酒、品尝当地特色美食等活动陆续开展开来，人来人往，熙熙攘攘，想起了小时候自己的村里，一有什么风吹草动，大批小伙伴马上叫上邻里街边，聚集起来看热闹。
而在围楼里，还有男女山歌对唱表演以及擂茶的现场制作流程展示，全南虽然在江南的赣州，却离广东非常的近，与韶关接壤，也有和广东一样的客家文化。
擂茶文化也是全南饮食文化的重大组成部分，用花生、绿豆、茶叶、山苍子等原料放在一个陶制的擂钵里边，擂约半个小时，然后用一把捞瓢筛滤擂过的茶，投入铜壶，加水煮沸，擂茶便做好了，可食用，可药用，可解渴，又可充饥，老少皆宜，全南旅游之必品特色美食，时隔20多天，还是忘不了那满堂的飘香。
而人群散去，才意识到自己置身于几百年历史的客家围屋，目前已列入世界文化遗产预备名录中，雅溪的围屋有“福星土围”和“雅凤石围”这两个，有点福建土楼的感觉，围起来也和土楼一样，为了防盗防寇，如此建筑也体现了客家先民的聪明与睿智。
走出围屋，继续闲逛，景区工作人员的古装让人有种时空错乱的穿越之感，除此之外，还能见到不少原著居民，以及他们的生活场所，优雅中又透漏着些许的人间烟火气息，十分接地气。
溪流蜿蜒，青山环抱，如此优雅的村落，怎能不让人心动，走着走着，耳畔回响起了木心先生创作的那首诗歌：“从前的日色变得慢，车，马，邮件都慢，一生只够爱一个人”，除了优雅，更多的还是那份平凡生活的感动吧。

【沙坝仔神农文化园：生态乡村独具风韵】

第二天我们前往沙坝仔神农文化园，感受全南深厚的农耕文化和浓郁的田园风情，沙坝仔是全南县的一个村，近几年经过改造发生了翻天覆地的变化，一大清早的沙坝仔神农文化园，清幽宁静，不被外界所打扰。 
天青色，等烟雨，没有阳光明媚的蓝天白云，却有水墨丹青的诗情画意，平静的湖面，如同一面镜子，倒影着天上人间，也倒影着我们内心的世界，微风不起一丝涟漪，却在我们内心泛起阵阵波纹，这到底是什么神仙地方？
农耕余韵，怀古遗风，深厚的文化底蕴、独特的客家风情、秀美的生态环境，藏于深山人未识，像是遗落在一个大山里的梦，梦境中青葱翠绿，农家点点，炊烟袅袅，原始古朴的气息让人沉醉其中，不知归路。
虽然我自身对什么文化园没什么兴趣，但这里却给我一种耳目一新的感受，在沙坝仔身上，我似乎看到了宏村的影子，白墙灰瓦，翠竹青石，写意无限。
也有幸参观了下湖边的民宿，确实有中国古典诗歌中的儒雅，但房间的整体装修还需要再提升下档次，有如此优雅的自然环境真的可以往高端民宿方向走，最后一张是在三楼阳台上拍到的景色，确实如山水画一般让人留恋往返。 

【葡萄采摘季：嗯，初恋的感觉】

来全南之前就听说这儿的葡萄又大又甜了，夏天刚好是葡萄采摘的季节，来得早不如来得巧，月秀葡萄成熟之时，我们如约而至，试吃了几颗葡萄之后，那种整个口腔的酸甜之感，就像初恋的感觉一样，妙不可言，于是每人抡一个框就冲入葡萄园。
这儿的葡萄有几种，我也叫不上名来，只看到颗大肉满的葡萄在我周边摇晃，仿佛进入了葡萄的海洋，而和朋友亲手采摘的乐趣，也融入了那酸甜可口的汁液中去，想想就令人垂涎三尺。
满筐圆实骊珠滑，入口甘香冰玉寒，来不及回家吃，当场又吃了好几个，果大甜香，硬脆爽口，给予我们极大的满足感。葡萄的储存期还是比较短的，建议在五天内吃完，当然越快越好，鲜味才不会因为时间的延长而消逝。
全南的田园生态也不止这些，还有江禾田园等优秀的综合体，确实全南拥有得天独厚的生态自然条件，将这些融入发展，一定会焕发出新的生机。
离开全南的时候，顺便路过了解放桥以及新城门，弥补当晚没有过来看夜景的遗憾，整个全南似乎铆足干劲在发展，听朋友说这两年的变化特别的大，突然感慨中国闪电般的基建速度，也正因为有无数个像全南这样大力发展的县城，才有今天中国在国际舞台上的地位和自信，国家的富强，离不开每一个中国现代化建设的人，全南的旅程也告一段落了，希望下次还有机会再来，欣赏到更加完美的全南。

【舌尖上的全南】

关于全南美食，因为周边青山环绕，而且靠近广东韶关梅州等地，口味和客家菜差不多，偏咸，同时也偏辣一些，但都可以根据个人口味改良，总的来说，还是比较习惯的。这里也总结下这几天吃到的全南特色美食。
1，雅溪古村陈家大院
这种类似果冻般的小吃就是全南特色小吃磨斋（推浆糍），色泽金黄，爽牙可口，具有“柴灰水”特有的浓香，是全南老百姓必备的年货。
这是糍粑，南方人已经很常见，用熟糯米饭放到石槽里用石锤捣成泥状制作而成的食品，既是客家小吃，又可当主食，在全南地区也非常受欢迎。
客家酿豆腐，豆腐中含有香菇、碎肉、葱蒜等佐料，也可以直接理解为“有肉馅的豆腐”，入口鲜嫩滑香，非常配饭。
其外还有各种当地特色美食，就地取材，最大程度地保证了食物的鲜味。
2，黄聂餐馆
这家是当地非常有名气的餐馆，靠口碑赢得当地人的广泛好评，甚至有临县的朋友千里迢迢过来吃饭。
龙骨生地汤，酿三样，焖唐鱼，炒牛三宝，泥鳅芋头，盐鸡，椒盐排骨，炸土虾，炒全猪，炒南瓜梗等等都值得一试，味道还不错，卖相也都下足了功夫。
不得不提的还有他家的早餐，也是做到整个县城都知道，为了保证食材的新鲜，他们凌晨3-4点就出去采购，猪红汤是猪血和猪肉混合，猪血晶莹透明，猪肉则软嫩蓬松，有辣和不辣两个版本，可以说是全南名副其实的网红早餐。
3，醉美全南宴
全友宾馆的“醉美全南宴”，刚刚在今年五月份第二届赣南客家美食节荣获“江西名宴”的称号，每个菜的份量都大得惊人，把我们吓一跳，摆拍也精致，推荐菜：黄精长寿鸡，乌桕坝熏鸡，天龙滋补鸭，多味蝴蝶，红扒山竹鼠，清蒸菜丸，脆炸南瓜花。

【关于住宿和门票】

全南毕竟是小县城，酒店业不是很发达，没有过多的选择，最好的应该我们住的希桥酒店了，4星级，每晚只需不到300元的价格，全南县城不大，酒店离中心位置只需10来分钟的车程，整体环境还是可以的。
交通上边都详细说明了，而门票目前也基本都无需门票，总的来说性价比还是非常高的。

【后记】

总体而言，全南还是超出我的意料之外的，原来广州周边还有如此秘境之地，忘不了那一份闲适和舒畅，全南，有缘再会。
