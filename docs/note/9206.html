抛弃男友，跟闺蜜逃离到首尔的三天两夜

游记 动物 摄影 自助游 购物 闺蜜游 2019/03/23
一直以来杰西我都是跟康纳一起出去旅游，这次终于有机会跟小姐妹出去玩个两三天了，经过我们的商量，最后决定去 韩国 首尔 来个三天两夜游，接下来就是属于姐妹俩的行程啦！

小浣熊咖啡厅Blind Ally

来首尔之前就决定一定要去小浣熊咖啡厅摸摸浣熊啦，因为之前在温哥华的时候有一只小浣熊溜到我家阳台上偷吃狗粮，当时就觉得太可爱了，但由于温哥华的小浣熊都是野生的，经常翻垃圾导致身上很脏很多细菌，我就没上前摸它，这次到了首尔终于有机会啦！
首尔有2家浣熊咖啡厅，而我去的这家是位于龙山区的Blind Alley。位置很好找，门口会有很多牌子指引客人去找到这家店，所以大家不必担心~
一进门就会有店员询问你是想要看浣熊还是只喝咖啡，如果只喝咖啡的话就只付饮料钱就可以了，而看浣熊的话就需要交门票钱，但会包含一杯饮料。
我去的那个时间小浣熊正好在休息，所以我和朋友就先坐下来喝东西~
旁边桌上有一只柯基在睡觉，但是很凶，会对人龇牙
然而当你手上有零食的时候，这只柯基不仅不凶了，还会表演握手和鼻子钻孔，真的是很势利了 
店里还有两头猪，店员一开始说是”mini pig”，意思是迷你小香猪，然而当这两头猪露出身体之后我开始怀疑店员是不是对”迷你”这个词有什么误解😂
这两头猪是可以摸的，毛比较硬，手感不是特别好，但算是温顺
店员还会让客人坐在垫子上，用毯子撘住客人的腿，然后在上面撒猪零食，这个时候我深刻地体会到什么叫“被猪拱”了 
这家店一共有三只小浣熊，两只深色一只浅色的，都很胖。我对比了一下它们一年前的照片，感觉是客人来太多给它们喂胖的 😂
进去小浣熊的活动区域之前要把包包、首饰寄存，只能带手机或相机进去，因为小浣熊们可能会因为好奇而去拿你的东西，例如我在抱浣熊的时候就差点被它“抢”了项链
这里的小浣熊每年都会去做健康检查还有打针，所以大家无需担心动物的疾病问题，但要是手上有伤口的最好还是别去喂食啦。

韩国校服体验

以前看韩剧的时候很喜欢女主角们的校服，所以我就在网上找到了这家可以租校服的店铺~
店铺里有分普通校服和高级校服，各有夏装和冬装两款，满足了大家在不同季节的需求。里面的场景布置也很少女心，有书桌和香芋紫色的椅子还有粉色储物柜。如果觉得场景太少的话还可以在租赁校服的时间内穿着校服去大学里面拍照哦~
建议大家用手机拍照就好了~相机的话有点难调色

可以看日落的屋顶咖啡厅

相信很多人有在Instagram上看过跟这个角度差不多的照片吧~
没错，就是在这家咖啡厅拍的
我跟朋友是在周五下午5点多到达这家店的，在点完饮料上楼找机位的时候发现 天台 已经坐满了人，而且还有很多人在排队等拍照。于是我们就下楼坐着了，发现楼下其实也很适合拍照，特别是在快要日落的时候，光线特别好。而在差不多6点的时候楼上拍照的人就会慢慢离开，这时候我们就可以上楼拍这个机位了~
而除了这个机位以外，在二楼平台上的这个位置拍照也是不错的哦，但地方有点拥挤，大家要给点耐心找到好的拍照角度。
顺带一提，这家店的cheesecake真的超好吃！！

穿韩服体验韩剧生活

到了首尔 ，最最最不能错过的就是穿韩服逛景福宫啦，一是为了体验韩剧生活，二是穿韩服进景福宫是不用买门票的哦~
在景福宫外的路上有很多家韩服租赁店，大家可以多逛几家去对比一下。但根据我的观察，每家店的款式应该都是差不多的，价钱我就不太清楚了。
我租韩服的这家店是服装发型分开收费的，服装的话传统款和改良款也是不一样价钱。我跟朋友选择了传统款，后来到景福宫内发现我们这个选择是对的，因为几乎所有人选的都是改良款，走两步就会撞衫哈哈 
景福宫不算太大，基本上韩服租两个小时就够了。我建议大家选择在刚开门或下午三四点的时候入场，那样人不会太多。我去的那天是周六下午三点多，感觉也没有特别多人
拍照角度可以选择仰拍
或者在偏殿拍
也可以在刚入门时侧面的小走廊拍
或者在犄角旮旯拍人像
地址：161 Sajik-ro, Sejongno, Jongno-gu, Seou
时间：9:00 – 18:00
韩服租赁价格：15000万韩元 / 人 / 2小时

新罗免税店

作为一个喜欢买买买的女孩子，怎么会错过免税店呢！尤其是这种不在机场里面的免税店，没有时间限制，想逛多久就逛多久哈哈哈 
这家新罗免税店位于 首尔 最棒的高端酒店新罗酒店旁边，这个酒店的迎宾馆是宋仲基和宋慧乔结婚的地方哦，那么来新罗免税店的我四舍五入就是跟双宋cp擦肩而过了！除此之外，酒店一楼有Pastry Boutique西饼店还有著名的自助餐、中餐、日料和 法国 料理餐厅等，大家要是在免税店逛饿了随时可以去新罗酒店大快朵颐。
虽然这家免税店距离我们住的酒店非常近，但我跟朋友还是拦了辆出租车前往免税店，因为新罗免税店拥有许多打车优惠呀！
