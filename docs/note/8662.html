【天鹅之旅首发】黔南深度游丨迷醉在贵州乡村的美食美景

游记 人文游 古镇 小众 摄影 采摘 2019/05/30

前言

神奇的喀斯特地貌，隐于山水的神秘村落，地域特色浓烈的贵州美食，曾经的南方出海丝绸之路的重要通道，宜人的气候，这些都吸引着我去黔南州探索一番。

黔南
旅途中的笑脸是一味强心剂，瑶族、水族、布依族、苗族，淳朴的族民们张开怀抱，热情的迎接四海八方到来的异族同胞。民族虽然不一样，但我们都是中华子孙，是血脉相承的亲人。
黔南
黔南
神奇的大自然，慷慨地赐予世人数之不尽的景观，让我们蒙尘的心灵和身躯有去处可以洗涤净化，是世外桃源大小井，是静美如诗的神泉谷。
黔南
黔南
夜幕降临下的都匀，西山大桥和石板街共同点亮了这座城市最迷人的一面。

黔南
重回猴场会议，在红色历史遗址回味那段战火纷飞的年代下坚强的心志和大无畏精神，忆苦思甜，让我更珍惜现在的生活。

推荐游玩线路

线路1：都匀—福泉—瓮安
前往【都匀】，欣赏夜景下的西山大桥、石板街、入住维也纳国际大酒店，第二天前往【福泉县】，问道福泉古城，去黄丝镇江边村体验新农村旅游；到达【瓮安县】，感受红色文化与千年古镇，入住宋钦客栈
线路2：龙里—贵定—惠水—长顺
前往【龙里县】游玩云栖谷、油画大草原，再到【贵定县】感受金海雪山和布依族景点，在【惠水县】好花红村，体验地道的布依族生活，【长顺县】的凤凰坝和深泉谷景致绝美。
线路3：都匀—三都—荔波—平塘—罗甸
从【都匀】前往【三都县】，参加了鱼鳅寨的枇杷节，下午去往【荔波县】瑶山乡拉片村，瞻仰邓恩铭故里，晚上抵达平塘并住宿，第二天前往【平塘县】六硐风景名胜区清凉清新，下午去【罗甸县】的大小井风景区，壮观的自然景色如同仙境一般，晚上到达龙里，参加龙门镇篝火晚会

繁华都匀，醉倒在你的夜色之下

黔南
都匀是黔中经济区的五大主要城市中心之一，不论是经济、政治、文化，在西南地区都有很重要的地位，而最近的一部热评影片让这座低调有实力的城市再次进入人们的视线。
黔南
黔南
最能代表都匀市夜景的，要算西山大桥，作为《无名之辈》的取景地，这里已经成为网红打卡点!在电影里，西山大桥屡屡出镜，整部戏的热点也在这里上演，当烟火升空，令人回味的不光是故事情节，秀丽的夜景也引得无数人开始搜索这里。
黔南
距离西山大桥不远处，有一条长约200米、由一万多块青石铺成的石板街。街道两旁商铺鳞次栉比，但这里并不是那种已索然无味的商业街，这条街在明清时期是重要的古驿道，徐霞客也曾来此处游历。
黔南
青瓦、红墙、雕花门窗，典型的明清建筑风格，经历了700年的风雨，这里依然古朴而典雅。夜幕之下，大桥与老街交相辉映，一起点亮这座繁华的城市。

黔南

来古城福泉，感受道教圣地之气

贵州福泉山
福泉历史悠久，早在殷商时期便有民栖居于此，战国时期的古夜郎国、东都竹王城遗址、被誉为“中国古代军事防御的绝妙制作”的福泉古城垣都在此处。而福泉因福泉山得名，福泉山因道教扬名。
贵州福泉山
贵州福泉山
著名的道家人物张三丰曾在此修炼，，关于张三丰的故事可真是多如牛毛，但应该都没有福泉这里的渊源深。
贵州福泉山
“山不在高，有仙则名”，张三丰登高望远，见此处自成一副“太极图”，认为是道教修炼的极佳之地，便在这里修道了八年之久，福泉山的山山水水也带上了仙气，一池一井都带着真人的影子，特别是那福泉井水，也有延年益寿，美容养颜的功效。
贵州福泉山
贵州福泉山
礼斗亭中是张三丰的自画像石刻，全国只有三处。张三丰碑林里保存着以张三丰诗为主的数十首经典诗词，还有百余首古今问人的诗文墨宝。
贵州福泉山
贵州福泉山
除了宗教地位外，福泉山的文化地位也非常之高，而整个福泉山也是山清水秀，游览一番令人神清气爽。
实用信息
地址：贵州省黔南布依族苗族自治州福泉市文化路
门票：不需要

黄丝江边，美好乡村

黄丝旅游度假景区
黄丝旅游度假景区
黄丝镇，江边村，波光粼粼的鱼梁江畔，一道帆水堰，现代人文工程的干预使得这里更加富饶。而真正让这里繁荣富强的是吃苦耐劳的江边人民，这三孔石桥叫做和善桥，始建于清光绪三十年，耗费千两白银，让两岸的人民可以畅通无阻。
黄丝旅游度假景区
黄丝旅游度假景区
集农产品与旅游于一体，葡萄长廊和草莓采摘园、樱桃园都是新农村建设的产物，一边赏着风景，一边享受采摘的乐趣，感谢乡村旅游的存在，是一处很好的缓解压力的场所。
黄丝旅游度假景区
黄丝旅游度假景区
这里也有个动人的传说，江边寨上游的三座巨岩相传为三仙女所化，为了帮助好心的年轻人摆渡，仙女们误了时辰而无法返回天宫，便化石于此，黄丝这里也跟沾了仙气一样，人杰地灵，美女辈出。
黄丝旅游度假景区
景美菜好，黄寺老罗桥附近有一家源缘生态山庄，我们在这里品尝到了地道的黄丝美食。酸酸辣辣的酸汤鱼、甜糯可口的粽子、鲜美的野生菌炒肉，都十分可口。一道简单的凉拌猪耳，让我真正明白了大家为什么都说贵州蘸水味道好，真是画龙点睛之笔，来黔南的朋友一定不要错过美好的蘸水。
实用信息：
交通：从福泉山到江边开车大概1个小时，可以走高速，会比较快
源缘生态山庄地址：包南线罗桥西北侧70米处

草塘——千年古邑

草塘千年古邑旅游区
草塘千年古邑旅游区
在瓮安县，曾经发现过距今5.8亿年前的动物群化石，是迄今为止发现最早的动物群化石，因此这里又被成为生命起源之地。而草塘阵（又名猴场镇）是拥有千年历史的古镇，在贵州商业历史上，有着很重要的地位。

草塘千年古邑旅游区

草塘千年古邑旅游区

邓恩铭烈士故居

后记

这次的黔南深度游，让我领略了黔南州丰富多彩的美好。这块土地上孕育出了多个少数名族，文化在这里流淌传承，旅游建设力度的加大，会让原本偏僻简陋的乡村焕发光彩，将深藏的魅力都释放出来，相信这里一定会越来越好，期待下次见面！
