说说这次旅行

DAY2 赴长白山，只为圆梦祖国名山

长白山，华夏十大名山之一，也是中国东北最出名的高山。石头继泰山、黄山、华山、嵩山、祁连山、武功山、西岭雪山，再一次攀登祖国名山——长白山。
长白山风景区
延吉有长白山直通车到达长白山风景区游客中心，车程四个小时。长白山分为长白山北景区、长白山西景区、长白山南景区，不得不佩服长白山充分开发赚钱。
长白山风景区
石头只去了长白山北坡北景区，大门票+环保车组合票190元，看天池还要在现场买倒站票80元，合计270元，长白山景区门票车费真是贵，去其他长白山景区还得重新买票。来一次的人，也许都不会来第二次，一家三口来一趟长白山门票车费也太贵了，成本太高。
长白山风景区
长白山风景区
长白山风景区
长白山北坡北景区包含了长白山天池、长白瀑布群、聚龙温泉群、小天池、长白山绿渊潭和长白山地下森林景点，景点太多，地方又大，建议驴友们合理规划时间，一天玩遍所有景点，还是很紧迫的。
长白山风景区

长白山天池

长白山风景区，最值得一游的就是位于长白山主峰火山锥天池体的顶部的天池。买了长白山大门票+环保车，是不能上天池的，必须购买上天池的80元车票。
长白山风景区
长白山风景区
长白山风景区长白山天池，是中国最高最大的高山湖泊，其实是一座火山口，经过漫长的年代积水成湖，还是松花江、鸭绿江、图们江的东北三条大江的发源地。
长白山风景区
长白山风景区
长白山天池，是很多国内外游客来长白山追梦所在，不过并不是每个人都能有幸看到长白山天池，据说只有30%的游客才能看到长白山天池。
长白山风景区
不得不吐槽，更坑的是景区明知山上下下雨雾特大，还继续送游客上山，也许看天池的80车票，不能错过赚钱的机会。
长白山风景区
这里建议山下下下雨雾大就不要买上天池80车票浪费钱，还有尽量中午之后去看电池，要比早上看到的几率大的多。

长白山瀑布

长白瀑布群，号称中国东北最大的瀑布群，是长白山的第一名胜。长白瀑布群来至于长白山天池，是松花江的源头，可玩性一般。
长白山风景区
长白山风景区
也许看过贵州安顺黄果树大瀑布和壶口黄河大瀑布的雄奇壮观、气势磅礴、气吐山河之壮。
号称中国东北最大的瀑布群——长白瀑布群，真的一般，水头很小，以为是山上下来的小溪。
长白山风景区
不过，周围四处环山，风景还不错，凉爽清新的空气，是暑假高温避暑好地方，值得到此一行。

长白山聚龙温泉群

长白山第一泉的聚龙泉，比长白山瀑布值得玩多了，可玩性值得推荐。据资料查询聚龙泉，是温泉群中水量最大聚龙泉、分布最广、水温最高的温泉。
长白山风景区
聚龙温泉群四周有无数条热流从地底涌出，热气腾腾，水温恒温。不过，聚龙泉水量还是太小，并不能用来当温泉泡，用来游玩摄影还是不错的。
长白山风景区

长白山绿渊潭

长白山绿渊潭，和小天池很相似，其实也是长白山山谷森林中一个池塘，水清澈到底，四处树木怀抱，崇山包围。

延边博物馆

延边博物馆，是一座集地方历史和朝鲜族民族特色于一体，位于延边朝鲜族自治州首府城市--延吉市。
延边博物馆
延边博物馆
延边博物馆 基本陈列有《朝鲜族民俗展》、《千秋正气--中国朝鲜族革命斗争史陈列》和《延边出土文物陈列》共三大陈列。
延边博物馆最大特点就是《朝鲜族民俗展》展陈从朝鲜族迁入到生产、生活、人生礼仪及文化、娱、朝鲜族文字、乐器、服饰等多个角度，全面展示了中国朝鲜族优秀民俗文化。
