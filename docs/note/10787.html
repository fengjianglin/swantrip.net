开篇语

她坐落在东南沿海，气候宜人无严寒，
一年四季都张开怀抱迎接着我们；
她是闽南文化重镇，
有着年轻可爱的一面，却也饱含着历史的韵味，

无一不让人欣然前往！

交通

【交通】
1.出租车、网约车
之前去厦门主城的时候，觉得有些景点里的略远，又懒得看公交线路的攻略，所以每次还是打个车，省点心~

 
2. 公交车
厦门公交车很发达，目前厦门市有公交线200多条。公交收费上车1元，分段计价，每跨一段加收1元，最高票价3元。轮渡口、厦门大学、火车站是主要的公交始发站。公共汽车市区专线服务时间自6:00-23:00。
以下公交车途经机场：27、37、41、81、84、91路。 
公交查询网站： http://xiamen.8684.cn ;
公交公司服务电话：0592-5852027、0592-5859275 


3. BRT——厦门新的空中通道
上车0.5元，全程3.5元。 
快速公交系统（Bus Rapid Transit）简称BRT，是厦门的一个交通特色，被喻为厦门市的空中交通系统。目前厦门的BRT线路有快1A线、快1B线、快2线、快3线四条线路。 
快1线 第一码头（05:45-22:15）——厦门北站（06:10-22:50） 
快2线 第一码头——同安枢纽站(侨英路口站、天马路口站、洪霞路口站、西吴路口站、美峰路口站暂不开通) 
快3线 第一码头——前埔 
另外还有二三十条BRT连接线，全程一票制1元，使用e通卡更有减免,0.8元。 例如： 
L1线 第一码头——第一码头 
L5线 二市——东渡 
L19线 县后——国内候机楼

住宿

【住宿】
在这里住宿比较推荐的几家酒店：
悦蓝湾庭岚坊客栈
位置：位于曾厝垵旅游风景区，周边遍布琳琅满目的特色小吃、民俗风情店。走出去不几分钟就到了曾厝垵大门，公交和出租车都很多，交通便捷。
环境：酒店后面环山，四楼有休闲大露台，闲暇之余喝个茶，赏个景也是一种享受。这里的确是个闹中取静的好去处，特别适合小情侣出游选择佳地。
卫生：客房空间大，布置的比较清新整洁，室内通透采光好，空气流通。
服务：极好，老板帮着抬行李上去。而且有问必答，怎么去景点可以问，非常方便。

客谈酒店

“客谈”酒店
位置：位于曾厝垵旅游风景区。走出去不几分钟就到了曾厝垵大门，公交和出租车都很多，交通便捷。 从机场出来打车很快就能到。
环境：闹中取静的好去处，特别适合小情侣出游选择佳地。
卫生：大气，装修豪华，有五星级酒店的感觉，干净！
服务：店里有人来接，服务都不错。老板有问必答，能减少不少麻烦，可以联系0592-2516089预订。

【景点】

【曾厝垵】
在十几年前，曾厝垵不过是厦门海边的一个小小的渔家村落，如今，走在风景如画的环岛路上，这里早已经成为这条路上一个不可或缺的风景。在厦门你随便问一个人，恐怕没有不知道曾厝垵的。
 爱上曾厝垵可以有很多原因，海肯定是第一个。如果你眯着眼睛，看过阳光下曾厝垵的海，你一定会爱上它。如果你看过黑色的大鱼和礁石的牙齿，你一定会爱上它。如果你看过旅人们在海边放孔明灯，光芒渐渐消失在夜空中，你一定会爱上它。
在海边的木栈桥上，你可以看到不远处的鼓浪屿和小金门，和全国最高的双子塔建筑—世贸大厦。而在眼前的沙滩上，有几块巨大的礁石立于大海之中。原来，在这里还立着一个由十几根红色船浆做成的雕塑，它是曾厝垵的一个地标之一，也是许多游客来这里必须要留影的景点之一。可惜在前不久的一场台风中，船浆雕塑被吹散，只留下光秃秃的一个基座，相信不久以后还会重新立起来。

【曾厝垵美食】



各种吃的，各种应接不暇，对于吃货会是天堂，你可以品尝到各类特色小吃，土笋冻，鸭肠，臭豆腐等等，有很多文艺的小店，明信片，地图，可以刻字的竹杯，同心锁总有一样值得你带回去的

.面对曾厝垵琳琅满目的美食招牌和褒贬不一的评价，游客不必纠结。即使多花费一些排队时间，也是值得的。


五兰沙茶面碱水粗面烫熟，快刀剪下加料，将浓香的沙茶汤头浇上去，做成了一碗地道的“招牌沙茶面”。


闽宗闽台肠粉(仅有一家，无分店)

曾厝垵的壹碗粥味道相当不错的！


店内提供各种美味粥，老板娘人很好哦！粥绵绵的，很入味，海鲜粥的海鲜非常新鲜，旅途吃太多油腻，不烦尝一口壹碗粥。

【环岛路】


曾厝垵紧邻环岛路，游客可以租辆自行车骑行，体验下移步换景的环岛路自行车道，也可以去往附近的音乐广场或书法广场，悠然自得地在海边小坐一会儿。

总的来说曾厝垵是一个特别适合热恋中的小情侣发呆、吃喝玩乐的地方，值得一去。

【鼓浪屿】

岛上交通
鼓浪屿是一个没有机动车和自行车的小岛，岛上唯一的交通工具是景区电动车。
除了观光的电动车，鼓浪屿是个适合漫步的岛。步行鼓浪屿，享受小岛的慢节奏，是个不错的体验。
1.行程安排
小岛上景点相对集中，很多人倾向于无安排的走走逛逛，这样更容易偶遇惊喜，但如果你是一个热衷于把事情安排的妥妥当当的人，那么可以按照这个推荐路线来游玩：
 首先看一下鼓浪屿的景点地图（地图一下渡轮就可以领取一份，关键时刻还是很重要的~）
2. 划重点
鼓浪屿比较出名的景点就是：
龙头街——风琴博物馆——海底世界——皓月园——菽庄花园（钢琴博物馆）——日光岩

鼓浪屿是个很安静的小岛，岛上唯一的商业中心就是龙头路。几乎所有的商场和小吃店都集中在这里，除了网红店（张三疯奶茶店、海小姐的玫瑰饼...），还有很多装修得很小资商铺，卖各种女孩子喜欢的小本子、小摆件，非常招人喜欢。

最有特色的是，这里大多数的店铺如咖啡馆、客栈、奶茶铺、杂货铺都有自己的印章，岛上发售的盖章本上每家小店们都各占一页，印有店铺的名字和地址，游客们可找上门在对应的地方盖上章，算是到此一游的印记。
游逛完龙头路，可以来到同样距离码头不远的风琴博物馆。

这座风琴博物馆是目前国内唯一、世界最大的风琴博物馆，馆内收藏有多种款式的风琴，其中要数中央中庭这部超大体量米高的诺曼·比尔管风琴。
开放时间：8:30-18:30
票价：20元，学生半价。
另外可以购买鼓浪屿众景点的联票，风琴博物馆的门票也包括其中。
喜欢一览众山小感觉的朋友，一定要登上日光岩制高点！
那里是欣赏鼓浪屿全景的最佳地点！
开放时间：
冬季：5:30-20:30，夏季：5:30-21:00。
游览时间一般为1-3小时
票价：日光岩门票60元（包含了山脚下的日光岩寺）学生票30元。
皓月园
皓月园座落在鼓浪屿东南，濒临鹭江，是为了纪念郑成功驱逐荷夷，收复台湾的历史功绩，福建省政府和厦门市政府，在此建造了郑成功纪念园。园中最大的亮点莫过于最大的一座中国历史人物雕像——郑成功雕像，郑成功巨型石像于1985年8月27日落成。


夜晚的皓月园景色优美，可以眺望对岸景色，非常适合饭后散步~

开放时间：7:00-21:00
游览时间一般为1-2小时
票价：成人20元，老人及学生优惠。
菽庄花园
菽庄花园夜市值得去的景点之一，整座花园面向大海，背靠日光岩！1913年，台湾富绅林尔嘉因为怀念台北的板桥而修建，并以其字“叔臧”的谐音“菽庄”来命名。内分成藏海园与补山园，两园内各建五景，包括四十四桥和十二洞天等。
开放时间：7:00-18:00
票价：菽庄花园门票30元，学生半价，开放到17:30.
这是联票内景点，所以还是建议购买鼓浪屿联票哦~

【厦大】


厦门大学依山傍海。它背靠五老峰，紧邻南普陀寺，出了白城校门是美丽的海滨浴场与胡里山炮台，一直是“中国最美丽的大学”之一。校内楼群多为欧式风格，色彩明艳，且取名“囊萤”“群贤”“芙蓉”“凌云”等十分特别，漫步校园之中，感觉轻松愉快。
嘉庚楼群前的芙蓉湖波光粼粼，气氛宁静，两旁各色花树，常有厦大学子驻足晨读。上弦场和建南大会堂是厦大的标志性建筑，鲁迅纪念馆、芙蓉隧道、情人谷（厦大水库）也是必玩景点。

厦大的建筑风格是建校人陈嘉庚先生独树一帜的“嘉庚风格”，“戴斗笠、穿西装”~
所以屋顶是类似南普陀的闽南的燕尾脊、马鞍脊和中国传统的歇山顶，楼身是西式建筑，也是把西方压在身下的意思。

【南普陀】
南普陀寺在闽南地区影响力很大，高僧辈出，佛教界都给予南普陀寺很高的评价。求神很灵！！！



这里无门票，请香也免费，另外在南普陀寺，不容错过的还有南普陀寺素斋，人气很旺，旺到需要先买单排队，素菜的份量挺多，口味也很不错的，来寺庙，吃顿素餐也是心灵洗礼的一种方式吧！
【沙坡尾】
第一眼看上去，这里十分的“旧”，好像就是老厦门的味道，混着些海的气息~但沙坡尾是厦门著名的文创区，类似于北京的798，上海的1933~

【中山路步行街】


中山路步行街始建于1925年，是厦门最老牌的商业街，可以购物，经常有人在这里购买一些海产品之类的。它也是厦门地标，类似于上海南京路步行街，只是相对比起来，它显得更有历史风韵~

【集美学村】

集美学村位于厦门集美区，是集美各类学校和文化机构的总称，陈嘉庚先生曾说“启迪民智，有助于革命，有助于救国，其理甚明。教育是千秋万代的的事业，是提高国民文化水平的根本措施，不管什么时候都需要。”陈嘉庚本着上述宗旨，在1923年创办离集美学村。现在的集美学村不仅是学习的地方，还是个著名的旅游观光地，学村内有归来园、归来堂、嘉庚公园等等著名旅游景点。
道南楼
道南楼位在龙舟池边，是集美中学的一部分，建筑上端是传统建筑风格，中下端是西洋式的建筑，寓意中国人把西洋人踩在脚下，叫做嘉庚式建筑风格。道南楼中的“道南”二字源于《论语》中“我道南矣”之句。此楼建于1962年春天，现在是集美中学的主教学楼。
龙舟池
位于集美中学道南楼正前方，当年陈嘉庚在集美修建了很多学校，他本人主张寓教于乐，学生在学习的同时也要加强体育锻炼，于是修建了很多篮球场很足球场，当然比较有名的就是龙舟池了，每到端午前后半个月就会在这边举行龙舟比赛，既锻炼身体又陶冶情操。
归来园、归来堂
归来园建于1983年，是为纪念陈嘉庚先生创办集美学校70周年而建的，占地面积9300平方米。是厦门的一座公园。
归来堂，建于1962年，位于归来园旁边，是一座别具风格的单层建筑。取白石砌墙，绿瓦盖顶，具有闽南特色的代表性建筑。
鳌园
鳌园是厦门嘉庚公园内的公园，位于厦门集美学村东南角海滨，因其形似海龟而得取名“鳌园”。它主要由门廊、集美解放纪念碑和陈嘉庚陵墓3部分组成。园内两厢刻着40幅古今历史人物的青石浮雕，人物千姿百态，立体生动，素有“闽南博物馆”之称。
嘉庚公园
嘉庚公园位于福建省厦门市集美区集美学村的东南海滨，是为纪念爱国华侨陈嘉庚先生所建。公园竣工于1994年，公园占地30000多平方米，建筑面积5500平方米。
