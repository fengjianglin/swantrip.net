武功山
行走在 武功山 途中，
看见路上有这么一句标语：
感谢你把自己的路，
走成了别人眼中的风景。
这句话，我一直记到了现在。
2018年6月我开始毕业旅行，7月下旬我从 庐山 下山来到了 武功山 山脚下的山水户外驿站。
那个一直背着户外登山包穿梭于各个城市的我，一度被很多人认为我是一个热爱徒步，热爱户外的旅行者，但我只是一个在各个地方旅行晃荡的小姑娘。
这一次是我真正意义上的徒步，第一次走完三天的徒步路程，以后的路还会走更远更远……
心姐还有贺大胆
我没有任何的户外装备，除了一双运动鞋，甚至背包里裝的都还是夏天的各种裙子，穿着仅有的稍微宽松的一条牛仔裤和买了一根木棍，和在驿站认识的心姐、贺大胆三个人结伴而行。
我在 庐山 上时，双腿已经被不知名的虫子咬得满腿是包，下 庐山 后是当晚的火车前往 萍乡 ，第二天一早到 萍乡 ，在火车站与人拼车到达 武功山 反穿路线的起点沈子村的山水户外驿站，到客栈之后直接倒头便睡，期望两腿的包能够快点消肿。
沈子村
沈子村
山水户外驿站的夕阳
下午遇到 海南 来的心姐和我同屋，心姐还捡了一个男生，在等待腿好一点的我第二天也跟着他们一起出发了。
楼顶 天台 拍摄的银河
一度觉得山水这个客栈的风景也很好，傍晚时分的夜色撩人，楼顶 天台 还有清晰可见的银河，白天还可在天然的小泳池玩耍……
当天晚上在拍夕阳余晖，晚上去找地方拍星星的时候，某个谎称村里人称贺大胆的跟着我，告诉我楼顶上可以拍，原来是窥觊我的摄影技术想叫我教他拍星空，现在的我不管走到哪，好像都在收摄影徒弟……

Day1 爬山的意义

三人小队晃悠悠的一大早便出发了，对于 武功山 其实我没有做攻略，因为知晓心姐都做好了攻略，还有贺大胆都是第三次来了，那我跟着走便是了，以至于我走过的地方都不知道名字。
沿着山水户外左侧的小路一直前行，不久后便是一直需要不断爬行上升的铁蹄峰。三天的路程，这一段是看见垃圾最多的，看到那么多垃圾心里挺痛。其实不用担心迷路，一路上都有前辈留下的记号。
一路上的垃圾太多了
到达山顶时，回头一看脚下的风景，瞬时觉得神清气爽，此时，听到贺大胆说了一句话，
“回头看山的那一瞬间，一切都有了意义。”
我想此刻我的心情，也正是如此。
很多时候，我们做着很多别人看起来不可思议的事情，只要是自己觉得快乐的，就坚持下去吧，生命那么短，我想去阅历山河，我想为自己认真地活一次。
夏日的 武功山 天气真的很多变，快到达金顶时，我们只是去上了个厕所，瞬间下起了倾盆大雨。
雨停后，上去看到的金顶也是一片大雾，没什么看头，继续走，一路上的风景随着天气的好转，越来越让人赏心悦目，特别是这满眼望去被雨雾笼罩的绿油油青草地。
经过吊马庄，路过帐篷基地，我们继续往前走，而这一段路是我觉得第一天最美的地方。
眼前一座又一座蜿蜒曲折的山头，在我眼中尽是柔美的曲线，伴随着黄昏渐近的柔和光线，以及 日本动漫中青蓝色天空，真的是每走一步都觉得舒心。
看着眼前越来越近的山庄，我们要前往落脚的是汇锦山庄，但是翻过了一个又一个的山头还没有到，看似近在咫尺，却远在天边，觉得比后来的绝望坡还要绝望，可能绝望坡我们是下行嘻嘻。
天要黑了，我们要赶路，所以觉得遥远。
到达汇锦山庄后，天气突变起大雾，日落是看不了了，在客栈里吃饭休息。山上条件有限，电是限时供应，一百块包吃住，我觉得还是可以的。
这里便是汇锦山庄了

Day2 我所乐于挥霍的时间

武功山 日出
看着天空从蓝变橙变红变黄，眼前的山与树静止着，与我们一样，等待太阳的光辉。我已经很久没有这样静静看过日出了，在早些年旅行的时候还很有激情和一大帮朋友早起看日出，现在大概是因为摄影让我愿意早起吧哈哈，去追逐日月星辰，乐此不疲。
所有你乐于挥霍的时间，都不能算作浪费， 比如 早起等一次日出。（贺大胆的文案）
今天的我是打算在山上休息好好的看风景，因为不想太急，我是来看风景的，不是来赶路的，后来他们俩也留了下来，大概没有想过要走完全程的我，也完成了这一次徒步之旅。
周六遇到很多来徒步的
看完日出回去补觉，心姐出去逛，没躺多久就被贺大胆大声呼唤我的名字把我喊起来……
心中一万个草泥马奔过……

我在武功山上拍婚纱

有的是时间，拉上我的徒弟贺大胆到山庄的后头山上拍摄我的婚纱照。
关于我的婚纱旅行已经拍了几个地方了，这次暑假在 贵阳 拍了之后，后面都没再穿。
武功山 的景色那么好，当然不想错过啊。我的婚纱旅行是带着婚纱，与路上相遇的人拍婚纱照，期待与你相遇。
之前穿婚纱走过的地方
拍完照后躺在山上的石头等待日落，如果生活的日子能够一直这么惬意就好了，脑袋放空，没有烦恼，呼吸自由的空气。
迷恋旅途，流连于自然中，只是想做一个内心纯粹的孩童。
武功山 日落

Day3 穿行于原始林木中

今天的日出没有昨天的好看，早饭后开始我们的第二段行程。
汇锦山庄出来后十几分钟便是绝望坡，我们是下坡，没有那么绝望，但是前方一个又一个的山，翻完之后还是挺绝望的，但这些远远比不上发云界之后的那一段路！
今天的行程基本是一直在走路，我连相机都懒得拿出来了。
发云界往后，有一段很舒适惬意的路，穿梭在茂密的草丛中，有种电影《蝴蝶》里，小女孩和爷爷在山间行走的错觉。
紧接着天气又开始变幻，开始下雨，而此时我们正在下一个陡坡，随后进入一个又一个密林，眼前再无高山草甸，穿行在看不见前方的原始林木中。
这里下去之后，再也没有这样的风景了
这一段看不见前路的方向，还有错综复杂，上行下穿的石头和烂泥路，给人迷失的感觉。
而心姐的大腿也在这里抽筋，走路缓慢，在高原上徒步叱咤风云（我夸张了哈哈）的心姐，没想到会在武功山 大腿抽了筋，多亏了还有一个男生可以帮忙背背包。
发云界之后的路基本上是这样的
我们的目的是到达新沈家大院落脚，还未到时，我们看见了一个小木屋，正好有人家，我们也都挺累的了，便在此留宿。这种小木屋很有深隐于林中之感，条件不好是次要，只怕是很少有人能耐得住这样的孤寂感。
住宿的小木屋

Day4 我们那多走的6381级台阶

我以为今天的路我们会走得很轻松，毕竟离沈家大院没有多远的距离，从小木屋走了一小时左右便看到了沈家大院。
一小时左右到羊狮幕后，就可乘坐小火车下山了。
但是我们没有看到小火车啊！鬼知道我们是怎么走到台阶那的，走了6000多级台阶下去的！！
比较痛苦的是心姐，拖着抽筋的大腿下台阶，真的是很不容易了。
如果上天给我一次重来的机会，我一定也会好好看攻略的，不至于大家都走错路。
下山的时候我们已经是在明月山景区了，想再走小路去坐小火车，无果，三人找车去高铁站，他俩坐高铁回 长沙 ，我继续坐车回沈子村，因为老娘的登山包放那呢。
没过多久，贺大胆退了票打滴滴过来和我去沈子村，于是我们回山水户外的小泳池玩水去了。
感谢捡我的心姐和贺大胆，也感谢我把自己的路，走成了别人眼中的风景，第一次徒步，就会有第二次第三次……希望未来我会去徒步更多的地方。虽然这一次 武功山 徒步我真的准备不充足，也没有任何的经验，所幸的是一路有伴，也没有遇到危险，感恩。

武功山三日徒步Tips

路线：
Day1 沈子村山水户外驿站—金顶—吊马庄—汇锦山庄
Day2 汇锦山庄—绝望坡—发云界—新沈家大院（也可在前面一点的小木屋落脚）
Day3 新沈家大院—沈家大院—羊狮幕—小火车或索道下山
装备：
请大家一定要有户外装备，不要学我！！！走起来舒不舒服的痛苦只有自己默默忍受！
山上有客栈，但前往的时候最好打电话确认是否开门，条件没有那么好，如果有帐篷，自己扎营也是不错的。
