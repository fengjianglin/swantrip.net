三亚，看不够、吃不够、玩不够的一座城

游记 亲子 海岛 自助游 2019/05/09
如果说， 中国 有哪一座城市可以让人产生一去还想再去的心境，在我的内心深处，这座城市非 三亚 莫属。
三亚 ，一年四季皆如春天般灿烂，皆如秋天让人有丰收般的喜悦，这里有蓝天白云，汪洋大海，碧海轮船，有无尽的美景，无尽的美食，让人永远都有来不够的喜欢。

行程

出发地： 广州 
目的地： 三亚 
交通工具：火车或飞机，飞机建议避开3-4月，因为此时 广州 多雨，飞机容易延误。
此次从 广州 到 三亚 ，其实也算是一场说走就走的旅行。有朋友说想去 三亚 玩，我说，可以。
然后，我们便马上买了车票。
再没有这样利落的了。
此次行程也并没有提前规划，只是走到哪里便是哪里。
比较随意。但是依然玩得很开心。

美丽之冠

美丽之冠 大树 酒店有6666间客房，但是这些客房由不同的酒店名称来经营，我们预订的早安 大树 酒店，平均一晚上150多元左右，标间，里面放置着2张1.2米的床。
前台服务人员态度很热情，我们在网上预订的房间，在前台办好手续入住，押金退房后会自动退回，退房的时候只需要把房卡留在房间就可以，假如住了一个晚上，还想再入住，只需要给前台打个电话说要续住就可以。很方便。
早安 大树 酒店每天24小时都有电瓶车接送，需要去哪一栋跟电瓶车司机说一声就可以。
不过，酒店房间也有缺点，必竟价格便宜，房间比较MINI，厕所也比较MINI，假如带着小孩住在这里，建议看好小孩不要在阳台上玩，因为我们住的楼层比较高，再加上自己也有一些恐高，站在阳台向外面看，还是觉得有些恐怖的。
酒店旁边有美丽之冠表演剧场，不过我们并没有去看。酒店附近有一条河，站在桥上拍酒店位置最佳，夜晚的美丽之冠也很漂亮，美轮美奂的灯光别有一种风情。
白天的 三亚 ，蓝天和白云总是准时出现，让人的心情不自觉地变得很开心。也许，喜欢 三亚 的理由，也与 三亚 的天气有关吧。

大东海

大东海 是离 三亚 市区最近的一片海。我们在的士上跟司机说要去 大东海 的时候，跟司机开玩笑要去
‘有几个棒槌的海；，司机小伙来自于 东北 ，可以说，在 三亚 的几天里，遇到的的士司机里面，90%的人都是 东北 人。司机听到“棒槌“笑个不停，说人家那是菠萝。
好吧，就当它是菠萝吧。
大东海 的海岸线很长，曾经以为 大东海 属于脏乱差的海，后来发现没有找对位置， 大东海 干净的海滩也是有的，并且人并不算多。
所以，假如想看大海，不一定要去 亚龙湾 或 三亚湾 ，其实 大东海 也很漂亮。
从1号港湾城过去， 大东海 的沙滩干净又舒服的。

一块豆腐

在 大东海 ，还真有一个菠萝。被称为1号港湾城。
我们打的经过的时候，看到 大东海 附近有个大菠萝，感觉很新奇，就进去逛了逛，3楼和4楼都是美食，一上电梯一抬头就发现了这家店。一块豆腐，名字起的很有意思。
店里生意不错，店面虽然不是特别大，但是感觉很温馨和安静，2点多都还有很多客人吃饭。我们随意点了几个，造型和摆盘很有西餐的风范，厨房也都是开放式的，全程可以看到厨师做菜。当时走到门口的时候，发现店里所有的植物每一片叶子都干干净净，我还奇怪怎么不落灰，后来发现，店里连植物都有专人伺候。
厨房的吧台上还养着豆苗，客人点就可以现割，实在是太好玩了。
在门口的玻璃房还能全程围观豆腐制作的整个过程，说实话，做豆腐我只在小时候看过，并且还只是看了个片段，在这里全场都可以看，原来豆腐也不那么容易做的，要先磨豆子，然后过滤，再煮开，这个时候就是豆花，豆花放温之后，再点上卤水，然后再放到豆腐槽里，然后再经过压制，豆花就会凝结成豆腐。当时不明白豆腐为什么分老豆腐和嫩豆腐，问了师傅才知道，豆腐压制太干拿出来时间长就会变老，为了保证店里客人吃到口感和鲜嫩的豆腐，店里每天都会现做豆腐。
真是佩服这种精神。
店里用豆腐做的菜品很多，而且味道极赞，值得一试。
地址： 三亚 大东海 区1号港湾城店

大悦洋海鲜自助料理

在大菠萝的楼上，还有一家海鲜自助料理很值得一去。
我们当初来 三亚 ，就一直想着要去吃海鲜，在一块豆腐店吃饭的时候，正好看到附近有一家海鲜料理，那么，既然来到了 大东海 ，吃一顿海鲜是必须的历程。
因为听说能看到海景。晚上便选择了这里吃饭。进店后，果然跟传说中的一样。透过超大的落地玻璃窗一抬头就是大海，甚至还可以出去在外面吃，一边看着碧海蓝天，一边吹着海风，感觉实在是太好了。
本来在 三亚 就计划吃海鲜，但是没有想到这里的海鲜品种实在是太丰盛了，而且超级肥美，还特别新鲜。
店里主打海鲜小火锅，一人一锅，一进门就是活蹦乱跳的新鲜海鲜，想吃就去捞着涮，任吃的感觉很任性啊。
冰鲜海货也都是个头很大，肉质新鲜，各种品种叫不出名字没关系，还细心的标注了名字，为这种贴心点个赞。
我们去的早，去的时候还没有正式营业，还在摆盘中，但是店里的工作人员依然很热情的招待，让我们先坐一会看看海景，感觉真的很亲切。
除了丰盛的海鲜自助，还有寿司，烤肉等等，酒水的可选择性也超级多，还特意设置了儿童美食区，准备了很丰富的小孩子喜欢的美食。
最后当然是捧着肚子扶墙出了。

 亚龙湾热带天堂森林公园

我们打的从 三亚 市区到达 亚龙湾 热带天堂森林公园的时候已经下午4点半了，在网上查了一下，5点半就停止游客放行。不过看看天色还早，艳 阳高 照，我们紧赶慢赶让司机帮忙买了票。好像 三亚 的的士司机买票可以优惠一点。
要游玩这里必须要乘坐观光车，因为山路狭窄，除了景区内的观光车，私家车只能停放在停车场而不可以进入景区内的。
我们只购买了车票和门票，玻璃桥和索桥门票都没有买。因此一路上到山顶，假如有买了其他景点的票，可以到站点上下排队上车。
山顶的风景应该属于最美的，站在最高处俯瞰，能够望到大海，以及森林公园之外的风景。
山顶有一个位置雕塑着一条龙，龙前围着很多游客，大部分人都在做着同一个动物：托龙。据说这个动作在抖音上很火，因此成为了一个网红打卡景点。
不过，最吸引我的莫过于站在观景平台上看大海。
三亚 的大海有一种天然的美，让人总会产生一种心矿神怡的感受。当大家都在做着托龙动作拍着视频的时候，我们来到观景平台跟大海合影。
这种时候，大海在眼睛里，在镜头里，也在心里。
我们按线路提示找到了拍摄《非诚勿扰》的一个景点：男女主角试婚的小木屋。
不过门票需要30元。
我们没有进去看，而是继续沿着山顶的道路前进。
在另外一侧，建有一片建筑，据说是 三亚 旅拍的打卡景点之一。
很多人选择在这里拍摄婚纱照。
最近几年，在 三亚 拍摄婚纱照的业务非常好，只要在优美的风景之地，必会遇到几位拿着相机，随时准备给游客拍摄美照的摄影师。这不能不说，是 三亚 的美，美到了全世界人民的心里，于是，人们来到这里，带动了这里摄影业的发展。
亚龙湾 热带天堂森林公园有一片山间的度假木屋，被称为“鸟巢“。假如用长焦拍摄的话会特别好看。虽然是度假小木屋，远远看着却真的很像在山间木间搭建的鸟巢的，给人很浪漫的感觉。
山顶上可以看的风景很多，例如还有魔幻影院，以及各种卡通气息浓厚的商铺。带着孩子在这里游玩，更让人能够感受到童真的乐趣。
我们在山顶走走停停，吃吃喝喝，真有一种舍不得离开的感觉。假如下次有机会再来这里游玩，一定要买全套票，不留遗憾地离开。

林姐香味海鲜

在 三亚 ，有一个地方，人人都熟悉，那便是第一市场。
第一市场是一个可以自己购买海鲜然后加工的地方。
第一市场附近最红火的海鲜店就数林姐了，开了两个档口，一边是新鲜的海货，一边是2层的用餐区。
海鲜价格跟第一市场的差不多，而且只只鲜活，明码标价，可以买店里的海鲜做，也可以在第一市场自己购买海鲜来店 里加 工，加工费也是明码标价，一目了然，让人感觉很放心。
海鲜的味道可以做多种选择，可以红烧可以清蒸，想吃什么味道都可以跟店里说，最后肯定会满意的。
我们人多，懒得去买海鲜，就自己在店里挑选了，上菜的速度很快，做出来的味 道真 的很不错。
香辣蟹膏满肉肥，辣味适中，本来以为小孩怕辣不会吃，但是竟然被小孩们一抢而光，真是让人惊讶万分，刮目相看。
螺肉的肉质很紧实，但是很鲜嫩，保持着海鲜的原味，也可以蘸着酱吃，别有一番味道。
粉丝扇贝的肉质很肥美，蒜蓉也很入味，好好吃，一口气吃了好几个。
鲍鱼的个头也不小，吃鲍鱼感觉确实红烧的更好吃，为厨师点赞，特别喜欢。
店里还有各种鲜榨果汁，味道很不错，新鲜的果实榨出来，极其实在。
店里老板人感觉特别好，当时不知道店里的茶里面那个圆圆的东西是什么，因为喝起来感觉很好，老板还热情的告诉我们哪里可以买到。

崖乡海南小吃

第一市场附近有很多好吃的，我们闲逛的时候，无意间发现了这家小店。
去到的时候店才刚开门，于是幸运的成为了第一拨客人。
店铺的装修是那种明显的小吃风格的装修，有点像隐藏在巷子里的美酒，偶然闯入，没想到竟然来对了。
在 三亚 每天都遇到 东北 人，这里的老板和老板娘竟然是 海南 本地人，人超级热情的，很随和。
我们3大2小大概吃了90多，吃的超级饱。
传统清补凉真的跟其他店的味道不一样，在 海南 的几天里我们几乎每去一家店必点清补凉，唯独这里的特别好吃，味道有明显的不同。老板说因为不是用椰子粉冲的，而是真正的椰子汁，同伴一直对这个清补凉夸个不停。
海鲜粉的料真的很实惠啊，不过稍微有点咸了。
芒果炒冰真的真的真的超级超级好吃，小孩们都没停过勺子，我也是第一次吃到这样的炒冰，过程充满了乐趣，用勺子一层一层刮着吃。绵密，而且芒果十足。 海南 的芒果确实不错。
虾饼又香又脆，趁热吃简直美味的无法形容。本来想多买几个打包走，但是又觉得热着吃才不辜负厨师的心意。
我到现在都还惦记着这家的虾饼和清补凉呢。

三亚湾的免费海滩

在 三亚 ，一定要近距离看海。看哪一片海，我们纠结了很久， 大东海 靠近市区， 亚龙湾 离我们有些远， 三亚湾 便成为最合适的选择。
三亚湾 的海从市区一直延伸到效外，我们选择了 三亚湾 路的这片海。长长的海岸线，沙子细腻而且沙滩干净，重点是人少。
我们沿海边订了一间民宿。淡季，很容易订到。路边有公交车到第一市场以及市区其他的地方，交通很方便。
过了马路便是海。
特别喜欢 三亚 的天气，每天都是蓝天白云，让人的心情格外畅亮。第一天到 三亚湾 海边的时候，让的士司机开快点，因为太阳快落山了，不过，最终只看到了夕阳的落辉。
晚上，我们赤着脚在海边沿着沙滩走，那种感觉有描述不出来的美好，不带相机，什么也不想，仿佛真的是走在人世间的桃花源里，耳边是风浪声，脚下是细腻，人也没有那么喧闹，很想让时光停留。
海边有一片椰林，椰林很长，很富有“ 三亚 特征”。早上，很早便醒来，趁着太阳没有那么强烈，便又来到了海边。
早上的大海，在寂静之下，有着不可忽视的力量，吸引着人们观赏它，并爱上它。
当早晨慢慢苏醒，大海又开始慢慢退潮，仿佛在腾出一个位置给喜爱它的人，仿佛是用一个怀抱轻轻拥着爱着它的人。
假如，时间充足，用一整天的时间接触这片海，都不会感觉到腻烦， 三亚 的海，总是让人看不够，爱不够。
重点是，长长的海岸线，全程免费哟。

天涯海角

都知道 海南 是 中国 的最南端，就像都知道 三亚 有个叫天涯海角的景点一样。
交通：
可乘坐6、16、21、24、25、26、29路到天涯海角景区下车
门票：
淡季:68人民币 (5月1日-9月30日 )
旺季:81人民币 (10月1日-次年4月30日 )
可以说，天涯海角是 三亚 的网红景点，也可以说，天涯海角是每一位来 三亚 旅游的人必来打卡的旅游景点之一。
也许有的人不理解，为什么天涯海角这个景点不过是几块石头，却要收费。姑切把它作为一种象征的存在吧。天涯和海角两块石头是爱情的象征。代表着有情人带着彼此的爱来到了天之边缘，海之尽头，有一种“ 天长 地久”的喻意。好像很久以前，有一部电影或是电视剧便是女主角来到了这个地方，寻找爱人的爱。
天涯海角同样有一片美丽的大海。帆船与椰林，沙滩与海水，每一种元素都是人们最爱的。也难怪喜欢这里的人这么多。
在这里游玩，有一种灵魂在天上飞的感觉。景色怡人，心情舒爽，实在是太惬意了。
天涯海角不仅仅只是几块石头，一片大海，它是一种精神的寄托，代表着无数有情人的心意。

三亚千古情

三亚 千古情票价300元左右。景区动物园和冰雪世界11点开放，其他12点开放。
游玩路线参考：
冰雪世界-----爱情街-----科技馆----水乐园-----动物园-----六间房广场------千古情剧场
三亚 千古情景区位于 三亚 市迎宾路333号的宋城旅游区，开辟有冰雪乐园、科技馆、彩色动物园、千古情景区、水公园等区域。
游客进入景区，首先可免费换上宋朝的衣服，或扮作阿拉伯人，或扮作 印度 人，或穿上狂野的野兽服装，或穿上草裙……穿越千年，回到那个 丝绸之路 的年代，仿佛自己便是那西域之路上的一员，去探索千年之前曾经的辉煌盛世。
天香炉的火依然在热情燃烧，一丛花树映日而红，温暖的 三亚 带来不一样的冬天。作为身处南方常年不见雪的人，见到冰雪乐园便仿佛看到了雪的洁白。孩子更是激动不已，吵嚷着要去抓雪、堆雪人。
冰雪馆内温度约零下10度，我们在入口换上棉衣，给小孩租了护膝和棉鞋、耳套，越过头顶的“冰锥”，便仿佛进入了一个被冰雪覆盖的童话世界。圣诞老人、白雪公主、 北极 熊、波波池、溜冰道……甚至北极 探险，惟妙惟肖的立体冰雕活生生一个现实的冰雪世界。景区内的游玩，尤如从温柔可爱慢慢过渡到惊险刺激。最受欢迎的莫过于从螺旋梯上溜冰了，景区贴心地把大儿童、成人以及幼儿、小儿童的溜冰道区分开来，人坐在冰上，像滑滑梯一样顺势而下，速度奇快，引来阵阵尖叫。而这样惊险的举动，在冰雪馆只是其中的一个项目。冰原魅影打造的鬼屋，更是能让游客在5分钟之内汗毛倒竖，冷汗淋漓。作为我这样的胆小鬼自然是不敢体验的。
租鞋子10元，租手套等10元。棉衣免费。
从冰雪馆出来，便来到了最为美丽的椰风街和爱情街。绣球花在头顶飘摇，街道两边商铺林立，商品琳琅满目，手工制陶、腊染、沙画、草编……从前，人们物质贫乏，商船把丝绸、瓷器、茶叶、四大发明……带去了海外，又从海外带回来象、香料、琉璃、珍珠……正是由于 丝绸之路 把各个地方从无变为了有，才有了今日了繁华。让人仿佛回到了古代的 海南 ，回到了那个海风阵阵、喧闹非凡的港口。
三亚 千古情有很多好玩的表演以及动物园和快闪等。
千古情最精华的地方在于看表演。
千古情整场表演有灵魂有故事，让人心伤让人惊艳让人开心，由《落笔洞》、《鹿回头》、《冼夫人》、《海上丝路》、《鉴真东渡》和《美丽 三亚 》几个篇章组成，整场演出使用现代的声光电技术及三百九十多套舞台设备、3000吨海水、全世界最大的透 天水 膜、1000多个水雾喷头，带来身临其境的绝妙体验。
一幕幕荡气回肠的故事，让人看完表演后依然久久不愿离开。恒瓦依旧，时隔千年，那些飘荡在宋代的英雄和灵魂，仿佛随着桃花，在那条丝路上开了千遍万遍……

后记

 三亚 是一个让人去了还想再去的地方，虽然因为时间问题，很多想去的地方没有去，可是，正好为下次再来做个借口。
三亚 ，我还会再回来的。
