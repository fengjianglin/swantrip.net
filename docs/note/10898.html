成都二环美食图鉴

游记 小众 美食 自助游 2018/10/29

吃货之都

在国内，几乎每一座城市都有自己拿的出手的特色美食。但如果当我们若聊起美食林立的城市，可能就不会超过十个吧？划重点：美食林立！也就是说这座城市的美食多到人短时间内是来不及吃完的。这样的城市里数成都、西安、重庆这三座可以说是独一档的水平了，应该毫无争议的吧？其他的还有一些城市大家就是仁者见仁智者见智了。成都和西安除了是美食之城，也是两座历史悠久、旅游景点众多的城市，可能也正是因为悠久的历史和文化才能沉淀点如此优质的美食吧？优质到单单可以只为了美食就去一趟的城市。作为一位细心的处女座吃货，更加偏爱辣辣的成都一些，所以今年也来了两次成都，然后心思缜密的整理了七家钟意的餐厅，即便你来一个星期，每天一顿挨个打个卡，都不带重样的。而且这些丰富的美食也会给你的味蕾带来无限的惊喜。
它们都有两个共同的特点，也是十分适合像我一样的吃货游客。一是人气高，每家店的就餐高峰期都会排上长长的队伍，需要错峰就餐；二是这些餐厅无一例外都是在离市中心很近的地方，其中六家都是在二环以内，还有一家在二环边上，不但好找，也很省时间，绝对适合来旅行的人。这七家餐厅分别是：马路边边麻辣烫（致民路店）、成都吃客（致民路店）、豪虾传龙虾（春熙路总店）、泰式集渔海鲜火锅（太古里点）、大龙燚火锅（太古里店）、寻欢焖烤羊肉串（九眼桥店）以及醉西昌（鸿云店）。其中上面按照顺序的前两家是在一个地方，三到五家也是在一个地方，最后两个分别在一个地方。
那么，到底有多好吃多有特色，就跟随我的筷子来一探究竟吧！

马路边边麻辣烫

马路边边麻辣烫，如果你是第一次听到这个餐厅名字的话，可能会和我第一次听到一样，以为这只不过是一家路边摊吧？其实事情并没有这么简单。马路边边是一家极具情怀的串串店，而在成都光串串就有热串串、冷串串等各种类型的串串，外地人根本傻傻分不清楚，不管怎么样反正只要好吃就是好串串！综合了网评和推荐你就会发现马路边边麻辣烫（致民路店）是成都最热门的串串店，还有网友表示这里永远在排队。从市区坐任意地方乘坐地铁从新南门地铁站B口出站，步行约百余米即到。
在经历了漫长又激动的等位过程后，终于轮到了我们。进入店门后就能从发现四壁上都有着各种充满情怀的元素。即使我不是成都人，里面一些关于80后的记忆碎片也可以让我产生共鸣，深深的回忆一番。蓝桌子，小板凳让我想起了小学时候在街边吃馄饨的感觉，我们那里可没有串串，不如成都人来的幸福。
马路边边的串串都是放在大的展示冰柜里自取的，大多数的价格是从5毛到2元钱不等，吃完按照竹签结算，如果是1块钱的串串，上面会有两根签这样子，以此类推，就不容易弄错了。有些不容易成串的菜品，比如鸭血、毛肚、鹅肠等等会有相应的容器盛好，到时候结算也是一目了然。
锅底和蘸料也有选择，我个人比较喜欢经典锅， 也是大众喜欢的，蘸料有辣椒粉花生碎、蒜泥香菜等，口味相对来说不是很辣，一般人都可以接受。所以马路边边作为一个积极接地气的串串店，已经在全国很多地方开了连锁店，这也是全国人民对于它口味的一种肯定吧！至少我们这次一行四个小伙伴都吃的赞不绝口。
啥也别说了，几个人坐在小板凳上不停的烫着串串，彷佛永远不会吃饱一样，时间过的也特别的快。吃的尽兴时再来一瓶北冰洋，可舒服了。如果要说这里的人均消费，感觉幅度会有些大，放开吃了一百块我自己也吃的完，佛系一点，二三十块也能吃的舒服。反正要提醒各位的是，如果来马路边边，一定要选择错峰就餐，比如上午11点左右或者13点以后，这样人相对较少，下午也是，推荐17点前，20点后的时间段。

成都吃客

说到成都吃客，我可是太喜欢了，以至于近年来了两次成都都打卡了这家成都最热门的川菜馆。只不过我第一次去到的是离宽窄巷子不远的魁星楼店。第二次来成都发现成都吃客已经开了分店，而且就在上面说到的马路边边的隔壁。那么就让我们来这新开的致民路店一探究竟吧！不知道还会不会像魁星楼店一样，下午从四五点钟开始就已经排满了长长的队。
不过这次来成都吃客和上回有些不一样， 这次来的也吃的是午餐，大概将近中午12点来的，正值就餐高峰期，外面排满了等位的人群。没想到和隔壁马路边边的等客混合在一起，显得更加壮观了，哈哈。等了将近一个小时终于轮到我们了，迫不及待的冲上两楼的餐厅。新的成都吃客相比于魁星楼老店要宽敞许多，餐桌摆放错落有致，吃客们都在埋头狂吃的状态中不能自拔。
因为是第二次来了，对这里的招牌菜也是记忆犹新，就给小伙伴们充当了点菜员。因为吃客是一家创意川菜馆，所以提前做好功课还是很有必要的。下面几款是它们家的必点菜，不信你来的时候看一看，这里几乎所有的餐桌上都会有这么几道。裸奔坨坨虾：顾名思义就是把虾壳剥干净，然后放在了飘满辣椒的红油汤汁里，把虾壳剥掉的意义在于顾客吃着更方便，而且也让虾肉更加入味，专家都说了多吃虾壳并不能起到补钙的作用呢。特色烤鱼片：这种鱼片几乎是没有刺的，然后穿在竹签上，上面洒满了让人望而生畏的辣椒，但是只要你吃上一串，这麻辣并存的口味又会让你欲罢不能。而且鱼肉的肉质细腻，也是其它肉质所不能取代的。麻辣兔头：兔兔这么可爱，怎么可以吃兔兔？别装了，在成都吃兔头是一件再平常不过的事情了，但因为我属兔才真的没吃，但小伙伴们都吃的不亦乐乎，一度怂恿我破戒，我差点就答应了呢。还有装在酒瓶里的牛蛙，牛蛙的鲜美自然不用我多去夸，这份吃起来要比前面几样都要辣，喜欢吃辣的小伙伴建议尝试一下。
除了这些必点菜品，我自己也有一些偏爱， 毕竟我作为一名游客来过两次，也是有一定的发言权的。烤脑花，这个也是川菜中的经典，只是有些小伙伴可能接受不了，如果你吃的时候不要想着它是什么，轻松的吃下去，也许就会打开新世界的大门。兔头不吃，但我可以用鸭头代替，同样的入味，这种本来在家里就很爱吃的卤味，在吃客也是不容错过。再有就是麻辣黄喉，之前我认为黄喉只有在火锅里才是绝配，但这麻辣的吃法配合黄喉特别有嚼劲的口感，也是极好的。最后，成都吃客的菜品都是偏辣的，吃的时候大家可以点一些甜品尝一尝，这样也可以中和下辣度，毕竟我们不是在成都大辣椒的光环下成长的。还有一些海鲜类也是我喜欢的，价格非常实惠，也是很推荐的。

豪虾传龙虾

说起来也比较奇怪，在成都火锅店也很多，串串店有很多，大家可能会各有偏爱吧！但是说到吃小龙虾，在成都还真是豪虾传一家独大。所以很爱吃小龙虾的我也是两度光顾了这家店，来到的是位于春熙路太古里的郁金香总店。众所周知，太古里也是成都最繁华的地方，而豪虾传则是闹中取静，因为它位于楼上的最里面，穿过熙熙攘攘的人群，来到这里的全是本着豪虾传来的了，选择一个靠窗的好座位，看着下面的人群，特别舒适。
那么这家曾经上过芒果台《天天向上》的龙虾到底有什么特点呢。一大特点就是辣，因为有了第一次来的前车之鉴，这次我们点的两份龙虾都是微辣的，其中卤香味的应该是店里最招牌，点的人最多的口味了，微辣的卤味非常适合大众，丰富的香料在嘴里不停的变换着，咀嚼着，很快就能把一份给吃完，最后盘子里还有颗卤蛋呢。我们还点了份蒜泥龙虾，我是很喜欢吃蒜泥的，生的蒜也可以接受，但不喜欢吃北方的糖蒜以及在菜里烧的口感特别面的那种。既然遇到自己喜欢的蒜泥，那就要多吃才行，几乎一盘都被我承包了。
因为正值秋季的缘故，豪虾传除了虾兵，还推出了蟹将，也就是香辣蟹。在著名的阳澄湖大闸蟹的故乡苏州一带，大家吃蟹的方式都更文艺，口味也多以鲜香为主。而在豪虾传的香辣蟹则要带上手套大口去咬的，肥美的肉和黄遇上刺激的辣椒，哪还有心思细细品味呀？毫不夸张的说这样的蟹我一个人也可以承包一大盘。

泰式集渔海鲜火锅

成都的火锅有千千万万的火锅店，知名的，小众的，但说的上来的都是辣辣的老火锅。但就在豪虾传马路斜对面有一家火锅店，它不是红彤彤的辣油火锅，也不是烫毛肚鹅肠的中式火锅，而是充满了异域风情的泰式集渔海鲜火锅，在来之前也就在网上看到过这家店，第二次来成都的出发前就已经把它列入了计划名单。可能因为它轻松的就餐环境和独特的东南亚口味，让吃惯了辣火锅的成都热有了耳目一新的感觉，所以才会火起来吧？
集渔也是一家服务性很强的火锅店，我们锅底要的是泰国最有名的冬阴功味，服务人员还会拿来一颗新鲜的椰子来往里面加椰汁，也使得锅底更加鲜甜可口。在等锅的同时，也会有其他的工作人员推小车来给吃客送甜点之类的。
集渔吃的是泰式海鲜火锅，因此火锅里的烫菜也是以海鲜类为主的，推荐点它们家的套餐，这些人少的话就可以吃到多种类的海鲜了。我这次来集渔是情侣两个人，点了198的套餐，然后又单点了两颗鲍鱼，菜量差不多就够了，荤素都有。
集渔的堂弟你是完全不用担心它辣不辣的，如果想要吃辣的小伙伴可以另外配上蘸料就可以了。烫菜也比传统火锅要来的精致许多，有些东西甚至不知道如何小手的关系，不过集渔的小姐姐都都说会帮客人下菜的，比如虾滑这类比较难搞的，还有些菜它们会在烫好后用专用的器皿给捞起来放好，特别贴心。
集渔的存在绝对是太古里的一股清流，如果把它比作一位美女，集万千宠爱于一身。那周边的大龙燚这些传统火锅就像性格直爽的壮汉一般，形成了鲜明的对比。

大龙燚火锅

刚刚有说到大龙燚，就在集渔和豪虾传的对面就有，三家店也形成犄角之势。作为游客，你一天都在春熙路太古里附近瞎溜达的话，那么这三家店可以承包了你的午餐、晚餐和夜宵了。大龙燚的知名度现在已经相当高了，不但有很多的明星来吃过，现在也在全国迅速的开到了许多加盟店，也证明了它的实力。既然这么出名了，应该也不会有人再帮它的名字读错了，“燚”（字读yi（第四声）由四个火字组成，并非读两个火字的“炎”呀！
电梯上两楼就来到了大龙燚太古里店的餐厅里，它们家的就餐还是蛮智能化的，可以提前通过微信公众号预约，点餐也可以使用扫码手机点餐，这些大大节省了人力物力，顾客本身也更加随意方便。而且太古里店里的餐桌摆放的间距很大，即便是在热闹的大厅也并不算嘈杂，大家都可以在自己的节奏里吃着火锅，特别好。
在大龙燚也有几样自家的拿手菜是来这必点的，比如大刀毛肚，嫩牛肉，雪花牛肉、鹅肠等，这些新鲜的菜品上桌后。打开自己面前的一小罐麻油，然后加上蒜泥香菜等其他蘸料，就等锅底开，然后大口吃肉了。
因为这段有好几个小伙伴一起来的，为了照顾口味就点了鸳鸯锅。都说鸳鸯火锅是成都人最后的妥协，但是我们这次在大龙燚点的鸳鸯锅还是感觉到了深深的”恶意“，在我们老家鸳鸯锅都是那种太极形状的，两边一样大。而问了才知道，成都的鸳鸯锅，都是把不辣的放在中间一个小圈。不过这样的设计也蛮符合当地特色的，吃辣的多不吃辣的少，一个小圈就够放了，况且这么小的空间，放进去的肉也不怕捞不到，反而是一种”善意“的设计。而且小锅里的汤上桌烧开后可以先喝上一碗，极其鲜美。
吃完烫菜有两样几乎是我到成都火锅店必点的美食，一个是红糖糍粑，一个是冰粉。红糖糍粑的软糯和香甜真的是太难望了，而冰粉里的聊很足，碗底也是有红糖的，把它搅拌开来吃上一碗，简直是火锅的绝配。还有一点值得一提的是，按照大龙燚的品牌效应，这里的消费水平并不算高，注册会员后还会有不定期的活动等，人均几十元就可以吃上一顿。

寻欢焖烤羊肉串

吃烧烤可能是很多人吃夜宵的唯一选择，特别是天气还不冷的情况下。所以来成都，有一天我和朋友在住的酒店打掼蛋到了23点左右，感觉饿了！然后就寻思着弄点夜宵吃，在成都这样一个美食之城呢个，如果我们还选择吃泡面，叫外卖这些，岂不是对这座城市的不尊重？于是，我们在当地朋友的推荐下，去到了寻欢焖烤羊肉串（九眼桥店），周边也有许多的酒吧等夜店，沿着致民路，也就是前面说的马路边边麻辣烫那里，一直往里走大约五六百米，转个小弯就到了。
它们家的门面真的不算很大，甚至还同挂着另外人家的招牌。出于好奇就询问了老板面店在哪里，是不是之前转让的店铺？老板给我的答案是否定的，原来烤肉店刚开的时候就是小本经营，和这家面店合作的这间铺子，现在已经过去了好几年，烧烤店也在当地小有名气，但这间店铺仍然按照这个模式经营到了现在。寻欢烤肉之所以被很多当地人接受热捧，是因为它们家用的是焖罐烤肉， 绝非市场上的炭炉烧烤架。
碳炉烤肉已经很流行了，那为什么还要用焖罐呢？焖罐的烤肉又有什么优势呢？不能仅仅只是因为换了个工具去烤就被人追捧把？当然不是这么简单。焖烤的方式烤出来的肉质更加水润，多汁入味。并且炉内的温度更高，即便是大块的肉也都可以烤熟。自己动手烧烤过的小伙伴肯定都知道，传统的烧烤架很容易把肉的水分烤干，而且火力不均匀时还会出现，局部烤焦但有的地方还不熟的现象。看来这种焖烤法还真的特别有创意！因此在寻欢除了我们常见的烤串外，还会有很多大块的肉串可以选择。特别喜欢它们肉串里夹两块菠萝的创意，烧烤的焦香加上菠萝的酸甜，别有一番滋味。
吃烧烤喝啤酒！如果你爱喝酒的在这里可以点上几瓶店家自己特别有情怀的啤酒，给我的感觉就像网红白酒江某白呀！

醉西昌

醉西昌是成都一家必吃的特色烤肉店，地图搜一下，就会发现在成都市内也有很多的连锁店。这次我们一行人去到的是中粮鸿云店，位于汇源北路333号的万象城3楼。这家店也刚出了二环，但并不没有很远，从市区过去也有地铁可以乘坐，因此我们还是可以把它给划分到二环美食图鉴里的。
醉西昌的人气绝对也是要排队的，因此我们这回也是错峰来就餐的，晚上21：30左右来的，才没有等位，餐厅位于商场的三层，进去后发现这里并不压抑，原来有着大大的露台，再通过巧妙的装修，宽敞的同时也不会显得太空旷。然后我们找了个大圆桌就开始点餐了。
说到“醉”，来醉西昌必点的一份是醉虾。虾是生的小虾，泡在辣辣的汤汁里，随手揪起一只小虾，晶莹剔透。看到是生的虾，有的小伙伴有点不敢下口。不过没关系，醉虾因为调料的作用下是没有腥味的，而且生的虾肉质更鲜美。
醉西昌除了醉虾，那就是吃特色的烧烤了。除了一些常规的烤肉一位，有的特色菜客是别的地方不容易吃到的，有猪鼻筋、烤小肠等。服务人员也是会为我们在烧烤盘上操作，上面有着出烟口，我们吃饭的时候就有个良好的就餐环境了。
吃过一波，服务人员给我们把铁盘换了一块，接着烤其他的菜品。这样之前粘在铁盘上的料汁就不会影响到后面菜品的口味。它们家的菜量也非常的实在，肉都是大块的肉，烤肉后蘸上各式的蘸料，吃起来特别的过瘾。
饮料推荐酸角汁，在成都的好几个地方都喝到了这种饮料，酸甜爽口，在我们常州但是没有见过，图个新鲜，也不失美味。
