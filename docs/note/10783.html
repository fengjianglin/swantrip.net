成都之美食美景篇（二）

游记 人文游 摄影 美食 自助游 2018/11/05
Friend：“嗨，有时间没，一起见个面约个饭”
Me：“最近两天比较忙，你这次成都呆多久，选个时间一起吃饭”
Friend：“只呆三天，这次过来就是为了美食而来，打卡成都的网红美食店”
甩过来一份列出来的要去品尝的美味
Me：“这么匆忙，成都能让你留下来的只有美食了，哈哈”
曾几何时，我也是因为爱上了成都的美食而爱上这座城市，尤其是现在，打卡美食比打卡景点跑的还要快。
有数据显示成都居于中国美食之都排行榜之首，我想这些数据统计不是凭空而来的。
我不仅是一个旅行爱好者，更是一个十足的吃货，看到美食抬不动脚的那种，每去一个地方的美食街，边走边吃边砸吧嘴的绝对少不了我。
每一次的旅行，美食攻略是必做功课的，一般都是在美食聚集地的周边选择住宿，这貌似已经是我的惯性了，比如去厦门入住曾厝垵，西安入住挨着回民街等。
你是在什么样的契机下来到成都的呢？成都之景想必大家来之前已经计划好的，那就让我这个贪吃鬼给大家推荐一些成都的网红美食店吧。
不过呢，美食开篇之前我还是想先说一下成都的美景，毕竟我曾经也是冲着这些景点奔到成都来的。
 

宽窄巷子

交通指南：地铁4号线宽窄巷子站，出站就是景点。
火爆的宽窄巷子景点现在是每逢周末挤一挤，每逢假期挤三挤。我带朋友过去宽窄巷子挤累了就会去一个地方站在高处看入口处川流不息的游客，在宽窄巷的一个入口有一家乐萃咖啡店，乘坐电梯到3楼穿过长长的走廊走楼梯上去顶楼，可以俯视整个宽窄窄巷子，一砖一瓦的屋顶，有些瓦片缝中冒着青青的嫩绿，与地面看到的是截然不同的另一番景象。
​
​

339电视塔

交通指南：周边地铁距此均有1公里左右的路程，市二医院站（3号、4号线）、红星桥（3号线）、玉双路（4号线），可选择方便的公交车出行。
因为电视塔高339米，故得名为"成都339"，远远的就能看到直矗高空的塔，因其中间的熊猫，我们还称它为“熊猫塔”，白天可观成都整体风貌，夜晚可赏成都夜景之美。
遗憾的是晚上10:30就关闭了变幻多彩的塔灯，饭后朝着这个方向散步，还没都到地方就看到已熄灭的灯，下次提早点时间过来。
 

网红爬墙熊猫

在春熙路的IFS顶楼有一只可爱的大熊猫，从地面可看到这只熊熊正爬在IFS的墙壁上，到底是什么在吸引着它呢，不妨我们也去看一看。
从IFS商场内可直接乘坐直梯到达L7，出电梯很惊喜的发现，原来顶楼是一座漂亮的空中花园，看来熊熊也是很喜欢这里的环境呢。

塔子山公园

交通指南：地铁2号线塔子山公园站
人民公园是大家众所周知的，距离成都东站只有1个地铁站的塔子山公园也是一个好去处呢，如果距乘车时间很充裕的话不妨过来小逛一会。
九天楼为成都市一标志性壮丽的园林景观，坐落于塔子山园内浅丘顶部，其名取自诗仙李白名诗《登锦城散花楼》中“今来一登望，如上九天游”，楼高达70米，远看似塔，近看似楼。
说完景点，我们再来唠唠成都的网红美食店吧。
 
西昌烧烤已是全国烧烤七大门派之一，曾上过上湖南卫视《天天向上》

醉西昌（鸿云店）

地标：汇源北路333号中粮鸿云13幢3楼(万象城)
楼下墙壁上的“醉西昌”三个大字的招牌特别的抢眼，乘坐电梯到三楼出电梯就是美味的醉西昌了。
就餐环境：分为室内和室外。室内环境宽敞，室外是露天的，环境很像是空中花园的感觉， 着盛开的鲜艳的花儿。想要坐在室外美美的吃烧烤必须的必要早点来，来晚了就只能坐室内或者排位了。
服务：这里的服务态度是相当的好，我们刚走进来，服务员就热情的接待了，就餐期间有什么需要都很及时的响应。
菜品：这里的菜品很丰富，咨询了老板点的都是这里的特色菜。
竹签大串：长长的签子很方便，大块大块的香猪肉，大口吃肉大口喝酒。
西昌邛海醉虾：看着虾子在里面蹦蹦跳跳的慢慢醉下去，中间打开碗盖后跳出了两只小虾。
酸辣烤大茄：本来以为茄子也是自己烤的，原来是烤好了的一份端上来就可以直接吃，酸酸辣辣的配料食欲大开。
西昌猪小肠：烤肥肠是需要技术的，如果烤太久肉质会变硬，烤到刚刚好的火候蘸上干的辣椒面吃着特别香。
蛋糕纸杯鹌鹑蛋：当端上来一排的鹌鹑蛋事，很像是摆放着的一排盛开的花儿，尤其是烤熟后，金黄的蛋黄更像是盛开的花心蕊特别漂亮。
酸菜五花肉：烤熟的五花肉油爆爆的裹上生菜，油而不腻特爽口。
另外我们还点了烤脑花（麻辣）、全味鸡中翅、大凉山煮小洋芋等菜品。
 
集渔是一直爆红我朋友圈的泰式海鲜火锅，因为爱吃海鲜所以爱上了海鲜火锅，因为喜欢酸辣，所以喜欢上了泰式的冬阴功汤。集渔是综合了我所有喜爱的口味，过来打卡是理所当然的。

集渔泰式海鲜火锅（太古里店）

坐标：太古里（下东大街169号晶融汇2期洋展购1号门），优越的地理位置交通特别方便。
十一点半开始营业，在未开始营业时门口已经排起了长隆，而且直到下午两点半依然还有很多客人在排队，生意火爆到不行。
楼上靠窗户的位置特别舒服，宽敞而舒适的座椅加上明亮的光线，为这顿美餐增加了更愉悦的心情。
服务：服务员根据我们的人数建议了合适的套餐，很详细的介绍了店铺的其他特色菜品，在帮忙煮食期间，更是服务细致周到。
菜品：点了招牌鲜料冬阴功汤锅，冬阴功汤微酸微辣很开胃，加了椰汁味更美。每次吃冬阴功汤锅，我都会喝上两碗汤的。
店里有六种口味的蘸碟，有两份蘸碟口感不辣，可根据自己的口味选择。
四人份套餐分量特别的充足，大活虾+天鹅蛋+竹荪镶虾+鱼滑+活鲍鱼+生蚝+海蟹+虾滑+肥牛等，各种海鲜的汇集，特别适合我这种情钟于海鲜的吃货。
咖喱蟹：香浓的咖喱完全融于蟹肉，那种浓郁的香真是爱不释口。
这里还有无限量供应的小吃、甜品，有酸梅汁、西米露、布丁等。
集渔是我最爱的泰式火锅。
 
说完泰式火锅，也来一个川味火锅推荐。

如在蜀老火锅（玉林店）

地标：玉林南路15号玉林生活广场一楼，距离赵雷《成都》歌曲中的小酒馆不远，吃完火锅散步去小酒馆坐坐也是一种不错的选择。
第一次过来被门外的排队阵仗给吓到了，屋内高朋满座，屋外也不例外，排队的客人队伍也是很庞大，所以第二次过来我就选择了中午，机智如我。
门外那个极具年代感的自行车超适合拍照，过来的朋友可以穿上搭配的衣服既可以吃美食又可以拍美照，两全其美了。
这里的特色锅底是奔驰锅，这次朋友选择了红锅。
铺盖毛肚：口感特别脆，在火锅里稍微涮烫个七八秒就可以入口了。
蜀小碗鹌鹑蛋、双椒千层是可以直接吃的，千层肚的味道辣到过瘾。坨坨牛肉放在锅里煮一下就可以吃了。
红酒浸肥牛：很特别的一道菜，盘子中央放置着一杯倒立的红酒，周围是一圈薄薄的牛肉片，待红酒慢慢浸入牛肉，放入锅中煮一下就OK了。
大刀腰片：腰片也是吃火锅很受欢迎的一道菜，最爱的还是蘸着干辣椒面吃。
茴香现炸酥肉：外酥里嫩，肥瘦相间。
上次的凉糕，这次的焦糖豆花更好吃，入口即化，入喉淡淡的甜味。
 
来成都怎能不吃串串呢，除了遍地开花的钢管厂串串，这里推荐一家排队排到口水流的网红串串。

马路边边麻辣烫（致民路店）

地标：致民路48号，距离新南门地铁站B口出步行两三分钟的路程。
火爆的生意在晚上九点钟都还有客人在排队，看这情况味道不摆了就知道绝对错不了。
老味道，最成都。
脱了漆的搪瓷杯，小小的蓝色方木桌，带有背靠的木凳椅，七八十年代的墙报，仿佛回到了儿时的农村，一切看起来是那么亲切。
掉了漆的破旧墙壁上的计划生育宣传语，与小时候老家马路边邻居家墙壁上的一模一样。
店铺里立体的墙绘，马路边吃火锅的既视感，完全是一幅吃饱了的小孩子在自由的玩耍，大人们坐在一起边吃边唠嗑的其乐融融的画面，地气十足。
马路边边除了美味的串串，我最爱的还有这里的碗碗菜品，之前吃过的碗碗土豆心中一直在记挂着。
​
这次点了的兔腰全被香辣的辣椒面包裹着，分量充足。
串串是自助取菜，一排排的荤素菜任选任挑。
油炸糖果，每次看到都会有一种深深的怀旧心情。
店内还提供自助充电宝，很是方便。
成都的美食实在是太多了，本文推荐最后一家美食店，川菜炒菜系列。

成都吃客(致民路店)

地标：致民路48号，距离新南门地铁站B口出步行两三分钟的路程。
店铺很有特色，每张餐桌隔墙上均是成都的一个景点，有创意。
楼梯间红色背景墙上的碗盘碟勺，上方悬吊着簸箕竹筐，错落有致的木制方块字，装饰的设计很不错。
裸奔坨坨虾：看到漂着诱人的辣椒就知道味道不会错，里面还有香芋头。
纸上焗鲜鲍：又一次吃豪华泡面，海鲜鲍鱼方便面，高大上吆。
麻辣兔头：怎么可以吃兔兔呢？去别的地方可以不吃，来成都是一定要尝一下四川传统名小吃兔头。
圣女果搭配桂花羹，甜甜可口，猜猜它好听的名字是什么呢？
粑粑鸭脑壳：辣味在舌尖跳跃，依旧停不下来对它的爱。
雀巢掌中宝：很有特色的一道菜，树枝上有一个鸟巢，里面盛装着诱人的美食。从菜品的外观设计到内在的舌尖美味，都给人留下深刻的印象。
​
甜品：搭配了红豆的豆花，融到心里的甜，朋友特喜欢盛装这甜品的小碗。
从菜品到服务，对这里一百分的满意。
成都的美食美景会使人舍不得离开，本文的美食美景介绍先到这里，请等待后续的分享……
