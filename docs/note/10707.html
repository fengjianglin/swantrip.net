这次的西藏之旅已有一个月，平日大部分时间在拉萨，去了一趟山南两天，林芝四天。
A：拉萨
拉萨周边景点很多很美的，强烈推荐：
1.帕邦喀。交通路线：公交车“娘热乡政府”下车后，直接百度地图，徒步2小时即到达帕邦喀山顶。
2.拉鲁湿地，这是一个封闭的保护区。如果有缘，愿你见到布宫在水中的倒影这幅画面。
3.市郊的扎叶巴寺、楚布寺、甘丹寺、热振寺等等寺庙，可以在早上五六点到宇拓路珠峰伟业路口搭车。每天早上都有的，大部分都是天亮前就出发的藏民。
4.特别喜欢每天闲着没事就去泡甜茶馆，转经，以及大昭寺广场晒太阳。
推荐茶馆：老光明甜茶馆，就在光明港琼茶馆对面那条巷子直走右边即到。
5.拉萨八廓街大街小巷里都有大大小小的很多寺庙，小寺庙都是不用钱的，都可以进去转转。就比如清政府斜对面也是两座小寺庙，但旅行的人们儿，大部分都忽略了。
6.这个季节三四月来到拉萨，床位价格大部分都在20元左右，非常便宜，环境也不错。如果可以尽量住在八廓街里头，每天穿梭在拉萨的老城里感觉很舒服的。
7.推荐几家拉萨好吃的地方，吉日二巷东素饭堂，布宫正门右手边酸奶坊，八朗学（面对东措左手边）巷子里的安多诺增藏餐、这条巷子也有一间酸奶坊、以及左手边第一间藏餐的快餐特别好吃。新平措上面的老鱼饭局，能够直接看到宏伟的布达拉宫。
B：林芝
去赏了一圈桃花，路线：拉萨—八一镇—雅鲁藏布江大峡谷深处达林村—八一镇—拉萨
八一镇住的是平措，新开的，热水、床铺都特别给力，一个床位40元。
达林村住的是藏民家，农家乐，包早晚餐一晚一共110元。
雅鲁藏布江门票260-290，特别是在林芝桃花盛开的季节里查得还是非常严的。
如果想要逃票还是得当地人带进索松村，大概100元一个人。
我走了4天，来回蹭的朋友车都很便宜，加上路上搭了周老师的摩托车，
一共大概只花了700元不到，花费不多但玩得很爽，所以说不要老是抨击“穷游”。
我没有吃差，也没有住很差，不就是跟人合住的青旅床位房，不就是搭了车但多少给了点油钱。
C：山南
山南才被成为藏文化的发源地，西藏的第一座宫殿第一块农田第一座寺庙全都在山南。
找了个周末来到山南，姐姐开着车载我去雍布拉康与昌珠寺。
从拉萨到山南泽当镇大概两个半小时，车费60元，镇上有直接到雍布拉康和昌珠寺的公交车。
山南的住宿有些贵，普通宾馆大部分也都在200元左右，而且并没有青旅哦。
但是此番去山南却燃起我对山南的兴趣，措美县、扎日神山等等都开始列入我的目标清单里。
好了，开始进入正题。

A《林芝春色醉迷人，世间能得几度闻》

——雅鲁藏布江大峡谷的桃花四天游
林芝春色醉迷人 ，世间能得几度闻。吾欲化作相思鸟，长住此间不欲还。
—— by 骑着摩托车搭我的周老师
青稞、桃花、沙漠、雪山、蓝天、白云
林芝桃花，我想了整整五年，每一年的春天到来都想来看一眼它。终于，如愿以偿，在这个最美的时间里。林芝的春天应该是中国醉美的春天，蓝天白云下的雪山，近处刚刚发绿的青稞田与盛开的桃花构成的景色让人久久不能忘怀。
米拉山口上的经幡阵
Day1，2016年3月31日
木子刚好要从拉萨到林芝，于是约了几个小伙伴一起拼车走起。我们的车爬上米拉山口，下着大雪，一会儿功夫身上满是小雪球，在山顶上耍了一小时有余，下山后，一路到八一镇，路两旁便是数不胜数的桃花。早上九点从拉萨市区出发，晚上七点到达八一镇。现如今拉萨到林芝的高速公路通了一大半，虽还是有限速，但路况好了太多，不久米拉山口的隧道也即将开通。
路边的风景，美若仙境。
在平措康桑青旅住了一晚的床位，不得不说他们家的热水太给力了。第二天一早，小伙伴想就近去嘎拉村，而我想去波密看桃花沟又或者深入大峡谷，于是一个人独自背着包出发。那也只能搭车了，徒步走在318国道上，心想搭哪个方向的车就去哪吧。
Day2，2016年4月1日
第一辆搭的是索松村的扎西，他送我到色季拉山口山脚下，说他要去河边修船了。如若可以回索松村再来搭我进去，我说，那要不你带我到河边吧。找到一个满是桃花的山坡，但是并没有什么看头。
继续在路边搭车，第二辆搭的是两位菜农，他们进苯日神山不远处买菜回拉萨卖。经过苯日神山的关卡，管理员愣是让我买票，我说我真的只是经过啊……两位菜农跟门票大哥说我一个姑娘徒步来的照顾一下呗，于是就放我过了，免了70元门票。
下了车之后，在满是灰尘的没有修好的狭窄的路上走，来了一辆摩托车，恩，就是它带着我走了2天的路，来自拉萨的周老师开着车带我，远至雅鲁藏布江大峡谷最里面没路了的尽头、多至前往大峡谷的河两边的路通通走了一遍……
由于摩托车方便停靠，我们俩几乎不错过任何一幅画面，觉得很是不错的角度便会停下来咔咔咔拍下来。

从这里开始进入了非常精华的一段旅程。我们来到大峡谷的门口已经是下午6点，估计景区门票也快下班了没那么严，当地人问我们是否进去，一个人100元带进村里。雅鲁藏布江大峡谷的门票是260-290元，其中30元是一顿午饭的钱。平时是肯定逃不了票，跟当地人的车进去索松村或者达林村一般就可以不用买票。但是若前往直白村那一块是肯定要买大峡谷门票的。
如果说索松村是人间仙境，那么达林村可以算得上是秘境了。从索松村再往里走7公里左右一个鲜有人知的村庄，它也是漫山遍野的桃花，但由于海拔较高开花时间较晚。我这边有达林村扎西的电话，他家可以住宿吃饭。一人一天110元，含一晚住宿+一早一晚。若在这个季节要前往索松村可给他电话，让他带你进去给个百八十块。但是一定要注意必须在早晚到达大峡谷，别大中午人家卖门票的值班时来。可加我微信9603887要电话。

Day3，2016年4月2日
凌晨四点半，扎西家的鸡开始在叫起床。清晨七点，天蒙蒙亮推开窗，天哪南迦巴瓦就在我面前。曾两次经过色季拉山口都未曾看过它的容颜，我已经做好与南迦巴瓦没有缘分的准备，却没想一早就给我来了个惊喜。在扎西家门口有块大地，有非常大的视野看着南迦巴瓦峰。
南迦巴瓦峰，中国最美山峰排名第一。海拔7782米，位居世界最高峰第15位，排在它之前的14座高山全为海拔8000米级，南迦巴瓦峰三角形主峰终年积雪，终日云雾缭绕，极少展露真面目。能看到南迦巴瓦主峰的几率极小，故其被戏称为“羞女峰”。

B《拉萨小时光》

——在拉萨一周记
喝过的美酒都忘记了，
只有青稞酒忘不了，
去过的地方都忘记了，
只有拉萨忘不了。
——朱哲琴《拉萨谣》
拉萨，你好，拉萨，我又回来了。
我知道我总会再回来，但是不知道这次回来得这么仓促。来时一张火车票，便带我回来圣城拉萨。在这个日光城里，每天最不缺的就是阳光。在午休或是周末，我涂上防晒就这样面对着太阳。晒黑，那又怎么样，我多么渴望在拉萨晒太阳。这一回，满打满算，我终于在拉萨呆够了春夏秋冬，就是怎么的也等不来一场大雪。
老拉萨人说，他是拉萨的，那就是以大小昭寺八廓街一带的人们。现在的拉萨比当年大了许多，可信徒们的虔诚依旧如初，拉萨的诱惑丝毫不减。
每日清晨醒来，坐上2路公交车，下车的茶馆里点一壶5元2磅的甜茶加一碗5元的藏面就是我的早餐，等着9:30的上班铃响起，打卡。今天做的也许是西藏民主解放第一村的山南克松村的拍摄计划，或者写的是318最美景观大道的微信号，更有甚者是跟着剧组跑这里拍摄那里录节目。
我最喜欢的还是一个人静静地坐在大昭寺前看着朝圣者一起一伏地朝拜。人说，拉萨是一个可以放下自己的地方。净化心灵的话说得太多就zuo了，可当你深度了解这座城市，你会发现，他确实如此。
三月的大昭寺，只有稀疏的游人，更多的是当地人们在朝拜的身影。
信仰总是不分老少，他正虔诚的三叩一拜。
阿佳用佛珠来数数，看自己磕了几个长头。
旁边磕长头累了的阿佳，正在翻着经书阅读。（相机摄）
如果说拉萨是雪域高原的心脏，大昭寺是拉萨的心脏，那么大昭寺里供奉的释迦牟尼12岁等身佛像就是雪域高原的心中之心了。大昭寺广场总有数不胜数的朝圣者，在周末晒太阳的时光里更是此起彼伏。总这么看着，总觉得，这世上，真的有一个地方，虔诚与信仰每日每夜都在。
泡甜茶馆，已经不止是拉萨人乐此不疲的生活方式，许多“拉漂们”也同样每日上演着这段剧情。八廓街附近有许多甜茶馆，有名的当属新、老光明甜茶馆、仓姑寺甜茶、邦达仓大院甜茶馆。让我最觉可惜的便是，一杯七毛钱的甜茶已不复存在。我们再也不用准备毛票，也看不到阿佳拿着茶壶在帮我们倒茶的流利模样。
回到拉萨，闲下来第一件事，我要去喝甜茶。甜茶是属于茶馆的，这它要是在家里做出来的，喝的总没那种feel。藏面和炸土豆，青稞饼和酸萝卜，那都是绝配。
牛肉藏面，一碗5元。
太阳从边边角角洒落下来，人们在甜茶馆里话仙。
喝完甜茶，出来八廓街绕上一圈。在这条古老的转经道上，我们总能听到那护手板滑过花岗岩铺就的路面那“刷刷”的声音，我是极喜欢随着转经的人潮在八廓街上绕上一圈又一圈的，偶尔看到喜欢的商店还会往里观望上一番。
转经的人们。
朝圣者与小黄，毫不违和的画面。
这挂满经幡的旗杆叫塔青。
在这个三月，拉萨也有了春的气息，万物复苏，梅花、桃花开得正旺。沿着布达拉宫转了一圈，特别是在宗角禄康公园，满满的梅花盛开，一次又一次地在布宫面前都不得不感叹这么神奇的建筑，就这样在山上挺拔而出。

傍晚下班之后步行到对面的布宫里头，转经道上仍有许多在卖琳琅满目的商品地摊，我挑了一个喜欢的小包包，看着人们走过转经筒转过六字真言。
有一天偶遇这么强烈且有活力的光束，很是开心。
龙湾潭面前的布宫，有许多鸭子和鸽子嬉戏玩耍，水倒映着蓝天，远处的桃花粉红粉红的。
周末之余花了一天来到帕邦喀，一个一看就很喜欢的名字。徒步两小时的山路来到寺庙里，发现满寺庙都是老桃花树，想来再过半个月肯定美如仙境。正打算进殿朝拜，有一家子人刚好从摩托车上下来问我说是要朝拜吗，走我们带你。我才发现那位父亲下车时是失去双脚的，膝盖以下都没有了。可是他依然攀登着楼梯，走向殿内，用藏语同他爱人说这尊是什么佛。孩子约莫两三岁的样子，很小。这时候进来一位阿里的阿佳，阿佳问他们写着藏文的第一块六字真言玛尼石在哪个殿内。他们用藏语沟通了一会儿，与一家人告别，阿佳喊我跟她一起朝拜。
在藏区的寺庙里，我最喜欢也最幸运的是经常与藏族人一起朝拜佛堂，他们不只会告诉你这尊佛是什么，也会告诉你一些他们知道的关于这座寺庙的一些故事，还是挺有意思的。有一回在阿里的科迦寺，我在黑暗中见到一尊刻在石头上的佛，他们居然让我用手机拍下才能保佑我。这是很神奇的，通常来说一进寺庙，相机或手机肯定是最好别拿出来了。
这是在桑曲寺山顶拍的，这两个寺都可以远眺整个拉萨城。
帕邦喀最出名的是这一座建在大石头上的殿。
小黄不知道在干嘛，伴随着桃花还挺美的。
整座寺庙拥有许多老桃树，虽然还没到季节但这时候也已经很美了。
周日决心翻墙也要进拉鲁湿地，我实在太想念它了。第一次来到拉萨，第一眼看到的布达拉宫就是在拉鲁湿地，布宫那柔美的一面在我心底是永远忘怀不了的。有一句话怎么说来着：如果你有幸在年轻时到过拉萨，以后不管到哪去，拉萨的经历都会跟着你一生一世。
第一次第二次第三次，来了三次了。依旧念念不忘。
初春的拉鲁湿地，还是明显感受得到拉萨的干燥。夏季的拉鲁湿地，满眼望去的绿色太舒服。深秋的拉鲁湿地，那是五颜六色的世界。冬季的拉鲁湿地，满是荒凉缺水却依然让人喜爱。
小牛哥，你好啊。

这个季节，水快满到路面上来。看着很舒服。
群山倒影下的拉鲁湿地。
仿佛走过这条道，我就能抵达布宫。
在拉鲁湿地度过一个完美的下午，一个人，一本书，一壶茶。后来，一位小朋友来与我一起玩。初中的孩子，带着脚踏车翻墙，也是够了！我以为他会比我识路要带我翻墙出去时才发现那是一些沼泽地，那怎么行，我坚决不走。我说你跟我走，我带你走不需要拔山深水的出口好吗？
最后顺利出来曲米公园，他偶遇小伙伴，玩耍去，我前往老鱼饭店，与旅行群里说好的小伙伴们一块约了个有情调的晚餐。夜幕刚刚降临，外面的布达拉亮起了灯光，我为这个景色拿起相机，却不慎将镜头摔坏了。当时是非常难过的，但是没办法的是事情发生了就发生了，努力挣钱买新的吧！

C《山南，藏文化的摇篮》

——山南周末两日游
西藏山南
在漫长的历史岁月中，山南因拥有众多个“第一”而被公认为“西藏民族文化的摇篮”。如西藏第一位国王——聂赤赞普；第一座宫殿——雍布拉康；第一座佛堂——昌珠寺；第一块农田——索当；第一座寺庙——桑耶寺；第一部经书——邦贡恰加；第一部藏戏——巴嘎布等，均诞生在山南。
从雍布拉康认识起山南
我的17-40mm镜头摔坏了，巴桑给我发了一个红包。说。你什么时候来山南玩，我带你去雍布拉康。我说下周末我去山南。早有耳闻雍布拉康乃西藏第一座宫殿，怎能不来。
周五晚，阿志做了衙口的紫菜煎以及咸鱼，这估计是我在西藏唯一能吃到的关于衙口的东西。说什么也要来一趟堆龙，阳光新城，如果接受公司的安排我也应该住在这个小区里。但是离大昭寺布达拉实在太远，一趟超过四十分钟的路程，有违我来拉萨的初衷。很满足地吃到的家乡的东西，喝着拉萨啤酒不知不觉晚了也没车再回拉萨。
第二天一早，我直接前往西郊客运站，这大概是我第一次这么没底的出门。没有充电宝，没有热水杯，相机的电池只剩2格，我还是去山南了。
巴桑是山南人。我还在路上，她已经在客运站等我的到来，我与她相识于三年前的秋池客栈，她是一位记者。在内地生活了11年有余，初中高中大学，毕业后回来西藏。见到我时总会说想念沿海的潮湿，那种可以闻到桂花味道的空气是她最想念的。不排斥海鲜，喜欢东南沿海。也许这是我们投缘的地方。三年来，我到过藏区不少时间，从未再次见过面，这次相见却丝毫无陌生感，一眼认出对方。
她开着车，我们在车上闲聊。我说我这周做的事克松村的民主改革，她说我老公他就是克松人啊。于是带我去克松村逛了一圈。一个新兴的，完全新建筑的村子。却是西藏民主改革第一村。
村子前的藏族阿佳们正好在播种青稞，一年之计在于春嘛。这个三月，天气还是有点凉。
我们前往雍布拉康，这是一座有故事的宫殿/寺庙。雍布拉康，藏语意为“母子宫”，是西藏历史上第一座宫殿，也是西藏最早的建筑之一，位于山南地区泽雍布拉康当镇东南，高耸于雅砻河东岸扎西次日山顶。相传是雍仲本教徒于公元前2世纪为第一代藏王聂赤赞普建造，后来成为松赞干布和文成公主在山南的夏宫，五世达赖时改为黄教寺院。 雍布拉康主要供奉释迦牟尼佛像。宫殿内的壁画上生动地描绘了西藏的第一位国王，第一座建筑，第一块耕地的历史故事。 雍布拉康分为两部分，前部是一幢多层建筑，后部是一座方形高层碉堡望楼，与前部相连。公元5世纪，藏王托托念赞时期，传说一本佛经从天而降，正好跌在雍布拉康宫顶，当时无人能识。有圣人断言，到了公元7-8世纪就有人能解读此书。所以这本书被很好地保留在雍布拉康。
山底看到的雍布拉康觉得just so so
往上爬的视野是越来越开阔，谁能知晓这个玛尼堆由多少人堆成。
从寺庙山顶，看山下的村庄，一条路，看着很美。
雍布拉康的建筑，前部的多层建筑。
山下的景色，虽说宫殿易守难攻，但有另一点，断水即断生命源。
爬上雍布拉康的山顶，美哭的视觉。
西藏第一座宫殿，在这。
在山顶的经幡阵旁，上来了一群活力十足的年轻小女孩。我让她们摆了个Pose，这么一张照。
阿嫲不让我拍照，说我没有买她的经幡。
我说就一张嘛，好不好，就一张。
雍布拉康半山腰全景。
满满的飞鸟。
逛完雍布拉康，我们往回走。打算去昌珠寺。昌珠寺位于山南雅砻河东岸的贡布日山南麓，距乃东县约二公里，属格鲁派寺院。建于松赞干布时期，据说文成公主曾在该寺驻足修行。寺中悬挂有一口铜钟。昌珠寺系吐蕃时期第一批兴建的佛教寺庙之一，重要建筑有大门、“乃定学”、措钦大殿名“乃定当”，大殿主像是莲花生。措钦大殿东西长45米，南北宽29米，里边有天井天棚，经堂有柱64根，主要佛殿三间，形式与拉萨大昭寺差不多。殿内有12座佛堂。
看这张图，跟大昭寺的正面真的像得不要不要，进去以后的格局几乎一样。
昌珠寺镇寺之宝——珍珠唐卡，是一件世界罕见的珍宝。这幅用珍珠串起成线条绘出的“观世音菩萨憩息图”（坚期木厄额松像），是元末明初的西藏帕莫竹巴王朝时期，由当时的乃东王的王后出资制成的。
整幅唐卡长2米，宽1.2米，镶昌珠寺珍珠唐卡嵌珍珠共计29026颗，钻石一颗，红宝石二颗，蓝宝石一颗，紫宝石0.55两，绿松石0.91两（计185粒），黄金15.5克，珊瑚4.1两（计1997颗）。这幅价值连城的唐卡能保存不易。
昌珠寺另一宝贝——释迦牟尼缂丝唐卡缂丝释迦牟尼像，像高2.92米、宽1.72米，释迦牟尼着红色袈裟，袒右肩，袈裟上饰八珍宝（即法轮、白海螺、白伞，幢、莲花、水罐、金鱼、吉祥结）；肌肤为金色，蓝色高肉髻，背光中遍饰莲花，结跏趺坐于覆莲座上；右手结触地印，左手结禅定印。像左上为太阳，正中为三足金乌，右为月亮，其中有玉兔在桂树下捣药。像上端有梵文两行。整幅唐卡织法平整、紧密，属明代缂丝像中的精品。
一趟山南行，让我最受感触的在晚饭的三个小时。我们在一家山南算是高档的咖啡厅里吃着西餐，当然味道就那样，价格却非常可观。巴桑喊了她的“男朋友”尼玛，然而尼玛是个女生。三个女人一台戏也许就是这样，一顿晚饭三个小时都结束不了。聊了太多太多，都让我对整个藏文化都改观的地步。即使之前做着克松村的民主改革，还以为对西藏历史有所了解，此刻我是觉得害臊的，因为西藏的历史，甚至没有一本书可以说得清楚，本来嘛，它就是在文成公主到来之后才有的资料，之前一切空白。
看了古格王朝的灭亡之谜很多年，却忘记古格王朝之前其实有非常多的朝代在西藏存在。历史既然是无法改变的，我们能做的便是记住过去，展望未来。佛教使人朝善，世界总会更美好不是吗？
每次看到磕长头的人，感动你了吗？
西藏是这样一个地方，即使我现在非常想回家，可它依旧有吸引力让我继续看着藏民们的生活习惯，我就是想了解他们的一生一世，也许，我上辈子真的是个藏族人。
