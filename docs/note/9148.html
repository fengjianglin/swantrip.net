#1MORE旅行新静界#烟雨江南︱遇城山沟，览春意芳华

游记 摄影 自助游 赏花 踏青 2019/03/31

序

阳春 三月，阳光洒满大地。
对 于田 野来说只要有阳光，便会有一切。
在城山沟这片土地上，山川大地如母亲般无私厚爱。
上善若水，厚德载物，道法自然，让万物随自然生长，让物产丰富大地。
山蕴玉而山辉，水怀珠而川媚。
两百亩桃花湖滋润了城山沟，映托了桃花林。
盛开的桃花就像阳光一样明亮，和阳光一样幸福。
满坡的映山红，说不尽物华天宝，话不完人杰地灵。
花已烂漫，如果不趁着阳光正好的季节去邂逅一场花语，是在是一种遗憾。

美图预告

俯瞰桃山，惊艳一芳华

城山沟景区坐落于著名的“帝乡佛国” 湖州 长兴 县内，东有两百亩的桃花湖，西有千年历史文化的城山清凉禅寺，据说《西游记》作者吴承恩在 长兴 县担 任县 丞时就曾经到过城山沟。
这次出行特意带了小飞机，从空中俯瞰城山沟，瞬间惊艳。漫山遍野的桃花、油菜花、映山红等，铺满了竹林绿植之间，从庭院到田野，从河岸到山丘，整个城山沟就是一幅立体花景。各种鲜花竞相开放，桃林下有金黄的油菜花，还有翠绿的野植，可以说是一层花海一层色，如诗如画，如梦似幻。

雨润山色，醉美城山沟

城山沟最得意的就是漫山遍野的桃花林， 阳春 三月春风徐来，漫山遍野盛开的桃花唯人们所倾心。走进城山沟桃花林，满含春意的 桃园 里，姹紫嫣红的桃花连绵起伏，微风吹来，香气扑鼻。带着欢声笑语来自四面八方的赏花客，徜徉在芬芳的桃树间，三三两两拍照留念，用自己钟爱的方式感受田野的气息和浓郁的花香。
沿着林间那条弯曲小道，在周围满是盛开桃花，各种不同的姿态，争相向你献媚。除了桃花林，山野之间还穿插着金黄的油菜花，还有掩藏在远山的竹林，层层叠叠，举起相机想要留住这美好的景色，却因为花开得太过喧嚣，竟让我有无从下手的茫然。这种感觉似曾相识，记得小时候家乡村头的 桃园 里也有这样的景色，那时候放学最喜欢的做的事情，就是三五两伴在桃花林里追逐，等到桃树成熟，会聚在一起在树枝间爬上爬下，有的时候被园子主人发现，叱吓声还未到耳边，小伙伴们早已跑的不见踪影，愉快的童年与桃花的美艳与成熟的喜悦交织在一起，至今记忆犹新。
话说等到九月金秋，当你再次徜徉于城山沟的山川阡陌间，可以想象得到，闪亮进视野的景象又变成瓜果飘香，硕果坠枝，到处洋溢着成熟丰收和幸福喜悦的景象，饱览开花结果的过程，采摘数以万计的水蜜桃子，定会让你也一样沉醉幸福，一样入迷丰收的喜悦。
江南 多烟雨，下午的时候天空开始下起了小雨。在朦胧的细雨中，春天的味道越发浓烈，就这样飘散在田园山野的淡淡清风里。春天的城山沟，诗意飞翔如鸟，乘着乡间民谣的翅膀，逾越过炊烟迷离的田园，走过春花飘香的阡陌，雨润山色，烘干姑娘含羞的一点心事，惊醒了一朵独放的桃花。
“桃花盛开桃林翩，桃花满园花似仙”，经历了风吹雨打，桃花正迎着春的时光，变换着成长的脚步，悄然绽放于新枝上，淡淡的、柔柔的。倘若你身临其境，一定会被那满眼盛开的桃花所震撼。成群结队的桃花挂在枝头，远远看去就像是一把把粉色的羽毛扇，其中有淡淡的红，浅浅的白，真实之中带着虚幻，是那么的好看，那么的令人难忘。
走近看桃花，却又是另一番景象，只见已经开花的就使劲的开，毫不隐瞒自己的力量，由一个个小花苞，成长为盛大的花碗，一层层包裹着，由内向外，粉红的颜容，微风吹来，里面的小花蕊轻摇着，更是赏心悦目；而那些花骨朵看上去也特别的懂事，一个个都昂首挺胸，仿佛是在时刻准备着绽放笑脸。
下山的小径途中偶见几树梨花，清雅幽香，与美艳粉嫩的桃花相比略显清秀。雨水打过，花瓣显得格外通透。
桃花湖上桃花岛，桃花岛与桃花林一条马路之隔。登岛远眺，远山如黛，雨润万物，密集的雨珠跌落在湖面，激起了密密麻麻的涟漪；微风拂柳，回荡之间，春意盎然。

春色氤氲，人花相映红

次日清晨，雨停之后桃花开的更加美艳，无尽的芬芳，让人对花的执着更加一往情深。走进一处桃花林，拾眸的一瞬间你会发现，万花丛中蜜蜂开始在辛勤的忙碌着，在满是色彩的花 田中 不细看都无法窥见它们的身影。幽幽的花香，伴随着熹微的 日光 ，几只蜜蜂在花丛中寻觅，这样的春色映入眼帘，不做态、不高贵、也不卑微，只是平静，平静的让人喜悦。
桃花与古色建筑相映成趣，时时刻刻彰显着诗意盎然的春韵。灰瓦白墙，花开蜂忙，宁静安详。也许只有城山沟这样真正的乡野田园才有这样的魅力，也许只有桃花有这样的柔情，穿越千年岁月，仍有淡淡花香。
从古至今，文人墨客都毫不吝啬对桃花的赞美。 比如 《诗经》有“桃之夭夭，灼灼其华”；苏轼有“且看桃花好面皮”；尤其是崔护的咏桃花诗“去年今日此门中，人面桃花相映红，人面不知何处去，桃花依旧笑春风”。被誉为借桃花喻美人的千古绝唱。
桃花的味道，就这样肆无忌惮的飘散在淡淡的清风里，是人在花中笑的甜美，是蜂忙采集的香馥，是 桃源 人家的淳朴，是 江南 大地的韵味，还是......
四季不尽的时光交替，唯一忘不掉的就是飘散在阳光中盛放桃花的香甜味，这种幸福满足感渲染满山 满城 ，让城山沟这片大地突然之间也醇美香甜了起来。当花开与阳光交织成诗，当色彩与 光泽 融汇成画，在如诗如画的节奏与韵律之间，在香甜醇冽的回味里，一切都让人记忆永恒，终身难忘。
红满破的映山红也已经开了，虽然只是一小部分，但这充满诱惑的颜色总能引人驻足。桃花、油菜花、梨花、梅花，在等到满破映山红盛放，城山沟总有看不完的惊喜。

攻略篇

1.【关于美食】
城山沟景区内有两个用餐点，一个是住宿的酒店餐厅，一个是景区内的鱼米之乡，里面的菜品都很接地气，味道也很棒，特别是鱼米之乡，取材都是当地的最原生态的食材，也没有什么美食大厨，都是当地的厨娘用柴火烧的，味道很纯正，家乡的味道。
2.【关于特产】
桃花的美不言而喻，它还有许多其他的作用，例如桃花本身可以美容，还可以做药膳，成熟之后的水蜜桃先甜可口，是人们最喜爱的水果之一。城山沟还有自己秘制的桃花酒，度数不高，喝起来有一股淡淡的桃花香。这些东西及食材都可以在景区对面的特产店里买到， 比如 桃花酒、桃花胶、当地的腊鹅等等，这些家里超市没有的，可以给亲人带一点。
3.【关于交通】
a. 杭州 —城山沟
高铁： 杭州 东站— 湖州 站—城山沟¥40.5/人
公交： 杭州 长运公路汽车站— 湖州 市—城山沟¥47/人
b. 上海 —城山沟
高铁： 上海 虹桥站— 湖州 站—城山沟¥120/人
4.【关于票价】
门票80元/人，不过现在有穿汉服免门票的活动，只要你穿着汉服就可以免费进入。

尾

万种风情，千般妩媚。行游城山沟色迷人眼。
十成春色，百倍闻香。醉看花海群芳漾心田。
在这里，有花海环绕群峰，与天空相连，恍然之间，隔世离俗已数年。
在这里，有芬芳遍野连绵，与自然相融，沉迷之时，流连忘返舍归途。
