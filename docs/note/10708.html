给我一片海，过一个人的海枯石烂

游记 海岛 潜水 2018/11/08

《靠窗的位置》

从福州出发，乘坐四小时飞机直达泰国甲米岛。有时候生活像长途飞机，很累很难熬，飞机不停你不可能跳下去，于是你只能望着窗口数着秒钟过，自我调节，或者打个盹，顺应了它。我倒是十分珍惜在飞机上的独处时间，读一会儿书，写一会儿字，不亦乐乎，几小时消纵即逝。
虽天公不作美，整个行程都在下雨，但是也不妨碍看到美丽的海岛的好心情。
一个日落
行前准备
泰国5-6月是雨季，先做好天天下雨的准备。而后是需要备一件长袖薄外套，车内的空调经常低得让你冷得不舒服。游艇上也需要穿薄外套，就算全身湿透了，长袖也比短袖来得温暖一些。另外就是可以用披肩来当薄外套用，无论是穿泳衣还是大太阳下，都很方便。
签证
落地签入境泰国，费用1000泰铢，可停留15天。
注意事项：
1.有效期6个月以上的护照及复印件
2.一张六个月内拍的两寸彩色免冠照片
3.入境后15天之内有效的返程机票复印件和确认过的机票
4.入境时个人须携带人民币不少于5000元（切记啊！海关偶尔会抽查的）
5.一张填写完整的旅游签证申请表。
交通
甲米机场：福州长乐机场有直飞泰国甲米机场，行程4小时。
普吉岛和甲米的公共交通相对来说不怎么便利，大多时候，出行还是要与当地出租车议价而行；还有一种车是Tuk Tuk 车，有点像我们这边的三脚虎，属于民间的一种类似公共交通工具，价格较便宜。也可以自己在国内就联系好车辆，到时候可以直接包车游。
另外出海，去海岛，建议可以直接在当地报一日游团。
住宿
在甲米镇上住宿酒店选择多，但是离海滩远。既然是去度假，选择海边还是比较好玩啦。
普吉岛的住宿，这次住的是班涛海滩上的arinara，店门口的海滩是特色。
特别提示：泰国酒店提供浴巾毛巾，但不提供牙刷牙膏梳子等私人用品，为了环保嘛，记得出门带上哈！

可以甲米自助，也可以甲米+普吉自助，或者可以半自由行跟团环游安达曼海概念—环游一圈，前后两天飞机，第二天环游安达曼海，不走回头，再回普吉岛自由行三天。

PP岛走起

皮皮岛位于泰国普吉岛东南约20公里处，是由两个主要岛屿组成的姐妹岛，这是一个深受阳光眷宠的地方，柔软洁白的沙滩，宁静碧蓝的海水，自然天成的岩石洞穴，未受污染的自然风貌，使得她从普吉岛周围的30余个离岛中脱颖而出，成为近年来炙手可热的度假胜地之一。
另小皮皮岛的是玛雅海湾，是电影《海滩》中拍摄外景的地方，由于海域比较深并且有许多珊瑚，也是潜水的胜地。明天为您播出。哈哈。
我们乘坐着快艇，摇曳在汪洋大海之中。昨天（6月8日）PP岛附近的快艇事情还在脑中挥之不去。危险却又刺激一直都是人们锲而不舍追求着的东西，无论是运动或人或事。
大海之中，所见之处只有远处一点点的岛屿，信号都没有，这时候你什么都没有，你只有这一船的人们相濡以沫。有时候想想，实在让人绝望，就如我第一次深处沙漠之中，无一不感触人类的渺小。

我们前往天堂湾，天堂湾位于小PP岛，三面环山，称为“天然游泳池”，其实是一个海水清澈的峡谷，峡谷的两岸相距不远，大概就是百来公尺，但两岸完全是悬崖峭壁，几乎与海面垂直；峡谷内没有任何可以停泊靠岸之处；下午，太阳斜着照射入峡谷，映照着变幻无穷的光波，景观特别壮丽。这里美丽、很宁静，宛若一个天然的巨大游泳池。当地人给他的名字是“天堂湾”。据说，在海啸时，这也是风平浪静，一个世外的天堂。

这么反反复复的天气里，我早不指望可以见到一场日落，谁曾想，如此幸运。游泳时看到对面的太阳一点点地往下沉，激动得回来海滩上就这么静静坐着。相机也没带，仅有的就是用手机来拍下这些美丽的时刻。
山的那一边被云染红了海水。女孩妖娆的身姿。
一边通红
一边深蓝
我们去看了一回泰国毒蛇研究中心，训练员是拿生命在工作啊。记得给小费哦。那天我一个人在海边，我也遇到了蛇，把自己吓破了胆。到这，我的第一次出国行也差不多结束了。
一次半自助的泰国行（福州往返甲米）终于在自己的护照上盖上了戳。
也算是探探路，往后怕是渐渐走向世界了。以前总想着走完中华大地再走出国门呢。
回家以后呢！
