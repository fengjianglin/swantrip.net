这个冬天不孤寂|梦回古城，穿越千年遇初雪

游记 人文游 古镇 客栈民宿 摄影 美食 2018/12/21

初见平遥

初见平遥便遇初雪，这是我古城之旅的惊喜。用一张图开篇吧！

前言

       山西有座古城, 一直保留着古城原汁原味的样子，它就是位于山西省平遥县的平遥古城。火车站离古城很近，10分钟就可步行到西门。在县衙，南城门等都有售票点，门票125元（相关证件可以优惠半价或者免票），有效期三天，是大通票，包含了20个左右的景点，都是名气较大的，但好些都是大同小异，时间紧就没有必要都去了。要想逛完整个古城，两天都是很紧张的。所以想真真切切的了解他，那就在冬季里来这里呆些时日，没有往日的喧嚣，仿佛整个平遥都是你的。古城里保存完好的明清古建筑里面至今仍有人居住，有着鲜活的人间烟火痕迹。我在这个古韵十足的地方，感受到了千年来人们生活的模样。
 

感动平遥：贴心接站的乔家庄园客栈

       在平遥站下车，陌生凛冽，举目无亲，第一件暖心的事情就是有客栈的接站服务，真切的感受到了平遥人民的淳朴和好客，当然临别也有送站，还赠送小礼物。11月的平遥天气已经很冷，但店家的房间有空调很暖和。客栈位置很好，在古城的四大城门的南门口，位于最繁华的南大街，离古城里各个景点都很近。我特意订的房型床是架子床，非常有穿越的感觉，尤其配合上灯光，效果很赞！城里的客栈一般的都不含早餐，然而幸运的我选择了乔家庄园，早餐虽然简单，但是也基本符合标准了，省去了我大早上就着冷风在陌生的街道觅食的尴尬。
       原谅我是个务实的人，人文的东西其实是这家客栈的价值所在。乔家厚德，生意越做越大，由开始单一的经营布庄扩大到经营棉花，烟草，当铺等。乔家埋入地下的银两，可以说不计其数，乔家所占的老宅铺店以及坟墓多次被盗不说，在1938年，从本庄园东院马棚内挖出金银财宝100万，捐献国家，最近在1998年，还在乔家山乔怀瑾老宅门前的打场上挖出金条18根，以上事实充分反映了乔家的经商之道，带来他们商业家兴业旺，也说明乔家是平遥著名的大财东，山西的大财东以及全国的大财东。
      现在看到的院落结构就是原汁原味，只是后期被改建为客栈，仪门、窑洞、土炕、灶台等等都有保留。关于这个门中门，在游览县衙等景点的讲解中学习了，容我现学现卖，古人认为邪走直线，吉走曲线，所以这扇门在平时一般不开，只有在逢年过年或者喜迎贵宾的时候才会打开，民间有“打开中门迎嘉宾”的说法。此前这家客栈还是一家博物馆，想必可探究的内涵还有好多呢吧。
 

感动平遥：醋酒飘香烟火气

      古城的建筑布局十分精整，东西南北四条主街道与分别衍生出来的八小巷七十二条蚰蜒巷构成了古城独有的道路交通体系。石板铺就的主街道至今平整坚实，让我十分佩服古代工匠的技艺。走在狭窄的小巷小弄中间，老房子门前有正在休息聊天的人们，时光在这里仿佛变慢。
 
        当清晨的阳光打在青砖墙壁和木质门板上时，明亮的光线里满满的是鲜活生活气息。想象着在这里生活的样子，我竟有些心满意足的感觉。淡季的缘故，虽然街道两边商铺鳞次栉比，货物琳琅满目，却显得冷清。
       
           
        由南向北，走到南大街的尽头，不由得在一家卖醋卖酒的店铺前驻足，名为——娘家醋坊|娘家酒坊，店员热情的招呼进去，虽是一家做生意的店，但是中间摆放的八仙桌和长条凳让人心头一暖，不禁想坐下来品一品这山西著名的陈醋。淡季人少的缘故，店员没有给我争创业绩的感觉，反而像是招待一位远道而来的客人，亲切不世故，也就是这种不刻意，我放开了品、放开了尝，左边醋右边酒，都很对胃口，那就蜂蜜醋来上三斤，玫瑰酒打上一壶！“人生得意须尽欢”嘛！还想说的是：包邮啊！平遥人的生意做得好，不完全是东西好，还有就是服务好，这就是晋商文化精髓的延续和继承吧。
 

感动平遥：玩穿越

       艺术总是神秘的，与众不同的，摄影师把场景设在了古名居的院落里，还给我讲述了古院的沧桑故事以及创作意图，以便能够更好的配合，有镜头感。我恍然也被带到了几百年前的大院里，成了小家碧玉的少奶奶，一颦一笑，一步一履都藏着故事。
 
       平遥古城远不止你们在攻略里看到的那些城楼、市楼、票号、镖局等等，来亲和门领略东城门的恢弘，来贺兰桥赏赏园林风景和小桥流水人家，小众的、有趣的、孤冷的，给人思暇空间的平遥，这里的神秘，只有来过的人才会知道，感谢吾生旅拍的王老师让我看到了别人没有看到的平遥古城盛景。
 

感动平遥：深厚的晋商文化

       在平遥古城不得不提的建筑是市楼，是平遥古城的标志性建筑物，位于古城中央，高楼耸立，雄踞全城。传说这里曾经是一口井，人们每天都来这里排队打水，由于等待时间比较长，人流集中，就会利用这个时间交换物品，继而形成了集市，古城最繁华的地方，后来建成此楼，命名为“金井”，延续至今。
 
市楼脚下的咖吧
        说到平遥就不能不说“票号”而说票号，就不能不提----中国第一家现代银行的雏形“日升昌”票号。当时，在“日升昌”票号的带动下，平遥的票号业发展迅猛，鼎盛时期这里的票号竟多达二十二家，一度成为中国金融业的中心。 它的业务在鼎盛时期都已经扩展到日本、新加坡、俄罗斯等国家。是不是要由衷赞叹一句：厉害了，我的国！
 
        据说‘日升昌’这个名字起的颇费心思。日升昌’。繁体字‘升’上面有一个‘日’字头（见上图匾额）。繁体‘日升昌’这三个字不管从左从右念都是‘日日升’。这名字起得可真的独具匠心。 
 
      日升昌在一百多年的经营中，始终坚持着存款自愿，取款自由，严格为客户保密的经营理念当时有“一纸汇票，汇通天下”的说法。这既是对日升昌票号生意兴隆，营业范围宽广的见证，也充分显示出日升昌票号在汇票防伪工程上所具有的特殊能力。
 
      日升昌汇票防伪的措施有很多，一般主要是靠：笔迹（专人书写）、印章、水印（四个角都有日升昌防伪水印---见图）和汉字密押防伪标识。通过以上这些防伪措施的实施，日升昌票号在长达一个多世纪的经营过程中未发生过一起误领、冒领的事件。
 
       日昇昌一百多年未曾出错的防假密押是：“谨防假票冒取，勿忘细视书章”，表示1至12个月；“堪笑世情薄，天道最公平。昧心图自私，阴谋害他人。善恶终有报，到头必分明”，表示1至30天；“生客多察看，斟酌而后行”，表示银两的1至10；“国宝流通”，表示万千百两。可以在这里写一张汇票留作纪念，笔记防伪，印章防伪，密押防伪一个不少，就是没有水印防伪。汇票长1尺2寸，宽8寸，填好的汇票从中折好，一撕为二，由客户和票务各持一半，票号这一半汇票通过政府邮政部门寄到客户指定的提款地的票号分支机构，客户及其亲属持另一半汇票可到那里提取银两。
 
协同庆票号
       刘庆和（字肃斋）因家贫，15岁辍学就商，曾在蔚泰厚票号学汇兑，后因太平天国革命爆发，江南大乱，各票号纷纷撒庄和裁减人员，刘便与同号好友孟子元离开蔚泰厚票号，并决心不再甘居人下，而另立门户。他们先经人介绍找到榆次聂店村富商王家，王家主持家务的王栋正愁办票号无合适人选，一见他俩曾在著名的蔚泰厚票号做过事，熟悉票号业务，是个人才，立即聘用刘、孟二人，由王氏联合平遥县王智村米家，二家共同投资办起了协同庆票号。
 
       协同庆地下金库，金银珠宝遍布在这壁垒森严的地下世界，在平遥一般的票号金库都是地窖式的，只有协同庆一家是地下窑洞式，已经很接近现代银行的金库了，在当时来讲是非常先进的理念。
 
       好多金子啊，虽然都是假的但仍可以想象当年平遥票号庞大的资金量。金山、银山，真教人看花了眼。这个超大号的金元宝叫“没奈何”意思是说哪怕有贼人闯进金库，也搬不走，无可奈何，只能放弃。
        硕大的地下金库里屹立着一根盘龙柱，据说是朱元璋赠给军师刘伯温的珍品，以檀木雕刻的神龙宝柱，朱元璋感念刘伯温和自己建立大明王朝的功勋，特赐神龙宝柱给刘伯温。刘伯温不忘祖德，将神龙宝柱运回平遥，并立于龙脉位。清咸丰年间，协同庆钱庄成立，财东和掌柜都对神龙宝柱珍爱有加，且绘彩上贡，据说协同庆起步晚，发展快，主要就是得益于地处平遥古城中心，兼有神龙宝柱相助。
 
同兴公镖局
 
       镖局，又称镖行，可以说是现代保安公司的前身，主营异地货款押运，看店护院，清朝灭亡后，镖局被现代警察机构及保安组织逐渐取代，淡出了历史的舞台。镖局押镖有“三宝”：镖旗、镖箱、镖车。镖车通常都是单轮的，方便在颠簸不平的山路里转弯和行走。镖车的两侧都插着镖旗，有些盗贼看到是大镖局所护的镖也就不敢来劫取了 。
 
       房檐上挂有两块匾，最上边那块黄色与蓝色镶嵌的“奉旨议叙”，是当年慈禧给二代掌柜王树茂的一块匾，一般的结构龙是高高在上的，而它是凤在上龙在下，表明当时是慈禧执政，一手遮天，上面写有四个字，但是她把头一个“奉”字写在了一旁，写得特别小，猛得一看就像三个字。当时慈禧是满人，满人最忌讳四字的竖匾，“四”的斜音是“死”，不吉利，所以把“奉”字写在了一旁。而她把“旨”字与“叙”字写错了，而且旨字写错的特别有意思，正确的“旨”上面是个匕首的匕，下面是个“日”，而她写成了“上面一个上，下面一个日  ”，说明她的身份和地位远远高于太阳，而叙字写错的原因是慈禧是满人，满人到了汉人的地方必须写汉文，她觉着这样有贬低身份的意思，所以故意写错的。下边还有一块横匾“庆衍花龄”这块匾是当年护送太后的镖师在过六十大寿时由各商家票号联合起来所赠送给他的一块寿礼，庆是庆祝的意思，衍是繁衍长寿的意思，花龄是六十花甲，下边这些小字是各商家票号的名字共有五十多家。
 

感动平遥：暖胃暖心的平遥美食

        平遥古城是一个适合生活的古镇，在饱览了晋商文化之后，集市上平遥牛肉、碗托、油茶等等，当地的特色美食还让我大饱口福。
 
       某点评上搜了一下，就选了五星的天元奎饭店，必定要探一探这好评如潮的究竟。店内装修风格古朴、原汁原味，旧时的物件、大红的灯笼，还有贴满了墙的顾客留言，我笃定这是一家有情怀的店。店里干净温暖，一锅热乎乎实实在在的铜锅牛肉之后，我已然忘记了这是冬天，掀起门帘出了门一股寒意袭来，才恍然明白。
 

感动平遥：初雪在平遥

        两天三晚，脚走的疼，但精神饱满，估计是被冻得。第三天一大早起床，收拾东西准备回家，推开门被眼前的景象惊呆了，外面银装索裹，美的一塌糊涂。顾不上火车的发车时间了，拿起相机奔跑在大街小巷，好是兴奋，昨日被雾霾笼罩的小城，今日被雪花所占有，临走前，她给了我莫大的惊喜，这会儿我更加爱上这座城。即便赶不上车，我也要看尽她的美。
 
      千百年来生活的烟火气，让这座千年古城生生不息的流传下来，慢悠悠的岁月，沉甸甸的历史，我还挺愿意在这种生活气息浓厚的小城老去。大家如果时间闲了，一定要带上最爱的人，在这里感受一下古韵十足的慢生活时光。
