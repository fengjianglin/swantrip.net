2019平遥中国年全新攻略【交通住宿、路线特产、美食、活动】

游记 人文游 古镇 春节 美食 自助游 2019/01/30

说说这次旅行

又是一年春节到，离家的人儿开始归，漂泊一年把家盼。归心似箭为团圆。马上到春节了，在外的让人都开始陆陆续续的回家，不知道大家过年有哪些好去处呢？除了是待在家和亲朋好友的聚会交谈，有没有想着约起来一起出去玩玩呢，找一个有特色的去处一定不失为最佳选择，那不如来平遥过平遥中国年吧，一定会让你的这个年与众不同。 之前在平遥体验过一次中国年，非常之震撼和流连忘返，这不今年的平遥中国年，我又准备去了，先来说说我以往过平遥中国年的经验吧，交通的住宿的游玩的美食的都有哦。
过惯了城市里平淡又普通的春节，除了在家吃顿团圆饭赏赏彩灯之外好想也没有什么特别的活动了。2019春节，来平遥过大年吧，千年古城平遥把自身独特的古韵之美与现代化的彩灯结合起来，同时也有许多民俗活动在这里开展。你想想，在浓厚的节日氛围下，漫步在古镇的街上，欣赏着千年以前的建筑群，观看着各种民俗活动，岂不是美哉。

交通攻略及注意事项

自驾游：开车到平遥古城之后车辆是不允许进入古城内的，不过不用担心，古城外有大型停车场，停车场有比较大的大南门外生态停车场、上西关停车场、下西关停车场、北关停车场、其中生态停车场、下西关停车场、北关停车场是比较大的，另外还有一些规模不大的个人停车场。
1、生态停车场：是平遥县政府花巨资修建的游客中心及停车场，位于平遥高速口出来后之行过了第二个红绿灯的右侧。停车场修建的挺好的，面积也大，但就是离平遥古城有点远，需要乘坐电瓶游览观光车抵达。
2、上西关停车场：上西关停车场比较简单，规模不大，位于平遥古城上西门外，距离平遥古城和又见平遥剧场比较近，从上西门进入古城的话刚开始看到的古城会感觉不是很繁华，那条街不是古城内的中心街道。
3、 下西关停车场：下西关停车场规模比较大，是2014年新建的停车场，位于平遥古城下西门外，和平遥古城的下西门和又见平遥剧场距离也比较近，进入下西门后人流量比较大，游客挺多的，而且下西关停车场与平遥县城的商业中心、小吃一条街等紧邻。
4、北关停车场：位于平遥古城北门外，规模较大，2014年翻新，是距离平遥古城城门最近的一个停车场，紧邻家庭旅馆一条街，这条街比较繁华，进入古城后是平遥古城北大街。
停车场收费标准：（一）7座以下，8小时10元，过夜30元。
                             （二）7座（含7座）——15座，8小时15元，过夜45元。
                             （三）15座（含15座）以上，8小时20元，过夜60元。
坐动车或火车：
一、坐动车是在平遥古城站下车，从平遥古城站乘坐108路公交到平遥古城，首班车是早上七点，末班车是晚20点26分，15分钟一班车，全程28公里，票价为一元。（注：乘车前一定要看清楚车站行走路线，以免坐反方向。）
二、坐火车的话是不用乘坐任何的交通工具的，火车站出来步行十分钟左右到古城下西门了，很方便呦。
来平遥古城旅游不管是哪种交通都很方便的，看每个人的出发地吧，平遥有高铁站、火车站、汽车站，各地基本都有直达的车。

年间住宿攻略

节假日的平遥古城格外热闹，越来越多的世界各地的人都汇聚到平遥古城来过中国年，人多意味着什么呢？意味着住的地方得提前订呀，要不这到地方了没有住的地儿多尴尬，一行人还要挨冻到处再寻找有空房间的客栈，但是像平遥古城这么大的旅游圣地，如果不提前预订的话，客栈酒店怕是没什么空房了，基本都爆满，很小的可能找到房间去住了，而且像这种情况，就算是有空房间，有的酒店见你没地方去，说不定还会问你多要点儿钱，然而你也没有办法，为了有个落脚处也只能接受了。所以说，定好哪天出发，一定要提前预订住的地方。
我和朋友去年住过的一家酒店就挺不错的，不仅环境服务大赞，而且就连地理位置都很给力，麒麟阁，城隍庙街76号，对门就是文庙学宫，再往前走上几步，右侧就是城隍庙了，完全就是住在古城里的感觉。看外观就能觉得麒麟阁与别家客栈的不同，很大气，都是复古元素，一进大门的关公铜像，有几根柱子上面有几条金色的龙盘旋着，把整个酒店的大堂都显得更加大气了。
环顾四周，怎么看都有一种熟悉的感觉，酒店服务员介绍说，这里的建筑风格都源自于故宫元素，怪不得呢，之前去故宫旅游时看到有的风格和这里的就非常相同，果真应了麒麟这个名字，典雅复古元素在这里展现的淋漓尽致。而且麒麟阁还是古城里唯一一家具有国际五星标准的酒店，这充分的说明了麒麟阁在各方面一定都做的非常不错，不然怎么就能做到是唯一一家五星标准的呢。
让我觉得麒麟阁和别处最大的一个不同就是，它有专门的文化休闲区，就在一进门大堂的左侧，那儿放着一部古筝，还提供一个台子给大家写毛笔字，累了的话还能喝茶呢，这儿还有一套专门喝茶的工具呦。
房型房间是最给力的，不管你是亲子游还是和朋友一起来，或者是一个人，这里都有最合适你的房型，房间很干净，各种设施也很给力，关键是床，特别的舒服，窗帘一拉，没有外边环境的嘈杂，即能进入深度睡眠状态，试想一下，在外边游玩一天之后回到酒店，一定累的看到床就想躺，舒舒服服的洗个澡，躺在软软的又舒服的床上，卸掉一身的疲惫，很惬意吧。
麒麟阁是一家以晋商文化为主题的酒店，不仅在建筑上包含晋商元素，在菜品方面也一直力求发扬和传承传统晋菜的本真味道，有时间的话一定还要再品尝上一番那儿的菜品。

游玩路线

春节期间在外打拼了一年的人们基本上都回家团圆了，休息下来和亲朋好友出来玩玩，也有从外地自驾过来的游客，所以说，每个景点人都是挺多的，我这儿有条路线可供大家参考，基本上把平遥古城内比较著名的景点都包含了，去年我就是这个路线来：
文庙——城隍庙——市楼——古县衙——华北第一镖局博物馆——协同庆——平遥古城墙——观看《又见平遥》
《又见平遥》是我觉得来平遥一定不能错过的一场体验，我拙劣的话语实在是不能把它描绘的透彻，当然也不可能描绘的完全。这场演出与以往我们看到的演出都不相同，没有固定的舞台，所有的演员都是边走边演，所有的观众也都是边走边看。也因此，又见平遥这个剧目更加的活灵活现了，演员有更大的发挥空间，观众也近距离的与演员亲自接触，【镖师洗浴】、【选妻】、【还魂 安息】、【面秀】……每一幕都是活的，不仅如此，还能亲自体验一把明清时期街上的热闹繁华，各种叫卖声还有各种揽客的场景，应有尽有。
来说说景点吧
文庙：印入眼帘的金字横匾，初晨阳光朦胧下的彩色琉璃瓦，门庭上镶嵌着的闪闪发光的壁画，让我不禁佩服古人的伟大工匠精神！
一股强大的魔力，让我不由自主的想靠近，想一探究竟，文庙学宫，这个让人流连忘返的国学圣地！文庙东面还有个九龙壁，也是平遥古城的一个标志性建筑呢，资料显示，“在在波涛起伏的巨浪里，居中的是一条黄色蟠龙，前爪作展翅状，后爪分撅海水，龙身环曲，瞠目张颔，威风凛然。左右两侧依序各有蓝、黄、白、黄八龙，白为升龙，其他均为降龙。九龙戏象万千。”
文庙没有现代化多余的掺杂，保留了这里的原汁原味，小课桌还在那里仿佛依稀可见当年这里教学的场景，有一面墙上还刻着历山西杰出人才的名字，是一个教育展示圣地，在这里有幸观摩了当年科举状元的真迹，登上阁楼二层，还领略到了古城的街景，这是在地上所不能感受到的，很是壮观。
城隍庙：这里向来也是以文化氛围浓厚而成九流百家的聚集之地，映入眼帘的城隍庙并没有那么有新意，反而还是旧旧的，仿佛吹大一点的风这木质架构的外观就要倒下去了一样，我俩蹭旁边一个旅行团的导游，听介绍说这里自从建成以来就保持这个样貌了。
虽然没有很新，但看着却极富年代感，试想能与这千年古城一样这么多年都保持风貌当真不易。庙内的格局据说是和平遥县衙相对应呢，庙门至献店之间还有戏台和钟鼓楼。献店的左右分别有灶君庙和财神庙，它们横向联接在中轴线上，从献殿以北的城隍殿延伸至寝宫，上建阁楼。灶君庙之后是道院。财神庙的正位是财神殿，上建真武楼，前有献殿，对面是一座建筑工艺精湛的戏台，别出心裁地建在拱券山门之上。
汉民族的古建筑群的各种形式屋顶应有尽有，每一处亭台楼阁都多姿多彩。
市楼：在古城南大街的中心，来来往往的游客只要是在南大街上走都能欣赏到市楼的美景，它高耸在古城的中央，就像是守卫着这座古城的士兵，紧紧连接两旁的商户，只要是从这里走过的人都必须经过市楼下方，也因此可以如此便利的看到市楼的独特风貌了。我和景蕊是在晚上看到的市楼，各式颜色的灯光照耀着，远远的就看到它光彩照人的样子了。
古县衙：对它的印象源自于古装剧，尤其是县衙里断案拍扁叫绝的样子实在让我印象深刻，平遥古城的县衙始建于元代，定型于明清时期，纵观整个县衙，遵循轴线对称的传统格局，是全国目前保存最完整、规模最大的县衙，是地方特色的综合载体。
爸爸去哪儿第五季曾在平遥县衙取景录制，jasper坐在椅子上趣味断案的场景可真是可爱，我可是爸爸去哪儿的忠实观众，所以来平遥古城怎么能少的了去县衙打卡呢。
华北第一镖局博物馆：在古城东大街上，里面充分而真实地再现了清代乾隆以来我国镖局的独特风貌。说道镖局，第一印象就是电视剧《武林外传》里佟掌柜她爹开的那个镖局了，它是在陕西汉中开设的镖局，虽说地方不同但也同样展示着走镖人勇敢不怕死的勇气精神。走进山西平遥的镖局博物馆，正厅牌匾上的四个大字“威震四方”把我给震撼住了，两边对联：一声哈武镖车走，年年江湖平安回。是希望走镖的镖师可以平平安安归来，极具威力，深入每个镖师心里。刀光剑影戎马行，江湖柔情知路遥。 
协同庆：来之前听说这里就是电视剧经常光顾的拍摄地，果然名不虚传，门楼前有两只端庄威武的石狮子，守着这个“地下钱庄”，晋商文化的影视基地。
钱庄，专门进行白银和铜钱相互兑换的一种金融机构，过去的这里一定是热闹非凡的吧。协同庆是中国票号业中一家独具特色的票号，创办于清咸丰六年，它的经营特色非常鲜明，主要有：一是以人为本，知人善任；二是注重资金灵活调度；三是突出重点地域经营，具有现代成功企业的重要特征。
 这里还有全国最大的钱币展厅，是研究中国近代金融、货币发展史的重要基地。最为壮观是这里有平遥古城最大的地下金库。可想而知这里曾经多么繁盛，钱庄产生于我国明代，协同庆钱庄是当时全国最大的钱庄。进入地下金库的时候还通过几处狭窄的地下台阶，方可进入地下金库。
平遥古城墙：很高，抬头望会觉得累，阳光刺眼的时候甚至都不能抬头把它看得完全，只是阳光下的这座城墙，美的让人移不开眼。一直以来都想把它探个明白，究竟是何等高超的技艺才始得这城墙这么多年都屹立不倒，完整的保存着。
来的时候在外边看它，就已经被深深的吸引了，当我站在它的脚下，才真正感受到这座守卫着古城的城墙时何等的古老沧桑又让人充满敬畏。淡淡的有些发青的墙砖，描绘着连绵不绝的城墙，望着看着，竟发觉熟悉的感觉越来越强烈了。由于雨水风霜的洗礼，有些墙砖已经凹凸不平了，我用手小心翼翼的触摸，这是何等想让人永久收藏的韵味呀。

平遥特色小吃

到一个地方旅行，肯定少不了要尝尝当地的特色小吃了，但通常情况下都是到处转悠到处寻找，走累了有时候也找不到要吃的那道小吃，当时我们去的时候就是走了好久才吃到几样，累就不说了，而且味道还不怎么样。后来再一次来平遥古城旅游的时候发现了个好地方，能一次把平遥的特色小吃都吃遍，叮叮叮，这里就是亲情炉食铺了，能一次性把平遥特色小吃都能吃个遍的小店。
亲情炉食铺在古城东西南大街的交汇处，我记得当时我和朋友是跟着地图去的，很好找，没费什么功夫。招牌很显眼推开门走进去的时候地下躺着一只猫，虽然吓我一跳但是很可爱，毛茸茸的，我们走进去的时候它还充我们喵喵的叫呢，像是在和我们说话表示欢迎，这个喵星人深得我们喜欢。店家的服务很是周到，一进门就招呼我们坐下了，还端过来热水给我们喝。我是最爱店里的装修风格了，文艺范儿十足，大到墙上的壁纸，小到店内的每一个摆件，都像是被赋予了诗意一般，精致又富有年代感。服务员小姐姐带我们坐到了靠窗的位置，不仅能观看到临街的美景，还能将整个店的景一览无余呢。
我们当时小姐姐推荐的双人套餐，其实想点黄酒牛肉套餐来着，但是看这个不如双人的菜品多，还是就选择了双人套餐。亲情炉食铺还有二楼，我们去的时候人不是很多，所以在点完餐等待的功夫 经过店家的同意之后我们去二楼参观了一番，同样的文艺范儿，木质架构的桌子看上去就很有感觉，个子高点的人仿佛能够得着屋顶一样。
墙上有个标语我很喜欢：“遇到最美的自己”。在旅途中不就是为了遇见之前没有看到过的自己。
小套餐端上来之后，觉得真是惊艳，什么样的小吃都有，比起上回来时到处奔波的累，这简直是太好了，双人套餐含有多达13份平遥特色小吃呢，有莜面栲栳栳、水煎包、平遥牛肉、熏蛋、牛肉拌面、香酥绿豆饼、独家秘制皮冻、平遥碗脱、一杯兆辉甜茶、一杯嘿！谷粒、还有鱼香肉丝米饭，种类的确很丰富，涵盖了平遥古城几乎所有的特色美食。
我喝的是嘿！谷粒，因为不太喜欢吃甜食，想着这个肯定比油茶口味又清一些吧，果然没让我失望，味道太棒了，一点都不腻，喝下去的第一口就咬到了一个葡萄干，还有杏仁花生什么的，不是特别甜，瞬间爱上。
这里的牛肉也很正宗，一丝一丝的，很入味儿，肉质一点都不柴。
碗托：不像之前吃过的没有嚼劲，亲情的碗托味道很清爽，搭配上黄瓜和胡萝卜丝的香甜清脆，在淋上一些陈醋酱汁，味道简直太棒了。
牛肉拌面：山西向来以面食著称，一小份儿牛肉拌面加着各种配菜，放上一点牛肉碎和花生碎，口感更加的筋道了。山西面食果然从来都没让我失望过。
 其他的菜就不一一介绍啦，总之都非常的美味，遗憾的就是没能早点碰上它，就冲着它的这个文艺范和店内的巧妙构思以及平遥特色小吃，我非常期待这次平遥中国年和它再次相见的时候。

平遥古城纪念品 

到一个地方旅游一定少不了带些纪念品回去，尤其是春节过来玩，纪念品尤其显得有特色，山西最著名的特产是什么，一定是醋啦，来平遥怎么着也得带点回去吧，去年我们过来玩的时候带了点陈醋和酒回去，反响很好呢，我记得是在古城里一家名叫娘家醋坊的店里买的，很古朴的一家店，环境什么的都不错。那个综艺节目《咱们穿越吧》也来过这里呢，从那以后娘家醋坊的名气就更大了，都成古城里的明星醋坊了。
其实也不无道理，娘家的醋味道确实正宗，口味也很多，有蜂蜜老陈醋、红枣老醋、苹果老醋、保健老醋等，娘家醋坊还有一个名字就是娘家酒坊，说是也是自家酿造的，常见的女儿红、竹叶青、黄酒都有，当时我们还免费品尝了一些，蜂蜜老醋我觉得是众多味道里最合我胃口的，其他的也都可以，我是不喝酒的，朋友尝了一些，说是味道还不错。最后我们买了蜂蜜陈醋和竹叶青酒，店家还给包邮了，很贴心。
看这店面就觉得有些年代感了，传统的木质牌匾，门口还有车车上放着几口缸子，就好像是古代街上兜卖的呢。
店内环境质朴复古，还有一块显示屏上播放着当时宋小宝他们来店时的场景画面，还有别的综艺节目的画面，可见娘家醋坊的名气之大呀。像古代店面里写着菜名的木头牌子，也在店里挂着，置身在娘家醋坊，仿佛回到了那个年代。

美食打卡

现在都流行什么网红烧烤店呐、网红酒吧、网红串串、网红麻辣烫……其之所以称为网红，无非就是因为去的人多，一传十十传百的，大家都听说过这里了，其实味道不一定像说的那样好，只不过打开了店的名气，所以称之为网红。我们上次去的这家网红餐厅，可真真没让我们失望，大家都听说过海底捞的完美服务吧，天元奎堪比海底捞，年间的天元奎饭店内座无虚席，门口等着得人也不少，生意非常火爆，真不愧为网红餐厅，店内装修的风格深得我心，就像是坐在古代饭庄客栈里与好友一起喝茶吃饭呢，带有年代感的桌子椅子，安放盆栽不知年代的石墩。
我记得我们点的菜是牛肉铜锅、长龙茄子、牛肉栲栳栳，当然别的招牌菜还有很多，人少也吃不了那么多，所以只点了这几道。
牛肉铜锅：里面除了牛肉之外还有炸过的豆腐，有粉条萝卜、表面撒着一丝丝香菜，这道菜最大的亮点就是，里面的牛肉不是刀切下来的那样规整，它是一丝一丝的，浸着满满的汤汁，咬下去特别的筋道。
长龙茄子：像一条鱼一样盘旋在盘子里，上边撒着青豆，裹着酸甜的酱汁，咬下去外边酥脆里头棉，口感非常的棒。
牛肉栲栳栳：果然是菜如其名，莜面圈之间靠的很紧，吃的时候得两个人合作才能给它揪下来，上面撒着牛肉碎末，放着一个油菜叶子，浇着味道浓郁的汤汁，吃起来满嘴的莜面香甜
美食打卡来这里一定不会错了，古城南大街的天元奎饭店。

平遥中国年，一定不虚此行

春节是中国人心里一个非常重要并且传统的节日，每年的这个时候不管是在外忙碌了多久的人都会赶回家过年，和家人团聚。平遥的中国年，让我看到了过传统中国年的样子，各种民俗文化活动都展现的淋漓尽致，人们脸上洋溢着的都是发自内心的那种最淳朴的笑。我体验着一场山西平遥的民俗文化活动之旅。2019年平遥中国年会更加具有全域性、体验性，不仅突出古城城区范围的年文化打造，而且把年文化主题向乡村延伸、向景点扩展。到时，还会有百名书法家现场泼墨送春联、旺火晚会、猜灯谜、游古城、送惊喜等活动。这届“中国年”新增快闪、随手拍、举办“网络春晚”等活动，将利用多种形式，更加融入时代的发展，为来平遥过中国年的游客提供一个充满创新、与众不同的“平遥中国年”。
来平遥过中国年，一定会让你不虚此行。
