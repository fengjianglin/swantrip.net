山西印象

周围朋友都说山西好风光，山西古迹多，我特别钟情平遥古城，到底啥样？参加团游！山西运城 关帝庙 平遥古城 乔家大院 雁门关 五台山环线夕阳游！ 自己来山西游览后，感到山西确实值得一游！不管是以前来过的壶口瀑布，还是首次登鹳雀楼、雁门关；王家大院、乔家大院，都是故事多多、历史厚重的景点。那个平遥古城，个人觉得夜晚灯光闪烁古色古香比白天更出色，白天跟着地接导游连轴转，看到的商店多，商业化气氛浓厚。

山西的气候

山西省，属于中温带和暖温带季风气侯区，即温带大陆性气候。春季日温差大，风沙多；山西地形多样，高差悬殊，因而既有纬度地带性气候，又有明显的垂直变化。

普救寺

普救寺位于永济市蒲州镇西厢村的塬上。进入普救寺大门攀登108个𨺗峭的台阶，上到屹立在寺中的莺莺塔，不仅形制古朴、蔚为壮观，而且以奇特的结构，明显的回音效应著称于世。现在塔侧两蛙之间以石扣击，塔上会发出清脆悦耳的“咯哇——”“咯哇——”的蛤蟆叫声，令团友连连称奇。塔后有西厢记的故事情节，各个房间一一参观！
普救寺始建于唐武则天时期，元代王实甫《崔莺莺待月西厢记》中说的“红娘月下牵红线，张生巧会崔莺莺”的爱情故事就发生在普救寺内。寺后是一个活泼的园林花园。因要下几十个台阶，就在上面观望，园内叠石假山悬险如削，莺语双亭飞檐翘角。荷花池塘上横架曲径鹊桥，亭桥相接、湖山相衔。莺莺的“拜月台”掩映在青松翠柏、千竿修竹之中。 据方志中称之“普救蟾声”，为古时永济八景之一。

解州关帝庙

到山西旅游的第一天下午来到运城，参观解州关帝庙，导游介绍关帝庙分为正庙和结义园两部分，它是现存规模最大的宫殿式道教建筑群和武庙，被誉为“关庙之祖”、“武庙之冠”。庙内悬挂有康熙御笔“义炳乾坤”、乾隆钦定“神勇”、咸丰御书“万世人极”、慈禧太后亲书“威灵震叠”等匾额。时间关系，也许导游经常来此，感觉没啥，带我们从前院主轴线上游览介绍，照壁、端门、雉门、午门、山海钟灵坊、御书楼和崇宁殿，两侧是钟鼓楼、“大义参天”坊、“精忠贯日”坊、追风伯祠；后宫以“气肃千秋”坊、春秋楼为中心，左右有刀楼、印楼对称而立。

鹳雀楼

鹳雀楼景区，是国家4A级旅游景区，位于永济市蒲州古城西向的黄河东岸、蒲州古城城南。因时有鹳鹊栖其上而得名，
鹳雀楼楼体高73.9米，我们运气好！乘电梯直达6楼，远可眺舜都遗址，近可瞰黄河之水天上而来。然后步行下楼，一层一层楼参观，四楼在修缮，其它楼层有展示，着重以河东文化和黄河文化为主题，还有反映河东人民勤劳智慧和丰富的民间工艺的制盐、冶铁、养蚕、剪纸、年画、社火等，这些都充分再现了悠久的华夏文明。鹳雀楼是黄河文化的标志和象征。
门票价格：60元
交通指南
线路一:到永济乘三路公交车到普救寺转车
线路二:到永济乘二路公交车；
建议游览时间：2~3小时；
唐代诗人王之涣的千古绝唱《登鹳雀楼》“白日依山尽，黄河入海流。欲穷千里目，更上一层楼。”即诞生于此。
新建的鹳雀楼为仿唐形制，外观四檐三层，内分六层，总高73.9米。一层主题千古绝唱和大唐蒲州盛景，二层悠远流长（华夏根祖文化），三层亘古文明，四层黄土风韵，五层旷世盛荣，六层极目千里，充分体现了唐代风韵和“欲穷千里目，更上一层楼”的意境。
景区规划以古典园林式分布，形成“四区十二点”的空间结构，是一个国家级旅游景区。
山水浏览区。包括:苍山苍林自然景观区、吉祥如意湖和鹳雀宛。
这是国内唯一采用唐代彩画艺术恢复的唐代建筑。

壶口瀑布

第二次来到壶口瀑布！仍然感觉震撼无比！上次是从陕西境内过去，这次从山西临汾吉县过去！正面观赏瀑布，巨大的浪潮扑面而来，更为壮观！若进龙洞亲吻壶口瀑布，更是惊险刺激！这是中国第二大瀑布，世界上最大的黄色瀑布。壶口瀑布气势恢宏；受壶口瀑布上游陕西境内云岩河流域影响，黄河壶口瀑布曾出现西边浊浪翻滚东边白浪滔滔的奇特景观。

王家大院

午饭后乘车赴灵石参观山西著名的民间故宫—王家大院。进入王家大院门楼斜坡上去，每院除有高高在上的祭祖堂和两旁的绣楼外，又都有各自的厨院、家塾院，并有共用的书院、花院、长工院、围院。周边堡墙紧围，门堡建筑群，是堡，又似城，依山而建。从低到高分四层院落排列，左右对称，中间一条主干道，形成一个很规整的“王”字造型。同时隐含“龙”的造型。站在城楼上观看王家大院壮观无比！王家大院的前堂后寝的庭院，充分体现了官宦门第的威严和宗法礼制的规整。王家大院的建筑群大小院落35座，房屋342间，好厉害哦！

平遥古城

古城有近三千年的历史文化名城，我国保存最为完好的四大古城之一，也是中国仅有的以整座古城申报世界文化遗产获得成功的两座古城市之一。早晨跟随导游第一站登城楼，远眺平遥古城全景：第二站游览平遥文庙，第三站参观地下钱庄，第四站游览平遥县署，路过平遥城皇庙！平遥古城保存了其所有特征。让我们游人大饱眼福！
交通情况
飞机：先飞至太原市转乘长途客车或火车前往平遥，从机场出来的机场集团公司站乘坐201路可以到达太原建南汽车站、太原汽车站以及太原火车站，从此三处出发均可直达平遥。
火车：平遥火车站位于古城的西侧，离古城西北城角仅300米。
客车：平遥古城公路比较发达，不过由于平遥汽车站主要运营省内线路，所以从太原中转客车去平遥最为合适。
门票：150元/人；
景区交通：电瓶车往返40元/人；
建议游览时间：一天~二天
早晨跟随导游第一站登城楼——南门城楼，远眺平遥古城全景。 平遥城墙总周长6163米，自由行伙伴有时间在城墙上走一圈，也是不错的安排。墙高约12米，把面积约2.25平方公里的平遥县城一隔为两个风格迥异的世界。城墙以内街道、铺面、市楼保留明清形制，城墙以外称新城。这是一座古代与现代建筑各成一体、交相辉映、令人遐思不已的佳地。

协同庆

协同庆是中国票号业中一家独具特色的票号。它的经营特色非常鲜明，主要有:一是以人为本，知人善任;二是注重资金灵活调度。游览明清一条街，来到协同庆门前排队，等待叫号入内参观。
参观中国银行的鼻祖—日升昌票号、协同庆钱币博物馆。
有管家向大家介绍票号情况，有互动，购买票号，去金库兑换银票、元宝。
协同庆钱币博物馆地下金库里的镇店之宝——龙柱。

平遥县署

参观平遥故宫——县衙。平遥县衙坐落于平遥古城中心，始建于北魏，定型于元明清，整座衙署坐北朝南，呈轴对称布局，南北轴线长二百余米。平遥县衙作为中国现有保存完整的四大古衙之一，也是全国现存规模最大的县衙。
参观平遥故宫——县衙。
县衙整个建筑群，主从有序，错落有致，结构合理，是一个有机的整体。

乔家大院

午饭后乘车赴太原，途中赴祁县游览北方民俗博物馆，央视热剧《乔家大院》拍摄现场-【乔家大院】，乔家大院是富有传奇色彩的清代富商民宅。以前看电视剧《乔家大院》的时候，乔家大院主人乔致庸的商业传奇让我惊叹不已。那时就想有机会实地探访一下，看看他如何让乔家在山西名噪一时，他家有三宝：九龙屏风、犀牛望月、九龙吊灯，还有子孙流洋带回家的宝贝一一火车钟。他提出来的“汇通天下”至今流传。
乔家大院交通：从平遥出发：可在平遥汽车站坐到太原的车（班次很多）中途在乔家大院下即可，参观完步行至下车点等车回平遥或去太原。
另外太原旅行社每天都有发往乔家大院的旅游团，车费在80元左右。
门票：138元/人；

旅游建议

1、来山西旅游，建议大交通还是乘飞机或者乘火车！因为直接从上海乘坐大巴去山西旅游太累人了，长时间坐大巴很辛苦的！
2、去协同庆参观，里面有互动活动，自己一定要听清楚，再出手！
3、一般夜晚入住平遥古城，一定不要错过夜游古城，再累也要去逛一逛，因为夜景比白天更有故事！自由行伙伴一定要在古城呆上两整天！厚重的历史要慢慢的品味！
