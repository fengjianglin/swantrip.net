武汉周边游3️⃣丨黄陂赏花和登山徒步，推荐这四个景区必须打卡

游记 五一 摄影 自然奇观 自驾游 赏花 2019/04/27

写在前面的话

这是我在武汉黄陂旅行的最后一篇游记，在最后一天半的旅行里，赏花当然是少不了的，还体验了木兰花谷的防震防空科普互动、去了贡品萝卜的产地脉地花都、攀登木兰山、还有木兰云雾山最美的漫山杜鹃红，可惜离开当天下了大雨，遗憾直升机不能起飞，或许这也是好事，上天或许眷顾，给了我们蠢蠢欲动下一次再来的机会，希望以后还有机会可以再去木兰云雾山坐一次直升飞机。

关于行程

D1 上午在木兰花谷和脉地花都两个景点看花，中午在脉地花都吃有机美食，下午去木兰山登山，之后前往木兰云雾山景区，景区内就餐和住宿。
D2 木兰云雾山赏漫山遍野的杜鹃花，围观云雾山直升飞机，景区午餐并退房返程汉口站回上海。

关于住宿

云雾山景区内有两处酒店可以选择，分别是：
云雾山大酒店，从景区建成那天，这座酒店就诞生了，距今已经有十年的历史了，许多旅行团来到云雾山都会选择这个酒店。
最具特色的还是要属景区里的梦幻房车营地了，这里有独具特色的大房车、小房车，同时还有舒适的营地大别墅和营地小别墅房型可供选择。

木兰花谷

抵达木兰花谷已是深夜，站在酒店房间的阳台上抬头看星，我们不禁惊呼这夜空中最亮的星，早上起来推开阳台门，漫山的红花映入眼帘，顿时不想再赖床了。木兰花谷依山傍水，宁静如画，四时之景不同，一年四季均有不同独具特色的生态花卉景观可赏，晨昏风光各异，郁郁葱葱，风姿卓美犹如仙境。在这个景区里，不仅仅有百花园，漫山各色花海争奇斗艳，还是一个国家级的人防科普教育基地，酒店一楼的防空防震科普馆也是一定要去打卡的地方。
一大早走入景区，人并不是很多，这里成了我们创作的舞台，我们来的时节，玉兰花大多退出了历史的舞台，但樱花和桃花正在盛开，整片的火红还是让人眼前一亮，我们徜徉在桃花林间，拾起散落在地上的花瓣，抛向空中，一场一场的桃花雨，玩的不亦乐乎。看着我们一帮女人穿梭在花海之中，一旁的小哥哥感叹：女人见了花，都走不动路了。
最酷的是位于酒店一楼的防空防震科普馆了，它是集防空、防震减灾、公共安全科普教育、宣传教育培训于一体的综合性宣教场所。采用仿真、控制、计算机、视频和实时通讯技术、将光电与影像融合于一体，让游客更加真实的体验灾难现场。
首先要穿越一条长达13米的影像隧道，这里运用3D影视特效与视频资料相结合，真实还原抗日时期武汉遭受日军大轰炸的惨烈情景，震撼。之后是空袭防护幻影成像体验厅、环幕地震体验屋、119报警训练厅、生化核武器展示厅、5D影院、VR模拟体验室等，互动性极强。
走进馆内，带上耳机可以体验不同的警报声音，而在环幕地震体验屋，感受8级地震时桥梁断裂、火车出轨、房屋塌陷的样子，身临其境了解灾难发生时的场景，掌握防震减灾技能。还有急救模拟，了解并亲自实践心肺复苏现场急救知识和方法。

地址：武汉市黄陂区长塔线独屋塆
门票：成人80元/人
开放时间：早8点至晚17点

脉地花都

 
冲着贡品萝卜，我们来到了脉地花都，这是黄陂的一个新兴景区，也是大名鼎鼎的脉地湾贡品萝卜产地。也是武汉以名贵花卉“牡丹”为主的花卉主题景区，园内荟萃了主要花卉植物以及世界上部份稀罕名贵花卉。脉地花都因滠水河的滋养，种植有数百亩牡丹花，四月底牡丹花盛放期，艳丽非凡，千姿百态，花香四溢。而现在这个时节，正是芍药花盛放的时候，好想再去看看国色天香。
三月来时，没有碰上名贵牡丹，却偶遇了一大片白色花海，我惊问着是什么花，竟是萝卜花，这也是我第一次见识萝卜花的样子，放眼望去一片洁白，体态玲珑，叶形雅致，在青草绿树之中格外扎眼，看多了三月的樱花与桃花，贡品萝卜扬名在外，就连这片纯白的萝卜花也是惊艳了四下，让人流连忘返忘记归期。
脉地花都还有野生烧烤、自助采摘和垂钓的区域，一家人或者三五好友来到这里采摘新鲜蔬菜，清洗以后在自然之中烹饪烧烤，也真是惬意。我们时间有限，于是午餐就选择了景区里的餐厅，一顿有机蔬菜大联盟的午餐，简单的食材，新鲜健康，在大厨的鬼斧神工之下，在这绿色田园之间，格外美味。
午餐期间了解到，脉地花都还有农副产品出售，萝卜油、土鸡蛋、千牡健康养生茶，有机蔬菜，非转基因水稻等等，吃了还要拿，最后带了两瓶萝卜油回家尝鲜咯。
小贴士：
地址：武汉市黄陂区长轩岭镇徐冲村1-12组

木兰山

“唧唧复唧唧，木兰当户织，……，愿为市鞍马，从此替爷征。”花木兰代父从军的故事早已家喻户晓，然而木兰故里，你知道在哪里吗？
木兰山——佛道两教圣地，这里是巾帼英雄花木兰的故乡，她生长在这里，从这里开始代父从军成为屡获战功的女英雄，最后魂落归尘亦是于此。先有此山，后因木兰得名，如今这座山成为黄陂著名的旅游景点。
昂立天宇气势磅礴，虽海拔不足六百米，却群峦秀峰、山势嵯峨、雄伟壮观。从一天门沿着步道楼梯而上，驿路蜿蜒，石阶虽陡却累的值得，山中自有神仙在，随着步伐的移动，木兰山古建筑群逐渐展现在眼前，这里建有七宫八观三十六殿，形成规模宏大的宗教建筑群，木兰庙会传承千年至今，每年十万余人参拜。
从古至今木兰山都是文人墨客争相拜访的名胜之地，正如明朝诗人朱臣颐的《木兰山》:"未有木兰先有山，山名偏借木兰补。木兰与山名俱在，山并木兰争万古。"，另一位明代诗人屠达曾说:"木兰为西陵最胜，盖三楚之极观。"，这里就是木兰山，它不仅风光迤逦，更重要的是这里汇聚了千年古文化，值得寻幽探胜。

地址：武汉市黄陂区前川街城北木兰山风景区
门票：成人80元/人，网络售价56元/人
开放时间：早8点至晚18点

木兰云雾山

云雾山素有“西陵胜地、楚北名区、陂西陲障、汉地祖山”的美誉。自然风光自是不用多说，景区里还有“十里花山，八里柳溪，一山两教，四季香火”，非遗泥塑、白鹭林、蜡像馆、冰雕馆、恐龙谷已开园可以一并游玩，但是最火爆最红的还是每年四月的云雾山杜鹃花海。
我们也是想把黄陂赏花之旅的最后一站留给这壮丽的映山红，或许是黄陂对我们的留恋，在黄陂的最后一天，大雨而至，哎，就像歌里唱的那样：爱就像蓝天白云，晴空万里，忽然暴风雨。早上从云雾山房车营地出来，大雨下个不停，我们甚至都想过要放弃这片映山红了，但最后冒雨深入杜鹃花海之中，却别有一番风韵。
乘坐景区观光车先来到的是花海，这里其实很大，大雨滂沱我们也只走了一小段，但足以尽收花海奇景了，它们就像绮丽的红色飘带环绕在青山幽谷之中，热情似火，又纯洁无暇。走在这木石板上，手扶着木桩，漫步在这花海之中，杜鹃花风子卓美，雨水落在花瓣上晶莹欲滴，耳边呢喃鸟语，花香弥漫，无比沉醉。
离开花海，驶向花山，在这里有很多花径，穿梭在杜鹃之中，在雨水的滋润下更显得娇韵动人，远山如黛，云雾缭绕，天是白色，山是绿色，花是红色，一层一层的颜色浸染在这片山峦之间，这大自然绘制的山水画，冒雨而行，一切都值得了。当然，除了可以坐着观光车深入花的海洋以外，还可以坐着直升飞机俯瞰，天公不作美，飞不起来，只好与网红几张合个影，假装自己坐了直升机。。。看来，还是要再来一次了。

结束语
旅行中总是会有遗憾的，我们五天的黄陂之旅结束了，或许是在黄陂旅行期间留有些许遗憾，又或许是同伴之间的欢乐多，坐着高铁回家时竟幻想着下一次来的样子，但也正是这份眷恋与遗憾，才能成全下一次的说走就走，不是吗？
