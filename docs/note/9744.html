#北京周末好去处#
如果一个城市没有愿意开小酒馆的人，那这个城市无论多有钱，都只是一个内心空虚的城市。
——by村上春树
周末闲暇，不想宅家？这六个小资目的地，让你非凡过周末！

今日美术馆

@疯人院SL：
今日美术馆是北京众多美术馆中比较小众的一个，属于公益性质的美术馆。
这里一共有三个馆，分别是三栋建筑，互不相通。一号馆是最大的也是最多人去的一个，其余两个馆相对人少很多。不过就展出的内容而言，个人认为每一个馆都有必要去。
动态
打卡北京超小众美术馆，随便一个展都够你拍上一整天

❗️今日美术馆（周末免费）

中国油画院

@阿蕾Lena：
油画院简约水泥楼是中国油画院的一部分，就在大家都打卡拍照的网红教堂旁边～ 很多宝宝们来油画院都直奔教堂打卡拍照，往往忽略了右手边的这座看似不起眼的教学楼。
这里拍出来的照片棱角分明，简约性冷淡的高级感满满！一点儿也不输三里屯的CHAO酒店拍出来的感觉！

当代MOMA

@Verstandfan：
如果对建筑感兴趣，不妨来这里逛一逛，环境很舒适。而对于大多数游客，如果想在北京寻一处小众而艺术的去处，当代MOMA是很好的选择之一。
这里汇集了精品商业与国际文化的开放社区，特别推荐其中的百老汇电影中心和库布里克书吧。
动态
在当代MOMA感受清新的艺术气息

当代MOMA是由哥伦比亚大学教授Steven Holl设计的一个建筑楼盘，由于其对城市空间立体的理念和设计，当代MOMA也可谓是建筑艺术的世界唯一，如果对建筑感兴趣的你，不妨来这里逛一逛，环境很舒适。而对于大多数游客，如果想在北京寻一处小众而艺术的去处，当代MOMA是很好的选择之一。

Page One

@树游记Treep：
在北京好久没看到这么让人兴奋的店了，真的是超级喜欢。
一层以白色公开区域为主，高高的白色书架非常壮观。 二层划分为很多区域，灯光颜色材质都不同，但是毫无违和感。儿童区书架还加了软垫，很贴心。 三层还比较空，有咖啡厅，以白色为主。二层和三层都可以看到前门，古建和现代的完美融合。

798艺术区

@六六壹：
798艺术区原为原国营798厂等电子工业的老厂区所在地，由于艺术家的“扎堆”效应和名人效应，当代艺术和798生活成了北京都市文化的新地标。
这里有成批的画展和各种艺术形式。不同时间段都会呈现不同的艺术品，值得常来常新。
