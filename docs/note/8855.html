【春日嬉游记】春游纳帕谷，品酒赏花两不误

游记 摄影 美食 自驾游 踏青 2019/05/01

美图赏析

Gourmet Vlillage星厨小馆

旧金山下飞机已是饥肠辘辘，立刻赶到星厨小馆来饕餮一番。它家离机场的距离不过十来分钟，非常适合晚班抵达的游客—例如我。这是一家经典的广东餐馆，从装饰，到菜单，到服务。店里客人不多，大叔看到我们，那是分外热情，用非常标准的广普跟我们介绍它家的招牌菜，这个也好七呀~那个也好七呀~好！果断来两个经典粤菜—避风塘炒蟹和椒盐鸭下巴。来加州，那是一定要吃大蟹的。这里一般吃的是Dungeness Crab，也就是珍宝蟹，产自太平洋。它家的口味没得挑，蟹的个头很大，壳子已经炸的酥脆喷香，而里面的蟹肉则仍然细嫩鲜甜。椒盐鸭脱一份八只，性价比可是相当之高了。表面微焦，鸭舌滑嫩。我们头一天没吃完，第二天还当了早饭，微波炉一热，那个香味，秒杀屋里的所有美式早餐。
 

Towneplace suites Marriottfoster city宾馆

它家离机场不远，总体设施令人相当满意，房间紧凑但是非常舒适，住suite的好处是可以冰个甜点热个菜之类，例如前一天的椒盐鸭脱。早餐免费，非常丰盛，可以自己烤华夫饼，还有现煮的燕麦粥、麦芬、贝果、煎蛋、肉饼之类，前台的态度极其sweet，看到我退房之后在大厅拍照，连连卖萌求给五星好评，哈哈。
 

Napa Valley纳帕谷

第一次听到纳帕谷，是从一部电影开始，它的名字叫做《云中漫步》。推荐这部电影的朋友一脸憧憬的描述了纳帕谷那些个烟雾迷蒙、美妙绝伦的清晨。纳帕谷以它的酒庄出名，没错，它是美国的红酒基底，盛产各种好酒。纳帕的酒庄有四百家，除了所产葡萄酒的种类和口味不同，酒庄本身也风格迥异，争奇斗艳。这次参观的，当属纳帕最有特色的酒庄之一——爱的城堡，Castello di Amorosa。这座城堡的主人叫做DarioSattui，他的曾祖父在1885年从意大利来到加州，开创了家族的纳帕酒庄生涯。
 
拐进庄园，先看到的是大片葡萄藤。四月份，蜿蜒的藤蔓上才刚刚爬上绿色，青翠可人。为了酿出高质量的葡萄酒，这些应该都是老藤啰。这里的葡萄架搭的极其工整，简直可以用横平竖直来形容，两侧的树木也修剪的工整而笔直，仿佛一个个庄严的卫兵。这座城堡真是高度还原了欧洲的中世纪城堡—除了修建了吊桥，护城河，防御塔，教堂，甚至连酷刑室也有。值得一提的是建造城堡的85万块墙砖都是从欧洲运来的，土豪的世界我们不懂。爬上城墙可以一睹整个纳帕谷的美景，群山连绵，绿草成荫，酒庄遍布，美。
 
它家人气很旺，参加了它的普通导览+品酒tour，30美金。导览图可以选择中文，这点很贴心，也是，扭头看看周围，确实有很多中国小伙伴，看来大家还是很爱这个酒庄的。第一个令人印象深刻的地方是城堡的宴会厅，两侧墙上的壁画极其精美，这可是两个意大利艺术家用了一年半的时间才完成的。最喜欢的是墙上的吊灯，造型都是飞龙，看上去阴森诡谲，却又透着说不清道不明的萌感。而那个老壁炉，已经有五百年历史了。这么肃穆的大厅，入口两侧的墙壁居然各画了半个马屁股。。。请把墙壁和大门结合起来，脑补一下整个画面。
 
酷刑室里的器具都是第一次见，有一件是主人花了上万美金从意大利买来的古董，叫做Iron Maiden，铁处女，是一个长满了刺的铁壳。我们的导游MM还试坐了一个酷刑凳，表示毫发无损。
 
 
最独特的体验当然是地窖品酒了。这座城堡有八层之高，四层在地面以上，四层在地面以下，房间多达102间。不少VIP会员们都在这里有自己的专属酒窖，有一些特制的酒瓶巨大无比，跟普通红酒瓶比起来就是姚明和小四，还有专门收藏的空酒瓶，上面的灰已经厚成了毛衣。参观中途先品一次酒解解渴，那是正在酿造中的红酒，味道实在是。。。涩。结束之后，大家则集中在一个洞穴之中，品酒正式开始。每人选择一个酒单，既有白葡萄酒，也有红葡萄酒，按照甜度排列，可以选择五种品尝，佐以长棍饼干。酒瓶摆成一排，塞子上有各种动物雕塑，无比精致。那么问题来了，这么多品种，究竟该怎么选？废话。当然是选最贵的。但我不是，我很没有出息的选了最甜的几种，包括最后的葡萄汁。结果，最让我心动的果然是它家的葡萄汁，可能是经过发酵的原因，跟超市里卖的完全不同，要不是飞机无法随身差点准备背两瓶回家。和纳帕不少低产量的酒庄一样，它家的葡萄酒不供应超市和餐馆，所以，只能在酒庄品尝和购买啦。
 
品完酒，可以去酒庄周围逛一逛，这里简直就是一个农家动物园，打着呼噜的猪，毛色鲜亮的公鸡。。。可惜，没看见我喜欢的黑脸绵羊。听说爱的城堡酒庄都是靠蝙蝠，猫头鹰这些野生动物消除虫害的。
 

Chengdu Style Restaurant川菜

晚上来到了让人敬仰的伯克利大学，它就坐落在同名的小镇上，朋友指着山上的房子，表示那就是大学的校舍了。路上能看到不少学生，其中许多事亚裔面孔。道路两侧的街道，矗立着好几层楼高的笔直树木。
 
这家川菜馆叫做古月飘香，很雅的名字。点了凉粉、豆苗、芋头烧鸡、干锅牛蛙和传说中的老妈蹄花汤，味道都比较正宗，就是分量有点偏少。
 
 

Monterey Downtown蒙特雷

蒙特雷小镇的downtown不是那么出色，感觉比索尔万和卡梅尔的风情差一些。街头停放着可爱的甲壳虫老爷车，格外显眼。美国的小镇，怎么能少得了画廊？走在路上，随时都能看见Art Gallery室外庭院中的雕塑。蒙特雷的亮点在于它的海港。这里停泊着鳞次栉比的游艇，海水湛蓝，野花盛开，不停的看到有人拖着游艇进进出出，真真是享受生活的好地方。艺术家、音乐家都在这里大显身手。请街头画家给你画一幅肖像，亦或是驻足聆听街头艺人的演奏，一天很快就过去了。走上栈桥，就来到了蒙特雷最热闹的商业街区。加州的酸面包浓汤可以尽情的免费品尝，家家户户做的口味都有不同。几磅重的大龙虾和螃蟹，就这样摆在摊头，现点现做，保证新鲜。
 

Days Inn Monterey Downtown宾馆

在蒙特雷住的是Days Inn，属于经济型酒店，不过该有的设施都有，咖啡壶，微波炉，免费早餐。地方就在蒙特雷市中心，离蒙特雷一家很不错的中餐馆Fullmoon很近。
 

Rivival Ice Cream冰淇淋

这里有家颜值很高的冰淇淋店，门口就闻到浓郁的华夫香气。它家的冰淇淋口味不仅多，而且奇，黄瓜味、蜂蜜味这里都能找到，香气纯正，不过冰淇淋的颗粒感很强。
 

Fullmoon川菜

这是蒙特雷小镇中一家正宗的川菜馆，中文名叫做富满。加州的华人众多，连海滨小镇的中餐厅都越来越多了。它家地处市中心，停车一般是街趴。餐厅环境虽然比较普通，菜肴的口味却相当不错。点了夫妻肺片，回锅肉和孜然羊肉，都很下饭。
 

Point Lobos罗伯茨角州立公园

这是卡梅尔小镇周边的一个州立公园。

Whaler's Museum捕鲸人博物馆

捕鲸人博物馆，一个并不起眼的木屋，讲述了加州的灰鲸从当年的繁盛，到几近灭绝，再到恢复生机的过程。没错，人类对自然的认识也是这么循序渐进的。接受巨大的馈赠—以为可以任意的破坏—遭到惩罚—懂得保护。这里特色是形状奇特的松柏，真是一身傲骨，野花也开遍山坡。
然后便来到了公园最北的观景点。这里海浪很大，拍击礁石激起巨大的白色浪花，听起来穿云裂石，看上去蔚为壮观。
 
 
最著名的景观当属China Cove。碧蓝色的海湾，慵懒的海豹，缤纷的野花，哦，还有鸟岛上遮云蔽日，让你无法忽视的群鸟。最美人间四月天，加州的四月，更是人间至美。罗伯茨角有如此美景，似乎必然人烟稀少，然而并不是。热爱运动的加州人民也纷纷出动来踏青。海岸旁边，有志愿者老爷爷支架摆摊，让你触摸各种动物皮毛，了解生态知识。海边部分步道关闭，因为这是麻斑海豹的哺乳季。没错，了解、保护，然后和谐共生。
开回蒙特雷的路上，经过一片金灿灿的油菜花田。漫山遍野的油菜花也许并不难寻，可是，蓝天碧海的布景，还能在哪儿找到？冬天开过两次一号公路，觉得美是美矣，却也未到国人每每提起必心向往之的程度。开了春天的一号公路，我服气了。单纯的山清或是水秀并不难，但是像加州这样，山与海，又与人家交融，美的如此大家闺秀，却很难得了。

Carmel Beach卡梅尔海滩

卡梅尔海滩是一条景观大道，这里一侧是开阔的沙滩，波涛汹涌的海岸，以及郁郁葱葱的山丘和密林，另一侧则是满坡的野花，盆栽般优雅的巨大松柏，以及价值上千万美金的豪宅。当你老了，就来卡梅尔住下，遛狗、看海，和你的爱人在海滨颤巍巍的散步，此生足矣。
 
走你到卡梅尔小镇的市中心，你就更能感到它的优雅、富庶和精致。路上处处是景，每一座小房子都有自己的风格，通透的橱窗里，五颜六色的珠宝、油画、玩具，让你目不暇接。店铺门口种植着加州特别适宜生长的多肉，可爱至极，不少我都从没见过，不由得看出了神。路边餐馆很多，地中海菜也能找到，可谓味蕾享福，钱包受罪。都说一号公路上都是欧式小镇，像丹麦小镇索尔万，西班牙小镇圣芭芭拉，其实也不尽然。虽然他们同样的美丽优雅，美国小镇却不似欧洲小镇那般匠气十足，阳台上未必要家家种花，也没有下午茶的n种吃法，在这里，想怎么过都行，舒适、平和。
