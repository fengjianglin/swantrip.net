然后问了一个 曼谷 通的小伙伴，说考山路早就过时啦，而且不安全，现在最火的是一个叫 拉查达火车夜市 的地方！

小瑞士绵羊牧场

望最平静的泰国湾

来到 东南亚 ，也带着看海的一种执念！不想舟车劳顿，也不想人山人海，所以我们选择了 华欣 的海。从 曼谷 到 华欣 就2个多小时多车程。我觉得我们这次最准确的决策就是住在了一个海边的酒店，这样阳台走出来就是 泳池，泳池走出来就是海的感觉，你们懂吗！
华欣海滩
第一次接触 华欣 ，也是因为一个朋友说去度假，所以印象中的 华欣 就是躺在海边晒晒太阳。
华欣 是 泰国 中部海滨小镇。距离 泰国 首都 曼谷 200多公里，与 芭提雅 隔岸相望，距离 曼谷 西南281公里，约3小时行程。
华欣 被称作是 泰国 最传统的海滨胜地，皇室贵族们每年都会到 华欣 住一段时间，当今泰皇就长期居住于此地的行宫。所以可见，这边的基础设施以及自然景观都会比较美丽。

然后来到 华欣 的海滩，你会发现 华欣 的沙滩呈两个颜色，一个是白色，另外一个是黄色。其实白色的部分被称为 白沙 滩，因为白色的部分被海水冲刷，是珊瑚贝类等破碎化的产物，主要成分是碳酸钙等，因此显白色。而黄沙滩是岩石破碎产生的，主要是石英，因此显黄色。

尝最爱的泰国料理

我觉得吧，来 泰国 ，随便吃什么都好吃！
就像我一个朋友说的，吃20块钱的东西，和吃200块钱的东西，味道是一样的，因为都很好吃，哈哈哈！
那么，这次因为自由行，所以吃的最多的就是夜市啦！
 拉查达火车夜市 
最喜欢 拉查达火车夜市 ，东西不贵还好吃！
来到夜市，一定一定要跑到旁边的一个商场，就在旁边的3楼停车场去看一下这里的夜景哦！我们去的时候下大雨，如果平时去，那真的是灯火通明，超级好看的！
拉查达火车夜市
这个夜市是我对比过最便宜的夜市啦！来此必体验：
 299泰铢的榴莲自助餐，
 20块钱就能买4对耳环的小摊
 600泰铢就能让你吃撑的海鲜大餐、
 非常漂亮的彩虹吐司

 曼谷 唐人街
听说街拍女王倪妮在 曼谷 唐人街拍出了各种大片，所以我们赶紧跑来看看！ 曼谷 的唐人街道气氛特别足，各种老 北京 牌坊，各种小吃中餐店一应俱全。
唐人街
但是博主来这里就光顾着吃吃吃，忘记晚上拍点夜景了，哈哈哈！
来这里必吃：
1.网红面包店。其实就是一个很简单的面包店，把面包烤起来，然后加入自己就喜欢的酱就是了。特别简单特别便宜，所以每天排队的人特别多。推荐牛奶味和巧克力味，味道超级超级棒。
2.酱油冰激凌。好像是唐人街的特色哦，在 纽约 唐人街也有哦！
3.还有很多海鲜排挡，味道也很棒。

 华欣 夜市
华欣 夜市算 华欣 最热闹的街市了，不过夜市不大。一边是卖手工艺品，另外一边卖吃的！
我们随随便便找了一家店，点了冬阴功。只能说这道 泰国 国民代表菜怎么吃都好吃！
这道汤的名字中，“冬阴”是酸辣的意思，辣味来自辣椒，酸味是来自柠檬与番茄。“功”是虾的意思，翻译过来其实就是酸辣虾汤。汤中的主料常采用明虾或墨斗鱼等海鲜。
 河畔夜市
河畔夜市最有名的还是这个大摩天轮了，让这个夜市的景色在众多夜市中脱引而出。相对来说，河畔夜市更规整，更偏向于现代化。以各式商铺偏多！
当然，吃路边摊夜市的注意了⚠️，肠胃不好不要吃哦！博主在 泰国 拉了2天肚子，虚脱回国！所以，这里还是建议去餐厅吃！吃了几家餐厅，就推荐2家特别特别的，哈哈哈哈哈！
一家是让我的阿久根本停不下来的Mango Tree餐厅，另外一家是让我们吃了4个小时的宫廷料理Osha餐厅。
 
 阿久吃虾停不下来，说的就是Mango Tree!
其实我们也是通过另外一个 泰国 土著的推荐才知道Mango Tree，这是一家连锁，据说每次去都要预约，不然根本没位置。因为正好要去河畔夜市，就选择了Mango Tree on the river这家，环境真实超级棒呢！河 景泰 式海鲜餐厅了解一下！
这是一家 泰国 料理餐厅，主打海鲜。我们点了6个菜，5个是虾，虾是阿久的最爱啊。平时一天只吃一顿的阿久，今天吃了两个小时！
泰国 菜的口味偏重酸、甜、辣、咸。 泰国 菜以酸、辣、咸、甜、苦五味的平衡为特点。以色香味闻名，第一大特色是酸与辣， 泰国 厨师喜欢用各式各样的配料如蒜头、辣椒、酸柑、鱼露、虾酱之类的调味品来调味，煮出一锅锅酸溜溜、火辣辣的泰式佳肴。招牌菜有冬阴功(酸辣海鲜汤)、椰汁嫩鸡汤、咖喱鱼饼、绿咖喱鸡肉、芒果香饭等。
 这个龙虾泰式炒河粉是真真真的好吃！特别特别香。以前不是很喜欢吃炒河粉，因为觉得油腻腻的。但是这个炒河粉将各种原料和配料融合得非常好，加上龙虾的鲜香，吃起来那味道太棒啦！
 烤虾的色香味也是超级棒！加上蘸酱，想鲜辣就鲜辣，想清爽就清爽哦！
 来之前就一直对咖喱蟹包邮一百分的期待！果然没让我失望！因为在 泰国 我就没吃过不好吃的咖喱！这里的蟹肉丰满肥美，做成咖喱蟹，饕餮大餐中的主角呀！
 据说 泰国 人民喜欢吃糙米饭，因为不容易发胖。所以他们会在香米 里加 一点糙米，吃起来更健康。
总之，两个人吃了人均200块的泰式海鲜大餐，炒鸡炒鸡满意了！
 
 吃了4小时的宫廷料理等你来战！
这个osha餐厅的话就像 泰国 的米其林一样，特别端庄华贵，要穿正装，也要提前预约。
整个店面装修非常华贵。菜单是定制的，一大特色就是男女菜单分开。所以每次男女的上菜菜品是不一样的，分前菜、汤、 沙拉 、小食、主菜、甜品等。
这个餐厅主菜品设计以及吃法，所以整个用餐过程有一系列的表演以及用餐点评，我们整整吃了4个小时。
嗯，她的特色就是能吃4个小时的宫廷料理，人均500，想要不一样的就餐体验，可以来试试 

附最全的泰国攻略

【关于行程】
一张表格告诉你我们的行程安排！绝对走心，因为真的，以前翻过几乎所有的游记，都没有人告诉我从华欣 怎么回 曼谷 ！去了铁道市场和水上市场，怎么回 曼谷 ！
【关于签证】
我的建议是：能提早办好就提早办好～
懒人签证法：找旅行社办好，一般一周就能出，签证费300元左右～
落地签：带上护照、一张2寸照片、酒店住宿信息和机票信息，2000泰铢签证费（身上要带20000泰铢同等现金）。
入境，填写入境卡和落地签申请表，提交审核拿到落地签整个大概需要2个小时的时间，大家衡量好哦！
【关于货币】
1元～5泰铢，国内 中国 银行可以预约和换汇，要前一天3点前预约第二天才能拿哦！这样的换汇是最方便哒～或者就是机场换，这样的话汇率会低一些～
当地ATM取钱一般都需要收手续费，30-50不等。
【关于交通】
指路就用谷歌地图，绝对出国好帮手！
打车就用grab， 东南亚 的滴滴，了解一下！（就是 曼谷 的司机方向感都不太好呀，每次打车都要等好久的说！但是我愿意打，因为这些私家车的司机态度要比出租车好的多的多！）
如果问路遇到语言困难，就用谷歌翻译！因为 曼谷 有些人的英语真的不好呀！
【关于住宿】
为什么要酒店列出来呢！因为赚了这么多年的钱，终于有一次的旅行自己舍得花钱全程住豪华酒店了！

【关于穿搭】
想了这么久，为什么每次出门都要穿的美美的？
因为每个女孩都希望：自己走在人群中，能被记住。
