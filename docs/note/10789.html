金秋川西 | 去稻城亚丁赴一场秋天的童话

游记 摄影 登山 自助游 自然奇观 2018/11/04

初次见面，你好稻城

两天的路程只为了 稻城 亚丁 ， 稻城 是 四川 省 甘孜 藏族自治州下属的“县”，“ 亚丁 ”是 稻城 县的一个村，在景区内。我们来的时候是工作日，游客依然不少，还有很多摄影队。
稻城 处于 川西 高原，雪山、冰川、草甸、林海、溪流、湖泊，这里拥有雪域高原的所有元素。还有仙乃日、央迈勇、夏诺多吉三座“神山”鼎足而立，圣洁巍峨。
进入景区，你会发现这里有神圣的雪山，辽阔的草甸，斑斓的树林，五彩的经幡，碧蓝的海子，真的是五步一景，十步一花，我们来的时候正好是金秋季节，松树的叶子全都泛起了金黄，仿佛置身画中。

路线短，但景色不打折扣的冲古草甸

第一天周姐帮我们安排了短线，用时和路程相比长线要短，攀爬难度也容易很多。
下了大巴车往前走一公里左右就能看到冲古草甸了，这就是《从你的全世界路过》的取景地，也是茅十八单膝跪地，向荔枝求婚的地方。
大片的草甸，配着碧水，很容易让人陷入那个浪漫的画面，就这样呆呆的看着它，旁边的溪流时而舒缓，时而急促，这真是大自然赋予我们的美妙乐章。

茅十八的求婚地——冲古寺

看过《从你的全世界路过》的对这个地方应该很熟悉了，茅十八去世后，荔枝一个人听着茅十八为她录制的私人导航提示：“荔枝，你又来冲古寺了吗？这里定位是冲古寺，我向你求婚的地方，抵达这个目的地呢，我就会对你说，因为是最蓝的天，所以，你是天使。荔枝，我在想，当你听到这段话的时候，是我们结婚一周年呢，还是带着宝宝自驾游呢，我站在那一天的天空下，和今天的自己，一起告诉你那个秘密吧，我要让全世界都知道，荔枝，我爱你，永远爱你。”
站在这里脑子里就有了画面感，仿佛亲眼目睹了茅十八向荔枝的求婚，来到这里，有雪山为证、碧水为盟、树木为鉴，该是多么纯粹的爱啊。
看着眼前的美景，会觉得不负今生。余生，我想和爱的人，走遍世界的山川河流。
怕打扰到僧人念经，我没有走进去，只是站在门口就感受到了严肃的气氛。

像翡翠一样的珍珠海

上山时我们走的是新铺的道路，都是阶梯，走起来很轻松。只不过绕的远，用时多，下山时候跟了一个当地人走近路，不到20分钟就下来了。
旁边的阿姨因为体力原因，又不想错过这番美景，雇了一个当地人帮忙背包和扛相机，村民一路对阿姨很照顾，全程牵着阿姨慢慢走。
去往珍 珠海 的路上卫生间很少，提醒爱喝水的朋友少饮水，也不要指望山上有商店可以买吃的，爬山可以带些士力架、葡糖糖补充体力，又不是来这里野炊、郊游，没必要带太多零食，竟增加负重。
我们只顾兴奋呢，一个个都没带吃的，中途休息的时候，和朋友说“我现在只想吃面包”，结果旁边的一位好心姐姐说“我带的东西多，你介意吃点儿吗？”妈呀，平生第一次“化缘”啊，感谢姐姐的面包给我提供了继续上山的动力。
路上遇到了很多可爱的小松鼠。
到了珍 珠海 ，感觉眼珠子都要瞪出来了，海子的水像翡翠一样碧绿，在阳光的照耀下blingbling的，完全不需要特效。只是爬这1公里的路感觉像是走平时道路的四五倍。
在山上还活蹦乱跳的我，下山直接挂了，发烧、头疼，出了景区赶紧找了家可以吸氧的酒店，当地居民说“还好机智的你来吸氧了，要是去医院，医生也是给你开单子去吸氧，有的也可能直接让你买机票回去了。”
觉得有点儿丢人，还是把脸蒙起来吧emmmm
我的身体情况自己清楚，确实是有些轻微高反，在山上没怎么吸氧，一路太兴奋了，消耗了不少元气，提醒大家爬上过程要保持平静的心态哦。
晚上没什么胃口的我喝了瓶酸奶，吃了自带的感冒药，担心第二天走长线再缺氧，当即决定租一台制氧机（300元）。睡一觉起来，啥事儿没有，连头疼都没有了，大概是有些适应了吧。

长线首站——洛绒牛场

全是草甸，有岩羊在远处吃草，下山回来的时候，景区内的马帮也都开饭了。看到马的屁股上都有一块块不长毛，我傻啦吧唧的问鹏哥“这马是有皮肤病吧，那一块怎么不长毛？”鹏哥一脸嫌弃的说“那是装马鞍磨的了。”心里忽然有些不舒服，心疼他们，所以景区规定200斤以内的游客可以骑马也算是爱护马了吧。

第一次见萌萌的岩羊

第一次见长这样的羊，看起来萌萌的，有点儿像鹿。景区里好多岩羊，拍照的时候不要打扰它们哈，毕竟是野生的，只有我们对它们友善了，它们才不会对我们示弱敌人，保持一种友好的关系对大家都好。

圣洁的雪山让人心生敬畏

一路雪山一路拍，简直太壮观了，镜头拍不下它们的魅力，只有亲自站在山脚下，对大自然的敬畏之心才会油然而生。

牛奶海的壮观值得这么远的路途

这路是真难走，长线的路不像短线的路都是新铺的，而且有护栏阶梯，这里都是野路，又窄又陡，一路全靠登山杖，到最上面的危险地段，才铺了一点儿木头，还有溪流，我们是不到4点下山，道路已经有薄冰了，走的时候一定要小心。
有几次想放弃，越往上走风越大，吹的人呼吸不畅，心想要不下次再来吧，可是转念一想人生有几个下次？要珍惜眼前机会（当然还是要是身体状况而定，不能强撑，更不能轻言放弃）。
一路很多下山的人会鼓励上山的人，“加油，没多远了”“坚持就是胜利，再有10分钟就可以看到漂亮的牛奶海了”“上去之后直走就到了”“上面的路比这里好走”，其实时间没有这么短，路也没有这么好走，不过还是要感谢陌生人善意的谎言，给了我心里暗示，才让我坚持走上来。
看到牛奶海的瞬间，想哭，既因为爬上来不容易，也因为牛奶海真的好看，清澈透蓝。
牛奶海是纯天然的古冰川湖，四周雪山环绕，和茫茫雪山相比，它的清莹碧蓝更让人眼前一亮，觉得爬这么久值了。海拔4500m，吹着7、8级的大风，望着眼前的美景，觉得活着真好。
一路和同伴调侃，我们从雪山脚下爬上去，能够爬到终点，以后就可以吹嘘自己是爬上半个珠峰的人啦（当然珠峰比这个要难爬很多）。
在前边耽误的时间有点儿多，担心赶不上最后一班电瓶车（6点半），就没有爬到五色海，算是一个遗憾吧，期待下次爬上去一睹芳容。
问了几个爬上去的朋友，他们都说五色海和牛奶海相比，略逊一些，大概是季节的原因，现在看上去海子的颜色没有五种，只是微微泛着碧绿，所以说，这里的四季都很美，各有各的魅力。
牛奶海也是因为每年开春的时候，海子周边的冰呈乳白色像极了牛奶而得名。
下山的时候轻松多了，一路没有休息也不觉得累，走到最后大概是麻木了吧。
出来的时候天已经完全黑了，所以，上山要趁早，可以有多余的时间看其他景致。
有一个叫 稻城 的地方，一定要和心爱的人去一次。

关于路线

1、短线： 稻城 - 香格里拉 镇游客服务中心（乘大巴车40-50分钟）-扎灌崩（大巴车终点）-冲古草甸、冲古寺、珍 珠海 （卓玛拉措）、仙乃日神山-返回 亚丁 村或 香格里拉 镇入住。
2、长线： 亚丁 村或 香格里拉 镇游客服务中心乘大巴车-扎灌崩-冲古草甸（乘电瓶车10-20分钟）-洛绒牛场（电瓶车终点、徒步起点）-牛奶海（徒步）-五色海（徒步）-原路返回洛绒牛场（乘电瓶车）-扎灌崩（乘坐大巴车）返回 香格里拉 镇- 稻城 。
返回的时候坐大巴车，看清指示标识，分“ 亚丁村”和“ 香格里拉游客中心”排队乘车。

关于高反

稻城 的平均海拔有4000米，会伴有轻微高反，可能出现胸闷、气短，头痛，甚至呕吐等症状，一般1-3天就能好转。严重贫血、高血压、心脑血管疾病的患者不宜 上高 原。出行前，服用红景天，并注意休息和锻炼。旅行中，多喝热水，不要暴饮暴食，保证睡眠，不要剧烈运动，勤添衣物，不要受凉感冒。

关于费用

门票150元+大巴车120元必买，第二天再进入景区还需购买大巴票，半价60元（需向工作人员展示自己前一天在景区的人物照），先坐大巴到扎灌崩，之后步行1公里左右到冲古草甸那边购买电瓶车票80元/张。
住在景区外，门票2天有效；住在 亚丁 村的（期间不能出景区），门票3天有效。如果从外地赶来不建议住在景区内，海拔较高，休息不好，整晚会伴着头痛、心跳加速、呼吸急促。

必备物品

1.进入 稻城 亚丁 景区少带些干粮，景区内商店很少，不要指望着能在景区内买到吃的；
2. 补水喷雾、防晒霜；
3. 多带些软管葡萄糖，感到疲惫时来一两支很有用；
4. 氧气瓶，觉得氧气瓶不够用的可以租一个制氧机（我是第二天走长线的时候租了一台，一路靠着它才走到牛奶海）。

注意事项

1.进入景区走长线的话，不建议骑马，忘了是500元/人还是300元/人，走的路也不长还要自己走，想骑马的话可以跟包车司机讲，让司机帮忙找马场，价格也便宜，180元/人；
2. 进入 稻城 亚丁 景区，不用穿的跟 北极 熊一样厚，爬山过程会很热，穿得太厚也不方便爬山，贴个暖宝宝也很暖和，最主要的是护好脑袋，本来就有些轻微高反，头顶像被重物压着一样，爬到牛奶海和五色海上面风巨大，再一吹很容易感冒；
3. 不要图省事，直接飞 稻城 机场，好几个朋友当时想着有机场正好直接飞来，结果飞机落地之后受不了高反直接买票回去了，有严重的，拉去 成都 医院，待了几天直接回家了，其实高反并不可怕，但是需要我们慢慢适应，坐车去就是一个适应的过程；
4. 高原地区紫外线强，一定要做好防晒，还有补水，菇凉们如果晚上不累最好敷个面膜，唇膏一定要带，我唇膏隔一小时涂一次，依然当不住嘴唇起皮。
