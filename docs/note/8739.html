黄山最该去的无敌美古村落，游人寥寥，让宏村西递一边儿呆着去吧

南屏村——千年古村

仿佛每一个名气大的地方都有那么一两个“功高盖主”的最知名景点，压过了群臣，盖过了帝王，成为了一个地方的代名词。而最知名的景点却往往不是最值得去的，它人头攒动，名气到达了抛物线发展的顶峰，商业化程度极高，失却了本身的文化，价值低到谷底。
第二次去黄山，诀别了94块钱门票的宏村西递，不再看村民表演生活给游人，不自己花钱找罪受，高价门票加缆车加景区小交通，还得累一贼死上黄山。从北京乘高铁5个半小时到黄山北，租车公司的人接我们去提订好的车，我和我爸的第25次旅行开始了新征程。
或许因为清明刚过，五一未至，我们开过的每一条路都空旷，寂寥，车极少，甚至有的路只有我们一辆车。天色阴沉，一个小时后在天黑前抵达了黄山市黟县碧阳镇南屏村。

中国古祠堂建筑博物馆

南屏与宏村、西递同为黟县下辖的村落，在县城西南4公里的地方，曾名叶村，从元朝创建至今有1100多年历史。全村1000多人，有36眼井、72条巷、300多幢明清古民居，至今保留着8个相当规模的祠堂。有属于全族所有的“宗祠”，也有属于某一分支的“支祠”，还有属于一家或几家的“家祠”。
宗祠规模宏伟，家祠小巧玲珑，形成了一个风格古雅颇具神秘色彩的祠堂群。这种祠堂群在全安徽省也仅此一家，被游客誉为“中国古祠堂建筑博物馆”。村内道路错综复杂，宛若一个大型迷宫，因此又被称为“百巷村”。
“邑俗旧重宗法，姓各有祠，支分派别，复为支祠。”从村头到村尾，大多坐落在村前横店街长约200米的一条中轴线上。
其中最出名的宗祠当属村心的叶氏宗祠“叙秩堂”，始建于明成化年间，距今530余年历史。1989年张艺谋导演、巩俐主演的电影《菊豆》80%的镜头都在此取景，现在仍保持着当年拍摄时的原貌。
南屏村的支祠与家祠大都由做官、发财者建造。叶氏支祠，即奎光堂，始建于明弘治年间，距今490余年历史。1999年，李安导演、周润发主演的电影《卧虎藏龙》就选此为剧中雄远镖局的内景地，章子怡和杨紫琼比武的地方就是叶氏支祠。
《复活的罪恶》、《大转折》、《徽商》等多部享誉国内外的名片，以及湖南卫视的《一年级》都在南屏拍摄，许多拍摄时的场景和道具都保持了原样，宗祠的墙上还挂着大量片场纪实的照片。南屏也被称为中国影视村。
如今的祠堂之中，门前，坐着一排排前来游学的未来“画家”，他们的眼睛好比尺子，落笔如有神，或速写出徽派建筑的神韵，或精确测量着古建筑的尺寸，画图纸，也时而嬉闹、说笑。

南薰别墅——深度体验徽派建筑

南屏村的古私塾园林和古民居建筑也比比皆是，“培阑书屋”、“陪玉山房”、“梅园家塾”……其中最著名，也是最能近距离体验的非居仁区42号的南薰别墅莫属。
始建于清道光年间，距今约170余年，因为它的大门不开在正厅的中轴线上，而是开在正厅左侧，使人有一步到堂之感，正对风光秀丽的南屏山而得名。
南薰别墅的大门正厅是明三间布局格式，宽敞明亮，光线透进宽大的天井，一直能照到厅堂后部。太师壁正中挂有“忠孝传家永，诗书处世长”金底黑字楹联。与所有徽州古民居一样，条案上摆放着东瓶西镜、自鸣钟，取其谐音“终身平静”。西边窗下分别是“吕洞宾三戏白牡丹”及“秋胡戏妻”图。
二楼为小姐的闺房，名为“万云轩”，绣楼的摆设体现了主人当时的富有，绣楼上的镜子及美人靠上的彩色玻璃都是当时主人经商所购的外国货。
如今这座中西合璧式的古宅建筑，早已是南屏闻名遐迩的民宿，作为民宿拥趸，听闻已多年，探访它也成了我行程中的重要一环。早晨和下午两次走到门前都大门紧锁，次日即将离开南屏，未来还不知何年何月才会重返这里，顿觉都到这儿了不进去看看太遗憾，于是在网上找了店里电话拨过去，一个大约四十来岁的男性接了电话，得知我已经有住处，而非要住宿，也丝毫没有抵触的只言片语，告诉我因为客人都是网上预订，清明刚过这天没有客人就锁了门，他在镇上，让我等一个小时。
第三次登门时主人依旧未归，走回住处的路上与一个年长面带笑容，颇有气质，肯定不是这个村子村民的人擦肩而过，我猜他可能是绣楼的老板吧，又不敢冒昧询问。几分钟后我再次拨通了电话，第四次登门时果然印证了我的猜测，这位彬彬有礼，温文尔雅，笑容可掬，声音颇为年轻的长辈，正是绣楼的主人，大家在网上评价中的“宫老师”。攀谈中聊到各自的职业，我们居然是同行，他曾是广东著名报业集团的特刊主编。我终于知道他这个“老师”并非教书的师父，而是取别人对媒体人的称呼。
宫老师带我参观了多个房间，一路都是笑着给我讲，给我开灯，让我拍照。言谈举止都透露出大文化人的气度。他比我爸年纪大，他做媒体的时候还没我呢。
他说一般是不让参观的，如果让门坎早被踩坏了。我问那为什么让我来呢？他说听你的声音和语言也不像坏人。他并不知道，未来的某一天，他的这个想法和推测，以及这一切会被晚辈媒体人写在游记里。
绣楼在深深的小巷尽头，从一大面墙都是爬山虎的小门进去，光线有些昏暗，房子像个迷宫，多个出入口连通着多个功能区，两个院子，5个厅，一个露台，6个房间，三楼是个公共空间，兼具了茶室、棋榻、书吧和咖啡吧的功能。古时候都是长辈住一层，因为楼梯陡。这座绣楼处处布满砖雕、木雕和石雕的徽州三雕，祠堂都是香樟木建造，用来防蛀。
房型多样，有外国人喜欢的我怕的老床房，有徽州大床房和阁楼双床房，四五百块钱的房价，是体验徽州传统建筑文化的好地方，宫老师说他开这个民宿就是玩儿，给看店的店长伙计挣个饭钱。他还带我看了一间宝贝大床房，一般人不让进，正是那间150年前德国玻璃窗的绣楼。

村外美景——只缘身在此山中

南屏是黄山众多收费景区中渺小的一个，名气被宏村、西递、徽州古城、呈坎、塔川压在身下，后起的免费碧山正在崛起。43块钱的门票在网上买最低30块钱到手，本就游人不多的南屏，清明后更是寥寥无几，规模宏大的古村落也恢复了它该有的模样。除了一些写生的孩子，只有零散的几个北京口音的大妈结伴而行，我和我爸这样的父女档是唯一。
黄山的各个景区村落长得极其雷同，区别是哪个门票贵，哪个商业化轻一些，哪个人少。在景区入口处可以免费约到导游，她们带着游人参观村子，讲解古今故事。而我，走到一半就不跟了，因为只缘身在此山中。
南屏比局部的精巧徽派建筑更美的景色其实在村外。雨后的南屏，如桃源仙境。远处的群山被烟云笼罩，与近处的菜园、庄稼，粉墙黛瓦形成一幅幅天然画卷。
广袤的油菜地，花已落幕，剩下枝条渐鼓的油菜等待出落成囊中饱满的豆荚，这是我第一次看见油菜花的菜，原来它不是平时吃的油菜，而是榨菜籽儿油的菜。耕牛独自在田间发愣，小鸟不时站在它身上嬉闹。一条条大河小河绕过村庄，倒映着绿树和云天。
在南屏的游玩儿项目中，不能错过的七约农场和染坊，我都错过了，最后一天的时候才知道。好在，我没有错过南屏最美的时候，雨后阴霾比响晴薄日更加洞彻人心。
徽公馆私房菜
南屏整个村子并没有正经的徽菜馆，我们开车去了6公里外的县城，在网络和朋友中都有好口碑的徽公馆私房菜。
安是安庆，徽是徽州，康熙六年就是1677年，两府首字合起来成了安徽。一般说的徽菜指的是徽州菜，黄山所在的地方正是古徽州所在，所以在黄山这边吃的才是真正的徽菜。记得那年在南京的饭桌上，陈晓卿还是小宽老师说过，因为当年盐商是巨贾，超有钱，有钱人家才吃得起盐，所以多放盐能显示有钱，就这样传到了现在。但是徽公馆的菜并没有，咸淡很合适。
徽菜以山珍野味著称，像笋啊就是从地里现拔出来的，非常鲜嫩。徽菜浓油赤酱，重油、重色、重火功，烹调方法上又擅长烧、炖、蒸。像徽菜里的咸肉、咸鸭、咸鸡、臭鳜鱼等都是腌制品，是经过“腐败”过程，最后成为腐而不败的名菜，成就了特殊香味儿。所以在民间，就有了一种对徽菜诙谐的说法“轻度腐败，破坏生态，盐重好色”。
臭鲑鱼和毛豆腐是游人来黄山不可不尝的美食，但是两个都点又吃不完，所以店里就有了一道徽州双绝，上边是臭鳜鱼，下边是毛豆腐。我在北京吃过的毛豆腐略有硬度，而徽州的毛豆腐跟王致和酱豆腐类似，内心绵软，口感顺滑，让你禁不住再来一块儿。鳜鱼身下有虎皮椒，虎皮椒并非辣椒的一种，就是普普通通的辣椒，因为遇热，表皮褶皱像虎皮而得名。微辣下饭，很好吃。
刀板香是徽州名菜，其实就是腌咸肉，据说是胡宗宪去拜访恩师时，师母将家中腌制的猪肉平铺于山笋上，放置在刀板上一同蒸，捞起切成薄片，和刀板一同端上桌，胡宗宪大爱，就命此名。后人有些改将其放在上等香樟木板上，所有油腻皆被木板吸走，保持肉的咸鲜却油而不腻。刀板香最丰盛的季节在清明前后，那时走进徽州任何一家农户家，墙上多挂着腌制好的刀板香。
为主食控强烈推荐一款！当地特色的焖面一定要吃！它是将面放在水中煮，半生的时候就捞出来，接着放锅里炒，加笋片肉丝儿、火腿、香菇和笋丁制成，面质软硬适中，有嚼劲儿。和面一同上桌的还有一种辣酱，不是很辣，提味儿下饭，超级好吃，一份儿量很大，碍于饭量实在吃不下了。
黑灰色那个小锅叫养生米糊，看着不好看，吃起来舒服，咸味儿的，包括主料大米在内的12种杂粮和蘑菇做的。

黟县晴园民宿

它距离屯溪国际机场和黄山北火车站一个小时的距离，相较于其他那些景区都算近的，作为首站最合适。北京直飞黄山的航班都在晚上，一来浪费白天光线好的时间，二来浪费订酒店的银子，坐火车白天到，然后租车自驾是最合适的选择。

前厅兼餐厅

南屏景区里的精品民宿并不多，我把住处订在了网上评价很好的晴园民宿，当我看到网友发的那些照片上有三只好看又乖的猫时，对这里更加充满了期待。
顺着导航拉着行李，沿着曲折的村中小路，走一会儿就到了民宿。三只猫只看到两只，白色的叫粽子，三花儿叫皮蛋，可怜的俩孩子蹲着门口，是迎来送往，还是伺机逃窜？当我在这儿经历了三天两晚后，我确定一定是前者，这儿太舒服，太好吃，还有爱它们的媛媛，就是猫天堂和人天堂。
初次见面的管家媛媛，穿着一身纯色棉布中式裙子，朴实大方，看到我们自己拉着行李找来了，像见到旧友一样，说怎么没叫她去接，我们甚至没有一秒钟陌生的隔阂，用我自带的面部识别系统得知，这个姑娘是善良的好孩子。
对于一个酒店发烧友来说，再没有比住上喜欢的店更开心的了。放眼望去，温暖、亲切，一眼望不到边的多空间，俩眼睛完全不够使的。
民宿由两栋相连的古民居组成，前边一个大院子，后边一个小院子，前厅和大公共空间的建筑，本是村里的老供销社上厅，后院是明末一位教书先生的旧居。
整个建筑面积900多平米，只有8个房间，所以公共空间就特别多，300多平米！民宿主牺牲了多少挣钱的空间给客人增加好的住宿体验，把酒店变成了乐园。
进门的前厅是吧台，办入住的前台，也是咖啡馆和餐厅。老木头桌上时时都有自取的祁门红茶、水果和饽饽……像在家里，饿了有的吃，渴了有的喝。
陶罐的干花与灯光配合默契，不论哪个角度都是好看的一副画。

2-3层健身器+书屋

两层半的建筑每一层都有公共休息区，二层是客房集中分布的地方，中间设有沙发区。
三层是一个独立的小空间，就是两层半的半，这是民宿的三个茶室中的一个，宽敞亮堂，视野开阔。健身的朋友可在这儿蹬蹬自行车，闲来无事的也可以免费取阅书架上的多种图书。

后院+茶室

整个民宿中最好看，最出片儿的地方是后边的私密小院儿，我在黄山的8天，拍到最满意的照片除了猪栏老油厂就是有着恰到好处禅意，融合潮湿斑驳墙体，颜色艳丽沙发的小院儿。
红灯高挂，傍晚时分灯光乍泄，傍晚时，掉点儿了，媛媛忙把塑料布苫在沙发上。
晴园民宿之所以叫晴园，因为民宿主名为舒晴，人们都管她叫晴姐，网上可以查到对她的介绍。她是早年北京的媒体人，后回到家乡和爱人开了这间民宿。自己深谙茶艺、经络、医术之道，在店里时就会为客人泡茶，聊聊周身的不适，不好的生活习惯，还会亲自上手调理肩颈、后背，整个就是免费的保健医生。

多功能公共空间

回到一层整个民宿最大的公共空间，地暖早已打开，这时我终于见到了第三只猫小夏，超级无敌好看的毛色，灰黑色美短，手感很棒。孩子们鬼着呢，知道哪儿暖和，平时没事儿就在地上趴着，像电褥子一样熥着。
三只猫里有两只是捡的，从不到一个月大小，一直养到现在。孩子都被媛媛宠坏了，有一天小夏特别不爱理人，小表情很生气，媛媛抱着哄了半天，还是没好。
村子里游学的孩子经过门口时都会好奇地看院子，我上小学中学的时候，也曾是他们中的一个，抱着画板跟着老师画色彩。南屏的商业化还低，经过两天的巡视，以我的审美取向，晴园是唯一复合要求的“咖啡馆”。这天下雨了，孩子们拿着器材坐满了晴园一楼的公共空间，他们依旧勤奋地照着手机里的照片，创作着。
公共区域采光通透，彩色玻璃直接过滤了阳光，自然光投射到室内，照在舒服的布艺和皮沙发上。喜欢喝茶的有榻榻米茶室，喜欢看书的，自己找个地方窝着，我爱看电影，晚上就听着淅沥的雨声，坐在大厅的地上暖暖的，大白猫趴在我腿上，看投影《碟中谍》。

客房

第二天，前边的客人退房了，后边的客人还没来，我就趁这个机会参观了所有房间。晴姐将自己的生活状态作为了八间客房的名字。
一庐独院蜜居大床房在院子里，是八间房子中最大的，25平米，也是唯一有超大落地窗和榻的一间，视野很好。
双邻古风单人间是唯一一间单人房，在二楼，我爸住的这间。老墙上开了个木头小窗户，用于采光，也方便看外头的风景。虽然只有15平米，由于设计的巧妙，一点也不觉得狭小。
三友茶榭标准间是个标间，在二层，这个房型我很喜欢。进门是个对坐的小床榻，L型的空间，经过门厅，拐弯儿才是卧室，保留了一定的私密性。
四季山景大床房也在二层，它的视野很好，可以看到窗外层层叠叠的徽派建筑屋顶。
五趣园景标准间是个标间，有两面景观窗，不论是躺在床上，还是泡在浴缸里，都能欣赏到徽州的美景。
六艺园景大床房也是观景大床房，都在二层，视野开阔。
七夕情侣大床房就是我住的房间啦，床品舒服，蓬蓬的松软。细节令我意外，住了那么多酒店，第一次见马桶圈的提手，干净卫生。
入住的时候，皮蛋比我还先进门，它先跳上沙发，又跳上床东看西看，一点儿也不认生。
我超级喜欢这个屋子！因为上下两组竖版的片子，都是在我的房间里拍的，窗外就是层叠交错的马头墙，俯视即可看到后边整个私密小院，茶室的红灯笼高挂，不论是阴雨蒙蒙，还是艳阳高照，都是绝美的角角落落。
住在这儿的第一晚下了一夜雨，声音很大，我坐在被窝里听着，想起18岁之前都住平房，想起雨打芭蕉的声音。那时候我们家院子有70多盆花儿，我屋外就是芭蕉扇和橡皮树，雨点儿打在大叶子上声音特别大。芭蕉扇是做夏天大蒲扇那种叫蒲葵。许多年后，在陌生的千年古村里，再次情景重现，特别感动。现在，当年那般期盼搬上的楼房，雨多大都不见。
八音园景大床房在一层大厅的一侧，门非常隐蔽，如果不细看根本不知道里头还有个房间，这间是最小的大床房，15平米。

早餐+晚餐

晴园还可以是很棒的餐厅。正餐按餐标配餐，或单点。媛媛在第一天我们还没到的时候就发给了我菜单，当地大姐做厨师，做着一手地道的徽菜家常菜，按照我们点的餐和到达时间做好。我和我爸两个人，饭量小，一荤一素一饭一汤，都没有吃完。
早餐比正餐还棒，每天都不一样，超级丰盛！有桂花糕，红糖发糕，芝麻饼，老面饼，肉馅饼，蒸饺，小笼包，粽子，艾叶粿，小菜，芝麻糊，红豆薏米汤，银耳羹，南瓜西米羹，米粥，皮蛋瘦肉粥，小米粥，蔬菜粥，红豆沙，水果，水煮蛋，茶叶蛋，咸鸭蛋等，吃完了早饭，到下午两点都不饿。
吃过早餐后，建议你可以体会一次正餐，对比之下如果还觉得早餐好吃，那有个法子，正餐让厨师大姐还给你做早餐的内容。如果是我，再去就这么干。
次日，民宿里的客人只剩下我、我爸，和刚来的一个单独的姑娘。第二天的晚饭，晴姐回到了店里，邀请我们一块儿吃晚饭，同坐的还有厨师大姐和媛媛，像一家人一样共进晚餐，晴姐的爱人还跟我爸喝酒……
我在酒店网站的评价里发现，出现最多的一个词居然是“媛媛”。主人在家才是民宿，晴姐偶尔在店里，但是媛媛一直在。在我看来这个店的最大亮点并非设计、古建、硬件，是店里唯一负责接待服务的姑娘媛媛，她像待来家的客人一样，聊天儿，爱猫，是民宿的灵魂所在。
住过很多酒店、民宿，这是唯一一个让我觉得不是在住酒店的民宿，让每个客人感觉，自己只是借宿在朋友家，这才是真正的民宿。
Tips：
1、南屏不大，距离屯溪机场和黄山北站比较近的景区。自驾过来约一小时，作为首站最合适。
2、公交要倒三次，一不方便二时间不靠谱，打车200吧，建议自驾，导航很准
3、门票提前订，最便宜30，不必在意景区营业时间，西、北两个入口全天都能刷身份证取票。实名制、多天进出均有效
4、景区俩免费停车场，分别在两个入口处，北边的大，西边小的距离民宿晴园最近。
5、附近有陶渊明后人居住的守拙园、关麓、黟祁徒步古道等景点，我一个都没去
