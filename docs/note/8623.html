【天鹅之旅首发】那山、那水、那花海，项王故里的自然之旅

游记 人文游 小众 自助游 2019/06/05
了解小城宿迁，是从它是西楚霸王项羽的故乡开始。
这座城的面积不大，却有着5000多年的文明史和2700多年的建城史，正儿八经的小小的身体，大大的内心。
除了悠久的历史，自然位置也是十分优越。地处长三角北翼、江苏北部，坐拥骆马湖，洪泽湖两大淡水湖，享有“华东净土、江苏氧吧”的美誉。
寻一个周末，与友相约，到宿迁，过两天安静悠闲的日子。住房车、徜徉花海、与项羽隔时空相约、大快朵颐的湖鲜宴……都是之前从未有过的体验。

一梦千年，项王故里的霸王柔情

中学历史课本中，以少胜多的名将。
“霸王别姬”的绘本故事中，情意绵长的痴情汉。
李清照名句“至今思项羽，不肯过江东”中惜败的西楚霸王。
项羽的故事为众人所知，战功赫赫的他，情真意切的他，凄凉败北的他，都只是他的一部分，而所有这一切加起来，并不足以概括这个传奇人物的一生。
2000多年前，出生于今江苏宿迁，他曾领军灭亡秦国，后四年的楚汉战争中，败于刘邦，自刎于江边。后人感怀，修筑项王故里，这也是中国首家以项羽和项家文化为主题的项家宗祠，国家AAAA级景区。
到达景区门口，立刻被门前这尊塑像吸引。项羽坐于马上，身形保持战斗的姿势。身下的乌骓马通体黑亮，如同黑缎子一般，唯有四只马蹄雪白。“踏云乌骓”的比喻甚是形象。明明只是一尊塑像，这一人一马的一身煞气扑面而来，让人没来由的备受压迫。
项王故里景区的建筑风格以楚地汉风为主，兼具宫廷建筑和园林建筑特色。换上景区提供的特色楚服，立刻就有了一种穿越回两千多年前的错觉。
楚服亦是汉服的一种，便是楚人所着的衣物。楚服颜色大多温婉，但还是凭直觉选一身正红色，只觉项羽的故事惨烈，与这火红色才最为相配。
景区的主体建筑是三进院落。前为高大的汉式石阙，象征项羽故居为帝王规格。中院以英风阁为主体，阁内为项羽高大塑像。四面墙上嵌着反映项羽生平十二幅浮雕，如项羽举鼎、吴中起兵、破釜沉舟、巨鹿救赵、鸿门设宴、垓下突围等等。
项府、项园、将署、项羽故居、项家宗祠、霸王古今馆、虞家老宅……围绕着项羽的生平，景区的设置可以说十分完备。从他的出生，到幼年生活，甚至少时即相识的虞姬生平，都能在这里寻到。
项羽生平最知名的两场以少胜多的大战——巨鹿之战和彭城之战，堪称军事上的神话，也是项羽战功赫赫的功章簿中，最不能缺少的两笔，在景区里，设有专门的展馆，为游客展示当年的战争盛况。
相传，项羽幼时曾种下一株槐树，据说原来的树干已死，这是从根部重发的。如今这棵有着2000多岁寿命的槐树被稳妥保护，在庭院中，带着一身岁月的痕迹，依旧生机盎然的存活着。能亲眼看到项羽手植槐，就已经不虚此行了。

有山有水有花田，这个景区四季有景

这个森林国家公园，因为大片大片的花田而远近闻名，公园内的“衲田花海”，总面积2220亩，由612块花田 “缝补”拼接而成，不仅面积大，而且种类多，足足有50多种花。春看百花争艳，夏有曲水风荷，秋赏红枫之恋，冬来梅村煮雪，四季都有独特的景致，新鲜又美好。
站在全长2.3公里的彩虹桥俯瞰花海，听花开花落，闭眼轻嗅，缕缕清香在身畔流淌。
三台山森林公园位于江苏省宿迁市，坐落在骆马湖畔，占地12.7平方公里，是国家级旅游景区。这里树木古朴葱茏，蓬勃参天，森林覆盖率达98%，负氧离子含量高达2万每立方厘米，远超一般的公园和城区，是苏北最大的天然次生林，被誉为中心城市的“氧吧”和“后花园”。
在三台山景区，最妙的是不仅可以在这郁郁葱葱的自然风光中徜徉欣赏，还可以塑在森林里。景区的房车营地中，几十辆房车披着卡通的外衣，特别受小朋友的喜欢。周末时爸爸妈妈带着小朋友，来这里看看花散散步，宿在房车里，也是一段舒心的体验。
除了安安静静的赏花拍照，在三台山森林公园，当然也要来一点儿野的。这超高的森林覆盖率，郁郁葱葱的森林公园里，最适合一场cs大战啦。配上景区的cs战服，子弹上膛，局面立刻紧张起来。
放下枪，还能立刻拿起手边的箭。在cs场旁边就是射箭基地。工作人员一步步教你拉弓，射箭，怎样才能射的更准。拉满弓，不知不觉间连眼神都犀利了起来。
或者带着宝宝来一次大冒险，树林之中设置的拓展训练项目，分为成人区跟儿童区。儿童区可供家长带着小朋友一起玩儿。适度的惊险刺激，加上趣味性，让活泼爱动的小朋友很是喜欢。
别说小朋友了，连我这个大朋友，也摩拳擦掌，即便穿着淑女风满满的连衣裙，还是装备整齐，跟在一众小朋友的身后，一个项目连着一个的玩儿了起来。
怎么滴，谁还没有一个爱冒险的灵魂了呢。
当然了，空中的拓展项目全程有工作人员在旁协助，提供专业指导，也安抚小朋友突如其来的紧张。每个项目都是连在一起，结束之后才能从空中下地。如果玩耍中想提前结束，也可以随时跟工作人员打招呼。
上完天，还能接着下水。花海旁的湖中，有着许多水上游乐项目。炎炎夏日中，穿上救生衣，好好在水中扑腾一阵儿吧。像是回到小时候，不怕把衣服弄湿弄脏，只顾玩儿的嗨不嗨爽不爽。

花开在侧，流水潺潺，衲田村里半日闲

若要我形容衲田村，想必“开在花田里的村子”太合适不过。一路走来，途径的花朵迷了眼，春日里的红绿相映，一点都不是红配绿的俗气，反而明亮又鲜活，让人觉得，春日就该是这个样子。
村里的老人说，衲田村寓意“缀衲百花色”，也是如此的生动形象了。除了这满目鲜艳的花朵，衲田村的建筑也是很招人喜爱。
村内的建筑立面选用当地石材，通过传统工艺将石头堆砌而成，古朴自然，并配以木材和简洁的立面开窗。建筑体块高低错落，既有古村落的韵味又体现着衲田拼合方式的特色。
古朴的建筑，曲折的街巷，潺潺的流水，与花儿们相得益彰。整个村子都散发着盎然的气息。

吃在宿迁，来一桌骆马湖鲜宴

毕竟三台山坐落在骆马湖畔，湖中鱼类多达56种，尤以银鱼、青虾、螃蟹、龙虾最佳。近水楼台先得月，一桌湖鲜才是最应景。作为江苏省的十大主题名宴之一，骆马湖鲜早就美名远扬。
在衲田村溜达半个上午，早就饥肠辘辘。一行人钻到村子的一家餐馆，一品湖鲜的鲜美。
骆马湖银鱼，通体洁白透明如美玉，形状如根根银条，肉质细嫩鲜美，有"水中白银"、 "水中人参"的美誉。湖鲜宴中用鱼最多，不管是生鱼刺身，还是清蒸、炖汤、鱼羹，都注重保留鱼肉的鲜美，不剥夺它本身的味道。
也有几道菜，以鱼为原料，加入了辣椒等调料，入口直呼好辣。若是细品，这辣味包裹着鲜嫩的肉质，倒也别有一番滋味。
大青虾，色青、个大、皮薄、肉饱，名闻遐迩。以此为主要原料制作的"醉虾"，别有一番风味。而小龙虾，不止是宿迁的特产，在整个江苏，小龙虾都是颇受欢迎的盘中物。
一顿酣畅淋漓的湖鲜大餐之外，当然少不了其他肉肉和菜品的作陪。索性点满了一大桌，算是在整个闲适的周末，一饱我这个吃货的口腹之欲吧。
最后的老婆饼跟肉松点心，是这餐湖鲜宴的小惊喜。没想到不仅湖鲜做得味美且别出心裁，连点心都做的创意满满，精美无比。鲜花树叶的装饰，也让人心旷神怡。
悠闲的两日弹指一挥，小城宿迁里，虽没有惊喜无限的旅游景点，也没有网红餐厅和酒店，但就凭它那原生态的花田，与这绵延2000多年的历史故事，它的魅力就足够。
寻一个周末来三台山游玩，赏山水和花海。或去项王故里，感怀霸王境遇。都是宿迁这座小城，给我的独特体验。
