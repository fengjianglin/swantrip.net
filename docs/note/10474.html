我带着老妈的亲子游

芽庄海滩
从我开始接触旅行之后，就一直跟爸爸妈妈将说将来有机会了，我带你们两个也出来玩。有机会、等下次就是一个很官方的推辞。所以为了这次旅行我做了很长时间的准备，机票大概是提前3个月就买了，因为计划了14天的行程，规划就用掉了一个月。
本次出行是穷游，我的文章没有什么豪华五星级酒店的推荐，没有动辄上千的蜜蜡沉香，没有教你如何打车（这个问题我去之前也很苦恼，一直怕被坑。但到了芽庄之后，会发现如果酒店位置订的好，其实真的不需要打车。）

初到越南你不得不知道的几件事

为什么在众多的选择中我选择了越南？很简单，因为我妈妈想去，我就是圆梦小天使。其次也想要去体验一下当土豪是个什么感觉。
说正事儿，初到越南需要注意什么呢？
一、签证
越南签证的出签率很高，可以直接通过天鹅之旅办理。最长停留时间：30天，办理周期：收齐材料后预计4个工作日（快递时间另算），受理范围：1）中国大陆因私护照2）港澳台因私护照。
需要提供的签证材料也很简单：1）护照必须在行程结束之后有半年以上的有效期；2）至少有连续两张（正反共四页）的空白签证页，空白签证页不包含备注页；3）需提供清晰版护照扫描件或者复印件。
简单的说就是将你的护照扫描件上传就可以在家等快递了。
购买链接：http://www.tuniu.com/visa/visa_210444927

二、机票
经过筛选，我选择了亚航。一个人单程的费用大概是900+ 一个人一程的费用就能少400，双人往返就少1600，算下来是笔不小的数目。廉价航空不会提供机上餐食，也没有行李托运的服务。这也就意味着你需要控制一个人的行李在7kg内，并且大小控制在56*36*23公分范围内。
西安咸阳国际机场
机场的登机口就有测量行李大小以及重量的秤，就我的经验来看，从中国出发没有人会真的让你挨个把行李称重。从胡志明回来的时候我的行李超重，但我巧妙的逃过一劫，后面会慢慢的讲。
三、货币
到越南去当土豪是当然没说的，买个东西动辄好几万、几十万越南盾。如何方便快捷的换算越南盾与人民币之间的关系呢？总不能一直那个手机算汇率吧。有个很简单的方法交个大家“去万乘三”，举个例子：东西售价10w越南盾，换算成人民币大概就是10*3=30元。即10w越南盾≈30元人民币。
越南盾可以抵达后在机场直接进行兑换，机场的汇率大概是1:3200，很多攻略都有建议说在市区换汇率会高一些，我的经验是建议在机场先换1000元人民币，市区很多地方都可以换钱，但很不正规，比如卖乳胶枕的地方、餐厅、酒店等。有的地方汇率还不如机场高，我住的第一家酒店汇率就是1:3150，第二家酒店是1:3300，碰到汇率高的地方建议一次性换差不多的钱，因为真的是很碰运气。
是的，土豪就是随手拿着几百万。当时我在机场换了500元人民币。
四、过关小费
也是来之前看各种攻略，说越南机场 过关是会被索要10元小费作为打点费。好巧不巧在来的途中我才发现自己身上并没有10元的零钱，给50吧又觉得舍不得，还好我机智。亚航的飞机上不提供飞机餐，所以就买了一袋零食，12马币。在亚航的飞机上，你可以使用各种流通货币，但找给你的钱都是以马币为结算单位的。所以我用50人民币买了零食，找给我16马币。
虽然觉得很贵，但是想想给了小费还落个吃的总比白白给出去50要好，忍了。
过关时特别紧张，结果人家小哥什么都没说，确认了签证就盖章放行了。真的就是凭运气，但还是建议大家带一些零钱，出门在外一分钱难倒英雄汉啊。
五、酒店
刚说到如果酒店位置定的好，基本省去了打车的费用，这话一点也不夸张。我这次在芽庄共住4个晚上，中间换过一次酒店，两个酒店距离不到一公里。毫不夸张的说这一趟如果出门都打车，差不多要花掉500块人民币。
给大家推荐一家：Galliot hotel nha trang
并不靠近海边，但房间的卫生和整个酒店的服务都做的很舒服，刚提到汇率比较高的地方就是这里了，如果不住在这里也推荐到这家的前台去换钱。房间卫生打扫的很仔细，而且第二天还给我们铺了花瓣在床上，第一天我大概早上九点多就到了，前台小姐姐还是很热心的帮我办理入住。
顺便再提一嘴关于房间应该放多少小费，这个完全凭个人的心意，我并没有给吗每个入住的酒店都放小费，而是我比较满意的才会放。每次放2w盾，合6元人民币。

芽庄哪些景点？

不得不说芽庄的景点门票都很便宜，大概是2w-4w盾，也就是6-12块人民币，跟价格成正比的就是景点的规模，一个地方差不多都是30-60分钟就可以游览完毕的。

芽庄国家海洋博物馆
千万不要被这个响亮的名头吓到，逛完反正个博物馆大概只需要半个小时。芽庄国家海洋博物馆成立于1923年，是越南最早设立的海洋研究中心，这里保存了6万多种海洋生物标本。不仅展示着有关海洋学的各种知识和主题邮票，各种大体型海洋生物的标本也可以看到，地下室还有一个水族馆。

龙山寺
建于19世纪后期，现在有很多僧侣在这里修行。寺内的汉字门联、龙纹雕柱等元素都透露着强烈的中国文化气息。在芽庄市区任何地方都能看见的24米白色大佛，就在这里，爬上后山的150级台阶，就可以看到庐山真面目。大佛安坐于莲花之上，俯视众生的欢乐与烦扰。如果选择从侧边的小路下山，还可以看见另一尊白色的卧佛。运气好还可以赶上参拜仪式，当地信众席地而坐虔诚礼佛，很是震撼。

I-resort 泥浆浴
离城市中心约4公里的I–resort所具有的安静气氛让游客们来到这里享受泥浴、矿泉浴服务时，获得身心的尽情放松。 由雄伟山脉和温柔的芽庄丐河包围、结合使用当地材料，如木材、石头、水椰叶的独特架构，I-resort度假村像一个绿洲一样，让游客们入门的第一眼就被打动。作为一种自然疗法，淳朴美丽的空间将有助于净化灵魂、给你痛快淋漓的轻松感觉，其矿泥浴服务将带给您全新的体验。
预订了从酒店出发，含接送的泥浆浴服务，某宝、天鹅之旅上都可以直接预定，跟客服沟通好接送的时间就可以了 。泥浆浴的选择有很多，国人最喜欢的一个是I-resort另一个就是一百蛋，我最后选择的是I-resort，很是不错，早上去的话人很少而且每个池子的泥浆都是现放的，感觉也很干净卫生。
至于可以在里面玩多久，这个完全看你的时间安排。你可以带上吃喝在里面带整整一天，如果赶时间的话单纯体验泥浆浴+换衣服洗澡的时间至少也需要一个多小时。

惊心动魄飞车党

在去吃瓦片烧烤的路上，我遇到了越南的飞车党，抢手机的那种。因为越南本就是小街道，瓦片烧烤的地点有有些偏僻。我拿着手机一路导航一路走，突然手背被种种的拍了一下砸在我的胸口。
脑子是蒙的，不知道发生了什么？回头看去，两个男人乘一辆摩托车飞驰而去的身影，突然反应过来，抢手机的！好在我的手机套在指头上，才免于一劫。这里也是要用亲身经历告诫大家一些预防飞车党的注意事项。
1、不要一边走路一边看手机，尤其是在背巷里。
2、如果一定要看手机，那就停在路上。有同伴的话同伴一定要帮忙看着来往车辆，千万注意前后都是男人并且四处张望的人。
3、不要把手机挂在脖子上，因为这样一点也不安全。
4、背包也尽力用手有意无意的保护着，毕竟包里肯定都是护照、签证等极其重要的物品。
最后希望大家在外旅游都能顺利平安，不要发生这样的事件。
