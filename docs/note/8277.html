游记 小众 情侣游 摄影 美食 自驾游 2019/07/31

我和她的最佳沟通方式——自驾游

每对小夫妻因为志同道合而走在一起，可能都会有属于自己的沟通方式。我和我家领导是因为旅行而相识，自然彼此最大的爱好就是牵手去旅行。我们喜欢在旅途中与彼此沟通，大到轻松愉悦的环境可以暂且放下生活中的琐事烦恼，小到顺便找借口吃顿好的改善下生活……
但是结了婚和谈恋爱的时候还是有所区别的，毕竟早上一睁眼就是茶米油盐酱醋茶，并没有太长的时间和太多的金钱到处游山玩水。因此周边的短途自驾游是我们维持感情的最佳沟通方式。当我们坐在一辆车里，抛开手机的羁绊，打开喜欢的音乐，敞开心扉的聊一聊，就是我们感情最好的黏合剂。
这次自驾游的地方有些特别，因为最近因为江浙沪持续高温，单纯的逛景点真的很晒很受罪。所以我规划了一条 常州 到 南京 的小众路线，绕了很多弯弯，但是把路上的风景发出来，很多当地的小伙伴都不相信还有这样的 江苏 ，所以有必要把这篇游记分享出来，让更多的小伙伴看见。自驾的时间不长，往返刚好十二时辰！

自我介绍

爱生活，爱旅行的80后，大学毕业后在家乡 淮南 煤矿工作了六年，看透-800m矿井下的阴暗和冰冷。特别是刚毕业的时候，可能是我整个职业生涯最灰暗的时刻，危险的工作环境甚至可能威胁到有生命安全。当然也要感恩这段工作经历，让我对生活充满了更多的热情和探索的欲望，现在辞职到了 常州 定居，追逐千里之外的诗和远方。希望有机会能够和各种有趣的人一起探讨旅行，结伴旅行。加好友或咨询线路问题欢迎私信。

注意事项

1、自驾出门，除了常规的身份证以外，驾驶证也一定不要忘了，自己的车还是租车行驶证也要带着。开车要穿舒适的鞋子，不要穿拖鞋或者女生穿高跟鞋。带适量现金即可，江苏的高速收费站以及各大服务区都支持电子支付，可以少备一些零钱，在一些小摊上买东西更方便一些；
2、夏季出行天气炎热，建议早晚凉爽的时候出门。这个时候拍摄照片也是最佳时机，中午天气炎热的时候可以找地方休息，露营或者找酒店钟点房都可以。切勿车停下开空调在车里睡觉，长时间这样容易一氧化碳中毒；
3、自驾去路况不熟的地方，建议及时更好导航，租车一般不提供行车记录仪，或者记录仪里没有内存卡，可以自备一张内存卡使用。导航支架一般租车也不会提供，可以带一个会有更好的视觉体验；
4、如果你和我一样喜爱摄影，一定要带好相机和充电器，还要有充足的内存卡哦。特别要注意的是，如果耗电快建议自己带车载充电器。

网红服务区

江苏 的路况真的超级好，这一点完全可以秒杀掉我的老家 安徽 全国的绝大多数省份。有朋友这样跟我说过：你的大家伙哈弗H9应该多开去越野，找烂路开！“对不起， 常州 乃至我生活的 江苏 周边真的没有什么烂路可以开。”但是我还是喜欢H9的稳重，媳妇喜欢它的视野，好路我们一样开。
我们早上出发， 第一站就是沪武高速上的滆湖服务区，它也是我们 常州 最早的“网红”服务区。最近很多网友都在社交平台上热议 苏州 阳澄湖服务区是“最美服务区”，有 澳门 威尼斯 人般的蓝天白云天幕。但我想说，在你们成为网红之前我们滆湖就已经是网红了，只是没有赶上全民分享小视频的好时代，只有小圈子里的人知道。暂且叫我们“过气”网红服务区吧！
滆湖的网红是因为它的外形酷似魔法城堡，高大的欧式建筑让司机还行驶在滆湖特大桥上就能远远的看到它，让人忍不住想要停进来。另外服务区里也有一个高速出口，驶出后就可以直接到 常州 的花博会，揽月湾等西 太湖 周边的旅游景点。对，没错！滆湖在 常州 一般都被叫做“西 太湖 ”。
服务区里面的环境已经再熟悉不过了，基础设施还是非常完善的，这里就不过多赘述了。这次路过主要就是想航拍一下它的大气。也意外的发现滆湖特大桥也是相当的霸气。欢迎大家有机会来 常州 西 太湖玩，这里完全免费。

大溪水库

从 常州 到 南京 ，其实沿着那条沪武高速直接开过去就到了，不过半个时辰的时间。不过我们是自驾游玩的，所以直接先岔到了 溧阳 。 溧阳 是 常州 代管的县级市，境内最出名的景点是天目湖和南山竹海，不过这些常规景点都不在我们考虑范围，我们导航的目的地是 大溪 水库。
大溪 水库的水质看起来也十分优质，蓝蓝的就算不像海，也很有几分夏日小清新的感觉。其实这里和天目湖很像，如果你不是一定要打卡景点，这里也会是一处不错的选择。
来到这里的一大原因是因为 溧阳 1号公路，这个也是火爆 常州 朋友圈的网红公路，因为它道路中间的实现是红黄蓝三种颜色组成，被广大网友称之为“彩虹公路”，这个配色也十分符合夏日清新的感觉。
溧阳 1号公路有很长，而且道路中央全是这样配色的实线。整个 大溪 水库的环湖路都是 溧阳 1号公路的内环线。来这里环境好，路过的社会车辆也不多，会有一种包场的感觉，夏日露营再适合不过。
而且在去往水库的路上还遇见了很多不错的景致， 比如 下面这面墙的壁画，特别喜欢上面的那条鱼，忍不住停下来拍照留念。而且路边也有很多野花开的正好，随手采了几朵送给媳妇，她也特别喜欢。
我们到 大溪 水库的时候是午饭后，正是一天最热的时候。于是我们找到了一片废弃的楼房前把车停下，在树荫下小憩了一会。温馨提示下，自驾游千万别在道路上随意停车以免影响交通，自己产生的垃圾也记得带走。
因为天热没有急着赶路，就在这片树荫下拍起了照片，自娱自乐一番。来看看我家的女司机，霸气十足！
我特别喜欢下面后视镜里抓拍的这张，表现力极好，彷佛整条gai上最靓的妞。因为H9的后视镜特别大，她才同意我发出来，说是镜子大显脸小哇！难道不是视野好更安全吗？哈哈。

高淳国际慢城

从 溧阳 大溪 水库到 高淳 国际慢城大约50km，路况很好。开到那里，也就已经从 常州 到达 南京 了，高淳 是 南京 的一个行政区。
之前也有听说过 高淳 国际慢城，但这次也是第一次来。可能也有很多小伙伴和我一样对“慢城”这个概念感觉很模糊，我的理解可能就是生态好，环境好，生活节奏很慢很舒适的一个生活圈。感觉很厉害的是早在2010年， 高淳 国际慢城被国际慢城联盟组织授予“国际慢城”称号，成为 中国 首个国际慢城，并将国际慢城联盟 中国 总部设置在此。
把车开到停车场停好，去售票处问了才知道这里只营业到下午16点。里面非常大，需要乘坐电瓶观光车，而且很多游玩项目也已经不营业了。所以也并没有赶着进去打卡拍照，那不符合我们轻松愉悦的旅行态度，也不符合“慢城”的态度啊！就在停车场飞了一把无人机。
我飞无人机的时候媳妇可没闲着，把我买的零食都吃光了！临走时还命令我给她拍几张照发朋友圈，得令！拍完赶紧去往下一站。

石臼湖

像预期一样，赶在日落时分从 高淳 赶到了 溧水 区。为的就是一睹石臼湖特大桥的绝美日落，之前在网上看到过这里的照片，实景还是更加震撼。
石臼湖特大桥足足有12km长，桥上是不许停车的。所以不管我们开什么车，遵守交通规则那是前提，于是把车停在了岔路口的空地上，和媳妇一起静静欣赏眼前的美景。
湖水很蓝，被厚厚云层挡住的夕阳努力挤出一些余晖晒在湖面上。看着眼前这开阔的景色特别治愈，哪怕我们现在离家很远，但我们在彼此身边，坐在车里也一样很有家的安稳。
到了这个时辰，把无人机仅存的一些电给耗尽，留下了石臼湖特大桥的美景，大家来感受下12km到底有多长吧！
开到桥的另外一端离 南京 城就更近啦！这个时候仿佛有种跑步胜利的感觉，媳妇说必须拍张照纪念一下。光线已经不好了，亏她想到坐到后备箱里用车灯补光！还特意换了套更合适的衣服。蚊子太多，赶紧按下快门，把车开走去 南京 寻找美食喽！

南京宁府口味龙虾

南京 美食吃什么，鸭血粉丝和咸水鸭？开什么玩笑，这么热的夏天当然是吃小龙虾的季节。如今华夏各地吃龙虾的热潮都在高涨，而且呈上升趋势。 江苏 最有名的龙虾基地是 淮安 的 盱眙 县。据自己网上视频里看到说这里从事龙虾行业的人占比极高，但吃龙虾的大户却是 南京 。因为 南京 距离 盱眙 很近，原材料供给方便，且这里人多呀！
朋友给我们推荐的是位于 南京 秦淮区的明瓦廊宁府口味龙虾。顾名思义，明代的时候这里是一座经营明瓦的场所，现在则是一条不折不扣的美食街。而宁府就在路口，很好找。明瓦廊的道路不是很宽，这里的门面看起来都不是很大，但店里还是很雅致的感觉。
店家打出的口号是“传承三代人，坚守真美味！甭管这么多，我们口味说话。”这里的十三香龙虾特别入味，吃完还得再往汤汁里下碗面才好。还有迷迭香红烧龙虾、红艳伴汤龙虾这些创意口味，第一次吃到！真的是360度满足味蕾。
两个人吃了五斤！是真的饿了。在小龙虾面前其它的菜也都成了浮云，味道好不说，你戴着手套剥龙虾的时候也没有办法拿筷子夹菜了，哈哈。其实南京的美食还有很多，包容性也特别强，所以各种口味的小伙伴都能找到自己爱吃的东西，只是我们刚巧来到了这里。所以大家按照自己喜欢的店去找就可以，另外南京停车真的不太好停，吃饭找一些大型的商圈，有自己的地下停车库的那种会比较容易停车。
吃完龙虾就选择住在了新街口附近，其实真的很怕在 南京 城区开车，道路比较复杂，且停车位十分难找。不过这一次的自驾游还是很美好的，毕竟路上的点滴都是我们人生中最美好的回忆。
