跳岛行，吉他、歌声一样不能少

逃离北京的雾霾、逃离高节奏的生活，逃离超负荷的工作……我只想不带脑子地旅行，给自己彻彻底底放个假，什么都不想，只是看着碧海蓝天静静发呆，享受阳光和沙滩。于是我选择了仙本那，一个临近赤道的小岛，单纯得只想看看这个世界究竟能有多美，是的，这是一篇仅关于【美】的游记，美到用任何的修辞来赞美都显得苍白，我不想努力晃荡肚子里那点少得可怜的墨汁来笔底生花，只想将仙本那的每一天，都过成诗。
清早醒来，躺在床上就能看到大海，慵懒地翻了个身，想着再睡一会还是去楼下寻觅早餐？眯着眼，看了看窗外蓝天白云，阳光正好，还是安奈不住想出去玩的心，果断起床，心说，幸福的假期这就开始了！
仙本那，仙境一般的小岛，在这里每天的生活都是从去码头坐船出海开始的。
早上7点多“海人”的司机就来酒店接我，同车的还有一对上海来的老夫妻，带着小孙女。我的酒店到码头其实只有5分钟的路，就在这短短几分钟里，大家就混熟了，他们比我早一天来，已经出海过一次，听说我和老妈是刚到，老爷爷就热情地介绍着我们今天的行程，酒店周围有什么好吃的，巴拉巴拉聊个不停。很快到了码头。老爷爷像是领队一样，熟练地带我们上了二楼，找到服务台，工作人员是位很漂亮的年轻姑娘，齐刘海、戴眼镜，娃娃脸，看起来像是台湾青春偶像剧里的大学生，中文很熟练，亲切的样子好像我们是来她家里串门的亲戚，阳光洒在她的笑容上，一股暖意让初来乍到的我瞬间觉得放松下来。
简单填写了表格核对信息后，就是等待8点多的船出发。一看表，时间还有1个小时，百无聊赖，没啥事可做，心想本来也是来看海发呆的，在哪发呆不是发啊。找了个能晒到太阳的长凳坐下，眼睛看着码头上来来去去的人，有打扮漂亮的游客；有晒得黝黑各种纹身的船员，各自忙着各自的事，脸上却都是开心欢快的神情，和游客之间打着招呼，几个皮肤嘿嘿一头黄毛的年轻人弹着吉他唱起歌曲串烧，从《流星花园》唱到《葫芦娃》，各种混搭，偶尔跑调，周围人也被这气氛感染聚拢起来，这唱的不是歌，是开心！
第一天来，我还在想，这码头怎么这么欢快？后来天天出海才知道，拿个吉他，一言不合就开唱是每时每刻都会上演的曲目，不管是在岸上还是船上，开心就唱，想唱就唱！歌声时刻不能少就对了。
自由、放松、欢快，嗯！是的！这就是我向往的日子，梦境般的生活，从歌声和微笑开始。
坐快艇出海，第一次感受到乘风破浪的感觉，像是在海上飞一样！海风吹在脸上混杂着浪花令我睁不开眼，小船被海浪推得老高又重重拍下，一船人不禁随着一浪又一浪飞跃尖叫出声，没多久衣服就全被海水打湿了，相当刺激，大呼过瘾。就这样“飞”了一个小时，终于跳到了我此行的第一个岛——马布岛Mabul Island，只一眼，就被她不可思议的美深深迷住了，这就是天堂的样子吧……
绿宝石般透明的海水在纯白色沙滩上缓缓拨动，身上晒得暖洋洋的，心理却生出一种美妙的宁静。
绿宝石般透明的海水在纯白色沙滩上缓缓拨动，身上晒得暖洋洋的，心理却生出一种美妙的宁静。
跳岛行，顾名思义就是一个岛接着一个岛地玩。在仙本那的前三天，我整天都在出海、跳岛、浮潜，看似都是岛，每个岛的美还都各不相同。接下来这个是马达京岛Mataking Island。
一路上有海人团队的导游和小黑哥们热情洋溢地陪伴，相处得像是从小一起长大的自家兄弟姐妹，各种疯闹，玩得相当欢乐。
才刚上岛没多久，我就在JIABI鼓励下，拉紧我的手，从3米高的月台上跳下海,纵身一跃，仿佛时间停止了，感觉过了好久才普通一声跳入宝石般透明的海水里，接受大海拥抱时只感觉好咸，海水近眼睛原来会杀得生疼，还好JIABI在身后迅速把我捞出水面，我只觉得天旋地转睁不开眼。真羡慕当地海边长大的孩子们，能在空中摆各种姿势飞身如海，潇洒自如，相比之下我就只有怂而已，冻得瑟瑟发抖爬上岸，我的导游小戴很贴心地准备好热茶递给我，招呼我去晒太阳取暖。
此行最美的沙滩要数军舰岛，纯净细软的白沙，光脚踩在上面软软的，偶尔还能见到几只色彩各异的海星趴着，那阳光将周围的一切照的发白，仿佛梦幻中的天堂。
我一会踩踩沙子，一会趟趟水，遇见海星就过去戳两下，想着反正一会还要穿着这身潜水，也不担心弄湿衣服，肆无忌惮地欢蹦乱跳，高兴了还躺在沙滩上迎着海浪打个滚儿，身上沾满细砂粒，就在海里扑腾一阵，这种什么都不用顾忌，没有约束，自由放纵的感觉真好。

坠入不愿醒来的海洋梦

来仙本那最好玩的也是必须必要玩的就是潜水！潜水！潜水！一想起来我就好兴奋！本人这辈子也就潜过两次，第一次是前几年在埃及的红海，当时只是很简单的体验了一下，战战兢兢下了海，潜水镜里不知怎的总是往里灌水，几乎不敢把头埋到水里，虽然没玩多久，但也被海里面的世界迷住了。
潜水会上瘾，来仙本那几乎天天潜，每天潜个2、3次，每次至少45分钟，还觉得不过瘾。想着以后考个潜水证，再来的时候一定要去传说中的诗巴丹深浅一次，感受鱼群在周围“飞”的壮观场面。
想起大海，表面上的碧海蓝天就已经很美了，随着咕咚一声，整个人没入海里，周围的一切一下子安静下来，只能听到自己缓慢规律且故作镇定的呼吸声，我感到自己进入了另一个奇幻世界，强压着狂跳的小心脏，完全被海里的美景震惊，本以为在海里看到的一切都会是蓝的，万万没想到竟是五颜六色。
海水荡漾折射着阳光，显得神秘又梦幻，色彩斑斓的珊瑚和各种鱼类近在咫尺，鲜艳得不可思议。
远远看到一群黄色的小鱼游过来，水流一动，像是惊到了它们，猛然一起转身，一大片鱼竟同时闪着金色的光，耀眼得像是水中放的礼花；
一只花里胡哨的鹦鹉鱼不紧不慢地游在我身下，一对紫红色的鱼鳍像飞鸟的翅膀一张一合，配上它翠绿色的身体更显好看，令我想起阿凡达森林里的神奇生物；
竟然海里的鱼也会卖萌，有种咖啡色的小鱼很愿意围着人，可能是觉得挨着大家伙有安全感，一会游到我腿周围一会又游过来紧贴着我肚子，伸手一捞就能摸到，小鱼躲一下，不一会就又凑上来，心说它在进化个几亿年能取代狗；
最激动是在汀巴汀巴看到海龟了，而且大大小小好多只，无法形容和海龟一起游泳有多开心，也不怕人，很好奇的样子，就在我周围游来游去。
我的潜水教练家诚说，如果有往北的洋流，还能看到海豚，不过这样的机会少之又少，他天天出海也就只见过一次。我说，我好羡慕你每天能在仙境里上班。

海上吉普赛---巴瑶族

巴瑶族（Bajau）是个海上的游牧民族，生活在菲律宾、马来西亚和印度尼西亚之间海域，“巴瑶”在印尼语里是“海上之民”的意思，常被比喻为“海上的吉普赛人”，他们没有国籍，也不让上岸，就住在海上的木屋里，靠潜海捕鱼为生。这样简陋的家四面透风，好多孩子没有衣服穿，看着怪可怜的，他们每天都能享受着天堂般的美景，却过着如此贫寒的生活。

红树林+长鼻猴+萤火虫

下午四点多，我们前往红树林，红树并不是红色叶子的树，听说它的树干砍断后，会流出血红色的液体，故名红树，是陆地向海洋过度的特殊生态系，突出特征是根系发达、能在海水中生长。
这条河一半是海水一半是淡水，深深扎在泥里的树根可以起到防风消浪、促淤保滩、固岸护堤、净化海水和空气的功能。所以当地人很笃定地说，有红树林保护这里不会海啸。
坐上小船，一船人的注意力全集中在茂密树林里，听说这里有3种猴子，长尾猕猴、短尾猕猴，还有一种是最稀少也最难看到的长鼻猴Nasalis larvatus。
现在正是涨潮季，能看到猴子的几率很小。我心里暗自祈祷，好期待看到长鼻猴啊，以前就算是去动物园也没见过，不吃香蕉，能反刍的猴子，公猴鼻子能张到7~8厘米长。好期待！好期待！好想看！越是这样想着，心情也紧张起来，眼睛紧盯着树林看，生怕错过。
船开得很慢，没一会就见树丛里动了一下，大家惊呼“看那！看那！”又不敢出声音太大，船立刻停了下来，远远看到一只小猴在树枝上窜，是只小猕猴，虽然不是长鼻猴，总算是看到一只，还是很开心的，这里的猴子纯野生，禁止投食，所以猴子还是很远离人的。
天色渐暗，一群群白鹭贴着水面飞远，像河面上白色的精灵，无声无息，不知要去何处，我们也回到岸边的小木屋享用晚餐，静静等待天全黑下来，再乘小船去看萤火虫。
萤火虫我还是在很小的时候见过一两只，从没见过成群的，不多时天就黑透了，周围没有灯光，只有很远处的公路上，偶尔有车驶过的光。大家又默默回到船上，之间此时的树林黑成一团，幽静神秘，时有树叶沙沙声，我猜是白天看到的猴子发出的动静。
正想着萤火虫在哪？只见我船上的小哥拿出个手电，对着远处的树丛一晃，手电光扫过的地方瞬间闪起了一片黄色的光点，那光微弱、缓慢，像极了夜空中的星光，几秒后又暗了下去，手电又朝另一处一晃，紧跟着又一片“星光”闪烁。
这时手电被绿色的布罩住，光线更幽暗了，小哥用手捂住，活动手指，幽绿的光从他指缝中流出，没想到这样一来，树丛里的萤火虫竟然闪着光，慢悠悠地向我们飞过来，那感觉像是天上的星星朝我飞过来了，好激动！有的落在船上，有的落在我身上，很快我就被萤火虫围住了，伸手一捞，很容易就抓到一只，好小，小到它在手心里爬完全没有感觉，屁股上的小灯忽明忽暗。灯光一晃，又飞来一群，超浪漫！
听说，手电模仿的光像是母虫，所以这些飞过来的都是公虫子，都是打算过来谈恋爱的。
我正托着手心里的萤火虫玩得高兴，突然天上一道闪电划破寂静的夜空，瞬间大雨倾盆！我记得电影阿甘正传里描写在越南雨淋里打仗，那里好好的晴天说下雨就下雨，说停就停，就好像拧自来水龙头，让人猝不及防。眼下的雨就是这样，我赶快让我妈躲进船蓬里，自己也拢起帽兜蜷缩在角落，刚才还在船上闪烁的萤火虫去哪儿了完全顾不上，只感觉暴风骤雨打在我后背上，拼命用手抓住帽子，雨水打得都睁不开眼睛，身上一下子全湿透了！
还好小船飞快开回了岸边，躲进木屋后大家早都淋成了落汤鸡，反而都挺开心，笑着说好不畅快！我心说来仙本那这几天，没有一天不是打扮得美美的出门，全身湿透回家的，好容易今天不

仙本那的一天

早上7点多，天已经大亮了，可街上的小店几乎都没开张，阳光撒向还未苏醒的小镇，明媚的清晨一片恬静。
我透过窗，看向马路对面的一排小楼，那正是海人工作人员的宿舍，心想着，这时候勤劳的小黑哥们应该已经在码头忙碌着准备船和潜水工具了，一会又要弹起吉他唱起歌了吧。连着三天在一起玩耍，一想起今天是在仙本那的最后一天，我又不出海，突然恋恋不舍，很想再去码头见到他们。
向右看，不远处就是海，有个长长的木桥通往海上人家，零星有几个人在桥上走着，看不清手里提着什么，那边我没去过，很好奇，好想也顺着木桥走到住家聚集的地方瞧瞧。
就这么躺在床上，看着窗外的海发呆，磨蹭磨蹭到了10点多，看街上的人也多起来了，决定出去逛逛。
仙本那不大，据说最繁华的区域有个地标建筑，叫屈臣氏，离我的酒店步行也就十来分钟的路。
昨天和我的潜水教练家诚一起共进晚餐时，他问我明天想去哪里逛逛？我说不知道啊，你是地主，这边有神马好玩的？他毫不犹豫，两眼放光！说我们这里最大的超市你去过没？去那玩吧！ORZ，我扶额，好吧，明天我去逛吧(捂脸小蜂)之前坐车去码头能路过，大概知道和屈臣氏都在同一个方向，听驴友介绍那边还有水果摊，于是一拍大腿，决定就去那了！
街上真是太晒的，没走多久已经晒出一身汗了，可能是天太热给热傻了，我和我妈误以为能抄个近路过去，却意外绕到了另一个市集，其实更像个农贸市场，主要都是摆摊卖菜、熟食和各种调料的。能更贴近地感受下当地人的生活也挺有意思。
中午时分，终于走回既定路线上，屈臣氏就是屈臣氏应该有的样子，最!大!的!超市我也逛了，没几分钟就出来了，你懂的。还是大海最美，一看表也该吃午饭了，我现在的位置正式网红肥妈馆附近，但我还是想选个离海更近的，风景更美的位置坐，于是选了肥妈隔壁的天下客午餐。
一边看风景一边啃着大螃蟹，再喝上一口榴莲冰沙，啊～～～心情真是格外的爽！
是啊，旅行终要结束了，我吸了一口冰沙，清凉瞬间传遍全身，闭上双眼，感受海风轻轻拂过时间，贪婪地享受着此刻的悠闲惬意。
看着天空中缓缓飘动的白云，海里飞驰而过的快艇，不由得回想起在仙本那的每一个瞬间，都美好得像是梦，一场蓝色的梦，那歌声，那笑容，是多么真挚和友善，遇到阳光，心里便会开出美丽花。
如果可以，我想一直沉浸在这蓝色大海的怀抱中，久久不愿醒来。

【DAY2 2019.1.22.】
仙本那码头---马布岛----卡帕莱岛浮潜
本次选的都是“海人”出海旅行团，一早就有司机来我酒店接，约好时间在前台等，司机会拿客人的名单对照，不会出错，开车大概5分钟就到码头了。码头很多家出海旅行社，一上二楼，第一家就是海人，他家品牌意识不错，标识很清晰，工作人员都穿同样的制服。
先来和工作人员核对姓名和当天的出行线路，注意要报自己的护照号，然后工作人员会在你手上贴个彩色的标签，不同颜色就代表不同的线路，一会会按照颜色召集人一起出发，大概每天都说是8：45上船，每条船上十来个游客，一个船长，中文导游1-2人，小黑（不会中文的工作人员）3-4人
潜水不用担心拍照问题，船上的小黑会用水下相机帮你拍好，回码头可以微信发给你。不用额外给小费，当然你愿意给自然是极好的。
不会游泳可以浮潜吗？答案是可以的，当然你心脏得够强壮，浮潜是穿救生衣下水的，而且还有救生圈，你只要把头埋在水里就行，手扶着救生圈，有小黑带着你游，比较安全。海上浪大，被海浪来回拍打不要惊慌，海水很咸，抢到水也不要惊慌。
下午一般4点左右回到码头，传照片，还是和码头的工作人员联系，司机会送你回酒店。海人的工作人员中文沟通没有问题，而且亲切热情，去了你就会被他们的热烈感染，非常难忘。

敦沙卡兰海洋公园     巴瑶族----爬山----军舰岛---浮潜----和潜水教练去渔人码头晚餐，感受当地人喜爱的菜肴
看见了吗？爬山！这是个很鸡肋的项目，600米的山，基本没路，下雨地滑，相当难爬，一上一下大概2小时，真心不想爬，但这条线路的军舰岛很美，纯净的白沙滩，清澈得透明的海，还有海上巴瑶族村庄。
