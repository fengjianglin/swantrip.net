大熊猫&小火车&“竹”的博物馆

庆幸这一次天气稍好
无人机的视角下，我们看到了竹林深处仿佛世外桃源一般“藏在深山人未识”的鸡鸣村。
由于它所处的位置位于苏、浙、皖三省交界处，自古就是官民交通要道上休憩的必经之处，据说，此村的大公鸡只要拂晓时分“喔喔喔”一叫，周围三省的老百姓就全都醒了，故得名鸡鸣村。

Some Tips

推荐住宿：
【南山竹海客栈】推荐理由：幽静的环境，庭院、天井，让你不虚此行
【天目湖御水温泉酒店】推荐理由：来天目湖就是要来这里泡温泉的呀~
关于门票：
【南山竹海景区】成人票90元
【天目湖山水园】成人票120元
关于交通：
江浙沪一带建议自驾，另乘坐高铁到溧阳站也是高效的方式，从溧阳到天目湖景区只需要1小时以内的车程，乘坐直达大巴费用10元，打车大概80~
