离开成都的每一天，想它。

游记 人文游 动物 摄影 美食 购物 2018/10/11

🍂 前话 🍂

我们似乎对天府之国成都都不陌生，不论是遍布大江南北的川味锅还是大熊猫都成就了成都的独立体格。成都作为很多旅行线路的落脚处我也几度匆匆一瞥，这行作为自驾川藏线的起点，终于实现了成都悠闲逛吃行。
在成都的街头走一走，感受温柔的包容和传统的延伸，成都有着时髦的精致和市井的地道。地震多发带却高楼密集，不管多偏僻商业都很完备，总不缺人烟，房价喜人，藏富于民，生活悠闲自在，除去湿度太大非常宜居了。西南的天气预报总是日不懂，每天阵雨却也日晒永远让我措手不及，在钻人鼻息的湿漉漉的桂子甜馨里铺开我的一行。

介于落地成都自驾进藏suv的租车费在2万+，包括拉萨空返的费用，而公路托运私家车成本不过三分之一。于是一早就奔赴龙泉驿区取车，车城规模叹为观止，果不其然成都作为西南、川藏各条经典自驾路线的衔接地也催生了租车物流服务的发达，坐上提前装满行李和零食可以换拖鞋的苏A，我们舒坦到想躺着开。于是也真切感受到三环的堵，和北京不同，成都的三环都是高架，友情提示每周1-5分别尾号16.27.38.49.50二环内限行。



青城山、都江堰都远离市区，儿时去过，时间宽裕的值得一游。顺路要去卧龙大熊猫自然保护区所以没去大熊猫繁育研究基地，后者在三环内。

名胜古迹诸如武侯祠、杜甫草堂自行了解一下。以下是常规市区游（无门票是重要原则哈）加小众景点。

醉西昌

醉美不过烤肉香！以大凉山彝族菜为标志，特色都雨露均沾了。「大凉山跑山生态小香猪」大块厚烤，猪肉自有的味道很突出，切开瘦肉部分意外的柔软，皮和油脂部分不会呈焦脆的五花肉口感。

「西昌邛海醉虾」用醪糟、醋、辣椒生腌，说是5分钟开盖，实则得等虾不弹动再吃不会四溅，虾体小而剔透，入味非常。

「包浆豆腐」是在云南常吃到的，以建水县的闻名，和建水铁网现烤豆腐不同，这里端上来的成品却是最酥脆，赏味期不长得趁热。

「生菜包牛肉」用蒜蓉调味，油润柔和。

「法式鹅肝配鸭肠」用锡纸盒烤，非常柔软丰腴，把菌类也放在鹅肝里烤蹭个出油。

大凉山酸奶倒是在泸沽湖喝到过，也算云川交界。

如在蜀老火锅

且不表火锅江湖的川渝之争，近年来，火锅串串在南京遍地开花，以为对这一口牛油锅审美疲劳，只安排了这一顿，也是别无分号不会有连锁的对比伤害。吃时方感决策英勇，这菜单喜人的价格和本地人热火朝天涮锅的氛围还是身临其境的妙。

「奔驰锅」39，「铺盖毛肚」39，「腰片」19，「鱼子虾滑」19，感人肺腑。

牛油块洗个澡，醇香柔和，辣度可以加，基础版吃起来口舌肠道都无负担，刚好可以放肆，辣度随涮食的增量一点点提升上来充盈整个胃袋，「毛肚」毛尖立体弹脆，腰片毫无腥臊，品质卓越，在香油里滚一圈保护肠胃也调和口味。

「鱼子虾滑」很紧实，一口下去有整颗虾仁。冒菜是这家特色，蜀小碗脑花很地道，土豆是顺滑的酥烂感。

「焦糖豆花」是甜点，清爽解辣，果冻般柔嫩剔透。

「手打豆花」以乐山闻名，水质不同所以风味有异，这款越煮越嫩，加辣椒圈调料吃起来别有风味。

番茄锅不像我们常吃的酱包调得浓郁，是清爽稀薄的酸度，加葱花喝起来有种妈妈的番茄蛋汤感。菌汤底也是很有诚意的高汤底，呈褐色菌味浓郁。

四川博物馆

馆不大，门牌是自封的博物院，然只有故宫博物院、台北故宫博物院、南京博物院三大博物院。语音讲解让逛博物馆成为有意思的事情，可以打发一上午。

成都吃客

成都吃客

成都吃客用地标性的路区分格局，也是中午时段就座无虚席，是一家比较新派的川味。

「纸上焗鲜鲍」用鲜甜的纯蟹肉蟹柳和小鲍鱼加方便面丝瓜与青花椒配搭。

「裸奔坨坨虾」去壳的串虾个大肉满，热油淋过表面酥香。

「碳烤五花肉」焦脆肥瘦有致，一把10串不过9元。

「特色烤鱼片」竹签黔鱼，属于鲶鱼品种，肉厚只有一根主刺，配菜有芹菜碎和菠萝丁，沾满辣油的口感很别致。

「豆捞烂稀饭」豆汤饭是用去皮的绿豆沙与小白菜丝熬成粥，非常细腻的沙沙口感有种豆子的清香，砂锅持续给温，表面粘稠的一层。

「相思红颜」桂花酱秘制小番茄。

泡桐树街

与人声鼎沸的宽窄巷子一巷之隔的是平行的泡桐树街，有一面涂鸦墙，尽显成都城市地标美食文化特色。都是些下午茶小酒馆，清幽闲适，更能感受到成都的慢生活，连这里的猫生都惬意许多。

贺记蛋烘糕总店

贺记蛋烘糕

贺记蛋烘糕前些天谢霆锋来过，更加推了一把。现烤的蛋糕体柔软边缘有微弱的松脆感，加料有甜咸口，本以为芝麻糖奶油会更适宜，没想到芝麻糖是干燥的撒一把，非常失望，奶油也是廉价的口味，倒是牛肉味的有惊喜。

马路边边麻辣烫

马路边边麻辣烫紧挨着成都吃客，都是工作日中午就排队的，晚上盛况如上。

店内装修成路灯下路边小板凳围炉吃热锅串串的模样，一进店就被热辣的底料熏着眼睛。签子一串5毛，大多荤菜也都是一根签。

「鸽子小郡肝」、「腰片」、「黄喉」、「小香肠」、「排骨」、「掌中宝」、「鸭舌」、「牛肉」、「豆腐」、「毛肚」都是必点。冒菜中「巴沙鱼」大块柔嫩质地最出挑，在成都，这样热火朝天的店，「冒脑花」更是没有过败笔。

Ifs国际金融中心

Ifs国际金融中心的爬楼熊猫了解一下。

集渔泰式海鲜火锅

集渔泰式海鲜火锅是标准的网红店了。11：30营业准时出现，已经排了三个拐弯的队在取号。晚上路过时已经有浓郁的冬阴功味散出来。三层空间，随桌服务。

锅底都是4小时熬制的野生鲫鱼原汤锅，汤色乳白浓郁。可以加不同调料成为川式鲜麻藤椒锅和招牌鲜料冬阴功锅。其中冬阴功用两种料包：泰国红辣椒、香茅、青柠、南姜，再加冬阴功酱料和现开椰汁。光是叙述眼见为实的加料已经足够有卖点。

双人餐不过198，AB餐的区别是「海蟹」/「扇贝生蚝」，包括用竹签串的「游水大活虾」，「青口」、「鱿鱼圈」、「龙利鱼」、「竹荪镶虾」、「虾滑」、「鱼滑」、「肥牛」、「午餐肉」。品质卓越，肉质鲜甜。加了份「青龙虾头盘」，还有「北极贝」随盘。

限量可选「起泡酒」、「焦糖布丁」、「冰淇淋」、「芒果饭」四选一，还有不限量的「酸奶」、「布丁」、「梅子冻」、「西米露」等桌边推车服务。

「焦糖布丁」和「布丁」都很浓郁胶着，「梅子冻」爽口，用杯子盛可以咻哦。

「蔬菜拼盘」里还有南极笋，分量可观。

「椰奶冻」有非常浓郁而本味的椰香，水果气泡水有颜有料。

是一家我想爆灯的网红店，小道消息南京总代已拿下，等着吃吧！

太古里

太古里和春熙路就好比三里屯和王府井，太古里大抵是年轻人最爱逛的综合体了，川西风格的青瓦坡屋顶与格栅配以大面积落地玻璃幕墙，传统又现代，低密度开阔的自由城市空间。又是工作日的人山人海，成都人都好闲适。
方所书店也是必打卡之一，有很多文创、森系服装、咖啡支撑，也有阶梯式阅读空间。
太古里还有linefriends，全国最大的muji，足4层，包括餐厅和家具软装。这一天终于有蓝天，而我们全程都在躲太阳。下午看了场江湖儿女，表示贾樟柯的片子还是不适合在电影院消化。

汶川映秀地震遗址

汶川映秀地震遗址，门票免费，观光车20元。从成都出发，走都汶高速往映秀去，震后10年了，一面是重建的宁静小镇一面是已爬满藤蔓的断壁残垣，依然让人震动。

卧龙中华大熊猫苑

卧龙中华大熊猫苑门票90元，网络票价75元。一路经熊猫大道前往卧龙保护区，走川西方向的推荐卧龙中华大熊猫苑，相比成都市区内的更近距离原生态且人少。有内环3馆外环6馆和熊猫幼儿园2个，从荒草的高度看，内环3馆和熊猫幼儿园2馆应该失修很久，毕竟来往人少又不在市区，连指示牌都没有。

开放时间为9：00-11：30，13：30-17：00，一定要早上去，下午的熊猫不活泼。

非常幸运赶早第一波还有直播在采访熊猫奶妈，于是饲养员挨个把小熊猫拖出来投喂。幼儿熊猫和成年熊猫画风差好大，大约就是我不做大哥很多年，看你那熊样的丧感。

还有一只叫滚滚的戏精特别多动，完成了爬树、打滚、奔跑、啃树皮、倒立对着树撒尿、泡澡一系列动作，感觉人生圆满了！
