携家带口丽江之行|探索彩云之南，行走丽江古城，梦回千古情

游记 主题乐园 家庭游 特色表演 美食 自然奇观 2019/07/14
所谓的旅游，便是从自己呆腻的地方去别人呆腻的地方，这些年，坚持带着父母去丈量祖国的大美河山，而前一阵厦门酷热的天气，便又动了出去避暑的念头。而那彩云之南的丽江，20度左右的舒适天气，便是最终的选择。这一站，携家带口，探索彩云之南，行走丽江古城，梦回千古情，感受玉龙雪山之奇秀俊美，当然还有那吃货必不可少的美食之旅。

【攻略篇】

1、如何抵达：丽江有机场，离古城大概40分钟车程，但是直飞丽江的航班，一般价格较高（至少从厦门飞丽江是这样的），一般比较省钱的方式是飞昆明，然后昆明坐动车直达丽江，今年昆明到丽江的动车开通了，只要3个半小时，票价220，非常方便。
2、关于天气：其实丽江最好的出行月份绝对不是七八月份，因为七八月份是暑假旺季而且又是雨季，基本上下雨的概率非常高，温度比起厦门来说倒是舒适很多，夜间在15度以下，白天也不超过25度，非常舒适。雨季要随身携带，雨来得急去得也快，“一山有四季，十里不同天”，听当地人说，甚至隔条街的天气都不同。早晚凉薄外套需要备着。
3、关于门票预订及当地团选择：丽江千古情景区推荐在官网预订，可以直接网上选座，到现场刷预订人的身份证取票即可，方便快捷。玉龙雪山的行程，可以自由行，但如果比较省事的是直接定个当地团，一分钱一分货，因为带父母出行，以舒适为主，我们定的是6人小团，有送雪山三宝（氧气瓶、防寒服、矿泉水）。
4、关于行程：
D1（7.8）：早班机山东航空SC4769 7:05厦门-昆明，12:32 D8780 昆明站-丽江站，16:00抵达丽江，打车入住考拉精品酒店，晚餐【石锅渔·山珍馆】，夜游【丽江古城】。
D2（7.9）：早餐【聚义堂·黎头米线坊】，逛古城游【木府】，午餐【聚义堂丽江腊排骨】，下午【丽江千古情景区】，晚餐【红景天土菜馆·野生菌火锅】
D3（7.10）：跟团【玉龙雪山】，观看【印象丽江】，晚餐二刷【聚义堂丽江腊排骨】
D4（7.11）：【丽江古城】闲逛，中国国航CA4452前往成都

【景点篇】

丽江千古情景区，寻找香巴拉的穿越之旅

对于千古情系列，其实并不陌生，先后去过三亚、杭州、阳朔的千古情景区，每次都有不一样的惊喜。恰逢雨季的丽江，其实在剧场里看一场七星级的演出，是最幸福的事了。毕竟带着父母，还是以舒适为主。
在丽江古城有直达景区的免费接驳巴士，出示订票信息即可，打车也不远，20元左右。跟所有的千古情景区一样，第一眼就充满了民族风情。天气不好，时不时来阵暴雨，但空气倒是格外的好了，入门先看到的是白塔广场，有祈福的含义在内。
定的是14：00的场次，还有些时间，便先沿着茶马古街逛逛，小哥哥的手鼓和歌声真的是极好啊，令人陶醉，都挪不开脚了。
隶属于宋城集团旗下的丽江千古情，演艺仍然是他的重头戏。其中的《丽江千古情》表演每天有3场，还有5D实景体验剧《大地震》以及大型实幕演出《丽江恋歌》。光看这三场演出就值回票价。其中《丽江千古情》和《大地震》都是在丽江千古情大剧院上演，因此不受天气影响。虽然之前看过三亚千古情、桂林千古情，但这次的丽江千古情在震撼之余，又多了感动，甚至是情不自禁的流下泪水。
《丽江千古情》是丽江文化的魂，用IMAX3D的大片视觉，重现《纳西创世纪》、《泸沽女儿国》、《马帮传奇》、《古道今风》、《玉龙第三国》等丽江长达千年的历史与传说，引领观众穿越雪山， 在旷远原始的洪荒之域、在泸沽湖畔的摩梭花楼、在挟风裹雨的茶马古道、在曼舞欢歌的古道重镇、在浪漫凄情的玉龙第三国、在世外桃源般的香巴拉相约一场风花雪月的邂逅，感受一个美丽的时刻。
最令人震撼的便是《马帮传奇》的乐章，千百年来，在崎岖的茶马古道上每天都上演着马帮们惊天地、泣鬼神的生命乐章，这天，肩负着朝廷和木府运输任务的马帮又出发了，经历了恶劣天气、山贼袭击，他们却是不屈不挠的坚守着这一段使命，着实让人泪流满面。而演员对角色的把握也非常到位，神态动作等都感同身受，配合音乐和特效的双重效果，整个演出现场真令人觉得不虚此行。很多观众也留下了感动的泪水。
今天，丽江作为古茶马古道上独具特色的中转集散地，也因此迎来了汉番辐辏、商贾云集的万千风华，处处丝竹管弦、曼舞欢歌。
当最后的经幡飘扬，白塔升起，每个人都不自觉的顶礼膜拜，对着白塔许愿，热泪盈眶。
看完《丽江千古情》表演，可以接着欣赏每天15:15《大地震》。演出真实再现了汶川大地震的惨烈场面，山崩地裂、房屋坍塌，3000立方大洪水瞬间倾泻而下，而且观众也可以参与演出，亲身感受演员的魅力，也经历一场不一样的心灵大地震。
当地动山摇之时，危急关头的母子、恋人做出了令人感动的举动，还有无数参加救援的解放军、医护人员，大爱无疆，众志成城。不畏艰险、气壮山河的抗震救灾证明，多难兴邦！面对任何艰难险阻，中华民族都不可征服！
愿世界和平，愿家人健康，大概是每个人心中最朴实而又最伟大的理想。
16:10分的大型实景演出《丽江恋歌》则讲述了藏族小伙阿布达与纳西族姑娘嘎吉米凄美而悲壮的爱情故事，风雨剧场里来自四面八方的惊奇和炫目瞬间将你包围，让你仿佛置身在火山喷发和山洪倾泻之中。
几场演出将你带入丽江千年的峥嵘岁月，不仅震撼也令人感动。而除了这些大型的演出，室外的几场滚动表演也是不容错过，入园要记得留意每场演出的时间节点。
首推锅庄狂欢和全民快闪。雨中围绕锅庄的火堆蹦迪，真是畅快。
还有跟随“四大才子”的全民快闪，将气氛推上高潮！在丽江千古情景区，所有人都可以参与互动，很好的将民俗和演艺结合在一块。
想不想体验一把木府乘龙快婿的感觉呢？每天两场“木府招婿”在景区上演，被绣球砸中就有机会参与哦。
还有流浪歌手驻唱，声线堪比大牌明星，唱得真的是极好的。
园区内也有各种小吃和民间艺人的表演，可以说是吃喝玩乐为一体。还有高科技的互动体验，大人小孩都玩的不亦乐乎。
景区位于玉龙雪山脚下，是一座活着的茶马古城，那措海、雪山沙滩、民族村落、走婚桥等主题区，古老的东巴文等让人感受一次寻找香巴拉的穿越之旅。
丽江千古情景区，来丽江旅行必打卡，值得花上半天时间来游览。不容错过！

丽江古城，登上世遗的古城

关于丽江古城，对于他其实褒贬不一，大概每个旅游景点都逃不开的商业化一直是困扰她的一大因素。4年前的匆匆一夜，留下了手机丢失的惨痛教训，在很长一段时间，对于丽江的印象总是不大好的。但这一次，慢下来，细细品味，发觉他的美好。住的酒店就在古城边上，因此几天时间几乎是闲着没事就往古城里转转。尽管雨一直不停歇，但却发觉她的别样美。
丽江古城，其实又常称为大研古镇，有时间的话还可以去另一个束河古镇。她的面积真的很大，比鼓浪屿岛面积还大。古城内都是石板路，数百年来被打磨得黝黑光滑，而古城内也有很多民宿，但如果要住里面，行李箱就比较不好拖行。第一天晚上夜游古城，很多人说丽江的夜才是精华，热闹、躁动，其实适合年轻人，但如果对于老人家来说，可能会更喜欢清晨的丽江，静谧祥和。
先说说丽江的夜，入夜时分的丽江古城，人渐渐多起来，尤其是酒吧街和大水车一带，几乎到了寸步难行的地步，这还是在飘雨的状态下，可想而知，如果天气好，是如何的热闹。每晚木府门口会轮番上演灯光秀，非常的美，值得一看，配合音乐的效果，仿佛穿越回木府土司的峥嵘岁月。
流光溢彩却又古香古色，这就是丽江古城的魅力，或是街上的那些油纸伞，或者是酒吧内透露出来的纸醉灯谜，都是那么的迷人。
作为地标的大水车，当然是必打卡的景点了。
而有个地方在大研花巷内的丽江之眼观景台，可以俯瞰到整个丽江古城的夜景，真的是超美了。相机无法记录的美好。
古城内的景点，必须要提的就是木府了。木府是丽江木氏土司衙门的俗称，位于丽江古城狮子山下，是丽江古城文化之“大观园”。整个建筑群坐西向东，是一座辉煌的建筑艺术之苑。“北有故宫，南有木府”，木府是一座辉煌的建筑艺术之苑，它充分反映了明代中原建筑的风采气质，同时保留了唐宋中原建筑古朴粗犷的流风余韵，而其座西朝东，府内玉沟纵横，活水长流的布局，则又见纳西族传统文化之精神。丽江旅游有一句话：“不到木府，等于不到丽江”。
从进入木府开始，就能感受到他的辉煌大气，不愧是丽江古城的“紫禁城”。396米的中轴线依次建有多个宫殿，层层往上，整体的建筑格局也是仿照紫禁城而设计，真可谓美轮美奂，无与伦比。
而推荐一定要登上最高点的，在狮子山极目下望，古城栉次鳞比的民居瓦房呈现出一片与天地混融的黛青色，苍苍茫茫宛若一幅巨大的水墨画，蔚为壮观。
慢下脚步，细细感受丽江古城的魅力。

玉龙雪山，雨季初窥神山之美，倾听印象丽江

或许南方的孩子总是对雪山有一种执着的向往，说起来，雨季绝对是上玉龙雪山最不好的时节，然而既然来了，总想着去看一看神山之美。雪山离丽江古城也不远，可以自由行前往，但跟团会比较方便些。山顶终年气温较低，要做好防寒保暖措施。跟团的好处就是氧气、防寒服都不用自己操心了。那么我们的雪山之行，就从氧气瓶开始吧。父母以60岁高龄也跟着我们登上了山顶，只要注意些，其实大部分人的身体状况都是可以的。
导游交代，氧气要从坐上索道就开始吸，每两分钟3-4次，一瓶大概可以按压45次左右。当然，依据个人身体状况不同，及时调整。我这种肺活量一直很小的人几乎全程都没有用上氧气，都给我爸了。上到高原切忌不要逞能，不要蹦跳，放缓行动速度。
索道下站是3356米，上到上站是4506米，1200米的高度差，在索道上升的过程中能逐渐感受。
老爸老妈在4506米的合影，这些年带他们又征服了一个新的里程碑。
从索道上站到最高点4680米，有木栈道通行，这一段向上的楼梯，我来回用了大概1个小时，还是有点喘，虽然天气不好，但在若隐若现中也见识了神山的冰川之美，惊叹大自然的造物，在自然面前，人类是多么渺小。
从玉龙雪山下来，去了蓝月谷，因为连日下雨的缘故，蓝月谷的湖水不是呈现蓝色而是如翡翠般碧绿。在这里还看到几对新人在拍婚纱。
壮观的大瀑布，气势磅礴。
《印象·丽江》在海拔3100米的世界上最高的演出场地，张艺谋执导的大片，全体演员都是非专业的，他们不是演员，却真实的表现了世世代代生活在此的少数民族的生活。骑马、喝酒、祈福等等这些都是融入到他们生活中的本色出演，却带给人更加具有冲击力的感官体验。
一个小时的演出，是一场真正意义上的荡涤灵魂的盛宴。

【美食篇】

对于吃货来说，每到一地，最不容错过的当然是当地的特色美食了。独特的气候、多元少数民族的融合，造成了她独特的饮食习惯。这次的行程，大部分还是安排在丽江古城内，三天吃吃喝喝，且听我一一说来。

石锅渔，一锅吃齐12种野生菌

石锅渔·山珍馆，位于四方庙会里面，装修色彩第一眼就爱上，打造了浓郁的少数民族风情，而店内到处是菌菇的图案。
来了云南就是要吃菌菇，作为原产地，无论是品质或者价位都比厦门好上许多。这家主打的便是野生菌菇锅。4个人点了一份398的套餐，相当划算。
汤底是老鸡半只，搭配12种菌菇，常见的竹荪、牛肝菌、羊肚菌、松茸、黄牛肝菌，还有从未吃过的金耳、猴头菇、老人头、皮条菌、荞面菌、谷熟菌，上桌服务员都会给你介绍。草帽当锅盖，在这里也见识到了！
菌菇下锅，配有定时器，一定要定时器响了才能吃。
凉菜四拼，包含茨菇、荞片、山珍菌、煎凉粉，第一次吃到凉粉用煎的，颇为惊喜。
水性杨花，一种云南特产的野菜，据说在泸沽湖盛产，也是第一次吃到，爽口。
豆面汤圆、山珍包、纳西黄金洋芋，都是手工菜，浓浓的纳西风情在里面。
菌香饭，主食搭配，用野生菌去炒饭，更加香，连一向不爱吃米饭的我都吃了不少
再搭配一杯店内自酿的梅子酒，简直完美！
一锅吃下12种野生菌，性价比超高，值得推荐！店内也有雪山三文鱼，可以做一鱼三吃，有机会也可以尝试。

清晨，食一锅云南米线

云南的小吃，就不得不提过桥米线了。来了丽江，当然要试一下。
黎头米线坊，位于木府斜对面，古色古香的店面，早餐吃完刚好去逛木府。
早餐的选择其实挺多的，但一心痴痴念念想吃米线，有小锅米线，也有仪式感较强的大锅状元米线。
先将小菜碟端上来，总共有8种，生鸡蛋、金针菇、火腿片等等。
米线上桌，要趁锅还滚烫将配料都倒进去，搅拌一下。
汤底是鸡汤，加上各种配料后更加赞，米线的分量很大，胃口小的两个人点一份就够了！
来云南，一定要吃一碗米线！

丽江必吃腊排骨，就来聚义堂

提起丽江的美食，腊排骨是必须要吃的，要吃必须得吃最好的。丽江三天，在这家15年老店聚义堂吃了两次，是真爱了。
木府店就位于木府旁边，逛完木府顺便吃上。整个餐馆的摆设和装修充满纳西族风情， 服务员都是身着纳西族民族服饰。
吃饭还能看表演：原汁原味民族歌舞，现场少数民族花式喝酒游戏互动：山高流水、杆子酒，很精彩。
主打便是腊排骨，腊排香而不咸，原汤鲜香可口，具有浓郁的野生菌味道。丽江独特的气候和天然绿色的环境，赋予其制作优质腊排骨的条件，在别的地方吃到的就不是这个味。
纳西烤肉，土猪肉烤得很香，可以包生菜吃。
可以搭配涮菜，当然少不了野生菌了，虫草花、菌三拼是好搭档。还有极品肥牛，造型很漂亮哦～
水性杨花和雪山野菜清香可口，几乎在丽江每顿都要点上这个水性杨花，厦门吃不上的美味。
服务方面，专业热情，介绍地道腊排从喝汤开始，主动用高温开水烫碗消毒，而且小姑娘笑起来真美。三天两刷聚义堂腊排骨，的确是值得推荐的！

黑山羊与野生菌的结合，食一顿土菜

丽江的小吃和土菜有很多，位于古城外围靠近四方庙会的红景天土菜馆就是其中一家。从丽江千古情景区坐接驳车回来刚好在附近，便来觅食。
整个装修风格也是充满浓郁的纳西风情，矮桌子竹椅子更接地气。
主打也是野生菌火锅，但它还有做一些丽江土菜，也有当地著名的黑山羊肉。点了份野生菌套餐小份，其实分量已经不小了，有8种菌菇，一起倒入土鸡汤底，十几分钟后便是一锅鲜美的菌菇汤。
套餐内搭配的纳西粑粑也是最爱，还有一小碟酱料可以蘸。
纳西米糕，看着像玉米棒，里面是米糕哦，口感软嫩。
搭配有菌香饭、传统小吃拼盘、水性杨花
特别推荐的就是这个黑山羊肉了，切成片状，烫个一两分钟就可以，没有什么膻味，口感纯正。
吃吃喝喝的丽江之行，最难忘的莫过于聚义堂的腊排骨，石锅渔的菌菇锅，黎头的过桥米线以及红景天的黑山羊肉了。

【住宿篇】

这次选择的是位于古城南门对面的考拉精品公寓，进出古城方便，而且又避免了古城的嘈杂拥挤以及行李箱不好拉的缺点。整个公寓虽然比不上一些民宿的花哨，但简单实用，而且房东也超热情，还有两只可爱的狗狗（虽然我不懂是什么品种）

【后记】

有人说丽江太浮躁，但这一次的丽江行，却让我感受到了这份来自纳西儿女的真诚。彩云之南，一个梦开始的地方，那里有能倾听神的声音的玉龙雪山，有山脚下感受寻找香巴拉穿越之旅的丽江千古情景区，有那在世俗与古老中交替的丽江古城以及来自丽江的种种饕餮美食。我想，我会回来的。
