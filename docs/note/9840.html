！前言！

2018年12月30日晚，想着2018年即将过去，想着要带着雪姑娘过一个不一样的元旦，便决定在2018年的最后一天带着雪姑娘一起去淮南跨年（因为带着雪姑娘，我们便选择自驾游，这样比较方便），之所以选择淮南，一是可以去千年古城寿县探寻最后楚都的秘密（作为春秋时期楚国都城最后之地，寿县也是楚文化的发源地，有深厚的历史文化底蕴），二是可以去“淝水之战”发生地八公山登山锻炼身体（八公山不仅风景优美，更是汉淮南王刘安招贤纳士之地）。

淮南客运中心汽车站在火车站对面，交通便利，班次也较多，另外还有前往新集、焦岗湖毛集的班车。

最佳旅游季节

春秋两季：淮南属于皖北，春暖花开和秋高气爽的季节最合适游玩，此时的淮南景色优美，气候宜人，很适合登高望远；
我们此次是冬季游玩，景色虽一般，但寿县古城是人文旅游，人不多，玩的也比较舒服，而冬季攀登八公山，倒别有一番景致。

行李准备

【背包】一个背包，2天的行程，一个背包足够（主要是装雪姑娘的东西）；
【衣服】换洗内衣和袜子（只住一晚，又是冬季，外套不用带），雪姑娘的浴巾和秋衣秋裤（秋衣秋裤冬季当睡衣睡觉）；
【鞋】以适宜登山的运动鞋为主（八公山虽不大，但台阶比较多，一定要穿合脚适宜登山的鞋）；
【雪姑娘物品】奶瓶、奶粉（不到3岁的雪姑娘现在只是睡前喝奶粉，用奶粉盒装就足够了）、湿纸巾、洗护用品（每次出去外面都用自己带的洗护用品）、保温杯（游玩时也随身携带）；
【洗漱用品】带够2天用的量就行，为了减轻重量，可以都带小样；
【其他】零食（自驾游玩，放在车上打发时间）、日常药品（感冒药）、保温杯、电子产品等。

寿县古城

坐落于安徽中部的寿县，是一座千年古城，寿县古称寿春、寿阳、寿州，寿春是楚国最后一个都城，在此为都十八年，历经四位楚王，这里也是楚文化的发源地；寿县是秦汉以来千百年见的郡、州、道、府所驻地，见证了著名的“淝水之战”；寿县古城墙是一座保存较完好的宋代城墙，而在最新的安徽行政区划分中，寿县从六安划归淮南，给淮南旅游增添了一抹亮色。
现在的寿县古城内属于寿县老城，仍有市民在这里生活，古城不大，道路不宽，两车道的古城内稍微显得拥堵，但因道路为方格路，步行去哪儿还是都很方便的。

寿县博物馆
寿县博物馆是了解寿县历史文化必去的地方，凭借身份证就可以参观，里面陈列了不少文物、著作和经典典故史料记载，作为安徽省最早的博物馆，寿县博物馆以收藏寿县境内出土的历史文物为主，馆藏了许多战国时期楚文化体系的文物，馆藏量仅次于安徽省博物馆，尤其是青铜器和楚金币的收藏是寿县博物馆的一大特色。
【开放时间】周一至周五上午8点至12点（11:10停止入馆），下午14:30至17:30（16:40停止入馆）；双休日和节假日上午9点至下午17点（16:10停止入馆）；周一闭馆。
【门票】参观者凭借本人身份证、驾驶座、社保卡、老年证、学生证等有效证件登记免费参观。

淮南八公山风景区

八公山，古称淝陵、北山、紫金山，位于安徽省中北部，大小山峰四十余座，淮河流经群山文化，曲折环绕而东下，淝河也飘然而来，傍依南麓注入淮河；八公山重峦叠嶂，形式险峻，山中有峡谷盆地可据，又有胡泊溪涧进退，龙藏虎卧，历来是兵家争雄之地；1600多年前曾发生以少胜多的“淝水之战”，留下了“风声鹤唳、草木皆兵”的传奇故事；
八公山自然风光优美，古泉遍布，苍松叠翠，宛如一颗璀璨的淮上明珠；八公山地质地貌奇特，蜚声中外的淮南虫化石，是迄今为止地球上发现最早的古生物化石。

淮南子文化园
淮南子文化是以《淮南子》的文化为基础，充分挖掘并展示《淮南子》丰富的文化底蕴，依托历史悠久、风景秀美的八公山文化遗址，构建的一个集文化观光、人文景观、休闲养身于一体的多功能、高品位的文化性主题公园，主要有主入口广场、次入口广场、人文广场、自然广场和养生广场五个功能区。
养生广场
位于淮南子文化园西南侧的养生广场，以《淮南子》中有关“养生”类的养生文化为主题，主要体现保健养生知识、医学药物、茶艺炼丹等方面的内容；广场中心是由64块象征八公山豆腐的花岗岩石组成，广场四周有六禽戏雕刻。

人文广场
人文广场以“道”为核心，将具有哲理思想的成语故事，以雕塑的形式与广场相融合，充分体现了《淮南子》“道法自然”的精神，广场以草书“道”为骨架，切合地形起伏，行程高低错落，生态自然的广场，并布置了八个在《淮南子》中均有记载的哲理成语：嫦娥奔月、后羿射日、女娲补天、一叶知秋、鸟尽弓藏、塞翁失马、削足适履、赴汤蹈火。

淮南子文化园主入口
山不在高，有仙则名，八公山之名源于汉淮南王刘安和“八公”得道成仙的故事，两千多年前，汉高祖刘邦之孙--淮南王刘安在此聚集天下贤士，讲经论道，著书立说，筑炉炼丹，寻求长生不老之术，他的数千门客中有8位最为著名：苏非、李尚、田由、晋昌、左吴、雷被、毛被，时称“八公”；主入口的刘安与八公雕塑是中央美院设计雕刻而成，而他们后面的《淮南鸿烈》又名《淮南子》，是“八公”及其他门客共同协助刘安编撰的中国第一部百科全书。

十二月令
《淮南子》集成了《吕氏春秋》十二纪和《礼记月令》的研究陈国，创立了《时则训》的纪年体系，十二月令壁画内容均取自《时则训》中某一句或某一段文字，构图采取一月一个小画面，十二个月连成一个整体组成一个大画面，每一个月的小画面又分为上中下三个基本部分：物候、天象或气候，人事代表家禽或吉祥动物。

二十四节气广场
《淮安子.天文训》中第一次完整、科学的记载了二十四节气，用北斗斗柄旋转的方向和度数，确立了纪年、四季、二十四节气的历法体系，二十四节气全年365 1/4日，十五日为一个节气，运行一周是二十四节气，核心为二分、二至、四立（四季的开始），二十四节气还反映了气候特征和物候的现象，如雨水和惊蛰等。

八公山地质博物馆是八公山国家地质公园的重要组成部分，坐落于八公山内的博物馆主体建筑依山临谷，周边环境优美，整个建筑为两层仿汉建筑，博物馆内有矿物岩石厅、古生物岩石厅、矿产厅、奇石厅、地质灾害及防治厅等展厅组成，另有影像播放厅、八公山地形地貌沙盘区以及相关科普教育等活动。
【门票】免费
【开放时间】夏季8点至17:50，冬季8:20至17点。
