高品质民宿不止台湾，浙西这家金宿颇具情怀，让耳目一新

游记 客栈民宿 摄影 登山 美食 自助游 2018/11/13

关于民宿一些肤浅的认知

我们都知道全国民宿最早起源于台湾，并且发展最好的也是在台湾。台湾的民宿可谓是在宏观与微观上的融合可谓是天衣无缝，在硬件设施与服务质量这一块也是双赢的。故而在全球有这样的一句话，那就是世界民宿看中国，中国民宿看台湾。
在最近几年的时间里，浙江根据自己的历史文化、农业状况、人均收入、资本市场以及国家政策导向诸多优势来打造出了属于中国特色的浙江民宿板块，其中叱咤全球民宿界的莫干山民宿体成为全国民宿业发展的标杆了。
浙江民宿在这样的前提之下发展成了属于自己的民宿文化特色，也是全国是仅次于台湾之后的最好的民宿体系了，这样的好不仅是在硬件质量上，更有在服务与文化建设这一块了。
既然我们都知道了民宿这个名词后，那么到底什么是民宿呢？估计很少有人回答出来或是回答的不尽全面。在浙江西南地区有这样的一个城市，知名度远不及杭州、宁波、乌镇、西塘等地方，但是它在浙西南却悄悄的崛起，成为浙江经济发展中不可估量的黑马，它就是衢州市了。
衢州是传统的农业城市，山水环境优雅，根据自身的山水资源并结合了本地真实情况发展绿色环保型的旅游产业，民宿的发展也成为一个比较成功的旅游产业了。
这一次来的浙西是以体验民宿为主题的旅行，感觉非同一般。

印象衢州圃舍

在衢州市柯城区的九华乡大云头村看到了许许多多的农村房舍，其中有一家民宿显得格外的耀眼，除了它与众不同的装修风格之外，那就是颇具民宿情怀与特色的氛围。
山野君我路过于此忍不住轻轻地推开院门进入民宿里面，给我的感觉真的很不一般。在农村民房里能够建设这样有特色的民宿非常少见。
在这里有优越的住宿环境，有优美的且无污染的山水环抱的环境，有农村村民里的田舍，更有宛如一家子人的民宿的小伙伴们与你不厌其烦的唠嗑着。感觉这才是真正的民宿，唯有的环境与范围方可敢对外大声官宣“我才是真正的民宿”了。那么接下来就跟着山野君我一起来看看这里的民宿吧，也许你也会觉得这样的民宿能够真正的定义得了“民宿为何物”的概念了。
这是一次因感悟民宿文化而起的旅行，其收获也颇多。那么就跟着山野君我一起来感受一下这个颇具民宿韵味的衢州圃舍民宿吧，估计你也会喜欢这样的感觉。
从外观总体上来看，这圃舍民宿与其他农民的房舍并无两样，唯独不一样的是颇具性冷淡与禅意绵绵相结合的宏观印象，第一时间感觉这是非常有感觉的民宿。艺术白气息的外围墙第一时间就给我留下了非常好的印象，就有让人想去入住的冲动了。进入院子内即可看到总高不过4层之高的民房村舍让人留下深刻记忆了。

民宿的公共区域

作为民宿的圃舍，它们的公共区域说大不大说小也不小。拥有小型的运动场地、菜园以及森林探险基地等配套设施。客房门口大约8米距离就是一个小型的运动场地，打打篮球或者跳舞之类的都非常的合适。在运动场地的最边上的位置就是一块几分地，可以种植很多蔬菜瓜果之类的，主人亲自种菜并烹饪，也可以自己亲自体验这样的生活。我想这样的民宿生活用陶渊明先生的“采菊东篱下，悠然见南山”应该是非常合适不过的了吧？
在一般人的认识当中民宿就是民房改建作为非公资本与非公管理体制的一种私营住宿的场所，所以一般都是只是简单的过夜休息之地，其实这个认识远远不够。古代的民宿与现代的民宿完全不一样了，因为时代发展与物质、精神双方面追求都不一样，产物也不可能完全一样的了。

圃舍的楼层与过道

在圃舍的走廊、过道，甚至是楼梯都能感觉到是一种靓丽的风景线，让人在步行的过程中即可感受民宿的氛围与入住的美好气息。走在三楼的客房的过道边即可鸟瞰到楼下的一群群的民房还有四周环绕的绿水青山，感觉真的可以用妙不可言来描述了。

圃舍的周边环境

圃舍民宿在九华乡的上云头村里几家民房改建而成的，所以周边都是被一群群新农村建设后与古村落相结合的村子房舍给包围了。周边的山水清澈、环境优雅，到处都能看到新农村的新面貌，顿然间感受到祖国的美好与百姓们的安居乐业。走在这样的小村子里居然也能够感受到身为中国人的幸福、骄傲与自豪。继续前进五百余米的路途即可见到衢州市的一座原始森林，森林的绿色植被极高、空气负氧离子含量浓，还有一个户外的体验项目，感觉很不错的。
绿色青山就是金山银山，在这里就已经应验了这样的真理。在浙江西南地区的衢州九华乡的上云村，就能发现这样的环境优美、人居环境极佳的农村面貌了。作为浙江四省通衢的衢州，农民问题是一个大问题。解决了农村问题就是解决了最基本的问题。
在衢州九华乡的上云村的村子里可以看到这样山水环绕的村民房舍，这样的居住环境让很多城里人都非常的羡慕不已了。浙西衢州的新农村的成功建设，不仅是我国新农村的典范，还是中国新农村建设中令人瞩目的成绩代表地之一。
中国的新农村建设提起还是在十多年前，在国家的支持与地方的努力之下已经取得了骄人的成绩了。在衢州这座小村子里即可见到日本国才会有的农村面貌，甚至这样的农村环境胜过日本很多农村地方。
作为中国农村是非常幸福的了，因为拥有这样优美的生态环境，能不说是一种幸福吗？农村环境的改善就是中国崛起与复兴的标志之一。你觉得 是不是这样的呢？
依山傍水的农村环境也许在国内也不少见，然而能够有这样的生态环境又能有如此宜人的新农村面貌与优越的经济发展水平，在国内可能不是很多。衢州上云村就是为数不多的新农村当中的那么一个。

走在村上的街道上，看到的不再是泥泞的土路而是干净整齐的水泥路或沥青路。村口里的房舍不再是昔日破旧不堪的房子而是舒适美观的别墅楼群。从这里的老房屋与新房舍的对比即可证明这个村子新农村建设的成功了。更值得一提的是这老房子与新房舍都是同一个农民主人家的。因为怀旧老屋，所以就把老房舍继续保留了下来。
我们眼前这里的房屋居然是远离衢州市区约20公里的农村里，昔日这座村子交通落后、经济水平欠缺，而现在则是交通四通八达，村容村貌改善了很多，衢州市区有直达这里的503路公交车呢，20公里的路程才40分钟票价才5元钱。
作为国家很注重也很关心的新农村建设问题，已经渐渐地在全国形成了特色，成绩也很突出。当你走在这衢州上云村这里，也许能够真正的体会到新农村建设的成功，同时也能够感受到这新农村建设的重要性。
这个新农村建设的典范的上云村，街道整齐干净而且功能设施齐全。连停车场都是以生态环保为主，其他的公共基础设施更是如此了。衢州这个村的新农村面貌可以代表着我国的新农村建设成功与典范，你认同吗？你认为还有其他的新农村能够与这个村比照的吗？

圃舍的地理位置与交通

最后呢，这家圃舍民宿虽然距离衢州市区约有18公里远，还是衢州市九华乡下的一个村子，但是交通非常便利，无论是自驾还是自公共交通都非常的方便快捷。自驾的话开个导航基本上无忧无虑的到达圃舍了。如果是乘坐公共交通工具也很方便，市区斗潭公交总站始发，终点站到上云村的503路公交车，在衢州市区可以乘坐。如果是在衢州火车站的话，建议乘坐7路公交车到达斗潭总站换乘503路即可了，503路公交车是按路段收费的，从上云头村坐到斗潭公交总站是5元钱，无法使用微信、支付宝等网络支付工具只收现金，要乘坐公共交通工具的朋友们一定要事先备好零散、足够的现金。从衢州火车站乘坐7路公交车到斗潭换乘503路公交车到圃舍的整个行程耗时大约50分钟到1个小时之间。
