龙游石窟，探秘世界未解之谜

龙游石窟
龙游石窟是位于浙江省龙游地区的地下石窟群，自从1992年被发现以来，经多家新闻媒体的报道，披上了一层神秘的色彩。从金华市区出发，行车一个多小时大约70余公里的路程到达目的地。此地处浙江中部，属衙州境内，一个叫石岩背的小山村，这个坐北向南、背山面水的小村落，自古就有“风水宝地’之说。站在村前眺望远处，一条衢江环绕而过，视野开阔，颇有神往缥逸的感觉。
龙游石窟
龙游石窟是衢州的特色，游客每次来龙游必须打卡的一个景点，他是世界的第九大奇迹之一，为什么这样说呢？当你来到景区后，你会发现它是从上往下挖几十米深，这是何等的智慧和坚持。所以说，古人的智慧和对于建筑的理解是特别厉害的。几千年前他们挖到这个，才能成就我们的历史景点。

龙游当地美食，让人欲罢不能

龙岩的美食是给我的印象最深刻的，这次旅行中我们吃了两个店，一是龙游小吃店，一家是龙游大酒店，两个店里的菜品风格大同小异，却又有着区别，龙游的菜真的太好吃了，龙游菜到底有什么，接下来上一些图与各位分享。

龙游肉圆
龙游肉圆也叫“山粉粿”，主要是由红薯淀粉及芋艿制作而成的。红薯淀粉在当地又叫“山粉”，因此得名山粉粿，是民间村妇制作而成。据说龙游肉圆起初是由葛粉制成的，但由于葛根粉资源缺乏，人们便把葛根粉换成了番薯粉，这样不仅能够在那个物质紧缺的年代填饱肚子，还能尝到像猪肉一般的美味。
制作龙游肉圆时，人们会在红薯粉里面加入一定比例的芋艿，少许肉末,还可以根据喜好加入豆腐或萝卜丝。龙游肉圆外形呈不规则半圆形，扁扁圆圆，高低不平，酷似农家菜里的狮子头，所以叫肉圆。龙游肉圆色如煮透了的茶叶蛋，暗红透亮，晶莹剔透。撒上点葱花点缀一下，在红绿相间之中透出浓烈的传统小吃魅力。龙游肉圆在红薯粉的作用下口感非常劲道柔软，嚼劲十足，且醇香诱人，咸淡适中，回味绵长。
葱花馒头
葱花馒头是龙游人逢年过节、吉庆宴席必用的传统小吃。龙游葱花馒头不仅因为可口味美，更因为它的圆润和饱满外形，被龙游人视为吉祥圆满的象征。制作葱花馒头的主要原料是猪肉、毛笋干、茭白、萝卜香干等组成，葱花馒头的馅料也随着季节的变化而有所变化。
制作好地道美观的葱花馒头第一要素是选择好弹性充足，而富有张力的淡馒头，只有足够的张力，才能保证撑起肚中的馅料，使外观足够饱满和圆润。其次是制作好美味、地道、鲜香辣适中的馅料，这是葱花馒头的灵魂，葱花馒头的活力和诱惑全在这一锅的馅料之中。
龙游葱饼
龙游葱饼，也叫烤饼、烧饼，是龙游街头巷尾非常常见的一种小吃。龙游葱饼起源于胡饼，后逐渐演变成烤饼，再由烤饼演变而来，与烤饼没有本质的差别，只是在烤饼里面放入了当地独特的野生香葱，也称“狗细葱”。此葱香味浓烈，余味绵长，所以特别香。
北宋靖康年间，皇帝昏庸无能，战乱不堪，人们为了在逃难时方便充饥，聪明的龙游人便发明了葱饼，将饼放在炭火边烤熟食用，这样子不仅便于携带也利于保存，行军打战或逃难途中不至于挨饿，慢慢的这种做法也就流传了下来。

前排人们放松的在湖面飘着，我们一共有3个船，有条不紊地隔着距离，也是方便我们去拍照。老大爷们划船都特别有经验，知道哪里水深或者浅，他会告诉我们哪里会比较安全。一般左右两边一定要对称而作，体重相差不大才能保障船的稳重，整个时间大概在一个小时，在大部分的路上都是平稳的漂浮，偶尔会有两到三个高低坡，整个行程都是特别放松的。
