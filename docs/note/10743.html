【首发】【江西羊狮慕】一个看见云海佛光都是家常便饭的景区

游记 小众 摄影 登山 美食 自然奇观 2018/11/06

前言

或许是这里的山实在太美了，所以我的无人机才不愿意离开，永远都停留在了这里。这是它飞行的第89次，它跟我走过沙漠，穿过云海，好几次差点以为回不来时候它还是回来了，而这一次因为贪恋佛光，贪恋山川云海，所以它抛下了我，选择日日夜夜在这里坐看云起。
这里是江西羊狮慕景区，一个24小时，1440分钟里，每时每刻都在不停变化的地方， 这里有壮观的日出日落，有永远翻滚着的云海，有出现频率极高的佛光，还有那跨越山谷的双彩虹，嶙峋峭壁等等美景，哪怕是常年做户外登山的人也会被这里的美景所惊叹。说来也是我们运气不错，短短3天里，竟然把这个季节能看到的奇观都看到了，也不枉我们走这一着了。

我们的行程

D1 启程-羊狮慕大峡谷观日落
D2 天子峰齐云台-谿谷道径-栈道（流云台-齐云台）-星空
D3 观日亭－归程

D1 初见羊狮慕大峡谷

我们到达羊狮慕时已是下午，当时天还嘀嗒嘀嗒的下着些小雨，但是依旧还是选择出门看看，毕竟有雨也是一种别样的韵味。或许正是这样佛系的心态，才让我们在哪里遇见了双彩虹。
司机师傅把我们放在了路口，穿过一道正在修建的建筑就可到达流云台。一眼看过去立刻被这里的气势给镇住了，栈道两侧是悬崖峭壁，中间大峡谷里沟壑种满了五颜六色的草木，美不胜收，旁边紧接着就是一望无际的云海，仿佛仙境一般。
此时，天空还淅淅沥沥的下着小雨，但是太阳已经逐渐从云层里跑了出来，正当我嘀咕这天气会不会有彩虹时，一转身，竟然发现山后出现了一条淡淡的彩虹，然后逐渐加深，五彩虹变成六彩虹和七彩虹，红茶橙黄绿蓝靛紫都出现了，形成一个跨山大彩虹。那时有飞无人机的小伙伴甚至还拍到了半球星彩虹和佛光。羊狮慕以这样的隆重的景象欢迎我们，怎能不惊喜意外？
当彩虹逐渐消失，我们以为一切该落幕时，远方又开始扫射出阵阵耶酥光，一道一道的扫射在山上。当你正痴迷于耶稣光独特的节奏时候它突然又消失了倏尔天空出现的一朵云仿佛就像雄狮吼天一般，一闪而过，又化作一条游龙，在山间嬉戏一会，摇着尾巴游走了。
羊狮慕景区之所以叫羊狮慕，一则是因为宋代大诗人杨万里求学期间游历此山，见山中奇石壮景而流连忘返，日夜思慕，尤其在观赏了鬼斧神工的石笋峰后，留下憾世诗篇:”笔锋插霄汉，云气蘸锋芒。时时同挥洒，散作甘露香”。景区因此得名‘杨思慕'。又因景区经年云雾蒸腾，常现'羊'、'狮'追逐嬉戏于山间的气象景观，而有‘羊狮慕'之名。所以我们看到的雄狮吼天的景象启示也是景区常见的景象之一。

于卧龙阁品山间珍馐

从城市到山林里，气温的骤然下降让身体变得暖和些，所以饭局时店里特意拿了珍藏的血藤酒出来招待我们。说起这血藤酒，功效还是很神奇的，可以治感冒，喝后身体会变暖和，平时不喝酒的几个人，酒为了暖合和治感冒都喝了下去，颜色似血透红，味道不像那些白酒那边呛和刺激，喝下一小口，身体酒就慢慢暖和起来了。
而桌子上的菜大多是山里的野味，不是海鲜大餐，却是十分难得的绿色蔬菜。像紫山药，野白菜，土山鸡等等，都是山上野生的或者养殖的，吃着非常健康，口感也特别清新。再加上师傅传统的江西土家菜的做法，我们一行人吃的都十分的愉快～

D2 追寻日出，踏访林荫小道，赏云海落日

天子峰日出

既然来了高山，便没有不看日出的道理。为了寻这个日出，我们五点钟起床，五点半就开始登山。那时天还没亮，四处漆黑一片，眼前都是白茫茫的雾气，顺着走道爬上了天子峰峰顶。
据景区的工作人员介绍，天子峰的峰顶齐云台是个相对不错的观日平台，空地虽然不大，也没观日台视线开阔，但是倒是个欣赏日出云海的好地方。刚登到山上时，周围云雾缭绕，一哈气都是白气，感觉进入仙山了，坐在有树枝的台阶上，上面时不时就会滴落很多水滴，没一会头发就都打湿了。我们都在说，等到冬天时候，这里的雾凇一定很漂亮，一定会是个粉妆玉砌的世界，约定着等入冬后再来一趟羊狮慕。
当天空渐亮，人眼视线范围内却还没等到太阳出来，这云层太厚，等不到日出那瞬间了，于是有小分队提前走了，我们几人坚持留了下来，才等到而后的佛光。以前别人都说佛光难遇，但是到羊狮慕没多久又再次看到了佛光，在羊狮慕，这个遇见的概率实在太高。如果你有无人机，不仅在日出不久后能遇见佛光，甚至到了九点十点你还是可以看的到，它就一直在山谷的那个地方，久久不散去。

谿谷道径

谿谷道径是羊狮慕的二期工程，这一条路线其实还未完全建成，还属于开发阶段，所以逛起来还是很“原始”的感觉。据了解，日后还会有建立个比玻璃栈桥还要有意思的项目，预计明年年底才对外开放，所以这个地方我们也只能从道路边上的简介了解到，并未有机会亲眼看到。而沿途却是看到了不少小瀑布，绿树青苔，这里是个未被开发的天然氧吧。
尽管现在不是水季，但是谿谷道径沿途一路都水流不断，数不清的小瀑布与野蛮生长的绿树组成了这里的奇景～如若你够仔细，脚下的砖有时还能看到生长的菌类，而树枝上，岩石表面随处布满了青苔，水的边上树还长了许多分枝，那是环境极好的地方才会这样。有工作人员告诉我们，那些常年在山上干活的民工，到了城里会不习惯，会有缺氧的感觉，他们长期在山林里呼吸新鲜空气，充足的氧份，所以才会产生这样的现象。
谿谷道径里目前中途设有2处可以休息的地方，一处是建在小瀑布之上的小亭子，人可坐在亭子里，水流从脚下留过，很有趣味；另一处是索桥前一块空地上摆设了亭子卫生间与几张木质桌椅。从这个地方下去，就能到达谿谷道径的底端。
那里是一个几十米高的大瀑布，水流从最上端直冲下来，一泻千里，不知最后是汇聚到何处。只见瀑布旁边小的观景台上只能允许一人站立，在那儿，似乎就达到天人合一的境界了。

栈道

栈道通常是从流云台开始，往右边走是凌云台，那里能云海的视野最开阔，往左手边走会靠近天子峰，天子峰山体上有很有意思的天然形成的壳子，上面写着天子二字，还有一个繁体的梦字，而发现这几个字的时间正好是我国推出中国梦的时期，天然的时刻与政策理念不谋而合，这才令这个山颇具传奇。当然这不是最神奇的事情，在羊狮慕景区，曾发生过很多传奇的故事，传奇到以至于听起来如神话故事一般。
比如景区在修建时，十几个民工因误食毒蘑菇中毒进医院，然而全部都痊愈了，后来继续在工地里上班，当时在他们被送进医院的时候，羊狮慕景区里面出现了火凤凰的云朵在空中展翅高飞，这个图片我们还看到了。或许是这福山知道修护他们的民工无恙才展露出此现象。关于羊狮慕的各个角落，总是有很多很神奇的事情发生，这或许是他们把他称为福山的原因吧。我们在羊狮慕听了很多的离奇故事，如果你也有兴趣，不妨让导游讲给你听。
越过天子峰，这里还有左将军右宰相，还有传到诵经的老者。他们一同守护着这座云海茫茫的佛山。据说再冷些的时候，栈道面前的这些云海还会高一些，走在栈道就仿若置身云海，我们一路打趣景区的工作人员，他们是每天过着神仙般腾云驾雾的生活。还时不时见个佛光参禅。没错，在栈道边上也是可以看到彩虹和佛光的，而且这里的佛光离人的距离最近，仿佛用手就能触摸到一般。

星空

生活在大城市里，看到星星是奢侈的，多少次走在城市的街道上，总是会被街灯和车流晃了双眼，而只有在深山老林中，才能一睹星空的美妙。我们随意找了块大草坪，把背包一丢，往草坪上一趟，漫天的星光便闪耀在眼前，这是久违的一种安逸安详与安静。

D3观云台日出

由于昨天没能亲眼看到日出，大伙并未甘心，为避免带着遗憾离开，我们依旧选择了早起。当我们出门后发现，其实另一波人马起的比我更早，甚至已经把绝佳的机位全部占领了。经过一番沟通才得知，原来他们是当地摄影协会的，也是来这里进行一个采风。
今儿我们日出的欣赏点，不再是昨儿的齐云台，而是视线更开阔的观日亭，这里虽然没有天子峰峰顶的海拔那么高，但好在视野开阔。一开始我们还在抱怨远处的流云不动，离的太阳升起的点远远的，而当太阳如同一个金黄色的蛋黄从云海平面跳出来的瞬间，周围的流云也躁动了，如一个个穿着华服的公主，分布在太阳的周围载歌载舞。
那一刻，我们真的满足了。它似乎是只知道我们只停留3天，所以在这有限的时间里竭尽全力的去绽放它的美。这3天，我们坐看云起时，横跨彩虹桥，守着日出日落，观星品茗，似乎从来还没一座山能让人感觉这样的满足。真不愧是武功山的姐妹山，风景独好，也难怪有人评论说，看了照片只觉得黄山第一，牛背山第二，羊狮慕第三。恕笔者走过的山川还不够多，虽走过五岳中的三岳等一些名山，但是黄山，牛背山，武功山都还没能去过，就目前走过的山川来说，羊狮慕还是排在前山的。之前去的名山，从没能看到过如此多的景象，唯有这里独有，这让我们不得不相信这里确实是一座福山，愿你来此山也能收获到你的福气。

后言

就这样，我们结束了3天2夜的羊狮慕之旅，这里太美实在怎么看夜看不够，我们小伙伴相约着，等到天气更冷，这里漫山雾松时，我们再一起前来观赏。下一次，你要与我们一同前行么？

关于美食

江西美食多数以辣为主，在江西，炒菜很少说没有不放辣椒的，所以整体口味是偏重些的。山里也很多野菜，口感特别好。还有用山里植物自制的茶叶，血藤酒都是不错的好东西，如果有机会一定要来尝尝。尤其天气冷的时候，一杯血藤酒下肚，身体立刻就能暖和起来。

关于交通

羊狮慕景区位于江西安福，最近的高铁站是宜春站，从宜春站过来是60多公里，开车一个多小时。

关于住宿

山脚有温泉酒店，山上也有几个酒店，如果阿你想观看到日出和星空，以及更好的欣赏日落，最好是住在山上。我们住在的是卧龙阁，一晚上518元，有床垫子，晚上睡起来特别暖和。

门票信息

羊狮慕景区
地址：江西吉安市安福县泰山乡
门票：250元（含索道,单纯门票110元）

实用小贴士

1.上山最后走索道，虽然后面有车道可以上山，但是相当不方便，最好的方式是乘坐索道。
2.山路景点之间有的距离比较远，但为方便游客，站点之间设置有游览车停车点，在站点等车便好。
3.山上温度比山脚温度低，尤其是早晚的时候，出门一定要穿上羽绒服。
4.山上的天气预报不是很准，有可能山脚是阴雨天，但是到了山上可能就是大晴天。
5.如果想看日出日落，一定要住山上，否则很多美景都看不上。
