盘点贵阳那些好吃好玩的地方

游记 人文游 美食 自助游 自然奇观 2018/12/03

关于美食 4家必去的餐厅

阿杜炒蟹（云岩店）

作为贵阳最火的人气餐厅之一，无论打开哪个美食榜单【阿杜炒蟹】都是绕不开的那家店。
TA火爆各大网络平台，甚至还多次上过央视。在贵阳说到吃炒海鲜，阿杜炒蟹绝对是很多人的不二选择。 
刚进入餐厅，就被那复古的工业装修风格格所吸引，各种形状的灯泡或高或低的悬挂着，黄色的灯光很容易将人带入食欲的世界。红砖墙上大大小小的螃蟹图案，又为餐厅添了几分俏皮可爱的元素。厨房是开放式的，可以看到食物的制作全过程。
香辣炒蟹是这家店的招牌菜。不过很多人都会担心食材问题，毕竟贵阳是大山的女儿，所以担心很正常，不过值得一提的是，阿杜炒蟹每天都会空运新鲜的食材到贵阳，而且只用进口的蟹，按季节性变化蟹的品种也会不同。
这道菜简直就是美味中的尤物让人无法拒绝啊。丰富的香辣味混合着新鲜宰杀的蟹肉，让人欲罢不能，店家很贴心，蟹钳已经被摧毁，所以可以轻松吃到蟹钳里面又厚又肥又嫩又白的肉肉，香气满满，怎么吃都不腻，就连炒在里面的葱姜蒜都让人舍不得放过，真是太过隐了。
香辣花甲，也是店内招牌之一，每一个花甲都充分吸收了酱汁的精华，肉质饱满紧实且弹牙，一口咬下，鲜香的肉汁飞溅而出，那味道简直绝了。
还有香辣螺丝，颗颗厚实又入味，每一颗都恨不得多嚼几下。
尖椒鱿鱼，虽然我深爱阿杜家的香辣和葱油味，但是比我更重口的不少人表示不够辣！那么我们来商量一件事好不好？我负责吃鱿鱼你负责尖椒好不好，拌饭特别好吃又过瘾哦！
火爆米豆腐，来自苗族的问候，作为贵州传统特色小吃，色泽金黄，咋一看以为是土豆，实际上是柔嫩中带着韧劲的米豆腐，拌上干辣椒，青葱等佐料，变得更加灵气有味。
值得一提的是阿杜炒饭，作为一道饭后餐点，味道却颇有几分喧宾夺主的意思，米饭里面配的牛肉粒，外焦里嫩又松软无比，配上可口的时蔬味道无与伦比，难怪每桌必点。

阿杜炒蟹
地址:贵阳云岩广场成都路1号1层
营业时间:17:00——1:00

柒食贵阳味（亨特店）

每到饭点，店门口便会排起长龙，可见【柒食贵阳味】在美食圈的影响力。店内空间汇聚了贵阳的多种路标——黔灵西路、贵惠路、盐务街、文昌南路、花溪大道等，精致而典雅的装饰布局蕴藏着老贵阳的文化底蕴。
都说贵阳人无辣不欢，那是因为你没有去过【柒食贵阳味】。如果你去过【柒食贵阳味】，你就会发现，原来贵阳人的回忆里有那么多不辣的菜，原来贵阳味“无辣也欢”
贵阳味并不是严格意义上的黔菜，它陪伴着生活在这座城市的我们长大，或许没有精致时尚的摆盘，但那种怀念的家的味道就是贵阳人记忆最深的味道。
有辣椒的贵阳菜虽然很带劲，但是一家人去吃饭总要考虑到老人和小孩的口味。柒食家是带老人小孩去正宗贵阳味的最佳地点。
贵阳人对贵阳菜还是那么钟情，在家懒得做饭的时候就适合带着一家人来这里感受下回忆里的老贵阳味道。
在吃麻辣上我们不输给四川成都，但是贵阳人的特色不只是麻辣鲜香，还有很多清淡的贵阳味。
吃多了麻辣的重口味又回来考虑下“养生”的贵阳菜。如果还有人跟你说贵阳人口味重，你就直接让他去【柒食贵阳味】让ta们好好了解一下什么是丰富多样的贵阳味。
tips:
柒食贵阳味(亨特店)
地址:贵阳文昌路70号亨特国际广场3层美食汇
营业时间:10:30——22:00

丝恋红汤丝娃娃（南国花锦店）

《大长今》里长今给爱人制作年糕时的样子，用心用情，满面含笑，笑容发光。美食，是这世上最好的情书，我喜欢你，希望你吃得好，吃得开心，希望可以永远这样共享美食、共度人生。
上过《舌尖上的中国》的丝娃娃，贵州最具有代表性的特色小吃之一。在街头巷尾总会是一道亮丽的风景。来了贵阳不吃一次丝娃娃都不好意思说自己来过这座城市。而在贵阳众多丝娃娃店里，丝恋红汤丝娃娃是这道风景中最独特的存在，宽敞、明亮、洁净的环境，贴心的服务和符合大众口味的菜品，使得它深受人们的喜爱。
丝娃娃口感细腻，而且吃的时候，嚼的程度不一样，吃出来的味道也不一样，连味道都富有层次感，真是让人憨爽。丝娃娃是由面皮和各种各样的蔬菜组成，将面皮把多种蔬菜包在里面，再蘸上红汤，那可是美味至极。
说到红汤，那可是丝恋家的独门秘籍了，他家的酸汤是用新鲜西红柿炒制后再和发酵好的酸一同再熬煮5-6小时，汤呈红色，所以叫做红汤。所以这样的酸汤鲜甜可口，不会太过刺激，更有人曾说来丝恋家要先干三碗纯酸汤，开胃舒爽！
丝娃娃的吃法是很讲究的，如果没有吃过的人，第一次都不知道怎么吃，服务员会很耐心的手把手的都你，正确的吃法是：把面皮放在手里，在上面加上自己喜欢吃的蔬菜，虽然种类繁多，但是也不要包得太满，否则皮就会破了，将蔬菜全部放好后，可以按照包饺子的简单方法，将它包起来，或者是像包小孩子的被子一样的手法。只要保证里面的蔬菜不倒出来就行了。
吃丝娃娃最大的乐趣就在于需要亲自动手包，来了贵州，一定不要错过哦，如果你们是夫妻、情侣那就更要去吃了，可以像大长今里面那样，一个负责包，一个负责吃，与喜欢的人一起共享美食，共度人生。
除了有地道的酸汤和新鲜的蔬菜，丝恋的各种特色小吃也是大家都爱去的原因，从贵阳街头的宵夜烤五花肉、烤脑花到瓮安的摊摊再或是安顺的冲冲糕都能在丝恋的所有门店吃到，不用行万里路便可尝万千滋味！
tips:
丝恋红汤丝娃娃
地址:贵阳中华中路130号南国花锦购物中心7层
营业时间:11:30——21:00

思飨季（南国花锦店）

思飨季是一家美食小店, 在南国花锦商场的B1层。思飨季主打的是街边各类小吃，餐位设置比较少，提倡大家‘边走边吃’，更加方便。”
店面虽小，但装修风格却很酷炫，文艺的动漫风，时尚又不失民族风情。铁质的餐桌、座椅以及周围墙上装饰的工用齿轮，其中最醒目的装饰就是墙上写的“思飨季，有小吃就有乡愁”。一个“思”字，诠释了对小吃的浓浓思乡情怀。“思乡恋情食飘香”便是对食物思恋最好的体现。“飨”，即体现了贵州人的热情好客又展现了对家乡美食的依恋。而“季”则是回归传统二十四节气对食材的考究，就连品牌的LOGO都以贵州古老文化的蜡染蓝为底，字体的一笔一落之间，尽显精心。
思飨季汇集了贵州各个地方的特色小吃。午后的闲暇时光,总需要美食来滋补身心。而思飨季，就是贵阳许多吃货们最爱的小食店，满足了吃货们不用东奔西跑就能享受一道道地道贵州小吃的美好愿望。
麻辣十足的炸土豆,松软入味的臭豆腐,香糯可口的烤卷皮,香脆可口的香肠洋芋丝,甜彻心扉红糖冰粉、绿豆汤,这些都是店里的招牌必点系列。此外还有掌中宝、橙汁儿等小食。
这些小吃中最让我惊艳的是炸怪味洋芋，洋芋外脆里酥、麻辣鲜香，洋芋经过油炸之后，将原有的洋芋香味紧紧的锁在了油炸之后得到脆皮里面，一口压下去，满满的香味就在整个口腔中乱窜，再加上酸酸辣辣的怪味，让人欲罢不能。

思飨季

青岩古镇(国家5A级)

青岩古镇是个600年前的古镇，白天比较热闹，游人熙来攘往，清晨和傍晚非常安静，也很漂亮！它不是建造在一个平面上而是建造在高低不平的山坡面上，从高处望去，整个小镇的格局给人一种在别的古镇中难以看到的立体美感。
所以您如果去那里逛的话，要背包去，穿轻便鞋。小巷子里全部都是有台阶的石板路，在傍晚的雨中散步是别有情调的。如果在里面住店的话，让店家接你进去，是可以免门票的。
古镇里，古建筑比比皆是，除以上八牌坊外，还有九寺：龙泉、慈云、观音、朝阳、迎祥（又名斗阁）、寿佛、圆通、凤凰、莲花；有八庙：药五、黑禅、川主、雷祖、财神、孙膑、东岳；有五阁：奎光、文昌、云龙、三宫、玉皇；有二祠：班麟贵土司祠、赵国澍祠；有赵状元（以炯）府、青岩书院、万寿宫、水星楼。还有一世界珍稀树木“青岩冷杉”；有神仙、黄龙、花山、璇宫四溶洞；有三叠系的古生物化石山。
古镇居民过着安静的生活，晚上六点多，许多店家陆续闭店休息了，小巷里非常安静，行人稀少，自己能听见自己的足响。店面大门还是老式的厚重门板，闭店时需要一扇一扇的安上，早上开店时再一扇一扇的取下来，立在一旁，一打开门板，许多店都连成一片了。
地址：贵阳市花溪区青岩镇
门票：10元；联票（含慈云寺、状元府、文昌阁、赵公专祠、万寿宫、龙泉寺、寿佛寺等小景点）：80.00元
开放时间：古镇全天开放，镇内小景点9:00-17:00。

黔灵山公园

黔灵山公园位于贵阳市区西北隅，园内可以爬山观景，也可以游湖、看钟乳洞，还可以去弘福寺礼佛，或者逛动物园。这座公园内容丰富，门票却很便宜，即使公园内要另外收费的小景点，也都只要几元钱，性价比很高。
黔灵山公园有东西南北四个大门，其中南大门是公园的正门，途经的公交车最多，上山索道也是在南大门这边的，所以建议游客从南大门进入公园。进入南大门，首先看到的是七星潭，泉水从麒麟洞流向公园南门，形成七个阶梯状的水潭。顺着七星潭走几步，就是上山路了。如果不想爬山，可以坐索道直达象王岭上的瞰筑亭，索道单程15元。
爬山的话，沿着九曲径拾级而上，山上植被茂密，空气很新鲜，一路上还能看到摩崖石刻群。山不是特别高，距离地面的垂直距离大约是200米。山顶有一座始建于清代的弘福寺，门票2元。据说寺里祈福很灵验，所以来拜的信徒很多。弘福寺后再爬一些山路，就到象王岭的瞰筑亭，在亭子处可以俯看贵阳群山中高楼林立的景色，是个不错的拍摄点。
若特别喜欢爬山，黔灵山公园的山是由几座山头连绵而成的，还可以去白象山、关刀岩、大罗岭爬爬，而一般游客看过弘福寺、瞰筑亭就差不多可以下山了。弘福寺及下山的这段路上猕猴特别多，不少游客来黔灵山公园就是为了看这些猕猴，猴儿们是公园放养的，一点不怕人，人反而要让它们三分，有照相机的手里要拿好，有食物的要特别藏好，因为猴子会来抢。
下山遇岔路，先走黔灵湖方向，黔灵湖边有小吃店，可以补充一点能量。黔灵湖是个人工水库，湖畔有一座革命烈士纪念碑。湖中可以划船，参考价15元/小时。到了夏季，荷花盛开，景色很美。
地址：贵阳云岩区枣山路117号黔灵山公园东门侧
门票：5 元/人（1 米以下儿童可免费入园，但须由成人陪同，60岁及以上老人免门票）
开放时间：6:30-22:00

花溪湿地公园

花溪公园很大，风景还是很美的，有溪流、小瀑、小桥，若是在晴天去，风景甚好，公园内最有意思的莫过于百步桥了，那些位于湖上的石板，每一步都得小心翼翼，有些地方旁边还是小瀑布，走着真是得注意每一个下一步，还有放鸽桥，曾在此放飞鸽子以纪念各国之间的友谊。
花溪公园位于贵阳南郊的花溪区，是贵阳历史悠久的自然景观公园。花溪公园融山水田园和民俗风情于一体，这里景色四季皆美，尤以春季群花与秋季梧桐夹道的黄金大道最迷人，而蜿蜒于花溪河上的百步桥也是一道美景。
花溪公园落成于1940年，1949年正式命名为花溪公园，园内主要以真山真水和田园风光为主，有芙蓉洲、百步桥、坝上桥、黄金大道、麟山、龟山、松柏园、碧桃园、牡丹园、竹莲池、憩园、戴安澜将军衣冠墓等景点。公园内有花溪河贯穿，而景点多错落于花溪河两岸。
花溪公园是一个山水环绕的地方，春季来此可以看到这里处处绽放着各种花卉，夏季则是非常适合避暑消夏的地方，秋季行走在铺满梧桐树叶的黄金大道上，其景可媲美童话世界。不过公园里最美的莫过于碧绿的河水和架于水上的各种小桥，其中最有意思的是盘曲在水上的一个个石蹬——百步桥。
来花溪公园可以随意地散步，也可以在湖中划船，或登上小山丘远眺。公园里也有提供自行车租借服务，不过由于公园里小桥很多，骑行的话要经常将车抬上抬下，多为不便。
花溪公园人文景观亦极丰富，既有著名作家巴金和萧珊1945年举行婚礼的东舍，还有周恩来总理曾下榻过的西舍，抗日名将戴安澜将军墓也位于此地。
地址：花溪区花溪大道南段3108号
门票：花溪公园分时间段收费，门票白天6元/人，清早、夜晚无门票，有自行车可租，双人自行车50元不限时，但建议最好不租，公园内桥多，需要抬上抬下
开放时间：8:30-18:30

红枫湖风景区(国家4A级)

红枫湖是一个人工湖泊，使修建红枫水库时蓄水而形成的人工湖泊，类似于千岛湖吧，湖面非常宽阔，大小岛屿星罗密布，四季都可以游览，景区都是满山红叶倒映在湖中，让人心醉神迷，景区吃的很方便，有贵州各地区的名小吃。
红枫湖位于贵阳市西郊，距省会贵阳28公里，是贵州西线黄金旅游第一站。景区面积200平方公里，是一个融高原湖光山色、岩溶地貌、少数民族风情为一体的国家级风景名胜区、国家AAAA级旅游区。湖边有座红枫岭，岭上及湖周多枫香树。深秋时节，枫叶红似火，红叶碧波，风景优美，故名红枫湖。
红枫湖系长江水系和珠江水系分水岭处的一个人工湖，青溪两岸层叠连绵的万重峰峦，被一池碧水淹没，形成了100多个大小各异的岛屿。泛舟此地，你不禁会想起郭沫若先生当年赞美千岛湖风光的一首诗句：“西子三千个，群山已失高，峰峦成岛屿，平地卷波涛。”
红枫湖里的水产资源也非常丰富，鱼的种类很多，而且肉嫩味美，在当地享有盛名。岛上的民族餐馆比比皆是。以“鱼”为主料的菜馔，就有上百种。游罢湖中美景的游客，若到岛上任何一家民族餐厅里，吃上一顿鱼席，既饱了眼福，又饱了口福，您将深感不虚此行。
尤其值得一提的是，红枫湖民族村。民族村由侗、苗、布依3个村寨组成，3个村寨都建在岛上，隔水相望，相互呼应。民族村不仅荟萃了大西南16个民族的原始风俗表演，而且还从全国各地引进杂技马戏、珍奇猛兽驯化表演等等。每周还举办一次“土风狂欢夜”，各民族的歌舞在这里各领风骚，令游人大开眼界。
