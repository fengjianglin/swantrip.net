西行散记：阳春四月，到若尔盖看一场雪吧

游记 人文游 摄影 自驾游 雪景 2018/12/12
川 西北 算是一场意外的旅行。
因为工作原因，毕业旅行的大 西北 路线泡汤了，那段日子刚忙完毕业设计，想着放松一下，给那段时间艰苦备战的日子阶段性的画个句号。于是约了几个好友，抓住青春的尾巴，去了一趟 川西 。
三天时间很短，但却感觉过了很久，一路上经历了很多事，新鲜的、幸运的、难忘的。毕业后的两年多时间里，生活平淡了很多，少了很多波澜壮阔。越是这样，每当回忆起之前那些说走就走的旅行，都觉得弥足珍贵。
L说，生活总是需要一些有深度的时间和经历，在以后回忆起来的时候才不至于那么平淡 。
想起心理学里的峰值尾值理论。说一个经历在事后给你带来的感受并不取决于整个事情持续性的平均体验，而是它给你带来的体验的高峰和收尾的平均值。
这样看来，这次的 川西 之行无疑是近些年生活里的一个“峰值”了，每次回望过去的时候，我都能看见它，那么璀璨的闪耀在记忆的长河里。
然后回到现实，告诉自己，工作生活再忙，也不要忘了远方。

红原：拥抱四月一场雪的温柔

集合那天早上天气很好，我们不急不忙的出发，
阳光透过车窗打到脸上的时候，有种特别熟悉的味道。
平原地区的四月已经是初夏，反而到 红原 一路的气候变化让人觉得仿佛经历了一个四季。
有 汶川 火辣辣的太阳，有 理县 把帽子都吹飞的大风，还有从刷经寺到 红原 县城路上遇到的沙尘暴。
第一天到 红原 已经是傍晚。
相比其他 川西 热门的城市，记忆中的 红原 是一个非常安详的小镇。
有安静的街道、有牦牛、有可爱的藏民家的小孩、有晚上一场突如其来的雪，以及第二天早上耀人的阳光。
▲ 公路边的羊群
从刷经寺到 红原 县城路上遇到的成群的牦牛，在车上记录的这个画面。可惜是4月份，草都是黄的，等到5/6月份春乱花开的时候，这大概就是一个“风吹草低见牛羊”的动人画面吧。
▲ 浮云牧场
经 汶川 、 理县 到 红原 ，海拔一直在升高，一路上记不清翻过来几座雪山。几个地方的景色的大相径庭。
路上遇到牧民家。其实很想体验下居住在里面的感觉哈哈
红原吉祥苑藏家客栈
▲ 藏民家的小孩
第一天到 红原 已接近傍晚，我们就近找了一家藏式旅馆入住。
旅馆里老板的两个孩子，非常可爱，姐姐很爱笑，感觉相貌并没有很明显的藏人特色。可能是生活在这边见得生人比较少的缘故，姐弟俩都显得有些羞涩。
（姐姐很爱美啊，而且镜头感十足，大家玩耍的时候一直不停让我给他拍照 哈哈）
小孩那种干净、无忧无虑的眼神真是很治愈。
▲ 姐弟俩在一起做游戏
▲ 一瞥
上楼的时候看见他自己在端水洗脚，看见我拿着相机，好奇又略带羞涩的望着我。当地的孩子很少看到那种被放纵的粘人的依赖，似乎骨子里都有一种在藏式粗狂家庭教育下培养起来的独立。
红原吉祥苑藏家客栈
简单的整顿好，吃了饭休息了一会后，L说到街上去转转，趁着天还有点灰灰亮。
红原 县城夜晚的街头基本上见不着人，店铺也早早地就关了门，
我们一行四人沿着河跟着有路灯的地方前走。
还没走到一半的地方，突然就飘起雪来了，
意外又惊喜，内地长大的孩子，从来都没见过几次雪，
更没想到还能在4月份看到雪。
几个二十几岁的人又蹦又跳，
高兴的像个小孩。
借助路边一个自动取款机窗口的微光，拍下的这张照片。
由于光太暗，原片简直产不忍赌，但就是特别喜欢这一张。
下了雪之后温度很快就降了下来了，雪在身上化了后感到阵阵寒意，最后冷的按快门都麻木了，于是不得不往回走。
河的对面，大概是这个小县城唯一夜晚还灯火通明的地方了。
不知道为什么，湖面上倒影泛起的红光，现在再看，竟有一种 巴黎 塞纳河的错觉。

▲ 被阳光惊醒的早晨
早上醒来的时候屋子里已经铺满了阳光。
梦里隐约感觉到雪下了一整夜，果然第二天早上推开窗便是一片白雪皑皑的城市，以及不远处在阳光照射下闪着白光的山峰。甚是可爱。
第二天推开窗看到的景象，
某人扒着窗“哇哇哇”的叫了一早上，
才发觉是不是应该去雪里走走。

路途：青春是条不回头的路

从 红原 出来之后，很长一段距离我们都是在徒步，
因为一路的风景太美了，害怕一旦坐车它们很快就会逝去。
那种对于美，渴望拥有，又害怕失去的感觉，其实像极了当时的自己。
干净辽阔的环境，会不自觉地让人放慢脚步去感受，所以这段路我们一路上都很少说话，沉默的走着,好像不思考点什么就对不起这段美丽又安静的路程。
想起之前看到的一句话说，二十几岁时，总以为二十几岁的岁月会一直持续下去。
这大概是我们都容易犯的错误吧。
小时候觉得长大很远很远，没想到一下成长的烦恼就来了，
大一时觉得大学很长很长，没想到一转眼就毕业了，
工作后觉得结婚很早很早，没想到周围的人都在谈婚论嫁了。
青春步履不停，却总有些来不及。
▲ 背影
周围的视野非常辽阔。
很喜欢的一个画面，总是让我想起《平凡之路》里面的歌词，
“我曾经跨过山和大海，也穿过人山人海....”
▲ 徒步走过的一段路
回想目前所有的旅途中，
最怀念的还是乘上当地人的三轮车。
前方是未知的风景，我们躺在车里的背包上跟当地人聊天，
三轮车轰轰作响，抬头就是一望无际的蓝天白 云和 耀人的阳光。
那种幸福大概是二十几岁是所特有，金钱也换不会来的。
▲ 我们的合照

瓦切塔林：与神秘的一场邂逅

因为一路并没有做详细的攻略，基本是路上看到那个喜欢的地方我们就停下来玩。
所以 瓦切 塔林 算是路上一个意外的惊喜。
瓦切 塔林 在藏语意为“大帐篷”，这是一处具有独特藏文化特色的人文景观，为了纪念第十世班禅大师颂经祈福之地 瓦切 塔林 ，当地藏民在微薄的收入中捐献了80%，在十世班禅大师讲经的宝座上建立了 塔林 和周围百十座大小不一的白塔，在 塔林 周围虔诚地挂起连绵的经幡，甚为壮观。（引自百度）
因为我们到的时候刚好接近正午，此时在这里诵经的人并不多。
瓦切塔林

印象中的 黄河 九曲第一湾应该是日落、草原，

街上的行人

干货&攻略区

因为我们一路并没有做太多细致的攻略，只是把大致的线路规划好，然后沿路看到什么好玩就玩什么。所以要说出行攻略，其实并没有太多特别细致的哈哈，简单根据经验说下了~
// 摄影器材 //
简单的佳能入门600D+18-135镜头、手机
// 出行提示 //
1、穿衣：四月 川西 依然很冷，温度在0-20度之间，出太阳就很暖和，下雨下雪就很冷，所以如果这个时间出行，厚衣服的必备的，同时也需要准备一些薄外套。
2、住宿：我们去到的几个地方包括 理县 、 红原 、 黄河 九曲、诺尔盖旅游相对都比较成熟，这段时间并非旺季，临时去找一般都有空位。不放心可以提前预定好。另外，如果时间充足，强烈建议在 黄河九曲附近住上几晚，非常值得，特别是想我们一样第一天去就遇到天气不好的情况。
3、饮食：上诉几个主要中转地方饮食都没有问题，可以尝尝当地特色的菜品。
4、交通：因为公共交通并不多，这边自驾是最方便的，关键是可以沿途边走边玩。
5、出行季节：按理说4月份并非走这条线路的最佳时节，通常6-7月份这边草绿了，花也开了会比四月份的景色漂亮很多，如果允许可以选择这个时间去。当然，每个季节都有其看点，关键要看自己时间以及想体验什么。
6、拍照：出行的一个共识，如果要拍摄当地人正面，尽可能的先征得对方同意。
// 一点乱七八糟的攻略 //
瓦切 塔林 是免费的
黄河 九曲第一湾的门票是70元，加上电梯票是105元。电梯可以直接上到最高处，在高原地区可节省不少体力。售票点离景区还有大概十几分钟车程，遇上节假日人多速度还会减慢。所以，要看日落的一定要算准时间，买对票啊！（不清楚具体日落时间的建议问问当地人）
