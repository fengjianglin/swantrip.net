行程规划

D1 重庆高铁前往成都，入住如家商旅酒店，前往牛灯影吃火锅，下午前往成都博物馆参观，晚上前往斗鸡饭场伙吃完饭，休息。
D2 睡到自然醒，中午前往烹火吃午饭，下午前往文殊院参观，晚上辟谷。
 

成都住宿

在外旅行，总是很纠结住宿，挑选一家心仪的酒店其实并不容易，这就像苏永康的一首歌《爱一个人好难》，或许一见钟情，再见无情，又或许日久见人心，看似不起眼的又总是可以爱的深刻。
成都
这次来成都选择住在了人民公园对面的一家如家商旅（人民公园地铁站宽窄巷子店），这是首旅如家旗下的中高端品牌，而且酒店出来就是人民公园地铁站出口，过个马路就是人民公园，步行10分钟即可到达宽窄巷子，骑行10分钟可以抵达春熙路、太古里，住在这样的地方，去哪里都方便。
成都
成都
酒店大门就在少成路上临街，非常好找，从成都东站打出租车到这里不过二十多块。走进酒店大堂，一阵香薰扑鼻，整个人好像立马轻松起来，前台的小姐姐们很热情的帮我们升级了豪华房。来之前就看到网上的评论说这里都是智能房，好奇之间已经打开了房门，不用插房卡，门一开电视自动启动，并且播报欢迎词，放在桌子上的小音箱也自动打开，悠缓的音乐响起，卫生间的墙面镜上安置了去雾和镜灯开关，床单和毛巾洁白干净，看着就心情极好，整个人栽进床里，好舒服啊。

成都美食

说到成都，品味当地美食必然是来到这里的必修课了，如果说来成都不吃吃吃，可能就跟没有来过成都一样，成都就是一个美食的天堂，不是我一个人这样说。以往来到成都总是火锅火锅火锅，好像有吃不够的火锅，但这次除了要吃火锅，还想感受一下不一样的成都风味，于是，这两天我来到了这里……
 
牛灯影私房老火锅（泡桐树街店）
刚刚在行程里已经简单介绍了我这两天在成都的吃食，先来说说这家牛灯影私房老火锅，讲真，这家火锅店就在宽窄巷子附近，我从酒店步行过来不过20分钟的路程，火锅店在泡桐树街临街，方便好找，来到这里看到门店并不是很大，正合我心意，我一直很喜欢在成都的路边寻找美食，小而精致的店自然是选择之一了。
成都
成都
牛灯影的内部装修充满了浓浓的老成都情怀，门口的老式自行车就可以看出，这里不仅仅只是一家火锅店，更重要的是传递一种怀旧情绪，走进餐厅内部，狭窄的过道一边堆放着各种老式行李箱、儿童三轮车等怀旧物件，我们坐的餐桌一边的墙面上，摆放着小型缝纫机、打字机、收音机等，尤其吸引我眼球的，便是那一盘一盘的流行歌曲卡带，都是80-90年代那些巨星的专辑，让我不禁回到了青春年少那时。里间两个房间桌椅干净整齐，墙上挂着一些老旧照片，加上火锅店的招牌，讲述牛灯影和老成都自己的故事。
成都
成都
听说以前这家店是专门做冒菜的，看来生意不错扩大了规模。讲到锅底，一般的火锅店都是牛油锅底，虽然这家也没有鸳鸯锅，但是锅底里大有文章，端上来的红油锅底，一层厚厚的红辣椒格外有人，当煮开了汤以后，锅底自带的牛肉就可以吃了，入口即化，绵软又有力，嘬一口满嘴麻辣。锅底料味道足，无论涮什么菜都会入味，除了牛肉锅底，这里的鲜毛肚、秘制牛肉、黄喉、脑花、贡丸都是新鲜必点的火锅食材，麻辣入味，怎一个爽字了得！

斗鸡饭场伙（同心路店）
说到斗鸡，顾名思义，自然是与鸡肉有关。都说这是成都餐饮界的一股清流，但我觉得这是一个将鸡肉和川菜做出了新鲜感的品牌，所谓新鲜感，就是创意啦，或者我是一个颜控，就像去超市挑选水果一样，长得好看的，或许就是首选的。
成都
这家斗鸡并不在新城市广场里面，而是在商场旁边的一条同心路上，门头不大，里面却别有洞天，整个店面装修风格古色古香，红彤彤的，很是喜庆，一楼的前台一边有一个小房间，果然店如其招牌菜——辣子鸡一样，火红火辣。

成都

烹火（小通巷店）
听说烹火燃烧了成都川菜和西餐互通融合的混搭美食一片天，川菜和西餐如何融合成当下人爱吃的口味呢？带着好奇心，我就将我的烹火处女吃给了宽窄巷子后面小通巷的烹火店。小通巷是条小巷子，骑行到这里，先看见的竟是门头烹火的披萨外卖站招牌，中午时分外面排队的人已经密集在店门口，可见这里的火爆。

成都博物馆

探访博物馆，是了解一个地方最好的旅行方式了，所以每到一个地方，我都会选择去博物馆泡上一个下午，这次也不例外。成都博物馆是成都时规模最大的综合性博物馆，位于市中心天府广场一侧，站在天府广场向西一眼望去，博物馆、四川图书馆、美术馆和科技馆抱团伫立在马路两侧，甚是壮观。成都博物馆是不需要购买门票的，只需要带好身份证刷证进入参观就行了，但是安检比较严格，如果自己的包包里有水、打火机等物品，可以在入馆之前存放在存包处，带着身份证进去参观就可以了，非常方便。
成都
成都
成都
博物馆一共有五层楼可以参观，主要讲述成都的历史文化以及中国皮影木偶文化，首层为大厅、放映厅、一号临展厅，地下一层为二、三号临展厅，地上二层至三层为成都历史文化陈列古代篇，四层为近世篇和民俗篇，五层为中国皮影木偶展，但因为临时关闭我们没有看到皮影木偶展。

成都的历史文化悠久丰富，《花重锦官城---成都历史文化陈列》几经调整后终于出炉剑客，展品时间跨度设定在远古到民国时期这个范围内，专题一是"九天开出一成都---远古家园篇"，铺叙成都独特的自然地理风貌;专题二是"不尽长江滚滚来--古代历史篇"，通过成都地区出土文物的精美陈列，勾勒从远古至清代城市文明进程的大线索;专题三是"玉垒浮云变古今---近世风云篇"，以历史档案为主，重笔渲染成都波澜壮阔的近现代历史;专题四为"锦城楼下二江流---城市变迁篇"，借助多媒体展示从古到今成都城市格局的变迁;专题五为"锦城丝馆日纷纷--民俗生活篇"，从市民耳熟能详的成都民俗入手，展现老成都的生活。通过这五个专题集中讲述成都的前世今生和这座城市成长的来龙去脉，成都历史进程中的重大事件和人物。

文殊院

以往来成都，多是集中在宽窄巷子、锦里等热门景点玩耍，文殊院却从来没有去过，所以这次来到成都，专门就设计了这样的历史文化路线，除了去成都博物馆，代表成都历史文化象征的文殊院也不可错过。
成都
文殊院也在成都的一环路以内，骑行就可以轻松到达，交通非常方便，而且这里是汉族地区佛教全国重点寺院，四川省重点文物保护单位，四川省、成都市佛教协会驻地，川西著名的佛教寺院，自然会吸引不少香客、游客前来。
成都
成都
说起文殊院，还是相当有年头的了，史料记载这座寺庙始建于隋朝，古刹佛家，经历过八百余年的风霜雨雪屹立至今，中间也经过多次损毁与修复，才是我们如今看到的模样。走进古刹大门，狭窄的道路两侧，寺院的红色围墙延伸而去，随之而去。

缓缓进入这座历史名刹，或许就是这份寺院固有的宁静，不禁敬仰之心也随之缓缓升起，跨入寺院正门，寺院坐北朝南，进山门往里走依次是天王殿、三大土殿、大雄宝殿、说法堂、藏经楼。与东西两侧的钟楼、斋堂、廊房等建筑浑然一体，庄严肃穆，古朴宽敞，两旁配以禅、观、客、斋、戒和念佛堂、职事房，各殿堂之间有长廊密柱相连结，形成一个封闭的四合院，所有房舍为木石结构，柱础石琢磨精细；柱上斗拱雕饰玲珑。
