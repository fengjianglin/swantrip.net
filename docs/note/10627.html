那些泡在天水的日日夜夜

游记 人文游 摄影 温泉 美食 自然奇观 2018/11/13
天水因“天河注水”得名，已有2700年的建城史。这天河之水有何神奇，这次甘肃天水之旅，我们先来到清水县，一起去泡泡——
漫步在清水县轩辕湖欣赏蓝天白云倒影如画，心情格外的畅快。​
和你一起等候日落后温暖的晚霞和魅力的夜色，要的就是这个等候和欣赏的过程。不需要刻意安排什么事，只需要静静的感受云涌、风起、水漾、心荡......风吹在脸上，才感觉时间的流逝。深秋的夜已是冷风瑟瑟，该去泡泡天河之水温暖你的身心。
【休闲温泉篇】
驱车来到清水县城东8公里的永清镇，这里海拔约1420米，地热资源丰富，是典型的温泉度假地。
才来到水边，你已陶醉包裹在温暖的水光里，不忍打扰，只是静静的欣赏、记录~希望带给你长久的回味。
快到半小时了，不得不把你叫起来喝杯水休息下。凝望+渴望，这样的场景无论如何阅读都是诗！
清水出芙蓉，你和温泉水分子相遇在一起，去演绎只属于你的水之韵律。
你会心舒心醉心的微笑表达着你的愿望——让幸福流动。
尽情的享受大自然的赐予，快乐的冲温泉、泡温泉，再来个温泉按摩浴吧。我知道你想问：是神马天河之水如此的神奇，让美颜欢笑、容光焕发？！
听我给你讲讲这个特“牛”的地方——清水温泉度假村。
清水温泉度假村建筑面积2.5万平方米，日可接待1000余人。这里森林叠翠、山峦俊秀、环境幽雅。除了温泉，还配有茶楼、网吧、VIP包厢、棋牌室、KTV包厢;度假村还设有游泳馆、健身房、篮球馆、保龄球馆、塑胶网球场、篮球场、羽毛球场、乒乓球室、垂钓中心等。餐饮部设有豪华及普通包间16个，经营川、粤及民族菜系，同时可容纳300人同时用餐。
这是清水温泉度假村的水疗中心，也就是温泉区域，白天的时候去看看最清楚。营业时间：10：00—22:00
清水温泉出口水温高达摄氏54度，含有氟、氢等多种矿物元素，对于人体有活血舒盘的功效，治疗神经官能症、风湿性关节炎、皮肤病等疗效显著。在冬季市非常适合养生休闲的地方
漫步在温泉区，热气腾腾、温泉泉眼翻涌，带给人更多的是温暖和惬意。
清水温泉还配有足衣浴、漩涡浴、光波浴、涌泉浴、芬兰浴......清水温泉的温泉水为高热矿泉水，水温54℃，富含人体必需的多种微量元素，尤其被誉为“生命之花”的锌含量居全国之冠,具有极高的医疗保健功能和矿泉水开发价值，商机无限。

秦家塬（图文/情流感）
你从来就有一双看世界的明眸和智慧的心灵，你知道天边的世界会更精彩。
秦家塬陇上风光（图文/情流感）
你徘徊在秦家塬陇上风光的路口等待温暖的手带你一起走，我只有手扶拖拉机来迎接酷酷美丽又善良的你。
但在赵川村的山顶却有足球场那么大的宁陵堡可以守候你一生的安全和快乐。去看看你满意不？
在赵川村宁陵有很多村名多含“寨”、“堡”，如魏寨、于庄寨、华堡、马桥寨。为什么有这么多的村称“寨”或“堡”呢?说来就话长了，来边看边给你介绍——
触摸历史的城墙（图文/情流感）
你现在触摸的就是我们宁陵马桥寨的古城墙，使用和万里长城有得一拼的土质磊成的城堡，带你穿越历史温情对话——清宣统三年《宁陵县志·寨堡》载：“(宁陵)旧无寨堡，承平志也。自咸丰三年，粤匪煽乱，捻匪蜂起，乡绅起义，遵谕团练。八年秋，创建华堡寨，兵粮储富，可持无恐。嗣是各团仿筑，保障居民，裨益非鲜，是宜志。”宁陵地方乡绅倡议办团练，筑寨堡，练武屯粮，以抵御捻军。
“平安无事喽!”这是电影《地道战》中的经典台词，表现的是民众夜间巡逻看家护院的情形。自清咸丰以来，宁陵县乡纷纷筹建寨堡。在城堡里面高声呐喊是有回音的，我知道你陶醉其间心灵像鸟儿般飞翔。
在我们宁陵县历史上城堡可是真多，据记载：咸丰八年(1858年)，全县共修筑寨堡47处，官道北28处，官道南19处，分别为：许庄寨、翟楼寨、赵村集寨、大孔集寨、桃堌集寨、徐黑寨、石桥集寨、孙迁寨、二堡寨、万集寨、后赵庄寨、柳河集寨、桃园关寨、杨楼寨、刘华桥寨、吴良吉寨、万庄寨、小孔集寨、于庄寨、坡里村寨、金厢寺寨、逻岗集寨、郑楼寨、胡大庄寨、赵尔庄寨、后吕河寨、张坡楼寨、阳驿铺寨、张瑞寒寨、平乐寨、谢集寨、郭屯寨、刘八楼寨、张弓集寨、毛楼寨、己吾城寨、黄岗集寨、大吕集寨、瓦屋刘寨、华堡寨、马桥寨、夏寨、史家集寨、沟厢集寨、魏寨、八里曹寨、吴祖师庙寨。这些寨堡是乡民防匪自卫的真实依据，也是目前许多村庄称“寨”、称“堡”的原因。
登上城堡的墙沿，整个张家川宁陵县尽收眼底，我能看到你会心的微笑和满意。据当地老人介绍：1947年国民党张岚峰部曾在马桥寨住俩月，重修寨墙，建防御工事，寨墙上垛的有夹道，双面都能打枪。当时三道寨墙，东西寨门。寨墙外的“海子”有五米深、两丈宽。解放后，寨墙被拆毁。
在兵荒马乱的年代，寨堡对社会的稳定和人民的生命财产安全，曾经起到十分重大的作用，到了和平年代，因失去原有的防御兵患土匪功能而逐渐被平毁废弃。现在好了，国家强大，人民平安富裕，已经不用再住在城堡里了，家家都在山下建有小楼房。
我们现在在城墙上站得高，随我看下面的建筑，那就是张家川的宣化冈。你可能会问？宣化冈是用来做什么的呢？为啥要建宣化冈。我们一起去看个究竟——
修建拱北是伊斯兰教苏非学派的特征之一。何谓“宣化”,即“宣扬、教化”之意。拱北是阿拉伯文的音译,为中国伊斯兰教哲赫忍耶门宦在其教主的葬地和道堂建造的圆顶建筑。中国伊斯兰教苏非学派修建拱北有比较悠久的历史。宣化冈在张家川回族自治县县城北约7.5公里的的查湾村，称为宣化冈拱北。
宣化冈自清光绪十五年（1889）开始兴建，历时数十年，经三代人建造，使宣化冈拱北成为具有一定规模的建筑群。总占地总面积19.94亩，共有院落24处，房屋300多间。拱北周围由苍松古柏等树木环绕，郁郁葱葱形成绿色天然屏障，中间矗立着中国宫殿式的宏伟建筑物，金碧辉煌，甚为壮观。故于右仁曾诗赞云：“金碧参差宣化冈”。
宣化冈拱北建筑，依山就势，错落有致，雄伟可观，具有中国宫殿式古典建筑特征，为甘肃全省名胜古迹之一。
距县城北约2公里多的赵川村畔，原有一雄伟的碑亭，内有石碑，谓“神道碑”(现存宣化冈)，此碑是陕甘两省官绅商学人士为元章、元超二人的功绩而树的。朝村北而上就可看见高约10米，宽约20米，底部用石条、上部用青砖砌成的一座门楼，中间开有一上圆下方，高约3米，宽约4米的洞门，其洞深约8米。门上悬一匾额，上书“万物静观”四字，门楼上额镌刻前清翰林哈锐亲书的“宣化冈”三个白底黑色大字。上面建有三间楼阁，飞檐翘角，起脊瓦寿，前后是八盘棂隔门窗，中间门顶悬一木匾，上书红底金色“观光楼”三字，四周有砖砌围墙，向南俯视，可看到张家川城区全景。
宣化冈碑廊（图文/情流感）
四座八卦拱北前面正中间立一石碑，是各省教下暨子侄等为元章、元超敬立的墓碑，碑正面中书“马大、三上人光烈、光?夫子之墓”。上首书“民国二十二年九月浣谷旦”，下首书“各省教下暨子侄等同敬立”。碑后面是“前清翰林庶吉士门生哈锐谨撰文，陆军上校愚侄何兆龙敬书”。墓碑前面埋有元章二弟元坤、堂弟元芳及元章三子仁武，元超的二位夫人及其子锡武、毅武、辉武、霆武、桓武、殿武的墓庐。坟园内栽有松柏、龙柏(紫丁香)树木，春季松柏青翠，龙柏花开，香气袭人。拱北之上有一大洼地，周围筑有砖墙，洼上栽有松柏、白杨、山杏及沙棘等乔灌木，每临春天，山花烂漫，树木成荫，鸟语花香，景色迷人。
入楼门便是一条石台阶路，路左首有一走廊直通“观光楼”，走廊西边有一花园，名叫“别有春”，园内栽有倒柳、竹子、葡萄，种有奇花异草，还建有三个雅致的箍窑，这是主人夏季静修避暑之地。路右首是碾麦的场院、沐浴室院。走上台阶路，是一条从东到西可通车马的巷道，巷道中设有上马石、拴马柱。巷道西有一总门，叫西门，出西门有一条大路直通北山公路。巷道东有一小门，叫东门，出东门有几条小路通北山堡子梁各地。巷道中间修有两间过道楼，登楼可观全冈的景色。巷道南边有七间大楼，是宣化冈私塾楼。靠楼房东边设宰牲院和宽大的蔬菜园子。
宣化冈内的拱北、经厅、礼拜殿等24处建筑群、300多间房屋，都是特请外地能工巧匠设计建造的。其建筑依自然地形布局，气势宏伟，门庭错综，结构精巧，曲径通幽，别具一格。厅殿和室内铺有名贵的和田地毯，摆设各式的古铜香炉、珠砂花瓶以及精制家具和古玩等。门上大多悬挂名人题赠的匾额。住室和客厅等房内挂有历代名人的字画和元章、元超二人亲作的治教省身格言及对联。书房内藏有珍贵的宋版史书和伊斯兰教的大部经典。
宣化冈拱北是哲赫忍耶学派的中心，也是广大哲赫忍耶学派教众心中的圣地；是宗教教理的传播中心，也是伊斯兰文化与汉文化的融合、交流的中心；是人才汇聚之地，也是观光游览的胜地；是近代回族与伊斯兰教学者、知识分子学习、交流的课堂，也是回族精英实现抱负和理想的舞台。宣化冈在哲赫忍耶学派的发展历史、在中国伊斯兰教的发展历史、甚至在回族的发展历史上都占有非常重要的地位。
快日落了，我们一起在马桥寨寨墙遗迹看夕阳吧。今日夕阳特别温暖而美丽。
在这历史感厚重的宁陵堡，我们应该学到了很多，收获了很多。是的，陇上江南~天水总会带给你无限的惊喜和温暖。
【唯美神话篇】
大晴天不是天天有，有时朦胧的山水也别有意境。当然不要下雨这是基本前提了。知道你一直喜欢登山，于是在天水的清水县带你去有“女娲补天”传说的花石崖登高望远。
这里：峰峦叠嶂、群山连绵、陡峭险峻，这里的山石色彩斑斓而被称为“花石崖”；这里：清泉汩汩，植物种类繁多，四季万紫千红而别称为“万紫山”。
这里有弯弯曲曲的健身观景栈道，这里有《西游记》中万子山得名的花石崖道观，这里有连绵不断层次分明姹紫嫣红的群山彩林......“北依关山望秦岭，花石斑驳殿堂隐。缘何飞洒及时雨，白云飘渺闻晨磐。”这首诗便是花石崖的真实写照。
登山途中，你坐下来问我： 女娲为啥要用这里的石头补天？难道石头防雨？！这问题可一点难到了我，花石崖是道教和佛教并存的寺观胜地，宗教建筑历史悠久，远在唐代，这里就建有庙宇，花石崖的所有建筑均依山势而建，错落有致、张弛自如、风格各异。瞧，下图密林中掩映的就是通仙阁花石崖的道观，我们还是请里面三清殿的道长为我们来解答吧。
这座木质结构的牌坊就是通仙阁，上书花石崖。是著名书法家赵治华先生所题写。入门回廊，就走在了通仙桥上。也代表了古人对成仙道的向往。
一直走来到静石台，到崖边是舍身崖。相传观音点化了无量祖师后，先跳了下去，无量祖师跟着跳了下去，最终升仙。现在这里建有万紫山胜迹亭，里面还建有重修花石崖记的石碑。​
祖师殿这里供奉的是无量祖师，就是我们经常说的周公。
三清殿的道长热情的带我们来到了一块平地而起的巨石之上，巨石高约30余米，范道长说这就是传说中女娲补天留下来的五彩神石。现在上面刻上了“补天石”三大字，是毛惠民先生所题写。显得简朴俊逸、气度恢宏。
我们也在补天石下留影，沾沾补天石的魅力和灵气。
然后道长带我们来到建在补天石顶上的磨针殿，磨针殿通过悬空铁梯上去。有点考胆量哟！
范道长告诉我们：磨针殿里有清代的塑像两尊，为“观音点化，祖师修道铁棒磨针”之形象。两侧有“太子游四门““祖师成道”的壁画。一位老婆婆坐地手拿铁棒在巨石上磨，一位少年旁立在问话，相传为“铁杵磨针”的传说。
范道长指着殿檐上说，这里有块匾额，我们打着电筒发现上面写着四个字：非人间世。匾额上的小字可以看出，清雍正四年（公元1726年），当地群众捐资重建了庙宇。
登上高处俯瞰来路，曲曲折折，山间万紫千红，非常美丽。
花石崖有一建筑群叫雷音崖，崖上有悬空云梯，钻过石洞，便进入观音佛洞。这里有寺、窟三座，都是悬空云梯与栈道相通，险绝奇伟，犹如天工造化，让人惊叹叫绝。我们在山间呐喊，回音涤荡。
在这里的佛龛深处的巨石中，有石缝里树根长成人根造型的奇树！
登高望远，我们来到了花石崖的佛陀寺。内有大雄宝殿，供奉着阿弥陀佛和文殊、普贤菩萨，左右偏殿分别为观音菩萨和地藏菩萨。
大雄宝殿的顶部雕刻非常精致美观，龙鳞都精细展现，堪称保护完好的精品之作。​
这里绚丽的自然生态，众多的名胜古迹和峡谷石崖，把花石崖装扮得如丝如画。
这就是美丽而传奇的花石崖， 女娲补天传说的原型”补天石”就在这里，这里的石头不一定能防雨，但是我们补天石去”补天”，祈福一路平安，大晴天！

【馋嘴美食篇】
甘肃天水张家川是陇东南唯一的少数民族自治县，也是中国回族比例最高的自治县。回族饮食文化博大精深，源远流长。早在明清时期，回族饮食文化就自成一派，形成了具有回族特有属性的饮食文化体系。
这次到张家川当地人告诉我们，有顿饭你们一定要去吃，叫“十三花”。 那到张家川为什么一定要吃这顿饭？先泡一杯正宗的三泡台，慢慢来品“十三花”。
三泡台（图文/情流感）
茶是回族迎宾待客的佳品，回族人沏茶比较讲究，其中以“三泡台”最为讲究。由于“十三花”的制作和上菜过程比较复杂，一般耗时较长，所以吃这顿饭都会先给客人上“三泡台” 茶和“果碟”。今天的这顿饭就从回民的三泡台聊起：“三泡台”是将上等茶叶、枸杞、桂圆、荔枝、红枣、菊花、冰塘等放入“三泡台”碗中，用滚烫的“牡丹花”水（沸水）冲泡即可。茶香和果香恰到好处的融合在一起，完美！“三泡台”碗是由盖子、茶碗和底盘配套而成，形象而美观。三泡台茶还有消食化痰、去除油腻的功效。
果碟小吃（图文/情流感）
果碟是由五种、七种或九种不同食品拼合而成的果盘小吃，水果比如葡萄、桔子，小吃就更多，我们品尝了：油饦、油糕、馓子、花花、锅盔等。
油糕（图文/情流感）
在品尝果碟等候的时候，回味斋的马老板非常热情而且口才好，给我们讲解十三花的来历以及特色：“十三花”是回族饮食中最具代表性的一种，“十三花”最早可追溯至明清时期，后被继承和发扬，一直延续至今，“十三花”因由十三种不同的菜组成而得名，之所以采用“十三”是由回族的宗教习惯决定的，回族宗教中历来有“采单不采双”的宗教理念。
油饦（图文/情流感）
张家川“十三花”以其品种多样、精美可口、制作精细、营养丰富、干净卫生、色、香、味俱全、油而不腻、老少皆宜、颇具回族特色而著称，深爱广大回族群众的喜爱，从而成为婚丧嫁娶，招待亲朋好友的必备美味佳肴。

花花（图文/情流感）
马老板还带我们参观了回味斋老师傅制作十三花的部分过程，非常难得。
这就是回味斋的大厨马师傅，曾经是张家川县委食堂的大师傅，从事回族清真美食40余年了。非常的慈祥，和蔼可亲。马师傅说，“十三花”从采购食材到端上餐桌要经过几天的时间，这次我们有幸品尝到了在宗教重大节日和婚嫁才能吃到的回族特色美食“十三花”真是太幸运了。
精选称重新鲜肉品（图文/情流感）
张家川人勤劳朴实，淳厚善良，崇尚礼仪，热情好客，设筵待客历来丰盛。制作“十三花”，从选料到称重配方都一丝不苟，非常讲究。
半成品（图文/情流感）
感觉十三香有点像西南农村的8大碗，但是有十三样精美菜肴，牛羊肉食为主。配好半成品菜品，请马老师傅和“十三花”的雏形合个影。马师傅介绍说：回族筵席名目比较繁多，其中以“九魁”、“十会”、“十三花”最负盛名：“九魁”即是八碗炒菜，中间一碗凉拌，若在“九魁”中再加一碗“八宝米”甜品就称“十全”，在“十全”中再加三碗诸如海参，鸡、鸭、鱼之类的菜便称“十三花”。
高汤调制菜品（图文/情流感）
然后，在我们的注视下，马老师傅把秘制的高汤汁烧开，然后依次加入热菜中，准备上桌。有清炖牛羊肉、牛肉丸子、牛排、红烧牛肉、拌肚丝、鸡腿菇炒肉、西兰花、八宝糯米饭......，好馋啦！
上桌（图文/情流感）
“十三花”用传统的木质方盘端上来啦，马老板亲自端上来的哟！大家都是一阵激动和小兴奋，想赶紧体验一下口感，口水都要流出来了，拿起筷子便开整！的确每一道菜品都是一场味觉的全新体验，就像我们在旅途中看到的美景一样，你不去体验哪知千秋不同的美色与美味。当美食触碰味蕾的一刻恰如美景展现眼前的兴奋，作为吃货+美食侦探的我，手中的筷子是根本停不下来的节奏。
在品尝“十三花”的同时再配以油香、花卷等回族的特色小吃。最后还有面食哟，面食一般是以“出汤面”为主，“出汤面”又称“另汤面”，“出汤面”多用手工擀面，刀切而成，然后，将切得又细又长的面条，煮好捞在“品”(专指捞长饭时用的小瓷盆)内，只加面汤不加调料，盛一“品” 面条摆在桌子中间，另将调配好的饭汤，用小碗盛上，每人一碗，吃饭者自捞面条，吃多吃少由自己，同桌摆上菜碟，既有“四碟”的，也有“八碟”的。用各色菜丝摆成菊花形、梅花形等各种花样，并配以盛有醋、盐、油泼辣子等调味品的碟子。我们实在是像花儿一样美丽又飘香的“十三花”美味吃得太多了，吃不下面食了，点到为止哈！
这就是我们在张家川回味斋的美食体验： 这是一场味蕾的全新触摸，回味无穷。所以到天水张家川一定要来吃这顿饭！当然如果人少，也可以品尝小份，从十三种菜品里选几样进行品尝，回味斋会尽力满足你的需求。 一个温情难忘的夜晚，一顿回味悠长的美食~十三花！
出门在外，我们要停下脚步，善于发现身边的美丽和温暖。我们用心体验在一起温暖如水的时光，你若盛开，天水自来！
