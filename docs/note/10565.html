［首发］你好，重庆，时隔4年，我们又见面了

游记 小众 摄影 登山 精品酒店 美食 2018/11/18

前言

2014年的时候，我还是个学生，代表着我们学校来参加创业之星全国大赛，那时，我见到了好多素未谋面的网友，我们一起比赛，一起吃火锅，一起逛景点，在这里珍藏了许多美好的记忆。没想到4年后我再次回到这个地方。不同的是我又见到了许多网友，而这次走的路线与之前也完全不一样，不单纯是只在主城区随意逛逛，而是跑到了附近的景点转着玩。
在重庆，我们打卡了火翻天的洪崖洞，打卡了《变形金刚4》的取景地天生三桥；打卡了《北京遇上西雅图2》取景地的金佛山，还在那遇见了佛光；打卡了张艺谋导演的感人的《印象武隆》；还去吃了2家网红火锅店，4天4夜，玩得简直不要太过瘾了。

我们的行程

D1 洪崖洞-红九格火锅（洪崖洞店）
D2 黑山谷
D3 金佛山-印象武隆
D4 天生三桥

D1 洪崖洞

初到重庆，迫不及待的就来打卡洪崖洞，这里被人们称为《千与千寻》取景地，在2016年以迅雷不及掩耳的方式走红了，随后到了风靡一时的“抖音时代”又更是助推了一把，这一个白天看着相貌平平，到了晚上却摇身一变，成了另一副景象的地方，一个身披金条的盛世公主的模样。
记得在2014年时候，笑容晚上打车看到洪崖洞店时候，司机也只是轻描淡写的说了一句，这是洪崖洞，一个夜景还不错的地方。没想到几年过去，这儿就变成了网红景点。当灯全部亮起来的时候，吊脚楼的屋檐边上等待都亮了起来，商铺的招牌也亮了起来，洪崖洞句变成你心里期待的那个千与千寻里的样子。
这里火到什么程度呢？洪崖洞里面的商业街人特别多的时候可能都是以龟速前进的，走路只能看到人的后脑勺。而桥的另一边和大桥上都挤满了人，尤其是在周末和节假日，不早点到是一定不会有位置的。好在这次笔者来的时候刚好不是节假日也不是周末，所以人还不是特别多，整座大桥都有很多空位。

红九格老火锅（洪崖洞店）

来重庆怎能不吃火锅呢！在对面看好洪崖洞了刚好可以到洪崖洞内部“爬楼”，其实也就在9楼而已，从地面坐电梯下一层就到了。这是红九格老火锅（洪崖洞店），在红九格门口可以看到洪崖洞音乐长廊，而对面大桥的美景也尽收眼底。
说起这红九格也是有故事的，他们一直打着这样的口号：“我们不卖鸳鸯锅，微辣是我们最大底线”。这家火锅店是本地人公认的辣，不能吃辣的朋友记得要点微微辣～不要以为红九格的口号只是说说而已，店里真的没有清汤锅。一盆红锅端上来，牛油上满的辣椒花椒，看着就非常过瘾。
最霸气的恐怕就是招牌牛杂，一个拥有一只大牛头的盘子端在火锅前，面前立刻就满了；鸡胗是最养眼的，里面会冒出烟气，很漂亮哇，请不要介意，笑容对吃的美观还是有一定讲究的；盛酒的杯子是最有意思的，里面还有一条锦鲤，倒酒时候仿佛还能游动；火锅呢，就是牛油火锅，还没融化的时候你可以看到牛油块上印着它们的口号。至于味道，那是倍儿爽，越吃越辣，越辣越想吃，直到停不下来。

D2黑山谷

说来也是缘分，笑容近段时间接触了不少原始森林，说好之后只泡温泉不爬山，结果还是爬山去了。这次爬的这山叫黑山谷，一个位于綦江区黑山谷镇里的一个5A级景区。如今景区最火的其实是奥陶纪，但是今儿咱们不说奥陶纪，我们来瞧瞧黑山谷，一个汇聚了渝黔喀斯特地貌风光精华的，每立方厘米约10万个负氧离子的，地处贵州重庆交界线的这个神奇的地方。
黑山谷有个最有意思的几处地方，一处是亭子上，走过亭子，便从重庆跨入了贵州境内；另一处是渝黔大裂谷，当地人也把这里称为大斜槽，由于7000年前的山谷断裂与日星月夜的流水侵蚀形成了如此神奇的地貌，更有意思的是在这短短的1000米的峡谷中，有不少地方可以让人两手都能摸到两侧的岩壁。岩壁一侧写着重庆，一册写着贵州，在大裂谷的中间就能办到一手摸重庆，一手摸贵州，非常的有意思。
不过最有意思和最好玩的当属景区的水上“吊桥”。这“吊桥”并不是想象中的悬崖峭壁上的悬空吊桥，而是在水面之上，悬挂在两侧岩壁的路，在景区里面有好几段这样的路。走在这样的桥上，无需特意去逛灯儿晃桥也能感受到灯儿晃，尤其是人多的时候，有的路段更是摇摇晃晃，趣味十足。
走累了可以找沿途的一些道路进行休息，看看身后滴水的岩壁或者面前的潺潺流水。在黑山谷，这里最不缺的就是喀斯特地貌的山以及大小瀑布。在这全长13公里的景区里，从电瓶车到步行到最后的缆车，沿途都能看到瀑布，较为有名的是飞鱼瀑，这里似乎是最大的瀑布之一，在瀑布对面的一张木椅子上还给游客标记了最佳拍照点。
整个景区的游览时间差不多3-4小时，需步行6公里，可乘坐6公里的电瓶车和1公里的索道。导游说，这里夏天的温度比市外的温度低上十度，尤其适合夏天避暑。不是夏天的时候也可以过来呼吸新鲜空气，放空大脑，与家人或朋友，寄情山水，也不为是一种享受。

D3金佛山

说来也神奇，那会金佛山上也没形成云海，只是薄薄的云雾缭绕奔腾，佛光就出现在草坪上，由浓烈变淡，又逐渐变浓烈，我们那一队的人都看到了这个景象，很是幸运。这完全是意外的景象，也或许是对我们的补偿？山脚下云雾太浓，把最为壮观的天然睡佛给遮挡了，也把山顶的金龟朝阳给遮挡了，金佛山这座有佛性的山也不忍了，所以它用这样的一种方式向我们补偿了，当然，这也可能是笑容一厢情愿的幻想。
但是说起这金佛山，确实很多地方与佛相关，从景区入口的大睡佛到古佛洞里天然的佛，无一不与佛相关。无独有偶，笑容问讲解员个大伙普遍都会问的一个问题，这里为何叫金佛山，走了很久都没有看到金色大佛呀。原来，说的不是人造的金黄的大佛，而是在景区上看到佛像在夕阳和日出时候会泛着金边所以称之为金佛山。
讲解员还介绍到金佛山还有许多杜鹃花，除了常见的红的粉的，还有会变色的杜鹃花，这种花盛开的时候是黄色的，随着逐渐凋落就变成了绿色的。据说曾经有位专家来对这里杜鹃花进行考察，在悬崖上采摘不小心坠落悬崖，所以后来为纪念它也给黄杜鹃花以他的名字进行了命名。当杜鹃花季节来时，整座金佛山就成了一片花海，美不胜收，难怪《北京遇上西雅图2》都要在这取景。
在黄杜鹃面前的绝壁栈道，其实也是金佛山的一大亮点。这里的栈道与悬崖峭壁形成了90度的直角，险峻异常。往栏杆外探头，看不见谷底，这么危险的动作大家还是不要操作的好。突然听见前方传来悦耳的歌声，原来是我们的讲解员唱起了歌，歌声通过弯弯的溶洞传播出来，又在溶洞里不断回响，听得我们如痴如醉。

古佛洞

古佛洞是金佛山里面比较有意思的一个景点，这里有天然形成的一座的大佛（从溶洞上掉下来的），专家调查称这是亚洲最古老洞穴，距今已经570万年了。如若想见到天降大佛，那须先走过“七十二道拐”，讲解员说人生也是如同这个溶洞一般，有许多曲折，假如你摸着这里的岩壁走过去，那么你就可以减少磨难～
于是我们几乎是一边扶着一边摸着岩壁进入到溶洞内部。传言曾经有人要迷失在溶洞内，是洞穴的仙鹤给予指路最终才走了出去。而那仙鹤如今在在溶洞内，有缘人可找到仙鹤。由于后来我们跟着讲解员走，所以也不知道这只白鹤到底在哪里，但是笑容并没迷失在这洞里，毕竟这里现在就只有一个出入口了。
沿着路一直走到头，你会有种突然闯入另一片天地的感觉，原本只能有2人通过的弯弯曲曲，凹凹凸凸的路的眼前突然出现了一大个巨大溶洞和水池，此时恐怕只有一个“哇”来形容内心的震撼。溶洞前方，左边，右边都是佛像，右手边端坐着的正是那尊从天上掉下来的大佛，你抬头看大佛正上方的溶洞,缺的那个形状正是这座大佛的形状。大佛宁静的端坐着目视前方，它的右后方有一个石棺，当然这并不是真正的棺材，而是一块大石头，上面还被贴了一些纸币，据说摸着石棺走，升官又发财～不管信的还是不信，摸摸总不吃亏，我们一行人就摸着石棺走了一圈。
大佛正对面的菩萨背靠着背坐着，周围围着许多和尚，他们身穿“迷彩袈裟”似乎在听佛法似乎又在各自忙活。这些佛像都是现在的人造出来的，并不是“古董”。参观完佛像后你可以走到水池旁边的岩石下去寻找娃娃鱼，池子里有比猫咪还大只的娃娃鱼，他们就躲在岩石下的水底，似乎也在跟那些雕塑和尚一样在参禅打坐～整个行程参观下来都不到一小时，还是结束得比较快的，结束之后我们就到达了山腰的酒店进行用餐了。

印象武隆

印象武隆一共除了开头和结尾，中间共有5幕，从前面的《父亲的船》到后面的《川江号子》，一幕接一幕，每一幕都有不同的惊喜。从《父亲的船》来说，父亲的声音悠扬，回荡在山谷，山上还飞来了一直来重庆不能不提重庆火锅，在第四幕《火锅摇滚》里，重庆人民翘着鼓欢快的跳着，都能感觉到吃重庆火锅的一种欢快之感；第五幕《太阳出来喜洋洋》那熟悉的调子，悠扬的旋律响起，山上同时出现了许多太阳，而后又不见了，不知是被射下来了还是去哪里了（一不小心暴露没认真看表演了，汗颜）；随后《纤夫的故事》，《哭嫁》又讲诉了重庆号子儿女们的感人故事。
我们这次坐的是贵宾席，是观影效果比较好的一个地方，安安静静的看完了整场表演。笑容最感动的是《川江号子》的故事。川江号子是当年木船航运时候兴起的，给在川江航运的船工们劳作时号工领唱，其他船工跟着唱的一种歌唱形式。它既是一种艺术，也是一个代号。当木船逆流而上或者遭遇戈壁时，这些川江号子拉着大绳，攀在岩壁上，一步一个节奏的，边唱着歌边拉着船直至船靠岸。他们几乎全身赤裸着，或顶着烈日或冒着大雨，以小小之躯撼动整艘大船。没有他们，就没有了当时的繁荣。
然而，当动力船时代来临时，号子们也逐渐消失了，如今仍然在世的号子已然白发苍苍，许多年后或许我们再也没法听到他们的声音，但是，我们会记得他们的故事，并把他们的精神传承下去，不是么？我看见前方工作人员举着的“音像机”，里面是我们的身影，挥挥手还能在电视机里找到自己，别忘了这里的印象武隆，别忘了这里的川江号子，别忘了，他们的精神。

D4天生三桥

天生三桥这个景点爆红，恐怕还要感谢老谋子（张艺谋）毒辣的眼光，也要感谢《变形金刚4》的宣传，以及周董的《满城尽带黄金甲》的大力支持，才让天生三桥有了如今的名气。尽管在最后变4的全球公映时并没能履约，在电影荧幕上打上“中国武隆”四个字，但这也足以让山城里的这个地方走上了世界的舞台。
初见这桥还觉得不像桥，等到下了电梯后步行走到桥底下就可以看到桥的样子了，在电梯脚下，还可以看到桥洞的一侧上面有石头天然形成的菩萨像。由于菩萨下岩石的座下非常像大象，所以人们就把这尊菩萨称为文殊菩萨。
我们一行人马下了山走到四合院脚下，抬头一望，两座大桥便出现在眼前。这些桥，有的方方正正，像是被人用刀子直接挖开似的，而有的又像天龙。山体陡峭，气势磅礴，难怪这些大导演都愿意把这里当作实景演出的取景地。
在这些岩壁的内部，你还可以根据讲解员的提示找到这里隐藏的4个泉眼，有的泉眼留出来的水很少，就跟雾气似的，所以也被人称为雾泉。而其他也被根据特性进行了命名。看第二眼泉的时候你可以抬头看到上方的大洞跟个方盒子似的，深处手掌往天空空白的地方挡去，就好像可以一手遮天。然而有的地方不需要特意用手去遮挡，山体间的空隙相机遮挡，形成了空中的大刀，鲤鱼等。
在天生三桥景区范围内的所有的山体中，最形象的是“大猩猩”山，这座山神韵最好，一眼过去就能看到大猩猩的形状，生动形象。不过当你走到这个地方时，意味着你在天生三桥的即将结束了。讲解员会带着你到之前所拍摄照片地方的取相片处领取照片后再把大家带下山。

怒火八零全牛串串（九街店）

回到市内后终于和另外的吃货小伙伴碰上了面，我们都觉得之前的火锅吃的意犹未尽，于是商量着要不要再去吃一顿，经过一番讨论厮杀，我们最终决定了去吃位于江北区的全牛串串。这是一家装修极具特点的串串店。一开始看到这名字时笑容还在想着，怒火需要降躁，所以后面跟着的应该是8度吧，冷些，暴躁就降下来了，不过实际这家店叫怒火八零，或许是个八零后的小哥哥开的店吧。
店的生意特别好，取得时候猜六点左右，门口都排满了队，等了好一会才有空位接待我们。店里面的装修风格一边是热火朝天，一边是小清新，一边又是性冷淡风的感觉。我们在刚空出来的桌子上就座好后就去旁边的冰箱拿了些串串。
串串都是整整齐齐的放在冰柜里，种类很齐全。在这么多串串里面，笑容最喜欢的是贡菜牛肉，一般大多数商家都吃不到这个菜，还在重庆火锅店里这还是比较常见的菜，但是当贡菜与牛肉撞一起，恐怕还是这家店自己独创，两个最喜欢的菜一起入口时，别提有多享受。
除了串串，我们还点了几个特别好玩的菜，最有意思的就是牛一根儿～这道菜是把牛髓包在锡箔纸里，非常好看，未揭开时上面还会做一个透明的气泡球，球上透着多彩的颜色，有点像咱们小时候吹泡泡的泡泡一般，等到泡泡慢慢膨胀成一个球体在破后，牛髓就可以下锅了。
吃完火锅还可以吃甜品，它们家的甜品种类多的差点挑花了眼，笑容最喜欢的是酒酿冰汤圆，而小伙伴最喜欢的是最近刚上的新品-胭脂和怒～一道颜值非常高的甜品。碎冰吃起来爽口，而丸子吃起来糯糯弹弹的，一口沙冰一口火锅，这滋味真的是不错。

后言

爬了3座山，看了1场表演，逛了1个洪崖洞，吃了1顿火锅1顿串串，重庆之行就这么结束了，觉得还没玩够就这么结束了。依依不舍的告别了老朋友与新认识的朋友们，期望下次旅行还能再见到你们！重庆，再见！我还会回来吃火锅的！
