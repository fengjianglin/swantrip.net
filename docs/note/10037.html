近于禅境，达于心灵，牛首山揽胜静心之行

游记 人文游 寒假 小众 自助游 闺蜜游 2018/12/26

【前言】

我喜欢旅行。经历不同人、事、物，给人生制造不同的记忆点，老了回忆起来，会觉得“啊，我的人生还挺有意思的嘛”，而不是“也没什么特别的，眼睛一睁一闭，一辈子就过去了”。
整理出来的是一段时光的记录，虽然有些细节已无从拾取，但留存下来的都是最美好的印记。
此次牛首山之行也是有缘，与佛结缘。所以，三亚结束度假直飞南京，开启两天一晚的牛首山揽胜静心之旅。

【实用攻略和注意事项】

1、尽量少带现金,贵重物品保管好。现在技术十分发达,刷卡结账依然成为了一种常态。因此在外旅行尽量选择少带现金。但是少带现金不等于不带现金,不然万一景点景区没有刷卡机就惨了。同时外出旅行,相机、电脑等贵重物品尽量不离身,妥善保管。
2、车票类:确定好出行时间,提前选择出行的交通工具,飞机还是火车,最好下载个出行app，这样订票更方便一些。
3、数码类: 单反相机，手机，笔记本电脑，移动硬盘，充电器，移动电源，插线板。
4、出门看好天气，最好要选天气晴朗的日子。
5、若是自驾游，提前确定路线；
6、景区内交通车20元可以多次乘坐，10元车票是单次乘坐，建议买20元的。
7、佛像是不能拍照的,礼佛的时候一定不要大声喧哗,佛家重地手机最好静音。
8、禅境大观《如莲佛颂》演出时间: 每日10:00、11:30、13:30、15:00
 
 

【关于住宿】

翠屏山宾馆
选择翠屏山宾馆的主要原因是交通非常便利，在机场直接乘坐机场大巴，票价20元路程30分钟即可到达翠屏山宾馆门口。如果行李特别多打车也不远。
距离南京南站高铁站六公里，不到十分钟即可到达。离地铁口不到两公里。
酒店离牛首山景区大概10公里，开车15分钟，打车19元左右。
酒店地址：南京市江宁区天元西路168号。
翠屏山宾馆属于老酒店，应该属于那种经常做接待用的酒店，虽然不像精品酒店那么高档，卫生还是比较干净的。房价400元。
环境不错，宾馆中间有一个湖。早起或者晚饭后可以在院子里散步。

【佛顶前苑：山水悠悠似江南】

来牛首山玩，大多数人会选择一路乘坐观光车，直接到达佛顶宫，而我则是从佛顶前苑开始游玩。
佛顶前苑，顾名思义就是佛顶前面的区域。这个是离东门最近的景点。由于这边是刚打造的景点，加上现在是严冬，来的人并不多，所以广场上的喷泉没有开放。
佛顶前苑作为通向佛顶寺、佛顶宫的重要通道，设置了天阙揽胜、清水观鱼、玉梅轩、集贤桥、停心湖、十牛图颂、法融广场、趣牛园以及晚红小径等十五处观景点。整体通过三个层次的景致， 前景“鱼红莲水映天”指的是前方的莲池，中景“雾起白虹飘玉带”，玉带便是中间的集贤桥和远景“寺宇山色有却无”，层层景致引人入胜。
佛顶前苑景色秀丽，环境幽谧，假山泉池，石桥流水，奇山异草，与远处的山、寺共同构成一幅山水画，有点小江南的感觉。
精巧的玉梅轩，像是天上的仙女在凡间的馆阁，书画琴棋诗酒花，这样的生活岂非天上人间。由于我们来的时候没有工作人员演绎，同行的Maggie演奏了一首古筝，在山水幽谧的佛顶前苑听到这样的古筝，仿佛是在天上，飘飘欲仙。
在浓淡适宜的安闲中，踱步集贤桥上，散心停心湖间，再多的红尘烦恼，便也都可以抛之脑后，此时此刻，只与秋水与长天共温情。
听了Maggie的古筝演奏，我们继续前行来到十牛图塑石景点，这也是整座牛首风景的预演，移步换景，一步一景，这座园地埋藏了太多可能，前一刻你还是独立桥头的伊人，看尽秋风舞落叶的风情。后一刻你就成了翻越山坳的侠士，你能读懂十牛图的奥秘吗？
来牛首山，佛顶前苑里最值得一赏的便是“法融广场”，牛头禅宗的创始人——法融，双目轻阖，衣带翩然，正襟端坐于停心湖边，周围或立或匍匐，聚集着各种神态悠然的动物。这就是“法融说法，群兽听禅”的传说故事，用场景再现的方式叙述给每一位过往的游人听。
景区草坪到处是小弥勒佛的雕塑，有看书的，敲木鱼的，形态各异的向大家展示着佛。
 

【佛顶寺】

不远处的建筑便是佛顶寺，位于牛首山主峰西南侧，坐西朝东，为唐代建筑风格寺院，肇始于南朝初年，在其房顶还可以看到2对栩栩如生的金黄色牛角，共有殿堂七座，按照其功能分为礼佛、弘法、斋堂、僧寮、茶苑五个片区,是舍利护持僧团的弘法道场。
门前有桥和河，墙壁有10座石质佛像，手势不一，令人不禁膜拜。桥前有上香及悬挂祈福牌的地方，每年会有不少游客前来上香和祈福。
寺院佛殿建筑以中轴线贯穿对称布局，典型的中国古代建筑文化对称之美，彰显大气磅礴，又因是舍利护持僧团的弘法道场，所以在大气和雅致之外，还有着其他建筑无法企及的清净庄严。
佛顶寺还是必玩的景点之一。 

【佛顶塔】

佛顶塔位于佛顶圣境内，是佛顶圣境的标志性建筑之一，塔九级四面，高88米，造型宏伟。这是一座新修建的高塔，与不远处的宏觉寺遥相呼应，双塔重现了牛首山历史上“双塔”的恢弘格局。无人机飞起来，把宏觉寺和佛顶宫与佛顶塔放在一起，感觉到大气和壮观。
由于目前暂未开放，无法进去参观，据说在塔内供奉着一尊毗卢遮那佛坐像，里面还有一口仿唐式全铜铸金刚钟及各种藏经书。如果能够登至塔顶，可以俯瞰整个牛首山美景，当然佛顶宫在塔顶看着也是最美，最壮观的。
 

【佛顶宫：近于禅境，达于心灵】

在牛首山公园山顶，有一个宏伟的巨大穹顶，这就是“佛顶宫”。整个佛顶宫总建筑面积近10万平方米，主体建筑共9层，地下利用矿坑建设了6层，规模宏大，可以说在世界上的佛教建筑中也是少见的。
从佛顶尊胜道走上来，然后乘坐电梯上来，即可到达佛顶宫的跟前。在这里不仅可以观赏到《禅境大观》演出，还可以看到世界唯一的一枚释迦摩尼佛顶骨舍利。
看报道说这是花了40个亿建造出来的一座宫殿。
这个巨大建筑，分为大穹顶和地宫两个部分，寓意外供养和内供养。
大穹顶形如佛祖袈裟覆盖在小穹顶之上，象征着佛祖的无量加持。小穹顶下部为莲花宝座造型，上部为摩尼宝珠造型，上下结合形成“莲花托珍宝”的神圣意象。
佛顶宫总共有9层，所以地上三层，地下六层。进入佛顶宫，要先经过一个“入地”通道。通道内有地、火、水、风四大自然现象的奇观以及佛祖八相成道的绘画故事。
通过最后一道门，便踏进佛顶宫地上一层。每一扇门，每一条通道，都充满庄严肃穆之感。
到达佛顶宫入口后，先映入眼帘的便是巨大的穹顶。仰天望去，神秘的天空苍穹似乎近在咫尺，点点的繁星似乎伸手可及，天人合一。
禅境大观的核心是禅文化，由三个部分组成，佛出生的禅境花园、成道的禅境花园和中间的莲花剧场，共同表现了佛祖一生的故事。
演出开始时，苍芎之上的灯光暗淡下来，围绕卧佛周围的喷泉开启，卧佛在音乐的韵律中缓缓旋转升起，前来瞻仰的信者手捧莲花围绕卧佛绕行三圈，口中颂着佛经诚心祈求佛祖保佑和许下愿望。
切记在这里不能大声喧哗，手机最好调至静音模式，场面比较严肃，不应嬉戏打闹。
 
卧佛可随台面进行360度旋转，寓意佛法恩泽四方。
 
礼佛结束，我们往更深层的地下五层走去，这里没有直达电梯，只能乘扶梯。一层一层下去，通向幽深之处，更显地宫之深邃。
 
到了地下五层，一进入楼道，当即就被震撼了，这般富丽堂皇，难怪造价如此之高。
圆柱体的底层一圈圆形的走廊里面，墙壁上摆满了各式木质的、玉质的小佛像，在灯光的映照下十分壮观。
楼层的中心是一座大舍利塔，大舍利塔四面有门，游人可以进入塔内观看舍利。不过这里供奉的舍利是仿制的。
佛顶宫之所以叫佛顶宫，因为佛顶骨舍利也供奉在这里。当代佛教界泰斗星云法师曾说，中国有了释迦摩尼佛顶骨舍利，可以傲视全世界。
据典籍记载，2500年前释迦牟尼佛涅槃后，留下了佛牙、佛指、佛顶骨等不同类型的佛真身舍利，佛祖的这些遗留物被信众视为圣物，争相供奉。其中以佛顶骨舍利最为珍贵。
2010年，佛顶骨舍利在南京盛世重光，2015年10月27日被正式迎请至牛首山文化旅游区佛顶宫内长期供奉。
释迦牟尼佛顶骨舍利，是世间所有舍利中最为珍贵的一颗，象征最智慧头脑，也是目前为止，全世界发现的唯一一枚，在佛教中是至高无上智慧的象征。
听导游说，佛顶舍利和小舍利塔每年只有7次瞻礼机会，除了春节、元旦、国庆还有佛教节日（释迦牟尼出家日、涅槃日、佛诞日、成道日）以外，都是不对外开放的，所以我们也并没有看到。
 
壁画，佛像，雕塑，布满整个大殿。一种庄严的气氛氤氲。让人的心也沉静下来……
 
置身于整个舍利大殿，让人不仅能够深刻感受佛国天宫的殊胜美妙，以及舍利供养的无限庄严，还能领略当代中国佛教艺术的丰富绚烂，以及制作工艺的集大成之美。
舍利大塔上的四波罗蜜菩萨，以及对称于环廊四周的四方佛、慧门十六尊菩萨、四摄菩萨和八内外供养菩萨，共同组成佛教密宗金刚界曼荼罗第四供养会。
的确叹为观止啊，像走进了一个金碧辉煌的极乐世界般……
来这里的人基本都会绕围环绕一圈,嘴里默念愿望和祝福,或双手合十或俯身三拜九叩,以表达心中的信仰。
听导游说这里还是很灵验的，来玩的朋友可以心怀虔诚前来拜拜！

【融咖啡：时光清浅，岁月静好】

走累了，来“融咖啡”坐一坐。点一杯香醇的咖啡和可口的小点心，体验这处集禅意与文艺于一体的咖啡馆，时光清浅，岁月静好。
静静的坐在弥散着咖啡豆的醇香氛围里，看一本书或是聊聊天，都可以惬意的过一个下午。现在，越来越多的摄影师和模特渐渐的将咖啡馆当成了一个非常浪漫的外景地.....
那如何在咖啡店拍出美美的照片呢？下面就告诉大家几个点。
1.假装看书
现在有很多咖啡馆都带有一种沙龙气质，也有很多的书籍可供顾客参阅。如果到这种咖啡馆进行拍摄，那么利用书做道具是不错的选择，假装看书。
2. 坐姿大长腿效果
竖构图将相机贴近地面位置朝上拍摄，腿部会相应的拉长，头部会变得娇小，整体看上去便会显得腿部曲线更修长。
3. 趴桌子
不单单是桌子，你利用好咖啡厅中桌子上的物品也会对整体画面起到意想不到的作用，既可作为照片的前景表现出景深，也可以很好的点缀画面。
4．手中玩道具
无论是咖啡店里的灯饰还是圣诞装扮，把它拿在手中，可以不用看镜头，专心玩你手中的灯泡，也可以排出不错的效果。
