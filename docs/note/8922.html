黎阳in巷，记录着徽州最纯正的记忆

游记 2019/04/25
青石铺就的长巷，飘散着古城淡淡烟火，偶有行人悠闲走过，把恍惚的记忆，遗落在时光里。这是一座闻着风，都可以做梦的城，我们时常会被一些细小的柔情与感动潜入心底，忘了自己其实也只是这里的过客。有一天从哪里来，就要回到哪里去，短短数十载的光阴，不过是跟岁月借了个躯壳，唯有灵魂可以自由带走。
夜色下的的黎阳in巷
 ”遥想当年郁达夫夜泊屯溪，感受徽州文化，发出了最真挚的关于徽州印象的感概，这也是著名作家对于徽州印象最最真实的写照和描述，黎阳in巷就是新旧建筑的有效融合。这已经是我第二次来到黎阳in巷，使我再一次来到这里的原因是这里的氛围以及美食。夜色下，和朋友在这里漫步，可能是因为刚下完雨的原因，这里的人并不是很多，似乎多了一份宁静。
很多人看不懂这是什么意思，其实前面两个大家知道的，“我爱”的意思，后面HS则是黄山的缩写，这意思就是我爱黄山。
为了更好的演绎徽派文化，黎阳in巷不但有有自己的美术馆、石雕工作室，还准备了很多民俗表演，这些都是徽派文化的传承，街上各种琳琅满目的店面，也吸引了很多名人来这里开店，例如陈赫。
在古代，黎阳是徽商，徽文化发源地，是徽州山水风光汇聚地。沿着黄山市母亲河新安江，不仅纵贯出当时鼎盛繁华的”黄金通道“，更因着两岸秀雅山水而赢得”唐诗之路“的美名。这里历史悠久，素有”唐宋之黎阳，明清之屯溪“的美誉。可以说，这里曾是徽州人文、经济中心地带，珍藏着辉煌与精粹。
黎阳in巷共由三条街区构成，分别是滨江酒吧休闲街区、特色美食街区及徽文化体验街区（黎阳老街），在建造的过程中最大限度地保留了老建筑群，也适当的加入了一些时尚园元素，安静闲适，妙趣横生，在这里你可以找到你所要的一切。
徽建筑作为传承徽州文化的重要载体，黎阳in巷在改造时大量保留了徽建筑中青石板铺地蜿蜒曲折的街道，也借鉴了徽州水系的风格，成为这里最精华的部分，就如开头所说青石铺就的长巷，飘散着古城淡淡烟火，偶有行人悠闲走过，把恍惚的记忆，遗落在时光里。
有一种记忆叫水墨徽州，
有一种印象叫黎阳in巷，
在古朴与时尚之间穿梭，
将岁月揉进美妙的春光，
一杯热茶温暖在手，
这样的日子，就刚刚好
徽自在，一处放松的地方
我去过很多地方，也住过很过民宿，大多数住过的民宿对于我来说，就是行程中暂时休息的点。第一次来到这家民宿，徽自在，它对我是陌生的，起初当我听到这个名字的时候，并没有联想到很多，而是把它当成一处普通的歇息点。可当我走进了这里，发现我对这里的看法完全是错误的。
长时间的高压环境使人身心疲惫，
想逃离这喧闹的“世界”，
去天边，去广遨的天地间，
在这里，可以安静的喝着茶，看着外面的风景，享受所有的一切。
拥有东方古朴人文美学装修风格的房间，
明月下手执书卷，听耳畔丝竹之音缭绕。
生活，休憩。此处，自在。
青石板的街道与竹影雕窗、长廊飞虹相得益彰，或晴天或落雨，再次而坐，时光就是用来如此浪费才不觉得浪费。在徽自在，安顿下来，我和我最好的时光。
房间很大，
房间是简洁的，
房间里的色调是素雅的，
房间充满了自然的味道。
洗浴间的干湿分离设计，色调使人进去后感觉清爽干净。
淡灰墙面柔和富有层次，与天然去雕饰的浅色原木家具相称。
伴随着自然的感受，自然将整个房间一分为二。
酒店多用木质材质的家具，整个房间都充满禅意。
面朝大海，春暖花开，给自己的心一个港湾，盛风盛雨盛欢笑。
给自己的心一片蔚蓝，寻寻觅觅，静思淡行。
临江一楼，徽菜系中的精华
作为安徽人，对徽菜系肯的是独有情钟，临江一楼恰恰把徽菜和美学结合，制作出不一样的徽菜。而餐厅的装修也是别有风格的，进去后放佛来到了博物馆。
他家的菜摆盘极为讲究，让我们享受到了地道的美食以外另外菜品更是世界上的享受。
临江大虾，这道菜是大家非常爱吃的。
​齐云山，道教的圣地
黄山不止有黄山，还有齐云山，而齐云山的在道教中的地位高于黄山。
齐云山，古称白岳，因遥观山顶与云平齐得名。中国道教四大名山之首，中国五大仙山之一。
树上挂着葫芦，这是齐云山上的标准。
在导游引领下，来到当年乾隆求子的太虚宫。太虚宫面向香炉峰，但见孤峰之上立有一亭一香炉，上下只能凭借一根铁索。背靠三座山头，似太师椅环绕稳固。这种建筑布局从风水学上讲就是“前有照、后有靠”。进得宫内，有女道士为我们讲解道教基本常识，我也求得红彩绸祈福消煞。此时，同行者中一人与道士发生口角，原因是道士向其要六百元功德钱。现在佛道寺庙商业化氛围太重，是否背离其境界呢？想想宗教也应如此吧，否则没有供养怎能香火长盛。只是觉得应该“随喜功德”，不应该被世俗功利过多熏染。但当下世风确是如此！登山时曾向一个当地农家乐的男主人问路，我不知他是否是道士，但是他的回来令我倍感失望。他说：“像你们这样
齐云山天街，这是山上道士的后裔所居住的地方。随着时间的积累，这里逐渐形成了一条街，站在山的对面看上去，就感觉像是在天上一般，所以这里被称为天街。
黎阳老街，重现徽州记忆的地方
一直以来对老街就是情有独钟，觉得它是一个时代的缩影，保存着原有风貌，一草一木，一水一屋，都仿佛还是那个时代的生命。走在老街的角角落落，也或许能找到前世的自己。
历史悠久的黎阳老街。它或许是暴露在人来人往的快门下，形成了一张张定格的代表性的画片，亦或是安静的独享沉淀的韵味。
走在古朴的街道上，感受老街古老，
也许你触摸的不再是石头，而是黎阳的历史，
这里没有城市的喧嚣，有的只是自然的宁静与超脱。
民俗与民居相结合的经典风景，游走水乡感悟水乡是很多人的梦想。慵懒的午后，满载回忆味儿的躺椅，这种闲适画面，是否曾在你脑中浮现？在黎阳老街中，体验这静谧的生活，会让你的生活更添一份浪漫。把那些逝去的似水年华，融入这乡间老街，没有太多奢侈，只是让你承载更多内心的安详。
老街定点也会安排文艺工作人员在街上进行快闪活动。
黎阳老街的一草一木仍在我脑中挥之不去，此时我发现，
我已经爱上这个地方，黎阳老街。有幽深的小巷，
有古朴的石桥，也有那份令人神往的宁静和慰藉，这就是黎阳。
舞台上文艺工作人员在上面为我们演奏了黄梅戏。
欲识金银气,多从黄白游。一生痴绝处,无梦到徽州。这是明代戏剧家汤显祖留下的千古绝唱，意思是，一辈子想去人间仙境，可做梦也没梦到人间仙境原来在徽州。写出了徽州之美是人的想象力抵达不到的地方，今天去品、去赏的就是徽州的黎阳in巷。
千年古镇，时尚新生。黎阳in巷以古今碰撞的建筑语言为特色，在黎阳in巷，有点穿越的感觉，一边是古色古香的百年老宅，一边是现代化潮流设计师的建筑，新老建筑有机的交融，让人既能体会到千年古镇的厚重历史，又能感受到现代都市的时代特征。
整个街区精彩纷呈，漫步闲逛，别有风情，目前保留九栋较为完好的徽派老宅，更有晴空、夜色、云海、流光为游者搭建360度的意境时空，处处皆是美轮美奂的天然布景。古风依旧，小桥流水间偶尔还会冒出阵阵白烟，游人如临仙境。
走在蜿蜒曲折的青石板，慢慢品味着黎阳这条充满特色而又富有历史底蕴的老街。你会发现黎阳建筑的墙都是非常高大，而且还没多少窗户，据说这是徽州商人为了保护家中女眷，断了盗贼的贪心，不让财物外露，就把墙壁砌得这么高。
一生痴绝处，无梦到徽州。走进黎阳in巷，就像走入的少女的内心。脚步在不知不觉中放慢，踏青、赏花，感受春的气息。充满仪式感的活动，不仅让人感受到厚重的文化积淀，更表达了对时尚新生活的追求与向往。
徽州乡贤馆，这里记录着徽州的人文史，
走进这里让我们了解到徽州的今生前世，
了解和认识了徽州的名人。
徽式老宅的墙体都是由木制的柱子支撑固定起来的，
墙上那些钉钉卯卯代表此处必有一柱，类似钢筋，
而那些砖土则相当于现在的水泥。
黎阳in巷现代建筑包裹着古建筑，
即是对古建筑的一种保护，也是一种创新的再造，
给人一种时空交错之感。新型徽派建筑映衬着牌坊，别有一番韵味。
老街共保留了八座有百年历史的徽派民居，
新老建筑的交融，使我不仅触摸到了历史的印记，
也感受到了是带所带来的变迁。岁月如墙上斑驳的印记，
再也回不去的往日时光，都沉淀在这一砖一瓦里。
樊登书院，洗化心灵的地方
在黎阳老街，还存在一处悠然自得的地方，那就是樊登书店
在喧闹的城市中，来到黎阳老街这安静的地方，在黎阳老街里的樊登书店更是这里的精髓所在，
与其说这是书店，更不如称之为书院，在这里有安静独特的环境，有各种图书，有各种精致的周边。
在这里喝一杯清茶，拍几张照片，三五个好友凝聚在此度过一个愉快的下午。
