浙江小城了不得，石板路一脚跨越三朝代，谜一样的石窟至今难解

游记 人文游 古镇 小众 美食 自驾游 2018/10/06
龙游县位于浙江省西部，是浙江东、中部地区连接江西、安徽和福建三省的重要交通枢纽，素有“四省通衢汇龙游”之称，“儒风甲于一郡”之誉。因龙游县南山区盛产竹子，享有“中国竹子之乡”的美誉。
龙游旅游资源丰富，景点众多，有龙游石窟、龙游民居苑、三叠岩、三门源、江滨公园、乌石山、六春湖等，此外，龙游的温泉、美食等也受到越来越多人的关注。美食有龙游发糕，龙游三头一掌等等。
【龙游民居苑】
相信大家都听说过越王勾践的故事“卧薪尝胆，三千越甲可吞吴”，这个故事都发生在浙江，而同样在浙江大地也有一个小国，在春秋时期有一个小国家与越国的命运紧紧相连，那便是姑蔑国，姑蔑国先是被越国灭掉，后楚国吞并越国，成为楚国的领土，后秦始皇扫六合统一天下，这里成为会稽郡的辖土，从侧面反映了从春秋割据到大一统的全面过程，那么这个姑蔑国在什么地方呢？今天牛哥就带大家去认识一下--浙江省衢州市龙游县。
牛哥今天就给大家介绍这个小城，龙游地方不大，民风淳朴，对于很喜欢古建筑的我，听说这里有古建筑群。牛哥便专程前往,龙游民居苑位于鸡鸣山旁，当地人也称之鸡鸣公园，牛哥了解到这些民居是当地本身就有明清一些古建筑，但都多破落不堪，后来当地政府出资修整，将这里改造成公园。
后从龙游县各乡镇各村搜集同样一些急需拯救的老建筑，通过一比一的方式整体搬迁到这里，日积月累就构建成了现在的龙游民居苑，现在这里有各时期古建筑民居四十三栋，现在城市的快速扩张使一些老建筑的位置尴尬，一方面老建筑体现了古人建造工艺的水平，断然拆掉未免可惜，但另一方面又影响着城市的整体规划，一些市政工程都难以展开，但龙游人民很有智慧，将古建民居做统一聚集，既可以是古建爱好者的乐园，又可以将古建完好保存下来及时修复，更重要的是城市规划更加容易，牛哥不得不给他们一个大大的赞。
而且园区内的青石板便道也是非常有历史感，这些青石板是从各个地方收集上来的，有不同的时期朝代，当我们今天再来游玩的时候，有可能自己的一小步，却跨越了好几个朝代，浓缩了多少的历史岁月，周边的仿古建筑也都是江南特色的味道，北方游客过来可以都印象中的江南小景，说起这些民居更是见证了荣华兴衰，这些建筑大多是浙商为自己家族修的宗族祠堂、深宅大院，这里充分的体现了我们中国人对家族血缘重视，讲究家和万事兴。
这些建筑中有名气的有：高岗起凤”、“翊秀亭”、“巫氏厅”、“鸡鸣塔’等，专家说这几十幢古建筑按现在的估价至少要上亿，很多建筑是无法用钱横了的。去这里游玩也较为方便，就在龙游市区，周一到周五对全球的游客免费参观，只有在周末节假日收取门票，十一出行，比起人多拥挤的景点，不如去趟清新秀丽的小众景点。
【龙游石窟】
提起石窟大家往往会想到莫高窟、龙门石窟，然而这个石窟和大家印象中不同，这里既没有香火，又没有佛像，还被成为世界第九大奇迹，那就是什么呢？请牛哥给大家慢慢道来。
时间还得说回上个世纪九十年代，在旅游一个叫石岩背的小山村，当地的村民一头牛跌入了水塘里，村民决定打捞出来，但是这水塘深不见底，靠人力无法打捞，这水塘也奇怪，塘口面积不大却不摸着底，村里更是传说这里连着大海，是龙王的一处行宫，但是村民就有大胆不怕邪的，几户人家筹钱1万元买来了抽水机等设备，决定用水泵把水抽干，用俗话说就是一定要看见牛的尸体，经过17个日日夜夜轰隆隆抽的水声过后，令村民惊愕的是，他们发现了一个巨大的地下石窟，石窟的底部离地面有数十米，有几根特别粗的石柱撑着石洞，里面有着巨大空旷的石窟，离洞口不远处有着一个一个很整齐的台阶，一直通到底部。
这几个村民到现在都不会想到，自己的“找牛”却不经意找到了世界上最大的人工开凿地下石窟，而这里被誉为“世界第九大奇迹”，因为龙游古代曾是姑蔑国，这个小国曾被越国灭掉，所以人人都传说这里是越王私练精兵的地方，只能在这地下悄悄练兵，准备一雪前耻，但要防备被吴王发现，所以只能秘密进行。
这里被发现后，当地政府继续查看了周边的类似洞口，大大小小有24个，分布在山周边，而山的旁边便就有河流，所以很难被发现，有的一些洞口被土石堵塞，有的便就是雨水存在里面。大的石窟面积达数千平方米，小的也有百平方米。再后来无数历史学家、建筑学家、考古学家等学者前来勘察研究，石窟很明显是工匠一锤子一斧子开凿出来，而且是经过规划的，而且石窟墙壁上雕刻着美丽的浮雕，石柱上也有龙形图案，本来一个巨大的石窟就非常惊奇，这些雕刻就更加扑朔迷离。
由于被成为第九大世界奇迹，关于龙游石窟的来历有很多种说法，大批专家和学者纷纷对石窟形成的年代及用途提出各种不同的推想和论证：采石场、墓穴群、藏兵站、储冰库、巨石文化、“道家福地”、“造反的采矿人安身之处”等等，众说纷纭。但目前最有权威的说法是：古代浙江地区采石场很多，龙游石窟最大的可能便是一个石头矿洞而已。当地人说当地江边的石洞旁，常年堆积着当地人称为“石坯”的红石头，而龙游境内的村庄，当年有许多的房屋也是用这种红色石头来建造的。至于到底是什么至今还是一道谜团
因为空旷壮观，说话时会有清脆的回音，景区离龙游市区不远，可以到北客站坐车前往，也可以从金华市区前往，大约一个小时的车程，建议大家自己亲自去见识下这感受下这神秘又壮观的石窟。
【石角漂流】
十一的假期即将结束，只剩下了两天的时间了，你是不是一直宅在家里，看着景区人山人海，高速路上汽车长龙，想不想去一个好玩又不拥挤的地方，牛哥今天就给介绍大家一个好玩的地方。
位于浙江省西部龙游县的石角漂流就是个绝佳的地方，漂流所在的是一个村庄叫石角村，石角漂流就是这样来的。石角村也称石墎村（当地方言“角、墎”同音），是一个古老而富有时代气息的村庄。村下多个自然村组成。石角村山清水秀、人杰地灵、民风淳朴、友善好客。
这个只有1500多人的古村落，却有一个著名的石角八景，它还曾是龙游4个重要古码头之一， 这里人文丰厚、景美如画，如今，村里把曾经的一个古埠头，予以复原、修缮，与沿江铺设的游步道融为一体，供游人观赏、村人缅怀，曾居于交通要道之上的古村，在历尽兴衰浮沉以后，正在以全新的姿态，惊艳亮相。
像世外桃源静谧在这里，但与其他漂流的不同的是，这里并不用皮划艇，而是用更原始的竹筏往下漂，用竹竿去控制方向，是不是特别刺激，湍急的水流涌动这船往前跑，也不用担心日晒，两岸还有碧绿高耸的青山，水滨边有着竹林片片，像隐居仙人居住过的地方，真是个人间仙境。小姐姐国庆节很优雅的坐在竹筏上放空自己。
因为是漂流运动，年轻人比较多，在这里邂逅小哥哥小姐姐的机会也是比较大，竹筏不时的左右倾晃，竹筏上被激流涌上水，还要控制好方向，小心周边的礁石，一旦落水，还有救生衣保护，所以大可放心，可以尽情尝试，不要害怕紧张，体验一把急流勇进，水上的速度与激情。
今年67岁的徐春权是石角村的一位资深老舵手。老徐16岁起就跟着爷爷学撑船。石角村村民 徐春权：从龙游到遂昌那个时候没路的，就是水路。回想起古埠头“渡口归帆”的这一景，老徐直言太美了。
全程有4公里长，漂流时缓时急，让大家劳逸结合，既可以欣赏风景又可以恢复体力，等待下一段的冲刺，说了这么多，大家有没有兴趣前往，约上自己的小伙伴，赶快出发吧。
【吃在龙游】
牛哥给大家介绍下龙游小吃，来到龙游当然要尝尝当地特色，有句话说的很好，民以食为天，所以当地的美食可不能错过，牛哥一一尝试了，给大家慢慢道来。
牛哥首先就尝试了龙游人经典特色的葱花馒头，葱花馒头绝对是龙游人早点的头牌，上好的猪肉与当地特色笋干切碎成末拌馅，然后放在小馒头内，在外地人看来像开口包子，咬一口，正是好吃到不得了，馅肥瘦正好，还有笋干的清香，再配一碗当地特色的米糊一起吃，真是好吃到哭，米糊味道鲜美扑鼻而来，有了米糊，吃起来不会觉得太干，这个CP赞赞赞！
紧
接着牛哥就尝试了当地很有名气的肉圆，肉圆也是由猪肉做成的，乒乓球大小的小肉丸实在让人不忍心吃，放一个在嘴里一嚼，Q弹有劲、咸淡正好，肉圆中还会搭配一些蔬菜，吃起来不会那么油腻，然后牛哥又品尝了龙游的发糕，发糕在当地读音与“福高”谐音，是吉祥福寿的象征，是龙游最具代表性的小吃之一。牛哥吃了一块，真的是松软可口、甜度适宜，老年人小孩子可以多吃点，方便消化又寓意好。
“三头一掌”大概是我们家乡最富盛名的小吃了，肉质细腻、疏松，并富含高蛋白。鸭头、兔头、鱼头，是谓“三头”，鸭掌，即那“一掌”。做菜时鸭掌、盐、味精、辣椒是基础，陈皮、八角、肉桂、千里香作为香料，更有白芷、甘草等几十味中药，一味一味按顺序放入小砂锅。打开火，“咕嘟咕嘟”一下午，那香气就从不断翻腾的气泡中，从不时被顶起的锅缝中溢出来，让人迫不及待地想吃到美食。
龙游小吃最有名的要数发糕，发糕制作工艺独特，配料考究，成品色泽洁白如玉、孔细似针、闻之鲜香扑鼻、食之甜而不腻、糯而不粘。其最大的特色是在制作过程中加入适量糯米酒发酵而成，营养丰富。因“发糕”为“福高”之谐音，寓“年年发、步步高”之吉祥涵意，成为当地百姓逢年过节必备名点。2007年原国家质检总局批准对“龙游发糕”实施地理标志产品保护。发糕也推出了真空包装，随时都可以买到。
青团许多地方都有，与众不同的是，龙游当地称之为清明果，而且是饺子状的，是清明节的特色小吃，它是用一种艾的植物的嫩叶和糯米粉制成的，牛哥顺手就买了两个，热气腾腾的艾草味漫散开来，像一块碧绿的宝石在手中，吃起来糯软香甜，里面的馅也是蛮美味的，当地还有猪肠，与印象中的不同，这里的猪肠是古铜色，同样还有鸭掌、鸭头也是这样，方便的大家带走的有豆豉，龙游本地话叫豆丝，香辣可口。
还有当地人从外归家时，一定要吃碗北乡汤圆，这种汤圆不是球状的，有一个可爱的小尾巴，诱人口水，用勺子盛上一个放嘴里，美味极了馋到哭。说了这么多好吃的，大家的口水都要留下来了，欢迎大家自己亲自前往去品尝一下。
【住在龙游】
龙游新国际大酒店，应该是龙游城区最新的酒店，地理位置还算方便，窗外就是相傍美丽的衢江，可以去江边走走。服务也挺好，这次正好赶上中秋节，服务员还送了月饼和水果。进去第一感觉，设备比较先进，智能马桶，感应灯。高铁站到酒店15元，去龙游石窟也很近，自驾游很方便。自助早餐，有龙游发糕，小点心，培根，咖啡，品种还蛮丰富的。
