超高性价比解锁普吉新玩法

游记 亲子 家庭游 海岛 精品酒店 2019/06/01

前言

我总是和朋友说，如果第一次出国，那 泰国 一定是极好的选择。其实在我眼中，即使不是第一次出国， 泰国 仍旧是旅游和度假极好的选择。
其实推荐 泰国 的原因有好多，首先是距离近，以从 广州 出发为例，两个多小时就能抵达，并不长的路程不会过于疲惫，影响接下来的旅游。
其次是消费， 泰国 的消费实在是非常低廉，以便利店为例，红牛的价格是10~15泰铢一瓶（人民币与泰铢汇率为大约1:5，价格视乎不同的地方价格会有一点区别），500ML的酸奶价格在20~35泰铢之间，吃饭的价格也非常实惠。
第三是服务态度， 泰国 一直有“微笑国度”之称，在这里，从酒店到小贩，都很容易让人感觉到这里的热诚。而且在 泰国 设置有旅游警察，如果遇到任何的旅游纠纷都可以向旅游警察进行投诉。
签证方便， 泰国 是落地签，不用提前各种烦人的签证手续，最近更是有落地签免费的优惠。
所以，我总是一次又一次地带着小肉包来到 泰国 ，去享受这里的美好感觉，和他一起，体验微笑国度的微笑与热情。
4月是 泰国 的宋干节，这是 泰国 人民庆祝从旱季转入到雨季的节日，在这个节日他们会走上街头，向路人泼水表达喜悦与祝福，因此，这个节日也被称为“泼水节”，在今年的泼水节，我特意带上小肉包，来到 泰国 ，去感受这个节日的热闹，也再一次感受 泰国 的美好与热情。
这次把目的地定在了 泰国 的 普吉岛 ，这个 泰国 南段的省份有灿烂的阳光与细腻的沙滩，有清澈的海水和各种美景，和爱玩爱闹的小肉包一起来，最是适合不过。

D1 广州飞往普吉岛

因为带着小肉包，所以选择了直飞的正点航班。
从 广州 到 普吉岛 ，有南航的正点航班，下午起飞，在接近傍晚的时间就会抵达 普吉岛 。
在飞机上会派发入境卡，填写完毕后直接提交会更加方便通关。 泰国 的签证一般不会出现什么问题，听说会有人检查身上随身携带的现金，少于4000RMB会被拒绝入境，但数次前往 泰国 并未碰到过相关的检查。
通关非常快，提取行李后已经订好的度假村Sunsuri度假村就已经安排了接机的人员，直接前往度假村。普吉的机场位于 普吉岛 北面，而这次预订的Sunsuri度假村位于 泰国 的南面，车程大概在一个小时左右的车程，算不上遥远。而且来到这里的时候，宁静而舒适的氛围让我感觉，这点路程实在是太值得了！
来到酒店，前台的服务员第一时间就送上了欢迎饮品和欢迎水果。热情的服务让我颇有一种温暖的感觉。
因为晚上，所以酒店的环境看得并不算特别清楚，但酒店的行李车把我们送到了半山的房间的时候我觉得酒店的规模实在是颇为不小。
预订的是酒店的套房，毕竟在我眼中，度假需要舒适的住宿环境，足够宽敞的空间和足够舒适的房间才能让度假真正成为度假。
酒店的房间没有让我失望，套房的面积有大约6、70平方，分为客厅和房间两个部分，客厅大大的沙发最是适合来个“葛优躺”。也准备大冰箱，在炎热的 泰国 ，往冰箱里塞满水果饮料和雪糕，回到酒店打开大大的冰箱，绝对也是极好的享受。
浴室同样很大，房间用了干湿分离的设计，非常方便。还有大大的浴缸，小肉包来个泡泡浴就是他喜欢的节目。
卧室也非常宽敞，一米八的超宽大床即使一家三口入住也完全不会有不够舒适的感觉。浴室采用了干湿分离的设计，还有大大的浴缸，在房间泡个澡也是非常舒适的享受。
更赞的是酒店的大阳台，能够看到酒店的泳池，看到奈汉海滩蔚蓝的大海，坐在阳台的沙发上，来一杯香浓的胶囊咖啡，也是极好的享受。
因为来到酒店的时间已经不早，所以晚餐就直接安排在了酒店的帆餐厅进行。帆餐厅提供自助餐，有泰式特色的各种美食，也有更适合大众口味的食物。不得不提的是这里的各种烤串和扒类，选好后会直接由酒店的大厨立刻烹制，非常新鲜。周末的晚上还有歌手的驻唱，浪漫的歌声配合美味的食物，Sunsuri的夜晚温暖而浪漫。

D2 酒店活动

第二天第二天并没有安排外出的行程，在酒店其实可以参与的活动很多，干脆就把整天的活动定在了酒店。
起来后参与了酒店的亲子搏击课，亲子搏击课非常有意思，酒店特意把上课的地方安排在户外，在凉亭下，跟随酒店特意请来的搏击教练，学习各种搏击的技巧。教练是专业的泰拳教练，在阳光下跟着专业的教练挥洒汗水，学习技能，确实非常有意思。
烹饪课会让家长和小朋友分别在大厨的指导下制作不同的美食。精通 泰国 菜的大厨教我做了包括冬阴功汤、鲜虾春卷、甜品等美食，虽然我并不擅长烹饪，但在大厨的指导下，我也感受到了烹饪的乐趣，我也做出了色乡味俱全的美味呢。而小肉包在大厨指导下制作的是他最喜欢的披萨，小肉包可以根据自己的喜好，加入数量不等的各种配料。小肉包就加入了很多的番茄酱和奶酪，都是他喜欢的食物呢。
在烹饪课结束之后，和小肉包一起，品尝自己亲手制作的美味，非常有意思，这种和他一起动手的快乐，总能拉近我们彼此之间的距离，能让我们有更多属于彼此的美好回忆。
因为正好是泼水节，所以酒店的泳池也安排了特色的活动，可以更好地感受泼水节的美好和热闹。
其实酒店的泳池本来就很大，有很多有趣的设施，4.8米的 高台 跳水最是刺激，从4.8米高的地方直接一跃而下，心脏仿佛瞬间跳离了胸膛。而旋转滑梯同样能感受极速的感觉，即使是小朋友也能感受滑梯的极速快感。
泳池同样非常贴心地分成不同的区域，有适合成年人畅泳的深水区，也有贴合小朋友玩耍的浅水区。而且不同的水深之间还用了围栏进行分割，不用担心小朋友不小心跑到深水区，发生不可预料的危险。
泳池每天还会组织各种有趣的活动，在服务员的组织下，皮划艇，水上排球等各种好玩的活动层出不穷，在酒店的泳池总会充满了欢乐的笑声。
如果喜欢安静，那躺在泳池边的躺椅上，享受属于自己的宁静，享受 普吉岛 的阳光也是非常舒适的选择。在泳池，还有泳池吧，提供各种特调鸡尾酒和各种饮料，点上一杯，泡在泳池中，享受冰凉的池水，享受美味的饮品，这种度假，最是闲适。
泼水节的活动更加有趣，先是有酒店的服务员表演 泰国 特色的舞蹈，接下来就是泼水节的环节。服务员会往游客头上浇水，也有调皮的服务员往水枪里装上冰水，跳起“战争”。泳池变成了打水仗的“战场”，非常有意思。
玩累了，当然要来个SPA。 泰国 的“马杀鸡”一直都闻名全球，手法舒适，同时 泰国 产出的精油质量也非常出名，配合 泰国 特色的按摩手法，能让肌肉放松的同时，也能让肌肤得到呵护，更能让精神得到放松。
在度假村内，就有提供SPA按摩服务的地方，玩了一天，其实也颇为劳累，这个时候当然要来体验一下正宗的 泰国 SPA。
如果不想带着小朋友按摩，可以把小朋友“寄存”在儿童俱乐部中，俱乐部有小孩子都喜欢的各种玩具，小朋友呆在其中也完全不会有无聊的感觉，更重要的是，在这里，还有专人照看“寄存”在这里的小朋友，不用担心小朋友有危险或是丢失哦！
而我选择了带着小肉包一起体验正宗的 泰国 SPA，毕竟对他来说，感受一下 泰国 的SPA，也是难得的体验。
这里的SPA中心装修非常精致舒适，采用了原木色作为主要的风格。每个房间的私密性也非常好，可以非常放心。
当然，技师的按摩手法也没有让我失望，按摩力度正好，而且拿捏的位置非常准确，可以很好地放松精神和身体。在香薰的味道和轻音乐的声音中，在半梦半醒间，完全恢复了体力和精神的感觉非常舒适。

D3 珊瑚岛——拉威海滩海鲜晚餐

因为酒店有住三天送珊瑚岛一日游的优惠，所以第三天的行程就定在了珊瑚岛。
虽说 普吉岛 的海水已是非常不错，但其实 普吉岛 离岸的小岛往往因为开发程度较低，水质也更加出色。珊瑚岛，皇帝岛，PP岛， 斯米兰 都是普吉这边水质清澈的小岛，因为带着小肉包，所以没有深潜的打算，珊瑚岛对我们的游玩来说已经足够。
要到珊瑚岛，需要从拉 威海 滩的码头上船。从酒店可以到购票点，购票点会有巴士（也许可以被称为巴士？ 泰国 特色的巴士，两边是护栏，人可以直接挂在上面那种……）送到码头，然后在码头乘船前往珊瑚岛。
当然，前往珊瑚岛的船比前往码头的“巴士”漂亮多了，有帆船和快艇两种，乘坐崭新的快艇，在蔚蓝的大海上乘风破浪，大概只需要十分钟就能来到珊瑚岛上。
珊瑚岛的水非常清澈而蔚蓝，站在岸边，就能看到海中游曳的小鱼。岸边的沙子同样非常细腻，踩在沙子上，脚底柔软的感觉非常舒适。
沙滩上有躺椅，租一个躺椅，躺在海滩边上，晒晒太阳，看看蔚蓝的大海，同样是极好的选择。
如果说珊瑚岛上最出名的，一定是这里的各种娱乐项目。在珊瑚岛上，从最基础的浮潜到深潜，再到刺激的水上摩托，快艇，水上降落伞，都可以选择。在珊瑚岛清澈的大海上，感受水上项目的刺激 和美好，也是很不错的体验。
午餐在岛上有提供自助餐，有肉类、米饭、水果无限量供应，虽然算不上丰盛，但已足够填饱肚子。
如果想前往皇帝岛，同样可以在珊瑚岛乘船前往，但因为实在不想太累，所以我没有选择“跳岛”，就在珊瑚岛享受属于自己的闲暇时光。
下午离开珊瑚岛，回到了拉 威海 滩。如果想回到售票点，同样可以在码头乘坐“巴士”前往，但其实在拉威海 滩，同样值得把游玩的时间花费在这里。
拉 威海 滩边上是一个海鲜市场，当地渔民每天捕捞上来的新鲜海鲜都会在这里售卖，各种海鲜非常新鲜，同时价格也非常划算。足有小孩子手臂大小的濑尿虾大概只要600泰铢一只，龙虾也只要500~800泰铢。在摊位上选购好新鲜的海鲜，就可以到旁边的大排档加工。喜欢海鲜的我当然不能错过，要在这里享受一顿属于自己的海鲜大餐！

D4 大佛 查龙寺 海豚表演

泰国 笃信佛教，在这里总不会缺少各种各样的佛寺。
我先是早起体验了一次 泰国 的布施，在 泰国 ，所有的男子在成年的时候都要出家一段时间，出家期间，只能通过化缘维持生活。
酒店每天清晨都会为附近寺庙的僧侣布施，还能得到他们的祝福哦！
而在 普吉岛 ，最出名的一定是这里的大佛，以及查龙寺。所以我干脆用半天的时间，感受一下 泰国 的佛教文化，感受一下这里浓郁的佛教文化。
普吉岛 的大佛建于山顶，大佛通体呈白色，非常漂亮。
大佛脚下种满了菩提树，菩提树上挂满了信众祈祷用的金叶子。金叶子上写上了信徒的心愿， 泰国 的文字虽说我看不懂，但却足以感受到他们对于幸福的期待，和对于美好的愿望。
在大佛所在的山顶，还能看到 普吉岛 的全景。俯瞰普吉，错落的楼房，远处的海岸线，组成了普吉美好的画卷，就这么呈现在眼前。
查龙寺是 普吉岛 最大的佛寺，同样是来到 普吉岛 不能错过的打卡点。
查龙寺旁边有一个清澈的湖，湖里养了颜色鲜艳的鱼儿和乌龟。买上一包鱼粮，在这里洒下到湖中，鱼儿和乌龟游出水面，抢食鱼粮的画面非常美丽。
湖边当地居民的小屋被粉刷上了鲜艳的颜色，和碧绿的湖水在一起，组成了一幅彷如童话的美丽画卷。
查龙寺的建筑同样充满了 泰国 佛教的风情与华美，红屋顶，白色的墙，白塔和庙宇在这里极尽华美。佛像用金色塑成，庄严而神圣的佛像端坐在莲花台上，静静地接受信众的朝拜。
有信众在佛像上贴上金箔，跪拜和贴金箔的画面，同样是这里难得的画面。
离开查龙寺的下一站是海豚表演。佛寺的静和海豚的动都在一天内感受，其实也是颇为有趣的一件事。
看过的海豚表演实在不少，本来并不打算前往，但看到网上的评价实在不错，所以还是决定到这里来看看。
没想到，这里的海豚表演，确实给我带来了无限的惊喜。
表演分为两个部分，第一部分是海狮表演，笨拙的小海狮甚至会算数，会向观众索要掌声，非常可爱。
而第二个部分的海豚表演更是整个演出的精华和高潮所在，训练员和海豚在表演中仿佛融为了一体，海豚除了常见的各种动作，甚至还能带着训练员在表演用的水池中驰骋，和训练员一起上演各种高难度的动作哦！
训练员站在海豚上，跳跃，破浪，各种动作优雅而帅气，不断引起场内的阵阵尖叫声。
在回程的路上还经过了这里的贝壳馆，贝壳馆收藏了各种珍贵的贝壳，小时候总是喜欢收集各种贝壳，能满足我一切关于海洋的期待与幻想，而在这里，无疑是关于童年回忆的一次追忆，也是关于童年回忆的一个小圆满。

D5 大树秋千，奈涵海滩，神仙半岛日落

大树 秋千是 普吉岛 新开的一个景点，从酒店出发前往，大概只需要半小时的车程就可以到达。
要前往网红秋千，要换成景区的越野车，走在颠婆的山路上，颇有一种乘坐过山车的刺激感。下车之后，就有柳暗花明的感觉，秋千位于大海边上，有拍照用的秋千，也有挂在 大树 上的 大树 秋千。
海边设置了很多拍照用的装置，在海边随手一拍，大海和拍照设施定格在画面中，无论是鸟巢或是秋千，无论是椅子或是玻璃球，都能构成美丽的画面。
要玩 大树 秋千，会有景区的员工帮忙把秋千推起，秋千的高度会甩得超过椰子树的高度，非常刺激。
离开 大树 秋千，我回到了酒店附近的奈涵海滩。
这个 泰国南部 最出名的沙滩，人少景美，远离尘世喧嚣，给人以宁静之感。只要住酒店，就可以享受酒店定时往返班车免费接送，每10分钟就有一班，非常方便贴心。
奈涵海滩的游客相对较少，海滩也更加安静，在这里，点一杯鸡尾酒，或是选一个清甜的椰子，听着海浪声，这样的时光，宁静而舒适。
时间来到傍晚，在酒店附近其实就是有“ 普吉岛 最美日落”之称的神仙半岛。
神仙半岛有一座山，站在悬崖边上，看夕阳把山和海都染成一片动人的金黄，绝对是极好的享受。在酒店连住两晚就可以提供免费接送到神仙半岛观赏日落。

D6 离开

在 普吉岛 的最后半天，没有再继续外出。就留在酒店，泡一杯咖啡，看看远处奈涵海滩的美景，发发呆，最后来到机场离开。
普吉的美景还有很多，这次来到普吉，也许我只是感受了普吉部分的美好，也许我还留有了不少的遗憾。但美好的回忆很多，包括酒店，包括这里的风景，包括这里的美食，包括这里的热情。下次来到普吉的理由有很多，但其实也许来到这里，并不需要任何的理由。
