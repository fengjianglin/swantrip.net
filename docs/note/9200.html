『 武汉 』匆匆走过，24小时的浮光掠影，一场楚汉大秀

游记 人文游 摄影 特色表演 美食 自助游 2019/03/24
传闻中， 武汉 是一座无论走在哪里，都能找到美食的城市。
近几年来，一直在世界各地到处奔波，却反而忽略了国内的许多著名城市， 武汉 便是这么一个只存在于我印象中的名词。
因为工作的关系来到 武汉 停留两晚，第一天深夜抵达，第三天凌晨离开，匆匆走过，只留下24小时的时光来领略这座城市的气息。
汉派早餐 过早
在 武汉 ，无论是否懂得当地方言，有一个名词是必须了解的，那就是：过早。
来到 武汉 的第一晚，曾经在 法国 留学的学长突然打来电话告知他已回国在 武汉 工作生活，老友相见，正好也让他带我领略一番这座陌生的城市。
还记得他形容 武汉 人时是这么说道：“ 武汉 人过的很休闲，最爱打麻将，但是如果让他们在打麻将和吃之间选一样，必然首选吃。”
可见…… 武汉 人真的都是天生的吃货啊！还有，为什么只有打麻将和吃两个选择？搬砖呢？
言归正传，来到 武汉 ，一定要去品尝一下当地的早餐，即是 武汉 人口中的过早。
在这里，过早已然成为 武汉 的文化。
每一个早晨，走街串巷，吃几样街坊小店中的特色小吃，或许能发现不一般的惊喜。
粮道街 跟当地人一起过个早
每个到 武汉 旅游做攻略的朋友必然看过热干面、豆皮等特色小吃的推荐，旅人们开始纷纷扎进“户部巷”这类网红小吃街道，孰不知真正的地道小吃，却隐藏在那些市井小巷当中，年复一年。
当我问到朋友想吃地道的小吃是否该去户部巷时，他微微一笑，说：“我们 武汉 人都去边上的粮道街。”
是的，就是这条延伸在众多古旧居民房当中的街道。走到这里，随处可见阿姨阿伯们提着大袋小袋的早餐和食材，在一家家店面之间串门，而这，也将是我 武汉 一日游的过早处。
粮道街离住处不远，需经过一段十分破旧的小巷，连如此小道中都设有早餐店，可见 武汉 人对过早是真爱呀。
要说 武汉 小吃文化为何如此深入人心，来看看下图的物价，或许有所明悟。在如今物价飞涨的城市，这里还保持着当年的味道，人们的生活习惯自然也能长久地保留下来。
在 武汉 ，每一条街道上都有各种餐馆，而粮道街更是汇聚了诸多当地著名小吃店。
这里不同于常见的餐厅酒楼，充满市井气息的破旧小馆反倒让我想起了小时的 沙县 小吃和妙香扁食。
这条街道的清晨似乎并不喧闹，很多小店甚至没有招牌，外人无从定位，全凭街坊邻里间熟悉的人儿惦着它们的味道寻去。
一、 天天红油赵师傅热干面
不知道如何形容一家小吃店竟能做到每天都大排长龙
这家远近驰名的网红小店如今每天从7点半起，就已人头攒动了。
从一早开门起，这里关于油饼包烧麦和热干面的吆喝声就从未停歇过，而这两样也是该店的招牌。
== 热干面 ==
每个来到 武汉 的游客都想要尝一尝最“好吃”的热干面，而赵师傅又成了吃热干面的不二之选。对于 武汉 人来说，可以从早到晚吃三顿热干面而不腻，而对我来说，似乎就只是拌面加辣椒油……
看看这排队的人群……而第一条队伍只是等着买饭票的，之后还要排队去领热干面和别的小吃……晕……果断放弃排队……
别问我是怎么不用排队吃到赵师傅的热干面的，外卖小哥了解一下……
== 油饼包烧麦 ==
虽然热干面个人感觉一般，但赵师傅的小吃当中，油饼包烧麦却是值得推敲推敲了。
看着摊好的面团，呲啦一声下入有过，待到咸香金黄之际，放入刚蒸好的烧麦，一气呵成，使得油饼与烧麦共同造就了一道新奇的美味小吃。
== 蛋酒 ==
加了蛋的酒酿圆子，于是有了个接地气的名字：蛋酒，过早时可以配热干面的汤，似乎酒精浓度比酒酿圆子高，不知道喝了这个会不会被当做酒驾呢……
二、 国民豆皮
== 豆皮 ==
来 武汉 前种草已久的小吃，这家小店门面很小，据说这对夫妻已经在此开了20多年了，也算是周边小孩子们最熟悉的味道了。
原以为豆皮就是豆皮，第一次见到还有点难以置信，一层金黄色的鸡蛋皮，包裹着铁锅炒香的糯米，搭配上五香豆干、香菇、肉末，铁锅里煎至焦香，撒上葱花，热气卷起了清香，加上脆皮和糯米的嚼劲，让你吃完还想再撸一碗。
三、 三胖腰子粉
== 腰子粉 ==
沿着粮道街走到接近尽头处，可以找到另一家知名小吃店，看着门前那几个获奖牌子就可得知。
看这名字，这家店老板难道是三个胖子？
餐馆的环境算是比较干净的了，很接地气的传统小店，我是第一个来的顾客，点了一份腰子粉，看着老板娘将腰子和宽粉一起下入沸水中。些许时间过后，腰子转变成了诱人的紫褐色，宽粉也因吸足汤汁而变得柔滑透明。
这家的腰子处理的很好，没有一点腥味，汤汁也特别鲜美，很大一晚，若再加上一盘肉，绝对够顶一份午饭。
除此之外，这条街上还有许多其它著名的小吃，如： 桂林 米粉、 重庆 牛杂粉、鸡冠饺、糊汤粉等等，然而，吃了几样小吃后实在太饱了，不得不放弃心心念念的糊汤粉，挂满了胡椒和鱼鲜的粉条，这绝对是胡椒星人的最爱吧。
昙华林 古道新韵的小文艺
昙华林离户部巷和粮道街都不远，是一片居民楼与商业街混搭的旅游街道，既保留了民国风格建筑，也有许多旧物小店。虽不如其他城市的文艺小清新街道那般人潮涌动，却也自带着一种小家碧玉，宁静的氛围。
或许是早晨，许多小店还未开门，这条街的小吃店不同于粮道街那般市井，而是带着点新潮。
随手拍了张小学，不知道 武汉 的市中心小学是否如 北京 、 上海 那般，周遭都是“名贵”的学区房。
在昙华林，你可以拍照闲逛，然后找一家小店，享受一个带有咖啡香的下午。街道上有不少挺有意思的创意小店，兜售一些手工艺品、旅游纪念品等等。
咖啡馆、创意小店林立，但街头仍有许多当地人摆摊，简易的糕点、早餐小吃，小时候常见的小玩意摊贩，虽然主打了文艺，但还保持着乡土味。
老 武汉 汽水，我承认我是被那条有点娘炮的龙吸引过来的……不同口味的汽水，当做尝试还可以。
被店门上的花猫吸引了目光，走进一看，原来店门口还蹲着一只小猫，爱猫人士表示必须过去撸一把。
看起来凶凶的猫咪，门上还贴着字：“勿撸猫，易伤人”，好吧，看来这猫大爷很牛逼。
如今国内的旅游，无论走到哪，都有那么几个地方，打着文艺小清新的招牌，最后打造成了一条只能用文艺形容的街道。然后无数网红或网不红的小年轻们争相来此打卡拍照，却也忽略了原本应有的慢生活。
不同于我曾去过的那些个城市，昙华林似乎有种不同的韵味，因为它还保留了一些老建筑。这是在如今不停跟风改造的趋势下，还难能可贵地保留了曾经的历史，传统的人文，当地人的情怀，与文艺小清新的概念相融合，反而让人觉得更加亲近。
昙华林周边区域有着悠久的历史，明清年间曾是总督府 和县 衙的所在地，清末又有大量传教士在此传教、办学，从而建造了一批具有西洋特色的建筑。在这里有50多处近代历史建筑，包括了教堂、医院、学校、城墙等，这些历史建筑信息可以在街道一端的导览牌查阅。
户部巷 小吃串串走一波
每一座城市，都有一个属于游客的小吃商业街，户部巷就承担了这一角色。
这样的街道往往打着当地名小吃的旗号，卖着大众都爱的烧烤串串、臭豆腐、烤鱼、炒栗子等常见的小吃，日复一日、年复一年地招揽着一批批游客来此。
去黄鹤楼的路上正巧会经过户部巷，不能免俗的，我也一头扎进了这条人头攒动，只有短短150米的“游客”街道上。
搜索 武汉 的攻略一定会提到户部巷，这里已经被打造成 武汉 最红小吃街，正如 成都 的锦里、 厦门 的台湾 小吃街那样，此起彼伏的吆喝声，排队等候吃小吃的游人们，还有时不时自拍、直播的小姐姐们……
入口处的一幅壁画， 武汉 黄鹤楼和鹦鹉洲大桥曾经的景致，相比于对面普通的游客小吃店，我对这幅图反而更有兴趣。
人山人海是这类小吃街的特色：
烤鸭肠，在 厦门 跟女票轧马路经常看到的街边小吃
这家臭豆腐排队很长，可以看到员工在巨大的油锅里现炸臭豆腐，味道还是不错的，喜欢的伙伴可以买一小份解馋。
一大片的烤猪蹄，加上一些香料和芝麻，看起来还是挺诱人的
来 武汉 ，鸭脖也许也是必点的小吃吧，烤的、卤的、各种口味的，反正自从有了周黑鸭、精武鸭脖这些品牌，鸭脖子在 武汉 人手中已被打造成了人见人爱的小吃美食。
题外话：一定要试试麻辣口味的鸭脖，因为，最不能吃辣的胡建人表示，吃完整整一根鸭脖后，嘴肿了……真的肿了……
居然还有麻辣小龙虾和麻辣鲜虾

湖北 是 中国 的产藕大省，各种未曾吃过的藕制食物，在 武汉 已然被玩出了新高度。抵达 武汉 第一晚朋友便请吃了别地吃不到的藕圆子和炸藕丝，晚上又被拉去吃夜宵喝了一碗莲藕排骨汤，现在在小吃街还能见到藕……
不行，不能整天吃藕，不然ch-ou丑……
小吃街里还有很多卖腌鱼、鱼干的，撒上那一大把辣椒油，看起来很能勾起食欲啊。
朝着黄鹤楼方向走就到了户部巷另一个出口，拍一张门口的特色，至于门的下半部分，很遗憾，只有人头……
黄鹤楼 午后的登高远眺
酒足饭饱，也许该走一走 武汉 的地标性建筑——黄鹤楼。
曾几何时，多少文人墨客在此提笔吟诗，留下了许多千古绝唱。
而如今，几经修缮的黄鹤楼早已没有了当年的传奇。
昔人已乘黄鹤去，此地空余黄鹤楼。说的也不外如是吧
黄鹤楼景区的示意图，如今的黄鹤楼门票已经涨到了80元/人，而许多当地的朋友也都说黄鹤楼公园里没什么可玩的，不需要进去，然作为游客，如此打卡地点若是有闲有钱还是可以进去参观一下的。
曾经赫赫有名的黄鹤楼与晴川阁、古琴台并称为 武汉 三 大名 胜古迹之列，与 湖南岳阳 楼、 江西 滕王阁并称 为“ 江南 三 大名 楼”。记忆里有太多的古诗词来自于此，而如今早已少了当年的文风雅韵，逃不过成为一座打卡景点的命运。
黄鹤楼始建于三国时代吴黄武二年(公元 223 年)，几经波折，多次损毁重修，现在的建筑是于1985年重修的，共5层楼，高50.4米。可以登塔俯瞰江岸全景，若天气晴好，还可眺望汉口。
在 武汉 的时间一直是天气阴沉，据说 武汉 到了冬天都是这样。
汉秀 目眩神迷的魔幻大剧
汉秀是万达集团斥资26个亿倾力打造的一场世界级水上大秀。由全球享负盛名的舞台艺术大师弗兰克·德贡先生亲自执导， 澳门 “水舞间”、 美国拉斯维加斯 “O”秀、“梦”秀均由其策划导演。整场大秀揉合了特技、高空极限运动、舞台剧、柔术、舞蹈等多种表演形式。运用了高科技的舞台设备、世界演唱会级别的声光电设备、全球第一座可移动/升降的座椅设施，是顶级科技与演艺的完美呈现，也是来到 武汉必看的世界级水舞秀。圣诞来此，正值汉秀四周年庆典，庆典上还展示了酷炫的黑科技秀、现场的乐队表演，令人赞不绝口。
我们常说的 武汉 其实很大，分为汉口、武昌、汉阳，一个区顶得上一个 厦门 ，当地朋友常说他们汉口的不去武昌、在汉阳的不去汉口玩。汉秀剧场则坐落在武昌区，与 武汉 大学和 楚河 汉街相邻，与黄鹤楼、户部巷等打卡景点均在一个区，如果只来 武汉 一天，那么日赏黄鹤楼、夜观楚汉秀是值得考虑的行程。
剧场总建筑面积约为8.6万平方米。地上八层、地下四层，通高77米，直径110米。其建筑造型概念源自中国 传统的红灯笼，设计构思极为巧妙，如今已然成为武昌区的一大地标建筑，多少摄影爱好者争相来此拍摄。同时，剧场除了每周都有的汉秀表演外，还时常有外邀剧目，诸如小野丽莎演唱会、 宫崎 骏音乐会、相声表演、巡演等演出。
（下图为官方宣传图，因去的时候周边马路在装修无法直接拍摄全景）
一楼入口处展示了汉秀表演的一些服装道具，其中，汉秀的服饰是由曾荣获奥斯卡最佳服装指导奖、最佳服装设计奖的叶锦添设计完成，每套服装手工打造，融合了现代艺术与“东方主义美学”是艺术领域的无价之宝。在此展示的是仿制品，演员身上穿着的服饰则更加华贵。
汉秀剧场大厅以金色为内饰，整体风格大气而奢华。一楼大厅的金色幕墙上雕刻了包括彩金鸳鸯、曾侯乙编钟、神鸟瑞象等独具文化特色的楚纹饰符号。二楼金色幕墙所雕刻的是三国典故，包括火烧 赤壁、草船借箭等，这些都代表了 湖北 非常悠久的荆楚文化历史。在四周年庆典之际，二楼还有一块摄影展示区。
汉秀剧场的舞台可谓是如今全球最为庞大、最具高科技的舞台，总计2000个观众坐席，是世界上第一座采用移动、升降座椅的水秀剧场。座椅区分为旋转座椅区和升降座椅区。表演开始时，旋转座椅处于闭合状态，在演出过程中旋转座椅打开、升降座椅垂直降下，整体观众席位呈270度开启状态，中间围合出演出水舞台。单此观众席的投资就已是天文数字，只为了让观众在观看过程中能够拥有最好的体验。
四周年庆上的特别节目，剧场乐队现场演出，为了配合演出，达到最佳效果，汉秀的每一场表演音 乐都是乐队现场直接弹奏的。
汉秀中的“秀”，指的是二十一世纪初继音乐剧之后，由 美国 最先兴起的一种新型综合舞台表演形式。通过将音乐、舞蹈、极限竞技、特技动作等多种表演形式的完美结合，再通过尖端科技的呈现出裸眼3D的舞台视觉效果，而此汉秀的表演又融合了 中国 楚汉文化的精华，为观众一同呈现出魔幻酷炫的声光大秀。
表演的全程没有任何语言和字幕，若认真欣赏也能代入其中。说的是一对年轻情侣穿越时空，来到了另一个星球的故事。
热情好客的红脸女王先是举办了盛大的庆典欢迎人间来的使者。
柔术大师丁丁的个人柔术表演，一众演员们的柔术功底亦是无比高超，将个人身体柔韧性的极致开发，并与舞台呈现的相互结合，为我们展现了一段精彩的表演
令人惊叹的体术，即使不停地锻炼都很难掌握这样的臂力 和平 衡感吧。
突然的画风急转，伴随着背景的电闪雷鸣，黑色星球的国王出现，紧接着是从天而降的战鼓，擂鼓阵阵，扣人心弦，仿若回到了战国时代。一群身材矫健的演员悬挂于战鼓之下，在架子上演绎各种高难度的动作。
这一阵战鼓声几乎将观众们的情绪带到了最高潮，随之而来的则是更为精彩的高空跳水，先由战鼓边上的两位小姐姐从10米左右的高度跳水开始，逐渐上升，知道最后的双人27.5米跳水，这可是世界跳水的极限，目前为止，全世界仅4人能完成这样的跳水，而其中两位，就在汉秀现场。
汉秀的演出可谓是高潮不断，刚刚结束了高空跳水，又到了高空芭蕾的时间，美女小姐姐在吊环上跟着音乐展示各种高难度的优美动作，令人目不转睛。
然而，真正精彩的还在于随后出场的悬发女表演。女演员利用天生长发与钢丝相连，完成高难度技巧动作：空中快速旋转、平衡、飞荡由于身体重量依靠头发进行悬吊后会造成头发松动，真真是命悬一线的表演。这样的表演全世界仅有少数人能够做到，而且对头皮损伤很大，每次表演前都要按摩头皮一小时，所以每天最多只能表演2次，是否能看到这样精彩的演出全凭运气啦。
现场的音乐依然急促，随之而来的是一段男女水舞表演，通过钢丝绳的悬挂，演员们时而在水中，时而在空中，不断演绎各种优美的动作。
在这里不得不提的是舞台的干湿切换，水深8.7米的水 池上 方，可以在1秒钟内迅速的干湿分离，如同奇迹一般，水舞台与干区舞台剧场秒速切换，令人目瞪口呆，至今不懂这样的技术是如何实现的。
汉秀剧场舞台上有3块巨型LED屏幕，总面积为225平方米，堪比IMAX巨幕。在演出过程中，LED屏在空中随意转动、飞行、拼接、拆分，将舞台空间放大。牵动巨型LED屏幕的是连接在他们后方的机械手臂，机械手臂展开可达28米。单个机械手臂重达近230吨，是目前世界最大的智能机械臂，可实现360度自由运动。
随着剧情的推进，黑红两位国王相继登场，男女主角的爱情探险也逐渐步入尾声，在异世界星人的见证下又再次重逢。
另一道酷炫的表演便是演出最后的室内水上炫技表演。这是世界上唯一一个室内水上飞人表演和摩托艇特技表演，摩托艇骑士如出水蛟龙，在水上跳跃、旋转、后空翻，引来巨浪翻腾，一个个惊险刺激的酷炫场景，加之裸眼3D的特效，仿若骑士大战蛟龙，引来观众的阵阵惊叹。
表演逐渐进入尾声，男女主角也最终回到了现实，90分钟的汉秀演出，为我们展现出了高科技的声、光特效，超越极限的人体挑战，中西元素的相互融合，舞蹈、杂技与极限运动的完美结合，无论是视觉上，还是听觉中，都是一场极致的盛宴。
演出的结尾，生活依然美好，让我们一起扭动起来。
