==美国亲历感受与故事==

❥钱少胆大办签证

当年，公司让我给客户办美国签证，无头苍蝇一样忙活了好久，客户没办，我倒把自己的给办了。 
美国大使馆门口存包的大姐的一手消息极具参考价值，大姐说：哎哟，你这单身，工作收入也不行，年龄也危险，拒签的几率大，前几天就有一个女孩跟你情况差不多被拒签了，哭得呀昏天黑地的…… 
我呸呸呸！其实我早就去过美国文化中心，亲自问过真正的使馆工作人员，说了自己的情况和疑虑，白人老先生说：No problem. 
房产证、结婚证、邀请函、资产证明、大额存款，一个都没有，但要有备而来。如果面试官问：你单身，如何证明你回国的约束力？答案我都准备好了。我会拿出一张彩色打印的照片，说这是我男朋友。事实上，这张照片没有用上。面试官根本就没问。 
面谈的帅小伙问我为什么要来美国？我当时准备了所有问题的答案，唯独没有考虑这个。我说的每一句话都关系到1000来块钱是打水漂，还是即将踏上快乐的旅程。这是一个带陷阱的问题，就像情人之间问“你爱我哪一点？”我忐忑地说想看黄石公园，好在他没有继续追问。 
明明就是旅游，为何如此心虚？办美国签证的人，犹如千军万马过独木桥！ 
准备了一大袋子的资料，一个也没看，面签出奇的顺利。 
当年是美国旅游一年有效期签证的最后一年，下一年办就执行10年多次的旅游签证，难怪那么容易办！

❥你见过凌晨四点的洛杉矶吗？

每个刚到地球对面的人都见过。凌晨三四点就醒来睡不着。于是起床去室外转转，会发现到处是溜达的同道中人。
∆小旅馆，凌晨四点来换班的人是外籍人

❥多找钱的店员 

早就听说老外的算术不好，没想到在仅仅21天的行程中，至少3个人给我找错了钱，2个人多找钱。在旧金山租自行车，帅帅的车行小伙子说自己是中美洲人，计算器都按不溜，因为太帅，又多找钱，原谅他了，教了他算帐，还给他多找的钱。 
话说，就美国人这算术水平，电商搞品类促销满减，恐怕会把美国人整疯了，还是打折最简单。但是，四大会计事务所有在欧洲的有在美国的，我们老祖宗给的天赋并没有可立足之用，还是恰恰因为美国客户需求太多了而成就了他们的事业？还是因为务实比机灵更有生命力？

❥无忧无虑的黑人司机

 司机是服务行业，开车是无奈的生存选择，各种交通新闻我真不忍看，所以女性开拖拉机都能成为货币图案，美女开货车，更是能多次上得了新闻头条。 
坐在洛杉矶的城市观光车里，黑人司机老哥用那说唱的口音介绍着这里的风景故事，望春风姐姐偶尔听懂一两句，都被这激情的介绍给迷住了。 
路过金门大桥边知名景点，老哥充满激情地说：准备好你们的相机！在停留的那几秒钟，他一连“唱”了好多个“GetGetGetGeGeGe……t------”提醒乘客拍照，可见每天经过无数次的景点在他心中依旧有很高的地位。
黑人老哥天生的音乐感和风趣，让大家在下车时纷纷开心地掏出小费。 
美国胖子多，见识了这红色雨衣展开居然是正方形的，上方中间1/3是开口的，露出头，左右上方各有1/2是开的，伸胳膊的，下方1/2是合的。这么简单却实用！

❥温暖得像上帝的帅哥店员 

 在去羚羊谷经过一家墨西哥卷连锁餐厅，一位店员，阳光帅气的脸真挚的笑容把每个女人都迷住了。我提了不加沙拉酱、番茄酱的要求，一一传达。
国内被网上传得很神的海X捞我只去过一次，一个男店员，面无表情不停地给戴眼镜的人送眼镜布，洗手间站着一位大姐，机械地给每个人递擦手纸，我问有wifi密码吗，都说不知道。我真的不需要那么多眼镜布，也不需要别人来递擦手纸，只需要wifi，却漫不经心地应答。国情无法提高服务业人员的从业状况，所有的贴心服务，都是消费预支的。 
 ∆世界上最弯曲的街道: 旧金山九曲花街，陡坡中有弯，据说车技很好的人没多少能挑战成功 
而在这家餐厅的阳光服务员，让我感受到美国的服务业，原来可以让人活得轻松些。不只这家，很多家店员都能提供周到的服务，在工作之余轻松地聊天。 
梅西百货的门卫都是西装革履阳光热情的帅哥！

 拉斯维加斯的市区和渔人码头，都是街头艺人的聚集地。COSPLAY成经典人物的艺人，打扮丝毫不含糊，小费拿到手软。

❥免费检测轮胎

 汽车的轮胎有一个刮痕，同伴很担心爆胎，一路上找汽车维修站，终于找到了。小哥把汽车升起，查看胎压，又检查了一圈，确定没问题。打算付费时，小哥说不收费。不是说资本主义国家啥都看钱吗？怎么能有雷锋呢？人家好歹忙活了一阵，过意不去，最后买了瓶可乐送他。 
∆一路平安的汽车

❥虚惊一场的夜晚

 洛杉矶中餐馆的小伙说这里治安不好，晚上很早就停止营业。半信半疑地逛到了晚上，果然，晚上7点钟，闹市区的门店都关了，只有路灯亮着，偶尔走过几个流浪者。很胆颤心惊，很担心万一有人拿枪打劫怎么办？美国打劫120元起价！行规是立刻孝敬他20美元，俗称救命20刀。尤其是我带的现金，紧张得不得了。 
圣迭戈入住荒郊野岭的宾馆，是一家几层楼高的小型旅馆，楼梯和走廊都是户外的。晚上大概12点回宾馆，看见一个人站在最高层，我和室友的房间也是最高层，必经过他站的位置。当时很暗看不清那个人，想等他走开再上楼，观察了半天他站那一动不动，只有硬着头皮走上去，魂都快没了，这时他主动打了招呼，我才敢正眼看他一下，原来是1米9多的黑人保安，虚惊一场。

❥渔人码头的贵妇乞讨者

 到了渔人码头已是晚间，同行伙伴开始拍照，一位戴着墨镜身穿长皮草的白净胖白人女性，手里还拎着几个购物包装袋，主动提出帮大家来一张合影。原来美国也有这么乐于助人的人啊！欣然接受，说了好多遍谢谢。没想到她说：能给我2美元吗？我还没吃晚饭。
哦，见招拆招，没吃晚饭，我请你吃吧，对她说：你喜欢吃赛百味（subway）吗？她说为什么去地铁（subway）? 你要回旅馆吗？趁着我这英语给她弄歧义的时候，仔细打一下算盘：景点的赛百味一个5、6美元，带她去吃，比给2美元亏啊！正巧包里有面包，拿给她，同伴也拿出苹果和饼干，终于打发走了。 

❥新兴的拉斯维加斯

 灯红酒绿的都城拉斯维加斯本不存在的，这里原来是沙漠，只不过停留的人多了，也就建起了城。 
当天还有个美国小明星路过，我也不认识。整个城里充满了荷尔蒙和金钱的符号。脱衣舞也看了，门票很贵，可是时差很严重，我已经睁不开眼睛了，掐手都不好使了，可惜全价票没看完全场。 

❥白人世界是否有缘分天空？

 此行一共有三位女性，一位有老公有房有车，另一位北京有房，而我，有流浪的心。都开玩笑说你在国内嫁不出去，不如在这儿偶遇个whiteman，再拿个绿卡…… 
都觉得白人高人一等，都认为中国传统价值观不喜欢的女性，找个老外是最佳的着陆点。 
“我纹身、抽烟、喝酒、说脏话，但我知道我是好姑娘。”——美国 
我不纹身，不抽烟，不喝酒，不说脏话，只是靠自己......貌似我不是好姑娘——在哪儿，你懂的
白人还真的以自己为傲，有个产品名字还叫whiteman呢。 
∆白色车斗的绿色logo: Whiteman
那么，如何帮助我在匆忙的旅途中结识到whiteman 是其他四个人的旅行额外任务。当年，我的手机还是诺基亚老款，还没有微信，也没有whatsup。同伴还热心地在老手机上居然鼓捣出了whatsup，可惜我当时没钱买美国手机卡，是不是不舍得芝麻结果套不住西瓜，错过了白种人的缘分呢？ 

❥白人，不过如此

 整体而言，美国人素质的确会高很多。二三十美元一晚的小宾馆的前台也非常热情。路人会用非常标准的慢速英语、非常耐心地帮你指路。 
看起来人畜无害彬彬有礼，却有着天然的优越感。在加油站上厕所后，店员会问：“你冲厕所了吗？”而不会问自己国家和其他国家的人。 
仅仅绿灯亮了我坐才车没有及时开走，一个白人男性车主瞬间爆发了路怒症：“Fuck you！”像极了美剧里的歇斯底里场景。 
在洛杉矶的观光车停下等红灯时，一个站在路边的白胖子，狠狠地朝观光车吐了一口痰。这样的人，也许在哪里都有差不多的概率。 

❥2起工人游行

 仅仅21天的旅行，就看见了2起工人游行。一家酒店门口，一排排工人举着牌子齐声喊着口号。另一家施华洛世奇门店前，也是工人在举牌。居然没有城管来制服，有时候美国很危险，有时候美国很太平。
感谢一路同行的伙伴们和汽车！三个人轮流开车，我洗车，另一个负责唠嗑，不让司机打瞌睡。

📷环球影城

位于洛杉矶寸土寸金地带，原本是好莱坞电影的拍摄片场，如今已成为一座主题乐园。
好莱坞电影与先进的特效相结合，打造出极具尖端科技的世界顶级游乐设施。
侏罗纪公园、木乃伊复仇过山车以及变形金刚、小黄人等4D乘骑设备，将游客带进虚拟时空中，体验惊险刺激的历程，直到游戏结束后才重返现实。
最具特色的是影城之旅，观览车带领游客穿越真实的好莱坞电影拍摄场，亲身感受空难、洪水、爆炸等特殊场景的拍摄过程。
乐园内每天会上演各种表演、可与随处可见的好莱坞人物合影，购买热门的周边，在环球影城中度过精彩绝伦的一天。

📷九曲花街

官方名称为伦巴底街，它是世界上最弯曲的街道。在花街高处还可远眺海湾大桥和科伊特塔。
市政府为了防止交通事故，特意修筑花坛，车行至此，时速不得超过5 英里，如今成为旧金山最吸引人的一条街。
在Hyde 街与Leavenworth 街之间的一个很短的街区内有八个急转弯，40 度的斜坡加上弯曲如“Z”字形的道路，可以来这里检验一下自己的驾驶技术，或者惬意地在其中漫步。
车道两边的花坛里种满了玫瑰，花开时节， 远远望去，有如一幅斜挂着的绒绣，美不胜收。
用时参考 1-3小时

门票 免费

📷马蹄湾

马蹄湾（Horseshoe Bend）是科罗拉多河在亚利桑那州境内的一截U形河道，也是格兰峡谷其中一小段，由于河湾环绕的巨岩形似马蹄，故名“马蹄湾”。也有人叫它科罗拉多河的Ω大拐弯。
想拍到马蹄湾的全貌就必须站在对面的悬崖上，但在这里一切都是纯天然的，没有门票、没有工作人员，当然也没有你想象中的护栏，所以游客们都非常小心地在悬崖边挪动着脚步，甚至没有人大声说话。马蹄湾的景色是千变万化的，正午的时候往往是一湾碧水，而到了傍晚，河水则会映射出四周岩壁的颜色，深邃中更具魅力。
电话 +1-928-608-6200
用时参考 1-3小时

📷渔人码头

渔人码头在旧金山家喻户晓。包括从吉拉德里广场开始一直延伸到35号码头的区域。
可以看到最热闹的39号码头的各色商店、餐厅、街头表演，也可以远眺远方欣赏恶魔岛、金门大桥、海湾桥的各色风景。
每年11月到次年6月之间正是品尝海鲜的最佳时节，这时候来到渔人码头，可以吃到上好的丹金尼斯螃蟹。
对于热爱购物的小伙伴，渔人码头不仅能够看到各种特色商品，还可以买到富有当地特色的纪念品。
在39号码头可以看到众多的海狮懒洋洋地躺在木筏上晒太阳睡懒觉哦。

📷黄石国家公园

黄石公园位于美国中西部怀俄明州的西北角，是世界上第一座国家公园，是美国国家公园里景观组合最丰富的国家公园，这片地区原本是印地安人的圣地，但因美国探险家路易斯与克拉克的发掘而成为世界上最早的国家公园，并在1978年被列为世界自然遗产。在公园中，可看成群的野生动物、奇妙的的风光、险峻的大峡谷和化石森林，让人着迷，让人流连。

1.搭机前往蒙大拿州毕林斯市(Billings)，走212号公经由被誉为全美国最美丽的公—熊牙公(Beartooth Highway)，距离黄石公园东北口约208公里。
2.搭机前往蒙大拿州波兹曼市(Bozeman)，走90号公距离北口约127公里到Gardiner，走191公到达西口约145公里，北口与西口都有很多的旅馆可供住宿。
3.搭机前往怀俄的科迪市(Cody City)，距离东口约85公里。
4.搭机前往州的盐湖城(Salt LakeCity)，再乘巴士从南口进入黄石公园，约628公里。
自驾注意事项：在黄石公园内驾车，除了要注意交通规则外，还要特别注意路两旁的动物。园内的限速是45mph（72.5公里/小时）。
∆黄石国家公园里一家酒吧的洗手间标志
门票 黄石国家公园通行门票参考（以下票种均为7日内有效）：私人/非商业车辆30美元/辆，摩托车25美元/辆；徒步游人，以及骑自行车，滑雪等15美元/人；此外也可以选择选择购买和大提顿国家公园的联票和全年通行证，更多票务信息和每年的免费入场日公告可在国家公园管理局官网查看。
开放时间 夏季一般为全天开放，其他季节、部分区域道路因气候等原因开放时间有所不同，建议提前查询官网。

📷大峡谷

大峡谷国家公园是世界七大奇景之一，位于亚利桑那州的西北部。
大峡谷国家公园分为南峡、北峡两个大部分，各有特色。(西峡属于印第安人保护地，并不在国家公园的范围内)
其中南峡是游客访问最多的路线，其自驾公路也修建得相当完美；北峡海拔更高，空气稀薄，但天空碧蓝如洗。
虽然大峡谷不是世界上最深的峡谷，但如果论起峡谷的壮阔程度和丰富多彩的景色，这里当仁不让可以称为世界第一，是美国最值得一看的国家公园之一。
电话 +1-928-638-7888
网址 http://www.nps.gov/grca/index.htm
用时参考 1天
交通 【长途交通】
机场：大峡谷机场(GCN)位于南入口处以南13公里处。每天都有航班同拉斯维加斯、菲尼克斯等附近各大城市相连接。从机场到景点之间有Tusayan-Grand Canyon shuttle公共汽车和出租车。
铁路：大峡谷铁路(Grand Canyon Railway)从威廉姆斯往返大峡谷南缘中心，每天两班，沿途风景很棒 (电话：(1-303)843-8724 网站：http://www.thetrain.com/ )
长途汽车：Arizona Shuttle提供从凤凰城、威廉姆斯和弗拉格斯塔夫等地到达大峡谷南缘的巴士
(电话：(800)888-2749 网站：http://www.arizonashuttle.com/ )
报团：可以在拉斯维加斯报名当地的旅行社，参加一日游等项目
【园内交通】
长途巴士：TRANSCANYON SHUTTLE长途旅游汽车5月中旬至10月中旬运营，是徒步游客往返南北缘的理想选择，需提前1到2周预定
(电话 (1-928)638-2820 网站：http://www.trans-canyonshuttle.com/ )
蓝线巴士（Blue Route）：连接园内Grand Canyon Visitor Center、Grand Canyon Village、其它几个野营点与住宿点；
红线巴士（Red Route）：基本线路，通往Hermits Res；
黄线巴士（Yellow Route）：Grand Canyon Visitor Center起点，可通往Yaki Point和Kaibab Trailhead两个私家车不准通行的景点；
紫线巴士（Purple Route）：从Grand Canyon Visitor Center经过南入口通往Tusayan；
登山快线（Hiker's Express）：为清晨登山者专备，五月和九月有早晨5、6、7点三班，六、七、八月有早晨4、5、6点三班，从Bright Angel Lodge开出，先到Backcountry Information Center，经Grand Canyon Visitor Center，最后一站到South Kaibab Trailhead。
【自驾提示】 如果要选择自驾租车，只能选择在拉斯维加斯或者是洛杉矶机场租车
门票
私家汽车门票:30美元；摩托车门票:25美元；行人和自行车门票:15美元 (1月1日-12月31日 周一-周日)
免票:15岁以下免票

开放时间 全天
tips: 南缘全年全天开放 北缘5月中旬—10月中旬开放

📷拱门国家公园

--拱门国家公园在犹他州莫阿布北部，是世界上规模最大的拱门状自然砂岩集中地，这里共有大大小小的砂岩拱门超过2000个。
--精致拱门是这里的镇园之宝，也是犹他州的标志，犹他州的车牌上都印着它的形象，02年冬奥会圣火传递还曾在拱门下通过。
--除各式拱门外，公园还有尖塔岩、平衡石等众多地质奇观，这些地质地貌早在3亿年前就已形成，甚至直到现在新的拱门仍在持续形成。
--建议花上一天时间徒步欣赏尖塔岩、窗口、魔鬼花园和精致拱门四个大区，黄昏时分还可以透过拱门欣赏壮美日落景象。
--游览公园的最佳时间是春秋两季，需自己带足食物，在入口处还可以向工作人员要一份中文简介以便细致了解和探索。

   ②火车：可乘坐美国国家铁路客运公司Amtrak的火车从大章克申或绿河到达莫阿布。
   ③其他公共交通：在大章克申和莫阿布之间有Grand County Travel Council公司提供的往来Shuttle车。
