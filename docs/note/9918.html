随歌诗达大西洋号巡游中国南海，五岁娃的邮轮初体验

游记 家庭游 小众 邮轮旅游 2019/01/06

前言

桃桃大王五岁了，是时候跟着他的旅行爹和美食妈一起闯荡世界了，之前尝试了两次港澳亲子游蛮顺利的，于是赶在双十一时订下去往日本的机票，精心计划着他的第一次出国旅行。然而，忽然接到一个邮轮体验师的邀约，四天后出发的歌诗达大西洋号菲律宾苏比克之旅，更诱人的是，同行人也享受体验师的价格，当时想都没想直接就报名了，顺带帮媳妇、老丈人以及桃桃大王也一起报上。于是桃大王的第一次出国提前了，由日本变成了菲律宾。

关于我

大家好，我是蕃茄哥哥，一名手持高压电工证的奶爸旅行家，目前是去哪儿网的聪明旅行家、飞猪旅行达人、天鹅之旅旅行大玩家、驴妈妈旅行达人、大众点评LV7美食点评与写手、去哪儿网试睡员、花筑酒店试睡员，yhouse生活大师与风尚达人，有经营多个自媒体平台，曾经踏足欧亚澳三大洲，有国外自驾游经验，平时的爱好就是码码字、修修图，分享一下自己的生活和心情。

行前准备

1、有效期内的护照、身份证。
2、除了日常换洗的衣物，最好再自备一套正装，以备出席船长晚宴。太阳镜，拖鞋、泳衣、泳镜、泳帽以及毛巾。
3、一些可以消磨时间的小玩意，例如一两本书或者是扑克牌。
4、相机、手机，我还带了带三个USB口的小米排插，事实证明真是太明智了，船上的插头虽然是通用的，但是三部手机、一台相机以及充电宝等其他用电器，真的是不够用。
5、牙膏牙刷记得一定要带，船上没有。沐浴露洗发水房间是有提供的，当然也可以自己带。
6、剃须刀，防晒霜等个人用品
7、晕船药，晕船的同学们最好自备晕船药，船上二层服务台也可以免费索取，但是本人没有证实。
8、一张有VISA标志的信用卡，这很重要，用以绑定房卡以支付船上的开销。
9、儿童睡袋（邮轮房间的空调开的贼冷）

邮轮费用

主要由舱位费 + 港务费 + 小费 + 其他费用四部分组成1、舱位费按照房型分为内舱房、海景房、阳台房以及套房。其中每种房型又有双人房、三人房和四人房之分。不同的房间价格不同，和酒店不一样，这里是按照人数收取，不是按每间房收取。2、港务费是邮轮停靠码头补给需要支付给码头的费用，由邮轮代收，一般是固定的。有一些旅行社会以赠送港务费的形式吸引客户，其实只是把房间和港务费进行捆绑销售，在房费上做了折扣。3、小费以每人每天结算，不同的邮轮公司小费标准不一样，歌诗达邮轮规定内舱、海景、阳台房型 每人每晚14.5美元，套房每人每晚17.5美元，儿童：4-12周岁减半，不足4周岁免小费；邮轮上绑定信用卡后每日夜间会自动扣取。4、其他费用：游轮上的私人消费（打电话、网络、洗衣服、购物、赌博、付费餐厅、酒吧咖啡厅消费、SPA、拍照以及参加活动等等）签证费，以菲律宾为例，提前办理大约200人民币，到邮轮上加急办理则是56美元。临时报名的活动，主办方通知来不及办理签证不能下船了。后来了解到拥有有效期内美国签证可以免签进入菲律宾，刚好去年全家办理了美国十年签，咨询了一下没有问题，只有老丈人一人没有美签留在船上晒太阳。陆上观光费（跟团的话很多都是包含在房费内，自由行随性的多，自己掌握支出。）

歌诗达邮轮大西洋号

歌诗达大西洋号邮轮（Costa Atlantica），隶属于欧洲最大的邮轮公司歌诗达邮轮，2000年建造完工，全长292.5米，宽度32.2米，最大航速24节，总载客量2680人，拥有客房1057间，有15层，底层A、B、C为船员工作室和休息室，其余为客房、功能厅和露天观景台，顶层拥有三个游泳池和一个小型足球场。

接下来就进入正题

DAY1 登船日

登船时间是14时，集合时间是12时左右，建议提前到达，早早吃完午饭便打了个出租车前往厦门东渡邮轮中心码头，邮轮在11号门，这里和去鼓浪屿的游客是分开的，不过上卫生间和接水则必须到6号门那才有。11号窗口已经在办理登船手续，秩序并不是很好，一堆大妈大声着喊着挤着。从候船室远远地可以看到歌诗达邮轮那标志性的黄色大烟囱。
登船手续很简单，递交预定的护照、核对姓名后，发给白色的登船卡和黄色小纸片各一张（团队出行没有黄纸），便可以到登船口排队了。关于行李，26寸以上的行李箱可以选择托运，26寸以下的必须自己提着上船。第一关是出境检查，大体禁止携带的物品和出国是一样的，有人书包里带了一瓶葡萄酒被没收了。之后是海关，海关人员在护照空白处盖戳放行，之后需要自己拉着行李走一段路到邮轮的泊位。中间还有上下扶梯，一定要带好孩子并注意安全。
邮轮一侧悬挂着悬梯，并且也有自己的一部安检设备，过了这关，便来到了邮轮内部，也就正式开始旅程了。船员们有个简单的欢迎仪式，帮你拍拍照片，和“船长”、美女水手拍拍照片等等，当然这些照片之后可以在三层看到，自愿付费购买。刚上邮轮先回房间把行李放下，顺便参观一下海景房。给了两个海景房，格局是一样的，朝向不同而已。床上整齐的被单上放着房卡、注意事项以及今日的日程表，稍作休整，广播便传来，所有人请携穿着救生衣参加演习，而且是必须。安全着想，全家人穿上救生衣带上房卡跟着人流去到三层外甲板集合，船员刷房卡签到（没有参加的话，下午会要求补参加），帮你整理救生衣，全程还是蛮严肃的，不允许使用手机。讲解员讲解逃生要领和救生设备使用方法后，会让大家自己操作一遍，直到都掌握了逃生技巧，才被允许放回房间。
第一天的任务就是熟悉邮局上的格局，以活动的主要路线，在这之前还得先去二层的服务台绑定信用卡，这很重要，船上的一切消费都是以美元结算，通过房卡支付，而房卡则需要与一张有VISA标志的信用卡绑定。换句话说，在船上，只要你想买东西，出示船卡即可，当然如果不急着支付，可以等大家都绑定好了再去 ，避开高峰期。
服务台就在二层的蝴蝶夫人酒吧边上，这里也是类似房间大厅的功能。蝴蝶大厅有三部观景电梯，在壁画中穿梭，就像真的到了意大利，非常梦幻。
拿着地图一边走一边逛，先去了二层船头的卡鲁索剧院，这里正在举行登船说明会，卡鲁索剧院是一个有三层船舱那么高的剧院，前面有个大舞台，所有歌唱类、舞蹈类、杂技魔术类演出都会在这里举行。说明会也会穿插一些意大利语教学，还是蛮有意思的。
在二层逛了一会竟然走进了提香餐厅，吃货的直觉真是可怕。提香餐厅是一个综合的点菜类餐厅，进入后服务员会安排座位，并且提供菜单，不同的是，菜单上除了特别著名收费的餐饮，大部分都是免费的，免费什么的最爱了，邮轮上的第一餐就在这里解决吧！我们把能点的都点了一遍，期间会有工作人员过来推销200多元人民币一份的波士顿龙虾，赠送一瓶香槟，其实性价比也是蛮高的，如果不是点了太多的东西，还真的有点心动。
吃饭时偶遇了邻座一位三岁的混血小男孩，当我们还在担心桃大王怎么应对的时候，他们已经使用”婴语”聊的不可开交了，其实混血小男孩可以熟练地使用中文，这真是太棒了。
邮轮一共有四个餐厅以及顶层用来承办星空晚宴的露天餐厅。四个餐厅分别是二层、三层的提香餐厅（其中三层是为套房客人专用服务的）。九层的波提切利自助餐厅（中间有一部分区域提供海底捞，是付费的），十层的意大利餐厅以及十一层的月亮餐厅。那么重点了解一下二层和九层餐厅就好，因为只有这两个是免费的，哈哈。记住二层是点餐的九层是自助的便行，二层会比九层多了一些仪式感。餐厅24小时无限量供应，不过每个时段提供的食物不一样。接水的话可以到九层接，有热水、茶还有咖啡，船上的水有一个味道，应该是净化水的原因，不影响饮用。值得一提的是，邮轮上会有船长晚宴（需要正装出席）、尊贵主厨晚宴、星空晚宴以及浪漫晚宴等等主题晚宴，可以酌情付费参加。吃过晚饭天色已经暗了下来，到顶层吹吹海风，意外地发现了这里还有一个足球场，球场有尤文图斯的标志，原来歌诗达和意甲尤文图斯足球俱乐部有合作，除了在邮轮上开办培训和体验教学，还开设了一个尤文图斯博物馆，展出和俱乐部相关的图片文字历史，而三层的商店也有售卖尤文图斯的球衣和纪念品。
大概溜达了几圈，媳妇带着娃儿回去睡觉了，我便独自去二层看表演，这里正在进行欢迎晚会，由巴西和意大利的DJ引领大家土嗨，大妈们都跳得异常兴奋。值得一提的是，登上意大利的邮轮自然少不了意大利元素，仔细逛逛会发现很多，例如披萨和男高音，这位保罗先生可是这艘船上的明星，洪亮的一首“我的太阳”把全场气氛推向高潮。
当然，也少不了斑马军团和足球教练的颠球表演秀。
稍微坐了一会便回客房了，因为孩子的缘故，我只好抛弃老丈人和他们母女两一起挤个房间了。客房除了大床还有个长沙发，沙发的下面有两床预留的被子，我就这么强行加床了。应该说，歌诗达在客房服务方便还是想的很周到的。客房服务员会在每天我们不在房间的时候收拾好客房，并且将第二日的行程单放在床上，晚上的时候又会把首日的消费结算单放在门边上的收件箱。船航行到台湾附近，中国移动的信号就消失了，自动连接了国际海事信号，听说这个打电话上网非常贵，而我们又没有买流量，避免孩子误操作，默默地把手机挂成飞行模式。第一晚邮轮非常平稳，伴随着海浪的声音，徐徐入睡。

DAY2 中国南海我来了

今天依旧是海上巡游，一大早醒来跑到阳台，祖国的南海一片蔚蓝，茫茫的大海一望无际，如果不是海风大了些，我想我应该可以坐在这发呆很久。当然，习惯了网络生活的我们一旦离开了网络，还真是非常枯燥，如果需要上网可以去前台购买网络，有即时通讯的套餐，也有全功能的，都非常贵，听购买的前辈说，买了即时通讯的，也是非常慢，所以放弃了，有位旅行社的小姐姐偷偷告诉我，支付宝的通信功能是可以用的，一试，还真可以，至少可以和家人保持联系了。
就这样坐在阳台晒太阳，看着茫茫大海，偶尔有个货轮经过都会兴奋半天。当然，对我这样的人来说，还有一种解闷的方法，那就是吃，万恶的九层自助餐厅，总是24小时不间断推出各种美食，做的精致又诱人，走过去就想吃，走过去就想尝，回家生生胖了四斤，这是后话。
用过早餐，老丈人便领着孩子去顶层的泳池游泳了，大西洋号虽然有三个泳池，但是每个都不是很大，孩子玩耍刚刚好，不知什么原因，滑道一直没有开放了，有点遗憾。而我便和媳妇抽身去十层的甲板拍照了，阳光明媚的话应该可以美美的照片，我这渣技术，大家凑合看哎。
五层的思高儿童俱乐部，也是一个遛娃的好地方，可惜当天是付费的小船长活动，所以没有参加了。这里需要说明的是，思高并不是24小时开放的，而是分时间段，早上，下午和晚上各一次，需要看好开放时间再过去，免得白走一趟。当然有时候也会有付费的活动，行程单上都会注明。应该说，思高的时间段并不好，特别是晚上八点半开门的那场，有早睡的小朋友基本上赶不上，比如我们家的桃大王。五天时间里只参加了一次自制服装的活动，不过蛮有意思的，来自马来西亚的小姐姐很有亲和力，小朋友很喜欢。
晚上娃睡了后，依旧一个人去了二层，玫瑰夫人酒吧总是有不同的表演，欧洲节奏大师演奏以及亚洲歌手献唱。而今天酒吧刚好有活动，100人民币左右，指定的啤酒饮料任饮，这下人更多了。
回到房间，发现门上的收件箱除了第三日的行程单，还多了两张蓝色入境卡，这就表示明天可以上岸观光了。

DAY3 苏比克，你好

大早听到锣鼓声，拉开窗帘忽然开到山，心想莫非是到 苏比克 了？走到阳台，果然看到了悬挂 菲律宾 国旗的引航船正指引邮轮靠港，而码头上的 菲律宾 人民正在载歌载舞表演节目，看看时间才6点多，赶紧把一家老小挖了起来，去了九层吃完早餐，便可以从二层的悬梯下船了， 菲律宾 的小孩子看到我家小桃子都围过来和他互动，把这个第一次出国的小伙子吓懵逼了。
先介绍一下苏比克港。作为前美军在亚太地区的重要海军基地，苏比克港一度热闹非凡，这里的房屋道路都是美式风格，被称为亚洲最像美国的地方。随着1991年皮纳图博火山爆发，美国海军基地报废，美军被迫撤兵，菲律宾借机要回了苏比克港的所有权，并建设成如今的苏比克自由港，曾经停靠航母的码头如今用来停放大型邮轮，只有远处两艘美国赠送给菲律宾的战舰还在告诉大家这里曾经是一个海军基地。
而苏比克港紧邻奥隆阿波市，从邮轮上远远就可以望见市中心的SM商业广场。估算着步行过去大概2公里远。那么苏比克玩什么？这里有一个大型野生动物园以及海洋公园，与香港那个海洋公园是同一家，但是游客不多。
码头上有几个摊位，可以索取免费的地图和咨询，出租车公司的代表也会详细讲解打车的事宜。稍微看了一下海洋公园和野生动物园的位置，再看看出租车的价格，果断放弃了。一来是怕时间不够玩的不尽兴，二来是之前做了攻略说这里的海洋公园并不是很理想，性价比不高。那就随便去镇上走走逛逛吧，这也符合我们这样的人，喜欢了解当地人生活，把苏比克闲逛权当美国旅行之前的一次预演吧。
关卡很随意的样子，几个手持枪械的阿兵哥把门，只要看到手里蓝色卡片便可自由进出，先去边上写着“SUBIC BAY”几个大字的地方打个卡，毕竟是网红打卡点，这样的字母雕塑有两个，另一个在远处的沙滩。边上还有一些雕像，也是蛮有特色的。从码头步行到SM商业广场，路上的人不多，房子有些陈旧，但是不会不舒服，过了一座桥，人便多了起来，行人很有秩序地分为左右两边行进。
到了商场，发现菲律宾一个很有意思的地方，十点开门，还没到点，所有人只能在一层等待，进入商场需要从指定的入口进入，有持枪的警察还是保安的负责安检，气氛相当紧张。十点一到，警察叔叔吹响口哨，这时候才可以进到里边的任何一家店里，当然更多人是杀向超市，而超市出口竟然也有持枪警察，这个国家出门逛街都搞得好紧张，超市的物品物价大概和国内差不多。
超市出来便到附近随意逛逛，在麦当劳和菲律宾大妈闲聊，大妈以为我是日本人，得知我是中国人的时候，大妈竖了个大拇指，咦，说好的菲律宾和中国的争议呢？看来是我想多了。街上到处可以看到这个花栗鼠头像的西式快餐店，看着挺好吃的想买，可惜没有换当地货币。临时被同意下船没有兑换当地货币，又不确定能不能使用美金。媳妇提议，趁着大家都出去了，赶紧回去船上再拍几张美照，顺便去二层餐厅找个好位置饱餐一顿，还省钱。说到吃，三个人一致同意了，反正我就这么被忽悠回去当摄影师了。
趁着船上人少，又拍了一波游客照。
傍晚跟团去旅行的人们陆续回到船上，邮轮随即要离开苏比克港踏上返航的征程了，刚驶出港湾不远，忽然风浪大作，邮轮摇晃得非常厉害，当晚广播取消室外的运动项目，原本还计划参加足球训练营的，只好带桃大王去邮轮内部探险，顺便看看表演什么的了。邮轮的三层其实也是我比较喜欢的一个地方，穿过史皮卡购物大道，这里有一个弗洛里安咖啡馆，是按照1:1的比例从意大利的威尼斯复制到这首邮轮上的，这对于去过威尼斯的我们也是莫大的吸引力，就像故地重游一般，门口的玫瑰花墙和摆放的贡多拉就值得花上一些时间拍照留念。每到固定时段，还会有一对外国夫妻在这个咖啡厅演奏欧洲古典乐器，据说他们已经在邮轮上演奏了几十年了。
三层再往前走依然是卡鲁索剧场的中层看台，刚好有表演，便坐下来欣赏一番，第一次感觉桃大王会喜欢这些歌唱和音乐剧，每一幕结束小朋友也会跟着观众鼓掌，看来孩子真的长大了。这场剧约有一个小时，几个表演都非常精彩，除了保罗的经典男高音，还有来自异国的俊男美女奉献的时代舞蹈，十分养眼，整台节目的水准也是非常高的，杂技表演的难度可是比陆地上难度还要高， 要知道船可是一直在摇晃，保持并找到那个平衡点，难度非常大。
看完表演回房休息，由于船颠簸的厉害，走路都很不稳，一直有撞墙的感觉，最痛苦的莫过于睡沙发上的我，海边长大的我从来都没觉得晕船这么难受，这次真真切切地体会了一把、尽管如此，还是在一阵“挣扎”之后睡下来。第二天到二楼前台一看，好多大妈围着前台要求换房，说晕的厉害，哈哈。这里教大家一些小绝招，头晕的时候除了吃晕船药，还可以嚼一些干性饼干，但是最好不要喝水。还有房间的眩晕感会比在公共场所强烈，可以试着去蝴蝶夫人大厅、咖啡厅、或九层等空旷的地方坐着休息一会。

DAY4 风浪中返航

很庆幸桃桃大王并没有晕船反应，小孩子也许天生就是一种不会晕船的生物吧，反倒是我们有些状态不佳。昨晚尝试了免费送餐服务，将送餐的牌子填好挂了出去，一大早果然很准时就送过来了，今天就享受了一把在房间用餐的感觉。
吃过早餐后，先去参加了离船说明会，说明会除了讲解离船的注意事项，也会介绍船员们的工作，感受他们的辛苦，如果没有什么事情，还是蛮推荐来参加这个说明会的。在这一天还有一个威尼斯狂欢节，以前去意大利错过了每年二月份的威尼斯狂欢节，这次终于可以在邮轮上得偿所愿，船上被布置上各种彩带和彩旗，所有的船员都穿上华丽的衣服，戴上威尼斯面具，小朋友也在思高工作人员的组织下领到面具，穿上自己制作的衣服，一起狂欢舞会。
由于风浪太大，安全原因室外设施都围了起来，甲板也关闭不让出去了，今天的主题还是探险为主吧。二层有个赌场，去澳门的时候桃桃大王被赌场保安拦了下来，这次是真真切切地进了赌场了。这里看戏的永远比主角多，只有寥寥无几的赌客在这里试试手气。
四层的秘密花园和小教堂也是我最喜欢的一个地方，入口有点隐蔽很少人会发现，也没有大吵吵闹闹的大妈和打麻将的大爷，非常安静闲适，坐下来休息思考人生看看书什么的也是极好的。教堂在秘密花园的尽头，同样是人迹罕至。
风浪这么大，九层的意大利大哥依然很尽心在为大妈们服务，带领他们做本次旅程最后的土嗨。
当晚，邮轮特别安排了欢送晚宴，地点在二层的提香餐厅，记得一定要早点过去。餐厅的工作人员会为大家带来一场别开生面的互动演出，效果非常的好，全场嗨爆。就连餐饮部的工作人员都在尽力逗乐大家，更何况其他部门的工作人员了，我想所有人都会对这场大秀永生难忘的。当然餐食也是非常独特的一顿，餐厅被布置成红绿白的意大利三色，已经排好的菜单，大家只需选择A餐还是B餐即可，这一顿也是免费餐饮中最豪华的一顿。值得一提的是，最后的那颗冰淇淋超级好吃。
明天就要离船，如果有托运行李的乘客，可以将行李打包好在入睡前放在房间门口，绑上提前分发的行李条，凌晨服务员会取走办理托运，规则依然是26寸以上的箱子可以托运。除此之外，每个旅客还会拿到一张不同颜色的纸条，这是下船时用来区别下船批次的，不同的颜色有不同的集合时间和集合地点。收拾好行李，便可以入睡了，当夜依旧晃得很厉害，有人说是因为不远处形成了一个台风的缘故。

DAY5 再见，梦幻海上之旅

一早醒来，发现船不晃了，拉开窗帘，赫然看到熟悉的海沧大桥，朝阳洒向海沧湾，将对面的高楼映得通红。厦门，我们回来了。将妻儿叫醒，洗漱完毕，便带着行李去九楼用餐。老丈人已经用完餐在那里和人闲聊。吃了好几天印度厨师做的中餐，忽然觉得早餐的稀饭特别好吃。伴着晨光，一边欣赏风景一边用餐，感觉特别好。
我们的纸条是粉色的，按照时间到一层的珊瑚大厅集合，跟着人群下船，热情的船员看到桃桃大王都过来同他道别，看得出小伙子还是有些难舍，下船后同我约定下次还要再带爷爷和阿嫲来坐邮轮。跟着人群，一路过安检，过海关，便又回到侯船大厅，门口就是出租车等候点，非常方便。搭公车的话则需要下一层。五天四夜的航程，非常适合带老人和小孩出游，邮轮上的服务非常好，吃的也挺ok，唯一不足就是小孩子的娱乐项目还是太少一些。而作为目的地，苏比克缺少一些亮点，略显单调，当然，作为一家意大利的邮轮公司，在邮轮上额外体验了一把意大利风情，重温在意大利的美好时光，倒是让人拍手称道，一趟航程感受两种文化，还是蛮超值的。
现在越来越有信心带娃出游了，三月，日本再见。
