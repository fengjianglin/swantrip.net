张家口——爱上一匹野马，想拥有这片草原

游记 摄影 精品酒店 美食 自然奇观 自驾游 2018/11/07

向往青青草原，骑上白马，策马奔腾

微风徐徐，细水长流在炎夏之中更是给人沁凉的感觉;绿意盎然的树叶带来清新的空气，摆脱了城市间的尘埃与烦恼。

远处骡马静静地啃草，满眼的绿色、野花的芬芳、牛粪的气息描绘着草原和山林的粗犷原始。由于地广人稀，这里安静得出奇，我甚至能够听见自己的心跳声。逃离了大都市的浮躁喧嚣的我们在这样的环境中倍感舒适惬意。
没有堵车，没有污染，景色绝佳，草沿天路如果不是世外桃源，那什么地方是呢？！
这里四季分明，光照充足，是观光度假的理想之地。密云向东向西，高差悬殊，景色各异。春日杜鹃映冰，夏季漫山碧透，秋来层林尽染，冬至冰雕玉砌。

自驾出行


好处：

1、尽可能的游览更多你想去的景点 ,自己决定行程，决定去哪个地方，决定哪个地方停留的时间，自由度更高。

2、随时可更改行程 。

3、能领略到更多当地文化风土人情，结交朋友 自驾游会有更多的机会和当地的人交流，或者是和同样在旅途中的朋友偶然相遇，或许就能认识新的朋友。

4、处处是风景，路边也会有惊喜，随时可能发现让你惊喜的风景，如果想停车就停车，拍照欣赏风景想看多久就看多久，总结就是自由！

出行路线

由于去张北草原交通不是很方便，可以选择租车、拼车或者自驾出行。
第一天：

从张家口市区出发，走西环路到锡海线，再上张石高速开车大概十九公里，到达草原天路东线，继续游览草原天路，后前往沽源县，夜宿沽源县。
第二天：

早晨从沽源县出发，开车大概七公里到达天鹅湖自然风景区，接着直接返回张家口市区，去桥东小吃街吃好吃的

行前准备

1.相机（够用的电池，相机包，一定要拍很多照片

2.衣物（这时间张家口天气相对比较好，不会太冷也不会太热，早晚温差还是有的，平时出门备着点衣服就好） 建议想好每天穿什么，不要带太多，拿着麻烦，行李重！拖鞋，运动鞋，平各一双足以。 要注意防晒。

景点拍照的话可以带一些好看的衣服。墨镜（凹造型）
3.个人用品 毛巾 牙刷 护肤品 防晒霜（安耐晒）防晒修护 太空毛巾 湿纸巾 充电宝 还有各种药（准备些感冒药，肠胃药，退烧药，有胃病的话带个胃药）身份证 银行卡 人民币（以备有些地方不可以手机支付）

张北草原

[张北草原]

地址：张北草原是位于张家口西北70公里的张北县赫家营乡三宝营盘村
境内的草原，由中都和安固里两大草原组成。在张北草原可以骑马射箭；也可以观看马术、摔跤、草原歌舞等草原传统娱乐项目。 爱上一匹野马，我的家没有草原。

我也想有片大草原策马扬鞭，来到大草原，首先看到是骆驼，这里除了骑马还有骑骆驼的。
驾驶在草原天路之上，与蓝天白云阳光共舞。张北草原天路有中国的66号公路之称，蜿蜒曲折道路，壮丽辽阔草原，清透湛蓝天空～～美丽至极!
离开市区，人非常少，山高林密、溪河纵横、人迹罕至。
驱车前往张北草原。
一路上各种指示牌，不用管向前走！
即便现在还不是草长得茂盛的时候。
张北草原是被音乐节生催起来的一片荒草地，至于草原天路，以前是为运载风力发电货车铺设的一条山路。
做风中的女子！
在车里打开车窗，乘着风，森林覆盖率达百分之八十以上，是天然氧吧、呼吸着大自然的产生的负离子。
或许是那种荒凉，那种空旷，那种萌芽生机，那种凉爽的天气吸引了我。

草原天路就似“美国醉美66号公路” 张北草原是位于张家口西北70公里。

由中都和安固里两大草原组成。
在张北草原可以骑马射箭；也可以观看马术、摔跤、草原歌舞等草原传统娱乐项目。 现在很多草原项目开始盛行
骑上马肯定少不了一首歌！

“让我们红尘作伴活得潇潇洒洒，策马奔腾共享人世繁华 对酒当歌唱出心中喜悦 轰轰烈烈把握青春年华 让我们红尘作伴活的潇潇洒洒 策马奔腾共享人世繁华 对酒当歌唱出心中喜悦 轰轰烈烈把握青春年华 喔...喔喔喔喔“
在这一望无际的草原上，少不了这不羁的浪子---马，他们矫健，骏美，奔跑在这犷野的草原上，油亮的肌肉在阳光下显得十分耀眼，而我有幸骑在这漂亮的精灵上，策马狂奔，沐浴在阳光下，那是怎样的何等的荣幸与尊享。
起先，不敢让马跑太快，又不忍抹杀他本应狂奔的青春，于是决定让他跑起来，拐向草甸，抓住了缰绳，深深一吸气，我发出了跑的号令，“驾！”

马儿似乎也感受到他的几分害怕，也没有跑得太快。脚几次从撑上掉落，总有惊无险地放回。

他的心态渐渐放平，开是追求更快的速度，可他的主人以担心安全为由制止了，只好让马减速，开始漫步山野。

在一片山坡上，马儿依旧眺望远方，心想，如果不是人类，马儿的命运应是狂奔在一望无际的原野，或闲庭信步，或狂奔于原野……
接着有的门牌上写的是草沿天路，有的写的草原天路，但其实指的都是同一条路。
爱上一匹野马，我来到了草原。顺着天路一路驱车，沿线山高坡陡、沟壑纵深、景观奇峻，展开了一幅百里坝头风景画卷
沿路是一条柏油公路，但海拔也有千米左右，深色柏油路与黄实线本身就是一条美丽的风景。

百里之间，左右徘徊曲折，剧烈地跌宕起伏，静谧深远，仿佛通向梦的彼岸。 一路行来，路况都很好，仿佛可以一直不停，开到天边。
滑草，草原摩托车，也可以惊醒体验一番。

最美草原期：
每年的6月——8月，是草原绽放的时节，一望无际的绿，是公路两旁最极致的美景

沽源县住宿安排

[汉庭酒店(沽源店)]

地址：沽源人民南街福源二区12号楼
[地理位置]酒店位于张家口，沽源县。要是找不到可以搜导航一下。地理位置非常优越，基本算是县城的中心，出门走几条街到购物商场，这个县城不大，去哪都挺方便的，比较近，非常方便购物十分便利，吃喝玩乐应有尽有，商场里面样样俱全，可谓说酒店的地理位置可是得天独厚的呢。
[酒店服务] 酒店服务很周到，到入住前台处登记，登记过后给房卡，我们自行到住宿楼层就行。还给我们提示的位置什么的。前台服务员态度都非常好。表扬
[酒店详情] 酒店是新开的，装修比较感觉，楼下是有免费停车位的，对于我们开车来游玩的，就比较方便。酒店隔音效果不错，走廊声几乎可以完全隔开。我们订的是高级大床房。房间呈现的是北欧风，温润的木头、明黄色的灯光、又使得整体柔和不少。
简约时尚，一起让烦恼清零，感受简单生活里的别样美好。这里有准备了生活用品，很贴心，有洗漱的牙刷和洗发水，沐浴露。房间门后都有贴着WIFI密码哦，注意查看。房间大小刚刚好，足够两个人来住。
房间里面的开关都是全自动的，还是比较方便，功能都挺全，床边也可以熄灯。而且每个开关都是分开的，可以只亮一处灯，不像其他酒店一打开灯整个房间都亮了。

房间也有衣服挂和桌子，桌子虽然不大，但是还是够放东西的，十分方便。
[房间卫生]被套整洁干净，白白的床单就是令人会有好感。这种装修让我们看着比较舒服，设计很简洁大方。床上放有两种枕头，一个是记忆棉，一个是腈纶棉，可供我们选择，而且被子还是挺舒服，挺厚的，毕竟是连锁酒店，统一的管理，统一的经营模式，质量什么都还是有保证的。

配套硬件设施齐全，有热水壶，还有矿泉水。有台灯，在写字台写字还是挺方便的。
卫生间还是挺干净的，就是稍微有点小，酒店是新的所有设备都很新，所以不会有陈旧的污迹什么的，就会看着会很舒服，简洁大方。
这应该是沽源县里最好的酒店了，有机会还会去那里的话，还可以继续住这个酒店，希望能保持到现在原有的环境。

天鹅湖景区

[沽源县天鹅湖自然风景区]

地址：河北省张家口沽源县天鹅湖度假村沽源天鹅湖
由张北县开车前往沽源县，天鹅湖位于沽源县的东北边十多公里处，那边还有闪电湖和大汗行宫，都在沽源和扎拉营村的中间，所以我们决定先去天鹅湖，事实证明，如果从景色上来说，天鹅湖比闪电湖强的不是一点点
误打误撞，根据导航我们实际上来到了天鹅湖的北面，放眼望去，湖面宽阔无比，沽源县城在对岸隐隐可见，有放牧的羊群，马群。

景区是没有人管理的，属于一个野景点吧。
景色不错，就是风有些大，空气清新，每片云朵都会变换出各种不同造型。 四周青山环绕，湖中碧波荡漾，生态环境良好。
但是没有看到传说中的天鹅，却有几只不知名的小鸟在湖面时起时落，飞掠而过。 听说现在不是最合适的季节，天鹅还早着呢，高原草甸也没到盛花期。

但是这时候去有个最大的优点，就是人少。可以安安静静的随意游玩。

事实证明也确实如此。
整个湖周围几乎就我们几个是来游玩的。 在湖边拍照的时候，无意发现对面的小山丘上有游客，似乎站上面看湖会更好看些。
询问村民路线后我们便驱车往山上开，果不其然，可以看到整个湖，高处瞭望景色。美观可唯更上一层楼。
最适合游玩时期：

每年的6月——8月，是草原绽放的时节，天鹅也回来了，这才能称得上名副其实的天鹅湖

张家口市区住宿

[驿家365连锁酒店(张家口红旗楼店)]

地址：张家口桥东区红旗楼冠垣广场23-25楼
第二次住这家酒店了，由于来张家口比较多，还是选择方便熟悉经济实惠的酒店。酒店在冠垣广场里面，入口在北富门，挺好找的

在张家口属于经济型酒店，位于张家口市中心，周边有很多商场，步行500米左右可以到达银座商场，吃喝玩乐都挺方便的，位置优越，打车都挺方便的，算是张家口的地标建筑
酒店登记入住前台服务态度挺好的，办理手续还挺迅速的。一楼好像是打台球的地方，不过酒店在高层，不会收到杂音的影响，酒店环境挺好的，很安静。

入住的是大床房（内宾），房间没有很出奇亮眼，不过还是挺好的，干净舒适，在25层，白天可感受阳光的明媚，晚上可享受满天星光的浪漫。

风景美，窗外风景美，映入眼帘的是车水马龙，深刻感受到张家口的繁华。房间明亮素雅，宽敞，床单洁白干净，眺望窗外，对面马路是全季酒店，所以说这边地段超好，去哪吃饭逛街都很方便。
卫生间干湿分离，灯光有点暗，我还是喜欢卫生间灯光亮一点比较好，不太喜欢浴帘，洗澡弄湿会不小心粘到身上。但是比较喜欢这的蓬头，水劲十足，这样很舒服。

不足的是连上酒店Wi-Fi却用不了，会不会是张家口地段网络不好，一直纠结着一点。

酒店没有提供免费的瓶装水，晚上太渴了，打电话给前台希望她们可以把水送上来，非常迅速就给送上来了，挺好的。这点十分满意。

美食大放送

[羊腿肉 ]

这家是在张家口沽源县里面找到的，刚从张北草原驱车到沽源县。超级想吃羊肉，便找了一家，听朋友说，这里家家都不错，可随便找一家。

都是自家养的羊，我们进去自行挑选，然后再店个 筱面 ，还有番茄炒鸡蛋，再到店门口买个玉米，张家口的玉米很好，都很鲜甜。
莜面,也叫裸燕麦面,又叫油麦面。

筱面就是筱麦，生长在内蒙古乌兰察布盟、呼包、山西雁北张家口地区。

莜面乃第一健康面食，已得到大家的公认，近来在阿拉善盟也得到当地居民的喜爱。
筱面是张家口的特色美食推荐大家尝下！安利！

蒸莜面不是蘸汤吃，而是把莜面放在蒸熟的肉汤里，加入土豆条，鸡蛋羹，腌韭菜等菜码放在一起和着吃 我: 只是莜面不容易消化（晚餐最好不要多吃），每顿量要少吃。
选了羊腿，羊肉含有丰富的蛋白质，其含量较猪肉牛肉高。

羔羊肉嫩多汁，想起都流口水！

一定要趁着热气，拿起刀，顺着刀花从上至下切下一大块。表皮底下是一层薄薄的油脂，散发着羊肉鲜嫩的味道！所有的等待，都变成满满的幸福感！

桥东区小吃街


所有地区都会有很多小吃街，去旅游怎么能不去小吃街。

这里晚上人很多，熙熙攘攘。

有点要注意的，这里人多手杂，注意自己的随身物品。
不过给我感觉这里东西很不卫生，街道脏兮兮的，几乎都在摊位上临街摆卖的，这里海鲜也比较便宜，卫生就不太好，建议对海鲜过敏的不要吃了。

沿街有很多特色饮品、糕点类的小吃不错，但是别多吃，感觉容易拉肚子。 我买了个降龙爪爪，还有奶茶，吃完回去就拉肚子了。
