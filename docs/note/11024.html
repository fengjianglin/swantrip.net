宁夏6日游：一群寂寞男青年居然在沙漠深处互~~~，不忍直视！

游记 自助游 自然奇观 避暑 2018/10/20

前言

有的事情现在不做，就一辈子也不会做了。看着身边越来越多的朋友，被家庭、被孩子、被房子、被工作所束缚。就想趁着自己还是无业游民，总想走完全中国，看完全世界。
对于大西北，能想到青海、甘肃、新疆、陕西、内蒙，唯独让人容易忘记宁夏。对于宁夏，有太多的空白，总想去看看。感谢朋友，给了我这次机会，让我近距离了解宁夏，也让我认识了一群戏无比多的戏精童鞋。
先来一组美图镇楼：
6天5晚，行程线路： 成都—银川—固原—中卫—青铜峡—银川。
备注：费用参考。强烈安利沙坡头和沙湖景区，玩的丧心病狂。

Day1:六盘山国家森林公园、火石寨

六盘山门票：55元
火石寨门票：55元
去一个地方，一定要首先了解它的地理位置与文化。
宁夏：北倚贺兰山，南凭六盘山，黄河纵贯北部全境，历史文化悠久，古今素有“塞上江南”之美誉，是中华文明的发祥地之一。 宁夏位于“丝绸之路”上，历史上曾是东西部交通贸易的重要通道。大家熟悉的西夏王朝，就是指宁夏。普及了大背景，就开始给大家介绍行程了。
固原，公元前114年建城，丝绸之路必经之地，明代九边重镇之一。固原知名的景点，有六盘山国家森林公园和火石寨。一个自然森林景观，一个自然地质景观。
六盘山想必大家不陌生，就是红军长征结束的地方。绿树、溪水、花草、虫鸣，给人一种回到四川的感觉。可能对于一个四川人来说，有山有水有鸟有草的景区，太常见了，但对于宁夏人民来说，是真正的塞上江南。当你从银川到固原的高速路上，你会看到荒芜的山丘，没长一颗草。这种强烈的反差，就够了。
六盘山国家森林公园
当天下了一场雨，当地的小孩都在捡类似木耳的食材。
上午冒着细雨游览了六盘山，结果下午到了火石寨，就是一片艳阳天。火石寨，就是丹霞地貌景观，传说穆桂英还在里面养过马。
火石寨有玻璃栈道，走在上面，再看看下面，提心吊胆。我真的超想坐旁边滑轮，结果没人陪我，我反倒觉得那个更好玩。
悬崖峭壁上的寺庙，很好奇当年是怎么修上的。
传说穆桂英养马的地方，外面还有一段古长城。
旁边长的野花，据说是本地特有的
里面真有马，但不是穆桂英养的，只是供人参观的。
一柱擎天，请问是怎么看出来是一柱擎天的，可能我眼瞎。哈哈。
然后一群人在这里浪，浪，浪。
登上山顶后，一览无余
第一天小伙伴找了农家乐，真心觉得超赞啊，老板娘人很美，很美，可惜没拍照。上的菜，也是超级多，这可能是我吃的最多菜的农家乐，大概有30个菜。全程吃饭的地方，我就只记住了这家，超级赞。
各种面食，吃的扶墙出~

Day2:沙坡头、通湖

沙坡头景区门票：80
通湖+篝火晚会门票：153
沙坡头位于宁夏回族自治区中卫市城区西部腾格里沙漠的东南缘，沙坡头集大漠、黄河、高山、绿洲为一处，具西北风光之雄奇，兼江南景色之秀美。
沙坡头有扶梯直达沙丘山顶，在整个沙丘的左侧
沙坡头集黄河、高山、绿洲、沙漠于一体，沙漠就在左边，为什么绿洲没有被淹没，大家可以猜猜？
到了沙坡头，各个文艺高冷的青年，分分钟变逗比青年，各种拍拍拍。我这种第一次见沙漠的人，也是激动的不得了，相机里不晓得弄了好多沙，超级好几好玩。
被沙漠车带进了沙漠深处，一群寂寞的男青年就开始了疯狂的拍拍拍。互相拍拍拍，哪还有什么旅游大V的形象，各种骚起来~
我曾走过人群，见过大海，也见过沙漠。
继续在沙漠中拉拉扯扯。
大漠，驼铃，电视中的画面来了
一群逗比青年，开始了疯狂的创作。
在沙坡头玩的太嗨，天黑了都不想走，但由于在网上订了通湖的门票，到了打卡。
通湖，顾名思义是湖水相连的意思。几百年前这里确有一片湖水，据说有两个喇嘛在距此60公里处的“太阳湖”边用铜壶取水，一不小心铜壶掉到了水里，几日后，有一个牧民妇人在这里发现铜壶，方知两地地下水系相通，因此取名“通湖”。
到了快下班了，索性就在门口看看了。
通湖门口的景色，沙漠，绿洲，湖，你能看出哪些是真羊？哪些是假羊么？
通湖的晚上也是很精彩的，但类似购物点，你懂的

Day3:黄河大峡谷、108塔、黄河楼

青铜峡黄河大峡谷长10多公里，宽50—100米，两侧的崖壁高30米以上，具有典型粗犷雄浑的北方黄河特色，被誉为“黄河中上游第一峡谷”。
在网上看了下黄河大峡谷的图，惊呆了。美美美。
越想去的地方，越得不得。当天，很不幸，沙尘暴。
我也说非常非常非常激动么？第一次看沙尘暴。
大家只好去了108塔，滚回去了~
一百零八塔是中国现存的大型古塔群之一，位于银川市南60公里的青铜峡水库西岸崖壁下，塔群坐西面东，依山临水，塔基下曾出土西夏文题记的帛书和佛祯，可能建于西夏时期是喇嘛式实心塔群。
佛塔依山势自上而下，按1、3、3、5、5、7、9、11、13、15、17、19的奇数排列成十二行，总计一百零八座，形成总体平面呈三角形的巨大塔群，因塔数而得名。
有沙尘暴，照沙漠都是灰的，想P呗，又觉得不真实。
灰蒙蒙的一片，就是沙尘暴。
傍晚的时候，确实觉得太无聊了，就跑去了黄河楼逛逛。
在黄河楼看青铜峡。

Day4:西夏王陵、贺兰山岩画、西北影视城

西夏王陵门票：90
贺兰山岩画门票：60
西北影视城：95
西夏王陵又称西夏帝陵、西夏皇陵，是西夏历代帝王陵以及皇家陵墓。王陵位于宁夏银川市西，西傍贺兰山，东临银川平原，海拔1130米至1200米之间，是中国现存规模最大、地面遗址最完整的帝王陵园之一，也是现存规模最大的一处西夏文化遗址。
西夏王陵营建年代约自11世纪初至13世纪初。西夏王陵受到佛教建筑的影响，使汉族文化、佛教文化、党项族文化有机结合，构成了我国陵园建筑中别具一格的形式。在中国119处国家重点风景名胜区中，西夏陵是惟一的以单一的帝王陵墓构成的景区。它承接鲜卑拓跋氏从北魏平城到党项西夏的拓跋氏历史。
景区占地面积58余平方公里，核心景区20.9平方公里，分布9座帝王陵墓，200余座王侯勋戚的陪葬墓，规模宏伟，布局严整。每座帝陵都是坐北向南，呈纵长方形的独立建筑群体，规模同明十三陵相当。吸收自秦汉以来，唐宋皇陵之所长，又受佛教建筑影响，构成中国陵园建筑中别具一格的形式，故有东方金字塔之称。
景区不去终身遗憾，去了遗憾终身。西夏王陵就是其中之一。远看像座山，近看一堆土。
到这种地方，还是要请讲解员，才知道里面的文化
光影交错的西夏王陵，合影走一波。
参观完西夏王陵，就去了贺兰山岩画。给人一种窒息感。
贺兰山在古代是匈奴、鲜卑、突厥、回鹘、吐蕃、党项等北方少数民族驻牧游猎、生息繁衍的地方。他们把生产生活的场景，凿刻在贺兰山的岩石上，来表现对美好生活的向往与追求，再现了他们当时的审美观、社会习俗和生活情趣。在南北长200多公里的贺兰山腹地，就有20多处遗存岩画。其中最具有代表性的是贺兰口岩画。

西北影视城，中国早期电影的摇篮，更因星爷的无厘头扛鼎巅峰力作大话西游而闻名天下，身临其境，别有一番感受。中国的电影从这里走向世界，《红高粱》，《西游记》，《大话西游》，《老人与狗》等等等等。中国的许多电影在这里拍摄。西北堡西部影视城在中国众多的影视城中以古朴、原始、粗犷、荒凉、民间化为特色,是中国三大影视城之一,也是中国西部唯一著名影视城。

Day5:沙湖

沙湖门票+船票+自助烧烤+篝火+游玩+其他乱七八糟：800
沙湖地处宁夏石嘴山市平罗县境内，距石嘴山市区26公里，距首府银川56公里。国道与包兰铁路傍湖而过，京藏高速公路直达沙湖。景区总面积为80.10平方公里，其中水域面积45平方公里，沙漠面积22.52平方公里。
沙湖以自然景观为主体，沙、水、苇、鸟、山五大景源有机结合，构成了独具特色的秀丽景观，是一处融江南秀色与塞外壮景于一体的“塞上明珠”，既有大漠戈壁之雄浑，又有江南水乡之秀美，被誉为“世间少有”的文化旅游胜地。
沙湖真的美到爆，沙漠，湿地，白鹭，游玩，露营，帐篷，星空，你能想到的都有。如果有机会，我还会去次宁夏，去宁夏一定要去沙湖。
