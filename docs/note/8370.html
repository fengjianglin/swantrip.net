#“帝”造浪漫，携爱出行#单车自驾：开往团山古村的百年米轨

来到建水，除了建水古城以外，还有很多值得游玩的地方，例如我国现存的唯一米轨小火车，坐着米轨小火车可以游览建水城外的璇旎风光，感受上世纪法国建筑风格的车站。这条米轨铁路可以从建水直达蒙自碧色寨，即电影《芳华》的外景地，而坐上米轨小火车，可以抵达有着云南最精美的古民居群之称的团山古村，这里是完整保存19世纪风貌特色的原生态村落，是一个地方民族文化与中原文化相融合，整体格局风貌完整、真实的传统风貌型历史文化名村。建水古城米轨小火车现在开通了临安站、双龙桥站、乡会桥站和团山站四个站点，穿过临安镇和西庄镇，这两个旅游小镇的众多历史文化和风景名胜，通过一条铁轨，连接在了一起。
1915年5月，个碧石铁路开工，这条名为个碧石的铁路创造了众多的奇迹和纪录，个（个旧）碧（碧色寨）石（石屏）铁路是中国最早自己修建的铁路，也是历史上最小的铁路，运行速度最慢的铁路，修筑时间最长的铁路，中国历史上唯一的民营铁路，唯一发行货币的铁路。这些纪录，记载的是云南铁路史上一段最值得骄傲的精彩传奇。百年之后的2015年，在个碧石铁路开建100周年之际，由昆明铁路局和建水共同合作的云南米轨铁路建水古城小火车，于5月1日在个碧石铁路建水东至团山间线路改造后的米轨铁路上开行。让人们可以亲自乘坐这条富有传奇色彩和历史底蕴的百年米轨小火车，驰骋在建水古城和团山古村之间，徜徉在历史与现实之间，流连忘返。
团山村位于建水城西十三公里的西庄坝子边沿的泸江河南岸，个碧石铁路从村口穿过，这里的自然环境和生态现状都非常好，从村口流过的泸西河，成片的水田，房前屋后的蔬菜，水果，院内的花草，有种田园风光的味道。村内具有大量文化沉淀的旧式建筑。团山村现保存完好的汉族传统民居和古建筑有21座，它们都已经被编了门牌号，成为无数户人家共同拥有的家园。团山村的后人们就在这令人羡慕的老宅内过着自给自足的农家生活，年年迎候春燕的归来，因为这些燕子也把它们的家筑在了雕龙画凤的檐梁上。

住宿餐饮

建水住宿有市区宾馆酒店和古城客栈两种。建水的宾馆酒店包括普通酒店、连锁酒店和星级酒店，可以根据需要选择，天鹅之旅网预定非常方便，价格也有很大的优惠。目前健硕最高星级是五星级的临安酒店，还有一些准星级酒店，这些星级及准星级酒店一般在设施设备方面都比较豪华。而古城客栈也非常推荐大家选择，因为可以住在古城里面，体验古城的环境和氛围，有些设施比酒店简陋些，有些现在已经装修得非常有格调，档次也不低，天鹅之旅网上也可以预定。
建水当地饮食最大特点是大多都是天然、无污染的绿色食品，这里的蔬菜种类齐全，甚至还有许多具有药用价值的野生蔬菜。特色食品有临安烧豆腐、狮子糕、泡糕等。很多人只晓得长沙臭豆腐，其实论味道，建水的烧豆腐一点不逊色，建水烧豆腐的正确吃法，慢慢用小火烤，外面有点金黄，里面蓬松，用手掰开，蘸点蘸水，好吃到爆。那别具一格的味道，绕颊的余香，常常令人回味无穷。建水烧豆腐的历史极其悠久，早在清代中后期就享有盛名。时至今日，建水烧豆腐已走向省城，遍及滇南城乡。建水城街巷、菜市、夜市、风景旅游景点的烧豆腐摊，人来熙住，成为古城中的一道亮丽风景线。

米轨火车临安站

建水古城很适合隐居，这里没有丽江的喧嚣，没有大理的浮华，宁静安闲，宠辱不惊。古城不大，主要是以临安路为中心的几条街道，这里是米轨火车临安站的旁边，在古城外的新区，但也全都是仿古建筑。
这里别看地处偏远，但是在元代就始建庙学，明洪年间建临安府学，万历所间又建建水洲儒学，清代先后建立了崇正、焕文、崇文曲江四个书院。
当时，建水有着临半榜之称，即云南科举考试中榜者中，临安府就占了半数左右，堪称云南之冠，在全国变不多见，所以建水从来就有文献名邦和滇南邹鲁的美誉。
建水有我国现存的唯一米轨小火车，可以坐着米轨小火车游览建水城外的璇旎风光，抵达有着云南最精美的古民居群之称的团山古村，这是来建水旅行最主要的游乐项目。我们是自驾车，所以这次没有乘坐米轨小火车，但是我们沿着米轨小火车的路线游览了其中三个车站，起点、沿途和终点的几乎所有景点，还登上了小火车一睹其真面目，收获满满还一分钱没花，真的是达到了穷游的极致啊。
首先是米轨小火车的起点临安站，建水以前叫做临安，所以在当地人们还是习惯叫这里为临安古城，车站叫做临安站也就不足为奇了。这里上世纪法国建筑风格的车站年代感特别强，候车室虽然有现代化的设施，但还是基本还原了百年前的面貌，走在其中有种穿越历史的感觉。
1915年5月，个碧石铁路百年前建成，创造了众多的奇迹和纪录，这是中国最早自己修建的铁路，是历史上最小的铁路，是运行速度最慢的铁路，是修筑时间最长的铁路，是中国历史上唯一的民营铁路，也唯一发行货币的铁路。
在这样怀旧典雅的环境中，会让人莫名地感觉一种从容悠静，建议大家不要匆匆的赶路，可以找一个幽静的角落休息一下，让日渐浮躁的心慢慢得安静下来。当然，这里布置在这个移动互联分享时代也绝对是大受欢迎的，很多人喜欢在这里摆个造型拍张照片，秀在自己的各种平台上。
进了站台，和我们想象中的火车站站台完全不同，看见就像个小区停车场。想想也是，这里只有一个目的地，来回两条铁轨，有时候很袖珍的米轨火车，当然和正常的火车站台是完全不一样的啦。
终于见到了传说中的米轨铁路，果然看上去就非常的迷你，顾名思义，轨距为1米，所以叫做米轨铁路。标准铁路的间距为1.435米，比这个宽的都叫做宽轨，比这个窄的都叫做窄轨，窄轨除了米轨，还有寸轨。寸轨比米轨还要窄，但一般用于矿山，真正运营的客运铁路，这条米轨铁路应该是最窄的了。运气不好，火车刚刚发出，也不会有车马上回来，所以决定继续行程，去追赶这传说中的米轨小火车。

米轨火车乡会桥站

很遗憾错过了米轨火车的双龙桥车站，不过不要紧，最有特色的乡会桥站没有错过。这个车站建于1936年，建成时有办公楼、候车楼、站台、货场，现在这些建筑都得到了保留和修复，还原了当时的场景。火车站前傍泸江河，河畔翠竹掩映，柳树成荫，后隔百余亩良田肥地与新房村相望，环境清幽，绿野满目。
这里和起点豪华气派的临安站比起来显得不太起眼，但却是整个个碧石铁路上最有特点的车站，像我们一样自驾游的朋友可千万不要错过了。车站的主体建筑站房，也就是售票候车楼，市巧妙的结合了中、法两种建筑方式和特点相的单檐现代建筑。
站房建筑状若炮楼，砖、石结构，法式建筑特点突出，色彩浓烈，外墙体统一为当时个碧石铁路站台建筑的黄色，具有醒目的视觉效果。中式木结构歇山屋顶，抬梁式与穿斗式架构形式，对称布局，部分小木作上仍然沿用中国传统建筑的斗、升等构件作装饰，其两侧还各有一面阔三间木构架与站房建筑相连接，布局稍靠后安排的侯车楼和耳房。
乡会桥火车站站房建筑是中西建筑文化协调融合的缩影，也是乡会桥火车站站房建筑最具文化个性和观赏价值的地方。候车楼的门开着，可以进去看看。
这里面还有贵宾间，这布置真的很有异国情调，仿佛置身于法国的小镇，让人能强烈感受到百年前这里曾经的繁华。
站内现在只有一条轨道，但是过去是有三条轨道的，一条供火车停靠站，一条供火车会车让道通行，另一条供车站调度、留置、货物装卸之用。
因新房村在上世纪80年代以前一直是原乡会镇公所、西庄区公所、西庄镇政府所在地，是建水西庄镇政治、经济、文化、交通中心及建水县域西部粮食、货物储运中心，故乡会桥火车站自建成后就一直是建水、石屏间重要的火车停靠车站之一，同时还是西庄镇主要的货物集散转运站。附近新房、荒地、高营等村寨村民都是在乡会桥火车站乘降火车，凭借中国第一条民营个碧石铁路提供的便利，往来于建水、石屏、个旧、蒙自、河口及昆明等地。

新房古村黄家祠堂

乡会桥旁有个新房村也是个保存完好的古村落，漫步新房村中，随处能够偶遇保存至今的百年老屋。这些老屋有的还在使用，世居其中的村民大多已经老迈，有的已经大门紧闭，门前芳草萋萋，只留古色古香的碧瓦朱檐向人们诉说着当年的辉煌，而有些则已经破败不堪，只留断瓦残垣，向人们展示着岁月的无情。
黄氏宗祠就在村子中间，已经有130年左右历史，依然维持着它的旧貌，雕梁画栋，富丽堂皇。门口处的石狮子栩栩如生，把守着这座老屋。
黄氏宗祠内凤在上，龙在下的旧式木雕是最值得一看的，但是很可惜的是我们去的时候没有开放，据说现在还没有开放旅游。但是站在门口，依然能够感受到黄氏家族在百余年前的家族盛况。

绍和山麓黄龙禅寺

继续出发，路过位于建水城西12公里绍和山麓的黄龙禅寺，这里据说也有着八百多年的历史，但是看上去十分破败的样子，很多古建筑由于年久失修变成危房，希望能早日修缮，恢复这里山寺珍泉景最幽的美景。

米轨火车团山站

终于来到了建水米轨小火车的终点团山站，运气非常好，远远的看见车展前停着黄色的火车，非常的高兴。
车头的东风标志很亲切，经历过60年代到80年代的人，对于中国老一代火车的模样，应该还记忆犹新。在那个时候，中方的火车主要是以内燃为主，其中早期的就是东风系列、东方红系列。东风系列运行时间最早，就算是今日，也可以看到有大部分东风系列的火车，依旧奔驰在祖国大地上。
米轨小火车前后两节车厢是敞篷的，专门供游客赏景拍照。这条线路两侧的景色很漂亮而且随四季而有着不同的变化。可以想象，鸣着汽笛的黄色车厢摇晃着身子，越过田野，穿过乡镇，在绿色的田野中欢快地奔跑着，展现着迷人的风采。
这种米轨小火车不仅是一个拍摄题材或者是一个旅游项目，更是包含了一段历史、一种文化和一种生活态度。
黄色车厢上的站牌写着临安—团山段，这段铁路见证了中国近现代史，也经历了由寸轨向米轨的转换，当初因为财力工程难度还有防止外来侵入者运输的顾忌采用了寸轨，但好在铺路留有余地是按照米轨建设，也为后来留下了卓有远见的基础。
米轨小火车古香古色，车厢不大，纯木质结构，有点象一个硕大的玩具，进入车厢让人仿佛进入一个童话的世界，回忆起无忧无虑的童年时代。车厢里的降温还是采用车厢顶的风扇，浓浓的怀旧的气息。
小火车从内到外都是以黄色色调为主，即区别于即将离开我们视线的绿皮火车，也不同于高速发展的白色或橙色的高铁动车，但这种颜色给人的感觉却非常舒服，也能迅速将人的情绪代入其中。
团山车站虽然是终点站，但是结构特别简单，只有一个建筑，集售票和候车于一体。

精美古村落团山

团山村位于建水城西13公里的西庄坝子边沿的泸江河南岸，这里的自然环境和生态现状都非常好，从村口流过的泸西河，成片的水田，房前屋后的蔬菜，水果，院内的花草，有种田园风光的味道。
团山村是完整保存19世纪风貌特色的原生态村落，是一个地方民族文化与中原文化相融合，整体格局风貌完整、真实的传统风貌型历史文化名村。
团山村面积不大，现在保存完好的汉族传统民居和古建筑有21座，它们都已经被编了门牌号，成为无数户人家共同拥有的家园，也成为了人们来寻幽览胜的旅游热门地。
游客中心不远就是团山村的东寨门，处于整个团山民居群的对外交通要冲，起着天然屏障，防御功用，建于清光绪三十年，建筑形制采用了二层三间牌楼式单孔拱形大门土木结构。楼上楼下设有瞭望哨、枪眼，有一夫当关，万夫莫开之势。
建议大家和我们一样，先不入东门，向右行，前往将军第。
团山村将军第建于清光绪三十一年，占地面积880平方米，建筑面积958平方米，计有大小天井7个，房屋38间。
房屋建盖者张和，字用礼。幼年丧父，全赖母亲抚养。后随堂兄张恺到个旧谋生。在矿山上成长起来的张和，性格豪爽，胆量过人，凡矿山发生的争吵和械斗，都由他一言定夺。
在1911年爆发的辛亥临安起义中，张和领衔率众平息匪乱，得到云南都督蔡锷的赞赏，委任张和为个旧国民军第四营管带兼稽查员，并授将军第匾额，悬于家乡。所以，团山人将其所住宅院称为将军第。
整个建筑保存是相当完好的，现在这里依然有张家的后人在此居住，而且分隔开来并不能全部游览。进去参观当地人十分友好，这样也挺好的，和房主聊聊还能多知道点这里的历史。
出将军第继续南行，来到张家花园。张家花园建于光绪三十一年，平面布局由由寨门、一进院、二进院、花园祠堂和碉堡组合而成，有大小天井21个，房屋119间，是一组城堡式私家园林，也是仅次于县城朱家花园的又一大型民居。
朱家花园涵盖了云南传统民居中“四合五天井”、“三坊一照壁”、“一进两院或四院”、“跑马转角楼”等主要建筑格局，可以说是一座云南民居的博物馆。
张氏家族的发迹和张家花园的建盖，都在清代末期。团山历史上是彝族的居住地，团山村是一个典型的滇南汉族移民村。个旧的锡矿，于清朝末年为滇南带来了巨大的财富。建水城里的朱家花园因此兴起，建水西庄镇的团山张家族人也以此发家。
在600余年的历史长河中，张氏族人遵“百忍”家训为安身立命之本，至清末，张氏族人积极参与个旧锡矿开发，挣得巨额钱财，皆回乡建盖豪宅，光耀门庭。先是在村中盖了五间两耳的住房。后因人口多，住房拥挤，又择村边的土地，建盖了张家花园。
整组建筑造型幽美，斗拱正檐，屋角起翘，屋面反曲有举折呈流线型，一色的灰调文，给人舒适之感；雕梁画栋，名人书画无一不备，极富装饰性；雕镂精巧的花格门窗、古色古香，是建水传统木雕中的精品；铺地、柱础、窗下坝墙、门垛和照壁都很有特色。总之，朱家花园建筑多而不繁、艳而不俗、朴实而不呆滞、简单而不粗陋，具有建水民居的主要特点。
这是团山村的北寨门，建于光绪三十年，因此地山青水绿，流水淙淙，所以也叫锁翠楼。此寨门为牌楼式单拱三层结构，楼上下设有瞭望哨和枪眼，易守难攻。
留苑大约建于民国初期，宅院面积不大，计有天井2个，房屋17间，解放后曾为村公所、办事处用房，现改名为留苑。
张氏宗祠是团山村张氏祭祀性祠堂，始建于清乾隆年间，建筑不算特别，却是族人议事和祭祖的场所，每年团山村所有张氏家族都要在此举行祭祀仪式活动。
秀才府始建于清末，光绪三十三年竣工，因屋主张桂金、张桂明早年参加临安府学考试，分获贡生、庠生，被村民称为大秀才、二秀才，所以当地人就叫这座宅院为秀才府。
司马第建于清光绪三十三年，因先祖曾有儒士身份和司马官衔，故名司马第。这也是为团山民居建筑中唯一不属于张氏家族的传统民居。
这里是营长府，此房的主人张国藩曾担任靖国军营长，所以大家都称这座29号民居为营长府。营长府建造于宣统二年，计有大小天井9个，房间64间，是一组由三进院和两层三间四耳土掌房组成院落。
