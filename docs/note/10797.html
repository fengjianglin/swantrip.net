二刷巴厘岛“小众自然探秘+特色民宿” 吃喝玩乐拍10日游

游记 客栈民宿 摄影 海岛 自助游 自然奇观 2018/11/03
二刷 巴厘岛 ，去入住了几家不同风情的特色民宿和一些隐秘小众的自然景点。景点来说我去了一些小众瀑布和沙滩， 巴厘岛 瀑布很多，在猫头鹰上找了很多瀑布，最终选择了两个照片上看上去颜值很高的瀑布，Kanto Lampo Waterfall 瀑布和Tukad cepung Waterfall瀑布，极美。沙滩除了 金巴兰 外，去了潘达瓦沙滩和黑沙滩，黑沙滩Keramas Beach也是在猫头鹰上找，沙子是亮晶晶的黑色，去这些小众景点最好记下英文名，包车时以免弄错，或者到达后直接给司机看再确认下。
关于出海，这次去的是佩尼达小岛，去打卡了很震撼的三个悬崖景点（精灵坠崖+天神浴池+天仙裂痕）实景比照片好看！除此之外，还去了乌布集市、 德格 拉朗梯田、天空之门和水宫殿。10天旅程，住了9天，其中的两天拿来泡民宿度假，因为民宿真的太美了啊，波西米亚风、露台帐篷想要住下来不走了！
大家可以先把游记收藏起来， 巴厘岛 那么火总要去一趟的，这篇整理了大家拿着就可以用的实用攻略！

关于地图

巴厘岛 是 印度尼西亚 群岛中的其中一座小岛，如图所示，版图很像一颗钻石，早期的经典景点都集中在南部下面用蓝色框出来的区域，不过现在的 巴厘岛 已经成为一个旅游热门岛屿，不仅对所有 南亚 区域的游客，欧美游客更是多到爆炸，每次去 巴厘岛 都觉得欧美游客数量多于本地印尼人，哈哈哈。所以说西方老外还很羡慕我们中国人呢，离东南亚这些美丽的小岛都很近，而他们却要 千山 万水才能到此。
如果你让我推荐 东南亚 的岛屿，推荐角度是有的吃、有的玩，那第一我推荐 巴厘岛 ，第二我推荐 普吉岛 。今天我们先讲巴厘岛 ， 巴厘岛有山区、有田园、有火山、有大海，还有数不清的有格调的小店，饭又好吃，人民热情，值得一来，图中是网络地图，有个基本的方位了解。
巴厘岛 区域分布以及小轿车所花时间，上北是罗维纳，可以去追随海豚；下南是乌鲁瓦图和 努萨 杜瓦， 巴厘岛 绝美的五星级酒店就坐落在南部，是看海看日落度假休闲的好地方；左西是海神庙， 巴厘岛 最重要的寺庙之一，也是一个老牌景点；右东有天空之门和水宫殿，路途遥远，包车也要花上一天时间。中间是乌布和库塔，这两个区域人流量非常之多，非常之多。可以说是 巴厘岛 的两大热闹的商圈，有数不清的小酒店和小民宿。库塔酒吧多一些，靠近大海；乌布工艺品与画廊更加文艺一些，远离大海却更加亲近热带森林 和田 园。

怎么选择合适的酒店？

我入住了4家酒店，本来预订的是3家民宿，1家酒店，可后来其中一家民宿（非常期待的一家）已经像其他三家那样 成功 预订上了，可在我到达时才发现被无情的取消了，新开的民宿就是不专业，既然已无空房为什么还要把招牌挂出来，我用booking好几年已是他们的G会员和旅行达人标签，第一次遇上这样的事情，无奈之外只能坐在民宿外迅速换一家，最后严格来说是2家民宿，2家酒店。
大家可以根据你入住的酒店选择附近的区域游玩，缩短车路时间，让旅行更加明智。举个例子：如果你住在库塔，就先玩库塔区域，不要去东部的天空之门，单程就3小时，来回6小时，一天就没了。选择住在靠近天空之门的酒店再去，我从酒店出发一小时就到天空之门了，节省时间。当然，如果全程只住一家酒店就不用考虑这些了。
如果全程只住一家酒店，怎么办？
你想要交通方便——推荐入住库塔区或是乌布区的酒店
你想要靠近大海且可观看日落——推荐入住乌鲁瓦图和努萨杜瓦
你想要离机场近——推荐入住 金巴兰 区域
你想要获得宁静和安静——推荐入住东部sideman地区的酒店，我入住的第四家就在那个区域
你想要泡酒吧——库塔 水明漾 区域
你爱户外徒步——阿贡火山附近，这个还是要有弹性，如果火山冒泡期间就不要去了，命重要。
你想要像茱莉亚罗伯茨《美食、祈祷、恋爱》里那样过上 巴厘岛 乡村田园生活，又能感受文艺气息，又能泡酒吧、泡小店，那你必须去乌布。
我个人推荐乌布，如果下回还去 巴厘岛 ，打算全程入住乌布的酒店，租个摩托车作半个月村民。下面是我此次入住的四家方位图。
