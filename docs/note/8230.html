草原的向往

打开手机的日历，时间竟不留余地的走入八月，
这个季节是夏的末、秋的初，也是可以与世间所有美好相遇的季节。
大概是漂泊久了渴望安定，安定久了又想要流浪，
又大概是京城炼丹炉般的气温让人日复一日的过着魔鬼般的生活，
对于草原的思念在这一刻愈发强烈，于是我开始搜索着与之匹配的目的地，
这一次，我想要来一场可以“放逐自己”的随性之旅。
似乎我对超广角景观、火山和花海有着一种执念，只要这样的画面出现就会默默添加到脑海的备忘录。
就这样，内蒙古就像是每年夏天的一门必修课再次出现在2019的旅行大纲中......
内蒙古有着草原一望无际的开阔画卷，也有沙漠戈壁的苍凉之美；
有着历经时代变迁的旷世火山群，也有成片的治愈系油菜花田
可以随时与牛羊群隔空对话，也能体验蒙古人的马背生活
这里没有城市的车水马龙，只有风与马奔跑的声音......

乌兰布统自驾穿越行程

翻看内蒙古的地图发现竟然没有去过乌兰布统！（虽然内蒙古很多地方还没有去），先把距离北京最近的“版块缺失”补上。本着“去流浪”的心情，既想体验传统，又想玩点儿不一样的，不紧不慢的自驾行程最适合我们。在锡林浩特还正巧赶上盛大的那达慕开幕。
[D1]北京-锡林浩特（那达慕开幕、蒙古马表演）
[D2]锡林浩特-达里诺尔湖自然保护区-达达线-克什克腾旗（达里诺尔火山群、达里诺尔湖）
[D3]克什克腾旗-乌兰布统影视基地-乌兰布统镇
[D4]马背上的一天（骑马穿越草原）-公主湖-乌兰布统镇
[D5]乌兰布统镇-红山军马场-蛤蟆坝-经乌线油菜花田-赤峰-北京
★交通：北京-锡林浩特、赤峰-北京飞机，内蒙古境内全程自驾
★住宿：锡林浩特和赤峰为市级城市，住宿选择比较多，不少五星酒店水准不错。此外都是景区周边的县级、镇级的地方宾馆。

蒙古人的精神文明

>> 锡林郭勒盟.草原那达慕

每年夏季草原上演的那达慕是众多蒙古人大秀炫技的“战场”，抵达锡林浩特后就赶上一场时隔七年的那达慕。武士们竞相展示着自己摔跤、射箭和赛马的技巧，这一万众期待的赛事是蒙古人期盼的，对于我们来说，自然就是走过路过不能错过了！
蒙古人热爱那达慕，整个赛事会持续3天。蒙语里那达慕是“游戏”的意思，为了娱乐，也为了亲朋好友相聚，更是草原牧民庆祝丰收、祈福草原一切风调雨顺的仪式。
蒙古的摔跤不分重量级，赛场上常常能看见块头大个的蒙古选手，他们身着摔跤手的服饰，穿着马靴不畏烈日炎炎，在宽阔平坦的草地上与对手一较高低，这过程的精彩大概只有身临其境才能体会到。
摔跤手的服饰其实有一些历史渊源，在蒙古传说里，一个亚马逊女子混入摔跤比赛，致使其他男性对手全部惨败，蒙古人自然觉得丢人，为了避免此类事件，摔跤手的上衣设计成敞胸露怀的款式，这样就能暴露其性别身份了。而他们肩膀上的五彩绸缎称为“将嘎”，象征着每一次胜利的荣誉，将嘎越多，这位参赛者的实力也自然不能小觑。
摔跤比赛也显示了选手的力量和技巧，决定胜负的关键就在于其中一位摔跤手是否有除了脚掌和手掌之外的身体部位触地，一旦触地比赛就会宣告结束，也意味着这位选手比赛失败。
当然比赛场上“摔出来”的豪情是蒙古人之间的交流方式，公平竞争、建立友谊、团结向上才是那达慕每一项比赛的精神所在。
📍锡林浩特西日塔拉

>> 锡林郭勒盟.蒙古马情怀

马是蒙古人的朋友，更是蒙古人的灵魂伴侣，没有马一路相伴的草原之旅也是不完整的！而锡林浩特夏日的城市夜生活有点不一样，大概是那达慕开幕竞技的“后遗症”，对蒙古人和马相影相随的那种精神文明充满敬佩，蒙古人从骨子里带出的硬汉气质从古至今都未曾离开，这部名为《蒙古马》的室内实景剧将蒙古人与马的故事展现的淋漓尽致，充满感动。
这部大型实景剧是内蒙古送给祖国70岁生日的礼物。看过不少依山傍水的古装实景剧，剧场内同时200匹蒙古马呼啸奔腾的场面实在令人激动。
这部剧一共14篇章，谱写了蒙古人在战争中与马相依为命的情感故事，以及与之牵绊的蒙古青年爱情故事。每一匹马都是戏精，从他们眼神流露出的感情，可以让人领略在战争年代，蒙古人与马、人与人、族与族之前的爱恨情仇
浩气凌然的氛围让人肃然起敬，有一种想要冲上去抱住人与马的冲动。
当演员拿着哈达偕同200匹骏马谢幕的那一刻，剧场内的掌声震彻整个剧场，我想每个人的情绪都在被这部剧带动着。除了情感，蒙古人不畏惧、勇往直前的精神也让人铭记于心。

与火山擦身而过

>> 克什克腾旗.第四纪火山群

我是个火山迷，没有理由！自从广大群众发掘到乌兰哈达第四纪火山群并且成功拔草后，我对内蒙古的感情瞬间UPUP！在整个行程中我最没有想到的就是再一次与第四纪火山群擦身而过，它就是达里诺尔火山群！
达里诺尔是个自然保护区，覆盖了达里诺尔湖和火山群，以及周边草原和沼泽。火山群在达里湖畔东侧2000多平方千米的火山熔岩台地，约有56座保存完好的火山口，形成东北九大火山群之一——达里诺尔火山群。
达里诺火山群的样子像极了乌兰哈达，休眠下的火山上长满草甸，却依稀能看到坚硬的熔岩石，四周是曾经喷射出的火山碎石，它们与骏马、牛羊和放牧人行程特别的“原上画面”。
据记载，在300万年前的第四纪，这个地区发生了两件事：一是更新世冰川来临，持续了100多万年，后来冰川逐渐消融。第二件事就是大规模火山喷发，形成了熔岩台地和一个个火山口。达里诺地区的火山曾经喷发过3次。
穿上蒙古袍的蒙古人在草原尽情奔跑，把时光拉回过去，这片草原是属于他们的——马背上的民族！
📍自驾定位“达里诺尔自然保护区”，快到终点便会看到一个个火山群，它们只是达里诺尔火山群的一小部分。

>> 克什克腾旗.达里诺尔湖

火山群距离湖区入口很近，可以说是必经之路。达里诺尔湖也是火山喷发后的产物
这里是丹顶鹤栖息的天堂。全世界的15种鹤中国占有9种，而达里诺尔湖可以找到5种！湖泊和沼泽的结合无疑是鸟类最爱，这里也成为观鸟的天堂，景区内还特别设有观鸟点。
在新生代第三纪末的100多万年前，中国北方温暖湿润，形成了很多湖泊，达里诺尔湖就是那个时期的产物，并且比现在的面积要大5-6倍。湖边是沉淀下来的盐渍，四周环绕沼泽地。
快艇是这里的热门项目，略大的湖风令湖边卷起浪花，拍击着河堤，一眼望不到头的达里诺尔湖有种海湾的既视感。
摩托滑翔机可以俯瞰湖面，480元/人，感觉超级炫酷有木有！
发现湖边沼泽一处废弃的船只，被我们评为“最有灵魂的打卡点”。斑驳的颜色很容易拍出灾难风大片。在这里我们开心的像个孩子。我想有许多风景，都是在路上才能体会得到的；每一个目的地都有让你难忘的一瞬间。
船只所在的不远处就是观鸟地，雁鸭和欧类是常客，每年10-11月会有大批白天鹅在此驻足。能否看到它们这个季节全凭运气。走在不高的芦苇地中间，天气变得温柔，像极秋天的模样，如果你爱旅拍，你会爱上这里！
📍达里湖分为北岸景区和南岸景区。北岸娱乐项目较多，观鸟长廊、自然博物馆、废弃船只都在这里；南岸包括曼陀山和碧海银滩，在曼陀山山顶可以观赏到达里湖、贡格尔草原、浑善达沙地景观。景区还有付费的摆渡车
🐦可以观赏的国家一级保护鸟类：丹顶鹤、东方白鹤、白头鹤、玉带海雕、金雕、白尾海雕、中华秋沙鸭、遗欧、大鸨、黑鹳
🐟达里诺尔湖盛产鲫鱼和华子鱼，每年冬捕的成果，镇上的餐厅可以品尝到不同做法的鱼。

>> 克什克腾旗.阴天下的达达线

对达里诺尔自然保护区的了解如若不那么深入，景区不远处的达里诺尔自然博物馆是个最佳的补习班！在不大的展馆里将达里诺尔地区的前世今生展现的淋漓尽致。地质迷、野生动物爱好者可以Get到很多内容，作为顺路的一站，还是强烈推荐来此学习一下。
蒙古包是打酱油的非参观区哈哈，觉得作为草原的必备元素随手抓拍。
博物馆的二层露台可以遥望距离我们最近的火山。这条公路与有着“最美自驾线”之称的达达线相连。达达线全长135公里，起于达里湖，终于大兴安岭南端的阿斯哈图石林。这一路时而曲折，时而笔直，天气好的时候伴随云卷云舒的景观和骏马牛羊的场面，一种画中行走的感觉。可是眼前的天气一巴掌将我们打回现实！美景都是别人相机里的，我们YY下就好了。
阴天下的火山更具有历经变迁的沧桑感
达达线两侧的草原有不少骑马体验，我和这只宝宝沟通了很久的感情哈哈，最后也没有骑哈哈哈。

久仰了！乌兰布统

>> 克什克腾旗.乌兰布统影视基地

从克什克腾旗的镇子开出来，又将开始一段漫长的旅程，开车的人颇为享受，坐车的人也一路被窗外的风景惊艳，只是一个弯道加一个直道或是多条弯道加多条直道的公路望不到尽头，此刻我们心中只有一个远方，那就是乌兰布统。
路途中总有惊喜，西拉木伦河的出现给了我们下车舒活筋骨的理由。蒙界内的每一条河流似乎都是九曲十八弯，它们的蜿蜒壮观是一张照片无法说清楚的。
开到乌兰布统界内时，也完成了阴雨转变晴天的过程。下车拍照，此刻我们也只是成千上万的游客之一。
乌兰布统很大，超过了我对常规景区所理解的概念，它可以一天走遍，也可以开上两天的车，是克什克腾地址公园九大园区之一。
来列数一下属于乌兰布统范畴的地方：花轱辘台、公主湖、盘龙峡谷、将军泡子、影视外景基地、跑马场、百草敖包、金莲花滩、熙水泉、小红山、大红山、军马场村、喇嘛山生态保护区、十二连营、赛马场、浩斯敖包村、德日乌苏牧场、蘑菇场泡子、五彩山、杨树背跑马场、红松湖、北沟、横立山、野鸭湖、夹皮沟、欧式风光区、跑马场、佟国纲将军墓、烟子窑湖、象群山、桦木沟、蛤蟆坝景区。
乌兰布统的地貌是丰富起伏的，色彩也是多样有层次的，每走过一处风景都是超广角景象，乌兰布统一年四季都在释放不一样的美。
如果说乌兰布统是一幅实景风光片，那这张照片的精髓就是乌兰布统影视基地。大概只有这样的景色才能得到国家的认可和影视剧组的垂帘吧！无数张摄影师相机里的定格都献给了乌兰布统，数十部影视作品的拍摄都曾在这里取景。
乌兰布统所谓的影视基地就是抬头可以仰望云卷云舒的天空，闭上眼睛可以闻见大地的芬香、感受清风拂面的凉爽，远处可以欣赏整片草原旷世之美的地方。这里没有景区到点关闭的大门，只有川流不息的车辆和忙碌的影视剧剧组。
我们没有深入基地内部，经过的时候有剧组正在拍戏，外来车辆不是很方便，靠近公路有个不高的山头已经搭起步栈道，走上去你的视野顿时plus两个加号。
《还珠格格》《康熙王朝》《新射雕英雄传》《狼图腾》。。。。。。这些承载儿时记忆的电视剧都曾在这里取景。看到这样的场景好想大喊：“尔康，你的紫薇还好吗”！🤣😂
山下遇到刚刚满月的小鹿，牧民家养的哦，呆萌呆萌的敲可爱，它唯一的本领就是不停地吃草，一个眼神都不给我💔。
深入影视基地内部，车辆和人群渐少，广袤的草原长着不高不低稀疏的树木，天色渐渐暗了下来，前方的晴空万里已被山上的十八级妖风吹走，眼下迎来的便是另一个远方飘过的乌云，也正是它们的出现，成就了不同风格的照片，6点多钟的草原在没有太阳的光芒下，依然清新脱俗。

上午的草原穿越本来计划会有野炊活动，鉴于大家骑马不易改为可以吹空调的农家乐用餐了。吃上饭的时候是下午2点多，今天中午的饭菜格外香，每个人平均进食2-3碗米饭，一桌子菜一扫而空🤣。对于下午的行程，蒙古领队说上午没有琦爽，下午要策马奔腾起来带大家飞，穿越更加原始的区域，我们跟拍车队就不能随行了。
三点出发，这一别就是五个小时后的酒店见了！没错！他们下午整整穿越了5个小时，这一路真的是策马奔腾了，趟河流走陡坡没信号，回来个个负伤，没摔几个跟头都不好意思说下午骑马了！
即便如此，他们一致认为这都是最难忘的体验，这才是真正内蒙古旅行应该有的样子。此时此刻为你们的坚强点赞！

接近尾声的自驾狂想曲

>> 乌兰布统.公主湖

旅途开始的时候觉得时间过得很慢，旅途行进到最后内心又在抱怨时间流逝的很快，今天的天气难得的给力，好风光+好天气可以说浓缩了这几天的所有精华。虽然期间经历爆胎的意外，但是解决问题的办法总是多过困难。
公主湖位于乌兰布统的核心位置，它不同于面积广阔的达里诺尔湖，公主湖是小鸟依人的，有着塞外明珠的气质。
说起公主湖的由来，还是康熙王朝的历史，康熙皇帝的三公主蓝齐格格被迫嫁给葛尓丹，途经内蒙草原悲极而泣泪成湖，便成就了今天的公主湖。她虽然违心远嫁，最终却被葛尓丹的真情所打动，有情人终成眷属。
公主湖在这个季节也是车流人流不息，站在观景台上远望，好像看到了这位公主父命难违、君命难抗的复杂心情。
航拍下的公主湖远比肉眼所见要大的多，蓝天白云衬托下的公主湖，有了牛羊的点缀才是真正的塞外风光。
📍河对岸游客过不去，建议带长焦相机

>> 乌兰布统.红山军马场

红山军马场曾是蒙古族重要的游牧区，在历史上也有一定的地位，距离乌兰布统镇很近。在建国初期这里还是重要的军事基地，边防部队的骑兵所骑马匹就由红山军马场输送，只不过现在已经失去了原有的作用，成为游客观光的地方。
说好的马匹呢，可能今天全部失联了~打卡留念
把车停在路旁，寻找这片草原的“野趣”

>> 乌兰布统.蛤蟆坝

蛤蟆坝位于乌兰布统比较偏远的一处，开往赤峰的路途正巧路过，这里不仅有草原风光，还有花海、白桦林和一处野生鹿园，很想在这里多浪费些时间。
从入口进入需要开车前行，每一个景点的距离都很远，但却是不会失望的精品之地！
【1】鹿场
从入口开车4.4公里便是鹿场，原以为这里饲养萌萌的小鹿，就像奈良一样可以与它们亲密接触，然而人家的鹿场是原生态的，下午时分小鹿们全都躲在看似近实则远的小树林里避暑，过着悠闲的“鹿生”，没有望远镜和长焦相机很难抓到它们的神态，也正因如此它们的生活得到了不被打扰的片刻宁静。
小树林里大概藏着4-5只鹿吧，是否是野生鹿我们也没去求证。
【2】蛤蟆坝水库
整个景区的核心，航拍下真的就像一只蛤蟆趴在水库中间。停车场两侧分别都有观景台，可以不同方位的欣赏“蛤蟆头”。
蛤蟆坝的景色实在太好看了，远方淡淡的五彩颜色就是花海，好想跨过山和大海走向你~
水库风光也不逊色，今天的风光大片一茬接一茬，导致发朋友圈都有选择困难症了。
这里本想拍一组灭霸同款照片玩玩，发现缺少钻石手套一点都不威武，拍完怎么感觉有点智取威虎山的劲头哈哈哈哈
📍蛤蟆坝值得去的地方：水库、鹿园、花海、摄影基地、白桦林（我好像都给写上了哇哈哈哈，但是真的每一处都很美）
🎫门票30元

>> 克什克腾旗.美哭的京乌线

在前往赤峰的路上我们不仅偶遇一片油菜花田（事实上这是第二次相遇，猿粪呐猿粪！），而且还途经了真正的“草原天路”。不得不承认内蒙古的公路是真正的拍照天堂，随时随地都有想要停下的冲动。小伙伴此时已经用尖叫声代替言语上的夸赞了，我们既不敢停在路边，又想欣赏这不可言喻的景色，只能放慢车速，把相机调制运动模式抓拍，镜头前的这条公路每一帧都配得上windows桌面。
经乌线是开往赤峰的必经之路，它的美缘起这片画卷绸缎般的油菜花田，第二次相遇终于迎来天高云低的样子。一侧是笔直的公路，一侧是拥有天时地利的油菜花田，幸福来得刚刚好。
这片油菜花不属于私人所有，而是内蒙古政府献给自驾游客的礼物。一米多高的油菜花在这里生长的非常好，每一个经过的车辆都很难不被这片风景吸引，停下来的人们小心翼翼的拍下这难能可贵的盛夏时光，印证着2019年旅途的过往。
真的很少见到这么高的油菜花丛，它们在风中摇曳并顽强的生长，此时此刻只希望它们可以永不凋零，成为这画卷中最美的风景！
从前面花丛中辟出来的土路走到里面的草原，有了草垛的点缀让这里的风景增添不少层次，小伙伴们真的都快被美哭了！！
此后遭遇了爆胎事件，在经历了路边换胎、补胎、打气、加油等一系列工作后，我们终于迎来“雨后的彩虹”——与天交相辉映的火烧云，当然也要感谢老司机的技能加持和左右互搏术！
这一拍也为整个旅途完美收官。

“当你看过世界，你才发现你要见的世面，是你自己的勇敢和自信。”

不可辜负的蒙古美食

达里诺尔湖的鱼
达里诺尔湖盛产鲫鱼和华子鱼，每年4月下旬和5月上旬，湖水解冻，华子鱼逆流而上开始产卵，到了12月中下旬，就是达里湖冬捕的季节，最多一网可以打出6万公斤鱼。当地蒙古人为了生态平衡，也约定每年冬捕数量不会超过30万公斤！
达里湖的镇上有很多可以品尝河鲜的餐厅，一顿全鱼宴也太爽了吧！
煎炸、炖、汤，这一桌全齐了
蕨菜：赤峰五大山珍，在多肉的内蒙，能有这样清爽的青菜实在太感动了！
内蒙的很多地方都有类似这样的老汽水（可能叫法不同），一定不能错过，有种淡淡的香槟和泡泡糖的味道~
奶茶与牛羊肉是绝配
在蒙古包里吃一顿传统蒙餐是来内蒙古必做的正经事！什么烤全羊、烤羊排、牛肉干、蒙古奶茶，小二，各给我们来一份！
蒙古人的待客之道是一杯热奶茶或一杯蒙古酒配上新鲜出炉的烤全羊在蒙古包里围成一圈，干了这杯我们就是兄弟！
内蒙古的牛羊肉质地都是上乘的，新鲜出炉的牛羊肉十分地道，我想任何人都不会错过吧~吃烤全羊也不需要含蓄，要符合蒙古人大口吃肉大口喝酒的特性，一口下去人生瞬间得到升华的感觉哈哈哈哈
对女生而言，蒙古酒可能太烈，奶茶就是最好的替代品，现在内蒙古为了迎合远方游客的口味，将奶茶口味已经改良许多，非常好喝，端上来顿时喷香四溢。

草原出行指南

内蒙古住宿
锡林浩特和赤峰都属于市级城市，住宿选择比较多，星级酒店和特色酒店并存。而乌兰布统景区这样的县镇级酒店条件一般，选择时多看看有关硬件设施的评价，也可以选择蒙古包特色住宿体验。
锡林浩特住在了蒙古利亚大酒店，赤峰住在了万达嘉华酒店，这两家酒店条件都不错，时尚与特色兼具。
★锡林浩特：蒙古利亚酒店
地址：腾图国际广场6号
★赤峰：富力万达嘉华酒店
地址：西拉沐沦大街12号
因为到达时间比较晚，所以没有特地拍照，国际五星水准。
蒙古利亚酒店开业不久，充满蒙古特色，一进大堂满满蒙古气息，很多会议婚礼都在这办。而且房间也很大，早餐更是蒙古特色
房间的陈列都以蒙古贵族的标准置办，四处都以草原和马为主题
蒙古茶和小食随时品尝~
卫浴隔离，卫生间还配备了洁身器！超赞！
自驾包车
五天的路线图和行程公里数在前面都已经分别列出，车辆部分我们一共10个人，一辆大众途锐SUV，一辆别克商务。大众途锐表现不错，洼地、小坡和沙地都能从容开过，别克商务就仅限于平坦的土路和柏油马路了，所以如果行程中是常规线路日常家用轿车完全没问题，如果想要玩的野一点一定要有越野车！另外一定要自带备胎，公路自驾还是有风险的！
天气与着装
内蒙古平均海拔在1000米以上，日间气温大概是25-30度，如果没有太阳还会低一些，晚上气温一般都是20度左右，薄外套、防晒服、长短裤还是需要的。
其他：防晒帽、太阳镜、防晒霜。如果有骑马等户外项目一定要穿长衣长裤，否则会晒得很惨！
