云南大理，一座声名在外的古城，这里四季如春，浓厚富有深度的历史，古朴幽静，这是一座来了就不忍离去，离去后又止不住挂念的古城，想来也是，自己走过那么多个城市，只有大理让我心生眷恋。这片土地，带着安静而神奇的力量，治愈每一位抵达的游子。

这是个充满包容的地方，没有无形中的隔阂，没有远道而来的疏离，即使满身旅尘，此处也让人心安。找一个时间踏上一段旅途，探访苍山与洱海的绝美秘境，感受这一场只属于你我的风花雪月。

注意事项

1：大理四季如春，常年气温一般在15-23度左右 只是早晚温差有点大 早晚需要穿一件外套 白天的话就是短袖/长袖/衬衣都是可以的。如果是要坐索道去苍山游玩，记得一定要提前带上一件羽绒服，因为苍山上面气温非常的低，当然了山上也是有出租羽绒服的地方，60块钱一件。
2：云南大理海拔高，紫外线特别强，防晒一定要做好措施，风也大，一定要带上墨镜和 高倍数的防晒霜，建议带点美白和补水面膜吧，还有滋润的护手霜或者身体乳，还有润唇膏可以随时随地补水哦。
证件：身份证、银行卡、部分现金等。
3、器材：风景优美带上一切拍摄的器材把，手机、相机、充电器、充电宝。
4、药品：晕车药、感冒药、消炎药、外伤药、红景天、葡萄糖…以及个人常用药品。
5、防晒用品：高原紫外线极强，建议带上防晒霜（spf50以上）、润唇膏、太阳镜和帽子以免灼伤皮肤和眼睛

风雪故城位于大理古城中心地段，近东门处，交通十分便捷。五分钟步行可达人民路，十分钟可到洋人街，缓慢的生活节奏是属于古城里的惬意，这个柔情似水的故城，随悠远的历史激起涟漪清波，让繁复的思绪在古城纵情地延伸蔓延开来。

菌珍坊，野生菌菇汤晚宴

风雪古城的晚餐——【菌珍坊】的野生菌菇汤晚宴。野山菌十分的新鲜，熬煮的汤锅非常具有胶质感，香醇浓郁、百种菌百种味，随意搭配更是滋味万干。健康养生，满满的氨基酸补给，就算全部吃完也不用担心会胖哦。
在大理的第一顿大餐呢，讲究的就是一个“鲜美”菌珍的滋味，鸡枞菌内厚肥硕，鮮甜香脂；见手青好像也是牛肝菌的一种呢，更丝滑柔嫩一些。当土鸡汤滚开后，一人一碗的喝起来就可以放入新鲜的菌菇，当然了这个时候如果你足够心细就会发现，手边只有汤匙却没有筷子，那是因为刚下进汤的菌菇是需要25分钟的沸腾，确保完全煮熟入味后才能让客人享用哦！满满的鸡汤加上菌菇的鲜美可想而知有多好吃。

服务员也都好周到热情，根据客人口味推荐菜品，尤其菌珍坊的小酒也值得品尝哦，自己是女生最爱的玫瑰酿，香味扑鼻的玫瑰加上甜而不辣的口感，搭配这一大桌的菌菇火锅实在是妙极！！最后推荐的菌菇炒饭也值得一试哦。

【宝成府】始建于1928年－1939年间，是当时有名的商人严子珍之子严宝成出资建的。严子珍曾出资建立喜洲第一所医院，还修建了五台中学，这所学校在二战时曾被华中师范大挙所用。严成捐资30万人民币，用于支持抗日战争，开且捐资请医生们救治受战争影响的伤兵和老百姓们。1950年之后宝成府则一直作为军队营地，所以在文革期间，才幸免了红卫兵的破坏。
宝成府是革命前修建的同奏建筑里最大的。林叠夫妇在成功修复并妥善经警理喜林苑之后（2008年正式开止），政府在2013年时邀请林登夫妇接手。

体验蓝续古法扎染，感受非物质文化遗产

【古法蓝续扎染】
第一批列入国家非物质文化遗产的扎染技艺本质上是一种防染技术，将手纺布不需要染色的位置结扎、捆绑，浸入由板蓝根等天然植物原料制成的染液，通过阳光固色最后做出样式各异的图案。

周城的扎染历史久远，针脚精细缜密，丰富的图案就有浓厚的白族文化特色。蓝续坐落在一个有百年历史的白族“四合五天井”百年古宅里。这里经常高朋满座，欢声笑语，大家通过亲自制作扎染，聆听老艺人现场分享精彩故事，感受和传播白族的传统文化。

扎染是周城白族人民明末清初以来的民间传统工艺，扎染工艺品集文化、艺术为一体。扎染布由手工针缝线扎，反复冷染浸制而成，显色青里带翠，凝重素雅。用扎染布制作的工艺品既充满浓浓的民族风味，也很具现代韵味。

以青砖、白墙、淡墨画为主的白族民族民居为主的双廊，历史悠久，文化灿烂。是大理地区重要的新石器时代和青铜器时代文明的发祥地之一，这里拥有保存完整的明清历史文化街区和百处典型白族明清传统民居院落。镇域三面环山，一面临海，西眺苍山十九峰，东靠鸡足山麓，集苍山洱海之精华，自然风光旖旎，民族风情浓郁，素有"大理风光在苍洱，苍洱风光在双廊"的美誉。

巍山古城

   执一片清瓦，煮一把清茶。繁华尽处，寻一处无人山谷，建一木制小屋，铺一青石小路，与你晨钟暮鼓，安之若素。
『巍山古城』据记载，早在汉武帝时，这里就已经开始设县，现存的巍山古城建于明洪武二十二年（公元1389年），至今已有600多年的历史。整个古城比较完好地保存着明清时代的棋盘式建筑格局。

相比大理古城，地处云南西部，属于红河源头的巍山古城格外的安静与祥和，整洁干净的街道，民风古朴，深厚的文化底蕴，造就了巍山人神闲气定的生活性格。
巍山古城
巍山古城，地处云南西部哀牢山麓，红河源头的巍山，是一座具有浓郁地方历史文化特色的城市，是中国历史文化名城，同时也是南诏国的发祥地。古城在唐代初就有村舍了，元代的时候，段氏总管开始建筑土城，到明朝才正式建砖城，至今整座县城较为完整的保持了600多年前建城时候的棋盘格局，是中国保存最完好的明清古建筑群之一。巍山还是云南推行土司制度时间最长的地区之一，其悠久的历史孕育了众多寺观庙宇。
巍山古城
巍山古城东靠文华山，南有宝山和锦溪河，西有阳瓜江，形成山环水抱态势和万山拱城的格局。据史籍记載，巍山古城在唐宋时为宗教场所。

进上坊
位于拱辰楼和星拱楼之间、即北街与后所街及关圣街的十字街口处的石牌坊、称之为进土坊。进士坊始建于明代，为万历丁未（1607）进士、四川重庆府巴县知县陈于宸立。原坊为面宽三间设斗拱的石坊，坊的明间前后各立一对高大石以辅助支撑石质立柱，柱顶亦雕有四个石狮，比例匀称，雕工精湛，采用青石建造，为一高两低三跨石坊。立柱顶端雕有莲花柱头。南面坊上题有“花封瑞凤”，北面坊上题有“蕊榜文龙”。进土坊无楼，仅有上下额枋、區额、雀替，次楼匾额处为吉样花饰，四柱前后有石痴抱鼓石，其上皆有雕塑，为浮雕。

巍山是南诏的发祥地和南诏文化的源流地，是云南省十大历史文化旅游项目一一南诏历史文化大景区的核心区。南诏在巍山经营92年，统治我国西南地区长达254年，给后人留下了众多弥足珍贵的历史文化遗产。修建于明洪武二十三年的拱辰楼、保存完好的古城古建筑群、长达二千米长的古街、土主庙等一大批历史实物形态，以及延续千年的土主崇拜，源远流长的彝族打歌、洞经音乐、扎染艺术、火塘文化、饮食文化、马帮文化，独树一帜的雕刻、绘画艺术等众多和谐生态文化，无不体现厚重的南语历史文化底蕴。

南诏古街由明清时期保留至今的月华街、日异街、北街和南街组成，由原文献楼（群力门）始，向南通过地官坊，穿过雄伟的拱辰楼，经进士坊到原古城中心星拱楼，直达原南门城楼。一条古街串连四座古楼，头尾衔接，相得益彰，它像一组音符发出高低音的共鸣。古街中轴线有直、有折、有曲，使近2000米长的古街灵活而不呆板，富有城市景观的情趣。
