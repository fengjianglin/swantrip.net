清东陵，世遗探秘

前一阵的《延禧攻略》和《如懿传》的播出，乾隆可谓是圈粉无数，他的爱妃们也着实让我们爱屋及乌了一回。皇室错综复杂的人物关系堪比函数的解题方程式了。生前的阶级位分除了影响自己的家族荣辱，也是关系到死后的如何安葬。
清东陵里安葬着清朝的诸位皇帝和小主们，一共15座陵寝，有5位皇帝、15位皇后、136位妃嫔、3位阿哥、2位公主共161人（人物方程式在脑中迅速出现）。目前并不是所有陵寝都对外开放，每个陵墓如果都细细了解安葬人物的生平故事，恐怕是一天也听不完的。
清东陵作为世界文化遗产名录中的一员，现在是国家5A景区。陵区以昌瑞山主峰下的孝陵为中轴线，依山势呈扇形东西排列，总体布局为"前朝后寝"。建筑风格自然是延续了宫廷风的红墙黄瓦，虽然没有故宫维护精心，但是整个陵寝更多了一分历史存在的真实感。
因为下午抵达的比较晚，只参观了乾隆的裕陵和慈禧陵。冬季，这里更加静谧，整个园区像是我们专属。
裕陵，没错了这里就是《延禧攻略》和《如懿传》中乾隆皇帝弘历的陵寝，气势当然要弘大些，前面的大殿展示着部分陪葬的出土文物，那些工艺在现在都是首屈一指的。
裕陵的地宫神秘，却又是一座艺术的殿堂。外面的寒风已经入骨的冷，进入地宫好像一个温暖的居室，冬暖夏凉是地宫的神奇之处。
地宫内布满精美的佛像、经文和石雕，庄严又神圣，在皇帝崩世后，才会将地宫的石门关闭。裕陵内除了葬有乾隆皇帝，还有他生前的两位皇后孝贤和孝仪，以及慧贤、哲悯、淑嘉三位皇贵妃。当然皇帝的陵寝也只有皇贵妃以上位分才有资格被安葬于此。
裕陵大殿展示的陪葬的一些遗物，想想都是几百年的东西，可以保存到现在的成色，已经很完美了！
慈禧陵位于裕陵的西侧，三殿用料考究，装修豪华，店内全部采用黄花梨木（据说现在都需要东南亚进口），柱身是鎏金铜龙并有砖雕贴金装饰，陪葬品自然也是价值连城。死去后的慈禧被安葬于此时，口中还放入了价值不菲的夜明珠，传说因为这颗夜明珠，慈禧奶奶的尸身多年未腐烂，肌肤保有弹性。慈禧奶奶果然是享尽了人世间的荣华富贵。
从慈禧奶奶的地宫上来，可以到清东陵的至高点宝顶观景，正赶上黄昏时分，延绵的群山和洒在清东陵的陵寝间落日，好像佛光普照。

泡汤才是冬天的正经事

泡汤是每年入冬后必要提上日程的一件事，从清东陵出来天色已晚，大家也已经冻得瑟瑟发抖，今晚的落脚处非温泉莫属了。距离清东陵30分钟车程的遵化市有一处中高端温泉度假村——汤泉宫温泉，开业时间不久。这个村庄的千年古泉，在帝王时代就是皇室最爱，汤泉出水温度68℃，养生效果棒棒哒，正因如此千百年后才被选中为温泉度假村。
度假村内的自助餐厅，晚餐菜品比较丰富，冷热菜、汤品、水果一应俱全，还有儿童专属的取餐台，很Q萌的感觉
室内温泉区打造成热带雨林风，出了自助餐厅就可以看到，有泡池和泳池，还有儿童专属的戏水区
二层是汗蒸区域，有不同功效作用的汗蒸房间，我比较钟爱的就是盐疗房了，很适合北方爱雾霾的天气。
汤泉宫室内外温泉大概有50个特色泡池，相比室内，我更喜欢依山而建的室外温泉，更有真正泡汤的意境。山上还特别建有VIP的小院泡池，私密感很强，中途遇到服务员会主动递上一杯热热的姜枣茶，真的太好喝了！总是为了这杯茶故意走到服务员的驿站哈哈。
山上的泡池大概有27个，可能这里的冬季太寒冷，室外汤泉比没有太多人来泡，每次走过山间这些热气腾腾的汤池，都感觉自己是穿越剧中的女主角哈哈哈。

后工业时代与文艺的碰撞

来到唐山，怎么也要去市区转转，都知道唐山是个工业化很浓的城市，市区的建筑风格将工业气息加入了文艺色彩，不再是曾经破旧的小城。去年新开的唐山工业博物馆走起了时尚科技路线；汉斯昆德故居的大House成为城市青年喜欢的新去处；开滦国家矿山公园让人真真切切感受了一把矿井作业。这里可玩可逛的地方恐怕一个周末的时间是不够的。
壹：中国（唐山）工业博物馆
唐山大大小小的博物馆不在少数，如果说将工业历史和现代科技融合的最好的就是中国唐山工业博物馆了，与其说是一个纯粹的博物馆，不如说这里是个综合性的工业旅游区。其功能定位为唐山工业博物馆总馆，从城市工业发展切入讲述近代工业，区别其他专业博物馆，对已有博物馆具有导引作用。
充满工业做旧风格的走廊，也是一处激发人好奇心的艺术作品，博物馆的建筑全部利用废旧钢铁设计，更加突出了这里的主题。园内有四栋灰色外墙的建筑是日伪时期的弹药库。
入门处的这台火车头是李鸿章视察我国第一台自产0号机车的首发情况的雕塑，象征中国第一条铁路的修建和第一台蒸汽机车的投入使用，也代表近现代工业的开端。中间配有LED电视，你可以看到工业博物馆的“序篇”。
走过唐山百年工业之路文化长廊，这是整个博物馆布局中的“一轴”，沿途可以了解中国工业历史的发展历程。
进入博物馆的主馆“工业之路”文明馆，大致有两层，一层脚下的工业历史沙盘诉说着从1876年开平矿务局筹建，到2010年唐山重点工业建成的里程故事，好像一块巨大的拼图，让我们从每个历史节点走过，由讲解员带领回顾着当时的情景。
展墙上的浮雕火车头，令参观者很有带入感。
3D图展示墙。This is Tangshan Station.
中国自建的第一条铁路是从唐山到胥各庄，1889年为慈禧太后专门制造了一辆“銮興龙车”，车身两侧镀金雕龙，内部用珍稀木材制作，还有“龙床”和“宝座”，相当豪华。博物馆将当年慈禧奶奶坐过的专属列车内景还原，并通过液晶电视播放当年的历史情景，参观者坐在列车里，时光倒退，体验感实在太棒了！

除了主馆“工业之路”文明馆，还有六个副馆：“工业之星”展演中心、“汉斯.昆德”生活馆、“詹天佑”实践馆、“茅以升”数字馆、“金达”音乐吧和“周学熙”会客厅。副馆都不大，更加注重主题性和体验互动性。副馆更加“时髦”些。

金达音乐吧的亮点是机器人送餐，真正现代生活与科技的结合。音乐吧艺术感很强，满满小清新的色调，不定期的live演出，让繁忙的工作放松不少。

贰：汉斯.昆德故居博物馆
昆德家族是来自德国的烧窑匠世家，至少五代人都是烧窑工匠。汉斯.昆德是1896年来到中国的，一开始只是在山东青岛胶州湾考察当地的土壤和矿物质，后在此创业，曾用当地原材料生产了质优价廉的水泥，在战乱期间，汉斯为唐山的工业有着功不可没的贡献，这里就是他和他的家族在华期间住的别墅，由唐山细棉土厂赠送，在经历了1976的唐山大地震仍然屹立不倒，可见整个建筑质地优良。
汉斯.昆德的故居不只是一座别墅大house，还有德国啤酒花园、丹麦小屋、阳光玻璃房、老唐陶办公楼和临街商铺等。将20世纪初期中国工业状况和那时引进的西方技术相结合，想象下100年前的中国城市有一处这样的居所，喝着红酒弹奏钢琴，闲暇时间还能去北戴河度个假，这可能是当时的中国人想都不敢想的事情，即便在今天，这样的小资生活也是属于少数人的。
别墅内光线通透，有一些汉斯昆德在中国工作和生活的历史片段，也有那一时期中国社会工业化的历史说明。另有两个房间将汉斯昆德的生活场景进行还原，让人无时无刻不感受到那时的奢华与生活的
原色木地板，暗纹壁纸，做旧的沙发和座椅，精美的摆件，无不彰显汉斯先生对生活品质的追求。这种欧式装修风格现在是多少家庭心中的dream house啊。
汉斯的别墅就是孙燕姿那首《完美的一天》最好的演绎
“我要一所大房子
有很大的落地窗户
阳光洒在地板上”
丹麦小屋，一栋百年老建筑，原启新瓷厂引进的第一批单买设备存放于此，当时有很多丹麦工人在此工作。现在被打造成休闲的咖啡厅，依然以丹麦文化为背景，供应北欧风情的咖啡和甜点，墙上的壁画仿佛走进安徒生的童话世界，享受哥本哈根午后的阳光。
丹麦小屋旁边的阳光玻璃房是它的延伸，因建筑为全玻璃建造，冬日的阳光照射进来温暖了整个房间，喝杯咖啡和朋友在这里度过慵懒的一天。
汉斯啤酒花园专门是以德式啤酒花园为设计理念，在唐山仅此一家。招牌当然是德国进口啤酒了，加上德国醇正的烤香肠，这里一定是青年男女夏天最爱的场所。

这里是我最期待的地方，源于对矿井和蒸汽火车莫名的喜爱，整个矿山公园可以说唐山甚至是中国工业史的微缩版，拥有140年的开采历史，一座座老旧厂房和工业器物零距离的展现在我们眼前，似乎百年时间未曾走远。曾经的矿山宝藏现在变身文旅基地，包括开滦博物馆主馆、井下探秘游、中国第一佳矿1878、电力纪元1906、蒸汽机车观光园、中国铁路源头博物馆等一系列展馆和景区。
最佳也是必游的线路推荐：开滦博物馆——井下探秘——中国铁路源头博物馆——蒸汽机车观光园
开滦博物馆
屹立于园区的工业烟囱，一种百年的沧桑感。
博物馆以实物展示、多媒体展示、场景还原等展现中国路况之源和唐山的城市记忆。一件件珍贵的展品诉说尘封往事。这里记录着开滦创造的20多个中国近现代工业史上的第一
达道，1899年，为联系西北井煤矿和唐山矿，在广东街马路路基开凿了一条南北走向的拱形卷砌式铁路桥涵隧道，这是中国早期铁路
镇店之宝来啦，尘封百年的“羊皮蒙面大账本”，真的超级大，出自歪果仁之手，侧面的花纹也是防止做假账而设置的，可见在没有电子高科技的时候，还是有很多别的方法防伪的。
里面的字迹工整的像是印刷出来，虽然过百年，但是仍然清晰，可见纸张和墨水都是品质极好的。据说对于记账人的要求也是很高的。
旧开滦使用的老式电风扇
博物馆内展示的一件件老物件都保存的很完好
井下探秘游是整个矿山公园的一个亮点，由位于博物馆地下数十米的煤矿井下老巷道改造而成。戴上矿工安全帽，走入幽深的井下隧道，一股股发霉的味道，从原始挖煤到现代化采煤的九个真实工作面让人真切的体验挖矿的不易和危险。
地下矿井有运输大巷、原始采煤工作面、落垛采煤工作面、爆破采煤工作面、普通掘进采煤工作面、高档普通机械化采煤工作面、普通机械化采煤工作面和开拓巷道，形成交错的地下迷宫。
真实的采矿人员场景还原
在战乱期间，地下矿井还有一个作用就是防空洞，里面的灶台和文革时期鼓舞士气的背景音乐好像一下穿越了回去。
地下休闲吧是井下探秘游的一处休息区，昏暗的灯光，工业桌椅，手捧一杯咖啡，比那些网红咖啡厅更具一份别样滋味。
最后搭乘和谐号矿井小火车驶回地面，伴随哐哐哐的火车轨道声音，结束了矿井之旅。
开滦国家矿山公园还有一处不能错过的地方就是蒸汽机车观光园，包括中国铁路源头博物馆、蒸汽机车展厅、唐山老火车站（复建）、复原龙号机车、唐胥铁路景观天桥等。从“矿井”中出来转到中国铁路源头博物馆，中国铁路的源头就是唐胥铁路啦。这里讲述蒸汽机车、中国铁路前世今生，算是开滦博物馆的一个续篇，适合铁路迷。蒸汽机车展厅内摆放着复原的“可以开动的龙号机车”，展厅外就是必打卡的“中国铁路源头”纪念碑。
园区复建的唐山老火车站是136年前的场景，放置着几列不同款式的老火车，从火车的外观可以追寻到它们的生命变革和经历的历史沧桑。它们真实的存在让人恍若昨日。
地址：唐山市路南区新华东道54号
交通：18路矿务局站，2路路北区委区政府站，K2路建国路站，6路车站路口站

海上温泉，月岛

晚上驱车前往曹妃甸，临海的新区少了夏天的喧嚣，把车停在三贝码头准备赶最后一班船上岛。整个码头安静的好像我们专属。三贝码头是唐山湾的游客中心，很国际化，虽然现在并无太多游人前往，但仍能看到坚守岗位的工作人员。
唐山湾有三个海滨小岛：月岛、菩提岛和祥云岛，从三贝码头出发都可以前往这几个岛屿。船程大概30分钟，冬天班次比较少，需要查看官网订票。行驶的都是封闭式游轮，并不会感觉颠簸，一路很平稳。
月岛上面五颜六色的度假小屋，入住率都不是太高，有些没有开放
这就是我们今晚的住所啦，华清月府温泉酒店，推开阳台的门就是大海，还有专属的温泉，冬季的大海虽然寒风凛凛，但是泡个海上温泉还是极好的。冬天的价格也比夏天便宜不少
温泉与大海，很有圣托里尼洞穴酒店的感觉
冬天的海是第一次见到，那是一种冰封的尘世，将整个世界都凝固住了
酒店四处都是冒着热气儿的温泉池
内部的装修风格很有东南亚的赶脚
月岛最著名的小木屋啦，分了很多区，每个区的小木屋都是不同特色的，冬天似乎不对外营业，夏天这里经常爆满订不到房间。拍婚纱照来这里取景也很棒！
月岛游客中心，也是码头，前面的风情街打造的很浪漫，夜景一定很漂亮。
地址：唐山市乐亭县
价格参考：10月20日—4月20日两岛套票580元/人
包含往返船票+1晚住宿（望海庐、闻涛阁）+温泉+（1晚餐1早餐）+菩提门票+菩提岛环岛小火车，如果是住华清乐府加50元/人，单房差是150元
Tips：套票5人才可发船

河豚小镇赶个海

来之前就听朋友说要来曹妃甸的十里海河豚小镇吃顿河豚宴，以前只在日本吃过，不知道国内的海边也有。整个河豚小镇其实是个小渔村，因养殖河豚出名，当地政府将这里打造成休闲度假和文化体验的生态小镇。渔村的设施被翻新后充满了趣味，很多设施还在筹划中，相信明年来这就又是一番新景象了。
整个小镇的墙壁都被涂鸦，以渔村的捕鱼故事为背景，很有美帝街头的感觉。
十里渔家是河豚小镇以河豚鱼为主题的餐厅，菜品都很新鲜，比起日本价格要划算很多，超推荐哦！
河豚小镇现在在打造文创产业了，渔村已经不是只会捕鱼养殖的渔村，在十里渔家的联排房子里，专门设置了文创区域，这些手办礼很适合年轻人，每件都很卡哇伊~~
十里渔家还有几间特色的民房改造成了河豚主题民宿，非常有爱，整个房间被河豚围绕，卫生间的香皂都是河豚君~原木色基调也很适合乡村调调。询问了老板大概是680元一个院子，夏天可以再来感受下。
整个河豚小镇并不大，一条主街贯穿，据说夏天慕名前来的人还是超多的
小镇还有一处很适合亲子的乡村动物园，有羊驼、大鹅、山鸡、兔子、山羊、孔雀、鸵鸟、矮马和三只超活泼的大金毛、暇步士和萨摩耶。每只动物都萌萌的而且很喜欢和人亲近，可以在这里泡上一下午。
重头戏来了，河豚宴，服务员将河豚拿出来给我们展示它生气的样子哈哈。很多人都担心河豚有毒，拜托河豚的种类就很多种，餐厅能出售的必然是经过加工才能上桌的，不然也是砸自己的招牌。而且食用河豚在日本就有很悠久的历史了，河豚的营养和药用价值也是毋庸置疑的！中国目前已经对河豚鱼的养殖解禁了，养殖和加工都有药监局监管，所以。。。放心大胆的吃吧哈哈哈！
河豚宴的菜品和在日本吃到的很像：有河豚鱼刺身、红烧河豚、河豚鱼皮、河豚鱼皮冻、白汁河豚、河豚天妇罗、烤河豚白子、河豚火锅、炭烤河豚鱼等。
红烧河豚，做法和普通的红烧鱼相似，吃起来汁鲜味美
炸河豚，将河豚切成块软炸，香酥美味~
贝类和螺类大集合，水煮的做法，蘸取酱汁比较鲜美
凉拌海鲜类菜品，每只贝肉分量都很足。
河豚刺身，摆盘很精致，是保留河豚原始味道和营养价值最好的吃法
东方虾，也是店长极推的，个头很大，冬天能吃到这么鲜的海鲜真是太开心了
地址：唐山市曹妃甸区十里河海养殖场
返京途中，路过曹妃甸新区在建的恒大度假区，据说在盖城堡酒店和大型的儿童乐园，还有美食商业街和养生温泉，未来这里就会成为京津冀地区看海最近的度假区~
