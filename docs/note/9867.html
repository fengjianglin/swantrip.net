珠海海泉湾暖冬度假全攻略

游记 主题乐园 亲子 摄影 温泉 美食 2019/01/09
当白日越来越短，寒冷逐渐现形，雾气在早晨的窗户上留下痕迹，你就知道，又到了泡温泉的季节。一个好温泉值得跋山涉水，让积攒一年的身疲心累都在热腾腾、暖融融中一瞬间烟消云散。
珠海 ，位于广东省珠江口的西南部，与澳门相连，是珠三角中海洋面积最大、岛屿最多、海岸线最长的城市，这座以“浪漫之城”闻名小城也是全国唯一以整体城市景观入选“全国旅游胜地四十佳”的城市。
珠海全年温暖湿润，冬天不冷，再加上清新的空气和整洁的城市环境，一年四季都让游客流连忘返。
珠海旅游资源丰富，能代表浪漫有风景优美的情侣路、如诗如画，空灵秀逸淇澳岛和珠海渔女等，但最适合冬天旅游的，还是来这里泡天然而成的海洋温泉。
海洋温泉神奇之处在于，它是天然而成,源自大海深处，泉水自喷高度达11米，是罕见的优质海底温泉。泉水清澈透明，出水口水温常年保持在78至83摄氏度。
港中旅珠海海泉湾的海洋温泉被称为“ 南海第一泉”，缘由在于其富含多种有利于人体健康的微量元素和矿物质，且是难能可贵的天然海洋温泉，也是目前国内为数不多以海洋文化为主题打造的海洋温泉公园。
听闻海泉湾正举行中国国际温泉节，节目很多，还有优惠，趁周末，便约上几个朋友，一起出发了。

DAY 1 : 渔人码头, 《浪漫珠海》晚会, 品尝广东特色火锅
珠海西方嘉年华 异国风情的渔人码头用无人机从高空俯瞰整个海泉湾度假区，最显眼的就是渔人码头了，依内湖而建的渔人码头，充满了西方嘉年华氛围，四周全是美轮美奂的异域建筑。
湖面上波光粼粼，有些北方飞过来的白天鹅在湖面上游荡。湖边热带树木林立，湖面波光粼粼，没有城市的喧嚣，多了一份清闲与自在。

舌尖上的海泉湾

这几天广东终于入冬成功了！天气降温，谁都喜欢热乎乎的美食，来温暖冬日低温中空虚的身体。
想寻找最能搭配冬日的美食？珠海海泉湾美食攻略，让你冬天不畏外界低温。
海泉湾荟萃了全国各地的无国界美食，我们首天品尝的是小鱼仔餐厅的广东特色火锅。
俗话说“冬令进补，春天打虎”。广东人注重养生健康的生活理念早已深深刻在他们的饮食文化中。在小鱼仔餐厅，与三五知己围坐一起，吃着热气腾腾的广东特色火锅，感受冬日里的传遍身心的温暖。
当晚，我们品尝到的是滋补清汤羊肉锅和清远鸡煲两种，
秋季必点的暖身御寒煲，精选鲜嫩羊肉，慢火熬制，搭配萝卜玉米和枸杞等食材。开了后先喝一碗原味汤，简直鲜甜得心都化了。
羊肉完全吃不出膻味，厨师调配了多种沾酱料，沾着吃，很香。
清远鸡煲比较清淡，鸡肉爽脆肉质嫩滑，沾酱料也很好吃。还有些鲜虾放入一起煮。海虾虽然被穿在竹签，但仍能看到海虾有抽缩。可以有多么的新鲜。海虾汤底，互为精华，这个鸡煲就是极佳的美味。
鹅乸饭，是必推美食，鹅肉选用开平的马冈鹅，鹅肉切成小粒状，放在瓦煲和米饭一起炆制，米饭吸收了鹅肉的香味，鹅肉吸收了米饭的饭香，每口都甘香无比。
小鱼仔餐厅的装修，从桌布到墙壁，都是用白色和蓝色的主色调，在这用餐会让人有种大海气息的感觉。
还在活蹦乱跳的生海虾，新鲜看得见。
广东火锅，少了不了潮汕牛肉丸。粒粒弹牙，好味！
“温暖一冬”广东特色火锅季是海泉湾在中国国际温泉节期间推出的美食项目之一，时间到本月31日前都有。

玩雪不用“挨冻”的冰雪乐园

零下8度，一座纯冰打造的酒吧，在这里喝点小酒，绝对有意思。
美轮美奂的冰雕作品 令人心生向往
对于广东人来说，冰雪游是这几年冬游的热门主题，因为身处南方的广东人对冰雪游非常向往，所以每到冬天，都会选择去国内一些可以看雪、玩雪的地方“挨冻”。到神秘岛不用“挨冻”就能实现冰雪游。
神秘岛主题乐园内的冰雪乐园是珠海首个华南最大的主题冰雪馆。它临海而建，总占地一千平方米，用冰量达800余吨，彩色灯管数千根，展厅温度零下8度，在门口换上大衣就可以进场玩真雪，看冰雕，不用挨冻。
进城堡大门时，瞬间穿越，一秒进入“梦游仙境”，多组经典3D卡通形象冰雕缤纷呈现，栩栩如生的站在你面前卖萌耍酷，想想心都要融化了！！！
汇集了众多冰雪雕刻大师，历时近三月之久，苦心构筑，精雕细凿，成功的实现了北冰南移的神话，达到了步步有景，步移景异的效果，令人陶然欲醉，流连忘返。
展厅内以真实展现的南极海洋生物、卡通人物艺术冰雕景观为主，生动展现大自然冰川的神奇与美丽，同时还有长达160米，落差达5米的冰雪滑道，刺激，惊险；
戏雪乐园可体验打雪仗，堆雪人。千里冰封，极目远眺，置身冰雕冰雪乐园，如身临其境感受大自然的熏陶。让你赞叹大自然奇妙景观的同时，亦可释放天性，嗨玩无极限！

所谓泡汤的最佳时节正是冬季，这个周末，和朋友来珠海的目的就是为了泡泡这里的海洋天然温泉。享受包裹如熊的冬季来一刻久违的放松。
温泉的美妙诱人之处自不待说，而现在越来越多酒店也配起了温泉，至于名副其实还是名不副实，据广东省温泉协会公布真温泉的标准是：
①水温：水温达到34摄氏度以上
②水质：对人体有用的化学成分主要是氟离子、偏硅酸、锂、锶、氡等几样指标。PH值及矿化度，也被列为重要的参考指标。
港中旅海泉湾海洋温泉是作为广东首批真温泉之一，也是这次旅游我选择它的原因。源自大海深处，泉水自喷高度达11米，是罕见的优质海底温泉。
泉水清澈透明，出水口水温高达80摄氏度以上。据权威机构监测鉴定：该温泉富含钾、钠、钙、镁、硅、氯离子等30多种有益于人体的微量元素。有纤体、美肤、活血、保健等功效。中国温泉泰斗陈炎冰先生曾亲笔题词“南海第一泉”。
4万平米的温泉浴场包罗了108汤形状不一、功能各异的温泉池，囊括了古今中外丰富多彩的温泉沐浴文化。
由室内温泉、室外温泉两部分组成。室内温泉分为欧风区、东方区、揽海区；室外温泉分为中心温泉区、世界温泉文化区、温泉康养理疗区、园林温泉区、空中温泉区等。欧风区以体现欧洲沐浴文化的以色列死海温泉、古罗马温泉、传统芬兰浴为主。
东方区则集中了东方国度---中国、日本和韩国的传统温泉项目，其中含中草药桑拿室、海洋矿物泥等。海洋园林温泉，通过各类中草药、精油、咖啡等风格各异的温泉沐浴方式，使游客获得更丰富的温泉沐浴体验。
临海而建的无边界观海池和充满童趣的儿童戏水天堂、急速漂流池、泡泡池等一系列老少皆宜的温泉设施均位于中心温泉区。温泉广场高低错落的水柱随着音乐此起彼伏，或轮番上演特色文化表演，精彩不停；
世界温泉康养区内拥有先进的康养设备，让您的身心达到高度放松，实现精神减压，保健养生；温泉康养理疗区则是采用独栋布局，尽享私密养生温泉带来的非凡体验。
空中温泉是海泉区的特色，致力于为会员提供尽善尽美的专属服务和私密空间的高级会所，位于海洋温泉四楼，营建于碧海蓝天之间，让会员体验海风、阳光伴浴的奢华。 空中温泉总占地面积约为1,200平方米，环境清新韵致、雍容华贵。
由温泉汤池、木桶温泉区、干蒸桑拿房、舒适宜人的休息大厅、露天休闲平台、男女更衣室组成，功能配套完善。
同时温泉客房还拥有地中海风格、东南亚风格、日式风格、中式风格等十余间装修风格不一的至尊温泉房及温泉海景套房。空中温泉格调高雅、自然简约，热情周到的贴心服务，体验专属的私密空间，彰显出会员的尊贵身份。

世界温泉沐浴文化之旅

除了泡汤，异国风情表演是不用错过的。进入温泉区首入眼帘的就是《南泉沐歌》表演，妖娆妩媚的异国舞姬用性感的舞姿展现着异国他乡的沐浴风情，鼓声隆隆翩若惊鸿裙袖飞扬仙姿玉色尽展汉代的繁华鼎盛。
《南泉沐歌》是以西汉时期海上 丝绸之路 为背景题材进行艺术形式创作的实景表演。古韵雄风的汉代使者用竹简记录着出使异国的风土人情，舞蹈将汉风与实地场景交织融合，舞者展现出壮阔开朗的气势。
土耳其风情舞，艳丽丰满的舞娘，穿着比较暴露，很有视觉冲击力。激情撩人的音乐，现在观看中这种土耳其风情舞动作大胆、奔放，幅度很大，特别是胯部的动作非常夸张。
它不仅仅是舞蹈，还是生活，是艺术，是让生活充满激情和阳光的一项运动。泡汤的时候，听听这种急剧的鼓点节拍，又是另一种享受。
红酒浴 舞娘跳完土耳其风情舞后，会集中到土耳其浴池中，围成一个圈圈，每个美女手持一瓶红酒，将酒缓缓倒入池中。
红酒温泉颜色醉人，酒香缭绕，浸泡在温泉里，酒液从皮肤表层渗入肌理，温婉迷人。游客可在现场观看红酒温泉的调制过程，还可小酌一杯红酒观赏一段迷人的肚皮舞表演。
据悉，在中国国际温泉节期间，还有日本民俗舞、唐风沐浴舞、健康养生操、八汤送福和趣味相扑挑战大胃王等活动，泡温泉的同时欣赏到这些精彩的演出，也值回票价啦。
DAY 3： 运动俱乐部、参观体检中心、吃湘西风味土钵菜
运动俱乐部按国际化理念设计，豪华时尚。有多功能室内体育馆、健身房、有氧舞蹈室、壁球室、射箭场、乒乓球室、拳击房、台球室以及户外网球场、五人足球场、篮球场等多个健身项目。
为客人提供完善的健身服务。我们体验了射箭，有教练在旁边悉心指导，还是蛮不错的，这里的设施都很新，适合现代都市健身休闲时尚的潮流。
海泉湾体检中心，集健康体检、健康促进、健康管理于一体。与南方医院建立远程医学联盟，实现医疗资源共享、远程对接，成为多种功能为一体的综合性医疗平台。
配备有国际最先进的大型体检设备，细致贴心的导检人员，体检服务能及时有效的健康体检报告和个性化的健康促进书，其功能主要是为个人和团体提供健康评估和保健指导。身在海泉湾，即可享受省会城市最顶级医院的优质医疗服务。
美食，是人生存的基本需求，更是提高生活品质的精神追求。中国地大物博，南北美食各有差异，其内在反映的正是不同地方的生活气息和民风民俗。湖南美食以辣为主，广东美食却以清淡原味为主。
中午在粤湘情中餐厅体验湘西风味土钵菜。野味十足的食材，加上鲜辣的辣椒，简直燃爆了冬天的食欲，好吃让你吃得根本停不下来。
吊烧鸡，粤湘情的招牌，鸡肉被火烤过后，逐渐被涂上一层动人的红色，散发出让人垂涎的香味。
烤制的鸡更加香脆，皮下的脂肪在被逼走，剩下薄薄的一层鸡皮变得香脆无比。因为温度足够高，肉汁被锁在鸡肉中，鲜嫩香口。
鲜人参老鸡汤，人参是名贵的中药材，以根供药用，具有大补元气，补脾益肺，生津固脱，安神等功能。鲜人参，加入老鸡，炖制而成的汤香甜浓郁，这汤冬天喝很时令，滋补身体刚刚好。
湘西土匪鸭，是湘西的一道经典名菜。鸭肉加上辣椒等调料在土钵中炖制，口感烂而不散，有少少微辣。土匪鸭选用优质肥鸭，佐以各种香料制而成色泽鲜亮，香味浓郁，辣而不燥，回味悠长。
?鱼头炖豆腐，鱼头富含胶质蛋白，脂肪和热量都很低，食之有美容润肤之效;豆腐的蛋白质和钙含量丰富，有清热润燥、降低血脂的作用，两者一起炖制，豆腐吸收了鱼头的鲜甜味道，是一道上佳的美味。
泥蒿炒腊肉，这个菜必点吓！泥蒿清香、爽口。与湖南腊肉同炒，可谓是完美的经典搭配。腊肉中夹杂着泥蒿的清香味，泥蒿中又增添了肉香味，这种强烈的对比和冲击，让人回味无穷。
在海泉湾度假的这个周末，我不仅在海洋温泉里暖回了冻傻的自己，在童趣满满的游乐场追忆了童年，更在不用迈出国门的情况下，从听觉、视觉和味觉上体验到了满满的异国风情，最大限度的舒缓身心是这里对我无形的馈赠。
这次我们体验的是港中旅珠海海泉湾度假区，其实港中旅还有青岛海泉湾度假区、咸阳海泉湾度假区、亳州海泉湾汤王温泉，据说都非常棒，期待下次能到其他城市海泉湾感受下。
