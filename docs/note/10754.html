4天3晚成都美食探店，顺便探访美到无死角的无影教堂

游记 人文游 小众 摄影 美食 自助游 2018/11/06
秋天，不负美食，再一次来到成都，我眼中的美食城！
谈美食色变，是旅行家们的喜好，一路走一路吃，才能吃出城市的味道！
上一站甘肃行，确实有些疲惫，给自己放个假，成都来补一补，补充一下打卡地。
成都，从某些方面是我心中的理想地，万事万物以逍遥的姿态展现着……

成都民宿是我的首选，有特色，更自在，家的感觉！

比起豪华和现代化的设施服务配备，住过那么多家后，我开始越发沉淀，用心去了解各城市的民宿，体验民宿，接地气，更好地去感受当地人的生活方式，越来越多的设计师也参与其中将民宿改造成理想的生活空间，新鲜感！也是吸引我的地方。其实一个人旅行，不需要太大的空间，舒适、安全就好！
到成都还是以春熙路和太古里为中心点，因为去哪里交通都方便。这一次选择的榻榻米，也是我比较喜欢的住宿环境之一，简单、还是简单……盘坐在阳台，泡一杯咖啡，写一篇攻略，才是成都的正确打开方式……
选择民宿，还有一个原因，旅途时间长渐渐养成了随时洗衣服的习惯，一台洗衣机刚刚好！再配有一台微波炉，即使我拍摄的再晚也不用担心回去后没有饭吃饿肚子睡觉，很现实，却是很多摄影师渴望能实现的。
读李白的诗：“九天开出一成都，万户千门入画图。草树云山如锦绣，秦川得及此间无。”便会让你爱上成都。想来感受成都的繁华与秀丽，想来领略这个城市独有的风姿，想来享受那闲适散淡中让人回味无穷的茶文化和麻辣美味，想来感悟那博大精深源远流长的佛文化和道文化的余韵。
于是，我来了，一而再，再而三！

成都美食多、美景多，吃不完、玩不腻，所以享誉世界

特别安排了4天3晚，只为这美食而来，穿插打卡地发现不同季节不一样的成都美。

DAY 1【成都吃客（致民路店） 】来成都做一名吃客

安顿好就马不停蹄的前往上了2018成都美食餐厅“必吃榜”且为成都武侯区美食榜第1名的【成都吃客】，美食之所以称为美食且被广为流传赞誉，大都与其匠心有关。
位置 /
致民路48号（距离地铁3号线新南门站b口步行180米）停车在路边，或者旁边的收费停车场。
营业时间：周一至周日 11:30-14:00 17:00-22:00
环境 /
这里可以说火的一塌糊涂，开门营业前就可以看到门前的小板凳坐满了等候的人，一波进去，瞬间！小板凳又都坐满……所以说，好吃这一口一定早早来排队！进入餐厅二层，原木色调很温馨，挂满了老成都的照片、写着老成都的大街小巷的名字，亲切感爆棚！
餐食 /
在餐厅吃饭，1个人，我还是喜欢让服务员推荐上菜，总会有不一样的惊喜，都是当季最热销的菜品。
银耳雪梨汤：老式的暖瓶装，到的时候就会看出浓稠程度，良心熬制！喝上一口，酸酸甜甜的刚刚好！绝赞！
小时候糖醋小排：分量十足，小竹筒却有种吃不完的感觉，地道的川味糖醋排骨，下面还有凉红薯。
特色烤鱼片：舰艇形状的成器，表层飘红，竹签串起的鱼片浸泡其中，闻起来就很香，撸上一串香辣可口，肉食鲜嫩，绝对的良心串！
蒜香排骨：趁热吃，甜甜的蒜香味，精小排烤出了肉的香味。
雀巢掌中宝：让我惊喜的一道菜，非常有创意！树枝托起营造出了一个鸟窝，鸟窝里是辣子鸡还有鹌鹑蛋装点，更形象！平时本身也很喜欢吃辣子鸡，要对这道菜评价高分，鸡肉炸的软硬度刚刚好，有辣味却不会十分辣，让人有吃了一块还想夹下一块的冲动。
纸上焗鲜鲍：鲍鱼、蟹、丝瓜，配双椒，底下是方便面，这道菜特别适合不能吃辣的朋友，当作主食为未尝不可！份量很足！
裸奔坨坨虾：我想这道菜光是看图片就会是大家必点的！去壳去虾线的大虾谁不爱？热油里浸泡，热腾腾的辣香扑鼻而来，却毫没掩盖虾的香味，一口一只，我是这样吃完的……底下还有芋头别忘了吃哦。
冷吃牛肉：单就为这摆盘创意点个赞！小细节也彰显出成都味道！吃起来又湿润的口感非常入味，越吃越香，辣的程度应该是成都人最喜欢的。
红糖豆花：餐前吃？不！餐后吃，解辣！的确解辣！菜单上有特别提示两款解辣菜品，这也是我第一次发现餐厅会有这样的设计，相信也是应大多数客人的需求而做的！豆花，本就是成都的传统小吃，爱甜品的你千万别错过！
相思红颜：如上解辣菜品之二，剥了皮的小番茄配上桂花，绝妙！酸酸甜甜惹人爱，光是回味就已经让你忘记了先前吃的辣的味道……

DAY 2【消失的教堂】一度成为网红打卡地

经常听朋友提起成都有一处消失的教堂，被薰衣草簇拥着，天际间若隐若现的一种“透明的艺术”，从不同的角度观察，会让人有一种视觉上的幻觉感！～
位置 /
中法农业科技园形象示范区（黄龙溪景区旁）
无人机刚起飞就被拦下了，此处据说有公司领导指示禁飞！很遗憾的没有拍摄到更美的风景！这么美的地方从空中俯视才能彰显法国人浪漫的设计感，平视感觉相差还是蛮大的！
说到这里，其实是浓缩了南法的生活方式，还原了南法的艺术场景，成片的紫色薰衣草地，有一种唯美主义的浪漫，一座表意性的建筑，更是一种传统的突破！
“建筑是对阳光下的各种体量的精炼的、正确的和卓越的处理。这些体量所表现的形象是明确的、可触摸的、而且是不含糊的。正因如此，它们才称得起是美丽的形象，最美丽的形象。”——勒·柯布西耶

【马路边边麻辣烫（致民路店）】没有最好，都是特色

成都排名第一的串串果然名不虚传，吃的是一种情怀，更是一种童年的味道！
位置 /
致民路48号（距离地铁3号线新南门站b口步行170米）
营业时间：周一至周日 10:00-01:00
环境 /
看到门前的凳子还没码好，本以为来早了，谁承想进去后看到的是坐满了食客，都说成都上午都喜欢喝大碗茶，对于串串的喜爱程度在我看来毫不逊色！让我感慨、感悟到成都人的另一种生活方式……
店内四壁上满满的都是情怀的元素，涂鸦画、老物件、我们童年时追捧的歌星、明星照……这醒目的大红标语太给劲了“发展才是硬道理”，的确啊，餐饮业长久的生存就是靠思维的进步和不断创新，满足一代又一代人的口味，在这里70后会觉得好熟悉，80后会认为自己穿越了，90后追求的是一种怀旧的时尚，00后们来吃的是一种信仰！
吃麻辣火锅要配唯怡豆奶！
首先从这个"辣"说起。有一种叫"辣椒素"的东西，它像蜡一样不溶于水。这辣椒素和舌头上的味蕾碰撞，就会产生神经信号传到大脑，这时候人体才真正感觉到了"辣"！所以，要减轻这种辣味的不适感，要想办法把辣椒素从味蕾上赶跑！科学证明，奶制品里含有大量的酪蛋白，而酪蛋白是很好的乳化剂，可以有效地把不溶于水的辣椒素包裹起来带走，所以奶制品有效！
唯怡：提到唯怡，大家都不陌生。这款豆奶产品在四川等市场十分流行，尤其在川渝餐饮渠道具有很高的市场占有率。
菜单：很简单，有点像小学课本的几页，可以单点一些菜品和主食等，串串自取。蘸料有辣椒粉花生碎、蒜泥香菜等等。
冰粉：这里的红糖是最正宗的老味道，我已经听很多成都的朋友回味提起过！饭前一碗冰粉，很开胃；饭后一碗冰粉，满足感爆棚。
串串自取：串串都是放在开放式冰柜里自取的，大多数的价格都在5毛到2元不等，品类不下几十种，相信总有几款适合你的口味，我建议勇敢的尝试一下你平时没有吃过的。
鸭肠：新鲜的鸭肠，只要涮个5、6秒钟即可，味道是最好的。鸭肠对人体新陈代谢，神经、心脏、消化和视觉的维护都有良好的作用。
冒脑花：口感细腻，近似豆花，嫩而不腥，一口一口香浓软糯根本停不下来。在中国饮食中，有个“以形补形”这样的说法，而这说法应用在猪脑花上面就十分正确了。吃脑补脑，是有一定道理的，猪脑含钙、磷、铁比猪肉多，还富含卵磷脂、蛋白质和脂肪，另含硫胺素、核黄素、抗坏血酸和大量的胆固醇等成分，这些成分，可以帮助提高记忆力。
兔肾：第一次吃这个东东，小心地放到口中，咬上一口比较弹，并没有吃出土腥味，稍微有点辣味，越吃越香。
辣炒花蛤：是一道由蛤蜊、葱姜蒜等做成的美食，口味咸辣。

【太古里】时尚成都，从这里开始

纵横交织的里弄、开阔的广场空间，为呈现不同的都市脉搏，有快里和慢里概念，树立了国际大都会的潮流典范。值得把玩的生活趣味、大都会的休闲品位、林立的精致餐厅、历史文化及商业交融的独特氛围，让人于繁忙都市中心慢享美好时光。
位置 /
成都市锦江区中纱帽街8号。
全球各大品牌的入驻是的太古里与世界接轨，成为时尚之都，独有的慢生活的精致与细腻，融进历史文化底蕴于一身，爱上这里，没有理由！
川西风格的青瓦坡屋顶与格栅建筑在保留了原来风貌的基础上配以几栋现代化建筑，将现代镶嵌在古典之中。绿树成荫、鸟语花香，呈现一种灵动的美！
热闹繁荣的商业街之中，深藏着一座千年古刹——大圣慈寺（简称大慈寺）。大慈寺是四川省成都市著名的千年古刹，古城“震旦第一丛林”，修建于唐天宝、至德年间，由唐玄宗赐匾“敕建大圣慈寺”始建，到唐肃宗为其亲书“大圣慈寺”而得名。内有壁画15000余壁，其中神佛画像及经变等14000壁，山水、花鸟、猛兽、亭台楼阁等1500壁，数量之多，吸收了唐宋壁画之精华，每一幅都有很高的艺术价值。

DAY 3【杜甫草堂】草堂虽小，却涌出了盖世的诗章

其实是来追剧的，前不久刚回看了一部中韩合拍片，由高圆圆和郑宇成主演的《好雨知时节》。还记得梭罗那句话：每个清晨都是一份令人愉悦的邀请书。人生无常，我们都需要勇气接受改变，更需要力量让明天继续。似乎，我也想从这里找到些什么……
位置 /
青羊区青华路37号。
开放时间：夏季 08:00-18:30；冬季 08:00-18:00
票价：50元
杜甫草堂是中国唐代大诗人杜甫流寓成都时的故居，杜甫先后在此居住近四年，创作诗歌240余首。现为国家AAAA级旅游景区，是中国规模最大、保存最完好、知名度最高且最具特色的杜甫行踪遗迹地。
2005年重建的万佛楼矗立于草堂东面楠木林中，它复原了历史文化名城成都“东有崇丽阁，西有万佛楼”之风貌。
绵延310米千诗碑成草堂一景，脍炙人口的传世佳作、经典卓绝的名家墨宝、古韵悠悠的鲁西青石、精彩绝伦的刻工技艺……
杜甫草堂博物馆依据杜诗描写和明代格局于1997年恢复重建了“茅屋景区”，重现了诗人故居的田园风貌，营造出浓厚的诗意氛围。
唐代遗迹进一步证实了杜甫草堂不是虚构，而是真实存在。挖掘出并展示的碗、盆、水井等都保存得很好，增加了杜甫草堂历史的厚重感。

【集渔·泰式海鲜火锅(太古里店) 】满足吃货的执念

坐拥太古里商圈，极火的一家泰式火锅店，听到冬阴功锅底已经召唤了我，旅行在泰国的时候，我称泰国美食为“彩色美食”，每一道菜品都非常漂亮，美味更不用多说！集渔怎么样？跟我来一起探一下吧。
位置 /
下东大街169号晶融汇2期洋展购1号门。
营业时间：周一至周日11:30-14:00 16:30-22:00
店有三层，我很喜欢这家的装修风格，首先很饱满，绿植很多，很浓的泰式风情格调。很多时候我们吃饭，吃的更多的是环境，好的环境品尝美味佳肴，才叫正！
选择了198元的2人A套餐，两大盘的新鲜海鲜、各式样的新鲜蔬菜，看着就过瘾。自然要选最畅销的冬阴功锅底。
底料是现加的，最后加一勺椰汁，瞬间香气已经扑面而来，勾起了熟睡中的馋虫……虽然大家都听过冬阴功这个词，但我相信还是有很多人没有尝过的。冬阴功汤是泰国和老挝的一道富有特色的酸辣口味汤品。这道汤的名字中，“冬阴”是酸辣的意思，“功”是虾的意思，翻译过来其实就是酸辣虾汤。不过冬阴功汤极辣，其中还放有大量咖喱，不过习惯了之后不少人倒是会贪恋上它。当然，极具营养价值！是秋、冬季最棒的滋补佳品。
除了哈根达斯和芒果糯米饭是一人一份限量的，其他服务员手推车里的小吃和甜品是无限量供应的。

【醉西昌（鸿云店）】吃的是一种异域风味

主打的是少数民族烤肉，常言道：有肉、有酒、有朋友！肉当先，看着碳火慢烤的滋滋的烤肉，就有一种幸福感！
位置 /
鸿云悦街13栋3F。
营业时间：周一至周日 11:00-14:00 17:00-23:30
醉西昌分碳烤和电烤两种模式，想体验最正宗的西昌味，我强烈推荐碳烤，虽然烤出的肉是差不多的，但我们唯独要吃的是那种“烟火味儿”，冒着火冒着烟，感觉才到位！三五好友围聚畅聊，要的是一种氛围！
西昌火盆烧烤，源自彝族人家围着篝火烘烤食物的简朴粗狂的饮食方式。而今，带着原始气息的火盆烧烤，已经慢慢在成都这座现代化都市扎下根来，只是多加了一个篦子，减少污染更安全。
包浆豆腐：外表金黄，撒上一些干辣椒，咬一口嫩嫩的滑滑的，浆汁充盈口腔。
竹签大串：大块小香猪肉，经过特别腌制，烤至金黄也可以多烤一会，干一点更美味。
生菜包牛肉：大片的黄牛肉附上一层蒜末，烤熟后蘸料生菜包着吃，有点典型的韩国吃法，但真的很香！
西昌猪小肠：有三种味道，原味、蒜香、韭菜。其实我平时不吃这些东西，但还是勇敢了一下，并没有想象中的难吃有异味，反而觉得很有嚼头，如果不是自己不能吃太辣的食物，我想我会全部吃光的。
酸菜五花肉：五花肉切的很薄，可见刀工健硕！薄，也是为了更好更快的的烤出滋味，烤熟后卷着酸菜吃，第一次这么吃，感觉很特别！～

DAY 4【东郊记忆】成都知名工业遗迹旅游景点

北京有798，成都的东郊记忆早有耳闻，都是“文青”聚集地，都是拍照圣地。如果追一下东郊记忆的历史，貌似与798又有相似之处，都是苏联时期建筑改造而成的工业园区。
位置 /
成华区建设南路中段4号。
每一座城市的工业历史都是悠久的，一句两句说不明白的，不妄言，我们只看今朝！如今的东郊记忆失去成都游玩必须打卡地，到处充满浓浓的怀旧感，随手拍几张美照发朋友圈，瞬间招来百个赞！没错，这里毫不夸张地说，保存的比目前商业化比较严重的798完好的多，更适合摄影师发挥创作的灵感！
火车头广场，在火车头广场有四川第一批蒸汽式火车头和两节绿皮车厢。历史与现代、传统与时尚、过去与未来的对话、碰撞，行走在东郊记忆里犹如一趟生动的“时光之旅”。

【豪虾传龙虾馆（春熙路总店）】季节大闸蟹才是王道

卤煮麻辣小龙虾的出现，让小龙虾的口味增到了4种。细数一下其他三种是十三香小龙虾、口味虾和油焖大虾。突然间感觉到自己的胃一点都不亏，这些口味在全国各地旅行时都吃过，对小龙虾的追捧也从来没有间断过，到了季节一定要吃！大闸蟹的季节，卤煮味道的也是很棒的！来豪虾传试试。
位置 /
下东大街36号郁金香广场2楼
营业时间：周一至周日 11:00-02:00
大闸蟹的季节：个个饱满黄多，关键是卤煮口味的，想必你也有兴趣来尝试一下。
店门前环境
红糖冰粉：来成都成习惯了，先来碗冰粉，我发现豪虾传就是豪气，这配料多放好多种，更好吃了！
蓝色可尔比思
卤煮极品小龙虾
蒜蓉粉丝蒸扇贝
蒜蓉虾球：小而Q，肉质嫩且饱满，在这里一年四季都能吃到这么好品质的，难得匠心！
虾汤藕片，脆脆的有点辣。
白饼泡虾汤才是正确的吃法。
卤煮大闸蟹：用卤煮小龙虾的方法来料理大闸蟹，只是蟹壳外面有麻辣之味，而大闸蟹内部只是进了底味，并不麻辣，吃到的依然是大闸蟹的鲜香，但是比清蒸出来的，更有味道。
热卤顺风：进补的秋季，推荐菜品之一猪耳朵，蘸料辣椒味道会更好。
卤煮甲鱼：外壳以下基本都可以吃，辣味十足，对面坐的成都朋友吃起来也是直呼：爽！
辣卤鲍鱼仔
招牌土豆饼：看一些点评，土豆饼的评价很高，果然不错！炸的金黄，外酥里嫩。

【如在蜀老火锅（玉林店）】品一顿温和的火锅

玉林，因一首《成都》爆火，小酒馆就在这一带，这一带更是成都“市井生活”的代表，周围餐饮更是林立！而如在蜀这时候提出了一个品牌理念：让一切回到简单，让火锅回到火锅！多么朴实的话，因为他的朴实，所以吸引到了我！
位置 /
玉林南路15号玉林生活广场一楼。
营业时间：周一至周日 10:30-00:00
店门前环境
环境装修风格融入了草堂风格，古色古香，并将公园的墙、小路、老的街道等等融入其中，在这样的环境里吃一顿地道得成都火锅……成都！没白来！
大堂环境
油碟：在川蜀地通常吃油碟，但这次我发现店家都更推荐原汤。
番茄鸳鸯锅
这一桌，真的是我一个人吃光的！对于火锅的喜爱已经不止两三天，也不止两三年……所以说我喜欢成都，是火锅留住了我的胃～而如在蜀，火锅的改良，更加温和的辣，最适合不能吃辣的朋友，所以我推荐！
满桌美食
铺盖毛肚：镇店菜品，甄选水牛牛百叶。水牛毛肚比黄牛毛肚好吃，煮的时候火候要大，时间要短。
桂花冰汤圆：别的店都在吃冰粉与火锅搭配，如在蜀的桂花冰汤圆，似乎更沁人心脾，冰爽可口。
鲜郡花：将鸡鸭的胃脏部位在做菜前用刀把它们切花纹状，煮熟后就会漂亮得像一朵花，因此叫做郡花。
红酒浸肥牛：红酒配红肉，搭配红牛涮锅也是有创意了。
