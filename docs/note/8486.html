初夏云南，是不一样的云南【自驾九乡石林普者黑】

游记 自然奇观 自驾游 2019/06/29
每到夏天，就很喜欢和几个朋友一起去自驾。这次，我们一致选择了去云南，在这个初夏的季节里，我们在阳宗海的水世界畅游戏水，在九乡溶洞里感叹大自然的神奇，穿梭在石林里感受着一座座石峰拔地而起，在荷花还没盛开的普者黑感受着世外桃源的魅力。

【预告片】

入住的阳宗海酒店附近有一大片草坪。
水世界里的欢声笑语，惊险刺激，让人念念不忘。
十年后再一次来到九乡，依旧记得每一处的风景。
溶洞的美，是大自然的鬼斧神工。
来过很多次云南，却还是第一次来石林。
到达普者黑的那天，偶遇了一场完美的日落。
上午下雨，下午蓝天白云，我们看到了不一样的普者黑。
彩云之南，这是一次不一样的自驾之旅。

【关于行程】

day1:成都-阳宗海（群樱荟度假酒店）
day2:阳宗海（华侨城水世界）-九乡-石林
day3:石林-普者黑
day4:普者黑-曲靖
day5:曲靖-成都

【在华侨城的水世界里畅游戏水】

从成都自驾去阳宗海大概花了十个小时，早上八点从成都出发，一路畅通无阻，全程高速，过了昭通，天气好得感觉每一处都是风景，还记得几年前从云南开车回川，昭通这一段的高速公路都还没有开通。
我们入住的是阳宗海华侨城的群樱荟度假酒店，整个酒店是欧式风格，面积很大，放下行李后我们就在酒店随意逛了逛。
酒店很大，稍不注意不看指示牌很容易迷路。红色的墙壁看起来很温暖，走在酒店的走廊里，仿佛走在欧洲的某个城堡里。
住的是湖景房，打开落地窗，就能看到远处的阳宗海，躺在阳台的躺椅上，好不惬意。
从房间出来，酒店的前方有一大片草地，还有一些马鞭草，非常好看。
在这里非常适合拍人像，建议大光圈和长焦镜头。绿草如茵，陶醉于美景之中。
离酒店不远的地方就是华侨城的水世界，整个水世界不光能游泳，还有温泉，每天下午一点还有水上世界开放，在这能玩上一天。
这里除了有公共浴池，还有私密性极强的套房，套房里不仅有温泉，还能蒸桑拿，里面还有机麻和ktv的设施。如果是一家人来玩，开一个套房，也非常不错。
我最喜欢的是这个无边泳池，远处就是阳宗海，趴在泳池边，看晚霞，别有一番滋味。
每天下午一点开始，这里就充满了尖叫声，欢笑声。在这里，体验了前所未有的惊险和刺激。
这个项目是需要自己扛着滑毯上去之后，听了工作人员指挥之后滑下来，虽然在下面看着不是很刺激，但是当站在上面往下看的时候还是需要勇气的。
大喇叭大概是最刺激的一个项目了，远远就听到不断的尖叫声。
除了各种游玩的设施，还有一个冲浪池也是深得大家的喜欢。
在这里，无边泳池，有特色的汤池，刺激的水乐区，夏天来到这体验畅游戏水的惊险刺激和乐趣。
 

【泛舟峡谷，穿越溶洞，丛林穿越】

十年前和家里人来过九乡，十年后再一次来到这，依旧是充满惊喜的。坐船游览在峡谷间，在溶洞里感受到了大自然的神奇，坐在缆车上将整片山林尽收眼底，每一处的风景都让人惊叹。
前往溶洞要到峡谷底，除了步行下山还可以乘坐电梯。
到达峡谷底，可以选择乘坐乘船游览荫翠峡，这是景区提供的，不额外收费。整个峡长600米，两岸古崖苍苍，，清幽迷人，如诗入画，我们去的这个季节是雨季，所以峡谷里的水都不是很清澈了。
坐船游览完之后就进入溶洞了，九乡溶洞分为四部分，而且这溶洞属于半开放式的，这是典型的喀斯特地貌，而且这里，还有很多的影视作品在这取景。
最壮观的是这个雄狮厅，这里总面积15000平方米，也是世界上最大的地下广场大厅，可容上万人，更为神奇的是，整个大厅顶部有一个完整的板块构成，板块上嵌有几个窝形的漩涡。在广场上向前看，便能看到一个像狮子一样的石头伫立在溶洞口，所以取名为雄狮厅。
从雄狮厅出来后就到了神女宫了，五彩缤纷的彩灯将整个溶洞照射的栩栩如生，美妙绝伦。
沿着步行栈道走，流水声一直伴随着。
雌雄飞瀑是溶洞里的双瀑布，水声震耳欲聋，整个溶洞里都充满着这个水声，和小伙伴说话都得靠吼。两条瀑布由三十多米高的悬崖上匐然而下，非常震撼。
要说整个景区最奇妙的就非神田莫属了。神田是大型边石湖群，面积100多平方米，深达10余米，错落有致，景观像梯田一样，奇伟壮丽，是世界上罕见的一大奇观，也被国际洞穴专家誉为世界之最。
整个溶洞游览下来并不费体力，最后的这一段是向出口走去，对我们来说也还好。但是如果实在爬不了梯子的，也可以选择坐轿子上去，明码标价的。洞口的风很大，凉悠悠的。
从溶洞里出来，还能选择射箭穿汉服或者丛林穿越，这些项目在五点之前都营业。我们去试了下丛林穿越，据说这是云南省内最长的。这个项目考验了体力和胆量，也告诉了我们要勇往直前。
从溶洞口到出口还有一段距离，这里可以选择步行、骑马或者是小缆车出去。特别喜欢这样的缆车，坐在上面特别惬意，还能有一览众山小的感觉。
从九乡出来后，我们就去了附近的农家乐，这个农家乐的风景特别好，要先走过一段吊桥，我们几个在上面晃来晃去，到了店内，一桌丰盛的菜肴就已经在等着我们了。

【世界自然遗产风光，石林】

虽然来了很多次云南，但是我却是第一次去石林。被评为“世界自然遗产风光”的石林的确让人眼前一亮，一座座石峰活灵活现，穿梭在石林里，一阵阵的现拉现奏的民族音乐悦耳动听。
整个石林景区很大，但是都有观光车游览，遇到中途感兴趣的点也能下车步行游览。我特别喜欢这里，后面是一座座石峰，前面有一个小池塘，石峰倒映在水面上，池塘里盛开着一朵朵的睡莲。
我们到景区的时候还是阴天，进了景区后蓝天就已经显现出来了。在蓝天下，每一座石峰更加活灵活现了。
这里叫做望夫石，仔细一看，的确像一个女人看着前方，所以取名为望夫石。
坐观光车到了大石林这边，步行进去浏览，这里早已经人山人海了。
导游说这是雄鹰展翅，而我们看到的上部更像是一只小狗的头。每个人的想象力不同，看到的呈现出来的都不同。
这里是最有名的一个地方，也是游客最多的地方，用广角拍照，下面全是游客。
对面的那个亭子就是望峰亭。
穿梭在石林间，每隔一段距离就会有当地的民族歌曲，都是现场弹奏的，导游说，每当有游客经过，就会有音乐响起。这跟音响放出来的完全不一样，让游客能近距离感受到少数民族歌曲的魅力。
除了弹奏歌曲，还有刺绣这些文化遗产的展示。不得不感叹，整个游览过程是丰富的。
据说这里在几亿年前，就是一片汪洋，经过漫长的地质演变，才形成了今天的石林。在这里，我们还看见了珊瑚化石，但是被无数人摸过之后，珊瑚化石已经不容易看出了。
穿过一条条小道，庆幸有导游带着我们，不然肯定要走错很多的路，登上望峰亭，这里是欣赏“林海”的最佳位置。
从高处俯瞰石林，是和在地上游览完全不一样的。怪石林立，每一座石峰造型都不同，跟同行的小伙伴说远处的那个好像是一个飞天猪。匆匆拍了两张之后就下去了，因为上面游客实在是太多了。
这是我最喜欢的一个点，湖水碧绿，平静的湖面还倒映着各种各样的石峰，蓝天白云下的这个景点让我狂按快门，朋友说这样的石林好美。
小石林的景区和大石林不一样，这边的景色更加秀丽，绿草如茵，这样的色彩让人看着神清气爽。
阿诗玛景区内最出名的就是这个神似阿诗玛的石峰，在湖边亭亭玉立，仿佛在等着她的阿黑哥。
从石林出来后，当地朋友推荐来了石林一定要去吃马肉。然后，我们就来了这家。
之前一直听说马肉不好吃，但是石林这边的马肉经过特殊处理之后，意外的美味。价格公道，每一道肉菜无论什么都是35元，特别喜欢马肉卤菜和爆炒的马肉，特别入味。
 

【心念念的普者黑，偶遇一场绝美晚霞】

对普者黑一直是心生向往的，奈何每次来云南都以各种方式错过。这一次终于要去了，心里说不出的高兴。
到达丘北，将行李放到酒店房间，原来打算出门吃饭的，小伙伴看到远处已经有红烧云了。大家又立马回房间拿相机拿航拍，开车朝红烧云的方向去。
一大片荷花还未盛开，想象着七八月如果荷花盛开，不知道这里有多美。远处云雾缭绕，橙色的云彩就在山顶，宛如一幅色彩丰富的山水画，美极了。
找到一两颗含苞待放的荷花，在夕阳的余晖下，画面显得温暖极了。
没有看到一大片的荷花，这些还未绽放的花骨朵也成了我们一行人的最爱。
一会儿的时间，整片天都被染成了橙色。我特别喜欢这个时刻，还未到盛夏的普者黑，吹了一阵阵的凉风，我们就这样享受着大自然的美。
到了最后，整个天空有紫色，橙色，蓝色，多姿多彩。各种颜色的晚霞倒映在荷塘中，美得令人陶醉。
大概是前一晚的绝美晚霞，第二天就开始烟雨蒙蒙了。我们泛舟于湖中，穿过一片片荷叶，小雨一会儿下，一会儿停的，别有一番风味。
普者黑整个景区是不收门票的，景区内有观光车，买了票之后当天可以无限次乘坐。在每个码头也有游船或者小舟提供，这些都是游览普者黑的不错选择。
我们第一个乘坐的是电动竹筏船，在西荒湿地码头乘坐的，全程绕一圈大概一个多小时的时间。沿途出了能看到各种鸟类动物，还有当地的小渔船。
路过了一片类似芦苇（我到现在也不记得叫什么，长在水中）的地，穿过的时候感觉像是电视剧里游击战的感觉。草的叶子上有的还有一团粉色的东西，导游说那是螺丝的卵，凑近一看，密集恐惧症就出来了。
途中还看到了一大片荷叶，中间有着零星的几颗荷花，用长焦一拍，觉得美极了。
七八月再来吧，那会儿就会看到满池都是红的，白的花。这个时候去只能看到满池的荷叶。
从西荒湿地码头到双甲山码头可以乘坐观光车。在双甲山码头这里有一片马鞭草，一眼看过去满是浪漫的紫色花海。走在花丛中，手轻轻扫过花，心情格外舒畅。
在双甲山码头有电动画舫船，全程一圈下来大概两个半小时的时间，我们没有坐到原点，在中途的一个码头就下了船，去了天鹅湖。
画舫船经过了普者黑村，很喜欢对面的那栋红房子，一个个落地窗外就是田园和湖景风光，好想去住一晚啊。
这个船是柳叶小舟，我很推荐这个游船项目，因为可以打水仗。一船的游客拿着盆子水枪各种武器互相泼水，在烈日下也觉得非常有趣。如果不想被泼水的，可以让船夫避开，但坐这个船基本也都是奔着泼水去的。
我们船有一块没了玻璃，被这群人一下将水泼了进来，船上的桌子椅子全湿了。泼完还大声的告诉后面的游客说“后面的兄弟，加油哦”。让没有打水仗设备的我们哭笑不得。
早上还烟雨濛濛，下午就已经有了蓝天和朵朵白云，这个时候的普者黑又是一番不同的景色。要说早上是山水画，那么下午就是油彩画。
从码头下来，就看到远处有一个观音阁。我们在桥上看了好久湖里的人打水仗，每一艘船都拿出水枪想要喷我们，奈何距离太远，每每射出后水都反射到他们自己人的身上。虽然没有去打水仗，但已经感受到了他们的乐趣，下一次，我一定带好装备，过去大战一次。
随后我们来到了天鹅湖，这是一个集喀斯特孤峰群、湖泊群、溶洞群、湿地生态群为主的生态型湖区。看天鹅有时间段，下午是两点到四点。成群结对的天鹅在水中漫游，高傲的伸长着它们的脖子。
除了白天鹅，也有黑天鹅，我觉得黑色的天鹅更加优雅。
到了下午四点，天鹅就要回家啦，有四个工作人员赶着回家，四对几十的这个场面颇为壮观。
我们看天鹅是从2号口进来的，要想看到三生三世十里桃花的拍摄地就必须从2号口走回1号口。一路上风景优美，看荷花的地方也有几个，但好几个荷花的观赏地离路边都有一段距离，开玩笑说咱们的70200都有些吃力，必须拿个600的焦段来。
远处的画舫船穿梭在荷叶之间，从船上看到的风景和岸边看到的风景又是不一样的。
终于走到了观光车上车点，坐上观光车，闭着眼睛享受着车速带来的风，任由头发乱飞，心里是开心的，我终于来了这个心念念的地方，但又有些遗憾待得时间不够长，希望下一次能多待几天。
 

【结尾】

我在异域风情的西双版纳生活过，我见过小清新的大力，我在热闹的丽江完成了我的高中毕业旅行，我和好朋友一起游览过金黄的腾冲，我也见过鲜花盛开的香格里拉，这一次，我见到了世外桃源的滇东南。
每一次的云南，都是不一样的云南，希望下一次的相见，还能有更多的惊喜和期待。
