答应我，此生一定要去睡一次莫干山

游记 人文游 摄影 精品酒店 美食 2018/12/24

一场逃离都市的山居生活·莫干山

《纽约时报》曾评选出全球最值得去的45个旅行目的地， 莫干山就名列其中。CNN更是将这里称为“除了长城之外，15个你必须要去的中国特色地方之一”。从上个世纪30年代开始， 莫干山便吸引着各地的游客纷纷前往。
莫干山 ，位于沪、宁、杭 金三角的中心， 浙江省湖州市德清县境内（高铁可以直接坐到德清站下），一到夏天， 莫干山更是江浙沪地区的避暑度假首选。

走进蝶来三舍·莫干

老远处就看见蝶来三舍莫干店啦，真的很喜欢这种白色的房子，莫名觉得很干净，就连空气中都弥漫着清新的香气。未来的48个小时舒适慢生活“滴”的一声，立马开启~
很喜欢这次住的民宿的名字，蝶来三舍，蝶来宛若新生，舍名舍利舍浮华。地址在莫干山民宿聚集地的旅游乡村劳岭村鸭蛋坞4号。
好一处诗意的栖居之地，让你只想静静享受内心的轻松和愉悦。除了美景， 莫干山最出名的就是民宿， 莫干山的民宿非常多，有2000多家。
天空飘起了小雨滴，小姐姐帮我们拎着行李，迅速将我们带进室内。
前台很快办好了入住，小伙伴已经迫不及待的想去看看我们的房间。
我们的房间是“隐舍”，和网图简直没差啊，一进门我就看了，有地暖，哈哈哈。
这个榻榻米赞爆了，觉得这上面明明还可以睡两个人，这个房间简直就是亲子套房呀。
房间配套科勒卫浴、 英国高端seven plus茶语系列的护理包、飞利浦的吹风机、超大索尼电视、Marshall音响、小冰箱等。
“隐舍”，向往高山流水，逃离都市压力，在莫干山“隐居”，“舍弃”即是“拥有”。突然睡意肆起，如果不是肚子在抗议，我怕是听着音乐就在躺椅上进入梦乡了。先把肚子填饱，才有力气睡觉呀。
第一顿的私房菜是红烧鲫鱼、香干水芹、干锅花菜、蒜泥小青菜，鱼很好吃，带点甜味，反而特别鲜嫩。舟车劳顿之后，吃点清淡的，人很舒爽。“春困秋乏夏打盹，睡不醒的冬三月”，吃饱了，我的冬三月已经蠢蠢欲动了~
一觉醒来想出去逛逛，雨还越下越大了。民宿的小姐姐说，下雨天，除了睡觉还可以喝喝下午茶，推荐我们尝尝她做的拿铁。
一本杂志，一杯咖啡，听着轻柔的音乐，或者什么都不做，只是放空，坐在窗前静静欣赏雨落，都是一个完美的下午。
有小盆友的家长也不耽误带娃，可以让孩子在儿童休闲区玩耍。
蝶来还有很多体验活动， 比如做手工香薰扩香石，这个真的吸引到了我们，我自己平时就喜欢做手工，蝶来的材料很全，模具、颜料、装饰物、小配件有一大堆。每个人的费用是70元。
我做了好几个扩香石，有小天使、发财图和暴力熊。不会做的也没关系啦，小姐姐会教你们哒~正好这时候雨也小了，我们想去附近的石颐寺水库走一走，小哥哥说在坝上可以看到对面的裸心堡。
很喜欢这个设计，在这里拍了好久，好的设计会说话，就有会不自觉的打动你～
蝶来有自行车，可以骑车在附近逛一逛，不过下着下雨，我们就步行前往了。
走过去15分钟吧，快到水库拍了一张海报，是不是还挺好看的，如果有蓝天白云，就更赞了。
照片的左边有个扶梯，爬上去以后就可以看到裸心堡啦。
说起裸心堡应该不会没人知道，现在她已经成为莫干山的一个地标了，周末5000+一晚的房价真是让人望而却步，但是这也无法阻止每位来莫干山的人都想看看她。迷雾雨潇潇中的她，在莫干山浩瀚无垠的竹海里变得更加浪漫奇幻了。
晚餐我们预定了云南野生菌菇养生小火锅，莫干店的店长是云南人，相信这个火锅不会让我们失望。这种湿冷湿冷的天气吃火锅，暖胃又暖心呀。
这边小伙伴们已经打起了游戏，坐等开饭啦。
云南野生菌菇养生小火锅的高汤是由羊肚菌，牛肝菌、黑松露、竹荪炖制而成，汤头鲜美，一口气喝了好几碗，食材很新鲜。
牛肝菌
店长自己调配的酱料，有云南的腐乳、单山蘸水辣椒面很好吃，不过我的心头好还是这一锅野生菌的汤，真的很美味。
想要泡澡的话可以问前台要一次性泡澡袋，袋子一定要固定好呀，水进到袋子下面后，袋子就会一直往下滑。。。
第二天早餐吃的是阿姨自己包的饺子，一定要夸一下调的酱汁太提味了，瞬间让饺子有了灵魂一样，这是我第一次觉得原来没有辣椒的酱汁也有一番风味。早上还有一杯牛奶，今天我们就要进山啦，虽然天气预报说上午会下雨。

竹海无垠、绿波万里的莫干山

其实莫干山有一条骑行道，从莫干山庾村出发，经过三莫线－五四村－何村－鸭蛋坞-分水岭-劳岭水库，最后折回庾村。要是嫌骑车太累， 莫干山这边也可以扫码租个电动车。
门票我们昨天在网上买好了，70两人，比现场买便宜20/人。从售票处到莫干山风景区里面还有一段路程，以前可以自驾进去，现在都是坐景区巴车，统一管理，也比较安全。车票在现场买，35一张。
车子开上山了，在一个落客平台停下。下来后告知我们还需要买车票， 莫干山风景区比较大，景点较为分散，这次买的是景区里面的巴士，不买不行，每个人都过来说一遍，自己走一天走不完的，还是坐车吧。每人15元，每个游览点都会停靠，每次上车都需要出示车票。
第一站是毛主席下塌处，是1954年3月毛主席在杭州主持制定第一部宪法期间到莫干山游玩时所住的宾馆——皇后饭店。
此栋建筑结构属于欧式乡村别墅，室内展示的是当年毛主席入住时的一些装饰摆设，朋友说这次来内容比之前看的丰富多了。
滴翠潭在武陵村口，系人工开凿，潭水清澈、内植睡莲。看点是潭边岩石上“翠”字，足足高20余丈，如三层楼，气度雄伟。钱君陶所题“翠”字，管理局花两年时间刻于崖壁。之前本想题“竹”字，觉得有点俗，后欣然改“翠”。
往前走一点就是蒋介石的官邸武陵村松月庐。目前只开放一栋可参观，不过室内没什么东西，倒是可以去2楼半月型的阳台上赏赏松树。
很好，我就假装我看到了。。。
莫干山可大致分为大坑景区、芦花荡公园、剑池景区和武陵村景区四部分。芦花荡也是人造的，现在也没有芦苇了。雾越来越大了，还下着雨，我们决定下山了。。。

莫干山小镇&乡村文创园

从游客中心出来就到莫干山小镇了，这个建筑旁边有个庾村老面馆，当地人推荐我们来尝尝，听他们聊天说游客和本地人去吃是两个价格喔。
炒猪肝面15，牛肉、牛肚面都是20。面是机械面，汤头有点像菜汤类的，肉给的还是挺多的面也不少，口感还行，可以尝一尝。猪肝面太甜了啊，看别人吃的挺香我们也点了一份，猪肝太甜了。。。
马路对面就是以前的 莫干山 车站。里面讲述了莫干山的交通发展史。
航拍视角的莫干山小镇~
朋友说这家梅干菜烧饼好吃，尝了一下，又加了糖，确实比咸口的鲜甜一些。
回到民宿放松放松，发发呆。
晚上的红烧肉肥而不腻，青椒牛肉丝和野生小杂鱼质嫩爽口 ，赞~
金色标的那瓶蜜桃味的好喝。
最后一天早晨的鸡汤面真的一绝，里面还有笋丝和肉丝，我们把汤都喝个精光，吃的干干净净。
再喝一杯小姐姐做的香甜拿铁~
离别不说再见。这一次来，天气不好，却看到了山水画般的莫干山，也是想让我们再来吧。下次来好好的航拍莫干山，一次旅程太圆满，就不会心心念念的了。
