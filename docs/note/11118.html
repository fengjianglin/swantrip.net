【天鹅之旅首发】慢游苏州太湖三山岛，看最美太湖日落

游记 徒步 摄影 登山 美食 自然奇观 2018/10/08

前言部分

虽然距离太湖不是很远，但一直还未前去一探，眼看秋天来了，天气凉爽了起来，但伴随着的阴雨天气潮闷也紧接着而来。于是，周末约了几个朋友准备去太湖转转，如果有可能环太湖一圈那就更好了。
  太湖，是中国五大淡水湖之一，湖岸线漫长，自然风光优美，且拥有众多形态不一的岛屿，其中这些岛屿中以太湖三山岛最为出名，三山岛由北山、行山、小姑山三座山峰组成，名字也由此而来。同时这里也是国家湿地公园，山水相依，面积虽不大却风光秀丽，明代称之为小蓬莱，故有“太湖蓬莱”的美誉，犹如世外桃源一般隐匿于太湖之中。
  在三山岛的那两天真的不要太惬意，清晨或傍晚岛上闲逛，中午找地方喝喝茶，骑骑单车采采果子，虽然不能环湖但也可以泛舟湖上，尽享湖光山色。这里也比较适合悠闲放松，比较慢节奏的生活，原生态的风景，一派返朴归真的乡野气息，如果能多住上几日也是挺不错的。

行程安排

DAY1：登山环湖，湖上泛舟，品尝岛上特色美食，住宿古井山庄； 
DAY2：探访岛上留存的明清古建筑群，岛上悠闲游，采摘枣子橘子。

登山观湖光山色

我是从南京出发，乘坐高铁至苏州站。从苏州站乘车到长圻码头，从这里乘船到三山岛15分钟左右即可到达岛上，是乘坐的那种快艇，速度相对比较快一些。
来到岛上可以买票乘坐游览车游玩，也可以徒步游览，由于岛屿并不是很大，建议徒步游览，当然如果不愿意走太多路也可以租这里的自行车，环湖路相对平缓，比较好骑。
  我们的游览起点是在这块牌坊前，开始徒步游览三山岛。牌坊上有对联：万顷碧波粼粼泊泊来，三春清风徐徐阵阵至，横批：湖光山色，往后可见“太湖蓬莱”四个大字，描绘的便是此处的太湖水以及岛上清风袅袅的景象，让人心旷神怡。
牌坊后为岛中最高峰北山，前为湿地公园，在整个三山岛内最常见的便是湿地风景。进入湿地公园需进过一条尝尝的木栈道，木栈道一侧为一大片荷花池，只是现在荷花已经凋零，但大片的荷叶看着也很美丽，另外一侧可以看到太湖的特色湖生植物，比如茨菇、荸荠等，并且有相关植物的介绍，方便大家了解。
跟我们一起同行的小姐姐，大家能猜到是哪个国家的吗？
木栈道的尽头是一块水上游船及可供休息的茶室，在这里可以亲自泛舟湖上，尽享太湖美景。也可以看到不远处的三山岛最高峰北山，海拔80多米，虽没有巍峨之势，但山势祈福平缓，层峦叠嶂，清代诗人吴庄曾这样美誉：“长圻龙气接三山，泽厥绵延一望间。烟水漾中分聚落，居然蓬莱在人寰。”
作为一处湿地公园，这里拥有多种多样的湖生动植物，比较方便自然学科的专家学者可以实地考察太湖鱼类天然保护区和太湖水鸟栖息地，研究湿地生物的多样性。
在湿地公园出口处还有一座木桥，为长寿桥。行至长寿桥，沉醉于风景之间，忘路之远近，疑非尘世，时闻林间鸟鸣，右折拾阶登中峰行山，山径缓缓盘旋，寓意健康长寿平安之意，表达当地人民对远方客人的祝福之情。
去往北山的途中可见一座寺庙一样的建筑，为娘娘庙，庙宇内供奉着吴王阖闾的女儿胜玉的雕像，庄重而又端详。这座寺庙始建于唐朝，在明嘉靖年间重修过一次，寺庙香火旺盛为三山诸庙之最，可以看到许多附近的居民前来上香祈福，在寺庙门口可免费领取三支香火。
娘娘庙后为一小段登山台阶，坡度较缓便于徒步，台阶两侧树木葱郁，阳光透过枝丫照射进来，照在人身上，感觉到暖暖的，其实我们去时还说有些微凉的。
沿着山间小道继续往上走，抬头可见半山腰间有一块巨石，一端矗立于悬崖峭壁之上，一端凌空山壁之上，远观与埃及金字塔的狮身人面像极为相似，从而得名狮身人面石，确实非常像，造型浑然天成，巧夺天工。如果你来过多次三山岛也许就不足为奇了，因为这里到处是奇峰怪石。
从这里开始再往上便是真正的爬山了，最先经过的是一线天，由两块巨石对峙行程了一条一个人勉强通过的石阶，共53级，取自佛教中的53参，参参见佛，在石壁之上镶嵌有铁环，以方便攀登，铁环碰撞石壁发出铿铿声，让人不得不小心翼翼攀登，至中间平台位置可见石洞之中供奉的佛像。
一线天附近多石洞，洞内摆放有多尊弥勒佛像。来之前也了解到一些，三山岛也是一座佛教圣山，到处有寺庙，有佛像，这次也算是一次朝圣之旅，当然道路有些难走也在所难免。
秋初的树叶有些已经变得微黄或红色，从茂密的树林之间可以远望太湖美景。我们来时经过的那片水域被分割成众多小的水域，如果台风来袭可以基本抵消台风对主岛的破坏。
三山岛多奇石，其中以这块“四世同堂”石称绝，意为这块石头共有四种不同地质年代的岩石组成，其中灰褐色的屑石英砂岩形成于距今3.6亿年的泥盘纪晚期，浅灰色的石灰岩形成于距今2.8亿年的石炭纪晚期，褐红色的火山角砾岩形成于距今0.8亿年前，白色半透明晶体的方解石脉形成于距今0.4亿年前，仅仅一块石头便记载了这里的地质变迁历史，可见其奇。
再往前走，可见人山桥，连通着两座山。到人山桥需要经过一段又窄又陡的台阶，这里为花石纲岩壁，外侧有护栏围挡，内侧有铁环供人牵拉，行至此处基本往下都是下山的路。
在人山桥一旁还有一座小亭子，为蓬莱亭，位于花岗石之上，这里也是观赏三山岛观看风景最佳的地点，站立亭中，眺望向远方，近处植物繁茂，远处湖光水色，八百里太湖一览无遗，仿佛置身于蓬莱仙境一般。
经过人山桥，即是板壁峰，有着“吴中第一峰”的美誉。整体为一块宽20余米，高10余米的峭壁奇石，是3亿多年前海洋中沉积的石灰岩受地壳运动挤压而形成，远看整块岩石如斧劈，近看石头纹理如刀刻，青苔斑驳，藤蔓攀附，峭壁挺拔，不得不让人称奇。
沿着山间小道下来的路上可以看到很多的橘子树，只是大部分还未熟，如果熟透了的话，可以自由采摘着吃，以及看到的枣树，都是可以自由采摘。岛上几乎一年四季都是花果累累，五月枇杷，六月杨梅，八九月份马眼枣，十月以后有橘橙。
很多人都是骑这种四人一起的自行车环湖游玩，比较方便，如果中途有美景可以随时下来欣赏，不愿意骑了也可以随处停靠，不需要归还到固定的点。
环湖路上景色比较多，有荷花，有波斯菊，一眼望去，大片大片的花海就像地毯一样铺在湖畔，争奇斗艳，色彩缤纷，将这里装扮成了一幅多彩的花卷，如果可以在空中俯瞰那就更美啦，令人流连忘返。
这种是波斯菊，与格桑花特别像，颜色也比较多，有黄色的，红色的，白色的，粉色的等等，到处炫耀着五颜的色彩，飘荡着令人陶醉的香气，不远处温柔的湖风迎面吹来，非常惬意。一般波斯菊可以长到1-2米的高度，走入其中，瞬间被花海包围，花香四溢，愉悦之情不言而喻。
路边还有一座姐妹桥，呈三角形南北走向，北边为一座普通的桥，但过了河中往南边各分左右，犹如婴儿连体，当地人把它看作了一对孪生姐妹，并取名“姐妹桥”，姐妹桥旁边的水池里种满了荷花，待夏季的时候，满池的荷花盛开，一定非常好看。
这位老先生是我们这次的倒影，今年已经72岁高龄，从小到大生长在岛上，对这里的每寸土地都是怀着一颗无比热爱的心。让这里的当地人做导游，最合适不过了，他们对这里的熟悉胜过其他人，对这里的感情不言而喻。
岛上还有唯一的一家咖啡厅，叫做遇见你de甜蜜小屋，玩累了不妨坐下来喝上一杯，享受下小岛的慢时光。
目前岛上大约有600多户人家生活居住在这里，很多人家为了给来此游玩的人有一处居住之所，开始做起了农家乐，方便了前来游玩的人。
临湖而居，每天清晨看看从湖上升起的太阳，傍晚看着太阳从湖水之中落下，想想就惬意。
在一户人家还遇到了几个外国小朋友，从聊天中得知来自于比利时，目前他们跟随父母生活在苏州，利用中秋假期来这里游玩放松几天。

太湖湖上泛舟

傍晚落日十分是湖上泛舟的最佳时间，此时的落日余晖映衬着碧波荡漾的湖水，可谓水天一色，湖光山色，非常美。我们在岛上码头租了一条竹筏船，慢悠悠的漂浮在湖面上，大约玩了1个小时，很惬意。
一定要去太湖边看一场日落，此时的八百里太湖水近在眼前，为你一人独享，远处的太阳徐徐落下，沉入湖中，太美！“落霞与孤鹜齐飞，秋水共长天一色”，用在此处，也最合适不过了。
在湖边坐着一直等到到太阳慢慢落下去消失不见，落日余晖下的太湖水让人心醉，起风了，天凉好个秋～
我们去的那天恰逢中秋节，傍晚时分，望向湖面，可见湖上一轮明月，古有“海上生明月，天涯共此时”，今有“湖上升明月，天涯共此时”。

探访明清建筑群

除了山水风光以外，在岛上还有三十多状保存比较完好的明清建筑，如建于明代的九思堂，是一座三品官员的私家住宅；建于清乾隆年间的清俭堂，也是其中保存规模最大的一座，现有房间54间，从厅堂及落地窗上的雕花不难看出其保存相对完好，目前这些建筑正处于闲置状态，真怕时间久了，坏掉就可惜了。
这里的明清古建筑是三山文化渊源流长的见证，其中清俭堂、师俭堂等至今保存完好。有一些房屋经过了修缮成为了一些影视剧的取景地，有些还是保持原来的模样，只是由于长时间无人居住，显得有些破败，如果加以利用，那就更好了。
房屋的门窗保存都比较完好，门窗之上的雕刻精美细致，古色古香，建筑结构整体大概由砖、瓦、木组成，一直以来对于古建筑就有着一股莫名其妙的喜欢，不仅仅是其外观，更是深层次所赋予的文化内涵。
白墙灰瓦及高大的马头墙，典型的徽派明清时期的建筑，一砖一瓦，一草一木，仿佛都向人们诉说着这里的历史，这里的时光，古建筑需要细细品味，慢慢体会其中最为本质的东西。

岛上特色美食

因为是湖中岛屿，所以岛上特色美食以湖产为主，我们吃饭基本上都是选择在三山岛大酒店用餐，比较容易找，登岛后即可看到一座牌坊，上面题有“太湖蓬莱”四个大字，这里便是用餐的地方。
10月正好是吃蟹的季节，早就听闻太湖大闸蟹的味儿赞，这次品尝后确如此言。个头大，颜色深黄是其特点，一般采用清蒸水煮的方法制作，所谓“秋风起，蟹脚痒”，现在的大闸蟹主要以尝鲜为主，最好吃的时候还要等到10月中旬以后，那时的蟹膏或蟹黄颜色更艳，更肥美，是吃吃蟹肉、蟹黄的最好时机。
这就是传说的“太湖一号”，个头大，味道鲜！价格比蟹还要贵上不少！这个季节的虾除了鲜嫩的虾肉外，还可以吃到虾膏，还真是第一次吃虾膏，味道很美。
来太湖当然要品尝下这里的太湖三白，其中三白包含太湖银鱼，太湖白鱼，太湖白虾，下图为太湖白鱼，肉质鲜嫩，鳞下脂肪多，是太湖名贵鱼类，夹一筷子放进嘴里入口即化，鲜美之味不言而喻。
岛上居民家养殖的土鸡，这个可是一道滋补美味，里面还加入了枸杞，吃鸡前先喝上一碗汤汁，汤汁入胃非常鲜美，随后再将整只鸡撕碎，汤汁的味完全融入到了鸡肉里面，吃起来爽嫩滑口。
太湖黄鳝也是太湖美食中的一绝，黄鳝味甘，性温，能够起到补虚壮阳，除风湿，强筋骨的功效，还有调节血糖的功效。当然黄鳝的肉也是超级嫩，一口下去，非常满足。
当然还有很多岛上的特产，比如菱角，马蹄，芋头，板栗等等，不仅可以生着吃，还可以煮着吃，这些食物在我很小的时候吃过，充满了童年时的记忆。

住宿那些事儿

岛上住宿多以农家乐为主，大部分都是当地人居住在此处的房子，为了方便前来游玩的人而改建成农家乐。这次我们选择住宿的是古井山庄，位置很好，背山面水，从码头步行过来大约5分钟即可到达。
从门头及庭院不难看出这里整体为中式古典设计，院落比较别致，小桥流水，假山花艺，果树盆栽，锦鲤在水池中游来游去，古色古香，精心设计并用心打理的庭院让人赏心悦目，犹如住进了世外桃源一般。
在院落内还有几棵石榴树，现在虽然个头很大但还未成熟，如果成熟了的话，可以采摘来吃。
我住宿在3楼的一处大床房间，房间内干净明亮，用品齐全。不得不说这里的装修真的是花了大价钱，清一色的红木家具及地板，与庭院遥相呼应，尽显古色古香。
除了房间内，中庭及客厅装修也是清一色的红木家具，典型的苏式庭院风格。在客厅内还要很多的摆架，里面有主人收藏的一些太湖石，太湖石也是此处盛产的一个品种。
我住的那间房屋窗户正对着院落，东面的窗户对着山。透过窗户可以看到远处零星分散着的人家，再远处便是太湖，打开窗户明显可以感觉到湿润的湖风从远处吹来。听朋友讲，这里的早餐很有特色也好吃，只是一早为赶着回程，没有吃上，下次来一定要尝一下。

旅行小贴士

1、交通指南
 自驾：沪宁高速沪宁路枢纽转苏州绕城高速(甪直方向)甪直枢纽(东山、西山方向)东山出口下，沿湖至东山长圻码头停车，快艇或渡轮至三山岛。
 公交：苏州火车站(502公交)至东山(629公交)至长圻码头(快艇、渡船)至三山岛。
 2、渡轮快艇 渡轮票价为20元每人次，航时为25分钟。快艇单程每艘220元，可乘坐9人，随到随上，10分即可到达，方便快捷。 
3、门票：三山岛门票78元/位。团体7折优惠，持有学生证、老人证等有半价优惠。
