从丽江到大理，一个人的慢旅行！

游记 人文游 古镇 小众 摄影 自助游 2018/11/29

【前言】

第一次来到云南时，还是在我的大学生涯，那时候“旅行”对我来说还是一件奢侈的事情，是需要我省吃俭用几个月才能完成的，而如今，“旅行”变的越来越简单，说走就走早已是我的日常，但一次真正纯粹的“旅行”却又成为了“奢侈品”，所以决定给自己放个假，再次来到我喜欢的丽江和大理，给自己一次只有我，属于我的旅行。

【出行攻略】

1.行程
D1:成都－丽江－束河古镇，住：束河陋室客栈（五月店）
D2: 白沙古镇 －束河古镇 住：束河陋室客栈（五月店）
D3:丽江－大理－洱海－大理古城 住：悠然间客栈
D4: 凤阳邑－大理三塔 住：悠然间客栈
D5: 大理－成都
大理和丽江是国内旅游发展的最早的目的地之一，旅游相关设施都非常的完善，所以来这里完全不需要跟团，毕竟云南的旅行团一直都被诟病，就不要给自己的旅行找不痛快了。
2.交通
1）飞机：除了节假日，成都到大理或者丽江的机票一直都不太贵，有时候甚至是白菜价，成都的朋友还可以关注一下附近的绵阳机场，买到白菜价的可能性更高。（成都到绵阳机场非常方便，普通火车18.5一个半小时到绵阳，站前公交车站坐79路2元十几分钟就可以到达机场）
丽江进大理出还是大理进丽江出，可完全根据机票的价格而定，本次出行选择的先飞丽江，然后从大理回，往返机票不到800元。
2）火车：目前只有大理有通动车，成都没有直达（目前知道的是广东、广西有直达），如果喜欢坐火车出行的朋友，可以把云南之行的起点设到昆明，成都有动车直达昆明（6个小时左右），昆明有动车可以到大理（2个小时左右）。
丽江和大理之间只有普通火车（每天有两趟），我坐的是一列双层火车，不知道是不是都是如此，丽江到大理之间强烈推荐坐火车，沿途都可以欣赏到洱海的无限风光。
3）丽江和大理市内景点之间的交通也很方便，丽江的公共交通稍弱一些，几个古镇之间有公交车，不过花费的时间比较长，建议选择滴滴，大理的公共交通比较完善，有景区直通车也有环洱海的公交车。
4）租车：如果你们出行是4-5个人，落地租车是最适合的游玩方式，人少的话也可以选择租电瓶车或者自行车。
3 最佳季节
云南气候宜人，是一个一年四季都适合前往的省份，不过四季中首推冬季，因为冬季这里不冷，绝大部分时间是晴天，洱海的颜色最美，还有成群的红嘴鸥，所以最佳时节我觉得是11月－3月。

更安静的束河古镇

十几年前第一次到丽江时，只知道有一个丽江古城，也就是出名最早的大研古真，慢慢的，大研古镇里的游客越来越多，有时显得有些吵杂，后来大家又发现了更安静的束河古镇，和更更更安静的白沙古镇，其实丽江古城本就是包含这三个古镇，这次丽江之行，就选择了没有去过的束河和白沙。
束河古镇，纳西语称“绍坞”，意为“高峰之下的村寨”，这里是纳西先民在丽江坝子中最早的聚居地之一。 当年徐霞客游芝山解脱林时，曾路过这里，在他的记述中这样写道：“过一枯涧石桥，西瞻中海，柳暗波萦，有大聚落临其上，是为十和院。”而“十和”就是束河古镇的古称。
开通于唐代的茶马古道，从滇南经过大理、丽江、迪庆、拉萨，止于印度，经过唐、宋、元、明、清千余年的运营发展，茶马古道促进了沿线各族人民的经济文化交流，束河古镇即是丽江坝子中保存完好的驿站，所以来到束河古镇感触最深的就是这里的马帮文化。
束河古镇和大研古镇一样，镇中心也有一个四方街，四方街的周围也是古镇里最热闹的地方，如果你初次来到束河，看到眼前景象会觉得这里似乎人也很多呀，和大研古镇差不多，如果你喜欢清静，建议你往茶马古道的方向走，人就会慢慢变少，可以好好感受束河的魅力。
慢步在坑坑洼洼的石板路上，似乎能感受到当年茶马古道上的热闹与繁荣，现在，繁华已尽，留下的只是历史的沉淀和岁月的变迁，让我们慢慢去品味。
束河西山的最南边是松云村，后边有石莲山，山上有个山洞，像老虎张开的大嘴，为了震住老虎的威风，当地人在洞口建筑了一个寺庙，叫石莲寺，其中有一棵柱子是用铁链拴着的。民国时期曾经办过夜校，读书人围火夜读，成了一道风景，叫做“石莲夜读”，为束河八景之一。
和茶马古道相反的方向就可以来到束河古镇的“九鼎龙潭”，潭水清澈晶莹，水草曼舞，潭边有一截断碑，年代已不可考，用石头敲击，据说会发出清脆的声音。旁边还有一座寺庙，名叫“三圣宫”，西殿供奉观音，北楼供奉龙王，南楼供奉皮匠祖师孙膑。就是这不大的地方，就有束河八景中的三景，“龙门望月，雪山倒映，断碑敲音”，值得细细欣赏。
束河古镇里最常见到的就是拍婚纱照的情侣们，各种风格轮番登场，看到甜甜蜜蜜的他们，忍不住想留下这些幸福的瞬间。
虽然现在又更多的游客选择来束河，但这里仍然比大研要安静的多，这里的空气都弥漫着悠闲的味道，千万不要匆忙的来去，一定要在束河住上一晚，早上在房间里等待清晨第一缕阳光洒进房间，照亮我们的心。

在陋室客栈，享受束河的清晨

想要真正感受一个地方的美，一定要在这里多停留一下，最好的就是住一晚，因为傍晚和清晨都是游客最少的时候。束河古镇，我选择了一家在前往茶马古道方向上的客栈，这里离热闹的四方街有一点点距离，非常的安静，是我喜欢的调调。
在网上预订之后，客栈的管家就马上联系了我，详细询问了我什么时候到，告诉我应该如果更快的找到客栈，大晚上的还专门来接我，太温暖了。
由于到的比较晚，天已经黑了，没能看清楚整个院子的模样，只觉得特别安静，倒在床上很快就进入了梦乡。第二天一大早，拉开窗帘，清晨的一缕阳光洒进房间，探头望出窗外，淡蓝色的天空下，整个古城也像刚刚睁开眼睛，空气中散发着一股慵懒又新鲜的味道，忍不住深呼吸几次，整个人感觉特别清醒。
经过管家的同意，就在院子里随意逛逛，看看不同的房间，每一个房间虽然大小格局有所不同，但都是非常的温馨，最棒的是楼上的每个房间都有一个小阳台，可以躺在阳台上晒太阳，非常舒服。
前台的公共区域很大，早上阿姨给我做了一晚热气腾腾的面条，好好吃，满满一大碗都被我消灭掉了，如果不想出门，完全就可以在这里和朋友聊聊天，喝喝茶，旅行并不一定要在各个景点间匆忙行走，找个舒服安逸的地方坐下来，静静的感受，也是一种不错的方式。

雪山下的白沙，最纳西范古镇！

赵雷唱到“再也不去丽江”，歌词所表达的种种都是对以往丽江的怀念和如今丽江的失望，然而丽江不是只有大研古镇，丽江有太多太多的美还未被世人所知。
一提到丽江古城，很多人以为就是指的大研古镇，其实被评为非物质文化遗产的“丽江古城”包含了三座古镇，分别是"大研古镇“，“束河古镇”和“白沙古镇”，其中的“白沙古镇”就像待字闺中的幺妹，质朴、神秘、迷人！
白沙古镇位于丽江城北约10公里，比大研古镇离玉龙雪山更近了一些，不用像在大研古镇要登上狮子山才能一睹雪山，在白沙的街道上就可以看的非常清晨，一览无余。
白沙古镇虽然是“丽江古城”中面积最小，名气也最小的古镇，但这里却是丽江第一大家族，即丽江土司“木氏家族”的发源地，这里曾是纳西族的古都，是丽江政治、经济、商贸和文化的中心，一直到明代初1383年木氏家族才迁到大研古镇 现在木府的所在地。所以这里可以称得上最具有纳西遗风、最原生态的纳西村落。
白沙古镇并不大，只有一条主要的街道，游客也很少，看到的大多是当地的居民，坐在牌坊下面聊聊天，卖卖菜，安逸自在。
街口有卖丽江的特色小吃“鸡豆凉粉”的，凉粉虽然到处都有，但“鸡豆”是丽江特有的一种豆子，因富含黑色素，因此用鸡豆做成的凉粉、豆浆外表呈现黑色，据说富含很多有利身体的营养物质，而且白沙的做法好像和大研的不太一样，我在大研古镇吃的就是直接凉拌的，这里还要先在火上烤一下，味道更赞一些。
白沙古镇商业及手工都非常发达，解放前白沙有150多家打铜的作坊，铜器主要以铜火锅、铜壶、铜盆、铜瓢、油灯等藏区及周边少数民族所需的物品为主，但现在白沙街上只有一家尚在坚持手工打铜，师傅非常认真的在忙碌，我的相机快门响个不停也丝毫没有打扰到他。
白沙古镇最有特色的就是随处可见的扎染布了，小镇不大却有很多家扎染作坊，房内房外到处都挂着五颜六色，图案各异的扎染布，在阳光照射下随风飘扬，特别美！
扎染的最基本原理就是把布的一部分扎起来，在染色的过程中被扎的部分无法染色或者染色较浅。原理看似简单的方法，但要染出非常复杂、漂亮的图案，还是需要高超的手艺才行的。虽然染布颜色多种多样，但我还是喜欢以蓝白二色为主调形成的作品，有一种青花瓷般的淡雅宁静之美，也特别适合这个古朴平和的白沙古镇。
别看古镇上游客不多，外国朋友倒是碰到好几个，看来他们比我们还喜欢找这种原始清雅的地方，而且我一直很佩服他们，多么偏远陌生的地方他们都能找得到。
古镇上还有一个地方可以去小坐一会儿，那就是丽江四大名人沙蠡的故居，小院古朴幽静，充满纳西族农家风格，故居主人和尚忠老人是沙蠡的弟弟，听着老人给我们讲他和哥哥小时候的趣事，再来上一杯老人亲手制作的海棠果汁，味道甘甜爽口，回味悠长。
如果你想要足够的清静和放松，可以选择在白沙古镇上住上几日，绝对的悠然自得，没有任何打扰。有人说白沙古镇是20年的大研古镇，语气中充满的悲凉，希望不是这样，可以留一片净土给我们！

洱海无限美，到挖色来喂海鸥

上关花，下关风，下关风吹上关花；
苍山雪，洱海月，洱海月照苍山雪。
著名作家曹靖华游过大理之后，对大理的风、花、雪、月四景感慨万千，留下了这首诗。如果说大家对丽江褒贬不一，但钟情于大理的人似乎很多，很多朋友都说大理是一个去了还想在去的地方，而其中最让人留恋的就是这宛若无暇美玉的“洱海”。
苍山洱海，山水相依。洱海并不是海，而是云南第二大淡水湖泊，因形状形似人耳而得名。洱海发源于洱源县的茈碧湖，源头出自罢谷山，北有弥直河注人，东南收波罗江水，西纳苍山18溪水，面积约251平方公里，环洱海一周大约是120公里。
洱海边最早出名的应该是杨丽萍的故乡——双廊镇，有一段时间，双廊完全是小清新的完美代言，来大理必须要到双廊感受什么叫“面朝大海，春暖花开”，但是后来它就变成了一个大工地，就开始了漫长的修修修之路，不过听说现在修的差不多了，如果下次有机会再来大理，可以再次纳入行程中。
这次选择了洱海边的另一个小镇——挖色，位于洱海东岸，其三面环山，一面临海，这里自古即为洱海东岸一个鱼米之乡，是洱海地区历史渊源长远的白族本土文化发源地之一。
挖色除了比大理古城、双廊更原生态更安静之外，还有两个最大的优势，第一，每年冬天，从西伯利亚飞来到洱海来过冬的海鸥几乎都集中在挖色镇；第二，这里位于洱海东岸，是洱海边欣赏日落的最佳位置。
海鸥从每年的11月份来到这里，到第二年的3月初离开，挖色镇的海鸥主要集中在镇口和小普陀附近，它们似乎已经习惯了游人们的争相喂养，吃的不亦乐乎，漫天都是它们欢快的叫声，感染了周围的人们，心情马上也变得明亮起来。
小普陀始建于明代，1982年重修，为亭阁式两层建筑。一层祀如来菩萨，二层祀观音菩萨，是一个很小很小的岛，离岸边不远 ，可以花15元坐船上岛，不过没什么必要，远观更美一些。
感受洱海的最佳方式也是租个电瓶车沿着环海公路走走停停，喂喂海鸥，拍拍照，等到夕阳西下时，在海边欣赏最美的日落，这才是真正的旅行。
太阳余晖的照射下，似乎感觉整个世界只剩下这美景和我们，伸开双臂拥抱着美好的一切，让这一刻永远的烙在我的心上。
挖色，刚刚进入大家的视线不久，它的美还是那么原始，那么纯净，希望大家能好好的保护着它，给远道而来的海鸥留下一片净土。

悠然间，苍山下的一个大院子

以前大家来大理，都喜欢住在洱海边上，面朝大海，春暖花开，但是后来整个洱海边的客栈整治，都关了，听说现在有一些已经通过政治陆续开始营业了，但已经住过洱海边，这次想换一种风格，所以听朋友的推荐，选择了这家离大理古城不远，面朝苍山的“悠然间”客栈。
从大理古城打滴滴过来，只需要九块钱就到了，选择这里就是想感受一下远离尘嚣，享受住在田园间的惬意感。悠然间就在苍山脚下，有一个很大很大的院子，蓝天白云下，一个晃动的秋千，一艘搁浅的小船，感觉这里的一切都是属于我的，张开双臂，拥抱这里的一切。
房间里是面朝苍山的一整扇落地窗，拉开窗帘，引入眼帘的都是明亮的蓝色、白色、绿色，在冬天阴郁的成都呆久了，最需要的就是眼前的这一切，可以一扫心中的阴霾，让整个人也变得明亮起来。
冬日里的丽江和大理，最不缺的就是阳光，这也是为什么会有那么多人喜欢来云南，喜欢来丽江和大理，这里有暖暖的阳光，有悠闲的时光，在大都市打拼太久的我们需要有这样的地方和环境来放松一下，如果可以，我只想在这充满阳光的房间里，什么也不做的待上整整一天，度过悠然自得的一天。

凤阳邑,大理仅存茶马古道上的石头村落

大理，自古即以“下关风、上关花、苍山雪、洱海月”的风花雪月四景著称，这四绝也确实名不虚传，让人流连忘返。而这次的大理之行，在当地朋友的推荐下，发现了另外一处原始而神秘的地方，那就是离大理古城不远的“凤阳邑”，一个隐藏在大理仅存的一段茶马古道上的石头村落!
茶马古道起源于唐宋时期的“茶马互市”，用于藏区和内地交换各自需要的物品。凤阳邑茶马古道位于下关至大理之间的大凤路中途，属于滇藏茶马古道上的一段，此古道始建于汉代，成型于南诏大理国时期，全长1900米，保存的较为完好，这也是大理境内唯一的古道了。
茶马古道经南诏国都太和城遗址，往北就进入了凤阳邑村，凤阳邑村原名“砖窑”，历史上曾经以烧制砖瓦陶罐为主，后来取“丹凤朝阳”之意，方改名为“凤阳邑”。村里最有特色的就是房屋都是用不规则的石头砌成，具备了大理“石头砌墙墙不倒”的建筑元素。
这里的民居大多是大理特色的土库房，檐口以下皆用条石和卵石堆砌而成；现在还有40多间土库式土房仍然保留着曾经的容颜，步行期间，斑驳陈旧的土墙、茅草舞动的瓦顶、光滑圆润的石地板，历史的沧桑在这里留下了深深地印记，仿佛能看到当年这里来来往往的马匹，托着古道两边不同民族的希望！
逛着逛着又发现来一个很有意思的地方，很多民居的围墙上都有长势茂盛的仙人掌，问了路过的村民，被告知墙顶种上仙人掌是有辟邪的作用，原来如此。
路边一座似乎已经荒废的房子上有一面的贝壳墙，贝壳墙顾名思义就是把贝壳和泥土混合后夯打而成的墙，这种墙更结实、更牢固，而且墙体强，不易裂，这种贝壳墙在沿河或者沿湖周边的居民经常使用，就地取材而且独具特色。
在村里闲逛时，碰到了三个小朋友，得知他们要去观音井打水，就兴趣盎然的跟着他们一起来了。观音井在路旁一个非常不明显的石梯下面，一个不算太大的长方形井口，进口上方有一座观音石像，故名“观音井”，村民饮水都从这里取用，井水没有被污染，可以直接饮用，小妹妹渴得一到井边就先咕噜咕噜豪饮的半勺。
小妹妹非常热情，非要拉着我去她们家玩，和他们聊了一会儿才知道，他们的爸爸妈妈都去大理古城打工了，只剩他们和老人在家，最大的男孩子还要负责给全家人做饭，真是懂事。
凤阳邑以前还因草帽交易兴盛而被当地人称“草帽街”，现在每天都仍然有白族的老奶奶在村中的小广场上编织草帽，老奶奶看到我非常热情的和我打招呼，还问我从哪里来的，脸上挂着幸福而满足的笑容，有时间的话可以就坐在老奶奶身边和她们聊聊家常，应该可以听到很多有趣而动人的故事。
凤阳邑目前还没有完全开放，民宿较少且都为本地人所开，还很少有游客会来这里，想要感受茶马古道的深厚历史，白族人民的古老宅院和纯朴风情，这里应该是不错的选择，没有外界的纷扰，早晨很容易就睡过头，醒来听到的只有鸟鸣声，抚摸一下留下深刻历史印记的石头墙壁，喝上一口观音井的甘甜井水，和白族奶奶唠唠家常，多么轻松惬意的别样体验！

崇圣寺三塔，为段誉而来

最早对大理的印象应该是来自金庸笔下的《天龙八部》，最爱里面痴情的段誉，所以这次特意来到三塔，看看这个段家包括段誉在内三十位皇帝出家的地方。
崇圣寺三塔位于大理古城西北部1.5公里处，西对苍山应乐峰，东对洱海，崇圣寺曾经是段家的皇家寺庙，不过曾经的崇圣寺早已不在，现在看到的是近些年重新修的，只有寺前的三座古塔是已有千年历史的古迹。
崇圣寺三塔始建于南诏王劝丰祐时期（公元824—859年），先建了大塔“千寻塔”，高69.13米，是一座座方形密檐式砖塔，共16层。而后又建了南、北小塔，均高42.19米，是一对八角形的砖塔，都是10级。修建三塔，除了佛家所宣扬的可以成佛外，还有一个重要的原因，是大理古为“泽国多水患”，而塔有镇水的作用。
游览三塔最佳的方式，是进入景区后，现在乘坐电瓶车到达山顶的重点，也就是崇圣寺的大雄宝殿所在的位置，然后再沿着寺中的台阶慢慢向下游览，最后到达三塔。
离三塔景区不远处有一个三塔倒影公园，这里也是观赏三塔非常好的角度，水中倒影更加生动，倒影公园是和三塔景区共用门票的。倒影公园关门时间比三塔景区早，一定要注意合理安排时间。
