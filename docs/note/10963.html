长江慢岛  沙上绿洲

湿地公园捉蟛蜞，芦竹苇荡观白鹭。
双山岛上慢生活，原始野韵沙上洲。
双山岛位于张家港，是"长江三鲜"的主要产地，十几公里的环岛岸线上保留了良好的生态环境和自然风光，目前岛上有双山慢街、滨江荷园、大伯墩生态湿地、西滩公园等景点以及极具特色的18洞国际标准型"水上"高尔夫球场和双山岛垂钓中心等旅游项目，适合观光休闲度假。
​芦苇丛，滩涂地，炊烟人家，落日江水，在双山，过的是江岛度假的慢生活。
大伯墩湿地公园，建于江堤外的滩涂上。公园最早于2011年开工建设，同年9月开放使用。整个公园加外围芦苇荡面积约2000余亩，公园内配套建设有观光栈桥、凉亭等休憩设施。秋天是观赏湿地公园的最佳时节，芦花开遍，鸟群翩飞，从栈桥看下去无数的小蟛蜞爬过，场景美不胜收。
​何谓“大伯墩”呢？相传大约在三十年代，长江的大潮将双山岛冲击成两块，中间水流称为大漕，如今还依稀可见。当时有一位农民在大漕的东面和西面各种了八堆芦苇，漕西的面积比较大，大家就称其为大伯墩，漕东的面积较小，便被称作小伯墩。如今，小伯墩已经在时代的变迁中消失了踪迹，大伯墩还能被我们寻到痕迹。
​相传大约在清光绪三十四年就有江阴、靖江等渔民相继来到岛上捕鱼、垦荒，成为了原始的垦荒居民。曾有诗人吟咏当时双山岛的情景：“四望无边际，中浮一片沙”。当时的人们称双山岛为：西开沙、巫山沙。直至民国年间，江阴的财主祝丹卿第一次前来岛上围垦圩塘，但并未成功，次年围垦时便在岛上举行祭滩仪式，祭拜时从胯下望见雄踞南岸（今江阴界）的长山和巫山的秀丽景色，遂取名“双山岛”。其蕴含菩萨护佑，基业如山永固之意。
双山竞秀，百舸争流。
菁菁沙洲，人间锦绣。
双山，不是山，它是张家港金港镇长江水域中的一座岛，由长江中的沙洲涨积而成，双山岛全岛总面积约20平方公里，相当于2800个标准足球场的大小，环岛一圈16.8公里，四面环水，绿树成荫，岛上没有一座工厂，保持着原始、恬静的自然风貌，是长江中下游地区原始生态环境保持最完好的“生态岛”之一。
菱角鲜美，鱼儿恋勾。稻禾飘香，清风拂袖。蓬莱何处？
此地就有。黄金有价，宝岛难求。​
在北滩芦竹园欣赏临江美景；
在西滩公园近距离观赏白鹭；
在大伯墩生态湿地沿着栈桥油廊观赏芦苇滩涂；
在双山高尔夫球场挥杆运动后再品尝当地美食。
双山岛由于“江心岛”的独特地理位置，盛产各类江鲜鱼虾，尤以长江三鲜“刀鱼、鲥鱼、河豚”和数量众多的螃蜞最为出名。长江三鲜，都属长江洄游鱼类，咸淡水两栖，每逢春季溯江而上，在淡水产卵繁殖后入海，与一般定居类江鱼不同。因此，不仅肉质特别细嫩腴肥，且营养丰富。
蟛蜞豆腐其实不是豆腐，在双山岛体验了这道菜品的制作过程，很新奇。​
高尔夫酒店提供地道的长江鲜特色菜肴，食材都是酒店生态园自己种植的。​
双山岛上河道成网，阡陌纵横，池塘密布，绿树成林，环境幽雅，空气清新，东暖夏凉，是“长江三鲜”（刀鱼、鲥鱼、河豚）的主要产地。双山岛地理环境独特，自然资源丰富，保留了恬静古朴的自然风光和良好的生态环境。
看看这长江里的大鱼有多大？​
鲜美的土鸡汤——​还想喝一碗！
菜品的摆盘非常精美，不忍下箸。​
在双山岛可以品尝到正宗的长江三鲜以及岛上出名的美食如双山黑羊、长江白丝、长江杂鱼等。
PS：
双山岛旅游度假区位于张家港市西北郊长江作业主航道之间，处在扬子江城市群位置中心，与张家港保税区隔江相望，距无锡硕放机场约55公里，一小时交通圈覆盖上海、苏州、无锡、常州、南通等长三角主要城市。

​登临香山赏美景，金凤凰村泡温泉

香山因吴妃采香而闻名。
著名文豪苏东坡晚年曾数度来梅花堂怡情养性，徐霞客大师也曾多次邀约友人一起来到香山观竹赏梅。
站在高处，可以看到凤凰湖的秀美景色。​
香山景区内有采香径、梅花堂、桃花涧、​香山湖、梅花园、樱花园等景点，构成一幅湖光山色的美丽画卷。
晚餐在金凤凰度假村品尝了当地的特色菜肴。
丰盛的菜品让我们大饱口福，分享一下前菜八小碟——
餐桌中间的装饰是用萝卜雕刻的，栩栩如生，非常精美！​
晚上下榻金凤凰度假村，享受源自地下1206米深处的“苏南一号”养生温泉。

恬庄老街古风雅韵

恬庄老街为明代奚浦钱氏创建，清乾隆、嘉庆年间达到鼎盛。
现存有完整的明清历史建筑群和历史街巷。
这条街仅存280米，由481块石板铺就而成。
榜眼府、杨氏孝坊、杨氏南宅、蒋宅等都是流传于世的珍贵古建筑。​
老街两边古色古香的大小弄堂还有居民生活在这里，恬静怡然的生活节奏也是一道独特的风景。​

河阳山歌余音绕梁

河阳山歌馆位于凤凰山西北侧，是为了保护和展示首批“国家非物质文化遗产——河阳山歌”而修建的。
馆内小桥流水、亭台楼阁，一派古典风韵。
山歌馆采用现代科技手段向人们展现了凤凰地区丰富的历史文化遗存。
现在也有部分非遗传承人在馆内坐唱并传承技艺。​

永庆禅寺历史悠久

永庆寺建于梁大同二年（公元536年），已经有1500多年历史，为“南朝四百八十寺”之一。
它曾经与杭州灵隐寺、镇江金山寺齐名。​
现在寺内建有山门、天王殿、观音殿、地藏殿、钟楼、玉佛殿、大雄宝殿、佛教文化苑、文昌阁、香花桥以及放生池等。​
登临高处，能远眺凤凰湖景色。

宴杨楼——沙洲人舌尖上的记忆

午餐在外表低调却非常接地气的宴杨楼品尝了当地沙洲人喜爱的美食。
沙洲味道在当地人心里是儿时的美好记忆，那些浓油赤酱的味道更是陪伴了老沙洲人的一生。
文火慢炖、沙洲味道、江南小厨、宴杨面坊、暨阳小吃这五个点菜窗口，菜谱是刻在小木板上的，可见它的专一和持久。​
开胃小菜、粉蒸肉、包子、小馄饨、响油鳝糊、松鼠鱼、红汤面​都是宴杨楼的招牌菜，满满一桌，大饱口福！

杨舍老街新旧更替

吃好美食，去杨舍老街逛逛。
说起老街，其实遗存的真不多了，仅有一个焦家老宅还基本保存完好。
政府在老宅附近打造了一个休闲美食商圈，现在成为张家港非常热闹的地方。​
商圈里面以餐饮为主，附带一些休闲娱乐项目。​
在橱窗里面还能看到传统用具的展示。​
慢享、慢憩、慢游、慢品。
慢的是节奏，远离喧嚣、暂别忙碌；
慢的是方式，走走停停、看看坐坐；
慢的是脚步，静的是心，养的是生。
期待再来港城，享受这样的慢生活。​​​​
