【天鹅之旅首发】珠海美岛数不尽，在桂山岛等木棉花开

游记 摄影 美食 自助游 2019/04/06

桂山美岛数不尽

你是不是一听到岛屿是不是就想到了阳光，沙滩与椰林？
而珠海的万山群岛则是众多风情岛屿中独具魅力的一座。在广东，各种山林度假温泉酒店数都数不过来，人们似乎总是想到国外去感受海岛生活。其实不用飞得太远，在家门口就有非常美丽的海景等着你！
当我们不断的往东南亚航线飞的时候，你是否忽略了我国漫长的海岸线，虽然说不同的纬度有不同的风情。这南中国海亚热带季风的浪漫，你又是否了解呢？
这一次我来到的桂山岛，以富有层次的村落为特色，渔港小镇风光格外吸引人。在小镇的背后，就是大片的青山，格外青翠。除了渔村，岛上废弃的别墅也是有种别样的味道。
时至今日，在岛上依然能看到当年香港富商废弃的一排排半山别墅，当无人机经过这片绿树丛时，我也不经意的被这眼前的美景打动了。无论时光如何流转，自然永远是最美妙的，还未盖好的别墅显得有些陈旧了，两旁的绿树几乎要把他们包围，这大概就是时间的力量吧。
 
在岛上，另一个值得参观的地方就是妈祖庙。妈祖庙已经有两三百年的历史，是万山群岛地区现存最大、保留最完好的妈祖庙之一，一度是周边包括港澳一带香火最旺盛的妈祖庙。我们一度想要从半路上攀爬上去，但是在错综复杂的道路之上，我们还是一次又一次的迷路了。据说前往妈祖庙的路上还有非常多的树荫，颇有一种探险的妙趣。
虽然没有走路到达妈祖庙，但是航拍的时候还见到了它，寺庙不大，他就在半山之上，作为全都绿荫，显得格外的静谧。
推荐大家岛上另外一个比较好玩的地方，它就是游艇码头了，这里也是一个帆船基地。
你还可以租上一辆四人自行车，它有着红白相间的帐篷顶，沿着海路缓缓而行。远处是驻扎的军舰与繁忙的海港，近处是安静平和的海，这样的安逸生活，想必是每个人都想追求的吧。
 
其实，在桂山岛上面还有许多的景点：桂山舰英雄登陆点、码头广场、海关大楼，北山生态园景区、网箱养殖风景区、沙滩泳场、妈祖庙、小东山生态乐园、神岛石、女娲像广场、蔬菜生态园......这么多好玩的景点，你是不是心动了呢？下面，你就跟随我的这篇游记，好好的感受一下桂山岛的魅力吧！
距离一个月不到，我又来到香洲港了，而这一次我准备去的是桂山岛，之前对珠海岛屿的了解实在是太少，只要你对它们有了初步了解，相信你也会和我一样，深深地沉醉在了这一座座珍宝一般的岛屿里。前两次来到珠海的时候，天气也是非常的棒，虽然广州珠海都是阴天，但也许当你来到岛上就能拥有碧海蓝天！
 坐船大概50分钟的时间就来到了桂山岛，桂山岛和别的岛屿还挺不一样，各种建筑特别多，据说以前香港还有很多富豪在这里建楼房呢，这样一个有历史感，而且拥有众多人文风光的岛屿，当然是得好好欣赏了。
从珠海香洲港出发，伶仃洋往外开17海里便是桂山岛，比起耳熟能详的外伶仃岛、东澳岛等，桂山岛离大陆更近，地理位置更为微妙，自古以来便是“省港门户”、“海上咽喉”，也正因了这重要的战略位置，桂山岛上驻扎着部队、海关等部门，过去甚少发展旅游业，更少了游客的熙熙攘攘。
 窗户之外是宽阔的海岸线吗，几只渔船停靠在岸边，有种恍如隔世的感觉。

超美民宿

 广州市到处都种满了木棉花，而我早已经看过她们盛开时的模样，没想到过了那么久，来到珠海竟然还能看到鲜红的花朵，感觉这一趟旅程让我穿越过了季节，望着窗外的鲜花竟然有些出神。
 坐着接驳的电瓶车从码头来到木棉雅舍，短短的的一段距离里先让我品尝到了一下岛屿上的风景。带着激动地心情走进了木棉雅舍，面对这安静又舒缓的环境，那种激动慢慢的转成了心满意足的感觉。
 
 
之前在香洲港公众号上就看到木棉雅居的缩略图，一眼就相中了它。因为它建在半山之上，而且外观设计非常新颖，让我非常诧异。我以为桂山岛是一个比较落后和原始的岛屿。但是木棉雅居的设计实在太棒了，冲着它，就非常值得来一次桂山岛。
 走进院落里，你就可以看到这两个高大的木棉树，周围有非常多的沙发和吊椅。躺在上面不是还有木棉花掉上来，木棉花是一种很有分量的话，掉下来就像羽毛球砸下来，假如刚好落到你面前，请不要惊讶。
 
 
其实我们来到岛屿的时候是大晴天啦，用较为静谧的色调展示了木棉雅居之后，相信你也沉醉其中开始幻想着海上的平淡生活了吧。回过神，其实桂山岛也是充满色彩，有着非常多好玩景点的一座岛屿呢~

实用攻略

 告别客房，我们来到户外区域，好好享受这亚热带海岛的阳光吧！
我们住的是顶层的海景套房这里的户外空间格外多，晚上走到阳台还能看到海边上一条明亮的白线，看一下地图那应是澳门。就在这短短的距离里，一面是繁华的都市，而我却在沉浸在美好的夜色中。
 
木棉雅居的1楼还是一个24小时不打烊的书院咖啡馆。民宿、书院、咖啡馆、院落的混搭，让木棉有着与生俱来的书生气，宁静致远又与世无争。
书院里除了码放着文史名著，各类成人、儿童读物、杂志等，还收藏着创刊至今近70年的《中国国家地理》！从院子的前后进入都能闻到非常浓郁的咖啡香味，就像玄关处黑底牌子上写着：这里是“我的梦想咖啡书店”，想必老板也非常满意自己的这个作品吧。

书吧咖啡馆

 听说书店的老板曾是都市中朝九晚五打拼的白领，能在大海之中建立这样一座迷人的民宿，是多少人的梦想呀！
 木棉雅居的咖啡屋非常明亮，盘面上整齐的摆放着各种咖啡工具，头顶还有四周都能看到摆放的书本，随手都能拿来看。
 
 来到这里你会发现大多数书籍都是跟旅游和人文有关的，老板真的是非常有文艺气息的一个人呢。看到这么多的国家地理还有旅游类书籍我心情也是十分的激动，顺着旋转木楼梯拾级而上，除去通顶的书架，还有那扇照亮整个空间的玻璃门，小小的露台就像一个瞭望点，外面的景色尽收眼底，而退一步就回到了温暖的书屋，真的非常带感！
 喝过咖啡，我们沿着缓坡上的建筑群里穿到了主街上，准备去灯塔边上瞧一瞧。
灯塔位于码头旁的半山上，走上去的石阶非常的干净，随着高度的攀升，你所能看到的东西也越多。
 桂山岛灯塔位于广东珠海市珠江口桂山岛，建于1961年，1998年重建。为白色圆柱形钢筋混凝土结构，塔高14.9米。2002年5月18日，中国邮政发行《历史文物灯塔》特种邮票，全套5枚将我国5处6座灯塔全部绘入邮票，其中就有这座桂山灯塔。

 桂山岛景点

你在这里可以望见更远处的澳门和艘艘货轮，十分的惬意。
 傍晚的时候，虽然天色还没有大晴，但也出乎意料的看到了落日，那淡淡的橙红色笼罩在安静的桂山岛上，思绪也慢慢的飘远了。
我们也沿着空空的街巷，准备到海星餐厅享受美食，慢慢的走着，身边就是微微的海风，美好的让我难以置信。
在桂山岛，最不缺的就是各种各样的好看的建筑。在这里无论是朴素大方的民房还是拥有华丽外表的海关大楼，各式各样的建筑在绿树与大海的映衬之下总是变得饶有风趣。这边的石头很多都是与山体的岩石同一种，既不沉暗也不会有太多的反光，显得非常朴素。
 
这里距离香港非常的近，可以说是一个当时的小天堂了。90年代中前期，一批批的香港“跑船佬”在这里歇脚上岸，找寻生活乐子，在那个年代，桂山岛甚至比伶仃洋这端的大陆还要繁华。
海星餐厅外围还有一片质地柔软的沙滩，走在上面很是舒服。
一天的劳累过后，饱餐一顿才是正经事。这里是海鲜的天堂，看似一样的食材，在每个厨师手下不停的变着花样。海星餐厅，听起来就充满海岛味，走进去果然不凡。桂山岛的海鲜，比起其他地方的要鲜美许多，海鲜小火锅的汤底还是清汤的，最大可能的保留了食材的鲜味。
桂山岛地处珠江入海口咸、淡水交汇处，水深大部份在10米-30米之间，在辽阔的水域里有着众多野生鱼虾贝类品种，较著名的有石斑、黄花、鲈鱼、泥鯭、石九公、狗爪、将军帽、海胆等海产类，具备海钓的优越自然资源条件，我们坐船的时候都能看到很多来海钓的人。
 煮过的汤底也变得非常鲜美，喝上几口简直那滋味简直美到了心里。
 海鲜锅我们最后下的是鱼片，一下子就可以捞起来吃了，十分鲜嫩。

海星餐厅

 铁板茄子
 灌汤包
水煮牛肉
超大只海蟹直接放进去煮就行了，吃起来格外鲜甜。
吃饱喝足以后夜色正浓，我们慢慢走回旅舍去，发现门口已经有一只温顺的狗趴在地上等候，十分可爱。
 
 暗夜里的木棉书院就好像一座灯塔，看到它就仿佛回到了家一样，温馨无比。
在桂山岛的一天，在静谧的夜色下按下了暂停~在微弱的灯光下，两旁的木棉显得高大无比，站在阳台上，望向远处，海面上的星星灯火透露着浪漫的气息。
