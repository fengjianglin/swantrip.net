环游中国到湖南，查了湖南比较火的广州周边景点便是郴州。中国很多城市都从未听说过，也从未去过，对我来说几乎是一直空白。可每一个地方都孕育过几千年文明，祖祖辈辈的人在这里生长，留下了灿烂的文化，深入挖掘和了解会发现每个地方的人文、风景都各具特色，每个地方都有它性格。
我们在郴州呆了大概一周左右，时间也比较自由。南方最近经常下雨，有时候遇到阴雨天气我们便也会在住的附近逛一逛，吃一下附近的小吃店，走在错落有致的小巷子里，看着儿童放学抱走邻居家的小狗玩。等天气晴了我们就坐上公交车，踏上新的旅程。
这几天，当地朋友给我们介绍了很多郴州的历史文化。和我们一起游玩了裕后街、看了湘南起义纪念碑，一起开车去了仰天湖大草原，遇到了特别美的黄昏和日落。感恩一路遇到这些温暖的人，让我们的旅途有了很多的故事，变得更丰富和精彩。
说说郴州
郴州市位于湖南省东南部，别名“福城”、“林城”，地处南岭山脉与罗霄山脉交错、长江水系与珠江水系分流的地带。境内地貌复杂多样，其特点以山丘为主，水面较少。山地丘陵面积约占总面积的近四分之三。
郴字只有郴州有，最早是出现在秦朝篆体，由林、邑二字组成，意为“林中之城”。是湖南的南大门。郴州是中国女排福地，30多年来，中国女排先后19次到郴州集训或比赛，她们和这座城市结下深深的情缘，每次集训后都能拿到国际比赛冠军！1981年，中国女排就是从郴州出发，开始“五连冠”的辉煌征程。
这里自然景观非常多，中国原始生态第一山——莽山、国家级风景区东江湖、苏仙岭，未被开发的丹霞地貌处女地高椅岭、高山草甸仰天湖草原，也有永兴板梁古村、桂阳阳山古村......郴州地热资源也比较丰富，温泉矿物质多，来这也可以体验下天然温泉。

游记部分

我们选择了最近比较火的三个地方，高椅岭，东江湖，还有仰天湖大草原。来了以后发现真的很美，第一次见到亿万年前的丹霞地貌，还有梦幻迷雾的小东江，蓝天白云的大草原，都深深震撼着我们。

交通

我们选择的是坐普快，价格是50.5元。从长沙出发，走了4个小时终于到了郴州。
外省的朋友可以选择飞机到长沙再乘坐高铁或者火车。高铁从长沙到郴州价格149.5元，时间一个半小时。郴州离广州比较近，广州的朋友可以直接乘坐高铁，也可以选择自驾。
不少人来郴州选择包车。包车有2个方法。一是如果你定的是民宿，你可以询问民宿的老板去哪里包车，他可能有认识的人。第二，查看马蜂窝、小红书上的攻略，会有包车师傅的评论和练习方式。你也可以选择租车，去美团、携程、马蜂窝等旅行软件或者软件看看当地的租车点。

住宿：

我们住在郴州市区，这里交通还是比较方便的。我们是住在五岭广场附近，也有很多公交和大巴车。
也可以选择住在资兴的东江湖附近，这样可以直接早上起来去看雾漫小东江。

沟壑纵横的丹霞地貌——高椅岭

第一个要去的地方是高椅岭。高椅岭风景区位于郴州市苏仙区，景区树石相间、红岩绿水、险寨奇涧，构成一幅幅美丽的山水画卷。山脚碧水环绕， 高椅岭风景区的优美之处更在于，墨绿色的“天池”环绕着一只在水面张牙舞爪的“八爪鱼”。丹峰林立、红崖绿水，山、水、泉、洞、寨、崖、坦俱全，是一处不可多得的丹霞地貌奇观。
我们先坐201路到大巴客运站，再乘坐501路公交到高椅岭站下车，路费是7块钱，但是花费的时间较长一些。
自驾可以百度导航到高倚岭杨冲土菜馆，往右边这条路开上去，上面有个停车场。这个路口需要徒步一段时间上山顶，年轻人可以从这边过去。第二个是导航到航桥口镇高椅岭村委会，山路相对缓一些，再往左走便可以看到巨蜥湖，这里就可以看到高椅岭的精华部分。
现在高椅岭是免费的，不过正在搞开发，白天经常会被打挡板隔开。可以避开工作人员上班时间进去。我们去的时候正好是他们中午的吃饭时间，但也有大批游客进入。预计到2019年10月份，这里就正式开发对外收门票了。
山顶上长了一些绿色植被，可以想像这么多年没有被雨水冲下的种子在顶部扎根，落叶后变成泥土，变成了一丛丛树林。颜色鲜艳翠绿，非常漂亮。
山下是常年积水形成的天然水池，水面碧绿，平静的像镜子一样。由于未开发完善，这里的楼梯没有完全建好，不过山顶的路是比较好走的，j建议穿运动鞋，不要靠近边缘处。高椅岭各项设施也不是很完善，没有垃圾桶也没有厕所，要注意环保。
高椅岭为砂砾岩层，数千万年前的地质结构仿佛穿越到了恐龙时代。石头经过雨水冲刷，留下深深浅浅的沟壑。
高椅岭丹霞地貌栩栩如生，当然少不了宛如神龙降世般俯卧在此的龙脊，听说每一个来到高椅岭的人都会走过龙脊，它会给走过的人带来福气，如龙脊一般一路顺风。

梦幻迷人的水墨画——雾漫小东江

下午离开高椅岭后，我们定了资兴市的民宿，乘公交车到资兴，准备第二天看雾漫小东江。
东江湖位于资兴市东江镇境内，是5A级风景区。景区很大，面积达200平方公里，碧波万顷，景象万千，水质达到了国家一级饮用水标准。穿资兴城而过，汇入湘江，再汇入洞庭湖，最终汇入长江，流向东海。
东江湖里面有雾漫小东江、龙景峡谷、兜率岛、拥翠峡、黄草镇、东京寨、白廊镇和坪石水上活动区等主要景区。购买门票后坐观光车进入景区，可在相应站点下车。
景区门票有3条路线可以选择。
线路1：价格80元，包含雾漫小东江、猴古山瀑布、东江大坝外景、龙景峡谷、观东江湖。
线路2：价格145元，含兜率溶洞门票，游船费用（普通游船）、及景区内景点参观（雾漫小东江、猴古山瀑布、东江大坝外景、龙景峡谷、游东江湖、兜率溶洞）。
路线3，价格220元，豪华游艇环游兜率岛费用、东江漂流（雾漫小东江、猴古山瀑布、东江大坝外景、龙景峡谷、游东江湖、兜率溶洞）。
据说之前的小东江是不收门票的，后来也被化进了景区中。我们买了路线1 ，价格80元，我们只看了雾漫小东江和龙景峡谷，早上5点钟就要起床，拍完照我俩又带着困意去龙景峡谷爬了山。
雾漫小东江它的形成原理和吉林雾凇一样，由“温差效应”造成。小东江水是从东江大坝底部100多米深的坝底流出来的，水温常年保持在8—10℃，而小东江水面的温度则在20℃左右。江水早晨上热下冷，傍晚上冷下热，形成温差，便形成一层水雾，再加上两岸植被繁茂，空气中水分充足，因而每年的5月—10月间，江面常常雾弥漫，在峡谷中袅袅升腾，倘若一条玉带。全国各地的摄影爱好者来小东江摄影创作，反映雾漫小东江的作品多次获国内、外摄影大奖。
小东江沿岸修了不少观雾平台，你可以坐景区的公交车到二号桥站点下车，然后一沿着栈道步行至四号桥。每天的早上和傍晚会各有一场表演。
沿途有渔夫撒网，随手都可以拍出大片。
如果遇上一个好的天气，东方山头上天空慢慢变亮，太阳睁开朦胧的睡眼穿过山峦，将光洒向江面，为云雾镀上一层金粉，江面渐渐变成调色盘，周围的云雾和倒影也染上红、紫、蓝、黄等色彩，如梦如幻，像一幅泼墨山水画。
江面云雾中有一艘渔船，带着斗笠的红衣渔翁每天都会在这里表演捕鱼撒网。金色的渔网在江中散开，撒向天空绽放开来，随着岸上长枪短炮的一阵咔嚓声慢慢飞入碧波中，激起一阵浪花........
东江湖风景区除了有雾漫小东江这个奇观外，还有龙景峡谷、东江大坝、兜率岛、兜率灵岩、东江漂流、三湘四水、东江湖文化旅游街(含东江湖奇石馆、摄影艺术馆、人文潇湘馆)等著名景观，以及豪华游艇、仿古画舫游湖和水上摩托、水上跳伞等丰富、惊险、刺激的水上乐园项目。

龙景峡谷

龙景峡谷就在东江大坝旅游码头旁边。龙景峡谷有个环线，进门走一段往左是小环线，经过水上浮桥，用时比较短，很轻松也不用爬山。
大环线也有两条路上山，楼梯非常陡走起来也很累。其实所有的景色都在右边，包括鸳鸯瀑、三叠瀑和龙吟瀑。建议从右边上去，因为后面没有什么景色，走到鸳鸯瀑就可以折返了。走完龙景峡谷大概一个多小时，里面就是小瀑布，个人感觉性价比不高。

纯净的童话世界——仰天湖草原

在这里我们遇到了此行最美的景色。周五刚刚下完雨，周六的天气阴沉沉也难以预测是否会下雨。我们的沙主本想带我们去市区转转，但是心心念念的仰天湖还没去。下午一点钟，豪豪便提议去看一下。
仰天湖离市区大概60公里，至少要开一个半小时。我们的沙主非常爽快的答应带我们去。山路不是很好走，很多急转弯，山路十八盘，曲曲折折绕上山。
仰天湖草原平均海拔1100多米，因草原腹地有第四纪冰川时期造山运动馈赠的死火山口汇集泉水形成的湿地“仰天湖”而得名，是中国最南端的高山草原。草原附近还有华夏奇观“仰天巨佛”，鬼斧神工的安源石林等景观。
仰天湖景区是受门票的，外面还有很大一片草原不收门票。景区里面是有个湖，外面景色也不错。
随着海拔越升越高，气候越来越凉爽，风越来越大，山顶植被也由树林变成了草甸。更惊喜的是，我们穿越到乌云之上，有大片的蓝天大朵的白云。连绵起伏的山脉上很多大风车在旋转，还有低头吃草的牛和马。
 
站在山顶，远处大片的白云浮在天边，像一朵朵棉花糖，仿佛伸手就可以触摸到。云层在光的折射下有明有暗，层次分明。有的地方连着土地黑压压的在下着雨，有的地方洁白无瑕，随风飘动，我的心情也像云一样轻松。
 
因为怕天黑路难走，也怕回去太晚，我和豪豪提议下山。我们的沙主却说：来都来了，看一下夕阳吧。
于是，我们在山顶继续吹着风，喂马儿吃草，用相机记录着云彩的变化。不知不觉，太阳西沉，整座山只剩下了我们三人。光线变得昏黄、柔和。这时候，山谷里涌出了云海，天边出现了彩霞。
我兴奋的叫了起来。从远山的尽头如诗如画般一轮瑰丽的红日正慢慢消失天际；蹄铃声声，唤起遥远的孩提梦幻，让人置身于金色童年的遐想之中;黄昏、日暮燃照着天海穹苍，派生出一种落霞与孤雁齐飞的意境;浪漫的气氛顿时漫延开来。
大自然赐予我们绝美的画卷，好在我们没有错过。

郴州美食推荐

1、鱼粉。当地比较有特色的便是鱼粉，粉有圆的还有扁的，10块钱左右一大碗，里面的鱼肉也特别多，汤汁非常鲜。比较正宗的就是栖风渡鱼粉。
2、烧鸡公。烧鸡公是郴州特色菜，这道菜一般选5斤以上的土公鸡为原材料，切成大小适中的数块，配上当地茶油加上干椒姜蒜烹制，口味鲜香麻辣，配上家里的大碗，乡土风味十足。
3、嗦螺。嗦螺是当地发风味小吃，盛夏十分流行。郴州人会在街边一边喝啤酒一边吃嗦螺。嗦螺味道比较香辣，没有腥味，轻轻一嗦肉就出来了，去郴州可以尝一下。
4、三文鱼。在资兴不可错过的便是特产三文鱼。东江湖附近很多饭店，几乎每家都有三文鱼主打菜系。其中三文鱼可以有很多吃法，一鱼五吃，把鱼皮、鱼肉、鱼骨、鱼头什么的分开烹制，（凉伴鱼皮，鱼肉刺身，鱼骨焦盐，鱼仔煎蛋，鱼头鱼尾清蒸，鱼骨还可以煲粥），味道也是非常鲜美的。
5、东江湖鱼。这里的鱼是在一级水质长起来的，碧绿的湖水甘甜可口，鱼肉紧致新鲜。这里有各种各样的鱼，火培鱼、黄尾鱼、翘嘴鱼、银鱼、桂鱼等等，也可以带一些回家尝尝。
6、临武鸭。是临武特有的一种土鸭子，和野鸭子一样的生长习性，喜欢在田间，水塘中生活，因此它全身是瘦肉，而且肉质细嫩，味道非常鲜美，是纯绿色的健康食品，深受大家的喜欢。
除了这些，郴州还有很多美食，东江湖烤鱼、血鸭，瑶家腊肉，清蒸翘嘴鱼，剁椒雄鱼头，米粉肉，米粉鹅，爆火肉、坛子肉。想吃宵夜可到东江吊桥旁有很多大排挡，有唆螺，口味虾等当地美食。这里水果也非常多，桃子、李子、杨梅、枇杷，每个季节都有不同水果，且价格实惠。
