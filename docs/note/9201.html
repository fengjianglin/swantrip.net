『 南法 』梦幻海岸与紫色薰衣草的追光之旅

游记 摄影 自助游 自然奇观 自驾游 赏花 2019/03/24
“很多人来此寻找光明。我来自北方，是一月那绚丽的色彩和白昼的光辉感动了我。” — 这是马蒂斯对尼斯的心声，却也是所有画家的心声，同时更是每位来到尼斯的游客的心声。
普罗旺斯，全世界独一无二的浪漫胜地，每年的6、7月份，当薰衣草的盛开为这片土地染上令人目眩神迷的紫色时，已然无法分辨那让你晕眩的味道，是薰衣草的芬芳，还是自己的心跳。
因为时间的关系，原计划7月初去南法的旅程提前到了6月底，所以行程中舍弃了花期较晚的索村，走了个顺时针的大环线前往普罗旺斯。幸运的是，一路天气晴好，美景不断，无论是路上的风景，还是南法的美食，都给这次旅行带来了无限惊喜。

蔚蓝海岸第一站

旅途的第一站从蔚蓝海岸最著名的度假胜地——尼斯开始，提前一天抵达的尼斯，迎接我的是一场迷人的落日，还有那久违的南法阳光。
第一天的日程可谓是昏头转向，为了倒时差睡到了大中午，匆匆赶去机场取车才发现在SIXTS租宝马居然需要5年驾龄……然后……在订单到期的前1分钟操作完成了退款，再然后，营业员小姐姐在我诚恳的小眼神下认真地为我搜寻空闲的车辆，最终拿到了一辆欧宝SUV，性能过关，大小也能满足我们4个人的行李。皆大欢喜，顺利取得车子后，还是默默地塞了一张消费给营业员，把人家感动的……
在机场又等待了一会，终于接到了另外几位小伙伴，开开心心的前往尼斯老城区。
事先定好的位于海边的民宿，地理位置不错，距离老城和海边都很方便。但是！必须得吐槽一下尼斯的道路设计，绝对不是理科生设计的，一路上绕来绕去的，加上各种维修和单行道，在这里绕了快1小时才找到路，一路上还要忍受后座两位小伙伴的撒狗粮，作为司机的我容易嘛……
夏季的欧洲落日时间总是特别晚，在民宿简单做了意面和牛排，趁着黄昏前来打卡尼斯的第一个景点：尼斯天使湾。
没错，以下这两人就是此次南法之旅一路上不停撒狗粮的小伙伴，好吧，看在你们为我发票圈征婚还成功介绍了漂亮小姐姐的份上，原谅你们了……
尼斯的天使湾尽头就是老城区的制高点：尼斯城堡山，这里关门很早，黄昏前就停止上山了，想在夏季去山上拍黄昏的朋友要好好规划下。退而求其次，于山下的观景台也是一个欣赏尼斯老城黄昏夜景的好地方，这个I LOVE NICE的雕塑也算是每个游客都要打卡的地方了吧，坐在海边，迎着海风，还可以随手拍几张游客照。
眺望着天使湾与老城的黄昏，很美很宁静，沙滩上是小酒吧和年轻人的聚集地，一人买上一瓶啤酒，坐在观景台边，欣赏这南法的落日，静静地等待夜晚的来临。
尼斯的第一天，就这样在焦急和宁静中度过了。
天使湾紧靠着英国人漫步大道，清晨时分，沿着海边散步不失为一个绝佳的选择。英国人漫步大道，法语名字为Promenade des Anglais，因为是英国殖民者于18世纪修建的，估计是当时的英国人都喜欢沿海散步吧。
蔚蓝海岸的海水果真是晶莹剔透，这里的沙滩很奇特，尽是鹅卵石为主，海边遍布着各种酒吧和露天休息场所。
沿着漫步大道行走，则可以看到许多蓝色条纹的长椅，这也是漫步大道上的标志性长椅，走累了就坐下小憩片刻吧。南法的阳光还是很强烈的，一大早就已然艳阳高照，许多当地人在这里跑步、骑车，不失为尼斯的一道风景。
尼斯海边的房屋，大多是冷色调和淡色调的，给人以清凉的感觉。
告别了第一天的匆忙和长途飞机的疲劳，一行四人元气满满的，迎着阳光，开始了蔚蓝海岸的自驾之旅。
从尼斯老城区出发，沿着海边的山路行驶，大概10分钟车程，可以抵达山间的一个观景台，在这里可以一览远方的尼斯城区和尼斯海港。阳光下的蔚蓝海岸，真的是放眼望去尽是一片蔚蓝的色彩。
同行的小姐姐，已经拿起相机狂拍不止了。
持续撒狗粮二人组
继续前行，便是联结在尼斯和摩纳哥之间的悬崖公路，公分三段，沿途尽是令人目眩神迷的地中海美景。
低断崖公路旁便是滨海自由城，离尼斯老城约20分钟车程。这里是一片美丽的海港，依山傍海，拥有着梦幻般的海滩，14世纪的老城街道仿佛在诉说着这里数百年来不变的故事。
滨海自由城著名的度假沙滩，就在小镇边上，刚好在被周边的青山环绕，许多外国人会选择夏日来此度假，住上个三五天，享受阳光所赋予的美好。

袖珍而富有的国度
进过一路的无敌海景，在中午时分终于抵达了另一个国家：摩纳哥。从尼斯进入摩纳哥需要经过一段隧道，最令人惊讶的是隧道里居然还有转盘！而且没有导航定位！只能靠识别路牌行驶，所以第一次过来的小伙伴要谨慎驾驶，千万别走错路了哦。
我们的车停靠在城堡山下的地下停车场里，作为一个袖珍小国，这里的停车位可是很稀有的，特地选择此处也是为了确保有停车位，而且比较安全。从停车场电梯上来恰好是摩纳哥海洋博物馆，背靠山崖的高层建筑，还是有几分壮观的。
这座举世闻名的海洋博物馆建立于1910年，是由阿贝尔一世亲王下令建造的，由于出发的太晚，时间关系我们只在外面参观了一圈，顺带跟周边的花园鲜花拍拍合照。
摩纳哥很小，仅仅1.95平方公里的领土，奠定了她世界第二小国家的地位，但就是这样无比浓缩的小公国，却充满着无限的美丽和色彩斑斓的风景。
这里山地起伏，最中心的当属摩纳哥城，这片被称为Le Rocher的地方是如今摩纳哥保存最完好的中世纪街道。往往来到摩纳哥，沐浴着这里的阳光，并不需要掐着时间打卡各种景点，我就喜欢这样漫无目的的在当地老街道上行走，跟来来往往的游人笑着打个招呼。
如此好看的中世纪街道，同样是不可错过的拍照场地哦。
如今的摩纳哥显然也已经开发成满是旅游气息的地方，大街小巷随处可见的是各种小商品纪念品店，就是不知道这些小玩意是否都是made in China呢？
从海洋博物馆前往王宫的途中会路过的摩纳哥大教堂，始建于1875年的罗马-拜占庭式教堂，可免费进入参观。
经过一段古朴的小路，就可以抵达摩纳哥城最高处的王宫城堡，巨大的广场和边上的观景平台自然也成为众多游客的聚集地。这座宫殿修建于13世纪，如今则成为阿尔贝亲王与查伦王妃的官方宅邸，据说每天11:55会有皇家骑兵团的换岗仪式，到的有点晚，错过了……
王宫可购票进入参观，成人€8，网上购票可以省去排队时间，更快的进入一窥摩纳哥的皇宫生活。
然而，对于我们已经看腻了欧洲各种奢华宫殿的人来说，还是王宫边上的炮台，如今已成为俯瞰整个摩纳哥城的观景台更为吸引人。即便大中午来此，都能看到这座环绕海港的山城的繁华和美丽。
王宫边上的观景台，不断有游客来此休息，嬉闹声总是不绝于耳，午后来此欣赏一番摩纳哥的城市美景，或者晚上时分在此欣赏夜景，必然是一个好选择。
在摩纳哥，或者说是南法，最不欠缺的就是豪车。不时有令人热血沸腾的引擎声响起，然后就是一辆辆酷炫的豪车飞驰而过。
随着豪车带来的兴奋，小姐姐们也开始在街道上各种摆拍了！
中午随便在城堡附近的街道上找了家环境不错的餐厅，南法的美食还是很不错的，经典的传统贻贝、吞拿鱼沙拉、玛格丽塔披萨，都是南欧常见的美食。而摩纳哥的消费相对于南法小镇来说确实又比较高，几份主菜合起来大约60欧元。
沿着这条长廊走下山坡，就是摩纳哥居民城区啦。
走到海边便来到了摩纳哥港，这里仿佛是一个停满了豪华游艇的炫富舞台，各色游艇令人眼花缭乱。
紧邻着港口的一圈马路则是世界一级方程式大奖赛的赛道，这可是每一个赛车迷的盛宴，赛道从蒙特卡洛大赌场为起点，环绕赛道78圈的极速赛事，可不是一般人可以参观的哦。
或许是港口已被豪华游艇所占领，亦或许这里离沙滩有一定距离，在这炎热的夏日里，还是有许多人在这座大型泳池里嬉戏玩耍的。
从城堡山绕过整个港口（在这炎炎夏日里走起来还是有点远的），远远地就能看到这座低调的建筑，谁能想到这可是摩纳哥城极致奢华的代表：蒙特卡洛大赌场。
大赌场的参观时间从早上9:00至正午12:00，而下午14:00开始则是赌博的时间，因此只是想进去参观的伙伴记得早上先来此。没能进去参观，又没钱去赌博，只好在门口看看停着的各种豪车了……

悬崖边的小镇
告别了摩纳哥城的奢华，我们迎着午后的阳光，来到了蔚蓝海岸另一热门的悬崖小镇：埃滋。
这座由岩石建成的小村庄就坐落于尼斯与摩纳哥沿途的峭壁之上，错落有致的石屋和蜿蜒曲折的小路由中世纪保存至今，加之那迷人的海岸线风光，每年都吸引着无数的游客来此。
这个金山羊城堡拥有着小镇上最著名的米其林三星餐厅，记得第一次带我奶奶来此游玩时想来试试，一看价格只能选择去隔壁吃披萨了……
在小镇上行走无疑就是一种享受，不时看到游客带着自己的小宠物在此散步更是每每吸引了我们的目光。
看这小姐姐蹑手蹑脚的在偷拍什么呢？
原来是一团蜷缩在草丛中的小猫咪，可是为什么要躲那么远拍呢……
小镇上随处可见的是各种植物还有这些一片片白色的鲜花，虽然不知其名，却也美艳无比，于墙头低垂，形成一道道风景，更加成为小姑娘们争相拍照的地方。
抓拍到一位气质超绝的外国美女，果然白裙子到哪都是绝配。
在埃滋小镇上，无论走在街道的哪里，都是古香古色的，没有什么现代化的设计，石头堆砌的喷泉、雕像，简洁而不失美观的木门木窗。
好吧，同行的小姐姐真的是绝不放过每一个角落，各种摆拍系列……
一路前行往山顶，途中只有2、3家小咖啡馆正在营业，阳光不时透过树林洒落在街道上，在这里喝杯咖啡、吃点东西，环境确实舒适，但食品味道真的很一般。
阳光明媚的地方，人们就喜欢穿花裙子，很有黑人大妈的既视感。
或许小镇的风景还不足以满足你，那么位于小镇山顶的热带植物园就成了埃滋必去的景点，虽然需要€6的门票，但也是物超所值，毕竟在这个小小的花园里，就能欣赏到美丽的地中海全景。
花园里的主要植物是仙人掌，因为夏日的到来，不时还有几簇正在开放的薰衣草。沿着一小段石阶向上，周边可没有任何遮挡，如果正午来此估计真会被晒死。
估计是位于小镇山顶的缘故，来此的游客相对于镇上狭小道路里的人来人往，这里相对少了很多人，大多是年轻的小伙子和小姑娘，然后各种自拍……
发现牵着狗来这里散步的游客还是挺多的，PS：这只汪星人的毛发好好看！
从山顶欣赏蔚蓝海岸的风景。夏日的南法，日落的很晚，基本都是晚上9:30以后才开始天黑，即使是下午来此，依然艳阳高照，如果想要感受这个小镇的静谧和黄昏的美景，不妨在此小住一晚。注意了，这里的奢华酒店可是很贵的。

从这里可以看到对面小镇的风景，南法的许多小镇都是这样，依山傍海，或者说是沿着山峦起伏所建成的，高低起伏很明显，蜿蜒曲折的羊肠小道同样也很锻炼车技。
摩纳哥和埃滋小镇的一日游结束，本想去山上拍照的，然而，正值世界杯期间，怎能不去当地酒吧看场球为法国助威呢？！
这里充满了香水的芬芳
在南法，普罗旺斯因其绵延不绝的薰衣草花田而成为浪漫的代言人，而这座位于南法内陆，距尼斯、戛纳仅半小时车程的山间小镇，则因其众多著名的香水工厂而成为法国的香水中心。格拉斯，拥有“世界香水之都”的美名，近30家香水工厂设立于此，生活在各色花香的环抱中或许正是这座小镇的写照。
告别繁华的尼斯，我们的下一站正是这座充满花香的古老小镇。从空中望去，尽是红瓦铺就的古朴小屋，至今仍完好地保存着中世纪的美貌。
小镇如南法的一贯作风，依山而起，蜿蜒的山路穿过每家每户，也许是对于阳光的热爱，更或是这里地广人稀，几乎每家每户都拥有自己的游泳池，简直羡慕。
我们住在距离镇中心约5分钟车程的度假村里，环境很棒，阳台正对着草坪，中午开车至此，大伙儿都忍不住葛优瘫在外面的椅子上晒太阳小憩。
酒店的泳池，度假村的必备，在这热浪夏日里是最好的降暑良药。然而，我们来的时候池子里全是中年油腻大叔……
== 加里玛香水厂 ==
位于小镇山腰处，入口是一个类似香水商店的地方，来此的主要目的是事先预约了香水制作，一个台子最多两人，一人制作，一人负责观看拍照。预约请上官网，自己搜索Galimard，价格好像是50欧上下。
香水调制台，会分为三个阶段的香精调试，根据自己的喜好，要闻过这三排所有的香精……只能说，前方高能，鼻子敏感者慎入~调香结束后，可以购买一个自己喜欢的瓶子，给自己的香水命名，印在香水瓶上，同时还能获得一张调香师毕业证书，然而妹子嫌味道太甜，于是辛辛苦苦调制的香水就成了家里的摆设品……男士们，想要讨好妹子还是带她们自己来买吧……
== 国际香水博物馆 ==
这里位于小镇的中心，周边并不好停车，我们将车子停靠在花宫娜边上的一片较大的停车场，然后步行过来博物馆。香水博物馆本身是一座14世纪的古堡，内部则展出了各个时期精美的香水瓶子和制造香水的机器。来此博物馆，所为的并不只是视觉上的享受，更是嗅觉上的旅行。
展馆整体来说还是比较大的，短短的参观时间，可以了解整个香水发展的历史，其中比较有趣的是随着中世纪以来的发展，出现的形状各异，越来越有情调的香水小瓶子。
感觉这是最好看的一个东西……

== 花宫娜香水厂 ==
个人觉得相比于先前的香水博物馆和香水制作，这个建于1926年的香水厂应该是来格拉斯比较值得一看的了，如果时间较少，不妨就选择来此参观，然后再利用傍晚黄昏于小镇商业接到上逛逛。
花宫娜是格拉斯最古老的香水厂之一，除了自由参观的一些香水瓶子等展品以外，还可以免费参加向导带领的Tour，参观位于工厂内部的生产车间，更详细了解香水制作的每个步骤，还能近距离看到一些香皂等产品的制作过程。
负一层开始的工厂内部参观需要讲解员带领，每隔一段时间会有英语、法语、西班牙语等不同语言的讲解。工厂内部车间不能拍照，仅外面一些废弃的生产容器和历史讲解区可以拍照。
带领我们的是一位漂亮的法国小姐姐。
Tour的最后就是一如既往的商场，毕竟免费的参观和讲解，主要还是为了吸引游客购买东西。说实话，这里的香水味道还行，就是太贵，不如去买自己喜欢的品牌的香水。

只想喝一碗马赛鱼汤
选择来到马赛，主要是为了进入普罗旺斯地区前停留一晚休息一下，其次就是来品尝一下传说中的马赛鱼汤。曾经路过马赛数次，却总是匆匆而过，从未好好玩过，如今，终于如愿能来马赛的市中心逛逛了。
我们的酒店位于马赛市中心外围，只要是怕了南法城市那不按常理出牌的曲折道路。地铁出来便是马赛老港，还记得当年学习法语时，刷了好几遍的无厘头法国电影《的士速递》有一段画面就是在这里取景的。
这里是一个充满活力的港口城市，街头巷里也都洋溢着她的历史情怀，作为马赛发展史上的摇篮，似乎整个马赛都是围绕着这一片海港而兴起的，20几个世纪以来，这片海港始终停靠这许多船只，繁华依旧。
如今的老港已然告别了曾经主要商业海运码头的身份，渔船、游艇和观光快艇成为了这里的新主人，少了一点工业气息，多了一份休闲与轻松。走在海边，听着海浪的声音，游艇随波摇曳，头顶的天空与大海相互交融，人们轻声交谈，不时传来阵阵笑声，在这阳光下，显得格外惬意。
作为马赛旅游的代表景点，老港终年吸引着无数旅客来此，在海边，经常能看到各地游客的身影。外国人的旅行模式也很令人感慨，带上一只狗，坐在海边，晒晒太阳，一晃就是一个下午，无比轻松自在。
既然是马赛的主要游客聚集区，老港周边的街道上早已换成了各种纪念品商店，餐厅、咖啡馆遍布，而我们的午餐就是在老港边的一家不起眼的小店解决的。专程跑来品尝传统的马赛鱼汤 Bouillabaisse，好吧，可能是事先没有做好攻略，又或者老港周边实在太过商业化，鱼汤的味道很一般，相对的还很腥，失望之余有点提不起拍照的兴趣，不如拍一张餐厅外的风景吧。没有吃到好吃的鱼汤对吃货来说绝对是个遗憾，今年7月再临马赛一定要好好吃一顿！
（传统的马赛鱼汤是马赛地区最有名的一道菜，千万不要为了省钱吃便宜的鱼汤，后来跟法国朋友聊天，据说传统的马赛鱼汤一般是2人起一份，正常情况下约5、60欧元每人，同时鱼汤会分两部分：一者为汤，浓汤，用面包加上大蒜汁蘸着汤吃；另一者则是煮好的鱼，现场去骨食用。所以，想要好好品尝大餐的朋友，就不要吝惜自己的钱包啦，放开来吃吧。）
港口处的观光列车，一路从老港开往马赛制高点处的圣母院，约1个多小时的行程，如果不想走路的可以选择坐小火车观光。
老港一侧的圣让堡，与对岸的圣尼古拉堡就像是老港的两个守卫，数百年来屹立于此。圣让堡上可以一览老港的景色，并且内部的城堡花园和庭院无需门票。然而，周二关门，很不巧的是，我们只在马赛停留周二一天，于是没能入内参观了。

从教堂边上的楼梯向下，是一片濒临海边的广场，前方是著名的新兴建筑：欧洲及地中海文明博物馆。
一旁的马赛-福斯港，蓝天下远眺大海总能给人以心旷神怡的感觉。
这座博物馆已成为马赛现代建筑的标注，内部展示了地中海地区的历史、文明和文化发展，更吸引人的是博物馆本身大胆的建筑风格，与周边的估计似乎有点格格不入，却又显得格外耀眼。
因为大家都觉得有点疲惫，没有继续认真参观，从博物馆边上的马路可以乘坐公交车，直达马赛的制高点：加尔德山顶（La Garde）。

罗马遗址+梵高足迹+米其林美食
“发现古罗马时期的阿尔勒，追随梵·高的足迹，并在普罗旺斯几间最棒的餐厅享用美食”，这是Lonely Planet上关于阿尔勒游玩亮点的介绍，很巧的，我们此行把这三项任务都完成了。
从空中俯瞰整个阿尔勒，有一种回到了罗马的感觉，巨大的圆形古罗马式斗兽场显得格外突出，古老的中世纪风格建筑又有种漫步在托斯卡纳小镇的感觉。
路过游客中心首先看到的是阿尔勒古罗马剧院（€6），是一座古罗马式的露天圆形剧场，其历史可以追溯到公元前1世纪末，如今依然在使用，夏季常被用来举行音乐会和歌剧表演。从街边可以俯瞰内部，没有专门进去参观。
阿尔勒的另一个著名罗马建筑：古罗马斗兽场。因为参观过罗马和维罗纳的斗兽场了，这里就不再专门进去参观啦。在斗兽场外围的街道、广场随便走走，等待午饭时间的到来。
这是建于公元2世纪初的斗兽场，迄今亦有近两千年的历史，算是罕见的保存良好的古罗马建筑。斗兽场长136米，横宽107米，能容纳2.6万多名观众，平日里进去参观的游客较少，如果厌倦罗马斗兽场的人山人海，可以选择进去参观。
阿尔勒以斗兽场为中心，向四周纵深着各种小道。作为一座旅游城市，小镇上的商业气息并不十分浓厚，依然保留着南法人的休闲懒散，很多商店甚至没有开门。除了斗兽场广场周围零零散散的纪念品店，这里最多的就是餐馆、咖啡厅啦，据说在这迷宫般的小路当中，藏着不少普罗旺斯一带最为杰出的法式餐馆。
斗兽场边上的小教堂：圣托菲姆教堂，教堂本身并不怎么出名，可以免费参观，而位于其内部的回廊则需要5.5欧元的门票，显然比教堂有名许多。
老城街道狭窄，自驾显然不是一个好选择，因而可以看到不少三轮车。
老爷车、复古车在南法也不是稀有物品，老城里更是随处可见，变成了各种适合拍照的道具。
在斗兽场周边，经常可以看到坐在阴影下表演的街头艺人，时而有孩童在此嬉戏，悠扬的音乐伴着小朋友们的欢笑，即便是炎热的夏日，都不觉得烦躁了吧。
欧洲城市的特点，到哪都能看见的涂鸦，还有彩色的门窗。
阿尔勒因为其保存至今的罗马时期建筑而闻名，同时，这里还有着传奇历史人物梵·高的生活足迹。当年，阿尔勒地区的阳光和明亮的色彩另梵·高为之着迷，于是，他带着无限的创作热情来到了这里。
感兴趣的朋友，可以在游客中心拿一份梵·高之路的徒步路线介绍，包括了梵·高生活和创作过的各个景点，其中著名的《星空下的咖啡馆》取景地便是下图中的小咖啡馆，位于老城中心，亦是梵·高之路中重要的一站。
黄色棚子的咖啡馆，现今已成为人们游览的主要景点，但却不是吃饭和喝咖啡的好选择。
除去城中的一些景点，位于城郊河边的一座断桥则是梵·高另一副名画《阿尔勒的吊桥》的取景地。需自驾前往，边上可停车。

从阿尔勒驱车进入普罗旺斯地区，阳光依然明媚，我们的目的地是那藏在群山里的一家小葡萄酒庄园，同时这里还是个小型度假村。事先在booking上预定的酒庄住宿，远远地看到这几个有点令人看不懂的雕塑就知道我们的目的地到了。
酒庄名为：Domaine des Peyre，坐落在Robion en Luberon小镇外围，坐拥一整片连绵的葡萄园。经过一个转盘走入一条小道，前方就是酒庄的入口处，1620年至今，可见这座酒庄的悠久历史。
庄园内的小院子，这里便是我们接下来两晚的住宿场所，院子里很安静，仅在晚上和清晨时分，住客们纷纷端着自己做好的食物在院子里或者阳台上吃饭晒太阳聊天。
为了不破坏小两口的私密生活，此次共定了一套两室一厅的房子和一个一室公寓，图为我们住的公寓，小窗台正对着后山的葡萄园，向外望去，一片郁郁葱葱。
房间的装修很简单，却又不失典雅，在这夏日的热浪中，躲在房间里反而显得很清凉。
每套公寓都有一个厨房，因为酒庄不提供餐厅，是的，他们只卖酒……

古老的庄园庭院，很适合小姑娘们拍照，佳能85的锐度是真牛逼……
酒庄里的小泳池，第二天午休后来这里泡了一会，水池非常干净，加之周边的环境优美，躺在这儿晒太阳，配上一杯小酒，还有边上的比基尼小姐姐，这是一种享受。

石头城与修道院
在吕贝龙大区有着许多精致的小镇，诸如著名的泉水城、红土城均在于此，距离我们住的酒庄最近的就是那山间的石头城。
石头城名为戈尔德，层层叠叠的矗立于这片高原的白色岩壁上，蔚为壮观。进入石头城，必然会路过小镇对面山崖上的公路，此处有一个小停车场（仅仅几个车位），一个观景平台，而这里，就是欣赏石头城全景的最美去处。
航拍石头城及其周边的田园，在吕贝龙大区，拥有无数这样的山顶小镇、遍地开花的葡萄园、还有那古老的修道院，以及动辄绵延数公里的薰衣草花田。
在观景台边上有几一片小树林、废旧的房屋，还有山坡下的小酒店，都是不错的拍照场所，看看我们的小姐姐穿上红裙子后的效果，怎一个美字了得。
黄昏下的石头城全景：
夜色中的石头城，当城镇灯光逐渐亮起的那一刻，小镇显得更加魔幻了。
第二天一早前往修道院时再度路过石头城，显然这里还是黄昏最美。
位于观景台边上的小屋，自家阳台就是一个绝佳的观景台。
经过20分钟环绕的山路，可以看见前方便是传说中的塞南克修道院了。这座彷如与世隔绝的修道院建于1148年，迄今已有数百年历史，而它最有名的地方则在于修道院周边的薰衣草，每年7、8月份，游客如潮，均是为了一览这里如明信片风光般的薰衣草景色。
由于坐落于山谷之中，此处的日照时间较短，6月底来此，薰衣草还未完全盛开，淡淡的紫色之间更多的是绿色。
最经典的明信片角度，花田不能进入，薰衣草亦还未完全开放。修道院内还有修士居住，外部可随意参观，若想进入修道院内部则需要提前预约报名参加导览游。
我们赶着日出来此，夏季的南法，日出时间约6点左右，7、8点钟，这里已经陆陆续续有不少游客来此。
后院的墙上爬满了常青藤，又是一个适合拍照的好地方。
没有看到盛开的薰衣草，略带着失望从修道院开车回酒庄，路上看到有个景点的牌子便决定顺道过来看看，这里在很多攻略中都没有介绍，是一个极具当地特色的小村庄：石屋村，法语名字为：Village des Bories
顾名思义，这个村子的建筑全是由一片片石头堆砌起来的，比石头城更加原始，也因此成为了一个很有意思的景点。进村参观需要门票（€6），日出时间来还没开门，要知道，法国人是很懒的……
在外围逛了逛，其实沿途经过的狭窄小路已然很有特色。这里的石屋均由这种条状的石灰石砌成，其历史可以追溯到青铜时代，后来一直有牧羊人居住于此，再后来就成了一片废墟了，直至20世纪70年代得以修复，成为如今一个供人探索的地方。
下午再度回到石头城，小镇似乎已经过了旅行团拥挤的时候，显得有点幽静。在镇中心广场边上约200米处有个大停车场，停好车，开始散步。
世界杯还在继续，因为开车，不能喝酒，但是叫上杯冰可乐跟当地人一块儿在酒吧里看场球还是不错滴。
石头城、红土城、泉水城是这一带最有名的几个小镇，游客很多，一家两口带上小孩的游客随处可见，拥有温暖阳光的地方，永远都是人们度假的首选。
在这里，几乎每一个小村庄都有属于自己的集市，一个说不上大的广场，几个出售当地农副产品的小摊位，例如奶酪、蜂蜜、橄榄油等，还有几个果蔬、热狗汉堡摊，组成了当地人们每周必逛的市场。
当地居民的生活很悠闲，每天都有新鲜的食物，三三两两坐在街头晒太阳、聊天，不时有游客过来，微笑的说上几句话，据说，法国人的寒暄都是从天气开始的。
镇上的小店，南法的餐厅一贯特色，人们都喜欢坐在街上吃饭喝咖啡。推荐品尝一下法国特有的可丽饼，法语：Crepe，在每个城镇中都能看到专门卖crepe的小店，便宜、好吃又地道。
石头城还是有不少当地居民的，黄昏来临，在街头走走拍拍，可以看到许多人提着刚在广场集市上买的菜慢慢散步回家。
石头城历史悠久，而人们依然保持着传统的生活方式，不知道是不是因为生活过于悠闲，小镇上的房屋显得有些老旧，甚至有不少已经没人居住的房屋和破旧的海报，但在阳光和青藤的衬托下，这里的墙壁和很多细节都成了值得记录的地方。
离开小镇时，停车场边上的一个类似庄园的地方，笔直的道路进去，看起来有点气派。
普罗旺斯山间有一片薰衣草花田
从石头城到瓦伦索尔需要横穿整个吕贝龙自然保护公园，这里的山峦起伏，因此每一个地方的日照和温度都不尽相同，这也导致了普罗旺斯各地长达一个多月的花期间隔。开过石头城，车子开始在山间小路上飞驰，阳光依旧强烈，而山路的两旁，开始闪现一片片的紫色，是的，那就是我们这些日子以来不断寻找的薰衣草花田吧。
恰巧路边有个通往田园农庄的小路，将车子往路边树林一停，也不用太担心传说中的南法砸车，毕竟这里不是瓦伦索尔一带人满为患的著名花田。或许是第一次看到绵延不绝的薰衣草，大家还有点兴奋，撑起伞开始在花丛中穿行吧。
因为此处海拔较高，薰衣草依然还未完全变紫，但也给我们带来希望，已到达瓦伦索尔的小伙伴发来贺电，听说那边的花田已经开始绽放了。
南法的路不好走，山林之间更是只有双向的车道，但贵在道路维护很好，平坦的山路在没有转弯的地方也能跑到个6、7十迈。
我们中午停车休息的地方，也许大多数人都没有听说过的小镇，这可是北斗发现的秘密地点，位于上普罗旺斯大区最西边的巴农 Banon，距离索村不远，镇子前方就是一片薰衣草花田，如若黄昏或者清晨来此，在那迷人的光线下一定能看到令人陶醉的美景，还好我们有长裙仙女小姐姐可以入镜。
小村庄巴农最著名的并非其薰衣草花田，而是当地特产的奶酪，名为巴农奶酪 Chevre de Banon，这可是一种珍贵的普罗旺斯美食，对于酷爱奶酪的法国人来说，专门来此品尝品尝奶酪也不失为一种体验。因此在小镇周边有许多草场和农庄，或许这就是用来饲养山羊的地方了吧。
在普罗旺斯的山间开车，这样的急转弯很多，两旁自然是郁郁葱葱的树林和田园，风景很美却也要时刻注意路况。
稍微往外走去，是一片更为广阔的花田，正午的阳光下，直射光导致了薰衣草看起来有点灰，所以说最佳的薰衣草拍摄时间是每天的清晨和黄昏时分。
无论阳光如何，小姐姐们对于鲜花的诱惑是无法抵挡的，不管怎样都要在阳光下摆拍几组才肯罢休。
一边是花田，一边是当地人居住的小屋，这就是南法简单的农场。
在这里，除了薰衣草，还有一种常见的鲜花，不知道叫什么名字，后来问了酒店的服务员也表示不懂……好吧，小哥哥你确定自己是南法人？总之，经常可以在薰衣草花田边上看到同样大片的该种花海。
南法最后的假日，漫步在薰衣草的芬芳里
在普罗旺斯的最后两天，贯彻了我一向的旅游方式，度假享受与摄影相结合，绝不走马观花，特地在瓦伦索尔周边的镇上酒店住了两晚，就为了守着那日出日落时刻最美的光线，先来一张小姐姐在酒店的自拍镇楼。
整个普罗旺斯最著名，也是最多人聚集的薰衣草花田，因其拥有连绵不绝，高低起伏的线条，加上远方的蓝天白云，还有不时出现的孤树，每一个元素都使得这里成为最佳的摄影基地，再来几张两天来拍摄的薰衣草。
后面攻略里有提到几个拍摄点，包括几个最佳机位的GPS定位，大家可以参考，事实上在D6、D8两条道路上都有很多花田，提前个1小时去寻找机位还是有意义的，带上发现美的眼睛去探索自己心中最美的花田吧。

在这里没有什么特别的攻略，两天的时光基本就是凌晨起床，开车半小时找花田拍照，8点多回到小镇在小咖啡馆吃早餐，然后回酒店休息游泳，酒店边上还有个温泉浴场，晚饭时间找家餐厅看世界杯，然后赶在日落前到花田继续拍照……

【攻略篇】蔚蓝海岸&普罗旺斯景点推荐
南法最值得玩的几个点，分别是蔚蓝海岸沿海的城市和海滩、南法山间小镇、普罗旺斯的薰衣草。
== 蔚蓝海岸 ==
1. 尼斯 Nice：南法最具浪漫气息的大都市，滨海大道的美景令人沉迷，全年沐浴着地中海明媚的阳光，使得尼斯成为众多情侣、家庭的首要度假选择。老城的餐厅、夜晚海边的小酒吧、沙滩、城堡山，这些都是来尼斯值得游玩的地方。
2. 埃兹小镇 Eze：法国南部拥有绝美风景的小镇之一，紧靠着地中海，坐落在山顶之上，前往摩洛哥途中可以经过游玩，山上城堡里还有著名的米其林餐厅，可以体验。
3. 摩纳哥 Monaco：依山而建的高楼大厦、奢华的大赌场、满是游艇的海港，这个全球第二小的国家同样也是一座繁华的大都市。市区住宿极贵，适合从尼斯出发的一日游。
4. 马赛 Marseille：被誉为“欧洲文化之都”的马赛，是一个充满活力的地方，海港的美景和正宗的马赛鱼汤足以吸引每个游客来此驻足。杰出的博物馆也是值得花点时间参观的，还有主座教堂山上的美景同样不容错过，自驾游前往普罗旺斯，首推从此进入。如果有时间，还可以去马赛边上的蔚蓝海岸国家公园。
5. 格拉斯 Grasse：香水小镇，南法主要的几个香精工厂都在这里，适合度假并且欣赏香水的制作与历史。这里可以体验香水制作，北斗还曾经在这里做了一瓶不可名状的味道的香水……
6. 戛纳 Cannes：个人不是非常推荐这里，城市比较拥挤，游客众多，主要是为了来看看电影节的红地毯，这里的停车费和餐饮住宿都很贵，如果时间不多，可以留着时间去边上的几个小镇游玩。
7. 此外还有土伦、芒通等城市，没有去过，属于较为冷门的地方，说不定可以在那找到惊喜。还推荐位于格拉斯附近的圣保罗德旺斯，也是山顶的城堡小镇，航拍效果很壮观。
城市之间车程：
尼斯 → 摩纳哥 40min
尼斯 → 戛纳 50min
尼斯 → 马赛 2h30min
尼斯 → 格拉斯 50min

租车流程：
1. 网页或APP选好时间和取还车地点，通过筛选功能选择手动/自动、车型、座位数等。
2. 货比几家，选择心仪的车型下单，下单时注意租车条款，包括租该车的最低持有驾照年限、额外驾驶人费用、里程限制、是否满油取还。
3. 填写好自己的个人信息，提交订单后驾驶员信息不可更改，只能取消订单重新预定。
几个tips：
1. 一定要买保险！一定要买保险！一定要买保险！啥也不多说了，直接从“超级补充全险”起步，至少起赔额为0的那种，事先买好，小擦小碰的都能迅速理赔。
2. 如果租车费用不是很高，建议直接全额付款，省得到租车公司还要交钱，避免交钱时多收各种七七八八的额外费用。
3. 欧洲大部分人喜欢开手动挡，想租自动挡尽量提早，且自动挡的费用会相对更高一些，晚定就没有了。
== 驾照认证 ==
一般来讲中国人去法国租车不能直接使用中国大陆驾照，需要办个法语或英语公证比较好，虽然在取车的时候店员完全没有查看，但是保险起见建议办理一个。
== 取还车注意事项 ==
1. 取车的时候，一定一定要注意检查汽车的情况，给你的取车单子上会标明车辆现有状态，比如划痕等，需要一一核对（示意图，可以看得出来）；
2. 要确认油箱是否满油；
3. 确认门窗、雨刮器等是否正常，还有空调是否正常运行（我看到有人说当时租了一辆空调坏的，于是大夏天的开了一路没有空调的）；
4. 注意查看车辆行驶证；
5. 确认还车地点，最后出发的时候在地图上保存车库的位置，免得出去好几天回来不认路了。
还车的时候没什么注意的，记得加满油就行。有刮擦的话一定要跟验车人员确认好，一般小的划痕等可以略过，明显碰撞就走保险吧，需要自己先付款，信用卡冻结一笔修车费用，回来在租租车网上申请保险理赔即可。
== 法国交规和注意事项 ==
1. 法国的交通规则和国内基本一样，左舵驾驶，左道超车，遵守行人优先原则，无论何时记得要礼让行人。
2. 遇到红绿灯和转盘必须停车观察，红灯情况下也不能直接右转。
3. 记得超车千万别右道超车，这样很可能被人投诉而面临罚款，当地人开车都很快，但是都遵守左边快车超车道，右边慢车道的规则，往往从左道超车后也会迅速回到右道上，而不会长时间占用超车道。
4. 限速：高速110-130，其它60-70，小镇和城市小路限速30，尽量不要随便超速，不过法国人开车飞快，如果他们想超车会打左转向灯，开的慢的小朋友让让就好了。
5. 法国很多地方没有红绿灯，都是转盘，全凭自觉，如果环岛内有车的情况下让环岛内车先行。
6. 远光灯：晚上没路灯的地方才开，对向来车记得马上关远光，法国人很自觉，基本看到对向车辆都会立刻关远光灯。
7. 加油：绝大多数是自助的，用带芯片的信用卡付款，确认清楚自己的车子是柴油还是汽油，别加错了。自助加油便宜，人工加油站费用一般比较高。
8. 停车：酒店基本都带停车位，南法小镇里面很难停车，其它山里和田园则有很多免费停车场、路边停车区等。注意：看薰衣草不要随便把车停路边，如果车里有包等东西，可能会被砸车！切记！
