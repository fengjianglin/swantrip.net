第一次冲绳五天四晚的行程安排

来过日本已经五六回了，冲绳县是第一次。爱拉的行程安排的是非常适合第一次来冲绳玩的，基本上来四五天的行程的话，需要打卡的景点都安排了，需要品尝的美食也都尽可能的甩开了腮帮子搓了。当然了因为是冬季的原因，不能尽情的玩海有点遗憾，这边的海水真的挺美的。如果夏季的话带宝宝海边耍耍再吃点海鲜，还是挺美的事。行程共计5天（2019.2.17周日~2019.2.21周四）
第一天：北京T3直飞冲绳那霸（住那霸市区）
第二天：那霸--冲绳美丽海水族馆--福木林--荞面街--今归仁城遗址--古宇利大桥--国际通附近逛街（全天包车）
第三天：万座毛--琉球村--海人食堂--残波岬--美国村--DFS（全天包车）
第四天：首里城--冲绳奥特莱斯购物中心--国际通逛街吃饭
第五天：退房出发去第一牧志公社市场--返回北京

DAY1:这三更半夜的是要闹奶出儿？

咱们接着上面接着聊啊，匆匆忙忙的起床，找出两个家里最大的箱子，整理了几身衣服，和一些基本的洗漱用品。我寻思着却啥都不用担心，大不了到了再去药妆店补货。反正也是要去买东西的。相机、护照、身份证，基本的都带了，没日元、没WiFi、没辙。这年头，有手机有卡有人民币，齐活就它了。
幸好平时出去的多，收拾行李这件事对爱拉来说已经是轻车熟路了，拎上箱子带上娃在跨上老公，爱拉的冲绳之旅就这么麻麻利儿利儿的开始啦~
从半夜两点多睁开眼，忙忙叨叨的总算是顺利屁股落了座，这才算是踏实了。原本想在T3换点日元来着，一打听，换一次手续费就要60人民币，想了一下还是决定先不换了。经过爱拉血泪教训，还是提前在国内银行换够了最划算。如果像我们这样临时抱佛脚的，机场换钱虽然手续费高，但如果算计好了每天用多少钱一共多少天，一次性换够了钱，交一次60元人民币的手续费也是划算的。就怕爱拉这种，去之前啥也没准备，在日本提款机多次提款，每次提款都根据提款的额度收取手续费，这样就贵了。
「重要提示」
以下是爱拉血泪教训的经验：冲绳挺多地方都是可以刷微信和支付宝的，刷微信和支付宝是实时汇率的，没有任何手续费，这个最值。爱拉因为现金不多，所以能刷微信和支付宝的地方都先用这个。按照爱拉的花费推算，一天大致需要8000日元的现金（主要是坐公交、自助贩卖机买水、打车、部分商铺、景点门票还有一些特别小的餐厅等）这样按照出行天数计算一下，实在没辙在机场一次性换也比来日本提款机提款划算。
没想到一点儿出发的心理准备都没有的贝壳儿小朋友，大半夜的被叫起床真的是干净利落脆。时间过得真快，刚刚翻看了贝壳儿小时候的一些视频，感觉昨天她仿佛还是个咿呀学语的小宝宝，转眼间已经出落成大姑凉了呢。关于入境卡，爱拉一般都会一上飞机就管空姐要，然后在飞机上就填好了。入境卡是一人一张的，我、大壳儿和贝壳儿一人需要填写一张。我们乘坐的是国航，所以有提供中文版的入境卡。上面有一条是要填写在日本的地址和电话，所以一定要记下你住的的酒店的地址和电话。
入境卡，飞机上的数量不多，所以想要在飞机上提前填好就要尽早管空姐要。如果飞机上的没有了也没关系，抵达日本机场后在机场领取填写也可以。
然后这个行李申报单，是一家人填一张就可以。就全在“否”上打钩就可以啦。
北京直飞冲绳的时间大约是4小时左右，飞机12：30灰常准点的抵达冲绳那霸机场。那霸机场不大，抵达就在一层。从那霸机场到市区的交通可以选择巴士，还有针对游客的路线巴士，往返于机场与酒店、海滩之间。从03年开始单轨电车开通后，从机场前往那覇市内的交通更方便了。
但是，爱拉选择了最不用动脑子的方式，就是出门一招手哈哈。
出机场找了一个提款机，换了15000日元，后面打车必须要花现金。
出来有很多出租车，打车非常方便。冲绳岛上的出租车起步价一般是550日元左右吧。我们住的酒店在那霸市区，距离那霸国际通步行大约30分钟，略远。距离机场大约打车5、6分钟吧，花费车费1110日元。
这里就是我们订的酒店啦，因为考虑呆的天数不多又是第一次来哪都想去，所以酒店嘛肯定就是早走晚归的睡个觉，所以订了这家。住下来感觉还是太小了，不太推荐。而且位置也一般般，去国际通略远，走着吧一次两次还行，天天走就觉得远了。从酒店打车去国际通也要超过起步价的，大约是900日元吧。这家酒店唯一的好处就是便宜，爱拉一家住了四晚一共才花了1543人民币。
这里是酒店的前台，每天的下午15：00才可以入住。不过可以提前办理手续，然后把行李寄存在酒店前台就可以出去逛啦。放下行李，我们一家决定溜达去国际通，反正下午也没啥安排。因为刚来，温度又舒服，类似北京的春天的感觉，有点小风不冷不热的，所以一路溜达一路玩也不觉得累。
好久没开过洋荤的贝壳儿，终于找到理由跟我要饮料喝啦。
一路溜达过来，看到琉球新报大厦这边正在办什么市集，索性漫无目的凑过去看看。是一些卖当地农副产品的市集。居然还来了一头大公牛。
贝壳儿挑了一瓶超级有营养的饮品，带汽的胡萝卜汁哈哈。我发现贝壳儿特喜欢吃有营养的实物，一般小孩子不喜欢的鸡蛋，贝壳儿小朋友要是没人拦着一顿能吃三四个。蔬菜也是特别喜欢胡萝卜。
出来以后过条马路，接着往国际通方向溜达。不远有一家Blue Seal，这个冰淇淋店的广告语是诞生于美国，成长在冲绳。所以来冲绳的话可以试一下。冰淇淋的口味品种繁多，大约有二十多种可以选择。里面还有很多冲绳特产的口味，比如芋头、紫薯、甘蔗啥的，这家冰淇淋店几乎遍布冲绳那霸，分店很多啦。
贝壳儿选的是这种外面卷薄饼的，里面是冰淇淋和奶油。其实我觉得普通的甜筒就挺好吃的。
出来不远是一家大商场，到这这里就算到了国际通的一头了，再往里走就开始进入国际通了。我们觉得有点走累了，也饿了，所以进来这里面逛下，顺便找点饭辙。
上面是商场的楼层安排，我们楼上迅速转了一圈。决定去地下一层逛下好吃哒~地下一层几乎都是卖食物的摊位，琳琅满目。
还有担担面和炸酱面
最后贝壳儿选中了这家店，我就知道有贝壳儿在她一定会选寿司。贝壳儿是个严重寿司控，就因为她的这个习惯，害的爱拉现在看到寿司就饱了。
我们选了这种榻榻米的座位，这家店不允许吃店外的任何食物，包括饮料。大壳儿点了一份红豚井，油炸的猪扒外面还裹着鸡蛋，配上一碗味增汤。这份套餐是1296日元。爱拉觉得挺好吃哒~
看到这个就知道是贝壳儿的了，严重寿司爱好者。她的套餐1620日元，量也算还行吧，反正贝壳儿就算是吃饱了吧。
这个是爱拉点的定食，好像是1728日元。里面有一小份的荞麦面，还有几块天妇罗和一小碗海鲜井，还搭配了蒸蛋羹和汤，品种相对丰富一些。
吃饱喝足已经下午16：00多了，因为早上起床比较早所以感觉有点累了，所以决定溜达回去酒店办理入住啦。
这就是我们的小房间啦，好像是6楼，有电梯。房间不大，挺干净的。这个房间最大的好处就是床大，三个人睡完全无压力。老板说每天早上离开酒店的时候，需要把毛巾筐和垃圾桶放到房间门口，晚上回来干净的毛巾和垃圾桶就会放在房间门口，自己拿回来就行。房间提供洗漱用品、吹风机、热水壶啥的。酒店不提供早餐，一楼有个开放式的厨房，咖啡是免费的。
卫生间超级小，特别不喜欢这个卫生间。不过一想到价格，算了忍了。
晚上爱拉给贝壳儿洗澡，贝壳儿跟着跑了一天洗完澡很快就睡着了。大壳儿自己出去在酒店附近的一家全家买回来点吃的，背着贝壳儿吃独食哈哈。

DAY2:超级充实的一天，包车向北

爱拉在冲绳的包车行程一共安排了两天，第一天：美丽海水族馆-福木林-荞面街-今仁归城迹-古宇利大桥
第二天：万座毛-琉球村-海人食堂-残波岬-美国村
我们是提前在某宝上订的包车，每天10小时的。早上九点司机会准时在酒店门口等着，晚上19：00回到那霸市区就可以，超时的话应该是每小时加收4000日元。在日本是不允许中国驾照自驾的，如果打算自驾的话可以去香港考一个驾照应该就没问题了。如果嫌麻烦包车是最省心省事的，有司机开车。
早上八点半收拾好自己，出门找饭吃。包车的这两天我们的早餐都是在酒店不远的全家解决的。包车的师傅是允许在车上吃东西的，爱拉不知道是不是包车都允许吃东西，也许是每个司机师傅要求不同吧，如果有需要在车上吃东西的话，建议提前问一下司机。
从酒店出来穿过一条街道就是全家，步行大约3分钟吧。
贝壳儿的早餐想都不用想一定是寿司和章鱼小丸子，每天如此她也不会觉得烦哈哈
买完吃的，拎着回酒店。时间还早本来计划在酒店吃完东西再出发的，没想到回来的时候发现车子已经在酒店门口等着了。司机和我们说可以在车上吃东西，这简直太棒了，这样就可以不用等吃完东西，马上就能出发了。司机的车上超级贴心，后座上还提供纸巾。从那霸市区到今天的第一站美丽海水族馆大约90公里左右，开车不走高速需要大约2.5小时，走高速需要大约1.5小时就可以。从那霸到冲绳北部有一条高速，这也是冲绳岛上的唯一一条高速公路。选择走高速的话高速费是由我们自己承担的。早上九点出发走高速，开车大约一个小时左右抵达许田收费站，花费高速费980日元。
出了收费站其实也就算是冲绳道的终点了。这边有一个许田许田服务区，在这里可以购买各个景点的门票，会比在景点门口购买有大约百分之15的优惠吧。
进入许田服务区里面的收银台上面就有关于各景点门票的价目表，需要注意的是在这里购买景点门票必须使用现金，不支持微信支付宝和刷卡。
买完门票，里面是一个小超市，有冲绳当地的一些特产和纪念品啥的，感觉价格还可以。
这个爱拉提前在冲绳的旅游书上看到过，叫做海葡萄，是一种海藻。这里可以买到，不过其实也不用特意买，在冲绳吃饭很多配菜都是这个。
因为出门太着急了，所以没给贝壳儿带袜子，刚好看到了买袜子了，赶紧给闺女来一双。
贝壳儿对冲绳限定的扭蛋机特别感兴趣，爱拉觉得这个限定的扭蛋机里的玩具没有一个好看的哈哈
从超市出来，门口有一家冰淇淋店，贝壳儿在冲绳的几天，每天都要吃冰淇淋。以至于回到北京还一直跟我念叨要再去日本，理由居然是因为能吃冰淇淋。这边虽然是冬末春初，按咱北京的天气还穿羽绒服呢，但是这里不用，虽然还没暖和到能下海，但体感温度也绝对相当于咱北京5月底的样子了。穿裙子吃冰淇淋绝对不过分。
这些是爱拉在徐田服务区买的门票，因为现金有限，所以就没都买。
离开许田收费站，前面就没有高速了，从这里到美丽海水族馆大约开车还需要45分钟吧。上午11：30抵达美丽海水族馆，司机停好车一路把我们带到大门口，才离开的。
美丽海水族馆爱拉感觉并不大，也许是逛过的水族馆太多了，没觉得这里有什么特别之处，当然了除了黑潮之海里面的几条鲸鲨确实挺震撼的，别的也没啥特别的。
水族馆官网：https://churaumi.okinawa

石台阶上面的这个门为瑞泉门，取赞美龙头里流出的水为吉祥之水的寓意。瑞泉门始建于1470年年前后，曾在冲绳战役中被烧毁，于1992年复原。

首里城
建议游览时间：1小时
门票大人：820日元
小童：310日元（6岁以下免费）
首里城是位于琉球群岛的琉球岛南部首里市以东一座琉球式城堡，建造于13世纪末至14世纪初。从15世纪至19世纪是琉球国的都城所在地和王宫，首里城是琉球王国的政治和权力的中心。首里城曾经四度被毁，最后一次于1992年重建。
首里城是琉球群岛的重要古迹，是以战后残留的原型为样板复制的唐朝风格建筑。首里城是琉球王国的宫殿建筑，是当时国王处理国家事务、接见使节和举行重要庆典的地方，它融合中国、日本及琉球岛的建筑特色。如今的古堡有北宫、南宫、首里门及多座城门。
2000年12月首里城被认定为日本第11个世界文化遗产。
贝壳儿非常认真的在一个个的积攒印章。
这里是首里城最重要的主殿，中间的大广场是皇宫里举行重要节日和活动的场所。

签证问题

日本的签证在之前的游记里爱拉也写过很多次了，这次和以往不同的是爱拉一家办理了三年多次签证。从某宝上搜一下日本三年多次签证就会出来一扒拉。大壳儿选的是中青旅办的，三个人一共花费997元。需要啥资料客服都会告诉你的，不用担心。只是开启三年多次签证需要到指定的几个县（冲绳县、岩手、宫城县、福岛县）第一晚的泊宿证明。泊宿证明酒店前台就能开。回国后寄回给旅行社就可以啦。

冲绳交通

Yui-Rail轻轨列车冲绳是一个木有地铁的县，整座城市里只有一条贯穿机场和首里城的Yui-Rail
轻轨列车，是东西走向的，从首里城到那霸空港。票价一般是大人230-330日元左右，孩子120日元左右，具体价位要根据距离计算，乘坐的远一些就会划算一点。关于孩子乘坐Yui-Rail轻轨列车购票的标准是按照年龄分的，每个大人可以免费携带两名不满6岁的儿童，小学生（6~11岁）半价，12岁以上就和成人一样为全价。乘坐Yui-Rail列车是一个挺不错的体验，这趟车主要是从西到东横穿冲绳那霸市区的车子，而且是高高的架在半空中的，所以乘坐体验特别有意思，可以俯瞰那霸市区。
单次乘坐的话在站内买票就行，如果乘坐次数多的话建议购买一日券或是两日券。一日券价格成人800日元/人，儿童400日元/人；二日券价格成人1400日元/人，儿童700日元/人。自购买后的24小时、48小时内有效，可以无限次乘坐。
出租车
冲绳出租车起步价为550日元左右，我们出机场打车去酒店，5公里路程花费：1110日元。不算便宜的，当然了如果人多，比如三个人以上就和公交车也差不太多了。冲绳的出租车挺好叫的，机场出来就有。大街上基本也都是可以招手上车的。当然爱拉也遇到过像是国内那种预约等客人的出租车，但是还是少数。基本都是招手就停的。
冲绳的出租车司机基本都是老人，个别的会一些英文，大部分都不会。所以可以给司机看地图导航或是使用翻译软件，都挺好用的。稍后爱拉推荐两款好用的手机软件。
公交车
这次爱拉就体验了一次公交车，好久没去日本了上车都快忘了怎么交钱了。分享下乘坐公交车流程：1、上车领取一张整理券（这张券上会有一个数字，表示你上车站的代码）2、看车子前面的屏幕，整理券上的数字对应的下面的钱数，下车时你整理券上数字对应多少钱就交多少就可以。3、下车时把整理券和对应的钱放入交钱的机器里（刚刚领整理券的旁边是一台收钱的机器）大票也不怕完全自动找零很方便。

在冲绳，人们说琉球语。琉球语是琉球人（包括奄美群岛及冲绳群岛）的语言，又被称为冲绳语。早期的琉球语接近闽浙方言，但是随着后来日本的入侵和殖民，日语侵入了当地的琉球语。琉球语现在被学者普遍认为是和日语不同的语言，但在日本国内则一般被认为是日语的方言。在冲绳，正式场合一般使用日语，而在民间则有些人使用带有琉球语口音的日语--冲绳日语。
