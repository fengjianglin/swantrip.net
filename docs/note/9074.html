去了宏村，一定要去西递 一个有故事的村庄

游记 五一 人文游 古镇 摄影 自助游 2019/04/10

概述

安徽的西递和宏村列入世界遗产。世界遗产评定机构的评语说，西递，宏村这两座传统村落保持了惊人的乡村住宅区域风貌，而这种风貌在过去的岁月里已大量消失和改变。其街道形式，建筑和装饰以及房屋与广泛的水系相结合，是个独特的遗存典范。我们游览宏村后去了西递，住了两夜，走走看看，和当地人聊天，很有收获。
西递有“桃花源里人家”之称。和宏村相比，西递古建筑更多，幽静，文化气息很浓，游客相对较少。村里成片灰白墙面，黑色马头飞檐民居。远处的青山，围着村子的小河，淳朴的村民，挂着红灯笼，贴着红对联的客栈一个接一个，街巷里幽静，古朴。
西递古老的徽州特色的商宅和官宅更多，当年徽商或官员老宅大门口有牌匾，标识清楚，是参观的重点。古老的宅子，当年奢华或风光已经不复存在，但是历史的印迹，文化的传承依然展现给后人。建筑的格局，一副对联，一样老物件等都在述说着当年主人或辉煌商业发迹，或沧桑曲折经历，或对子孙循循教导，故事多多。
进入西递景区，有导游免费带领参观，大概一个多小时，然后可以自己在村里走走看看，拍照。导游讲的很快，有兴趣的，没有听过瘾的可以留等待下一趟讲解。每个景点，都有扫二维码的，关注后可以看简介。
进入西递景区门票104元/人，老年人和学生半票。住宿在西递村里客栈的游客也要买票的。
从屯溪去西递交通很方便。屯溪汽车站有空调对号大巴开往宏村，中间停靠西递。屯溪到西递行驶一个小时。歙县也有公交车到西递，歙县可以参观徽州古城。屯溪汽车站有自助售票机，快捷。窗口买票排大队。
 

胡文光刺史牌楼

当我们接近西递村时，被眼前这座雄伟挺拔，烟熏色，满目沧桑，造型大气堂皇的牌楼吸引。蓝天白云，远处青山郁郁葱葱，田园里农夫耕作，水池边灰白墙面，黑色马头飞檐民居连绵，红色对联和灯笼点缀，令人感觉恬静，平和，安详，韵味十足。西递被誉为桃花园里人家，果不名不虚传。
 
胡文光西递村人，明代嘉靖年成为举人，一直在朝廷做官，后来当了大官。胡文广开挖水渠，发展农业，注重教育，当地百姓富裕兴旺。后来皇帝恩准西递老百姓为胡文广建了这样一座功德牌坊。这个村曾经建了13座大大小小的牌坊，大多是表达忠孝贞洁的，文革中除了这一座全都被毁坏了。当年胡文光刺史牌楼当时写满了革命口号，涂满了红色，才勉强被保留下来了。现在胡文光刺史牌楼默默地站立在碧水池塘边，注视着过往的西递乡亲和游客，叙述着西递古老文明故事，给前来参观的游客留下一幅轻柔又沉重的水墨画般的美景。游客的参观也从这里开始。胡文光刺史牌楼在村口，再往里走才到验票处。
细看牌楼有四根方石柱，五层楼阁，上上下下布满石刻。两只狮子守卫两侧，浮雕双龙盘踞，牌楼面板上刻满了花卉，动物，上面依稀还可见荆蕃首相和登嘉靖已卯科朝列大夫胡文光字样。上面还留有红漆，是文革留下的印记。后来听了讲解员介绍，对这座特别的牌楼更加怀有敬重感。
 
胡文光刺史牌楼附近的小广场，沿街有很多小吃店，舌尖上中国介绍的毛豆腐，臭鳜鱼，面条，盖饭全有。
 

膺福堂

膺福堂是西递景区的主要景点之一。 1664年修建，已有三百多年历史。房主胡汝川曾经任职后部尚书，二品官。建筑高大，八字楼门，三层飞檐，足以说明房主的地位。导游说，古代百姓宅院飞檐超过五层是犯上的。墙面布满了花鸟雕刻，寓意吉祥如意。现在开放的是庭院和堂屋。门和窗框全部是木格镶以木雕，木雕玲珑有趣，富有情趣。
导游特别介绍了膺福堂的正屋门槛，门槛高低是光第标志。古时候，家里婚嫁喜事和达官高人来访才能跨门槛走正门，通常都是走边门的。
堂屋正中八仙桌上摆着镜子和花瓶，这是古代有身份人家堂屋装饰的标配，寓意平平静静。那个筒状瓷器是放置官帽的，客人来访如果没有看见官帽放置在那里，说明主人不在家，不宜久留。
堂屋两侧高悬两块红底金字牌子，已经很旧了，询问主人后得知是上朝时使用的，很珍贵。
正屋里陈列着几件老家具和用品，其中一个轿子很小，主人说那时街巷很小，这是一座平时出行使用的轿子复制品。
现在住在膺福堂的是胡姓70岁老先生，是第九代传人，桌上供着他的妈妈遗像。老先生说儿子是老师，很想让他传承这份文化传统。这位老先生很健谈，讲起家谱历史如数家珍。膺福堂里现在经营古玩和老物件。老先生很实在，再三讲很多不是真品，你喜欢，有眼缘就买。
厢房，后面的屋子及楼上现在不对外开放。大门口挂着五好家庭和十星级文明户的标牌。

敬爱堂

敬爱堂大门口有胡氏宗祠大匾，敬爱堂的牌匾在里面。这座建筑原本是胡氏大宅，建于明朝1600年。后来胡家三兄弟携资归故里，扩建为胡氏宗祠并立巨匾，取名敬爱堂，表达了兄弟间互敬互爱，家人和所有族人之间和睦相处的意愿。
敬爱堂应该是西递村里最大的古建筑，但是不奢华，粗旷肃穆。导游介绍建筑前后深62米，宽30米，总面积有1800平米。一进门迎面是两侧六米多高的整块青石的方形柱子。粗大的横梁是樟香木的，不会长蜘蛛。粗大的圆形柱子是银杏树木材，柱子下方垫着荷花型石墩，防潮防水。
敬爱堂前后三厅，最里面的叫享厅，供奉祖宗。上方挂着大匾，写着“百代蒸尝”四个大字。导游说，这是告示后人，每年腊月二十四和七月下旬都要来祭祖，代代相传。
 
中间是祭祀厅，胡氏家族讨论重要事项，惩戒不良行为和婚嫁在这里进行。最外面是礼仪堂，接人待客，不同身份享受不同待遇。
祖宗牌位对面，上方有朱熹墨迹，一个两米见方的孝字，字画合一，很吸引人。在导游的指引下，抬头望去，从右方看是一个笑脸，拱手下跪，意思是尽孝可以抬头做人，是争气儿孙。另一侧看起来像一个猴，挥拳踢脚，意味不孝只是畜生。这个有趣的孝字，西递村户户皆知。客栈的主人专门介绍我们去看看这个孝字，很多摊位都买这个孝字。
 
敬爱堂里墙上贴着巨幅墨宝，忠孝节廉。两侧靠墙各有一个半圆形的桌子，看过其他老宅，知道通常是主任或长辈在家，或年节，两个半圆形桌子会拼在一起，意味团圆。
一位女士是敬爱堂工作人员，胡氏三十代传人，她说起敬爱堂如数家珍，不时打开手机让我们看储存的家族信息。她说，当年西递村商贾，文人和在朝廷供职的住家集聚，敬爱堂是胡氏家族当年地位显赫的象征。胡氏家族倡导的互敬互爱，和谐相处的道德风尚滋养了一代又一代人。我们去的时候，敬爱堂正在展出黟县四季美景摄影展作品。

旷古斋

旷古斋建筑外观气势宏伟，二层楼，左右三开间。导游特别介绍了旷古斋的门楼，门罩。高高的门楼，两层飞檐，外墙呈现花鸟雕花。徽派建筑的门楼门罩是权势地位的象征。门楼上方有富贵两个字，屋顶一角里有狮子雕刻，表达房子的主人招财进宝和以商入仕，以仕保商的勃勃雄心。
厅堂八仙桌上方挂着中堂画，两侧是镜子和瓷瓶，中间是座钟，寓意终身平平静静，子孙万代保平安。圆筒瓷罐是放置官帽的，也是古代女子出嫁陪嫁品。
中堂画两侧是木刻楹，主人看我们有兴趣，阴阳顿挫地诵读一遍，朗朗上口，是家训，也是日积月累的人生真谛。
世事让三分天宽地阔 心田存一点子种孙耕
这幅对联在履福堂也看到过，看来很被认可，是功成业就人家的自勉和对子孙的教诲。
黑底金字的是一个改字联，孝弟传家根本 诗书经世文章
很有意思的是章字下面的早字出头，表达的是唯有读书才能早出人头地。
西递村里家家户户门口贴对联，增添了西递村的文化气息。对联是中华文化的独特语言，文字对仗，平仄相拗，内涵丰富，充满哲理。旅行走过很多地方，第一次看到那么多的对联，西递村可谓是对联博物馆。
在一些景点出售西递村对联集。翻翻看看，记下几条：
以八千岁为春，之九万里而南 
书是良田，传世莫嫌厚产 仁为安宅，居家何必构高堂
事业从五伦做起 文章本六经得来
读书好，营商好，效好便好 创业难，守成难，知难不难

追慕堂

追慕堂建于1794年，是胡贯三为纪念祖父和父亲崇高尚义和激励后代而建。1992年西递村投资重建了门楼。在西递和敬爱堂工作人员聊天，她告诉我们，追溯胡氏家族，原本胡氏后裔住在现在的江西婺源一带，先祖是李世民，她打开手机让我们看历史学家的论述。
在追慕堂里供着李世民塑像。两侧墙上挂着字屏，是李世民手迹名言。
以铜为镜，可以正衣冠；以古为镜，可以知兴替；以人为镜，可以明得失。
去奢省费，轻徭薄赋，选用廉吏，使民衣食有余，为君之道，必须先存百姓。
追慕堂三进开间，层层抬高，有威严肃穆之势。祠堂青石板铺地，巨石立柱，廊子上有李世民英勇作战故事图。屏风上记载着胡氏家族历史。天井敞亮，沿房檐层层挑高。古时候楼宅森严，天井通风进光，还可以接盛雨水。追慕堂内容丰富，如果对胡氏家族有兴趣，看一个小时都不多。
 祠堂文化是中华文明一部分，祠堂寄托了归属感，离家背井的徽商，徽官在祠堂寻找亲情友情，传承忠孝节廉的精神，维系着家族。当年婚庆嫁娶，种族商议重大事项，惩戒违反族规道义都在祠堂进行。西递村胡氏祠堂很完整，保护的不错。

履福堂

履福堂在一个很窄的小巷里，建于清康熙年间。看起来这个宅子屋内虽然有点破旧，但是书卷气很浓，是书香门第之宅。导游介绍，主人胡积堂是收藏家和鉴藏家，曾藏有千件字画，被收入中国美术名人大字典，至今北京故宫里保存着胡吉堂的藏品。
前厅正中挂着松鹤中堂画，上方悬挂“履福堂”匾额，两侧和东西列柱上木楹联，写着：
孝梯传家根本，诗书经世文章
世事让三分天宽地阔，心田存一点子种孙耕
第一等好事只是读书，几百年人家无非积善
快乐每从辛苦得，便宜多自吃亏来
八仙桌上摆着古代装饰的标配，镜子，花瓶，中间是自鸣中，寓意平平静静，终身平安。两侧的筒形瓷罐是男人放置帽子的，称为帽罐。
后厅从房顶吊着一把木质古扇，上面刻着清风徐来四个古隶字，旁边拉动绳子，板扇轻轻来回摆动。导游说古时候乡村有很多降温防暑的方法，除了这种大型扇子，还有天井里大缸盛水，墙边小溪流水等。
我们和现在的主人聊天，他指着挂起的木框雕刻的窗框，门框老物件，介绍说以前门框上刻有完整孝义的故事，现在不完整了。我们饶有兴趣的问起网上介绍这家的原主人是收藏家，有一幅猫戏蝶的画非常有名。主人笑着说，几十年前，家里要用钱，母亲几百元钱卖给人家了。猫戏蝶图的收藏者后来还来过。墙上有一幅谭词同书写的琴书两字，询问是否真迹，老先生没有作答。
现在的主人持家并卖古玩和工艺品，介绍商品很客气。他拿着一把铜质琵琶型老式锁头介绍，这是一款手工制作，独一无二的的工艺品。上面有龙凤造型，琵琶是琴，这把锁有忠贞爱情之意。

瑞玉庭

走进瑞玉庭，大红灯高挂，导游说，瑞玉庭建于1853年，原主人是徽商世家，曾经是当地六大首富之一。看过了几个景点，老宅的结构和装饰，一般能辨别出是商人还是还是文豪。瑞玉庭门罩上有元宝形富贵二字，呈现了宅主的追求。门罩上依稀可见花草动物，生活场景的雕刻，丰衣足食，喜气洋洋。庭院里花繁木盛，墙上很多诗词，格言和小画，处处体现主人经商门道和为人处世的伦理。
有一幅对联被称为是西递第一联，
快乐每从辛苦得 便宜多自亏吃来
这是一副错字联，“快乐每从辛苦得，便宜多自吃亏来”引起大家的兴趣。在导游指引下，看到“快乐每从辛苦得”“辛”字上多加一横，多一份辛劳，“便宜多自吃亏来”“多”字少的那点移到“亏”字去了。这样的增减笔画，一定是主人的感悟和人生真谛，希望子孙后辈做人、经商，为官，享受快乐少一点，付出辛苦多一点，贪图便宜少一点，甘愿吃亏多一点。现在的房东说，朱镕基来西递参观，也赞赏这幅对联。现在“快乐每从辛苦得，便宜多自吃亏来”被做成各式纪念品出售。
 
环顾屋里处处是对联
事业从五伦做起  文章本六经得来
寿本乎仁乐生于智  勤能补拙俭可养廉
 天井里假山盆景错落有序，花草茂盛，现在的主人还经营着书籍，文房四宝，小纪念品。老人的儿子到外地工作去了，屋里供着他的父亲遗像，他坚守着这份产业，接待游客，传承着文化传统。

住宿：流水人家客栈

我们在西递住在流水人家客栈。客栈位置在主要景点胡氏宗祠附近，西递村居委会对面。从西递景区入口步行十分钟就到了。客栈有7间客房，堂屋很大，八仙桌置正中，旁边还有茶座。学美术的大学毕业生小悦经营着客栈，妈妈帮助打理。客栈还经营着古玩。家里一条叫西点的小狗对住客不认生，跑来跑去。
我们住在二楼的雕花床客房，老妈妈陪我们进屋。雕花大床不很宽，床框雕满花，镶着镜子，这种床在古董店和电影里出现，现在我们要享用了。老妈妈说，这是我结婚时用的，令人惊讶。老妈妈介绍了热水使用方法，怕我们从北京来怕冷，又拿来一床被子，打开空调。客房不算大，干净整洁，温馨。卧具是浅色花布的，还有两个靠垫。墙面和屋顶都是松木板子装饰，两面的窗户是木格窗框。蜡染蓝花布落地大窗帘和桌布很优雅。仿古桌子装饰着金属拉手，两个老式凳子，还有一个椅子。有烧水壶，门后面有挂衣钩，靠桌子墙上有两个电插座。客栈想得很周到，基本满足了旅行者需求。WI-FI 不错。
卫生间有淋浴，洗脸盆干净，有简单洗漱用品，热水充足，无异味。
我们放下行李到三层的晒台观景。晒台上晾晒着店主家的火腿，腊肉。早晨雾气还没有退去，眼前徽派建筑粉墙黛瓦，层层叠叠马头墙，点点红灯笼，红对联点缀。远处的青山还朦朦胧胧。慢慢地青山露出来了，团团雾气飘逸。我们第二天又去晒台观景拍照。
客栈可以点餐，网上有介绍老妈妈很会做饭。当天我们晚餐点了臭鳜鱼和青菜。臭鳜鱼很入味，青菜是他家地里种的，非常可口，吃了比平常多的米饭。大米也是他家种的。他家的春笋炖肉很鲜美。
正餐必点臭鳜鱼。

西递村餐食

街巷里有很多客栈兼饭店。靠近景点入口有一排饭店，舌尖上中国介绍过的毛豆腐，还有面条，包子，馄钝等。印象最深的还是毛豆腐和臭鳜鱼。毛豆腐必须现煎的才好吃。
我们离开西递村时，看到村口一块刻着西递村三百多户一千多人庄严承诺的巨石，他们承担着保护和延续文化遗产的重任。村里所有旧房改造都要经西递镇政府批准，并接受村民监督。
