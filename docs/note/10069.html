有点话要说

当我敲下游记的第一个字，我得庆幸，我此刻还健康到有心情记录下这9天的旅行，这次的旅行从第三天开始，我就开始无比渴望的回到祖国的怀抱。当然不是因为和肖总吵架，毕竟回国还是睡在一个家里。
第三天开始，我就得了怪病，夜里憋气到睡不着觉，白天吃饭吞咽困难加憋气，作为一个年纪轻轻涉世未深的小姑娘，简直就是紧张到觉得自己要客死他乡，无可奈何第四天跑到了 布达佩斯 看急诊，至于细节我后面再说，总之医生180cm＋，英俊潇洒，温柔体贴，顺便告诉我，怀疑我是返流性食道炎，给我开了一种完全不知道是什么的小药片，告诉我吃了这个神奇的药片，你可以支撑到回国去瞧病。最后靠着这个神奇的小药片＋每餐喝汤吃稀烂的面疙瘩，我也算坚持完成了整个行程。最后回来，我还完成了一件所有姑娘都梦寐以求的事，我瘦了10斤。
当然，我不得不佩服我自己，哪怕拖着虚弱的身躯，我还是坚持拍了一大堆乱七八糟的照片足够我完成这篇游记，并且回来发现，我还把肖总和小伙伴们照的美美嗒帅帅嗒，如果不信你就往后瞧，如果真的美美又帅气，记得夸我。
反观我自己，因为这个神奇的返流性食道炎，我吃不下饭，喝不了酒，要知道，吃不了饭也就算了，我竟然喝！不！了！酒！在这样两个啤酒红酒超级棒的国家，喝不了就算了，还要眼睁睁看着别！人！喝！简直惨绝人寰。不过幸运的是，因为这样，我再也没有了喝酒过后的水肿大脸，我这以前略显粗壮的胳膊啊、腿儿啊，也都细溜了不少。省去了我多少液化的工作。
总之，这还是值得你看下去的游记，让我给你还原一个慢节奏的旅行。

关于住宿

这次的住宿选择了 Airbnb ，虽然 英国 已经有过体验，但那一次预定以及和房东的交流都是由摄影师小伙伴来完成的，所以这次也可以算是我自己第一次的全体验。
而首先，你要明确， airbnb 不同于酒店，入住和退房时间差别会很大，而且需要提前和房东商量好这些，否则过早过晚的到达都会影响房间的交接。而退房时间更是重要，毕竟大部分的房东不会提供行李寄存这样的福利。再有就是公寓不比酒店，很多隐藏在小巷子里，没有明显的标志，寻找起来会稍费点气力。
但是 airbnb 的好处自然也很多，好多小伙伴应该都已经安利过了，我就不多废话了。只说这次我们的经历。

行李的准备

1.插座
捷克 和 匈牙利 都是标准的 欧洲 插座，230v，50Hz，样子就是那种两个圆形的孔。一定不要忘记带转换插座，另外相机手机什么比较多的话，再带个插线板更好些。
2.现金&刷卡
捷克 是使用捷克克朗，1人民币是大概3.7克朗左右。
匈牙利 是福林，1人民币大概43福林。
两种货币在国内都不太容易换到，所以可以在国内换欧元或美元，到那边再换成相应的货币，或者直接去那边的ATM机取钱， 捷克 和 匈牙利 都还比较容易找到有银联标识的提款机的。
刷卡在两个国家都比较方便，超市、商铺、吃饭的地方基本都是支持刷卡消费的，不过基本没有什么支持银联的，Visa、Master Card还是必须要带的。
另外两个国家的物价都比国内一线城市要低一些。
3.电话卡
亲测联通的电话卡在两国都是可以正常使用的，每天26不限流量的套餐，但是网速挺慢的，大概也就是2g水平，不如小伙伴在＊宝买的 欧洲 几国通用电话卡好，他们的价格还比我的便宜不少，750M，28天有效，主叫200分钟，才78人民币。
后来我们在 布拉格 买了当地的电话卡，服务商是Vodafone，网速也比联通快很多，价钱有点记不住了。
4.证件、保单
护照这种重要的证件最好复印好带一份，另外这次看了趟急诊发现，保单最好也带着，或者手机里面存份电子版，需要的时候才知道重要性啊。

而 布达佩斯 却刚好并不算是我旅行清单中多么靠前的城市，也许正因为不熟悉也并没有多么的向往，这座城市留给了我更多的空白，最后，这些空白在我的憧憬中满满填满幸福的回忆。
其实说起 匈牙利 、 布达佩斯 ，我绞尽脑汁，脑子里有下面三个联想：
1.陈 佩斯 ，和他不知道长什么样子的哥哥，陈布达。
2.小学时候音乐课上听到过的蓝色多瑙河。
3. 匈牙利 的红酒据说非常不错。
在我离开的时候，三个最深的回忆：
1.黄色电车和绿色的自由桥， 布达佩斯 的颜色。
2. 渔人堡 上面 布达佩斯 、多瑙河的金色夜景。
3. 匈牙利 的红酒据说很好喝，还是据说。。。这可能不是回忆，是怨念。
 当阳 光透过我们贵族公寓的窗子洒入，窗外的多瑙河波光粼粼，如此良辰美景，我端着水杯一仰脖，塞下帅哭医生发给我的神奇小药片，崭新的一天开始吧。
公寓的门口就有 布达佩斯 传奇般存在的2路电车，它沿着多瑙河行驶，途径众多景点，这个电车可是被评为过世界十大最棒的电车旅行经历的，不过乘坐的时候一定请记得看好随身物品。
说到了这，一定要再啰嗦一下车票问题。
布达佩斯 的公共交通，地铁、电车、公交车车票是通用的，而布达佩斯的车票比 布拉格 买起来还费劲，因为只有在地铁站里面才有卖，电车站和公交车站都是不卖票的。公交车司机是可以卖单次票给你的，但是比正常的单次票要贵0福林。所以建议到达布达佩斯以后先去找个地铁站买好了车票。
车票大概是有这些种类：
单次票，350福林，但是不可以换乘。1次换乘的票是530福林。
次票，3000福林。
24小时卡，1650福林。
72小时卡，4150福林，
7天卡，4950福林。
坐车的时候一定要记得上车打卡，车票也一定要保存好，千万不要逃票啊，如果被查票员查到的话可是要罚款的。
但是刚刚来到 布达佩斯 的第一天，我可不会放过沿着多瑙河散步这么浪漫的事情。
九座桥梁横跨多瑙河连接着布达和 佩斯 ，我们公寓门口的伊丽莎白桥刚好是最中央的那一座，往北是链子桥，往南是自由桥，也是九个当中最有名的三座。
按照计划，往北边的链子桥走去，大概需要20分钟，有多瑙河相伴倒也并不无聊和漫长。走累了河边的长椅可以随意的休息。

链子桥旁乘坐16路电车在Szentháromság tér站下车，再次回到了城堡山上，这次的目的地是 渔人堡 。车站旁边有一家邮局，如果你和我一样对寄明信片情有独钟一定不要忘记带着来。
渔人堡 的对面也是 布达佩斯 标志性的马加什教堂，不过我们上来的时候已经关门，只能外观感受一下。
因为历代 匈牙利 国王的加冕仪式都在这里举行，所以这里也有一个名字叫做加冕教堂。
16世纪的时候， 匈牙利 沦陷，被 土耳其 人统治，在那期间这里还被改作称为过清真寺（此时眼睛不由飘过到肖总那里），当然后来又被改建了回来，并且重新融入了巴 洛克 和新哥特风格。
现在看去轻盈美丽的一座教堂曾经经历过辉煌，也见证了略显悲惨的历史，这样的建筑没能走进参观，还是多少有些遗憾的。
马加什教堂
教堂的旁边就是 渔人堡 了，这里其实就是一个观景台，两层的白色建筑群，融合了新哥特、新 罗马 式以及 匈牙利 当地特色的建筑风格。中世纪的时候，这里的渔民负责保卫这一段城墙，渔人堡也就因此而得名了。
到了这里，游客明显多了一些，但是也绝对没有多到影响参观的体验。

DAY7--国会大厦金光闪闪

旅行接近尾声的时候，疲倦感多少有点袭来，更何况这天的第一个景点是国会大厦，这个距离步行就有点困难了，这种河畔的景点，2路电车绝对是最好的选择。
到了 布达佩斯 的第四天，终于在门口的车站第一次等车。。
尽管已经初秋， 布达佩斯 还是给足了面子，阳光不止是耀眼，带来的温度也是喜人的。特意准备的裙子也终于不顾肖总的各种反对被我穿了出来，老公觉得你冷这事儿该怎么解。
非让我穿外套，自己却穿短裤这事是不是该谈谈。
2路电车到Kossuth Lajos tér M站或者Szalay utca站下车就可以到达国会大厦了，如果坐地铁的话是M2号线到Kossuth tér站下车，如果是从链子桥走过去的话，大概15-20分钟也可以走到的。
已经连续两晚远远望到国会大厦的夜景，这天也终于要得见真容，不同于夜晚的金灿灿，下车后整栋建筑出现在眼前，庄重感体现了出来。新哥特风格的外观有点类似 伦敦 的议会大厦，同样许多的细长型尖塔，不同的是中间的圆形拱顶带来了些柔和优雅。
国会大厦有96米高，因为马扎尔人王国在西元896年建国，而且又在建国1000年也就是1896年开始建立这个国会大厦，所以96米的高度也是意义非常。

相比外观夜景的金灿灿，这个据说用了40公斤的黄金为建筑材料的国会大厦绝对是奢华的金碧辉煌。
在1896年开始建设的时候，国会大厦就已经全面采用了电灯，电梯，机械通风，冷暖空调等先进设备，所以这个大厦绝对不是一个金色的样子货，而是从内到外全方位的奢华。

最后一个下午景点的选择又很随意，谷歌了一下，发现 匈牙利 国家歌剧院似乎不远，也是决定过去看看。路上偶然发现了一家 土耳其 餐厅，肖总也是乐开了花，终于可以放心的大口吃肉大口喝酒了。
下图的辣椒看起来很普通，但是据小伙伴说，这是他们人生中吃到过的最辣的辣椒，好吧，我也是疑惑脸。
熟悉的 土耳其 味道肖总表示很满意，回民的口味果然与众不同。
看到我在拍照，一个歪国人走了过来，一张嘴竟然是中文，告诉我们他曾经去 中国 学习过，很喜欢那里，并且主动要求给我们拍合影，而且竟然拍的这么好！！
继续向着歌剧院的方向走去，沿路慢慢出现越来越多的海报，也预示着我们也离歌剧院越来越近，偶尔还能听到歌剧的声音，不知道是有人在练习或者在表演。
匈牙利国家歌剧院
匈牙利 的歌剧院是茜茜公主的丈夫约瑟夫出资建造的，后来茜茜公主也经常来到这里欣赏歌剧。
如果坐地铁的话是M1线到Opera站下车就能看到了。

走进了歌剧院，本想买张参观票进去游览一下，毕竟也没有去看场歌剧的计划，可是谁知，不做攻略的报应再次来袭。我们到达的时间是大概4点半左右，最后一场的内部参观时间是在4点钟，再次丧失了一次感受艺术熏陶的机会。
遗憾的最后只能去逛逛纪念品店，谁知道这时候传来了歌剧的声音，想来应该是上一个团队参观的歌剧片段欣赏，我也只能安慰自己还是听到了歌剧声的。
歌剧院基本上每天下午的3点、4点都是有英文参观团队的，门票2900福林，然后参观的过程中还会有一个歌剧片段的欣赏，对于我这种没什么音乐细胞的人来说，实在没有勇气去听一场完整的歌剧，如果睡着了多丢人。。所以这种参观本应该是最好的。

没能参观歌剧院，只好沿着安德拉什大街逛荡，不过这条大街也是不错，林茵大路，两旁是新文艺复兴宫殿和房屋，这条也是 布达佩斯 的主干路，地底下还运行着 欧洲 大陆的第一条地铁线。
遇到该拐弯的路口，能看到 中国 银行的大字不。
应该是到了下班的时间，街道两旁的餐厅、酒吧渐渐满了起来，不同于夜晚的多瑙河畔更多是些上了年纪的人在悠闲的喝红酒，这边的年龄明显小一些，啤酒也更受欢迎一些，看样子更像是下班以后朋友一起出来放松的。
欧洲 的国家果然都很热爱足球，很多的酒吧都有足球的直播，不过说起 匈牙利 足球，我真是一个球星都说不出来呢。
散步到有点疲惫，查了个公交车回到公寓，拉开窗帘，伊丽莎白桥在夕阳的映衬下上演着最美的一幕。
伊丽莎白桥
旅行接近尾声的时候总是免不了惆怅，说白了还是要回去上班了。惬意的状态在回到生活和工作的城市总是难以找到，恐怕同样的景色挪回到了 北京 ，我也没有同样的心境去欣赏，这也是旅行对我而言最重要的意义吧。
公寓对面的自由山还没有上去过，据说那里的夜景也很美丽，可到了最后一天，更希望的是享受最后的这个夜晚，而多瑙河旁的餐厅绝对是个不错的选择。
我们只吃了这一家餐厅，提不上推荐什么的，饭是什么味道我也记不清了，反正不难吃也不好吃，价格肯定要比一般的餐厅贵些，不过人均也不到200人民币的样子，这个气氛和环境在后海恐怕也要人均300+。
小乐队的表演也不算是糊弄事，花不了多少钱，点一曲喜欢的歌来听，这在 北京 恐怕我是绝对干不出来的，可是这个环境下却觉得挺不错。
吃饭的过程，讨论起下次一起再去去哪儿哪儿哪儿，算不算是故意逃避旅行要结束的悲伤呢。
不过我想的是，既然说要一起去，算不算是给予了我这个半吊子、拖后腿的小领队好评呢。
