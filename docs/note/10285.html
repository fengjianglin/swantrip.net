去浙西隐世小城，寻找山野小时光~

游记 人文游 客栈民宿 小众 暑假 美食 2018/12/07
阳光、微风、草帽、小野花、碎花裙、树影的斑驳……这，大概就是夏日里的小清新。
关于这个夏天的回忆，我想，就从这座低调却能在旅行者心中开出花来的浙西小城—— 衢州 开始。
这里，有久违的烂漫星空，天空是蓝蓝的，溪水是清澈的；去荷塘钓小龙虾，在夜晚邂逅萤火虫；泡在井 水里 的西瓜和饮料，都是小时候的味道……
（△同行小伙伴@灵光 拍摄）
在 浙江 无数的诗意山水中， 衢州 柯城，算不上最耀眼的小城，它太低调，也许一不小心你就会将它错过。
然而，这座小城却一直自顾自低调地美着……
这里有着世外 桃源 的氛围，一座山、一川溪水、一个村庄，更像是一幅画；
这里有穿越千年的围棋文化、儒家南孔文化，有传承千年的非遗民俗，千年一轮回；
这里有难以忘怀的味蕾记忆，远离尘嚣的慢生活，带你梦回田园；
在这里，你可以热恋一片花海，寻觅一尾鱼，飞跃一片丛林；
在这里，你可以回到小时候，奔跑在荷塘边，钓个小龙虾，热了就跳进溪里游个泳；
……
我想，小隐隐于山，大概说的就是这里吧！
其实，这样的日子，离我们并不远， 杭州 出发，高铁1小时or自驾3小时，就能抵达这个世外之地。

/ 落弈烂柯，围棋仙境 /

提到柯城，人们就会联想到千年前那个古老的围棋传说——王质烂柯。这个神秘的故事，像一粒冥冥之中播种的文化种子，在这片历史久远、文化积淀深厚的地域扎根发芽，造就了这片神奇的围棋仙地。
据北魏郦道元所著《水经注》：晋时有一叫王质的樵夫到石室山砍柴，见二童子下围棋，便坐于一旁观看。一局未终，童子对他说，你的斧柄烂了。王樵回到村里才知已过了数十年。因此后人便把石室山称为烂柯山，并把烂柯作为围棋的别称。
“仙界一日内，人间千载穷”。山上的一盘棋，道遍了世事沧海桑田；黑白跳动间，落尽了人间繁华。这使得烂柯山透着传说的神秘感，以及淡泊明志、隐居生活的气息。
虽说是山，但其实烂柯山并不高，拾阶而上，一路树影斑驳，眼前的绿色让人忘却夏日的炎热。不一会就到了山顶~
山顶有一处石梁，东边石壁上刻有“天生石梁”四字，巨大的天然拱顶下有一个巨大的围棋盘。炎炎夏日，外面 日光 强烈，石梁下却是凉风习习，是个纳凉的好地方。
你还可以爬上石梁去望远，目之所及是绵延的不知名的小山，透着南方特有的秀美。
Tips：
地址： 衢州 市柯城区石室乡
门票：20元
开放时间：08:00-17:00

/ 南孔圣地，全国唯二 /

衢州 孔氏家庙是全国仅有的两个孔氏家庙之一，距今已有800多年的历史，被誉为“南孔圣地”、“东南阙里”。
曾在 日本 佐贺 去过当地的孔庙，不过那只 日本 人靠自己的想象建的。 衢州 这个孔氏家庙，却是孔子后裔所建。据史载，北宋末年，金兵南侵，孔子第四十八代裔孙孔端友，抱负着孔子和元官夫人的一对楷木像从 山东 曲阜 南迁至 衢州 ，建立孔氏家庙，这就是南宗孔氏家庙。
对啦，孔庙最特别的就是它的门票，是一本半个手掌大的论语。
步入庙门，便见数株百年以上的银杏树，枝叶披离，挺拔摩天，一种厚重的历史沧桑感油然而生。
这里分为孔庙、孔府和后花园三部分。大成殿是孔庙的正殿，建筑十分雄伟，殿中供奉孔子坐像。此外，还有五支祠、袭封祠、六代公爵祠、思鲁阁等。
孔府看似不大，但一进进往里走，还藏着一个后花园呢！后花园中有亭台阁榭、池塘草坪、假山花木，还养着孔雀呢！
Tips：
地址： 衢州 市柯城区新桥街
门票：10元

/ 树上探险，丛林飞越 /

出衢城往北约12公里，在庙源溪和大候溪交汇处，便是神秘的九华大荫山。这里树木茂盛苍翠，林中鸟语阵阵，山间清泉咚咚，走入山林，清新幽静，有着“天然氧吧”之称。
这里还有目前 亚洲 最大的森林穿越项目——“飞越丛林”，由 欧洲 著名设计师拉斐尔设计，建设者全部来自 法国 的专业安装团队，全部设备、建材、木条、绳索、甚至小到一颗螺丝钉都全部由 法国 原装进口。
“飞越丛林”乐园是一项在树上进行的探险项目，它是用各种各样的障碍环节将树连成一条线路，玩家需通过悬空桥梁、网道、步道、木桶、 泰山 秋千、飞狐索道等各种趣味环节，进行树木间的探险。
根据难易程度，树上线路共分为绿、蓝、橙、红、黑五条路线，其中黑线最难，绿线最简单，一旦进入就没有退回的余地了，所以请根据自己的实力来合理选择哦！
此外，这里还有速降、攀岩、愤怒的小鸟、蹦床、比翼双飞等各种项目。
在整个运动中，汇集了高空、速度、力量、毅力等户外探险所必备的元素，喜欢刺激、敢于挑战的小伙伴们，一定不能错过啦！
（△搞笑向，自娱自乐的小伙伴们）
Tips：
地址： 浙江 省 衢州 市柯城区九华乡新宅村 
门票：50-200元
开放时间：周一至周五：09:00-17:30； 周末、节假日08:30-17:30

/ 赏荷钓虾，夏日童年 /

夏日，是蝉鸣、蛙声、海风与荷塘。
拿上钓竿，带上草帽，去荷塘边钓个龙虾，伴着阵阵蛙声，那一瞬间，仿佛回到了童年的夏天。
“接天莲叶无穷碧，映日荷花别样红”，杨万里笔下的美景，不仅存在于西子湖畔，还藏在“梦里柯乡”。
每逢盛夏，万田乡的荷塘村数百亩荷花亭亭玉立，在夏日的骄阳里争奇斗艳，竞相盛开。迷人的水韵，醉人的荷香，成为了荷塘村一道靓丽独特的风景。这座 江南 小村落里，正孕育着无数人心中可以寻觅的诗和远方。
除了荷花，这里还有很多不知名的小野花，都是夏日里迷人的小清新。穿上花裙子，在花丛中拍美美的照片，谁还不是个小仙女呢？！
Tips：
地址： 衢州 市柯城区万田乡荷塘村 
门票：免费

/ 石梁溪畔，追忆武林 /

沿着石梁溪一路行走时，浪漫的堰坝与开阔的空间，让人忍不住去放空，只想看水看山看时光飞逝。
石梁溪是柯城最美溪流之一，全场12.1公里，沿溪能看到海龙堰、关王堰、司马堰等16个堰，这些堰坝错落有致，临近看时，水流湍急，烟涛微茫；远望时，滚滚水流倾泻而下，是夏日亲水解暑的好地方哦~
此外，这里还能欣赏到金庸文化广场、滨水公园、碧水潭等景点。金庸先生高中时曾在石梁求学一年多，他武侠小说中的“石梁帮”、“桃花岛”、“十三 太保 拳”等创意，就来源于少年时石梁溪两岸的记忆印象。
石梁镇不只是拥有自然的风光、武林的豪气，还有着文艺的气息……
谁能想到，在石梁镇城西村中，有两栋用猪棚改造的艺术画廊呢！原本一个800平方米的养猪场，如今却成了挂满油画、国画和各类装饰画的艺术画廊，令人深感惊喜。
行走石梁溪，需要慢慢地品味，才能体味到这其中的各种滋味哦！
Tips：
地址： 衢州 市柯城区石梁镇
门票：免费

/ 桃源七里，诗意田园 /

说到七里乡，便想起了周杰伦的《七里香》。
窗外的麻雀、温暖的阳光、饱满的稻穗、院子落叶、窗台蝴蝶……周杰伦《七里香》中描绘的诗意田园，在七里乡就能找到。
七里乡位于石梁溪的源头，进入七里境内，顺着 盘山 公路一路往上，这里的溪水比石梁温柔，两岸山峦叠影、茂林修竹，俨然一副天然的山水画卷。
这里是 衢州 的后花园，小气候、原生态、农家屋、高山菜…远离尘嚣，古村落与青山绿水造就一派世外 桃源 的景象。住一次老房子改造的民宿、尝一尝农家的有机蔬菜，感受山里人家的淳朴与热情。入夜，还可以看一看久违的星空~
桃源七里亦有丰富的景点资源：黄土岭有三叠龙潭，杨坞有森林氧吧，大头有石尖问顶， 新店 有林海绿道……你可以在这里来一场有氧游、戏水游。这里，是夏日里的诗意栖息地，也是是理想的归隐之地。
Tips：
地址： 衢州 市柯城区七里乡
门票：免费

/ 绘艺余东，绿里画乡 /

走在余 东村 清静的街巷中，夏日的骄阳变得热烈，街巷两旁的墙画仿佛开出绚烂的花朵来。
或牧童放牛，或乾隆下 江南 ，或天鹅游湖，色彩浓烈的画作印在村民院落的外墙上，田园风采的气息随着夏日的微风扑面而来，让人仿佛走入了摩西奶奶五彩斑斓的世界。
不过这里没有摩西奶奶，却有被誉为“田园毕加索”的画农们。
即使村里最老的老人，也记不清余东画画的年岁了。从呱呱坠地时，眼前就满是五颜六色的画面，手执画笔的传统被一直保留了下来，成为余 东村 民们农闲时打发时光的最佳方式。
跟着当地的农民画老师们，学着画画，虽然我画了个蛇精脸，不过还是得到了老师的好评呢！
Tips：
地址： 衢州 柯城区沟溪乡
门票：免费

/ 寻一尾鱼，品鱼子酱 /

对鲟鱼初次有印象，是源于《十二道锋味》。当时，有一集的主题就是“为爱烹饪鲟鱼大餐”，谢霆锋、阿Sa在 西班牙 的鲟鱼养殖场挑选了一条六七十公分的鲟鱼，取出新鲜诱人的鲟鱼鱼籽。
鲟鱼，据称是世界现有鱼类中体型大、寿命长、最古老的一种，有“水中熊猫”和“水中活化石”之称。鲟鱼的营养价值及美味，在我国古代就早已享誉大 江南 北。
到石室乡吃鱼，是到 衢州 不可错过的一件事。当然，家门口的鲟鱼场，也是老饕们的福音！
你可以约上三两好友，去鲟鱼场，亲自选一尾肥美的鲟鱼，就近找个餐馆代为烹饪。根据鲟鱼的属性，用来清蒸再以葱油泼之，最是美味啦！
当然，你还可以在这里品一品鲟鱼鱼籽酱，配上烟熏鱼、苏打饼干食用，再喝上一口气泡酒，高级下午茶大概就是如此啦！
Tips：
地址： 衢州 市柯城区石室乡

/ 古城老街，深巷美食 /

有人说， 衢州 本是座美食城市。想要尝遍 衢州 当地的特色美食，就去 衢州 古城的老街深巷中吧~
水亭门，古城最沧桑的一面。古老的城墙，白墙黑瓦的建筑，石板路铺成的小路……它就像一本书，把曾经的风雨烟云不动声色嵌入字里行间。
当然，在这里，你不仅可以感受到 衢州 柯城古老的一面，也能在老街深巷中，找到当地最好吃的美食！
“三头一掌”是 衢州 最出名的风味小吃，分别是兔头、鸭头、鱼头和鸭掌。鲜、咸、辣三种味道交杂，让人忍不住大快朵颐。
此外，还有 衢州 烤饼、 衢州 麻饼、鸡蛋饼、凉拌粉干等特色美食。
炎炎夏日，记得一定要来一碗水晶糕，甜甜的糖水，清凉的薄荷味，晶莹透亮，糯软耐嚼，是夏天清热解暑的好东西哦！
Tips：
地址： 衢州 市柯城区水亭门

关于住宿

▼圃舍·源溪民宿
地址： 衢州 市柯城区九华乡新宅村
费用：500+元/晚
圃舍保留了原有房子的结构体系，利用旧房子改造空间，使用传统的建筑材料还原乡村生活氛围，运用原石、原木、菜田等自然元素刻画自然景观。
民宿共有15间客房，木质小清新的装修，有点日系muji的风格。房间内各种设施一应俱全，还有提供水果哦！民宿还有一个私享庭院、溪味餐厅、书友书屋。入夜，和小伙伴们一起在庭院烧烤、喝啤酒、看世界杯，大概是这个夏日最燃情的夜晚了！
▼天光云景
地址： 衢州 市柯城区七里乡 新店 村
费用：400元/晚
天光云景民宿由13栋白墙黑瓦的徽派建筑组成，每栋楼都融入一个文化主题，分别为琴、棋、书、画、诗、酒、花、茶、香、曲等具特色的 中国 文化元素。在不破坏乡村自然结构的基础上，结合当地的特色，打造“清幽雅静”“ 中国 式雅致”的主题文化民宿村。
早晨，民宿会准备好一份早餐，有粥有粗粮还有水果，味道很不错！民宿周围就有很不错的风景，白天可以戏水赏花，入夜可以看漫天星空！
▼祥云阁
地址： 衢州 柯城区七里乡大头村上村28号
费用：400元/晚
这家民宿是利用古民居改造的，保留了老房子的传统风格特色。远远望去，泥黄色的外墙，返璞归真；走进院内，花香扑鼻，后院还有石磨、蓑衣等，还有儿童活动区，就像是一户寻常人家，让你逃离城市喧嚣，回归田园生活。
