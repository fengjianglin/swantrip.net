触摸香格里拉遇见2019，难忘的独克宗花巷跨年之旅

游记 人文游 元旦 古镇 摄影 特色表演 2019/01/12

前言

这次香格里拉之行，朋友问了大家一个问题：为什么要来香格里拉？这里对你意味着什么？如此认真的问起来，倒也说不出一个很确切的答案。
就像王力宏所唱的“你是心中的日月，落在这里，旅程的前后多余，只为遇到你 ”，大概就是一种感觉，一种向往，也许仅仅觉得“香格里拉”这四个字听起来足够美好，无形之中就在脑海里构想出一幅乌托邦的景象。
香格里拉，这次是第二次来访。对于喜欢的地方，我不介意多次踏足，不同时间不同心境，自然会有不同的解读和感受，所以这次对香格里拉也有了更深的了解。
这里有着西藏的气息，却比西藏多了一些浪漫的色彩，就是它独特的气质，吸引着我们前来朝圣。虽然短短几天的行程，却留下了太多太多难以磨灭的记忆。
在松赞林寺，晒着暖暖的太阳，看着僧人们为迎接格冬节而排练节目。
独克宗花巷，在浓浓的藏式氛围里，与千人共同参与跨年长桌宴。
在蒙古哥的店里，把酒言欢，一起聊到2019年的来到。
在龟山公园，迎着新年的第一缕阳光，转动世界上最大的转经筒祈福。
我们在这里度了一个难忘的跨年之夜，看着时钟的指针从2018走向2019，感慨甚多，过往记得也好，忘记也罢，开心的，悲伤的，无论好与坏最后都会被淹没在岁月的洪流里。
但是特别的日子里，我要选择特别的方式来度过，正是这样的仪式感使得这个年跨得意义非凡，这样的经历，我也相信会一辈子念念不忘。

独克宗花巷

从丽江到香格里拉，风景在车窗外一路移动，不到4个小时的车程，却发生了翻天覆地的变化。比起丽江的石板小巷，香格里拉多了些许磅礴的大山大水，再加之有着浓郁的藏式风情，味道立马又很不一样了。
我的第一站是独克宗花巷，位于独克宗古城的北部。
一走到大门口就看到工作人员正在忙着明天的跨年千人长桌宴，其中一个见到我后，立马询问我是订了哪个酒店，我回复光年酒店之后，她就顺手接过我的行李说，走，我带你过去。
瞬间，心里暖暖的。
在大堂办理完入住，她又把我送到房间才离去，真的不得不感叹，超级贴心细致的服务呀。
冬天的香格里拉，夜里的冷是刺骨的，裹着几层衣服依旧还在寒冷的边缘颤抖着，但我还是想好好逛一下花巷，这是我对新到一个地方的致敬。
只见精致的藏式房屋，分布在巷子两边，在夜里静静凝视着，诉说着无声的言语。
夜色渐浓，却也掩盖不住随着风飘扬的七彩经幡，它每拂动一次就像意味祝福传送了一次，此时此刻，我才坚定我站在了香格里拉的脚下。
我喜欢一座城的晚上跟清晨，因为这两个时候万籁俱静，仿佛一切都是尘埃落定的样子。
翌日，则在蓝天大太阳里醒来，高反什么的对于我来说，都是不存在的，而且光年酒店里有着全天候弥散式供氧系统，所以更加没有感觉。拉开窗帘就感受到阳光的热烈，于是立马洗漱，立马吃早餐，立马出去逛一逛，不能辜负如此好的天气。
看着明媚的太阳，晒着暖意融融，有那么一瞬间的恍惚，仿佛昨夜的冷是一种错觉。这里的天总是蓝得那么的纯粹，一朵云都没有。
如果说想以最直接的方式了解香格里拉的藏式风情，那么独克宗花巷就是一个很好的窗口。
这是一个以藏文化风情为主题，整合藏区非遗体验、藏文化情景商业街、第5颗陨石系列酒店、原生态藏餐美食、《遇见·香格里拉》演艺秀等多种商业业态于一体的藏文化情景度假目的地。
从门口的大戏台开始，就置身于这个处处充满藏式氛围的小镇。大戏台上，景区会定时进行文艺汇演，游客们可以即时享受一场视觉盛宴。
绕过戏台，往两边的商业街逛一逛，也是有很多有趣的发现。
闲逛中，就看到这家彝族老奶奶开的店。
香格里拉的主体民族是藏族，除此之外还有纳西族、彝族等十几个民族，每个民族也有自己不同的文化特色。
老奶奶这家店就是经营着彝族特色的服饰，衣帽，生活用品等等，一眼望去，都色彩斑斓，做工精致。
如果想拍上一组民族风的照片，买上一套倒也是不错的选择。
紧接着一路逛到非遗体验馆，则可以看到尼西的黑陶、奔子栏的木碗、藏式的弦子、还有唐卡等等藏式非遗文化。
唐卡画院，进来刚好就看到一个藏族的小哥哥正在专心地画着唐卡，每一笔每一画都是那么的小心翼翼。
唐卡有着吉祥的寓意，也是艺术美的体现，它的画作历时短则半年，长达十几年。
虽然我并不能完全读懂它的含义，但也丝毫不影响我欣赏它的丰富绚丽的色彩，以及宗教的神圣，当细细端详上面的佛像时，会发现无论站在哪个角度，佛的眼睛都是在看着我，充满悲悯。
尼西黑陶，尼西是香格里拉的一个藏式村落，有着悠久的制陶历史，制作出来的陶瓷都非常精美与别致，尤其是烹饪起食物来，有着普通器皿所没有的香味。
奔子栏的木碗，每个都是独具匠心，说是碗，有时候觉得是艺术品更多。因为是手作，所以带着温度。
江川铜锅，江川拥有着深厚的古滇青铜文化，但知道它的人并不多，正是因为它的偏僻和落后，才保留下了流传千年的铸铜工艺，记得之前在抚仙湖，便吃了当地很出名的铜锅鱼。
这锅，便是江川的铜锅，煮出来的鱼肉鲜味美，鲜甜至极。
藏香，多数用于佛教祭祀活动，也有少量用于家居的净晦辟邪。它的制作工艺流程蕴含着藏文化的精髓。
随后又逛到了蒙古哥的店里。
记得那天看到他时，他正在桌子旁雕刻着字，神情专注。
听到我们进来后，抬头笑着欢迎我们；他的眼里带有光，直觉告诉我，他是一个有故事的人。
是的，确认过眼神，真的是有故事的人！于是，听故事也成了逛独克宗花巷的亮点之一。
蒙古哥来自内蒙古，所以大家都亲切叫他蒙古，“SH.XM画廊”是蒙古哥和他的搭档阿娇共同经营的一个“心灵疗养站”和“生活体验馆”。阿娇平时主要负责画画，蒙古则负责店里画画以外的所有事物，协助阿娇更好的创作。
因为热爱自由，想寻找心中想要的生活，做真正想要做的事，蒙古跟阿娇2010年从杭州来到了云南，刚开始是住在虎跳峡徒步路线的第二个村庄上。艰苦的环境，贫瘠的生活条件，但他们的心灵却从所未有地感受到自由，在那里，阿娇潜心画作，蒙古则打一些零工赚取生活费用。最穷困的时候他们身上只有20元，但是他们从未放弃过希望，甚至觉得每天都充满了期待。
后来经济条件慢慢好转，机缘巧合之下他们便搬到了香格里拉的独克宗古城，在最美的巷子上搭建起他们的第一个心灵疗养站——SH.XM画廊。
这是一个远离一切的第四空间，来到这里每个人都可以有着不同的解读方式，也许是轻松的，也许是安宁的，也许是愉悦的。在这里，香格里拉不止是香格里拉，而是你心中的理想国。遗憾的是2014年1月古城的那场大火，把他们的苦心经营的理想国给烧没了，所有家当以及累积了十多年的画作都在大火中化成灰烬，只救出他们养的一条狗。
如此跌宕起伏的人生，如果不是亲耳听到，我都不相信这是真实发生的事。再后来，独克宗花巷的开业，让他们的理想国得以延续，就是我们现在所看到的这家店。
可以看得出，店里的设计都别具匠心，慢慢探寻，总能发现不一样的惊喜。在吧台旁边，是一个T型舞台设计，连接着鱼池，从T台则可以走向“香格里拉不只是香格里拉”的小屋，在这个小屋里，你可以做一切你想做的事，无拘无束，自由自在，发呆一整天都是可以的。
而在天花板上则挂着好多小小的盒子，有两种颜色，一种是花的，一种是纯色的。蒙古哥说，你可以把你的心事或是秘密写了放在花色的盒子里，永远不会有人打开，而我也会帮你永久保存着。而纯色的盒子，你可以写上你的梦想、想实现的事，这些盒子其他人都可以打开，也许某天真的有人可以帮你实现你的梦想。
真的太多有意思的设计了。
而在阁楼上，还别具心裁地设置了座位，用蒙古哥的话说，就是可以换一个角度看世界。最后，我们一群人也在阁楼上拍了张合照。新的一年，愿我们都能以全新的方式去拥抱这个世界。

噶丹松赞林寺

松赞林寺有小布达拉宫之称，是云南省最大的藏传佛教格鲁派寺院，也是康藏地区有名的“十三林”之一。坐在车上远远看着它的时候，它座落在蓝天之下，半山腰之上，恢宏壮观，金色的屋顶显得格外引人瞩目，越是走近越是觉得震撼。
来到这里，总是让我想起在拉萨的那些时光。庄严的藏式寺院，很神圣也很神秘，作为一个行外人，只能以自己的方式去解读，看起来美，就够了。
这种美，是跟整个大环境融合在一起，一切显得那么的和谐而美好。
行至松赞林寺的大门口，抬头就可以看到牌匾上写着“噶丹松赞林寺”几个大字，这是由五世达赖所赐的寺院名称。在佛教故事里，天界分为“六重天”其中第四重叫“兜率天宫”，而“噶丹”就是“兜率天宫”里的藏语称呼。“松”在藏语里是“三”的意思，“赞”是指天神游戏过的地方，而“林”为幸福的乐园。所以，“噶丹松赞林寺”就是“众神游戏的兜率幸福乐园”。
过了大门，需要踏上147台阶，才能抵达扎仓大殿。平均3千4的高海拔，爬着这100多级的台阶，也是气喘吁吁，走至半路，回头一望，就可以眺望到远处豁然开朗的风景，一下子就心旷神怡。
继续往上爬，耳边传来寺院里独特的音乐声，一打听，才知道原来是院里的僧人们正在为即将到来的格冬节排练节目。
以顺时针的方向走到扎仓大殿前的广场，便见到一群僧人围在一起跳着独特的舞蹈。
每个动作走位都有章法，随着音乐舞动着，虽然不懂，但是热闹欢乐的氛围还是可以感受得到的。他们脸上眼里都洋溢着笑意，在阳光中闪闪发亮，有时候也会互相讨论一下。
后来查了一下，格冬节在藏语里，“格”是9月的意思，“冬”准确的音应该是“度”，是汇集的意思。
以前很多牧场、田地都是寺院的财产，租给平民放牧耕种，每年到了藏历9月，所有的租都收齐了，寺院就会举办庆贺丰收的活动。
盛会还未正式开始，还在排练的阶段，也吸引了很多当地藏民前来观看。
他们三三两两坐在台阶上，小板凳上，晒着太阳，看得入神。
驻足观赏了好一会之后，继续逛一下其他地方。
在走向释伽牟尼殿的台阶上，有一群藏式的老奶奶在晒着太阳，她们穿着藏族的特色服饰，一脸平和安详看着来往行人，也吸引了好些外国游客前往去跟她们合影。
老奶奶们虽然有点害羞，但也没有拒绝，开心地跟游客一起照相。
她们都是脸上都带着慈祥和蔼的笑容，看着就很想拉着她们闲话家常，可惜语言不通，鸡同鸭讲，全程靠比划，也是很有意思的体验。
当然，笑容就是最好的语言了。
寺院里僧人众多，走几步便能碰上，有时候也会微笑致意，或是好奇的看着我们。
站在寺院内的置高处，可以看整个香格里拉的县城尽收眼里。
而在寺院的正前方是一个湖泊，冬天了，湖水都结冰，跟周边干枯的树枝组成一幅冬景图。
远处的山上，也有不少积雪。
冬天的风景，没有春夏秋色彩的鲜艳和浓烈，但是看起来很素净，万物处于沉睡的状态中，是那么的安静。
我喜欢这样的香格里拉。
往回走的时候，碰到一只小狗在欺负着一只小黑猪，这个画面也是很好笑，于是立马拍下来。

千人跨年长桌宴

长桌宴不稀奇，但是数千人的跨年长桌宴，就非常值得期待了。为迎接2019年的到来，独克宗花巷特别组织了这场盛大的跨年盛会，让广大游客可以在有着人间净土之称的香格里拉，体验最淳朴，最自然，最浓郁的藏式生活。
晚宴即将开始之前，独克宗花巷的卓玛小姐姐们便身着盛装，在门口载歌载舞迎接四方来客~
老奶奶们在墙边晒着太阳，聊着天，悠闲自在，她们还是晚上篝火跳锅庄的主角之一呢。
其他宾客们也陆续就座，现场还邀请了香格里拉特殊学校的同学们一起参与这场跨年盛会。
小朋友们都对这场长桌宴充满了期盼与好奇~
工作人员忙碌的身影也处处可见，这个时候他们正忙着在上菜，青稞酒，藏式八大碗等一大波当地的美食正在上线~
跳锅庄的奶奶们围坐一起，等待开宴，她们穿的衣服都很漂亮~坐在一起就是一道风景线~引得我忍不住多拍几张~
这也是我第一次同来自国内外不同地区的陌生人一起吃饭，虽然都不认识，但坐在一起之后就像朋友一样，其乐融融的，相信此刻大家感受到快乐的频率是一样的，真的很神奇呀～
长桌宴的主菜是藏香八宝火锅，要说这道菜的特别之处，首先就是锅了，就是前面所提到的尼西黑陶，煮出来的汤底味道鲜美浓醇，非常好喝，好喝到导致我们直接把汤底喝到一滴不剩。
配菜也是很丰富，有炸香酥肉，彝家红烧肉，小炒藏香猪，吹肝凉三丝，花生拼虾片，银耳扣南瓜，和很开胃消食的奶渣，吃太多肉之后吃上一口，酸酸的，纯正的藏式风味~
无论大小节日，都需要酒来点缀，而在高原，肯定是喝青稞酒了~
宴会期间，还加入了情景演绎，把游客们也带进了属于香格里拉的故事里~
首先是独克宗花巷的城主亮相，拿着一碗酒，先干为敬以表示对大家的欢迎。
卓玛小姐姐也纷纷端着酒，热情招呼着大家~
彝族老奶奶笑得合不拢嘴，一直叫身边的客人多吃点~
大家就在愉悦的氛围里吃吃吃，喝喝喝~
随后一阵悦耳的歌声从楼上传来，原来城主的女儿在招亲，对着楼下的扎西们说，谁能接上她唱的歌，她就嫁给谁。
最后就是这位帅气的扎西抱得美人归。
城主家双喜临门，更加开心叫大家一定要喝得尽兴！不醉不归！
所有人都在热闹的氛围里结束长桌宴，之后，等待我们的就是围着篝火跳锅庄。
锅庄是藏族的一种传统民间舞蹈，节奏活泼而热烈，不受时间、地点、人数的限制，男女老幼皆可随意加入或退出跳舞队伍。
随着音乐的响起，扎西跟卓玛们都甩着长袖，迈着轻盈的步伐，围成一个圆圈跳起来。
他们的身姿时而优美，时而豪迈，非常欢乐，所以现场也有很多游客加入他们的队伍中，跟着他们的步伐，一起围着篝火欢跳。
原本有点冷意的夜晚，瞬间就暖起来了，快乐也会传染，所有人都融入到舞蹈的海洋里，现在回想起来，每一幕都还是记忆犹新，历历在目。

新年祈福&穿城跑

2019年的第一天，自然还是在香格里拉的光年酒店里醒来，想着今天还有新年祈福活动，便觉得很是兴奋，睡不着了。赶着太阳还没有出来的时候，赶紧起来收拾一下，往月光广场走去。
世界上最大的转经筒就在这里，它总重60吨，高达21米，相当于几层楼高，至少需要三、四人合力才能将其转动~
据介绍，这个转经筒为镏金纯铜制作，筒内藏有经咒、六字真言124万条和各种佛宝16吨，它位于龟山公园的山上，需要爬一段楼梯上去才能抵达。
跟着一大群人一起爬上去，倒也不觉得累，很快就爬到山顶。
站在最高处，可以清楚看到独克宗古城的全貌，估摸着有些人家正在做着早饭，炊烟袅袅，烟火气满满。
然后我们就一起转动这世界上最大的转经筒，转的时候我也默默许下了新年愿望~
转完转经筒之后，参与此次活动的游客和当地藏民重新回到月光广场集合，整装待发，参与全城穿越跑，以活力满满的面貌迎接新年的到来。
这是一项全民参与的健身活动，全程5公里左右的距离，一路穿过古城古街，经长征大道折返，最终抵达终点红太阳广场。
穿城跑期间，警察叔叔也为运动健将们开路，保驾护航。
当天气温很低，但是每一个人都热情高涨~
所有人都坚持跑到终点~
卓玛们则早已在红太阳广场，唱着欢歌，迎接归来的运动员们~
最后，大家都在广场上开心的合影，记录属于他们胜利的这一刻，新年的第一场狂欢也拉下了帷幕。

关于食宿

整个行程入住的是位于独克宗花巷的光年酒店，它是第5颗陨石系列里主题酒店之一；因为香格里拉地处高原，所以酒店拥有全天候弥散式供氧系统，能快速缓解高反，实现人体功能恢复，入住客人就完全不用担心高反的问题了。
然后关于陨石，还有一个美丽的传说，在很久之前，有5颗陨石落入香格里拉，代表着勇气、智慧、爱心、健康的其中四颗分别被当地名门望族寻得，代表幸运的第五颗陨石，则遗落在独克宗花巷，所以成为了系列主题酒店的命名。
入住以后，觉得各方面都非常满意，首先，环境很赞，浓浓的藏式风情，非常别致；其次设施很赞，办理入住之后，就有一条短信发到我的手机里，登入酒店提供的网站设置一下，进入房间就可以用手机微信操控，不用每次开门都要找房卡，真心太方便了。房间里还很贴心地准备了加湿器，这对于干燥的高原地带，且开着暖气的我们，十分有必要。
酒店里也有餐厅，早餐非常丰富，也有午餐跟晚餐供应，出品都非常赞。
酒店前台~
我们入住的是标间~
还有一处飘窗，可以喝茶，很舒适
卫浴都是科勒的~洗浴用品都很精良~
酒店里的餐厅，玻璃的天花板，可以晒着太阳吃着美食。
接下来会有一大波美食介绍袭来~
牦牛刺身，海鲜刺身就吃多了，牦牛的还是每次吃，据餐厅介绍，刺身只选用牦牛的里脊位置，所以一只牛只能切出2.5斤，可以说很珍贵，口感也是很特别，很紧实，这个可能也是跟牦牛长期生活在高海拔地区，且是散养有关系。
白切尼西鸡，就是高原上的走地鸡，用白切的烹饪方式保留了鸡本身的鲜美
花巷宝塔肉，是酒店的特色菜，是一个宝塔的形状，非常别致~
羊肚菌炖盅，非常鲜美的一道汤，营养价值也高
乾坤金牛掌，是牦牛的掌，饱含胶原蛋白的胶质

藏红花娃娃菜，一道很适合女生吃的菜~

果盘也是充满艺术感~草莓非常甜~
餐厅晚上的时候还能喝喝小酒~
适合三几朋友小聚一下~

高原旅游注意事项

1、首先就是高反的问题，因为香格里拉的海拔有3400左右，有些人会有高反反应；关于这个问题，首先就是放松心情，刚到的时候不要进行太激烈的运动，行动尽量缓慢一点，过个一两天就可以适应的了。如果反应真的很严重，可以采用吸氧的措施，但不建议一来就吸氧气，因为会有依赖性。
2、高原紫外线比较强，女生们要带高一点度数的防晒，还有气候比较干燥，一定要带面膜跟身体乳。
3、关于天气问题，香格里拉的冬天多是晴天，且阳光充裕，就是早晚温差有点大，要注意保暖。
4、尊重藏族的风俗习惯，行路遇到寺院，玛尼堆，佛塔等宗教设施，要从左往右顺时针绕行。
