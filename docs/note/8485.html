周末在韶关亲近自然，泡巴厘岛风情温泉，享静谧天然氧吧！

游记 亲子 家庭游 暑假 温泉 自驾游 2019/06/29

序言

炎炎夏日酷暑难耐，越来越多的人喜欢自驾，寻一片森林美景，找一家度假酒店，在郁郁葱葱的绿意中过一个清凉周末。位于珠三角的 韶关 ，就有这样一个不错的避暑胜地，这儿绿树成荫，空气清新，可以感受到 巴厘岛 般的度假氛围，不光可以与大自然亲密接触，还可以游泳泡温泉，适合带小孩子一起来，是一个老少皆宜的旅行目的地。
很多人觉得泡温泉在冬天比较适宜，但夏天泡也有很多好处，由于气候炎热，人的体内会产生很多毒素及油脂，适度泡温泉不仅可以缓解压力，还可以塑身美颜。 韶关 这边都是真温泉，水量大、水质好，真正做到了回归养生的本质。

森林温泉浴，休闲养生地

上个周末，我带着家人体验的蓝山源岭南东方温泉酒店就非常的不错，这里依山傍水，位于“世界过山瑶之乡” 乳源 ,从 广州 城区出发，3小时左右车程即达，也可以坐高铁到 韶关 站再通知酒店来接。这里是一个由温泉、湖泊、山地、植被景观为主体的自然风景区，置身其中，沐浴天然氧吧，可以给肺一场彻彻底底的洗礼。
该酒店是当地最好的五星级度假酒店之一，依山面湖、设计风格以现代感 东南亚 巴厘岛 为主，设计中采用大量的大自然材质，体现异国情调以及神秘、宁静、清雅、放松的度假氛围。集天然温泉、休闲度假、中西餐饮、绿色会议、 康乐 养生于一体。
这里的温泉中心以 东南亚 巴厘岛 风格建造，总共有45个不同主题的温泉汤池。
温泉水质天然，元素丰富，属于偏硅酸弱碱性碳酸泉，俗称“小苏打”，富含偏硅酸、锂、锶、钡、氟、锌、银、硒等48种对人体有益的矿物质和微量元素。温泉水温常年保持在52℃-56℃之间，是 广东 地区唯一可以在下雪天泡温泉的汤池。
泡汤池依据地势高低起伏而设，在自然山脊中错落分布，格调新颖、恬静幽雅。有各种功效的中药养生温泉，美容功效的鱼疗池、纳米池，药浴、瀑布浴，极具热带风情的 苏门答腊 水疗池。每一处都可以看到不同的山野景色...
个人最喜欢的还是这种铺满了玫瑰花瓣的汤池，玉水清澈，水质温和，不仅能提神醒脑，还能美容养颜。阵阵清香环绕鼻尖，与最爱的人在里面嬉戏玩耍，何尝不是一种惬意的放松呢？不过建议大家浸泡时间不宜过长，以每次5-10分钟为宜，记得及时补充水分。
对了，带小孩的朋友还可以来泡果冻池额，池子里放满了红色和蓝色的Q弹小珠珠，待在里面，不仅解暑，还可以释放天性，返璞归真，非常好玩。
泡完温泉，最适合去蒸个桑拿，不同功效的热辐射能量房就隐藏在这样的一栋栋小房子里，进去待一段时间，即可为身体排浊忘忧，恢复活力。
白天嫌太热的小伙伴，也可以在山脚的动感欢乐天地儿童池泡水，清幽静谧的环境令人心情舒畅。
戏水区的水比较浅，放置了很多卡通玩具，很适合大人陪着孩子一起玩，享受清凉的亲子时光。也可以三五好友在里面打水仗，欢乐的笑声包围着整个游泳池,荡漾着欢乐。
颜值最高的当属山顶的圣泉 · 无边界泳池，堪称整个度假区的网红景点，游到边缘，仿佛与周边的山谷浑然一体，女孩子还可以来此拍摄背影照，一定会惊爆朋友圈。
泡温泉或者游泳累了，也可以来半山腰享受一次理疗，这里的中式按摩服务时间是中午12:00-晚上24:00，不仅能中式全身按摩，还可以精油护体，敷一次黄瓜面膜，让肌肤恢复弹性。

巴厘式古法SPA

众所周知， 巴厘岛 是全球公认的“spa天堂”，而蓝山源岭南东方温泉酒店的SPA中心聘请的都是来自 巴厘岛 的专业水疗师，无需远赴国外就能享受到纯正轻柔的推、按、揉手法。
在轻音妙曼、温馨宁静的雅致空间里，瞬间烦忧尽忘、倦容已去，浑身散发着精油的香气，不仅可以改善身体皮肤粗糙状况，排除身体毒素，还可以让人有深度放松的感觉。

休闲娱乐项目

晚上则可以来豪华 KTV与三五好友高声歌唱，释放内心的激情。这里的开放时间是20: 00-01: 00，大房 1880 RMB ( 最低消费 ) 中房 1380 RMB ( 最低消费 )。
喜欢打麻将的也可以去棋牌室，营业时间 15:00-01:00 58 元 / 小時。

罗曼斯中餐厅

一段完美的假期时光，美景与美食都不可辜负。当原生态的上乘食材遇见顶级的五星级厨艺，就是一场舌尖上的盛宴。 罗曼斯中餐厅以特色地方菜为主打，同时精心提供各式 潮州 菜、客家菜以及瑶乡特色美食。中餐大厅可容纳250 人同时就餐，并设有VIP 餐饮包厢，湖畔餐厅...
这里的食材都是新鲜采摘，特别是那些严选的山珍，说不定昨天都还在南岭的原始森林里。窗明几净的就餐环境静谧优雅，仿佛身处山水画之中！推荐几道特色菜，来此一定不要错过...

清蒸黑皮鸡枞菌,一定要重点说一下，不仅含有人体所需的蛋白质，脂肪，还含有多种营养成分，是体弱、病后、老年人理想的食疗佳品。黑皮鸡枞菌肉细嫩，气味浓香，味道鲜美，是食药用菌中丰富资源宝库中的一支独秀。它的提鲜美味之功效，远在味精之上，因为它的谷氨酸钠含量较高，所以，无论炒菜还是清蒸或是做汤，其滋味都很鲜美，是餐桌上不可多得美味佳肴。

登巴萨西餐厅

主要以西式点心为主，当地农家小食为辅的自助餐形式。营业时间 （早餐）07:30-10:00，（宵夜）21:00-01:00。
开放式设计与独特的主题分区，融汇时尚与经典风格，环境幽雅，氛围浪漫，是品味异国风情、美食的最佳场所。

房型推荐

我当时入住的是豪 华山 景房，房内空间很大，宽敞明亮，全景式落地窗可以将窗外的山景一览无余。桌上还配有一套精致的茶具，可以在此泡一壶香茗，卸下一身的疲倦与烦忧。清晨，当远处的云雾从山谷飘来，简直美如画卷，令人沉醉。
阳台上摆放的按摩大浴缸，在夜深人静的时候躺下去，放置一些舒缓的音乐，或是听蛙鸟虫鸣，那种亲触自然的美好让人只想静静发呆去好好享受。
洗浴间采用干湿分离的设计，洗护用品应有尽有，配套设施全是五星级标准，透明玻璃隔板显得整个房间更加通透大气。
视野最好的就是这间九十平米的豪华湖景套房了，享受着舒适的大床，在清朗月色中入眠，在清脆的鸟语中醒来，享用完美味的自助早餐，泡着房间温泉池，仿佛过上了神仙般的日子。
敞亮的落地窗对着群山绿树，波光潋滟的蓝山湖一览无遗，群山不语人自醉。
双床房也不错，视野开阔。
酒店的走廊楼梯也设计的很有意思，在室内就可以看到窗外满山翠绿的好风光，很是养眼。

周边环境

在酒店的度假好时光，无时无刻都可以与森林共同呼吸，享受最纯净的空气，暂且忘却工作，扔掉烦恼，面山静坐，修心养气。
这里的荷花正直盛开的季节，犹如亭亭玉立的少女，给宁静的池塘增添了无限的生机，来此转转，浪漫情节油然而生，让人陶醉，流连忘返。
一直沿着芦苇荡的小路向前，又仿佛到了一个诗意的世界，翠翠绿绿的环境漂亮极了。
在山的尽头还藏着一个瀑布，虽没有飞流直下三千尺的震撼，却能带来清清凉凉的感受，在山脚激起的千波万浪，水雾蒙蒙，仿佛在为自然拍打最美的音符。

结束语

两天的度假时光太过美好，以至于心里似乎离不开这山清水秀的度假酒店了！享受在森林里泡温泉、游泳，蒸桑拿、做SPA、吃美食...下一次，继续带着家人来这里过一个轻松的假期吧。

注意事项

1、充分休息
虽然泡温泉可以修身养性，强身健体，但是也是需要耗费体力的。 所以，泡温泉前必须要充分休息好。
2、泳衣泳裤
泡温泉一定要准备好泳衣泳裤，男生宜带平角泳裤，女生则可选择 大方 得体的泳衣。
3、密封袋
对于公务繁忙，即使在泡温泉的时候，也不愿放弃商机的 成功 人士，泡温泉前可准备一个透明的塑料密封袋，便于将手机等贵重物品放在袋中，以便于把握一切商机。
4、毛巾浴袍
由于泡温泉通常会不断地更换池子，在出水后用毛巾擦拭身上的水以免受凉非常有必要。所以带上干净的毛巾和浴袍是必不可少的。
5、保温杯
泡温泉过久会有有胸闷、口渴、头晕等现象，因此，备好水杯及时出池补充水份才是王道。
泡温泉有何讲究：
1、肚子饿的时候，不可以马上浸温泉，因为空著肚子泡温泉很容易会有头晕，想要吐及疲倦的情形。
2、如果坐了很久的车或是走了很远的路，非常的累了，不可以马上去浸温泉，不然会越泡越累。
3、睡眠不足或是熬夜，如果突然浸温度很高的温泉，可能会发生休克或是脑部缺血情形。
4、心情很兴奋或是很生气，心跳变快的时候，也不适合浸温泉。
5、刚吃饱饭或是喝完酒，不可以马上去浸温泉，不然会有消化不良及脑溢血的情形。
6、营养不良或是生病刚好，身体很差时，千万不可以去浸温泉。
7、有心脏病、高血压及有动脉硬化的人，要浸温泉之前，要先慢慢地用温泉擦身体，再泡温泉，不可以一下子就去泡温泉，才不会影响血管的收缩。
8、有癌症、白血病的人，不适合浸温泉，因为容易刺激新陈代谢，使身体很快变衰弱。
