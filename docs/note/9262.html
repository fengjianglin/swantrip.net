【天鹅之旅首发】春游曼妙三千漓，沉醉山水之境

游记 客栈民宿 自助游 2019/03/15
大家都知道桂林山水甲天下，而阳朔又汇集了桂林的万千美景。这一次，我要带你来到三千漓山水人文度假区，感受桂林美景精华中的精华。 意盎然的时候，正是踏春出游的极好时刻，阳朔距离广州只有两小时高铁，立马就成为了出门度假的一个好选择。
 “远方”不远，日新月异的交通工具让我们越来越能够到达自己心目中的远方。从桂林出发仅20分钟、广州出发2个小时即可抵达，得天独厚的交通优势，让三千漓成为阳朔非常值得打卡的网红目的地。你不仅能够获得栖居山水间的独特住泊体验，还能获得集吃、住、行、游、购、娱于一体的山水文化浸入式体验。游在阳朔，宿在三千漓，是现代人绝佳的出行选择。 
“三千漓”取名来源于佛经“弱水三千只取一瓢”，我们相信，人们走遍大山大河，阅尽世间风景，终究回归内心深处一片波澜不惊的澄澈净地。三千漓山水人文度假区位于兴坪古镇，坐拥漓江山水绝境，以“渔樵耕读”为文化主题，让你这一趟阳朔之旅不再只是走马观花。
 三千漓着重关注人的内心真实需要，提倡删繁就简、返璞归真的生活哲学。这种简约大气的风格从我们办入住的时候就感受到了，高高耸起的大厅，吊顶的灯饰仿佛一把把油纸伞，十分有禅意。
 在办理完入住以后，我们就前往景区的展示中心了，在这里你可以看到三千漓未来的规划，更直观的感受三千漓的布局，看到未来三千漓还有那么多好玩的，脑海里立马充满了各种幻想呢。
 
 经过三千漓充满艺术气息的旋转楼梯，我们就可以来到景观超好的二层区域，这里也有非常不错的用餐区域。
 
走遍大山大河，阅尽世间风景，终究回归内心深处一片波澜不惊的澄澈净地，这里的山水，有足够的魅力，让世人在走遍千山万水之后，蓦然回首。我想，三千漓就是那个我心目中值得久留目的地吧。
 随着时代的发展，我们甚至连接触纸币的时间都很少了，但这幅全国人民都印象深刻的美景其实就在三千漓可以看到。如果还不过瘾，你甚至可以乘坐直升飞机，你不仅能够以独特视角俯瞰整个兴坪、饱览漓江大转弯的完整景致，用手机拍下的高空大片更能分分钟引爆朋友圈点赞狂潮！
下面这个餐厅环境是绝对要大力推荐的，明亮的空间内可以品茗，亦可以静坐阅读，木质家具给人一种安静沉稳的感觉。
 有山，有水，山可作肴，水可作酒，能品尝一番如此美味，此生无憾!
 就连餐厅外的露台都是一个风景绝美的拍照圣地！
 在这么美的就餐环境里，想必品味起美食来都更有一番滋味了吧，远山青黛，在暖烘烘的室内感受一道道美食，实在是惬意无比。
来到阳朔一定要好好品味当地的美食，比如说：阳朔啤酒鱼、兴坪将军鸭、黄焖甲鱼、干锅黄焖鸡、猪肚焖鸡......看到这里你是不是都已经垂涎三尺了？
 
 用过午餐后，我们就先去入住了。 三千漓山水人文度假区内的三千漓酒店，目前有三个主题开放入住，每一个主题，都是从广西本地的非物质文化遗产中提取各种元素，在漓江最美的山水风景线中，营造不可多得的闲逸时光。
渔家傲主题酒店门前还有两只鱼鹰，花草就放在一艘小木船内，十分的有意思。主题酒店内，先进入眼帘的是一个景观精致的小庭院，连砂石都拜访得格外有意思，层层楼间设置着宽敞的大露台及休闲桌椅，抬眼群山环绕。
 我这一次入住的是渔家傲主题酒店。鸬鹚捕鱼流传千百年，是漓江沿岸独特的渔文化。渔家傲主题运用竹筏、竹篓的元素进行装潢，光影间仿佛穿梭百年，感受得到渔火明时，渔翁忙碌的情景。
 
每一间客房都宣示着独一无二，从装修到家具到每一个细节的设计与装饰物，都充分展示了艺术家的个性与想法，这才是真正让艺术与酒店相得益彰的典范。 
大面积留白的空间设计，再配上一幅充满禅意的画作，让你对旅程的美好充满了想象~
 
静坐在屋内，渔灯常亮之处皆是漓江最淳朴的气息。传统延续，精髓入心，生活在别处的诗意，也在这小小的渔灯里。
 
 【衣帽间】
鸬鹚捕鱼流传千百年，是漓江沿岸独特的渔文化。渔家傲主题运用竹筏、竹篓的元素进行装潢，光影间仿佛穿梭百年，感受得到渔火明时，渔翁忙碌的情景。
此外，渔家傲主题酒店还游泳大阳台的房型，望得见山水 
 参观完渔家傲之后，我们就来到了另一家别具韵味酒店，它叫做天文天纹锦，这里的色调更加沉稳，难怪住这边的朋友都说更喜欢，来到之后果然不同凡响。
天纹锦酒店用色以广西“壮锦”为色彩点缀，在房间软装上合理应用，凸显少数民族的特色，融合在地文化和素雅的民族情调。
 花间集，它以花为主题。房间用淡雅之蓝色宁静怡心，点点花束挑动氛围，眼见之处的设计，都希望通过用心的装点，带来一花一木的细腻。
 
沉浸在花间集的环境中看窗外云卷云舒，烦恼和生活中的一切不如意，都不会打扰你的享受。风月对望，在房间的露台上与朋友、亲人饮茶赏花，看着烟雨漓江，平静，惬意。
 看过了那么多美好的住所，其实，在未来的三千漓，还有独具特色的船屋、泡泡屋，和琴、棋、书、画、诗、酒、花、茶等主题酒店将陆续开放。独特的设计，让酒店不仅仅是酒店，更是让住所与这一派宁静山水融为一体，给你带来一种前所未有的生活体验。参观过酒店之后，我们来到有着各种体验活动的创意街区。
在三千漓，还有一个不可错过的就是它安逸的购物街区，这里不同于阳朔西街的浮躁。这里只有坦然和从容，行走其间，你可能还会看到穿着古风服饰的游客穿梭其中，每一个人面容上的那种喜悦都能感染你。
目前园区设有曹邺诗文展、剪纸展、摄影展三大展览，游客可根据个人喜好免费参观。置身三千漓，你将不再是匆匆而过的旅人，你还可以做一回当地人，与桂林山水的人文底蕴进行一次亲密的接触：做一件阳朔兴坪古镇的草编，绘一把扇面精美的画扇，织一匹花样别致的壮族锦段，手工技艺亲身体验，器物之美就在手作的温度中绽放、盛开。
 首先我们看到的就是特别有味道的团扇馆，据说这里的团扇子还出口国外，一件件都非常精致。
邱广初团扇创始人为国内首位圆竹剖丝团扇工艺的创制者邱广初先生，他的团扇以竹起家，对竹无比钟爱，秉承初心，每个步骤都精益求精，不仅传承濒危的团扇竹艺，又坚守中国美和桂林文化，现逐渐成为国内最具影响力的团扇品牌。
 
团扇又称宫扇、纨扇。从名字便可知道，它一开始是供宫廷贵族使用的。从商代出现到如今，历时千年自不必说。随后传入日本，自成一脉，融合了自己特色成了“和式团扇”，自此从中国团扇里完全脱离出来。如今制作优良的团扇更是因为稀缺和难制作，只能充当小众雅玩。而在三千漓，你不仅将精致的团山买回去，更可以亲手制作，感受diy带来的乐趣。
在咖啡厅休息了一下之后，我们又前往广西民族博物馆了，这里有很多少数民族的展品，对于少数民族文化有着深厚爱好的你不要错过了！
广西民族博物馆首个文创基地落户三千漓，基地重点展示广西12个世居民族的特色服饰、历史文化、人文风情，以及多个系列、数百种的文创产品。
看过了少数民族的服饰，我们也是时候感受汉服的魅力了~
 
最近汉服复兴越来越盛行，无论是在抖音还是在街边，都能看到穿汉服的小姐姐、小哥哥，你是否也心动了，想要试一试汉服，体验一把衣袂飘飘的风韵？
 在古艺文化体验中心，体验一把穿汉服射箭，仿佛穿越到古代的闲逸时光。在古艺文化体验中心，体验一把穿汉服射箭，仿佛穿越到古代的闲逸时光。
很多人都想尝试穿汉服，却不知道在哪购买，在三千漓其实就有非常多汉服可以挑选，无论清纯还是艳丽，你全都可以尝试一遍！
接下来就可以去美美的拍照啦，店内的布景都是古风古色的，相信你无论在哪一个角落，都可以拍出完美的照片。
 
 接下来就是欣赏仙女的时间！这些都是我们的朋友，并不是专业的模特，每个女孩子换上汉服后，瞬间变身为大家闺秀，大方得体。
 除了汉服体验，在三千漓，细字闲抄也是一家大小都适合的休闲方式。对于大人来说，可以通过起笔落墨在心经和历代禅诗的字里行间完成一次修心历程；对小朋友来说，也是一个练字和学习安静的地方。
 三千漓除了有许多安静的、充满历史感的展馆之外，还有许多充满动感的活动，很时候适合一家人来玩。
三千漓的必玩项目！VR深海探险模拟海底环境，极大程度调动起游客的视觉、听觉、触觉，游客仿佛真实遨游于形形色色的神秘海域，与大白鲨和各种缤纷鱼群擦身而过，并经历海底崩塌、逃险等激烈体验，享受到一趟乐趣无穷的海底冒险。
风物集是景区内最大的一间伴手礼特产商店，阳朔有名的花果茶、三花酒、松花糖、辣椒酱、九龙藤蜂蜜、金顺昌糕点都可以在这里找到哦。
 
在三千漓各有特色的店铺里呆了一下午后，我们来到了漓江边上的又一轩餐厅品味美食~
细长线悬挂下的吊灯与壁饰都体现着包厢的精致感，总之，“漓江又一轩”是一个经得起细细咀嚼的工致之处。漓江又一轩店主打“原生态健康美食”，菜品精致，环境优雅，一、二楼可同时容纳600人用餐，不论个人用餐还是团体聚会都是不错的选择。
在晚上，花间集的餐厅也是一个不错的选择，它更为小巧，可以根据需要摆出不同的造型，甚至办一个简约的小party也是非常不错的。比如说我们刚到的那天晚上就来到了花间集餐厅。
在这里开上几支红酒，然后取上最喜欢的食物，和朋友们一起欢快的度过了晚餐时光，在有些微醺的时候，我们看到户外看到美丽的灯光，更觉梦幻了。
 
你看到过阳光下的青山绿水，却未必感受过月色下的漓水山境。夜幕降临，璀璨华灯映照下的三千漓更显五彩缤纷，梦幻绝伦。
 
 看到这么浪漫的灯火，其实三千漓在夜间还有更加好玩的一些活动，那就是放孔明灯。
我们小心翼翼的将孔明灯展开，慢慢点燃底部燃物，灯型渐渐撑开，手捏柔柔面料，默默许愿，虔诚祈福，灯体徐徐滑出手心。
在微风轻拂下冉冉升起，带去美好的祝福，慢慢地飘向遥远的星空，点点落落，与繁星无分彼此，点缀起今晚并不空寂的夜幕。  
 晚安，三千漓
 
桂林米粉早已遍布了全国各地，但是你知道桂林米粉的原始做法么？在三千漓的早上，我们来到桂林米粉文化中心，看到他们今天仍然坚持桂林米粉的古法烹饪，实在大受震撼。
 一碗粉的出炉需要14道工序、200小时的发酵。只有尝过一口，才明白他们这份“守住地道桂林味道”的良苦用心。店内还有手工榨粉的传统器具，可供游客亲自体验。
 鲜榨粉工艺制作出来的米粉，经过自然发酵，饱含许多对人体肠道有益的天然益生菌，能促进肠道健康！这也就是日本说的“酵素粉”。
最后，米粉在巨石和师傅的同时施力下，从榨机口处缓缓挤出，在600斤左右的重力作用之下，进一步加强了米粉的Q弹、劲道。几个人站在大石块上的场面你见过么？也许，桂林米粉光是因为有这道工序，一碗粉便有了更多的意义吧！
最后，我们一起去看了阳朔最美的山水画卷，兴坪码头距离酒店不过几分钟的路程。兴坪古镇侧枕漓江，漓江之水蜿蜒流于该镇西南部，境内江段长达 20多公里。两岸群峰连绵，如剑芒排扦，奇特怪异，万态千姿，重峦叠嶂望不到尽头，不愧是人间仙境呀！
 兴坪的奇山异水吸引了许多中外名人到此一游，其中一位最重要的就是我们敬爱的周总理。 画家徐悲鸿到此写生，也说：“阳朔美景在兴坪。“听到这些介绍，我才知道，原来三千漓展馆里介绍过的历史，就真真切切的发生在这片山水中呀。
 兴坪镇全镇有九马画山、螺丝山、莲花岩、天水寨、灵宝阁、兴坪古街、老寨山友好亭、总统渔村、黄布滩、罗田大岩、西塘天湖、崧坪龙、腾蛟庵、大源林场、大河背风光等景点。在兴坪码头搭乘游船就可以一览无遗啦！
弱水三千，只取一漓，三千漓美的不止是山水，更是生活。跟我一起，享受一场诗意的周末吧！
