【DAY 1】

定了个飞成都的班机，一个人晃晃悠悠的去机场，到成都刚好是中午，拿了行李，联系好出租车，直奔这次下塌的酒店----香格里拉酒店，放好行李办理好入住就直奔第一站宽窄巷子
宽窄巷子是成都最为出名的景点之一，也是很多第一次来成都的童鞋几乎必去的地方，就好像重庆的磁器口一样。宽窄巷子主要由宽巷子、窄巷子和井巷子组成，是由政府打造并扶持的历史文化名城保护街区。过去这里曾经是成都遗留下来的较成规模的清代古街古坊.
现在这里已经是遍布着清末民初风格、青黛砖瓦的四合院落，是成都著名的历史文化名城保护街区之一
宽巷子集中了整个街区最多最完整的老建筑，多数的旧时门脸都保存完好。在这里可以品碗茶，吃正宗川菜，体验老成都的风土人情。窄巷子既有清末民初的建筑，也有早期西式洋楼，是最能体现宽窄美学的地方；以西式饮食文化、艺术休闲、健康生活为主的成都休闲生活品味区。井巷子是典型的民俗成都缩影，除了引人入胜的砖文化墙，也汇聚了成都特色小吃、民俗玩意儿，展现了地道老成都生活的独有味道。
 
缝纫机、茶壶、灶台、留声机、老式的电话等砖文化墙上的这些老物件可以直接将人带进那个年代。
这里是成都的生活缩影，代表了最成都、最市井的民间文化，曲折蜿蜒的小路两边布满特色商店和成都小吃，喝茶、掏耳朵、川剧变脸等活动应有尽有，能够很好地体验巴蜀的人文特色和民俗文化。
时间跟天气的原因，没有在宽窄巷子这边好好逛一下，希望下一次的成都之行能在此品美味，品茗茶，品人生，感受老成都的“慢生活”
 

锦里古街

锦里是成都市著名的步行商业街，号称“西蜀第一街”，被誉为“成都版清明上河图”。传说锦里曾是西蜀历史上最古老、最具有商业气息的街道之一，早在秦汉、三国时期便闻名全国。今天的锦里依托成都武侯祠，以秦汉、三国精神为灵魂，明、清风貌作外表，川西民风、民俗作内容，扩大了三国文化的外延。在这条街上，浓缩了成都生活的精华：有茶楼、客栈、酒楼、酒吧、戏台、风味小吃、工艺品、土特产，充分展现了三国文化和四川民风民俗的独特魅力。 
 
街道两边售卖的都是颇具当地特色的手工艺品，布偶、剪纸、手机壳、泥塑公仔……还有众多老成都手艺人在吹糖画、捏面人。此外，各色美食绝对令你垂涎三尺，凉面、豆花、鲜花甜品等，各具风味
沿街一直走，红灯高挂，旌旗飘摇，小桥流水，亭台掩映，人们或于桥上闲叙，或临窗品茗，漫步其间，优哉游哉的，觉得自己都被成都这休闲之风感染了……
在这里购物，总会有一种令人流连忘返的感觉，而且这里还汇集了四川非常具有特色的知名小吃。如果你很久没有大吃一顿了，那么来四川锦里，不妨疯狂的吃一次吧。
商店里卖的些是筷子、茶叶、灯笼、蚕丝被和土特产。一条街色彩缤纷的花灯和幌子，图的是个热闹。而街上最吸抓眼球的，还是那种生活方式，老的少的在酒吧里嗑瓜子和打牌，喝着外国酒，说的还是绵软成都话。
百年木板门，千载石板路。射弩、织锦、客栈，铜锣声声，追寻逝水年华；刺绣、竹编、当铺，花轿悠悠，勾起尘封记忆。
古街上，人们随着人潮流动，路边的各种布置无不显示一种精巧、一份精心。
就这样嬉闹着松弛地在锦里闲逛，怀旧的人情感有了出口，爱吃的人满足了口腹之欲，三国迷在这里充分感受三国文化，其实人们更多的是体验一种传统文化，领悟生活的多元。锦里表现的是现实人间的景象，内核却是千年的文化。 

【抱汁汁】

当花甲海鲜和火锅底料相碰撞，这简直就是为咱们吃货量身打造的专属美食啊有没有
靠这单品就吸引了无数的吃货排队去吃，这家店就是春熙路的抱汁汁，它的店面并不大，甚至可以用mini来形容，可是喜欢吃的人真的是非常的多。
它拥有的头衔非常多：米其林二星大厨、腾讯美食推荐、新浪美食推荐......光看到这里，都会忍不住让人驻足品尝。
虽说是靠单品打天下，但是抱汁汁花甲的味型还是非常丰富，专属四川重口味好吃嘴的老火锅味和鲜椒口味，吃起来相当过瘾，蒜香和糖醋味，老少皆宜。除了花甲之外，还可以跟鲍鱼拼，跟大虾拼，跟圣子拼，跟鱿鱼拼，吃法多多
 吃海鲜嘛重点就在一个鲜上，抱汁汁的花甲个头大，采用青岛海域新鲜捕捞的海鲜空运到成都，到达成都的时候依然鲜活无比，这样的海鲜能保持其自然的鲜味，吃起来口感自上乘。
 
主厨亲自调制出来的卤水，将所有海鲜全都过一遍，然后搅拌均匀，即能保持海鲜的本质口感，又有汤汁浓郁丰满的口感，两者混合搅拌之后，海鲜很入味，抱汁汁的每一份菜品都是客人现点现加工，这样才能保证最佳的口感。
做好了打包拎起来便可以继续逛街，边走边吃很过瘾，不要太幸福！
 营业时间：11：00-22：30
地址：春熙路联升巷10号
 

【爱妻牛排火锅】

因为爱所以让原本的平淡的生活变得甜蜜
从这家点起的一个虐狗的名字【爱妻牛排火锅】就能看到爱情最美好的模样。
老板开店的初衷是因为老板的老婆怀孕了，想吃火锅，为了让老婆随时吃到他爱的火锅，老板便开创了这个超级好吃的牛排火锅，赤裸裸的秀恩爱以及虐狗，
傍晚的时候小伙伴们一起去爱妻牛排吃饭的时候，成都的天气有点凉，衣服穿的有点少，感觉有些冷，店员贴心的帮我们煮了可乐姜汤，好有爱心好温暖。
底料采用最好的牛油和传统黄菜籽油，来爱妻的朋友们，要提醒大家
一定要提前一天为最佳，因为这家店采用的是纯新鲜耗牛，新鲜的耗需现做现煮为佳，才能达到牛排提前煨至入味，他家的牛排都以斤卖，
肉也非常实在一大坨牛肉骨肉相连，一般提前会放入锅里煮好（这就是客人要提前预定的原因）。
梅林午餐肉就是除牛排外第二道看家菜了。纯手工制作的午餐肉肉质紧致，香气饱满，一片片烫着吃，也要把握着时间才能有更实在鲜嫩的口感体验。食材不稀奇，讲究了吃法才算觅得了火锅真经，才够刺激。如果说，热干面要配蛋酒，片儿川要配油渣，驴肉火烧要配羊汤……那么，正宗的四川火锅一定要配上一碗蛋炒饭。饭粒要颗颗有型，蛋花要嫩而不碎，一点点香葱就是提香法宝。
 
爱妻牛排火锅的配菜，例如川式火腿、腊肠、牛百叶、蕨菜......因为锅底实在是太好吃了，所以导致下锅的配菜全部都让人食不停口。美味红糖锅盔 美味的让我一直停不了筷子。
 
 营业时间：11：30-13：30，16：30-20：00
地址：十二南街2号附2号（四川音乐学院附中对面，地铁3号线B出口500米）

【四川美术馆】

四川美术馆是 成渝地区，乃至西南地区比较大的综合性美术馆。旧馆建于1991年5月，地处中国西部政治、经济、文化中心成都市中心地段，地理位置十分优越。它位于天府广场西侧人民西路20号，南临蜀都大道，东接人民南路，毗邻人民公园，省展览馆，市体育场及省文物大楼，是一个集文化、商业、休闲娱乐的黄金地段。
 
美术馆外周边绿草坪上摆放了6座风格各异的雕塑作品，已成为成都乃至四川的文化标志性建筑，，整个建筑外墙均由黑灰色大理石和弧形玻璃幕墙组成从正面望去发现很像一个睡着的“如意”造型。进去参观是不需要门票的，凭身份证换票进去就可以
 
我们去的那天美术馆正在进行书画展览，随性欣赏了一下，画作都很大师范，不过因为没有涉及到书画领域，虽然是名家，自己孤陋寡闻倒也并不认识。书画展也只是某一时期的展览，在来美术馆之前最好查查主题，选择自己喜欢的主题前来参观。
 
在美术馆里拍照，尽量不要开闪光灯，避免影响其他欣赏画作的人，动作也要尽量地轻，有禁止拍照的展厅也不可以拍照。
地铁2号线在人民公园站下车步行即到
开放时间：周二至周日 09:00-17:00周一闭馆

【人民公园】

在成都，最能展现成都人慢生活的悠闲状态的就是公园，而说到公园，最具有代表性的便是人民公园。这是成都人喝茶悠闲爱去的街心公园，是一座集文化文物与休闲娱乐于一体的综合性园林。
公园内由金水溪、盆景园等自然景观组成，沿岸种植的多种植被和溪流相辅相成。
金鱼岛和永聚茶社融为一体，打造赏鱼与品茗的文化场所，目前岛上保有十多种珍品金鱼。
公园内最著名的要数鹤鸣茶社，近百年来风格不改，标配是竹椅、矮桌、盖碗茶，最拿手的茶是“香片”。 除了喝杯清茶，还可以看冲茶表演；或者找张躺椅睡起，喊个采耳师傅来掏耳朵；打几圈麻将或打牌……这就是真正意义上的"偷得浮生半日闲"吧，
正好赶上菊展。菊花的品种据说上千种，颜色也丰富多彩，甚至还有七彩菊花，给秋冬装点了亮丽的妆容。
在人民公园里，你会不自觉地感受到成都  的“慢”，三三两两的人群在金黄的银杏树下漫不经心地走着;公园的长凳上坐着人，有一句没一句地闲聊着;，慢慢地溜达，即使一句话也不说，看着鱼池  里畅游的鱼，看着树上的金黄，看着地上的清菊，有时候停下来，慢下来，，思考一下，体会到生活的美好，而是为了行走得更远，更好。
 

【张烤鸭】

一说起烤鸭可能大家的第一反应就是全聚德，但是老一辈的成都人都知道
有一家店才是真正属于成都的烤鸭味道，它的名字就是 【张烤鸭】
传统的成都冒烤鸭烤制的方式跟北京烤鸭最大的不同，就在于“外烤而内煮”：烤之前在鸭腹内放入十几种秘制香料，用本地的小黄姜和大葱再加成都人最爱的郫县豆瓣，然后灌水。：土灶里的温度烧到一定程度之后，熄灭柴火，用阴火来烤制鸭胚。在300度的高温下，鸭肚子里的水和香料翻滚沸腾，鸭子从外到里，从里到外地成熟，同时去腥增香。张烤鸭的烤鸭，选用的鸭都是最优秀的等级，刚出炉的烤鸭全身都是「枣红色，油光润泽的」，放入口中，飘着淡淡的木材的清香，皮质清脆，肉质鲜嫩，回味不尽。
烤鸭的土灶下面还有一个机关，放置着一个黑砂窝子，正好用来接烤制的过程中，鸭肚子里的水和鸭子皮下融化的油脂。这些收集而来的鸭油和汤汁再加上土鸡、土鸭和棒子骨熬出来的就是冒烤鸭的精华——卤水。成都冒烤鸭一般只烤九成熟，最后的熟成必须靠卤水来冒制。烤好的鸭子皮酥肉嫩，滚烫的卤水馥郁浓香，两者完美结合才能成就一碗地道而传统的成都冒烤鸭。一入口更是鲜嫩多汁
［片皮鸭］是北京烤鸭的吃法，师傅就在桌子旁边现场片鸭子，片好的烤鸭趁热包着面皮夹着配菜沾上面酱卷起来吃，又香又嫩，味道特别地道。
坛子肉窝窝头不同于其他窝窝头的配菜，张烤鸭选用的是安岳的坛子肉，这种肉本就很有意思。在安岳当地，奶奶辈儿的能干人会把新鲜的五花肉装在坛子里，配合着酸菜秘制发酵，几个月之后再取出来招待客人，五花肉在腌制发酵的过程中产生的益生菌，让肉质发酵得肥而不腻，馥郁浓香。
 营业时间：11：00-14：30，17：30-21：30
地址：大业路古卧龙桥街9号附17号（距地铁1/2号线天府广场站出口步行730米）
 

【香格里拉大酒店】

 这次来成都旅行选择下榻的酒店——香格里拉大酒店，之所以选择香格里拉酒店，酒店的地理位置很好，交通非常便利，从双流国际机场打车25分钟左右。挨着兰桂坊，旁边有酒吧一条街，可以15分钟慢悠悠的闲逛到太古里。距离宽窄巷子也很近，打车15分钟左右。
二层的酒店大堂宽敞明亮通透，色调明快，配上白色水晶灯饰，整体富丽堂皇，奢华典雅 。
这次选择入住的是豪华阁江景房，可以直接在35层豪华阁大堂办理快速入住手续，不用在二楼大堂排队等，方便快捷，大大的提高效率节省时间。
暖调色系的客房很温馨舒适，配有浴缸和干湿分离的卫生间，免费高速无线网络覆盖客房，临着锦江，拉开窗帘就能欣赏到美丽的江景。
豪华阁位于酒店的36层，只针对行政楼层以上级别的客人开放，早餐丰富，用餐环境安静、舒适，招牌中餐厅香宫，也有很多不同菜式可以选择，有广东小伙伴喜欢的经典粤菜，也有地道新派川菜。​​
酒店拥有四个餐厅和酒吧，蓉咖啡西餐厅拥有9个风格迥异的开放式厨房，世界各地的国际美食汇聚成饕餮大餐；招牌中餐厅香宫，提供传统经典粤菜及地道新派川菜，并设有包间供商务宴请，秋蟹正肥，酒店特色玫瑰盐焗大闸蟹非常值得一试；Mooney’s 酒吧是夜生活的开始，现场乐队弹奏着悦耳明快的音乐，舞步和着节拍，音乐伴着磁性嗓音，点燃了一屋子人跳舞的冲动和激情。此刻，可以在舞池大展身姿，也可以点上一杯鸡尾酒，点上一桌子的食物，静静地吃，默默地喝，看着人笑，看着人闹，仿佛世界和你有关，仿佛世界和你无关。
4楼是我很喜欢去的地方，一侧可以做SPA，一侧可以运动健身。环境非常不错，健身设备也很齐全，除了各种类型的健身器材，还有室内游泳池、桑拿、蒸汽浴、漩涡浴以及按摩设施等，每个健身器材的旁边都细心的放置了毛巾跟瓶装饮用水，贴心的准备了消毒喷剂。运动完之后跟小伙伴一起去健身房对面的CHI SPA预约一个活力香薰按摩，放松肌肉，
CHI是香格里拉的自由品牌，结合中国传统的养生理念进行身体护理。有11间宽敞舒适的私人护理套房每个房间都是private rom ,私密感觉很强，按摩之前可以挑选你喜欢的精油，理疗师按摩时手法非常细致，照顾比较周到，按摩之前会询问身体哪个部分需要特别留意。按摩的时候理疗师会重点按
听着舒缓的音乐，跟着理疗师的手法慢慢放松，整个过程下来，神清气爽。在香格里拉的住店客人可以享受9折优惠。

【关于交通】

成都市区这几天都游玩，都是地铁+共享单车+打车，这几种方式交替进行了地铁目前已开通一号、二号、三号、四号、七号以及十号线，可直达市区部分热门景点以及火车站、飞机场等地。公交系统也很发达，绝大多数景点已经地方都有公交车可达，需要自备零钱，共享单车随处可见，市区附近的景区可以踩单车会更快捷一点，宽窄巷子、锦里景区都有景区直通车，想去熊猫基地、青城山、都江堰、峨眉山等景点都可在这些地方乘坐
