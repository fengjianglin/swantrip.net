#天鹅之旅首发#【天柱山】不只是山，这里总有一种美让你流连

游记 五一 家庭游 摄影 登山 自助游 2019/05/15

前言

在安徽西南，有一座小城，它素有“皖国古都、安徽之源”的美誉。这座城，以山为名，承载着无数的历史与故事。
-
它是中国禅宗文化发祥地之一，有1500多年的三祖禅寺；
是“千年陶乡”，有六千年历史痘姆古陶非物质文化遗产传习基地；
是三国佳丽大乔、小乔的生长地，自东汉至南宋先后为州、郡、府治所近800年；
它还是京剧的发源地，长篇叙事诗《孔雀东南飞》的故事就发生在这里；
小城有被誉为天然艺术博物馆的山谷流泉文化园，园中藏着数百位名人石刻；
还有峰雄水秀、让世界瞩目的地质公园——天柱山。
-
这座城，名为：潜山。

DAY1

山谷流泉文化园

武汉到潜山全程二百六十多公里，两座城市之间有大巴车直达，如果选择火车出行，从安庆中转也很方便。
这次潜山之行，第一站去了山谷流泉文化园。
山谷流泉文化园在安徽省潜山市天柱山镇风景村内，它位于山谷寺佛殿后面。山谷流泉指的是山谷寺西侧的一条山溪，它发源于及第庵的左山，曲折潜流至凤凰山下，上游名潺潺溪，因水声潺潺而得名；中段因有一块形似卧牛饮水的巨石横于溪间，所以被称为石牛溪。下游则是山谷流泉，它自北而南，迳入潜河。
这里石壁陡峭，松萝丛覆，空气清新，泉流其间，宛转激回，山溪两岸摩崖与溪间巨石上石刻众多，其中以山谷流泉处最为密集，也最为精彩。
虽然经岁月变革，自然风化以及早期损毁，有一部分已不复存在，但现在山谷流泉现存的石刻，还是存有从唐、宋、元、明、清，直到民国历朝历代的各种石刻400余方，数量之大，保存之完好，是安徽之最，在全国也排得上前茅。
草书、隶书、楷书、行书、篆书、真体，摩崖石刻上各种文字题刻，书法纷呈。
内容从天文地理到经济政治、军事、人文，涉及方方面面，琳琅满目。
题刻的作者数量也颇多，经考证的就不下百人。像唐代的李德修、李翱，宋代的王安石、黄庭坚、苏东坡，张同之，明朝胡缵宗，清代张楷等名宦大家都题字于崖谷之上。
“水无心而宛转，山有色而环围。
穷幽深而不尽，坐石上以忘归。”
据传，王安石在潜山任官期间来到山谷流泉，拥火夜游，观字听泉待了很久，次日又再赴此地写下了这首著名诗篇。
山谷流泉中以唐代石刻最早，又以宋代石刻数量最多，诗词歌赋、纪事题名，各具特色，各有千秋。
如今的山谷流泉，是一处珍贵的书法艺术宝库，更是一座天然的史料档案馆，沿着幽静的小路，边赏石刻，边呼吸负氧充足的上好空气，享受泉边的清爽，实在悠哉美哉。

痘姆古陶

从山谷流泉文化园出来，去探访了痘姆古陶非物质文化遗产传习基地。
中华文化博大精深，悠悠几千年的历史长河中，手工业的地位不容小觑，很多传统的老手艺都如同瑰宝，熠熠生辉。随着工业越来越发达，现代化进展越来越快，不少传统手工技艺逐渐被机械所替代，慢慢退出历史舞台，那些被传承下来的手工艺就更加显得弥足珍贵，这次在安痘姆乡邂逅的痘姆古陶就是传统手工艺中的一块宝藏。
潜山市痘姆乡有“千年陶乡”之称，它是痘姆古陶的发源地。经专家考证，痘姆陶器手工制作技艺与薛家岗遗址出土的新石器时代晚期陶器一脉相承，同根同源，距今已有六千年历史，这项传统手工技艺也被列于“国家非物质文化遗产”。
痘姆古陶素而不拙，简而生雅，色中藏色，不管是制作过程还是陶器本身，都美的让人生叹。
选泥，采泥，晒土，搅拌，过滤，垒索，和泥，制坯，刻画，上釉，晾干，进窑，烧制，直到变成成品，每一件痘姆古陶的完成都需要足足十几道工序，耗时25天以上，还要中和季节，温湿度，光照等条件，才能保证其品质，可以说是十分的艰辛了。
潜山痘姆古陶非物质文化遗产传习基地上有一座长近百米、形似卧龙的古窑，当地人称其为痘姆龙窑。
走进龙窑内部，两侧窑洞内壁上松木焚烧留下的油脂在窑洞的砖石上烙下厚厚的、黑色的印迹。龙窑很大，一次可以烧制三千多个大小不同的陶坯，据说每次开火都会举办一次小小的点文仪式，可惜我们没赶上，留点遗憾，下次再来。
基地上正在制陶的大师傅。
痘姆古陶的制作过程中除了水以外，什么都不加，全凭师傅们的经验和手艺来成就，所以我们在传习基地看到的每一个师傅都聚精会神，精力集中到让手拿相机的我们都不敢随意咔嚓，怕打扰到师傅们手上的艺术。
基地里四处走走看看，各种陶器随处可见，缸，盆，坛，钵，胎质坚硬，纹理简朴，釉色光亮，外观大气、质地厚重、气质沉稳。
因陶得名，凭古生辉，希望痘姆古陶这个传承了几千年的技艺越来越好。

潜山天柱山王府国际大酒店

当晚入住于潜山市天柱山王府国际大酒店。
夜幕中的潜山市天柱山王府国际大酒店静谧、时尚。
通往房间的过道宽敞亮堂。
房间装饰简约大气中混着时尚感。
床品舒适，床垫软硬度适中，睡眠感受极佳。
这一顿，在天柱山王府国际大酒店的中餐厅用餐。
作为潜山数一数二的酒店，环境方面自不用说，菜式也很丰富，我们人多，有机会多叫了些菜。有酒店招牌菜式王府宫廷雁鹅、国宴油条豆腐之类，也有像红烧臭桂鱼、锅仔毛草菇氽肉这样的当地特色美食，有时令菜式薄脆田螺肉、韭香土豆仔，亦有滋补菜品浓汤小米鲍、飘香牦牛骨等等，厨师手艺颇佳，这一顿，一大桌人个个都吃得很满足。

二乔公园

饭后去二乔公园小逛。
《三国志》中记录二乔故里出现的地点仅一处，即皖公山下的皖城，潜山在周代为皖国封地，都城被称为皖城，也就是今天天柱山下的潜山市区，于是潜山有二乔故里之称。
二乔公园是潜山的一张名片，也是潜山第一个免费向市民开放的综合性公园。
公园很大，布置也颇有味道，湿地、林木、水榭、廊桥、观景台、还有古胭脂井，据说，在这里可以触摸到“古皖文化”。
胭脂井是二乔故里的招牌。《安庆府志》载：“汉末乔公有二女居此，皆国色，孙策克皖娶大乔，周瑜娶小乔。二女以残脂粉投井中后，井水带脂粉色，故名胭脂井”。
乔家祖居被掩埋多年之后，到1995年被潜山文物部门发掘，使千年古井重见天日，并公布为重点文物保护单位。
廊桥水榭，幽静的园区，站在小假山上，听着流水潺潺的声音，看着周边的虫鸣鸟叫，公园对面，是现代的高楼大厦，古老与现代在这里完美交融。

DAY2

天柱山

前面有提到过，潜山这座城是以山为名，所以除了是市名外，潜山其实还是一座山的名字。这座山还有别名叫万岁山，也叫皖山，安徽省简称中那个”皖“字就出自此处，这些名字说起来可能知道的人不多，但要说到这座山的大名估计就无人不晓了，此山如今被称为：“天柱山”。
天柱山海拔1488.8米，高耸挺立，因主峰如“擎天一柱”而得名。它是安徽三大名山之一，与黄山、九华山齐名，有着“江淮第一山”的美誉。翻开历史回头看，早在西汉时，汉武帝刘彻曾登临安徽天柱山后，就将其封为“南岳”。
天柱山拥有独特的自然景观，它的景色之雄奇可以和黄山相媲美，山上群峰连绵风光旖旎又有庐山之幽秀，百态奇石又兼华山之险峻。山上飞瀑流泉，峡谷幽洞，险关古寨，天气适宜时，更有云海，日出，雾淞，佛光等令人为之惊叹的盛世景像。
天柱山全称是”天柱山世界地质公园“，整个景区分东关、西关两个部分，天柱山第一日，选择了西关路线。
对于登山爱好者而言一般徒步上山约需2.5小时左右，我们选择了更适合大众人群的，更为舒适的方式，从大龙窝索道乘缆车上山，省时省力的同时，还能从更高的角度欣赏沿途的美景。
到的当天，刚下过雨，雨后的天柱山云雾缭绕，仿若仙境，从索道上山开始，就有了神仙般的视角，云在身边绕，仿若触手可及。
奇峰出奇云，秀水含秀气。
青冥皖公山，巉绝称人意。
象鼻石
戴着”帽子“的飞来石是飞来峰的特征。
飞来峰
在这里，奇形怪状的岩石比较常见，正等着想象力丰富的你去给它们取个动听的名字......
伫立在神秘谷谷口的一石像，人们敬为“皖公神像”。
皖公是春秋时期皖国的仁慈君主，他执政期间，关心百姓冷暖疾苦，体察民情，深得拥戴，人们尊称他为“皖公”。为弘扬皖伯大夫治皖有方，清明德政，天柱山的别称”皖山“和为“皖公山”就因他而得。据说，皖公逝后， 他的神灵真的在天柱山作化神像，他终归踪皖山皖地这方热土，默默呵护着皖山的安宁祥和。
破顽石而生的天柱松，扎根在绝壁石缝中，让人敬仰。
天柱峰是整座天柱山的主峰，它海拔1489.8米，雄绝险峻，势如擎天一柱，天柱山因此而得名。唐代大诗人白居易在《题天柱峰》的诗中写道：“天柱一峰擎日月，洞门千仞锁云雷。”
峰顶刻着“中天一柱”“孤立擎霄”八个大字，每字五米见方，气势磅礴。
蓬莱峰，西、南、北三面崖陡壁削，巨壑万仞；峰腰时有烟云缭绕，峰顶时隐时现，宛若蓬莱仙境。
青龙峰，人称“青龙背”，峰顶之青苍石冈，狭长陡险，似青龙卧伏；峰之西巅，怪石高翘，有如青龙昂首，故名青龙峰。

炼丹湖

很多人都知道天柱山山峰峻美，岩石隽秀，然而，很多人却不晓得这里的水也颇有看头。在天柱山良药坪，藏着一个如明珠似翡翠的高山平湖，它海拔1100米，面积有34亩，蓄水量多达8万立米，是我国第三大高山平湖，在全国各地一众名山中，能与天山天池和长白山天池媲美。
这个地方原名为良药坪，因汉末明道左慈曾在此起炉炼丹而得名“炼丹湖”。左慈当年的炼丹房和炼丹炉还在。
从炼丹台上看出去的风景，天柱山西关群峰在湖畔生出新的意境。
而炼丹湖，湖平如镜，水面上烟云袅袅，波光山影，幽明幻变，蓝天白云、群峰苍崖倒映其中，让湖面显得格外的静美雅致。
绿水青山中那片片红颜，是正开着的高山杜鹃，一丛丛、一片片，给这样的山水增添了别样的生机。

天柱山炼丹湖度假酒店

这一夜，入住炼丹湖畔的炼丹湖度假酒店。
酒店依山傍水，布局错落有致，看得出来，在建造设计过程中应该是花了不少心思的，放眼望去，酒店的建筑和周边的美景很好的融合在了一起。
整个酒店由水天居客房，木屋别墅和翡翠楼餐厅三部分组成。
客房内饰是大气的金色调，美而不俗。房间宽敞明亮，各种用品用具一应俱全，床品舒适，拉开窗帘，窗户外面就是炼丹湖美景。
这一顿，吃的是当地特色菜式，菜式简单，味道靠谱，一顿饱餐后回房间美美的睡上一觉，第二天去爬东关。

DAY3

天柱山（东关）

炼丹湖畔的东西关界碑。
天柱山素以“西关秀，东关雄，南关神秘北关幽”闻名，和西关相比起来，天柱山东关地势更陡峭险峻，上山时对体力的挑战略大。但去了就会发现，穿梭在云端的惬意，以及满眼高山峡谷的壮丽唯美，能将所有的辛苦都幻化成“值得”二字。
沿着石级缓缓而行，转一个弯，穿一道岭，面前就又是一番新景色。
崖陡壁削，巨壑万仞。
东关的美，不张扬，不妖媚，美的空灵，美的静雅脱俗。
站在高空放眼望去，山峰之间不时有烟云缭绕，时隐时现，宛若仙境。在这里，每一眼都是空前的视觉享受，每一秒，都会被大自然的鬼斧神工所震憾。
顺便说一声，天柱山东关正在修建一条悬挂式高空栈道，待它修成，以后赏东关美景就不用翻山越岭了，不久的一天，老人小孩都可以赏这样的美景。

天仙峡

潜山的一处广告牌上说：好玩的山是天柱山，好玩的水在天仙峡。
天仙峡地处天柱山腹地，七仙女和董永的爱情也和这里颇有渊源，影视剧《天仙配》曾在此取景。天仙峡内峡谷清幽，四季有花海，水生态资源丰富。
景区山谷竹林中有长达1688米的玻璃水滑道，惊险刺激。
还有网红款呐喊喷泉，彩虹滑道等项目，据说不久，7D高空玻璃桥也要开始迎客了，期待！

可以先坐高铁到合肥或安庆站，再转汽车前往潜山县。
亦或从合肥转火车到潜山县（天柱山站）。
上一篇 东方夏威夷，玩转舌尖上的三亚 下一篇 周末轻奢游澳门，畅享食住行乐购
