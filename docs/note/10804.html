偷得浮生半日闲，与张家界的两日邂逅

游记 摄影 精品酒店 自然奇观 行程规划 2018/11/03
‘偷得浮生半日闲，哪得云雾半绕山’，经历一周的繁重工作，在周末时候享受没有工作的繁忙，没有人流的拥挤，游览与大山之间。和奇峰险地相约周末张家界，美妙乐于难以言表。
缩小的人间仙境，掌中的梦幻奇景，一遍遍的在眼前浮现至今难忘，真的是奇妙的体验。站在崖边，俯仰于天地之间，真的有种征服天地的豪情。

古香自然溪布街

这次趁着周末，从成都飞到了张家界。到达机场后，酒店的工作人员早早就来机场等候接机了。我们住的酒店叫云栖山房歇心，就在溪布街里面，晚上到了酒店就开启了逛吃逛吃模式。
几个人在那里一边欣赏夜色，一边尝着特色小吃，美滋滋的生活。整个溪布街都是那诱人的香味，让人垂涎欲滴呀!
夜幕走在热闹的古镇里，古典与现代的结合，越加的迷人。几人有说有笑，拍照留影，忙着不停。
干净街道两边古韵十足的溪布街建筑，在橘黄色的灯光下，太适合放下所有的情愫，做一个时光的聆听者，在有月的夜晚里慢慢的，借着风轻轻的倾述着，漫步在青石板路上，真正寻得着小桥流水人家的古朴。
长穗灯笼三层高，把酒临风叙往日。在溪布街找了间清吧，坐在靠溪的一边，和朋友一起喝了杯酒，聊起了过往，好生惬意。
溪布街，一个充满遐想的民族殿堂，有着浓厚的土家风俗人情。路过一些名特产工坊，里面的店员身穿土家民族服饰，进去逛了一圈，买了点当地的手信，想把这份异乡情愫寄往归途。
入秋的张家界还是比较凉的，人比较少。夜晚，走在空荡的街上，独揽夜色中的溪布街，倒是别有一番趣味

展开的画卷，十里画廊

十里画廊位于索溪峪景区，是索溪峪景区的精华之一。据说是因为谷内风景如画，远远望去犹如舒展数十里的山水画卷，因此被称为“十里画廊”
早上在酒店吃完早餐便出发前往张家界森林公园，从酒店出来直接打出租车到武陵源景区标志门，记得上车的时候要问好师傅能不能打表。进入景区之后还要乘坐景区的大巴到达十里画廊。
走完十里画廊并不算太远，不过为了省点体力留给后面的行程，我们上行的时候徒步，下行的时候坐的小火车，这样既可以边走边看风景，也不会太累。
十里画廊向远处眺望，能看到那高大雄伟的山峰耸立在远处。即使隔着很远，还依然能看到它们气壮景色，不由得惊叹，大自然真的是鬼斧神工。
形态各异的山峰相对而视，各自诉说自己的故事。真的是‘远近高低各不同’，各具形态，生动活泼，活灵活现。
每一座山峰都有着自己的名字，每一个名字都有着属于自己的故事。他们以不同的故事，让人体会到当初创造它们的自然的功力。我们几个人每看到一个，都会发出惊叹声。
宽阔的山谷像个敞开怀抱的老人，迎接着所有到来的人们。希望里面的景色，能够有人去欣赏，能够有人去品味。
高大耸立的山峰直上云宵，要是云雾天的时候，云雾环绕，到是真有点像孙猴子插入天空的金箍棒。
走完十里画廊，我们下行的时候坐上山林间开辟出来的小火车。在路上，享受十里画廊的峡谷风光，到是再美不过。

幽静安宁金鞭溪

穿过十里画廊，走进金鞭溪，远远就看到碑石。
清澈的溪水流下，哗哗的水声响彻在宁静的世界。水花四溅，在空中形成了一条珍珠项链，清新剔透。溅起的水珠落下，泛起点点涟漪，晶莹多彩。身边是绿树白花，远处是层层叠叠的山峰，真的太享受了~
长长的走廊，沿着小溪边搭建。让我们能更好亲近这地方，亲近自然世界。
路上还有一些石头组成的路，如棋子的石头，一不小心都有可能掉到溪水中。到是有几分小危险。可惜现在有些凉，没人在溪水中游玩。
穿行宁静的溪边，带给我们一份幽静和清凉。澄净的溪水，游弋的鱼儿，美艳的花草，让心也透明了起来。千里汇聚在一处，‘千里相会’的故事，让我们在这里停留不少时间。

奇峰千石袁家界

袁家界位于国家森林公园北部，被称为神秘的“潘多拉世界”。
从金鞭溪坐观光车来到百龙天梯，百龙天梯是张家界的一道宏伟靓丽的风景线。百龙天梯垂直高差335米，与山一样高的户外电梯，高耸入云，有着欲与天公试比高之感。，只有乘上电梯才可以到达袁家界和天子山。
排队等待上百龙天梯的人非常多，都想要体验一把在空中，望向外面的感觉。
层层叠叠的石峰，远远望去，那是异常壮观。挺立的姿势，有将、有兵，有鼓、有旗，有车、有马，犹如聚集在演兵场，威武雄壮严阵待发的军队。
石峰上的柱子太多，也是没办法一一记住。到是对它们挺立的姿势，我还是非常深刻。
每一处的石柱看上去相似，在仔细看，到是有些不同。一个个中间都有着自己独特风景，需要去寻觅。
近距离下的石柱上面郁郁葱葱，陡峭异常。恐怕也只有天地之间的飞鸟才能在它们上面停留。
细窄的石柱和旁边石柱没任何支撑，可依然挺立着。细细的地方，真的有些怕支撑不住。这或许就是它们生存的平衡之美。
乾坤柱到是有几分定乾坤的味道，也是《阿凡达》中那座悬浮的“哈利路亚山”取景地，面对这片峰林，任何人都会被震撼，被征服！
一个个的石柱高矮胖瘦长短不一，千姿百态，各不相同。不管是从什么地方看去，都会有不一样的味道，需要你去体会，琢磨。
横跨在两山之间的天下第一桥，不得不感叹大自然的鬼斧神工。
中午在这里的肯德基吃的午饭，点了一份泡椒鸡肉饭，微辣。荤素相间的味道，是我在肯德基吃过最好吃的一顿，特别满足。累了半天吃上一顿饱饭，才有精力接下来的行程。

巍峨石峰天子山

吃过午饭之后，继续坐大巴到天子山，大概40分钟的车程，刚好在车上睡了一会儿。
御笔峰数十座错落有致的秀峰突起，要冲蓝天，有种御笔悬挂的景致。
在御笔峰斜对面的仙女献花，石峰俏丽可爱，姿态妖娆。山上灌木郁郁葱葱的生长着，风一吹犹如低头要像世间撒花的仙女。
从“仙女献花”一景向前走几十步，回眺右侧深谷，只见石峰林立，其中有一峰恰似骏马一匹，骏马旁边有一怪石如武士一般，正握紧缰绳在驯马，故名“武士驯马”。
从另外一处看，真的有种骏马嘶鸣，武士叫吼的味道。当初起名字的人，肯定是高人。不然怎么能想到这么契合的名字。
从天柱山下来坐索道下山，结束一天的行程。
穿梭于山峰之间的索道，犹如漫步云端，更加近距离看到旁边的山峰上的景色。
繁忙的索道从山上飞向山脚，那一座座山峰犹如不舍的朋友，慢慢消失在我们的视线当中。

绚丽多彩黄龙洞

第二天再酒店里面吃过早饭，我们就出发坐出租车到达黄龙洞。
以高阔的洞天、幽深的暗河、密集的石笋而出名的黄龙洞，面积很大，美轮美奂的景色，那是异常迷人。
从大门口道洞口很长一段路，旁边亭台阁楼，连环水车，到是也有些看点。只是对于期待黄龙洞的我们，到是显得有些兴趣索然。
进入黄龙洞就有景区专门的工作人员带领，分批进入洞里。刚走进洞，我们就有种走到另外一种世界，恍如走入地下龙宫的错觉。
​
走一段路后就可以坐船游览，还好是分批进来的，所以我们不用等待，直接就坐上了。
坐在船上欣赏着溶洞又是一种别样的感觉，灯光倒映在水里非常的美妙。
刚走到里面，就被里面五彩缤纷的世界震撼。一个个石钟乳在灯光下，散发出自己的魅力。
各种不同的石钟乳有大有小，在不同的灯光下，五彩斑斓，异常好看。拥有着几万年历史的见证者，在我们面前揭开面纱。
密密麻麻的石笋石柱，看上去非常壮观。
和石峰林一样，这里的石笋石柱也有着自己的名字，每个上面也有着自己的故事。千姿百态的形状，自然有自己的历史痕迹需要对别人讲述。
定海神针是黄龙洞里最高的石笋，非常的纤细，所以年被投保了一亿人民币，之后一下就火了，牛逼的营销手段不得不服。
一片片石头翻卷，像是一个个丘陵。大自然的鬼斧神工还真的是厉害，想要看什么的时候，就随手把这千丘田放在眼前，让世人观赏。

刺激的张家界大峡谷

离开黄龙洞之后，我们就包车前往张家界大峡谷。张家界大峡谷集山、水、洞于一身，是张家界地貌的博物馆。峡谷里植被茂盛，空气清新，飞流瀑布、小溪泉水随处可见，犹如一个天然大氧吧。
呼吸着新鲜空气，望着远处的山峰，我们踏上了著名的玻璃桥上。(不过这里要求，上玻璃桥不能携带相机，所以我就用手机拍的)
远远望去，玻璃桥上不少人站在上面。
踏上玻璃桥，依着群山，留下美美的照片留作纪念。打卡玻璃桥的任务，也算是达成，可以放肆玩一会儿。
两天的旅行生活就在玻璃桥之后画上了完美的记号，也是一个美好的周末时光。
见过石峰林，游过黄龙洞，踏上了玻璃桥，这个周末绝对是不虚此行。能在一周工作结束之后有这么放松的体验，自然是最好的生活方式。

攻略

关于酒店：

这次到张家界是冲着森林公园和网红玻璃桥来的，因为张家界森林公园在武陵源，而玻璃桥在慈利县，所以我们选择住在了武陵源，这样离景区比较近搭车也方便。
在网上订酒店时，看到这家酒店的名字就被吸引住了，“张家界云栖山房·歇心禅意酒店”很有诗意的感觉，来到发现果不其然。酒店外观以白色为主调，装修风格像是日式和新中式的结合，简约素雅。
走进里面，映入眼帘的是一席茶桌，阵阵淡淡的原木香，身上疲倦也是跟着少了些许，不由得被它吸引住，驻足停留。
一进去就有热毛巾和甜品迎接，十月份的张家界天气还是有点冷的，热毛巾暖手很贴心。
有别于其他酒店，这里是没有前台的，整个大厅摆满了新中式的家具，可以坐在沙发上一边办理入住，一边享受热毛巾和甜品带来的温度，办完入住手续还可以在这里休息、喝茶、看书，十分悠闲。
穿过庭院乘电梯上楼，整个庭院的风光展露无遗，青瓦黛墙，白色的墙壁搭配原木窗棂，完美运用了新中式的精髓，每一个细节都让人眼前一亮。
跟随前台，手持佛珠，用它打开原木的房门，走进的瞬间，心一点一点的欢喜跳跃起来。
我们住的是见法阁复式套房，古味十足的床是整个房间的亮点，仿照古代床帷幔的设计样式，有机棉麻混纺材料的床垫，搭配柔软的被套，房间飘散着阵阵自然芳香，躺在床上，分分钟陷进温柔的梦乡。
在窗台一角，隔着原木屏风是TOTO品牌大浴缸，从张家界森林公园回来的晚上在这舒服的躺着泡个澡，闭上眼，静静的享受这难得的惬意时光，洗去一身疲倦。
简约风格的楼梯，搭配着禅意十足的茶几，夜晚，和好友一起，泡一壶茶，共话往事，轻触流年，甚是难得。
不仅每个房间都配有茶几，酒店还设有专门的茶室，素如积雪的茶杯，温润通透的茶汤，浅酌慢品，禅茶一味，心外无物，一切都是那么从容自在。 
听酒店员工说，每一个茶杯都是酒店老板亲自去江西景德镇考察后定制的，看得出酒店老板对于每一处细节的用心。
另外，凡是在酒店住宿的客人，都可以免费到她们的禅修室抄经，还可以看到美美的山景呢。
早晨迎着第一缕阳光，在这里抄上一份心经，在笔墨纸间感悟中国文字的韵味，同时也让忙碌、浮躁的心，慢慢沉寂下来。
沐手、净身、焚香、啜墨，再一笔落下，那一刻我做到了收摄身心，闻着墨香，手握笔毫，盘膝静心，在习字中颐养最平静真实的自己。
一个周末的时间，逃离城市喧嚣和烦扰，放下心上的枷锁，让自己沉浸、专注于一件简单的事情——与自己对话，与自己相处，那便是最大的幸福。
