温馨小TIPS

1、 北京 的大部分景点都聚集一个地方，如果你决定环绕天安门附近走，能尽量选择地铁就选择地铁，因为天安门广场附近限行的地方比较多 ，公共交通不仅线路稳定，而且绿色出行更环保。
2、 北京 地铁通往机场特别便捷，有机场线 ，可以从东直门搭往首都机场。景色还一级棒，一度以为自己去了个草原~赚到了，哈哈
3、 北京 有两个机场，一个首都机场 ，一个南苑机场，首都机场比较大，距离市区也较远，但准点率也相对高些。当然，订越早的航班，准点率越高，我很幸运地订的是晚上的机票，却也提前了 40分钟到达北京 。
4、通往 北京 的航班一般档次也比较高，例如我这次搭的南航，是比较新的机型，飞机除了配有充电插头、耳机，还有电视屏观看综艺、电影、纪录片等等，从 广州 飞往 北京 是个很愉快的享受过程。比较搞笑的是，从尾部望去，九成的乘客都是在看复联（笑哭）。一般选择A380机型的，都不差，可以在航空网站上看到自己航班的机型和准点率，再进行选择。
5、 北京 相对于南方温差还是比较大的 ，有阳光和没阳光的时候，温度差距挺远，常备多一件，是非常需要的。
