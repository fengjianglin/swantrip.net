朝拜礼佛，感悟一份禅意

游记 人文游 摄影 精品酒店 美食 2018/12/29
仓央嘉措曾说：世间事，除了生死，哪一桩不是闲事？为俗事所累，不得安静，是现代人的通病。焚一柱清香，静静的听法师讲一段经，人是未来佛，佛是过来人，我在佛前许愿，愿我早脱离欲海的浮沉，有一日如你般睿智。
人是未来佛，佛是过来人。世界万物皆有因果，如今的日子，总是穿行在人群之中，忙碌于日常琐事。岁月悠长，但唯独没有留给我们静坐回味的时间，期望能有一片土地，走进后能短暂遗忘烦恼，品味来时过往的人生。而我一直在寻找，从未遇见。直到有一天走进牛首山。这片佛教圣地恍惚间像是曾经遇见，努力回忆却无法追踪，豁然知觉这便是我脑海期许的那一片净土，那一片能让我静静聆听内心的圣地。
牛首山又名天阙山，是金陵四大名胜之一，因山顶南北双峰形似牛头双角而得名。一座牛首山，半部南京史。牛首山风光秀美，素有“春牛首”之美誉，古有牛首烟岚、献花清兴、祖堂振锡等金陵美景，还遗存了很多历史古迹，如岳飞抗金故垒、摩崖石刻、弘觉寺塔、郑和文化园等，吸引无数帝王将相、文人墨客在此修身养性、品茗作赋，留下诗词400多首。近年，世界佛教界至高圣物——释迦牟尼佛顶骨舍利于南京盛世重光，经宗教界、文化界、文物界研究同意，南京市委市政府决定建设牛首山文化旅游区，长期供奉佛顶骨舍利。牛首山文化旅游区以“补天阙、藏地宫、修圣道、现双塔、兴佛寺、弘五叶”为核心设计理念，全面保护牛首山历史文化遗存，修复牛首山自然生态景观，利用原有矿坑建地宫，长期供奉佛顶骨舍利。整个文化旅游区涵盖佛顶圣境、宝相献花、隐龙禅谷、谧境禅林、天阙小镇五大功能区，致力于打造融佛禅文化、金陵文化、生态景观为一体的生态胜景、文化圣境、休闲胜地。
 
大交通
 
 
牛首山位于南京市江宁区，坐落在整个南京市的西南方位。南京的交通较为便利，飞机和高铁都能够到达，距离牛首山较方便的是南京南站和南京禄口国际机场。
南京南站：华东地区最大的交通枢纽，国家铁道枢纽站，是亚洲第一大火车站和亚洲第一大高铁站。目前是京沪高速铁路、沪汉蓉高速铁路、宁杭高速铁路、宁安高速铁路、宁合高速铁路重要枢纽，南京南站每天过往停靠的高铁比较丰富，大家可以根据自己的出发地适当选择。
南京禄口国际机场：南京禄口国际机场位于南京市江宁区禄口街道，每天国内国际通航班次也较为丰富，远距离的游客也可以考虑乘坐飞机前往。

景区大厅售票及优惠政策如下
除了景区当天购买，还可以选择从网上提前一天购买门票价格可以享受一定的优惠。
1、 佛教圣地不同于其他旅游景区，游览参观期间大家要做到文明敬佛，佛教是一种文化，不论信与否都不要在景区中对佛评论
2、 部分景点如佛顶寺不可对佛像拍照，进入寺院不可大声喧哗，不踩寺院中的门槛
3、 牛首山上有八座蕴含东方禅意的公厕，每一处的别具风格，也是2016年全球最美公厕入选公厕，有兴趣的可以参观
4、 景区比较人性化，较长的台阶都装有电梯，因此不需要很大的体力
5、 佛教文化游较深的底蕴，建议请导游或者租赁导游机听介绍游览，避免走马观花
6、 景区的东入口是景区正门，周边设施比较完善，还有星巴克、哈根达斯这种大品牌入驻，适合休闲放松，也有部分餐饮提供
7、 景区不论是购买门票还是其他消费都支持微信、支付宝、刷卡等，因此不需要准备大量现金，只需准备少部分应急即可
景区东入口有自己的餐厅是对外开放的——牛首山食堂，里边有两种就餐方式一种是自助餐一种是桌餐。这里给大家重点推荐一下牛首山食堂的桌餐，包间在装修上较为中式，纯木质的装修给整个包间提升了一个档次，房间里还装饰有牛首山的风光图片。
再来说一下整体的餐品，说到南京的菜品风格肯定要提到的是金陵菜，是指以南京为中心，一直延伸到江西九江的菜系，是苏菜的四大代表菜之一。金陵菜在民国时期发展到鼎盛，因菜品受到名门望族的青睐，故在摆盘上比较精致，食材选择也是上等。而这些烹饪风格一直延续到如今，在牛首山食堂也同样不例外。
在口味上遵循以咸为主，咸甜适宜、酸而不涩、苦而清香、辣而不烈、脆而不生、浓而香醇、肥而不腻、淡而不薄。所以来南京一定要品尝一下地道的金陵菜。
除了景区的餐厅以外，我入住的翠屏山宾馆也是可以提供餐饮服务的，分为自助和桌餐两种形式，不论是自助餐还是桌餐菜品也都非常精致，环境也不错，适合聚会。
菜品如下
目前牛首山景区还没有自己的酒店住宿，距离牛首山最近的酒店便是翠屏山宾馆，约20分钟的车程，酒店属于园林式设计，楼层并不是很高，庭院中适合休息散步。房间布局比较合理，每个房间都有独立的阳台，阳台可以看到酒店的花园，视野很不错，没有城市酒店拥堵的感觉。
房间装饰上属于传统商务风格，配套设施非常全面，床垫的软度适中，卫生间干湿分离，酒店的服务人员态度非常友好，并且酒店在价格上也比较合理，非常适合旅行住宿。
牛首山的游客中心分为上下两册，下层为商业餐饮区域，上层为售票大厅、资讯台和休息区，售票大厅设有自动售取票机可以选择从上边购票取票，减少排队的时间，另位售票大厅也可以租赁自助导游机，休息区除了有座椅也设有自助的存包柜，不仅可以存放小件物品还可以存放行李箱，非常方便。如果对景区的分布不够熟悉还可以选择从书刊架上自行领取景区的导览图。
从游客中心出门后便是检票口，检票口左侧是景交车的乘车处，第一站没有选择乘坐景交车，而是步行走进佛顶前苑。佛顶前苑是通往佛顶寺的重要通道。
走进佛顶前苑像是一幅天然的山水画呈现在眼前，很有江南气息，虽已是寒冬，但江南的冬天并不像北方那样枯燥，景色依然美丽。
佛顶前苑
放眼望去，不见任何寺院，唯一能看到的便是山脚处露出的半边佛顶塔，给内心的感觉像是走进了一座公园，令整个牛首山显得有些神秘。
沿着弯曲小径继续向前，近处的亭子中传来阵阵琴声，彩色的绸缎随风飘荡，这般景色恍惚间像是让人走进某个仙境。
佛顶前苑
佛顶前苑最值得驻足思考的地方莫过于这个十牛图塑石假山，十牛图的思想基础，源自《六祖坛经》的见性法门。这是印度大乘佛教真常唯心思想的推演。这种思想，强调每一位众生的本性（或佛性、自性、如来藏心）是真常清净，具足无量功德属性的。如果能在次悟出一番道理，便是最佛教最透彻的领悟。
佛顶前苑的草坪两侧还会有一些小禅师的石像，雕刻的惟妙惟肖，十分可爱。
佛顶前苑的尽头便是佛顶寺，佛顶寺佛顶寺为佛顶骨舍利护持僧团弘法道场。是牛首山主要建筑之一，占地面积六十八亩，建筑面积近四十亩，仿唐风格，传统中轴线格局。共有殿堂七座，外围有僧竂、斋堂等建筑。
不同于其他寺庙的建筑风格，佛顶寺的建筑风格是典型的唐风建筑，牛首山的寺庙并非是今天才有，最早可以追溯到南朝初年。唐朝贞观时期，寺院规模宏大，名为佛窟寺，为禅宗之前“牛头禅”发祥地。北宋后易名宏觉寺。南宋毁于战乱，明代复兴。时至1856年，寺院毁于洪杨之乱，香火中断百年。而今牛首山再次重现世面，在文化界知名人士和佛教界高僧建议下，以供奉于牛首山地宫的佛顶舍利命名，取名佛顶寺。
佛顶寺共分为南北两个片区。北片区属于唐风宫殿式布局，包括礼佛区和弘法区；南片则属于院落式布局，是僧人饮食起居的空间，不对游客开放。
 
寺庙的大门被称作山门。山门又称三门，分别是中间的空门，左边的无作门和右边的无相门，象征着三种解脱。正中的空门，通常供大德高僧和大功德主进出，普通的信众和居士，则从两边进出寺庙。
空门门楣上方匾额“佛顶寺”三个大字由中国佛教协会名誉会长传印法师亲提。门楣下方的这幅楹联，由江苏省楹联研究会会长秦志法创作、江苏省书法家协会主席尉天池书写。
进入山门后，依次是天王殿、伽蓝殿、大雄宝殿。佛顶寺的大雄宝殿内供奉的是“释迦三尊”，分别是中间的释迦牟尼佛，佛左侧的文殊菩萨和右侧的普贤菩萨。从大雄宝殿出来，依次是观音殿、法堂。在中轴线的末端是法堂，也叫讲堂，是高僧演说佛法、皈戒集会的地方，它是佛顶寺中仅次于大雄宝殿的第二大主要建筑。
祖师殿内供奉了祖师的牌位，最前面的是法融禅师，后面供奉有牛头禅的二祖智岩、三祖慧方、四祖法持、五祖智威和六祖慧忠；后面供奉的是中国禅宗初祖达摩、二祖慧可、三祖僧璨、四祖道信。
牛首山最核心的区域——佛顶圣境区域。包含了无忧广场、佛顶尊胜道、佛顶宫、佛顶塔以及佛顶广场等部分。正前方是万字墙，墙上“佛顶圣境”四个大字据说是台湾佛光山的星云大师亲笔书写的。
从远处望去，佛顶宫和佛顶塔完美结合在一起，沐浴着一天从清晨的第一缕阳光到日落的最后一缕阳光。
走到无忧广场。眼前的无忧门，是根据佛祖在无忧树下诞生的故事来设计的。无忧门采用花岗岩雕刻而成，在门楣上方是一尊太子像，据佛经记载：太子在降诞后一手指天一手指地说道：天上天下，唯我独尊。太子的形象是采用汉白玉雕刻而成的，刚好与无忧门形成一种鲜明的对比。在太子像的周围还雕刻了九个龙头，因为太子出生时天上出现了九条龙，口吐香水为太子沐浴，称为九龙灌浴。
佛顶塔，塔高88米，9层四面，是仿唐式的建筑，塔的1-7层主要是各种经藏的不同展示，所以佛顶塔就是佛教三宝中的“法宝”。目前这一块区域还没有开放。
佛顶圣境的另一个主题园区便是佛顶宫，整个佛顶宫是以供养为主题的，释迦摩尼佛的佛顶骨舍利便在此供养。单体建筑面积达到了13.6万平方米，相当于19个足球场的大小，分为九层，地上八层，地下一层。九层又被分成三大文化空间，自上而下分别是“禅境大观”、“舍利大殿”、“舍利藏宫”。
禅境大观的核心是禅文化，由三个部分组成，佛出生的禅境花园、成道的禅境花园和中间的莲花剧场，共同表现了佛祖一生的故事。沿着佛陀修行的足迹，让人们感悟由释迦牟尼人成佛的奇迹。每天的10:00至10:30、13:30-14:00、15:00至15:30，11:30至12:00在禅境大观内举行每场约30分钟的“如莲佛颂”佛禅仪式及舞台多媒体秀表演。仪式表演形式中结合燃灯祈福、献花礼佛、大型舞台机械舞美视觉艺术展示，可以感受一场恢弘绮丽的视听盛宴。
从禅境大观中走出来乘坐电梯会可以下到佛殿层，佛殿层距离地面约33米，核心空间是千佛殿和万佛廊。千佛殿长廊两侧是千佛墙，一共2800尊佛像，表现的是大日如来形象。佛像的材料是金色年华大理石，下部的基座是珊瑚红大理石。
千佛殿最核心的点也是整个地宫中最气派的建筑便是舍利大殿，它代表的是大日如来。大塔高21.8米，由塔座、塔身、塔刹三部分组成。走进舍利大殿，心灵像是瞬间得到放空，这大概便是佛的魅力吧。
因为佛顶宫舍利安奉在牛首山佛顶宫是佛教界、文物界和社会各界对牛首山的重托，必须保证安奉环境的清净、庄严、如法如仪。根据《南京市佛顶骨舍利保护管理办法》，每年元旦、春节、国庆假日期间和释迦牟尼出家日、涅槃日、佛诞日、成道日，佛顶骨舍利在佛顶宫五层千佛殿对外供奉瞻礼。不过在舍利大殿中有佛顶骨舍利被放大后的图片，可以瞻仰。
从千佛殿走出便进入了万佛廊，万佛廊是一条环形的回廊，展现的主题是佛本身故事。在回廊的内外两侧有佛龛，这是其中的一段，佛龛内供奉的是文殊菩萨（古法琉璃.兔）、大势至菩萨（锡青铜.马）、不动明王（紫砂.鸡）和千手观音（德化白瓷.鼠），他们是十二生肖本命八尊中的四尊。这些便是佛教中所信奉的本命佛。
除了这些核心景点以外，牛首山景区还有郑和文化园、牛首捌厕、牛头禅文化园、岳飞抗金故垒等可以参观，由于我这次的时间比较紧张，就没有继续游览，毕竟旅行总要留下一些景点，念念不忘必有回响。
