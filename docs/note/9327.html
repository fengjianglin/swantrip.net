前言
朋友，你知道江苏境内有几座长江大桥？告诉你，七座。它们是南京长江大桥、南京长江二桥 、南京长江三桥、扬中长江大桥、润扬长江大桥、江阴长江大桥、苏通长江大桥(在建），另外还有一座上海崇明到启东的崇启长江大桥。
毛主席说“一桥飞架南北，天堑变通途”。现在，七桥飞架南北，天堑变成坦途。
以前的苏北不是这样。
从青年时的我爷爷揣着一把菜刀和几个志同道合的亲友坐着木舢板船从扬州沿长江顺流而下闯荡十里洋场的上海开始，我算第三代移民，但只回过一次老家。那时我只有十几岁，利用暑假和姐姐去接在苏北玩的奶奶和弟弟回上海。我们在上海坐的是晚上的火车，第二天早晨才到镇江，再坐小火轮船到永安，途中过邵伯闸还下船步行了很长一段路，到永安人民公社快天黑了。回来的那天是吃了晚饭步行一个多小时到永安镇，住到靠近河边的一家小旅店，第二天凌晨上船。船上很拥挤，为了给奶奶找个座位，姐姐急得哭了起来。中午到镇江，再坐火车到上海已是晚上。一路车船劳顿，苦不堪言。唐代诗人李白诗云：蜀道难，难于上青天。而去苏北，也难！
从此以后，我对这个老祖宗的栖息地又敬又畏，难得回去一次。
半个世纪过去。年轻朋友害怕了吧，真的半个世纪过去了，如今的长江（江苏段）上已经架起了七条“彩虹”。苏北 ，就像插上翅膀的天鹅，飞了起来。
朋友，请跟随我们这些“少年”馿友，去苏北腹地掠影玩耍吧！

第一天。早晨，在人民广场上车，两个小时就过了江。实际上过了江就到了南通，就像做梦一样，现在的交通真的不敢想象，恍如隔世。第一个景点是南通城市绿谷公园，公园被河水环绕，风景很美，是苏北地区最大的私人花园，融中西风格于一园的独特园林。

第二天。吃了特色鱼汤面后游甘港村。这是新农村发展的样板村，村里保存着许多以前的东西，看看也很有意思。院子里有石磨，水车供游人拍照、互动，大家掙抢着推石蘑、踩水车，摄影又摄像，玩的很开心。没想到这倒成了这次旅游的亮点，现在想起了还兴致盎然，意犹未尽。这些东西如果只让看不让动，那就索然无味，没什么大意思了。

这是里下河与沿海交界处的历史悠久的安丰古镇，文物古迹很多，游客要慢慢观看，细细品味才能领略到古镇的味道。
我喜欢一个人慢慢的在古镇长长的深深的长着青苔的石板路的巷子里转悠，望着旁边石拱桥下的溪水在缓缓流淌，你会和历史不期而遇，你会想起童年的幼稚，青年的英姿，中年的负重和老年的无奈，你会觉得这一生就像眼前这条古巷，被历史的风雨侵蚀得满目旧痕又不堪负重。而巷子尽头是一片亮亮的天空，鼓励着你、引诱着你朝前走、朝前走。。。。。。
人生就是这样，朝前走，朝前走。。。。。。

老年的欢乐就在宫商角征中。苏北人才济济，据说梅兰芳、筱文燕、徐志摩等许多文艺名人都是苏北大平原培育出来的。苏北民歌、扬州小调响彻中国大地，一首《茉莉花》世界共唱，全球缭绕，经久不衰。近来一首《烟花三月下扬州》更是风靡上海的大街小巷。
这时候，捧一杯龙井酽茶，坐在他们旁边，慢慢呡着清香的茶水，悠悠的欣赏着苏北平原特有的、淳朴的、不加修饰的豪爽而粗犷的乡音，真是天上人间，不知有多享受。
遗憾的是，匆匆拍了两张照片，连个调门都没听出，就被爱人匆匆拉走，追赶队伍去了。这也是跟团游的硬伤。

如皋水绘园，苏北平原上的一颗璀璨的珍珠。始建于明朝万历年间。水绘水绘，就是以水绘画出映入水中的倒映，是很有情趣的公园。内有许多亭台榭阁等古迹，融诗、文、琴、棋、书、画于一园的特色，很有看头。走在园中，你无意中也会沾染一些文人的书卷气，变得彬彬有礼，文明大方。你可能还会听到董小宛和冒辟疆的缠绵爱情故事。

南通滨江公园，在狼山风景区。浩瀚的长江，使人心胸开阔，精神振奋。我喜欢看江，看海，看洋。坐在岸边的石头上或站在船弦旁，我会呆呆的久久的看着，看着那水天一色的风景。许是城市太压抑，高楼大厦太多，人车太吵杂，生活节奏又太快，人情味又太单薄，这时，最好的解脱方法就是看海，看洋。看到大海，混沌沌的大脑就像扫完垃圾的电脑，一片清爽。
我喜欢大海！

编后
匆匆的苏北大平原三日游结束了，令我回味无穷。那些老黄历的记忆应该抹去了。我感叹苏北的变化，也感叹人世间的沧桑，更感叹自己的碌碌一生。在退休之年能走走看看，玩玩转转，吃吃尝尝，此生足矣！
