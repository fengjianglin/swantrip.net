行前攻略

签证，万能的网上，因为我是12月底才临时决定了这趟 日本 行，元旦大使馆又休息，一月初才能递签，于是签证选了特急六个工作日出签，1163元 orz 财产证明只要余额十万以上的银行活期流水就行。不过日本 签证现在确实是很容易了，小伙伴是刚办的空本护照加 福建 户口的学生狗，这么难的条件也是顺利出签了。还有个小插曲就是，学生狗还要提供放假证明，而厦大这学期2.14就开学[呵呵]，于是我们办签证的时候行程是写的2.6-2.13，实际是2.12-2.22去的，机票预订单让TB代做，酒店就在jalan随便订几个，记得出签以后要取消掉。所以亲测实际出行时间是可以跟办签证的时候写的时间不一样。
机票，因为怕拒签只敢等出签了再买机票，选了特急还是1.13才出签，曾经在1月初看到往返全日空直飞两千三的机票，已经涨到了三千五 = = 只好又买了中转。国泰航空官网直接搜， 厦门 往返 东京 中转香港 ，来回只要2374，中转时间也没多久，大家直飞太贵的时候可以去国泰官网碰碰运气。【我最近又要在国泰航空官网买机票，但是发现现在搜机票需要翻墙才能打开了 = =。】
酒店，真的强推jalan【 www.jalan.net 】，首先jalan真的比其他各大订房网便宜！而且预订的时候不用先付款不用信用卡担保，只要填个人信息就可以，这次感受到一个很棒的是jalan还可以选各种不同的方案，像我们这次飞驒牛早餐，鹤雅的晚餐都是在jalan订房的时候就选好了真心非常值。建议用日文原版注册个账号登录预订，不要用中文版，中文版少很多酒店而且方案的翻译很奇怪看不到具体方案内容，日文版也基本都是中文的没什么难度，实在不行边复制翻译也都大致能看懂。
换钱，我会把所有的花费都列出来，吃的也每餐都大概估计列出来，算出来总共大概要多少钱先在国内换好，我在建行换的，提前在建行官网预约了换日元的。结果没想到我们的零食吃太多了...最后钱竟然不够了，小伙伴就在 日本 的ATM取了钱，有银联的标志的ATM就能用银联卡取钱，取了两万日元，被收了23人民币手续费，所以应该是在国内换比较值？
JR PASS，就是 日本 JR铁路通票，全国通票只能提前在国内买，也是万能的网上就行。网上提前买好兑换券，到 日本 再用兑换券换成JR PASS。根据自己坐了多少钱JR决定要不要买，我们是坐了四千块左右的JR，JR PASS只要一千四，疯狂省钱啊哈哈哈哈。JR PASS还有各种分区域的，像如果只在 北海道 玩的也可以买JR 北海道 周游券，按自己需求选择好了真的非常省。详细可以在JR PASS官网了解一下【 http://www.japanrailpass.net/cn/ 】。JR 北海道 周游券官网【http://www2.jrhokkaido.co.jp/global/chinese/railpass/rail.html 】。
巴士，长途巴士基本都要提前预订的，像我们这次 东京 到高山的巴士，在巴士官网就能预订了，浓飞巴士【 http://www.nouhibus.co.jp/ch_k/tokyo.html 】，其他各种巴士搜一下官网基本都能直接在官网订。也可以在这个网站【 http://japanbuslines.com/tw/?_ga=1.57926506.1529011881.1452834520】，浓飞巴士官网点了网上预约就是自动跳到这里，我也就在这里订的，大致看了下好像可以订全 日本所有线路的高速巴士，用信用卡付款，付款完会给你发预约 成功 的邮件，把邮件打印出来到乘车处给工作人员看就可以。
北海道 道东交通，真的是被这个折腾死，火车巴士班次少，一个地方到另一个地方一坐就是几小时，一些巴士一天只有一班车错过了就没了。在 北海道 JR官网【http://www2.jrhokkaido.co.jp/global/chinese/index.html 】搜火车的时刻表，再结合japanican上的 北海道 东部快速巴士【 http://www.japanican.com/cn/special/area/eastern_hokkaido/index02.html 】，斜里巴士【 http://www.sharibus.co.jp 】， 网走 巴士【 http://www.abashiribus.com/ 】，阿寒巴士【http://www.akanbus.co.jp/ 】， 札幌 巴士【 http://ekibus.city.sapporo.jp/ 】，好不容易做出来一个在有限的时间玩最多地方的行程.....orz
北海道 到 东京 ，我们去程是坐JR去，回程是从 札幌 飞回 东京 的，直接全日空官网买的机票【https://www.ana.co.jp/asw/wws/cn/c/ 】， 札幌 飞 东京 只要622元，是对有往返 日本 机票的外国人的特价好像，很多廉价航空加上托运行李的价钱也都要五百多了何不多几十块感受一下全日空，而且全日空也还有另一个好处，后文会提到。
白川乡 点灯，一年只有七天，因为出 白川乡 的巴士下午五点多就是最后一班了而点灯是五点半开始，所以要看点灯只有几种办法，一种是住 白川乡 ，点灯日住宿听说是提前半年开始预订并且瞬间抢光，一种是浓飞巴士会出一个小团，坐巴士下午从高山出发去 白川乡 看点灯晚上回高山的，japanican上也可以搜到，也可以直接在浓飞巴士的官网预订【 http://www.nouhibus.co.jp/new/teikan.html 】，这个是提前两个月开始预订，也是瞬间抢光 = =。我决定去 日本 的时候这些都被订完了，又在japanican找到了 东京 出发的两天一夜团，这种就没法自由玩高山了，最终又在万能的TB发现了高山出发拼车去 白川乡 看点灯的团，终于 成功 订下 白川乡 的行程，不过这个拼车真是非常无语详见后文，如果想看 白川乡 点灯的建议提前做好行程，提前半年订 白川乡 的住宿或提前两个月订浓飞巴士的团。
japanican是有中文的，当地团那里搜一下 白川乡 就可以找到。浓飞巴士官网的话就是这两个是去看点灯的团，上面那个是可以网上预订的提前两个月开始，下面那个是只能电话预定的不会日语就没办法了。
知床 五湖暴走&流冰漫步一日游预订： 知床 观光协会官网【 http://www.shinra.or.jp/ryuhyo_walk-goko.html 】，里面还有很多 知床 其他的一日游、半日游各种，点预约按钮填一堆信息以后他会给你发一封邮件，有一些注意事项和可能会再问一些问题，回了这封邮件以后他会再给你发一封确认邮件就是预约 成功 了。这个是会来酒店接送的，当天就到酒店前台，让前台帮我们找教练，教练会有一份名单在上面找到自己名字就可以出发啦。冬天的 知床 五湖是封闭的没法自己去的，一定要在 知床 观光协会官网报名了才能统一带进去。
