玩转·厦门|绝对不能错过的吃喝玩乐

游记 情侣游 摄影 美食 自助游 2018/12/23

前 言

人应该去旅行的，趁你年轻的时候，你还有本钱耍个性，装潇洒，请你果断的离开睁眼闭眼看见的水泥森林，逃开纷纷扰扰的勾心斗角，去一个自己喜欢的城市享受独处的岁月静好，然后精心的拍一些美的想哭的照片，留给老年的自己，自豪的拿给儿孙，看，这些都是我走过的地方。 ​
 

关于住宿交通

 1.关于住宿
        厦门的民宿业真的特别发达，厦门本地的民宿大多选择余地很多，从一百二百元到上千元不等，可以根据自己的经济实力进行选择。其中民宿一定要选择在岛内，因为大部分的景点游玩的地方都在岛内。另外不推荐去鼓浪屿上特别住一晚上，感觉意义不大。最后一定要选择拍照好看的房间。在后面文章里会介绍一家力荐的酒店，在抖音上超火的一家酒店。
 
2.关于交通
        厦门高崎机场建在岛内，算是中国为数不多敢把机场建在市区的机场了，所以飞机降落的时候特别酷，下面的居民楼清晰可见。厦门站和厦门北站距离很远，厦门站在市区，厦门北站在集美，所以买票的时候一定要擦亮眼睛啦。
厦门的公交系统还是比较发达的，很多旅行专线的公交，准备好零钱啦。
 

厦门必打卡景点-厦门大学

对厦门大学情有独钟的我们在达到厦门的第一时间，就来到了这里。可谓是超级兴奋。
全国最美大学之一，一个大学能作成一个旅游景点也是特别难得。建南大礼堂和颂恩楼的历史文化风韵，芙蓉湖和情人谷的旖旎风光，一起来看流星雨的上弦场，还有涂满涂鸦的芙蓉隧道，都是厦门大学的特色。10月底厦门大学需要开始在公众号里预约参观了，需要身份证和人脸进行识别，一天1000多人次可以进入，记得提早预约哦。
厦门大学最出名的芙蓉隧道还有厦大拱形的树。
从厦门大学后门出去就是白城沙滩，出门就能看到大海的学校可以说很羡慕了。

厦门必吃美食-堂宴·老厦门私房菜

德云社同款的一家厦门私房菜听说来厦门一定要吃堂宴,不单单是口味地道,还有优秀的服务，里面还有很多老厦门的失传菜。
这家店是在大众点评上必吃榜搜索到,结果出奇的满意,超性价比,难怪排名如此之高,来厦大的朋友可以顺路来吃,在厦门大学对面的一个美食广场里,远远的就可以望见了,位置比较好找,菜品超棒，性价比很高。
堂宴的服务超级好,每个工作人员都非常贴心,每道菜上来都会介绍当前道菜品,还会考虑地方口味等因素贴心的准备了小蘸料,每时每刻,心系食客,及时了解就餐情况并对应补给满足需求。
黄鱼煮面线强烈推荐的一道菜,汤超级超级鲜,乳白色的一大碗,分量非常足,面线和汤可以无限量的加,超满足,鱼肉很紧实,差不多四五条的样子,面线很细,很劲道,跟之前吃过的不太一样,爆炸来好吃,感觉是非常地道的家菜馆,很有本土特色,汤头应该是熬了超级久，,码到这里口水都要快流下来。店员考虑到口味原因,推荐了蘸料酱,黄灯笼辣椒酱,超级好吃。和黄鱼面线搭配一起,就是一份神奇,完美
香酥芋泥鸭一直非常喜欢香芋,芋泥的东西都非常爱,这款菜品好棒,满足了所有的爱,据说是土鸭去骨后,用秘制酱卤制,芋泥先蒸后碾碎再摔打,最后油炸成型,芋泥里滚了美味的鸭肉,虽然对鸭我爱的一般,但这款爱的深沉,酥脆的外壳,绵软的芋泥,里面包裹着扎实的鸭肉,沾上特制的鱼肝油橙汁酱,酸甜不油腻,实在太棒了。其他菜品就不一一介绍了，上一波图嘻嘻。
在去完厦大后就直奔事先预定好的酒店，对于酒店其实有着自己的一些标准，毕竟住在旅行中占了很大的一部分。我们在来之前考虑再三还是定了这家艺术气息浓厚的酒店，据说是厦门老牌酒店鹭江旗下的新品牌。

厦门鹭江·佲家

鹭江佲家距离曾厝垵商圈不到一公里的位置’坐拥绝佳美景°前有奔流不息的海’后有连绵不绝的山,左手微风,右手阳光,好不惬意。
前台 check in之后搭乘电梯到自己的房间,推门而入,房间内满目的清新干净的色调让人心旷神怡,湖蓝色的抱枕与亮橘色的布艺为这一片宁静增添了一丝活泼与俏皮。
精致山景双床房内有两张1米2单人床,如有需求也可以电话至服务中心,让客房服务人员帮忙把两张床拼为张独一无二的2米4超大床。梳妆台应该是女孩子们最关心的一个部分,自带LED灯的小梳妆镜一方面不会吵到还在沉睡的同伴,另一方面也可以在黑暗中依然照亮你的面容,让你的妆容精致完美,不要太贴心哦
洗手间设备一应俱全’宽敞的洗手台,温馨的香薰小瓶’高档的牙刷用品,更有让人瞠目结舌的高科技感应电吹风等待你去挑战哦
Ps:为了大家的安全着想,酒店的访客管理也是非常严格的哦,如果约了朋友临时过来酒店一起享用下午茶,需在一楼做好访客登记,离开房间后需捋回执交还给前台。

厦门必吃-鹭江·佲家丨璞艺自助餐厅

鹭江宾馆姐妹品牌,高端五星级度假型酒店~鹭江.佲家酒店,位于厦门思明区龙虎山382号,紧挨曾厝垵旅游区,科技中学斜对面
原本就对鹭江宾馆印象极佳,可以说是心中厦门自助餐第一把交椅,肯定不会放过这个机会,来试试它的姐妹品牌酒店整体风格有别于鹭江宾馆沉稳大气历史感厚重,新酒店融合了很多年轻元素+新派艺术气息,让人感觉欢快年轻!
自助午餐试营业期间活动价140,送即位菜品红烧鸡丝翅和冰镇鲍鱼!这也是我最看重的重头戏!其实自助餐到现在已经千篇一律了,真正能决出高下的,就是在这些有仪式感的菜品上!印象中,鹭江宾馆是厦门最早也一直坚持半自助的自助餐厅,最早的主厨推荐菜品还不限量,现点现做,那真的是很美好的美食记忆!扯得有些远!佲家的即位菜品品质还是不错的!鸡丝翅中鱼翅汤很好喝，,认可!冰镇鲍鱼口感也不错,个头也超级大!摆盘都很好看,热菜挺多,印象比较深的是汤品很多,金线莲猪肝沿汤,沙茶汤,石橄榄排骨汤,这些都是好料啊!肉类也不少,猪牛羊鸡肉都全了!味道也超级棒海鲜就是各种鱼虾贝类!但是没有螃蟹!这点之前有人诟病过,不过个人现在比较喜欢有温度的料理。

厦门必打卡景点-鼓浪屿

 
去鼓浪屿一定要提前几日在公众号（厦门轮渡有限公司）买船票，对于游客来说，有两条线路，一条是从东渡国际邮轮码头到三丘田码头，另一条是从东渡国际邮轮码头到内厝澳码头。大部分游客，旅行团都是前者的线路，我个人比较推荐后者的线路，避开人群才是王道。
日光岩是鼓浪屿的最高点，在日光岩顶可以看到整个鼓浪屿岛，日光岩上特别拥挤，所以推荐赶早上日光岩最佳。
 
避开人群，鼓浪屿上的居民区其实尤为好逛。鼓浪屿上不知名的码头同样充满了有趣的小惊喜。

厦门鼓浪屿必吃美食-挑食咖喱馆

在鼓浪屿闲逛偶然看到了这家挑食,没想到挑食还有咖喱馆,不愧是鼓浪屿上的网红店。
进入门店就像进入森林中的小房子般感觉,有庭院有绿植有木屋,复古的中式装修风还多了丝回忆感。海鲜咖喱馆主打的自然是咖喱,好在我们人多,可以多尝试了几份融入不同食材制作的不同口味咖喱,不过它家咖喱确实出彩,不管加入什么食材都很好吃,黄咖喱
味道浓香粘稠,红咖喱辛辣刺激。炸猪排也是强烈推荐的,外表金黄酥脆,内在鲜嫩肉
汁,不蘸酱都超棒的呦。还有店内的红咖喱波斯顿龙虾,咖喱用的是红咖喱，但却不会很辣，在配上超级新鲜的波斯顿龙虾，肉质很紧实，很下饭。嘻嘻红咖喱面包蟹，超大的一只面包蟹，蟹膏丰满装满整个蟹壳，良心之选。
推荐： 红咖喱面包蟹/印度红咖喱海鲜煲/沙爹牛肉串/沙爹鸡肉串/锡兰白咖喱鸡/秘制红咖喱牛腩/沙爹猪肉串

厦门鼓浪屿必吃美食-挑食|沙茶面（龙头路店）

在之前吃过挑食家的海鲜餐厅,对他家各方面印象都很不错,搜索时发现他们新开了家沙茶面店,还有优惠折扣,在鼓浪屿觅食的我就顺势寻了过去。店铺位置在龙头路主干道侧边的小巷中,闹中取静,导航很容易就找到。门面就非常讲究,整家店虽然不大却处处透露出细节:古早时期闽南特有红砖白墙装饰风格,配上复古门窗臬椅,加入各种做旧效果,连白墙上斑驳的岁月痕迹也模仿得惟妙惟肖,可见其用心。
吃沙茶面的碗勺都是复古鸡公碗,让顾客仿佛置身处古宅,真实还原老厦门的用餐环境。沙茶面用料新鲜实在,点单时就能看见小哥在台面上切新鲜的鱿鱼,干净卫生都看在眼里。当然沙茶面最重要的是汤头正不正宗,我平时不怎么爱吃沙茶面,也能感受到浓郁的沙茶飘香。最欣赏他家面里放的鱼丸,没吃前以为是普通火锅小鱼丸,没想到是老字号龙头鱼丸同款的手工包芯鱼丸!都不用去排队了。他们家的五香炸的刚刚好，外酥里嫩。服务态度也很棒,总体体验棒棒哒
推荐菜：沙茶面/土笋冻/五香

拍照打卡-反正鼓浪屿店

这家【反正】的环境无可挑剔,就是藏身的太过隐蔽,所以客人不多,可谓是真爱党才能寻得的爱丽丝仙境啦。很适合拍照氛围安静,特别适合炎热的下午在这里小憩避暑带着朋友过来鼓浪屿游玩,附近景点溜达完,顺带寻过来拍拍照,吃个下午茶。院内有一座现用作家庭旅社的别墅,绿植将庭院裝饰得郁郁葱葱,而【反正】则藏身在庭院的更深处。
店内空间不大,布置看似简简单单,却赏心悦目。可供选择的甜品种类还不是特别多,但都超级精致。38元甜品加饮品的团购很超值。个人更喜欢那一款看似比较简单的【黑白】,口感细腻,甜味怡当,让人心生愉悦。饮品的颜值都很高,名字也很好听,应该是苏打水调制,再装点薄荷叶与柠檬,清爽解渴,味道很夏天。
推荐:黑白

厦门必打卡植物园

 万石植物园因为沙生植物区的网红仙人掌和雨林世界里的雾气朦胧而出名，真的特别适合拍照。植物园一般推荐逛这两个比较有名的区域，因为植物园特别大，记得一定要买票乘游览车到最上坡的沙生植物区，再慢慢走下来到雨林世界。记得一定要看清雨林世界里碰烟雾的时间，避免跑空。
      沙生植物区是最推荐游玩和拍照的地方了，因为地方特别大，拍照可以有效避开人群，同时里面有几个小型温室状的博物馆也特适合拍照。
沙生植物区的植物配上蓝色白云简直绝配。

厦门必打卡景点-八市

      厦门一共有1到9个菜市场，也就只有八市突然红了，八市承载着厦门人浓厚的厦门记忆，也是当地人最爱去的市集，现在的八市不单单卖海鲜，还藏着许多古早味的老店。
八市的海鲜都是最新鲜的，第一时间捞上来的海货，直接在摊位上开卖。
八市的傍晚是最繁华的时候，摊位出摊，游客，本地居民都聚集在此。
 

厦门必打卡景点-中山路

厦门最老牌的商业街，人流旺，商品多，名气大，不论往昔还是如今，好似纽约的曼哈顿、东京的银座、香港的中环。到厦门，中山路是不能不去的，因为它代表了厦门的繁华，富有和时代韵律。
一头连着厦庇五洲客的宾馆大厦，一头连着门泊万倾涛的碧波大海，与海上花园“鼓浪屿”遥遥相望。中山路的骑楼街是厦门建筑文化的代表，这些建筑绝大部分是二十年代华侨回乡建造的，2017年全部重新整修了一遍，恩……变得，各位自己体会……

厦门必吃美食-宴遇(中华城店）

游厦门的时候寻思着吃些好吃的,翻开大众点评的必吃榜,发现宴遇也入列了,虽然这家不是必吃的那家,却是离中山路最近的,交通最方便的。整装待发的前往拔草。
餐厅名字就是一个充满了想象空间的文艺小资风的名字,正如许多人心里很期盼的那样,美食如此,人生也如此。为了配合这场景,餐厅的装饰也是小文资,从灯光到餐具还有菜盘。
在宴遇等餐的时候,餐前商家给准备了一个惊喜,是一个盒子里放了两个小瓶,有一瓶是“毒药”。这个创意很好,其实毒药是苦瓜汁。
好吃的菜实在是太多,只能选一点点,胃还是有限。
招牌爆汁小乳鸽招牌的菜系,乳鸽躺在干草的锅里,就像躺在了窝里。带上手套撕开,香味飘了过来,肉汁的鲜味带来一丝喜悦。
猪猪流沙包一道网红的点心,特别是一些视频的网站的网红,每一次吃这个网红们都会虐待一番,筷子前捅了一下,我也想试试,水平不够,效果太差了。
服务非常不错.最后送的水果也是巧心思.花开富贵
还有其他菜品就不一一介绍了
推荐：魔鬼辣子鸡配芒果蛋黄/招牌爆汁小乳鸽/藜麦沙拉盆栽

老思西沙茶烤肉店
地址：南中广场对面
是一家家传老店,坐落在厦门老巷局口街街口,远远路过便能闻到一阵烧烤的鲜香。不过,烧烤其实际上不是小店最有力的招牌,而是搭配在一起蘸食所用的特制沙茶蘸酱才是精髓所在。这里的沙茶酱都是热腾腾地上桌,充分地将酱料的鲜香激发了出来,完全胜过超市卖的罐装酱料平淡的口感。此外,思西沙茶酱虽然十分香醇,却并不黏腻,甚至可以直接饮酌,咸甜适中,香味十足。来厦门特色尝鲜,必然是不能错过这里的沙茶了。

厦门必打卡景点-沙坡尾

可坐公交车在中华儿女美术馆站下车，青年文化、老城文化、闽南文化。在这一条老旧的街区相互交织，曾经的冷冻厂变成了艺术西区鱼铺变成了咖啡馆。裁缝店变成了创意汉堡店，街道两旁古朴的骑楼，创意商店门口新奇的装饰。艺术西区、手艺厦门、避风坞及演武大桥
都是极好的拍摄地。
所尝

厦门必吃-莲花公园煎蟹

离厦门站近,导航走差不多10分钟。餐厅环境不错,店里服务也不错,入座就给了开水烫碗筷,还有架子方便夹碗;给了一次性手套,也是考虑周到;店家那里免费停车,还有待客泊车。
莲花公园煎蟹~厦门老牌煎蟹餐厅,有着二十二年的经营历史,厦门餐饮界众所周知的煎蟹一哥,一直是以食材新鲜,味道一流,价格实惠。
香煎蟹，蟹肉鲜甜,红膏包满,个头挺大,芝麻油香迷人底下放了姜片煎，两者味道融合在一起味道超级棒，很满足。
炒米粉用的是兴化细米粉,超级好吃,今天一吃惊艳到了,瞵间回味起十几年前的味道,太神奇了,味蕾竟然比记忆还提前感受到这种老厦门人才能吃出来的味道。
白灼海虾惊艳不亚于炒米粉,因为厨师把虾的白灼熟度控制到非常极致,粉红带点鲜白,干净剔透,拨壳时自然顺滑,入口能吃到天然的鲜虾甜味,实属难得,从虾的色泽到拔壳再到入口,口感超越斑节虾。还有很多菜,我就不一一介绍了,大家自已来试试,一定不负众望。
推荐:莲花煎蟹/秘制旺螺/风味姜母鸭/特色炒米粉/文蛤土猪汤/特色海蜇头

厦门必吃美食-香遇巴厘印泰餐厅

可以说是厦门迄今为止年岁最悠久的东南亚餐厅了，常年在大众点评的必吃榜上，在这里,可以品尝到来自于印尼、泰国、新加坡等多个国家的菜品哟!而且最有意思的是,这家店老板的经历也是挺有故事的,香遇巴厘印泰餐厅的菜系主要分为三大类:地道的东南亚菜、港式烧腊、日式料理
咖喱皇炒蟹是香遇巴厘的王牌菜品,来店里的顾客几乎桌桌必点。它的亮点在于其金黄可可口味的咖喱汁运用了三十多种原产地为东南亚的正宗香料,口味香醇,带有微微辣昧却又有着椰奶的香甜。大伙最喜欢的吃法,就是再来一份泰国香米饭,将满满精华的咖喱汤浇入搅拌!而主菜也是挑选优质的蟹,大个的蟹腿,用钳子剪开就会看见里面白嫩嫩的蟹腿肉,吃起来也是蕴含了咖喱汁的美味。
东南亚菜擅于运用各种香料,其经典一是咖喱汁,二是沙嗲酱,三就是冬阴功汤了!泰式冻生腌虾这道菜品既有着日式刺身的感觉,又有着泰式的风味。新鲜的大虾进行开边处理,放在冰盘中既能杀菌又能保留着鲜美度,虾肉晶莹剔透,入口冰爽且Q感十足!而上
面覆盖着一些酸甜的泰式辣椒,酸辣中又带有一些清甜。明炉青柠桂花鱼有别于一些餐厅选用低价的鱼类,香遇巴厘印泰餐厅用的是优质且成本较高的桂花鱼(但是卖价也不贵哟)!搭配有酸甜可口的泰式辣椒、以及产自泰国的新鲜青柠檬片,在明炉的盛具中冒着热气,鱼肉慢慢地吸收着汤汁的昧道,微酸微辣!印尼沙嗲肉串在香遇巴厘也能品尝到哟!

厦门必吃美食-亚尖海鲜大排档·创于1993年（湖里店）

厦门网红级别的海鲜大排档，因为他们家的服务和相当地道的菜品，最主要的是他们家的海鲜现捞烹饪，新鲜度满分。备受网友的好评率超高，所以就决定过来试试。位置在南山路上的路边,有一个不大的灯牌,但正面的招牌还是有比较大的字,邻近一个超市。
店内有三层楼，,但我们只在一楼了,座位不算太多,大小桌都有。一楼主要还是海鲜展示区比较大一点儿,店家也主要推荐这个。不需要提前预约,验证也比较方便快捷。
炒时蔬,空心菜,蒜味儿的,很爽口。海苔角,脆脆的海苔包裹着肥肥油油的肉馅,鼓鼓的很喜人。水煮三鲜,,虾又鲜又大,肉多又实在,特别是藤椒的味道更是令人极大的满足。虾兵蟹将，这道菜真的是满足了我对海鲜的所有遐想，实在是太好吃了。姜母鸭,需要些时间才上桌,热气腾腾,锅还在滋滋作响,姜和酱汁的味道将鸭肉浸透,入口香甜软烂,大爱。
商家都很热情,服务的相当周到。

拍照打卡-环岛路烂尾楼

人少又很适合拍照的地方吧让旅行不留遗憾~
环岛路的烂尾楼公交车可以坐到曾山站
现在有专门的人在看管，进去拍照需要收费，五十左右。去年由于厦门金砖会议的召开，烂尾楼重新粉刷了一番，天晴的时候别有一番地中海的风味。
