在北方最美海岛睡炕房、赶海，三天两晚玩儿转刘公岛

游记 人文游 小众 摄影 美食 自助游 2018/10/31
海景大片儿不只有东南亚才能拍到，美景海岛更不是出境游的专利
这座历史上赫赫有名，如今常常被旅行者忽略的小岛
就要成为新晋网红度假胜地
不在南方，却在北方，让我们一同去揭秘它

【美景】

东泓炮台
这一张意境唯美的照片，如果不说，很多人会以为是日本，实际上是在中国拍的哦。
若不是亲身经历，我坐在家里也很难想象在北方，从不曾入心的角落有着触动你的一隅。
它就是我们中学历史课上不可逾越的一页，甲午海战主战场，北洋水师的诞生地——刘公岛。
100年以后的战场，不再是软弱可欺清政府丧权辱国牺牲的资本，当年的英租界殖民地在强大中国的保护与治理下，变身为人们享受平静生活的避世之所。
位于山东威海的刘公岛由于受到海洋的调节作用，气候冬暖夏凉，全年平均气温12度左右。
岛上完好地保存着英租界时期的建筑。疗养院变身为大炕民宿，军营化身为商铺，英国商人的海滨别墅，成为了可供大家租住的共享别墅，当年英国人的高尔夫球场，如今成为中国现存最古老的高尔夫球场，并被改建成高尔夫博物馆，供客人用餐休闲，当年的两栋龄球馆，现在是刘公岛管委会的办公区……
1891年李鸿章第三次巡阅北洋海军来威海的时候，威海卫海军基地已经初具规模，这时刘公岛已经有完整的炮台群和一些临时性堡垒防御工事。至今岛上仍存有东泓、南嘴、迎门洞、旗顶山、公所后、黄岛6座炮台。
而这一组小清新的照片，恰恰就是在当年“重金属” 的东泓炮台拍摄，这里可是看海景的最佳位置呢。
东泓炮台是威海地区28处甲午战争遗迹之一，位于刘公岛的最东端，约建于1889年，又叫东风梢、东口炮台。站在东泓炮台朝东南方位极目远眺便是韩国、朝鲜、日本。
甲午威海卫战役中它发挥了优势，协同日岛、南帮炮台重创日军的进攻，击中日舰多艘，后来被南岸日军的炮火摧毁而弃守。战后英国强租威海卫，炮台得到了英军的重视和利用。
炮台设有24厘米口径克虏伯要塞炮和12厘米口径速射炮各2门，7.5厘米口径行营炮6门，小口径速射炮4门。炮火火力可及刘公岛东部海面与南北两海口，主要与日岛、南帮炮台一起封锁南口。
炮台之下凿山建了互相贯通的半地下兵舍11间，每间约22平米，隐蔽坚固，有7个大门，位于坑道出口处并与之连接，兵舍互相贯通，方便进出，守军可以从兵舍经过坑道直达炮位。坑道为拱圈顶，用花岗岩砌成，里边有完好的通气设备。
坑道内设有炮弹以及发射药包提升井、灯龛、弹药吊装挂勾等设施，方便运输、操作弹药等活动。同时坑道内设有弹药库，弹药库旁的窗户处是灯龛，在灯龛里放煤油灯，从而提供室内照明，灯龛顶部设有通气孔，分流灯火的热量和烟气。
刘公岛国家森林公园
哪怕是来过刘公岛的熟客，也大都局限于岛的前端，餐饮游乐聚集区的码头附近，实际上，会玩儿的人都往后走。
乘坐景区的游览车可以轻松抵达没有游人的“后山”，在环岛的平整小路上驰骋，体会凭海临风的快感。
很多人不知道，这里是刘公岛国家森林公园，它是国家林业部命名的全国第一个国家级海上森林公园呢。80多种树，50多种野生花草，50多种鸟。
沿着蜿蜒的环岛路，仿佛步入"世外桃源"，想要找到一块儿裸露的土地可不容易，每一寸都被青草覆盖。公园总面积4000多亩，占全岛面积的74%，森林覆盖率高达87%，树种以黑松为主。
据说在树林中有百余头野生梅花鹿，我们时时刻刻擦亮眼睛寻找着，每个人都不敢说笑，生怕惊扰它们。
功夫不负有心人，我们遇到了不是一只，而是一群，长着雄壮鹿角的公鹿与几只母鹿在树林间穿行。
海边设有一个凉亭，供游人休息赏景。
沿着小路顺势而下就可以到达海边，风浪大石必惊涛拍岸，每个石头上都爬满了各种蜗牛和类生蚝的生物。带上小桶，今晚的晚饭就有着落了。
我们找到了可爱的小螃蟹，当然在我的建议下放生了。还找到了很多被海水冲刷干净，晶莹剔透的好看的石头。
那辽阔的海平面，深深浅浅的蓝，与渐浓的秋意打了个照面。
小路上散落着无数颗完整的松塔，呈现出金银铜三种颜色。在城市中这样的场景是多么难见，带上孩子，来一场另类的“采摘”。松塔带回家可以用银色、金色的颜料涂抹外表，做成很森系的装饰品，用心观察的话，你会在很多咖啡馆里发现它们的身影。
这里也是适合拍唯美大片儿的地方，只是记得如果你要来的话，多穿些厚衣裳哦。
雨中夕阳
爱雨的人总能和雨有个相逢，走到刘公岛前面的铁桥岸边，大陆的云彩已经厚得透不过气，岛上的天却还是忽明忽暗，蓝天蓝云白云粉云交相呼应。
每一个人都看过下雨，却不是每一个人都看过雨从云彩中掉下的样子。这就是只缘身在此山中吧。
它是我见过的北方最美的海岛，有着平和中最动人的夕阳。一边是灿烂的晚霞，一边是阴云密布。雨水把云彩都拉下了天际。
太阳躲在山后避雨，分不清这是傍晚还是艳阳初升。
自然之美也美在转瞬即逝，如此浓密的天气可遇而不可求。你看那召之不来挥之不去的云层，与太阳交手，天际被染成了玫红又镶了金边儿。
第二天此时我们再来，晴空万里早已没有了这样的韵味。不过7个人有7个人的玩儿法，坐在石头台子上，拍下了我们动作最统一的一张背影。

【美食】

去山东旅行，海鲜当是大家心心念念的一环，那么在山东能吃到什么海鲜，怎么吃，这次我们的威海之行给了我答案，不仅让我们打开了眼界，还亲手赶海，捡拾了一桶海鲜，仿佛大海就是厨房和粮仓，那种富足和满足感爆棚，乐趣多多。
中国高尔夫博物馆
在刘公岛一片临海的开阔地，绿草茵茵，微微的波澜起伏，仿佛是一个高尔夫球场。没错，它居然真的是！而且是中国现存最古老的高尔夫球场。
据资料说，1898-1930年威海卫成为了英国的租借地，在1898-1940年，刘公岛成为了英国皇家海军的驻地。英租时期威海卫的两任主要行政长官都来自现代高尔夫的故乡苏格兰，一位是骆仁廷，刘公岛威海卫高尔夫俱乐部的首任会长，另一位就是末代皇帝溥仪的英文老师庄士敦。
1901年英国海军在刘公岛东部东泓地带修建了有苏格兰传统林克斯风格的标准9洞高尔夫球场，并且成立了威海卫高尔夫俱乐部，后来又改称“皇家海军联合服务高尔夫俱乐部”。
威海的高尔夫运动有着浓厚的苏格兰味道，比赛采用苏格兰圣安德鲁斯皇家古老高尔夫俱乐部的规则，所用器械全部从英国进口。
1930年威海收归中国后，东门外和刘公岛两处高尔夫球场，分别由英侨和英国海军续租使用。英军掌控的刘公岛高尔夫球场一直都热闹非凡，每到暑期的时候，来打球的海军军官和他们的家属就络绎不绝，光1933年夏天，会员就高大345人。
威海当地居民管打高尔夫球叫“打蛋”，胡桃木高尔夫球杆为“蛋棍”。1940年11月15号英国海军撤出了刘公岛，太平洋战争爆发后，旅威的英侨都被日军投进了集中营，威海风行一时的高尔夫运动就此走向没落。
高尔夫传入中国后，就威海而言不仅起步早，且直到1933年，英国驻威领事仍在劝诫威海人应当“为这里作为中国仅有的几块高尔夫绿地之一而骄傲”，印证了高尔夫在威海风行的程度，和在当时中国的位置。威海至少应该是现代高尔夫传入中国最早的地区之一。
2009年，刘公岛管委会为了恢复历史原貌，在原址重新修建和恢复了9洞高尔夫球场，全长3029米，并修建了一座现代化的高尔夫会所。刘公岛高尔夫球场因此成为中国最早建立和唯一现存的苏格兰林克斯风格的高尔夫球场，成为“中国高尔夫朝圣者的圣地”和“中国的圣安德鲁斯”，这座建筑就成为了中国高尔夫博物馆。
走进这座中国高尔夫博物馆，不仅可以了解世界最古老的高尔夫历史，看到当年溥仪和妹妹们，以及梅兰芳打高尔夫球的英姿。
更重要的，这座当年英国殖民者娱乐的地方，如今成为了刘公岛上最大的餐厅，享受刘公岛特色大餐的地方。
很多人提到山东，都会不由自主地想到海鲜，来刘公岛就可以满足你对海鲜的想象。
这里的人们吃海鲜不用什么碗筷刀叉，直接把各种各样的海鲜铺在旋转的玻璃餐台上，一大片海鲜瞬间变成海鲜的海洋。
海鲜种类繁多，比如蛏子、钉螺、红红火火的赤甲红螃蟹、花蛤、夏威夷贝、海虹、丰满的蚝（山东人管它叫海蛎子，这里的口音也被称为海蛎子味儿）……
这些海鲜都仅仅是用清蒸的方式弄熟，不加任何调料，保持海鲜本身的味道，蘸上醋和芥末之后，滋味更加浓郁，吃海鲜吃到饱是什么感受，这一片海鲜的海洋给你答案。
大家肯定会发现餐桌中间巨大的馒头，那是山东有名的花饽饽，蓬松得像面包，发面的香气甚至盖过了海鲜的肥美，我说你不信，一个面团能比海鲜好吃？你吃了就知道。在北京山东的馒头店可都是排队的，一人都买好几斤。
要想吃这桌海鲜需要提前预订，按照你订的海鲜种类多少，以及重量定价格，而且海鲜是时价，所以没办法说价格了，总之比在大城市，在京便宜海了。
即便正餐不吃海鲜饕餮盛宴，也可以点到很多品种单做的海鲜。
到山东当然要尝尝货真价实的山东大葱，白白净净的葱白儿，蘸着山东的酱，配上花饽饽，简直享受，大葱也变成了水果一般的甜辣滋味。
还有餐后吃的一碗看起来清汤寡水的宽面，面条柔软不糟，咸淡适中。面上不放酱油醋，而是配上一小碗虾和鸡蛋在一起做成的虾酱，有着浓浓的海的味道。

刘公岛人家
刘公岛上好吃的还真不少，我们在刘公岛景区内的丁公路，就是背向码头右前方的地方，最边儿上的这家餐厅吃饭，店名是刘公岛人家，它家主打柴锅鱼和贴饼子。炒菜味道不错，推荐点一盆热汤面哦。
另外点餐前确认好菜价。
泰祥鲅鱼水饺
我的多年老同事是山东长岛人，常听她说小时候在岛上生活的场景。傍晚该吃饭了，她妈就叫她拿着小桶去海边捡海鲜，仿佛大海就是她家的仓库，取之不尽用之不竭。想象着满沙滩的贝壳、螃蟹、海螺……都是自己的感觉，赶海二字，适中在我脑海里，成为了一种特别期待又没实现的向往。
世界旅行多次也没能体验到赶海，这一次刘公岛之行终于圆了我这个梦。
傍晚，我们来到海边，我们和餐厅的大姐们一起捡拾生蚝、螃蟹、各种贝类、海螺……
天色渐暗，看谁眼尖，这是一场眼力和时间的竞赛。
红边白色的小桶那是头天我们路过海边时看到别人的战果，硕果累累啊，满满一桶梭子蟹。但到次日我们来的时候，只捡到了很少的两只。
周末的时候，岛上的驻军也休息了。你会发现解放军也来这里捡海鲜，拿回军营饱餐一顿，改善生活。
带着沉甸甸的战利品，走几步就到了晚餐的地点，泰祥鲅鱼水饺餐厅，让店里的厨师处理制作。这家餐厅正对着刘公岛码头，非常好找，据说是岛上很火的一家。
这里可以吃到山东省级非物质文化遗产，著名的鲅鱼饺子，它是山东饺子中是出镜率最高的一种。
以新鲜的鲅鱼为原料，加上适量的五花肉馅儿和韭菜调制而成，调制鲅鱼饺子馅，“搅”力是非常重要的，鲅鱼去刺后剁成泥，用筷子朝着一个方向搅动，不换方向，这样搅出来的鲅鱼馅Q弹有力。
鲅鱼的肉质细腻、味道鲜美又营养丰富，富含蛋白质、维生素A、矿物质等营养元素，有补气、平咳的作用，对体弱咳喘有一定疗效；鲅鱼还有提神和防衰老等食疗功能，经常食用对治疗贫血、早衰、营养不良、产后虚弱和神经衰弱等症会有一定辅助疗效。
我们终于在这儿喝到了传说中的，网上看过多次的崂山可乐，因为里边加入了乌枣，丁香，良姜，白芷，砂仁等草药，所以有健脾暖胃，祛寒燥湿，散风除湿，通窍止痛的功效，味道类似可乐，也有一些草药的味道，还挺好喝的。

【住】

刘公岛英商海滨别墅
之前已经讲述过了刘公岛的风景和美食，那么在岛上可以住哪儿呢。岛上有两处特色的住宿，一个是豪华型，充满历史感的别墅——刘公岛英商海滨别墅，这是英租时期英国商人留下来的。
整栋别墅有三个大床房卧室，一个超大的英式古朴客厅，所有家具和陈设都带着浓浓的欧式韵味，仿佛走进了历史中。
别墅里用具和功能一应俱全，独立的厨房、餐厅，特别适合一大家子人或者团建，办活动、小晚会，一定别有风情。
别墅外边的小路直通海边，海浪拍打着礁石，礁石上布满了生蚝，人们可以在此捡拾自然和大海的馈赠。
小路的台阶旁有野生的枸杞，非常大又丰美，随手摘下一串儿，这是我第一次享受野生枸杞的滋味。
刘公岛宾馆
因为别墅只是整栋出租，所以人没有那么多资金不够雄厚的话，就可以选择刘公岛宾馆。大家在酒店预订网站上搜索到的刘公岛宾馆只有一个链接，实际上叫刘公岛宾馆的实际上有两个，一个在岛外，一个在岛内，是同一家。岛外的那个是普通的商务酒店，岛内的是特色炕房住宿。
酒店所在的平房区，原是英军的休养所，现在院儿内种满了无花果树，成熟的果实鲜嫩肥美，非常好吃。

【逛】

滨影农家院
刘公岛虽然是个景区，其实还是有原住民居住的。刘公岛东村一共有49户136人，常驻的四五十人，他们都在岛上从事服务行业。我们就探访了这家名为滨影农家院的主人，老大爷是岛上年纪最老的居民了，为我们讲述了他在岛上的生活。
他家陈设简朴，还用着当年的三屉柜橱。大爷还向我们透露了一个小秘密，岛上的那些野生梅花鹿，实际上当年是圈养的，一天因为他的疏忽，让梅花鹿跑了，于是久而久之就变成了野生的。
海峡两岸交流基地
从没有想过，刘公岛的海峡两岸交流基地里居然有两只超胖超好看的大熊猫，它们是来自四川雅安碧峰峡基地的一对国宝大熊猫，两个大宝背靠背，实际上谁也摸不到谁，两个孩子被一道玻璃墙阻隔。
我们看着饲养员给它们喂食竹子，它们吃得太香了，咔嚓咔嚓先把外皮剥下去，咬下鲜嫩的部分，再把外皮咬下去，周而复始，一会儿一根儿鲜笋就吃完了。
另外旁边还有个鹿园，那里有宝岛台湾赠送的梅花鹿和长鬃山羊，以及从江苏大丰来的国宝级珍稀动物麋鹿均在此安居。有一只鹿因为小时候身上沾染了人的气味，再送回母鹿身边，就不被认可了，所以是人给养大的，它不怕人，跟人很亲近，至今其他小伙伴仍旧另眼相看。
萝卜车
在刘公岛上能玩儿什么，最最推荐的就是这种慢悠悠的智慧萝卜，岛上工作人员管它叫“萝卜车”。很安全，车速很慢，急性子的可开不了哈，油门刹车是同一个，踩到底和完全松开都是停车，半踩是加油。租赁就在码头对面的空地上。
车内空间不大，每辆可以坐两个人，景区内的马路平坦，可以开着它环岛呢。
定远舰景区
定远舰景区不在岛上，而是在到外的刘公岛客运中心，要参观的话走的时候，或者还没进岛的时候去看。定远舰的主体是按原貌复制再现的清末北洋海军旗舰“定远”号。
历史上的“定远”舰是19世纪工业文明的产物，因为武备强大、防护完善，一度被誉为“亚洲第一巨舰”、“遍地球一等之铁甲舰”，见证象征了中国海洋上一段辉煌的往事。
这艘军舰又因浓浓的甲午风云和在甲午海战中的不屈表现，赢得了“永不沉没的‘定远’”之称，成为国人心目中的一座海上图腾。
复制的“定远”号纪念舰，完全依据历史资料按1：1比例进行复制建造，最大程度再现了历史原舰的风采。舰上各种武器装备一一复制，齐全逼真，许多著名的历史场景也都纷纷还原再现，如临其境。
登上这艘战舰，恍若跨越时空，百年往事历历在目，甲午英雄的故事、铁甲巨舰的雄风、黄海大战的惨烈、亚洲第一的强国梦都都在这里重现。
参观过程中才得知，小日本儿把定远舰的舵轮拿走当咖啡桌，铁甲做了门板，炮弹至今仍立在日本海军墓地前。定远舰是当年的航空母舰，听说现在威海湾北洋海军沉舰水下考古调查已经取得了阶段性成果，根据声呐探测和水下探摸，有很多处疑点，其中一个长50多米、宽20多米，有可能就是它！
它昂首静泊在威海湾畔，与刘公岛隔海相望的“定远”，龙旗猎猎、身未动威先行，仿佛蓄势待发，将要继续那未尽的航程。
期待定远返航！
岛上的英租时期老建筑
岛上遗留着英租时期的很多栋建筑，这一栋漂亮的红房子就是英国海军上将别墅旧址。
这一排房子原是英国海军的保龄球馆，可以想象他们当时的生活还挺多彩。现在这里已经成为刘公岛管委会的办公地点。
这个白房子是共济会会馆旧址。
刘公岛国家森林公园
刘公岛的森林公园内不仅有６座清朝时期古炮台，还有北洋海军忠魂碑、刘公像、刘公亭、动物园、观海楼、五花石、梅花鹿园、板疆石、钓鱼台、听涛亭等20多处自然景观。
乘坐缆车可以轻松上山，如果想爬山的话，可以从动物园的入口进去，走过高高的台阶，直通忠魂碑。
山顶的风光相当美，蓝色堪比地中海。
这里是山顶的炮台和军械库。
山上种着很多橡树，我们在路边捡了很多橡果。
中国甲午战争博物馆
中国甲午战争博物馆在刘公岛上其实是很多个地点集合起来的统称，比如有9座海岸炮台。以旅顺口口门为界，口东5座、口西4座，从东至西分别为：老蛎嘴后炮台、老蛎嘴炮台、摸珠礁炮台、黄金山副炮台、黄金山炮台、老虎尾炮台、威远炮台、蛮子营炮台、馒头山炮台。其中除老蛎嘴炮台为穹窑式外，其余均为露天炮台，共配置火炮58门，其中200mm口径以上9门。
还有这个陈列馆，馆内收藏展示着很多珍贵的历史文物，比如济远号前双主炮、"济远"铁锚、英国皇家海军训练用鱼雷等海底出水巨型文物，北洋海军将士委任状、望远镜、指南针、怀表等珍稀遗物，使观者重温历史，睹物思人，炮台上复制的甲午战争时期克虏伯大炮，外观逼真，功能齐备，可以操纵演练，再现北洋水师辉煌历史，增进国防海防知识。
陈列馆以《甲午战争：1894—1895》为基本陈列，展出珍贵甲午战争历史图片650多幅，复制了大量甲午战争时期的武器装备，还原再现了多个超写实人物塑像场景，如：“金州曲氏一家投井场景”、“李鸿章在马关谈判场景”等，开辟国内首个“黄海海战”3D影视厅，声光电与多媒体复合再现“威海卫保卫战”震撼人心的战争场面，还有大量反映甲午战争的巨幅油画和巨型雕塑。
整个展馆分为“序厅”、“甲午战前的中国和日本”、“甲午战争”、“深渊与抗争”、“尾厅”五个部分，非常大。
​还有水师学堂，丁汝昌寓所，龙王庙，刘公庙、北洋海军将士纪念馆。
户外区域还有这个陈列着巨大的打捞上来的舰船遗迹。
