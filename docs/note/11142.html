＃悦野旅行＃走进秀丽自然风貌，夜游璀璨珠江

游记 客栈民宿 小众 徒步 摄影 自然奇观 2018/10/07

前言&预告

从化，
是广州的后花园，
是中国温泉之都，
是北回归线上的明珠。


从化有着优越的地理位置和生态环境，东边与增城区接壤，南跟白云区毗邻，西和清远市、花都区交界，北面同佛冈、新丰县相连。无论从哪个位置出发，都十分方便。

北回归线横跨境内南端的太平镇，气候温和，雨量充沛。境内有100多个湖泊水库，12万公顷青山以及森林覆盖率67.2%。成全了拥有“中国最佳旅游度假胜地”等众多美誉。除此之外，还有近几年火热起来的从化溪头村，素有“广东最美乡村”的美誉，绝佳的自然环境让人蠢蠢欲动。

最美乡村溪头村

溪头村早已安利很久，无论在电视旅游节目上还是在网络文章中，不少它的身影，然而让我把它记住的，是源于一张星空的图片，图中是一个宁静安详的村庄，一间间古朴的红墙瓦房，屋顶是南方独特的三角形，夜色下村庄的点点星火和星空中闪烁的繁星交相辉映，有种说不出的静谧美。

而图片的注释，写着拍摄地是广州的溪头村，那时候由衷地喜欢上这个地方，没想到在高楼林立的省会，还有这么难得的自然生态景象。平日里抬头瞭望夜空，那一闪一闪的，只有高层建筑上的航空示警灯，以及飞机的闪烁指示灯，而星星的踪影，却无迹可寻。
驱车前往溪头村的路上，会有一段盘山公路，一边靠山一边是悬崖，蜿蜒而上，虽不比四川等山区道路险峻，但还是要分外谨慎，双向共两车道，还会有好多个急弯，注意对向来车。
人生中总会有无数个惊喜的瞬间让你热泪盈眶，旅行上也会遇到一些美景让你欣喜若狂，而这些美好的景色，多半是来源于自然的鬼斧神工。转向下山路的时候，突然看到群山中间，有一汪碧绿的河水，Tiffany般的蓝绿色，如同山峦灵动的眼睛，让人着迷，旁边还有一间间独立的三角形小屋，此景就像油彩画中的世界。只可惜在山路中不宜停车，无法定格这画面，就让它留存在脑海。
带着期待终于来到了溪头村，人类都有一双发现美的眼睛，在众多的宣传广告中，这里难免变得有些商业化，一辆又一辆的旅游大巴，进入牌坊前的一段路，就有食肆在招揽生意，这里游客虽多，倒还不缺停车场，随着大哥的热情，就停在了其中一块空地，收费比市区也便宜多了，然而停好车没走几步，就发现有一个大型的游客停车场，似乎还不收费的，突然就感觉上当了！所以警惕后面的游客，不用担心没有车位，牌坊前几十米处就有一个大型的景区停车场，目前还不设收费。
一条约2米宽的小路，基本没有支路，一直到尽头两边都是摆满各种货品，跟大致到旅游区都有点相似，街道两旁一栋栋到小房子显然有些年纪了，而居住到基本都是这里长大到居民。
看到最多的是腊肉腊肠和鱼干，所谓秋风起晒腊味，广东人多都会有这喜好，一阵风吹过，也还真香。
除了一些干货，一路还能到自家居民在做竹木工艺品，边做边卖，这个是最具有当地代表性的。直接用当地的竹子进行加工制作，就地取材，又能为当地人带来不菲的收入。
走到尽头，就能看到这汪半圆的谭，沿着半圆，架起了一个个蓝色的小帐篷，自家房子不在主干道的，估计就担着竹杠来到这摆摊了，这热闹的架势，有点是旧时的市集。
走进一家农家店，品尝一筒原滋原味的竹筒饭，打卡盖子后菜香四溢。竹筒饭的做法颇具有野炊特点，用一节竹筒 ，装进适量的米和水，放在火堆中烤熟即可，很适合野外。
桃胶也是这里的特产，村民们自产自销，除了可以看到在售卖的，也能看到还摆在路边晒的哦。
千里迢迢来到这里，当然还有目的地，就为了这一碗阿娣家的豆腐花。一家82年的老字号了，来来往往的人民都会停下来品尝一碗再继续前行。
豆腐花香滑可口，可以拌红糖也可以拌着姜汁吃，吃完一碗还觉得未够瘾，那就来两！
一路穿过一条条小巷子，突然眼前葱绿，青山环抱，奇石错落，周围的景色浑然一体。
继续前行就来到了溪头谷口，迎面就是一座小山，山上林茂竹翠，中间裂开一条沟谷，一道银白色的溪流瀑布就从沟谷里飞流而下，泻落下面一个碧波荡漾的潭。潭水冰清玉洁，碧绿湛蓝，手忍不住去抚摸这潭清澈见底到水流，水凉丝丝的。
小孩和狗都忍不住来到潭水的怀抱中，这天然的山水真的很难得。
山水随着小溪流出，水灵动了整座山。在阳光的照射下，显得波光潋滟。

东南亚风的房间

与外婆家结缘于网络上的一张图片，其实不是图片特别漂亮，也不是摄影师拍摄的，只是图片中宁静安逸的环境正合我心意，向往度过一个舒适惬意的周末已久，便二话不说预订了外婆家，只为寻找市区难得的悠闲清净。

前往度假村的路上，从市区进来发现路越走越偏僻，车子似乎开在了乡间小路上，两旁是昏暗的路灯，偶尔对象驶来的车辆闪着远光灯，才让我们确定这不是一个稀无人烟的地方，再看一下导航上的地址：广州从化区江埔街锦一村外婆大道37号。确认无误，就只能悬着心即使往前开去。
一路进来让我们对这个度假区多了几分期待，好不容易看到漆黑的夜空下出现了亮光，“广州外婆家森林生态度假区”，终于到达目的地！
门卫放行后继续驱车向上，经过了一条上坡路，早上终于看到了度假区的庐山真面目，园内群峰环抱，植被覆盖率高，山水资源丰富，可谓是好山好水好风景。
预定的房型是高级双床房，这间客房与前台不在同一栋楼层，而是位于山坡上的外公楼，前台check in效率十分高，快速核对了预定姓名了给房卡我入住，连押金都不需要交。
每一个房间的命名都极具诗意，符合度假区的意境。我们入住的房间是「留月阁」。
推开房门觉得十分惊喜，大大的落地玻璃窗，外面就是树林，一抹碧绿印在眼里，早上有阳光照射的时候树影婆娑，葱葱郁郁，让人觉得很贴心，而且十分亲近自然。
房间大部分都是木质的家居，与度假区的主题十分贴切，木质的桌椅、木质的屏风、木质的衣柜和床板。有种还原原始时代的感觉。
值得一提的是双床房竟然两张床都是1.8米的大床，具体尺寸我分不清，反正挺大的。两床中间是用屏风隔开，很复古，感觉房间睡4个人，一家四口出行也是妥妥的。床上放着四个中高枕头，以及一个复古的红色抱枕。床垫软硬适中，一整晚睡眠质量很高。
再来看左边，一个落地的摇椅，摆放在墙角，坐在摇椅上，看一本书，品一壶茶，晒着午后的阳光，看着青葱的数目，这就是外婆家的日子~
房间面积约40多平方米左右，墙壁贴着橙黄色的墙纸，图案像是铜钱。打开门，门背后就是鞋柜，里面放着四双非一次性拖鞋，鞋柜旁边就是衣柜了，衣柜里面除了放有浴袍，还有两张稍微薄一点的棉被，以备客人不时之需，的确在现在这个季节，昼夜温差还是比较大的。
两张床的旁边都分别放有一张床头桌，上面有房间灯光的控制按钮，十分便利。
早上醒来，玻璃窗外数目葱葱郁郁，加上这木质感的房子，仿佛置身于东南亚热带雨林的树屋之中。
卫生间可圈可点，整体面积比较大，干湿区分离，单单是湿区的面积已经不小，约有平时使用的1.5倍。抬头看，顶上是玻璃设计，外面看到的都是树林。
早上卫生间光线十分充足，其中一半有帘子遮住了，另外一边估计也是可以合上的，只是没发现帘子的开关控制。
其次，酒店十分注重环保，配备的洗漱用品是非一次性的，而是牙博士的旅行套装，刷完牙可以直接打包带走，留待下一次出行使用，传递环保信息。

广州外婆家森林生态度假区

外婆家生态度假村内配有一栋三层楼的中餐厅，从客房区过来直接连通餐厅的三楼，门口的对联十分有诗意。这里可提供早餐、午餐和晚餐，都是一些粤菜，月均100左右。味道比较地道，早餐的话是现做的，普通的炒粉炒面和粥类。
餐厅门口有一副用木刻制的对联，「放怀风月里，胡生笑谭中」十分有诗意，emm...如有错误请纠正。
餐厅里面都是桃红色木质的桌子和长板凳，窗外满眼青葱，在这样的地方用餐倍感诗情画意，仿若古代里的亭台楼阁。
度假区种满了树和竹子，在阳光照射下，树影婆纱，此时深呼吸一口，满满都是自然的气息。
沿着石阶走，所到之处都能给人惊喜，或是桥下的小桥流水，或是岩石边的溪流瀑布，或是眼前这如穿越般的滴翠亭。
这充满田园风格的大院子，原来就是客房的前庭，酒店客房分几个片区，每一块都似乎是独立的，围绕着自然的主题，但风格各不不相同，这么大一块活动空间，不得不由衷感叹， 这真是名副其实的生态度假区。围坐一团晒太阳闲聊的人们，印证了闲得偷生半日闲的惬意。
最让我意外的还是看到了蓝孔雀，笼子内一公一母，看来是要让它们在这繁衍后代了。自从小学时期去动物园开过孔雀之后，就再也没有看过了，已经对孔雀陌生，印象中就是有很美的羽毛，这次近距离的观赏，刷新了我的认知。
一首周杰伦的一路向北，竟然成功让孔雀开屏，看来小子也是杰伦的粉丝哦！孔雀羽毛展开的一瞬间，真的有把我惊艳到了。华丽的羽毛，上面一个个的图案，就像一颗颗翡翠的蓝宝石。
孔雀让我看得目不转睛，还看到了了孔雀求偶的过程，求偶时孔雀尾屏竖起稍微倾向前，尾羽不动地抖动，而且频率非常高，发出清脆的声响，简直太cool了！
度假区内泳池肯定少不了，这么大的靠山泳池比较难得，在群山怀抱中畅泳，别有一番⻛风味，而且还看到有温泉池在建。

天然氧吧大夫山

跨越了大半个广州从北到南，只为再一次亲近自然，大夫山是一个生态型森林公园，无论是情侣还是一家老少都十分适合，这里有山有湖，山水秀丽，自然的风光再一次让我着了迷。大夫山可不仅仅是爬山，骑行的更占大多数，我们选择徒步上山，一路都是骑自行车的大人小孩。
从大夫山的北门进来，走不远就能看见云岫湖，一排排挺拔的大树格外俊俏，把湖水映照得碧绿无比。
有这么美丽的背景，我已经忍不住开始摆拍起来，虽然今天特意穿了一身休闲装来爬山，但也毫无违和感。
金秋十月，广东的天气开始有丝丝凉意，一早一晚特别明显，在市区中心温度可能会稍微高一点，但是在郊区会略低，所以10月出行一定要备一件外套，风衣再也合适不过，此行带了法国品牌Lafuma(乐飞叶），对这件衣服真的爱不释手，完美的融合了户外与生活，自然与城市。
一开始我还真小看了Lafuma这件衣服，后来晚上的小插曲让我真心膜拜，防水性能不多说，晚上下起了蒙蒙细雨，全当雨衣使了，用餐完毕时，服务员收碗碟不小心把菜汁洒到衣服上，我心想完了，白花花的衣服黄了一片，油很难洗掉，没想到纸巾一擦都干净了，为这高科技含量的衣服质量点赞。
话题扯远了，继续回来看山清水秀的大夫山。
公园还有多个湖泊，聚秀湖、红山湖、竹山湖和凤山湖等，云岫湖只是其中之一。
秋天来这儿也非常适合拍照哦，背后的树叶子红了一片，也是另外一种美。
无意中发现了湖中有一只游动的东西，竟然是一只鸭子，看来还是挺机灵的，竟然完美的躲过了我的镜头。
走在园内，红的蓝的黄的帐篷各自扎好了营地，大人舒适地躺在帐篷内，小孩则在外头嬉戏，其乐融融，又是大自然里的一幅风景画。

夜游珠江

璀璨的珠江夜景和婀娜多姿的小蛮腰吸引了不少游客慕名而来，在广州，也要算这两景点最有名气了。而这里的珠江夜游其实不仅仅是西江，还包括东江、西江和北江等支流，汇合至广州河段，游船的话全程会途经海印桥、广州大桥、海心沙、广州塔、猎德大桥几个著名的景点，全程约70分钟左右。
每当暮色降临，广州塔对面的CBD大楼逐渐亮起了灯，就像花儿竞相开放，似乎也为广州塔而绽放。
隔海眺望，对面是CBD，而背后就是广州塔，小蛮腰与珠江新城、花城广场、海心沙岛隔江相望，由于高度优势，在广州很多地方都能开到它的身影，当然啦，站在此地算是最佳的观赏位置。
广州塔加上146米高的天线桅杆，总高度600米，是目前中国第一高塔，世界第二高塔，仅次于东京晴空塔。平日广州塔是有多种颜色变换，而在国庆节这一天，为衬托出喜庆之日，仅有红色和彩色两种颜色哦。
猎德大桥
游船分别从大沙头码头和天字码头起航，路过码头时刚好看到游船靠岸下客。
在珠江边行走，就像观赏游船show，一艘艘颜色各异，形态万千的游船在江上行驶，也在这江水映照得五颜六色，色彩纷呈。
当中要数这艘金色的南海神最出众了，金色的船声特别亮眼，打造得好像一个财神爷的形象。
游走完 “珠江走廊”也算是此程完美的ending，夜晚两岸灯光璀璨，动人夺目，白天在丽日蓝天下也能观赏到另一翻秀丽的景色哦。广州美景背后的历史耐人寻味，还值得人们一一探索，去阅尽广州说不尽的文化风情。
