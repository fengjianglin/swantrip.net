沙巴怎么玩？
看了上面的预告片是不是无比心动了呢？
没错，这次北斗旅行的地方就是位于马来西亚的沙巴。
说起沙巴可能很多人并不是十分熟悉，沙巴州是位于马来西亚东部的一个州，早期称为北婆罗洲。这里位于赤道以北6度的热带气旋区之下，巧妙地避过了台风圈的侵袭，因此还拥有“风下之乡”的美誉。
沙巴州很大，又分为东西沙巴，举世闻名的仙本那就位于东沙巴，而我们此行的目的则是位于西沙巴的亚庇，即沙巴州首府。
沙巴州的面积很大，典型的地广人稀，旅游资源可以说是丰富至极。这里拥有世界上最古老的热带雨林，稀有的动物世界，罕见的花草植物，还有仿佛永无边境的海洋生态以及丰富多彩的珊瑚礁。这里不但是潜水爱好者的天堂，更是探险者的乐土，你可以登上巍峨的深山，还可以深入丛林，去与当地稀有的长鼻猴打个交道。并且，作为一个多种族和谐共处的地方，沙巴更是个多元文化大融合的聚居地，独特而又淳朴的原住民文化加之多姿多彩的传统节日庆典，绝对能给每一个游客带来非同一般的体验。

避开了国庆旅游的两大看点：人山与人海，选择在十一之前一周前往沙巴度假绝对是明智的选择。
还记得第一天见到导游的时候就问我们，是不是10月1日起是中国的假期，沙巴这所有酒店已经全部订满了！不禁感慨，沙巴不愧为世界著名的度假天堂，果真火爆。
选择沙巴作为度假目的地，除了是签约旅行家考察团的邀请之外，本身也对于这里的海岸线充满了期待和向往。

航空公司安排住在机场不远处的一家度假酒店，好像挺高大上的，可惜没有时间享受，没睡多久就要继续赶路前往机场啦。
这里补充一点，东南亚的室内空调简直冷的可怕，无论是吉隆坡还是亚庇的机场，就像是一个巨型的冰窖……我这穿着短裤短袖就出发的人，在机场候机的几个时间段简直是一场噩梦……小伙伴们来东南亚玩记得要带一件长裤，最好加一件薄外套哦，不然一定会被冻死……
沙巴亚庇 多元文化大融合
一路辗转，终于在23日中午顺利抵达亚庇，这也是此次沙巴之旅的主要目的地。与同行的小伙伴们顺利会师，第一件事情，必然是吃饭！终于可以开始尽情享用东南亚的美食啦！
第一天中午，我们前往位于亚庇一家当地的海鲜面汤店，叫做Fatt Kee Seafood Restaurant（发记鱼杂）。可以说，这里的面料和鱼杂都是亚庇数一数二的，选用独特的番茄汤底，加上自选的鱼头、鱼杂、鱼片，十分美味。强烈推荐鱼头，很大一份。
可能是前一天飞机延误导致晚上没有吃够，解决了一大碗面之后，我们再接再厉，来到边上的另一家餐厅Kah Hiong Ngiu Chap（家乡牛什）。类似一家大排档的餐厅，一份简单牛杂面就能吃到牛肉、牛肉丸、牛肚、牛腱、牛舌以及美味的白萝卜。
酒足饭饱，驱车前往位于亚庇郊区处的马里马里文化村。看着车窗外的风景逐渐从城市变成村庄，再到不断飞驰而过的山林，听着导游讲述着当地土著人和猎头族的传说，这才反应过来，原来我已经来到了另外一个国度——马来西亚沙巴。
== 马里马里文化村 ==
旅行的第一站，我们来到了距离亚庇市中心约半小时车程的文化村。文化村的名字十分有趣，在当地语言中，“马里马里”就是“来呀来”的意思，有欢迎我们到来的意思，念起来更带着一点诙谐。
在沙巴州，有着大片的原始森林，更有许多是未经开发的热带雨林，仿若是亚洲的亚马逊，这里面充满了神秘与危险，同样的，这片热带雨林中也生活着无数的原始部落人群。在这个文化村里，则主要展示了人口数量较多的五大原住民的生活风俗。
文化村坐落在一条狭窄的河道旁，走过大门经过一道吊桥，展现在眼前的就是原始的森林世界，以及点缀在绿树之中的茅草屋。在文化村里，游客们可以参观沙巴州最具特色的五大原住民房屋，感受他们的生活习俗和文化氛围，并且亲自体验各种民族传统技艺，比如竹筒饭、酿酒……
原住民自酿的米酒，跟我们农家自酿的甜酒有点相似，略带甘甜。团队中的酒鬼领队面面大姐喝过之后直接放弃治疗了，当天晚上在酒店硬是拉着我们陪她喝酒……
拥有三百多万人口的沙巴州，共有三十多个种族，八十多种方言，可以说是马来西亚境内文化最多元的州。在这片美丽的土地上，不同的当地原住民族群友好融洽和睦共处，许多族群至今能保持着他们世代相传的古老传统。
例如龙古氏族（Rungus）世世代代居住的长屋，这是由竹子搭建而成的一种高脚楼，往往一个家族的男女老少全部居住在这一个长长的房屋当中。
在文化村里，游客可以亲身体验到当地氏族古老传承的各种生活技艺，比如用竹子酿制蜂蜜、古老族群的鳄鱼图腾、有点像小浣熊干脆面的炸饼，还有吹剑、钻木取火、树皮制作的衣裳等，一次游览下来，绝对让你大饱眼福。
第一次体验这种原始的吹剑，看我们团队最年轻的小美女吹剑的姿势，还是带有一点霸气的。嗯，多练练，以后可以吹的更厉害……
原始部落传统的盾牌和刀剑制作，还有皮革打造的衣裳，坚韧无比，就是下面这两位小骚包，装逼的动作能再浮夸点么？
在文化村了最为惊人的是沙巴原住民中有众多猎头族，不知道是否传说夸大其词，但是据说这些种族会猎取外人的头颅作为装饰，在他们的居民房里也能看到许多头骨制成的挂饰。而有些氏族还会将族人的骸骨装在一个个坛子里，存于族群驻地的大树之上，十分神奇。
一路上导游还给我们讲了许多关于猎头族的故事，其中比较有趣的是跟原住民之间不能说byebye，否则很可能被他们猎取了脑袋……是不是听着有点毛骨悚然呢……
在欣赏过五大原住民的衣食住行之后，大伙儿终于可以坐在阴凉处，喝上一杯东南亚风味的柠檬水，欣赏他们不同部落的传统舞蹈。
东南亚的原住民也是美女众多哦，有没有人想要来当压寨女婿的？
表演最后的重头戏是竹竿舞，游客也可以上去体验玩耍，就是怎么感觉跟云南佤族的竹竿舞很像呢？
在欣赏完精彩的表演后，还可以继续在文化村中享用当地原住民精心准备的点心、下午茶，为这次的观光画上一个美好的句号。

== 普陀寺 ==
参观完马里马里文化村，意犹未尽，我们开启了下一段行程，亚庇的城市观光，由于时间的关系，我们主要参观了这里两个著名的寺庙。
沙巴普陀寺（Puh Toh Tze Temple），这是由当地华侨捐赠修建的一座佛教寺庙，始建于1980年，沿袭了中式寺庙的传统建筑风格，是当地华人的佛门圣地。据说，寺庙里的建筑原料全是从中国运过来的。
在寺庙的大门口站立着一座巨大的观音像，似乎和国内的观音像有些许不同呢。
普陀寺距离亚庇市区约15km，车程在15分钟左右，可能是远离闹市区的缘故，这里显得十分清幽，更突显佛门圣地的感觉。这座寺庙的设计风格与台湾的寺庙十分接近，从天王殿步入，再来到大雄宝殿。
普陀寺的铜钟，运气好的话或许可以看到寺院人员撞钟。这座巨钟是由当地华人以及世界各地的信徒合力捐赠的，铜钟上刻有各方捐赠人士的大名。
作为猫奴，一定是逢猫必拍的！
此行的年轻小美女一枚——小V童鞋，据说是参加鞋厂七夕活动抽中了大奖免费来玩的，然后跟另一个同样抽中大奖小帅哥一起来的，小帅哥自称大V，然后……然后这两人就这样组成了couple！是真的组成了couple！！！什么时候我也能参加这样的活动？！既解决单身问题，连蜜月也解决了……
== 水上清真寺 ==
接下来来看看坐落在里卡士湾，距离普陀寺不远，于1997年建造，目前是亚庇最大的回教堂。
清真寺又被称作“水上清真寺”，因其建立在一片人工湖上方，典型的伊斯兰教建筑特色，白色的高墙配上蓝色的巨型圆顶，倒映在湖水当中，与天空中的蓝天白云融为一体。
清真寺内部空间巨大，可同时容纳1万2千多名回教信徒，同时，这里还是著名的日落观赏圣地，在夕阳下染上一层金光的水上清真寺，一定能让每个来到此地的游客此生难忘。
清真寺是对外开放的，但是进入内部参观需要花3马币以租借回教服装，作为屌丝我们还是选择在外面拍拍照就好了。
再来看看从空中航拍的清真寺，是不是觉得更加震撼？！
世界三大最美落日地
第一天傍晚，听说我们将会去欣赏传说中世界三大最美落日的观赏地，同时也将是此次我们在亚庇市居住的酒店：香格里拉丹绒亚路酒店。

酒店内还有一个巨大的无边泳池，放松身心的好地方。
面面一路上一直说要拍比基尼美女，可是我们在泳池等了好久都没有遇上比基尼美女，怎么都是大叔和小孩子居多呢……没办法，只好扒几张酒店的宣传照给大家饱饱眼福了。
这里的度假设施真心完善，无论走到哪都有极其浪漫休闲的躺椅可以坐着晒晒日光浴，然后谈谈情、说说爱。让我们从天空的角度看看酒店的泳池和娱乐实施吧。
这是隶属于酒店的一片私人海滩，远处就是最美落日的地方啦，海滩左边则是举世闻名的落日酒吧。
重头戏来了，先来看几张落日酒吧的宣传照，是不是美的不要不要？这里每到黄昏时分都会有无数游客争先来此欣赏落日，但是就如酒吧则必须是酒店顾客，并且要提前预约座位的。看看那浪漫无比的海边躺椅，心动不心动？那么，心动不如赶快行动！
这次在亚庇住的几个晚上，其中两晚正好有时间可以来此落日酒吧欣赏落日，看着漫天的火烧云，简直美哭了！据酒店人员介绍，这还只是最为普通的火烧云，运气好的时候，可以看到染红整个天际的火烧云绝景。
看完火烧云，喝完鸡尾酒，大家也都已经饥肠辘辘了，一天的文化体验还真是耗费体力和脑力啊。第一天晚上在酒店自助餐厅用餐，同时这里也是大家每天享用自助早餐的地方。

街头市井 亚庇城市闲逛记
亚庇市，又称为哥打京那巴鲁（Kota Kinabalu），位于婆罗洲西北方的海岸去，是一个宁静而又生机勃勃的现代化城市。
亚庇市的起源可追溯至1881年，在第二次世界大战期间，城市的建筑几乎毁于一旦，如今可以看到许多当年战火遗留的废墟，而全亚庇市也只有三栋建筑逃过了炮火的洗礼。
这座艾京生钟楼便是三栋建筑之一，不是很高，很有特色的白色木质钟楼，就这样静静地矗立在亚庇市区山头之上，可以说是该城市中最为古老的建筑物，于1983年被列为历史文化古迹。
亚庇的0km标志，整个城市的战后重建，都是从这个点开始进行距离测量的。在这个标志的边上就是沙巴旅游局，同时也是沙巴最古老的三栋建筑物之一，于1991年起，称为沙巴旅游局办事处。游客们来此参观的同时别忘了拿一些旅游宣传资料哦。
一早上的重点闲逛景点：加雅街，更是当地著名的商业中心。每到周日，这条街道就会变得人满为患，周日集市的魅力不仅吸引众多当地居民来此采购，同样也吸引了无数游客来此一览沙巴最为接地气的周日集市。在这里感受沙巴的热闹景象，还可以淘到各种手工艺品、纪念品、小吃美食、盆栽宠物，一切你想的到和想不到的，应有尽有。
当地有名的印度老爷爷，会写多种语言的毛笔字，可以让他帮忙设计一个写着某人名字的书签，只要5RM哦，旅游出行，准备一个物美价廉的小手信送给朋友也是不错的选择。
老爷爷给自己取了一个中文名字，叫做黄菠萝，不知道是因为自己太黄，还是太爱菠萝，总之听到这名字大家都笑了……但不得不佩服老爷子的毛笔字水平，比我写的中文字还好看太多太多了……
这四个字有意思了……我们乍看之下都念成“我怕太太”，结果这位印度大叔义正言辞的告诉我们，这应该读作：“太太怕我”……
当地市场，可以买到各种蔬菜瓜果。
如果你来到亚庇正值周日，那么一定要来看看这个周日市集，感受一下沙巴市的人潮涌动，以及当地的人文文化都是不错的选择。
第一次见到这样子的鲜榨果汁机，就是不知道好不好喝，不敢尝试……
星辰大海 古达的银河璀璨
来到沙巴旅游，很少有人会前往古达（Kudat），甚至很少有人知道古达这个地方。
即使在这里玩了整整一天，即使在这里看到了灿烂的银河，品尝了美味的大餐
但是如果你问我古达是个怎样的地方，有什么好玩的？
那么，我依然感到很迷茫，不知从何说起
我只知道，这里真的很美，这也就足够了……
在即将离开亚庇之前，我们先去品尝了当地一种很有特色的手工面，味道简直没翻了！难以想象，一道简简单单的炒面竟然有如此美味！导致我们后来到部落吃午饭都没有任何食欲了。
一路向北，我们开启了前往沙巴最北角的旅程
汽车缓缓地驶出了城市，窗外的风景再次变成一片青翠的山林
午后的热带阳光，显得有些灼热，在昏昏欲睡之时，我们来到了途中的一个民俗村。
如今，沙巴州的许多原住民依然保持着他们世代相传的生活传统，居住在这样原始的木质高脚楼里面。
午餐是原住民自制的土菜，连米饭都是用芭蕉叶包着的。至于味道嘛……嗯……不做评价……
原住民的高脚楼，可以进入参观，如果民俗村中这栋可供参观的土楼已经对外开放，游客们还可以选择在这样的房间里体验一晚。走在这竹子之上，我们一直都在讨论，当地的小孩子在上面跑来跑去难道不会夹到脚么……
有几个小孩子穿着部落特色的民族服饰欢迎我们，十分热情地跟我们一直说hello，导致面面激动的跑去拿相机来拍娃……
原住民中年级最大的老奶奶，因为年龄无法外出打猎的老人们就这样留在家中，做一些手工艺品，这里出售的很多纪念品，全是当地老人家纯手工制成的。
从亚庇到古达的一路上，有许多的原住民和古老的村落，他们沿袭着世世代代传承的生活传统文化，保留了最朴实简单的生活态度。同样的，每一个村落也有他们各自的特产和特色，比如我们经过的一个蜜蜂的村落，这里的蜂蜜是他们的特产，也是引领全村发家致富的宝贝。
刚走进村子，其实最吸引大家眼球的不是那一排排蜂蜜，也不是那琳琅满目的手工艺品，而是这只骄傲的斗鸡。难得看到一只这么奇特的马来西亚土鸡，大家的第一反应竟然一致统一，那就是：这样的土鸡一定很好吃！
当地居民给我们展示养殖密封的蜂巢，据说这些蜜蜂是不会蜇人的，看我们的美女小V就这样不穿戴任何保护措施，只手提起这密密麻麻的蜂巢……
相信要是有密集恐惧症的人来此，绝对会很刺激。
古达，这个位居沙巴州最北角的地方距离亚庇市区还是相当远的，整个车程算起来要接近4小时左右。而我们此次居住的地方，就是位于古达著名的日落观赏点Tip of Borneo边上的：古达浪天涯度假村 。
嗯……不得不说，这个度假村的名字，很浪，很符合我的胃口。
度假村看起来十分的原始，没有什么现代化的建筑，在绿树环绕之间，点缀这几栋小木屋，加上一道木桥，就构成了整个度假村。
度假村的环境可以说是十分诱人的，来此一日绝对能让你享受到与世隔绝的宁静~
为何说是与世隔绝呢……因为古达这里的网络实在是……完全没网……
这是我们住的度假小屋，一人一间，环境不错，出门就能看海，唯一的缺点是蚊虫有点多，还好我有先见之明，带了驱蚊水！
小屋门口的猫咪，发现用vivo近距离拍摄也是很给力啊！
让你们看看什么叫真正的海景房！从度假小屋出来，走下山坡就是度假村的私人海滩啦，这里也是我们第二天出海去潜水的地方。
碧海蓝天，或许说的就是眼前的场景，早已分不清哪边是海水，哪边是天空，眼前，尽是一片蔚蓝色的世界。
搭建在大海中的小木屋，是不是就是渔民居住的地方？
这里拥有一大片的原始森林，从天空中航拍，可以看到树林与大海相互交接的美景
停靠在海上的渔船，前方是海岸边的珊瑚礁，而另一半则是大海。由此可见这里的大海有多么的清澈，竟然可以之间见底，真的是如玻璃般透亮。
如果说，丹绒亚路看到的日落是全沙巴州最为柔美的，那么，在古达欣赏的日落则是最为壮阔的，虽然没有看到令人叹为观止的火烧云，但是眼前千奇百怪的礁石，在夕阳的余晖下，在海浪的涛声中，却显得格外壮丽。
日落时分的古达海边，这里的海岸线经历了海水常年的冲刷，行程一个个大小不一的洞窟，无数迷你的虾蟹在这里穿行寄居，站在这片礁石中央，看着远方缓缓下落的红日，仿佛自己就在外星世界一般，显得格外魔幻。

生活在礁石洞穴中的小螃蟹，好像抓几只回去炒了……
再来几张夕阳下的古达海边风景，这样的美景，即使用手机都能轻松出大片。
傍晚时分，当月亮逐渐高悬，当晚霞逐渐褪去鲜红，我们也到了该告别这片海岸的时候，因为接下来还有一个重要的事情在等着我们，那就是海鲜大餐！
最后来一张海边的全景
当晚的美食，虽然没有什么令人惊艳的摆盘，但是几道简单的海鲜，就另得我们每个人都食指大动，并且，这里的食材实在是太新鲜啦，而且分量又足，简直美味！
说实话，这次难得来海岛旅游，我只想好好地度假休闲，从此再也不做风光狗了。然而，无奈的是，古达这里没有网啊！于是隔绝的生活让我们不得不走出房间，告别宅男，扛起相机脚架，再度来到下午拍摄的海岸边拍照。
虽然远方光污染强烈，但是不得不赞叹沙巴的空气清新，即便是星月齐鸣的夜晚，已然能够看到如此璀璨的银河。
我们的大V小V，一对小情侣，跑去草地上谈情数星星去了……我和面面两人只好默默地来到海边拍银河，谁让我们是风光狗呢……不过好在，对于风光狗的最大安慰就是，拍到值得珍藏的风光大片，而这张银河与月亮同台演奏的风光交响曲，就是这次古达之行最值得收藏的作品啦。
夜渐深，月儿早已隐藏起来，银河依旧高悬，感觉只要抬起头，伸出手，就能触摸到空中那触手可及的星河。
小情侣们谈情结束一同来到了海岸边，陈小羊也修完了图过来一起看星星，酒鬼面面则跑回度假村拿出我们事先买好的黑啤，就这样大家坐在悬崖边上，喝着酒，聊着天，看着星星在空中飞舞。
一直以来，都很喜欢旅行的感觉，旅途中的许多不期而遇，无论是人，亦或是事，而那一刻，业已忘却了颠沛流离的艰辛，忘却了一路走来的磨难和挫折，只将曾经的美好在心中沉淀，洒下一路芬芳。
珊瑚与海 浮潜初体验
据说，沙巴州有许多适合潜水的地方。
从我们居住的度假村海滩出发，十几分钟的船程，便可以抵达当地著名的潜水点啦。
在度假村中可以租借各种装备：脚蹼、蛙镜等，还有必不可少的救生衣……
由于当年为了当学霸，导致如今的高度近视，我只能自己带了一个游泳镜过来，直到要下水的时候，度假村的潜水教练才告诉我他们有带度数的蛙镜出租……可想而知，我当时心中得有多少只草拟吗奔腾而过啊！
话不多说，潜水开始，当快艇在海中央刚刚挺稳，所有人都已经开始迫不及待、跃跃欲试的想要跳下水了~
来看看这个度假村的潜水教练，多么淡定娴熟的潜水技巧……
这里的海水很浅，也就3、4米深左右，辅一下水，就可以看到各种各样的珊瑚丛，还有在珊瑚丛中游荡的鱼儿，还有海星，甚至是海胆。
这哥们的动作简直是太帅了，虽然我自认是水中小白龙，浪里小魔王，但也做不到这样气定神闲的潜入水中啊……难道是因为我没戴蛙镜和脚蹼的缘故？看来下次再玩浮潜的时候要把装备都备上。
水下拍摄基本都用gopro完成的，不过由于大家都顾着跟海底的鱼儿玩耍了，基本上也没拍到几张照片，特别是这里有好多的小丑鱼，居然一张都没拍到……T T
下水之前，教练员会仔细检查每一个人的救生衣是否穿戴正确，然后大家可以在水中一路前游，到不远处的无人小岛上。
我和面面也算是艺高人胆大了，直接在海中脱了救生衣，开始了自由潜。还是海底的风景迷人啊，更加近距离的欣赏这些千奇百怪的珊瑚丛，这还是我第一次如此近距离看到这么可爱的海星哦。
或许是对于自己技术太有信心，这次连脚蹼没穿，水母服也没穿，就这么直接跳下海了~途中最郁闷的是在游往无人岛的途中，居然有一片海域珊瑚距离海面只有不到1米，一不小心把膝盖划伤了……现在想想，面面那包成兔子一样的水母服虽然搞笑，不过人家既不怕划伤，又能防晒，还是值得一买的……
在大海中漂泊了一个小时左右，我和面面还有大V三人均是直接游到海岛上的。今天的天气似乎格外晴朗，天高海阔，配上阳光沙滩，还有比基尼美女，简直完美
这就是我们传说中因为活动而凑成一对的大V小V童鞋，羡慕嫉妒恨……
这里的海水真心清澈见底，在阳光的照耀下，犹如玻璃一般。
无人岛的风景，是一片片的礁石，在石头缝中或许你还能发现意想不到的惊喜，比如说——生蚝

蓝天白云碧水青山
在沙巴亚庇的最后一天，行程安排的是前往亚庇市区不远处的东姑阿都拉曼公园，这是由5个大小不一的海岛组成的国家公园，而这里同样拥有着世界级的潜水热点。
从港口乘坐快艇出发，在碧海蓝天中飞驰，可以看着天空中的朵朵白云，今天，似乎又将是阳光明媚的日子。
东姑阿都拉曼海岛公园共由5个海岛组成，分别是：加雅岛、沙比岛、马努干岛、马穆迪岛和苏禄岛。20分钟左右的飞驰，我们抵达了此行的第一个海岛，加雅岛。还未上岛，大家便被眼前碧绿的海水所吸引，而海中，特别是木栈道边上，成群结队的鱼群，简直叹为观止。

关于签证
电子签！电子签！电子签！重要的话要说三遍！
从去年开始，马来西亚对持中国护照游客实行电子签政策，大大增加了前往马来旅游的效率。只需上马来西亚移民局官网申请即可：https://www.windowmalaysia.my/evisa/，分为eVISA和eNTRI两种，所需材料如下：
1）个人2寸照片；
2）护照扫描件；
3）前往马来西亚的往返机票订单。
其中最最牛逼的是，申请eNTRI居然是秒出签，材料一递交，立刻就会发电子签邮件给你，不需要等待，完全可以来一场说走就走的旅行！
言归正传，关于电子签（eVISA）使用条件如下：
· 有效期3个月，最多逗留时间30天，不能延期；
· 入境前，必须用A4纸打印出电子签。
关于电子免签证登记（eNTRI）使用条件如下：
· 单次入境，最多逗留15天；
· 持有中国护照公民可申请，不包括港澳台；
· 入境前，必须用A4纸打印出电子签；

实用信息
== 交通 ==
这次大家统一都购买的马来西亚航空，这里贴一下新马航的官网：
https://www.malaysiaairlines.com/cn/zh_CN.html
此外在沙巴亚庇，城市里随处可以打车，价格很便宜，并且在马来也可以使用Uber
== 气候 ==
沙巴位于热带区域，气温常年高温，在低海拔地区平均温度在32℃左右，而高原区域的平均温度较低，约为21℃。
== 货币 ==
马来西亚通用货币为令吉（RM），和软妹币的兑换比例约1：1.5，Visa和MasterCard等信用卡在大部分酒店和较大型餐厅可以使用，大部分当地商贩多使用现金，个人感觉不需要提前在国内兑换好先进，可以抵达机场后ATM取现或者当地的兑换点兑换。
== 语言 ==
马来语虽然是当地的官方语言，但是大部分人都会说英语，而且沙巴华人数量很多，基本无论到哪都能遇上讲中文的人，所以无需担心语言问题。
== 网络 ==
以前一直使用漫游宝的，这次忘记申请了，只能在福州机场买了一张上网卡（当时也是第一次看到机场有自动售货机卖上网卡的，就抱着试一试的态度）。这种卡只能上网，不能通话，不过网络速度还不错，而且不限流量。
到沙巴如果时间长的话，建议买一张当地电话卡，价格实惠点。当然，自己带一个移动WiFi也是不错的选择。
