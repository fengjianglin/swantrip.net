玩-小东海

炙热的午后，来到小东海，静静地它和椰子树下的那些歪果仁一样，悠闲而惬意，这里更像是一个私密的小海滩，半山半岛洲际酒店的后花园，走出酒店就可以随时投入海浪的怀抱。
有了这些高大威猛的椰子树，不用畏惧当下酷暑难耐，反而觉得凉爽舒服了许多。
如果再来三亚，一定要在附近住上一晚，这里海水清澈又人少，适合度假休闲，这里也深受歪果仁的青睐。
这边的位置虽然不是最佳观看日出和日落的地方，但是这边有很多礁石，等到海水退潮，可以收获许许多多的贝类、海草等小海鲜。
在距离这片海域不远处的鹿回头公园，确是观看日落的好地方，但我们并没有进到景区内，就在景区门口公交站的观景平台上，看了一次免费的日落。

吃-椰子鸡

视频-1分钟了解椰子鸡

早就听闻海南的椰子鸡有名，恰逢这次去三亚旅行，一定要去品尝下地道的椰子鸡，趁着在大东海游玩，查了下附近椰子鸡店。
进入店里，首先从砍椰子开始，现砍现做，为了保证食客能够吃到新鲜的食材。汤底的食材就来自椰子的椰汁和椰肉，只有来海南才能吃到正宗的椰子鸡。
椰子全身都是宝，最熟悉的就是椰子水了，可以直接插上吸管当水喝，既解渴又防暑降温，而椰子肉可以用来做各种小视频，椰球、椰糖、椰子粉等，而椰皮的粗纤维可以作为药材，有降压降脂的功效。
不得不说，这家店从外观上就给人一种明朗、干净、温馨，简单的桌椅和布局，小清新的风格，没有印象中火锅店浓浓的锅底味道，而是一种飘香四溢的椰子香，不愧是“三个椰子一只鸡”嗲嗲在当地独具一心的文化守望和承袭，遵循原汁原味的“裸烹”，用东郊椰子代替了传统的锅底，简简单单还原食物最本真的味道。
让小妹帮忙调制一份酱料，沙姜、蒜蓉、指天椒、什锦酱、小金桔，这些具有海南特色的食材，被称为海南四宝组合，还可以根据自己喜欢的口味调制，麻将、花生碎、香葱、酱油、香醋等十几种，总能调制一出独一无二，喜欢吃的口味。
每一种都想尝一下，不知不觉点了一车的菜，野生竹荪、有机木瓜、手打马鲛鱼鱼滑、还有清凉Q弹的手工椰子冻。
每一个锅底都放入了2个椰子水，待客人上桌，再倒入第3个椰子水。里面还有椰丝和红枣。
每个桌子上都有一个计时器，如果是煮文昌鸡的话，只需要3分钟时间，不用担心会因为忘记时间，而把鸡肉煮老，因为那个计时器会到点发出“嘟嘟嘟”的声音。
菠萝饭，据说每家椰子鸡的店做的还不大相同，我吃的这家店做的比较清淡，是真的用新鲜的菠萝肉搭配糯米烹制的，喜欢重口的亲们慎点，如果像这样一大桌子菜，两个人吃没有那么大食量的话，可以点小吃类的椰子糕、椰子饭，份量小份，还可以多品尝几种。
手打马鲛鱼鱼滑，选用深海马鲛鱼，手工打制，摆盘就非常精美，像勺子一样，可以直接挂在锅边，1分钟左右当粉色变为白色就煮熟了，Q弹的马鲛鱼鱼肉，很鲜也很有嚼劲，新鲜实在，上面鱼子富裕味蕾微妙细腻的层次感。
野生竹荪，下锅就熟。营养丰富，香味浓郁，滋味鲜美，自古就列为“草八珍”之一。竹荪搭配原味椰子鸡，更加营养鲜美。
绣球菌，一个小罐子装的，像银耳一样，乳白色的一种菌类，吃起来口感虽没有银耳滑溜溜的感觉，但却没有蘑菇那种味道，被称为世界上唯一的阳光蘑菇，当人们开始越来越注重食材和养生滋补，而绣球菌每天十小时的光照，就是时间沉淀的秘境美味，特殊的生长环境与漫长的生长时间赋予了绣球菌。
海南有机木瓜，开始以为是土豆，知道是木瓜后，差点生吃活嚼，才知道是生木瓜，煮了才更有味道哦！
酸角汁，喝出了小时候的味道，小吃后经常吃酸甜的酸角话梅，那时候虽然包装很简单，但是味道却很纯，多少小时候的味道随着时间的流逝而一去不复返了，真的很怀念。
海南手工椰子冻，入口就有细腻嫩滑的感觉，甜而不腻，醇香丝丝缠绕舌尖，上面乳白色的奶酪，有着较为浓郁的奶香味，下面就是透明状固态的椰子水，不仅保留着椰汁上的一股清爽之感，而且含在嘴里还有由内向软颚外扩张的Q弹感。

住-珠江俪豪

三亚的民宿非常多，大部分还都属于民宿的初级阶段，类似短租房，家庭旅舍之类的，建议出行前在网上多甄别，参考评论和评价，以防采坑。
我入住的是珠江俪豪小区，马路对面就是万人海鲜广场，吃还是很方便的，从这里打车到第一市场10分钟，这个小区在三亚属于中高档，环境卫生都不错，小区环境优美，住的比较放心，因为选的高楼层，可以远眺远处的西岛，晚上也很安全，小区有室外游泳池，入住的客人都可以免费进去游玩。

玩-椰梦长廊

长达20公里的椰梦长廊，像是一条绿色丝带，顺着三亚湾滨海大道一直延伸下去，白天这里遮阴纳凉，傍晚这里日落红霞，如果赶上一个好天气，这里的日落美景丝毫不逊于国外的海岛。在椰梦长廊的另一侧则是休闲度假区，布局巧妙，风格各异的幢幢别墅、座座宾馆等园林式现代建筑处处林立。椰梦长廊营造了优美的生态环境和最佳的休闲度假环境，是三亚市旅游精品的典范。
除了长长的绿色走廊，与之相连的是银色的沙滩和湛蓝的大海，去的时候，赶上了好天气，在太阳光的照射下， 细软的白沙，清澈的海水，每一来到的人都按耐不住兴奋的心情，和大海来一次亲密的接触。
长长的海岸线，散步玩耍，停了许多的渔船，木质的渔船特别适合作为拍照背景。喜欢拍照的MM不要错过。
门票：免费
交通：椰梦长廊距离三亚市中心约3公里，交通非常方便，可以乘坐新国线双层观光巴士或者乘坐206路公交车，在大东海公交站牌乘坐8路车都可以抵达椰梦长廊。
Tips：椰梦长廊最好是傍晚来，这里是观看日落的绝佳地点。

吃-海鲜

澳龙、老虎斑、和乐蟹、皮皮虾、基围虾、白昌鱼、芒果螺、圣子王、花螺、海胆、生蚝、鲍鱼、带子，13种海鲜，满载而归，挑选完海鲜，到门口去称重，餐厅工作人员会跟你确定每一种海鲜的做法。
蒸、炒、炸、煮等不同的做法，可以根据自己的喜好选择不同的口味，川味是店里的招牌口味也是我的最爱。
除了海鲜，店里还有鲜榨的果汁、炒冰，有西瓜、橙子、芒果，推荐芒果汁，满满的一大扎，搭配美味的海鲜，最合适不过了。
蒜蓉鲍鱼：如果觉得蒜蓉生蚝吃不过瘾，那一定要点几个鲍鱼解解馋。
香辣和乐蟹：分大小，喜欢吃蟹黄多一些就选择小的母蟹，喜欢蟹肉多一些就选择大一些的，我们点的是大的，香辣的口味完全没有破坏了海鲜的鲜味，咬上一口都是厚实饱满的蟹肉。
芒果螺/花螺：都是微辣的做法，吃螺最好的配搭是冰啤，尤其在夏日的夜晚，真的是美味。
清蒸老虎斑：之前吃过石斑鱼，这次想尝个老虎斑试试，一看鱼眼珠就能辨别鱼的新鲜与否，我们想要清单一些，所以选择的是清蒸的做法，还可以红烧并加一些微辣。
椰饭：每次来三亚必点，百搭的椰子，不但椰子汁可以用来做果汁、椰子冻、椰子鸡…还可以做这样的椰子饭，吃完上面的糯米，下面的椰子肉也是回味无穷，带着淡淡椰奶味道，非常好吃。
清蒸基围虾：基围虾的做法很多，白灼、椒盐、姜葱、辣炒，可以选择其中一个口味，也可以一虾两吃，建议尝试辣炒的做法，毕竟香辣是小胡子家的特色嘛，白灼有专门的蘸虾的调料，但是个人觉得完全可以不蘸，因为本身虾就很新鲜了，原汁原味的味道最好了。
椒盐皮皮虾：因为家在沿海城市，所以每年春末夏初都是吃皮皮虾的季节，当然来海南吃海鲜是不受季节的限制，吃惯了清蒸，这次试一试椒盐的做法，怕我们不会剥，工作人员特意来教我们如何剥皮皮虾，当然对于我这样的“吃货”来说，剥虾是游刃有余的事情，但是不得不说，椒盐的皮皮虾比清蒸更容易剥皮，而且吃起来更加入味。
澳龙：清蒸澳龙，最喜欢吃的就是龙虾腿，又嫩又鲜，如果想喝海鲜粥的话，可以告诉店员，将龙虾头剥下来用来熬粥，之前在广西东兴吃过沙虫粥，就一直对海鲜粥念念不忘，非常好喝。
四角豆：四角豆对于很多北方人来说，可能会很陌生，但对于我却又熟悉又陌生，因为很多年前我就很喜欢这个蔬菜，加上蒜蓉简简单单的清炒，却能品尝出菜的清香来，在三亚这道菜是最寻常不过了，寻常到像北方的大白菜。几乎是每周必点的。
人均:132元
营业时间：10:00—23:00

玩-皇后湾

这里是我这次来三亚，最喜欢的地方了，它不需要门票，也不算严格意义上的景点，就是一处平静的海湾，就连拍打的浪花都那样的温柔。
来的时候正值傍晚，还怕赶不上日落，匆匆的来，虽然没有下雨，但是来海南的两三天每天都有厚厚的云，若隐若现的晚霞，趁着这最后一点微弱的光，再好好感受一下这份安宁。
去年10月因为出差，和当地的同事一起拜访过海边的民宿和客栈，清晰的记得就是背后那家店，十月的午后要比三月来的更炙热，但有了屋檐的遮挡，在微风的吹拂下，还是很舒服的，也就是那一刻，我才得以片刻的休息，这时耳边传来悦耳的风铃声，伴着Five hundred miles优美的旋律，眼前是一幅现实版的海岛人的幸福生活。此刻，岁月静好，真希望我不是来工作的，而是在度假。
天时、地利、人和，如果都能同时在线，那是多幸福的事情，可惜了我这个见水就按耐不住ziji的人，这次不能下水，只能在沙滩边静静的看着你，你们...
皇后湾如何去？
蜈支洲岛停车场附近，有个后海村，刚进村口有个藤海小学，过了小学右转，再直行就能通往海边，海边有很多民宿，还有带泳池的中高档的，喜静或者喜欢冲浪、浮潜的小伙伴，可以选择在这里多玩一些时间，或者住上一晚。
后海村除了有皇后湾，它还有一个重要的地标—蜈支洲岛的码头。

玩-水稻国家公园

vlog-玩转水稻国家公园

本来是为了寻找花海的，稀里糊涂的就来到了水稻国家公园（这就是不提前做攻略的下场）门票90元（加电瓶车），真的有点小贵啊，而且来的不是季节，想看的花也没有看到。建议想来的亲们，下午四点左右来玩，玩2个小时，晚上可以观看表演。各种套票等实用信息，请移步文末的实用信息部分。
进入景区，先上电瓶车前往景点。
祈福园，传说中的十大全域旅游婚纱摄影基地，如果来对了季节，应该就能看到千亩花海，除了三角梅额美人蕉，荷花、格桑花、菊花、紫荆花、沙漠玫瑰，紫薇花，这些无缘看到的姹紫嫣红景象，就留给大家来揭秘吧！
五色长廊里种植着瓜果蔬菜。
来到南繁水稻科研基地，也就来到了全长1.5公里的百龙步道，这是全球首次以1：1复原的大型恐龙模型，让博物馆走进大地，让科普走进田间，上百种形状各异的仿真动感恐龙，不但能灵活的转动身躯，还能模拟发声，和这些远古时代的精灵们来了一次近距离的接触。
水稻国家公园水稻科研基地占地面积150亩，主要对征集全国科研研究所、种业公司的水稻新品种和新科技在该基地做展示、对比和选育，项目由国家杂交水稻工程技术研究中心、国家粳稻工程技术研究中心全程指导。
起初是奔着看花海来的，无意中来到水稻国家公园，见识到南繁水稻的种植，不同品种的花卉，鲜有的瓜果蔬菜，走在万亩水稻，穿行于各型各类的大小型恐龙之间，别有一番乐趣，小巧的如精灵般，大如擎天柱般，对这个远古时代的精灵又有了更深的认识。
