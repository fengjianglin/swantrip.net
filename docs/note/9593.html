初见*便是碧海蓝天

( 海南 ) 真的是一个被上帝宠坏的地方，
上帝把宜人的气候、清新的空气、和煦的阳光、湛蓝的海水、柔和的沙滩、风情万种的少数民族、美味的海鲜……都赐予了这座( 海南 ) 岛南端的海滨旅游城市~
如果到了这里百分之九十的人都会选择去这里—— 蜈支洲岛 。
蜈支洲岛 紧紧靠着公路，交通十分便利，我们坐着车很快就从酒店来到了这里。以前一直看照片看岛，来了之后才感觉：不愧是五A级景区。
湛蓝的天空与大海浑然一体，望不到边际。礁石保持最原始的形态散落在靠近海岸处，一些石头上长满了紫菜，听闻一些海民最喜欢采下新鲜的紫菜晒干以后烤着吃，这种吃法也出现在《舌尖上的 中国 》里，被烤的酥脆的紫菜以最原始的口味进入到口中，那滋味应该不同于超市经过加工运输过的味道，不知这次旅途可以不可以品尝到。
岸边是最繁华的地方，商家给游客提供了丰富的游玩资源。水上摩托惊险刺激，是不少年轻人来这里必玩的项目之一，即使排了再长的队伍也阻挡不了人们的热情。还有潜水，沙滩排球，各种度假别墅供人选择，古韵的沙滩休息配置也很抓人眼球。这里也有专门给小朋友设计的游玩设施，爸爸妈妈们可以放心的让孩子在这里玩耍。
你也可以选择去一些石头比较多的海岸捡贝壳、海螺，当个纪念品带回去也是很不错的。
忽略我的脚，专心看美景
情人桥，原是座铁索桥，是当年守岛部队的海上瞭望点。有些小姐既想过桥到瞭望点里体会一下，又怕掉进海 水里 ，过桥时紧紧抓住朋友的手不放，因此这桥又被戏称为“情人桥”。后来该岛为客人安全着想，故将原来的铁索桥改造成如今的木板桥，但依旧是情侣们最向往的地方。
其实，很多人说 蜈支洲岛 又名情人岛，是龙王的女儿与其爱人化成的石头岛，龙女为了爱人而对抗父亲逃出龙宫却与爱人一起被化作石头的故事凄美让人唏嘘，而这岛上也有不少应景的设施吸引着一对对恩爱的情侣。
我向来就对这种凄美的悲剧有着很浓厚的兴趣，在这里放上这段故事供大家欣赏一下吧：
很久很久以前，一个年轻人在海上打鱼，突遇风浪把船打翻，将他漂到一座荒岛上。在岛上，他以打猎为生。一天，他到海边打鱼，忽然发现一位美丽的姑娘正在沙滩上拾贝。姑娘主动上前与他聊了起来。原来她是龙王的女儿，因为贪玩跑到岸边。
两人聊的特别投缘，从那以后俩人约定每天在这里见面。日子一天天过去，俩人谁也离不开谁了，后来俩人就在一起生活了。白天一起打猎，夜晚一起在沙滩漫步，过着只羡鸳鸯不羡仙的生活。
转眼一年过去了，小龙女想家了，可是她已经离不开小伙子了，但又怕父王知道此事后会怪罪。最后俩人商定，由小龙女回去把此事告诉龙王，请求他成全他们俩，三天后在他们最初见面的地方再重逢。自从姑娘走后，小伙子每天都站在那里盼她回来。三天很快过去了，小龙女还是没有回来。小伙子心急如焚，每天站在海边大声呼喊姑娘的名字。
原来姑娘回去以后把她和小伙子的事情告诉了龙王，龙王大怒，下令把她关了起来，不准她再到人间。姑娘日夜思念小伙子，终于有一天趁看守不备跑了出来。龙王很快知道了，在后面紧追不放。眼看着这对痴情男女就要相拥了，在后面紧紧追赶的龙王大怒，大喊一声，用了一个定身术，将两人变成了两座大石头。千百年过去了，经历了潮起潮落洗礼的两座大石头，依然矗立在那里，静静相望，近在咫尺却远在天涯。后来，人们为了纪念这对痴情男女，把这里叫做“情人岛”。如今，这里也成了青年男女耳语、听涛的好地方。
最后，当然是撒狗粮+一波美照

神秘幽静的热带雨林

这里是典型的热带雨林植被，一落地的那一刻，眼睛里已经充斥了绿色，不同于北方植物给人的那种淳朴，耐旱的感觉，呀诺达的植物是热情洋溢，“水灵灵”的，许多在北方只有及人腰的植物到了这里变的巨大无比，甚至“遮天蔽日”，感觉随便摘一片叶子下来都可以当遮阳伞。得天独厚的地理环境甚至赐予了这里温泉的礼物，也给了这里丰富的水果资源，仿佛受尽了上天的宠爱。
纠结繁杂的树枝树根，“高高在上”的椰树，是这里的特产。雨林谷盘根错结、鬼斧神工、石径通幽、九曲回肠、森氧绿肺、禅意风生，其中林蟒绞杀、空中花篮、老茎生花、藤本攀附、高板根、根包石等六大雨林奇观形神兼备，展现了独特的根石雨林景观。
人们相信，如此完美的地界一定会有神灵的护佑，于是衍生了许多祈祷，祈福类型的景点， 比如 说幸福天道。
人们将自己美好的愿望写在木牌上，与特制的小草帽一起，用最吉祥的红绳系在过道绳索上，希望可以求得自己愿望实现，在美景的衬托下，好像真的可以感受到庇护小岛的神力。
美景不会夺走美食的魅力，这样一处仙境怎么能没有美食呢？
这里的药膳自助餐与养生宴都是很出名的，而丰富新鲜的食材、少数名族自家酿的酒以及缺少不了的本土特色小吃听着就让人垂涎欲滴，可惜由于时间不够，没来得及品尝，真的很遗憾了。大家如果有机会来的话千万不要错过啊！
优秀的水土造就了一个人间仙境，这里的水都是那样的吸引人。在层层的树荫下，倒一杯清水，坐在桌旁，闭上眼睛，才能感受到那种最纯真的水土魅力吧！

走进第一市场

终于到了我们这次旅行的主题：吃！
第一集贸市场是 三亚 最大的海鲜交易市场，不过如今所提及的“第一市场”，已经不仅是指这个集贸市场了，而是更宽泛地代指集贸市场及其周边的商业街，包括 新民 街、椰风巷、河西路一带街区，是海鲜加工店的云集之地。
第一市场是许多游客吃海鲜的首选之地，这里的海鲜新鲜又便宜，在集贸市场里买了海鲜之后就近找加工店加工，比去大饭店吃要便宜很多，而且市场内外也可以买到品种多样的水果以及 海南 特色小吃。
水果：你看我诱人不？
珍珠项链：我就是你富贵的象征！
这个汇集了无数商贩的交易市场是购买海鲜的绝佳场所，价格低廉，但十分新鲜，那种海物刚被打捞上来的腥气弥漫在空气中却不引人厌烦，甚至可以直接说这就是海的味道。有的蚌类还在向路过的游客喷着水，螃蟹张牙舞爪的向人们挥舞着它有力的钳子，殊不知自己就快被做成桌上佳肴。扇贝吐着舌头，各色海鱼游来游去彰显着自家的生命力，但在每个人眼里它们全部是鲜美的肉。

味蕾上的海南味儿~

很多游客会提前在网上先认准口碑好的加工店，到了这里先去找加工店，就在第一集贸市场对面的那一条街。然后可以由店里的阿姐带着去选购海鲜，阿姐比较有经验会还价。
这家店好评是比较多的，我们到来时店里人也不少，也算是证明了这里评分高吧，所以你动心了吗？
接下来，就是美食时间了！在采购完海鲜以后，就等着做好的海鲜出炉了！
第一道：蒜香鲍鱼
怎么样？是不是光看到图片就已经流口水了？
鲜嫩多汁的鲍鱼配上粗犷的大蒜
再加上点缀的翠绿的葱花
以及还没来得及消散的大海的味道
浓郁咸香的口味并不会觉得油腻，
除了好吃我已经来不及想出其他形容词了
第二道：蛏子
厨房阿姐（也有可能是大哥）将蛏子壳“废物利用”摆出了绝美的摆盘（彩虹屁）
小辣椒的辣与蛏子的香组合，将菜的颜值与口味提升了一个档次
色香味俱全的蛏子，开胃的爽口辣椒很是符合这里热情的风土人情。
第三道：海虾
你没有看错
没有多余的配料
没有累赘的点缀
只有最原始的海虾！
最古老的烹调方法在今天任然适用，跨越了几千年在今天赋予这道菜新的生命
鲜红的虾身夺人眼球，处于海虾最淳朴的味道弥漫在四周
灯光照在虾上反射着的都是带着美味的光
本姑娘甚至不舍得剥去虾壳，巴不得享受它最完整的味道
第四道：芒果螺
其实一开始我很担心这道菜的味道，毕竟是海鲜与水果的混搭，一个不小心就会成为黑暗料理，但当芒果螺上来后，这种忧虑完全打消。
先来说“色”，鲜红的辣椒，黄色的芒果以及海螺的混搭，加上一丝丝的青椒，居然是一次 成功 的撞色挑战。
再来说“香”
看不见的香气有着真实的嗅感，本应是浓郁的水果并没有抢了海螺的主场，而是与海螺被烹饪过的香气相得益彰。
最后，最重要的“味”，咸与甜完美的结合在了一起，在咀嚼过程中交替着刺激着味蕾，海味与水果的味道结合，非常可口美味。
真的推荐大家尝试一下！
第五道：皮皮虾（富贵虾）
没想到表情包也会这么好吃，但吃的时候要记得挑去虾线。
因为皮皮虾肢节较多，吃起来比较麻烦，可以用剪刀将虾身两边的皮减掉，比较好剥。
皮皮虾肉质鲜嫩，吃起来根本不觉的占肚子，容易吃多（ 比如 我...）
在这里特别给大家提个醒，因为虾与一些水果不适合一起吃， 比如 葡萄、石榴等，在吃晚饭到市场上吃水果是要小心啊！
第六道：
粉丝与扇贝的组合可能在全 中国 都是个潮流，这里也不例外。
做熟的扇贝呈现出激发食欲的橙色，在吃完粉丝后，再吃下面的扇贝肉，颇有些柳暗花明又一村的意思，而且把扇贝肉嗑下来的一刹那颇具成就感
个人感觉，加些醋可能会更好吃~
第七道：石斑鱼
不要被它的大嘴吓到，这道菜外貌与味道成反比。
谁能想到一条黑乎乎的鱼肉质这样鲜美可口。关键是它高蛋白但低脂肪！怕胖的姑娘们大可放心。而且它的胶原蛋白含量很高，所有也叫作“美容鱼”，营养价值特别高，被誉为“ 中国 四 大名 鱼之一”也就不足为奇。
对于我来说，一条鱼根本不够吃，因为真的很好吃啊！
第八道：狗粮
其实是给大家看一下这次盛宴的全景啦！各种菜配上 海南 特制的辣酱，真的绝了！
海鲜配啤酒也是不可忘怀的搭配啊。前面我们已经说过，这里市集上的海鲜很便宜，在市场对面有很多加工店，所有费用加起来比外面的酒店便宜不少，而且量可以自我把控，是海鲜爱好者的绝佳选择。

关于乡愁的崖乡小吃

在海鲜加工店饱餐一顿后，就是吃小吃的时候了（没错，我还能吃下！）
于是我们来到了崖乡小吃！
这家店颜值是不是很高！好了，话不多说，开始我们的小吃点评！
第一道：海鲜汤粉
海虾，扇贝，粉丝，青菜等组成的组合熬出的汤，鲜美可口。刚刚的海鲜主餐可能是“妖艳诱惑”，那么这道就是“小家碧玉”，被熬成白色的汤散发着清香，很适合大餐过后对胃进行滋补，暖暖的汤入腹很是舒服。
第二道：榴莲雪媚娘
又白又软的团子很可爱，像个软乎乎的奶团子，激起了本仙女的少女心。
榴莲被包裹在甜香的糯米冰皮里，缓冲了榴莲有些冲鼻子的味道，咬一口下去，软糯的榴莲果肉与外表椰丝进入口中，甜到了心里。
吃一口，仿佛周身都在冒着粉红泡泡~
真心劝在座的受不了榴莲的小伙伴，这个小零食绝对可以改变你对榴莲的认识！
第三道：牛油果汁
牛油果，是一种很健康，糖分含量很低的水果。很多减肥的人都在吃它，在这里，变成了果汁，但依旧很具有营养价值
淡绿色的果汁颜色也在体现着它的营养价值。
第四道：鸭掌
不知各位小时候有没有看过哥哥演的《家有喜事1992》，我一直就对那里面的“百花炖鸭掌”很好奇，如今终于吃到了。
鸭掌没有多少肉，皮厚，但入口口感很好，富有弹性的皮有恰到好处的嚼劲，而且蛋白质含量高，低脂肪，又是一味减肥的好菜！
虽然是荤菜，但很爽口，泡椒与小辣椒配合鸭掌变成这样一道小吃食，的确精妙。
第五道：椰奶清凉补
哈哈，又是一道属于女孩子的甜食。
Q弹的紫薯丸子，补齐的红枣，甜甜的葡萄干以及糙米、椰丝加上椰奶，网红零食也是有内涵的！
凉凉的口感中和了海鲜大餐的急躁，给这里炎热的气候带来一丝凉意，不过晚上不要喝太多哦！
舀一勺放入口中，刚退下去的粉红泡泡再一次涌了上来，不过，这份椰奶清凉补糖分含量可是不少，想减肥的姑娘们不要喝太多，剩下的，就给男朋友喝了吧，让他也陪你一起冒粉红泡泡~
