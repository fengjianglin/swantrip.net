2019重庆低调赏花地出炉，跟我们来场寻香之旅吧！

游记 五一 小众 摄影 赏花 踏青 2019/04/28

寻香之旅

明暗相转，四季更替。
迎着春的气息，我和Pom Pom开启了长达2个月的寻香之旅，在一片片绿浪中寻找绵醉到心里的芬芳。 

照母山丨红梅花开，早春寻香

元宵节吃过汤圆，这年算是过完了。赏花，便成了接下来的头等大事。早春时节，略带料峭寒意的微风吹醒了照母山的红梅，一个个长着圆圆脑袋的花骨朵，像夜晚的火焰一样悄悄绽放着。
我和PomPom作为第一波赏花大军中的主力军，挑了个周末去照母山打探花情。红梅的风姿确实很美，成片的梅林，一众盛开，如画似海。
听说，红梅也有很多品种，按色调来分就有宫粉、骨红和朱砂红，颜色最淡的就是宫粉。照母山的梅，我估摸着应该是宫粉这个品种吧。
花骨朵是深红色，绽放后呈淡粉，特别有感染力，就连如雨如雪的雾似乎也泛着粉的调子。
梅开得热闹，人气自然也是旺的。
最会抢占拍照机位的，当属挥动丝巾的大妈们，一群人“霸占”着最佳拍照点。这一次，大妈们又把丝巾玩出了新花样儿，把好几条丝巾捆在一起，每人拉一根，组成一朵花儿。我倒是觉得，大妈争奇斗艳的样子比红梅更好看。
Tips：
地址： 重庆 市渝北区金开大道西段88号 时间：周一至周日8:00-AM18:00PM 交通：乘坐轨道交通5号线在幸福广场站下车，顺着陶然居古镇上山后，往右侧沿着公路走，穿过节孝牌坊后，就能看到一个飞机模型，在模型后100米就是红梅苑
注意：①私家车只能停在大门附近的停车场，不能开上山。停车都是免费的，而且车位充足
②想要避开人流，那就早起吧，7点至8点这个时间段，几乎没有人，可以随便拍

五一村丨偶遇油菜花，田园野趣

我和Pom Pom打算去五一村，看大片的粉红樱花海。一路自驾过去，接近目的地，交通就瘫痪了，众多私家车一字排开，堵了接近一公里。我们果断弃车，拿着相机，扛着三脚架，步行前往。
好不容易走到门口，却被告知要收门票费，30元一个人。我们在大门口略略瞅了一下，然后秒变“小气少年”，一致决定不进园。既然来了，我提议在园区附近走走。
果然，最好看的风景，都是在不经意间出现：乡间田野点缀着一片金黄的油菜花，像是给大地母亲披上了一件黄金圣甲，令人心旷神怡。
4朵花瓣组成一个娇小的花朵，许多花朵叠在一起又堆成一团花球，这就是油菜花，既不雍容华贵，又不光彩夺目，可是却美得很朴实。柔柔的风拂过花海，一株株油菜花随着风的方向左摇右摆，真是可爱极了！
Tips：
地址： 重庆 市北碚区 澄江 镇五一村樱花橘乡 交通：乘坐轨道交通6号线至北碚站，再换乘511路在烟湖滩站下车
注意：①地方比较偏僻，建议组队自驾前往，导航直接搜“樱花橘乡”
②樱花园景区门票30元/人
③景区门口的铁板鱿鱼沙茶面超好吃，鱿鱼14元/串，沙茶面18元/份
④许多农民在景区门口摆摊卖菜，菜很新鲜，价格也不贵，春芽10元/两，自己做饭的宝宝们，也可以带上一些回家

龙头寺公园丨艳阳高照，踏青赏茶花

我和Pom Pom去龙头寺公园时，是个周末，天气良好，阳光明亮，因此赏花大军的队伍数量亦是有增无减：有的人带着孩子，在 春日 的和风里放着风筝；有的人精心准备了各种卤菜、小吃，在草坪上惬意地野餐；还有的人不失童心，拿着小网在池塘捕鱼。
对呀，这不就是春天吗？
我们再说说龙头寺公园里的花！这里不似照母山有成片的梅林，气势恢宏，但胜在花的品种较多，山茶、梨花、李花，还有红梅竞相盛放，散布在公园各个角落，为 重庆 的早春增添了不少缤纷。
梨花、李花洁白如雪，冰清玉洁；茶花红得不惨一点杂质，虽然鲜艳却又不失纯净，不管是哪个品种都有各自的美。
但若论哪种花适合拍人像，我更偏爱茶花。因为梨花、李花的枝干较高，适合用仰拍的角度，只有像小女孩一样骑在爸爸的肩上，才能与灿烂的花儿亲密接触。
Tips：
地址： 重庆 市渝北区龙塔街道五黄路(近兴盛路)
交通：在五里店乘坐轨道交通6号线（北碚方向）至红土地站下车，步行至龙头寺公园（乘坐乘轨道交通10号线可以到龙头寺公园站，但此站离公园的北门、南门都比较远）
时间：全天开放
门票：免费

碧津公园丨双色桃树，红白碰撞

碧津公园是 重庆 的“老资历”公园之一，承包了许多渝北人整个童年的回忆。从1986年到现在，公园的颜值不断刷新，如今的大门已被修建成仿古的建筑，颇具气势。
如果你期待的是成片花海，碧津公园或许会略失所望。但单从春色浪漫来说，这里又是超级适合打卡的地方。
我和Pom Pom在一个风和日丽的清晨赶到这里，沿着“桃花山”的指示牌往里走，约摸5分钟的路程，就与如云霞般美艳的桃花邂逅。
桃花是最喜欢争春的花朵，即使叶子还没睡醒，花骨朵也迫不及待从树枝里挤出来。古人常用桃花来形容妙龄女子，因为它的风韵十分独特，小巧的花瓣，白中泛着微红，像极了害羞少女的脸颊。
碧津公园里还有两株“双色桃树”，一根枝条上长出两种颜色截然不同的桃花，火红艳丽与洁白纯美碰撞在一起，形成了一幅有趣的画面。
若是赏花新鲜劲儿过了，去湖边、拱桥上闲逛吧！高大巍峨的碧津塔倒映在湖水中，四处环绕着绿树红墙，此景此景把你一秒拉回小学语文课本的情景中。
Tips：
地址： 重庆 市渝北区两路镇胜利路172号
交通：①乘坐轨道交通3号线至碧津站，从1号口出步行5分钟到公园。（后门）
②乘坐609路公交车至碧津公园站，下车后步行2分钟到公园。（正门）
时间：8:00AM-18:00PM
门票：免费

金果园丨四月蔷薇，浅笑而生

寻花之旅的最后一站，我和Pom Pom定在了北碚的金果园。这里是一个低调的赏花地，距离市中心50公里。
每到莺声燕语在哝哝作响的时节，金果园的蔷薇花就拼命绽放，召唤还未脱离休眠状态的人，来场开春的短途旅行。
重庆 的春很短暂，淅淅沥沥的春雨过后，便是连日的高温。即使是在衔泥筑巢的四月，气温也能轻易达到30℃。这次出发前，我们褪去厚厚的大衣，换上了一身轻薄的春装，以最舒适的状态去蔷薇的怒放。
初见蔷薇之时，还以为是玫瑰，枝干上同样都有恼人的刺，同样都是芳香清幽，带一点淡淡的甜味。仔细观察，才发现两者又有些许不同，蔷薇的花瓣更小，叶子也比玫瑰更薄。两者的颜值难分高下，但我更偏爱蔷薇，隐隐地能感觉到它有股傲然的气质。
在金果园赏完蔷薇，别急着走，采摘新鲜的水果也是不容错过的玩法。这个季节，果园里已经成熟且可采摘的是樱桃，漫山遍野的樱桃树，看上那哪棵就赶紧行动吧！
Tips：
地址： 重庆 市北碚区 澄江 镇 缙云 村20号（三花石沿212国道往 澄江 镇方向）
交通：乘坐轨道交通6号线至北碚站，4-2口出，再步行至五路口站，换乘510/511/556/518路公交车至电镀厂站下车（出了北碚地铁站后，也可以打车前往金果园，费用15元左右）
时间：9:00AM-17:00PM
门票：30元
