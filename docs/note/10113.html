星空日出，雾凇云海，南方西岳大容山浪漫之旅

游记 摄影 自驾游 露营 2018/12/22
星空日出，雾凇云海，这些壮观美丽的自然景观，总能给人一种心灵的震撼，如果能带上她一起，远离都市喧嚣，寻找属于我们两个人的空间，一起数星星，看银河，看日出。。。这是多么浪漫的事，于是就计划去了一趟 广西 玉林 的大容山国家森林公园。
大容山国家森林公园位于大容山腹地内，距 玉林 市40公里，距 南宁 220公里，公园面积4825.0平方公顷。大容山山脉呈东西—南北走向，长46公里、宽25-30公里，方圆1150-1400平方公里。被五代十国的南汉高祖刘龑封为“南方西岳”。
大容山海拔最高为 桂东 南第一峰1275.6米。森林覆盖率为92.5%，因为这里独特的地形和气候条件，所以看云海，星空，和日出的概率很大，同时这里还是著名的露营天堂和汽车营地，冬无严寒，夏无酷暑，吸引了大批游客过来度假。
根据《 广西 通志》记载“山高五百余丈，周围千余里，以其辽阔无所不包无所不容而得名。”大容山由此得名，可见其壮阔无比，登此山可一览众山小。

第一天 龙门景区

公园有五大景区： 龙门 景区（容山风情小镇）、六二丫游客接待服务中心、天湖景区和莲花景区，望君山景区（未开发），共有20多个景点。第一天我们中午到达大容山，下午就先去游玩 龙门 景区。
树包石，一块大大的石头被树根密密包围，见证自然界强大的生命力。
龙门 景区位于景区门口附近，包括了：容山风情小镇、容山度假村、养生餐厅、原生态溪谷长廊——九瀑谷、大容山桫椤谷漂流。首先我们来到了九瀑谷，九瀑谷位于大容山山脚，海拔300多米，全长3公里，是一条原始生态的溪谷长廊。进入九瀑谷入口映入眼帘的成片的野竹林，它们就像优秀的士兵成群结队伫立路旁，展开手臂，欢迎客人的到来。
桫椤树，与恐龙同时代的植物，是已经发现唯一的木本蕨类植物，极其珍贵，堪称国宝，被众多国家列为一级保护的濒危植物,有“活化石”之称。
观音园，这里有一尊高10.8米的观世音 菩萨 。每年农历二月初二都在这里举行一次庙会，四面八方的村民、商人、老板等都来这里赶会，在这里烧香、许愿很灵验。
九瀑谷并不是它有九条瀑布，九是多的意思，说明它是由多条瀑布组成的一条溪谷。
溪谷以谷幽、水清、石美、藤悬、林秀为特色，瀑、潭、滩贯穿整条溪谷，藤蔓蔽日，花香荡漾，泉水叮咚，溪流潺潺。被旅游专家誉为“岭南第一谷”。
一路向前，各种形状奇特的石头夺人眼睛，看这个像不像一个蟠桃，沿河谷山泉淙淙，细瀑飞流，潭水清沏，泉边芳草萋萋，野花飘香，茑啭鸟啼。
鹤瀑布，瀑布里的这块大石头，就像鹤的身体，它的头已经深深地钻进水潭里去寻找食物，这是九个瀑布当中面积最大的一个，远远望去犹如一条练悬挂在山崖上。
鸳鸯瀑是一个小巧玲珑的瀑布叫双头瀑，它流水飞快，一左一右，一大一小，仿佛像一对热恋的情侣故名鸳鸯瀑。
这瀑布就在桥的旁边，每每经过桥时，瀑布的水珠都扑面而来让人倍感清凉，疲劳也就一扫而去。
银线吊葫芦，看这个瀑布底下，酷似一个葫芦的形状，上面两条瀑布首尾相连，形成一条银线，如从天降，吊着一个巨大的葫芦，非常形象，且葫芦与福禄谐音。
莲花瀑布----银河瀑布座落于诸峰环绕的高山盘地的大莲池，水从两座站 之间飞泻谷底，形成落差有 500M的九级瀑布。瀑布凌空飞泻，翻滚着白色的雾花，悬挂着如银似玉的珠帘，象千万银珠从九天撒落，发出雷惊般的响声，山鸣谷应。
江家大院
江家大院，它的旧址坐落在最美乡村—— 江西 婺源 ，“江家大院”始建于清 咸丰 年间，距今170多年，是由当地的一江姓县太爷所建，费时7年才竣工。在2002年的时候，古玩收藏家王宽把房子买了下来，花了两个多月的时间，把木屋化整为零，运回 广东 放在仓库保存起来，江家大院得以保存。而他的厉害之处在于能够将古民居进行整体挪移复原。
江家大院气势磅礴，它是包括正厅、中厅、厢房、院落、天井等五进两层结构的古建筑。
江家大院是典型的清代徽式建筑，气势磅礴，砖雕、木雕、石雕巧夺天工，精美细腻，各种动植物雕刻的栩栩如生，这种追求天人合一的设计使大院显得更加静谧雅致。
参观完江家大院，我们在养生餐厅用餐，这里的每一道菜都是精选大容山养生食材，由五星大厨悉心料理，还原食物本身美味，天然美食，绿色养生。
到大容山能吃到的野菜有三十几个品种，有酸、甜、苦、各种味道。颜色有红、黄、紫、绿五彩缤纷。这是养颜野菜羹

住龙门民宿

与酒店相比，我更喜欢住民宿，这里的民宿有着传统民居的质朴素雅与现代人的品质舒适，雅俗相和，体验最原生本真的民俗悠闲生活。四周环境苍翠宁静，当然想热闹的话这里还可以举办篝火晚会。
民宿内配套有无边际游泳池，健身房，书吧，咖啡吧，电影院等
房间宽敞开阔明亮，布局合理，原木家具给人一种不以言说的朴质与宁静,房间简约而不简单，窗外风景更是美如画。
卫生间干湿分离，功能设计合理好用。

第二天 莲花景区

莲花景区位于大容山主峰莲花山一带，景区面积1247平方千米，平均海拔1100米，是一处世外 桃源 。至身其中，仿佛走进了 欧洲 的山水画卷。景区内最具特色的景点有：欧陆风光、 桂东 南第一峰、莲花瀑布、南流江源头、风车阵、高山溪谷、原始森林、天泉、南方西岳封禅台、古战场遗址等。
欧陆风光——它位于大容山莲花景区的核心景观的高山草甸上，海拔约1000米。四面环山，草甸与森林、湖泊、溪流交织相连，风光迷人，设置婚纱摄影基地、城堡酒店、教堂等项目，让游客体验绿野仙踪的曼妙之旅。
在天湖水库的旁边是成千上万亩的柳杉林，绿意浓浓，幽静至极，林中空气清新，散发着花草的香味，林下松针如毯。在林中漫步，深深吸一口空气，清新的空气沁人心脾，清爽之感油然而生。
这里有一泉眼——天泉，在莲花景区小莲塘东面有一股溪水从地缝中流出，一年四季长流不断。此泉水味甘甜，据水质部门检测，泉水中含有大量有益健康的矿物质微量元素。
在草地上，树上处处弥漫着浓郁的圣诞氛围。
为了营造浓郁的圣诞气氛，工作人员用心地把圣诞节的元素装点着园区的每一个角落，巨型的圣诞树，充满欧陆风情的基督教堂前，圣诞老人驾着的驯鹿雪橇带着一大袋礼物来到这里探访我们咯。
迷蒙的雾气里，我们发现了不远处的柏树上闪耀着跳跃的色彩，散发着浪漫的童话气息，绝对是拍照打卡的绝佳圣地。
一群逼真的呆萌驯鹿玩偶，远远望去就像真的一样，像在奔跑慢慢消失在雾中。
据说天气好如上图般美丽，这次雨雾实在太大了，留点遗憾下次再来。
下午6点灯饰悄然点亮，带有东方风情的园内景观，将这里的圣诞市集点缀的无比迷人，彷如走入一部老电影中。。，梦幻，如梦似真。
高山冰雪狂欢节期间更有冰屋帐篷体验，冰雪COSPLAY秀，冰雪祝福等节目在教堂举行，雪花飘起，欢乐走起。
夜幕降临，园区的灯光秀悄然而至。无数盏彩灯点缀着梦幻般的园区，让我们彷如置身于一个充满 北欧风情的圣诞乐园，灯光随着音乐的律动，变换着色彩和展现方式，仿佛是奇幻的欧若拉。漫天飘雪从天而降，更是让这里充满浪漫的气息，让大家感受浓厚美好的圣诞时光。
亮灯后在莲花餐厅享用BBQ啤酒盛宴，菜式多样，还有BBQ，啤酒，水果等，让你大快朵颐。当然期间还有圣诞老人派礼物，歌舞，魔术表演等，让晚宴高潮一浪接一浪。
期间还有多个游戏，大家的欢乐互动顿时变成一个欢乐的海洋。

晚上住房车

晚宴结束后我们回到了汽车营地，这里可提供服务包括住宿、露营、餐饮、娱乐、拓展、汽车保养与维护等。
我们这次住的是房车，因为第一次住房车心情还是有点小激动，房车虽小，但五脏六腑俱全，犹如一个浓缩版的家一样，里面有如厅，房，厨房，洗手间等功能区域，还有冰箱电视机等，功能设备还是挺齐全的。
这算个小小的厅吧，很温馨舒适，可以在这里办公，打牌什么的，还可以开窗看看外面美丽的风景，不会觉得无聊。
房车里还有电磁炉等，可以在上面做菜做面吃哦。
在汽车营地旁还有7间独立成栋集装箱住宿，每个集装箱都有观景台，房间里空调，热水器，更有落地玻璃窗可以一览大容山的美景。
天气好的话在这里露营，躺着也能看星空了，想想也浪漫。

第三天 莲花景区

第三天一早我们就去寻找南流江源头，我们顺着指示牌走进了这个原始森林。
原始森林的神秘感油然而生继续往原始森林深处走，脚下原始植被茂盛，几乎看不见土壤，石上地衣、苔藓生于其上，随处可见青腾缠绕树上，有的长成环状，远远望去还像一条条蛇缠绕在树上。
由于雾气很大，林内树木繁茂，根深叶密，更加让人有一种神秘幽深的感觉。
南流江，是 广西 独流入海第一大河，是 广西 南部独自流入大海的河流中，流程最长、流域面积最广、水量最丰富的河流，自北向南流，故名南流江，而在这里我们找到了南流江的源头。
原始森林被列为 广西 自治区级大容山自然保护区的核心区。走进原始森林，但见数目高大挺拔，万木峥嵘，虬枝交叉，有的树木两人才能环抱，树高二三十米，直插云天，树龄已逾百年。
看完南流江源头后我们来到了七一哨所，因建于1971年而得名，当时为了贯彻毛主席关于民兵工作的“三个落实”的精神，并根据 玉林 军分区， 北流 武装部的指示，于1971年8月份建成哨所第一层，1976年扩建哨所的第二层，具有防空，防敌，防特及监控火警等多种功能，该哨所也为当时国防备战以及林场的护林防火工作发挥了重要作用。
最后我们向 桂东 南第一峰出发，虽然雨雾很大，但犹如仙境般的感觉也是另一种美哦。
山上除了高山草甸外，还有野杜鹃，如果是三四月来的话看到的又是另外一种景象，绿油油的草地，还有开满杜鹃花的山头，所以一年四季会有不同的景色。
经过一段时间攀爬之后终于 成功 登顶了，海拔1275米，是 桂东 南地区最高的山峰，耶。
天气好的话在这里可以一览众山小，还可以看到云海哦，当然还可以看到大容山别致的风景，一座座风机矗立在山头上，随风转动。
