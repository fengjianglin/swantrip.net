广东之行

游记 主题乐园 动物 温泉 自助游 2019/02/17

前言

遥想第一次出远门，就是在广东，一转眼间，十多年就过去了。
如今再次来到广东，蓦然回首，无限感慨；一路前行，却又充满惊喜。

行程

D1：永州-广州南G6074
广州塔 
D2：广州长隆野生动物园
广州-恩平
D3：恩平-江门恒大泉都
D4：江门恒大泉都-恩平西门客运站
恩平西门客运站-广州滘口
广州南-岳阳东G1140

交通

To广州：1、航空：广州白云国际机场网络已覆盖全球200多个通航点，其中国际及地区航点超过80个，通达全球40多个国家和地区；
2、铁路：京广铁路是中国南北铁路交通主干线，连接京沈、京承、京包、京泸、京原（平）、石太、石德（州）、陇海（连云港-兰州）、浙赣、湘黔、湘桂、广九等线。此外还有广州至茂名的铁路干线-广茂铁路、广州至汕头的铁路干线-广梅汕铁路，这两条铁路沟通了广东省东西的交通。
3、公路：广州市公交运力不断增强，先后修建了多条城际间高速公路，市内也修建了高架路、公路桥、过江隧道等。
To广州塔：乘地铁3号线、APM至广州塔站下。
To长隆：乘地铁3号线或7号线至汉溪长隆站下，再坐乘长隆免费穿梭巴士。
To恒大泉都：恒大泉都紧邻沈海高速（开阳段）圣堂出口，距离广州市区仅不到2小时车程，自驾走S43广珠西线高速-G1501广州绕城高速-G15沈海高速-恒大泉都。
如果不是自驾，也可从广州各大汽车站坐大巴车到恩平，再打车前往恒大泉都。

美食

粤菜即广东菜，是中国传统四大菜系之一，由广州菜、潮州菜、东江菜三种地方风味组成，取百家之长，用料广博，选料珍奇，配料精巧，善于在模仿中创新，依食客喜好而烹制。
既来之，又怎能少得了当地美食？

住宿

D1：入住金圣宾馆，地址：广州市天河区黄埔大道西187号，点评：位置还行，离正佳广场较近，交通也很便利，各方面条件一般，胜在经济实惠。
D2：入住恩平银城宾馆，地址：恩平市恩新西路26号，点评：恩平银城宾馆位于恩平步行街，交通便捷，但设施比较陈旧，略有异味，胜在经济实惠。
D3：入住恩平恒大泉都，地址：恩平市良西镇恒大泉都，点评：恒大泉都拥有400床豪华套房以及近800床精品客房，最具吸引力的莫过于温泉别墅，环境优美，私密性强，自带私家庭院及温泉池，既可与朋友一起BBQ，也可随时泡个温泉，甚至是享受影院的视听盛宴。
不管是硬件还是软件，都堪称无可挑剔，细节彰显品质，匠心致以初心，一切美好得如同童话一般。

第一天

2018年5月24日。
离开永州，继续选择南下，来到了心心念念的广州。
来到广州，自然不能错过广州塔。
世界大多数电视塔均以地名命名，广州塔也不例外，以600米的总高度问鼎中国之最，位列世界第四。
值得一提的是，荷兰设计师夫妇马可.海默尔与芭芭拉.库伊特将广州塔设计成一座椭圆形且扭曲上升的建筑，扭转的地方又格外纤细，像极了女性的小蛮腰，故而得了个“小蛮腰”的昵称。
广州塔集都市观光、至高游乐、时尚餐饮、婚庆会展、影视娱乐、环保科普、文化教育、购物休闲等于一体，从下到上分为5个功能区。
其中，A区为负2~7层，设有游客中心、一楼、二楼登塔大厅、城市规划展、珠江摄影观景平台、江畔吧、宴会庆典大厅、多功能展厅、多功能会议室、旅游咨询中心、汇礼轩商场等；
B区为17~22层，设有金逸电影空中会所，由一个4D及2个3D影院组成，有最先进的视听设备。
C区为29~33层，D区为65~68层，这里是天梯的起点和终点，栈道共35层，1096级阶梯围绕塔身核心筒螺旋而上，跨越总高度166.4米，是世界上最长的螺旋式空中云梯。
E区为97~110层，107、108层分别设有以星空为主题的室内观光大厅，全透明钢化玻璃构造的“悬空走廊”是拍摄留念的绝佳地点。此外，这里还有世界最高的邮局和纪念品售卖柜台。
广州塔的独特之处在于塔顶开设户外观光平台，利用塔顶区域开发游乐项目，坐拥3项世界之最：世界最高横向摩天轮，世界最高惊险之旅极速云霄，世界最高户外观景平台488摄影观景平台。
广州塔摩天轮位于广州塔塔身顶450米高空处，是世界上最高的摩天轮。
与一般竖立的摩天轮不同，广州塔摩天轮不是悬挂在轨道上，而是沿着倾斜的轨道运转。摩天轮共由16个观光球舱组成，观光球沿着与主体结构呈15度倾斜的轨道运转，使用国内首创“刚性调平装置”确保平稳。
摩天轮素来是浪漫的象征，如此与众不同的浪漫场所，自然就成了广州最浪漫的定情求婚圣地。
广州塔极速云霄位于455~485米的天线桅杆，是目前世界上设置于最高出的垂直速降游乐项目。吉尼斯世界纪录委员会甚至专门开设“世界最高惊险之旅”项目，广州塔“速降体验”成为首个且唯一获得者。
极速云霄与其它垂直速降游乐项目最大的不同是，它有两组座椅，一组坐跳，一组站跳，短短1秒多钟内体验30米的急坠，既可以凌空俯瞰羊城美景，又可以体验高空速降，玩的就是刺激和心跳！
“云霄488”户外摄影观景平台坐落于广州塔天线桅杆惊险位置的488米处，是目前世界上最高的户外观景平台，2013年6月荣获吉尼斯世界纪录。
登上观景平台，立于城市之巅，羊城全貌尽收眼底，感受一番“会当凌绝顶，一览众山小”的快意。
此外，广州塔塔身共有46环6696盏LED灯，每当夜幕降临，灯光璀璨夺目，变化无穷，与珠江新城景观交相辉映，美不胜收。
十一年前的匆匆一瞥，还未来得及停留，就已挥手作别。
而今登上广州塔，满城灯火尽在眉眼之间，独不见当时月。彼时此刻，昨我今我，在时间的荒野中重重叠叠，仿如灯塔之光，一路向前漫溯。

第二天

5月25日。
来到广州，同样不容错过的，还有长隆。
每一座大城市，都有标志性的主题公园，上海有迪士尼，杭州有宋城，深圳有世界之窗，广州则有长隆。
广州长隆旅游区包括长隆欢乐世界、长隆水上乐园、长隆野生动物世界、长隆国际马戏大剧院、长隆飞鸟乐园等景区，各个园区分开独立运营。光是长隆野生动物世界，都能逛上一整天。
长隆野生动物世界于1997年开园，凭一己之力掀起了全国性的野生动物园热潮，创造了单日入园游客8万的奇迹，一跃成为亚洲野生动物园的领头兵，甚至打破了主题公园五年宿命论。《爸爸去哪儿》大电影以及《奇妙的朋友》都慕名而来，在此取景拍摄。
长隆野生动物世界以大规模野生动物种群放养和自驾车观赏为特色，是全世界动物种群最多、最大的野生动物主题公园，拥有中国国宝大熊猫、澳洲国宝考拉、马来西亚国宝黄猩猩、泰国国宝亚洲象、洪都拉斯国宝食蚁兽等世界各国国宝在内的500余种20000余只珍奇动物，堪称“国宝乐园”。
园区主要分为步行区和乘车区两个区域，都由一系列展区组成。
步行游览区就像一个巨大的容器，将全球各地的风情网罗其中，既有充满热带风情的非洲长廊，也有雾气弥漫的亚马逊雨林，更有来自“远古”的侏罗纪森林，还有大象园、百虎山、金猴王国、金蛇秘境以及集万千宠爱于一身的长隆大熊猫中心……
在这里，你不仅能大开眼界，还能跟各种各样的动物亲密互动，譬如用香蕉投喂大象，拿树叶喂食长颈鹿，给可爱的老虎宝宝喂奶……
此外，这里还有各式各样的演出，身怀绝技的动物们轮番上阵，颠覆传统的表演模式，充分展现了各种动物在自然环境中的行为状态，令人叹为观止。
对于“老司机”而言，自驾游览区无疑是策马奔腾的极佳去处，不用出国，就能轻松穿越“五大洲”，看遍世界珍奇动物。
这不，你刚在澳洲森林看到袋鼠奔跑，下一个场景立马切换成“网络神兽”羊驼在南美雨林漫步，转眼间变成“熊出没”的狂野地带，紧接着大群大群的长颈鹿、斑马、角马在一望无际的东非草原上穿行……野性自然之美如同花蕾一般绽放，动物原始风貌一览无余。
这还不算完，景区为了给游客带来全新的观赏体验，打破传统的观赏动物模式，特别设计了从空中观看野生动物的缆车项目，长达2.7公里的行程，最低点距地仅1米，俨然与兽同行，与鸟同飞，无论是水里游的天鹅，还是地上跑的斑马，还是空中飞的鹦鹉，都尽收眼底，说不出的惬意。
在动物王国的一天，走过丛山，路过草原，穿过雨林，寻觅自然深处的风情，中国大熊猫、澳大利亚考拉、孟加拉白虎、泰国亚洲象、马来西亚马来貘、非洲黑犀牛、塞拉利昂倭河马……来自世界各地的珍稀动物齐聚一堂，千姿百态，妙趣无穷。

第三天

5月26日。
遇见不同的人，邂逅不同的风景，收获不同的成长——这大抵就是旅行的意义。
心之所向，再遥远的风景，也会义无反顾地抵达。
恒大以房地产起家，对于房地产业早已驾轻就熟，但显然并不想局限于此，相继完成了地产、金融、健康、文化旅游四大产业布局，并在文化旅游产业上加码，一步一步地构筑起庞大的商业帝国。
恒大泉都可以看作地产与文化旅游的联姻，直接定下了“全球首席世界温泉度假城”的目标，集结三大功能中心，荟萃十三大国际风情酒店，融汇八国风情温泉会馆，俨然形成了一个完整的体系，集观光、游乐、休闲、运动、会议、度假、体验、居住于一体，乐享品质，尊享生活。
初入恒大泉都，先是惊叹于园林之美，随后折服于建筑之美。
气势磅礴的温泉别墅、优雅大气的光景洋房、随处可见的欧式建筑，传承欧式的古典和浪漫，宛如充满异域风情的生态大观园，绿影婆娑，繁花盛放，鸟语花香，相映成趣，瞬间有一种置身于欧式小镇的感觉。
在接待中心，浪漫元素也是应有尽有，喷泉、雕塑、鲜花一个都没少，尽显梦幻之感。
而一支乐队正在台上欢歌，俨然为本次行程拉开了序幕。
此时已近晌午，享受完美味佳肴后，我们就开启了正式地探索之旅。
首先路过的是生态果园，枝繁叶茂，硕果累累，自带吸睛效果，不妨挎上菜篮子，亲自去采摘新鲜水果，与大自然来一次亲密的接触，体验一下“自己动手，丰衣足食”的农家之乐。
其次是射箭场，射箭原本就属于“六艺”之一，随着社会的进展，这一项传统技艺已经不可避免地走向没落，有此机会，当然不能错过，不光是感受射箭的乐趣，同时拥抱源远流长的射箭文化。
最后则是来到CS基地，已经有一拨人正在开启团队游戏，我们虽没有身着迷彩服，手持模拟设备，来一场酣畅淋漓的CS大战，却找到了另外的乐趣。
训练基地是一处废弃的碉楼，具有鲜明的特征，运用外国建筑中的穹顶、山花、柱式等建筑元素大做文章，形成了千楼千面的建筑式样，墙体比普通的民居厚实坚固，窗户则比民居开口小，都有铁栅和窗扇，外设铁板窗门，角堡内开设了向前和向下的射击孔，进可攻，退可守，集防卫、居住和中西建筑艺术于一体。
可以说，碉楼是时代发展的产物，与当地的自然地理、人文环境及社会经济密不可分。
复古的建筑、老旧的教室、参天的古树……既有一种浓浓的年代感，又弥漫着一股荒野气息，简直就是拍照的圣地。
旅行容易让人上瘾，拍照亦是如此，一旦确认过眼神，根本就停不下来！
我希望，每一个时刻，都像彩色蜡笔那样美丽；我企盼，每一段记忆，都像照片一样定格；我期待，每一次回眸，都有你的身影。
随着生活节奏越来越快，越来越多的人处于亚健康状态，健身就显得尤为重要。
恒大泉都运动中心分3层分区打造，除了设有篮球场、网球场、游泳池等标准化运动场馆外，还设有桌球室、棋牌室、瑜伽室等，配备的都是专业运动标准的器材，力求提供一个多元化的运动中心。
生命不息，运动不止。如果一个人觉得不够燃，不妨邀上三五好友，挥洒激情，绽放活力。
恩平既是温泉之乡，又怎能少得了温泉？
恒大泉都是恩平地热国家地质公园里四大温泉之一，拥有三大天然泉眼，贯穿园林，纯净如玉，水温达48~53℃，富含钙、钾、氮、氢、硫、氡等多种有益身体健康的元素，对神经性骨痛、消化道、风湿等多种疾病具有特殊疗效，同时还能起到舒筋活络、强身健体、润肤养颜、安神定神、抗衰老等保健作用，乃是名副其实的优质医疗温泉。
恒大泉都以温泉为主题，特意打造了一个充满异域风情的特色小镇，包括意式温泉水疗和海洋温泉疗、纯正芬兰特色盐浴、土耳其蒸房理疗和鱼疗、泰式SPA护理及、纯正日式裸汤等等，不用出家门，就可一网打尽世界各地的佳品温泉。
在芬兰温泉馆，体验的是迷人的北欧风情，从特色盐浴的无尽乐趣，到冰天雪地的浪漫刺激，感受冰火两重天的畅快；
在日式温泉馆，有各具特色的绿茶汤、红酒汤、啤酒汤等天然温泉汤浴，弥漫着浓浓的北海道气息，体味的是日式汤浴的魅力；
在意式温泉馆，明净湛蓝的天空、绿意盎然的灌木、缭绕的天然雾气，仿佛徜徉在地中海的碧海蓝天之下，感受的是托斯卡纳风情；
在土耳其式温泉馆，伊斯兰风格的城堡、独特的浴疗温泉，再配上芳香湿蒸的特色理疗，细味的是棉花堡的雅韵；
在泰式温泉馆，以泰式风格为主调的建筑，将热带丛林风情的泰式鲜花浴、精油浴和草药熏蒸浴，与特色芳香SPA完美结合，在袅袅升腾的香气中，尽享东南亚风情！
时尚与浪漫共舞，生态与人文交织，浸于温泉水中，洗去尘垢，消除疲惫，抛却俗事，心安自在，怎一个惬意了得！
真正的平静，不是避开车马喧嚣，而是在心中修篱种菊。
生活不在远处，脚下即是前方，寻一处惬意居所，觅一段惬意时光，静放一帘幽梦，与自己来一场心灵对话。

第四天

5月27日。
在星光中入睡，从晨曦里醒来，既可以选择在林荫小道漫步，也可以来一场晨跑，呼吸湖水与园林带来的负氧离子。
恒大泉都以欧式造园手法打造有80万㎡皇家园林，拥有2.8公里奢华景观大道和8大主题景区，以三大景观大道为轴心，以上百种稀缺植被为背景，运用情景融合的手法巧妙的将山、水、园林穿插在各栋异国风情的建筑之间，与社区环境浑然天成。
通往泉都的入口处，两侧叠石为瀑，潺潺的流水掩映在绿树繁花之间，错落有致，营造出峡谷森林的错觉，仿佛迷失在绿野仙踪的世界里……
正所谓湖光山色，有山有林，又怎能缺少湖泊的踪影？
五大湖系精雕细琢，巧妙布局，优美的湖面沉静闪动，悦耳的流水喷泉激情跳跃，根本不需要想太多，或赏景，或听歌，或散心，仿佛整个世界都安静了下来。
可以说，每一滴水、每一棵树、每一幢建筑，都具有美学造诣，就好像美妙的音符一样萦绕在恒大泉都。
社区内配套优异，周边景点更是引人入胜！
花海世界以四季时花观赏和水上乐园为主题，融合养生文化与田园生活情趣，集自然观光、生态农业、花卉观赏、文化艺术品鉴、休闲游玩、科普教育、餐饮美食于一体。
步入景区，其内植有数以百计的花卉植物，俨然成了“花卉王国”，长达200米的林荫道上，花开得正好，在阳光的衬托下，犹如阿娜多姿的少女，微风拂过，漾起一阵诱人的芳香，花不醉人人自醉。
与此同时，花海世界拥有以机动游乐为主的20多个适合亲子、激情游乐的项目，包括打钱罐、投木桶、跳跳球、贪食大嘴巴、投篮球、飞镖红心、幸运射击、愤怒的小鸟、迷你保龄球、打鲨鱼、弓弩手、钓鸭子、扎气球、大炮神射手、投炸弹等，无论是大人还是小孩，都能拾回童趣，乐而忘返。
在这当中，花海摩天轮显得尤为引人注目，高达120米，拥有24个透明吊舱，别具童话色彩。
乘坐山顶摩天轮，登临花海之上，将花海世界的旖妮风光尽收眼底，满满的都是喜悦。
此外，景区还建有仿古观光长城，全长2.7公里，宛如一条青龙盘旋在崇山峻岭之上，蔚为壮观，不用去帝都，就能一睹长城的雄姿。
走进仿古长城，里面是一个以爱国教育为主题的长城文化展览馆，馆中展出了我国历史上每一个里程碑的时刻。
置身馆内，每个人都可以感受到悠远文化的熏陶，仿若在历史长河中漫游一圈，切身体会着中华文明的博大精深，一种自豪之情油然而生。
如果你觉得这还不过瘾，可以恐怖世界里，四周一团漆黑，完全不知道下一秒会发生什么，也许是转角处的一声尖叫，也许是突然冒出来的“厉鬼”，也有可能是背后凭空出现的一只手……近距离地体会来自背后和空气里的恐怖元素，绝对让你头皮发麻，双腿绵软，脊背冰冷，却又大呼过瘾！
然而，美好时光总是短暂的，转眼我就要离开。
爱的，不爱的，一直都在告别中。告别中，不越界，知进退，也许才有徘徊后的万里晴空。
生活在别处，筑梦向远方，纵有别离，也要轻歌曼舞，砥砺前行。
