关于交通

交通卡，（T-money卡）
如果想要玩转 首尔 ，首先，你必须要有一张交通卡，交通卡可以在每个地铁站专门卖卡的机器上买到，不过这种机器没有中文服务，不过看英文也基本可以轻松搞定的。
充值的话，也是在地铁站里面可以轻松完成，而且充值和买单次票的机器有中文服务哦。
首选：地铁
首尔 的轨道交通可以说是四通八达，基本地铁可以满足你到达任何景点、商场的需求。但是这也带来了线路图的密密麻麻，尤记着上次一个人在地铁里搭错车，带着哭腔的跟肖总抱怨，肖总当时冷漠的回答我“谁让你自己跑去 韩国 的..."。这次虽然带着他一起，他还是不忘找机会奚落我，有了男人是不是就是不一样啦，不会走错路了吧。。。我也只有点头认错的份了。
地铁收费是19岁以上1,350 韩元(刷卡1,250 韩元)；13至18岁1,350韩元(刷卡720韩元)；7至12岁450韩元(刷卡450韩元)；6岁以下免费。10～40km，追加100韩元/5km，超过40km时，追加100韩元/10km，如果你是购买一次性地铁的交通卡，记得出站的时候退卡然后退得500韩元的押金。
第二选择：出 租车 
首尔 的出 租车 普遍来说都是非常整洁干净的，而且司机师傅也都非常亲切，但是硬伤是普通出 租车 的司机基本不会英文，也许偶尔你能碰到个会说几句中文的师傅。不过所幸，现在网络很发达，只要Google好地址给师傅们看，师傅基本都会用自己的导航输入好地址，准确的把你送达目的地。
首尔 的出 租车 不算贵。
普通出 租车 （除黑色的），起步价3000韩元，超过2公里以后，每增加142米加收100韩元。以低于时速15公里的速度行驶时，每35秒加收100韩元。0:00-4:00之间乘车时需多支付20%的深夜附加费。还有，出 租车 普遍都接受交通卡支付的哈。
模范出 租车 （黑色的），司机会些英文和中文，起步价5000韩元，超过2公里以后，每增加164米加收200韩元。以低于时速15公里的速度行驶时，每39秒加收200韩元。
第三选择：公交车
不是迫不得已的情况下，并不建议搭乘公交车，主要原因是车上没有中文报站，如果你和我一样，刚好google到必须搭乘公交才能到达的话，一定要随时关注地图，不要坐过站哦，而且 首尔 的公交车，在准备下车的时候，也是要自己按铃提示给司机师傅的，要不有可能会不停车的哈。如果坐公交，刷交通卡会便宜100韩币。

关于怎么花钱

现金
国内换少量现金，或者不换也可以， 仁川 或 金浦 机场换少量现金orATM取，剩下的建议带着人民币去明洞 的换钱所换钱，汇率最合适，当然每家换钱所汇率略有不同，如果你有时间，可以多逛逛找家最合适的。
刷卡
首尔 作为国际大都市，咱们国人最爱的大买场，银联刷卡这种后勤服务做的也是极为到位。大型商场基本都接受银联，有时还会有特殊优惠活动，买东西时候记得多问句，可不可以银联支付哦。不过711等便利店还是只接受VISA这种国际信用卡的，现在基本所有银行都提供全币种信用卡，出境刷卡更合适些。
支付宝、微信
相比一年多前的 首尔 ，现在 明洞 、林荫路这种大型购物集中地，更多的出现了支付宝和微信支付的标志，支付优惠多多，我天朝真是越来越强大了哈哈哈。

关于退税

首尔 非常非常多的店面都可以退税，只要在同一家免税商店内消费超过30000韩元（可以是当日累积消费），并且三个月之内携带所购商品出镜，那么就可以办理退税。 韩国 的消费税不算高，在扣掉必要的退税手续费后，大概还可以退8%。
具体的退税操作（适用于退税金额5万韩币以上的）
1.办理登机手续时只换取登机牌，不要托运行李。
2.在海关检查台向工作人员出示所购商品和退税单，经确认后加盖“物品出境（ALL GOODS EXPORTED）”图章。
3.在大型物品专用柜台托运行李并接受出境审查。
4. 完成出境审查后，前往相应的退税柜台（Gate 27~28），出示退税单和护照即可领取税金。
不过如果退税金额在5万韩币以下的，就可以省去海关检查和盖章的步骤，也就是说可以直接托运行李然后去自动退税机（KIOSK），机器就在安检入口的旁边，完成扫描以后，安检出境，然后到退税柜台退钱就好了。

DAY2的第二个景点定位 清溪川 。地铁2号线，在市厅(City Hall)站下车，5号口出来就能到了。
清溪川 其实就是 首尔 市中心的一条河流。这个小河曾经被覆盖成为暗渠， 清溪川 的水质也因为废水的排放而变得恶劣。在1970年代，更在 清溪川 上面兴建高架道路。2003年7月起，在 首尔 市长李明博推动下进行重新修复工程，不仅将清溪高架道路拆除，并重新挖掘河道，并为河流重新美化、灌水，及种植各种植物，又征集兴建多条各种特色桥梁横跨河道。修筑河床让 清溪川 的水不再轻易流失，在旱季的时候引汉江水灌 清溪川 ，让 清溪川 长年不断流，分清水及污水两条管道分流，保证水质保持清洁。这个工程总共耗资9000亿韩币，在2005年9月完成。想来这样巨大的治污决心，真的很值得学习。
