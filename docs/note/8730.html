一篇美食与美景的安逸游记

【交通篇】那么远，这么近

桂林的历史，万年前的文明就不说了，早的可以追溯到始皇帝开发岭南六郡，无数将士的汗水开通的 灵渠，至今仍发挥着它的功能。大自然总是吝啬的，给了桂林如画的美景，却给了她不便的交通。幸好，现在交通的便捷，高铁，飞机等等高速的交通工具，使得桂林不再遥远。一张机票，两个小时就足够我从东南的 福州到达桂林了，天涯变咫尺。所以，桂林的旅行不再是一场有闲有钱的梦，利用周末的时间，一场山水之旅就可实现。【飞机】 桂林两江国际机场是位于广西壮族自治区桂林市 临桂区两江镇的一座4F级干线机场，距桂林市中心约 28公里，面积406平方公顷，是广西唯一的4F级机场和广西民用专用较大的航空港。机场大巴（市区线） 两江国际机场-桂林民航大厦 根据航班动态滚动发车 间隔时间：根据航班时间调整 票价:20元 【火车】桂林三个火车分别位于不同的区、县，其中距离市中心最近的是 桂林站，其次是桂林北站，最后便是桂 林西站。每个火车站都有往返市区的巴士，最远的桂林西站也就1个小时左右。其中 阳朔往返桂林主要还是以桂林西站和北站为主。【市内交通】桂林风光主要集中在桂林市区及周边县区，而且每一个景点之间都有一定的距离，所以自驾是游览桂林风光的最佳选择。当然退而求次，拼车或包车也是一种选择。市区的景点可以选择自行车或者直接步行，比如 东西巷到 独秀峰、 七星景区等都只要十几分钟的路程，沿江漫步也是很舒适的，处处是美景。

桂林的夜与美食

桂林是一座适合居住的城市。每天迎来无数南来北往的游客，却也不太拥挤。放下行李，已经是晚上9点了，来到了桂林的庙街。桂林文化宫庙街是桂林美食的新地标，在庙街，不用出桂林也能吃到全国各地500+的美食。夏初，漫步在不甚拥挤的庙街，感受着这座城市的温度。
庙街共两层，一楼是汇聚了知名品牌的美食广场，二楼是风格各异的主题餐厅。夜晚的庙街，是不是有点梦幻的感觉呢，吃货的天堂。地址：桂林市秀峰区文化宫内。

玻璃桥

清晨，睡到自然醒，在路边的小店里点上一份正宗的桂林米粉，只要4.5元，实惠的不敢相信。饱饱的吃过一碗，就随意而走吧。
桂林米粉是广西桂林地区历史悠久的特色传统名小吃，以其独特的风味远近闻名。其做工考究，先将上好的早籼米磨成浆，装袋滤干，揣成粉团煮熟后压榨成圆根或片状即成。圆的称米粉，片状的称切粉，通称米粉，其特点是洁白、细嫩、软滑、爽口。其吃法多样。制作最最讲究卤水的制作，其工艺各家有异，大致以猪、牛骨、罗汉果和各式佐料熬煮而成，香味浓郁。
两江四湖环绕着桂林城，两江四湖指漓江、 桃花江、木龙湖、桂湖、榕湖、杉湖，中国独一份，可说是最美市区了。来到桂林，不一定要追着某个景点走的，市区内不高的建筑，保护着这个世界上独一无二的喀斯特地貌奇景。打开手机地图，发现一个有趣的名字，玻璃桥。
桂林玻璃桥位于榕湖，桥体采用工艺复杂，造型考究的水晶玻璃制品为建筑构件，集中体现了现代玻璃工艺的水平，是目前为止我国最早采用特种水晶玻璃承重体系的实用性桥梁。
下次再去桂林，一定要去看看夜晚的玻璃桥，那定是璀璨夺目的。
沿着水岸信步而行，不多远就是双塔公园。象山是桂林的城徽，而双塔则是新桂林的标识。日塔、月塔与象山上的 普贤塔、塔山上的寿佛塔，相互呼应，相互映衬，有“四塔同美”之说。桂林 日月双塔号称世界第一铜塔，是 桂林旅游的一个绝美夜景观赏地。
满城绿意，生活在其中，没有大城市的快节奏，很是安逸。

【住在】春天里

小隐隐于野，大隐在于市。中国人向有隐士的情结，隐士的最高境界自然不是一人独居深山野林之中，穿梭在车水马龙的城市里或许你的心已经有些倦怠。在我看来，隐，更多的一种心境，对生活的淡然，于尘世间追求内心的安宁。
荻花秋，潇湘夜，橘洲佳景如屏画。碧烟中，明月下，小艇垂纶初罢。水为乡，篷作舍，鱼羹稻饭常餐也。酒盈杯，书满架，名利不将心挂。
这次在桂林入住的是仅有六间客房的【隐想 春天里】民宿，民宿位于穿山东路28号彰泰春天北苑1栋，从 桂林机场到春天里也仅需1个半小时的公共交通，打车的话50分钟足够了。不赶时间的话，我更建议坐在公交车上看这座宛如镶嵌在优雅盆景中城市在以前缓缓而行。
春天里所在的小区，不知道是不是沾染了太多桂林山水的诗意，景区的大门竟如此典雅。春天里没有很大的招牌，我跟着导航前行，慕然回首，原来已经到了。
隐享· 春天里民宿以中式合院住宅的形式，通过建筑、连廊与茶室将庭院围合，将假山、水景、植物引入其中，为住客提供幽雅、私密的庭院生活，使宅院成为室内与室外融合、宅院合一的居住单元。据说，这里的每个细节，装饰都是设计师老板的亲自布置。
春天里的庭院不大，中国古典园林的设计精髓在这里淋漓尽致的体现着，移步换景，假山流水，营造了一幅古典的山水田园居的画卷。
一棵枫树，给春天里平添了几分春意，精致的屋檐，有那么一刻，我以为回到了古代隐士的后花园，纵然外面是车来车往，在春天里，却可以独享一份宁静，不觉间，我竟走神了。
听说，桂林的雨已经下了大半年，果然，凌晨又是一场夏雨如期而至。雨，落在春天里，洗净了尘埃，分外的清新，住下了就不忍离开。
春天里的客房布局不一，各具特色，每个独立的套房都采用了落地窗设计并且 采用了炭黑色的玄武岩将目光从房间引向设有落地窗的榻榻米，在这里客人可以一边泡茶一边欣赏私家庭院的景致。
房间的布置非常精致，可以看出每个细节都是经过精心的设计，就连小小的冰箱，每个房间的造型也是不一样的。冰箱里的饮料、啤酒，还有房间的零食都是免费的，再不担心半夜找不到吃的了。
我选择的是电影房，来一场刺激的爆米花电影，再配上啤酒，泡面，可以一天不出门了，一个人的旅行也不会太无聊。
挺喜欢这个音箱的，音质很不错，打开音响，连接上播放器，悠扬的旋律在房间流淌着。 春天里带给人的居住感觉更多的是一种平静，时光，缓缓的，不骄不躁。
房间用品超过我的想象，除了酒店常规的洗漱用品吧，春天里甚至还提供了面膜，想的足够周到，指甲刀等用品也是一应俱全的。
到会客厅读一本书，闲时，和那只叫花花的小猫一起发呆，聊聊天，拍拍照。旅行，不必总太匆忙。
在春天里，我们可以品一壶茶，欣赏着设计师老板收藏的各种好玩的物件。春天里还有一套特殊的套房，300多平方的面积与其说是一间房，更像是一座小小的私家庭院，也是个人的修身养性之所在。
我们都沉醉在 春天里了，优雅的环境，是天然的取景地，很容易就拍出让人心动的美美照片，有时间的话，甚至可以穿上民宿免费提供的汉服，在这古典与简约融合的环境里，一定是不着一丝穿越的痕迹。
我喜欢夜色下的宁静，橘红灯火中的一点暖意，不论人生还是旅途，我们需要的莫不过如此。

良辰美景，如斫西餐厅

突然觉得自己没文化了，如斫，咋读的呢？努力的回忆，原来是如zhuo。据说这名字的来历跟老板有关系。老板有一位喜欢收藏古琴的朋友，朋友曾跟我、他提起的“斫琴”让老板印象深刻，斫是砍伐的意思，但加上琴字却变成一件古雅的事情，老板特别喜欢这个斫字，所以凑成了如斫。好吧，这是一个很有趣的名字，会不会是一家很特别的餐厅呢？
如斫西餐厅位于桂林市中心漓江精华段，桂林七星区临江路，餐厅的对面就是 叠彩山，夜色之中的叠彩山色彩斑斓，充满着梦幻的气息。
如斫是一家法式西餐厅，却有着最古老的汉唐风韵，人的想象力是无限的，高古石刻、汉唐雕塑是我国雕塑的颠峰，也是文化的源头。把这些“老东西”搬运到此，自然陈列，呈现出最真实、自然的一种状态，食客可以运用自己的经验与知识各自感悟，与古人对饮，感受文化之美。
魏晋之前的中国文化，可说是创造力与想象力的巅峰，那些保留下来的雕像，石碑无不彰显着那个时代的浪漫气息，有着一种脱俗的灵魂。
神秘古风的南北朝佛像与“工业风”“高级灰”等后现代主义相结合，馆藏级的高古雕塑陈列在现场，古老的能量、漓江的山水，梦幻的不太真实。
在这里，很合适来一组充满文艺气息的旅拍。中西融合，古今贯通的场景，你的相片可以是穿越历史长河，回到那个神秘而辉煌的时代，也可以在充满后现代气息的场景中光彩绚丽。
神秘主义和浪漫主义的碰撞，就在如斫餐厅。

吃在桂林

桂林最驰名的美食自然是桂林米粉了。当然，枕着这一片美丽的山水，桂林是从来不会缺少美食的。我们一行人这次挑选的都是桂林最富盛名，也有着悠久历史的酒家。
第一站 【澳门酒家】澳门酒家 南航店位于翠竹路上，走进澳门酒家，人气很足的，能够在餐饮市场繁盛几十年，自然是需要真材实料的。店内装修豪华面积达到两千多个平方，据说有六个篮球场那么大，能同时容纳一千个人用餐。
满满的一桌，看中那道菜就归你清盘了。这里有桂林的特色菜田螺酿，漓江虾，还有酒店的特色招牌菜，价格适中分量又足，实在是吃货们的最佳去处。
这是秘制猪手拼，看着油润润的，顿时食欲大振。
就开吃了吗，动筷之前，让相机先行吧。桂林极富盛名的十八酿，就是十八种酿菜。所谓酿，是当地人的一种吃法，就是把调料混合在肉馅里，然后填入在各种蔬菜或者贝壳类，可蒸，可闷，可煮。各种酿菜自由搭配。
天下第一排，光看图以为排骨会很柴，没想到吃起来却让人惊艳，分量也是很充足的，很适合家庭好友的聚餐。
话不多说，直接上图吧，原谅我没有记住美味的大名。
第二站【桂客 膳坊】
游玩东西巷之后，我们转战逍遥楼下的桂客 膳坊。膳坊距象山公园、七星公园、 伏波山公园、两江四湖均10分钟内的步行时间。包厢以桂林市各县名称命名，每个包厢的墙壁上都画着对应县区的名胜旅游区的风景，既美观又富有心意。我们的包厢名叫【阳朔】，最喜欢包厢里的那幅画，阳朔山水甲桂林。
桂客膳坊在传统中融入了现代，酒店里没有传统的点菜单，直接用手机点单，好处是可以亲眼看到我点的菜的模样。
这是一顿有故事的美食大餐。每道端上来的美味佳肴，都能娓娓道来一段好听的故事，要不，美食后面怎么当的起文化二字呢？
在桂林地区除了米粉外，在 恭城县和 灌阳县还有一种以米粉为主重要的饮食，那就是油茶，油茶其实可以算作为一个总的称呼，又可以分几个细的做法：油茶、香菇茶、糖茶、开口茶、炒米茶。每个地方又有不同的做法，但大致相同，都是以茶叶和老姜为主，只是辅料有些不同，作为每天早上的主食，必不可少。
油茶，不小心就喝了两碗，如今的油茶，不仅是老人家的最爱，也慢慢吸引了众多的年轻人。
十八酿由豆腐、田螺、菜包、油豆腐、豆芽、番茄、老蒜、冬瓜等等菜一起酿制而成的，总之，在阳朔人的眼中，什么东西都是可以酿的。
桂林不可错过的漓江虾 漓江虾肉质细嫩纯正，原汁原味，使本菜肴色泽鲜红如宝石，口感稣脆嫩爽，香而不浓，从中尚可品尝出漓江水的清纯与甘甜。
还是继续上图吧，眼见为实，若有机会，自己去品尝更美。

桂林东西巷

桂林逍遥楼，历经多次重修，依然是桂林登高望远，欣赏美景的绝佳去处。始建于唐代的逍遥楼，一度与黄鹤楼、岳阳楼、滕王阁等江南名楼齐名，自古就是桂林文化地标。
“蓬莱凝望眼，隐隐海霞红”，古人吟咏了上千年的一方胜景，终于再现于桂林 山水间，昔日桂林老八景中的訾洲烟雨、东渡春澜、 尧山冬雪尽收眼底。
逍遥楼旁，是桂林的古街东西巷，它见证了桂林城自唐代武德年以后近1400年的历史兴衰。人们从城门走过，仿佛走进了桂林的古老历史。
东西巷历史文化街作为明清时期遗留下的唯一一片历史街巷，它包含正阳街东巷、江南巷、兰井巷等桂林传统街巷。
窄窄的巷子上空挂满灯笼，到了晚上一定是流光溢彩的，本来想晚上再来的，可惜要赶飞机，留待下次吧。
走进文化街，色彩很是鲜艳，不一定要购买东西，也可以在这里拍到有趣的场景，经过改造的东西巷，不再破败，有失必有得，这是一个略为艰难的抉择吧。
作为桂林的一条文化古街，初到桂林的旅行者，还是值得一探的。游玩时间1-3小时，门票0.

七星景区

桂林市区内有很多景区可以去，比如 象鼻山，人名币的背面、连徐霞客都无缘登过的王城等等，都可以步行或者骑行前往。在桂客吃过午餐，还有大半天的时间让我挥霍，不如去 七星岩看看吧。从逍遥楼出来，走过一座桥，就是七星岩了。一路上都是风景，如果有更多的时间在桂林逗留，我会沿着这江水信步而行，一座美丽的山水城市。
选择去七星景区，因为它是桂林市面积最大、历史最悠久、景致最多最集中的综合性景区。历史文化的精深博大在七星汇聚，自然与人文的完美融合在七星展现，堪称桂林山水与文化的缩影、桂林城市绿肺，集桂林山、水、洞、石、庭院、建筑、文物之大成，是桂林山水精华景观的缩影。
公园正门走不多远，就是花桥。每年春夏，花开烂漫，桥畔繁花似锦，桥两岸遍植桃花、翠竹，小桥掩隐于“满溪流水半溪花”之中，花桥因此而得名。
七星公园因山而得名，它荟集了骆驼峰，普陀山、七星岩、 月牙山以及桂海碑林等，是游人的必游之地。
七星景区因七星山而得名。普陀月牙， 一东一西， 相互连属， 分布如太空北斗七星， 山名以此而来。山体面积51.5公顷， 以岩多洞奇著称。
七星岩是七星景区的核心部分，它其实是一个岩洞。七星岩长约1100米， 最宽处54米， 最高处27米，是距今100万年的一段古老的地下河。
这里就是七星岩的入口，要分批而入的。略坑的地方，它竟然还要再买一次门票，所以，你若问我里面壮观吗，我还真不知道，都是门票惹得祸。
雨后的禅寺，还是很清幽的，据说这是桂林最大的寺院，虽然并不信仰佛教，还是适合拍拍照。
“美景若画入廊漫， 携梦仙境醉悠然 ”，来了不想离去的城市，容我再约你一次。
