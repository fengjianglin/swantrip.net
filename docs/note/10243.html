【序】这座熟悉又陌生的城

总觉这座城并不属于我。
锦里
锦里
生于斯，长于斯，三十余载，我还是对她这样陌生。我走过丽江的篝火，走过曼谷的灯红酒绿，走过符拉迪沃斯托克海边的冷风，走过瓦拉纳西的恒河堤岸，走过科莫多岛滚烫的砂石……总是那么快就融入那些陌生的异世界，就像变色龙幻化着迷彩的体鳞，让自己看起来那么的入世。
但是，为什么我行走在这座生我的城市的街头，我总是有种自己不过是个过客的失落。
人说，成都是最包容的城市，她的好客不如重庆一般直接而热烈，也不如东北一般三杯酒下肚就是共生死的兄弟，但是当你觉得做了糗事惴惴不安的时候，身边总是会出现一张绵软的笑脸，一笑而过。
或许，只是我自己不足够勇敢吧，所以我总是害怕着，害怕着得到那些归属感，害怕把自己坚守的道就这样光明地横铺脚下。
得到了，下一步不就是失去吗？
哦，所以，我其实是在怕失去。
这座城，承载我太多的悲欢离合，看惯了我的聚聚散散，每当我撕心裂肺的时候，她依旧那副绵软的笑脸，看着我。
朋友说我是个不入世的人，我很诧异，因为我一直觉得我俗得很足够很彻底。我不过是一个不想长大就干脆一直保持童真的人，不过是一个和所有人一样渴望着能牺牲一切去守护某一个人的人。

★市博 九天开出一成都

是这个时代变化太快，还是我们固守着自己的世界不愿意去改变？
我想，那些坚持怀旧的人，大概在那个曾经熟悉的世界里，总有些不愿意失去，却早已不能再拥有的过去，就像埋在心底的初恋，也像早已化作一帧帧黑白照片模样的童年。

市博物馆
面对历史长河的时候，我觉得自己连一粒沙都不是。
沙不管如何总会随着江水奔流入海，而自己却短暂得像滚滚江水前进时偶尔泛起的一圈涟漪，或许会一环一环激发那些后续的波浪，到底却画了一个圆以后，便消失不见。
所以，我觉得我的第一站应该前往市博，因为爱她，便要先了解她；而要学会敬畏，就该用解读历史的方式去仰视她。

这个时代，这个年岁的我，太浮躁，又太渴求，于是总觉得逛博物馆有些力不从心。
