夏宁葆在南高加索的梅斯蒂亚深度游。

游记 客栈民宿 登山 自助游 自然奇观 雪景 2019/06/19
我们继续在格鲁吉亚进行深度游，这次是到南高加索的梅斯蒂亚去旅游。梅斯蒂亚世界级的缆车风光令人惊艳而出名。我们每人花了38元人民币去看雪山，大家一定觉得奇怪，38元能够看什么？大家跟着老夏去看看：这是一座被雪山包围的小镇，被称为高加索深处的世外桃源，每年都有大批徒步爱好者不远万里来到这里。这里一年四季都可以欣赏斯瓦涅季的壮丽山脉，给人一种强烈的视觉冲击感。
这里的碉楼主要集中在河谷两岸,很多是三五个一组相互呼应,也有独立于山头的,在碉楼集中的地方,数十座碉楼连绵起伏,形成蔚为壮观的碉楼群。这些土黄色的高大石头作品,每一座都经受了百年风雨,棱角分明地巍然屹立。站在碉楼下,会恍惚感觉在历史隧道里迷失了自己。
这里的碉楼最多也最出名,有世界上最集中的古碉群,共100多座,堪称碉楼博物馆,有“千碉之乡”的美誉。碉楼的外型是四角型，碉楼由石块砌成。墙体表面光滑,缝隙紧密,棱角笔直,可见建筑师具有高超的设计功力和工艺水平。
我们在巴统玩了3天以后，预定的奔驰9人商务车前往450公里以外的梅斯蒂亚。早上7：30从巴统的民宿，直接送人到梅斯蒂亚城镇中心的民宿。两头不用走路，虽然我主动多给了司机一些钱。但是对于我们老年人来说，绝对值得。服务态度很好，经常主动停车，然后告诉我们哪些地方风景不错。如果我们自己拖着行李箱，去巴统火车站有17人的依维柯公交长途车每天2班，中途在祖格迪迪停车，上厕所，以及等候上下旅客。总共时间大约7小时，每人25拉里。想乘坐位置舒服一些，只有寻找7人商务车去祖格迪迪。每人20拉里，路程3个小时。再从祖格迪迪寻找商务车去梅斯迪亚，也是每人20拉里，路程又是3小时。那么我从头到底包车，7个人应该是280拉里。司机半路上还买了2瓶水给我喝，很客气。所以我一共给了300拉里，兑换成人民币750元，7个人分摊，每人107元人民币。毕竟人家服务到家。而我们图的是舒服，省心，一步路也不用走。完全与旅行团一样。从巴统的A民宿直接到达梅斯蒂亚的B民宿，下面就是我包的奔驰9人商务车。

突然看见有人骑马过来，不知道大家注意没有：走在前面的大汉，这是马匹的主人。旁边的女人和后面的青年是一对游客。
因为上山下乡的时候我养过马，一般会不会骑马的标准：就是能够驾驭马匹去任意一个地方。不会骑马的人，你走不了200米，马匹就会拐弯逃跑回家或者就地打滚，钻树丛等等。。。所以你看这个女人的缰绳，或者叫襻头是牵在主人手里的，女人的马是跟着主人的马一起走的。由于主人在前面走，后面的小伙子的马才会跟着走。否则马明白，回家是要挨揍的。我告诉大家一个题外话：马是训出来的，需要大声呵斥和鞭打。牛和毛驴是哄出来的，否则不会有牛脾气，驴脾气的说法。

这是我在网上预定的民宿，3间房间，7个人，2天一共250拉里。兑换过来平均每人每天44元人民币，不含早餐。
安排好了住宿，我们立刻到房东家的后山上去玩玩。雪山的下面，有一片片草场。草场上疏疏落落的长着一些松树，地面开满了各种各样的小野花，黄的，红的，蓝的，紫的，像漫天繁星。鸟儿婉转的歌声，清脆的马蹄声，还有叮叮当当的马铃声，非常和谐的交织在一起，象一曲动听的乐曲。

这里的碉楼叫：斯凡塔。这是这个地区原始居民区最显著的标志。该塔由世代生活于此的斯凡人（Svans，指生活在斯瓦涅季地区的格鲁吉亚人）于8世纪为了保护家庭而建造，整个塔完全依靠人力修建，没有任何的机械化设施参与。也因此斯凡塔象征着家庭的实力，实力越强越富有，家庭成员中男性就会越多，斯凡塔就会越高。
斯凡塔一般有4-5层，如今或用于饲养牲畜，或用于储存粮食，古时用于瞭望防御的斯凡塔通常比普通家庭的斯凡塔更高，可达28米左右。严格意义上讲斯凡塔没有地上的直接入口，要通过秘密通道或与斯凡塔连接的房子内部进入，现在部分斯凡塔为了方便旅行者进入参观开通了入口，一般收费2拉里/人。在此推荐一个开放的斯凡塔，名为Khergiani塔。 
这是哈特斯瓦力滑雪场刚刚建造成功的缆车出发站台。缆车分成2段，第一段运行大约10分钟，到达站台下车。然后横着走300米。自动刷卡，进入第二段的站台。继续运行大约15分钟，到达终点站。
这样的来回4段缆车，价格一共每人15拉里。兑换人民币38元，实在是便宜得不得了。下面是缆车刷卡。
原来这里是个滑雪场，冬天开放滑雪场。冬天一天无限量上下是80拉里，夏天一般都是观光客，也可以乘坐一半，无论第一段，还是第二段，单独一段上下是10拉里，二段上上下下是15拉里。
我看了不少游记。2017年过来的人説：当时缆车费用来回只有：3拉里，2018年是10拉里，现在2019年已经涨价到了15拉里。不过对于中国人来说，比较国内的景区缆车价格，呵呵。。。
缆车的生意一般般，反正是游客稀少。缆车比人多得多。
这是第2个缆车车站，换乘以后上更加高的山顶。2个缆车站台距离300米。
坐在缆车上面可以看着近在咫尺的大雪山，感到一阵欣慰。我挑选的日子真好，现在是5月20号，最近白天的温度已经到达28--30度，晚上温度是5--10度。冬天的雪山积雪融化得很快，用不了15天，这些雪山基本上都会变成绿色的森林。因为都是在海拔4000米--5000米之间，最多露出来一个雪山顶。好看的程度就会大减。
滑雪场因令人惊艳的缆车风光而出名，下山缆车正对高加索雪山，给人一种从空中接近雪山的机会。缆车上部站台拥有景观无敌的观景台，附近的Zuruldi餐厅人气很旺，冬天的时候餐厅正中心会燃起熊熊炉火，将屋子里变得暖意融融，看着外面大雪皑皑的雪山，非常惬意。站在观景台上，一年四季都可以欣赏斯瓦涅季的壮丽山脉，给人一种强烈的视觉冲击感。
站在山顶看看四周围，真是千里冰封，万里雪飘。
每年雪季滑雪场会开放，其他时间缆车照常开放。缆车开放时间：每天10:00-17:00 ，17:00-18:00只有下行的缆车运行。
非常壮观的仙境。加上今天天气很好，运气不错。
这是：Zuruldi餐厅。几个外国妞与我打招呼。人家的体质就是好，短衣短裤还打赤脚。我们却是冲锋衣。。。。
整个餐厅，观景台就是这些人。我已经把所有的游客统统拍了进去。只有三三两两的欧美游客，我没有看见其他中国游客和旅游团。
翻过重山，掠过冰川，白云朵朵，云时常缠绕在头顶，这里的蓝天总是如此的“不真实。也许只有亲身体会，方能感知这无以言表的奇妙。四周围的雪山就像波涛一样涌向天际。
远远望去，高耸的山岭间丝丝白云缭绕不散，仿佛一条晶莹耀眼的龙在蓝天下飞舞。我们坐缆车上山，从海拔3000多米的高处往下看，山中人迹罕至，只有一群牛在吃草，悠闲地甩着尾巴。
我们坐着缆车上了雪山，一开始还好，但是到了后来，山上的冰雪越来越多了，树上也结着许多的雪，遥望远处的山顶，白茫茫的一片，真让人觉得来到了人间仙境，据说，那里的雪还是常年不化的呢。

两边是一大片郁郁葱葱的原始森林。一棵棵树，高大挺拔，直入云霄，好一派古木参天的秀丽景色。

这是一家网红饭店。里面，外面统统坐满了人。因为晚上还有表演活动，所以这家饭店人特别多。
在这里碰到2个从中国来的小青年，一男一女正在吃饭。他们在第比利斯的青年旅社认识，一同相约到这里来玩。住宿在青年旅社，上下铺那种房间。每天每人13拉里，兑换成人民币32元。在他们的介绍下，我单独叫了：一份红焖牛肉7拉里，一份蔬菜蘑菇汤7拉里，一份米饭4拉里。一共17拉里，兑换成人民币42元。结果拼命吃，也只吃掉一半，因为分量太多了。由于我们在格鲁吉亚已经17天了，知道格鲁吉亚的规矩。一般大饭店，人多一起吃饭超过50拉里是要加18%税的。还有可能另外收取小费，每人1拉里。所以我们7个人各自分开吃饭结账。他们2人一起吃饭的，没有加18%税，只收取2拉里小费。我因为一个人吃饭，连1拉里小费也没有收取。第2天晚上，我们继续在这里吃饭。我有经验了，只叫了一份红焖牛肉，7拉里。另外1拉里买了一个大馕，掰了一块，吃的很香。
