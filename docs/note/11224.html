秋初季节空空如也莽莽撞撞地一次邂逅—水下孔

游记 古镇 国庆 徒步 摄影 自然奇观 2018/09/30
与一望无极的平畴旷野相比，山带给人怎样的审美？跌宕、曲折、奇险、幽秘…在西北，大山绵延叠嶂，很容易叫人熟视无睹。突然对山动情，是在几年前，爬完泰山之后，每一座山，都是造化摆在人世间的神物。高山仰止，须是在你动心之后才能体会到的。这一次空空如也莽莽撞撞地走进了磐安县水下孔风景区。
对于磐安，对于水下孔景区估计大家都不太熟悉；它只是金华的一个小县城，一个横店的外景拍摄取景地之境地之一。但这里现在以“养生旅游”名义悄然兴起，吸引着许多上海、杭州等城市游客前往避霾，在没来之前还以为这里只是空气清新，不想这里的自然风光也是一绝。两旁高大的山峰对峙，峻拔的高山巍然屹立，奇峰纷呈，怪石错叠。沿路总会碰到小溪和瀑布，水清澈、自然...

交通那些事

前往的这个低调的风景区，你可能需要从义乌转一次大巴，甚至需要赶早班车，尽管如此，但当你到了这里，你会忍不住的想要请几天假期留在这里做几天山里人。
当然了，自驾可能是你最为舒适的方式，车载音响里大声地播着音乐，嘴里哼着各种风格的曲调，一路上走走停停，看看路边风景或是抻抻腿、伸伸腰以解旅途之疲惫。这种想唱就唱，想停就停的恣意也只有自驾旅行中才能体会得到。

住宿和地道美食一起解决了

很多人选择自驾来这里小住几日，相对比这里得农家乐还有很多可以选择的，大多数都集中在乌石村的的周边，这里交通方便，距离游玩的景点也很近。
农家乐的基本设施都很齐全，电视机、空调、数字电视、网络光仟...基本的一些设施都有，但需要注意的是自己带好洗刷用品哦。
在这里选择住宿除了舒适度之外，淳朴实在的老板娘一定要有烧制美味可口农家菜的绝活。因为这样你会省去一大部分的事情。省去找饭店的时间，还不担心饭菜不地道。
来的时候问了几位来过这里的朋友，一致推荐了磐新农家乐，除了地理位置和设施，最重要的是阿姨烧制的一手好菜。所以的食材都是自家种植的，有些还是山里的野味，所以根本不用担心食材的问题。特色菜推荐：土鸡煲，笋干火腿煲，石斑鱼和小溪鱼，还有蕨菜，梅干菜，香菇等等，对了，香菇是这里的特产，可以带点回去哦。
这里的农家乐设有卡拉OK 、棋牌、自动麻将等娱乐设施，晚上回来之后也不用担心无聊，唱k、BBQ都可以搞一波。

关于行程这点事

行程自由一点，根据自己的时间安排，景点之间距离很近，景区都不大，时间可以安排宽松一点。周末两天的时间可以，再多几天你也不会无聊。

鬼斧分天地，神工曲山水——水下孔风景区

水下孔景区属于浙中大峡谷。从入口处望下去，这是一个圆形的巨穴。这里有件很有意思的事情就是与其他的爬山方式不同，在这里进入景区你首先需要走过一个5D玻璃天桥，通过高清户外互动显示屏，以人屏互动为根基，加上5D震撼视频源，让你身临其境，屏里的画面，随你而动，让你看着玻璃爆裂，闭眼还能听到碎裂的声音，心理承受力差的不要轻易尝试。通过玻璃天桥你可以选择乘坐华东最高的观光电梯下行游览，顺便来一场视觉享受，远望水下孔景区四周繁茂的森林、俯瞰壮丽的瀑布、欣赏美丽的山水风光，还可以挑战一下自己；
除了电梯，你也可以选择徒步前行最后乘坐观光电梯回到出口，还有不同的是这个景区是先下山然后在上山，徒步爬山可以一边前行一边欣赏满眼的绿色，相对比大城市单一的钢筋石板和灰蒙蒙的雾霾，这满眼的绿色和新鲜的空气似乎显得弥足珍贵。
从景区入口处下去，幽谷深深，沿着山坡修建的屈折小道进入峡底，好在外侧的一旁都有扶栏围住，但仍有胆小者不敢俯视。整个景区是冰川期后期冰川融化而成，历经地质变迁的峡谷，它形成了独特的地貌。澄溪之水忽从高四十余米的断崖跌落穴底，形成七级首尾相接，各异其趣的瀑布，飘者如雪，挂者如帘，断者如索，穴中雾气弥漫，轰声灌耳，撼人心脾，让人叹为观止。整个水下孔景点以“七瀑一湖”为架构，主要景点有罗汉瀑、飞龙瀑、姐妹瀑、五星潭等。
可能是由于好久没有爬山，二个半小时的游览过程显得略微有些吃力，但看到清幽的水下孔，周围高大的山峰峙，峻拔的高山巍然屹立，奇峰纷呈，怪石错叠。山路并没有外表看起来那么难走，平缓的坡度，可以尽情的欣赏沿途的风光美景。走在栈道上，一阵暖风徐徐吹来，沿路的绿色，好像久违的朋友。沿路总会碰到小溪和瀑布，溪水清澈自然，这潺潺的流水，不知道哪里才是它的源头...
飞瀑嗽玉，青山嵌翠，进入谷底，向上望悬崖绝壁，向下望，峡谷之间，巨石累累，中间夹着溪流穿过，有潺潺流水，也有深潭，如五星潭，整个潭有几块巨石团团围成，这些巨石组成呈五角星形状。据导游介绍，潭虽不大，但潭深可达数米，故在旁边游览，更要注意安全，防止失足落水。到五星潭后需要原路返回，从对面峡谷往上沿着山栈道向上行走。
这样从山上下到谷底，又打弯从谷底继续前行，边走边领略怪石、山水、飞瀑等风景，呼吸着深山中的新鲜空气，因为不是周末也是阴雨天，所有游客团队相对不多，我想幸亏这次团队不多，如果团队多的话，那么游览的人绵延不绝，熙熙攘攘，对大自然的幽谷宁静是一种破坏，绝非是游览的最佳时机。
离出口不远有一个观瀑亭，可以在这里坐下休息，见一彩色剧照悬挂在亭旁，原来这里是台湾电视连续剧《完美新娘》的剧照，曾在此拍摄取景。除此之外《刁蛮俏御医》《新还珠格格》《大唐女将樊梨花》《倾城绝恋》都有在这里取景拍摄，其实这里也是横店外景取景地之一。
不知道走了多久，渐渐听到水撞击岩石的声音，循声而去，声音越来越大，终于我看到了这里最壮观的瀑布——罗汉瀑。自50米高处崖壁凌空跃下，水帘银珠飞溅，峰鸣谷应，十分壮观。突然想起李白的诗“飞流直下三千尺，疑是银河落九天。”水顺着山势而下，山崖边还有顽强生长的小草、小树，那种壮观是无法言表的。
大概这个季节才是看瀑布最好的时节，降雨多，水量大！在这里奔腾的瀑布一泻而下，激起的水汽笼罩山谷，一时间暑气全消！深山碧水流，夹岸树林苍翠，碧潭飞瀑，山清水秀迭串而成，错落有致。如果天气好的话遇到彩虹也是分分钟钟的事情。
天空下起了蒙蒙细雨，于是我们选择了乘坐电梯回到出口。电梯很平稳，有一种是透明玻璃的可以看到整个水下孔景区的山水、峡谷、飞瀑、洞泉、树木、花草、还有历经冰河期的巨石散落在峡谷中，巨石形态各异，像龟、像虎、像鳄...
站在电梯的平台看着自己走过的路，才发现，原来它们是那么威严，那么高大。在这奇伟的大山之间，每个人都如蚂蚁一般微不足道，仿佛尘埃里的一粒沙，在山峰之间飞舞，路过光怪陆离的大石头，走过瀑布，在水下孔自由飞舞，烦恼、困苦，在这里一切不复存在。

火山台地，空中乡村——乌石村

来到水下孔景区，怎么可能错过乌石村，一个用石头建起来的村子。走进乌石老村，最大的亮点就是建筑，全部采用的全是石头：石路、石屋、石墙、石碾、石凳、石槽…一座典型的石头村。而这些石头全是千年石化的黑色火山岩石，据分析这种黑色火山石，有多种微量元素，对人体健康有益。整个村庄形如“燕窝”，是保存完整的以中国古代风水理论建造的村庄。
一些古建筑悬挂着灯笼，与这里的建筑和氛围格外的和谐。走到村子的东北角上，绕过一个老祠堂，在这个位置上往下望是深谷，往对面望有一座山，山上是盘龙梯田，梯田一圈又一圈盘旋而上，蔚为壮观。
整个乌石村建在地处560多米的高山台地上，，素有“空中乡村”的美誉。在这里可以看看去古寺院、三合院、四合院...这些地方去看看，在这里不管你怎么走，怎么拐，你总会发现一些惊喜，沿着长长的石路拾阶而上走进村，一下子被包围在石头世界里，令人恍然有时光倒流之感。
沉默的石头见证着古村落曾经的辉煌与变迁，讲述着这里千百年的传奇故事。这里的石头是有灵气的，它们被风吹过，被雨淋过，被敲打过，被搬动过……然而，每一道时光留下的痕迹，它们都默默地记录着。它们就是一部多姿多彩、回味独特的文化史。
傍晚时分炊烟升起，只是闻着炊烟的味道，就可以想象到老人家煮的白米饭，那么老人家今天会吃什么菜呢？夜晚十分，抬头便是久违了的星空，凉风习习，瞬间觉得自己回到了小时候。
温馨提示：
这里的温度相对比城市温度要低几度，毕竟是在山里，所以在前来之前最好是准备见厚点衣服哦。

The end

在回程路上，不由神情迷离，想像着这幅令人心醉的山水画卷，对大自然美的赞叹自此在心间长久存留。如果说，每个景区都有自己的性格的话，那么水下孔一定是低调内敛的性格。只有走近他，才能真正的了解到他的魅力所在。
