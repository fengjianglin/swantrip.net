【关于交通】
1. 从桂林到阳朔主要有自驾游、坐高铁和坐客运车。其中本地人建议自驾游，外地游客建议坐高铁，高铁到阳朔站约30分钟，但是阳朔站离阳朔市区有点远，也要坐客运车大概1个小时才能到，没有桂林直达阳朔市区的车。坐客运车大概1个小时左右直达阳朔，票价30元。
2. 阳朔客运站有直达桂林两江机场的车次，时间在每天的7点、9点、11点半、13点、14点30、16点、17点30、19点，票价60元，用时90分钟，建议要提前规划好路线和时间。
3. 坐竹筏游漓江一般的路线是杨堤码头——九马画山——兴坪古镇，需要在阳朔客运车坐客车到杨堤码头，车票15元，漂流到九马画山后乘坐观光车到兴坪古镇，兴坪古镇有直接回阳朔的客车，车票10元。
4. 十里画廊是不允许外来车辆进入的，游玩的最好的方式就是租一辆电动车或者单车，街上有很多租车点。

【关于美食】
阳朔美食有很多的，也很有自己的特色，像啤酒鱼、桂林米粉大街小巷全都是！
1. 啤酒鱼
啤酒鱼这道菜是阳朔有名的特色菜，用料极为讲究。鱼是漓江清水鱼，啤酒和鱼肉的香味保证可以让你胃口大开，我们是在西街刘师傅啤酒鱼吃的，两个人买了一个129的套餐。
2. 桂林米粉
在桂林，当地人每天的早餐几乎就是一碗热乎的桂林米粉，这一在阳朔发挥得更加淋漓尽致。选择一家小馆落座，矮凳木桌，一碗米粉配一根油条或一颗卤蛋，会吃的再加瓶豆浆，当地体验从这里开始。吃粉一定要找小巷子里的吃才正宗，价格也实惠，我吃了一碗卤菜粉加一颗卤蛋才5.5元。
3. 素炒漓江虾
素炒漓江虾是桂林典型的风味菜，以桂林漓江里的河虾为原料加入少许桂林三花酒爆炒而成。菜肴色泽鲜红如宝石，口感酥脆嫩爽，香而不浓，从中尚可品尝出漓江水的清纯与甘甜。这道菜在我们吃的啤酒鱼套餐里，个人觉得一般般。

桂林阳朔，钟灵毓秀的山水之地

阳朔县，位于广西省东北部，县城距桂林市区65公里，属于亚热带季风气候，体感较为舒适。全县有汉族、壮族、瑶族、回族等11个民族，同时也有很多外国人来此旅游居住，主要得益于秀丽的自然风光。
俗话说：桂林山水甲天下，阳朔风光甲桂林。这足以表明阳朔的地位，也为我们展示了一个山明水秀的桂林。
当你踏上阳朔的那一刻，你就能感受到这是一个怎样的人间仙境。漓江、聚龙潭、蝴蝶谷等众多景区让你眼花缭乱，啤酒鱼、叫花鸡、竹筒饭让你垂涎欲滴。这是一个风景胜地，一个你非来不可的旅游度假胜地。

Day1:桂林站—阳朔汽车站—桂林千古情景区

第一天我们是从桂林站打车达到桂林客运站，然后再坐车去往阳朔，大概花了一个小时左右就到达了阳朔客运站。
下车以后我们就在旁边的小餐馆吃了个午饭，点的是广西特色的螺蛳粉，加上酸豆角、酸笋、腐竹之后，味道简直棒极了。一口吃下去，嘴里弥漫着的是螺蛳粉带来的酸、辣、鲜、烫，刺激着味蕾。
吃饱休息过后，我们便出发前往我们的第一个目的地——桂林千古情景区。
桂林千古情景区位于桂林市阳朔县千古情大道，于阳朔县城次新区域，西街以西，从城区打车大概10分钟左右的路程，交通较为方便。
从正门进去，第一眼看到的便是恢弘神圣的巨像，屹立在阳朔这片被大自然恩赐的地方，令人心生敬意。
与其他千古情景区相比，桂林千古情独特的地方就在于它的自然风光及其优美，尤其是高耸的山峰，平地拔起，千姿百态，有奇、秀、险的突出特点。
进入景区以后，可以看到各具特色的民族古建筑，给人一种古朴大气的感觉。小吃街也有各式各样的美食，方便游客品尝阳朔具有地方特色的美食，比如好吃的桂林米粉和“桂林三宝”之一的三花酒。
来千古情景区是一定要看千古情演出的，号称一生必看的演出，可以给观众带来唯美的视觉盛。宴。
“千古情”是“中国演艺第一股”宋城演艺打造的旅游演艺品牌，是由黄巧灵导演，拥有《宋城千古情》、《三亚千古情》等千古情系列，展示了城市独特的文化底蕴和历史内涵。可以这样说，千古情已经成为了中国演艺的一大品牌，拥有了一大批热爱传统文化的忠实粉丝，在世界各地享有很高的知名度。
大型歌舞《桂林千古情》的演出时间在每天的12点、15点、19点半，我们观看的是下午15点的场次。千古情大剧院占地面积很大看千古情演出的人很多，观众席密密麻麻地坐了好多人，可见人气之高。
我们进去坐好位置以后，中间的大屏幕便为我们讲述了千古情的历史渊源和简介，为我们普及了相关的知识。《桂林千古情》是黄巧灵导演运用全新的艺术表现方式，别开生面地展示了民族优秀传统文化，彰显了宋城品质、国际水平。
全剧分为《远古的呼唤》、《大地飞歌》、《米粉传情》、《靖江王府》、《刘三姐》等幕，从古至今再现了一段三生三世的桂林绝恋。运用现代的高科技技术，千古情演出给我们带来了一种穿越时空的新鲜感。
通过演员精湛的演技和现场的精心布置，给我们营造了一种扑面而来的历史感，仿佛把我们拉回那个时代，去感受当时的人们的生活方式。
看完演出以后，我们从剧院出来，沿着街道还能观看其他类型的演出。水上飞人项目让我们近距离的感受了一把水的温度，顿时引起了现场的欢呼，也调起了大家的激情活力。
莫土司招婿为我们展现了那个时代的招亲方式，演员打扮好以后抛绣球，底下的游客通过接绣球的方式，融入到这个节目之中。接中绣球者换上新郎的服饰，体验了一把成亲的滋味，其乐无穷。
在风情街表演的《民族快闪秀》和《秦兵巡城》也是相当精彩，值得一提的是歌舞表演，穿着民族风采服装的演员们尽情的展示丰富的舞蹈动作，让人拍手称道。突然，混杂在人群中的保安也加入其中，跳起了时尚的街舞，惊讶了在场群众。后来才发现，原来他们也是乔装打扮的演员，可谓是啧啧称奇。
在6点半的时候，我们还在大剧院里观看了一场由现场招募而来的演员配合的演出——《大地震》。大家积极踊跃的参与进来，通过这种形式的互动，生动地表达了大地震以后的现状，也让表演观众近距离的感受了地震，感触很深。
看完演出以后，正值晚饭时间，我们便去了阳朔西街吃当地富有特色的啤酒鱼。选中一家评价较高的餐馆以后，我们点了一个套餐，有啤酒鱼、竹编漓江虾、糍粑、上汤生菜等，啤酒鱼味道很正宗，很鲜嫩，吃起来的口感很棒。漓江虾也很好吃，值得称道。
吃完饭以后，我们定好的民宿老板便来了接我们去住的地方——竹窗溪语，一个充满禅意的宝地，一个与自然风光完美融合的胜地，是居住与观赏的绝佳地方。

Day2:竹窗溪语禅艺酒店—十里画廊—遇龙河

竹窗溪语位于十里画廊景区附近，我们在车上一路还能看到美丽的风光，过足了眼瘾。
民宿在鸡窝渡村122号，我们到了以后，第一眼看到的是泳池，随即衍生出了想要游泳的念头，但是因为气温的缘故，便打消了这个念头。
然后远望过去就是遇龙河畔的大好风光，秀美的山峰尽收眼底，充满了诗情画意。整个酒店的布局很合理，采光条件好，还带有服装试穿美拍，十分人性化。
酒店的管家也很贴心，为我们设计了最优的游玩线路，还提前问我们到达的时间，同时提供了接送服务。名宿还提供了租车服务，有电动车和自行车，方便了游客的游玩。
进入自己的房间以后，里面的布置更加让人满意，面积很大很整洁。打开天窗，还能看到远处的自然风光，是一个欣赏日出日落的好地方。房间提供的洗漱物品还带有名宿的风格特色，方便携带，又有收藏价值。
房间还能品茶，闲暇的时候坐在窗边，和好友欣赏外面的美好景色也是一个不错的选择，让人心旷神怡，心情大好。
第二天清晨下了小雨，烟雨朦胧，一派自然风光的气息，我赶紧拿着相机去周边的农田、小池塘里拍了一些照片，成片效果还是不错的~
之后我们租了辆电动车，便开始了我们期待的游玩之旅。按照管家的意见，我们先是沿着十里画廊，一路欣赏两边的美丽景色。秀丽风景尽收眼底，不愧是十里画廊。
骑车大概10分钟，我们就到了拍摄的最佳地方——工农桥，很多的游客都驻足在此拍照。工农桥位于遇龙河、金宝河、田家河三河交汇处，是遇龙河漂流的终点。桥下可以看到很多的竹筏，游客惬意地躺着，尽情地看着河两岸的风光，桥上的人则在观看起伏的群山，山水相映，风景很好。
​拍完照打卡以后，我们继续骑着小电驴前进，不久就到了蝴蝶泉，从外面看就像一只雕刻在山峰上的巨大蝴蝶。蝴蝶泉位于阳朔月亮山风景区“十里画廊”的精华旅游地段，有阳朔唯一的原始吊桥，景区由蝶洞、蝶桥、蝶山、梁祝表演、兰蜂园、蝶缘、蝶厅七大部分组成，是一个鸟瞰田园风光的最佳观赏台。
再往前走去，我们就到了图腾古道景区，停好车以后，沿路走去，看到的是古老的图腾柱、石磨、石器、弓弩等。这个景区为我们展示了原始人的生活方式和场景，蕴含了深刻的文化内涵。
中午时分，我们就回到工农桥前面不远处的小摊铺吃午饭，点了竹筒饭、红薯、漓江鱼等，味道很不错，我们吃的津津有味，很满足。
然后饭后还在附近买了草莓，可以自己采摘，草莓大棚就在不远处，很方便。草莓长圆锥形，大如鸡蛋，果肉软多汁、味甜，吃起来实心味美，水嫩多汁。
酒足饭饱以后我们踏上了新的旅途，以工农桥为起点，向前方进军。不多远就到了聚龙潭，一个由黑岩、水岩组成，因岩外奇峰竞秀宛如神龙腾云、岩内乳石多姿巧似游龙戏水而得名的国家4A级旅游景区。
聚龙潭景区最别致的地方就在于它得天独厚的钟乳石溶洞，巧夺天工，造化钟神秀。经历亿万年的化学、物理反应而形成，真所谓千姿百态。溶洞内较著名的钟乳石景观有“杨贵妃仙浴”、“海上明月”、“水晶龙宫”等。
游玩过聚龙潭以后，我们马不停蹄，继续朝着下一个目的地——金水岩前进。几百米以后，我们成功抵达金水岩，它是阳朔最大、最有特色、最令人开心刺激而且是世界闻名的地下河水晶岩洞。
金水岩景区位于阳朔月亮山景区历村，由旱洞区、地下河水晶区、泥浴区、岩洞温泉区组成。我们先是进去观看了精彩的民族演出，然后沿着观景天桥入洞，进入旱洞区，能看到“海底龙宫”、“猴王宝座”、“皇宫御伞”等天然形成的原始溶洞。
我们最关注的就是泥浴区和温泉区这两项，怀揣着好奇之心进去，只不过因为没有备好泳衣，只能放弃。看到外国游客带着小孩畅快地泡在温泉浴里，我们好生羡慕。
很有意思的是，溶洞里还有卖夜明珠卖酒的，还有专门的摄影师拍照的，只不过洗出来要收取洗印费。此举可谓是因地制宜，很有商业头脑了。

Day3:竹筏游漓江——兴坪古镇——阳朔西街

都说桂林山水甲天下，既然见识了秀美的山峰，那肯定也得去看看桂林的水了。
所以第三天我们的目的就是乘坐竹筏游漓江，去近距离接触这让人流连忘返的漓江水。
我们先是从阳朔客运站坐车去杨堤，中间大概花了一个半小时到达。杨堤景区北起杨堤官岩村，接兴坪景区，约18公里江段，是漓江的黄金水段。
买好票以后，我们和另外两位游客一起乘筏漂游。穿好救生衣以后，在工作人员的安排和指挥下便开始了我们的漓江之行。
坐在竹筏上，感受着两岸的自然风光，感觉到整个人都心情舒适了很多，有一种天朗气清的美妙感受。竹筏是电动的，两边还时不时地能看到观光游览船进过，可见游客数量之多。
漓江全长164公里，有山青、水秀、洞奇、石美“四胜”之誉，从桂林至阳朔的83公里漓江河段，是漓江精华，还有“深潭、险滩、流泉、飞瀑”的佳景，是岩溶地形发育典型、丰富和集中地带，集中了桂林山水的精华，令人有“舟行碧波上，人在画中游”之感。坐在竹筏上，遇到水流湍急的地方，还会有倾斜的感觉，就很惊险刺激。
大概45分钟左右，我们就到了兴坪古镇。对于这个地方唯一印象最深的地方就是，这里是20元人民币背面图像的取景地。走进里面，才会发现这里不仅有壮美的自然风光，还有丰富的人文底蕴。
兴坪镇位于阳朔东北部，距县城25公里，依山傍水，风景荟萃，粉墙乌瓦石板小巷，是漓江沿岸最美丽的古镇。
兴坪古镇的九马画山、螺丝山、莲花岩代表的是大自然赐予的风景胜地，能给人带来很好的享受；古街、渔村、古村落建筑群等历史古迹则是人文底蕴的象征。
我们在杨堤买的竹筏门票可以凭票乘坐观光车，观光车的司机为我们讲述了兴坪古镇的历史，也从历史的角度给我们讲述此地的名人逸事，讲的很有意思。
观光车只能把我们带到桥头，然后我们就得靠自己走了。我们沿着原路走了大概10分钟，就回到20元人民币的取景地，大家纷纷拿出20元人民币进行对比，发现果然一模一样。还有很多人纷纷合影留念，以此记录下这一神圣的时刻。
走近兴坪古镇，一股夹杂着历史的气息扑面而来，各类砖瓦结构的古建筑大部分保存完好。古街的东南侧狮子旦为熙平县城遗址，城墙轮廓尚清，随处可见古砖瓦陶瓷残片。
兴坪古镇是一座有一千多年历史的古镇。离阳朔约十公里。周围群峰竞秀，有“三岩、五井、十二山”等名胜。兴坪古镇现尚有众多的文物古迹，主要集中在兴坪古街及离镇约2公里远的渔村。
这里的古桥、古渡、古亭、古戏台、古庙、古寨、古树和古村落建筑群，比较完事地保持了原有的历史环境风貌，身临其境，仍可领略“老街长长，古巷深深”的意趣。
古街有汽车站，我们玩完以后就坐兴坪——阳朔的直达车回阳朔了，期间花了半小时左右，比来的时候快了很多。
回到阳朔以后，我们就打车去了最后一个想去的地方——阳朔西街，一个阳朔重要的旅游景点，来了自然不能错过的好地方。
阳朔西街位于阳朔古镇中心，宽约8米，长近800米，是县城内最古老的街道，有1400多年的历史。
晚上的西街依旧繁华，人山人海，挤都很难挤进去，可见游客对它的重视程度。西街分布着很多的历史遗迹，也有很多阳朔本地的传统美食，很值得品尝一下。这里也是外国游客聚集地，街上有很多西餐厅，不过沐沐是不爱吃西餐的，还是喜欢次螺蛳粉！

DAY4:阳朔西街——桂林两江机场

在阳朔停留的最后一天，下午的飞机回太原了，趁着上午的空闲时间决定再去阳朔西街逛一逛，顺便再买一些特产带给同学、舍友们吃。白天的阳朔不似夜晚那样的喧闹，但是游客仍然不少，古色古香的街道里有行走的游客、有卖特产的小铺、有卖传统工艺品的小铺。
我们买了一些绿豆糕、芋头条、桂花糕、香芋糕等，就回去客运站乘坐大巴去机场了。我们走的那天下起了大雨，兴许是不舍得我们走吧。
这是我们第一次来到桂林，真实感受到了“桂林山水甲天下，阳朔山水甲桂林”的魅力，期待下一次能够再次去到桂林~
