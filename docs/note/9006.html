走读西安—穿越周秦汉唐，体验文物修复，破解古都密码

游记 亲子 小众 摄影 美食 2019/04/18

前言

西安，作为世界四大古都之一，历史上曾多次被作为都城。先后有西周、秦、西汉、新莽、东汉、西晋、前赵、前秦、后秦、西魏、北周、隋、唐13个王朝在此建都，因此，西安所沉淀的历史文化是难以被代替的。
这次行程你将会体验到：
钻木取火、原始房屋搭建、原始植物印染、观兵马俑、制作兵马俑排兵布阵
寻盛唐“黑科技”、仿古人“雁塔诗会”、周原考古、汉阳陵修复文物，在文博界专业老师带领下破解古都密码........
丰富多彩的游戏及场景设计，专业的老师讲解和服务，严格控制的团队规模，让历史变得生动有趣，让旅行变得回味无穷。
此次活动将沿着 原始社会到 盛唐的时间轴，针对亲子家庭不仅邀请了文博专家，通过生动有趣的讲解把历史呈现给大家，还设计了“ 原始人的一天”、 “周原考古”、 “寻秦记"、“ 又见大唐”、 “汉阳陵文物修复”、“ 汉字历史小课堂” 、“雁塔题名”、“ 舌尖上的陕西”等活动带领大家全面地了解华夏文明。
我们还特别设计了学习小册子，帮助大家更好地参观和回顾！

原始人的一天——半坡博物馆

在半坡时代的母系氏族，人们只知其母，不知其父。过着日出而作，日落而息的生活。
那个时代对我们而言，遥远得不可想象、无法触摸。
可是在半坡博物馆，让我们有机会靠近它，了解它，甚至体验它！
半坡遗址揭示了距今大约6000多年前的一处典型的新石器时代仰韶文化母系氏族聚落的社会组织、生产生活、经济形态、婚姻状况、风俗习惯、文化艺术等丰富的文化内涵。
钻木取火、原始房屋搭建、原始服装秀、观摩体验尖底瓶打水、原始植物印染，这些原始项目，可能只有在这里才能体验到了。

周原考古——周原博物馆

周原是周文化的发源地和灭商前周人的聚居地，素有“青铜之乡”的美誉 。
参观周原遗址，在周原考古学家的带领下，带上一把洛阳铲，模拟考古发掘、清理工作。
不论是看《鬼吹灯》或者是《盗墓笔记》，或者其他盗墓相关的小说，一定会提到的就是洛阳铲，这铲子到底怎么用？为什么一定需要它？去找到这些问题的答案吧。
既是参观了青铜器的故乡，那对金文就需要有一定程度的了解。因为金文是青铜器上的文字。
这次行程让专家带你一起解读金文密码，书写长卷金文。

寻秦记——兵马俑

俑坑中的兵俑千人千面，装束、神态各异，或威武雄壮、或轻松惬意，或整齐列队……有初上战场的青年，也有久经沙场的老兵……
何不亲手制作一个属于自己的秦佣士兵？
做一个秦俑需要几个步骤？做完就知道咯。
然后根据他们的官衔排兵布阵，率领你的千军万马！
跟着团队，不必担心被骗子带去假的兵马俑啦。

大汉风采——地点：汉阳陵

西风残照、汉家陵阙，别以为阳陵和兵马俑一样，它可是非常罕见的能下墓葬坑参观的皇陵。而且这里有东方的蒙娜丽莎和维纳斯！
兵马俑的队伍中有一部分是女子，大多面目清秀，身材匀称，但也有一些颧骨突起，面貌奇异，去观察观察，是不是这样？
汉阳陵出土的汉俑只有真人的三分之一大小，大约60cm高， 赤身裸体且没有双臂。研究其原因，这些陶俑在刚刚完工时其实都身着各色美丽的服饰，胳膊是木制的，插入陶俑肩部的圆孔，以便木胳膊可以灵活转动。
但经过千年的风霜，衣服与木胳膊都已腐朽，只剩下了裸露而残缺的身躯。
修复就成了一个巨大的任务，大家来变身成为文物修复师，来还原它们的故事。

又见盛唐——历史博物馆、大雁塔广场

大唐是开放的、充满活力和生机的，历史是鲜活和有趣的，而且处处是“黑科技”
杨贵妃同款的香包，无论球囊如何转动，香料都不会洒出；
一只小小的虎符，却能调动千军万马，秦军锐士全部都要听从它的调遣。
长安自古繁华，大雁塔广场，寻觅西游记的踪迹，小雁塔仿古人来一场雁塔题名。
再来一场古人的游戏，和现在的哪一项运动一样？

汉字小课堂——地点：碑林博物馆

碑林博物馆陈列有从汉到清的各代碑石、墓志共一千多块。这里碑石如林，因此被称为碑林。
你能发现碑林里中国最有名的“错别字”吗？
卫夫人是怎样教王羲之练书法的？肚痛帖、争座位帖.....
原来大师也这么任性！
观摩拓碑全过程，感悟文化传承。

穿越古城——古城墙

西安城墙，是古都西安的标志性景观。
全长约14公里，是在唐代皇城的基础上建成的。
登上城墙，俯瞰西安城，途经太极宫、大明宫.......每一个地方都诉说着城市的过去。
学着用古老的乐器—埙，吹奏时代的乐章。

美食是最深的留恋
