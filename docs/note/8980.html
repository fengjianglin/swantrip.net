没去益田西街，不算到过阳朔！

桂林山水甲天下，阳朔山水甲桂林。
到桂林旅行度假，当然不能不到阳朔，这里集结了全桂林最精华的山水自然风光，还有最经典的美食。
阳朔益田西街和西街紧密相连，是阳朔的中心地标之一。西街遍布酒吧，卖的是各种市井杂货和美食，益田西街更专注主题文化的展示，小资风情更加浓郁。
站在湖边，清澈的湖水将青山、蓝天、白云和建筑物倒映其中，画面美不胜收。
因为山色的衬托，益田西街每一个角落都呈现出如画美感。
在益田西街行走，犹如穿行于古老与现代之间的时空隧道，这里既有独具创意的现代化楼房，也有传统中式大楼，既有平坦的水泥地板路，也有一步一段故事的石板路。不同的建筑在这里交相辉映，但不显突兀。
坡屋顶、小青瓦、吊阳台、白粉墙……桂林的特色建筑在这里一一呈现。山、水、田园、岩、竹五大院落风格建筑群是益田西街的主体建筑内涵所在。
益田西街的设计兼顾视觉审美又贴近现代潮流，其空间开阔而不局促，商业门类包罗万象。像我这类游客，最不喜欢的是一直赶路的行程，益田西街提供各种休闲场所，非常适合我们。
在外边或益田西街小逛一会，累了找间咖啡馆、豆浆馆，或某间特色饮料店坐下来喝杯咖啡、豆浆，养一会神是最惬意的选择。
出门在外，慢下来，才能回味风景之美，才能享受生活。
益田西街总面积5万平方米，包括泰国精品酒店和文化街区两部分，其中文化街区又分多条街道。
街区的商业门店包括美食店、精品店、工艺店、婚礼旅拍店、咖啡店、超市等等，如果喜欢逛街的话，这里至少可以呆两个小时到半天时间。
不管怎样，作为阳朔的文化地标，游客到阳朔如果不到益田西街，行程是不完整的。
益田西街有很多奇特的雕塑和形象可爱的大公仔，这些公仔都是绝佳的拍照配角，很多游客选择和这些公仔合影留念。
沿着台阶往二楼方向走，可以看到萌状可掬的大熊猫和古风浓郁的彩色伞阵，楼梯上描绘着帅气的牛郎和美丽的织女，这个角落为益田西街增添了浪漫文艺气息，特别受到女性游客和情侣的欢迎。
站在伞阵下拍一张文艺照片，或者坐在阶梯上相视而笑，属于自己的小美好定格成永恒。
惊喜地在街区内发现冰泉豆浆馆，从小就喝它，特别有感觉。
每一座雕塑都有自己的艺术特色，他们或站、或卧，抒发创作者的艺术情怀。对于游客来说，这些雕塑可以引发无限联想，哪怕只是在旁边拍一张打卡照，也是不虚此行。
每一个物件都足够特别。在一些区域还有近代生活风情展示，这些生活场景很多上了年纪的游客比较熟悉，年轻人可能第一次见。通过蜡像重现旧时生活场景，也是一种文化传承。
随意走进街边某间门店总是会有意外的视觉收获。
古风的服饰、萌态可掬的精品公仔、精致的茶壶、民族风情的包包、会奏乐的小盒子……
游览的时候巧遇桂林首届三月三全民网红盛典，不时在街头上碰见打扮活泼的美少女，她们都是新晋的网红。
到了晚上，益田西街更加热闹，观众里三层外三层围满了中央舞台，美丽的网红小姐姐轮番登场，将现场气氛燃爆。我在一楼找不到位置观看，只能跑到二楼远距离欣赏。
晚会尾声进行了网红实训基地挂牌仪式，当地想通过互联网＋的模式打造新网红，有志往网红方向发展的小姐姐机会来了。
据说益田西街每个周末都有广西特色舞蹈、戏曲等表演，个人尤其喜欢戏曲，下次找时间来看。

最爱的刘姐啤酒鱼

每次去桂林，都要去寻觅美食。
阳朔哪个地方美食最多？当然是县城的西街啦。这里因老外众多而出名，如今已经成为大小品牌美食餐厅“战斗”的地方，因为竞争较大，很多餐厅都认真经营，出品的美食令人回味，其中包括我每次到阳朔都要吃的刘姐啤酒鱼。
我们去的是刘姐啤酒鱼西街总店，一如既往地人山人海，露天餐区和室内就餐区都坐满了食客，其中有大多数人是慕名前来，也有不少像我这样的回头客。
每次来刘姐，啤酒鱼是必点的，它不单是刘姐的招牌菜，也是阳朔最有名的特色菜。
桌上烧制是刘姐啤酒鱼最大的特色，工作人员将现杀的剑骨鱼摆在一个碟子里，开炉热锅到合适时间慢慢将鱼放进锅里。
啤酒鱼光配料就有十几种，最独特之处在于采用非遗酱料调制出独特酱汁，很好地保证了啤酒鱼的美味和特色。
剑骨鱼和配料安放完毕，接下来的事情就是安心等吃。工作人员贴心地在砂锅盖上放置了一个沙漏，来回倒两次，然后开盖，鱼便能吃了。
还没动筷子，啤酒鱼的香气便扑鼻而来，夹一片鱼肉放在嘴里，香、嫩、滑，还有一点点脆，让人回味无穷，以至于第二天中午，我们又去重新点了一次。
七仙峰茶叶和漓江虾混炒，既有茶叶的甘醇香味，又有河虾的香味，入口酥脆，令人回味。
牛腩船上糕：中间是牛腩，旁边是船上糕，里面是芋头和蒜叶汁、糯米，油炸后外酥内软。
山海一家亲：内有芋头、腐竹、豆腐、木耳、花甲、蒜蓉等，这个菜微辣，各种食材的鲜味完全融合在一起，特别喜欢里面的豆腐和木耳，尤其是豆腐，很滑很嫩。
田螺酿，桂林地标美食，也是阳朔十八酿之一，这些田螺最大的特点是个头大，个别田螺的尺寸都有网球那么大了。田螺酿最大的特色是里面的肉并不全是螺肉，而是先把螺肉掏出来，混合了剁碎的猪肉、香菜及其他调味料、酱汁一起烧制。这道菜也是我们每次必点的名菜。
走地咕咕鸡：选用刘姐啤酒鱼集团自家养殖基地－青江渡生态农庄养殖的走地鸡，这些走地鸡散养在山谷里，喂的是玉米，养足一年才会对外售卖。
走地咕咕鸡用高压焖制而成，开锅后外观呈金黄色，肉质Q弹鲜嫩，鸡肉的原生甜味保留其中，很棒！

遇见安安感受夜之魅

晚餐过后，我们到刘姐啤酒鱼对面的遇见安安酒吧嗨皮。
这是一间清吧，每天都有歌手驻唱，歌唱类型以民谣及流行音乐为主，现场观众还可以与歌手愉快互动，也可以点歌。值得一提的是驻唱的歌手很帅气。
酒吧的氛围很好，灯光不太暗也不太亮，适合三两好友静静喝酒，静静听歌，兴起了搞点小娱乐也是不错的选择。
特别喜欢酒吧工作人员调制的美酒，一边品味美酒，一边倾听美妙的歌声。夜，如此美妙！

漓江边的休闲下午茶时光

西街很热闹，距离西街不远处的雅堤西餐厅Cafe却独享一方清宁。
雅堤西餐厅Cafe位于漓江龙头山迎宾画廊出口，周围竹林绿树掩映，意境深幽。这里是阳朔唯一一家江边餐厅，提供咖啡、茶饮、酒水，还有各式西餐。
吹着清风，一边欣赏漓江喀斯特美景，一边品味咖啡、美酒、美食，感觉无限写意。
餐厅出品的咖啡很棒，甜品也很不错，据说主厨是一位女性，一生颇具传奇。
特别推荐黑森林蛋糕、木糠杯（奥利奥）、拿铁咖啡和云南顶级原豆咖啡。
蜂蜜百香果：选用广西本地百香果，加入柠朦片调味，酸酸甜甜，口感极佳，我们先是要了一壶热饮，后来又要了一壶冷饮，两者各有特色，相对来说，更喜欢冷饮。
西冷牛扒：特地交待工作人员要七成熟，牛扒上淋上黑椒汁，咬一口，香味盈满喉腔，牛肉很新鲜，入口Q弹，赞一个！
黑椒牛柳炒意粉：黑椒和牛柳的份量很足，意粉口感很好。
四位好友的休闲下午茶时光，当然少不了红酒作伴。
黑胡椒披萨：份量很足，饼皮薄底香脆，咬一口，浓浓的芝士味盈满喉腔。

阳朔大桥上欣赏漓江秀色

桂林旅游，一般都是到阳朔看漓江风光，或到遇龙河看“水墨画图”。游漓江，骑行、徒步或坐船是最流行的方式，其实站在阳朔大桥的人行道上也可以欣赏到迷人的河岸风光。
也可以从大桥两边的入口走到河边，用相机或手机拍下迷人秀色。
但见河上游船徐徐前行，岸边白色民居连绵成片，喀斯特群峰层层叠叠，最后隐于朦胧之中。站在河边观光，有如身在画境一样，这也是漓江让人迷恋的地方。
阳朔大桥长度为207米，已经有将近100年的历史，是阳朔最长的公路桥，承担阳朔县城往兴坪等地的交通功能，每天都是车水马龙，热闹非凡。
阳朔大桥虽然是绝佳的观光点，不过一直不红，原因是大多数游客直接去玩遇龙河竹筏漂流，或到兴坪坐船，在县城附近看风景的人非常少，其次是阳朔大桥作为公路桥，不适合吸纳太多游客在桥上观光。
站在桥上，或走到岸边，眼前的景色美不胜收，望着层次感丰富的山峰和平缓的河水，你就会明白世人为什么会说“阳朔山水甲桂林”了。

阳朔秘境－原生态的青江渡生态农庄

在阳朔住了两个晚上，最后一天的上午出发到郊外探访青山里的秘境－青江渡生态农庄。
青江渡生态农庄位于阳朔福利镇094省道白马山村内，我们的汽车沿着不太宽阔的村道拐了好几道弯，上了几个小山坡始到达，感觉就像在寻找世外桃源一般。
一下车便被眼前的景色震住。但见田野雾气朦胧，山花迎风绽放，喀斯特群峰从近至远连绵起伏，若隐若现，仿如一幅水墨画图，又让人想起各种田园诗。
翻过一个山坡后，我们来到青江渡农业养殖基地，小狗看到我们，表情有点呆萌。
走地鸡是养殖场的主角，这些鸡散养在山谷里，喂的是玉米粒，不喂饲料等带激素的食物。
这些走地鸡养足一年才会拿出去卖，除了供应青江渡农庄，还供应刘姐啤酒鱼等网红餐厅。
从山谷回到山坡上，我们沿着石阶深入青江渡生态农庄，在草木掩映的地方出现几间吊脚木屋，这些木屋是农庄的就餐包厢，居高临下，可以欣赏到另一个山谷里的湖景。
在湖边的草木之中出现数个烧烤场，环境优雅，在此烧烤，氛围特别好。
农庄餐厅的大厅宽敞明亮，可容纳团队就餐，游客也可以选择在室外就餐，一边品尝美食，一边欣赏湖景，惬意。
我们选择的是古典气息浓郁的室内包厢。
包厢走的是简约风格，没有过多的点缀，反而衬托出山林雅趣。
鸡肉是刚刚从养殖基地抓来的走地鸡，没有经过复杂的腌制，只是将鸡脱毛、洗净、切好，用来当作火锅的主菜。
身着民族服饰的工作人员细心地帮我们把鸡肉放进汤锅里。
汤底用的是山泉水，没有复杂的佐料，目的是保留鸡肉的鲜甜味和山泉水的清甜味。
其实，美食做到最后，吃的就是原味，食材保留原味又好吃除了考究做法，更重要的是本身的质量。
当！当！当！鸡肉熟了，肉质鲜嫩甜美，汤底非常好喝，一口气喝了好几碗。
我想，如果你也来到青江渡，也一定会喜欢上这里的鸡汤火锅。
米饭是用柴火烧的，特别香。
发现湖边还有滑草场和儿童娱乐中心，适合一家大小一起来玩。听工作人员讲，前一天他们刚刚组织一场600多名小学生掏鸡蛋的活动。
民以食为天，如果放着美食不吃，去桂林阳朔旅游也就没有意义了，你说是吗？

关于住宿

这次入住的是后院旅舍，毗邻阳朔西街，但享一方清宁。整座旅舍的布置很雅致，喜欢无处不在的古风！ ​​​​
房间宽敞明亮，配置齐全，最令我满意的是冰箱里提前备好了饮料，这些饮料都是免费的。
早餐是地道的桂林米粉，粉很爽口，肉很鲜，最喜欢的是汤底。
位于益田西街的AMARI阿玛瑞酒店也很不错，这家酒店是泰国品牌，内部装修很有特点。

曾被称为洋人街的西街

西街并不是因地处阳朔县城的西部而得名，而是早些年到此流连的洋人较多而得名。别看西街现代元素丰富，其实已经有1400多年的历史。
这条石板街道始于隋朝，长度不足一公里，早期只有几间店铺，居住在此的大多数为农民，现在几乎都是生意人，西街的概念也扩充到周边的分叉街道。
很多人可能不知道，徐悲鸿大师曾在西街居住过一段时间，孙文先生也曾在此演讲，这里也是外国要员访问阳朔时必到的地方之一。
西街汇集了众多阳朔知名美食餐厅，还有一些中小型美食城、精品商店、手信店、特产店等等，其中包括我每次到阳朔都会吃的刘姐啤酒鱼。
西街还是酒吧集散地，夜幕降临后，各种酒吧生意兴隆，养活了不少歌手，一些歌手在西街磨练出实力，成为网红，或者专业歌手。西街还有很多画快像的画师，市井风情浓郁。
艾粑也是主角之一。
西街都卖些什么？除了各大古镇都有的工艺品、糖制品，壮族元素是一大亮点，这里可以看到很多绣花钱包、小挎包等物件。
辣椒店也是网红之一。阳朔出品的辣椒酱加入了蒜蓉，香味浓郁又不至于太辣，深受东部游客喜欢。 其实，在西街找家合适的店，喝杯咖啡，小坐片刻也是极好的。

关于交通

飞机：桂林两江机场，抵达后可乘坐机场巴士直达阳朔汽车总站，车程约一个半小时。阳朔到桂林机场最早一班汽车七点开出，最晚一班汽车晚上19：00开出；桂林机场到阳朔最早一班汽车9：30开出，最晚一班汽车晚上20：00开出，具体发车时间可能会有所调整，详情可咨询巴士公司。
汽车：桂林汽车南站有车到阳朔汽车总站，全程约60分钟至70分钟。班车为双向对开，早上六点开始发车，最后一班车晚上十一点发车，具体时间可能会随时调整，详情可咨询汽车站。
高铁动车：坐到阳朔站，出站后乘坐巴士到阳朔凤鸣停车场，车程约五十分钟左右，不堵车的情况下有时候四十分钟就到了。高铁巴士为双向对开。
