海洋光谱号邀约，送你一场海上奢华之旅

游记 摄影 特色表演 美食 邮轮旅游 2019/03/20

写在前面的话

大学一毕业，我就进了一家航运公司工作，一做就是七年，七年之痒，我离开了公司，但在这些年里，却因为工作的关系，对巨轮有了感情。公司的巨轮是拉货的，不能带人去旅行，所以我总是有一种对可以带人去旅行的游轮有着很强的新鲜加好奇感。去年乘坐了一次皇家加勒比海洋量子号，总算是见识了巨型游轮的壮观和豪华。
但是，敲黑板的地方来了，一艘新的豪华至臻的新游轮，皇家加勒比超量子系列第一艘游轮“海洋光谱号”（Spectrum of the Seas），即将于2019年6月盛大开启啦。

海上盛宴——海洋量子号的记忆犹新

乘坐游轮并不是一件只有爸爸妈妈才会喜欢的旅行体验，对于我来说，在海上漂泊的日子，对自己是一种疗愈。
大船开启，平稳的驶向深邃的茫茫大海之上，手机信号渐渐消失，我进入了一个没有网络的时段，剩下的几天，拍拍拍、吃吃吃、玩玩玩、晒晒晒就是我们全部的生活，怎一个享受了得，也正因为有了这样的体验，下船的时候，我竟伤感起来，因为这样的好日子告一段落。
住的地方很高级
上了船，第一个五星好评，来自于客房，我们选的是阳台客房，房间内都是选用高品质的材质、皮革和木材涂装，结合当代艺术所打造而成，洗漱用品是欧舒丹的，卫浴是汉斯格雅的，床褥软硬适中，非常舒服，打开阳台门就可以看星星看日出日落，好不惬意。
当然复式套房，可谓是量子号上的豪宅了，复式打造，开阔海景，艺术装饰，和按摩浴缸，羡慕的不要不要的，心里愤愤的：下次一定要在海上泡个澡。
玩的花样很多元
安置好自己，已经迫不及待的要上甲板去参观传说中的酷炫娱乐设施了。先坐电梯来到15层的北极星，这个获得了吉尼斯世界纪录最高游轮观景台的巨大球体可以360度的欣赏海天一色的蔚蓝，我觉得最佳体验时间是日落时分，唯美极了。
我是一个不大爱运动的人，所以船上的极限运动体验我没有参加，这里有个小贴士给你们，就是在游轮上想要体验到各种娱乐运动项目，一定要提前预约时间，否则，只能跟我一样，在甲板上看看别人玩儿了。预约最简单的方式就是上船前提前下载好这条游轮的app，到了船上，动动手指就能知道最新的游轮体验项目和预约时间了。
吃喝的种类多到爆
正餐可以选择的种类特别多，19家餐厅三百位国际厨艺大师的饕鬄盛宴可以选择，中餐、西餐、日料、火锅、创意菜等等，对于我们这种喜欢实惠的大胃王来说，自助餐就是最佳的选择，这艘船上的自助餐在14层的帆船自助餐厅，提供早午晚三餐的供应，重要的事情说三遍，船上有九家另付费餐厅，而自助餐是属于免费提供的10家餐厅中其中一间，这里中式、西式、日式、小食、饮料、茶点应有尽有，我们每次都吃到撑，而且在甲板玩累了的时候，来自助餐厅拿杯饮料休息休息也是极好。
晚上当然一定要体验一次船上的机器人调制的鸡尾酒，位于5层的Bionic Bar，坐在下面，看着台上的两个机器人大臂挥动，这就是科技的力量，连调酒都可以如此，未来的世界难以想象啊。
在船上发呆时间总是很多，在量子号的时候就听说了皇家加勒比新鲜打造的光谱号正在建造中，于是翻了一些资料来看，这一大波干货，看的我激动不已。。。

海上城堡——光谱号邮轮诞生全面升级 

最近，我看到一则报道，也是关于这条游轮的，就在今年2月28日，皇家加勒比宣布旗下新造船“海洋光谱号”完成新的建造里程碑，于德国帕彭堡的迈尔船厂正式浮出干船坞，这也就意味着，这条巨龙要远航了。
但是船舶浮出船坞后，还将经过一系列的海上航速、承受力和操控性能的测试，并于4月举办盛大的交付仪式，此后光谱号将驶往巴塞罗那港展开为期47天的奥德赛环球之旅，最终将于2019年6月抵达上海，开始她的首个中国航季，6月我们上海见！
航运公司出来的人，对船舶的参数总是很敏感，看到下面这组数据的时候，我也是惊呆了：
光谱号，是皇家加勒比的首艘超量子系列邮轮，建造于德国，总吨位168900吨，如果按照一辆微型家用汽车是1吨来算，这座海上城堡的总重量相当于一万六千多辆微型汽车拼在一起的规模，这是一个相对抽象的概念。但这条船有348米长，41米宽，活动空间足够大，载客量高达五千多人，并且舱房2124间，其中有一间二百多平米的超级豪华的完美家庭套房，可一次容纳11个人居住。
相比起我乘坐过的皇家加勒比海洋量子号邮轮，海洋光谱号有了全新的升级，这些新升级的项目，也是我极其期待的，先来一大波预告：
标志性设施升级：量子号上最炫的一个设施就是北极星观景台了，而在光谱号上，又增加了一个直径为11米的中空橙色球体——南极球，可以模拟穿越地心，极其向往。
客房升级：在保留量子号海上豪宅的复式套房的基础上，光谱号从13-16层，设立了皇家府邸，提供专属客房、餐饮和休闲服务，当然完美家庭套房也做了全面升级，261平米，3间独立卧室，配有卡拉OK、观影、游戏等娱乐，最酷炫的就是深处海平面的无敌海景浴室，以及从二层卧室直通客厅的儿童滑梯。
美食升级：同时容纳1844人就餐的免费餐厅；自助餐厅扩容近20%；同时在保留了了量子号人气餐厅以外，另外增加了正宗川菜料理和时尚下午茶。
娱乐升级：三场娱乐大秀，无论从设备技术还是表演形式都进行了升级；我这种麦霸也是有福气了，因为光谱号增加了卡拉ok，可以在大海上一展歌喉了；科技也做了突破，增设可直接通过肢体及动作就可以体验的AR虚拟娱乐互动，这是新鲜玩意，在陆地上都很少能体验的到，到时候一定要提早预订。
下面这一波光谱号升级增量攻略，妥妥的马克下来吧。

超凡美食——你就是游轮里的美食博主 

民以食为天，所以美食这一块格外留意了。噔噔噔噔。。。33种餐饮选择，中欧日等各种菜系选择多多，在量子号现有的基础上，升级并突破了6种美食概念：
川谷荟（收费），位于游轮4层，这是皇家加勒比的首家正宗川菜，鲜香麻辣、浓香四溢，想必海景之清爽碰撞上川菜之火辣，必将造就最难忘的游轮回忆，而且餐厅还配备有贴心家庭包房，与中式川菜成就绝配。我是个喜欢吃辣的姑娘，估计这里会是我主打的用餐餐厅了哈哈。
日本铁板烧（收费），位于14层，喜欢日料的不能错过啊，想想烤肉，炒饭，各种铁板烧，就在铁板诱人的滋滋声和引人入胜的厨艺秀中送到我们的面前，这不仅仅是味觉的享受，也是一场铁板上的舞蹈表演。
海涵火锅（收费），这是一块位于14层甲板帆船自助餐厅内的独立用餐区域，还可以一边吃火锅一边尽享船尾270度无敌海景。每次旅行回家，都一定要吃一顿火锅解馋，这次好了，在海上就可以吃到正宗的麻辣火锅，划重点：每天这里提供的新鲜肉类和海鲜，是不限量供应的哈哈，千万不要担心今天吃了，明天就没有了的尴尬。
味好面吧（免费），位于14层，北方人爱吃面，有这样一间面馆儿，心底是极其暖心的，想想那爽滑劲道的面条，淋上一清二白三红四绿的汤头和配菜，在热气腾腾中仿佛汇聚了一碗乾坤，定是可以在朝阳中开启崭新而美味的一天，亦可在深夜时分安慰我们的辘辘饥肠。
戏水美式快餐（免费），这是个儿童用餐区，位于儿童游泳池边14层甲板，餐厅里配有绚烂色彩的童趣造型桌椅，感觉是在游乐场里吃饭一样，而且餐厅的新鲜食材是经由大厨匠心巧手，为了悉心呵护每位宝贝的营养用餐，也是很贴心的服务了。
咖语茶道（收费），位于游轮4层皇家大道，集茶吧和咖啡馆为一体，不但有中式纯茶，咖啡，还有当下最流行的奶盖茶饮，吃完川菜到这里喝喝茶，倒是还蛮四川的，赞！
帆船自助餐，除了这些升级加强版的美食，我最关心的自助餐厅也是得到了升级，还是位于14层，总面积近3200平方米，比之前的帆船自助餐厅扩容近20%，能容纳千人同时用餐，刚刚提到的日本铁板烧、味好面吧和海涵火锅，以及索伦托餐厅，就在帆船自助餐厅内，中西荟萃，想吃什么点什么，友情提示，饭点的时候人一定很多，早点去占位啦。
主餐厅，位于3层和4层甲板，这是皇家加勒比将超量子系列4间独立主餐厅合并成为3层挑高，同时可容纳1836人就餐的免费主餐厅，行程中每天提供不同的中西两种菜谱，单点，或者自由选择分餐，还是家庭分享，随意选择。
我就在想，这么多美食，每一餐、每一道菜、每一个甜点都拍一组美图发到朋友圈，足够羡煞众人了吧。

超凡娱乐——新鲜、刺激、有高潮 

运动运动更健康
量子号有北极星，光谱号有南极球，这个南极球颜值就很是酷炫，与船艏北极星相对，位于船尾左舷、直径为11米中空橙色球体。球外是悬空玻璃步道，球内设有4个蹦极床，可以模拟穿越地心之旅，伴之以VR科技创造出的视觉历险体验。如果是家庭出游的话，一家人先登上北极星远眺，360度欣赏无敌海景，再穿越到南极球，戴着VR眼镜穿越地心，历险之后去蹦蹦床，酣畅淋漓。
出了一身汗也不怕，15层不是有免费的甲板冲浪吗，换上泳衣，在极速奔腾的12米长造浪模拟器上挑战冲浪的极限，爽！攀岩设施、慢跑步道也都是免费的哦。
当然全家一起出游，最开心的就是在一起享受乐趣，船上15层和16层的海上多功能运动馆应是一个不错的选择，因为有碰碰车、滚轴溜冰、桌上冰球，还有高级的击剑和射箭项目，LEVEL提升！最重要的是这里增设互动娱乐厅AR墙及AR地板，无需借助手机或AR设备，宾客即可直接通过肢体及动作，体验结合虚拟与真实的高科技超强互动。老年人站在外面看着馆里的孩子们开心的笑脸，那种感觉定是非常幸福的了。
不过对于我来说，运动是其次，海上疗愈才是重点，这就包括一个很重要的疗愈方式——马杀鸡。量子号的14层有一个巨大的玻璃屋顶景观泳池，晚上在这里游泳的时候抬头就可以看见漫天繁星，好浪漫啊。而按摩、针灸、活肤、除皱、紧肤等高级理疗项目，在这艘万吨游轮上也是必备的了，所有的疲惫和心塞，从这里丢进茫茫大海，好舒服！
晚上是狂欢的时刻，去到位于4层的明星时刻ktv，开一间包房，我很想知道，在这丰富的歌单和专业的音效背后，有没有麦霸的歌声可以穿过巨轮，在茫茫夜色的大海之上，传向外太空哈哈。这个位置我马克了，就算没有人陪我去，我也可以在公共练歌房唱给别人听咯，兴许还能成为海上光谱号的歌后呢。。。
视觉的顶级享受
光谱号将以超强演员阵容，在270度景观厅和皇家大剧院上演全新设计和制作的三场娱乐大秀，以数码宽屏投影为背景，通过充满异域风情的舞蹈表演，加上现代声光和智能机器人屏幕造势，打造全新感官体验。
台上一分钟，台下十年功，真是做什么都不容易，我看到这些演员背后的故事，也是极其感慨。演员们要前往位于迈阿密的办公室被层层面试筛选，每艘船都会配备一支专业的舞台技术团队。真冰冰上表演队伍、水上芭蕾及高台跳水表演的团队这些也都是需要严格精选并且严格训练，在四到六周的密集培训后，才能送上船表演的。
每场演出前的6小时，都要按照规定进行彩排。在没有演出安排的日子里，每天都要进行形体基本功的训练；参加舞蹈技巧提高教学课；负责演出服装的清洁与维护。每一场绚烂之下，都有一些感人的背后故事，了解到这点，等到看演出时，对他们最好的回报，就是认真的欣赏，用最热烈的掌声送给他们。

航线推荐——不用签证就可以去日本 

我查阅了一下光谱号的官网，目前有10条精彩推荐的航线，而且已经有的航线已经售罄了。。。基本上这十条航线都是从上海出发的，目的地都是日本，包括长崎、冲绳、福冈、大阪、京都、神户、鹿儿岛、熊本等，大家可以根据自己及家人的时间选择，并且在官网上直接预订哈哈，好人做到底，我看到的航线链接直接赠送了：https://www.rcclchina.com.cn/Rccl.Cruise/CruiseSearch/Search?offset=0&Keyword=海洋光谱号
处于对熊本熊的喜爱，我决定我的光谱号处女航，要选择可以落地熊本的航线，登陆熊本以后，带着熊本熊打卡这座美丽又气势豪壮的海港小城。

万事俱备，只欠东风

当然，这艘巨轮还在最后的冲刺阶段，6月份就可以露出庐山真面目了，至于这些丰富的内容有多震撼有多好玩，还需要你自己上船去体验一下了，我自己已经马克好了坐标，就等着她的首航了。
