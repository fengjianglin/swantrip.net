五天四夜，带你领略黔南好风光 ​​​​

游记 五一 人文游 登山 自然奇观 自驾游 2019/04/26
人们常言道：贵州黔南美，不到黔南非贵州。黔南布依族苗族自治州，是贵州省辖自治州，也曾成为南方出海丝绸之路的重要通道，也是黔中通往川桂湘滇的故道。这里是布依族，苗族，水族，毛南，瑶族等43个少数民族的居住之地，得天独厚的优势以及特色的民族风情，让黔南州变得远近闻名。
这次从广州出发，走过黔南的惠水:都匀、三都县、荔波以及平塘，五天四夜的旅程，带大家领黔南好风光，具体景点如下:
1.惠水:好花红民族村寨
2.贵定:金海雪山
3.都匀:中国茶文化博览园——秦汉影视城——杉木湖
4.三都县:水族文化馆——非遗水族马尾绣
5.荔波：小七孔景区——荔波冰雪世界城
6.平塘：平塘国际天文体验馆——水司楼

一、惠水

位于贵州省中南部，自五代起设南宁州，宋代置“八番”地，明代建程番府，定番府，自今已有一千余年的历史。世居在这块土地上的布依，苗，汉，毛南，壮，彝，水，回等民族，经世代开发，将惠水建设成为贵州著名的稻粟之地，桔果之乡。
1.〖好花红民族村寨〗
惠水的好花红民族村寨，是一个以布依族为主体的民族村，是著名布依族民歌《好花红》的发源地，也是著名的“中国金钱橘之乡”和“中华布依第一堂屋”所在地，名称“三块金牌”。它也是“全国首批少数民族村寨”以及被评为国家4A级景区。
在好花红里，最有名的莫数“中华布依第一堂屋”。它是一座具有200年历史的二层楼三合院，房子的第一层主要是立柱建筑，不是吊脚楼，又恰似吊脚楼。从房屋构造可见房主家产殷实，受汉风影响，两种建筑风格在历史的推进中互相影响着。从天井正中建筑的石梯已被侵蚀，同时可见些许苔藓，无不渗透着时光的痕迹。顺着16级石条踏步上二楼，一米多宽的走廊可供客人在此眺望远景，整座堂屋，显得雄浑而古典。
“中华布依第一堂屋”内收藏着丰富多彩的布依族文化古物，如布依传统纺织、织布、刺绣、服饰、炉灶、生产工具以及生活用品等，其中年代久远的为清代物品。​
另外布依文化底蕴深厚，民风淳朴，这里也是体验布依族民风民俗的好去处。​

二、贵定

1、金海雪山
初听到“金海雪山”，还以为是真的有雪山之景，后来了解到，原来每年的阳春三月，贵定县盘江镇音寨村，万亩金黄的油菜花好似金海，千顷雪白的李花、梨花就像雪山，它们交相辉映，因此被人们形象地誉为“金海雪山”。
“金海雪山有着“中国最佳生态文化旅游目的地”、“中国娃娃鱼之乡”、“中国苗岭贡茶之乡”、“全国森林旅游示范县”等众多荣誉称号。
步入景区广场，可以看见一座大型的雕塑像，上面有老人和小孩，另外在下面还可以看见玉米、茄子、水牛、大白鹅等农村常见的果蔬和动物作为底衬，这个被称为【农乐图】。而在【农乐图】旁边有一个一块【二十四节气图】石碑。而“二十四节气”也被列入联合国教科文组织人类非物质文化遗产代表作名录。
往园区门前走，就会看到最让人为之惊叹的便是矗立在广场内的这12组布依族的图腾灯。这图腾灯中间是竹子，左右两边是凤凰。至于布依族为什么用竹子当图腾，传说是在很久以前，有一个美丽的布依族少女到河边洗衣服，只见上游顺水漂来一节竹子，少女想捡来做洗衣服用，可是一不小心就被尖锐的竹节把脚划破了。这一划不要紧，少女回家以后竟然就怀孕，生了一个很聪明的小孩，这个小孩就叫布罗陀。布罗陀长大以后教会了布依族人学会用火、栽种稻谷、吹奏乐器等这些智慧和文明，从那以后布罗陀就被奉为布依族的祖先和智慧之神，因此竹子也就成了布依族的图腾。
景区内设有稻梦空间、忙牛阵、布依乡愁园、天下粮仓、庆丰大道、红绸丝路、田园水吧、水巷街市、演艺中心、水龙门等项目。
入到园区内，就会看到由这168个真牛头组成的【忙牛阵】。为啥叫【忙牛阵】呢？主要是因为布依族是中国最早种植水稻的民族之一，而牛在农业发展中占重要的地位，除了是农耕助力之外，还是布依族重要的图腾之一。
【天下粮仓】听说里面共建12座大粮仓。可惜去时还是建设当中，没看到。或许等下次再就可以目睹了。
【音寨】，这是一个有着600多年历史的布依族村寨，是“全国农业旅游观光示范点”，也是黔中最美好的田园民族自然景观之一，已被列入我省13个重点保护民族村之一。最为遗憾的是因当天时间关系并没有进入村子好好逛逛，只能留待下次了。
【水龙门】贵州及中国第一座跨河玻璃桥，长84米，距离河面高6米。在桥上可以眺望到清澈见底的瓮城河，两岸风景尽收眼底。听说每年夏天，很多游客都过来在河下面嬉水游乐。
从玻璃桥走过，发现这个大水车这边的风景真不错。绿草茵茵，田地里盛开的土豆花、萝卜花和马儿绘成一副天然的风景图。
【四季花谷】在贵州，花海并不少见，毕节的百里杜鹃、韭菜坪，荔波的薰衣草，兴义万峰林，甚至金海雪山春季的油菜花等都已独具特色各领风骚，很多游客也只因春天的花海慕名而来。而四季花谷，主要也是以花卉为主题。我们去的时候，恰逢遇到盛放满草地和山坡的“虞美人”。微风吹过，放眼望去，那些红的黄的白的虞美人像蝴蝶般在舞动，煞是迷人。
【匠人鲜花小镇】，里面汇集了民宿、匠人街、美食街以及屯堡文华园等设施。匠人小镇以非物质文化遗产为支撑，通过匠人手工技艺文化的展示和传承，让游客感受到匠人传统文化的博大精深，惊叹古人的神奇技艺的同时并感受到各个民族的独特智慧。
【民俗文化大观园】选择这个景点，是因为这里是贵州新晋的一个网红景点。
刚抵达园区，便看到【拦门酒】了。还别说，这米酒挺甜的，我连喝了好几碗，嘿！​
步入园区的民俗风情街，会看到一些由木头建造的房子。听说这些木头都是深山搜集过来复原的，所以基本上每一栋都有上百年的历史呢。难怪这样看着一点都不像新建的感觉了。​
来到观景台，看到了一条水系，水系旁边有很多建筑，原来这里是要沿着水系规划贵州的18个世居民族村寨，还有酒吧一条街，两边还有花海，估计明年就会陆续开放，细问之下才清楚，原来格鲁格桑贵州民俗大观园中的大观园就是指的18个村寨了。​
入园转身看到那个巨型大铜鼓啦。这个就是世界上最大的铜鼓。因为旁边有新搭建的马戏团帐篷，只有侧面拍一个。​
园内的摔碗酒和祈福墙。摔酒碗可以尝试下，另外祈福墙可以把祝福和心愿挂上去。​
除了这个铜鼓，还有这座铜鼓楼也是非常有名。每晚六点半左右都会在这里举行篝火晚会。六点左右时分，当广场的音乐响起了，陆陆续续游客都汇聚到了广场，等待篝火晚会的开始。6点半，篝火晚会准时开始，主持人上台做了景区介绍过后就是精彩的表演，第一首就是布依族的迎客歌，用布依族的语言唱的阿哥阿妹，另外还有与游客的竹竿舞等。最后就是跟着布依族同胞的舞步，一起围着篝火跳了起来……
总体来说，可以在在观园内观赏和参与各民族歌舞表演、民族工艺品制作，以及品尝民族风味食品，到处一游也是不错的选择。

三、都匀

都匀，简称“匀”。“都匀”一词在布依语里面是“彩云之城”的意思。广州有直达都匀的高铁。
1、中国茶文化博览园
中国茶文化茶博园仿1915年巴拿马太平洋万国博览会中国馆古建筑群1:1.8而建，总占地136亩，建筑面积1.7万平方米，总投资4.2个亿。茶博园是黔南州委、州政府为贯彻落实习近平总书记关于“把都匀毛尖品牌打出去”的指示精神，并以2015年正值都匀毛尖茶荣获1915年巴拿马太平洋万国博览会金奖100周年为契机，在原883兵工厂选址建设中国茶文化博览园。
茶博园内主要景点包括：嘉南坊、长秀阁、熙春阁、仰望塔、鱼钩亭、晚芳楼、云和楼、鹤楼轩、三昧轩、勤辛堂。​
在离茶博园大概2分钟的车程，还有一片大面积的茶园。这里便是盛产“毛尖”之地。可惜我们去的时候已过了采摘最佳时期，但即便如此，还是能看到很多当地的阿姨在采摘剩余的，阿姨还给我们展示如何采摘茶叶的技巧。感兴趣的就要等待明年了。
2、秦汉影视城
秦汉影视城是都匀经济开发区三大影视基地之一，是集剧组拍摄一站式服务，弘扬体验汉文化为一体的影视体验基地。秦汉影视城，耗资60亿，主题为“梦回长安”，以秦汉时期建筑风格为主，城中有堪称目前全世界最大的汉代宫殿未央宫、经典的秦汉建筑和饮食、有怀旧的家居、以华夏礼仪文化为背景的汉服等可供参观和租赁体验，更有上千年前汉代气息的宫殿。
独特的魅力吸引众多影视剧组过来取景，如《寻秦记》《斗破苍穹》《凰权·奕天下》《将夜》等。来到影视城，可以穿着汉服，体验一下瞬间穿越汉代的感觉，感受一下汉代的千年文化气息。​
3、杉木湖
杉木湖可以说是都匀另一个悠闲和集生态、文化、休憩、娱乐、健身、防灾功能一体的综合性市政公共公园。这里不但可以沿着阶级爬山锻炼身体，一路上还能遇到不少小动物，如孔雀、鹦鹉以及造型逼真的恐龙谷。再有，还能玩到现下最火的彩虹滑道与玻璃滑道呢，非常刺激。
在杉木上还有一座玻璃桥，可以在桥上眺望远处的湖光山色。​

四、三都水族自治县:

三都水族自治县，隶属于贵州省黔南布依族苗族自治州，位于黔南布依族苗族自治州东南部，是中国唯一的水族自治县。要想了解水族的文化历史，就一定要去【水族文化博物馆】。
1、水族文化博物馆
去了贵州才知道有水族这么一个少数民族。贵州三都水族文化博物馆，就是水族文化发展的象征和文化建设的重要标志。
博物馆以图片、文字、实物等多种形式结合来展示水族概况、美丽家园、民族历史与社会发展、水族文化、习俗与节日、欣欣向荣的水家新貌等六部分内容。另外水族铜鼓文化、剪纸艺术、水族舞蹈、水歌、水族丧葬、水族卯节等系列珍贵资料及藏品都尽收于馆内。
在博物馆内，能够很好地了解水族文化历史以及多彩的水族风情。
2、非遗物质【马尾绣】
马尾绣，是水族妇女世代传承的以马尾作为重要原材料的一种特殊刺绣技艺。是民间传统工艺，被誉为刺绣的“活化石”。这种水族独有的民间传统工艺，分布在三都境内三洞、中和、廷牌、塘州、水龙等乡镇的水族村寨，是贵州省三都水族自治县一种现存最古老而又最具有生命力的原始艺术。
主要产品形式有马尾绣背带和马尾绣花鞋等。现在马尾绣已入选首批国家级非物质文化遗产名录，想看这些非遗物质的手工艺品的，可以前往三都县中心城的【韦桃花】和【水族作遗项目体验中心】可观赏到。

五、荔波：

1、小七孔景区
被列为世界自然遗产荔波的小七孔景区，有着“地球腰带上的绿宝石”，同时也是中国最美丽的地方之一。景区因一座建于道光15年的小七孔古桥而得名。
沿着涵碧潭上游一直走，长达1.6公里之间，高高低低的河床，层层叠叠的瀑布，淙淙哗哗倾泻而下，或倾珠撒玉，推雪拥云，或如匹练飘逸，似银河泻地，形态各异，气象万千，这便是小七孔景区内有名的“68级瀑布了。​​
过小七孔桥，曲行百米许，即见拉雅瀑布。拉雅瀑宽10米，落差30米，逼近仰视，但见瀑首悬蓝天，旁缀白云，几疑天河自空而降，瀑势如山倒，吼声状雷，颇为壮观。瀑布溅喷的水雾飘飘洒洒，纷纷扬扬，人在瀑下，清凉的气息扑面而来，一洗征尘的暑热和劳乏。​​
鸳鸯湖，由两个大湖、四个小湖串联组成的一个奇妙的水网，是一组高原喀斯特湖泊。湖水如茵，可在湖上荡舟游览。​
来到小七孔景区，你定会被其美色所惊艳，里面翡翠般的溪水涓涓细流，瀑布气势磅礴，群山树林环绕植被茂盛，是一座天然的大氧吧，难怪被游客誉为“贵州小九寨沟”。而这里，也是避暑度假的好去处。​
2、荔波冰雪水世界主题乐园
冬天过去了，还想再感受滑雪的乐趣？可以来荔波冰雪水世界主题乐园体验一下。
荔波冰雪水世界主题乐园位于贵州省黔南布依族苗族自治州荔波县县城新区，是一个集冰雪世界，水上乐园为一体的主题乐园。
进入冰雪圣境，除了可以看到很多特色冰雕，其中便有仿照东北的“雪乡”情景。​
在里面，还有一条总长200米雪道，采用国际上先进的人工造雪和制冷技术。而场内温度始终保持在-8℃左右，365天感受滑雪运动的魅力。​
除了冰雪世界，游客来此还可体验冲天回旋、大喇叭、造浪池、梦幻螺旋、彩虹滑道、大小水寨、旋转木马、冰雪世界等水上世界游玩项目。​

六、平塘：

1、【平塘国际天文体验馆】
贵州〖平塘国际天文体验馆〗，是游＂中国天眼＂的必游项目之一，是宣传“中国天眼”，并普及天文学常识的自然科学类展馆。
天文体验馆内容十分丰富，馆内设有文字介绍，模型展示，黑洞，宇宙奥秘小电影等，它们为四方游客演绎着“宇宙、生命、人”的永恒话题。无论大人或小朋友，都可以了解和学习到丰富的天文学知识。
最后一站来到了独山一睹静心谷【水司楼】之容貌。
静心谷位于独山县影山镇，是翰林的故里，净心的福地。
“天下第一水司楼”高达99.9米的隼牟结构建筑，是现今世界上最高的木质水族楼，除水族特色外，建筑还融合了苗族、布依族特色，是净心谷景区重点打造的标志性建筑。​

七、关于美食

常言道：“民以食为天”。到某一个地方去观光游览，除了玩，当然就要好好的品尝一下具有本地特色的佳肴了。贵州有“三天不吃酸、走路打蹿蹿”的民谣，贵州菜肴突出酸，“酸汤鱼”是必要一试，其次在荔波品尝到的簸箕宴也非常有特色。另外还有虾酸、折耳根、腊肉等这些菜式都是当地的特色美食，前往贵州时都不可错过。
