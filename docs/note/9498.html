【初识】

大唐芙蓉园，一座大型的仿唐园林建筑，转一圈慢慢观赏需要1~2个小时，占地面积1000亩，其中水域就占了1/3，是中国最大的仿唐皇宫建筑群，气势恢宏，大气凛然，目前的灯展也是一大亮点。
陕西历史博物馆，免费不免票，一天只发6000张门票，收藏珍品多达170多万件，其中第五展厅门票单独卖300元，想逛还不一定有得逛的博物馆，被誉为“古都明珠，华夏宝库”。
小雁塔，避开大雁塔的人潮，独爱小雁塔的清静，唐朝时代的遗珠，地震震掉塔尖的小雁塔有种残缺美，还有关于地震雁塔“三离三合”的风云往事。
大华1935，1935年投入生产的长安大华纺织厂，以另一种形态承袭着近代工业的文明，身处闹市却又清幽宁静，朴实中带有几分复古情怀，避开人流的好去处。
曲江书城，大唐芙蓉园附近的网红书店，高颜值面积很大的一所书城，网红打卡地。
华清宫，历史上周、秦、汉、隋、唐等历代帝王都在这里修建过行宫别苑，更有“烽火戏诸侯，褒姒一笑失天下”的历史典故和震惊中外的西安事变于此上演。女娲补天、秦始皇温汤戏神女等传说也与华清宫有关，集众多典故传说于一身，华清宫也成为了中国唐宫文化旅游的标志性景区。
永兴坊，抖音红遍全国的毛笔酥和摔碗酒就在这里，汇集了陕西各地特色美食，回民街如果太多人，就来永兴坊吧。
言几又，西安新开高颜值网红书店，知名日本设计师池贝知子设计，网红打卡地。
秦岭野生动物园，秦岭的自然风光配上庞大的动物种群，这是西北最大的野生动物园，可以看到黑猩猩、山魈等稀有物种，以及坐观光车近距离观赏老虎等刺激体验。

【大唐芙蓉园：中国最大的仿唐皇宫建筑群】

这是第一个在外过的年，西安这两年越来越有年味，第一站大唐芙蓉园，畅游于绚丽多彩的芙蓉园，看灯火阑珊，品盛世繁华。大唐芙蓉园是一座大型的仿唐园林建筑，进去之后才发现里面非常的大，转一圈慢慢观赏需要1~2个小时，大唐芙蓉园占地面积1000亩，其中水域就占了1/3，不愧是是中国最大的仿唐皇宫建筑群，气势恢宏，大气凛然。
除了各种亭台楼宇，今年春节也举办了为期两个多月的新春灯会展，从1月28号一直持续到3月10号，夜晚的大唐芙蓉园灯火辉煌，璀璨迷人，有种梦回大唐盛世的感动，各种水陆花灯分布在芙蓉园内各个角落，配上激情四射的激光灯，满满的年味气息。
大唐芙蓉园立源于唐代芙蓉园遗址以北，由中国工程院院士张锦秋，按照唐代皇家建筑精髓规划设计的，于2011年被国家评为5A级景区。
而新春佳节的灯会，把这种盛世繁华推向高潮，5000年的历史文明，加上西安厚重的文化底蕴，配现代化的灯饰，传统与现代的完美融合，给了我们深深的民族自豪感，让我们不会忘记世界上曾经还有一个如此强大的朝代存在。

【陕西历史博物馆：想逛未必逛得到】

印象中的博物馆，总是和枯燥无味挂钩，我本身对博物馆并没什么多大兴趣，但西安的陕西历史博物馆，却改变了我的看法。这估计是西安乃至陕西最“任性”的博物馆，免费不免票，每天限量发门票，其中一个展厅的门票还卖到了300块，在节假日期间，想看都未必看得了。
原本以为博物馆应该没什么人看，但大年初一来就吃了闭门羹，来的时候免费的门票已经没有了，只能通过排队的方式或者购票的方式进馆，看了下门口排长龙的队伍，预计一个小时才能进馆，纳闷什么时候博物馆已经成为了网红景点了。
问了一下才知道，陕西历史博物馆每天只发6000张门票，上午3000张下午3000张，可以在网上提前预定，但看了下接下来几天免费的门票都预约完了。于是我们无奈只能逛了下西安其他的景点，在第一天晚上无意中看到有剩余的票，立马预约第二天早上的。而第二天早上过来，门口依旧排长龙，这是何等神奇的博物馆，网红店都没这么火。
当然还有一种可以立马进馆的方式，就是购买门票，博物馆里面有5个展厅，1~3号展厅是免费的，4号展厅收费30块，5号展厅竟然卖到了300元一位，卖这么贵门票的博物馆还是第一次见到。
一到三号博物馆主要的主题分别是陕西古代文明、大汉雄风、盛唐气象，而4号展厅是大唐的遗宝馆，展示的是何家村窑藏出土文物，5号展厅是唐代珍藏的壁画，有600幅精品壁画之多，据说都是绝品。
进入博物馆内，那人可不是一般的多，基本上可以用寸步难行来形容，但中国上下五千年的精华都在这里了，建议请一下讲解，不然自己是看不出其中的精髓的，这里展示的宝贝都是国家级别的，有的价值连城，收藏的珍品也多达了170万之间，每隔一段时间博物馆会更换文物。
对博物馆没什么兴趣的我竟然破天荒关了近三个小时，还只是免费的123号馆，不愧是“古都明珠，华夏宝库”，这里展示部分的文物造型奇特，设计精美，巧夺天工，不仅具有极佳的观赏性，还有极强的实用性，展示了古代人民超前的智慧。
很多外国人来中国的第一个城市就是西安，而西安必打卡的地方就是陕西历史博物馆，这是我国第一座大型现代化博物馆，它浓缩了中华民族的历史文化精髓，从远古时代到1840年，时间跨度长达一百多万年，这里也是是了解中国古代文明和艺术的一个窗口，陕西历史博物馆，绝对值得一看。

【小雁塔：“三离三合”的风云往事】

1487年，陕西发生六级大地震，活生生的把一座古塔震出了一条裂缝。1521年又一次地震，把这条一尺宽的缝又给震回去了。而这样的离合，竟有三次之多，现如今依旧屹立不倒，没错，说的就是西安最神奇的千年古塔――小雁塔。
来过西安游玩的人都应该知道，西安有大小雁塔，因为大雁塔太多人了，所以我们跑来小雁塔看一下，网传“大雁塔经济，小雁塔人文”，看样子的确没错，清幽宁静的环境逛起来也舒爽，小雁塔至今已经有1300多年历史了，历经漫长岁月的洗礼，如今风采依旧。
小雁塔建于唐景龙年间（707年），原本在荐福寺内，后来因为唐末战争，寺庙不见了，塔却存活了下来保存至今。小雁塔高15层，约43米，是典型的方形密檐式砖塔，大理的三塔寺就是仿造大小雁塔而建的，小雁塔是又佛教传入中原，而后渐渐融入了汉族文化并成为标志性建筑。
这座千年古塔不仅在建筑历史上有重大的意义，还有很多令人难以置信的风云往事，从建塔至今经历了10来次地震，不仅没有被震倒，还震出了“三离三合”等神话般的事迹。
1487年，陕西6级大地震来袭，周边很多建筑都未幸免于难，纷纷倒塌，小雁塔虽然没有倒塌，却惨遭毁容，自上而下出现一条一尺多宽的裂缝，一尺约0.3米，可谓是肉眼可见的大裂缝。而到了1521年，历经34年，又引来了一次大地震，神奇的事情发生了，小雁塔不仅安然无恙，还把之前一尺深的裂缝震合了！
而如此神奇的事情竟然连续发生了三次，明嘉靖三十四年，华县大地震，小雁塔一分为二，嘉靖四十二年“塔合无痕”，没错又合了，康熙辛未“塔又裂”，“辛丑复合”，这不是传说，这是真事！感情就像情侣谈恋爱般分分合合，“三生三世十里桃花”？
当然分分合合也不是毫发无损的，在1556年的地震中，就把最上边的两层楼震没了，玩大了吧。不过目前看，竟然几分残缺美，后期工程的修缮，都没有把小雁塔最上两层还原，如今的小雁塔，历经岁月的洗礼，依旧带着神圣的光环。
而关于小雁塔的“三离三合”，后期专家们修复的时候才发现其中的奥秘，原来古代的设计师将塔基用夯土整成一个半圆球体，地震来袭，小雁塔将地震压力均匀分散开来，以柔克刚，化险为夷，于是乎，小雁塔如“不倒翁”般存活至今，在此不得不佩服古人高超的智慧。
而小雁塔的旁边，还有西安博物馆，来小雁塔逛的朋友也可以顺便看一下，人没有大雁塔和陕西历史博物馆多，还有微波粼粼的湖水，宽阔的视野令人心旷神怡。
西安博物馆虽然没有陕西历史博物馆震撼，但里边好的文物也不少，各种佛像书画贯穿其中，细细品味还是可以了解很多西安的历史文化的。

【西安明城墙：中国现存规模最大的古代城垣】

全长13.7公里，走一圈需要3小时，作为中国现存规模最大、保存最完整的古代城垣，西安城墙绝对必须来看下，而其中最热闹的，无疑是永宁门这一片，过年的西安城墙热闹非凡。
这两年的西安，可以说是后来居上，连本地人都跟不上这么快的节奏，17和18年的春节，西安喊出了“西安年最中国”的口号，并把西安装饰了一遍，整座城市焕然一新。
过年的西安，不管是大唐芙蓉园，还是大唐不夜城，抑或是华清宫、明城墙，都张灯结彩挂满了灯饰，各种灯光秀表演更是让人心潮澎湃，在年味越来越淡的今天，西安敢为人先，为全国人民营造了一座极具年味的城市。
十三朝古都，中华民族的文化精髓都在西安，以前这里是汉唐盛世，以前这里有全世界最强大的王朝，外国人来中国旅游，必看的一座城市绝对是西安，深藏于骨子里的文化内涵，让城市发展更加有延续性。
人生第一次汉服就献给了大西安，感觉还少了一把剑，夜色西安，令人狂欢，也令人着迷，今夜，西安注定无眠。

【大华1935：春节里的慢时光】

小伙伴在大明宫国家遗址公园传开被游客“攻陷”的消息，于是我们转战隔壁的大华1935，人挤人的西安？不存在的，整个文创园都属于我们的了！
1935年投入生产的长安大华纺织厂，经过几轮岁月的洗礼，已经变成文创园，配以西安元素，是一个节假日依旧能够慢下来的地方。
废弃的钢铁插上想象的翅膀，变成了一件又一件艺术品，安静的大华1935，还缺少悠扬的音乐，不过很喜欢这样安静的氛围，和自己对话，和这座城市对话，这里是西安的文化创意园。
身处闹市却又清幽宁静，朴实中带有几分复古情怀，新旧元素于此交替，传统与现代正在交融，嗯，一大波年轻人正在赶来.......
于细节中捕捉生活的美好，慢慢品味，拾遗一份不被打扰的小确幸，静下心来，才能找到自己，春节的大华1935创意园，微风不燥。

【曲江书城：整个阅读室我承包了】

来一座城市，总喜欢去他的书店，因为书店的样子，可能就是这座城市的样子，满城的书香气，引领我来的曲江书城，这里就在大唐芙蓉园附近，去大唐芙蓉园的朋友可以顺便过来看一看。
一大早的曲江书城，几乎没什么人，墨绿色的色调，沉稳而宁静，美观而不张扬，书店很多，足足有三层之高，伴着书店舒缓的轻音乐，细细品味一个个艺术般的角度。
艺术的空间构造让人着迷，淡雅的国画添加了一些中国风元素，除了卖书，这家也卖咖啡，卖花，卖文创品，但没有杂乱无章的感觉，一切似乎井井有条。
最喜欢的还是它家的阅读室，一大早没几个人，都被我承包了，作为一家网红书店，肯定有不少人过来拍照，而独立的阅读空间，能够阻隔拍照的人，给阅读者创造了一个阅读的空间，整体而言，这家书店还是蛮不错的。

【华清宫：中国唐宫文化旅游的标志】

时隔一年多，再遇华清宫，竟有种说不出的感慨，但可以肯定的是，现在比以前更加漂亮了，满山的灯饰，点亮了骊山。文化的缺点之一，就是相对的枯燥，现在的年轻人追求的是刺激，传统文化不太能够吸引到他们的眼球，让文化接地气，以喜闻乐见的方式传播开来，让更多的年轻人融入进来，才能更好的传承中华传统文化。
现代的灯会，也是西安的包容，古都以前给人的印象就是刻板，不鲜活，但如今的西安真正动了起来，时代进步了，西安也应该与时俱进，文化应该以现代的方式来传播，才能更为让人接受，而不是以前古板式的教学。
华清宫从来不缺乏浪漫，童话世界的样子，或者就是眼前这般模样，温馨唯美，璀璨迷人。
斑斓的光影下，看到的是有活力的文化，西安也不再是围城，在年味越来越淡的今天，西安的春节氛围，仿佛又让我们回到了从前，将中国传统佳节和历史古韵以及现代科技结合在一起，如今的西安，绽放出前所未有的魅力。

【永兴坊：西安不止摔碗酒】

因为抖音，摔碗酒火了，毛笔酥火了，永兴坊火了，西安也火了，因为特地过来永兴坊看看这网红摔碗酒到底为何物。
春节期间的永兴坊，挤满了游客，火爆全网的摔碗酒更不用说，想围观拍个照片都难，一碗5块钱，游客拿过米酒，举过头顶，配合着激扬的音乐，一个简简单单的小仪式之后一饮而尽，然后将碗重重的摔在地上，主持人念道：岁岁（碎碎）平安，流程结束。整个过程不过10来秒，而排队可能要耗上1到2个小时之久。
那么这个摔碗酒又是什么来头呢？和西安到底有什么关系呢？摔碗酒在网上有很多种说法，一种是土家族两兄弟有了矛盾，为了以后的发展，喝酒摔碗将恩怨一摔而尽，另外一种是壮士断腕、舍生取义前饮酒摔碗的豪迈，“黑色酒碗手上端，黄泉路上不孤单。 喝了摔碗酒，黄泉路好走”，还有一种是关中习俗，人去世后的丧酒。
但无论哪一种说法，都好像跟西安深厚的文化底蕴没有什么直接关联，在摔碗酒的现场，感受得到的只有“碎碎平安”这种寓意，也难怪西安本人会说，摔完酒代表不了西安，毛笔酥更代表不了西安。有部分人还不知道是什么玩意，反而是外地人纷纷过来问才知道的。
不过细想还是挺郁闷的，“世界第八大奇迹”兵马俑没火，有着“古都明珠，华夏宝库”的陕西历史博物馆没火，感动千年的《长恨歌》没火，上过《舌尖上的中国》的肉夹馍没火，反倒是以前门可罗雀的摔碗酒火了。
不可否认，摔碗酒如果没有在西安这种特殊的历史文化城市，是火不起来的。有了西安厚重的文化底蕴的加持，店家摔碗氛围的渲染，群众盲目跟从的心理作怪，网络的传播，摔碗酒一摔而红。而作为游客，能够图个吉利，有了平安的精神寄托，加上摔碗摔的舒爽，围观群众的起哄，独一无二的消费体验，怎么能不火呢？
摔碗酒火得同时，也不禁对网络的风气感到惋惜，一个视频，如果没有人的一赞再赞，一评再评，是不可能指数级别的传播的，大众的兴奋点在于刺激，在于新鲜事物，而不是中华上下五千年优秀的传统文化，比起摔碗酒，兵马俑、长恨歌、博物馆的文物等等不是更应该得到人们的关注吗？
当流量减少、当群众兴趣减弱、当潮水褪去，摔碗酒还能摔多久，还能火多久，褪去浮华，摔碗酒还剩下什么？最后你会发现，唯有文化永流，那些沉淀几千年的优良文化，才是我们更应该追随的。来西安，别只知道挤永兴坊的摔碗酒，西安还有很多璀璨的文明等着我们去探索。
当然永兴坊也不止摔碗酒和毛笔酥，还有众多陕西特色美食。长武酥肉、魏府枣汁、临潼柿子饼、陕南魔芋、陕西纸卷、肉夹馍等美食琳琅满目，永兴坊汇聚了陕西全省的美食，让人应接不暇，如果吃全这条街所有的美食，估计要花上好几天的时间。
当然也有各种漂亮的大小饰品，永兴坊是唐长安城108坊之一，各种建筑古色古香，有种重回大唐盛世的感觉。

【言几又：逛出了“菜市场”的感觉】

去年西安新开了很多网红书店，其中最火的一家莫过于西安高新区的言几又书店，从颜值上来看，的确超过了绝大多数的书店，这是一家看了照片就决定要来的书店。
言几又书店离市中心还蛮远的，而且没有地铁直达，它位于西安的高新区，因为过年和位置偏僻的原因，这里的马路几乎见不到路人，庆幸自己等一下又可以霸占整个书店了，但推开门的一瞬间还是傻眼了，原来街上的人全都挤到书店来的，看着人来人往，心情有几分复杂。
不得不承认这家书店真的是颜值很高，看到图片就决定要来了，尽管交通很不方便。进入大门，映入眼帘的是两层楼高的H型书架，上面密密麻麻的摆满了书，4500平方米的空间配上独特的书架设计，感觉跟个迷宫一样，很多的通道和门都是用书堆叠出来的，给人感觉仿佛在书海中畅游一样。
书店分为上下两层，中间有一条曲线的楼梯连接起来，融入了生活美学，看起来非常有艺术气息，而楼梯上方挂满了“书页”，密密麻麻的“飘浮”在空中，加上透明玻璃的映衬效果，给人的视觉震撼非同一般，这个书店是日本著名设计师池贝知子耗时2年设计的，融入了艺术、生活、哲思、文学之美，自然也成为了西安的网红拍照地。
书店还经常举办各种签售会和座谈会，在电子商务的冲击下，实体书店已经倒闭的差不多了，混业经营然成为时下的趋势，书店不是慈善机构，它也要赚钱，所以卖各种商品也无可厚非，但商业气息的入侵和视觉上的冲击，已经让读书的氛围发生了一些改变。
你会惊喜地发现，咖啡桌上坐满了人，看书的寥寥无几，全都在聊天，而进店有一半以上的人都拿起手机在拍照，还有专门带闺蜜去拍的，书店充斥着嘈杂的声音，可能是新开的缘故，也可能是春节的缘故，这个感觉更像一个“菜市场”，加上这金黄的色调，更加多了几分浮躁。
卖咖啡没问题，融入生活美学的设计也没问题，但如果一个书店的设计过于抢眼，大于看书人的欲望，很容易让读书人的人产生浮躁的心理，而且会引流一些专门过来拍照的人，这部分人是不看书的，纯粹只是拍照，如果商店没有很好的规则，嘈杂的氛围会一直延续下去。
在此也强烈建议书店将阅读区域再完善一下，把熙攘的人群和阅读的人群隔离开来，创造一个安安静静的读书环境，这点曲江书城做得就比较好了，它专门有个阅读室，里边都是看书的，没有人说话，阅读氛围非常好，这是一家好书店，也希望它能带给读者更棒的阅读体验。

【秦岭野生动物园：西北最大的动物园】

作为西北第一大动物园，还是有蛮多可圈可点的地方，这里倚靠秦岭的自然风光，给动物们营造了一个优越的生存环境。动物园非常大，需要坐观光车才能进行游览，当然较近的地方还有机会看到各种有的动物。
我也不知道它们在干嘛，一不小心拍到这么美的画面，不打扰了你们继续。而这种红屁股花脸蛋的猴子叫山魈，脾气暴躁，据说经常打老婆而被分居，猴子见过不少，但种类这么多的猴子还是第一次见。
这种很迷人长相有种说不出感觉的猴子也非常讨游客喜欢，挤了大半天才拍到这一幕，一个个蜷缩着身子在发呆。还有黑猩猩！第一次看到黑猩猩还是有几分激动的，左边那只以销魂的睡姿躺着而无视游客的欢呼，而右边这只趁人家睡着了光明正大地偷吃东西，太有趣了，看了好久都不舍得离开。
接着坐车游览，第一站是大熊猫，果然是国宝，一直在吃竹子，动都不想动一下，而另一边的熊猫则在睡觉。短暂停留之后，我们去了最刺激的虎园，没错，还拿着肉诱惑它们，一只只趴在玻璃上，看看哪个孔有食物就去哪边，如今近距离的观看老虎真是惊险又刺激。
当然还有豹子和野狼，经过之时虎视眈眈地盯着我们，虽然是玻璃围起来的，但还是要注意安全，别把手指伸出投食孔了。

【后记】

几天的西安之旅也在这刺激奇趣的秦岭野生动物园画上了圆满的句号，虽然不是第一次过来的，但如此有年味的城市还是第一次感受到，西安年最中国，希望西安发展得越来越好，让世界更多的朋友们知晓还有这么一座活力的文明古都。
