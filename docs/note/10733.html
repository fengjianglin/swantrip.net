游记 摄影 登山 自然奇观 行程规划 跟团游 2018/11/07

前言

如果说冬天代表是白色，夏天的代表是绿色，那么秋天的代表一定是金黄色，深秋的季节，则是万树由绿逐渐转黄，金色洒满大地的曼妙时刻。每年的11月份，在 洛阳嵩县白河镇的400余株古银杏树迎来它们一年中最美丽的光景，这些千年银杏树高大挺拔，蔚然成林，姿态优美，叶形优雅，实属国内外罕见，构成了一处独特的古生物奇观，具有极大的科研、观赏价值。其中云岩寺67株千年银杏古树，2007年7月以“古寺庙内千年银杏树最多列入吉 尼斯 世界纪录”。

行程计划

这次行程目的地是到 洛阳 白河的云岩寺赏千年银杏美景。由于云岩寺距离 郑州 需要约5个小时车程，且由于地处伏牛山坳，如果没有很好的阳光，银杏叶就没有漂亮的景致，而云岩寺的向阳面最好的观赏时间则是上午至中午，下午由于阳光会被伏牛山挡住。失去了阳光照射，银杏的光彩则会大大减少，虽然太阳遮挡的那一段时间，金灿灿光纤是另一种迷人的风景，但是那千钧一发的时刻，可遇而不可求。于是乎，在第一天需要为行程搭配一个景点，缓解第一天下午的空档期，这个地点，则选择距在离云岩寺40公里外的老界岭。老界岭是传统老牌的伏牛山景区，南阳唯一的5A旅游景点，随着洛阳的；旅游资源强势崛起，目前南阳的各个景点略有落寞，游客稀疏了 很多。如今的老界岭在11月后，甚至打出了免费的大杀器，非常适合这次行程，景色好了，团员们会称赞这次活动地点安排，景色一般，作为免费景点，也不会说什么。并且，由于这里是老5A景区，景区建设是比较健全的，毕竟云岩寺附件没有住宿条件，这边则有上百家农家乐。
那么这次行程则为：
D1 郑州 —老界岭 老界岭农家午餐，游玩老界岭，夜宿老界岭农家；
D2 老界岭—云岩寺 游览云岩寺赏银杏 下午2点返程回 郑州。
洛阳
郑州-老界岭-云岩寺出行线路
我们的行车线路为 郑少洛高速—洛栾高速—栾川下高速沿着S249栾州大道—庙子乡转国道G311一路上山过隧道即到老界岭 老界岭有山路至白河镇云岩寺。
百度地图截图显示，从地图看老界岭与云岩寺，其实就是被伏牛山主峰鸡角尖相隔开。这边有一条山区公路可以直接到达云岩寺，比较便捷。

行程准备

这次出行前，旅行社给每位团员发了小提示
1.二日游住宿山里农家宾馆 条件不比市里酒店，因此自带洗漱用品；2.要带上厚外套 或者 轻薄的羽绒服，因为夜里山里会很冷；3.早上要带点零食，因为到达老界岭可能要到中午了，别在车上饿着了
猫猫的准备
1，拍摄设备：这次银杏树拍摄非常珍贵，一年只有2周时间，如果遇到大风降雨，那么最多会给三天拍一下落叶满地的景色，为了尽量少留遗憾，能带的设备还是要尽量背着，不过时间还是比较紧迫的，只有大概半天时间，带定焦设备，其实并不适合，而且由于只有一个背包，叶不能带太多东西，最终猫猫准备了1635 70200 两只传统的旅行镜头，都是万金油类型的镜头，适合各种题材拍摄。数码单反机身一个，为了保险，带了两块电池，CPL滤光镜一片 小型三脚架准备一只“”马小路01“”三脚架配手机支架，万一拍个延时什么也可以应付。
2，服装准备：猫猫穿了一件户外的速干衬衣，外面是一身软壳衣裤可以防风保暖。由于包包空间有限，已经不能带更多的衣服，里携带了一身迪卡侬滑雪高弹贴身衣，很像秋衣那种，可以在睡觉的时候换上保持身体温度。白天如果天冷了，下雨了也可以穿在里面，多一件总比少一件强。
3，其他准备：两瓶矿泉水，一块面包，一瓶酸奶，以及一些膨化小食，这些装在一个塑料袋子里，放在车上不拿下来，剩下就是充电器，洗漱包，卫生纸，现金200元零钱，塞在包包的边边角角。

游记正文

第一天，老界岭观伏牛主峰鸡角尖

周六早6点，猫猫洗漱完毕乘坐地铁前往集合点，今天赶上郑州戒严，需要外出的市民全部挤上地铁，早上第一班地铁就有菜市场大减价的赶脚，废了半天劲，到达集合地点，今天的团员素质很高，基本上没有迟到，7点清点人数，人员齐备，我们一行50人浩浩荡荡乘车开始进行我们的行程。
户外团也是团，路上就不再描述太多，行程需要5个小时，大家都是早起到达集合点，等大家落座后，无非是领队介绍一下行程，讲解一下景点，游玩的注意事项，乘车的规矩以及卫生等等这些，然后就是一些互动的小游戏活跃气氛，让这些临时安排在一起的团员之间不会那么的外气，尴尬。最后告知一下行程一共会有几个服务区可以供大家放松休息一下。最后还拿出一些小礼品，拍照的红围巾，几瓶红酒，魔术头巾等大家参与互动节目，让气氛达到高潮。之后进入服务区让大家放松一下，再之后大家就开始闭幕养神，这条高速服务区特别少，每个要间隔快两个小时。大概中午12点左右，我们的车辆到终于到达此行的第一个目的地，老界岭。
老界岭介绍
老界岭位于 河南 省 南阳 市 西峡 县北部，整个景区地貌由 中山 组成，主峰走向北西--南东向，支脉呈羽状向南延伸。山势陡峭，沟壑纵横，峰峦迭嶂，气势磅礴。主峰鸡角尖，位于太平镇乡 东北 部，是西峡 、 栾川 、 嵩县 三县界山，海拔2212.5米，为群峰之最，因山峰高矗，远看似向东引颈高歌之雄鸡而闻名。晴日山体呈青紫色，清晰雄伟；阴时云雾缭绕，时隐时现，其景壮观异常。
老界岭地理位置非常特殊，是 黄河 、淮河流域分界线，同时还是 洛阳 、 南阳 两地市的分界线，又是三县（ 西峡 、 栾川 、 嵩县 ）的分界线。现在老界岭的老界岭主峰鸡角尖（海拔2212.5米）归属龙峪湾景（ 栾川 县），次峰玉皇顶（海拔2203米）归属白云山景区（ 嵩县 ）。老界岭被一分为三，现在的老界岭景区已经大不如从前， 洛阳 为 栾川 嵩县 修建了高速公路及旅游专线，而老界岭则是依靠国家修建的国道、省道才能到达，最具特点的主峰没了，而且距离距离省会 郑州 则又是最远的一个，游客数量已经大大减少。
河南的很多山区自然景区，在以往的光景，冬季都有封山育林期，一般是12月-次年清明前，在金秋11月，老界岭现在已经属于绝对淡季，门票免费，这也为今天这个添头景点，增加了一丝吸引力。毕竟大家不是冲你来的，这里也是老5A景区，各方面的设施是非常完善的，现在当个开胃菜只能说它生不逢时。
我们一行到达预定的农家，然后分房间，猫猫很幸运的分到了最顶层，倒是安静，然后大家收拾东西，猫猫把除拍照以外所有的设备都留在了农家，然后进行了简单的午餐，午餐没啥说的，大家都饿了，很多人早上都没怎么吃饭，猫猫房间分的晚，下来的更晚，至于前几个菜长啥样子也没有看到，不过这都不是重点。我们饱餐战饭后，开始这次行程的暖身之旅，登顶老界岭。

云岩寺介绍
河南 云岩寺位于 河南 洛阳 嵩县 白河乡下寺村，整个区域分上、下寺两部分，面积约五平方公里。寺院背依伏牛山主峰龙池墁（目前 河南 省唯一一块没有被开发的原始森林）云岩寺目前仅下寺存留有一正殿和左右几间厢房。上寺地表建筑已基本无存。云岩寺周围的千年古银杏树据传为寺里的僧人所种，分布在寺庙方圆5平方公里的范围内。共有古银杏树400多棵，500年以上树龄的古银杏树413株，千年以上的古银杏237颗。其中，最大的一棵树龄达到2300多年，高36米，胸径7.5米，遮阴面积最大可达680平方米。还有3棵黑银杏，1棵雌雄同株，在 亚洲 绝无仅有。

返程

返程就没太多可以说的，唯一可说的就是，这边不算正规景区，每年云岩寺最美的时光也就是两周左右，当地小交通也是向附近景区借，导致运力严重不足，由于小交通压力太大，运力不够，景区看人太多了就不再售卖小交通票，只拉已经购票的游客，相当于甩客行为。不过村里领导号召全村及附近村的出来致富，村民都开着小车、摩托来拉游客，价格倒是公道，单程10元。猫猫为了节省时间，在村里找了个摩托车直接帮我带出景区，节省了不少时间。出来时候，景区门口已经堵车堵成一锅粥，中午1点，大批的游客才刚刚赶到，让本来就没有停车场的云岩寺，变成了到处都是停车场，里面的车出不来，外面的车进不去，僵持着。两条出山的道路，路边停满了小车，足足有两公里长。
返程，我们的大巴位置很好，我们的领队经验比较足，我们来的早，停的位置比较好，并且提前掉好了头，可以顺利出车。不过还是车多的问题，加上下午突然变天，下起了小雨，据说鸡角尖、白云山、老君山等景区山顶开始飘雪花，出山的盘山公路遭遇大堵车，我们的领队，司机多次停车，下去协调车辆，疏导交通，导致我们返程的时间比来的时候，多了快两个小时时间。
至于费用，总共花费340元，猫猫跟户外团，团费200元（包含来回车费、住宿费、一中一早两顿饭、老界岭小交通），自费120元（70元索道，30元晚餐，40元云岩寺交通费）

总结几条贴士

1、深秋时节山区温度较外面低很多，早晚已经低于0度，中午也就10度，需要多备一件厚外套保暖，最好是长一点的，晚上还可以当被子。
2、这边山区信号很差，在国道盘山公路也多次没有信号，更不要说云岩寺这种不正规的景点，上寺就完全没有信号的世外桃源，并且上寺和下寺中间也多次没有信号，所以最好有移动联通双号，可以的话移动电信联通三号。
3、不要单人前往上寺较原始区域，要成队前往，前后照应，那些所谓的山道，没有保护不说，有些地方都是人踩出来的，不是道，而且河里石头很多，万一歪一下，叫天天不应，叫地地不灵。并且上寺的游客已经很少了，能不能得到救治就很难说。
4、云岩寺较为原始，户外鞋非常重要，一定要注意防滑，猫猫一路见到很多小姐姐穿的漂亮小鞋，一走三滑的，如果只为拍照，在下寺那边比较安全，拍拍照就行了，不要向上寺去了。
5、山区物价比较贵，建议尽量自带干粮。这边可能出山，进山相对艰苦，物价或者说对游客的要价明显高于附件洛阳几个景区，而且山沟沟里面，没有选择，而且不到镇上是连小卖部都找不到的存在，如果心疼玛尼石，那就多带点干粮。
6、拍摄的时候，最好把小道都走几遍，来回走，下寺用石板铺好的路，在银杏林就有山中下三条，而且银杏林大概也就几百米长，走一遍也没多长时间，来回走，寻找不同的角度及风景。
7、云岩寺这段时间摄影师比较多，可以看到很多拿着长枪短炮穿着马甲的摄影师，多观察大师的机位与角度，也可以直接请教，都可以排到漂亮的照片。
8,、遇到人多时候，大光圈镜头是很好的选择，可以虚化背景，显得不那么乱，拍人像也是，虚化一些背景，可以把穿帮的其她游客虚化掉。并且使用一些中长焦的镜头，拍摄一些窄幅面的景色，也可以合理的避开大部分游客。
