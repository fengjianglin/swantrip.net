#冬日极客#四姑娘山二峰攀登|感受海拔5200米的世界

游记 徒步 摄影 登山 自然奇观 雪景 2018/11/15
提到四姑娘山，相信大家都不会陌生，无论是作为一个世界自然遗产、国家级风景名胜区，还是小说《盗墓笔记》中提到的冒险天堂，又或是驴友们心中的十大登山名山，四姑娘山总有着自己独特的魅力，吸引着不同类型的游客去一窥这“四个妹妹”的真容。
四姑娘山
四姑娘山由绵延不绝的山峰组成，从大姑娘山、二姑娘山、三姑娘山到幺姑娘山，四座山峰一字排开，长年被冰雪覆盖，虽然都可以攀登，但难度却依次递增，特别是最小这个妹妹，简直不好惹，其超高的难度让很多大神都望而却步，可谓是“可远观不可亵玩焉”。本次我所攀登的二峰，在四个妹妹中难度排行第三。

行前准备

1.身体状况评估：
户外运动是体力活，身体状况极为重要。虽然二峰的难度并不高，但是从未去过高海拔地区，或者是仅仅是自驾/火车等方式到达过高海拔地区的朋友需注意高反的问题，毕竟高反严重了之后会危及生命安全，可以先从牛背山、四人同等3000级山峰开始攀登，如果确实想挑战五千山峰，那么大姑娘也是一个不错的选择。此外，还注意是否有心脏病等长期性疾病及感冒等突发性疾病，一旦入山，要送医可能相对麻烦，因此，建议评估身体状况，在一个较为良好的状态才选择攀登。
2.天气观测：
现在的天气预报能较为准备的预报一周之内的天气，在决定登山之前应该留意天气，雨雪天会极大的增加登山难度，就算是大晴天，也需要注意冰雪覆盖程度，如果积雪太厚，任然会具备一定的危险性。去过两次二峰，第一次没有冰雪覆盖，明显就轻松很多。
3.装备准备：
托包：55-70升，用于马托，可以把睡袋及食物等较大、较重的东西放在里面。
冲顶小包：1个，用于冲顶时装随身物品。
四姑娘山
衣物：冲锋衣裤、抓绒衣裤（冬季）、轻便羽绒服、雨衣、帽子、魔术头巾、手套、太阳镜、唇膏、防晒霜50+++、头灯、雪套、冰爪。（装备好后如下）
四姑娘山
四姑娘山
鞋袜：高邦登山鞋，袜子2-3双，鞋子应该充分磨合过，确保脚的舒适。
四姑娘山
登山杖：2只
睡袋：舒适温标-5到-10
路餐：随个人习惯自备，建议轻巧易食、水、饮料。
公用装备：对讲机、气罐、炉头、应急氧气等
公用药品：主要为跌打损伤外用药，其它药品自备（体质因人而异避免药物过敏）
其它：能量胶、保温杯、暖宝宝、个人洗漱用品及身份证等活动相关物品。
特别提示：户外运动具备一定的危险性，一定要购买保险以防万一。
这次二峰攀登，穿戴了狼爪（Jack Wolfskin）的新款装备，德国的高端户外品牌，功能性和美观性都有，而且衣服很轻便，还有一个户外双肩包，使得旅途轻松不少。冲锋衣是三合一的，外层防风防水，内胆是羽绒材质，保暖御寒刚刚好，室外可以套在一起穿，室内则可以拆卸自由组合，可以单独穿里面的羽绒服，也可以单独穿外层的外套，这就是为什么越来越多的人选择购买三合一冲锋衣的原因了。我所穿的这件衣服，本身是滑雪服，这个外套内衬里面是有裙带的，收紧以后，可以防雪进入。这个设计非常不错，所以也就穿着来爬雪山了。
最近狼爪的公众号里还有相关活动，#冬日极客#活动，前往狼爪门店或者网上店参与活动，即送狼旗一幅。带着狼旗精心拍摄你的冬日出游照于2018年12月12日前上传至JackWolfskin公众号的活动链接，还有机会赢取免费北欧极光游和1000元装备基金，大家可以去参与哦，千万别错过啦！
户外运动具备一定的危险性，所穿戴装备不仅仅是为了好看，更多的是要关注功能和功效，因此，在进行户外的时候，选择恰当的装备非常重要，好的装备会让旅途事半功倍。这件衣服就具有Recco救援系统，遇到雪崩等意外时，能得到更好的救援。

登顶与下撤，感受5200米+的世界

原定计划是凌晨2点起床，结果1：20就醒了，倒不是兴奋，是因为昨天没休息好反而睡不着了。左边有人高反，一晚上不睡觉不停进进出出，右边有人感冒，一晚上不是咳嗽就是打呼，住通铺就是这样，你无法选择和谁一起住，就看天意了。
吃完早饭，凌晨3点从大本营开始攀登，四周伸手不见五指，全靠头灯一点光亮，伴随凛冽的寒风，很多人还没有攀登就放弃了。继续向上攀登，回头看下面点点光亮，再走再回头，下面却是一片黑暗了。向导告诉我，他们都放弃了，每次都是这样的，很多人会选择放弃。
雪很深，上周由于这个原因，几乎没人登顶，我的鞋子太大了，普通的冰爪穿不上去，没有冰爪，走一步滑半步，行程更加艰难。历时近3小时，终于来到垭口，这里有一个平台，可以稍作休息，正好临近日出，便拿出相机拍照。

回到大本营，收拾东西回撤，这个时候可以选择骑马了，毕竟冲顶又下来，体力也差不多了，而且都是来时的路，也没有新风景了，不过嘛，我还是选择了走回去，毕竟我是一个不骑马的人（穷人）。
户外是运动，依旧应该以安全为主，每个人的情况都不一样，天气与环境也不一样，中途放弃的人也没什么，山就在那里等你，下次再来就好，强撑也不见得是什么好事。
都说户外有毒，会让你不由自主的爱上它，虽然在路上，可能会走到崩溃，还会不断地问自己为什么要来找罪受，然后，看到雪山，看到日出，看到云海，一切抱怨又烟消云散了。
下次去哪里，我也不知道，一切都是未知的。看天气，看心情，时候到了，那就走吧。
