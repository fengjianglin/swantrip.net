杭州市新晋小众旅游景区——鸬鸟镇旅游攻略

游记 客栈民宿 美食 自助游 露营 2018/12/01
我在鸬鸟镇景区的一幅写生（微博ID已改为 旅游画手刀刀同学
写在前面的几个问题
1.鸬鸟镇在哪？
鸬鸟镇地处杭州最西北山区，毗邻竹乡安吉。从市区乘车大概两个小时可以到达。
2.为什么要去那？
杭州的西湖雷峰塔你们还没看腻吗？城市的堵车限号你们还能受的了吗？周末打卡难道就是河坊街南宋御街吗？（😂这几句好像广告词啊）正经脸，原汁原味的苏杭小城是需要去乡村体验的，不同于乌镇等比较著名的景点，鸬鸟镇没有经过过度开发，是真正的民风淳朴景色秀丽，很适合说走就走放松身心的旅行度假。

旅行干货

一、交通
我在山西，所以大交通是飞机
飞杭州时候在飞机上画画（微博id已改为旅游画手刀刀同学）
坐飞机到达萧山机场后，我们打车去河坊街转了一圈，然后才叫了个滴滴快车到达鸬鸟镇预定的酒店，用时一个半小时，费用280（高速费30，用时超出预期是因为酒店订的比较偏）。
据我了解杭州市区内没有直达的大巴或客车，而且滴滴顺风车很少，经权衡之下滴滴快车是性价比最高的选择。
这里我强烈建议大家自驾或者租车去，鸬鸟镇的交通并不便利，几乎没有出租车通行，滴滴打车更是没有司机接单。不同于杭州市的单双号限行，鸬鸟镇没有这些限制，且路面宽敞车辆少，不必担心堵车的问题。镇上景点密集，景区之间驱车最远不超过十五分钟。智能地图推荐高德，跟随导航行进不会迷路。
二、住宿
镇上多以民宿为主，从平价到高档一应俱全。到达小镇的第一天我们入住了房车公园营地，体验了一把房车住宿的乐趣。
入住第一天就迫不及待的画了可爱的车车
房车内部空间虽然比较狭窄，但是餐台，冰箱，烘干机，厕所，电视等一应俱全
清晨把车帷拉开，外面的风景很美
正好那天外面还在下雨，屋内干燥清爽，体验感满分
酒店信息：杭州鸬鸟房车露营公园
第二晚我们入住了山沟沟景区的民宿，让我欣喜的是民宿卫生条件不错而且价格也不贵，可以说是性价比很高了。
民宿内环境还是很不错的~
我们的房间正好对着景区，窗外就是蓝天白云小溪
其他几晚都是在这里住的，性价比真的真的非常高~

由于民宿之间相隔不远也很容易找到所以在这里我就不放详细地址啦，大家感兴趣的可以直接打电话向店主要具体地址，按导航开车不超过十五分钟就能到达哦
关于住你可能要注意：
1. 小镇民宿性价比较高，参考价格100-200元；特色/精品酒店参考价格500-1500元。
2. 入住特色酒店（如房车公园，牛路野营）需提前一周预定，并说明用餐情况。
3. 晚八点之后大多数民宿就会关门，所以尽量不要过晚入住；小镇照明设施不完善，太阳落山后黑压压一片，应尽量避免夜间出行，注意安全。
4. 山区多雷电，很容易造成电力设施故障，如遇雷雨天气应及时联系酒店或民宿工作人员，提前做好准备，以防手机或电脑供电不足的情况。
三、美食
为了体验当地居民最原汁原味的美食，我们此行基本上都选择在了居民家中进餐。
第一天到达鸬鸟镇时已是傍晚，下榻的酒店已经不提供餐食，在酒店工作人员的建议下，我们步行来到不远的农家乐。
饭店阿姨带我们来到自家厨房，让我们自主选择了食材，由于本次我们的队伍只有两个人，所以阿姨炒了一荤一素两个菜，尖椒鸡蛋和毛豆茭白肉片。
接下来的几天中我们每到达一个目的地都会就近找路边村民小馆吃饭：
辣椒猪舌和炒烩菜
当地烟熏过的猪舌肥而不腻，连着猪脆骨吃甚至还有点清爽
炸银鱼和胡萝卜豆角炒肉
银鱼是在山沟沟河滩里现捞的小鱼，一口一个很脆很下饭
还有一些当地特产一定要尝尝看~
这里的年糕是烤着吃的很清爽，一定要试试~
差点忘记推荐鸬鸟镇最最有特色的特产——蜜梨
鸬鸟蜜梨果形美观，肉质细腻，汁水丰盈，松脆甘醇，咬一口真的很解渴
当时我们过去正好是杭州最热的一段时间，随身背的水真的不够喝，但是一路都能看见果园，一边走一吃，非常nice

关于吃你可能要注意：
1. 不管是住在民宿还是酒店中，都要提前和工作人员说好进餐时间，口味偏好以及同行人数。大多数情况下如果没有提前预定，下榻的酒店不会额外提供餐食。
2. 尽量在饭点进餐。当地景区的餐厅都是村民自己开的，一旦错过饭点（中午一点之前，下午七点半之前），就很难看到开门营业的饭店。
3. 江浙地区饭店里的辣椒酱基本都是酸口的腌辣椒，醋也是不太酸的米醋，这和北方的口味差别还是蛮大的，需要提前做好心理准备。
4. 镇上小饭店的菜基本都会切好乘在盘子中放到冰柜里，点餐基本不用看菜单。如果碰到语言不通的情况，用手指指你要吃哪个盘子里的东西老板就会给你炒。因为是家常菜所以基本上不需要等待，顶多五分钟就可以吃到现炒的当地美食。
四、玩儿
鸬鸟镇纯玩的地方并不是很多，基本以自然景观欣赏为主。不得不说小镇上的风光真的是美如画，走到哪随手一拍都是绝美的风景，甚至不必刻意寻找旅游景点，在乡村小路上随便走走都会心旷神怡
在镇里东南的野天堂景区有个沙滩越野项目，到达野天堂小镇以后乘坐班车（只有一路）仅需要五六分钟就可以去往营地。由于到达时正好下雨停电所以我们没有来得及体验，但是据负责人介绍，沙滩越野对于户外爱好者和周末自驾游游客来说还是非常值得体验的
沙滩车很新很酷，半个机车迷的我随手画下来了
机车很酷有没有😆
车道沿途景色很美~
除了野天堂小镇外，太平老街和山沟沟景区也十分值得一去。
太平老街算是镇上的一条商业街，特产食品，水果，日用品，生活服务等都比较齐全。如果需要物资补给，去老街就对了。
山沟沟景区绝对是我五星推荐的地方，到达时由于刚下过雨，整个景区显得澄澈干净，溪流两旁的民宿错落，小路盘结，很有一种爱丽丝漫游仙境的感觉。
刚下过雨，太阳藏在云后，丁达尔效应美美美！
装修精美的民宿，山沟沟景区开发不久，所以民宿区也很新
民宿区还保留着当地居民最原始的生活状态
关于玩你可能要注意
1. 山区雨多路滑，一定要注意安全，避开湿滑的地方；家长要看好小孩，景区开发不久安全设施亟待完善，自护工作一定要做好。
2. 野外蚊虫多，长衣长裤长袜一定要穿好带好，风油精花露水也是包中必备。
3. 江浙地区多雨，来之前需查看天气状况，此行我们没有提前做好攻略遇上连天大雨，山沟沟景区涨水很多地方没有玩到，仙山谷漂流和野天堂沙滩越野更是没机会体验。
4.太平老街的店铺在快傍晚时就没人值守了，所以尽量选择在白天游览并补给物资。
5.鸬鸟镇开发时间短，许多自然景观都没有遭到人为污染和破坏，所以即使不去热门景区，周末开车出来随便逛逛走走也是极好的选择。
