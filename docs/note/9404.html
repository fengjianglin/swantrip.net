北纬22℃的风，在东澳岛开启一场轻奢的奇妙之旅

游记 亲子 海岛 精品酒店 美食 自助游 2019/02/27
珠海，素有“百岛之市”的美誉，在珠江辽阔的水域上，大大小小的一百四十多个岛屿星罗棋布，其中大部分集中于东部海域的万山群岛。在珠海众多的岛屿中，东澳岛天生丽质，以独特的自然风貌，坐拥15公里的黄金海岸线，迷人的钻石沙滩，闻名遐迩，吸引了众多海岛爱好者的前往探寻。
行程路线
day.1 入住格力东澳大酒店-享受海洋热疗-晚餐-参加蒙特卡洛酒吧主题派对
day.2 环岛游-体验空中飞人和射箭-地中海咖啡厅DIY烘焙-游览铳城等历史遗迹
day.3 蓝色海岸餐厅早餐-蜜月阁徒步-海星餐厅午餐-返程
关于交通
1.坐船抵达
东澳岛地处万山群岛的中南部，乘船抵达是最为便捷的。珠海横琴码头到东澳岛，船程是35分钟；珠海香洲港到东澳岛，船程是50分钟；深圳蛇口港到东澳岛，船程是70分钟。船票可在各港口码头或格力东澳大酒店的官方微信公众号预定，亦可在港口码头的售票处进行购买。
温馨提醒：乘船需要过安检，查验身份证原件和船票，大件行李需要放置在船舱的后部，每张船票对应一个座位号。如果是晕船的小伙伴，建议提前吃一点晕船药，遇上海浪太大的话，轮船确实会比较晃，需要稍稍注意。
2.坐直升机抵达
珠海九洲机场开通了到东澳岛的直升飞机航线，航程约10分钟，每天仅有一个班次往返，建议提前2-3天预定，可致电0756-8855888（转分机号6252） 。乘坐直升机出行的体验更为独特，美丽的港珠澳大桥和珠海歌剧院一览无余，旖旎的海岛风光尽收眼底，可以更好的满足部分旅行者的需求。
关于酒店
东澳岛上可选择的酒店并不多，民宿主要集中在东澳湾码头附近，由本地居民经营，住宿条件相对一般，但空调和热水也都还是有的。如果是来岛上度假，期望有更为舒适的住宿环境以及齐全的配套设施，格力东澳大酒店一定是作为首选，距离码头仅3公里，而且酒店给客人提供免费的往返于酒店和码头的接驳车接送，班次时间可在酒店官方公众号获取。
同时这是一家充满了地中海风情的酒店，设有餐厅、酒吧、无边泳池、私家沙滩、悬崖式海洋热疗、会议室、运动中心、健身房等一系列配套设施，十分的完善，除此之外，酒店的服务也是颇为热情，从码头搭乘接驳车至酒店门口时，酒店的工作人员热情的挥舞着双手，欢迎客人的到来。
行前准备
1.由于需要坐船到达，容易晕船的小伙伴建议提前服用晕船药。
2.泳衣泳裤是必备的，酒店的悬崖式海洋热疗是一定不能错过的体验。
3.一定要带上证件，登船前工作人员会核对船票和身份证。
4.东澳岛属于珠海区域，但由于距离港澳较近，偶尔会切换到其网络，建议提前关闭手机漫游，以免产生额外流量；一般情况下手机都是有信号的，另外，酒店也有高速WiFi，所以完全不用担心网络问题。
day.1 入住格力东澳大酒店-享受海洋热疗-晚餐-参加蒙特卡洛酒吧主题派对
一次美好的旅行，优越的住宿环境是尤为重要的。这个周末我和朋友开启了一场轻度假的海岛旅行，从广州出发，约一小时的车程即可抵达珠海站，随后前往香洲港坐船，历时50分钟船程便抵达了景色迷人的东澳岛。由于这次周末来东澳岛是度假休闲放松，我们毫不犹豫的预定了坐拥15公里黄金海岸线、迷人钻石沙滩的格力东澳大酒店。
到达东澳湾码头的时间是下午3点半左右，下船后就能看到酒店的接驳车已经在码头等候，坐上酒店的电瓶车，吹着海风，沿着沿海主干道，历时约10分钟，便来到了格力东澳大酒店。热情的工人员主动帮忙拿行李，另外还有装扮成大螃蟹卡通玩偶的员工为客人的到来表示热烈的欢迎。
步入酒店，宽阔敞亮的大堂展现在眼前，巧妙的利用了玻璃天窗采取自然光，让人感觉颇为舒适。再往前是天际吧，全透明落地玻璃的设计，在这里可以欣赏到绝美的海景风光，同时可以享受菲律宾乐队的美妙歌声。
酒店共有南沙湾和南山咀两座主楼，分别位于迷人的东澳岛南山湾沙滩双翼，拥有沙滩、森林、园景及海景景观的各式客房345间（套）。我们本次预定的是南山湾带有私家泡池的海洋热疗房，位于酒店1层，客房面积约60平方米，带阳台及独立海洋热疗泡池。
客房空间开阔宽敞，铺有干净且舒适的地毯，整体风格较为温馨舒适，同时营造出了一种度假的惬意。进门的右手边依次为衣柜、迷你吧台、储物架、休闲座椅等，拖鞋、浴袍、保险柜等配备齐全。再往外则是阳台及独立海洋热疗泡池，下午4点开放，足不出户，便可享受海洋热疗泡池带来的奢华体验。
进门的左手边则是独立的卫生间，浴室采用了半开放式设计，配有独立浴缸，浴帘可通过智能开关控制。洗漱台洗漱用品齐全，且摆放整齐，免费供客人使用。沐浴液洗发水润肤露等都是大瓶装，更好的符合了海岛旅行的使用需求。
在客房稍作休息后，已经按奈不住我想要去感受一番酒店极具特色的悬崖式海洋热疗，换好衣服，便赶紧过去享受啦。悬崖式海洋热疗，位于酒店二层，非住店客人是188元/位，住店客人是138元/位，当天可无限次使用。
古有“一览众山小”的豪情万丈，现有悬崖海洋热疗的柔情似水，悬崖不仅有惊心动魄，也有浪漫舒心，这是一种十分奇妙的体验。云岫漩涡池、人参池、当归池、牛奶池等等，各式各样的泡池应有尽有，更好的满足了客人的需求。泡上差不多一小时，能很好地促进血液循环，加速新陈代谢，有效改善体质，增强抵抗力。
云岫漩涡池，池内仿佛有一股神秘的魔力在搅动着暖洋，旋转的水涡好玩炫酷，顺势按摩疲惫的身躯，神奇又有趣，洗去所有的疲惫。人参池、当归池，进入冬季后，天气渐渐变冷，泡一泡不仅使身体更强壮，还能提高免疫力。
盐蒸房，别样的天然喜玛拉雅盐疗，房内连空气都饱含硒、碘、钙、钾等矿物质。有效的盐蒸可以消除疼痛和寒凉，还可以美容瘦身，舒缓减压，加热的矿物盐还能杀菌嫩肤呢。无边际池，下午可以欣赏到海岛绝美的日落和极致的海景；而夜幕降临之时，无边际池则变换着灯光，更添了一份浪漫和美好。
蒙特卡洛酒吧风格独具，环境优雅，是东澳岛上最有情调的酒吧，每晚都有乐队驻唱，可以说是岛上夜生活最丰富的一个地方。酒吧位于酒店一层，提供最前卫的各款饮料及特色鸡尾酒等，均为现场调制。
驻唱乐队来自菲律宾，女生是主唱，男生负责配乐，偶尔和音，非常合拍，会唱一些欢快的欧美英文歌曲，也有国内的流行歌曲等，女生的音色非常好，听起来也非常舒服。乐队主要是暖场活跃气氛，如果是周末入住的话，酒店会专门开展适合亲子家庭参加的活动，同时会有小丑互动，赠送各种小礼物等等，以便让客人更好的参与同时得到尽情的放松。
day.2 环岛游-体验空中飞人和射箭-地中海咖啡厅DIY烘焙-游览铳城等历史遗迹
用完早餐稍作休息后，我们决定搭乘酒店的电瓶车开启环岛游，游玩当天是多云，不过岛上的风还是挺大的。沿着沿海主干道一路游览，途径叠石、观景平台等，师傅会主动把车停下来，给我们一些时间欣赏美美的风景，以及拍些美美的照片作为最好的旅行记录。全程下来，半个小时左右。
由于第一天我们预约了空中飞人和射箭，环岛游览之后，便移步至运动中心进行体验。据介绍，酒店的空中飞人项目是目前广东省唯一的一家，颇具特色，有教练协助指导，第一次玩还是挺激动的，站台有10米高左右，需要自己一步一步爬上去。
在空中做一系列动作的伸展则听取教练的口令即可，挺好玩的，非常值得体验。在空中飘着的时候，感觉十分放松，为了过足瘾，一连玩了三把，不过还是有点小遗憾，没能完成后空翻的动作。射箭也还挺好玩的，教练会指导拉弓及架箭的姿势，教你如何瞄准，只要手不太抖还是挺容易中靶的。
下午我们在地中海咖啡厅和小朋友一起体验了DIY烘焙，咖啡厅以欧洲皇家餐厅的装饰布置为主，提供经典意式咖啡、精美点心，开心时光下午茶。小朋友们自己动手，利用模具尝试做各种形状的曲奇饼，然后放入烤箱进行烘焙，20分钟即可新鲜出炉。自己做的曲奇饼还是挺香的，味道也还不错，吃上两块刚刚好。
体验完DIY烘焙，时间还比较早，我们便决定去北岛感受一下南沙湾新石器时代遗迹，铳城的残墙，烽火台遗址，将军石林，“武当胜景”和“万海平波”的摩崖石刻，舒展着东澳悠久的历史画卷。
东澳铳城遗址是一座古城堡，距今已有270年的历史了。站在城墙上看着一望无际的大海，很有点凭海临风的感觉。石城墙，城门设于东南面，炮台设在西北角，且在铳城东北约40米处建有立锥形的烽火台一座。在城墙上观海景实在是太美，不禁让同行的朋友拿起相机帮着拍摄了几张美美的照片。
东澳岛“武当胜景”和“万海平波”的两处摩崖石刻，刻于清代嘉庆年间。当时，大海盗张保仔统领“五色帮”部队4万人以及4膄大船，打着“反清灭洋”、“劫富济贫”的旗号，称雄于南海海上。酒店也贴心的策划了“张保仔”寻宝活动，比较适合亲子家庭参加，一方面是体验寻宝的乐趣，也更好的丰富了岛上活动的多样性。
day.3 蓝色海岸餐厅早餐-蜜月阁徒步-海星餐厅午餐-返程
第三天的早餐依旧是在位于酒店二层的蓝色海岸餐厅享用，餐厅为开放式设计，空间十分宽敞，可同时容纳四百多人，室外平台可观赏无敌海景，靠窗的位置视野非常讨喜。现蒸鸡蛋肠粉、各式点心、热汤米粉、新鲜水果、咖啡等等，种类较为丰富，应有尽有，很好的满足了客人的需求。
用餐完回房间稍作休息后，决定步行前往蜜月阁看看。位于沙滩西北部的山峰上有一座楼阁，其名为蜜月阁，历经1008级台阶可至。登阁远望，澳门的凼仔大桥依稀可见，珠海也在朦朦胧胧中。蜜月阁顶层的中心悬挂着一口巨大的“一见钟情”钟，用力敲击，钟声悠悠，回荡在蓝天碧海之中。蜜月阁上钟情声，一见钟情到客船。多少年后，这钟声或许还会在有缘人的心中飘荡吧。
关于美食
1.在酒店用餐
酒店主要有两个餐厅，分别是位于2层的蓝色海岸餐厅和位于3层的海星餐厅。蓝色海岸餐厅为开放式设计，空间十分宽敞，共设有460个座位，靠窗的位置可欣赏到迷人的海景。蓝色海岸餐厅主要提供自助早餐，海鲜、粤菜、台湾菜等，以围餐为主，自助晚餐则需要视情况而定。
酒店的海鲜均由本地渔民直供，将军帽、皮皮虾、大蛏子、石斑鱼等等，在三楼的蓝色海岸餐厅可自由挑选，足不出户便可品尝到东澳岛最新鲜美味的海鲜大餐。除此之外，还可以吃到特色的台湾菜，三杯鸡、椒盐四季豆、蔬菜高汤煲鸡等，都是不可错过的美味，吃货可以大饱口福了。
海星餐厅则主要提供地道北方面食，精致面点，共设有158个座位。同时可以承接小型宴会，以更好的满足客人对独立空间的需求。第三天享用早餐的时间比较晚，游览完蜜月阁稍作休息后，中午则选择了在海星餐厅用餐，吃上一碗地道的红烧牛肉手工拉面，然后愉快的准备返程。
2.在码头用餐
东澳湾码头可选择的餐厅有好几家，都是本地人经营的，可以在档口买海鲜来加工，工费按每斤15元算。我们一行四人，要了一斤皮皮虾、四个大扇贝、一斤花甲、一只八爪鱼，在距离档口最近的一家餐厅加工，具体名字我也记不太清楚了，另外点了一份油麦菜，味道都做得挺好的，性价比也很高，吃完刚刚好。
后记
面朝大海，春暖花开。趁周末时光静好，即刻启程，约上三五好友，远离城市的喧嚣，在格力东澳大酒店开启一场轻奢度假之旅吧，来这里感受悬崖式海洋热疗的舒心惬意，一览无余的全海景尽收眼底，这才是美好生活该有的样子。
