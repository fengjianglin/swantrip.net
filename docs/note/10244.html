探寻广西秘境，四天三晚发现最美广西

游记 人文游 小众 精品酒店 美食 自然奇观 2018/12/11
【前言】
广西，山的国度，水的沃土。群山环绕，风景绮丽。在这里，天长蓝、水长清、人长寿、花长开。已记不清第几次过来广西，每一次到达都有不同的体验。去过桂林看山水甲天下、黄姚古镇看历史遗迹、三江看民俗风情，这次来到广西百色更是犹如探寻秘境，百色市有百色起义纪念公园等红色旅游资源，天坑、峡谷、原始森林等绿色资源。
这次自驾主要去的景点有吉星岩、旧州古镇、鹅泉、通灵大峡谷、古龙大峡谷、百色起义纪念馆、粤东会馆、浩坤湖、罗妹莲花洞、大石围天坑群。
【旅行指南】
飞机：百色机场已开通航线有：上海、广州、天津、深圳、重庆航线。
铁路：途径百色全境的云桂铁路直通云南昆明、广东广州等城市。百色至南宁高铁每天开行10对动车，从广西首府南宁乘坐高铁到百色大约1.5小时。
公路：从广西首府南宁沿高速公路自驾车到百色大约需要2.5小时。
吉星岩景区门票：100元 在德宝县城乘开往燕峒的公交车到达吉星岩景区，乘车票价5元。
旧州古镇门票：48元、鹅泉门票：48元、通灵大峡谷门票：115元、古龙山峡谷漂流门票：190元、百色起义纪念馆：免费、粤东会馆：免费、浩坤湖门票：50元、罗妹莲花洞：80元、大石围天坑群：90元
广西喀斯特地貌是最为隐秘的奇观，碳酸盐岩统治者广西西北部庞大的区域已有一亿年以上。北回归线横贯广西中部。这里热量丰富 雨水充沛。千百年来，碳酸盐岩遭遇流水侵蚀地壳隆起，凿刻出参差错落的石山之海，大自然从来不缺探寻者。岜盆山 碳酸盐岩构筑的山峰。喀斯特地貌中的水下洞穴神秘莫测。碳酸盐岩的主要成分碳酸钙，很容易被水溶解，地下水沿着岩石裂缝不断溶蚀岩石裂缝，形成复杂的地下溶洞和地下河床。

VOL.1：百色市→德保吉星岩→靖西·旧州古镇

百色市→德保吉星岩（距离100km，车程约1.5小时）→靖西·旧州古镇（距离50km，车程约1小时）住宿/晚餐：归顺山庄
【德保吉星岩，岩溶造型博物馆】
吉星岩景区属于溶洞类自然风景旅游区，主要有序洞、通天洞、龙王洞、惊天洞、无名洞、摩天洞等六个景点组成。这是我第一次到溶洞游玩，以前的想法是洞里面能有什么好看的，不就是一些五颜六色的灯照着石头么，看了吉星岩才发现自己的多么孤陋寡闻，他可不是仅仅是灯与石头组合那么简单。
他是大自然的瑰宝，是大自然赠与地球的礼物，是千百年来石头与山的不断碰撞，才产生出这么神奇的景观。最让人难忘的是水中石头的倒影，犹如看到了另一个奇幻的世界，让人震撼。
住宿/晚餐：归顺山庄
旧州古镇归顺山庄有很多野菜的菜品可以点，菜品很有当地特色，口味很赞，人均约60~80元。

VOL.2：靖西·旧州古镇→通灵大峡谷→古龙山峡谷风景区→鹅泉→百色市

靖西·旧州古镇→通灵大峡谷（距离约30km，车程约50分钟）→靖西·古龙山峡谷风景区（距离2km，车程约5分钟）→鹅泉（距离约40km，车程约1小时）→百色市午餐：群聚缘农家乐
晚餐：农耕园
住宿：百色梦之缘酒店
【绣球之乡，靖西旧州】
在广西省靖西市，有一处旧州古镇，此地自古以来以制作传统工艺品---绣球而闻明于世。心灵手巧的旧州人家家户户都会制作绣球，小到7、8岁娃娃，大到6、70岁老奶奶都会制作绣球，因此这里也被称作“绣球一条街”。
这里与以往所见到的古镇不同，没有喧嚣的街市，没有人来人往的游人。旧州古镇街道两旁几乎没有商业店铺，居住的都是镇子里的村民。青墙灰瓦，灯笼与绣球悬挂，清静而古朴。
绣球文化已融入到当地生活中，在广西壮乡成了吉祥物，是传递着爱情，亲情，友情的民族文化使者。村民们在自家门口坐着静静地绣着绣绣球，不得不赞旧州人的智慧，一针一线不断变换，精美的绣球就在旧州人的手中诞生了。
村头有一处明清古建筑文昌阁立，立于绿水之中，相传是文昌阁是用来镇住河水。
穿过文昌阁旁边的石桥，是一片一望无际的花海。花海中雏菊、向日葵争相斗艳。此花海游客甚少，显少有人穿梭。天气也是非常争气，山上还有弥漫的晨雾，此时的花海像桃花源让人向往。花田远处，石山遍布，如同一片浩瀚的峰林石海。
村子里有一位姓赵的爷爷，平时喜欢写生和画油画。我们在门口徘徊，犹豫能不能进他家房里，“请进”赵爷爷非常和蔼的邀请我们进屋参观。他是民间艺术家，刚开始是自学，经常到户外写生，专门画广西的景色。在房间里挂满了他的作品，以广西景色为主。最吸引人的是一张村口场景的照片，刻画的栩栩如生。如果再有机会，很想在这里多住一些时日，融入当地生活
【通灵大峡谷】
初到通灵大峡谷被植被茂密的丛林吸引，穿过丛林是峡谷的入口。从入口望去可以看到大自然的奇观，洞口与洞口相连，犹如通天的秘境。 
通灵大峡谷峡谷内呈现一派生机盎然的盛景，原始的古树，高深的峡谷，涧水潺潺，整个峡谷弥漫着沁人心脾的活力与灵气。这里荟萃了两千多种植物，其中许多属于国家重点保护的珍稀植物，如出现在侏罗纪时代与恐龙同时生长的桫椤、观音莲子座蕨类植物和金丝李、枧木、桄榔树、润椭、火焰树、奇异的咬人树等，使通灵大峡谷蒙上了神秘的面纱。
峡谷尽头一处高山瀑布，瀑布飞流而下，让人不由想起诗句“飞流直下三千尺，疑是银河落九天”的壮观场景。据说如果在雨水旺季瀑布会更加壮观，有时阳光照射还会出现彩虹悬挂在瀑布上的奇观。
【古龙山地下河景区，隐秘的】
通灵大峡谷附近是古龙山地下河风景区，位于广西省靖西市胡润镇境内享有“山水似桂林，气候胜昆明”的美誉。景区内一年四季都有漂流项目，古龙山与各峡谷隔山相连，地下暗河溶洞相通。乘坐皮艇可全程游览，夏季雨水充沛一个多小时就可以游完全程，冬季雨水少漂流需要3个多小时。漂流时会经过迎宾洞、百福洞、水帘洞等三个溶洞。经过三个溶洞，犹如黑夜到天明像是过了三天三夜。
河水清澈见底，两岸景色秀丽，葱葱郁郁的原始森林美不胜收。为了维护生态环境其中迎宾洞、百福洞并未大量开发。乘坐皮艇漂流至未经开发的两个溶洞时很很神秘，洞内漆黑一片，全靠船夫头顶探测灯的一束光灯进行照明，两侧不断传出水拍打石壁的声音，像探险一样。一边感叹大自然的神奇，一边感叹，这么美的地方竟然显有人知。第二个洞口出来时的水刚好被阳光直射，这时水的颜色犹如蓝宝石，清透洁净。
上岸后会有景区直通车可以乘坐到大门，车满了之后就开始前行，几分钟后停车，船工会一起坐车车折返，然而他们却都是站着的。乘客会在船工之前坐车。后来的一趟车也是，乘客坐着，船工站着。不知道说什么，看着他们其实蛮辛苦的，但是他们一直坚持着，他们的生活方式。
【最美乡村，鹅泉】
未入鹅泉先闻其桥，在众多鹅泉摄影作品中，最显眼的就是这座古桥，夕阳下的古桥，赶牛回家的村民让人向往。这次来鹅泉也是为了目睹它的风采。鹅泉内有一座建于清朝期间的古桥，桥长约60米，宽1.5米，全为石灰料石组成，是由15个涵洞的石拱桥组成，鹅泉古桥为靖西县保留得比较完整的古桥。
初到鹅泉时在车上刚好看到村民赶牛经过古桥的那一幕，立即停车驻足拍照。由于天色已晚，未能拍到最美的鹅泉。也许有遗憾才会有怀念，才会促使自己有再次踏上鹅泉的念头。
鹅泉是世界第二大跨国德天大瀑布的源头，鹅泉水长年不凅，水质清澈如镜，景色优美，因鹅山像一只鹅躺在泉水之上，故名鹅泉。相传古时这里住着一个姓杨的婆婆，偶于路边拾得鹅蛋两只，等了许久没人来领，带回家用体温孵出小鹅精心护养，一天放牧于池塘，骤然风雨大作，池塘陷为大深泉，小鹅不知去向。老婆婆因忧郁而死去，后为小鹅收葬，因名鹅泉。鹅泉风景区现有主要景点有鹅泉跃鲤、古桥，岜搭书、叫喊岩等。被鹅泉的景色吸引，山光水色，甚为清幽，喜欢摄影的可以来这里创作。
午餐：群聚缘农家乐（通灵大峡谷景区附近）
晚餐：农耕园（百色市）
农耕园的客人蛮多的，点了柴火地锅鸡和一些其他菜品，柴火鸡是现场做的，店员从点火开始做，大概需要20分钟能够吃到菜。口味还是挺不错的，建议你们尝一下花生芽和藕尖，特别好吃~
住宿：百色市梦之缘酒店
梦之缘是百色市新开的酒店中最大的一家酒店，酒店设备一应俱全。床垫特别舒服，这一晚睡得特别好。

VOL.3：粤东会馆→百色起义纪念公园→浩坤湖→乐业故事小镇

粤东会馆→百色起义纪念公园（车程约30分钟）→浩坤湖（距离65km，车程约1.5小时）→乐业故事小镇（距离120公里km，车程约3小时）
午餐：锦绣壮乡食府
晚餐/住宿：乐业小镇富氧山庄
【粤东会馆】
来到百色城内就不得不看丰富的红色旅游景点，百色起义纪念馆和粤东会馆。粤东会馆是中国工农红军第七军军部旧址。
1929年12月11日，邓小平同志成功发动发百色起义，同时，中国工农红军第七军在百色成立。军部设司令部和政治部，司令部就设在粤东会馆。对邓爷爷当时所用的办公桌以及卧室床铺都保留完整，依然可以看到当时简朴的生活。
相传以前有很多广东的的商人到广西经商，这里是粤商赴百色经商的落脚点，很多地方都还保留着岭南的文化和建筑，百色市粤东会馆就是一座保存较好的岭南风格建筑。
粤东会馆内布局完整，结构严谨，雕粱画栋，古色古香，天井、厅堂、阁楼，月门、甬道、青砖，极具岭南建筑特色。
【红色记忆，百色起义纪念馆】
来到纪念馆前,广场上屹立着邓小平同志的铜像,一手叉腰,一手高举,革命领导人的形象跃然眼前。上过阶梯来到纪念馆门口,只见整个纪念馆的外形就像是坚实的堡垒一般,宏伟壮观,让人一看就联想到了革命。
江泽民同志题写的“百色起义纪念馆”七个大字也苍劲有力,金色的字体也更显庄严。
纪念馆进门是赤红红七军石浮雕，走入洁净的每个展览厅：起义厅、英雄厅、将军厅、分为3层。有在广西各个县参与起义就义英雄名单。300多行，每行12个名字，3000多人。
百色起义是中国革命历史上重要且辉煌的一页。百色起义纪念馆通过《百色风雷》、《革命英烈》、《邓小平与百色》、《建设新百色》等篇章共计1400多件（幅）革命文物和图片资料的展览，展示形式新颖，借助声光电高新技术以及场景、影视等多种展示手段，使游客感觉仿佛超越时空，回到那群情激奋，战火纷飞的战争年代，可受到一次“血与火”的洗礼。生动的再现了百色起义和右江革命根据地斗争的壮丽画卷。
百色起义纪念公园位于百色城东面的后龙山、盘龙山、迎龙山上，是在整合百色起义纪念馆、右江民族博物馆、百色起义纪念碑、红军桥、铜鼓楼、南阁亭、两广青年友谊园等革命历史文化资源的基础上形成的主题性公园，成立于2008年7月，是中央确定的全国12个红色旅游重点景区之一“两江红旗，百色风雷”景区的核心区。
【大山中的仙境，浩坤湖】
“水笑山欢景致新，鱼腾鸟唱氛围亲。人间仙境北方是，敢比西湖韵味真。-------赞凌云浩坤湖”
通往浩坤湖的山路环绕，还未靠近景区就被窗外山下的一幕吸引力住。只见山下湖水碧绿，山间一处村庄在阳光的映衬下犹如仙境。
浩坤湖湖面约20平方公里，湖面宽阔平静，水质清澈碧绿，湖面四周山峦起伏，几处小岛竖立湖中间，自然风光美不胜收。看到如此美景，旅途的疲惫也随之散去。
午餐：锦绣壮乡食府
晚餐/住宿：乐业小镇富氧山庄
夜宿故事小镇，小镇的空气含有极高的硒离子，休息一晚褪去满身的疲惫。故事小镇的晨曦令人陶醉，清晨推开窗，望向静谧的山间，仿佛时间是静止的。

VOL.4：乐业故事小镇→罗妹莲花洞→大石围天坑群→百色市

乐业故事小镇→罗妹莲花洞（距离20km，车程约50分钟）→大石围天坑群（距离21km，车程约40分钟）→百色（距离190km，车程约4.5小时）
【罗妹莲花洞】
从乐业故事小镇到罗妹莲花洞的路途中一直在云雾缭绕，到达洞口前天空仍然雾气弥漫。洞外冷风瑟瑟，走进溶洞后温度瞬间暖和许多。罗妹莲花洞是一个溶洞内地下河不断侵蚀而形成的，洞内的地貌与其他溶洞有所不同。这里的钟乳石底部有一个个像莲花盆形状，内部有莲花盆达200多个，最大的莲花盆直径达9.4米，在世界上排名第一，为莲花盆之王。
洞内乳石悬泻，石帘惟妙惟肖，石瀑栩栩如生，造型奇特，其发育的莲花盆数量及规模堪称世界之最。洞内还可以见到大大小小散落的类似鹌鹑蛋大小的穴珠，穴珠是在地下河溶洞滴水坑中形成的具有同心圆结构的球状碳酸钙沉积物。
【大石围天坑群】
大石围天坑区位于，广西西北部乐业县，方圆20多公里的土地上分布着28个深陷石灰岩地层中的巨大坑洞。乐业地区降水量丰富，经多年流水侵蚀，地下溶洞顶部坍塌，形成了世界范围内罕见的天坑群。在地质学上被称作喀斯特漏斗群。
天坑是世界上最复杂的地形之一。天坑四周被悬崖绝壁环绕，暗河密布。地下河是碳酸盐岩地区地下水汇集和排泄的通道，在水流的作用下，碳酸盐岩被雕塑成千奇百怪的造型。数百万年来，天坑的底部一直隐匿在岩层中，近几年天坑的秘密才开始逐渐揭开帷幕。探险者们在大石围底部发现了十多万平米的原始森林，以及盲鱼，洞螈等被认为已经绝迹的远古动物。旱季洞穴的河水很浅，人可以徒步涉水。但是没有人知道，这些水来自哪里。旱季 洞穴的河水很浅，人可以徒步涉水。但是没有人知道，这些水来自哪里。
大石围天坑最大深度有270米，相当于90层楼高，一眼望去，令人生畏。大石围的天坑如同有吸引力一样让人发颤，看着脚下的深渊，再看看自己，在大自然面前人是那么渺小。
带着欢喜与兴奋离开了大石围天坑群，此次对秘境广西的探寻也告一段落。广西这个山美水美的地方真是值得深入的探寻。走这条线你可以看到诗人眼中的广西，山水画卷的原型，完成了我对广西的所有想象中的美景。回程的途中看着鹅泉的照片，幻想着如果能在此处生活十天半月岂不是人生幸事，广西再见，还会再来。
