海坨山谷：北京周边的瑞士风情小镇，假装在国外，炫爆朋友圈

游记 人文游 摄影 精品酒店 自驾游 避暑 2019/07/29
距北京市区两个半小时车程的地方，有一个世外桃源般的“瑞士风情小镇”，她就是海坨山谷。海坨山谷是奥伦达部落又一城，是继原乡美利坚之后的又一力作。
海坨山谷具备得天独厚的自然环境，坐落于海坨山风景最美的地方——平均海拔1300米的山谷中，整个度假区沿26公里山地而建，总占地约40平方公里。这里不仅仅环境优美，更是夏日避暑的好地方，受山地气候影响，这里的气温比北京市区要低5~10℃。这里早晚的温差较大，到这里游玩，在欣赏美景的时候，也不要忘了带上长衣长裤，冲锋衣什么的也备上，总不会有错的。另外，记得带上雨伞，山里的天就像小孩的脸，说变就变的，可能刚刚还晴空万里，天空飘过一片云，就带着雨来了，这里的雨来的快，收的也快，运气好的话，还可以看到雨后的彩虹。
海坨山处在北京市延庆区与河北省张家口市赤城县交界处，是北京第二高峰，延庆第一高峰。海坨山主峰海拔2241米，山脊是赤城县和北京市的分界岭，山阴为大海陀国家级自然保护区，山阳为松山国家级自然保护区。这里山高林密，保存有完好的原始次生油松林，是北京周边地区最完整的自然生态系统。
敲黑板、划重点的时候到了：来海坨山谷完全不需要购买门票，但一定要记得提前预约，重要的话说三遍，提前预约，提前预约，提前预约。

初识海坨山谷，从瑞士飞来的一颗明珠

有太多人向往瑞士，是因为瑞士自身就构成了一个浪漫的童话世界。推开门，打开窗，映入眼帘的就是一幅美丽的风景画，每个角度都是风情，每走一步都是风景。
瑞士风情也就变成了人们梦想中的庄园，完全可以称得上是庄园生活的巅峰。高耸的尖塔、独特的坡屋顶、经典的建筑比例，都会给人留下极为印象的深刻；瑞士风情的建筑色彩古朴雅致，同时拥有层次感极强的门和窗、充满自然野趣的装饰、独具风格的细部构建每一个细节的处理，都有一种大巧不工、天然去雕饰的意味。
奥伦达部落海坨山谷的出现，就如同一颗从瑞士摘来的最闪耀的珍珠，将它安放在北京与张家口交界的地方。从此，看瑞士小镇，不必大费周章地跨越欧亚大陆，在家门口就可以享受到假装在国外，炫爆朋友圈的感觉。
海坨山谷以瑞士小镇为蓝本，从建筑风格、建筑品质、景观设计等全方位将瑞士风情完美融入规划设计之中，借助有利自然地势，打造极具层次感的瑞士风情建筑群，完美诠释纯正庄园生活的隽秀典雅。

露营季，草地上长出五彩的帐篷

炎炎夏日，住帐篷、撸串、喝啤酒才是最正确的打开方式，如果你认为夏季理当如此，欢迎你来海坨山谷。这儿有大片大片的草地，有各种各样的大大小小的帐篷，帐篷花花绿绿的，有蓝色的、有黄色的、有五彩的，一层层的从这头延展到那头，如同夏日草原里的菇，如同夜空中最亮的星，一闪一闪的眨眼睛。
自驾而来，想要帐篷露营的，别忘了带上帐篷，到了这儿，您相中的哪一块地儿，只要还没人占，你就可以大大方方的在这里安营扎寨，分分钟构建一个小家。安顿好之后，可以看看景、发发呆，静享一个人的美好时光。
当夜幕降临，气温慢慢回落，一天中最舒适的时候到了，不要急着躲进帐篷吧。这时，你是不是应该燃起小炭炉，整个小烧烤，喝点小啤酒呢？天地为幕景，月色星辉佐酒，三五好友为伴，天地任我遨游。
当然也可以选择在这里租帐篷，悄悄的提醒一句，别忘了提前预约。

美食街上的饕餮盛宴

如果你不想自己动手解决吃食问题，那就来美食一条街吧。
西点、饮料、地道中式小吃、原浆啤酒、烤肉，想怎么吃就怎么吃，总有一款适合你。
餐饮一条街的就餐环境是开放式的，白色的遮阳伞为你遮风挡雨。清风徐来，花香为伴，吃着美食、看着美景，与小伙伴打打屁，畅想一下未来，人生之惬意也就如此了吧？
餐饮一条街享饕餮盛宴，丰俭由己，10元20元就可以吃饱，二三十元就可以吃好。在这里一定要给来这里的朋友们一个小提醒，记得带现金。你想想，在如此优美、田园的地方，用现金来消费是不是更有感觉呢？万一赶上个手机信号不好，电子支付不灵的时候，还是现金来得更靠谱些。

黑松林骑行主题公园，追逐山风的地方

吃得饱饱了，就去黑松林骑行主题公园散散心吧。
黄土小路蜿蜒曲折，穿过了低矮的灌木丛，穿过了格桑花海，通往千顷黑松林的深处。在最奢野的地方，做一做有氧运动，找山间的野趣，在林海里听涛，听风拂过黑松林的声音，听黑松林的歌谣，此时，我就像风一样自由。 

房车营地，五星级的住宿体验

除了帐篷，住在房车营地也是很不一样的旅行体验。
海坨山谷的房车营地可以用一个“壕”字来形容，它里是亚洲最大的房车营地，每个房车营位的单体面积都能达到夸张的600㎡以上，这简直就带四合院出行啊，烧烤、露营、逗弄宠物、玩一场勇敢者的游戏，想怎么嗨就怎么嗨啊。无论您是携家带口，还是与爱自驾的玩家交友，都是妥妥的。
房车看上去不大，走进去却别有洞天，无论您是会客、沐浴，还只是想美美的睡上一觉，都是妥妥的。
上百辆房车停在不同的位置，有不同的区域，栗林涧、冰川原……每个区域都有一个美美的名字，每个区域都有不同的风景。

彩虹牧场，孩儿们的乐园

城里的孩子，对狐狸、老虎、猴子并不陌生，可半散养状态的兔子、小香猪看过的却不多，彩虹牧场因此变成了孩子们的游乐场。
还记得童谣里是这样唱的：“小白兔，白又白，两只耳朵竖起来……”事实证明，兔子不论黑白，耳朵都是竖着的。这的兔子不怕人，不主动亲近，也不远离，自顾自的与自己玩耍。工作人员介绍，以前这儿养了好几只兔子，因为是半放养状态，现在跑得只剩下这两只了。
这儿兔子是散养的，鸡是悠闲的，小香猪呢？呃，它们在睡觉，怎么逗弄也不醒。

芬芳海坨心，天鹅翩翩舞

海坨山谷的中央区域有一大一小两个湖区，无名。有人管它叫海坨之心，是因为从山顶上往下望的时候，湖区犹如人的心脏。也有人叫它天鹅湖，是因为这里有黑天鹅常年在这里停留，黑天鹅才是这里的原住民。
据说，黑天鹅对环境相当挑剔，不是水草丰美的地方，它们是绝不会停留。这里的黑天鹅不怕人，即使游人靠近湖岸，它们依旧不会丢了自己的优雅，该做什么就做什么。如果你投食给它，碰见了它们喜欢的，它们也不会磨不开面子，是该吃就吃，该喝就喝。
带小朋友来的，在七彩牧场看过小兔兔、小香猪之后，一定要带他来这里转一转，和黑天鹅的亲密接触不可错过。

瑞岚酒店品茶、读书，静享安逸时光

瑞岚酒店是是典型的瑞士风情酒店，彩色小楼建筑，各色的绿植、鲜花，带露台的房间，还有不同主题的公共会客厅。现在开放的会客厅有书吧、陶艺馆、茶馆等，会客厅面向所有游客开放，就算不住在瑞岚，也可以来这里读读书、品品茶，小憩一下。
我最喜欢的是海坨山谷的IF书吧，书店的命名源自是英国作家、诺贝尔奖获得者吉卜林写给他12岁儿子的诗《IF》。作者用诗歌表达了对儿子精神独立和自我修养的勉励，希望他面对梦想和平庸，面对挫折和美好，面对庸碌和作为，都能坦然待之。
IF书吧设杂志主题房、复古主题书房和影音主题书房等三大主题书房，囊括三种不同的阅览和视听风格，近2000册书籍，涵盖生活美学、艺术、文学等，还有近200册的杂志期刊。
可能，你会和老友一起重读一本世界经典，重温一首经典老歌；也可能，你在山谷这间书吧里，听最喜欢的作者，分享创作经历。

1473咖啡馆——北京周边地区海拔最高咖啡馆

拾级而上，绿植相伴，花香袭人……
1473咖啡馆所处的山脊，是北京延庆与河北赤城分水岭处，1473咖啡馆也是北京周边地区海拔最高咖啡馆。咖啡馆外的观景平台，与2022年冬奥会高山速滑隔空相对，是冬奥的最佳观赛区。
1473咖啡馆的小木屋由芬兰重松家族的传承者托马斯亲手设计、打造，使用与英国女王狩猎小屋同款的建材，据说木屋的每一块木料都漂洋过海来自芬兰。
1473咖啡馆室内可容纳40人，室外可接待30人左右，产品主要以饮料、咖啡为主，其他还有鲜榨果汁、气泡水、茶、牛奶饮品等。将咖啡慢慢的搅动，咖啡的香气四散开来，阳光温暖而舒适，岁月静好。
1473咖啡馆比实际海拔多一米，多一米阳光，多一米海拔，多一米的富足，多一米的悠闲……所有的美好源自多一米包容。因为多一米，仰可揽日月星辰，俯可瞰海坨山蜿蜒的盘山路。

白桦林音乐节点燃夏天的夜晚

这是乐队的夏天，也是海坨山谷的夏天。7月20日，海坨山谷白桦林音乐节上，面孔乐队的出现点燃了海坨山谷的激情夜。
不到现场，你永远不会懂得这群摇滚老炮儿的激情、执着、偏执与热爱，他们看似放荡不羁的外表下，深深的藏着他们对于生活的热情，他们用摇滚、用音乐来表达，现场的乐迷随着音乐呐喊、摇摆、欢呼……
面孔乐队，成立于1989年的老牌摇滚乐队，三十年的风风雨雨，洗尽了铅华，看似磨平了棱角，主唱陈辉的表达多了一分幽默，多了一分温柔，但摇滚的内核永远不变。音乐会现场,面孔主唱陈辉在与观众互动时说：“只要观众还在喜欢，我们会永远唱下去，我永远不降调。”永远不降调与海坨山谷的诉求不谋而合，这就是所谓的相互吸引吧？一定是的，这不是《梦》，也不是《幻觉》。
如果您错过了7月份的海坨山谷白桦林音乐节，也不要着急，8月份的刺猬，9月份的Click15在海坨山谷与您相约。
