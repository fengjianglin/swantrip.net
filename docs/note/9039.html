探索泰国新晋网红海岛——歌谣岛

游记 摄影 海岛 精品酒店 自助游 闺蜜游 2019/04/14

前言

泰国是个神奇的国家，哪怕你去过很多次，它也能不断带给你惊喜，很多人都说泰国是出国旅行最有性价比的国家，这里有纯朴的民风、美味的食物和多样的风景，而且还能落地签免除签证烦恼，无疑成为了中国人出国旅行的后花园。
这一次，
我将带领你们揭秘一个泰国新晋网红海岛——歌谣岛，隐藏在原始森林中的歌谣岛，位于泰国普吉、甲米地的中间，虽然接近著名的普吉岛，但它保持着原生态的气息，拥有纯白的私人沙滩，茂盛的热带雨林，整个岛屿游客很少，仿佛是被世间遗忘的小角落……
明媚的阳光，柔软的沙滩，蔚蓝的大海，吃不够的海鲜和水果，人生梦想List里总是少不了这一条——去海岛度假的愿望，人们终其一生都在追求最理想的生活状态，在晴空碧水的山脚下听涛看海，何等惬意！

行前准备

一、货币兑换：
建议提前在国内银行预约兑换货币，因为到泰国过关时有可能会抽查携带现金数量（1W泰铢/人），汇率都会实时变化，我去的时候汇率是1人民币=4.6928泰铢，仅供参考；ps：在泰国支付宝和微信还是比较普及的，毕竟中国人旅游的人多，一般便利店或者商场都支持，再带上一张VISA或者银联就更完美了，当地有很多银行可以取钱。
二、必备APP:
1.Google Map（导航必备） 2.Google 翻译（语言沟通的救星） 3.Grab（泰版滴滴） 4.行程助手（行程规划、记账本）
三、物品准备
1.防晒霜，泰国的温度很高，建议选择SPF50+；
2.防晒衣，材质轻薄凉爽，方便携带；
3.驱蚊水一定要带好，那边蚊子很多，尤其海边有各种虫类；
4. 住宿的房间里一般都配备有洗漱用品、拖鞋和水杯，但还是建议自己自备一套洗漱用品和毛巾。
5.护照复印件打印两份分别放在随身的包和行李箱里，以防不时之需；
6. 相机尼康D850，镜头15-30mm、50mm,70-200mm，无人机大疆御、三脚架等拍照设备和各类充电器；
7．女生建议带一些适合拍照的裙子，海边很适合飘逸的连衣裙，别忘了再带上个草帽和太阳镜，光照很强，经常会睁不开双眼；
8.网络：可以到当地购买电话卡，小娴是在网上直接租了“环球漫游wifi”，网速还不错的，可以提前先在公众号里预定，然后去机场取机，最多可以连接5台设备，续航时间为4-6小时，值得推荐。
9.小仙女们最最重要的，化妆包。
泰国的温度很高，不建议画大浓妆，薄薄的底妆，会让显得比较干净。在这里推荐两款我包里必备的底妆CP，Za防晒妆前乳＋Za清透蜜粉，她们是我心中的油光byebye神仙组合～ 先说妆前乳，使用它之后非常清透，可以很好控油，还有SPF32+，抵抗日常阳光，非常适合去海边使用，之前也买过小香的同款，但觉得旅行中用这个足矣，小香掉了容易心疼（夸我贤惠）；另一款是清透蜜粉，如果肌肤状态还不错的小仙女可以在用完妆前乳后，直接叠加蜜粉，她的粉质非常细腻，不容易脱妆，还能有高级的哑光质感！Za控油妆前乳搭配使用简直是完美了！

四、关于签证
泰国实行落地签的政策以来可谓是相当便利了，如果时间充裕的话可以在网上提前办好，价格一般在250RMB左右，一般建议提前20天办理，准备好护照和2寸照片寄过去就行。
时间来不及的话可以落地签，费用是2000泰铢，如果要走快速通道的话是额外支付200泰铢，要求是带好有效护照、往返机票（可电子票）、入住酒店信息（可电子信息）、入境卡（航班公司会给）、签证申请表（现场会发）、40*60mm白底照片、2000泰铢签证费用，模板可以参考以下的图片。
PS:2019年四月底前去泰国都是免签证费！！！
五、关于机票
机票建议提前订好，东南亚机票经常会有打折活动，但提醒大家的是在选择航班的时候要注意行李问题，某些廉价航空虽然机票便宜，但是一般不带（或者带很少）行李额度，加上额外的行李费用可能就不一定划算了。

探索新晋网红海岛

阔别三个月，再次和巴黎老师相约在了泰国，我俩年前就在“遇岛”预定了此次的歌谣岛行程，开启了一场私密的海岛之旅
飞机从三万英尺高空缓慢下降，望向窗外透过云层就能隐约看到大面积的绿色，随着飞机滑行落地在普吉岛机场，空气中满是暖暖的海风，脱下了身上臃肿的外套，投入到了这场与夏天的约会中去。
规划师早早为我们安排好了接送机的车辆，从普吉岛机场开车到我们坐船的码头只需要半小时的车程，到了码头之后还需要换乘快艇才能到达我们的目的地，四十五分钟的船程，吹着海风，仿佛是一艘时光穿梭机，带我们来到了一片与世无争的世外桃源……
歌谣岛上享有秀丽的攀牙湾风光，举目望去，皆是美景，攀牙湾地质构造形成于侏罗纪时代，对环境要求极其挑剔的犀鸟见证着这里最原生态的环境。
对于未曾踏足过歌谣岛的人来说，它真的很神秘，“人少景美”是我对它最简单粗暴的形容，葱郁的森林、粗犷的海岸线，那是一种原始的生命力，是一种未经人工雕饰的自然野性。
岛上有两个酒店，分别是天堂度假村和树屋酒店，周围被热带雨林包围着，拥有绝佳的私密性，能满足各种人群不被打扰的度假需求。
唯一的代步工具就是电瓶车，穿梭于度假村之间，一下船他们就在码头等着为我们搬运行李，很是贴心。
岛上的时光很惬意，除了游泳晒日光浴，还可以体验到丰富的活动：药草蒸汽浴、泰式瑜伽、皮划艇、自行车环岛等，时间在这里就像静止了一般……
阳光很强，也抵挡不住歪果仁们一颗爱耍的心，轻划独木舟探寻大自然的鬼斧神工，会经过那些构造神奇的岩石和被掩藏起来的泻湖，即使晒黑十度也是值得的。
度假村里的SPA也非常值得体验，可以选择在室内也可以选择在海边，而我俩选择了在海边和大自然零距离接触，吹着暖暖的海风享受着泰式马杀鸡，按摩完整个人都打满鸡血了呢！
岛上的生活很惬意，这里没有灯红酒绿，纸醉金迷，这里有阳光小屋、蓝蓝的海、暖暖的阳，每一帧都是绝佳的取景地，随手一拍都是满屏ins风。
这里默默表扬下Za，有了这对油光byebye CP，即使在这么热的暴晒下也依旧不掉妆，出汗后，用蜜粉轻轻压下，肌肤感觉毫不粘腻，妆容轻薄，一点也不用担心会有可怕的厚重感～
这里的中国游客很少，更多的是来这里度假的欧美人，晒着日光浴吹着海风，当我们快被晒脱皮时他们似乎一点都不怕热！
偶遇了两枚超级无敌小可爱，镜头感十足的兄妹俩，看到我们就say hello，忍不住给他们拍了好几张照片，愿我们都能像个孩子一样天真烂漫。
每次看到这些萌娃就舍不得离开，脑海中闪现的永远都是好想把他们抱走哈哈哈哈哈，咋感觉自己那么像个人贩子！！
他们的爸爸妈妈就坐在海边谈情说爱，任由宝宝们自由玩耍，这种自由式教育值得国内很多家长学习的。
当然，除了在岛上游玩，还可以选择跳岛游，所谓的跳岛游，就是在岛屿之间来来往往地旅行，从A岛“跳”到B岛，再从B岛“跳”到C岛，乐此不疲地把一座座岛屿逐一收集到自己的地图上，这是如今关于海岛游最时髦的词汇。
攀牙湾以其壮观的石灰石岩层、洁净的海滩、泻湖和洞穴而闻名，乘坐长尾船，换上泳衣，跳岛去附近的海域，下水与海里的生物亲密接触，这才是来海岛正确的打开方式……
一天时间总共跳了五个岛，每个岛上都有自己的特色所在，喜欢浮潜的小伙伴一定不要错过这样的体验。
视线所及之处，都是温柔的颜色，感受着印度洋吹来的海风，沉醉其中不能自拔。
这是酒店为我们跳岛游而准备的午餐，三明治蔬菜沙拉水果等等很是贴心。
最后一个岛上还生活了很多monkey，他们不仅会爬树，还会游泳，最关键的是还要跟我们抢香蕉吃，很是有趣。
海风中夹杂着大海特有的清新和湿润，结束了我们一天充实而又疲惫的跳岛游……

海岛上的度假时光

海岛度假，那么酒店当然至关重要了，颜值高、品质硬、服务好的酒店，可以在旅行中留下更美好的回忆，歌谣岛上共有两个度假村，天堂度假村坐落在山坡绿荫之间的小城堡，在阳台上就可以俯瞰攀牙湾的美景，距离海滩仅几步之遥。
我们第一晚住的是天堂度假村，房间内满是清新的色调，与时尚大气的装饰搭配得恰到好处，让人心情瞬间舒畅起来。
原生态的自然环境，半开放式的洗浴间，让风景不加阻挡地自由行走，和传统酒店相比，十分具有当地的特色。
最美的风景里永远藏着一张床，光阴不可虚度，假期不能浪费，既然要躲懒，总该找一个最安静美好的地方。
餐厅也是极其具有ins风，看风景的角落多到你想不到，无一例外的，都是拍照的绝佳位置。
人少景美的小众地，不会有莫名其妙的人头闯进你的镜头里，每一天都能远离尘嚣，感受到浓浓的自然气息。
第二晚入住的是树屋度假村，大概是从小渴望拥有一个在森林中掌管星辰、与鸟儿对言的生活，对树屋总是有着无法言说的向往……
天堂度假村的边上就是树屋度假村，拥有上下两层的独栋树屋，四周只有鸟叫声，与朋友聊天偶尔沉默时，能听到风从树林间穿行而过的轻快声。
树屋酒店独特的魅力深得我心，上层是卧室，下层是宽敞的休息区，随手都是一张能吸引人前来的照片，光这树屋颜值，就值回房价。
每个房间里都是一整个墙面的落地窗，不放过窗外的每一处美景，只要你想，随时都可以步入大自然中……
特别钟爱那个浴缸，想必俘虏了很多女生们的内心，放满水，倒入浴盐，卸下都市中的铠甲，全身心松弛下来，泡一个温暖的热水澡，洗去一身的疲惫。
卫生间空间足够大，原生态的木质设计，环保而自然……
很喜欢房间里的各种摆设，足够舒适，足够安静，足够温暖，仿佛空气里都是细微的精致。
整个树屋度假村只能允许16岁以上成年人入住，绝对是为情侣夫妻们提供了绝佳的私密空间。
私密宁静的休息区如同海畔的一方伊甸园，吧台里的饮料可以随意喝，私人泳池可以随意游，伴着音乐晒着日光浴，在微微醉熏之意中慢享惬意时光。
清新、幽静、简单，耳听目视的种种舒心，都一一归为美好，让人觉得全身心放松。
树屋酒店内无与伦比的高颜值泳池，就算不会游泳，也想跳进去HIGH一场，眺望远处的山景，还能听到山中小鸟的叫声。
在航拍的视角下，每张照片都美出了新高度，刷爆朋友圈……
在这里，不用担心头顶是否有雾霾，不用担心挤不上地铁，可以穿着长裙，在海边的秋千上，看看海，吹吹风，是最好的咖啡伴侣……
幸运的我们在海边看了一场唯美的晚霞，如果不是一天的精彩在此刻迸发，晚霞怎会如此绚烂，如果不是一天的喧嚣在此刻沉寂，晚霞怎会如此宁静，变化无穷的云霞，使天空充满了神秘色彩……
夜幕降临，海风轻拂，洒落了一捧星星，浪漫在其中闪烁，暖色的灯光更显得这座岛屿静谧而美好。
这里的夜晚非常安静，有着绝美的山景作为天然的妆容，简朴而不失优雅，简约而不失格调，深深地印在了我的脑海里……

浪漫与美味的并存

是谁来自山川湖海，却囿于昼夜，那是美食与爱……此刻的我，上妆已经近12小时了，检查了下自己的妆容，因为调皮玩了下水，所以妆稍稍有点化了，Za蜜粉轻轻一压，30秒完成补妆，肌肤呈现出可爱的奶油肌状态，很棒～ 小仙女享用晚餐～ 
天堂度假村有两家餐厅，一家餐厅丰富多样，提供海鲜、西式、泰式餐以及主题自助餐，另一家古朴浪漫，来自欧洲的主厨在开放式厨房烹饪美食甜点，总有一款适合你的味蕾……
如果你是个浪漫主义者，更是不要错过和另一半在海边浪漫的烛光晚餐，这定是一个难忘的回忆……
树屋度假村的中心拥有专门的餐厅和小酒吧，夕阳西下，晚霞满天，在景观最好的位置，静静地看天边的云彩变幻着，听海浪拍打礁石的声音，享受这一刻的美好时光……
餐厅提供各种泰国当地的特色菜品，在泳池旁边，听着音乐，享受着此刻的美味，每时每刻，都有一道唤醒回忆的菜。

结束语
