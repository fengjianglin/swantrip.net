#首发#自驾河西走廊，从张掖到敦煌探寻丝路精华

游记 五一 人文游 摄影 自然奇观 自驾游 2019/04/29
#春日嬉游记#

写在最前面

“丝绸之路”，一个经常在我们生活里听到的名词。它起源于西汉时期，当年汉武大帝派张骞出使 西域开辟的以首都长安为起点，经甘肃、新疆，到中亚、西亚，并连接地中海各国的陆上通道。它的最初作用是运输中国古代出产的丝绸，因此得名“丝绸之路”。
我这几年也陆续去到了西安、西宁、敦煌甚至是中亚的哈萨克斯坦等地，对“丝路”的概念也越发清晰，但是想深入了解到它的意义真的还只是皮毛。这不，这一次又出发来到了大西北，沿着“河西走廊”，通过自驾的方式探寻“丝路”上这一小段的精华景点。
大西北地广人稀，行程做起来并不复杂。简单一点说就是飞机达到张掖，然后去到七彩丹霞景区，再经嘉峪关到敦煌，游玩鸣沙山，玉门关以及雅丹魔鬼城。 行程看着简单，但是过程却挺辛苦的，因为每天都要有大量的车程，这里也给我们的司机师傅点个赞。

注意事项

旅行可以说走就走。这次旅行也是收获颇丰，就连飞机上看见终年积雪的祁连山也特别震撼！但说走就就走并不代表毫无准备，贴别是大西北的气候环境和我生活的江南都有很大的差异。一定要注意以下几点：
1、甘肃的空气相对干燥，不管是身体还是皮肤都要及时补水，我这次去了五天居然还干到有点流鼻血。一款保温杯，保湿的水乳和面膜都可以准备起来；
2、大西北地广人稀，不像大城市一两百米就有超市、便利店，要准备足够的水和干粮，这可不是小事情，可能到了饭点都看不到饭店，有钱花不掉。我这次就准备了海底捞的自煮小火锅和方便米饭，随时随地就能来一盒。嘿嘿，你有一次旅行可以顿顿海底捞吗？这次我就给自己安排上了。
3、大西北不光气候干燥，紫外线也特别强烈，很容易晒黑晒伤。想起我刚毕业那会愣头青一样的去青海湖环湖骑行，就直接晒脱皮了。所以这次除了遮阳伞/帽，防晒霜，还准备了一些配饰。这不，我和我的小伙伴这次都戴上了海伦凯勒太阳镜，绝对的一举两得，不但眼睛得到了有效保护，拍起照来凹造型也是格外美丽！
4、因为上面也说了要带这么多东西，准备一个大的行李箱和背包也是很有必要。因为飞机上一些东西是必须托运，一些是必须要随身携带。比如超过100ml的护肤品等液体必须要托运，而相机的锂电池、笔记本电脑和充电宝都要随身携带。如果到了地方发现行李箱空间不足没法带特产了也没有关系，我的箱包一直都是在京东上买的，一是信任品质，二是看中物流。比如旅行途中不可能长时间等快递，如果我的背包带子断了，要买新的又苦于没有商店，京东下单次日即达，根本不会耽误使用。
好了，闲话不多说，我们开始介绍几处景点吧！

张掖七彩丹霞

这次来玩，已经是第二次走进张掖七彩丹霞，相比于去年十月份的旺季，这次来的体验感非常强，不但观赏栈道上不拥挤，乘坐景区的摆渡车也不需要排队，还能拍出大气的空景出来，绝对的美滋滋。
七彩丹霞共分为五个观景台，每个观景台之间都是有摆渡车乘坐的，在每个观景台也都可以看到神奇的彩色地貌，但是形态却是很不一样的。参观要依次乘车进行，分别是一、二、三、五、四号，最后的四号观景台也是最美的一座，赶时间的游客可以一路乘车过来，同时这里也是景区日落的最佳观赏点。天气好的时候可以把握一下。
可惜我们来的这一天是个阴天，但却惊喜的发现了气候的不同，呈现出的景象却另有惊喜。雨后的天气更加通透，石头的颜色对比也相对明显。不过下了雨还是有点冷的，刚好到了吃饭的时间，赶紧打开海底捞，倒入冷水就能直接加热。暖和暖和！也引来一票游客的羡慕眼光，哈哈！

鸣沙山月牙泉

鸣沙山位于敦煌市，敦煌市也是丝绸之路的节点城市。同时鸣沙山也是全亚洲距离城市最近的沙漠，毫不夸张的说，在敦煌市区某些地方，一抬头就能看到沙漠，还是蛮奇特的。鸣沙山我也是第二次来了，对景区里也是非常熟悉。
鸣沙山里怎么玩呢，最经典的莫过于骑骆驼，以及登沙山俯瞰月牙泉。不过首先要去租一个鞋套，15块钱！这里鞋子里面就不会进沙子了。来的时候和七彩丹霞的天气正好祥饭，特别的热，太阳也非常的刺眼。小伙伴们一顿操作，都把自己遮得严严实实。建议太阳镜，帽子和围巾都在来之前买好，景区门口虽然都有卖，但是质量绝对不敢恭维。
我们先是登上了沙山，沙山看着不高，但是爬起来绝对是很累的，毕竟一步一个脚印在这里也可以感受到的。爬山得时候也要戴好太阳镜，不但阳光刺眼，还会有一些细沙会吹起来，以免吹进眼睛里。戴不戴太阳镜真的也是蛮有区别的，即便是拍照，不戴眼镜在阳光直射下会睁不开眼，拍出的照片都会很难看。
登上沙山是征服的快感，观了景之后，眼睛里就只有冰可乐了！一路小跑下山，冲进了小卖部，哈哈……
喝完冰可乐休息一会，就去骑骆驼啦！通常一支驼队五个人紧挨着走，都可以和自己同行的小伙伴一起！骑骆驼危险系数很低，毕竟“沙漠之舟”是人类的好朋友，但是骑在背上较为颠簸，不太建议拿相机拍照，以免掉地上摔坏。如果想要拍照，可以找牵骆驼的工作人员帮忙拍照，不过再私下给她20元的小费。当然也可以有一个小伙伴不骑骆驼，在半路等着给同伴拍照。没错，我就是那个中国好伙伴！

玉门关

玉门关是从敦煌到雅丹魔鬼城的路上经过，就顺道去了一下。玉门关地处河西走廊最西端，距市区约90公里，因为当年西域输入玉石时取道于此而得名。这次从市区一路开车过来，发现两边土地几乎寸草不生，偶尔能见到的一点点水就算作是湿地了。
如今的玉门关遗址只存有断壁残垣的小方盘城。城楼也许可以灰飞烟灭，但玉门关曾经作为汉代重要的军事关隘和丝路交通要道，它的历史意义已经得到了永存！
玉门关专程从敦煌过来的话不是特别值得，但是顺路过来看看，或者当作一个落脚点都是极好的。在这里没有多呆，就去往了雅丹魔鬼城。不过这一路上的戈壁公路十分开阔，是拍摄公路大片的好地方，我们也找到了安全的地方，拍了一些！

雅丹魔鬼城

这次的雅丹魔鬼城我是第一次来，所以神秘感让我更加期待。魔鬼城里也并不是因为有魔鬼，只是它是一种独特的风蚀地貌 ，形状怪异，因此叫做“魔鬼城”。
雅丹魔鬼城的面积很大， 我不确定有多少。我们先是乘坐了景区内的大巴，然后再换城了越野车游玩。虽然地貌特征都是一样的，但是游玩的乐趣还是蛮丰富的。因为它们就像天上的云朵，形态各异，总能想象出一些不一样的东西。
它们有的像孔雀，有的像外星人，还有的壮观的像一个大型的舰队。能够保存下来如此完整的奇特地貌，本身来说就是一个难能可贵的事情。
景区内是不允许近距离走进地貌岩石下的，一是怕被破坏，二也是害怕有石头掉落或坍塌砸伤了游客。在规划好的停车点观景是极好的，搭配着越野拍拍照同样会很酷。
