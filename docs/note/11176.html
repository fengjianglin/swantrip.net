船帆石
从凯撒所在的垦丁大街出发，一路向南。
往鹅銮鼻公园方向，这一路上浓缩有巴士海峡的众多迷人的小精灵。
船帆石
路上首先能看见的就是一块造型奇特的巨石，由于其轮廓恍如一艘艘进港的船只，后来代代相传过后，当地的人们就将其称作船帆石。
船帆石
在船帆石旁，就是垦丁国家公园的贝壳砂岛保护区，当海浪不分昼夜的拍打着海岸，打碎了珊瑚、贝壳及其他生物的壳体，海流和潮汐将它们慢慢的搬到砂岛，就形成了这片可爱又美丽的砂岛，在这里仿佛可以让时光倒转看看贝壳砂的前世今生，当然这里的夕阳也一样让人迷醉。

屏东国立海洋生物博物馆
海生馆分为三大展馆，分别是台湾水域馆，珊瑚王国馆以及世界水域馆。
而台湾水域馆被垦丁人亲切的称为小水滴的旅行，从山到海的展示方式，生态及景观完全模拟实地景象，就如小水滴的旅游，从高山上汇集融雪、雨水而成的小溪，顺流而下，冲刷切割河床，挟带各种物质，逐渐堆积到平缓的中下游最后蜿蜒的流入大海，借着小水滴的旅行诉说台湾河川到海洋生态的多样性，这就是台湾水域的集成。

✔ 入台证
关乎于你的城市是否为开放自由行城市，如果是的话那就特别的好办，办理好台湾通行证以后直接到旅行社申请办理入台证即可，一般一星期就能出签证。如果很不幸你的城市不属于开放自由行城市，那么你只能报旅行团跟团或者去参加美容团啦！
✔ 一些碎碎念
1. 插座： 台湾的插头和我们两座插头是一样的，不需要带转换插头，但是需要注意的是台湾电压只有110V，少量酒店会提供220V的插座。
2.购物：垦丁基本只有伴手礼，小吃之类的购买，其他大牌子的商场购物就基本不用想啦，就算是高雄的国际机场，免税店也非常的小，只有日常常备的烟酒和化妆品。
3.衣物：垦丁基本一年四季都是夏天，温度比较高，但是早晚温差还是存在的，十月份去最好要带一些薄外套。
4. 防晒： 垦丁的太阳不是盖的，紫外线非常的强烈，防晒产品一定要带够，而且要多涂一些！
5.现金：虽然支付宝和微信已经在逐渐普及，但是相比大陆肯定还是有些差距的，现金还是要常备一些在身上，以防万一。
6.药物：好吃的太多了，肠胃不好也是正常的事情，止泻药，晕车药什么的最好也要带一些。
