二、旅行整体穿搭

>> 游乐场

白色蕾丝过膝裙+樱桃红针织外套+游乐场的发箍头饰+浅色平底鞋
作为小仙女，白色蕾丝裙是最温柔的百搭款，长过膝盖显得更稳重，搭配樱桃红的针织外套，更显甜美。在游乐场游玩时，再买一个应景的发箍头饰，更显俏皮~
 

>> 海岛

蕾丝上衣+黄色裤子+复古丝巾+大帽檐草帽+爱心眼镜
在海边，俏皮风适合走可爱线路的小仙女。嫩黄色的衣服在蓝天碧海绿树中非常显眼，配上加分的波点元素，复古的丝巾也可以在用来遮盖湿掉的头发。这一身非常适合去海岛玩的小仙女们~
 

三、旅行重点单品推荐

 

>> 草编包

草编包是一个非常实用且百搭的单品，空间很大，可以放女生很多随身的小物品，尤其是去海边时的瓶瓶罐罐，可以都有了安身之处~
而到了比较有欧式风情的地方，草编包搭配浅色的长裙，又有了不一样的味道，是不是非常优雅呢~

>> 吊带、背心

背心是一件非常突出女生小性感的单品，尤其搭配上色彩相称的A字裙，会非常有女人味！
如果觉得背心+A字裙麻烦的话，可以选择吊带长裙，也不要太长，过膝盖就可以，可爱又不失小性感~
 

>> 复古丝巾

丝巾的话，参考国外博主叠成三角形系脑袋上，罗马假日的feel就来了~
丝巾也可以开发其他装饰功能，可以放包包上，做短围巾，甚至做腰带，是一件非常增加色彩感的单品！

>> 毛绒帽子

冬季来临，这些毛茸茸可爱的帽子，除了出游，平时也可以戴着拍照，尤其圣诞节将至，两个小毛球减龄满分。
如果去比较冷的地方出游的话，这样一款帽子，搭配浅色的衣服，和雪景相衬，让人想起了电影《东京爱情故事》中莉香在一个大雪天去了完治的家乡的场景。
 

>> 发饰

喜欢温柔可爱的风格的话，这样简单、婉约的发夹是很不错的选择。
建议选择和衣服同色系的，或者全身没有亮色的时候，选择颜色比较亮的，整个人就会感觉活泼起来~

四、牛访谈

1.【牛班长】可以看出你很喜欢复古的风格，可以给大家一些建议嘛？

【怡妞子】时尚总是在反复，如今虽是2018年，流行的元素却是一直上演着70、80年代的复古风潮。
复古总是给人感觉耐看又有韵味。
服饰：可以选择过膝连衣裙，碎花图案，绿色、深蓝都不错，或者我穿着这种酒红色，复古风浓重的。
妆容：眼妆尽量淡一些，红唇，可以把嘴巴画的饱满一点！
姿势动作：高冷or随性自然。高冷的适合姿势动作幅度小一些的，偶尔看看别处，拨弄头发，借助旁边的建筑；随性自然的可以仿照倪妮的姿势，做一些大幅度的夸张姿势！我感觉这种挺难的，看似是抓拍，其实都是摆拍，还摆得那么自然。时间充裕的情况下可以多拍拍~

2.【牛班长】如果去日本的话，你最推荐什么？

【怡妞子】去日本一定要去体验花魁！店里的衣服租的，特别奢华！像龙袍一样，买的话一套下来至少人民币几十万！
花魁套餐的专用装饰花会根据不同季节进行更换，春天是樱花，夏天是绣球花等等……我樱花季去的，就是樱花背景哦~

3.【牛班长】如果让你选一个充满童趣的旅行地，你会选择哪里？

【怡妞子】和海豚亲密接触的水上乐园！新加坡旅行必游！来这里摸摸海豚，和海豚亲亲抱抱举高高，也是很多游客来新加坡一个很重要的原因！
在这里工作人员会给你讲解很多海豚的知识，如何分辨海豚公母，海豚如何呼吸，还有跟海豚的各种互动，学到了很多！也特别特别开心！大人小孩都可以参与~

4.【牛班长】年底了，小仙女们会选择去香港买买买，你会推荐大家在买买买的间隙去哪里玩呢？

【怡妞子】香港的哈利波特魔法商店！店里卖哈利波特周边，电影迷可以来这里看看全是正版商品。
老板要贴心的准备了拍照打卡区，眼镜、帽子、魔法棒，黑色的魔法袍长的和短的，还有小围巾。
这家店最近太受欢迎，老板本来要关门后来决定到2019年2月关！想去的快啊！

5.【牛班长】天冷了，国内有没有非常适合的雪景推荐大家去的呢？

【怡妞子】内蒙古根河的布冬霞猎民点，部落以展示敖鲁古雅鄂温克原生态民俗为特色，是根河市特色民俗家庭游接待户，是呼伦贝尔优秀家庭游接待户。部落由两户猎民组成，分别为猎民肖良柱和肖良库哥俩，女主人张卫东和布冬霞都是纯正的敖鲁古雅鄂温克。
他们放养着80余头驯鹿。在这里可以看到纯正的鄂温克民俗、吃上特色鹿肉串、列巴、蓝梅果子酱、喝一口猎民自已泡制的白酒、品一口肝复、灵茶，听一首女主人动听民歌和驯鹿来一个亲密接触，真正地体会一下森林民族敖鲁古雅鄂温克的纯净生活。
游客可与驯鹿近距离亲近，亲自喂养驯鹿，与驯鹿合影，感受驯鹿的温顺神秘；在这里让你做一天真正的敖鲁古雅鄂温克人。
