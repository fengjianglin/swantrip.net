【天鹅之旅首发】沉醉在浪漫的花海中，中国首届郁金香花博会等你来

游记 摄影 精品酒店 美食 自然奇观 赏花 2018/12/26
 看到郁金香，喜欢李白的人会说起他的诗句“兰陵美酒郁金香，玉碗盛来琥珀光”，的确，一年四季盛开的花有各的美丽，但这次葵园盛大的郁金香展则能让你充分感受到郁金香的魅力。
 
 含苞待放的郁金香花蕾娇美可人
而已经绽放的则花朵饱满、色泽艳丽
 百万葵园12月就绽放的郁金香，可以说是全球最早开花的花展，本次展出的郁金香全部由荷兰引进，30多个不同的品种。来到现场的朋友们可有福啦~
 
 今天刚好是周末，来到百万葵园游玩的亲子家庭也很多，到处都是欢声笑语。园子门口的小何边上种满了柳树，你能想到这时候已经是南方的冬至了么？不要奇怪，在百万葵园就是有那么多的美色，让你以为置身于春天里。
 百万葵园不仅适合情侣出行，更适合小孩子来玩，园区内处处都有小朋友喜欢的玩偶雕塑，如果一个个拍照估计都要花上一天的时间。
 
 进入园区你才会发现，原来花之恋酒店的另一面有这么美的花田呀，甚至还可以乘坐小火车围绕其中名，无论大人小孩，能够坐着充满童趣的小火车穿越，一定是一件非常美好的事情吧。
 
 平时在葵园还会上演有趣的表演，喜欢的朋友可以在此观看。
 
 
接着，我们穿过各式各样的花圃，来到了今天的重点~“彩云飞”。”“彩云飞”设计手笔出自"莫伯治工作室"，是葵园10年以来投资最大的一个登峰造极之作，今天，我们就要在这里欣赏到众多的鲜花啦。早在2004年百万葵园是华南地区首次举办郁金香展览，经过十几年磨炼，现在用意想不到的时尚与芝术融合一体呈现令人陶醉旳浪漫。
 一走进来就有种来到大温室的感觉，头顶还有一簇簇蓝色的小花往下垂，格外好看。大厅的设计独具匠心，挑高开阔的屋顶，阳光铺洒，厅内处处是布景，样样是道具，相机随便一拍都是美景。大厅的气温全天保持在18-25℃之间，不管室外有多冷，置身其中都会令人产生“整个世界都成了春天”的错觉，是携带一家老幼前来赏花观景的最佳之选。
 一个个悬挂在天花板的花挂，就好像是让整片天空都布满花的感觉，真的美到让人无法移开视线，和以前赏花的习惯不同，以前都是看着地上遍地缤纷。现在是天与地连成一气，真的美到无法无天啊，连我这个平常没那么爱花草树木的人，都兴奋的不得了，如果是平常就爱拈花惹草的人，来到这里真的就好像到天堂了吧。
 据说这里创意灵感源于日本花鸟园，日本美学是植物美学，"彩云飞"也不例外，加上独具匠心的设计，让世界各地的鲜花凌空飞越24米的跨度，在15米的高空中飞舞。
 这次花博秀创造了“天地合一”的效果，地面铺陈了数千棵粉红色的郁金香，一看就少女心爆棚，在半空亦垂吊着来自世界各地的鲜花与之相辉映，如来自南非的蓝雪花、美国的紫云牵牛、巴西的龙翅海棠，旨在多维度、多角度地展示鲜花美态，让游客与花海相拥。
 成片的郁金香，漂亮的大风车
你是不是感觉到了
美丽的“风车、郁金香王国”
-----荷兰？
 有“世界花后”美誉的郁金香原产地为荷兰和土耳其，它的花期为每年的3-4月。那么，百万葵园是怎样让郁金香在冷冬时节逆开花的呢？
 原来，为能让这次展览顺利举行，在数月前，葵园的园艺师们就在育苗基地全力奋战了。高级花艺师宫益涛表示：郁金香属于北方花卉，若想在南方种植，得选择处理过的种球。百万葵园根据开花的时间选择五度处理球，接着把它放在冷库里让其生根，出芽之后，再放到温室里正常养护。
让郁金香逆季开花，在花卉界历史上还是头一回，堪称奇迹！整个过程还得时刻根据天气变化来控制温度，才能让花开得不早不晚。
“赏花”的花艺展你就去得多了，“收花”的花艺展你又见过没？在花博秀期间，百万葵园送出超过十万株郁金香，平均每天有300株花送给爱美的女士，高质的花盆还让人舍不得丢弃。
 花博秀以花卉、文化和艺术为元素，缔造一场集观赏性和趣味性于一体的艺术盛宴。在华美的旋转楼梯旁，还摆放着一排色彩斑斓的英国卫兵，可谓是一个拍照的好景点呀。
 本次展出的郁金香全部从荷兰引进，涵盖了30多个不同的品种，种类丰富，花色多样。不同姿态的郁金香均令人沉醉，含苞待放的花蕾娇美可人，而已经绽放的则花朵饱满，色泽艳丽，优雅高贵的气质恰如一位风姿绰约的佳人。
 
 这一次展览围绕郁金香打造的主题场景多达三十个，有用郁金香做成的“爱的公式”：只等你带上最爱的人来；欧式风情的花篮：精致的热气球造型花盆把郁金香带到了半空中，挂在粉色小木屋栏杆上的花篮
 镶有一万颗施华洛世奇水晶的高定青花瓷礼服，由著名服装设计师郭培所创作，与现场的郁金香相映得彰；在浪漫的婚礼舞台上，一层层的阶梯铺满了不同品种的郁金香，步步生香；还有用郁金香串成的门帘，贯穿其间，恰似“一帘幽梦”。
 除了主体大场景，还有全手工制作的摩天轮、抓娃娃机、书屋、花房等迷你道具打造的微场景，搭配郁金香，实现艺术的跨界融合。
 你现在看到的一盆盆郁金香已经是剪下来插在水中的了，刚开始的时候花骨朵儿还是弯着腰的，但是过不了多久，鲜花就吸饱了水分，纷纷挺立起来，非常的神奇~
 捧着鲜花合影，绝对是分分钟出大片
 连扶梯都雕刻着郁金香花纹的艺术大厅，与郁金香有关的细节无处不在。现场看到有不少美术生在写生，想必这一场郁金香花博秀如此新颖动人，一定能激发出他们的创作灵感吧~

 置身花海中，仿佛拥有了整个春天
  怒放的郁金香，承包你的整个冬天！
 本次展览以花卉、文化和艺术为元素，缔造一场集观赏性、趣味性和知识于一体的艺术盛宴，也是非常适合亲子家庭出行的。
不要怀疑你的眼睛！
来到这里感受醉美花海吧！
景美+人美，怎样拍，怎样好看…
 
 带着小朋友认识各种鲜花的品类，细嗅花香，也是一次非常不多的亲子互动机会。
 
 
 
 
 现场有超级多小孩喜欢玩，喜欢拍照的场景
 
 
 
 
花博秀展览期间，葵园景区内还有百万种正值开花期的鲜花，如葵花灿烂暖寒冬、薰衣芳香醉人心、玫瑰深情爱永恒、超级海棠红透天等等。接下来我们来到了薰衣草花海，这里地中海  风格的建筑配上梦幻的紫色薰衣草，仿佛身处异国。
 成片薰衣草的开放时有如蓝色的波浪，壮观而神秘。每当微风轻轻吹过，一整片的薰衣草花田，犹如深紫色的波浪层层叠叠地上下起伏着，超级美丽。浪漫的紫色花海，醉人的花香。你，怎么能错过呢？ 
 浓郁的薰衣草显示着自然赋予的色泽，美丽且恬静。阳光下，当人们置身于薰衣草的清香中，便会迎来久违的欢愉与自在。不管什么季节，这里都有一片花海，不同的花卉为不同季节代言，这就是百万葵园的魅力所在啦。
 薰衣草花田充满了美丽浪漫的气息，下面就带大家看看萌宠，保证能把你逗得乐哈哈。
 
 看完可爱的小狗表演，我的目光又投向了远处的白鸽，一起跟我走过去看看吧。
 白鸽历来是和平与幸福的象征， 纯白鸽广场 纯白鸽广场 是人类忠诚的信友，设在入口处的白鸽广场给远道而来的嘉宾们增添了一道亮丽的风景线。
 鸽子的感情专一，是一夫一妻制。公鸽、母鸽共同筑巢。和原本象征着爱情的鸳鸯差的可远了，这些纯白色的鸽子才是爱情的信使~
 看过了郁金香和薰衣草，我们自然是要来看看葵园的这些向日葵啦。这里的葵花常年开放。葵园共引进了来自18个欧洲、日本等观赏性向日葵品种，并优选培育出一批适宜亚热带种植生长的品种。
 向日葵又名“太阳花”和“太阳神”，它象征着阳光与希望，也寓意着一种无言的感恩与爱。看过了百万葵园的众多葵花，相信你的心情也会变得如同葵花一般，充满阳光~
一轮葵花向日开，粒籽密盘甜睡酣。数不清的向日葵竞相绽放，这里俨然变成金色的海洋。
 看过了园区里的那么多种鲜花，你是不是也已经仿佛踏进了春天里呢？其实百万葵园还有多种玩乐设施，第八空间动漫港、逗趣松鼠全接触、葵花丛林过山车、薰衣风情海盗船、玫瑰三弄弹跳机，绝对让你流连忘返。
 很多情侣来此合影留恋
 各式各样的童话城堡，浪漫的感觉让人迷醉
 
 
 
在花海中，小孩子玩的都非常尽兴，非常疲惫了，结果看到有一个动漫手办馆一下子又兴奋了起来
这里可以看到非常多可爱的米老鼠，还有当前非常热门的漫威动画人物。
这些精美的手办系的小孩子都不转睛驻足不走了呢
在百万葵园的鲜花动漫港里，有无数画面与情节，让人恍惚间，置身于一个没有忧愁的世界，许多我们儿时的梦想，心灵深处那遥不可及的奇妙世界变得触键可及。
 
 这一次是我第二次来到百万葵园了，百万葵园花之恋酒店的客房色彩斑斓，而且每一层楼都有不同的主题，这样多姿多彩的酒店，住一次怎么能够?这不，趁着中国首届郁金香花博会的契机，我们一行人便入住了百万葵园的郁金香客房。
  光是听花之恋酒店这个名字就觉得非常的亲切，我们仿佛进入的是一座游乐场，像国王的城堡，里面住着童话故事里的王子与公主，不仅高端大气上档次，同时也充满了童趣与浪漫的气息。
郁金香主题房的配色以橙红色为主，最最让我惊奇的是，房内各处细节都能体验郁金香元素，与别的楼层房间区分开来。仔细留意房间内拜访的都是真的郁金香呀，睡了一晚上之后，惊喜的发现它们都盛开了，这样的喜悦应该会让每一个住客都动容吧。
 傍晚时分，看着屋外美丽的日落，远处的鱼塘好似一片海洋，有种烟波浩渺的美感。
殊不知，到了夜幕降临的时分，这里的景色又会呈现出另外一番模样，你期待么？ 
 【郁金香房不同于上次入住的樱花套房，多了一丝明媚的色彩，显得更为欢快】
 冬日里入住百万葵园的花之恋酒店，四处都是春意盎然的样子，仿佛春天就要来到了呢。
 枕头的细节之处也做的非常到位，这刺绣非常清晰优雅
 参观完色彩缤纷的客房，我们来到一楼的自助餐厅，用过餐以后，你就会知道为什么那么多客人喜欢一来再来了，留住客人的，不仅仅是优质的客房，还有这花之恋酒店的餐饮。
在花之恋酒店的自助餐厅，甚至专门为小朋友开辟了一个儿童食物区。
在自助餐厅，总是有一些美食成为人们的心头大爱，我到餐厅以后就明显感受到了下面这道美食的热度！好多人都询问葵花鸡在哪个位置可以吃到，甚至还有人为找不到而闷闷不乐。但是不用担心，葵花鸡都是无限量供应的，而且位置也非常醒目哦！葵花鸡不是“新品种”而是以喂养鸡的独特饲料而得名。它是以新鲜的葵花盘、叶、杆等炮制而成的饲料喂养的，甚至饮用的水都是用葵花杆榨成的汁。
餐厅的搭配都是以亮色为主的，整个配色很明亮，也充满了童真。
葵花鸡海鲜自助餐
 在烧烤档可以选自己喜欢的食材让厨师现烤，也可以拿现成烤好的。
 海鲜自助当然少不了吃鱼生，你可以选择现切或者直接取走，味道都很鲜美。这里的美食琳琅满目，荟萃中西的食品，在这里，绝对让你享受到舌尖上的盛宴。 
 用过晚餐，还是非常值得到户外散散步的。晚上的花之恋酒店与白天截然不同，绚烂的灯光，迷人的夜色，更显浪漫。吃过自助餐以后当然是出来散步啦，夜里的点点灯火把百万葵园打造的如梦似幻。无论是星光点点的大片花圃，还是笼罩在粉色氛围下的秋樱，把“浪漫”二字诠释得淋漓尽致。
 双旦期间来百万葵园，温馨浪漫的气息更是充满了整个园区，无论是酒店内的圣诞装饰，还是户外的灯饰，梦幻般的场景让每个来到百万葵园的游客都倍感欢喜。
 装满配饰的圣诞树
 走上这条绚烂的花桥，就可以走到前方超美的花圃上方，这里安装了数以万计的灯饰，好像置身于星辰之上，梦幻极了。
 
 面对如此震撼的美景，想必你也心动了吧。
行走路上我们可以看到非常多世界地标，他们都是用灯光打造的，梦幻无比！比如  巴黎  铁塔或是伦敦  眼，还有发着光的南瓜车，都会让人想要坐上去一次。
 看过了花之恋酒店外围的点点星光，我们回到温馨的房间内，躺在舒适柔软的大床上，安稳的睡上一觉。睡得格外香甜~
