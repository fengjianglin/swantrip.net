前言

现在家里的亲戚都有了各自的还小，我们一致觉得小孩和小孩之间一定要经常有互动，因此家庭旅游一直没有间断过，时而小规模，时而大规模，这样家庭成员之间的关系会更加融洽，大家也都很享受家庭旅行的乐趣。从今年的国定假期来看，五一小长假是没有了，所以其他几个小长假就更加弥足珍贵，元旦期间，组织了一次从上海到常州的两日游，不带老人，只带孩子，之所以选择常州，一是因为上海到常州的距离不远，3小时内可以到达，二是常州的路况不错，非常适合自驾，三是常州的淹城野生动物园非常适合年龄小的孩子游玩。 

关于交通

一大家子人出去旅行，最方便的肯定是自驾，上海到常州距离大约有180公里，最方便的线路就是走G2京沪高速转常台高速，正常行驶时间约为2小时30分到3小时，高速公路过路费为80元。常州市区内的停车场费用比上海便宜多了，就拿我们去的江南环球港来说，停一天也只需要5元，去淹城动物园，停车费是免费的。
假如不是自驾游，从上海到常州有高铁，有到常州站的，也有到常州北站的，时间约1小时左右，二等座票价为74.5元。常州市区内交通就要选择出租车，毕竟带着孩子，不带孩子的情况下，可以乘坐公交车。  

携带物品

亲子游旅行要带的东西比较多，特别是孩子还很小的时候，出去旅行携带的物品就要分两部分了，一部分是自己的，一部分是孩子的。我们自己的那一部分，需要带换洗的衣服、洗发水和沐浴露、牙膏、牙刷（自己带，环保）、毛巾、睡衣睡裤、充电宝、身份证、雨伞等，相对来说还是比较简单的；孩子那一部分需要准备的物品就多了，现在也有经验了，会带一个小的行李箱，里面基本都是放孩子的用品，我们是出行两天，孩子换洗的衣服要带2套、袜子、备用鞋子、睡袋、尿不湿、奶瓶、奶粉、日常D3、鱼肝油、保温瓶、辅食搅碎器、湿纸巾、柔纸巾、PP霜、润肤油、毛巾、婴儿车、玩具、被子、床单等等。这些东西基本可以放满一个20寸的行李箱，还会携带一个母婴双肩包。

D1：梅村服务区

原本我是想上海到常州也不是特别远，直接开到那里再集合吧，但是最年长的哥哥说在一个服务区碰头，一起吃个午餐，从上海到常州，中途只有一个服务区，就是离常州市区60公里远的梅村服务区，所以我们就决定在梅村服务区集合，顺便在服务区吃个中饭。以前经过这个服务区的时候没有觉得这个服务区规模有多大，但实际让我有点惊讶。这个服务区的停车区域面积非常大，进入服务室内区域，琳琅满目的餐饮商家和纪念品商家，肯德基、面包店、小笼汤包店、炒菜店应有尽有，规模已经接近于杭州湾大桥两侧的五星服务区的水平了。 
大部分游客会看中入口处的几家店铺，那几家店铺等候的人是最多的，假如不想长时间等候的游客，就往里面走走，可以看到一家“驿客餐厅”，这个餐厅像大食堂一样，是自己拿菜结账的，就餐面积非常大，座位肯定是有的，价格不是很贵，菜品的种类繁多，味道也还可以，支付的时候，只需要把餐盘放在扫描器上，就能立刻知道价格，结账的速度也快，这样很简单的一顿午餐就不需要浪费很多时间，才能节省更多的时间花在游玩上面。 

D1：常州香格里拉大酒店

常州香格里拉大酒店位于常州武进区西湖路2号，是一家五星级酒店，位于常州武进开发区，走G2京沪高速转常台高速后，在常州南高速口出高速，酒店离高速公路出口非常近，出高速后，转个弯就到了。酒店的停车场主要是地面停车场，地下停车场虽然也有，但是车位数不多，所以一般不会去停地下，酒店内停车是免费的。我觉得常州香格里拉大酒店很适合亲子游的酒店，酒店大堂很亮堂，右侧是住客休息区，休息区旁有一个室内小动物区域，养着几只鹦鹉。酒店的一层有咖啡吧、日本料理店、酒店的餐厅和一个小型的儿童游乐区，三层是酒店的水疗中心和健身中心，酒店餐厅旁边有一个小湖泊，湖内可以看到鸭子和天鹅，在酒店的植物带，还养殖者孔雀和鸽子，这里像是一个小小的动物世界一样，而小朋友会很喜欢这种小动物。综合看来，酒店内的娱乐设施很一般，但是动物区非常适合小朋友。 
Tips: 酒店办理入住的速度一般，办理入住的时候需要支付押金，押金500元/间，建议使用信用卡，退房的时候即时解除冻结，很方便。酒店的婴儿床是免费的，但是不能预订，是先到先得的。 
我们预定的是大床房，房间内有一张六尺大床，一个两人沙发，一个单人沙发，一张写字台。硬件设施有冰箱（里面的饮料是收费的）、吹风机、热水壶，中规中矩吧。卫生间的面积比较大，有淋浴室和浴缸，洗澡的时候水量很大，很舒服。因为我提前告知酒店需要婴儿床，所以酒店还提供小朋友的沐浴露、牙膏牙刷，很贴心。我在预定之前对比过几个平台，天鹅之旅的价格算是性价比最高的，一晚的价格在500-600之间，所以我就在天鹅之旅预定了，而且体验非常好。

D1：江南环球港

江南环球港是江苏省常州市的一个新的地标，是一个集购物、餐饮、娱乐为一体的一个大型游乐购物体验区，位于常州市新北区通江中路598号，并非在市区，乘坐当地交通并不方便，对于自驾游的游客来说比较方便，沪蓉高速常州出口出来就是江南环球港了。我们从常州香格里拉大酒店自驾前往江南环球港，大概是需要40分钟的时间，是走高速的，高速费用为5元，如果不走高速的话，可能需要1小时左右。江南环球港的内部建筑很奢华，似乎是仿照澳门威尼斯人度假村来装饰的，内部给游客一种很清新、自然的感觉。环球港的面积也非常大，有环球广场、太阳广场、花园大厅三个大型中厅，第一次去的游客八成会在里面绕晕。江南环球港在装饰上的精华位于顶层，近距离的看到玻璃穹顶周边一圈都是具有欧式风格的油画和雕像，有种欧式宫殿般的感觉。顶楼靠近美林菜谱旁边，有一个透明的玻璃桥，透过玻璃桥，可以看到下方的儿童游乐区。
Tips: 江南环球港的面积非常大，有地面停车场和地下停车场，通江中路正在修路，所以江南环球港的停车场要从环球港邮轮酒店那里进去，不要被导航误导了哦，可以定位邮轮酒店停车场的，江南环球港停车费用很便宜，8小时内5元。
但是要说到江南环球港最吸引人的地方，不是装饰、不是美食，也不是购物，而是江南环球港的游乐项目，其中最有代表性的就是世界港口小镇，最有名的项目就是江南环球港顶层露天区域的摩天轮。我们去的时候正好是下雪，摩天轮存在停运的可能性，所以工作人员就建议我们微信购票，如果停运的话，是可以退款的，一成人是60元，身高120以下的孩子是免费的。大概在等待了将近一个小时候，我们终于乘上了摩天轮，比较特殊的是，摩天轮的众多舱位中，有12个舱居然还带有K歌功能。摩天轮转一圈大概是20分钟左右的时间，夜观常州下雪的景色也还挺有意思的。

D1：美龄菜谱

家庭游在餐饮选择方面是比较纠结的，因为人多，某些餐厅可能没有包房或者大桌，我们这次出行一行有10个成人和5个小孩。那么多人，在用餐方面还真是不方便，好在找到一家餐厅，餐品适合孩子，也有环境比较好的包房，这家餐厅叫做“美龄菜谱”，从餐厅的名字来看，就是江浙本帮菜的口味，美龄嘛，当然是想到宋美龄啦。这家餐厅在上海也有一家，我曾经去品尝过，味道的确是不错。从地下停车场有直达“美龄菜谱”的电梯的，非常方便。店内的装修风格有种20.30年代的感觉，比较复古，店内的墙面上挂有一些旧社会的照片用来装饰，还能看到一些复古的家具来装饰呢。整个包房空间还挺大的，一半是餐桌，另一半是休息区，顾客的座位都是沙发样式的座位，很柔软，坐的非常舒服。
推荐的美食有香芒汁熏鱼、红烧肉、南瓜慕斯等等。香芒汁熏鱼就是江浙本帮菜中的油爆熏鱼，只是里面含了一些水果的清香口感。红烧肉是本帮菜中的代表作，浓油赤酱，肥瘦相间，还能加红烧蛋。南瓜慕斯的样式比较独特，口感很清香，适合小朋友吃。我们还点了油爆虾、素爆鱼、菠萝古老肉、鸡汤等，味道中规中矩。人均100元的费用性价比不算特别高，但就餐环境还是挺优雅的。
一个小插曲，从江南环球港返回酒店的时候，由于是雪天，高架上出现十几车的连环撞车，在高架上一动不动的堵了一个多小时，游客都下来看雪景了。

D2：常州香格里拉大酒店早餐

“早晨起来,拥抱太阳,让身体充满,灿烂的阳光”，虽然元旦期间常州冰天雪地，但是早餐用完后，相信一定可以充满能量。常州香格里拉大酒店的餐厅位于一楼，每间房间早上有两人份额免费早餐可以享用，如有超出的住客，将收取一人124元，6岁以下的小朋友是免费的，6岁到12岁之间是半价。12岁以上的孩子就要全价了。进入餐厅并不是用早餐券，是报房间号。餐厅给我第一印象是区域很大，看上去，仅仅是看上去餐品应该挺多样，但实际早餐的餐品并不是很多，装饰的内容大于实际的餐品内容。一些常见的早餐还是有的，例如面包类、饮品类、面条、包子、粗粮、粥、炒菜、煎蛋、松饼等等，但要说有什么特别的地方，还真说不出来，有点不符合五星级酒店的派头，早餐的内容我觉得也就四星级酒店的水平。有可能的话，尽量找靠窗的座位，因为旁边就是常州香格里拉酒店内的小湖泊，品尝早餐的时候还能观赏外面的景色，很惬意。

D2：常州淹城野生动物园

常州淹城野生动物园是江苏省常州市春秋淹城旅游区的重要组成部分，动物园的面积不是特别大，大概总面积在2000亩，比上海野生动物园还要小，但是可玩性要比上海野生动物园多，原因一，淹城野生动物园是有自驾区的，自驾车是可以开进去和动物近距离接触的；原因二，淹城野生动物园内的表演很多，每个场馆的表演时间都是错开的，而且当中有间隙，这样游客看好一个表演后，有足够的时间去看另一场表演。因此淹城野生动物园是一个非常适合亲子自驾游的景点。
我们在常州入住的酒店是常州香格里拉大酒店，从常州香格里拉大酒店自驾前往，有七公里的路程，大约行驶时间20分钟。淹城野生动物园的门票价格为成人180元（网购票170元），儿童票90元，自驾车票200元（网购票190元），自驾车票是不包括驾驶员的，例如一成人自驾景区，需要支付成人门票+自驾车的车票，车内可以随便坐多少人，毕竟是在景区里面，人再多，交警也不会来管你。自驾车入口和非自驾车入口不是同一个入口，在进入景区后需要识别入口。
不是自驾的游客就不能近距离的接触动物了？显然不是，在景区内有观光小火车，门票包含小火车的费用，无需额外付费，小火车上的游客人数满了就会发车，如果不满的话，也是等到指定时间发车。小火车经过的区域和自驾游经过的区域是一样的，只是停留的时间比较少，互动性不足。
自驾车和小火车的车辆行驶区域，我觉得可以划分为三个区，第一个是食草动物区、第二个游客喂养区，第三个区域是猛兽区。食草动物区和猛兽区是有围栏或者河道隔开的，真正可以近距离接触动物的区域是是游客喂养区，可以接触到的动物主要有鹿、鸵鸟和羊驼。进入园区，首先看到的是食草动物区，动物的种类不多，可以看到有野马、矮马、羚牛等动物，很多游客会停靠在路旁下车喂食，但不需要停留很久。 
车辆行驶到喂食区后，几乎都会停留一会，让小动物们靠近，然后去摸摸它们，给他们喂食，没带食物的游客也不用担心，这个区域是有出售胡萝卜的，10元一袋。喂食区中，鹿的鹿角是被锯掉的，为了不伤人和破坏车辆，挺可怜的。鸵鸟的嘴巴也是比较危险的，我们经过喂养区的时候准备了一些面包，专门用来喂小鹿的，当看到鸵鸟过来的时候，我们马上把车窗给关上，鸵鸟的嘴巴的确是有点吓人。到了喂养区的后半段，出现的都是体型比较大的鹿，头能伸进车厢，也比较吓人。而乘坐小火车的游客，只会经过鹿区和羊驼区，有鸵鸟的地方不会经过，毕竟鸵鸟的嘴巴容易伤人。羊驼区域的羊驼数量也很多，不过好像有点高傲，你不给它们喂食，它们是不会理会你的。
最后一个区域是猛兽区，淹城动物园内猛兽区的动物有狼、黑熊、狮子、白虎等，其中狼、黑熊、白虎的数量非常多，景区内提供喂食服务，可以在工作人员那里购买活鸡或者肉块。活鸡的价格是50元，我们经过狼区的时候，就看到一个顾客买了一直活鸡，准备投放的时候，群狼都集中到了一起，扔过去的后，场面有点血腥，1秒内，一直活鸡连骨头都找不到了。带孩子去的游客就尽量避免这种画面吧。相比之下，黑熊区要比较和谐，游客向他们投食的时候，他们还会站起来标识“友好”，就和马戏表演里面的一样，很听话的感觉。
常州淹城动物园的成人票票价是180元，在没去过之前，我会觉得门票价格略贵了，但是去游玩过后，你会觉得这个票价贵是贵的有道理的。原因一，门票是包含景区内的观光小火车的，乘坐小火车可以和部分动物近距离的互动。原因二，景区内有好几场表演，门票都是包含的，其中最亮眼的表演是森林大马戏，大马戏表演不是随便忽悠忽悠人的。表演的时间可以关注淹城动物园的官网。大马戏表演的时间是40分钟左右，表演者们除了中国的马戏精英，还有来自俄罗斯、白俄罗斯、乌克兰等多国马戏精英。森林大马戏表演分为几个专场，第一场表演是马术表演，表演者们骑在马上进行各种高难度的动作，测骑、双人叠高骑、三人叠高骑、瞬时的下马和上马等等，让观众都喂他们捏了把汗。第二场表演是动物表演，首先出场的是河马，体型巨大的河马会慢慢走向舞台中央的平板桥上，随后小狗、小熊、驴、猴子、小矮马、小猪、小老虎会出来和大家打招呼，小猴的戏份是最足的，会在河马的身上进行吊环表演。第三场表演是高空特技表演，两位表演者在空中不停的“飞舞”，到了高潮部分，舞台上的会同时燃放烟花，渲染效果。第四场表演是魔术表演“大变活人”，运用了烟雾和激光效果来渲染气氛。最后一场表演，我叫它“无敌风火轮”，表演者在旋转的轮框上走动，最后的时候，表演者甚至还蒙上眼睛，在轮框的顶部走动，太精彩了，所以淹城动物园的这场大马戏表演千万不要错过哦。 
常州淹城动物园内的动物，不仅仅是在野生动物观光区才有的，景区内还有火烈鸟馆、大熊猫馆、羊驼馆、涉禽馆、河马馆、猩猩馆等，最受欢迎的应该是是国宝大熊猫馆，大熊猫馆有室内和室外区域，馆内有两只大熊猫，我们去的时候，一直大熊猫正在吃竹子，另一只大熊猫则准备睡觉，睡觉的那只大熊猫真的是逗乐了所有的游客，抱着一根树干，睡觉的姿态太萌了；而正在吃竹子的大熊猫吃到一半，走到室外区域去了，游客也跟着出去，没想到，它只是出去拉了个粑粑，又回去继续吃竹子了，这两个大熊猫太喜感了。
常州淹城野生动物园的表演，除了森林大马戏，还有丛林熊风、动物大悦宾和非洲印象。林熊风表演，顾名思义，主角是黑熊，表演时间大概是15分钟左右，黑熊和它们的驯兽师会进行投篮、打扫卫生、溜滑轮、骑自行车、爬杆子、走钢丝等表演。
动物大悦宾在一个小型剧场里举行，观众的座位也不错，这里出现的都是比较小型的动物，例如矮马、羊、羊驼、小狗、猴子、鸭子、袋鼠、鹦鹉等动物，驯兽师们会和观众互动，例如请几位小朋友去骑小矮马，每个动物出场的时间不多，节奏挺快的，表演时间大约是20分钟。带小孩子去看表演是最轻松的了，他们看的开心，家长也不会很累。
看完动物大悦宾后，这次旅行也结束了，带小孩子出去旅行，景点不能安排的很多，行程不能安排的很满，要以休闲游的方式进行，大家都玩的很开心。
