一路王下去---一个处女座的湖北自驾老友记！

游记 主题乐园 人文游 特色表演 自然奇观 自驾游 2019/04/24

真正的老友，是一生的感动！

人活着，圈子不要太大，容得下自己和一部分人就好；朋友不在于多少，自然随意就好。好朋友不一定天天见，但一定会常想念；真情谊不一定浓似酒，但一定会心里有。
很多人都看美剧都是从《老友记》开始的,我们都幻想着有这样一群朋友，像瑞秋一样独立 ,像 罗斯 一样长情 ,像钱德勒一样蠢萌 ,像莫妮卡一样努力 ,像乔伊一样仗义 ,像菲比一样善良 ,像他们一样相爱！ 之所以称之为“老友”就是天大的不满，也不留隔夜的仇，I‘ll alway be there for you，这大概就是真 友谊 吧。
而我的这篇“老友记”是几个高中相识到现在近二十年的小伙伴，其中一段的旅行故事！

第一天，说走就走，结伴而行！

全程依然手机拍摄，其实有相机的，就是都过去三年多了，在孟老板单反里的照片我到现在没拿到，原因是孟老板说十年前他们在仙壕的相机里面的照片还没拿到呢，也是够够的了，哎！！大伙将就着看吧！
我们几个年初就商量着要找个地方转转，一直纠结到快五一，小伙伴们商量着利用小长假时间自己开车吧，巧合仙壕五一要去 武汉 吃喜酒，就这样，我们四人成行，占仙壕的光去 大湖 北溜达溜达。。暂定的是 宜昌 荆州 和 武汉 。先到 宜昌 ，30号下午开始出发！
在高速上自拍，当时发的朋友圈说：老大都是坐后面的，今天低调，链子不带了，哈哈哈哈！下面评论就是一大堆怼我的。。。
因为我们是中午就出发了，从 南京 到 宜昌 有800多公里，停停歇歇要开到凌晨才能到，中途在服务区麦当劳拿出来垫垫肚子，给我们的食物先来个合影如何？
开了差不多4个多小时，差不多进入 湖北 界了，夕阳下的高速上，风景开始俊俏起来，可是此刻大伙又饿了，没办法，车子上有个最大的吃货---仙仙壕！就这样，一人一袋方便面，一根火腿外加一块鸡腿，这下能再坚持三个多小时了吧！
我们一路停了好几个服务区，的确路程比较远，而且特地出发早不怎么赶，再加上当时的小长假大家还没那么多意识去 自驾 旅行，恩！厕所的提示语还是挺有趣的！
就这样兜兜转转，路上还开错一段的情况下，终于在凌晨近1点我们到达 宜昌 ，行李都没去放下酒店，我们直接转悠找有没有夜市大排档，还不错，给几个资深吃货在酒店不远的小巷子里找了一块宝地（当时很少看攻略，都是随游）
停好车，找了家比较全的，先来一份烤鱼，生活在江边的 宜昌 人这鱼做的真不错！外脆内嫩，四个人除了孟老板不怎么吃鱼，我们三人把它吃的干干净净。
海鲜，烧烤，鸭脖，一应俱全，再加点啤酒饮料，伴随着微微江风，那生活真是美滋滋的，可能也是大家饿坏了（其实就是能吃，10小时内吃了四顿），光盘，打着饱嗝儿，结个账就去酒店入住休息了。。要不是时间太晚，真想再去江边走走，留点精神明天用吧，想不到第二天是暴走的一天。。

第二天：三峡人家，三峡大坝

五一的早晨，推开窗帘，好天气就好心情，看到 宜昌 段的长江水还算绿，也算惊奇，后来直到 武汉 ，长江水就黄了。。。把小伙伴们一一叫醒，吃个早饭准备上路了！
第一站：我们这代人小学课本都学过的：葛洲坝枢纽！那时候第一次听说经过的轮船是怎么进去闸门被抬升，下降。
经过西陵峡来到三游洞，三游洞的名字有两个典故，唐代诗人白居易、白行简、元稹三个人曾一同游过此洞，人称“前三游”；到宋代，苏洵、苏轼、苏辙父子三人也一同来游过此洞，人称为“后三游”。因为此行的主要目的地是 三峡 人家，所以就赶路继续前行，路上仙仙给我们来一段骑马舞！像不像鸟叔？
三峡 人家景区门口当时是有免费的停车场和转拨车的。
小伙伴们再次合影！
三峡 人家风景区位于 宜昌 市夷陵区三斗坪镇石牌村， 三峡 大坝和葛洲坝之间，面朝长江背靠 三峡 之一的秀美的西陵峡。因独特的地形地貌和特殊的地理位置，兼容了巴文化和楚文化，是有特色的风景和文化融合的景区。李白的“春水月峡来”，欧阳修的“江上挂帆明月峡”，陆游的“ 巴东 峡里最初峡，天下泉中第四泉”，还有现代大诗人郭沫若的“唐僧师弟立山头，灯影联翩猪与猴”，陈毅元帅的“ 三峡 天下壮，请君乘船游，西陵甲 三峡 ，忘返兴犹稠”都是说的这里！
看到这面国旗，有没有一种战狼2镜头的感觉？身后即是“明月湾”
三峡 人家风情项目又分为 水上 人家、溪边人家、山上人家和今日人家。实际上没有那么繁琐，只要记住 水上 人家和山上人家就可以了。
月影阁码头，爬山去山上人家及返回景区大门在此下船或上船。上山有徒步路，也可以乘自动扶梯，30元/人。过去索道可以直达山顶，现在扶梯只到石令牌。要想到山顶灯影石，还要爬一段山路。
来一碗甜酒粮！
对这些人造景点没啥太 大兴 趣，演出也就大概看了一会儿，爬这么高，主要是去看灯影石！四块兀立山巅的奇石，酷似《西游记》中唐僧师徒西天取经的形象，每当夕 阳西 照，晚霞映衬峰顶时，远远望去，它们仿佛灯影戏幕上的人物造型，惟妙惟肖，“灯影石”即由此得名。当年郭沫若先生游历灯影峡时，曾题诗赞叹：“唐僧师弟立山头，灯影联翩猪与猴”。
浩浩长江奔涌至石牌河段时，在这里向左急转弯110度,改向东去，形成一个巨大的月牙形，这就是“长江三峡 第一湾”的明月湾！
来到大诗人们纷纷作词吟诗的美景圣地，小伙伴们怎能不摆拍，不合影？！
30度的天爬了这么久山路，小伙伴们累了也饿了，一人搞碗儿两面吃吃,15元一份口味一般。坐在索道里填饱肚子下山。。。前段时间看游记说现在因为索道经营纠纷已经停运！
来到明月阁码头，准备乘船去水山人家！在龙进溪水与长江的交汇处，许多 三峡 人祖祖辈辈生活在船上，常年以打鱼为生，在 水上 流动，和风浪搏击，与渔船为伴，形成了“ 水上 人家”。
三峡 人家“一肩挑两坝，一江携两溪”，这里石、瀑、洞、泉……多种景观元素巧妙组合，山有山的伟岸，水有水的柔媚，洞有洞的神奇，瀑有瀑的壮丽，石有石的气质。
在近溪的一边，一道水坝已筑起，截住了清幽的龙溪水，使溪水抬高了1-2米，溪水碧绿，林木苍翠，斑驳沧桑的龙溪桥横跨其上，几只小渔船静静地泊在桥下，“龙溪桥下春波绿，惊鸿照影来”，如梦似幻，令人如痴如醉。
久违的古帆船、乌篷船安静地泊在 三峡 人家门前，后面一段因为塌方已成断头路！
溪水呈现出蓝绿色的宝石一般的色彩。
溪边少女挥着棒槌在清洗衣服，江面上悠然的渔家在撒网打鱼！
清凉的溪水泡泡脚，缓解疲劳，亲近大自然！
竹林芭蕉簇拥的木栈道。游览 水上 人家实际上就是在峡谷中沿着山体走一个U字。白墙青瓦石板路，小桥流水吊脚楼，枯藤老树喜鹊窝，机枪碉堡旧战壕，奇石溶洞古城堡，在您进入景区的一刹那，您的心将注定要长留此地，整个景区弥漫如仙境般的梦幻景致，这里的美让人心悸，这里的美让人窒息!
如果在过去，这里不失是一处世外 桃源 ，返程的路上发现这里也有大多运输船通过，希望尽量避免对这里的污染和过度开发吧！
基本不走回头路，离开时在胡金滩码头！稍作休整去下一站： 三峡 大坝！
因为考虑到第二天再去 三峡 大坝还要赶路 荆州 就比较匆忙，所以看着时间才3点多，去大坝大约1个小时车程，所以就临时决定今天就把大坝行程走掉，说不定能看到夕阳下的大坝（还真的看到了）
俯视 和平 视西陵峡！
这一座山峰像不像一个躺着的人？睡佛？我不知道它叫什么峰！
三峡 大坝旅游区拥有坛子岭园区、185园区及截流纪念园等园区，又叫俯视，平视，仰视三观 三峡 大坝，第一站是坛子岭，截流石，四面体单边长2.5米，重达23吨，由混凝土浇筑而成，是长江截流时的主要投抛料。截流四面体纪念石是实际使用的2倍。因四面体各角可分散江水的冲击力，抛入江底后不易冲走，可有效提高 龙口 封堵速度。先后在葛洲坝工程和 三峡 工程大江截流中立下汗马功劳，是 三峡水利公共的科技创新成果之一。
俯视 三峡 ，当时 三峡 大坝景区是收门票的，一起连景区大巴好像是155，现在据说只收大巴费用，国人凭身份证网上预约免门票！
又到了补充能量的时刻了，景区的东西不是一般的难吃，但是天气太热，降降温是必须的！
经过大坝的船都要经过船闸一步步抬升或下降，据说要1个多小时才能通过。。。
接着是平视大坝，此刻夕 阳西 下， 三峡 风景显得格外美妙！
将来这里还将有升船机，类似于电梯一样（2016年造成了）！
最后一处是截流纪念园，通过工程遗址展示、场景演出、幻影成像等多种表现手段和大量史实史料,全面展示长江截流的壮美场景,生动再现截流 成功 的历史时刻,为您揭示出截流壮举的伟大意义和深刻内涵。
工程车也是摆造型的好地方！！！
此刻夕阳在 三峡 大坝上方，无与伦比的美丽~
仰视 三峡 大坝！
夜色下的大坝！
结束返程，晚上继续找当地特色菜馆，这家馆子的鱼在当地很有名，就看门口排队的车就知道了。。。吃饱喝足，奔波的一天也结束了，休整睡去！~~

第三天：荆州古城耍关刀！

在 宜昌 住了两晚，酒店对面的江滩公园还没走过，我也不太习惯睡懒觉，其他三个小伙伴还在熟睡中的时候，我已经早起去江边溜达了，对面就是我们住的酒店！
江边的钓鱼人，很多很多！
江水不浑浊，鱼长的都是这么漂亮，哈哈！一直憧憬着自己老了，也能到处逛逛，钓钓鱼这种慢慢悠悠的生活状态！
时间溜达的也差不多了，回头和小伙伴集合去，准备今天的行程！
集合完毕，附近找了个早餐馆，在 湖北 吃的最多的就是面食，不过这豆奶口味真是一级棒，后面在 荆州 和 武汉 再找也没找到。。。吃饱喝足继续赶路出发。。。
荆州 离 宜昌 不远，车程大约2小时不到就到达，还好昨天先去了 三峡 大坝，不急不慢的开到已是中午，不离吃货本性，继续找吃的，听 湖北 的朋友介绍黄家塘的米粉最好，也顾不得这几天吃的最多的就是面食，还是想去尝尝！
一碗米粉，一杯绿豆汤，一块饼，那满足感。啊！
荆州 古城墙！~
我喜欢逛博物馆的习惯大概也是跟这几个小伙伴久了，到了古城，怎能不去看看 荆州 古城博物馆，这里面最有名的就是出土的“西汉女尸馆”可惜不能拍照！
到 荆州 ，怎能不去古城墙看看，去古城墙，怎能不去关羽祠，提到关羽也就不能不提到 荆州 ！
荆州 百姓为纪念关公，在卸甲山修建关羽祠，以关公的忠、义、仁、勇精神，教育后人“读好书、说好话、行好事、作好人”。
哈哈哈，看到这个时候我们非要拍下来发给此行没有来的“龟龟”
有三国特别蜀国文化的纪念馆必有 桃园 文化！哈哈哈！
横刀立马属关公！我也就合影留恋一张！
可是我们的胖胖孟老板，在商谈细节的重要性，他这身材，不提个刀，不值得到此一游！萌萌版的“关老爷”好像三国演义中的关羽成天也就一件绿色战袍，你赢了！
三人合影~~~
后来越看越觉得奇怪-----对啊，这哪是 桃园 三结义啊，这明明就是交通灯的三色啊~呵呵哒！
我们都是安静的宝宝！！有没有点致青春里面的场景！
因为要在晚上之前赶往 武汉 ，所以在大约4点多结束 荆州 的行程，出发去 武汉 。。。。
刚才还说我们是安静的宝宝，这就玩起来了！好吧，你们是真爱！
看到标志性建筑--黄鹤楼，好吧， 武汉 我们到了！！放下行李，我们即可出发前往觅食！
到了 武汉 ，怎能不去户部巷，到了户部巷，怎能不去蔡林记，到了蔡林记，怎能不尝尝热干面，虽然吃过就感觉到各大景点都一样，坑人，哈哈哈哈！
填饱肚子，我们去哪消消食呢？见多识广的仙仙说，要么去江滩公园转转吧，好啊，见识下大 武汉 的江景（见后才发现真不抵我大蓝鲸）
漫步在汉口沿江大道上， 从江 汉关到市政府办公楼（前 德国 领事馆），12幢风格各异的百年老建筑连绵排列开来，每一座建筑的形成，都会固化一段不可改变的历史，
“两江四堤八林带，火树银花不夜天”，这是一位诗人对 武汉 江滩美景的赞颂。在 武汉 三镇的水景中，武汉 江滩可谓是这座滨江城市中一道最美丽的风景。
深夜，已经有人放飞孔明灯，星星点点很是壮观，不过这种现象在 南京 的江边是禁止的！
在江滩公园溜达了1个多小时，吃货们又喊饿了，没办法，开到商业街，找了个馆子夜宵。。。
这么多，不得不说真能吃，好了，结束一天的奔波，回去睡了~~明天走 武汉 ！

第四天：武汉城里逛逛！

我们住的酒店在汉阳火车站附近，清早起来在房间窗台就能看到一列列火车进出。。
这家面馆在酒店后面巷子，口味还不错，最起码肉量是足的。。。
早上仙总没跟我们一起，他要睡个懒觉去参加同学的婚礼，我们三人就同行去了第一站：黄鹤楼！
黄鹤楼景区进门要坐一个景区电瓶车才能走到黄鹤楼脚下。。来，三人行自拍！
黄鹤楼对于大家再熟悉不过了，最熟悉的不过那句“故人西辞黄鹤楼，烟花三月下 扬州 ”名句了，对，我们不是辞而是 从江 苏而来！合影！
进入楼阁，就看到墙面上那只巨大的黄鹤！
黄鹤楼经历过多次损坏和重建，现在的位置也不是古代的黄鹤楼原址，去的时候正好也在修复。。。
黄鹤楼上远眺 武汉 长江大桥！
当年岳飞收复 襄阳 六郡曾多次来到黄鹤楼。。。。黄鹤楼三楼大厅壁画上，刻有岳飞披载盔甲，腰悬宝剑，左手扶剑鞘，右手欲拨剑，再现了当年岳飞登临楼头，缅怀祖国沦于敌手的大好河山，抒发“还我河山”的伟大政治抱负。
遥望中原，荒烟外，许多城郭。想当年，花遮柳护，凤楼龙阁。万岁山前珠翠绕，蓬壶殿里笙歌作。到而今,铁骑满郊畿，风尘恶。
兵安在？膏锋锷。民安在？填沟壑。叹 江山 如故，千村寥落。何日请缨提锐旅，一鞭直渡 清河 洛。却归来、再续汉阳游，骑黄鹤。----岳飞《满江红·登黄鹤楼有感》
离开黄鹤楼，来到著名的 武汉 大学牌楼下拍几张，也算到此一游，这已经是2013年翻新后的样貌！ 新建 位置位于八一路！
迎接回仙壕，准备午餐，附近某饭店来一顿正宗 武汉 鸭子的多重吃法，具体的。。。见图！
东湖因位于 湖北 省 武汉 市武昌东部，故此得名！
乘船游湖！其实东湖周边也有绿道，方便骑行！
磨山是东湖的重要组成部分，她三面环水，六峰逶迤，既有优美如画的自然风光，又有丰富的楚文化人文景观（磨山楚城）
散落在园林中的吊床也是东湖公园一景！
此行的最后一站， 湖北 省博物馆，东湖， 武汉 大学以及博物馆离的都不远，所以游览起来比较方便！
湖北 省博物馆的四大镇馆之宝为：越王勾践剑、曾侯乙编钟、郧县人头骨化石、元青花四爱图梅瓶。去博物馆很少拍照，所以照片有限，三分看七分听！
下午四点多左右准备回程，附近找了个麦当劳解决晚餐问题，最后去了当时还不算太火的周黑鸭，买了鸭脖和鸭脆骨！回程路上解馋！也算圆满！
