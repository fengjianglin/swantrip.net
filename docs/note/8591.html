海上漂流记，初登「海洋光谱号」

游记 摄影 购物 邮轮旅游 2019/06/10
序：光谱驾到
第一次的邮轮之旅从海洋光谱号开启，也从此迷上了邮轮之旅
刺激个性的甲板冲浪
轻奢极简的美味佳肴
纸醉金迷的夜生活
精彩绝伦的梦幻演出
一场场一幕幕，浮现在脑海中，爱上了这里风轻云淡的生活，挥挥手就能够得到的海风与蓝天。
就这样踏上了我的第一次邮轮之旅，开启了我的海上漂流记，
之前总是看到一些航海传说，今天终于可以前往真实的体会下。
一次放空自我，人与自然的海上盛宴即将开启。
旅行Tips：
出发地： 上海 吴淞口国际邮轮港，如有大行李可托运上船
登船手续：护照，登船证件打印版
穿衣：长裙，礼服，运动装，泳衣，帽子，眼镜，披肩，长外套
最重要就是防晒一定要做好，邮轮户外运动直晒的时间很多，所以防晒一定要做好。
购物：免税店
wifi：邮轮有Wi-Fi一个账号课免费绑定4台设备
美食：4楼5楼主餐厅，还有甜点店，14层的自助均为免费餐厅，房间内可以24小时拨打免费送餐电话。
提前下载Royal app，可以在上面预约游玩项目和一些邮轮上吃喝玩乐的信息。
初见光谱号
还未到码头，未见皇家 加勒 比光谱号，便已感受到了豪华与霸气的气息，码头两旁插满了光谱号的旗帜，用最大的热情来迎接这第一批客人，让邮轮上的每一位客人都有一种宾客如归的感觉。
夕阳夕下，落日余晖，光谱号在夕阳的映衬下，像是一位海洋王子，温文儒雅而又活力四射，来迎接即将要登陆的宾客们，带我们开启一场海上旅程。
华灯初上，光谱号上霓虹灯逐渐开始被点亮，耀眼夺目的灯光终于在夜色降临后，彻底爆发了，迷幻的灯光诱惑着你我的灵魂，看到这醉人的风景，有一种，就不醉人，人自醉的感觉，就这样一直沉浸在他诱惑般的美景中。
到达房间后，托运的行李稍后会直接送到房间门口，房卡在房间门口的卡槽内，光谱号上的房间也是超豪华版的，硬件软件齐全，房间设有阳台，如果喜欢安静，可以在房间内点杯酒，一个人看尽海的尽头，从日出到日落，从白天到夜晚，尽享一个人惬意时光。
拍照估计是女孩子的最爱了吧，邮轮上晚上的灯管特别棒，绝对是拍人像必打卡，这处走廊位于5楼270度餐厅的右手边，有个去舱外的门，从那里出来后，便是此走廊，最重要的是人超级少，可以在这这里尽情的摆pose拍照。
皇家加勒比光谱号甲板拍照打卡地
拍之前，小伙伴们一直强调要拍出有故事的感觉，眼神要深情迷离，不料，一路太欢乐了，眼里出不来故事，最后只能是换了风收场。
在顶层享受甲板上的惬意时光，正好赶 上海 上日落，海平面上的一抹夕阳，夹杂在天空和海平面之间，像是一道穿越时空之门，载着你我的梦想与快乐继续前进。此时，不需要过多的话语，静静地望着远方的海平面，享受大自然带来的美景。
一个人躺在甲板上的躺椅上，享受海洋带来的惬意时光，闭上眼再此晒个 日光 浴也不错 ，海风轻拂过我的脸庞，阳光毫无遮挡的洒在我的全身，一个与大自然的亲切接触，就这样毫无保留的享受的上帝赐予的阳光。
甲板上还有我们的国宝熊猫的雕像，超级可爱，还有波点拍照墙，怎么拍都是最美。
这里在泼点墙的旁边，刚出电梯就可以看到，在 北极 星那边的电梯，像是一个鱼缸一样，在这里你就是一条美人鱼，哈哈哈。
火车模型这里必须打卡，在这里拍出来的人像腿超级长，随便一拍就是大长腿，女孩子一定不要错过。
This is me,This is you，超级上相的背景墙，打卡拍照根本停不下来
户外运动集结地
爱上了，爱上了，一个旱鸭子，爱上了冲浪，一开始纠结了好久，一直不敢上去，最后应了一句话“来都来了”，于是上去排队，开启今日邮轮的第一项运动，小心翼翼，战战兢兢的准备上场，上场前需要看一个安全手册，上面提示到，当被水冲倒时，请双手抱头。
直到上场前还是害怕的，全身在抖，哈哈哈，在旁边的轨道上，教练信息的告知动作，准备好后，便开始上场，没两秒钟，就被冲回了轨道，也是重新上场，第二次上场简直不能太开心，完全不没有被冲跑，跟随的重力上上下下在摇摆，其实原理跟滑雪有点像，如果你回滑雪，那这个就会很简单。
此冲浪项目分两个阶段，一个是俯卧式，一个是站立式，去之前就说，能完俯卧式就不错了，站立式就不尝试了，但当你经历火俯卧式的冲击，完全没有抵抗站立式的诱惑，所以继续选择了站立式。
但不得不说站立式，真的超难，完全站不住，给自己起了一个外号叫“撒手没”，教练拉着还好，可以正常站立，只要教练一撒手，立马被冲到了上面，瞬间消失的画面中，画面太快，嗖的一声就没了，一开始还很害怕被冲到上面，怕会发生危险，但后来体验过后，发现完全没有问题，连被冲倒的瞬间也是一种享受。
甲板冲浪的身后便是跳伞，如果不敢在高空中跳伞，那就来这里体验下模拟高空中跳伞的感觉吧，项目体验前，会有半个小时左右的专业培训与讲解，之后便可以模拟体验在高空中飞翔的感觉了。
南极球，户外VR运动，带个VR眼镜，享受空中跳跃带来的刺激感，欣赏不曾看过的美景，跳的越高，看到的越远，远方的大海，深山，一览无余。
北极 星，一次15分钟，登上南极星，享受至高点带来的满足感，南极星会绕着邮轮，移动90度，俯瞰整个光谱号和海平面， 天水 一线的场景，更加真实的展现在眼前。
甲板上的漏天游泳池也不绒错过，旁晚时分，夕阳渐渐浸入海中，海上的一切都暗了下来，然而，此时，穿上最爱的泳衣，跟随着邮轮一起在海中漂流。
室内的碰碰车，也是最爱玩的项目之一，车超级好开！！，好开！好开！重要的事情所三遍，一场速度与激情就在此展开了。
小伙伴在玩击剑的时候笑喷了，因为要戴头盔，所以要摘掉近视眼镜，于是展开了两个“瞎子”之间的决斗，对方的剑完全碰不到双方，不是实力不行，而是眼力不行，哈哈哈。
超燃节奏的演出
演出可谓是整个邮轮的点睛之笔，一场视觉盛宴就此展开，从此便一发不可收拾，一场名为中外的 丝绸之路 的演出开启了你我的演出之旅，超美的舞感和动人的歌旋，渗透着身体里的每一个细胞，演出主要融合了中西元素文化，其中融入了神话故事，阿拉丁神话再现，一场爱情与历史的文化之旅跟随光谱号的起航一起开启了。
“魅舞今昔”此剧的名字可谓是形容的非常贴切，一场从开始后，目光自始至终就没有离开过舞台，全场演出精彩纷呈，掌声欢呼声不断，也绝对是男孩子们的最爱，再多的介绍我就不说了，我相信你们都懂得，当然也是女孩们的最爱，有时候就在想，这个世界真的很精彩，很奇妙，竟然会有如此美好的事物。
为“光谱号”专门打造的“光谱奇侠”，像是一场科幻片再现，剧情精彩绝伦，现场效果也可谓是形象逼真，四位光谱奇侠就像是从电影中总出来的人物一样，梦幻而有真实的为在场的每一位观众演绎着他们心目中的光谱奇侠。
关于美食
除了演出，美食也是此次行程中最为重要的一部分，银卡臻享餐厅，位于船头位置，餐厅的柔和的灯光简直美到爆，正好是日落黄昏，落日透后玻璃若隐若现的散落在餐厅中的每一角落，这么美的场景怎么不庆祝一下,cheers～
此餐厅的分为两部分，一部分为主食，会有工作人员过来为你点餐，另外一份部分为自助模式，虎虾，爱吃海鲜的一定不能错过，肉质鲜美，鲜嫩多汁，可是是海鲜中的极品，所以点餐的话一定不要错过～
自助的部分的美食也是色香味俱全，基本以海鲜居多，龙虾，虾，三文鱼等应有尽有，满足了我这小小的味蕾。
14层的自助餐厅，满眼的美食简直令自己手忙脚乱，无从下手，也是游轮上免费的餐厅之一，面食，海鲜，寿司，中式，西式等全部供有，只能你来拿，另外餐厅还有鲜榨橙汁，无糖可乐，啤酒等饮品提供，这些需要跟工作人员另外单点就可以。
四楼和五楼的主餐厅，可单点，也可直接自行自助，不得不说在这里的这几天体重暴涨，因为根本停不下来的节奏。
