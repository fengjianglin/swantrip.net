提起白娘子第一个总会想到杭州，但提起西施却很少人会想到诸暨，诸暨也被称为杭州的后花园，这次随行的有一位非常nice的乌克兰小姐姐，她对中国的文化历史非常感兴趣，所以就让我们一起开启这次的诸暨之旅吧~

越王勾践，卧薪尝胆之地——诸暨
西施范蠡，东施效颦之事——诸暨
豆腐西施，秀色可餐之食——诸暨

唐突西子，豆腐西施，东施效颦，甚至连惊心动魄这四字都出自于此，可见诸暨不仅是一座风景如画的历史之城，更是一座满腹经纶的才华之城。西施生于2500年前的春秋时代，当时诸暨乃越国都城（后迁会稽），对于诸暨有太多太多想听的故事，也有太多太多想看的风景，所以带着这些期许来到了这里——诸暨。

杭州西湖惊现四大美女之首——西施

这次去到诸暨也是个巧合，但也就是这份巧合，让我了解到，原来，你是这样的诸暨。
此次旅行的目的地锁定在杭州，既然到了杭州，怎能错过这花俏烟柳的西湖之景，漫步于西湖湖畔，领略着如诗如画的江南水景。
湖中若隐若现的船只，慢慢的向这边驶来，从远处看去，桥头上面站了一个长发女子，一开始以为是白素贞，直到靠岸，才发现是越过西施。一位拥有沉鱼落雁,闭月羞花容颜的古装女子，站立在桥头，眺望着远方，宛若战国时期西施再现的场景。
也就在此时，对这片越国故地产生了浓厚的兴趣，于是转战诸暨，杭州从终点变成了起点，驱车一个半小时到达诸暨，到达诸暨的时候天已经快黑了，考虑到明天的行程，所以今晚的入住直接锁定在东白湖风生态旅游区的斯宅村——吾爱庐民宿。
因为个人比较喜欢原生态的地方，所以当晚到达后满满的都是惊喜，刚下车便已经喜欢上了这里，绿树成荫，小桥流水，所以决定明天一早来欣赏下这里的鸟语花香与青山绿水。
办理好入住后，终于可以开餐了，这里的美食也极力推荐，不知道是饿了，还是饭菜太好吃，红烧肉，水库鱼头，马兰头，桂花年糕等全部吃光。
来到诸暨怎么不尝试下西施豆腐，超好吃的一道菜，此后，这道成了每餐必点。结束了一天的行程，可以美美的睡上一觉了。

依山傍水，群山环绕，隐匿于山谷中的「东白湖旅游区」

早晨很早便起床了，直到今天早晨才目睹到斯宅村内的精致建筑，房屋以徽派建筑为主，青砖白瓦，充满了老建筑的气息，后来了解到，这里有张爱玲与胡兰成的一段乱世情殇，全国重点文物保护单位斯宅千柱屋-发祥居，距今180年的风水宝地——华国公别墅，摩崖石刻等古迹。
为了欣赏清晨的诸暨，虽然很累，但早晨还是挣扎着离开了床，天亮的很早，差不多4点多天已经亮了，5点半太阳已经升起来了。清晨的斯宅村，布谷鸟的叫声遍布了整个斯宅村，伴着灵动的鸟叫声，下面这张图就是这次民宿的大门口了。
一条水流潺潺的小溪贯穿于每个角落，一条蜿蜒曲折的公路连接于各个美景之处，斯宅村位于山谷中，依山傍水，群山环绕，一片绿意盎然之景，闻着花香，听着鸟叫，走在山谷之中，一切都是那么惬意悠然，宛若一片室外桃源。
斯宅村其实就是以斯姓宅第建筑而命名的地方，这里远离城市喧嚣，许多古迹都保留至今，村子里以老人居多，依然过着日出而作，日落而息的生活。
走过桥便是斯宅村的唯一一条公路，公路不宽，但很平坦，公路两边住满了人家，有晒干了的凉菜，有悠闲的土鸡，一切都是那么的惬意。公路靠近小溪的一边都有到小溪的台阶，走下去，清澈的溪水，流淌在整个山谷中， 清净澄澈。
恋恋不舍的结束了今日清晨之旅，回到民宿吃完早饭便开始了今天的第一个目的地—— 斯宅村千柱屋
千柱屋又叫斯盛居，建于清代嘉庆年间，距今已经是二百二十年的历史，一座以孝为主，有着悠久的人文气息，千柱屋为当地巨富斯元儒的住宅，“千柱屋”全部呈砖木结构，气势宏壮。到达千柱屋后正好有一位百岁老人站在门口，一片祥和之景。
千柱屋占地6900度平方米（也就是相当于我们现在入住两室一厅的房子10间左右），但这里有118间屋子，400多个人，弄3 2 条。内含1 0 个天井，屋柱盈千，所以名为“千柱屋”。
斯宅村的人非常注重礼节与孝道，有着传承至今的人文气息，进入千柱屋后第一眼看到的便是这一扇大门，此门只有在出殡的时候才会打开，平时不管什么情况都不会特例打开，可见此屋主人是个恪守礼节之人，此习俗传承至今。
斯宅有着“坠地做男儿，读书即不成名，亦当择术济人”之家训，“为官宜衷，为友宜信”之家规，可见当时斯宅之人为刚正不阿，大义炳然之人。说到这里其实斯宅有个流传下来的故事，在东汉时期，斯宅始祖——史伟，因擅自赦免一批被冤犯人，被人伺机弹劾，判为死罪，史伟之两子（长子十六，次子十三）不忍父亲背叛死罪，两人联名泣血上书，争着愿捐躯以代父身，最后了解其缘由后，下旨赦免其死罪，官复原职，但是当时的刑法规定，朝廷官员，犯死罪获得赦免，须改姓。因此“史”改成“斯”，变成了御赐，这便是斯宅姓氏的由来，也是斯宅为何孝道如之重缘由。
听到关于斯宅的故事对这座宅子的主人肃然起敬，也愿这座宅子的礼孝之节继续传承下去。
从孝道之门指出来看到的便是这座百马图，但是仔细看，这里的百马图只有67匹， 百马图中无一雷同，神形逼真，但缺少个33匹的确有些遗憾。
千柱屋的木雕主要为典型的徽派作品，遍布于古宅的各处，尤以雀替和冬瓜梁上的雕刻为最，夜市不可多得的艺术珍品。门上的木雕可以看出来之前为彩色，后因某种原因掩盖住了。
从搓衣板可以看出这里的人们依旧遵循着古老原始的生活状态，如果想体了解下，真正的古村落的生活，那这里一定不能错过，
继续往前走会看到一个类似走廊的过道，从这里出去，在往上走，可以看到千柱屋的全景。
Tips:
门票：免费
参观时间：全天
游玩时间：2个小时左右
结束完千柱屋前往下一站风水宝地——华国公别墅，途中经过斯民小学，已有115年历史，由斯氏家塾发展而来。
继续前行，千柱屋，斯民小学，华国公别墅这三个距离都很近，正好一趟下来都游览完，10分钟后到达华国公别墅，建于清道光庚子年，距今已有180年历史。
华国公别墅前面一片碧绿溪流，需要步行一小段路后才可以到达，背靠象山，门前有一片绿树遮挡，像是隐匿于山中的一处神秘之所。
斯宅村随著可见小溪与过道，请忽略我凌乱的秀发~~
华国公别墅身后有一篇油菜花，喜欢的拍照的朋友不能错过。
华国公别墅的由来是因为其父酷爱读书，想要建家塾，为培育族人成才，一生克勤克俭，然终未竟，其子故而再此建为别墅，为继承其父志向。
走进里面一股书香门第既视感，充满了文雅气息。
这里便是私塾，之前上课的地方。
走上二楼，轻轻推开古老的窗户，远处的青山清晰可见，鸟语花香伴着潺潺流水，绿水青山伴着清风拂面。
出来的时候发现门口有一处池塘， 后来了解到此为泮池，在清规戒律很严的清朝，建造泮池是社会地位的象征，没有一定的级别是不允许建造的，因此可见建造者在清朝有一定的地位。
Tips:
门票：免费
游玩时间：08:00-17:00
博物馆基本是去每个城市必去的地方，花上一上午的时间，来更好的了解城市文化与人文气息，于是决定去往不远处的越红博物馆。
此博物馆不单单是博物馆，他中国非物质文化遗产之一，有着茶工艺的整个流程，在博物馆对面就是采茶的地方，换上采茶的衣服，开始准备采茶。（换上衣服后感觉自己像是丐帮弟子）
一般是采摘一芽一叶或一芽二叶初展的芽叶，就是师傅手中的芽叶，背上我的小背篓开始采茶咯~
体验完采茶，跟着师傅去往茶仓，本来以为就是一个单纯储藏茶的地方，但是进去后，完全被惊讶到了。
古越茶仓是由他们自己制作的一处贮藏茶叶之处，曲线形的线条以及特殊的茶罐材质，在镜子下反射下像极了艺术展厅，一个茶仓都可以制作的如此精细，想必茶叶的制作工艺也一定更加细腻。
下面就是师傅示范如何揉茶，这也是茶工艺最重要的工艺流程了，
跟着师傅采茶，揉茶，下一步就是品茶了，感受当地的独特文化，品一壶红茶，留一抹清香。

世界之最「会稽山古香榧群」

来诸暨怎能不去看香榧，世界上最美的人——西施在诸暨，最贵的坚果——香榧在诸暨，香榧发展至今已成为中国文化遗产，从南宋时期开始保留，最好的就是在诸暨。
会稽山古香榧群位于会稽山脉上，面积约400平方公里，有结实香榧大树10.5万株，其中树龄百年以上的古香榧有7.2万余株，千年以上的有数千株。
香榧分为雌树和雄树，婀娜多姿的为雌树，挺拔单一的为雄树，这个还是比较好分辨出来的，下图就位香榧雌树。
蜿蜒曲折的山沟像是被震开，时隐时现的溪水像是从天而降，将此处一分为二，沟两边的香榧相映成辉，行程“夫妻恩爱”、“兄弟连心”等各种姿态。
这是香榧树的叶子，有想象含羞草的叶子，下面第二张为香榧的果实，香榧一般要三年才会结果，香榧子营养价值极高，当时价格也最贵，大概在150元左右一斤。
从山上看过过去，满山遍野的香榧树装饰了这座山，茂密的丛林，当阳光透过树木，微微洒下光，一片绿影仙踪之境浮现在眼前。

清水浣纱 沉鱼落雁——西施故里

终于来到了盼望已久的西施故里，西施春秋末期出生于越国句无苎萝村（今浙江省诸暨市苎萝村，西施也被称为荷花神女，关于西施的故事大家应该都耳熟能详，因长相貌美被送往吴国。
刚到西施故里门口有打年糕，搓糍把，做清明裸等传统民俗活动，可见诸暨人的传统文化之深。
西施故里总共分为好几个部分：中国历代名媛馆，范蠡祠，民俗馆，郑氏宗祠，购物一条街，西施殿，浣纱江景带等。
这是一个关于西施传说的雕刻，讲述了西施从诞生到送往吴国再到吴王灭亡的故事，雕刻做工精美，形象生动，如果时间不紧迫，一定要驻足于此，拜读西施传说，同时欣赏下雕刻之精华。
这是西施传说的第一幅图，夜明珠从月宫掉落凡间，施母在江边得到夜明珠，金鸡飞向施母，西施出生，此雕刻栩栩如生，惟妙惟肖，可谓是西施故里特色。
沿着江边栈道，继续前行，正好看到一些小朋友们穿着古时候的衣服在背诗，在继续往西施往浣纱女群雕的方向走，可以看到诸暨的标志性建筑浣纱大桥，
西施故里步行街，里面有一些关于西施比较有特色的商品或者小吃
西施故里可谓是风景如画，绿树成荫，柳絮漫天飞扬，像是在人间四月，飘起了雪花，阵阵风来，吹动着柳叶，波荡着湖水，古老的房子依旧伫立与此，经过百年的风水雨打，已经渐渐出现了历史的痕迹，但就是这些痕迹，把这里点缀的更加有韵味。
湖中的凉亭，湖上的小乔，诗情画意般的美景，在湖水的倒影中映射在眼眸中，让人不得不为此停留，静静的坐在凉亭，欣赏此时这片宁静。
提起范蠡，可能就会想到他与西施的爱情故事，但历史上并非如此，但历史记载，范蠡与西施不是同一辈人，他俩在一起的可能性很小，但是因为人们总是向往美好的事物，所以把西施和范蠡这两个优秀的人写到了一起，同时也希望西施能有一个好的归宿结局。
西施殿这次主要的目的地，位于诸暨市城南苎萝山下，浣纱江畔。西施的沉鱼落雁、闭月羞花之容吸引了全天下人目光，西施的大义炳然，爱国情怀震撼了全天下人的心怀，在古时候女子的地位可谓是极低，所以成大事者女子占很少数，但是就是这样一位奇女子，为越王勾践的复国大业做了巨大贡献，相传吴国灭亡后，西施回国，越王设宴前来欢迎，传说总是美好的，但历史记载可能有一些偏差，只有一些西施被沉入江底的模糊记载，关于西施最后到底去哪了，一直是个谜，所以让我们留一份美好给她。
中国有十二花神，而西施就是十二花神中的荷花神。相传2000多年前西施远嫁吴国那天，村民们在浣纱江两畔点亮无数荷花灯，为西施娘娘祈愿，也可能因西施清水浣纱，像极了盛开的荷花，粉嫩的脸颊，倾城的容颜。
轻轻走上前去，驻足于此，静静的欣赏这荷花神女，绝代佳人的容貌，也暗自感叹西施的 勇敢与坚强，早就了越国人民的安乐生活，一位这样的奇女子，怎能 不引得天下人为她敬仰。
在往上走为古越台，古越台依山而筑，正对门楼。台内奉越王勾践，两侧立谋巨范蠡、文种，台中上悬"卧薪尝胆"匾额，尽显励精图治、灭吴雪耻之志。造像高大魁梧。系樟木雕刻。
西施殿上面就是伫萝山，也是西施出生的地方，在进去可以看到上面有做伫萝亭，驻足于此，仿佛回到了旧国，西施的历史画面随之映像在眼前。

单一具体票价：西施殿45元 中国历代名媛馆45元 范蠡祠30元 郑氏宗祠20元 民俗馆20元
另外，学生、教师、老人、军官可凭证明各自身份的证件进行打折优惠。
拥有诸暨市本地户籍的人可凭身份证免费领票参观。
交通：
自驾车：杭金公路，沪昆高速公路穿境而过。距上海3小时车程，杭州1小时，宁波2小时，绍兴1小时
公共交通：诸暨火车站距西施殿1公里，乘车有1路、2路、113路
结束了一天的行程，回到酒店很快的办理好入住，房间有一个大大的落地窗，桌子上有茶品和水果赠送，很温馨的酒店。
偶然发现酒店的四层有一处露天天台，有一点点像苏梅岛的W酒店的感觉。水中的沙发拍照很有意境，因为赶着要去吃饭，所以忘记拍了，喜欢的拍照的宝宝们可以留出一些时间在这里拍照哈~
拍完照后直接去一楼吃饭，不知道怎么的特别喜欢诸暨的美食，这里不仅风景迤逦，连美食都让人垂涎欲滴，必点西施豆腐，5颗星，最爱的红烧肉怎能放过，油而不腻，下饭首选。

“诸暨白塔湖，浙中小洞庭”——白塔湖湿地公园

白塔湖国家湿地公园，位于浙江省诸暨市的北部，北连杭州萧山，东接绍兴，是钱塘江流域保存相对完好的湿地之一。也是诸暨市最大的湖畈和生态湿地，有着“诸暨白塔湖，浙中小洞庭”之美称，
白塔湖与78个岛屿，形态各异，湖内河网交错，自然曲折，呈现"湖中有田、田中有湖、人湖共居"景象，水陆相通。
乘船到达白塔湖的岛上，恰巧看到渔夫们正在捕鱼，渔网渐渐缩小，鱼儿一个一个的跳出到水面，场面着实壮观，就这样坐在河边，欣赏着捕鱼的全过程，也是白塔湖的一处风景。
白塔湖有一处桃花岛，有着2600多株的桃花，当时去4月中旬去的，桃花已经凋谢了
自驾路线：
杭州方向到白塔湖国家湿地公园：杭金衢高速行驶至临浦路口下高速，走03省道东复线（塘湄线）到店口，再走枫湄线到杨梅桥至金家站。

“斯里兰卡”火车带你穿越儿时年代「米果果小镇」

米果果小镇与白塔湖国家级湿地公园隔湖相望，这里不仅是4A级风景区，更是一个教育基地，孩子们的天堂，孩子们失去的童年，要在这里找回来，进到小镇里面连我们这些大宝宝们都喜欢的不得了
米果果小镇最喜欢就是这个小火车了，最先经过的是一片紫藤大道，紫藤花的花期应该是在夏季，现在还未完全盛开，到了夏季，这里一定美的不可方物，然后会经过一片花海，小火车在花海中穿梭，像极了斯里兰卡的小火车，清风拂面，花香飘来，伸手就可以够得到的美景。
小火车开到北门，运气好的话，还可以看到白鹭，还有天鹅表演，寓意着宝宝们像天鹅一样，可以自由自在的在天空中翱翔，并且安全落地的全过程。
这里的教育基地在诸暨也是非常有名，由武汉大师专业老师为孩子们研究课程，专门为每个年龄段的宝宝设计贴身独有的体验课程。
Tips：
门票：成人通玩票128元 儿童通玩票98元
华东国际珠宝城，一提到珠宝，可能女孩子的两眼都会散发着光芒，估计看自己男朋友都会发光，诸暨也被誉为“珍珠之都”，所以这里的珍珠绝对值得一提，拥有世界最大的珍珠市场，是世界上最大的淡水珍珠原珠集散地，是我国淡水珍珠养殖产量最高的养殖地区，也是我国珍珠加工技术最先进的地区之一。
因为是原产地，所以价格在市场上算是物美价廉，珠宝城珍珠样极多，基本上只要你能想到的，这里全部囊括，一颗一颗珍珠映入眼中根本挪不开我那轻快的步伐。
