== 关于租车 ==
虽然以前在欧洲生活了3年了，但这次是我第一次来欧洲自驾，而且还是刚刚过了实习期。意大利租车有一个要求，驾照至少一年以上，并且年龄低于26岁会多收费用，原因是：你还小……
租车的平台有很多，这次我选用了租租车，因为感觉这是目前用过最为方便快捷，而且跟进服务十分完善的一个平台。全球极大租车公司，如AVIS，Hertz等，在租租车都能查到，可以直接对比价格，也很容易筛选需要的车型。
欧洲大部分人喜欢开手动挡，因而中国人租车尽量提早，不然很可能没有自动挡，外加自动挡的费用会相对更高一些。此行总共6个人，考虑到大家都有一个大行李箱，所以就定了两辆车，均是欧宝英速亚，但是实际给的确实丰田的混动版Yaris，整体空间比较局限，但3个人肯定没问题。事后发现，还好当初定的是小型车，罗马城市街头简直太难开车了，随处停止各种摩托小汽车，很容易一不小心就把车刮了。
再说说怎么在租租车预定，一般你只需要选择好时间和取还车地点，筛选自己喜欢的车型就能下订单，异地还车有较高的额外费用。要注意租车公司的取车条件，还有额外驾驶人，里程限制，是否满油取还这些要求。下订单填写好自己的个人信息即可，可以预付全额，也可以部分付款，但是在一定时间内都能全额退款，不是很贵的情况下，建议直接全额付款，省的到租车公司还要交钱，甚者会给你收各种七七八八的额外费用。
== 驾照认证 ==
需要说说中国驾照在意大利的认可情况，在意大利大使馆的官网上有明确说明：持中国驾照在意大利驾车，必须经过使馆认证。所以说网上那些所谓的国际驾照认证件其实在意大利是行不通，有些时候可以顺利提车，但是如果外遇到事故、警察，则必须出具使馆认证，否则可是无照驾驶哦！
关于驾照认证办理，上海和北京有所不同，我是在上海办理的。
首先，需要先办理意大利签证，然后才可以去办理驾照公众。
地点：贵都大酒店6楼的外事办
材料：护照（原件+复印件，须有有效申根签证）
驾照（原件，正本+副本的复印件，正反面印到一张纸上）
上海办理无需无须翻译驾照，北京需要
费用：代办费100，认证费223
时间：从我提交材料到收到短信通知整整两周
== 意大利交规 ==
意大利的交通规则和国内基本一样，左舵驾驶，左道超车，遇到红绿灯和转盘必须停车观察，红灯情况下也不能直接右转。记得超车千万别右道超车，这样很可能被人投诉而面临罚款，当地人开车都很快，但是都遵守左边快车超车道，右边慢车道的规则，往往从左道超车后也会迅速回到右道上，而不会长时间占用超车道。
还有一个重中之重，那就是意大利全国随处可见的ZTL限行区，一般在老城区都会有，是一个白底红圈的牌子，看到这个标识，千万别再往前开了，这是外来车辆不能进入的区域，误入的话会面临高额罚款哦。
== 取还车注意事项 ==
取车的时候，一定一定要注意检查汽车的情况，给你的取车单子上会标明车辆现有状态，比如划痕等，需要一一核对（示意图，可以看得出来）；
要确认油箱是否满油；
确认门窗、雨刮器等是否正常，还有空调是否正常运行（我看到有人说当时租了一辆空调坏的，于是大夏天的开了一路没有空调的）；
注意查看车辆行驶证；
还车的时候没什么注意的，记得加满油就行。我们去还车的地方位于佛罗伦萨ZTL区域外，居然周六没人上班，自己把车停进停车场，钥匙仍在他们办公室门口的箱子就行了，给意大利人的随意给跪了……
== 高速 ==
意大利的高速有一部分是收费路段，也有不少是免费的。在进入高速路口处在机器上拿一张单子，出去的时候给工作人员或者放到自动缴费的机器上。如果遇上收费路段，要注意看进入匝道上方的牌子。
黄色牌子：标着TELEPASS，这就是国内的ETC，千万不要走进这条道路，不然我也不知道你会遇上什么样的麻烦……
蓝色牌子：标着CARTE和银行卡标志的，是使用信用卡付款，无人工服务，需要自己使用信用卡交钱。
白色牌子：这里需要注意了，白色牌子分两种，一种是只有硬币和纸币标志的，说明是自助缴费，需要用现金，可以找零，尽量别用大面额钞票；另一种还有一个手的标志，是人工收费的，此次走的几段缴费高速，我全是通过人工收费的。
意大利的高速整体来说比较便宜，我们这次从罗马开到佛罗伦萨，经过了3段收费高速，总的合起来也就20欧元不到。
== 加油 ==
欧洲的加油站有很多是自助的，但也有不少人工服务的地方，此次出行我们专门找的都是人工服务的加油站，避免出错。
切记租车后核对好车辆使用的游，主要是柴油、95、92，跟国内一样。我们这次租的小车虽然是Hybrid车型，不过加的是95油。跟加油站工作人员说清楚后，直接给现金即可。
如果遇上自助加油的，要注意了，这需要自行选择加多少油的，而且，意大利加油的奇葩之处在于先交钱后加油，并且不找零！不找零！不找零！所以，切忌一次性交太多钱，根据自己车子的油量，一点一点慢慢加吧。
== 如何停车 ==
关于意大利的停车，其实相对于国内来说更为轻松一点。一般是白线停车位为免费车位；蓝色停车位为收费车位；黄色停车位，要注意了，这是给残疾人或者一些当地居民使用的，千万别乱停！
在大城市里，基本上不用指望能找到白色停车位了，甚至几乎城中心都是ZTL区域，随便驶入会面临高额的罚款。关于付费停车，一般是停好车后，寻找周边购买停车票的机器。机器显示全是意大利语，可以向当地人咨询，自己核算好要停的时间，然后交相应的钱（收硬币或者5元、10元），就可以拿到一张小票，记得放在挡风玻璃上，方便警察核对。
== 导航和保险 ==
保险：购买了租租车上面的全额补全险，一天多加几十块钱，可以从0欧元起赔，开起来也放心。
导航：这里说一下租租车可以提供免费的导航，需要提前申请，好像不能托运；我们这次全程使用的是google地图，事先下载了整个意大利境内会开过的地方，一路上也没什么问题，可以离线使用。

== 住宿相关 ==
1. 罗马酒店：Gonfalone 6，含早餐，坐落在距离纳沃纳广场不远处的老城区，步行前往梵蒂冈也只需要10分钟；
2. 天空之城：Civita B&B，住在城堡当中，房间很舒适也很有当地的风格，老板态度非常好；
3. 皮恩扎：Podere Spedalone，一个坐落在田园中的农庄，需要自驾前往，农庄很大，有一个不错的游泳池；
4. 锡耶纳：Hotel Il Giardino，就在老城城门边上的酒店，阳台可以直接远眺城市风光，还有一个巨大的露天泳池；
5. 圣吉米尼亚诺：Hotel Villa Belvedere，距离圣吉米尼亚诺老城后城门大概10分钟步行，一个很精致的别墅型酒店，巨大的庭院可以享用早餐，喝点小酒，还可以在游泳池晒晒太阳；
6. 卢卡：B&B Villa La Perla，坐落在卢卡和比萨中间的一个小酒店，是一对夫妇经营的，态度特别好；
7. 佛罗伦萨：Hotel De Lanzi，距离佛罗伦萨市中心圣母百花大教堂仅仅1分钟的步行距离，地理位置非常好，而且也不会嘈杂。

罗马 在历史浮沉中不朽的荣耀
都说罗马是一个小偷横行的危险城市，还记得在我第一次单枪匹马独自一人来罗马旅行之前，学校的好几个同学都跟我讲述了他们在此被偷盗的悲惨往事。
“什么？！你要自己去罗马？注意好行李，容易被抢。”
这是当年每一个去过罗马的小伙伴听说我要来罗马游玩的第一反应。
如今，这已然是我第三次来到罗马，对于这座城市的许多街道，依稀还有着难忘的回忆。在这里，我也曾遇上过小偷，甚至还被一群小阿堵在过火车站边上一个狭小的卫生间里，好在每次都能化险为夷，可以尽情地享受罗马的自由。
抵达罗马的第一个早晨，与同行的牛哥一起，凌晨4点，来到了罗马废墟边上的高地，等待着第一缕晨光洒落在古罗马帝国的历史遗址上。
“什么？！你疯了？这么早出去不怕被抢么……” 
我和牛哥都是货真价实的风光摄影爱好者，我更是专业的旅拍摄影师，这是第一晚牛哥听说我要4点出去拍日出的第一反应。还好，很庆幸的，我们最终是成功起床，并且看到了常人难以见到的美景。
很少有人在旅行的时候能如此早出晚归，强忍着困意于凌晨起床出门是一件十分艰难的事情，然而，当看到眼前空旷而静谧的场景，当听到漫天飞舞的海鸥在天空中鸣唱，不禁有一种热泪盈眶的感动。
在罗马，甚至是意大利每一座城镇的街头，随处都能看到散落着的古罗马遗址，除却世人皆知的罗马斗兽场，这片紧邻斗兽场的古罗马废墟则是罗马最著名的旅游景点之一。
这里曾经代表了罗马帝国时代的辉煌，是古罗马帝国城邦建设的中心区域，我们常说的“条条大路通罗马”便是指当时的罗马城内所有大道均以古罗马废墟为核心散开的。和我们古老的中华文明一样，当时的罗马人们也都认为罗马是世界的中心。
晨曦中若隐若现的朝阳，两旁便是这里比较著名的几个凯旋门。说起凯旋门，人们往往会想起巴黎香街上的那座巨大凯旋门，但是在罗马城中同样藏着许多或精妙或古老的凯旋门遗址。其中，坐落于斗兽场边上的君士坦丁凯旋门最为著名，也是罗马城中年代最久远（建于315年），保存最为完善的一座凯旋门。
从边上的高地俯瞰到的是古罗马广场，也是当年古罗马的经济中心，如今遗留着许许多多神庙、建筑等遗址，来此参观最好事先看看古罗马的历史故事，时间充足的朋友，值得在此流连好久。
广场中比较有名的就是提图斯凯旋门和塞维鲁凯旋门，上面同样有着诸多精美的浮雕。
远方废墟中的帕拉帝诺山 Palatino 是当年城市的中央区域，可以俯瞰整座罗马古城，更是当年几位罗马大帝的皇宫所在地。
6点多，从废墟前往图拉真广场途中遇到带着爱犬出来跑步的帅气大叔。对于罗马人来说，生活节奏依然是十分缓慢的，能够如此早起锻炼的更是寥寥无几。
这些罗马城中别具一格的“大头松”，生长在断壁残垣之间，绿意盎然与古老的城墙交相辉映，十分精致。
这里是位于威尼斯广场后方的高地，这座建筑的背后就是我们欣赏古罗马废墟晨光的地方。
清晨的卡比托利欧广场，对面是罗马市政府以及一尊马可奥来里欧皇帝的骑马铜像，两旁则分别坐落着保守宫、元老宫和新宫，共同组成了卡比托利欧博物馆，主要展出古罗马的各种雕像雕塑。据说，该广场建筑的所有细节，均是米开朗基罗所设计。

一路前行，离开古罗马废墟不远，就能看到高高耸立的图拉真柱。这里，便是位于威尼斯广场边上的图拉真广场。该广场于公元112年建成，是为了纪念图拉真大帝远征罗马尼亚获胜而建。广场边上的主体建筑是两所巨大的图书馆和两座宏伟的大会堂。
图拉真柱Colonna Traiana，是广场中最为显眼的建筑之一，于113年落成，以纪念图拉真征服达西亚，由大马士革建筑师阿波罗多拉建造。石柱高30米，共有18块大理石砌成，其表面刻满了精美的浮雕，展示着达齐亚战争的恢弘场面。其柱体之内，有185级螺旋楼梯可直通柱顶，喜欢登高者可以登上柱顶的观景平台参观。
罗马城中心最大的广场，当属下图中的威尼斯广场了。这座巨大的圆形广场正前方则是建立于1911年的维克多·埃曼纽尔二世纪念堂，是为了庆祝1870年意大利统一而建造的。无论远观还是近看这座纪念堂，无不感叹它的恢弘大气，16根巨大的大理石圆柱，更是纪念堂最为吸引眼球的亮点。
一个早上的闲逛，此行意大利之旅的小伙伴们也纷纷顺利抵达酒店会和，稍作休整，一同到位于纳沃纳广场边上一家评价很高的披萨店享用意大利最地道的美食。不同于国内必胜客的披萨，这里的披萨均是薄底的，一大片一大片放在原始的火炉里进行烘烤，加上浓郁的芝士，简直美味！
下午时分，继续开启罗马老城区的暴走模式。罗马城的老街道都很狭小，大部分全是单行线，行走在青石子铺就的道路上，可以欣赏两旁多彩又古朴的建筑。
位于纳沃纳广场上的四河喷泉方尖碑，喷泉的雕塑分别象征着四条天堂河流(多瑙河、尼罗河、普拉特河与恒河)以及已知世界的四个角落（亚洲、非洲、欧洲和美洲）。
继续前行，一会儿就能到达罗马必游景点之一的万神殿。果然如同想象的一样，整个万神殿门口广场上全是游客，连拍照的空隙都没有！
万神殿是不用门票的，直接进入即可，这也是迄今为止唯一一座保存完好的罗马帝国建筑，更是古罗马建筑艺术史上的一座瑰宝，其中最大的特色是入口处的古希腊式三角形门楣以及对称的巨大大理石圆柱。
当步入万神殿内部，无论是谁都会被这巨大的拱形穹顶所吸引。
今日的万神殿有别于前几次来时的样子，因为今天的人实在是太多了！还是很怀念当时寥寥无几的几个人，站在这个空旷的大殿里，抬头仰望着天空中的那一道光束。
白天，巨大的穹顶上会有一道阳光射入殿内，随着时间的推移，光线在万神殿中游走，按照24h的时间打在殿内环绕着的天使雕像上。我们到的时候，恰巧是整点时间，于是拍下了这一幕，有点《天使与魔鬼》的既视感。

梵蒂冈 一个关乎信仰的国度
在梵蒂冈以及罗马的街头，总是能看到广场上矗立着高耸的方尖碑，上面刻着无法看懂的玄奥文字，仿佛每一个方尖碑都在诉说着自身千百年前的历史故事。它们就这样静静地矗立在这座繁华城市的各个角落，却又是那么和谐地融入平淡的城市生活当中，毫无唐突。
今天，我们主要的目的地将围绕在梵蒂冈城，这个充满信仰的国度。我们决定一大清早出发，这样可以避免大教堂超级长的排队时间，结果不知道是因为酒店早餐太好吃，还是沿途的风景太吸引人，出行的小伙伴们展开了各自的超级拖延技巧……导致我们从原定的8：30抵达大教堂排队变成了9：30才到，此时，教堂前已然开始大排长龙了……
梵蒂冈的圣彼得广场，一个可以同时容纳50万人的巨型广场，乃是罗马教廷举办大型活动的地方。面向着高大的圣彼得教堂，两侧是由巨型大理石柱组成的半圆形回廊。每根大理石柱顶上，还各有一个圣男圣女的雕像。
一直都很喜欢这些精美的雕像，可能是由于顾着排队聊天的缘故，一时忘记了多拍几张雕像的特写。
广场的左右两侧，则各有一个造型精美的喷泉。据说整个意大利城市中的喷泉水均是可饮用的，不知道这里的喷泉水是否可以饮用呢？
强烈建议夏季来意大利游玩的小伙伴，记得携带一个空瓶子，这样在炎炎夏日里可以更加方便地随时补水。

守卫着梵蒂冈教堂的瑞士士兵。据传，于1506年间，一支150人组成的瑞士军队穿越了人民之门(Porta del Popolo)进驻梵蒂冈接受当时的教皇尤利乌斯二世的祝福，至此教廷护卫队均由瑞士士兵组成。
但凡来到梵蒂冈的游客，都会想要多看几眼这些身着红黄蓝三色条纹古代骑士服装的卫兵，手握长戟的卫兵们各个高大威武，加上那顶蓝色的小贝雷帽，显得更加帅气。现如今的梵蒂冈士兵，俨然成为了来梵蒂冈旅游的一道亮丽的风景线。

大教堂的穹顶，个人感觉并不是多么的惊艳，后面会看到佛罗伦萨圣母百花大教堂的穹顶，那才是一个精美，绝对震撼的巨作。
要登顶大教堂其实需要一定体力的，即便是你乘坐电梯，仍然有2、3百级台阶，而且越到后面越窄越难走，许多人走到一半就已经气喘吁吁走不动了，更别说我的书包里还有两台相机、3个镜头……这个时候就展现了摄（风）影（光）师（狗）的牛逼之处了，一路向上，没有停歇，直奔顶部平台。
可惜，我们到的时间已经比较晚了，爬到教堂顶的时候依然快11点，属于接近正午的阳光，非常强烈，加上楼顶处没有任何遮挡，完全处于暴晒中。本来想说9点前登顶拍几张晨光中的梵蒂冈的，可惜远望落空了，实际看到的是这样烈日炎炎的画面。
这是两年前下午来大教堂的画面，正巧接近即将关门的时候，不用怎么排队就进来了。
如果有人想要登顶参观的话，建议选择一大早或者黄昏将关门的时候，可以减少排队。如果正午过来，绝对排队排到你中暑，因为这时候往往是旅游团最密集的时间段。
来到大教堂内部的时候已经11点了，我们还要赶在12点前往参观梵蒂冈博物馆，所以时间不是很充足，只能草草地参观一番。
梵蒂冈是世界上最小的主权国家，以圣彼得大教堂的穹顶为中心，成就了天主教世界的中央之都，并且收尾着各种艺术宗教宝藏。毋庸置疑的，作为天主教世界的中心，圣彼得大教堂乃是世界上最大，也是最为奢华的教堂，这其中蕴含了数百年来无数位天才艺术家的心血结晶
圣彼得大教堂最早于公元349年由罗马皇帝君士坦丁开始建造，但这期间也陷入了一段荒废期，直至15世纪开始重新修复，经历教皇尼古拉斯五世和尤里乌斯二世最终完工，其中高达119米的中央穹顶则出自米开朗基罗之手。除却在登顶时观看到的中央穹顶之外，教堂内部还有许多漂亮、精美的穹顶结构。

意大利葡萄酒分为4个级别： 即日常餐酒VDT、地方餐酒IGT、法定地区DOC和保证法定地区DOCG，其中，DOCG是最高级别的酒。

历史上，锡耶纳是由雷慕斯的儿子所建，因而在城镇街道中往往能够看到母狼喂养双胞胎的雕像。从12世纪起，随着商业与贸易的发展，锡耶纳逐步变得兴旺起来，最终达到了足以和佛罗伦萨相抗衡的实力，在13世纪期间，还曾经爆发过多次战争。尔后的14世纪，锡耶纳进入其鼎盛时期，在九人议会的统治下，相继建造了锡耶纳大教堂、市政厅、田野广场等举世闻名的绝妙建筑。
在一场可怕的瘟疫之后，美蒂奇家族成为了这座城邦的统治者，这也直接造成了这里长达多个世纪的经济衰弱。反观历史的发展，也许这也是一件好事情，经济的衰弱使得城市发展十分缓慢，全城也一直保持着当年的古老模样，二战期间的薄弱抵抗，也使得这座城市免于战火的侵害。

千塔之城的迷人天际线
埃尔萨谷（Val d'Elsa），一个从名字中就能感受到它的魅力，田园风光、美酒美食、教堂博物馆，这里应有尽有，但是最吸引人的却是这里的中世纪小镇——圣吉米尼亚诺，也是我曾经最喜欢的一座托斯卡纳小镇。
这里是托斯卡纳最为标志性的城镇之一，只需驱车来到周围的山丘上，便可看到一座座位于小镇里高耸的中世纪塔楼，高低的起伏构成了这里最为迷人的天际线。从步入小镇的城门开始，随处可见的是这里保存完好的中世纪风格建筑。
San Gimignano 曾经是一座伊特鲁里亚村庄，因其地处法兰西珍那古道之上，已成为自由城邦的圣吉米尼亚诺得到了良好的发展。当地的富裕家族日渐增多，有钱有闲的人们开始互相炫耀自己家族的权利和财富，慢慢地，谁家建造的塔楼更高成为了攀比的标准。也许正是因为人们的虚荣心，才使得如今小镇保留下了浓厚的历史感以及中世纪街道的迷人风景。
小城内两个紧邻着的著名广场之一，大教堂广场。广场边上就是建立于11世纪后半叶的罗马式大教堂，这座教堂被称作学院教堂，指的是最初掌管这里的神父的学院。从下午开始，就有许多游客和当地居民在教堂前的楼梯上坐着乘凉，这个时候买上几颗意大利Gelato，岂不美哉！
托斯卡纳的建筑都很有特点，往往每个地方的市政厅边上总有一座高大的塔楼，而这座格罗萨塔楼则是这座千塔之城中唯一一座可供游客登顶的地方。该塔高54米，也是小镇中最高的一座塔楼，门票：€5。
在圣吉米尼亚诺这座千塔之城里，随处望去就是一座高耸入云的高楼。
作为一个刺客信条的忠实粉丝，圣吉米尼亚诺的千塔之城风景对我来说同样有着特殊的意义。在第二代的刺客信条中，游戏曾还原了这里的宏大场面，而游戏中的主角也曾经攀爬过这里众多的塔楼，每座塔楼都高度还原了现存的样子。因而，每次回来这里，我都会探寻一下那些曾在游戏中见到过的场面。

提起比萨，每个人最先想到的必然是举世闻名的比萨斜塔。同样是第二次来此，每次都因为奇迹广场上密密麻麻的人群而对这座城市和建筑奇迹没有什么特别的感觉。
比萨位于佛罗伦萨西面，紧邻卢卡小城，为了让大家看一眼这座传奇建筑，我们驱车前往卢卡的途中，在比萨停留了一个中午，先来看看奇迹广场上的全景吧。从左往右依次是圣若望西礼堂、比萨墓园、巴萨大教堂、比萨斜塔。

比萨洗礼堂，这是一个圆形的穹顶式建筑，始建于1152年，100多年后经过重新设计，于14世纪完工。洗礼堂楼上有一个美术馆。
比萨大教堂始建于1063年，是由雕塑家布斯凯托·皮萨谨主持设计的，平面上看呈长方的拉丁十字形。其多层叠加的白色、绿色大理石外观，彰显着教堂的庄严肃穆。教堂在竣工时曾一度为当时欧洲最大的教堂，由上至下共分四层，内部的木质天花板呈金色，传说这上面可是镀了24K金哦！
前方整齐的一排建筑，是比萨公墓，同大教堂、洗礼堂、斜塔共同组成了奇迹广场的一系列建筑。墓园内部存放有数百座墓碑，由一个回字形回廊组成，如果在晨光来此，或许能看到独特的回廊景色。墓园的中庭著有许多精美的壁画，其中最著名的是《死的胜利》、《最后的审判》和《地狱》三幅。
比萨斜塔，这座举世闻名，在全球独一无二的建筑，作为大教堂的钟楼，其名气却远超大教堂。这座高56米的钟楼，在1372年建成时已经开始倾斜了，由于这里的土壤层松软，导致斜塔随着时间流逝越来越严重。
除了比萨斜塔的倾斜令人惊艳不已，它的著名之处还在于当年伽利略在此做过的自由落体实验，而这段实验故事，也出现在了全国各大教科书当中，最终比萨斜塔成为全国皆知的著名建筑。

恬静小城 卢卡
卢卡是意大利著名音乐家普契尼的家乡。这个恬静的小城是所有想逃避都市烦乱，享受自由空气的理想去处。在绿树成荫的城墙上骑自行车已经成为了很多游客来此必做的一件事儿。只需辟出一天的时间，还你一世难求的安宁。
托斯卡纳拥有无数迷人的小镇，除了此行去过的几个景点城镇之外，还有类似吸血鬼之城的沃尔泰拉，精致文艺的科尔托纳，而卢卡则可以说是休闲惬意。
卢卡，她就是这样一座慢节奏的休闲小城，不同于锡耶纳休闲中带着夜生活繁华的喧嚣，在这里，你完全可以沿着文艺复兴的城墙上阴凉的小道，缓慢步行一圈，花上一个下午的时间，走遍卢卡的大街小巷。而或许，在某个不经意间的拐角，你就能发现令你惊奇的艺术品。
卢卡的老城区由一个16世纪建筑的城墙所包围，这片城墙至今仍然完好地保存着，而墙上的林荫道如今已成为卢卡人们每个夜晚的散步小道。城墙沿途散步着三座角楼，至今仍完美展现了角楼当年的模样，我们则通过西南方的一座角楼进入了老城区。

佛罗伦萨，这是一座无论我来过多少次都永远看不够的地方，无论是走在这里中世纪的大街小巷，还是徘徊在各大著名博物馆的光影当中，亦或是站在阿尔诺河的桥上欣赏着千年不变的落日黄昏。这座从文艺复兴时期开始蓬勃发展的城市，在每一个角落里都能找到文艺复兴的踪迹。同时，佛罗伦萨作为意大利第二大都市，这里更是各大时尚名品的聚集地。来到佛罗伦萨，无论是探访文艺复兴的历史，还是欣赏翡冷翠的迷人风光，或是大快朵颐的品尝美食美酒，更可以在无数名品商店里挥金如土。
我们自驾行的最后一站便是佛罗伦萨啦，在老城区外围的还车点归还汽车后，打车来到了城中心的酒店。酒店位置就在圣母百花大教堂边上，绝对的中心与便利，出门一分钟，就能看到大教堂广场啦。

来到佛罗伦萨，大教堂的穹顶可以说是必须参观的景点。这座“布鲁内莱斯基穹顶，可以说是文艺复兴时期最为精美的杰作之一，在经过400多级台阶后，首先能够欣赏到的是绘制于16世纪晚期的无比华丽的壁画，描绘了末日审判的场景。
每个来到此处的游客，无不在抬头的一瞬间，便陷入无比震撼之中。整个壁画精妙万分，每一个人物都形象生动，从天界、人间到地狱，分别描绘了启示录中末日审判的三个阶段。在基督教义中，人们认为，当世界终结前，上帝将要对世人进行审判，即末日审判。凡信仰上帝和耶稣基督并行善的人皆可升入天堂，不得救赎者则被丢入硫磺火湖中永远灭亡。

领主广场的大卫雕像
领主广场（Piazza della Signoria），佛罗伦萨旧宫前的“L”形广场，乃是佛罗伦萨共和国起源与历史的焦点，至今仍享有该市政治中心的名声，也是佛罗伦萨游客的主要聚集地。在广场上有许多引人注目的雕像，其中之一，就是位于旧宫入口的大卫雕像复制品。在整个佛罗伦萨，一共有三个大卫像，米开朗基罗的原作则保存在学院美术馆中，另一个复制品则位于米开朗基罗广场上，这在我们下一天的行程中都会参观到。
旧宫前面还有一个巨大、精美的海神喷泉，6月份去的时候正好在维修，没能看到，下图是之前来佛罗伦萨时拍摄的。这座喷泉是巴托洛米奥·阿曼纳蒂的设计，后由其助手詹波隆那建设的。
海神尼普顿的面孔很像科西莫一世，暗喻佛罗伦萨的海上统治权。雕像竖立在八角形喷泉中间高高的底座上，底座装饰着被铁链锁住的神话人物斯库拉和Charybdis。
在维琪奥宫的入口一侧是米开朗基罗的大卫的复制品，而另一侧则是巴齐奥?邦迪奈利于1533年完成的雕塑《海格力斯和凯克斯》。这是希腊神话中的两个人物，海格力斯是大力神，凯克斯是一个会喷火的盗贼。
广场一旁的佣兵走廊，这里陈列着众多的雕像，圆拱下方，最左边是本韦努托·切利尼花了几乎十年的时间完成的名作《珀耳修斯高举美杜莎的首级》。
这是詹波隆那的另一杰作《强掳萨宾妇女》，塑造的是罗马士兵劫夺邻邦萨宾妇女的情节，整个雕塑巧妙地将三个形体的动作缠绕在一起。
《墨涅拉俄斯扶起帕特洛克罗斯的身体》，其中墨涅拉俄斯，是希腊神话中斯巴达的国王，美女海伦的丈夫，而帕特洛克罗斯则是在特洛伊之战中牺牲的希腊英雄之一。

来到美术馆附近，远远地就能看到排满大半个广场的队伍，而且还一眼看不到头，不禁庆幸自己提前做好了准备。在入口处对面的售票厅兑换了门票后，在预约队伍等待片刻便可准时入场。进入美术馆的第一道作品是在领主广场见到过的《强掳萨宾妇女》原作。
每天都有无数游客来参观学院美术馆，究其原因在于这里藏有文艺复兴时期最著名的标志性巨作——米开朗基罗的《大卫》原作。从转角进入展厅的第一刻，就能看到前方玻璃顶下显耀的雕像，而眼前人头攒动的画面更体现了人们对这一巨作的钟爱。
无论如何，来此一览大卫雕像的真迹还是值得的，虽说只是一座大理石雕像，但是上面各种栩栩如生的细节无不令人叹为观止。
除了《大卫》雕像之外，美术馆里还展出了许多学院学生们和其他名家大师们的雕刻艺术作品，同样值得驻足一观。
美蒂奇宫，坐落在学院美术馆不远处，票价€7，乃是当年美蒂奇家族早期的住宅，由老柯西莫委托米开罗佐修建的。当年的美蒂奇家族，从银行发家，聚集了巨额的财富，一跃成为佛罗伦萨真正的统治者。
宫殿中的三王礼拜堂则是这里最为精彩的地方，其顶部的壁画精美绝伦，展现的主题是《东方三博士去往伯利恒》。
参观完博物馆宫殿，午饭在边上一家很著名的ZaZa牛排用餐，点了他们最特色佛罗伦萨烤牛排，还有龙虾意面，都非常好吃！
下午参观的是以前一直没有机会参观的皮蒂宫，位于阿尔诺河南岸，这座巨大的文艺复兴风格宫殿曾一度作为佛罗伦萨统治者的宅邸。
皮蒂宫的购票排队并不算太长，但宫殿十分巨大，参观时间很长，所以我们也只买了皮蒂宫的门票，没有再购买宫殿后面的波波里花园门票。
如今的皮蒂宫已改造成了一个巨大的宫殿式博物馆，内部展出了一些银器、雕像等艺术品。
这是小时候的米开朗基罗雕像。
当年拿破仑的浴室。
作为美第奇家族和城市统治者的宅邸，这里面的装饰可以说是极尽奢华，宽敞明亮的宴会厅，加上晶莹闪亮的巨大吊灯，不禁想象当年的贵族们在此宴会跳舞的场景。
皮蒂宫的二楼为现代艺术画廊，展示了佛罗伦萨一些艺术家们在18-19世纪期间的艺术作品。
最后一个下午，我们在傍晚打车来到了位于佛罗伦萨东南角山腰处的米开朗基罗广场，这里可以看到最后一座《大卫》雕像，而在这座巨大的广场上，更可以尽情地欣赏佛罗伦萨壮阔的全景。这个时候，不妨在露台边上的小酒吧，点上一杯Mojito，再吃点小点心，静静地等待夕阳的到来吧。

每一个观景平台必然是游客聚集的地方，而米开朗基罗广场更是佛罗伦萨不得不来的景点，络绎不绝的游客，也总能看到许多身材高挑、身姿妙曼的金发美女~
据说，在米开朗基罗广场上，可以看到佛罗伦萨最美丽的日落，真的是这样吗？大家不妨亲自来此虚度一个黄昏吧。
夜幕降临，当华灯点亮这片城市，大教堂的灯光闪烁着迷人的光芒，远方是在夜晚的云雾中若隐若现的山头，还有迟迟不肯散去的晚霞和蓝天，一切，显得那么的美丽和平静，真希望能够就这样一直坐着，欣赏着。

最后，用一张佛罗伦萨的夜景全景作为结尾吧！
