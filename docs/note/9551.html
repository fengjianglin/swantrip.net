凤凰涅槃后的塞尔维亚，让人迷恋的不止是世界网坛传奇德约科维奇

游记 自助游 2019/02/02
时间在一座城市里已经流逝
而在另一座城市则刚刚开始
所以，你可以在两座城市之间作
穿越时间的旅行
你可走向未来或回到过去
── 米洛拉德·∙帕维奇《哈扎尔辞典》
巴尔干，是我未曾触及的土地：那是一片被世界历史教科书称作“世界的火药桶”的地方，那里有一段不同寻常的历史停留在记忆深处，那一片沧桑、神秘而还有点遥不可及的地方。
2019年的第一次旅行，是一场穿越时间的旅行，是一次在现实里触摸过去的旅行。
2019年1月27日，我最喜欢的网坛一哥诺瓦克·德约科维奇在墨尔本举行的澳大利亚网球公开赛上，再次战胜了纳达尔，勇夺今年首个大满贯冠军。也就在这场巅峰之战前夕，我走进了小德的家乡—塞尔维亚。关于曾经的南斯拉夫主体国家塞尔维亚，只有走进了，你才知道那里不只是德约科维奇。

关于塞尔维亚，你知道多少？

1. 塞尔维亚是第一个对中国实现免签的欧洲国家，从2017年1月至今已经有两年时间了；
2. 欧洲的母亲河多瑙河流经塞尔维亚；
3. 塞尔维亚是前南斯拉夫的主体国，“巴尔干火药桶”、“萨拉热窝事件”、“中国驻南联盟大使馆被炸”都与这个国家有关；
4. 世界上最大的东正教堂所在地---塞尔维亚；
5. 首都贝尔格拉德曾《孤独星球》评为全球十大夜生活之都；
6. 目前世界排名第一的网球明星诺瓦克·德约科维奇的家乡就是塞尔维亚；
7. 塞尔维亚还盛产帅哥美女，除了小德，维密天使米兰达·可儿也是塞尔维亚人。

塞尔维亚在哪里：

塞尔维亚位于欧洲东南部，处于巴尔干半岛的心脏。多瑙河（Danube River）从塞尔维亚穿流而过，两岸汇集了令人惊叹的地质、文化和历史遗址。从拜占庭到奥斯曼，从东欧剧变到科索沃战争，这里一直聚集着世人的目光。
1987年德约科维奇出生于塞国首都贝尔格莱德，我的塞尔维亚之行也就从这个欧洲最古老的城市开始。

塞尔维亚怎么去：

令人无限期待的巴尔干之旅，其实我从打卡土耳其航空商务舱的那一刻就开始了。土耳其航空公司（Turkish Airlines Inc.）成立于1933年，总部设于伊斯坦布尔。对于土航向往已久，是因为它曾经连续6年被评为“欧洲最佳航空公司”、“最佳商务舱餐食”以及“南欧最佳航空公司”。在朋友们的口中，土航高大上也充满了神秘。也正好趁着这次机会，给小伙伴们“揭秘”一下我眼中的土耳其航空商务舱。
1.敞亮大气舒服：土航商务舱的座位间隔相当宽敞，空间也大，特别设计了放鞋子的地方，放置杂物的箱子也够大，特别适合我这种东西多的人。
2. 中文 中文 中文：重要的事情说三遍。从上海飞伊斯坦布尔的这条航线，飞机上有中文广播，提供中文菜单，机上娱乐设备有中文选项，所以小伙伴们再也不用担心自己的英文水平不够啦！
3. Wifi Wifi Wifi：土航的国际长航线都提供Wifi服务，在商务舱只要输入自己的座位号和姓名就可以免费上网，经济舱客人可以付费上网；这样，在万米高空，你也不用担心会和国内的亲朋失联，关键可以随时和你喜欢的人聊天，再漫长的旅途也有了寄托。
4. 美味与颜值并存：既然率获“最佳商务舱餐食”，那么最精彩的部分绝对是机上美食。小食、前菜、主食、甜品、美酒一样不落，摆盘和口味都像极了四道菜的烛光Fine Dine，关键来为你点菜送餐的不是空姐，而是帅帅的戴着高帽的大厨。把超级大厨请到飞机商务舱做美食，土航应该是首创吧！
真的是烛光晚餐哦，因为在餐盘里真的有只可爱的小蜡烛，只可惜身边那个“TA”不知道在何方？
另外，用早餐的时候，我特地选择了“在线电视”，竟然是我喜欢的澳网直播，虽然不是德约的比赛，但是一边看澳网，一边用早餐的感受也是很特别；而且我点了粥，满足一下中国胃。
令人惊艳的土航伊斯坦布尔商务舱休息室：
早就听朋友说过土航在伊斯坦布尔的商务舱休息室令人惊艳，这次打卡下来的确名不虚传。
一进入休息室，你会立刻被各种半圆形或圆形结构的设计所震撼，再加上时有时无出现在你面前的各种植物，有穿越时空隧道的感觉，也有迷失方向的感觉，总之相当梦幻了。
整个休息室的设计感很强，很现代也很实用，而且也大得惊人。休息室有上下两层，一层我都觉得我会迷路；去下层要通过一个网状交错的区间，那里就像是进入异度空间的通道，我被那个地方迷住了。
休息室按照功能分为电影区、餐饮区、储物区、小型套房区（需要另外收费）、植物园区、简餐区、上网区、高尔夫体验区等，一应俱全，转机4、5个小时待在里面也不觉得无聊。我都觉得自己可以在里面吃喝玩乐一整天。
所以，如果你有机会到伊斯坦布尔，这个土航的商务舱休息室，千万不要忘了进来打卡哟。
其实选择乘坐土耳其航空公司还有一个重要的原因，  土耳其航空全球目的地为306个，包括257个国际目的地和49个国内目的地，在土耳其国内肯定首屈一指不用说了， 土航飞往的国家数量和国际目的地也多于全球其他任何一家航空公司。当然因为首都伊斯坦布尔刚好在欧亚的交汇处，这个绝佳的地理位置让土航比其他航空公司更有地理上的优势。

塞尔维亚初印象：诺维萨德

抵达贝尔格莱德后，我们立马驱车去了一小时车程外的诺维萨德（Novi Sad）。诺维萨德是塞尔维亚北部伏伊伏丁那自治省省会，它扼守多瑙河上下游和两岸的交通运输，就如同扼住了塞尔维亚这个“欧洲十字路口”的喉咙，在交通不发达的时代，曾是兵家必争之地。
屹立在多瑙河右岸的彼得罗瓦拉丁要塞（Petrovaradin Fortress）是近观诺维萨德历史的窗口。早在罗马帝国时期这里就已经有大规模防御工事，要塞居高临下易守难攻，很多世纪来一直是兵家必争之地，经历过很多次战争，现在已经功成身退成为观光景点。
要塞上有座白色灯塔，长短指针和正常时钟相反，被称作“醉钟”；黄色的大楼现在已经变身为城市博物馆和画廊，你可以走进去感受艺术气息；要塞还是每年7月，巴尔干地区颇为有名的EXIT音乐节举办地。
斯雷姆斯基•卡尔洛夫奇（Sremski Karlovci）距离诺维萨德仅8公里，它位于弗鲁什卡格拉山坡和多瑙河岸的一座巴洛克小镇，是塞尔维亚最重要的精神和文化中心之一。
我们去了斯雷姆斯基•卡尔洛夫奇最著名之一的酿酒厂Zivanovic，据说他家的酒早在上个世纪就被带上了欧洲的豪华邮轮“泰坦尼克号”；他家还有自酿的各式蜂蜜，老板拍照右边那个酷似城堡的东西，就是蜜蜂的家。
地窖里储存着好酒，还有一只积满灰尘1910年的橡木桶散发着古老的气息；我们专门品尝了当地最出名的Bepmet 甜香葡萄酒，那是由葡萄、蜂蜜加上27种植物酿造而成，据说配方传男不传女，只有当地14个家庭的男子知道该酒的配方。
诺维萨德被称作欧洲青年之都和文化之都，同时这里的悠闲和散漫也对游客有着强烈的吸引力。
市政厅
天主教玛丽大教堂
老城区中心的自由广场（Liberty Square）从十八世纪以来一直是集会和举行庆典的场所，并且在不同历史阶段广场有不同的名字，奥匈帝国统治时期叫弗兰茨·约瑟夫广场，也就是以茜茜公主丈夫、奥匈帝国皇帝弗兰茨一世的名字命名的。一战后更名为革命广场，二战后才改称自由广场。自由广场周围分布着各式各样的建筑，市区主要的景观都在这附近。
鸽子，是诺维萨德市政广场的小可爱
礼品店
从自由广场出来是城内最古老的街道Zmaj Jovina 街，然后是诺维萨德大（Dunavska Street）。这条五百米不到的诺维萨德大街上分布着大大小小的图书馆、博物馆、美术馆和画廊和很多露天的餐厅和小酒馆。我想夏天会更热闹吧！

冬日，行走在贝尔格莱德

贝尔格莱德坐落在萨瓦河与多瑙河交汇处，历史与现实、信仰与热忱、悲情与荣耀在这座城市不断地相互叠印，掺杂有一点点颓废的忧伤。尤其是在这样的冬天，地上、树上、教堂的房顶上都落满了雪花，灰色的天空让整个城市显出欧洲特有的中世纪阴冷与忧郁。
我不知道，是不是只有扒开厚雪才能够发现深埋在这座城市里的精神与信仰，去寻找作家陈丹燕在《捕梦之乡》里形容的“穷而性感”...... 还有那些“颓唐而古风盎然的街道散发出一股奇异的自由与失落混合的迷人气氛”，在这个冬天大概是找不到了吧。
我住的贝尔格莱德市中心万怡酒店出门就是共和广场(Republic Square)，只是广场被围起来正在进行翻新和修缮，米哈伊洛大公雕像上满是雪花，连日下雪工程也停了下来；共和广场通常是贝尔格莱德城市之旅的起点，在拐角处还时常能看见打着黄伞的导游领着游客讲解。塞尔维亚国家博物馆、贝尔格莱德国家剧院等数个城市建筑均位于广场周边地区。
共和广场往西就是贝尔格莱德最著名米哈伊洛大公街（Knez Mihailova Street), 我觉得这条不算太长的步行是塞尔维亚的缩影。在贝尔格莱德待了3天，我几乎每天都会这条街走一遍。我喜欢驻足聆听街头艺人设备简陋却是用心的演奏，看橱窗里各式本地的和世界的品牌是如何觥筹交错的共生，也喜欢欣赏夜色近晚华灯初上时，情侣相拥快步行走的亲密样子。
米哈伊洛大公街上，汇集了无数的咖啡厅、文艺小店、服装店，但是我觉得最多的应该是书店。文学巨匠们大幅海报醒目地挂在墙上，看得出来这个民族对知识的尊重。虽然你不懂塞尔维亚的西里尔文，你也一定能这些间隔不远的一个个书店里，感受得到这个民族对历史对文化的尊重。作家陈丹燕老师追随着《哈扎尔辞典》的地理路线，在土耳其和塞尔维亚待了好几个月，写出的那本《逐梦之乡》也被译成了塞尔维亚文字豁然摆放在大公街的书店里。
米哈伊洛大公街往西走延长线上，就是面对萨瓦河与多瑙河交汇处贝尔格莱德老城贝城最重要 卡莱梅格丹城堡（Kalemegdan Fortress)。城堡分上、下两城，上城顶部和下城底层原有的宽敞绿地与独特古朴的石头建筑都已经被白雪覆盖，两河交汇处也是冬日的一片苍茫。
城堡从罗马人开始兴建，到被匈奴人摧毁，后来被拜占庭人重建再又被奥斯曼人占领，再后来几易其主，到如今卡莱梅格丹既是一座古堡，更是历史博物馆、艺术殿堂、音乐厅、剧院、动物园、幼儿园、美食城。如果有机会沿着城堡的通道或小道漫步并仔细阅读一路上的碑文，你就会被带到过去的时光隧道，感受曾经的历史并解读许许多多历史名人轶事。
通过几座木桥，看到塞尔维亚自然博物馆、历史博物馆、军事博物馆、天文台以及画廊都在这里。甚至连那个不起眼的枯井也不要小瞧，它可是充满神秘色彩和装载许多城市传说的古罗马时代的水井。
作为欧洲最古老的城市，贝尔格莱德有一处地方你不得不去，那就是圣·萨瓦大教堂（St. Sava)。它有很多名头，比如是在巴尔干地区最大的教堂，是世界上10个最大的教堂之一，也是世界上最大的东正教教堂，它是为纪念塞尔维亚东正教会的创始人和中世纪塞尔维亚的重要人物圣萨瓦而修建。
圣.萨瓦大教堂
从位置上看，它主宰着贝尔格莱德的城市景观 ，也是贝城最宏伟的欧式建筑 。只是，不巧我们去的时候，圣萨瓦教堂正在进行大修，除了能看外观以外，里面什么也看不到，我只能想象着里面的金碧辉煌。
从圣萨瓦教堂回酒店的路上，可以选择途经 特斯拉博物馆（Nikola Tesla Museum)。这里保存着Nikola Tesla的骨灰瓮，这位科学家的发明成就宣告了21世纪的到来。特斯拉对世界的电气化和未来的技术进步做出了重要贡献，最有趣的是游客可以参与到特斯拉生平最重要的发明体验中去。
在酒店附近的小店买了一张当天公交卡，290第纳尔（20元人民币左右），坐上中间有个转盘的三节式公交车去距离贝尔格莱德几公里以外的泽蒙，我赶脚有点穿越...
这种公交车估计在中国的三线城市都不大能见到……
蒙泽的特色是临近多瑙河狭长紧密的街道中，遍布的酒馆和咖啡馆，只是冬天街上的人不多，不像夏天在露天和咖啡聊天那么有feel。
戛尔朵什塔Gardos Tower
泽蒙有个著名的塔，戛尔朵什塔Gardos tower。走过白雪铺满的石梯，这个塔就豁然眼前，again塔楼又是不开放的，我只有悻悻地拍下外观。不过，就算站在塔下，也可以将蒙泽的美收入镜头中，古老的教堂尖顶仍然是冬日里一抹曼妙美景。

德约科维奇，塞尔维亚的传奇

我敢说有很多人因为德约科维奇才知道了塞尔维亚，小德可以说是赛国的推广大使；钟爱网球的我也是因为小德，才充满了对塞尔维亚的探索之心。作为“小德的迷妹”，从蒙泽回贝城酒店的路上，我特地去德约在市区的 诺瓦克餐厅（Novak Café）打了个卡。
餐厅上方有个巨大的No1灯箱，让你在夜里也不会错过；进门后，那个征战大师杯时，上海人民送的兵马俑小德会让我忍俊不禁，哈哈！他的奖杯几个陈列柜都放不下，餐厅里到处都是他的图片；到现在，他已经赢了700多场大小比赛。
想想小德也真是不容易，他出生后四年南斯拉夫内战开打，加盟共和国之间的五场战争一直延续到1999年。“我们会去轰炸最多的地方练球，猜测他们不会两天内轰炸同一个地方。”小德在自传里说。就是那个在被轰炸过的废墟里练球的男孩，成为了网坛霸主，从他在球场上不屈不挠奋力扑救每一个球的动作中，我们可以读懂来自贝尔格莱德的坚毅与力量。
2012年他与纳达尔苦战5个多小时的巅峰之战，贝尔格莱德的每个人都记忆犹新；今年的澳网，德纳第53次对决，德约直落三盘获胜，小德绝对是塞尔维亚乃至整个巴尔干的骄傲与传奇！
游走在贝尔格莱德，每一条街巷或是每一个转角，即使是冬天都充盈着深厚的历史感与生命的张力；什么都是有诗意的，甚至冬日灰白的天色和不停的飘雪，也错落成一道道印刻在城市历史里的时光印记！

在塞尔维亚吃什么？

塞尔维亚人喜欢吃肉：羊肉、牛肉、猪肉、鸡肉、鱼肉、鸭肉他们都喜欢... 做法其实和西餐都差不多，烤、腌、蒸、炖等，只是分量更大也没那么精致。他们认为食物的分量一定要足，才能显出热情和友好；点的这些菜，要么全部吃光，吃不完的肉也会打包回家。难怪塞尔维亚男男女女都长得特别结实，这真的和他们的饮食习惯有关。

Sokace餐厅蛮有特色，很多旧时物件诉说着历史；墙面正中间放有反抗奥斯曼帝国英雄的照片，以及其他的纪念图片、物品等...
5A Soba 贝尔格莱德餐厅
地址：Rajićeva 12, Beograd 11000, 塞尔维亚
5A Soba这个的意思是老板开的第五家Soba餐厅，他家装修现代而叛逆，是贝城年轻人喜欢去的地方。关键，东西实在太好吃了，特别是羊肉里外都嫩而入味，相当好吃；还有墨鱼汁意大利Risotto，做得香糯美好，很久没吃过那么有滋味的意大利饭了；我们四个人点了四种肉，全部光盘。
Dva jelena 贝尔格莱德餐厅
地址：Skadarska 32, Beograd, 塞尔维亚
这家餐厅坐落在贝尔格莱德Skadarlijia波西米亚去，每天晚上8点之后就有乐队演奏传统的塞族民谣。贝尔格莱德曾被《孤独星球》评为“全球十大夜生活之都”，即使是寒冷的冬夜，餐厅里也因为音乐而变得热闹温暖！
塞尔维亚是欧洲物价和消费最低的国家，一杯咖啡一般在人民币12元左右；像上面说的几家很好的餐厅，人均也就在人们80-100元；我去了贝尔格莱德市区的超市，价格也是很平。
充满魅力的塞尔维亚所拥有的历史积淀、丰富内涵和美丽风光，远远不止贝尔格莱德和德约科维奇，我期待着一个不再寒冷的季节再次光临。因为免签，待阳光正好，再来一场说走就走的塞尔维亚之旅吧！
