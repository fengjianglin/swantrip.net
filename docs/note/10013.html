行程规划

周末出游，我们选择了自驾。周六一早从 昌平 出发，走京藏高速转张石高速到 蔚县 东出口出高速，途径代王城镇，然后到暖泉镇游览古城。晚上住 蔚县 。第二天从 蔚县 穿越四十里峡，走飞狐古道，主要游玩飞狐峪·空中草原，下午回京，全程580多公里。
在我们规划的行程中，本来第二天下午还可以游览 蔚县 东南部的茶山村，但因为带着老人孩子，体力都有限，所以就放弃了。不过不完美的旅行才是最好的，把遗憾留给下一次。

代王城，你的古老有谁知

公元前423年，赵武灵王始设代郡。秦始皇统一 中国 后，将天下分为三十六郡，代为一郡，领县十八，郡治即今代王城。汉以前的经济文化、军事中心一直在代王城，前后延续了千余年，从南北朝始置蔚州后，代王城便沦为一方区域的商贸古镇。
进入代王城镇的时候恰逢正午十分，镇上的孩子们都放学了，堵车堵的那叫一个狠。
代王城
小镇不大，一边开车一遍欣赏景色，终于找到了代王城遗址。镇内现存的三面戏楼，明初木构龙王庙、地藏寺、财神庙、雍正圣谕碑，大小堡城，大量的古民宅等遗存，均蕴藏着大量的历史文化信息，具有重要的历史研究价值。

实在没有时间去一一逛了，如果没有带娃，不需要午睡，下午的时间完全可以逛遍这古镇。
看到了这座玻璃宝塔？恭喜你，就快走到了。
全景非遗奇绝演艺打树花——《天下第一堡》是暖泉古镇必游的“重头戏”。每天将全国首台融合了打树花的演出呈现给游客。这部舞台剧主要讲述了暖泉古镇的一个凄美的爱情故事，我一开始觉得孩子那么小看这种剧肯定坐不住，所以都没在网上订票。没想到她真真的坐了一个小时，很让我和她爸意外。不过耀眼的灯光、华丽的服装、变幻的舞台再加上美妙的歌声，确实连小朋友都被震撼住了，很值得一看。
羊咩咩和大马也出场了。
最后十分钟上演的“打树花”更是让人拍案叫绝，150一张绝对值回票价。
据说当年暖泉镇有好多铁匠作坊，每逢年节，富人们燃放烟花庆祝，铁匠们也同样渴望着热闹喜庆，但铁匠买不起烟花，他们从打铁时四溅的火花中得到灵感，把熔化的铁水泼洒到古堡城门上方的砖墙上，好似朵朵烟花盛开。这种特别的烟花吸引了越来越多的普通老百姓，其热闹喜庆的氛围绝不输给拥有烟花的富人们，因此便有了每逢过年富人放烟花，穷人打树花的民俗。

飞狐峪，去空中草原骑个马

第二天一大早，我们就出发去了飞狐峪，天公不太作美，有些阴沉，草原拍出来不是那么美丽。
飞狐峪·空中草原
飞狐峪是一处以峡谷和高山草甸风光为主的自然风景区，也是 张家口 著名的旅游胜地。
景区分为三个部分，第一部分是飞狐峪，这是通向空中草原的一条峡谷，峡谷中有公路，可以沿路开车参观。开过飞狐峪后来到马蹄梁，这是景区的第二部分，就是山的垭口，视野非常开阔。经过马蹄梁再向前十几公里，就到了开阔平坦的空中草原地带了，也是景区最深处的一部分，这里可以观景摄影，可以骑马游玩，清风拂过清凉的山岗，惬意无比。
