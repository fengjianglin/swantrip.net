前言

有这样一种花十月怀胎一朝分娩，它被称为女人花；
有这样一种花从一而终不喜挪地，它被称为君子花；
有这样一种花漫山遍野嫣然怒放，它被称为映山红。
是的，它还有一个最耳熟能详的名字——杜鹃花。
毕节市位于云贵高原中部的隆起地带，最高海拔2900.6米，算得上是贵州屋脊。毕节境内地势落差大，群山耸立，绝大部分区域属于喀斯特地貌。因高海拔低纬度的特点，毕节冬无严寒、夏无酷暑、雨水日照充足，孕育了毕节特有的花海景观。其中最有代表的要数中国最大且品种最多的原生杜鹃林——百里杜鹃。
每年3-5月是百里杜鹃的花期，每到那时候，各地的游客蜂拥而来，都为了一睹这几千公顷的大型花海。快快跟随我们一睹百里杜鹃的真容吧。

Day1 百里杜鹃景区一日游

百里杜鹃，杜鹃花林地面积7000多公顷，杜鹃花密集处3000多公顷。被誉为“最大的天然花园”。百里杜鹃景区分为普底景区、金坡景区、仁和景区、大水景区，此次我们去的是普底景区和金坡景区。
百里杜鹃普底景区
百里杜鹃花海的环形赏花线路连绵几十公里，可最精华的，必须是五彩路这一段。当地有这样一个说法“百里杜鹃甲天下，黄坪归来不看花”。说的就是五彩路这一段，我们百里杜鹃的第一站正是这杜鹃花最集中的五彩路。
为了赏花，景区还修了穿梭在杜鹃林间的木栈道。行走在栈道上，林间花团锦簇，虽然还未到盛开期，但是万绿丛中的那星星点点的杜鹃花和密集的花骨朵，已经可以想象出盛开期的场景，必是移步异景，如临仙境一样了。苍劲扭曲的杜鹃树干和五颜六色的花朵更是形成了极大的反差。
小姐姐都忍不住在这花丛中玩起来了~~~
举世闻名的红军二万五千里长征，贺龙、肖克率红二、六军团曾在乌蒙山区与敌周旋，景区内的“黄家坝阻击战纪念碑”贺龙铜像及新落成的红军广场，均记录着1936年2月红军长征途中在普底的战斗故事，这里也是周边学校的爱国主义教育基地。
乘着景区电瓶车，一路盘山来到醉九牛停车点。两座雕塑被杜鹃林簇拥着，定睛一看，一座由六头牛和一座由两头牛组成的雕塑在林中一左一右卧倒。咦？不是醉九牛吗，好像还少一只呢？原来还有一头牛躲在了六头牛后的林子中。
说到醉九牛还有这么一个有趣的故事：明洪武17年，奢香夫人为维护西南边陲的安宁，进贡朱元璋的贡品中有一群水西壮牛走到此地，不肯走了，它们被这色彩艳丽的杜鹃花迷住了，醉倒杜鹃花林中。于是便有了这醉九牛的地名。
站在醉九牛观景台放眼四周，那巍然屹立的黄家坝阻击战纪念碑仿佛对我们诉说着1936年初红二、六军团进军黔西北、创建黔大毕革命根据地与数倍于红军的国民党反动派在此鏖战，英勇的红军浴血奋战，一次次打退敌人的进攻，取得了黄家坝阻击战的胜利，保卫了新生的红色政权的可歌可泣的故事。还有那漫山遍野的杜鹃花林，让我们看到了一个雄奇壮美、大气磅礴的毕节。
数花峰是普底景区的制高点，因此在数花峰上修了一个形似盛开的杜鹃一样的360度观景平台。站在平台上，无论从哪个角度望去，都有一种众山环绕的感觉，此时的杜鹃还未全部盛开，放眼望去，还是红色点缀在绿树中；待到盛开期，放眼望去应该会是绿色点缀在一片红色、紫色、白色的花朵中吧。
驱车来到奢香军营山地公园，从普底过来并不算远，地处百里杜鹃“方家坪”。公园是以奢香文化为主线，结合彝族古军事展示、山地运动于一体的公园。公园有大片的草原，花开时节，各色的花点缀着路的两边，公园有骑马、滑索等体验。还在车上时就被这庞大的军营寨子的大门吸引了，门口两侧并排站着十几名铜制的士兵雕塑，威武雄壮。
百里杜鹃金坡景区
午饭后，乘车前往百里杜鹃金坡景区。百花坪是金坡景区一大看点，这里有花神奢香的巨型雕塑、祭花神主题公园、花神庙。每年杜鹃花开时节，当地就会在这里组织祭花神仪式，届时成千上万的居民和游客都会汇聚于此，参加烧香祭祀，祈求花神保佑风调雨顺，国泰民安。那个时候的百里杜鹃也是杜鹃花开得最美丽最妖娆的时候，还会评选出当年最美的杜鹃王。
相传很久很久以前，天突然破了五个洞，洪水淹没了大地，这世界上的人除了神王吱嘎阿鲁以外，其他的人都被淹死了。为了补天上的漏洞，吱嘎阿鲁用他的神鞭从很远的地方撵山来填。他的举动感动了上苍，于是上苍策举祖就派了他的女儿——漂亮的仙女玛依鲁下凡。仙女与神王结为夫妻，他们生儿育女，共同治理泛滥的洪水。当他们后代兴旺、天下归于太平时，上苍策举祖却要召回玛依鲁。玛依鲁舍不得离开丈夫和她的子孙们，但天命又不能违，为了守护着丈夫和子孙们，让他们随时都能看到她，她便化身成了杜鹃花。杜鹃花成了彝族人民心中的圣花，玛依鲁就成了彝族人民心中的“花神” 。
在“花神”雕塑的附近就是祭祀花神主题公园 。 每年杜鹃花盛开的时候，彝族人民都会选择良辰吉日在这里祭拜花神。花神玛依鲁的坛位前，千里眼在观东西南北的天象，顺风耳在听四面八方的民情，还有那一尊尊击鼓、吹长号、吹唢呐、诵经祈祷、载歌载舞的铜像雕塑，向我们展现了这里祭祀“花神”时的欢乐和热闹场面。
从古至今，在彝族的习俗中，祭祀花神是每年很重要的一项仪式，所以每年都会举行盛大的祭“花神”活动，成千上万的人汇集到这里祈求“花神”保佑风调雨顺，国泰民安，六畜兴旺，福德安康。今年祭祀花神的日子是3月28日，我们这次没能亲眼目睹那隆重和热闹的场面，不免有些许的遗憾，但从百花坪上那一尊尊栩栩如生的雕塑不难想象那隆重和热闹的场面。
杜鹃花神庙里供奉着“花神”像，每年杜鹃花开时节，当地的老百姓都会到庙里烧香祭祀祈福。花神庙旁有一座吊桥，桥的两边全被杜鹃树围绕着，虽然距离盛开期还早了一点点，但是这并不妨碍这里的美，夕阳的照耀下，小姐姐在桥上被杜鹃树环绕着。运气好还能看到鸟儿栖在枝头，或是山中的锦鸡在林间穿行。
走在两座山谷之间的吊桥上，桥两旁的杜鹃花大多含苞待放，在桥上静静地赏花、看景，无论远眺还是近观都别有一番滋味。只是不知道桥上看风景的人是不是也成了别人的风景。
在开往马缨林的电瓶车上导游讲了一个故事：相传古时候有一位美丽、勤劳、善良、聪明的彝家少女“索玛”爱上了一位英俊、勇猛、健壮的牧羊小伙“阿哲”，在那由父母决定婚姻、个人毫无自由的年代，这对少男少女私定终身，在一个月明星稀之夜选择了离家逃走。他们靠采野果、食山芋、饮山泉度日，日子虽然过得艰难，但相亲相爱的他们却感觉无比幸福。谁知好景不长，头人看上了“索玛”，为能将“索玛”占为己有，头人在采野果的路上害死了“阿哲”，逼“索玛”与他成婚。忠贞不渝的“索玛”跳下悬崖为“阿哲”徇情。被“索玛”鲜血染红的杜鹃花变成了火红的马缨杜鹃，她的灵魂也变成了美丽的杜鹃鸟。这个传说虽然太过悲凉，但火红的马缨杜鹃如团团烈火般在枝头绽放确实让人震撼。听完导游的故事，让我们更期待看到如火般团团绽放的马缨杜鹃了。
马缨林是百里杜鹃景区中马缨杜鹃最集中的地方。盛开时的马缨杜鹃花如团团烈火在枝头燃烧，远远望去犹如山林涂了胭脂，正是因此那些红色的杜鹃才被称为"映山红"。站在马缨林的景观台上，虽然没有数花峰那样居高临下，但是却是和杜鹃花的另一种亲近。由近及远的杜鹃林像是波浪一样，在眼前起伏。
马缨杜鹃树身高大，最高的可达十来米，花冠浓密，多的一株上能开二、三百朵花。由于花朵美丽、颜色鲜艳，马缨杜鹃具有较高的园艺价值。马缨杜鹃不仅有较高的欢赏价值，它的花、叶、根都可入药，彝药中就常常拿它来治疗一些疾病。
阳春三月，乍暖还寒， 夕阳西下，微风习习，置身杜鹃花树丛中，听小鸟啼唱，树枝沙沙，顿觉心清气爽，宛若世外桃源。
百里杜鹃花冠大酒店
酒店位于百里杜鹃管理区鹏程管理区花海大道游客接待中心。为4星级酒店，算是当地很不错的酒店了。房间宽敞，灯光的明暗合适，寝具没有什么特殊的味道，有折痕。卫浴也都很新，清洁得很干净，淋浴有隔水层，设计的挺周到。
配套设置也都十分齐全，配有全日餐厅、多功能厅、会议室、棋牌娱乐等。
另外要提一下的是，杜鹃花期的时候，住在花冠酒店，有直达百里杜鹃普底和金坡两个景区的摆渡车。（发车时间 周一至周五 8:00-18:00，周末及节假日7:00-19:00）
百里杜鹃花冠大酒店餐饮
花冠酒店的餐饮相比花舍来说就更加家常一些了，口味也都迎合了大多数人的口味。
推荐的菜有：
藕夹

Day2 在风车音乐节上撒个野

如果说，百里杜鹃是一个纯天然的景区，那么彝山花谷景区就是一个人工的大花园。在这里有着和百里杜鹃不一样的景致。说到毕节，大家会记得两个关键词：百里杜鹃的花海和夏季的避暑圣地。毕节夏季气温不超过30度，是避暑的理想圣地，去彝山花谷就再自然不过了。

因为花谷中植被繁多，负氧离子含量极高，含量在每立方厘米6.9万+个。我们到的时候花谷的花田正在为夏季的花海做准备，郁金香等花卉刚刚种下，夏季盛开的时候，花谷一定和世外桃源一样吧。
在爬山上玻璃栈道的时候，偶遇几棵盛开的梨树，白色的梨花衬得小姐姐也娇柔了许多。
“心”型观景平台和玻璃栈道更是不能错过的体验，胆子大的大步向前，胆子小的踌躇不前，走过去的都是好汉一条。
在植物迷宫里穿梭，永远不知道下一个转角能不能遇到对的路。花费了一些时间到达迷宫的制高点，在上面能看清迷宫里的人是如何走得晕头转向的。
结束了彝山花谷的游览后，赶来观看今年杜鹃花风车音乐节的第一场表演。当听说有音乐节的时候还是小激动了一把，以为可以看老萧，结果音乐节是从3月23号开始，3月24号，4月5，6，7号都有演出。而我们去的3月23日是罗志祥压轴，同时还有miss-mix乐队、第三区乐队、三维鱼乐队、战旗乐队、简恺乐。而信，许巍，萧敬腾，孙燕姿分别是后面几场的压轴明星。

晚餐和住宿订在了彝山花谷的花舍酒店
花舍养生度假酒店
花舍位于彝山花谷景区内，属于度假轻奢民宿，均是以木质结构为主的独栋2层小别墅为主，每栋别墅有2-4个房间不等。特别之处是，酒店园林设计很用心，整个酒店就像一个大花园一样，二层的房间推开窗户可以看到成片的杜鹃林，远处还可以看到景区的玻璃桥。
配套设置也都十分齐全，配有全日餐厅、多功能厅、会议室、SPA养生馆、茶室、棋牌娱乐室等服务项目。
花舍养生度假酒店餐饮
花舍的餐厅主要以川菜的口味为主，口味符合贵州的辣，就是放辣椒毫不吝啬的那种。比较特色的菜有：
香酥野菜，将野菜裹上面糊炸至酥脆，醮这辣椒面吃。
脆梢土豆丁：脆梢就是猪油渣，用猪油炒出来的土豆，自带香味。
推荐的菜有：
鲜椒牛仔骨
干锅虾
夫妻肺片
干椒炒牛肉末
