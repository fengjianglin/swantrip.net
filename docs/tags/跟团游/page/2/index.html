• 2019/07/12

约旦河发源于北面的叙利亚境内的赫尔蒙山，向南流经以色列，在最南端的约旦境内注入死海，全长360多公里。这条不长的河流却有一项世界之最，那就是它是世界上海拔最低的河流。在以色列境内的约旦河其水平面比海平面还要低200多米。
