贵 阳

丝娃娃 / 避暑 / 林城 / 大数据
﻿• 甲秀楼曾改过名，为“来凤阁”，清代时期经历多次重修后，又将其恢复原名。

﻿• 上有天堂，下有苏杭，气候宜人数贵阳。这里全市夏季平均气温 23.2℃，森林覆盖率高达 48.66%，名副其实的“天然氧吧”。

• 贵阳有名的小吃“湖南面”在湖南是没有的。据说是来贵阳的湖南人最先做出来的，所以叫湖南面。

• 虽然是省会城市，贵阳却是贵州省面积小的市州，仅为 8034 平方公里。

﻿• 不要看到宝山路、普陀路、南浦路、林城西路……就觉得贵阳的路名好听。还有粑粑街、黑羊巷、油榨街、猪屎巷……和“著名”的菊花洞路。

﻿• 在贵阳，90% 的人患有腰部疾病，其他 10% 的人不会打麻将。

﻿• 你只知道白天的黔灵山不分周末、工作日，经常都是人人人，其实晚上的黔灵山照样全是人人人，并且六点以后就不要门票了哦~

毕 节

花海 / 豆干 / 豆豉
• 毕节一年四季都有花海可看，油菜、杜鹃、玫瑰、玉簪、韭菜、薰衣草、腊梅……各种花，应有尽有。

• 毕节是三省红都，长江以南最后一块革命根据地，其厚重历史文化，在全国都具有唯一性。

• 大多数人只知道白天的百里杜鹃可以看花，却不知道晚上还能夜游~

• 毕节彝学研究院，是中国唯一的彝学研究院。

• 你以为毕节只有一种“豆干”吗？毕节还有糍粑豆干，手撕豆干、荞灰豆干、臭豆干……

• 贵州省唯一市区天然瀑布——响水滩，就在毕节母亲河——倒天河上！

黔 东 南

苗乡侗寨 / 红酸汤 / 绝美小镇

• 岜沙苗寨是全国唯一一个用镰刀剃头的地方。

• 2008年的奥运圣火曾经过朗德苗寨，这里正是清咸丰、同治年间苗族起义领袖杨大六的出生地。

• 中央电视台鉴宝栏目曾鉴评摆贝苗寨的百鸟衣，一件达 60 万元。

• 黔东南革家族自称是后羿的后人。

• 余秋雨曾到黔东南做过长达 6 天的考察，事后，他感慨：在这里看到了一整套非常完整而系统的精神哲学。

• 传说有龙神居住过镇远铁溪龙潭，龙潭也是因此而得名。

六 盘 水

凉都 / 羊肉粉 / 烙锅 / 滑雪场
• 你以为六盘水只有梅花山滑雪场？不，这里还有玉舍滑雪场、野玉海滑雪场、盘州滑雪场~

• 六盘水梭戛苗寨（又称梭嘎苗寨），是亚洲第一座生态博物馆，寨子民族文化深厚，堪称人类活化石。

• 堪称“西部一绝”的水城烙锅始于清代，至今有 300 多年历史。

• 被称为“中国凉都”的六盘水，是全国唯一一个以气候特征命名的城市。

• 贵州湿地公园很多，但知道明湖是省内第一个获得正式授牌的国家湿地公园的人肯定很少。

遵 义

红色景点 / 虾子羊肉粉 / 鸡蛋糕 / 茅台

• 虾子羊肉粉里面并没有虾子，而是因为这个粉源于遵义虾子镇。

• 道真县是遵义乃至贵州仅有的一个以人名命名的县，尹珍（字道真）是正安（今道真）人，是贵州最早见诸文字，最先走出大山、叩问中原文化的著名儒学者，曾任尚书承郎、荆州刺史等职。为纪念汉代学者尹珍（字道真），将县名定为“道真”。

• 遵义有 30762 平方公里，相当于 937.9 个澳门，113.2 个台北，27.8 个香港。

• 国内唯一旅游洞穴宿营地，就在遵义市绥阳县十二背后景区。

• 遵义新舟机场曾是中国西南地区重要的大型军用机场，后改为战备机场，现为 4C 级民用航空机场。

铜 仁

米豆腐 / 苗文化 / 空心李

• 梵净山是中国唯一上榜美国《国家地理》杂志全球“最值得到访的 28 个旅游目的地”的旅游地。

• 2018年央视春晚贵州分会场，肇兴 5 分钟惊艳全世界。当时场上有一条飞舞的巨龙，这巨龙就是来自于铜仁松桃寨英镇大水沟村的滚龙。

• 铜仁松桃拥有三个超大型锰矿床，其中，普觉锰矿（整合）超大型锰矿床新探明锰矿资源量 1.92 亿吨，堪称亚洲第一！

• 玄武岛——贵州唯一个取名“岛”的地方，在铜仁市万山区鱼塘乡。

• 界牌苗寨不仅是传说中除贵州西江千户苗寨之外面积较大的原生态木楼苗寨之一，还是苗族战斗英雄吴良保的故乡。

• 铜仁玉屏县是贵州省最小的县。

黔 西 南

峰林 / 糯米饭 / 兴义羊肉粉

• 去万峰林景区内是不要门票的，想看到景区全景图才需要买票乘坐观光车沿途欣赏。

• 著名的政治家、教育家、中国同盟会先驱、孙中山先生的追随者、国民政府交通部长——王伯群，就是黔西南州兴义人。

• 黔西南州还叫中国金州，因为这里黄金分布广、储量大、品质高。

• 安龙一中校园内的兴义府试院，是清道光二十二年（1842年）由兴义知府张锳（张之洞之父）主持建成。

• 全贵州，只有兴义万峰林的蛋炒饭炒好后是用“锅”或者“塑料盆”端上桌的，而不是用碗或盘子。

安 顺

瀑布 / 裹卷 / 蜡染


• 安顺蜡染被称为“东方第一染”，早已进入第二批国家级非物质文化遗产保护名录。


• 中国乃至全亚洲的“最后洞穴部落”在安顺紫云，它叫——中洞苗寨。


• 安顺市的旧称原为“普定”，这一名称最早出现于元朝，直到清康熙 10 年，才由“安顺”取代“普定”。而普定今属安顺境内。


• 格凸河曾经有一种职业叫“背尸”，如今这些背尸人成为了攀岩表演者。


• 紫云县有一个神秘的村寨——巴身小。“身小”是一个字，在字典中完全查不到，当地人把它读作“làng”。

黔 南

盐酸 / 都匀毛尖 / 天眼

• 世界上最长的年节是黔南水族的“端节”，从农历八月到十月。

• 三都县姑鲁寨的登赶山上，有一面崖会生蛋，叫“产蛋崖”，距今共“产”蛋 60 多枚。

• 福泉以前叫“福泉县”，九六年才设立的县级福泉市。

• 平塘的“中国天眼”相当于 30 个足球场那么大！

• 黔南州建州 60 周年的时候，卡蒲乡 98 名毛南族妇女在 3 个月内绣出了一个“世界最大的虎头帽”。

贵 安 新 区

大数据 / 区域示范基地 / 万亩樱花

• 建在贵安新区的苹果公司主数据中心，投资达 10 亿美元，相当于人民币 60 亿！

• 很多人都知道花溪大学城，却不知道花溪大学城其实归属贵安新区。

• 贵安新区是国务院批准的第八个国家级新区！

• 今年是贵安新区成立的第 4 个年头。

• 在贵安，有个洞叫“大花洞”，是早年村民们为躲避山匪而修建的。
