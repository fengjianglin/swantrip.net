【天鹅之旅首发】最美时节，邂逅烟雨无锡！

Posted on 2017年3月31日 by 天鹅之旅
前言 三月，江南的杨柳吐出了新绿，桃花诗意了园林。在中下旬，最浪漫的樱花也抢闸盛放！ 为赏春，我来到了无锡，来到了太湖鼋头渚…… 鼋头渚 关于无锡 无锡，简称“锡”，古称梁溪、金匮，江苏省地级市，被誉为“太湖明珠”。无锡位于长江三角洲平原腹地，江苏南部，太湖流域，京杭大运河从中穿过。无锡北倚长江，南濒太湖，东接苏州，西连常州，构成苏锡常都市圈，是长江经济带、长江三角洲城市群的重要城市。无锡是国家历史文化名城，自古就是鱼米之乡，素有布码头、钱码头、窑码头、丝都、米市之称。无锡是中国民族工业和乡镇工业的摇篮，是苏南模式的发祥地。

Posted on 2017年3月31日 by 天鹅之旅
一、【序言】 对于上海，还是比较陌生的，之前一直以为魔都是个大都市，从来不喜欢大都市，所以都很少去。自从去年上海迪士尼开业之后，对于我这个没有童年的孩子，至少说小时候从来没去过游乐园，更不知道什么是摩天轮，旋转木马。恨不得分分钟钟飞到魔都去实现童年的迪士尼梦想。刚好接到携程的邀请，参加2016年中国CTF旅行者大会，刚好也在上海举办，立马答应了。从大学开始自助旅行，直到去年辞职专职旅行，成了全职的旅行自媒体，在外人看来真的很风光，可以到处走，到处看不同的风景，可是风光的背后，只有自己才清楚的辛苦，很荣幸在大会上获得携程2016年年度优秀签约旅行家，真的是对我这一年以来最大的肯定了，真的真的当时都感动的哭出来了。忙到现在才有时间来整理这段上海之旅，给我留下了深刻的记忆。
