桐庐的石舍和金华的浦江其实是隔壁邻居。去石舍的慢生活体验区这条山路修得很不错，其中一条大峡山隧道长达三公里，可见山有多深，由于山的阻隔，雾霾基本进不去，那边只有蓝天白云。当我们桐庐地区探索得差不多了，就开始往更远的浦江辐射出去了。 比如前年冬天，下完雪，我们去爬的马岭。

假装坡很陡

假装堆了个大雪人
马岭上面有个美女峰。正常情况，车子开到山腰，才开始登顶美女峰。去年去了三次，没有一次登顶，前面两次雪太厚，车不能上山，我们只能徒步前往山腰处，没等到山腰，我们已经湿了鞋，太阳也开始落山了，只能打道回府。第三次，雪化了，我们又去登顶，结果山路太泥泞，走了一半放弃了。我说：急什么？反正美女就在那儿又不走，什么时候去看都行~
眼前就是马岭美女峰
仔细看！有没有发现山顶上有一块孤独的小石头？那就是美女峰了。虽然我一直无法看出哪儿有美女。听浦江人说，从浦江这边望过去，就是美女的样子，所以，一直以来，我们看的只是美女的后背么？

终于，第四次登顶美女峰，那块神奇的大石头，像是飞来石。浦江马岭美女峰，不需要门票，山间小路，自带水果零食，随时随地来个呼吸新鲜空气的半日游。

我家男掌柜问：亲子乐园，我带你玩合适么？我一脸奸笑，心里想：有什么不合适的呀？又不是一两次被人家误会是父女~男掌柜朝我一声冷笑，一个大转弯，把我甩歪了，报复心理太强了！哼~
本来以为桐庐境内公路都修得不错，出了桐庐就不行。因为我们每次去富阳龙门搬气泡米酒的时候，富阳路段大坑套小坑，颠簸一路，酒都撒在车斗里，去一次就要洗一次车。但从富春江慢生活区一直开过来，都是山路，但路都不错。一路风景也特别好，灰尘也很少。骑摩托车最怕路烂灰尘多，在山区里的一条公路，附近散落着大大小小的自然村落，又没有工业污染，就是闲逛都觉得舒服。
下一站：江南第一家
离桐庐市区70公里。镇上走走不需要门票，有三个室内景点需要买一张联票，一般旅游团会带游客去参观。车子停在门口，虽然第一次来，但突然感觉这地方似曾相识，好像琼瑶剧里面的某个场景。
一连串的牌楼，每个都不一样。霸气极了~
我喜欢这个写着有序的斗拱木结构牌楼。
糖画，小时候没少吃。那时候一毛钱可以转一次转盘，大家最喜欢转到龙。因为图案复杂，用的糖比较多。但大多数时候，都是舍不得吃的，实在忍不住了才把龙须咬掉一点点，除非拿在手里磕到了哪儿，半个龙掉在地上了，只能心疼得把剩下的半条龙咬碎了吃完，吃完以后会觉得异常的轻松，又可以放手去玩了，这就是甜蜜的负担吧~
这鸡有点丑，但好在就算磕坏了，还在袋子里，这点比以前人性化了不少呢~
古色古香的村子，一条小溪贯穿，两岸住着村民。溪水清澈见底。
街边的铺子也古色古香的。
浦江郑宅镇的郑义门，号称“江南第一家”。900年历史，至今依然生生不息。
她的传奇故事在于：其一，这一家族十五代同居生活，300年不分家，鼎盛时期郑家3000余人同吃一锅饭；其二，郑义门173人为官，官位最高者位居礼部尚书，却没有一人因贪墨而遭罢官。
正月十五，村里都会舞龙灯。这里舞的龙叫板凳龙。我们在桐庐的茆坪村也见过板凳龙，但是在白天看的，没见过灯点起来的样子。跟村民打听，天黑了 以后板凳龙就会在这个牌楼前的广场上表演。
这个牌楼是2008年新造的，听说郑氏后裔遍布很多地方，有一支在韩国混得不错。每年的二月初八是郑氏祖宗生日，国外的子孙们都会赶回来摆祭。郑氏家训非常有名，大家可以上百度搜一下，作为一个郑氏子孙，有一种强烈的归属感和家族荣誉感，所以斥资100万也是很正常啦~
天色渐暗，这时候的郑义门显得特别高大雄伟。
一群摄影爱好者已经在广场上有模有样的摆好三脚架，抢好了机位。我们不紧不慢的在小溪边剥着橘子等敲第三遍锣。是的，必须打锣三遍，才能召唤板凳龙，这就是一种仪式感。
差不多6点左右，华灯初上，锣打了第三遍。因为板凳龙点灯，所以在晚上会更好看。
先是20多位村里的长者提着灯笼走了出来，他们是专门为龙带路的。然后一条龙从广场中间路过。。。。龙先要在村里和祠堂前绕一圈再回到广场上来。
在郑义门的柱子中间来回穿梭。数不清到底有几节板凳，反正非常非常多，真的有种神龙见头不见尾的感觉。
用tv模式拍到一张清晰的，整条龙在广场折了5个来回。这还没拍全。村民说，每家出一个人舞龙的，具体多少没数过，但少说也有150条板凳。当时围观群众也非常多，围得水泄不通。那些一早就在广场上抢机位的摄影爱好者们，哼哼~可以想象得出，你们今晚也拍不出什么像样的片。哪像我们这种老司机，早就和村民搞好关系，到人家二楼占据有利地形了~哦也~
龙头长这样，一边走还一边不停上下摇摆，摇晃着铃铛发出响声。需要3-5个年轻力壮的男人一起抬着。别说是舞起来，就是抬着走一圈已经非常累了吧。听说，舞龙之前都要先拜拜祖先，只要够虔诚，祖先自然会保佑，哪怕跌到都不会有伤。
虽然看不出是龙头了，但线条还是挺美的呀~村里一位长者对我们说，村里有个习惯，每10年要换一次灯。每次附近村子会有十几条各式各样的龙灯汇聚到这儿来，非常热闹的。下一次换龙灯是2026年。敢情去年刚换过啊~我们是没赶上趟呗~不要紧，我们可以等！
晚上了，门口的牌楼也都灯火辉煌呀~比白天更耀眼。
正月十五过完了，这个年也算是完美结束了。我们俩开着侉子吹着冷风回到桐庐的家里已经是九点多。虽然到家又冷又困，但心情还是很不错的不是么，如果我们还在魔都生活，上哪儿去看这么原汁原味的民俗啊是吧~
前面说到了二月初八郑氏家族摆祭，时间是还没到，但杭坪镇的摆祭就在正月初十九。也就相隔没几天，我们又一次驱车来到浦江。感觉浦江的人文比桐庐多多了。
每年这个时候，来杭坪镇参加摆祭的人和车多到需要交通管制，就能看出来两点，杭坪镇是个大村子，摆祭的名气也非常响，不仅仅是村里人，还有很多外来人前来祈福的。
讲真，虽然我家侉子开起来风很大，冬天坐还挺冷的，但真的很拉风，并且一路畅通，交通管制也对我们网开一面，让我们一路开到村子里，而不是和别人一样，走着进去。也有可能我们带着狗，看起来更像是回家的样子。
杭坪镇的摆祭是浙江省的非物质文化遗产。 每年正月十九摆祭仪式由杭坪村的六个社轮流主持。 迎送关公是杭坪村正月十九“摆祭”中最为热闹隆重的，每年正月十九日上午8时，由当年轮流到的主持社召集6个社的人在广场集中后，去离村1公里的关公庙迎接关公。

讲真，因为人实在太多了，好不容易挤到供桌旁边拍了两张不模糊的。大家凑合看吧~远处的正是关公。第二天大概也是八点左右又要大张旗鼓、吹吹打打滴送关公回关帝庙。届时会给参加的每个人发放馒头，寓意着沾点吉祥喜气。所以，第二天来的人会更多更热闹。经常是洋洋洒洒好几千人，一路跟着到关公庙。
村口的戏台已经坐满了人，虽然戏还没开始演。浦江这边喜欢看婺剧。屏幕上已经打出了剧目名称。反正我们也听不懂婺剧，那就早点走吧。
我敢保证，他们不知道两个拉在一起是什么意思~
下一站：薛下庄，林间小道。
只是在路上多看了你一眼，突然发现有个小公园，还有沙滩和划船。调头下去看看呗，反正天气这么好，也没啥事儿。
第一次走在沙滩上的哈叽，只能用疯一样的狗子来形容，开心得已经不知道怎么办了。所以狗还是要多带出去玩玩，整天在家在小区里玩也是憋屈。
小溪比较宽，像小河了。停着几只小船。
20元一次。小孩最喜欢了。
青山绿水间一条水坝上几个年轻人在搞烧烤趴。阵阵香味传过来，真是的，还让不让人好好的散步？
真山真水中的小村庄，建设得比高档小区还高大上。
你有本事以后拍照一直这么斜着，倒也算是个特色。哈叽表示：我只是路过抢镜。
村后山上一条步行道。毛竹都舍不得砍，很有味道的生态小道。
走了十多分钟，根本看不到尽头，哈叽宝宝已经拖着大舌头，表示好累嗷，喝水喝了好几次，都没缓解它喉咙里发出重重的喘息声，我们打算回头吧，下次再来。组织几个小伙伴，我们也搞个烧烤趴，吃饱了再上山消消食。

席场桥，一座看起来就比较古老的廊桥。 壶源江第一桥。是浦江地区唯一一座廊桥。让我惊讶的竟然是这条不是小溪，ta叫壶源江！
关于席场桥，还有一个传说：相传建桥以前，壶源江虞宅附近没有木桥，行人来往十分不便。南宋宝庆年间，当时虞宅村叫毛宅村，村内有个员外，姓毛名日严，家道殷富，乐善好施。于宝庆元年正月十三请来一批员外共商造桥大计，并在自家厅堂里办了几桌酒席，酒席上大家商谈建桥事宜，并把桥命名为“席场桥”。
　　席场桥曾建了三次，前面两次都被叫做“篾缆精”的妖怪毁掉。日严公两次建桥失败，有些心灰意冷。一天夜里，日严公梦见观音菩萨对他说：“篾缆精已被我收服在富春江江底，再也不会出来害人。5000两银子放在你厅堂里，你要尽快把桥修好。”日严公披衣到厅堂一看，果然有一只金光闪闪的箱子放在那里，立即打开箱子，里面装满了白花花的银子。清点了一下，刚好是5000两银子。
所以桥上供奉的正是观音。写着慈航普渡。
看到这里，亲们是不是也想来一场说走就走的金华游了？实际上金华还很大，这些只是这条公路沿路可以游玩的几个景点而已。还有很多山清水秀的好玩地方我们没去到呐~我们想开发一条自驾游路线，让杭州上海的小伙伴们到桐庐玩了几天以后，一路往浦江玩下去。舒舒服服的，也不是一直在开车，消费也都很合理。在城里憋屈坏了的小伙伴，带着你的熊孩子和熊狗子出来玩吧~
别以为你们吃饭的时候把我放在凳子上，我就真的下不来，我只是给你们一点面子~哼~
现在我再问你金华有什么？脑海里搜索到的关键词不会只有火腿了是不是？如果还是满脑子只有火腿，那么我只能说，你是个吃货！
